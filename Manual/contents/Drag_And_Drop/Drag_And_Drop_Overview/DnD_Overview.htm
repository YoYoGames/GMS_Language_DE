Drag And Drop™ Übersicht

Drag and Drop™(DnD™) ist ein visuelles Skripting-Tool, mit dem Sie Ihre Spiele erstellen können, ohne tatsächlich Code einzugeben. Das soll nicht heißen, dass Sie keinen Code programmieren, wenn Sie DnD™ verwenden, denndas tun Sie ja, nur dass Ihr Code auf visuelle Weise mit "Blöcken" und nicht mit Text erstellt und präsentiert wird. in einer visuellen Art und Weise unter Verwendung von "Blöcken" von Aktionen anstelle von Text. Aktionen sind einfache AusdrückeoderAnweisungen, die miteinander "verkettet" werden können, um eine Instanz eines Objekts etwas tun zu lassen, wenn sie in einem Raum platziert wird. Sie könnten zum Beispiel eine Aktion in einem "Create Event" haben, um eine Instanz in eine zufällige Richtung zu bewegen eine Instanz in eine zufällige Richtung zu bewegen, und eine andere Aktion in einem Kollisionsereignis, um sie an den Dingen abprallen zu lassen, mit denen sie kollidiert (weitere Informationen zu den verschiedenen verfügbaren Ereignissen und wie sie zusammenarbeiten, finden Sie im Abschnitt über Objekt-Ereignisse).

Das obige Bild zeigt ein typisches Objekt, das auf einem Arbeitsbereich geöffnet ist und bearbeitet werden. Es hat ein "Ereignis"-Fenster und ein "Code"-Fenster, die mit ihm verkettet sind, und im Code-Fenster können Sie sehen, wo wir unser DnD™ aufgebaut haben. Nachfolgend finden Sie weitere Informationen zu jedem der dargestellten Bereiche:

Ereignis-Registerkarten

Wenn Sie Ihrem Objekt ein Event hinzufügen, wird eine neue Registerkarte im Codefenster geöffnet, um den Inhalt dieses Events anzuzeigen. Sie können mehrere Ereignisse hinzufügen und mehrere Registerkarten gleichzeitig geöffnet haben, zwischen ihnen wechseln und sie bearbeiten wie Sie möchten. Sie können die Registerkarten auch jederzeit schließen, um Unordnung zu vermeiden (durch erneutes Auswählen des Events im Objekt werden sie wieder geöffnet), und Sie können auf klicken und die Registerkarte aus dem Fenster in ein eigenes Codefenster ziehen, wobei das Schließen des Objekts oder des Codefensters das neue Fenster nicht schließt. Registerkarten können auch nach links oder rechts gezogen werden, um neu sortiert zu werden.

Aktion Übersicht

Wenn Sie Ihrem Ereignis Aktionsblöcke hinzufügen, werden diese in "Kurzform" als einfache englische Übersicht über die Vorgänge angezeigt. Die Übersichtsreihenfolge ist die Reihenfolge, in der die einzelnen Aktionsblöcke für dieses Ereignis ausgeführt werden, und Sie können auf auf ein beliebiges Element in der Liste klicken, um die Ansicht der Aktionsblöcke zur Bearbeitung an diese Position zu zoomen.

Aktionsblock Arbeitsbereich

In diesen Arbeitsbereich ziehen Sie Aktionen aus der Toolbox , um Ihren DnD™-Codezu erstellen. Klicken Sie auf auf eine Aktion und ziehen Sie sie in diesen Bereich, um sie der Liste der Aktionen hinzuzufügen und sie zu bearbeiten. Die meisten Aktionen haben verschiedeneArgumente (Parameter), die Sie setzen können, und die meisten nehmen auch Schlüsselwörter, Instanzvariablen oder sogar GML-Funktionen - zum Beispiel können Sie die Aktion " Richtung festlegen" aufrufen und die GML-Funktion random(val) verwenden, um eine zufällige Richtung zu setzen.

Wenn Sie Ihrem Code Aktionsblöcke hinzufügen, werden diese visuell "verkettet", um die Reihenfolge anzugeben, in der die verschiedenen Aktionen ausgeführt werden, wobei die Art und Weise, wie diese Kette erstellt wird, vom verwendeten Ausgangsblock abhängt. Ein Beispiel, ein If-Variable-Block zwei Ketten haben - eine zu den Blöcken, die aufgerufen werden sollen, wenn die überprüfte Bedingung true ist, und dann der nächste Block danach den. Weitere Informationen zum Konstruieren von Aktionsblöcken in Code finden Sie in den unten verlinkten Abschnitten.

Während Sie sich im Arbeitsbereich des Aktionsblocks befinden, können Sie die mittlere Maustaste drücken und gedrückt halten, um die Ansicht zu schwenken, und das Mausrad , um die Aktionsliste nach oben oder unten zu scrollen. Wenn Sie gedrückt halten, während Sie das Mausrad bewegen, dann wird die Ansicht nach links/rechts gescrollt, und wenn Sie / gedrückt halten und das Mausrad bewegen dann wird die Ansicht vergrößert und verkleinert, so dass Sie mehr vom Arbeitsbereich mit den Aktionen sehen können. Sie können auf auf eine beliebige Aktion aus der Aktionsliste auf der linken Seite klicken, um den Fokus auf diese Aktion zu setzen, wodurch der Zoom und die Position zurückgesetzt werden. Beachten Sie auch, dass Sie die Größe der Aktionsblöcke ändern können, indem Sie die Maus zu den Rändern des Blocks bewegen und dann - wenn sich der Cursor ändert - in die gewünschte Richtung ziehen.

Sie können auch mit der rechten Maustaste auf irgendwo im DnD™-Arbeitsbereichklicken, um das RMB-Menü aufzurufen, das hier ausführlich erklärt wird.

Werkzeugkasten

In der Toolbox werden alle DnD™-Aktionengespeichert, wobei verschiedene Abschnitte Sammlungen (Bibliotheken) von Aktionen enthalten, die ähnlich oder verwandt sind. Jede Sammlung von Aktionen ist auf ein bestimmtes Thema oder einen gemeinsamen Zweck bezogen und Sie klicken einfach auf und ziehen die gewünschte Aktion aus der Toolbox in den Arbeitsbereich des Aktionsblocks, um sie dem aktuellen Ereignis hinzuzufügen. Um die Dinge einfacher zu machen - da es eine Menge Aktionen gibt - können Sie die "Such"-Leiste oben verwenden, um nach einer bestimmten Aktion zu suchen oder die sichtbaren Aktionen zu filtern (geben Sie z. B. "Zeichnen" ein, um alle Zeichenaktionen zu erhalten), und Sie können auch auf klicken und Symbole aus einer beliebigen Bibliothek in Ihren Favoriten-Ordner ziehen, um häufig verwendete Aktionen zusammenzuhalten. Dies ist besonders nützlich wenn es bestimmte Dinge gibt, die Sie immer wieder machen werden. Der Favoriten-Ordner kann durch Klicken auf das kleine "Mülleimer"-Symbol geleert werden.

Eine vollständige Liste der verfügbaren Aktionen sowie der Argumente, die sie annehmen und wie sie verkettet werden können, finden Sie im Abschnitt Drag-and-Drop-Referenz.

Sie können mit der rechten Maustaste auf eine beliebige Aktion in der Toolbox klicken, um das folgende Menü zu öffnen:

Hier haben Sie die folgenden Optionen:

  • Verwenden Sie<Aktion> : Damit wird die Aktion, die sich unter der Maus befand, als die rechte Maustaste geklickt wurde, dem aktuellen Ereignisarbeitsbereich hinzugefügt.
  • Favorit erstellen/entfernen: Damit wird die Aktion zur Favoriten-Bibliothek oben in der Toolbox hinzugefügt oder entfernt. In dieser Bibliothek können Sie eine benutzerdefinierte Gruppe von Aktionen erstellen, die Sie regelmäßig verwenden. Dadurch wird der Zeitaufwand für die Suche nach diesen Aktionen in den eigentlichen Bibliotheken reduziert.
  • Favoriten löschen: Wenn Sie dies auswählen, werden alle Aktionen aus der Favoritenbibliothek entfernt.
  • Kürzlich verwendet löschen: Wenn Sie dies auswählen, werden alle Aktionen aus der Bibliothek "Kürzlich verwendet" entfernt.
  • Alle einklappen: Mit dieser Option werden alle DnD™-Bibliothekeneingeklappt, sodass nur die Bibliothekstitel sichtbar sind, nicht aber die Aktionsknoten. Jede Bibliothek kann wieder aufgeklappt werden, indem Sie auf sie klicken.
