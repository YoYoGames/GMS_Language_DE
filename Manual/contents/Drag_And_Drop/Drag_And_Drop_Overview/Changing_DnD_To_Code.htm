Ändern von DnD™in Code

Ein weiteres Merkmal von DnD™ist, dass Sie es in GML-Code(GameMaker Language) umwandeln können und auf diese Weise sehen, was genau hinter den Kulissen passiert, wenn Sie Aktionen verwenden. Bevor Sie die Aktionen jedoch tatsächlich in GML umwandeln, können Sie zunächst eine Vorschau mit Hilfe der rechten Maustaste Menüoption Live-Vorschau:

Dadurch wird ein neues Fenster geöffnet:

Wenn Sie DnD™-Aktionenhinzufügen, ändern oder entfernen, wird die GML-Live-Vorschau aktualisiert, um Ihnen den tatsächlichen Code zu zeigen, der der "hinter den Kulissen" erstellt wird. Der Code in der Live-Vorschau kann nicht direkt bearbeitet werden, aber Sie können Abschnitte auswählen und zum Einfügen in GML-Skripte oder Code-Aktionen (z. B.) kopieren.

Wenn Sie sich entscheiden, dass Sie das DnD™in Code umwandeln möchten, nachdem Sie die Vorschau gesehen haben, dann können Sie dies erneut tun, indem Sie mit der rechten Maustaste in einem einem beliebigen Event-Arbeitsbereich mit Aktionen klicken und In GML konvertieren wählen.

Wenn Sie dies zum ersten Mal tun, erhalten Sie eine Warnmeldung, die besagt, dass dies eine einseitige Konvertierung ist, da Sie Sie können Aktionen in Code umwandeln, aber Sie können sie später nicht wieder in einzelne Aktionen zurückwandeln. Wenn Sie hier auf "okay" klicken, wird die Konvertierung für Sie durchgeführt.

Der resultierende Code verwendet {}, um einzelne Aktionen abzugrenzen, und Sie können klar erkennen, welche Aktionen sich auf welche Funktionen oder Variablendeklarationen innerhalb des Codes beziehen. Wenn das DnD™komplexer ist, wird auch der Code komplexer, aber es gelten dieselben allgemeinen Regeln, und der Code wird sequentiell genau so aufgebaut, wie Sie das DnD™geschrieben haben. Wenn Sie lernen, mit Drag and Drop zu programmieren, kann dies ein wichtiges Hilfsmittel sein, um später mit GML weiterzumachen, aber es ist keineswegs obligatorisch und Sie können immer noch großartige Spiele mit DnD™ machen !Es ist auch erwähnenswert, dass die Konvertierung zwar Prozess eine Möglichkeit ist, nach der Konvertierung von Aktionen in Code wieder zu DnD™zurückzukehren, indem man das Rechtsklickmenü im Code-Editor benutzt und "In DnD™ konvertieren " wählen .
