Menüoptionen der rechten Maustaste

Wenn Sie an Ihrem DnD™-Codearbeiten, können Sie mit der rechten Maustaste auf die Aktion, auf die Arbeitsfläche oder auf den Text in der Auswahlliste links klicken klicken, um das Menü der rechten Maustaste zu öffnen:

Dieses Menü bietet die folgenden Optionen:

  • Ausschneiden / Kopieren / Einfügen: Damit können Sie jede Aktion, die im Arbeitsbereich ausgewählt ist, ausschneiden, kopieren oder einfügen. Beachten Sie, dass Sie + außerhalb der Aktionen verwenden und ziehen können, oder / + auf einzelne Aktionen anwenden können, um mehr als eine auszuwählen und den Befehl auf alle anzuwenden.
  • Kommentar / Kommentar entfernen: Mit dieser Option können Sie Kommentare zu Ihren DnD™-Codeblöckenhinzufügen und entfernen. Wenn Sie Kommentar wählen, wird rechts neben dem ausgewählten DnD™-Blockein Texteditor geöffnet, und Sie können dort einen beliebigen Text hinzufügen, sei es eine Erinnerung, etwas zu ändern, oder ein hilfreicher Kommentar für ein Tutorial oder sogar Anweisungen für Ihre Teamkollegen:
    Sobald Sie den Kommentar hinzugefügt haben, können Sie ihn anklicken und ziehen, um ihn vertikal und horizontal zu positionieren - allerdings Wenn Sie den DnD™-Knoteneinklappen, wird auch der Kommentar eingeklappt und eine Zusammenfassung seines Inhalts angezeigt (in diesem Fall kann er nicht neu positioniert werden). Wenn der Kommentar nicht mehr benötigt wird, können Sie entweder den gesamten den gesamten Text im Kommentar löschen oder die Option " Kommentar entfernen " aus dem Menü verwenden. Beachten Sie, dass beim Kopieren eines Knotens der Kommentar mitkopiert wird, und wenn Sie den Knoten von DnD™nach GML konvertieren, bleibt der Kommentar erhalten und dem erstellten GML hinzugefügt, wie im folgenden Bild zu sehen ist:
    Wie bereits erwähnt, kann ein Kommentar nur auf einen einzelnen Knoten angewendet werden, aber Sie können ihn auf mehrere Sie können ihn jedoch auf mehrere Knoten ausdehnen, indem Sie einfach Zeilenumbrüche einfügen, um ihn zu erweitern - auch wenn Sie dabei den Kommentar möglicherweise neu positionieren müssen, um Überschneidungen mit nachfolgenden Knoten zu vermeiden:
    Sobald ein Kommentar hinzugefügt wurde, können Sie die Option "Kommentar" erneut aus dem RMB-Menü auswählen oder durch einen Doppelklick auf die Bearbeitung aktivieren.
  • Deaktivieren / Aktivieren: Mit diesen Befehlen können Sie die ausgewählte Aktion (oder Aktionen) deaktivieren oder aktivieren. Wenn eine Aktion deaktiviert ist, ist sie weiterhin im Arbeitsbereich sichtbar, wird aber nicht ausgeführt, wenn Sie das Spiel kompilieren. Auf diese Weise können Sie Aktionen zum Testen und Debuggen hinzufügen/entfernen. Das folgende Bild zeigt einen DnD™-Codemit deaktivierten Aktionen:
    Sie können sehen, dass die Aktionen "If Score" und "Go To Next Room" deaktiviert wurden - sie wurden visuell ausgegraut - und werden daher beim Testen oder Kompilieren des Spiels übersprungen.
  • Haltepunkt umschalten: Mit diesem Befehl schalten Sie einen Haltepunkt für die ausgewählte Aktion ein oder aus. Ein Haltepunkt ist einfach ein Punkt im DnD™-Code, an dem der Debugger anhalten soll, wenn das Spiel im Debug-Modus ausgeführt wird. Eine Aktion, der ein Haltepunkt zugewiesen wurde, wird hervorgehoben, wie im Bild unten gezeigt (die Aktion "Wenn Gesundheit" wurde als Haltepunkt festgelegt):
  • Alle Haltepunkte löschen: Mit diesem Befehl können Sie alle Haltepunkte aus dem DnD™-Codeentfernen.
  • Alle einklappen / Alle ausklappen: DnD™-Aktionenkönnen entweder eingeklappt oder ausgeklappt werden, wobei eine eingeklappte Aktion nur den Namen der Aktion, aber keine der internen Werte anzeigt, um den verfügbaren Platz für den Arbeitsbereich. Beispiel:
    Wie Sie sehen können, nehmen die Aktionen viel weniger Platz ein, und Sie können immer noch anhand der kurzen Beschreibung rechts sehen, was sie tun. Beschreibung auf der rechten Seite. Sie können die Option "Expand All" wählen, um dies für jeden Knoten im Arbeitsbereich rückgängig zu machen.
  • Layout zurücksetzen: Dadurch wird der DnD™-Arbeitsbereichzurückgesetzt, die Zoomstufe auf 100 % gesetzt, der Fokus auf die aktuell ausgewählte Aktion gesetzt und alle eingeklappten Aktionen aufgeklappt (und alle anderen).
  • Live-Vorschau: Wenn Sie diese Option wählen, aktivieren oder deaktivieren Sie das Code-Vorschaufenster von DnD™zu GML(GameMaker Language). Dies wird auf der Seite Ändern von DnD™ in Code näher erläutert.
  • In GML konvertieren: Damit werden alle Ihre Aktionen im aktuellen Event in GML-Code(GameMaker Language) konvertiert. Seien Sie dabei sehr vorsichtig, da Sie GML nicht zurück in die DnD™-Aktionen konvertieren können, aus denen es erstellt wurde. Sie können das Ereignis wieder in ein DnD™-Ereignisaus Code umwandeln, aber jeder Code wird einfach in eine Aktion "Code ausführen " verpackt. Dies wird auf der Seite Ändern von DnD™ in Code näher erläutert.
  • Fenster: Dieser Bereich öffnet das folgende Untermenü:
    . Die hier verfügbaren Menüpunkte sind:
    • Zum Objekt gehen: Damit wird der Arbeitsbereich auf das Objekt verschoben, das das Ereignis aufruft
    • Geerbtes Ereignis öffnen: Diese Option ist nur verfügbar, wenn das bearbeitete Objekt ein untergeordnetes Objekt eines übergeordneten Objekts ist, und öffnet das bearbeitete Ereignis des übergeordneten Objekts und verschiebt den Arbeitsbereich dorthin.
    • Einzelner Bereich / 2 Spalten / 3 Spalten: Diese Optionen können verwendet werden, um den aktuellen Arbeitsbereich in 1, 2 oder 3 verschiedene "Fenster" aufzuteilen. Jeder Bereich ist ein separater Arbeitsbereich, auf den Skripte oder Ereignisse auf sie gezogen werden können, damit Sie 2 oder mehr DnD™-Codeblöckegleichzeitig bearbeiten und vergleichen können. Das folgende Bild zeigt zwei Skripts in einem Fenster, die in einem eigenen Arbeitsbereich angezeigt werden:

Die folgenden beiden Optionen sind nur verfügbar, wenn das RMB-Menü auf einem DnD™-Knotengeöffnet ist:

  • Schwenken: Wenn Sie diese Option auf einem beliebigen Knoten auswählen, wird der Arbeitsbereich so geschwenkt, dass dieser Knoten in der Mitte liegt.
  • Hilfe anzeigen... : Wenn Sie dies auswählen, wird das Handbuch in einem neuen Arbeitsbereich auf der Seite geöffnet, die für den ausgewählten Knoten relevant ist. Sie können das gleiche Ergebnis auch mit auf dem Knoten erreichen.
