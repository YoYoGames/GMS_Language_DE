Zustand der Zeitleiste einstellen

Mit dieser Aktion können Sie eine Zeitleiste starten, stoppen, pausieren oder fortsetzen. Die Zeitleiste muss zuerst mit der Aktion Setze Instanz-Zeitleiste für die Instanz gesetzt worden sein, und dann können Sie diese Aktion jederzeit verwenden, um ihren Betriebszustand aus den folgenden Optionen ändern

Option Beschreibung
Start Starten Sie die der Instanz zugewiesene Zeitleiste (wenn die Zeitleiste angehalten wurde, wird dadurch die Startposition wieder auf den Zeitpunkt 0 gesetzt).
Lebenslauf Fortsetzen einer angehaltenen Timeline (wenn die Timeline angehalten und nicht pausiert wurde, bewirkt dies nichts).
Pausiert Halten Sie die der Instanz zugewiesene Zeitleiste an (im angehaltenen Zustand werden die Momente in einer Zeitleiste nicht mit den Spielframes weitergeschaltet).
Stopp Stoppt die der Instanz zugewiesene Zeitleiste.

Syntax:

Argumente:

Argument Beschreibung
Zustand Der Zustand, in den die Zeitleiste gesetzt werden soll (siehe Tabelle oben)

Der obige Code des Aktionsblocks prüft den Wert, der in einer globalen Variablen gespeichert ist, und wenn er wahr ist, wird die Zeitleiste angehalten, und wenn er falsch ist, wird die Zeitleiste wieder fortgesetzt.
