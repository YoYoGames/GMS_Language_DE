Zeitleiste der Instanz einstellen

Diese Aktion weist der Instanz eine Zeitleiste aus dem Asset-Browser zu. Sie geben die zuzuweisende Zeitleiste an und können dann wählen, ob die Zeitleiste beim Erreichen des letzten Moments in einer Schleife laufen soll (das nächste Bild nach dem letzten Moment wird in einer Schleife zurück zum Moment der Zeitleiste zurück) sowie ob die Zeitleiste sofort gestartet werden soll oder nicht. Wenn Sie die Zeitleiste nicht sofort starten, können Sie sie mit der Aktion Zeitleistenstatus setzen später in einem anderen Ereignis oder Skript zu starten.

Aktion Syntax:

Argumente:

Argument Beschreibung
Zeitleiste Die Timeline-Ressource, die einer Instanz zugewiesen werden soll
Schleife Flagge, um die Zeitleiste nach dem letzten Moment in einer Schleife laufen zu lassen (standardmäßig ist die Flagge gesetzt)
Start Zeitleiste Flagge, um die Zeitleiste sofort starten zu lassen oder nicht (Standard ist markiert)
