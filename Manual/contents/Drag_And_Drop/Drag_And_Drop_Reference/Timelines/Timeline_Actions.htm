Zeitleiste Aktionsbibliothek

Zeitleisten sind ein leistungsfähiger Mechanismus zur Steuerung von Ereignissen in Ihrem Spiel, und sie basieren auf "Momenten", wobei ein "Moment" einfach ein einzelner Spielschritt ist. Diese Zeitleisten werden in der Regel über den Asset-Browser auf ähnliche Weise wie ein Objekt erstellt, d. h.: Sie erstellen eine Zeitleiste im Zeitleisten-Editor, Dann fügen Sie in jedem Moment, in dem sie eine Aktion ausführen soll, etwas Code oder eine DnD™-Aktionhinzu, die sie ausführen soll.

Nachdem eine Timeline-Ressource erstellt wurde, können Sie sie mit den folgenden Aktionen anhalten, starten und manipulieren:

Instanz-Zeitleiste einstellen
Timeline-Moment setzen
Timeline-Geschwindigkeit einstellen
Timeline-Status setzen
