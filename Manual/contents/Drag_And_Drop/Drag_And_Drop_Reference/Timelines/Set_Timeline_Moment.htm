Timeline-Moment einstellen

Diese Aktion wird verwendet, um eine Zeitleiste vom aktuellen Moment zu einem neuen Moment zu überspringen. Zeitleisten bewegen sich durch verschiedene "Momente" vorwärts, und jeder Moment nimmt typischerweise einen Spiel-Frame ein (wie das Step-Event eines Objekts). Sobald eine Zeitleiste begonnen hat Sobald eine Zeitleiste gestartet wurde, bewegt sie sich durch die Momente, bis sie das Ende erreicht, an dem sie entweder zum ersten Moment in der Zeitleiste zurückkehrt oder ganz anhält. Dieser lineare Ansatz für Zeitleisten kann jedoch mit dieser Aktion geändert werden, um jederzeit einen anderen Zeitpunkt zu setzen. Zeitpunkt zu setzen, so dass Sie Ihre eigenen Zeitleistenschleifen erstellen können, oder Abschnitte einer Zeitleiste unter bestimmten Umständen überspringen können, usw... Mit dieser Aktion legen Sie einfach den Zeitpunkt fest, zu dem innerhalb der Zeitleiste gesprungen werden soll, und der nächste Spielframe wird diesen Moment (beachten Sie, dass Sie der Instanz mit der Aktion Instanz-Zeitleiste setzen eine Zeitleiste zugewiesen haben müssen).
