Partikelsystem aktualisieren

Mit dieser Aktion wird das angegebene Partikelsystem gezwungen, die Position, Farbe, Alpha usw. zu aktualisieren. Sie geben den eindeutigen ID-Wert für das zu aktualisierende System an (wie er beim Erstellen des Systems mit der Aktion " Partikelsystem erstellen" zurückgegeben wurde), und jedes Mal, wenn Sie die Aktion aufrufen, wird das angegebene System um einen Spiel-Frame aktualisiert, d. h. Sie können diese Aktion z. B. innerhalb einer Repeat-Schleife aufrufen und das Partikelsystem mehrmals in einem einzigen Spiel-Frame aktualisieren können (nützlich, wenn Sie einen Level mit einem Effekt versehen wollen, bei dem die Partikel gleich zu Beginn vorhanden sein müssen, wie z. B. Schnee oder Regen). Sie können diese Aktion auch verwenden, wenn das Partikelsystem mit der Funktion "Partikelsystem anhalten " angehalten wurde, um die Partikel im System manuell zu aktualisieren.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Der eindeutige ID-Wert für das zu aktualisierende System
