Partikel-Alpha einstellen

Diese Aktion legt das Alpha für die Erstellung des Partikels sowie das Alpha über dessen Lebensdauer fest. Sie geben zunächst den ID-Wert des Partikeltyps an (der mit dem Wert übereinstimmen sollte, der beim Erstellen des Partikels mit der Aktion " Partikeltyp erstellen " zurückgegeben wurde) und können dann die zu verwendenden Alpha-Werte festlegen.

Sie können drei Alphawerte auswählen, zwischen denen der Partikel während seiner Lebensdauer überblendet. Wenn ein Partikel z. B. die Alphawerte 0,5, 1 und 0 hat und eine Lebensdauer von 60 Spielframes, dann hat er bei seiner Erstellung (Frame 0) auf 0,5 Alpha, dann wird er auf einen vollen Alpha-Wert von 1 überblendet (Bild 30), bevor er mit einem Alpha-Wert von 0 unsichtbar wird (Bild 60).

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der eindeutige ID-Wert für den Partikeltyp, für den die Farbe eingestellt werden soll
Start Das anfängliche Alpha zur Verwendung
Mitte Das dazwischenliegende Alpha zur Verwendung
Ende Das endgültige Alpha zur Verwendung

Beispiel:

Der obige Code des Aktionsblocks erstellt einen neuen Partikeltyp und weist seinen eindeutigen ID-Wert einer globalen Variablen zu. Anschließend werden alle Eigenschaften für den Partikeltyp festgelegt.
