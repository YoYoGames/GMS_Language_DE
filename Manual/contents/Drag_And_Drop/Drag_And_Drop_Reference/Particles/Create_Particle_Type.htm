Partikeltyp erstellen

In Ihren Projekten können Sie mehrere verschiedene Partikeltypen haben, jeder mit seinen eigenen Eigenschaften wie Richtung und Farbe usw., die mit den verschiedenen Partikeltyp-Aktionen eingestellt werden. Sie müssen jedoch zunächst einen Partikeltyp mit dieser Aktion erstellen, die im Wesentlichen eine kleine Menge an Speicher zuweist, um alle Partikeldetails zu halten, so dass Sie die Partikel jederzeit in der Zukunft verwenden können. Sie geben eine Variable an, um den eindeutigen ID-Wert für den Partikeltyp zu halten, und dann in allen zukünftigen Aktionen, die sich auf diesen Partikel beziehen, können Sie die Variable aufrufen. Sie können diese Variable als temporäre lokale Variable kennzeichnen, aber das bedeutet, dass sie am Ende des Aktionsskripts oder Ereignisses verworfen wird, was bedeutet, dass der Partikeltyp Typ immer noch existiert, aber Sie keine Möglichkeit haben, darauf zuzugreifen, so dass Vorsicht geboten ist, wenn Sie dies verwenden (im Allgemeinen müssen Partikeltypen nie in einer temporären lokalen Variablen gespeichert werden, aber für bestimmte Dinge, wie das Zeichnen von Partikeln auf eine Oberfläche, kann dies erforderlich sein).

HINWEIS: Ein einmal erstellter Partikeltyp bleibt für die Dauer des Spiels im Systemspeicher resident. Wenn Sie ihn also zu irgendeinem Zeitpunkt neu erstellen müssen (zum Beispiel, wenn Sie die Aktion " Spiel neu starten " aufrufen), dann müssen Sie den aktuellen Partikeltyp zerstören, da Sie sonst ein Speicherleck erhalten. Dies geschieht, wenn eine Ressource dem Speicher hinzugefügt und dann "de-referenziert" wird, was bedeutet, dass GameMaker Studio 2 keine Möglichkeit mehr hat, mit dieser Ressource zu interagieren mehr. Dies führt dazu, dass der Speicher durch unzugängliche Ressourcen verstopft wird, was wiederum zu Problemen in Ihrem Spiel führen kann. Um einen Partikeltyp zu zerstören, wenn er nicht mehr benötigt wird, verwenden Sie die Aktion Partikeltyp zerstören.

Sie können auch den für die Partikel zu verwendenden Überblendungsmodus einstellen. Standardmäßig werden sie mit einem "normalen" Überblendungsmodus gezeichnet, d. h. die Farben werden nicht durch das, was unter dem Partikel gezeichnet wurde, verändert. aktiviert ist, werden sie mit additiver Überblendung gezeichnet, was bedeutet, dass die Farben je nach dem, was unter dem Partikel gezeichnet wurde, verändert werden. Additives Überblenden ergibt sehr helle und "leuchtende" Effekte und ist ideal für Explosionen, Laser, Feuer usw., während normales Blending besser für Dinge wie Regen, Rauch und Trümmer geeignet ist.
