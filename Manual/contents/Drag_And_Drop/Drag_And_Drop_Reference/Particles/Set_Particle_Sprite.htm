Partikel-Sprite einstellen

Diese Aktion kann verwendet werden, um das Sprite festzulegen, das ein Partikel beim Zeichnen verwenden kann. Sie geben den eindeutigen ID-Wert für den Typ an, für den das Sprite festgelegt werden soll (wie bei der Erstellung des Typs mit der Aktion Partikeltyp erstellen zurückgegeben) und die zu verwendende Sprite-Ressource. Wenn das Sprite über Unterbilder verfügt, dann wird das Partikel mit der im Bild-Editor für das Sprite eingestellten Geschwindigkeit durch die Bilder animiert. Beachten Sie auch, dass, wenn Sie einen Farbwert für den Partikel festlegen (mit der Aktion Set Particle Colour ), diese Farbe mit dem verwendeten Sprite gemischt wird, ähnlich wie das Einstellen der Bildüberblendung auf einer Instanz eine Farbe mit dem Instanz-Sprite mischt (verwenden Sie Weiß für keine Überblendung).

HINWEIS: Wenn Sie diese Aktion verwenden, sollten Sie nicht die Aktion " Partikelform setzen" verwenden, da Sie nur ein Sprite oder eine Form haben können, aber nicht beides.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der ID-Wert für den Typ, auf den das Sprite gesetzt werden soll
Sprite Das Sprite, das dem Partikeltyp zugewiesen werden soll
