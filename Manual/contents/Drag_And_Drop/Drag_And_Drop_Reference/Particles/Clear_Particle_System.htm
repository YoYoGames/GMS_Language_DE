Klares Partikelsystem

Mit dieser Aktion können Sie das angegebene Partikelsystem in den Standardzustand zurückversetzen, indem Sie alle Emitter entfernen (es besteht auch keine Notwendigkeit, die Aktion "Partikelemitter zerstören " aufzurufen, da dies automatisch erledigt wird) und alle Partikel vom Bildschirm löschen. Sie geben den eindeutigen ID-Wert für das zu löschende System an, der zurückgegeben wurde, als Sie das System mit der Aktion " Partikelsystem erstellen" erstellt haben. Vorsicht ist geboten bei der Verwendung von Diese Aktion ist mit Vorsicht zu genießen, da sie alle mit dem Partikelsystem verbundenen Emitter zerstört - das bedeutet, dass jede Instanz im Raum, die einen Emitter verwendet, eine Fehlermeldung ausgeben kann, da dieser Emitter nicht mehr vorhanden ist; stellen Sie also sicher, dass die Aktion nur aufgerufen wird, wenn Sie keine neuen Partikel von diesem Emitter erstellen werden (weitere Informationen zu Partikelemitter finden Sie unter Partikelemitter erstellen).

HINWEIS: Diese Aktion löscht nur die sichtbaren Partikel im Raum, aber sie löscht weder die Partikeleigenschaften noch entfernt sie diese aus dem Speicher. Dafür sollten Sie die Aktion Partikeltyp zerstören verwenden.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Der eindeutige ID-Wert für das zu zerstörende System
