Partikelsystem zerstören

Diese Aktion "zerstört" das angegebene Partikelsystem (d. h.: gibt den vom Partikelsystem verwendeten Speicher frei). Sie geben den eindeutigen ID-Wert für das zu zerstörende System an, der zurückgegeben wurde, als Sie das System mit der Aktion " Partikelsystem erstellen" erstellt haben. Diese Aktion sollte immer dann aufgerufen werden, wenn Sie ein Partikelsystem in Ihrem Spiel nicht mehr benötigen, oder wenn Sie das Partikelsystem neu erstellen möchten (z. B. kurz vor dem Aufruf von Restart Game). Beachten Sie, dass wenn das Partikelsystem nicht als persistent gekennzeichnet wurde, brauchen Sie diese Aktion nicht aufzurufen, da das Partikelsystem (und alle damit verbundenen Emitter) automatisch auf der Raumseite aufgeräumt wird.

WICHTIG! Diese Funktion zerstört und entfernt auch alle Emitter, die möglicherweise erstellt und mit dem zu zerstörenden System verbunden wurden (weitere Informationen finden Sie unter Partikelemitter erstellen ).
