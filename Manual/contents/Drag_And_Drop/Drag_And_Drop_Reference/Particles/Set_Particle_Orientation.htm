Partikelausrichtung einstellen

Diese Aktion wird verwendet, um die Partikelausrichtung einzustellen - den Winkel, in dem die Partikelbilder gezeichnet werden - der statisch sein oder so eingestellt werden kann, dass er im Laufe der Zeit zunimmt/abnimmt. Sie geben den eindeutigen ID-Wert für den Typ an, dessen Größe eingestellt werden soll (wie er zurückgegeben wurde, als Sie geben den eindeutigen ID-Wert für den Typ an, für den Sie die Größe festlegen möchten (der zurückgegeben wurde, als Sie den Typ mit der Aktion " Partikeltyp erstellen" erstellt haben), und geben dann einen Bereich von anfänglichen Startausrichtungen an, dann den Betrag der Zunahme und schließlich das "Wackeln". Die Ausrichtung, mit der ein Partikel erzeugt wird Die Ausrichtung, mit der ein Partikel erzeugt wird, wird so berechnet, dass 0 rechts, 90 oben, 180 links und 270 unten ist, so dass eine minimale Einstellung von 0 und eine maximale Einstellung von 360 Partikel emittiert, bei denen jeder um einen zufälligen Betrag um 360° gedreht wird (beachten Sie, dass dies keinen Einfluss auf die Partikelrichtung hat).

Mit der Option zum Erhöhen einer Partikelausrichtung können Sie einen Wert eingeben, der bei jedem Spielbild der Partikellebensdauer zum Basisausrichtungswinkel addiert wird. Wenn Sie diesen Wert also z. B. auf 0,1 setzen, wird dem Bildwinkel in jedem Spielframe 0,1 Spiel-Frame. Beachten Sie, dass Sie hier auch negative Werte verwenden können, um den Partikel auch in die entgegengesetzte Richtung rotieren zu lassen, so dass positive Werte den Partikel gegen den Uhrzeigersinn rotieren lassen und negative im Uhrzeigersinn. Beachten Sie, dass die minimale Ausrichtungs Dekrement -0,01 und das minimale Inkrement 0,01 ist.

Sie können auch die Option "Wiggle" für die Partikelausrichtung einstellen. Dies ist ein Wert, der zufällig zwischen 0 und dem von Ihnen angegebenen Wert generiert wird und dann bei jedem Schritt der Partikellebensdauer zum Partikelbildwinkel addiert oder subtrahiert wird, wodurch das gezeichnete Bild in seiner Drehung schwankt. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Schließlich können Sie wählen, ob die Ausrichtung relativ zur Partikelrichtung sein soll oder nicht. Das bedeutet, dass, während der Partikel eine Richtung (und Geschwindigkeit) eingestellt hat, das Partikelbild auf der Grundlage dieses Vektors ausgerichtet wird. Beachten Sie, dass, wenn Wenn Sie die Geschwindigkeit des Partikels reduzieren und diese während der Lebensdauer des Partikels den Wert 0 erreicht, setzt die fehlende Geschwindigkeit die Richtung auf den Standardwert von 0°. Orientierungswinkel.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der eindeutige ID-Wert für den Partikeltyp, für den die Ausrichtung festgelegt werden soll
Min-Winkel Die minimale Ausrichtung für den Partikeltyp (von -360 bis +360, Standardwert 0)
Max Winkel Die maximale Ausrichtung für den Partikeltyp (von -360 bis +360, Voreinstellung 360)
Erhöhen Sie Der Wert, der bei jedem Spiel-Frame zur Partikel-Typ-Orientierung hinzuaddiert wird (kann negativ sein, Minimum +/-0,01)
Wackeln Der maximale Wert, der pro Schritt zufällig zur Partikelausrichtung addiert oder subtrahiert wird (+/-20).
Relativ zur Richtung Ob der Partikelbildwinkel relativ zur Bewegungsrichtung des Partikels eingestellt werden soll (aktiviert) oder nicht (deaktiviert).
