Partikelgeschwindigkeit einstellen

Diese Aktion wird verwendet, um die Partikelgeschwindigkeit einzustellen, die statisch oder so eingestellt werden kann, dass sie mit der Zeit zunimmt/abnimmt. Sie geben den eindeutigen ID-Wert für den Typ an, dessen Größe eingestellt werden soll (wie beim Erstellen des Typs mit der Aktion " Partikeltyp erstellen" zurückgegeben) und geben dann einen Bereich von Anfangsgeschwindigkeiten an, dann den Betrag der Zunahme und schließlich das "Wackeln". Die Geschwindigkeit, mit der ein Partikel erzeugt wird, wird als Pixel pro Spiel-Frame berechnet, so dass eine minimale Einstellung von 2 und eine maximale Einstellung von 5 dazu führt, dass Partikel, die sich mit einer Geschwindigkeit zwischen 2 und 5 Pixeln pro Spiel-Frame bewegen.

Mit der Option zum Erhöhen der Partikelgeschwindigkeit können Sie einen Wert eingeben, der bei jedem Spielframe der Partikellebensdauer zum Basisgeschwindigkeitswert hinzugefügt wird. Wenn Sie diesen Wert also z. B. auf 0,1 setzen, wird die Geschwindigkeit bei jedem Spielbild um 0,1 erhöht. Beachten Sie dass Sie hier auch negative Werte verwenden können, um den Partikel zu verlangsamen, obwohl die Geschwindigkeit nie unter 0 gehen wird. Beachten Sie, dass die minimale Geschwindigkeitsverringerung, die akzeptiert wird, -0,01 und die minimale Erhöhung 0,01 ist.

Schließlich können Sie die Option "Wackeln" einstellen. Dies ist ein Wert, der zufällig zwischen 0 und dem von Ihnen angegebenen Wert generiert wird und dann bei jedem Schritt der Partikellebensdauer zur Geschwindigkeit addiert oder subtrahiert wird, wodurch das gezeichnete Bild in seiner Geschwindigkeit schwankt. Geschwindigkeit. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der eindeutige ID-Wert für den Partikeltyp zum Einstellen der Größe von
Min. Drehzahl Die Mindestgeschwindigkeit für den Partikeltyp (Voreinstellung 0,1)
Maximale Geschwindigkeit Die maximale Geschwindigkeit für den Partikeltyp (Voreinstellung 1)
Erhöhen Sie Der Wert, der bei jedem Spielframe zur Geschwindigkeit des Partikeltyps hinzuaddiert wird (kann negativ sein, Minimum +/-0,01)
Wackeln Der maximale Wert, der pro Schritt zufällig zur Partikelgeschwindigkeit addiert oder subtrahiert wird (+/-20).
