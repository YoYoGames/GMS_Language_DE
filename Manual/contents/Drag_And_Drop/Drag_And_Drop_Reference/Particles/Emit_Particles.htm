Partikel emittieren

Diese Aktion wird verwendet, um Partikel aus einem Emitter zu erzeugen. Sie geben die Partikelsystem-ID an, zu der der Emitter gehört (wie von der Aktion Create Particle System zurückgegeben), dann die eigentliche Emitter-ID (wie von der Aktion (zurückgegeben von der Aktion " Partikel-Emitter erstellen") und schließlich den zu erstellenden Partikeltyp (zurückgegeben von der Aktion " Partikeltyp erstellen"). Sie können dann "Burst" oder "Streamen" der Partikel aus dem Emitter wählen sowie die Anzahl der zu erzeugenden Partikel angeben.

Wenn Sie sich für das Streamen von Partikeln entscheiden, müssen Sie diese Aktion nicht in einem Game-Frame-Ereignis wie dem Step-Ereignis oder dem Draw-Ereignis platzieren, sondern rufen sie einfach einmal im Create-Ereignis auf (z. B.) und der Emitter wird in jedem Game Frame, bis Sie diese Aktion erneut aufrufen, um sie zu stoppen (indem Sie den Zählwert auf 0 setzen). Wenn Sie sich dafür entscheiden, die Partikel platzen zu lassen, können Sie diese Aktion in jedem beliebigen Ereignis aufrufen und die Partikel werden einmal erstellt.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Die ID des Systems, in dem sich der Emitter befindet
Sender Die ID des zu verwendenden Emitters
Typ Die ID des zu erstellenden Partikeltyps
Veranstaltung Der Ereignistyp für die Partikel, entweder "Burst" oder "Stream".
Zählen Sie Die Anzahl der Partikel zum Bersten oder Strömen

Beispiel:

Der obige Code des Aktionsblocks erstellt einen Partikelemitter, legt seine Region fest, und setzt ihn dann so, dass er einen Strom von Partikeln aussendet.
