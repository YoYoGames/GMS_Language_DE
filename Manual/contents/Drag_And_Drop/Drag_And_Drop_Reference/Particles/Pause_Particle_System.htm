Pause Partikelsystem

Mit dieser Aktion wird das angegebene Partikelsystem angehalten. Sie geben den eindeutigen ID-Wert für das anzuhaltende System an, der zurückgegeben wurde, als Sie das System mit der Aktion " Create Particle System" erstellt haben. Ein pausiertes Partikelsystem zeichnet weiterhin die Partikel, die gespawnt wurden, aber sie aktualisieren nicht mehr ihre Farbe, Position oder Alpha, noch beenden sie ihre Lebenszeit und verschwinden, bis Sie das System wieder aufnehmen. Sie können wählen, ob Sie das System anhalten oder fortsetzen möchten, indem Sie die Option Option "Zustand".

Beachten Sie, dass dadurch weder Emitter angehalten werden, die Partikel strömen, noch wird ein Emitter gestoppt, der Partikel ausstößt - Emitter erzeugen weiterhin Partikel, wenn sie aufgerufen werden, aber wenn das System angehalten ist, tun die Partikel nichts und werden genau dort gezeichnet, wo sie erzeugt wurden, bis Sie den Pausenzustand zurücksetzen Wenn das System jedoch angehalten ist, tun die Partikel nichts und werden genau dort gezeichnet, wo sie erstellt wurden, bis Sie den angehaltenen Zustand zurücksetzen oder " Update Particle System" aufrufen.

Aktion Syntax:

Argumente:

Argument Beschreibung
System Der eindeutige ID-Wert für das System, das pausiert werden soll
Zustand Ob das Partikelsystem angehalten oder fortgesetzt werden soll

Beispiel:

Der obige Code des Aktionsblocks prüft eine globale Variable, und wenn sie den Wert true zurückgibt, wird das angegebene Partikelsystem pausiert, andernfalls wird es fortgesetzt.
