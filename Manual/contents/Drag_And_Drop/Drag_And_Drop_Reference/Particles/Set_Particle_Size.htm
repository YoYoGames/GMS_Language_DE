Eingestellte Partikelgröße

Diese Aktion wird verwendet, um die Partikelgröße einzustellen, die statisch oder so eingestellt werden kann, dass sie im Laufe der Zeit erhöht/verringert wird. Sie geben den eindeutigen ID-Wert für den Typ an, dessen Größe eingestellt werden soll (wie beim Erstellen des Typs mit der Aktion " Partikeltyp erstellen" zurückgegeben) und geben dann einen Bereich von anfänglichen Startgrößen, dann den Erhöhungsbetrag und schließlich das "Wackeln" an. Die Größe, mit der ein Partikel erzeugt wird, ist ein skalarer Wert, wobei 1 die gleiche Größe wie das Basispartikelbild ist, 0,5 wäre die halbe Größe des Basisbildes und ein Wert von 2 wäre das Doppelte. Wenn die minimale und die maximale Größe nicht gleich sind, wird die Anfangsgröße für jeden erstellten Partikel ein zufälliger Wert zwischen diesen beiden Werten sein.

Mit der Option zum Erhöhen einer Partikelgröße können Sie einen Wert eingeben, der bei jedem Spielbild der Partikellebensdauer zum Basisskalarwert addiert wird. Wenn Sie diesen Wert also z. B. auf 0,1 setzen, wird bei jedem Spiel-Frame 0,1 zum Skalierungswert hinzugefügt. Beachten Sie, dass Sie hier auch negative Werte verwenden können, um das Partikelbild zu verkleinern. Beachten Sie, dass die minimale Skalierungsverringerung, die akzeptiert wird, -0,01 und die minimale Erhöhung 0,01 beträgt.

Schließlich können Sie die Option "Wackeln" einstellen. Dies ist ein Wert, der zufällig zwischen 0 und dem von Ihnen angegebenen Wert generiert wird und dann bei jedem Schritt der Partikellebensdauer zur Größe addiert oder subtrahiert wird, wodurch das gezeichnete Bild in seiner Größe schwankt Größe. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der eindeutige ID-Wert für den Partikeltyp zum Einstellen der Größe von
Min Größe Die minimale Skala für den Partikeltyp (Standardwert 0,1)
Max Größe Die maximale Skala für den Partikeltyp (Voreinstellung 1)
Erhöhen Sie Der Wert, der bei jedem Spielframe zur Größe des Partikeltyps addiert wird (kann negativ sein, Minimum +/-0,01)
Wackeln Der maximale Wert, der pro Schritt zufällig zur Partikelgröße addiert oder subtrahiert wird (+/-20).

Der obige Code des Aktionsblocks erstellt einen neuen Partikeltyp und weist seinen eindeutigen ID-Wert einer globalen Variablen zu. Anschließend werden alle Eigenschaften für den Partikeltyp festgelegt.
