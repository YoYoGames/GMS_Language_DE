Ziehwert

Mit dieser Aktion können Sie eine String-Beschriftung und/oder eine Variable oder einen Wert auf den Bildschirm zeichnen. Das Argument "Beschriftung" ist eine Zeichenkette, die dem Wert vorangestellt werden soll, und kann bei Bedarf leer gelassen werden, aber wenn Sie Text einfügen möchten, sollte dieser in Anführungszeichen eingeschlossen werden in Anführungszeichen gesetzt werden. Der Wert kann eine Variable, ein fester Wert oder eine andere Zeichenkette sein - in diesem Fall sollte er ebenfalls in Anführungszeichen gesetzt werden -, darf aber nicht leer bleiben. Dann geben Sie die Position an, an der die Zeichenkette gezeichnet werden soll, entweder als absolute Position oder als eine Position relativ zu der Instanz, die den Text zeichnet. Beachten Sie, dass das Aussehen des Textes beim Zeichnen von der verwendeten Schriftart, der Farbe und der Ausrichtung abhängt. All diese Dinge können vor dem Zeichnen geändert werden, wie im folgenden Beispiel gezeigt.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Beschriftung Die zu zeichnende Beschriftung (eine Zeichenkette), kann leer gelassen werden
Wert Der zu zeichnende Wert (Real, Variable oder String)
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll
