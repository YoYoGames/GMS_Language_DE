Sprite zeichnen transformiert

Diese Funktion zeichnet das angegebene Sprite ähnlich wie die Aktion Draw Sprite, jedoch mit zusätzlichen Optionen zum Ändern der Skalierung, der Überblendung, der Drehung und des Rahmens des zu zeichnenden Sprites. Das Ändern dieser Werte verändert nicht Sie können alle verfügbaren Sprite-Eigenschaftsvariablen anstelle von direkten Werten für alle Argumente in der Funktion verwenden. Das folgende Bild veranschaulicht wie sich verschiedene Werte auf das Zeichnen des Sprites auswirken:

HINWEIS: Die Farbüberblendung wird nur für das HTML5-Ziel empfohlen, wenn WebGL aktiviert ist, obwohl Sie die Überblendungsfarbe auch dann festlegen können, wenn sie nicht aktiviert ist, und das Sprite wird dann wie üblich überblendet. Alle Überblendungen auf diese Weise erzeugen jedoch ein Duplikat Sprite, das dann im Cache gespeichert und bei Bedarf verwendet wird. Dies ist alles andere als optimal, und wenn Sie mehrere Farbwechsel verwenden, verlangsamt sich die Leistung Ihres Spiels, es sei denn, Sie aktivieren WebGL. Wenn Sie WebGL nicht verwenden möchten, können Sie die die Schrift-Cache-Größe einstellen, um dies bei Bedarf zu begrenzen, indem Sie die Funktion sprite_set_cache_size().

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Das zu zeichnende Sprite
Rahmen Der zu zeichnende Sprite-Frame (verwenden Sie die eingebaute Variable image_index für den aktuellen Frame)
X Die x-Position, an der innerhalb des Raums gezeichnet werden soll
Y Die y-Position, an der innerhalb des Raums gezeichnet werden soll
Skala X Der Skalierungsfaktor für die X-Achse (verwenden Sie die eingebaute Variable image_xscale für die aktuelle X-Skala)
Skala Y Der Skalierungsfaktor für die Y-Achse (verwenden Sie die eingebaute Variable image_yscale für den aktuellen Y-Maßstab)
Drehung Der Zeichenwinkel für das Sprite (von 0 bis 360, gegen den Uhrzeigersinn rotierend, wobei 0 rechts ist - verwenden Sie die eingebaute Variable image_angle für die aktuelle Rotation)
Farbe Die Farbe, die mit dem Sprite überblendet werden soll (Standard ist weiß, und Sie können die eingebaute Variable image_blend für die aktuelle Farbe verwenden)

Beispiel:

Der obige Code des Aktionsblocks prüft eine Variable und wenn sie auf true gesetzt ist, wird das Sprite mit zufälliger Skalierung entlang der x- und y-Achse gezeichnet und mit der Farbe rot überblendet, andernfalls wird das Sprite mit den Standardtransformationen gezeichnet.
