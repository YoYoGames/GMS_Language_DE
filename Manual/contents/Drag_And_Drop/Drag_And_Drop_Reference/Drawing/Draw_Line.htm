Linie zeichnen

Mit dieser Aktion wird eine Linie an einer bestimmten Position im Raum gezeichnet. Sie geben die anfängliche x/y-Position und dann die endgültige x/y-Position an und die Linie wird zwischen diesen Positionen mit der aktuellen Zeichenfarbe gezeichnet. Die Positionen können entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die Start-X-Position
y Die y-Startposition
x2 Die Endposition x
y2 Die Endposition y

Beispiel:

Der obige Code des Aktionsblocks zeichnet eine rosafarbene Linie über den oberen Teil des Raums.
