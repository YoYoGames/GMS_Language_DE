Sprite zeichnen

Mit dieser Aktion wird ein Sprite irgendwo im Raum gezeichnet. Sie geben das Sprite, die Position (entweder eine absolute Position innerhalb des Raums oder eine relative Position zur zeichnenden Instanz) und den Animationsrahmen des Sprites an, wenn es mehrere Frames hat (beachten Sie, dass Bildframes von 0 aufwärts nummeriert sind). Wenn Sie ein animiertes Sprite haben und Sie wollen, dass es die Frames so zeichnet, als ob es dem Objekt zugeordnet wäre, dann verwenden Sie -1 für den Frame-Index oder die eingebaute Instanzvariable image_index, Beachten Sie jedoch, dass, wenn der Instanz kein Sprite zugewiesen ist, sie nicht animiert wird, und wenn das zugewiesene Sprite eine andere Anzahl von Frames hat als das gezeichnete, wird das Sprite mit zusätzlichen Frames oder fehlenden Frames gezeichnet.

WICHTIG: Diese Aktion zeichnet das Sprite nicht transformiert, auch wenn Sie die Bildüberblendung oder den Maßstab geändert haben. Siehe dazu die Aktion Sprite transformiert zeichnen.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Das zu zeichnende Sprite
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll
Frame-Index Der Rahmenindex zum Zeichnen
