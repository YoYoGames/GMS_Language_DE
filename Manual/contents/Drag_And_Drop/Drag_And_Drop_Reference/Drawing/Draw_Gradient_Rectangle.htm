Verlaufsrechteck zeichnen

Mit dieser Aktion wird ein Rechteck an einer bestimmten Position im Raum gezeichnet, wobei ein Satz von Mischfarben verwendet wird, um einen Farbverlauf zu erzeugen. Sie geben die obere linke Position und die untere rechte Position an, zwischen denen das Rechteck gezeichnet wird, und die Position kann Die Position kann entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft. Sie stellen dann die Farben ein, die von jeder der Ecken aus überblendet werden sollen, und können auch einstellen, ob das Rechteck gefüllt oder umrandet werden kann, indem Sie das Füllfeld am unteren Rand.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Links Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet werden soll
Rechts Die richtige Position zum Beenden der Zeichnung von
Unten Die unterste Position zum Beenden der Zeichnung von
Oben links Die zu überblendende Farbe aus der linken oberen Ecke
Oben rechts Die zu überblendende Farbe aus der rechten oberen Ecke
Unten links Die zu überblendende Farbe aus der linken unteren Ecke
Unten rechts Die zu überblendende Farbe aus der rechten unteren Ecke

Der obige Code des Aktionsblocks zeichnet ein Rechteck mit Farbverlauf relativ zur Instanzposition mit etwas Text darüber.
