Farbverlauf Ellipse zeichnen

Mit dieser Aktion wird eine Ellipse an einer bestimmten Position im Raum gezeichnet, wobei ein Satz von Mischfarben verwendet wird, um einen Farbverlauf zu erzeugen. Sie geben die obere linke Position und die untere rechte Position des Bereichs an, in den die Ellipse "passen" soll, und die Ellipse wird zwischen diesen beiden Positionen gezeichnet. Die Position kann entweder eine absolute Position innerhalb des Raums sein oder eine Position relativ zu der Instanz, die die Aktion aufruft, und Sie können die Farben so einstellen, dass sie vom Rand der Ellipse und von ihrer Mitte aus überblenden. Die Ellipse kann gefüllt oder umrandet gezeichnet werden, indem Sie das Füllfeld am unteren Rand aktivieren.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Links Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet werden soll
Rechts Die richtige Position zum Beenden der Zeichnung von
Unten Die untere Position zum Beenden des Zeichnens von
Zentrum Die Farbe zum Ausblenden aus der Mitte
Kante Die Farbe zum Ausblenden vom Rand
