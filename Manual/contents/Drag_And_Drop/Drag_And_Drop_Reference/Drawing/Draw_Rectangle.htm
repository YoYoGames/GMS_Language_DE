Rechteck zeichnen

Mit dieser Aktion wird ein Rechteck an einer bestimmten Position im Raum gezeichnet. Sie geben die obere linke Position und die untere rechte Position an, zwischen denen das Rechteck gezeichnet wird, und die Position kann entweder eine absolute Position innerhalb des Raums sein, oder eine Position relativ zu der Instanz, die die Aktion aufruft. Sie können auch festlegen, ob das Rechteck gefüllt oder umrandet werden kann, indem Sie das Füllfeld unten aktivieren.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Links Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet werden soll
Rechts Die richtige Position zum Beenden der Zeichnung von
Unten Die untere Position zum Beenden des Zeichnens von

Beispiel:

Der obige Code des Aktionsblocks zeichnet zwei Rechtecke, eines grün und gefüllt, und eines gelb und nur umrandet, an einer Position relativ zu der Instanz, die die Aktionen aufruft.
