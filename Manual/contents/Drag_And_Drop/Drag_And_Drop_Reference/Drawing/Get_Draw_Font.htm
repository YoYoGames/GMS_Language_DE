Get Draw Font

Mit dieser Aktion können Sie den ID-Wert der Schriftart abrufen, die aktuell zum Zeichnen zugewiesen ist. Der zurückgegebene Wert ist entweder -1 für keine zugewiesene Schrift oder ein Wert >= 0, der eine Schriftkonstante aus dem Asset-Browser darstellt. Der Rückgabewert wird gespeichert Der Rückgabewert wird in der von Ihnen angegebenen Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren).
