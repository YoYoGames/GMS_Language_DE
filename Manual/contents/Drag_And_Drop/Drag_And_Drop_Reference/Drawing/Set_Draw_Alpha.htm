Zeichnen Alpha einstellen

Mit dieser Aktion können Sie das zu verwendende Alpha für beliebige Formen (Rechteck/Ellipse/Linie) oder für jeden Text, der anschließend gezeichnet wird, einstellen. Beachten Sie, dass dies nicht vor jeder Zeichenaktion aufgerufen werden muss, sondern nur einmal zu Beginn des Spiels eingestellt werden kann. Alpha ist die Einstellung, die steuert, wie "transparent" Dinge gezeichnet werden und basiert auf einer Skala von 0 bis 1, wobei 0 völlig transparent und 1 völlig undurchsichtig ist.

Aktion Syntax:

Argumente:

Argument Beschreibung
Alpha Der zu verwendende Alphawert (von 0 bis 1, Standard ist 1)

Der obige Code des Aktionsblocks setzt das Zeichen-Alpha auf einen Zufallswert zwischen 0 und 1.
