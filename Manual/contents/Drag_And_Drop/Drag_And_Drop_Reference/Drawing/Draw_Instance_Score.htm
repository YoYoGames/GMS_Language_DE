Draw Instance Score

Mit dieser Aktion können Sie den Score-Wert der Instanz zeichnen. Wenn Sie eine beliebige Score-Aktion in GameMaker Studio 2 verwenden, wird der Instanz, die sie aufruft, eine neue Instanzbereichsvariable namens score hinzugefügt. Diese Variable kann dann geprüft und gesetzt werden, wie jede andere Variable auch. Im Fall dieser Aktion wird die Variable an der angegebenen Position auf dem Bildschirm gezeichnet, zusammen mit einem beliebigen Beschriftungstext, den Sie angeben möchten. Der Beschriftungstext wird vor dem Punktwert gezeichnet, und wenn Wenn es sich um Text handelt, sollte er mit "" umschlossen werden, und beide Texte werden mit der aktuellen Zeichnungsschrift, Farbe und Ausrichtung formatiert.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Beschriftung Die Zeichenkette, die zusammen mit dem Score-Wert gezeichnet werden soll (vorangestellt und kann leer gelassen werden)
x Die x-Position, an der die Partitur gezeichnet werden soll
y Die y-Position, an der die Partitur gezeichnet werden soll
