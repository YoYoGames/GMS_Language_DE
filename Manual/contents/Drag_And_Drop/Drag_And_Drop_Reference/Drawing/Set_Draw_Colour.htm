Set Draw Farbe

Mit dieser Aktion können Sie die Farbe einstellen, die für beliebige Formen (Rechteck/Ellipse/Linie) oder für jeden Text, der anschließend gezeichnet wird, verwendet werden soll. Beachten Sie, dass diese Aktion nicht vor jeder Zeichenaktion aufgerufen werden muss, sondern nur einmal zu Beginn des Spiels eingestellt werden kann. Wenn z. B. Ihr gesamter Text gelb sein soll, können Sie diese Aktion im Ereignis "Create" des ersten Raums des Spiels aufrufen, und alle Texte (und Formen) werden dann von da an gelb gezeichnet. Wenn Sie jedoch mehrere verschiedene Farben verwenden möchten, dann müssen Sie diese Aktion vor jedem Text oder jeder Form aufrufen, die eine andere Farbe benötigt.

Die Aktion hat unten ein Kontrollkästchen für Alpha verwenden. Damit wird die Alphakomponente der Farbe auch zum Einstellen des Zeichenalphas verwendet (ähnlich wie bei der Aktion " Zeichenalphas setzen ").
