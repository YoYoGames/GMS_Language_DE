Transformierten Wert zeichnen

Mit dieser Aktion können Sie eine String-Beschriftung und/oder eine Variable oder einen Wert auf den Bildschirm zeichnen. Das Argument "Beschriftung" ist eine Zeichenkette, die dem Wert vorangestellt werden soll, und kann bei Bedarf leer gelassen werden, und der Wert kann eine Variable, ein fester Wert oder eine andere Zeichenkette sein - in diesem Fall sollte sie in Anführungszeichen gesetzt werden und kann bei Bedarf auch leer gelassen werden. Anschließend geben Sie die Position an, an der die Zeichenkette gezeichnet werden soll, entweder als absolute Position oder als Position relativ zu der Instanz, die den Text zeichnet. Zusätzlich können Sie die Skalierung des Textes festlegen, d. h. die Streckung entweder entlang der horizontalen oder der vertikalen Achse, sowie die Drehung (die Winkel für die Drehung des Textes sind gegen den Uhrzeigersinn, d. h., wenn Sie den Wert 90° festlegen, wird der Text nach links und -90° nach rechts gedreht). rechts). Die Skalierung erfolgt mit dem angegebenen Faktor, d. h. eine Skalierung von 0,5 halbiert die Breite des Textes und eine Skalierung von 2 verdoppelt die Breite, wobei 1 der Standardwert für keine Skalierung ist. Beachten Sie, dass das Aussehen des gezeichneten Textes von folgenden Faktoren abhängt der verwendeten Schriftart, der verwendeten Farbe und der Ausrichtung abhängt. All diese Dinge können vor dem Zeichnen geändert werden, wie im Beispiel unten gezeigt.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Beschriftung Die zu zeichnende Beschriftung (eine Zeichenkette), kann leer gelassen werden
Wert Der zu zeichnende Wert (Real, Variable oder String)
x Die x-Position, an der gezeichnet werden soll
y Die y-Position, an der gezeichnet werden soll
x-Skala Der x (horizontale) Skalierungsfaktor, Voreinstellung ist 1
y-Skala Der y (vertikale) Skalierungsfaktor, Voreinstellung ist 1
Drehung Der Drehwinkel für das Zeichnen des Textes

Beispiel:

Der obige Code des Aktionsblocks legt eine Farbe und eine Schriftart fest und richtet dann den Text aus so aus, dass er von der Mitte aus gezeichnet wird, bevor er tatsächlich den Wert "current_time" zusammen mit der Beschriftung "TIME: " zeichnet, die auf die doppelte Größe skaliert ist und aufgrund der zufälligen Winkeländerung leicht "vibriert".
