Textausrichtung einstellen

Mit dieser Aktion wird die Schriftausrichtung für alle nachfolgenden Aktionen zum Zeichnen von Text festgelegt. Sie können die horizontale Ausrichtung auf links, mittig oder rechts und die vertikale Ausrichtung auf oben, mittig oder unten einstellen und der Text wird relativ zur x/y Position der Zeichenaktion ausgerichtet, wie in den folgenden Bildern gezeigt:

Text wird horizontal nach links ausgerichtet
Text wird horizontal in der Mitte ausgerichtet
Text wird horizontal nach rechts ausgerichtet
Text wird vertikal nach oben ausgerichtet
Text wird vertikal zur Mitte ausgerichtet
Text wird vertikal nach unten ausgerichtet

Beachten Sie, dass damit die Ausrichtung für alle nachfolgenden Zeichentextaufrufe festgelegt wird. Sie können diese Aktion also z. B. einmal zu Beginn des Spiels aufrufen - in jedem Fall, nicht nur beim Ereignis "Zeichnen" - und alle Zeichentextaktionen werden die festgelegte Ausrichtung verwenden. Wenn Sie jedoch mehrere Zeichentext-Aktionen auf unterschiedliche Weise ausrichten möchten, müssen Sie diese Aktion vor jeder Zeichentext-Aktion aufrufen, die Sie anders ausrichten möchten.

Aktion Syntax:

Argumente:

Argument Beschreibung
H.Ausrichten Der Wert für die horizontale Ausrichtung (Links, Mitte, Rechts)
V.Ausrichten Der Wert für die vertikale Ausrichtung (Oben, Mitte, Unten)

Beispiel:

Der obige Code des Aktionsblocks setzt die Schriftart, die Zeichenfarbe und die Ausrichtung für einen Text der gezeichnet wird.
