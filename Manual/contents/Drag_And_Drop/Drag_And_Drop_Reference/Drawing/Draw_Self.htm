Selbst zeichnen

Wenn Sie dem Draw Event eines Objekts irgendeine Art von Aktion oder Code hinzufügen, teilen Sie GameMaker Studio 2 mit, dass Sie die Kontrolle übernehmen und ihm sagen, was es von nun an zeichnen soll (standardmäßig zeichnet eine Instanz das ihr zugewiesene Sprite im Objekteditor, wenn es keine Aktionen im Haupt-Zeichen-Event gibt). Das bedeutet, dass ein zugewiesenes Sprite nicht mehr gezeichnet wird, es sei denn, Sie sagen es explizit, dann würden Sie diese Aktion verwenden. Dies zeichnet einfach das zugewiesene Sprite für das Objekt mit allen Transformationen, die möglicherweise in früheren Ereignissen gesetzt wurden (wie Skalierung, Überblendung oder Änderungen in Alpha), auf das aktuelle Teilbild.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.
