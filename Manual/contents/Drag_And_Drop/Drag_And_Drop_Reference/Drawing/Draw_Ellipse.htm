Ellipse zeichnen

Mit dieser Aktion wird eine Ellipse an einer bestimmten Position im Raum gezeichnet. Sie geben die obere linke Position und die untere rechte Position des Bereichs an, in den die Ellipse "passen" soll, und die Ellipse wird dazwischen gezeichnet. Die Positionen können Die Positionen können entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft, und Sie können auch festlegen, ob die Ellipse gefüllt oder umrandet werden kann, indem Sie das Füllfeld unten aktivieren.

HINWEIS: Diese Aktion ist nur für die Verwendung in den verschiedenen Draw-Events vorgesehen und wird nichts zeichnen, wenn sie anderweitig verwendet wird.

Aktion Syntax:

Argumente:

Argument Beschreibung
Links Die linke Position, von der aus gezeichnet werden soll
Top Die oberste Position, von der aus gezeichnet werden soll
Rechts Die richtige Position zum Beenden der Zeichnung von
Unten Die untere Position zum Beenden des Zeichnens von
