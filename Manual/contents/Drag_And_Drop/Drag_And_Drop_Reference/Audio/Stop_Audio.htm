Audio anhalten

Diese Aktion kann verwendet werden, um die Wiedergabe eines Tons zu stoppen. Sie wählen den Sound im Asset-Explorer aus und alle Instanzen dieses Sounds werden gestoppt (wenn Sie also denselben Sound für die Wiedergabe in drei verschiedenen Events eingestellt haben, werden alle gestoppt).

Aktion Syntax:

Argumente:

Argument Beschreibung
Ton Die Tonquelle zum Anhalten

Beispiel:

Der obige Code des Aktionsblocks prüft, ob eine Maustaste gedrückt wurde, und wenn dies der Fall ist, prüft er eine globale Variable, um festzustellen, ob sie nicht wahr ist. Wenn sie nicht wahr ist, wird ein Ton abgespielt (und in einer Schleife wiederholt) und die globale Variable auf wahr gesetzt, andernfalls wird der Ton gestoppt und die globale Variable auf falsch gesetzt.
