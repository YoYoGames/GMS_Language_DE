Master-Lautstärke einstellen

Mit dieser Aktion können Sie die Lautstärke für das gesamte Soundsystem einstellen. Der Lautstärkewert, den Sie angeben, kann zwischen 0 (stumm) und 1 (volle Lautstärke) liegen und die Skala ist linear, so dass ein Wert von 0,5 der halben Lautstärke entsprechen würde. Beachten Sie, dass sich diese Aktion auf alle Instanzen aller Klänge, die gerade im Raum gespielt werden, und alle zukünftigen Instanzen aller gespielten Klänge beeinflusst. Diese Aktion ist auch unabhängig von der eingestellten Lautstärke für jeden einzelnen Sound, wenn Sie also einen Sound auf 0,5 Lautstärke eingestellt haben eingestellt haben und dann die Master-Lautstärke auf 0,5 einstellen, wird der Sound mit einer Lautstärke von 0,25 abgespielt.

Argumente:

Argument Beschreibung
Band Die neue Master-Lautstärke für alle Töne (von 0 bis 1, Voreinstellung 1)

Beispiel:

Der obige Aktionsblock-Code prüft eine globale Variable, um zu sehen, ob sie wahr oder falsch ist. Wenn sie wahr ist, wird die Master-Lautstärke für das Soundsystem abgerufen und in einer temporären lokalen Variablen gespeichert. Der Wert wird dann überprüft, um festzustellen, ob er ungleich 0,5 ist, und wenn dies nicht der Fall ist, dann wird die Master-Lautstärke des Soundsystems auf 0,5 gesetzt. Wenn die globale Variable auf false ausgewertet wird, dann wird die Master-Lautstärke auf 1 gesetzt.
