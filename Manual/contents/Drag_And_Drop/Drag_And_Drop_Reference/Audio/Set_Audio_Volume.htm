Audio-Lautstärke einstellen

Mit dieser Aktion können Sie die Lautstärke eines bestimmten Tons einstellen. Sie wählen den Ton aus dem Asset-Explorer aus und stellen dann den Lautstärkewert ein. Die Lautstärke kann zwischen 0 (stumm) und 1 (volle Lautstärke) liegen und die Skala ist linear, so dass ein Wert von 0,5 der halbe Lautstärke. Beachten Sie, dass sich diese Aktion auf alle Instanzen des Sounds auswirkt, die derzeit im Raum abgespielt werden, und auf alle zukünftigen Instanzen des abgespielten Sounds.

Aktion Syntax:

Argumente:

Argument Beschreibung
Ton Die Klangressource zum Einstellen der Lautstärke von
Band Die neue Lautstärke für den Ton (von 0 bis 1, Voreinstellung 1)

Beispiel:

Der obige Code des Aktionsblocks prüft eine globale Variable, um zu sehen, ob sie wahr oder falsch ist. Wenn sie wahr ist, wird die Lautstärke des angegebenen Tons abgerufen und in einer temporären lokalen Variablen gespeichert. Der Wert wird dann überprüft, um zu sehen, ob er ungleich 0,5 ist, und wenn nicht, dann wird die Lautstärke des Sounds auf 0,5 gesetzt. Wenn die globale Variable den Wert false, dann wird die Lautstärke des Sounds auf 1 gesetzt.
