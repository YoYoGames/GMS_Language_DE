Wenn Audio wiedergegeben wird

Diese Aktion kann verwendet werden, um zu prüfen, ob eine Instanz einer beliebigen Sound-Ressource gerade in Ihrem Spiel abgespielt wird. Sie geben die zu prüfende Sound-Ressource aus dem Asset-Explorer an und die Funktion gibt truezurück , wenn sie abgespielt wird, und false Die Funktion gibt true zurück, wenn sie abgespielt wird, und false, wenn sie nicht abgespielt wird. Wenn Sie allerdings das Flag "not" markieren, wird dies umgekehrt und die Funktion gibt true zurück, wenn der Sound nicht abgespielt wird, und false , wenn nicht.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, werden nach dem "if"-Block ausgeführt.
