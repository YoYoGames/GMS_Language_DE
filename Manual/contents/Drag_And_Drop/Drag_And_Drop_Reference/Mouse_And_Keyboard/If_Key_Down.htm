Wenn Taste unten

Mit dieser Aktion können Sie die Tastatur abfragen, um zu sehen, ob eine Taste gerade gedrückt gehalten wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt wird, true zurück, andernfalls false. Wenn Sie jedoch auf den Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt wird, und false, wenn dies der Fall ist. Wenn Sie nur auf einen einzelnen Tastendruck prüfen müssen, verwenden Sie die Aktion If Key Pressed.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Alle Aktionen, die an anderer Stelle entfallen, werden nach dem "if"-Block ausgeführt Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Taste Der Name der zu prüfenden Tastaturtaste.
Nicht Negieren der Prüfung (aus true wird false und umgekehrt)
