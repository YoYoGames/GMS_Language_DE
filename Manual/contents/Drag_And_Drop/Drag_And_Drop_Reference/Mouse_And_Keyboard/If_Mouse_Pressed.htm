Wenn die Maus gedrückt wird

Diese Aktion prüft, ob eine Maustaste zum ersten Mal gedrückt wird. Sie gibt bei dem einen Spielschritt (Frame), bei dem die Taste zum ersten Mal gedrückt wird, den Wert true zurück und zu allen anderen Zeitpunkten false. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und prüft, ob die Taste nicht gedrückt wurde, und gibt false bei dem Spielschritt (Frame) zurück, bei dem die Taste gedrückt wurde, und true zu allen anderen Zeitpunkten. Wenn Sie prüfen müssen, ob eine Taste gedrückt gehalten wird und nicht nur nicht nur das anfängliche Druckereignis, dann verwenden Sie die Aktion If Mouse Down.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Maustaste Der Name der zu prüfenden Maustaste (links, Mitte, rechts).
Nicht Negieren der Prüfung (aus true wird false und umgekehrt)

Beispiel:

Der obige Code des Aktionsblocks fragt bei jedem Schritt den Zustand der Maustaste ab, und wenn sie gedrückt gehalten wird, prüft er, ob die Maustaste beim ersten Drücken gedrückt wird. Wenn die Prüfung bei der ersten Betätigung der Maustaste wahr ist, wird die Instanzgeschwindigkeit gesetzt und die Sprite-Überblendung wird auf rot geändert, während die Taste gedrückt ist. Wenn die Maus nicht gedrückt gehalten wird, wird beim Loslassen der Maus eine Prüfung durchgeführt, um die Überblendungsfarbe auf Weiß zurückzusetzen und die Geschwindigkeit auf 0 zu setzen.
