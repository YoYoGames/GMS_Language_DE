Wenn die virtuelle Tastatur angezeigt wird

Mit dieser Aktion kann geprüft werden, ob die virtuelle Tastatur des Betriebssystems angezeigt wird oder nicht. Die Aktion wird als true ausgewertet, wenn die Tastatur offen oder geöffnet ist, oder false, wenn die virtuelle Tastatur geschlossen oder geschlossen ist. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und Sie können den Rückgabewert true / false in einer Zielvariablen speichern, die als temp (lokal) gekennzeichnet werden kann.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if" zu true ausgewertet wird, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.
