Wenn Maus unten

Mit dieser Aktion können Sie die Maus abfragen, um zu sehen, ob die ausgewählte Taste gerade gedrückt gehalten wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt wird, true zurück, andernfalls false. Wenn Sie jedoch auf den Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h., die Aktion gibt true zurück, wenn keine Taste gedrückt ist, und false, wenn sie gedrückt ist. Wenn Sie nur auf einen einzelnen Tastendruck prüfen müssen, verwenden Sie die Aktion If Mouse Pressed.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Maustaste Der Name der zu prüfenden Maustaste (links, Mitte, rechts).
Nicht Negieren der Prüfung (aus true wird false und umgekehrt)

Beispiel:

Der obige Code des Aktionsblocks fragt bei jedem Schritt den Zustand der Maustaste ab, und wenn sie gedrückt gehalten wird, prüft er, ob die Maustaste beim ersten Drücken gedrückt wird. Wenn die Prüfung auf den anfänglichen Tastendruck wahr ist, wird die Instanzgeschwindigkeit gesetzt, dann wird, während die Taste gedrückt ist, die Sprite-Überblendung auf rot geändert. Wenn die Maus nicht gedrückt gehalten wird, wird eine Prüfung beim Loslassen der Maus durchgeführt, um die Überblendungsfarbe auf Weiß zurückzusetzen und die Geschwindigkeit auf 0 zu setzen.
