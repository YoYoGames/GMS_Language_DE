Bei gedrückter Taste

Diese Aktion prüft, ob eine Taste der Tastatur zum ersten Mal gedrückt wird. Sie gibt in dem einen Spielschritt (Frame), in dem die Taste zum ersten Mal gedrückt wird, true zurück und zu allen anderen Zeiten false. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und prüft, ob die Taste nicht gedrückt wurde, und gibt false bei dem Spielschritt (Frame) zurück, bei dem die Taste gedrückt wurde, und true zu allen anderen Zeitpunkten. Wenn Sie prüfen müssen, ob eine Taste gedrückt gehalten wird und nicht nur Wenn Sie prüfen möchten, ob eine Taste gedrückt wird und nicht nur das erste Druckereignis, dann verwenden Sie die Aktion If Key Down.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Alle Aktionen, die an anderer Stelle entfallen, werden nach dem "if"-Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Taste Der Name des zu prüfenden Schlüssels.
Nicht Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Beispiel:

Der obige Code des Aktionsblocks fragt den Tastaturstatus bei jedem Schritt ab und prüft, ob eine Taste gedrückt gehalten wird, prüft er, ob die Taste beim ersten Drücken gedrückt wird. Wenn die Prüfung auf den ersten Tastendruck zutrifft, wird ein Ton abgespielt, und während die Taste gedrückt gehalten wird, wird die Sprite-Überblendung auf rot geändert. Wenn die Taste nicht gedrückt wird, wird eine Prüfung beim Loslassen der Taste durchgeführt, um die Überblendungsfarbe auf Weiß zurückzusetzen.
