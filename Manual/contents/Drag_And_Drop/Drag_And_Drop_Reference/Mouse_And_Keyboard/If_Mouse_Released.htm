Wenn die Maus losgelassen wird

Diese Aktion prüft, wann eine Maustaste losgelassen wurde. Sie gibt bei dem einen Spielschritt (Frame), bei dem die Maustaste losgelassen wird, true zurück und zu allen anderen Zeiten false. Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und prüft, ob die Maustaste nicht losgelassen wurde, und gibt false bei dem Spielschritt (Frame) zurück, bei dem sie losgelassen wurde, und true zu allen anderen Zeiten.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Maustaste Der Name der zu prüfenden Maustaste (links, Mitte, rechts).
Nicht Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Beispiel:

Der obige Code des Aktionsblocks fragt bei jedem Schritt den Zustand der Maustaste ab und wenn sie gedrückt gehalten wird, prüft er, ob die Maustaste beim ersten Drücken gedrückt wird. Wenn die Prüfung auf den anfänglichen Tastendruck wahr ist, wird die Instanzgeschwindigkeit gesetzt, dann wird, während die Taste gedrückt ist, die Sprite-Überblendung auf rot geändert. Wenn die Maus nicht gedrückt gehalten wird wird beim Loslassen der Maus geprüft, ob die Überblendungsfarbe auf Weiß zurückgesetzt und die Geschwindigkeit auf 0 gesetzt wird.
