Virtuelle Tastatur ausblenden

Diese Aktion kann verwendet werden, um die virtuelle Tastatur auf dem Gerät, auf dem das Spiel läuft, auszublenden. Der Aufruf dieser Aktion erzeugt ein asynchrones Systemereignis, bei dem die async_load DS-Map mit den folgenden Schlüssel/Wert-Paaren aufgefüllt wird:

  • " event_type" - dies wird " virtualkeyboard status" sein, wenn es durch Aktionen der virtuellen Tastatur ausgelöst wird.
  • "screen_height" - die Höhe der virtuellen Tastatur (in Pixel). Dieser Wert ist 0, wenn die Tastatur unsichtbar ist.
  • "keyboard_status" - der aktuelle Status der Tastatur, zurückgegeben als eine der folgenden Zeichenketten:
    • "versteckt"
    • "versteckt"
    • "zeigend"
    • "sichtbar"
