Gamepad-Trigger erhalten

Mit dieser Aktion können Sie den Wert der verschiedenen Auslöser eines bestimmten Gamepads abrufen. Sie geben den Gamepad-Index an (dies ist die Nummer des "Steckplatzes", den ein Gamepad belegt) und wählen den zu prüfenden Auslöser (links oder rechts). Der zurückgegebene Wert liegt zwischen 0 und 1, wobei 0 für keinen Druck auf den Trigger und 1 für vollen Druck steht. Dieser zurückgegebene Wert wird in der Zielvariablen gespeichert, die Sie für die zukünftige Verwendung angeben. Beachten Sie, dass diese Funktion die Schwellenwerteinstellung nicht berücksichtigt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Auslöser Der zu prüfende Trigger (links/rechts)
Ziel Die Zielvariable, in der der zurückgegebene Wert gespeichert werden soll.

Beispiel:

Der obige Code des Aktionsblocks erstellt eine lokale (temporäre) Variable und verwendet sie dann zum Speichern den aktuellen Wert für den linken Trigger. Wenn der Triggerwert über 0,5 liegt, wird eine Instanz an der Position der aufrufenden Instanz erzeugt.
