Wenn Gamepad-Taste unten

Mit dieser Aktion können Sie ein Gamepad abfragen, um zu sehen, ob eine bestimmte Taste gerade gedrückt gehalten wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste gedrückt wird, true zurück, andernfalls false, wobei Sie mit dem Modifikator not klicken, können Sie prüfen, ob die Taste nicht gedrückt ist, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt ist, und false, wenn eine Taste gedrückt ist. Wenn Sie nur auf einen einzelnen Tastendruck prüfen müssen, verwenden Sie die Aktion If Gamepad Button Pressed.

Wenn es sich bei der zu prüfenden Taste um eine analoge Taste handelt, wird sie erst dann als gedrückt betrachtet, wenn der Tastenwert über den mit der Aktion Gamepad-Tastenschwelle festlegen eingestellten Schwellenwert gestiegen ist.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Taste Der Name der zu prüfenden Schaltfläche.
Nicht Negieren der Prüfung (aus wahr wird falsch und umgekehrt)

Beispiel:

Der obige Code des Aktionsblocks fragt bei jedem Schritt den Tastenzustand des Gamepads 0 ab und wenn eine Taste gedrückt gehalten wird, prüft er, ob die Taste beim ersten Drücken gedrückt wird. Wenn die Prüfung auf den ersten Tastendruck zutrifft, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt gehalten wird. Wenn die Taste nicht gedrückt wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
