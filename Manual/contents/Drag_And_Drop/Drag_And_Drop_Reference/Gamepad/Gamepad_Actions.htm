Gamepad-Aktionsbibliothek

GameMaker Studio 2 verfügt über eine Reihe spezieller Aktionen, mit denen Sie sowohl analoge als auch digitale Steuerelemente von mehreren angeschlossenen Gamepads erkennen können. Für diese Aktionen müssen Sie einen Gamepad-Indexwert angeben, d. h. eine Zahl (von 0 an gezählt), die das ausgewählte Gamepad darstellt. Beachten Sie, dass, wenn ein Gamepad an Ihr Gerät angeschlossen wird (oder es entfernt wird), dann wird ein asynchrones Systemereignis ausgelöst (um damit umzugehen, müssen Sie jedoch möglicherweise Code verwenden).

Die Gamepad-"Steckplätze" sind von 0 an indiziert, wobei die Steckplätze 0 bis einschließlich 3 nur für Xinput-Gamepads sind , d. h. Xbox360-Controller und kompatible Geräte. Sie können jedoch auch die Steckplätze 4 bis einschließlich 11 für DirectInput-Gamepads überprüfen, was bedeutet, dass Sie viele andere Controller-Modelle erkennen können, wenn sie über diese Steckplätze angeschlossen sind.

Eingangskonstanten

Wenn Sie mit den Gamepad-Aktionen arbeiten, können die Eingaben von Achsen, Tasten oder Hats kommen, die GameMaker Studio 2 den folgenden eingebauten Konstanten zuordnet (beachten Sie, dass "Hats" in der Regel nur bei Nicht-Standard-Controllern erkannt werden):

Konstant Beschreibung
gp_face1 Obere Taste 1 (diese entspricht dem "A" auf einem Xbox 360-Controller und dem Kreuz auf einem PS-Controller )
gp_face2 Obere Taste 2 (diese entspricht dem "B" auf einem Xbox 360-Controller und dem Kreis auf einem PS-Controller )
gp_face3 Obere Taste 3 (diese entspricht dem "X" auf einem Xbox 360-Controller und dem Quadrat auf einem PS-Controller )
gp_face4 Obere Taste 4 (dies entspricht dem "Y" auf einem Xbox 360-Controller und dem Dreieck auf einem PS-Controller )
gp_Schulterl Linke Schultertaste
gp_Schulterlb Linker Schulterauslöser
gp_shoulderr Rechte Schultertaste
gp_Schulterrb Rechter Schulterauslöser
gp_select Die Auswahltaste (dies ist die PS-Taste auf einem DS4-Controller)
gp_start Die Start-Taste (dies ist die "Optionen"-Taste auf einem PS4-Controller)
gp_stickl Der linke Stick gedrückt (als Taste)
gp_stickr Der rechte Stick gedrückt (als Taste)
gp_padu D-Pad nach oben
gp_padd D-Pad nach unten

gp_axisrh Rechter Knüppel horizontale Achse (analog)

Die folgende Liste zeigt die aktuelle Kompatibilität zwischen den Plattformen (beachten Sie, dass sich diese mit zukünftigen Updates ändern wird):

  • Windows wird vollständig unterstützt, wobei bis zu maximal 11 angeschlossene Geräte gleichzeitig zulässig sind. Beachten Sie, dass unter Windows die ersten 4 Gamepad-Steckplätze (0 - 3) über die XInput dll verwaltet werden, was bedeutet, dass nur XBox-Controller zu 100 % kompatibel sein können und dass Sie für andere Controller-Typen die restlichen Gamepad-Steckplätze (4 - 11) überprüfen sollten.
  • macOS wird unterstützt, wobei maximal 4 angeschlossene Geräte auf einmal erlaubt sind und diese Geräte NUR vom Typ Playstation3oder Xbox 360 sein können. Bitte beachten Sie, dass die Option " Buildfor Mac AppStore" in Mac Game Optionen ausgeschaltet sein muss, damit die Pad-Unterstützung funktioniert.
  • Ubuntu unterstützt auch die GamePad-Eingabe, aber Sie müssen möglicherweise zusätzliche Bibliotheken aus dem Ubuntu-Repository installieren. Sie können dies ganz einfach tun, indem Sie ein Kommandozeilenterminal öffnen und Folgendes eingeben, wodurch die GUI-Unterstützung für den Joystick sowie den Joystick selbst installiert:

sudo apt-get install jstest-gtk
sudo apt-get install joystick.

  • HTML5-Spiele unterstützen Gamepads in den meisten gängigen Browsern, außer Safari.
  • Die Gamepad-Unterstützung erstreckt sich auch auf iOS mit dem iCade-Gehäuse. Die linke Achse entspricht dem Stick-Controller (obwohl der Eingang digital und nicht analog ist), die vier "Face"-Tasten entsprechen den Tasten auf der Vorderseite des Gehäuses, und die vier Schultertasten entsprechen denen auf der Rückseite des Gehäuses. vier Schultertasten entsprechen den Tasten auf der Rückseite des Gehäuses.
  • Android export unterstützt NYKO-Controller und generische Bluetooth-Controller, aber nur, wenn sie aktiviert sind, d. h. Sie müssen die Option "iCade/Bluetooth" im Abschnitt "Allgemein" in den Android-Spieloptionen aktivieren. Sie benötigen API-Level 12, um vollständig zu funktionieren, und es ist zu beachten, dass GameMaker Studio 2 alle Bluetooth-Geräte, mit denen Ihr Gerät gekoppelt ist, als verbunden registriert, unabhängig davon, ob es tatsächlich verbunden ist. Daher sollte dies bei der Zuweisung und Überprüfung von "Slots" berücksichtigt werden.
  • Auf der PS4 müssen Sie, wenn Sie das Touchpad-Tracking verwenden möchten, die Funktionen für die gerät_maus_* Tasten (dafür gibt es keine DnD™-Aktionen).
