Wenn die Gamepad-Taste gedrückt wird

Mit dieser Aktion können Sie das Gamepad abfragen, um zu sehen, ob gerade eine Taste gedrückt wird. Die Aktion gibt bei jedem Schritt, bei dem die Taste des Gamepads gedrückt (und gehalten) wird, true zurück, andernfalls false, obwohl Sie mit dem not-Modifikator klicken, können Sie prüfen, ob die Taste nicht gedrückt wird, d. h. die Aktion gibt true zurück, wenn keine Taste gedrückt wird, und false, wenn eine Taste gedrückt wird. Wenn Sie nur auf einen einzelnen Tastendruck prüfen müssen, dann verwenden Sie die Aktion if gamepad button down.

Wenn es sich bei der zu prüfenden Taste um eine analoge Taste handelt, wird die Betätigung erst dann erkannt, wenn der Tastenwert über den mit der Aktion Gamepad-Tastenschwelle festlegen eingestellten Schwellenwert gestiegen ist.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if" als wahr ausgewertet wird, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Taste Der Name der zu prüfenden Schaltfläche.
Nicht Negieren der Prüfung (aus true wird false und umgekehrt)

Der obige Code des Aktionsblocks fragt den Tastenzustand des Gamepads 0 bei jedem Schritt ab und wenn eine Taste gedrückt gehalten wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Wenn die Prüfung auf den ersten Tastendruck zutrifft, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt wird. Wenn die Taste nicht gedrückt wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
