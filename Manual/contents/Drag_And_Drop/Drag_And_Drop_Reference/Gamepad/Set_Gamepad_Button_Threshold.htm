Gamepad-Tastenschwellwert einstellen

Mit dieser Aktion können Sie den "Schwellenwert" der analogen Tasten des Gamepads einstellen. Sie geben den Gamepad-Index an, der eingestellt werden soll, und stellen dann einen Wert von 0 bis 1 ein, und wenn der Betrag der analogen Tasteneingabe niedriger als der angegebene Wert ist, wird die Gamepad-Taste Beachten Sie, dass dies eine globale Einstellung ist, die sich auf alle analogen Tasten auswirkt, die an das Gamepad angeschlossen sind. Dieser Wert wird bei allen Prüfungen auf "gedrückt", " gedrückt" und " losgelassen" für das angegebene Gamepad verwendet, wird aber von der Aktion " Get Gamepad Trigger" ignoriert.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Totzone Der Schwellenwert (0 - 1)
