Gamepad-Anzahl abrufen

Mit dieser Aktion können Sie die Anzahl der Gamepads ermitteln, die mit dem Gerät verbunden sind, auf dem Ihr Spiel läuft. Die zurückgegebene Zahl liegt zwischen 0 und 12.

Aktion Syntax:

Argumente:

Argument Beschreibung
Ziel Die Zielvariable, in der der zurückgegebene Wert gespeichert werden soll.

Beispiel:

Der obige Aktionsblockcode ermittelt die Anzahl der angeschlossenen Gamepads und weist den Wert einer eine temporäre lokale Variable zu. Diese wird dann überprüft, ob sie größer als 0 ist, und eine globale Variable entsprechend gesetzt.
