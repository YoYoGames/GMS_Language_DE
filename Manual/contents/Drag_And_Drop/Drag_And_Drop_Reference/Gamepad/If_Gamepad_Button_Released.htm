Wenn Gamepad-Taste losgelassen wird

Diese Aktion prüft, wann eine Gamepad-Taste für den angegebenen Gamepad-Index losgelassen wurde. Sie gibt true zurück, wenn die Taste bei einem einzigen Spieltick losgelassen wird, und false zu allen anderen Zeiten. Wenn Sie den Modifikator not aktivieren, Wenn Sie den Modifikator not aktivieren, wird diese Aktion umgekehrt und prüft, ob die Taste nicht losgelassen wurde, und gibt false bei dem Spieltick zurück, bei dem sie losgelassen wurde, und true zu allen anderen Zeiten.

Wenn es sich bei der zu prüfenden Taste um eine analoge Taste handelt, wird das Loslassen erst dann erkannt, wenn der Tastenwert unter den mit der Aktion Gamepad-Tastenschwelle festlegen eingestellten Schwellenwert gefallen ist.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block zu wahr ist, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if"-Block ausgeführt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Gamepad Der Gamepad-Index.
Taste Der Name der zu prüfenden Schaltfläche.
Nicht Negieren der Prüfung (aus true wird false und umgekehrt)

Der obige Code des Aktionsblocks fragt den Tastenzustand des Gamepads 0 bei jedem Schritt ab und wenn eine Taste gedrückt gehalten wird, prüft er, ob die Taste zum ersten Mal gedrückt wird. Wenn die Prüfung auf den ersten Tastendruck zutrifft, wird ein Ton abgespielt und die Instanzgeschwindigkeit auf 10 gesetzt, während die Taste gedrückt wird. Wenn die Taste nicht gedrückt wird wird beim Loslassen der Taste eine Prüfung durchgeführt, um die Geschwindigkeit wieder auf 0 zurückzusetzen.
