Schalter-Aktionsbibliothek

Die Switches-Aktionsbibliothek enthält die Aktionen, die für die Erstellen von Switch-Anweisungen in Ihrem Spiel-Aktionscode. In einer Reihe von Situationen möchten Sie Ihre Instanzen eine Aktion in Abhängigkeit von einem bestimmten Wert ausführen lassen, und obwohl Sie dies mit einer Reihe von aufeinanderfolgenden "if"-Anweisungen tun können, kann dies Dies kann jedoch unübersichtlich werden, wenn die Anzahl der möglichen Optionen zwei oder drei übersteigt. In diesen Fällen ist es besser, die "switch"-Aktion zu verwenden. Eine switch-Aktion funktioniert wie folgt:

  • Sie liefern den Wert a (normalerweise aus einer Art Ausdruck oder Variable)
  • Der Wert wird mit dem Wert verglichen, der den verschiedenen "case"-Anweisungen zugeordnet ist
  • Ist der Wert gleich dem Fallwert, wird der Inhalt des Falles ausgeführt, ansonsten wird der nächste Fall ausgewertet, usw..
  • Sobald ein Fall ausgeführt wurde oder alle Fälle ausgewertet wurden und fehlgeschlagen sind, wird der Code vom Ende des Schalters
  • Wenn alle Fälle fehlgeschlagen sind und Sie einen Standardfall haben, dann wird dieser ausgeführt, bevor der Code ab dem Ende des Schalters weiterläuft

Die Bibliothek " Schalter" verfügt über die folgenden Aktionen:


Schalter
Gehäuse
Standard
