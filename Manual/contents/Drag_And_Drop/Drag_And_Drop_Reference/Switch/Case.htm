Gehäuse

Diese Aktion erzeugt einen neuen Fall innerhalb einer Schalter. Sie geben den Wert für den Fall an und dieser wird mit dem Eingabewert für den Schalter verglichen, und wenn sie übereinstimmen, werden die unter dem Fall hinzugefügten Aktionen ausgeführt. Aktionen, die zu einem Case hinzugefügt werden, um nur in diesem Case ausgeführt zu werden, sollten an der Seite der Aktion abgelegt werden, ähnlich wie Sie eine Aktion zu einem "wenn" hinzufügen würden:

Beachten Sie, dass ein Case-Wert eine Konstante sein muss, die ein beliebiger Datentyp sein kann (Real, String, etc...), solange er beim Kompilieren zu einem einzigen Wert ausgewertet wird - Sie können also keine Ausdrücke verwenden, die von Variablen abhängen. Beachten Sie auch, dass Case nicht außerhalb eines Switches verwendet werden kann und der Versuch, dies zu tun, zu einem Fehler führt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Konstant Der konstante Wert, der gegen den Eingangswert Switch ausgewertet werden soll

Beispiel:

Der obige Aktionsblockcode ruft den in einer Variablen gespeicherten Wert ab und vergleicht ihn dann mit den verschiedenen möglichen Fällen. Wenn einer der Fälle mit dem Wert übereinstimmt, dann wird die Aktion unter diesem Fall ausgeführt, andernfalls wird der nächste Fall ausgewertet. Nachdem der Schalter gelaufen ist, wird ein Alarm auf 30 gesetzt.
