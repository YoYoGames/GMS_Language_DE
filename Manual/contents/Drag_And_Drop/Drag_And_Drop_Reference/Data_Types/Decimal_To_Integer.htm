Dezimal zu Integer

Diese Aktion nimmt eine Dezimalzahl und wandelt sie in eine Ganzzahl um. Sie geben den zu konvertierenden Wert an und wählen dann die Art der Rundung, die durchgeführt werden soll, um ihn in einen Ganzzahlwert umzuwandeln. Die verfügbaren Rundungstypen sind:

  • Runden: Nimmt eine Dezimalzahl und rundet sie auf die nächste ganzzahlige Zahl auf oder ab. In dem speziellen Fall, dass die übergebene Zahl genau eine halbe ganze Zahl ist (z. B. 1,5, 17,5, -2,5 usw.), wird die Zahl auf den nächsten geraden Wert gerundet. Die Zahl wird auf den nächsten geraden Wert gerundet, z. B. 2,5 wird auf 2 gerundet, während 3,5 auf 4 gerundet wird. Diese Art der Rundung wird Banker-Rundung genannt und ergibt bei einer großen Anzahl von Iterationen oder bei der Verwendung von Fließkomma-Mathematik eine statistisch bessere Rundung als die traditionellere eine statistisch bessere Rundung als der traditionellere Ansatz "aufrunden, wenn über 0,5 und sonst abrunden".
  • Ceil: Nimmt einen Dezimalwert und rundet ihn auf die nächste Ganzzahl auf. Im Gegensatz zu round (oben) wird hier der Dezimalteil des Wertes nicht berücksichtigt, so dass eine Zahl wie 5.0000001 immer noch auf 6 aufgerundet würde. Es ist erwähnenswert Wenn Sie die Zufallsaktion " Zufallszahl ermitteln " verwenden, um einen zufälligen Dezimalwert zu generieren, und dann diese Aktion aufrufen, um ihn aufzurunden, können Sie unerwartete Ergebnisse erhalten. Die Zufalls Aktion könnte möglicherweise 0 zurückgeben, und wenn Sie "Ceil" darauf anwenden, erhalten Sie nicht 1, sondern 0. Wenn Sie also eine zufällige Dezimalzahl von 0 bis 9 erhalten und diese Funktion verwenden, erwarten Sie möglicherweise, dass Sie ganze Zahlen von 1 bis 10 erhalten, aber in Wirklichkeit könnten Sie ganze Zahlen von 0 bis 10 erhalten. Dies ist eine entfernte Möglichkeit, sollte aber bei der Verwendung dieser Aktion berücksichtigt werden.
  • Stockwerk: Dies nimmt einen Dezimalwert und rundet ihn auf den nächsten ganzzahligen Wert ab. Im Gegensatz zum Runden (oben) wird der Dezimalteil des Wertes nicht berücksichtigt, so dass eine Zahl wie 5,9999999 trotzdem auf 5.

Nachdem Sie den Rundungstyp ausgewählt haben, müssen Sie die zu rundende Zahl angeben (dies kann eine Variable sein) und dann eine Zielvariable, an die der neue Ganzzahlwert zurückgegeben werden soll (die als temporäre lokale Variable gekennzeichnet sein kann).

Aktion Syntax:

Argumente:

Argument Beschreibung
Abrunden Die Art der Rundung, die verwendet wird, um die Ganzzahl zu erhalten (siehe Beschreibung oben)
Nummer Die Dezimalzahl zum Runden auf eine ganze Zahl
Ziel Die Zielvariable zum Speichern der zurückgegebenen Ganzzahl
