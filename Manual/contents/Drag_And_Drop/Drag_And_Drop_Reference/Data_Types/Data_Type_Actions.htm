Datentypen Aktionsbibliothek

GameMaker Studio 2 hat eine Reihe verschiedener Datentypen, wie Integer, Strings, Pointer, etc... und einige davon können in andere konvertiert werden. Dieser Abschnitt des Handbuchs zeigt die verschiedenen Aktionen, die für den Umgang mit und die Konvertierung von verschiedenen Datentypen zur Verfügung stehen.

HINWEIS: Mehr über die verschiedenen Datentypen erfahren Sie hier.

Die Typenbibliothek verfügt über die folgenden Aktionen:

Dezimal zu Integer
Zeichenfolge in Zahl
Nummer in Zeichenkette
