Wenn die Sequenz vorhanden ist

Mit dieser Aktion kann geprüft werden, ob das angegebene Sequenzelement im Spielraum vorhanden ist. Sie geben die Element-ID an, wie sie von der Aktion " Sequenz erstellen" zurückgegeben wird, und wenn das Sequenzelement im Raum existiert, dann die Aktion " true" zurück, andernfalls wird "false" zurückgegeben.

Wenn Sie das Argument " Not" markieren, prüft die Aktion, ob kein solches Sequenzelement vorhanden ist, und gibt, wenn keines gefunden wird, true zurück.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if" Block, diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if" zu true ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, nach dem "if" Block ausgeführt werden.
