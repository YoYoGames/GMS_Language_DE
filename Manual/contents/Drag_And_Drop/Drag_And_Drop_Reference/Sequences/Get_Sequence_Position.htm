Sequenzposition holen

Mit dieser Aktion können Sie die aktuelle Position für das angegebene Sequenzelement im Raum abrufen. Sie geben die ID des Sequenzelements an (wie sie von der Aktion " Sequenz erstellen" zurückgegeben wird), und diese Aktion liefert seine X- und Y-Position in Zielvariablen zurück. Beachten Sie, dass die Zielvariablen als temporäre (lokale) Variablen gekennzeichnet werden können, die für die Aktion erstellt werden und in allen nachfolgenden Aktionen für das Ereignis, in dem sich die Aktion befindet, verwendet werden können.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variabel Die ID des Sequenzelements, dessen Position ermittelt werden soll.
Ziel X Die Zielvariable zur Aufnahme der x-Position.
Ziel Y Die Zielvariable zur Aufnahme der y-Position.

Beispiel:

Der obige Aktionsblockcode ruft die aktuelle Position des Sequenzelements ID ab, die in der Variablen "my_seq" gespeichert ist. Diese Werte werden dann verwendet, um die Position der Sequenz festzulegen und sie um 10 Pixel nach rechts zu verschieben.
