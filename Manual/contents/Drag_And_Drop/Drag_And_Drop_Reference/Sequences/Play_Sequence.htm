Abspielsequenz

Mit dieser Aktion können Sie die Wiedergabe einer (zuvor angehaltenen) Sequenz starten. Sie geben die ID des Sequenzelements an (wie von der Aktion " Sequenz erstellen" zurückgegeben), und die Aktion startet die Wiedergabe.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variabel Die ID des abzuspielenden Sequenzelements.

Beispiel:

Der obige Code des Aktionsblocks prüft, ob eine globale Variable wahr ist. Wenn dies der Fall ist, wird das in der Variable "my_seq" referenzierte Sequenzelement angehalten, andernfalls wird es auf Wiedergabe gesetzt.
