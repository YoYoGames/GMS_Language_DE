Sequenzposition einstellen

Mit dieser Aktion können Sie die Position für das angegebene Sequenzelement im Raum festlegen. Sie geben die ID des Sequenzelements an (wie von der Aktion " Sequenz erstellen" zurückgegeben) und dann die neue X- und Y-Position. Wenn Sie das Kontrollkästchen "relativ" ankreuzen, dann werden die angegebenen Werte für X und Y zur aktuellen Sequenzposition addiert, andernfalls handelt es sich um eine absolute Position.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variabel Die ID des Sequenzelements, dessen Position gesetzt werden soll.
X Die neue X-Position im Raum für die Sequenz.
Y Die neue y-Position im Raum für die Sequenz.

Beispiel:

Der obige Aktionsblockcode ruft die aktuelle Position des Sequenzelements ID ab, die in der Variablen "my_seq" gespeichert ist. Diese Werte werden dann verwendet, um die Position der Sequenz festzulegen und sie um 10 Pixel nach rechts zu verschieben.
