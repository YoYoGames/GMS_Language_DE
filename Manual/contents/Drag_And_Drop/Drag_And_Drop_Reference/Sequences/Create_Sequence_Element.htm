Sequenzelement erstellen

Mit dieser Aktion können Sie ein Sequenzelement einer beliebigen Sequenz aus dem Asset-Browser erstellen. Sie geben den Namen der Sequenz und eine x/y-Position innerhalb des Raums an, an der sie erstellt werden soll, sowie die Ebene, auf der sie erstellt werden soll (diese Ebene muss eine Asset-Ebene sein). Sie können die "Relativ"-Flags ankreuzen, um die Sequenz relativ zu der Instanz zu positionieren, die die Aktionscodeblöcke ausführt. Wenn Sie eine Sequenz im Spiel auf diese Weise erstellen, erhält sie einen eindeutigen Element-ID-Wert mit dem Sie die Sequenz dann in weiteren Aktionen identifizieren können. Daher erlaubt Ihnen diese Aktion, eine "Ziel"-Variable zu definieren, die diesen ID-Wert enthält (wie im Beispiel unten gezeigt). Beachten Sie, dass das Ziel als eine temporäre (lokale) Variable gekennzeichnet werden kann, die für die Aktion erstellt wird und in allen nachfolgenden Aktionen für das Ereignis, in dem sich die Aktion befindet, verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sequenz Der Name der Sequenz, von der ein Element erstellt werden soll.
x Die x-Position im Raum, um die Sequenz zu erstellen (kann relativ zur aufrufenden Instanz sein).
y Die y-Position im Raum, um die Sequenz zu erstellen (kann relativ zur aufrufenden Instanz sein).
Ebene Der Name (eine Zeichenkette) der Asset-Ebene, auf der die Sequenz erstellt werden soll.
Ziel Die Variable, die als Ziel für den Rückgabewert dienen soll.

Beispiel:

Der obige Aktionsblockcode erstellt eine neue Sequenz und speichert ihren eindeutigen Element-ID-Wert in einer Variablen. Diese Variable wird dann zum Anhalten der Sequenz mit der Aktion "Sequenz anhalten " verwendet.
