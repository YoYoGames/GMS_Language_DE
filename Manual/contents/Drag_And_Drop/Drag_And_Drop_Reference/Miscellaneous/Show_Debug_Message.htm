Debug-Meldung anzeigen

Mit dieser Aktion können Sie eine Debug-Meldung an den Compiler-Ausgang senden. Diese Meldung kann entweder eine Zeichenkette oder eine reelle Zahl sein. Wenn Sie jedoch beides verwenden möchten, muss die Zahl in eine Zeichenkette umgewandelt werden (siehe die Aktion Number To String). Beachten Sie, dass Sie mit dieser Aktion mehrere Meldungen ausgeben können, indem Sie auf das Symbol klicken, um die Aktion mit weiteren Ausgabefeldern zu erweitern.
