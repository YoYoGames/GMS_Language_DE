Spiel laden

Mit dieser Aktion wird ein Spiel geladen, das zuvor mit der Aktion " Spiel speichern" gespeichert wurde. Sie geben den Dateinamen der zu ladenden Datei an (als String und einschließlich der Erweiterung) und das Spiel wird geladen und läuft weiter von dem gespeicherten Punkt aus. Beachten Sie, dass dies nicht als universelles Speicher-/Ladesystem konzipiert ist und der Versuch, ein Spiel zu laden, das in einem früheren Durchlauf des Projekts gespeichert wurde, zu Fehlern führen kann (vor allem, wenn Sie Dinge wie Datenstrukturen oder Partikel verwendet haben), und daher sollten Sie nur versuchen, Spiele zu laden, die im Verlauf eines einzelnen Durchlaufs gespeichert wurden (für Dinge wie Checkpoints). Beachten Sie auch, dass die Datei erst am Ende des aktuellen Ereignisses oder Skripts geladen wird, so dass alle Aktionen nach dem Aufruf der Ladeaktion noch ausgeführt werden. Einen umfassenderen Ansatz zum Laden gespeicherter Spieldaten finden Sie im Abschnitt " Datei-Aktionen".
