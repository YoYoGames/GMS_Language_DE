Beenden des Spiels

Mit dieser Aktion können Sie das Spiel beenden, indem Sie das Ereignis "Other - Game End" auslösen. Beachten Sie, dass dies nicht in dem Moment geschieht, in dem die Aktion aufgerufen wird, sondern erst am Ende des aktuellen Ereignisses oder Skripts, so dass alle Aktionen, die Sie nach dem Aufruf dieser Aktion haben, weiterhin ausgeführt werden, bis das Ende des Ereignisses/Skripts erreicht ist.

Bitte beachten Sie, dass diese Aktion die folgenden Einschränkungen hat:

  • Es meldet einen Fehler an das iOS Ziel, da es gegen die Bedingungen ihres Marktplatzes verstößt.
  • Es hat keine Auswirkungen, wenn es auf den HTML5- und Windows UWP Ziele.
  • Es funktioniert auf Windows (einschließlich der für Steam erstellten Spiele), Android, Ubuntu (Linux ) und Mac.
