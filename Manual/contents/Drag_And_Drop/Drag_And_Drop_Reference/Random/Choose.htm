Wählen Sie

Mit dieser Aktion können Sie eine Reihe von verschiedenen Optionen angeben (dies kann eine reale Zahl, eine Zeichenkette, eine Ressourcen-ID oder eine Konstante sein), aus denen die Aktion eine zur Rückgabe auswählt. Sie können zusätzliche Optionen hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken und einen weiteren Wert angeben, der zurückgegeben werden kann. Der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre lokale Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren).

Aktion Syntax:

Argumente:

Argument Beschreibung
Option Die Option, die von der Aktion "Wählen" zurückgegeben werden kann.

Beispiel:

Der obige Aktionsblockcode wählt eine von vier Raumressourcen aus und speichert die zurückgegebene ID in einer temporären lokalen Variable. Diese wird dann verwendet, um den Raum zu wechseln.
