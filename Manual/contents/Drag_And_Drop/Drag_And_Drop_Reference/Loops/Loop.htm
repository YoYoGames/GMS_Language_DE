Schleife

Die Loop-Aktion führt eine Endlosschleife aus, bis eine Break-Aktion aufgerufen wird. Bei dieser Aktion sollte darauf geachtet werden, dass unabhängig von den in der Schleife platzierten Aktionen immer ein Break aufgerufen wird, da sonst andernfalls wird das Spiel auf dem Ziel-Betriebssystem einen "Nicht reagierenden" Fehler verursachen und nicht mehr funktionieren. Sie fügen diese Aktion in den Aktionscode-Arbeitsbereich ein und fügen dann die Aktionen hinzu, die innerhalb der Schleife ausgeführt werden sollen, ähnlich wie Sie eine Aktion zu einem "if" hinzufügen würden, d. h., Sie fügen sie neben der Aktion ein und nicht unter ihr:
