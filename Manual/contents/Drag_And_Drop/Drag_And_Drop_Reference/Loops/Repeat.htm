Wiederholung

Diese Aktion wiederholt alle ihr hinzugefügten Aktionen so oft, wie angegeben. Sie geben an, wie oft die Aktionen wiederholt werden sollen, und fügen dann die Aktion zur Wiederholungsschleife hinzu, ähnlich wie Sie eine Aktion zu einem "wenn" hinzufügen würden, d. h.: Sie fügen die Aktion Sie wird neben der Aktion eingefügt und nicht unter ihr:

Alle auf der Seite hinzugefügten Aktionen werden in die Wiederholung einbezogen und ausgeführt, bevor das Ereignis oder Skript mit den darunter hinzugefügten Aktionen fortfährt (wie im folgenden Beispiel gezeigt). Beachten Sie, dass Sie jede Schleife vorzeitig beenden können, wenn Sie die Aktion " Abbrechen " aufrufen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Zeiten Die Anzahl der Wiederholungen (ein ganzer Integer-Wert)

Beispiel:

Der obige Code des Aktionsblocks ermittelt die Anzahl der Einträge in einer Liste und speichert den Wert in einer temporären lokalen Variablen. Diese Variable wird dann verwendet, um die Anzahl der Durchläufe einer Wiederholungsschleife festzulegen. In der Wiederholungsschleife werden die Einträge aus der Liste in eine andere temporäre Variable eingelesen, die dann verwendet wird, um die Instanzen zu zerstören, auf die sie verweisen. Nach Beendigung der Schleife beendet ist, wird die Datenstruktur freigegeben.
