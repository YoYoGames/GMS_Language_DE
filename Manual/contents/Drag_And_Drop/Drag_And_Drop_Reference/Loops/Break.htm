Pause

Diese Aktion kann nur innerhalb einer Schleifenaktion (z. B. Repeat oder While) verwendet werden, und ihr Aufruf bricht die Schleife in diesem Moment ab. Normalerweise wird eine Schleife entweder eine feste Anzahl von Malen ausgeführt oder bis eine bestimmte Bedingung erfüllt ist, aber Sie können die Aktion "Abbrechen" verwenden, um eine Schleife zu jedem beliebigen Zeitpunkt vorzeitig zu beenden (Aktionen innerhalb der Schleife nach dem Abbrechen werden nicht ausgeführt).
