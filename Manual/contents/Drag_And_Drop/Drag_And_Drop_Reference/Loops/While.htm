Während

Die While-Aktion wird verwendet, um Schleifen auszuführen, die nur beendet werden, wenn eine bestimmte Bedingung erfüllt ist. Sie können z. B. eine Variable verwenden und sie auf "false" setzen, dann die While-Schleife ausführen und diese Variable überprüfen lassen, dann zu irgendwann in der Schleife setzen Sie die Variable auf true und die Schleife wird beendet. Beachten Sie, dass unabhängig davon, wo Sie die Bedingung innerhalb der while-Schleife setzen, alle Aktionen innerhalb der Schleife bis zum Ende ausgeführt werden, bevor der Rest des Skripts oder Ereignis fortgesetzt wird, das Ändern der Bedingung führt nicht zum sofortigen Verlassen der Schleife.

Wenn Sie eine While-Aktion in Ihren Code einfügen, müssen Sie die zu prüfende Variable, die Art der Prüfung (kleiner als, größer als, gleich, etc...) und den Wert, gegen den die Variable geprüft werden soll, angeben (Sie können auch markieren, dass die While Schleife nicht prüft, ob die Bedingung nicht wahr ist). Dann fügen Sie die Aktionen hinzu, die innerhalb der While-Schleife ausgeführt werden sollen, ähnlich wie Sie eine Aktion zu einem "if" hinzufügen würden, d. h.: Sie fügen sie an der Seite der Aktion ein und nicht unter ihr:

Alle zur Seite hinzugefügten Aktionen werden in das While eingeschlossen und werden ausgeführt, bevor das Ereignis oder Skript mit den darunter hinzugefügten Aktionen fortfährt (wie im Beispiel unten gezeigt). Beachten Sie, dass Sie jede Schleife vorzeitig beenden können, wenn Sie die Break-Aktion aufrufen.

Argumente:

Argument Beschreibung

Ist Der Typ der durchzuführenden Auswertung (kleiner als, größer als, gleich, größer oder gleich und kleiner oder gleich)
Wert Der Wert, gegen den die Variable geprüft werden soll
