Für

Die For-Aktion wird verwendet, um Schleifen auszuführen, die nur beendet werden, wenn ein Wert die angegebene Bedingung erreicht. Sie geben einen Anfangswert als Variable an (die als temporäre lokale Variable gekennzeichnet werden kann - das ist normalerweise das, was Sie wollen), dann geben Sie die Bedingung, gegen die die Variable geprüft werden soll (z. B. ob sie kleiner als ein bestimmter Wert ist), und schließlich den Betrag, um den die Variable bei jeder Iteration der Schleife erhöht werden soll. Der allgemeine Ablauf ist also wie folgt:

  • Sie initialisieren eine Variable auf einen Wert
  • Sie legen die Bedingung fest, mit der diese Variable geprüft werden soll
  • Sie stellen ein, um wie viel die Variable bei jeder Iteration der Schleife inkrementiert werden soll
  • Sie fügen dann Aktionen in die Schleife ein, und diese werden bei jeder Iteration ausgeführt, bis die Bedingung false zurückgibt oder eine Break-Aktion auftritt.

Beim Hinzufügen von Aktionen, die innerhalb der Schleife ausgeführt werden sollen, werden diese so hinzugefügt, wie Sie eine Aktion zu einem "wenn" hinzufügen würden, d. h.: Sie werden an der Seite der Aktion und nicht darunter eingefügt:

Alle auf der Seite hinzugefügten Aktionen werden in das For aufgenommen und ausgeführt, bevor das Ereignis oder Skript fortgesetzt wird mit den darunter hinzugefügten Aktionen. Beachten Sie, dass Sie jede Schleife vorzeitig beenden können, wenn Sie die Aktion "Break " aufrufen, und wenn Sie For-Schleifen verwenden, müssen Sie nicht inkrementieren und können sie auch dekrementieren lassen, solange die Bedingung die Sie verwenden, angemessen ist. Setzen Sie z. B. die Anfangsvariable auf 10, setzen Sie die Bedingung so, dass sie auf weniger als 0 prüft und setzen Sie dann den Inkrementwert auf -1 und die Schleife wird 10-mal iterieren, wobei sie von 10 bis 0 rückwärts zählt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Initialisieren Initialisieren Sie die Variable, die zum Zählen verwendet werden soll, auf einen Wert (kann als temporäre lokale Variable gekennzeichnet werden)
Zustand Die zu prüfende Bedingung, um die Schleife zu unterbrechen
Inkrement Der Betrag, um den die anfängliche Zählvariable bei jeder Iteration der Schleife erhöht (oder erniedrigt) wird
