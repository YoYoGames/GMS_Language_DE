Schleifen-Aktionsbibliothek

Die Loop-Aktionsbibliothek enthält die Aktionen, die zum Erstellen von Schleifen in Ihrem Spiel-Aktionscode erforderlich sind code. Eine Schleife ist einfach eine Möglichkeit, denselben Satz von Aktionen in Abhängigkeit von bestimmten Bedingungen ein- oder mehrmals zu wiederholen, und ist in jeder Situation nützlich, in der Sie dieselben Aktionen immer wieder wiederholen müssen.

Die Loop-Bibliothek verfügt über die folgenden Aktionen:

Schleife
Wiederholen Sie
Während
Für
Pause
