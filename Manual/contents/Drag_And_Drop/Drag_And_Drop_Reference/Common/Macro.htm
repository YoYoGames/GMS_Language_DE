Makro

Diese Aktion erlaubt es Ihnen, ein oder mehrere Makros zu definieren. Makros sind einfach eindeutige Konstantenwerte - oder Ausdrücke, die zu einem eindeutigen Konstantenwert ausgewertet werden -, die dann in ähnlicher Weise wie eine reguläre Konstante verwendet werden können. Sie können auch einen Funktionsaufruf verwenden, um einen einzelnen Rückgabewert zu erzeugen; in diesem Fall wertet das Makro die Funktion bei jedem Aufruf aus. Beachten Sie, dass Makros einen globalen Gültigkeitsbereich haben, d. h. sie können nach ihrer Deklaration von jeder Instanz, jedem Skript oder Aktionsblock in Ihrem Spiel verwendet werden, aber sie können nach ihrer Definition nicht mehr geändert werden. Beachten Sie, dass Sie zusätzliche Makros in derselben Aktion definieren können, indem Sie auf das Plus-Symbol neben der Aktion klicken. Weitere Informationen zu Makros finden Sie hier.
