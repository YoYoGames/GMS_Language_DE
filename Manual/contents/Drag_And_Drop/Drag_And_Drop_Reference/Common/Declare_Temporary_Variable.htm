Temporäre Variable deklarieren

Diese Aktion wird verwendet, um eine temporäre Variable (auch lokale Variable genannt) zu deklarieren und ihr einen Anfangswert zu geben. Dabei handelt es sich um eine Variable, die nur für die Dauer des Ereignisses oder Scripts existiert, in dem sie deklariert wurde. Temporäre Variablen sind sehr nützlich, um Werte zu speichern, die nur für einen einzigen Vorgang benötigt werden, und verhindern, dass Ihre Objekte mit Variablen "vollgestopft" werden, die vielleicht nur einmal verwendet werden. Sie sind auch sehr nützlich, um komplexe Operationen aufzulösen, oder um Werte in anderen Instanzen zu setzen, wenn Sie den Aktionsbereich ändern. Das liegt daran, dass als temporär deklarierte Variablennichtauf eine bestimmte Instanz, sondern auf das Ereignis oder Skript selbstskaliert sind. Wenn Sie also eine temporäre Variable erstellen und dann den Aktionsbereich mit der Aktion " Anwenden auf..." ändern, können Sie die temporäre Variable weiterhin problemlos überprüfen, ohne ihren Aktionsbereich zu ändern.

Es ist erwähnenswert, dass wir, wenn wir von "Werten" sprechen, nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder alles andere sein kann, das eine Funktion zurückgeben oder verwenden kann, sogar ein Ausdruck. Weitere Informationen finden Sie im Abschnitt über Datentypen.

Beachten Sie, dass viele Aktionen ein Kontrollkästchen "Temp" haben, mit dem Sie eine neue temporäre lokale Variable erstellen können, die den Rückgabewert der Aktion enthält. danach. Beachten Sie auch, dass temporäre lokale Variablen auf das Skript oder Ereignis skaliert werden, in dem sie erstellt werden, und daher von anderen Instanzen im selben Block verwendet werden können, wenn Sie " Anwenden auf" verwenden.

Sie können auch weitere Variablen in derselben Aktion hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere temporäre lokale Variable Namen und Wert geben. Weitere fortgeschrittene Informationen zu Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen Variable, die zugewiesen werden soll, oder der Name einer vorhandenen Variable, die geändert werden soll.
Wert Der Wert, den die Variable enthalten soll.
