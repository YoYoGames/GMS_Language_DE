Eine neue Funktion deklarieren

Diese Aktion kann verwendet werden, um eine neue benutzerdefinierte Funktion zu erstellen, die dann später in Ihrem Code jederzeit aufgerufen werden kann. Eine benutzerdefinierte Funktion ist eine Funktion, die Sie definieren, um eine Aufgabe auszuführen, die nicht von den verfügbaren DnD™-Aktionen oder GML-Runner-Funktionen abgedeckt wird, und kann global, instanziell oder lokal sein (weitere Informationen zum Anwendungsbereich finden Sie hier ) und sie kann auch als Konstruktor verwendet werden, um Strukturen zu erstellen. Weitere Informationen über die Verwendung dieser Aktion zum Erstellen globaler Funktionen finden Sie im Abschnitt über das Erstellen von Aktionsfunktionen. Sie können die Funktion auch als statisch kennzeichnen, was bedeutet, dass sie nur einmal definiert wird, wenn sie zum ersten Mal verwendet wird.

Um diese Aktion zu verwenden, können Sie sie zu einem Instanzereignis hinzufügen und dann der zu deklarierenden Funktion einen Namen geben. Anschließend geben Sie die erforderlichen Argumentnamen für jeden der Parameter an, die die Funktion annehmen soll. Wenn die Funktion als Konstruktor verwendet werden soll Wenn die Funktion als Konstruktor verwendet werden soll (d. h., sie soll eine Struktur erstellen), müssen Sie das Flag " Konstruktor" aktivieren und dann die Funktion mit der Aktion "Neu" aufrufen. Wenn die Funktion nur lokal in demselben Ereignis und nirgendwo anders verwendet werden soll, können Sie auch das Flag Temp auswählen. Nachdem Sie den Namen und die Parameter definiert haben, können Sie die Aktionen, die beim Aufruf der Funktion ausgeführt werden sollen, in Ihrem Code hinzufügen, indem Sie sie an der Seite von der Aktion hinzufügen, wie in der folgenden Abbildung gezeigt:

Beispiele für die Verwendung sowohl als reguläre Funktion als auch als Konstruktorfunktion sehen Sie unten.

Argumente:

Argument Beschreibung

Argument0 ... ArgumentX Die verschiedenen Argumente Namen für jeden der Parameter, die an die Funktion übergeben werden sollen.
