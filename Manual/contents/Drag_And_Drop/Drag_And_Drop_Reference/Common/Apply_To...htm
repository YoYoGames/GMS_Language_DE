Anwenden auf...

Diese Aktion kann verwendet werden, um den Aktionsumfang für alle weiteren Aktionen zu ändern, die mit dieser verkettet sind. Wie bei einer if-Aktion, wenn Sie weitere Aktionen rechts von der Aktion Anwenden auf... ablegen, werden sie von dem Objekt oder der Instanz mit dem Geltungsbereich ausgeführt. Der Gültigkeitsbereich wird aus dem Einblendmenü ausgewählt, das angezeigt wird, wenn Sie auf den kleinen Pfeil oben rechts neben der Aktion klicken, und auf diese Weise können Sie Auf diese Weise können Sie einem anderen Satz von Objekten - oder sogar einer einzelnen Instanz, wenn Sie die Instanz-ID haben - mitteilen, dass mehrere Aktionen ausgeführt werden sollen, bevor Sie mit dem Code des aktuellen Objekts fortfahren.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "Gilt für"-Block, diese an der Seite der Aktion abgelegt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden jetzt von dem Objekt oder der Instanz ausgeführt, auf die der "Gilt für" Block, während alle Aktionen, die an anderer Stelle abgelegt werden, nach dem "Gilt für" Block ausgeführt werden.
