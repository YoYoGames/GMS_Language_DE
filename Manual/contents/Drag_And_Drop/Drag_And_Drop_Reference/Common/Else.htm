Die "else"-Aktion wird immer nur in Verbindung mit der "if " -Variable (oder einer anderen "if"-Aktion) verwendet und ist eine Möglichkeit, eine Reihe von Aktionen zu definieren, die ausgeführt werden sollen, wenn die "if"-Auswertung falsch zurückgibt. Sie könnten z. B. eine "Wenn"-Aktion haben, die prüft, ob die "Health"-Variable des Spielers kleiner oder gleich 0 ist, und wenn diese Auswertung "true" ergibt, zerstören Sie die Instanz. Wenn Sie jedoch eine andere Aktion ausführen möchten Wenn Sie jedoch einen anderen Satz von Codeaktionen ausführen möchten, wenn die Gesundheit des Spielers nicht kleiner als 0 ist, können Sie eine "else"-Aktion einfügen und dann die Aktionen angeben, die ausgeführt werden sollen, während die "if"-Auswertung "false" ergibt.
