Skript ausführen

Diese Aktion kann verwendet werden, um eine Skriptfunktion oder eine benutzerdefinierte Funktion aufzurufen. Im Allgemeinen würde diese Aktion nur für Legacy-Support verwendet werden und sollte nicht wirklich in aktuellen oder neuen Projekten verwendet werden. Zur Verwendung der Aktion zu verwenden, geben Sie das aufzurufende Skript-Asset an - oder den Variablennamen der benutzerdefinierten Funktion, die aufgerufen werden soll - und liefern dann alle erforderlichen Argumente, d. h. alle Werte (Strings, Ganzzahlen, Ressourcenindizes usw.), die Sie in das Skript oder Funktion eingeben möchten, damit es sie verwenden kann. Sie können bei Bedarf weitere Argumente hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken.

Einige Skripte und benutzerdefinierte Funktionen können auch einen Wert zur späteren Verwendung in der Aktionsblockliste zurückgeben, und so können Sie die Zielvariable angeben, die diesen Rückgabewert speichern soll. Diese kann vorher erstellt worden sein oder eine neue temporäre lokale Variable sein (wenn Sie das Kontrollkästchen "Temp" aktivieren). Sowohl das Ziel- als auch das Argumentfeld können leer gelassen werden, wenn sie nicht benötigt werden.

Beachten Sie, dass es im Allgemeinen besser ist, die Aktion Funktionsaufruf zu verwenden, um benutzerdefinierte Funktionen aufzurufen, und dass Sie diese Aktion im Allgemeinen nicht verwenden oder ein Skript-Asset direkt im Code aufrufen sollten.

Aktion Syntax:

Argumente:

Argument Beschreibung
Skript Der Name des aufzurufenden Skripts oder der benutzerdefinierten Funktion.
Argument0 ... Argument3 Die verschiedenen Argumente (Werte), die an das Skript oder die Funktion übergeben werden sollen (nicht verwendete Argumente können leer gelassen werden)
Ziel Der Name der Variablen, auf die die zurückgegebenen Werte ausgerichtet werden sollen (kann leer bleiben)

Beispiel:

Der obige Aktionsblock-Code ruft eine benutzerdefinierte Funktion auf und übergibt zwei Argumente für eine zufällige Position innerhalb des Raums. Er gibt nichts zurück.
