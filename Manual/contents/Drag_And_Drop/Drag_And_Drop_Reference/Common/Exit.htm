Ausfahrt

Diese Aktion führt je nach dem Kontext, in dem sie aufgerufen wird, eines von zwei Dingen aus. Wenn sie innerhalb eines Ereignisses aufgerufen wird, verlässt sie das Ereignis und fährt mit dem nächsten fort, wobei sie jeglichen Aktionsblockcode überspringt, der danach platziert sein könnte. Jedoch wenn sie innerhalb einer Aktionsblockfunktion verwendet wird, dann wird nur die Funktion verlassen und zum Ereignis zurückkehren, das die Funktion aufgerufen hat, wobei alle nachfolgenden Aktionen ausgeführt werden. Beachten Sie, dass der Aufruf in der Mitte einer Funktion keinen Wert zurückgibt (verwenden Sie dafür die Return-Aktion ).
