Wenn Variable

Diese Aktion wird verwendet, um den Zustand einer gegebenen Variablen gegen einen gegebenen Wert anhand eines bestimmten Ausdrucks zu prüfen. Eine "if"-Anweisung gibt immer entweder "true" oder "false" zurück, und die Ausdrücke für die Überprüfung können einer der folgenden sein folgenden sein:

  • Ist gleich - Die Variable und der Wert sind beide genau gleich
  • Kleiner als - Die Variable ist kleiner als der Wert
  • Größer als - Die Variable ist größer als der Wert
  • Kleiner als oder gleich - Die Variable ist kleiner als oder gleich dem Wert
  • Größer als oder gleich - Die Variable ist größer als oder gleich dem Wert

Wenn Sie das "Nicht"-Argument markieren, dann werden die obigen Ausdrücke negiert, z. B. wird "gleich" zu "nicht gleich". Beachten Sie auch, dass weder die Variablen- noch die Wertargumente auf numerische Werte beschränkt sind sondern auch eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein können, das eine Funktion zurückgeben oder verwenden kann (weitere Informationen finden Sie im Abschnitt über Datentypen ), aber Sie können nicht zwei Sie können jedoch nicht zwei verschiedene Datentypen vergleichen, ohne einen Fehler zu erhalten - wenn die Variable z. B. eine Zeichenkette enthält und Sie versuchen, zu prüfen, ob sie gleich einer Zahl ist, erhalten Sie einen Syntaxfehler.

WICHTIGER HINWEIS: Aufgrund von Problemen mit der Fließkommagenauigkeit kann die Prüfung, ob zwei Werte genau gleich sind, falsch sein, da einer genau 1 sein kann, während der andere der andere 1,00000000000001 sein kann. Dies kann vermieden werden, indem Sie vor der Prüfung die Aktion " Decimal to Integer " verwenden oder die Ausdrücke "größer als" oder "kleiner als" verwenden.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, werden nach dem "if"-Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variabel Der Name der zu prüfenden Variablen.
Nicht Wird gesetzt, um zu prüfen, ob der Ausdruck nicht zu true ausgewertet werden kann.
Ausdruck Der Typ des Ausdrucks, der für die Prüfung verwendet werden soll.
Wert Der Wert, gegen den die Variable geprüft werden soll.

Beispiel:

Der obige Aktionsblockcode prüft eine Variable "dir", um zu sehen, ob sie gleich 10 ist, und wenn Wenn ja, wird eine Instanz von "obj_Squirrel" erstellt, wenn nicht, wird die Instanz zerstört.
