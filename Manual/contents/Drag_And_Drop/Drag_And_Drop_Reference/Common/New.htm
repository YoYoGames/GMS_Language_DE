Neu

Diese Aktion wird nur verwendet, wenn Strukturen mit einer Funktion erstellt werden, die als Konstruktorfunktion gekennzeichnet wurde (siehe Deklarieren einer neuen Funktion für weitere Informationen). Diese Aktion nimmt die Funktionsmethode für eine zuvor definierte Funktion sowie eine Reihe von Argumenten entgegen, die Sie bei Bedarf mit dem Symbol auf der linken Seite erweitern können, um weitere hinzuzufügen. Diese Argumente sollten den Eingaben entsprechen, die von der Funktion benötigt werden, und werden verwendet, um die zu erstellende Struktur zu füllen. Die Struktur wird an die Zielvariable zurückgegeben, die als temporäre lokale Variable gekennzeichnet werden kann. Variable gekennzeichnet sein kann oder nicht.

Aktion Syntax:

Argumente:

Argument Beschreibung
Skript Der Name des aufzurufenden Skripts oder der benutzerdefinierten Funktion.
Argument0 ... Argument3 Die verschiedenen Argumente (Werte), die an das Skript oder die Funktion übergeben werden sollen (nicht verwendete Argumente können leer gelassen werden)
Ziel Der Name der Variablen, auf die die zurückgegebenen Werte ausgerichtet werden sollen (kann leer bleiben)
