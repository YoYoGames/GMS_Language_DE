Variable zuweisen

Diese Aktion wird verwendet, um entweder einer Variablen einen Wert zuzuweisen (und sie dabei zu erstellen) oder um den Wert einer vorhandenen Variablen zu ändern. Sie geben den Namen der Variablen an und dann den Wert, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. erforderlich. Wenn Sie einen neuen Variablennamen angeben (d. h. eine Variable, die noch nie zuvor in der Instanz zugewiesen wurde), wird die neue Variable erstellt und ihr der von Ihnen angegebene Wert zugewiesen, während, wenn Sie den Namen einer vorhandenen Variable angeben einfach den Wert in den neuen Wert ändern. Beachten Sie, dass die Überprüfung von Is Relative den Wert zum bestehenden Variablenwert addiert (verwenden Sie eine negative Zahl, um zu subtrahieren) und Sie können auch Expression s und zuvor erstellte Variablen als Wert verwenden.

Es ist erwähnenswert, dass wir, wenn wir von "Werten" sprechen, nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann. Im Fall einer Zeichenkette wird die Überprüfung von Relative Wenn Ihre Variable also "Hallo" war und Sie ihr mit der Relativprüfung "Welt" zuweisen, wird der endgültige Variablenwert "Hallo Welt" sein. Es muss darauf geachtet werden, Es muss jedoch darauf geachtet werden, dass die Variablenwerte beide vom gleichen Datentyp sind, da der Versuch, z. B. eine Ganzzahl und eine Zeichenkette zu addieren, zu einem Compilerfehler führt.

Beachten Sie, dass Sie weitere Variablen in derselben Aktion hinzufügen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und einen weiteren Variablennamen und Wert angeben. Weitere fortgeschrittene Informationen zu Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen Variable, die zugewiesen werden soll, oder der Name einer vorhandenen Variable, die geändert werden soll.
Wert Der Wert, den die Variable enthalten soll.
Relativ Prüfen Sie, ob der Wert relativ zum vorherigen Wert eingestellt werden soll.
