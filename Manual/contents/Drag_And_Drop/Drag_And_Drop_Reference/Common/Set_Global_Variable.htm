Globale Variable setzen

Diese Aktion wird verwendet, um entweder einer globalen Variable einen Wert zuzuweisen (und sie dabei zu erzeugen) oder um den Wert einer bestehenden globalen Variable zu ändern. Globale Variablen sind Variablen, die zu keiner bestimmten Instanz "gehören", sondern zum gesamten Projekt "gehören" und als solche von jeder Instanz, jedem Skript oder Code aufgerufen und geändert werden können. Weitergehende Informationen zu globalen Variablen finden Sie hier.

Sie geben den Namen der globalen Variablen an und dann den Wert, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. Wenn Sie einen neuen Variablennamen angeben (d. h. eine Variable, die im gesamten Spiel noch nie zuvor zugewiesen wurde), dann wird die die neue globale Variable und weist ihr den von Ihnen angegebenen Wert zu, während es bei Angabe des Namens einer vorhandenen globalen Variable einfach den Wert in den neuen Wert ändert. Beachten Sie, dass das Aktivieren von " Relativ" den Wert zur bestehenden globalen Variablenwert addiert wird (verwenden Sie eine negative Zahl, um zu subtrahieren) und Sie auch Ausdrückeund Variablen, die zuvor erstellt wurden, als Wert verwenden können.

Es ist erwähnenswert, dass wir, wenn wir über "Werte" sprechen, nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann. Im Falle einer Zeichenkette verkettet die Relativprüfung den Wert mit dem, was die Variable zuvor enthielt. Wenn Ihre Variable also "Hallo" war und Sie ihr mit der Relativprüfung " Welt" zuweisen, wird der endgültige Variablenwert "Hallo Welt" sein. Es muss Es muss jedoch darauf geachtet werden, dass die Variablenwerte beide vom gleichen Datentyp sind, da der Versuch, z. B. eine Ganzzahl und eine Zeichenkette zu addieren, zu einem Compilerfehler führt.

Sie können auch weitere Variablen in derselben Aktion hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere globale Variable Namen und Wert geben. Weitergehende Informationen zu Variablen finden Sie im Abschnitt über Variablen und Variablenumfang.

Aktion Syntax:

Argumente:

Argument Beschreibung
Name Der Name der neuen globalen Variable, die erstellt werden soll, oder der Name einer vorhandenen globalen Variable, die geändert werden soll.
Wert Der Wert, den die globale Variable enthalten soll.
Relativ Aktivieren Sie diese Option, um den Wert relativ zum vorherigen Wert zu setzen.
