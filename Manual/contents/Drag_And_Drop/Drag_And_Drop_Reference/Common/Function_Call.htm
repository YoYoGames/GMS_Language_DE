Funktionsaufruf

Mit dieser Aktion können Sie eine beliebige GML-Funktion oder eine benutzerdefinierte Funktion aufrufen. Sie geben die aufzurufende Funktion an und liefern dann alle zusätzlichen Argumente, die sie benötigt, d. h. alle Werte (Strings, Ganzzahlen, Ressourcenindizes usw.), die Sie in die Funktion eingeben möchten, damit sie diese verwenden kann. Sie können zusätzliche Argumenteingaben hinzufügen, indem Sie auf das Plus-Symbol neben der Aktion klicken, und Sie sollten sicherstellen, dass die Anzahl der Argumente mit der der verwendeten Funktion übereinstimmt (d. h.: wenn die Funktion 3 Argumente benötigt, sollten Sie 3 Argumente angeben). Einige Funktionen können auch einen Wert zur späteren Verwendung in der Aktionsblock Liste zurückgeben, und so können Sie auch die Zielvariable angeben, die diesen Rückgabewert speichern soll, die zuvor erstellt worden sein kann oder eine neue temporäre lokale Variable sein kann (wenn Sie das Kontrollkästchen " Temp" aktivieren). Sowohl das Ziel und das Argumentfeld können leer gelassen werden, wenn sie nicht benötigt werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Funktion Die aufzurufende GML-Funktion.
Argument0 ... ArgumentX Die verschiedenen Argumente (Werte), die an die Funktion übergeben werden sollen.
Ziel Der Name der Variablen, auf die die zurückgegebenen Werte abzielen sollen (kann leer gelassen werden).
