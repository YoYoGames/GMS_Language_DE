If-Ausdruck

Diese Aktion wird verwendet, um zu prüfen, ob ein Ausdruck entweder wahr oder falsch ist, und Sie können hier jeden logischen Ausdruck prüfen, solange er zu dem einen oder anderen Wert ausgewertet wird. Zum Beispiel könnte jeder der folgenden Ausdrücke verwendet werden:

x < (Raum_Breite / 2)
a == b
++val >= 100

Wenn Sie das Argument "Nicht" markieren, werden die obigen Ausdrücke negiert, z. B. wird "==" zu "not=". Beachten Sie auch, dass der Ausdruck nicht auf numerische Werte beschränkt ist und Sie Strings, Zeiger, Ressource-IDs oder alles andere prüfen können, was eine Funktion zurückgeben oder verwenden kann (weitere Informationen finden Sie im Abschnitt über Datentypen ), aber Sie können nicht zwei verschiedene Datentypen vergleichen, ohne einen Fehler zu erhalten - wenn Sie z. B. versuchen, einen String und eine ganze Zahl im Ausdruck zu verwenden, erhalten Sie einen Fehler. Weitere Informationen zu Ausdrücken finden Sie hier.

WICHTIGER HINWEIS: Aufgrund von Problemen mit der Fließkommagenauigkeit kann die Prüfung, ob zwei Werte genau gleich sind, falsch sein, da einer genau 1 sein kann, während der andere der andere 1,00000000000001 sein kann. Dies kann vermieden werden, indem Sie vor der Prüfung die Aktion Dezimal zu Ganzzahl verwenden oder die Ausdrücke "größer als (>)" oder "kleiner als (<)" verwenden.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if" Block, diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle abgelegt werden, nach dem "if"-Block ausgeführt werden. Beachten Sie, dass Sie Ausdrücke in derselben Aktion "stapeln" können, indem Sie auf das Plus-Symbol neben neben der Aktion klicken und einen weiteren zu prüfenden Ausdruck angeben. Dadurch wird geprüft, ob alle angegebenen Ausdrücke in einer Prüfung den Wert "wahr" ergeben, und der hinzugefügte Aktionscode wird nur dann ausgeführt, wenn dies der Fall ist (siehe das unten stehende Beispiel).
