Wenn Datei vorhanden ist

Mit dieser Aktion können Sie prüfen, ob eine Datei mit dem angegebenen Namen existiert. Sie wählen die zu prüfende Datei aus (als Sting und einschließlich der Erweiterung) und die Funktion gibt true zurück, wenn sie existiert, und false, wenn sie nicht existiert. Beachten Sie, dass die Funktion " true" zurückgibt, wenn die Datei im Spiel-Bundle als "Included File" vorhanden ist oder wenn die Datei bei laufendem Spiel herausgespeichert wurde (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Der obige Code des Aktionsblocks erstellt eine Variable zur Aufnahme von Pufferdaten und prüft dann, ob eine Pufferspeicherdatei zuvor erstellt wurde. Wenn die Datei vorhanden ist, wird sie in die erstellte Puffervariable geladen. Wenn sie jedoch nicht vorhanden ist, wird sie erstellt, beschrieben und dann gespeichert, damit sie beim nächsten Mal geladen werden kann.
