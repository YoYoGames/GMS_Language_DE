Ini-Datei öffnen

Mit dieser Aktion können Sie eine Ini-Datei zum Lesen und/oder Schreiben öffnen. Wenn die Ini-Datei an dem Ort, den Sie prüfen, nicht existiert, kann GameMaker Studio 2 eine erstellen, aber nur, wenn Sie Daten in sie schreiben. Wenn Sie nur Informationen aus der der Ini-Datei gelesen haben, werden die Standardwerte für die Lesefunktion zurückgegeben, aber die Ini-Datei wird nicht tatsächlich erstellt.

Bitte beachten Sie, dass immer nur eine Ini-Datei geöffnet sein kann und dass Sie die Ini-Datei schließen müssen, wenn Sie mit dem Lesen/Schreiben der Ini-Datei fertig sind, da die Informationen erst dann auf der Festplatte gespeichert werden. gespeichert werden (sie werden im Speicher gehalten, bis die Datei geschlossen wird).
