Ini-Datei lesen

Mit dieser Aktion können Sie ein Stück Daten aus einer Ini-Datei lesen. Sie müssen die Ini-Datei zuvor mit der Aktion Open Ini File zum Lesen/Schreiben geöffnet haben, und dann geben Sie mit dieser Aktion den Typ der zu lesenden Daten (entweder eine reelle Zahl oder eine in "" eingeschlossene Zeichenkette), sowie die Abschnittsüberschrift und den Schlüssel innerhalb dieses Abschnitts, von dem die Daten abgerufen werden sollen. Sie können auch einen Standardwert angeben, der entweder eine reelle Zahl oder eine Zeichenkette sein kann, und der Typ sollte mit dem in der Aktion eingestellten "Typ" übereinstimmen. Diese Standardaktion wird von der Aktion zurückgegeben, wenn einer der folgenden Fälle eintritt:

  • Die Ini-Datei, die Sie geöffnet haben, existiert nicht
  • Die Abschnittsüberschrift, auf die Sie verweisen, existiert nicht
  • Der Schlüssel, von dem Sie versuchen, Daten abzurufen, existiert nicht

Unter jedem der oben genannten Umstände erhalten Sie den Standardwert zurück und nicht einen Wert aus der Ini-Datei. Welcher Wert auch immer zurückgegeben wird, wird in der von Ihnen angegebenen Zielvariablen gespeichert, die auch in diesem Moment erstellt werden kann, wenn sie als als temporäre lokale Variable.

WICHTIG! Der Versuch, Daten aus einer Ini-Datei zu lesen, die zuvor nicht mit der Aktion " Ini-Datei öffnen " geöffnet wurde, führt zu einem schwerwiegenden Fehler und zum Absturz Ihres Spiels.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der Typ der zu lesenden Daten, entweder eine Zeichenkette (in Anführungszeichen ""), oder eine reelle Zahl
Abschnitt Der Abschnitt der Ini-Datei, aus dem gelesen werden soll (als String)
Taste Der Schlüssel innerhalb des Abschnitts, aus dem gelesen werden soll (als String)
Standard Der Standardwert, der zurückgegeben wird, wenn entweder die Datei, der Abschnitt oder der Schlüssel nicht existiert
Ziel Die Zielvariable zur Aufnahme des Rückgabewerts der Aktion

Beispiel:

Der obige Aktionsblockcode öffnet eine Ini-Datei zum Lesen und verwendet dann die daraus gelesenen Daten zum Setzen von 3 verschiedene globale Variablen, bevor er die Datei wieder schließt.
