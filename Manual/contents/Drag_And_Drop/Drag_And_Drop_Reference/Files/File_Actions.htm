Dateien Aktionsbibliothek

Die Datei-Aktionen befassen sich mit zwei verschiedenen Dateitypen - Pufferdateien und Ini-Dateien - und verfügen außerdem über bestimmte generische Datei-Aktionen zum Umbenennen oder Kopieren vorhandener Dateien.

Pufferdateien werden erstellt, indem Sie die Daten aus einem zuvor erstellten Puffer auslagern und diese Daten zu einem beliebigen Zeitpunkt in der Zukunft wieder in einen Puffer laden können. Weitere Informationen zu Puffern finden Sie hier: Puffer-Aktionen.

Ini-Dateien sind kleine, leichtgewichtige Dateien, die mit den meisten Plattformen kompatibel sind. Sie sind ideal zum Speichern kleiner Informationen, wie z. B. Oberflächeneinstellungen, lokale Highscores, Leveldaten usw... und sind sehr einfach zu verwenden. Ini-Dateien müssen nicht müssen nicht vorher erstellt worden sein, um diese Aktionen zu verwenden, und Sie können aus einer nicht existierenden Ini-Datei lesen und erhalten einfach einen Standard-Rückgabewert (den Sie angeben), allerdings empfehlen wir, dass Sie zumindest eine "Basis"-Ini-Datei ini-Datei zum Öffnen und Ändern zu erstellen, bevor Sie die Aktionen verwenden. Diese Basis-Ini-Datei kann durch den einfachen Aufruf der Aktion "Open Ini File" gefolgt von der Aktion " Close Ini File" erstellt werden, da das Schließen der Datei auf die Festplatte geschrieben wird, oder Sie können eine solche Datei in die eingeschlossenen Dateien des Asset-Browsers aufnehmen. Wenn Sie eine im Asset-Browser enthaltene Datei als Basis-Ini verwenden, sollten Sie auch den Abschnitt den Abschnitt des Handbuchs über die Funktionsweise des Dateisystems lesen.

HINWEIS: Für Spiele mit Lokalisierungen ist es sehr wichtig, dass akzentuierte Buchstaben aus externen Dateien gelesen werden können. Das bedeutet, dass Sie die ini-Datei zunächst im UTF8-Format erstellen und sie dann in GameMaker Studio 2 als eingebundene Datei hinzufügen sollten einfügen, damit sie beim Ausführen des Spiels exportiert und anstelle der standardmäßigen Ini-Datei im ANSI-Format verwendet wird, die von den Ini-Funktionen erzeugt wird, wenn zuvor keine mitgeliefert wurde. Auf diese Weise können Sie sie korrekt lesen und schreiben, wobei alle Akzente und nicht-römischen Buchstaben beibehalten werden.

Puffer speichern
