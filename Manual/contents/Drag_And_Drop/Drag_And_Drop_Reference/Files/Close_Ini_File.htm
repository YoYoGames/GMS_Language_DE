Ini-Datei schließen

Mit dieser Aktion wird die aktuell geöffnete Ini-Datei geschlossen. Diese Aktion sollte in dem Moment aufgerufen werden, in dem Sie mit dem Lesen oder Schreiben in eine geöffnete Ini-Datei fertig sind (Ini-Dateien werden mit der Aktion Open Ini File geöffnet). Wenn Sie diese Aktion nicht verwenden, nachdem Sie die Aktion Write To Ini File verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher gehalten werden, bis diese Aktion aufgerufen wird, woraufhin sie aus dem Speicher und in die Datei auf der Festplatte geschrieben. Beachten Sie auch, dass Sie einen Fehler erhalten, wenn Sie versuchen, eine Ini-Datei zu öffnen, ohne zuvor eine andere (oder dieselbe) geschlossen zu haben.
