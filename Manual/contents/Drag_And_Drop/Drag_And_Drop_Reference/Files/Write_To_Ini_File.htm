Schreiben in die Ini-Datei

Mit dieser Aktion können Sie einen numerischen Wert oder einen String in eine Ini-Datei schreiben. Sie müssen die Ini-Datei zuvor mit der Aktion Ini-Datei öffnen zum Lesen/Schreiben geöffnet haben, und dann geben Sie mit dieser Aktion den Typ der zu schreibenden Daten an (entweder eine reelle Zahl oder eine Zeichenkette, die in "" eingeschlossen ist), sowie die Abschnittsüberschrift, den Schlüssel innerhalb dieses Abschnitts und den Schlüssel des Abschnitts. Daten (entweder eine reelle Zahl oder eine in "" eingeschlossene Zeichenkette), sowie die Abschnittsüberschrift, den Schlüssel innerhalb dieses Abschnitts und den Wert für den Schlüssel. Zur Veranschaulichung der Struktur einer Ini-Datei sehen Sie dieses Bild:

Wie Sie sehen können, werden Ini-Dateien in Abschnitte aufgeteilt - wobei jeder Abschnitt einen eigenen Kopfnamen hat - und dann wird jeder Abschnitt anschließend in verschiedene Schlüssel/Wertpaare aufgeteilt (ähnlich wie eine Map-Datenstruktur). Beachten Sie, dass Sie Ini-Dateien selbst mit einem beliebigen Textverarbeitungsprogramm (wie Notepad unter Windows) selbst erstellen und sie dann als eingebundene Datei in Ihr Spiel einbinden, um daraus zu lesen und zu schreiben.

WICHTIG! Der Versuch, Daten aus einer Ini-Datei zu lesen, die zuvor nicht mit der Aktion " Open Ini File " geöffnet wurde, führt zu einem schwerwiegenden Fehler und zum Absturz Ihres Spiels.

WICHTIG! Dadurch werden die Daten nur in den Speicher geschrieben. Um die Daten endgültig in die Datei auf der Festplatte zu schreiben, müssen Sie Ini-Datei schließen aufrufen, sonst werden die Daten nicht gespeichert.

Aktion Syntax:

Argumente:

Argument Beschreibung
Typ Der Typ der zu schreibenden Daten, entweder eine Zeichenkette (eingeschlossen in Anführungszeichen "") oder eine reelle Zahl
Abschnitt Der Abschnitt der Ini-Datei, in den geschrieben werden soll (als String)
Taste Der Schlüssel innerhalb des Abschnitts, in den geschrieben werden soll (als String)
Wert Der zu schreibende Wert, entweder eine Zeichenkette oder eine reelle Zahl, basierend auf dem Argument "Typ" (oben)
