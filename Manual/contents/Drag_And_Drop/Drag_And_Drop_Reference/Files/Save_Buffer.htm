Puffer speichern

Mit dieser Aktion können Sie eine Datei speichern, die die Daten aus einem Puffer enthält. Sie geben die eindeutige Puffer-ID an (wie sie beim Anlegen des Puffers mit Create Buffer zurückgegeben wurde) und geben einen Namen für die Datei an (als String, mit der Erweiterung Ihrer Wahl) und der Puffer wird in diese Datei geschrieben, die dann mit Puffer laden wieder geladen werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der eindeutige Puffer-ID-Wert des zu speichernden Puffers
Dateiname Der Name (als String) der zu speichernden Datei

Beispiel:

Der obige Code des Aktionsblocks erstellt eine Variable zur Aufnahme von Pufferdaten und prüft dann, ob eine Pufferspeicherdatei zuvor erstellt wurde. Wenn die Datei vorhanden ist, wird sie in die erstellte Puffervariable geladen. Wenn sie jedoch nicht vorhanden ist, wird sie erstellt, beschrieben und dann gespeichert, damit sie beim nächsten Mal geladen werden kann.
