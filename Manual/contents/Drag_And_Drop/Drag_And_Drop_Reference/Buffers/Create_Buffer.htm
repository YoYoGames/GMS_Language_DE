Puffer erstellen

Diese Aktion kann verwendet werden, um einen Teil des Speichers als Puffer in Ihrem Spiel zuzuweisen, wobei die Funktion den eindeutigen Puffer-ID-Wert zurückgibt, der in einer Variablen gespeichert und für alle weiteren Funktionsaufrufe auf den Puffer verwendet werden sollte. Sie können eine globale Variable, eine Instanzvariable angeben oder die Aktion eine temporäre lokale Variable zum Speichern der Puffer-ID erstellen lassen. Beachten Sie, dass Sie zusätzliche Puffer erstellen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere Variable auswählen, die die Puffer-ID enthält.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erstellten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden; wenn Sie also mit dem erstellten Puffer fertig sind, sollten Sie ihn wieder freigeben mit der Aktion " Puffer löschen" (z. B. im Ereignis " Sonstige - Auf räumen"). Dies ist besonders wichtig zu beachten, wenn Sie eine temporäre lokale Variable, denn wenn Sie den Puffer nicht vor dem Ende des Ereignis- oder Aktionsskripts wieder entfernen, können Sie nicht mehr darauf zugreifen und später Probleme bekommen.
