Suchlaufpuffer

Diese Aktion kann verwendet werden, um eine Position innerhalb eines Puffers anzufahren (zu suchen). Sie geben den eindeutigen Puffer-ID-Wert an, der von der Aktion " Puffer erstellen " zurückgegeben wird, und dann den Offset-Wert in Bytes, zu dem verschoben werden soll. Der Offset-Wert wird Der Offset-Wert wird vom Anfang des Puffers aus angewendet, d. h. der Aufruf dieser Funktion verschiebt die Lese-/Schreibposition an den Anfang des Puffers, bewegt sich dann um die angegebene Anzahl von Bytes weiter und positioniert die neue Lese-/Schreibposition dort. Wenn Sie jedoch den Offset als relativ kennzeichnen, wird die Lese-/Schreibposition von der aktuellen Position aus vorwärts (oder rückwärts, wenn der Offset ein negativer Wert ist) verschoben.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der Pufferindex (gespeichert in einer Variablen)
Offset Der Offset (in Bytes) relativ zur aktuellen Suchposition
