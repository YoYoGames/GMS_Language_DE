Puffer kopieren

Mit dieser Aktion können alle Daten, die in einem Puffer gespeichert sind, in einen anderen kopiert werden. Wenn Sie zwei Puffer verwenden und von einem in den anderen kopieren, müssen beide Puffer zuvor mit der Aktion Puffer erstellen erstellt worden sein, erstellt worden sein und Sie geben die eindeutige Puffer-ID als Von/Bis-Argumente an.

HINWEIS: Sie können nicht in denselben Puffer kopieren.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der Pufferindex (gespeichert in einer Variablen), der von
Kopieren nach Der Pufferindex (gespeichert in einer Variablen), der in

Beispiel:

Der obige Aktionsblockcode erstellt einen neuen Puffer und speichert dessen ID in einer temporären Variablen. Anschließend wird geprüft, ob der in der angegebenen globalen Variablen indizierte Puffer existiert. Wenn dies der Fall ist, wird er in den temporären Puffer kopiert und die Lese-/Schreibposition auf den Anfang zurückgesetzt. Wenn er nicht existiert, wird er erstellt.
