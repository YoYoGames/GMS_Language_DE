Puffergröße holen

Diese Aktion kann verwendet werden, um die Größe eines Puffers in Bytes abzurufen. Sie geben den eindeutigen Puffer-ID-Wert an, der von der Aktion " Puffer erstellen " zurückgegeben wird, und geben dann eine Zielvariable an, an die der Größenwert zurückgegeben werden soll. Die Zielvariable Variable kann als temporäre lokale Variable gekennzeichnet werden. In diesem Fall wird sie für Sie erstellt und am Ende des aktuellen Aktionsskripts oder Ereignisses wieder aus dem Speicher entfernt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der Pufferindex (gespeichert in einer Variablen)
Ziel Die Zielvariable, die den zurückgegebenen Wert aufnehmen soll

Beispiel:

Der obige Code des Aktionsblocks ermittelt die Größe des Puffers und verwendet dann diesen Wert, um die Lese-/Schreibposition auf das Ende der aktuell gespeicherten Daten zu setzen.
