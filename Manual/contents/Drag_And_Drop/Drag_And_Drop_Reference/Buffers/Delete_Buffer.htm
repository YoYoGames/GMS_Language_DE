Puffer löschen

Diese Aktion kann verwendet werden, um einen Puffer aus dem Speicher zu löschen, wodurch die zu seiner Erstellung verwendeten Ressourcen freigegeben und alle Daten entfernt werden, die er derzeit enthalten kann. Sie geben den eindeutigen Puffer-ID-Wert an, der von der Aktion " Puffer erstellen" zurückgegeben wird. Beachten Sie, dass Sie mehrere Puffer zum Entfernen auswählen können, indem Sie auf das Plus-Symbol neben der Aktion klicken, und eine weitere eine weitere Puffer-ID hinzufügen.

HINWEIS: Es ist wichtig, dass Sie dynamisch erstellte Ressourcen immer aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der Pufferindex (gespeichert in einer Variablen)
