Schreibpuffer

Diese Aktion kann verwendet werden, um Daten in einen Puffer zu schreiben. Sie geben die eindeutige Puffer-ID an, in die geschrieben werden soll (wie von der Aktion " Puffer erstellen" zurückgegeben) und dann den zu schreibenden Wert, der entweder ein Real oder ein String sein kann, aber beachten Sie, dass Beachten Sie jedoch, dass alle Werte als Zeichenketten gespeichert werden. Wenn Sie also aus dem Puffer lesen (mit Puffer lesen), müssen Sie eventuell eine der Datentyp-Aktionen verwenden, um den Wert von einer Zeichenkette zurück in den ursprünglichen Wert zu konvertieren. Denken Sie auch daran, dass Puffer sequentiell gelesen und beschrieben werden, so dass das Schreiben in einen Puffer die "Suchposition" um eins verschiebt, was bedeutet, dass Sie zum Schreiben verschiedener Daten einfach die Aktion so oft wie nötig aufrufen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Puffer Der Pufferindex (gespeichert in einer Variablen)
Wert Der Wert, der in den Puffer eingefügt werden soll
