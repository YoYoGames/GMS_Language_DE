Wenn die Instanz existiert

Diese Aktion kann verwendet werden, um zu prüfen, ob eine Instanz eines bestimmten Objekts im Spielraum vorhanden ist, oder ob eine bestimmte Instanz selbst vorhanden ist. Sie geben das Objekt aus dem Asset-Explorer an, und wenn eine Instanz im Raum vorhanden ist, gibt die Aktion true zurück, andernfalls wird sie false zurückgeben. Beachten Sie, dass Sie hier eine Variable mit der eindeutigen ID für eine bestimmte Instanz verwenden können (z. B. wie sie von der Aktion Objektinstanz erstellen zurückgegeben wird) oder sogar die Instanz-ID Wert verwenden können, der vom Raumeditor zurückgegeben wurde, als Sie die Instanz hinzugefügt haben.

Wenn Sie das "Not"-Argument markieren, dann prüft die Aktion, ob keine Instanz vorhanden ist, und wenn keine gefunden wird, wird true zurückgegeben, und wenn sogar eine vorhanden ist, wird false zurückgegeben.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle an anderer Stelle ausgelassenen Aktionen nach dem "if"-Block ausgeführt werden.

Dieser Wert wird in einer temporären Variablen gespeichert und dann geprüft, ob der Wert einer Instanz entspricht, die sich gerade im Raum befindet (wenn keine Kollision gefunden wird, enthält die Variable das Schlüsselwort " noone"), und wenn dies der Fall ist, wird eine Variable in der Instanz gesetzt, mit der kollidiert wird.
