Objektinstanz ändern

Mit dieser Aktion können Sie eine Instanz in eine andere Instanz ändern. Sie müssen das Objekt angeben, von dem die Instanz in eine andere Instanz geändert werden soll, und die Aktion ändert die Instanz, indem sie das Zerstörungsereignis und das Aufräumereignis der der ursprünglichen Instanz und das Ereignis "Erstellen" der neuen Instanz aus. Beachten Sie, dass, wenn Sie keine Richtung oder Geschwindigkeit usw. angeben, die neue Instanz diese Werte "erbt". Wenn die ursprüngliche Instanz zum Beispiel einen image_angle (Rotation) von 180, bewegt sich mit 2px pro Spieltick und hat eine image_speed (Animationsgeschwindigkeit) von 0,5, dann wird die Instanz, in die sie wechselt, diese Werte beibehalten, sofern sie nicht explizit im Ereignis "Create" der neuen Instanz festgelegt werden.

Es ist erwähnenswert, dass das Ändern der Instanz bedeutet, dass Sie bis zum nächsten Spieltick keine weiteren Aktionen mit dieser Instanz durchführen können, insbesondere der Versuch, auf Variablen usw. zuzugreifen, da dies einen Fehler verursacht. Grundsätzlich gilt, dass Sie Sie ändern die Instanz, aber sie ist bis zum Ende des aktuellen Schritts nicht wirklich verfügbar, so dass ein direkter Zugriff auf eine der darin enthaltenen Variablen (z. B. durch den Aufruf von inst_changed.x) nicht funktioniert.
