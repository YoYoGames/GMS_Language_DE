Instanz Skala einstellen

Dieser Aktionsblock setzt die Werte image_xscale und image_yscale für die Instanz. Die Eingabewerte hier sind Modifikatoren, die auf das der Instanz zugewiesene Sprite angewendet werden, wobei ein Maßstab von 1 (der Standardwert) keine Skalierung (1:1), kleinere Werte skalieren nach unten (z. B. 0,5 halbiert die Breite des Sprites), und größere Werte skalieren nach oben. Sie können negative Werte verwenden, um das Sprite zu spiegeln und zu skalieren, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das (in diesem Fall wird das Sprite nur gespiegelt oder um seinen Ursprung gedreht, ohne Skalierung). Sie können die horizontale oder vertikale Skalierung relativ zu den aktuellen Werten einstellen. In diesem Fall addieren oder subtrahieren Sie den neuen Wert von den aktuellen Skalierungswerten.

Beachten Sie, dass eine Änderung dieser Werte auch die Art und Weise ändert, wie die Instanz Kollisionen erkennt, es sei denn, Sie haben im Objekteditor eine separate mask_index (Kollisionsmaske) angegeben. Bitte beachten Sie, dass damit Änderungen in dieser Variablen sichtbar werden, sollte die Instanz entweder kein Draw-Ereignis haben (und somit wird das Sprite von GameMaker Studio 2 standardmäßig gezeichnet) oder mit der Aktion Draw Self gezeichnet werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Horizontal Der zu verwendende vertikale Skalierungsfaktor (Standard ist 1).
Vertikal Der zu verwendende horizontale Skalierungsfaktor (Standard ist 1).
