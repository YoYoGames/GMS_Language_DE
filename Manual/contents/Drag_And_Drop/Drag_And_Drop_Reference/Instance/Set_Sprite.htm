Satz Sprite

Dieser Aktionsblock setzt den sprite_index der Instanz auf ein anderes Sprite aus dem Asset-Browser. Sie wählen das Sprite aus, zu dem gewechselt werden soll, und geben dann das Bild (Animationsframe) an, das angezeigt werden soll. Beachten Sie, dass Sie, wenn das Sprite nicht animiert werden soll, nachdem dem Setzen des Rahmens nicht animiert werden soll, müssen Sie die Aktion "Setze Animationsgeschwindigkeit " verwenden und den Wert auf 0 setzen, oder wenn das Sprite weiterhin animiert werden soll, setzen Sie den Wert auf die eingebaute Variable image_index stattdessen.

Wenn Sie das Sprite ändern, wird auch die Kollisionsmaske der Instanz geändert, es sei denn, Sie haben im Objekteditor auch einen separaten mask_index angegeben. Bitte beachten Sie, dass Änderungen in dieser Aktion sichtbar sind, sollte die Instanz entweder kein Draw-Ereignis haben (und somit wird das Sprite von GameMaker Studio 2 standardmäßig gezeichnet) oder mit der Aktion Draw Self gezeichnet werden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Sprite Das neue zu verwendende Sprite (-1, wenn Sie das Sprite entfernen möchten).
Rahmen Der Animationsrahmen, der anfangs angezeigt werden soll (wenn keine zusätzlichen Rahmen verwendet werden, verwenden Sie 0).
