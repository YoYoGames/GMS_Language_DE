Elternveranstaltung aufrufen

Dieser Aktionsblock führt das Ereignis des übergeordneten Objekts aus, von dem die Instanz ein Kind ist, bevor er mit der Ausführung der Aktionen in der eigentlichen Instanz fortfährt. Wenn Sie einer Instanz ein übergeordnetes Objekt zuweisen, werden alle Ereignisse dieses Objekts an die untergeordnete Instanz "vererbt". von der untergeordneten Instanz "geerbt". Wenn Sie dann jedoch dem untergeordneten Objekt ein Ereignis hinzufügen, überschreibt es das übergeordnete Ereignis. Mit diesem Aktionsblock können Sie ein untergeordnetes Objekt dazu zwingen, sowohl das übergeordnete Ereignis als auch das Ereignis in der Instanz selbst auszuführen. Weitere Informationen zu übergeordneten Ereignissen finden Sie im Abschnitt Objekteditor.
