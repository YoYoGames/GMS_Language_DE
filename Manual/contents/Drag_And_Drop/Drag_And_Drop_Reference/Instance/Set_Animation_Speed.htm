Animationsgeschwindigkeit einstellen

Dieser Aktionsblock setzt die image_speed (Animationsgeschwindigkeit) für das der Instanz zugeordnete Sprite. Sie stellen die idealen Bilder pro Sekunde im Sprite-Editor ein und können dann mit diesem Wert um ihn zu ändern. Die Standard-Animationsgeschwindigkeit ist auf 1 eingestellt, d. h. das Spiel versucht, die Anzahl der Bilder pro Sekunde beizubehalten, die Sie für das Sprite definiert haben, aber Sie können Werte größer als 1 verwenden, um schneller zu animieren (es werden möglicherweise Bilder übersprungen), oder dezimale Werte, um langsamer zu animieren (Frames werden über mehrere Schritte angezeigt) oder sogar negative Werte, um die Geschwindigkeit der Animation umzukehren.
