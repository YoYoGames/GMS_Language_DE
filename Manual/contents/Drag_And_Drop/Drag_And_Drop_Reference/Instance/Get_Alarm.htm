Alarm erhalten

Dieser Aktionsblock ruft den Wert eines Alarm-Countdowns ab und speichert ihn in der angegebenen Zielvariablen. Wenn Sie die Zielvariable als "Temp" kennzeichnen, wird eine neue temporäre (lokale) Variable erstellt, um den Rückgabewert für Sie zu speichern, wie im folgenden Beispiel gezeigt im folgenden Beispiel.

Aktion Syntax:

Argumente:

Argument Beschreibung
Alarm Der Alarm zu bekommen.
Ziel Die Zielvariable, die zum Speichern des Alarmwerts verwendet werden soll.

Beispiel:

Der obige Aktionsblockcode erstellt eine lokale (temporäre) Variable und speichert den aktuellen Wert von des Alarms[0]. Dieser wird dann überprüft und wenn er kleiner als 0 ist, wird der Alarm auf 60 gesetzt.
