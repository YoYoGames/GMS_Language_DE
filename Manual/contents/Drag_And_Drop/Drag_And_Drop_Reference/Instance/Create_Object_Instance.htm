Objektinstanz erzeugen

Mit dieser Aktion können Sie eine Instanz eines beliebigen Objekts aus dem Asset-Browser erzeugen. Sie geben den Namen des Objekts und eine x/y-Position innerhalb des Raums an, in dem es erzeugt werden soll, sowie den Namen des Layers, auf dem die Instanz erzeugt werden soll (es muss sich um einen Instanz-Layer). Sie können die "relative"-Flags ankreuzen, um die Instanz relativ zu der Instanz zu positionieren, auf der die Aktionscodeblöcke ausgeführt werden. Immer wenn Sie eine Instanz im Spiel erstellen, erhält sie einen eindeutigen ID-Wert, mit dem Sie die Instanz dann Diese Aktion erlaubt es Ihnen daher, eine "Ziel"-Variable zu definieren, die diesen ID-Wert enthält (wie im folgenden Beispiel gezeigt). Beachten Sie, dass das Ziel als temporäre (lokale) Variable gekennzeichnet werden kann, die für die Aktion erstellt wird und in allen nachfolgenden Aktionen verwendet werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Objekt Der Name des Objekts, von dem eine Instanz erzeugt werden soll.
x Die x-Position im Raum, um die Instanz zu erzeugen (kann relativ zur aufrufenden Instanz sein).
y Die y-Position im Raum, um die Instanz zu erzeugen (kann relativ zur aufrufenden Instanz sein).
Ebene Der Name (eine Zeichenkette) der Instanzebene, auf der die Instanz erstellt werden soll.
Ziel Die Variable, die als Ziel für den Rückgabewert dienen soll.
