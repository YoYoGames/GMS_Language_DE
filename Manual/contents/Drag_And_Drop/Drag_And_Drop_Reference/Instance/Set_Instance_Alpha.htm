Instanz Alpha einstellen

Dieser Aktionsblock setzt den image_alpha-Wert für die Instanz. Der image_alpha-Wert ist der "Transparenz"-Wert für das Zeichnen jedes Sprites, das der Instanz zugewiesen ist, und hat einen Wert zwischen 0 und 1, wobei 0 vollständig transparent und 1 vollständig undurchsichtig wäre. Beachten Sie, dass, wenn das Sprite transparente oder halbtransparente Pixel hat, deren Transparenz um den Betrag skaliert wird. Wenn Sie also einige 50 % transparente Pixel im Sprite haben und das Instanz-Alpha auf 0,5 setzen, würde das endgültige Alpha für die transparenten Pixel 25 % sein. Sie können diesen Wert als relativen Betrag einstellen

Bitte beachten Sie, dass die Instanz entweder kein Draw-Ereignis haben sollte (so dass GameMaker Studio 2 das Sprite standardmäßig zeichnet) oder mit der Aktion Draw Self gezeichnet werden muss, damit Änderungen in dieser Variablen sichtbar werden.
