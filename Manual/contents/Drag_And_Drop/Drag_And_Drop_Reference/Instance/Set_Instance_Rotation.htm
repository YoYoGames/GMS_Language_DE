Drehung der Instanz einstellen

Dieser Aktionsblock setzt den image_angle (Drehung) der Instanz. Der Winkel wird in Grad gemessen, wobei rechts 0° ist, oben 90°, links 180° und unten 270°. Setzen Sie diese Variable auf 0, um die Instanz zurückzusetzen, was bedeutet, dass sie jedes zugewiesene Sprite so zeichnen, wie es im Sprite-Editor definiert wurde. Beachten Sie, dass die Instanz entweder kein Draw-Ereignis haben sollte (so dass GameMaker Studio 2 das Sprite standardmäßig zeichnet) oder mit der Aktion "Draw Self" gezeichnet werden sollte, damit Änderungen in dieser Aktion sichtbar werden. mit der Aktion " Selbst zeichnen ".

Aktion Syntax:

Argumente:

Argument Beschreibung
Winkel Der neue zu verwendende Winkel (von 0 bis 360).
