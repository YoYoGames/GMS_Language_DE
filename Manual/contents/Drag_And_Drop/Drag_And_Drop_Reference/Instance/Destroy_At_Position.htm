Zerstören an Position

Dieser Aktionsblock zerstört alle Instanzen, die an dem angegebenen Punkt gefunden werden, indem er das Destroy-Ereignis und dann das Clean Up-Ereignis für alle gefundenen Instanzen aufruft. Beachten Sie, dass diese Aktion auf der Kollisionsmaske der Instanzen basiert, und wenn ein Teil der Maske mit dem Zielpunkt überlappt, wird die Funktion diese Instanz zerstören (wenn die Instanzen kein Sprite oder keine Maske angewendet haben, dann werden sie nicht zerstört).

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die zu verwendende x-Position im Raum.
y Die zu verwendende y-Position im Raum.

Beispiel:

Der obige Code des Aktionsblocks prüft auf einen Tastendruck (die Leertaste) und wenn ein solcher erkannt wird wird er alle Instanzen zerstören, die an der relativen Position der Instanz gefunden wurden.
