Objektinstanz zerstören

Sie können diese Aktion immer dann aufrufen, wenn Sie eine Instanz "zerstören" möchten, wodurch sie aus dem Raum entfernt wird. Beachten Sie, dass, wenn Sie eine Instanz zerstören, ihr Zerstörungsereignis unmittelbar nach der Aktion, die die Zerstörung aufruft, aufgerufen wird. Beachten Sie auch, dass, obwohl das Destroy-Ereignis in dem Moment ausgeführt wird, in dem die Aktion aufgerufen wird, die Instanz nicht sofort aus dem Spiel entfernt wird, sondern weiterhin Sie führt den im aktuellen Ereignis enthaltenen Code bis zum Ende aus. Erst wenn das aktuelle Ereignis beendet ist, wird die Instanz aus dem Spiel entfernt, und zu diesem Zeitpunkt wird das Ereignis "Aufräumen" aufgerufen.
