Gesundheit erhalten

Mit dieser Aktion können Sie den Wert der Instanzvariable abrufen, die für die "Gesundheit" des Objekts angelegt wurde. Sie liefern die Zielvariable, um den zurückgegebenen "Health"-Wert zu speichern (die als temporäre lokale Variable gekennzeichnet werden kann), Beachten Sie aber, dass die Variable für Sie erstellt wird, wenn sie nicht vorher gesetzt wurde (mit Set Health), und die Aktion 0 zurückgibt (im Gegensatz zu regulären Instanzvariablen, die einen Fehler ausgeben, wenn Sie versuchen auf sie zuzugreifen, ohne sie vorher gesetzt zu haben).

Aktion Syntax:

Argumente:

Argument Beschreibung
Ziel Die Zielvariable zum Speichern des zurückgegebenen "Health"-Werts (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblockcode ruft den Wert für "health" ab und speichert ihn in einer temporären lokalen Variable. Diese Variable wird dann verwendet, um eine Gesundheitsleiste für die Instanz zu zeichnen.
