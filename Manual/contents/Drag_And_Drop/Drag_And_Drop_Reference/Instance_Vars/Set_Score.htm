Spielstand einstellen

Mit dieser Aktion können Sie eine Instanzvariable für "score" erstellen oder setzen. Sie geben den Wert an, auf den die "score"-Variable gesetzt werden soll oder der zur "score"-Variable hinzugefügt werden soll (wenn Sie das Kontrollkästchen " relativ" aktivieren, wird der Wert zum aktuellen "score"-Wert hinzugefügt Wert addiert). Beachten Sie, dass im Gegensatz zu regulären Instanzvariablen der Variablenname für "score" intern gespeichert wird, sodass Sie zum Abrufen des Score-Werts die Aktion "Get Score" verwenden müssen.

Argumente:

Argument Beschreibung
Ergebnis Der Betrag, auf den die Punktzahl gesetzt werden soll (oder auf den die Punktzahl addiert werden soll, wenn " relativ" aktiviert ist)
