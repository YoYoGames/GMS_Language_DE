Ergebnis abrufen

Mit dieser Aktion können Sie den Wert der Instanzvariable abrufen, die für den "Score" erstellt wurde. Sie geben die Zielvariable an, um den zurückgegebenen "Score"-Wert zu speichern (die als temporäre lokale Variable gekennzeichnet werden kann), aber beachten Sie, dass Beachten Sie jedoch, dass die Variable für Sie erstellt wird und die Aktion 0 zurückgibt, wenn sie vorher nicht gesetzt wurde (mit Set Score) (im Gegensatz zu regulären Instanzvariablen, die einen Fehler anzeigen, wenn Sie versuchen, auf sie zuzugreifen ohne sie vorher gesetzt zu haben).

Aktion Syntax:

Argumente:

Argument Beschreibung
Ziel Die Zielvariable zum Speichern des zurückgegebenen "Score"-Werts (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblockcode ruft den aktuellen Punktwert ab und speichert ihn in einer temporären lokalen Variable. Diese Variable wird dann in eine Zeichenkette umgewandelt und in einer anderen temporären lokalen Variablen gespeichert, bevor sie auf den Bildschirm gezeichnet wird.
