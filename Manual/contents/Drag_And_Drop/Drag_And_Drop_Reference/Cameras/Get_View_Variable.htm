Ansichtsvariable holen

Mit dieser Aktion können Sie den Wert bestimmter eingebauter Variablen abrufen, die sich auf einen bestimmten Ansichtsanschluss beziehen. Sie wählen die abzurufende View-Port-Variable, geben dann den View-Port als Ziel an (von 0 bis 7) und schließlich die Zielvariable, die den zurückgegebenen Wert enthalten soll (diese kann als temporäre lokale Variable gekennzeichnet werden). Der Ansichtsanschluss ist der Bereich des Bildschirms, der zum Zeichnen einer bestimmten Kameraansicht verwendet wird (wie im Raumeditor eingestellt), und die verschiedenen Variablen, die Sie auf diese Weise abrufen können sind in der folgenden Tabelle aufgeführt:

Variabel Beschreibung
Camera Der zurückgegebene Wert ist der eindeutige ID-Wert der Kamera, die dem Anschluss zugewiesen werden soll (weitere Informationen finden Sie hier ).
Visibility Dies gibt einen Wert von true oder false zurück, wobei true bedeutet, dass der Ansichtsanschluss sichtbar ist (d. h., dass er auf den Bildschirm gezeichnet wird), und false bedeutet, dass nichts gezeichnet wird.
View Port X Coordinate Dies ist die X-Position des View Ports innerhalb des Spielfensters.
View Port Y Coordinate Dies ist die Y-Position des View Ports innerhalb des Spielfensters.
View Port Width Dies ist die Breite (in Pixel) des Ansichtsports.
View Port Height Dies ist die Höhe (in Pixel) des Ansichtsports.
View Port Surface ID Dies gibt den eindeutigen ID-Wert für die Oberfläche zurück, die dem Ansichtsanschluss zugewiesen wurde, oder -1, wenn keine Oberfläche zugewiesen wurde.

Aktion Syntax:

Argumente:

Argument Beschreibung
Variabel Die eingebaute Ansichtsvariable zum Erhalten
Ansicht Der anzuvisierende Ansichtsanschluss (von 0 - 7)
Ziel Die Zielvariable, die den zurückgegebenen Wert aufnehmen soll (kann als temporäre lokale Variable gekennzeichnet werden)

Beispiel:

Der obige Aktionsblockcode ruft die aktuelle Kamera-ID ab, die dem View Port [0] zugewiesen ist, und prüft dann, ob sie mit der in einer globalen Variablen gespeicherten identisch ist. Wenn dies nicht der Fall ist, wird die Kamera auf die neue eingestellt.
