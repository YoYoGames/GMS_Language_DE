Ansicht einstellen Variable

Mit dieser Aktion können Sie bestimmte eingebaute Variablen einstellen, die sich darauf beziehen, wie der Ansichtsanschluss die Kameraansichten des Raums anzeigt. Sie wählen die zu bearbeitende Ansichtsfenster-Variable aus, geben dann das Ziel für das Ansichtsfenster an (von 0 bis 7) und schließlich den neuen Wert für die Variable. Der Ansichtsanschluss ist der Bereich des Bildschirms, der zum Zeichnen einer bestimmten Kameraansicht verwendet wird (wie im Raumeditor eingerichtet), und die verschiedenen Variablen, die Sie auf diese Weise einstellen können, sind in der folgenden Tabelle aufgeführt:

Variabel Beschreibung
Camera Hiermit können Sie das Ansichtsfenster so einstellen, dass eine andere Kameraansicht angezeigt wird. Sie müssen den eindeutigen ID-Wert der Kamera angeben, zu der Sie wechseln möchten (siehe hier für weitere Informationen).
Visibility Dies ist der Sichtbarkeitsschalter für den Ansichtsanschluss und erwartet einen Wert von true oder false, wobei ein Wert von true den Ansichtsanschluss sichtbar macht (d. h., er wird auf den Bildschirm gezeichnet) und ein Wert von false nichts auf den Bildschirm zeichnet nichts auf den Bildschirm.
View Port X Coordinate Dies ist die X-Position des Ansichtsports innerhalb des Spielfensters. Beachten Sie jedoch, dass dies nur dann erforderlich ist, wenn mehr als ein Ansichtsfenster aktiv ist - zum Beispiel, um einen Split-Screen-Effekt zu erzeugen - und wenn nur ein Fenster aktiv ist, erhalten Sie unerwartete Ergebnisse.
View Port Y Coordinate Dies ist die Y-Position des Ansichtsports innerhalb des Spielfensters. Beachten Sie jedoch, dass dies nur erforderlich ist, wenn mehr als ein Viewport aktiv ist - zum Beispiel, um einen Split-Screen-Effekt zu erzeugen - und wenn nur ein Port aktiv ist, erhalten Sie unerwartete Ergebnisse.
View Port Width Dies ist die Breite (in Pixel) des Ansichtsfensters. Die Breite des View-Ports (oder kombinierter View-Ports, wenn mehrere aktiv sind) definiert die Breite des Spielfensters oder der Hintergrundleinwand zu Beginn des Spiels, so dass eine Änderung dieses Wertes nach dem Start des Das Ändern dieses Wertes nach dem Spielstart hat keinen sichtbaren Effekt auf die Größe des Spielfensters, es sei denn, er wird zusammen mit der GML-Funktion window_set_size(). Beachten Sie auch, dass, wenn Sie eine größere oder kleinere Portgröße als die der Kamera zugewiesene haben, die Kameraansicht verkleinert - oder vergrößert - wird, damit sie passt.
View Port Height Dies ist die Höhe (in Pixel) des Ansichtsfensters. Die Höhe des View-Ports (oder kombinierter View-Ports, wenn mehr als einer aktiv ist) definiert die Höhe des Spielfensters oder der Hintergrundleinwand zu Beginn des Spiels, so dass eine Änderung dieses Wertes nach Das Ändern dieses Wertes nach dem Start des Spiels hat keinen sichtbaren Effekt auf die Größe des Spielfensters, es sei denn, er wird zusammen mit der GML-Funktion window_set_size(). Beachten Sie auch, dass, wenn Sie eine größere oder kleinere Portgröße als die der Kameraansicht zugewiesene haben, die Kameraansicht verkleinert - oder vergrößert - wird, damit sie passt.
View Port Surface ID Mit dieser Variablen können Sie festlegen, dass der Inhalt eines gegebenen Ansichtsports auf eine Oberfläche gezeichnet wird, wobei der von Ihnen angegebene Wert derjenige der Oberfläche ist, auf die der Ansichtsport gezeichnet werden soll. Für weitere Informationen siehe hier.
