Position entlang der Bahn holen

Mit dieser Aktion wird die Position einer Instanz entlang eines Pfades ermittelt. Der zurückgegebene Wert ist normalisiert von 0 - 1. Wenn Sie also z. B. ein Ergebnis von 0,5 erhalten, befindet sich die Instanz genau in der Mitte des Pfades. Beachten Sie, dass Sie die Aktion " Start Following Path " aufgerufen haben müssen, um der Instanz einen Pfad zuzuweisen, damit diese Aktion einen anderen Wert als 0 zurückgibt. Der zurückgegebene Wert wird in der Zielvariablen gespeichert, die Sie angeben, oder Sie können die Variable als temporäre lokale Variable kennzeichnen, um sie zu erstellen und den Wert zuzuweisen.
