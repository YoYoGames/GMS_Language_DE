Bahnfolgegeschwindigkeit abrufen

Diese Aktion ermittelt die Geschwindigkeit in Pixeln pro Spielframe, mit der eine Instanz einem Pfad folgt (beachten Sie, dass Sie die Aktion "Start Following Path" aufgerufen haben müssen, um der Instanz einen Pfad zuzuweisen, damit diese Aktion einen anderen Wert als 0 zurückgibt). Der zurückgegebene Wert wird in der Zielvariablen gespeichert, die Sie angeben, oder Sie können die Variable als temporäre lokale Variable kennzeichnen, um sie zu erstellen und den Wert zuzuweisen.
