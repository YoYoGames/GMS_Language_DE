Start Folgender Pfad

Mit dieser Aktion wird eine Instanz gestartet, die einem Pfad folgt, der im Asset-Browser erstellt wurde. Mit dieser Aktion können Sie Ihrer Instanz mitteilen, welchem Pfad sie folgen soll, mit welcher Geschwindigkeit sie dem Pfad folgen soll (gemessen in Pixeln pro Spielframe), wie sie sich verhalten soll, wenn sie das Ende des Pfades erreicht, und ob sie der absoluten oder relativen Pfadposition folgen soll. sich verhalten soll, wenn sie das Ende des Pfades erreicht, und ob sie der absoluten oder relativen Pfadposition folgen soll. Beim Einstellen der Geschwindigkeit können Sie negative Werte verwenden, um zu signalisieren, dass die Instanz dem Pfad in umgekehrter Richtung folgen soll, und es ist erwähnenswert, dass die Geschwindigkeit, die Sie hier einstellen, durch den Geschwindigkeitsmodifikator modifiziert werden kann, der im Pfad-Editor für jeden Pfadpunkt eingestellt ist. Wenn Sie also die Pfadgeschwindigkeit auf 2 einstellen und einer der Punkte im Pfad einen Geschwindigkeitsmodifikator von 200 % hat, dann beträgt die tatsächliche Geschwindigkeit an diesem Punkt 4 Pixel pro Spielframe.

Nachdem Sie die anfängliche Pfadgeschwindigkeit eingestellt haben, können Sie mit den folgenden Optionen festlegen, wie sich die Instanz verhalten soll, wenn sie das Ende des Pfads erreicht (beachten Sie, dass, wenn eine Instanz das Ende eines Pfads erreicht, dies auch ein Ereignis "Other - Path Ended" auslöst ):

  • Stopp: Beendet den Pfad und hält die Instanz am letzten Pfadpunkt an.
  • Neustart: Setzt den Pfad vom Start aus fort und springt wieder an die Startposition, wenn der Pfad im Pfad-Editor nicht als geschlossen markiert ist.
  • Schleife: Fahren Sie von der aktuellen Position aus fort, um den Pfad erneut abzufahren.
  • Rückwärts fahren: Den Pfad wieder rückwärts fahren (erreicht durch Umkehrung der Pfadverfolgungsgeschwindigkeit), und wenn er den Anfang des Pfades erreicht, wird er wieder rückwärts fahren, usw...

Schließlich können Sie den Pfad als absolut oder nicht absolut kennzeichnen. Standardmäßig beginnt die Instanz, dem Pfad von ihrer aktuellen Position aus zu folgen (die relative Position), aber wenn Sie das Kontrollkästchen " Absolut" aktivieren, beginnt sie, dem Pfad genau so zu folgen, wie Sie ihn im Pfad-Editor entworfen und platziert haben (die absolute Position). Wenn Sie jedoch das Kontrollkästchen "Absolut" aktivieren, folgt die Instanz dem Pfad genau so, wie Sie ihn entworfen und im Pfad-Editor platziert haben (die absolute Position). Aktion Syntax:

Argumente:

Argument Beschreibung
Pfad Das Pfad-Asset zum Starten nach
Geschwindigkeit Die Geschwindigkeit (in Pixeln pro Spielframe), um dem Pfad zu folgen
Ein Ende Die Reaktion auf das Erreichen des Endes des Pfades
Absolut Ob an absoluter oder relativer Position gestartet werden soll
