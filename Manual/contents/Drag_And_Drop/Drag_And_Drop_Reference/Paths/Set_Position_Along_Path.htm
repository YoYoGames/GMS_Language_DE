Position entlang der Bahn setzen

Mit dieser Aktion wird die Position einer Instanz entlang eines Pfades festgelegt. Der Wert ist normalisiert von 0 - 1. Wenn Sie ihn also z. B. auf 0,5 setzen, wird die Instanz genau in die Mitte des Pfades verschoben. Beachten Sie, dass Sie die Aktion " Start Following Path " aufgerufen haben müssen, um der Instanz einen Pfad zuzuweisen, damit dies einen Effekt hat.

Aktion Syntax:

Argumente:

Argument Beschreibung
Position Die Position (von 0 - 1) entlang des Pfades, um die Instanz zu platzieren

Beispiel:

Der obige Aktionsblockcode startet die Instanz entlang eines Pfades und erzeugt dann einen zufälligen Dezimalwert zwischen 0 und 1. Dieser Wert wird in einer temporären lokalen Variablen gespeichert und verwendet, um die Position der Instanz entlang des Pfades festzulegen.
