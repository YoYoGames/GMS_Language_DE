Bahnfolgegeschwindigkeit einstellen

Mit dieser Aktion wird die Geschwindigkeit in Pixeln pro Spielframe festgelegt, mit der eine Instanz einem Pfad folgen soll (beachten Sie, dass Sie die Aktion "Start Following Path " aufgerufen haben müssen, um der Instanz einen Pfad zuzuweisen, damit diese Aktion Wirkung hat). Sie können negative Werte verwenden, um zu signalisieren, dass die Instanz dem Pfad in umgekehrter Richtung folgen soll, und es ist erwähnenswert, dass die hier eingestellte Geschwindigkeit durch den im Pfad-Editor für jeden Pfadpunkt eingestellten Geschwindigkeitsmodifikator geändert werden kann, wenn Sie also Wenn Sie also eine Pfadgeschwindigkeit von 2 einstellen und einer der Punkte im Pfad einen Geschwindigkeitsmodifikator von 200 % hat, beträgt die tatsächliche Geschwindigkeit der Pixel pro Spielframe an diesem Punkt 4.

Aktion Syntax:

Argumente:

Argument Beschreibung
Geschwindigkeit Die Geschwindigkeit (in Pixel pro Spielframe), mit der die Instanz dem Pfad folgen soll

Beispiel:

Der obige Aktionsblockcode startet eine Instanz, die einem Pfad folgt, und generiert dann einen zufälligen Wert zwischen 2 und 4, bevor er diesen Wert verwendet, um die Geschwindigkeit festzulegen, mit der sich die Instanz entlang des Pfades bewegt.
