Aktuellen Raum holen

Diese Aktion gibt den eindeutigen Raumindexwert für den aktuellen Raum zurück. Sie müssen eine Zielvariable bereitstellen, die den zurückgegebenen Wert enthält, und diese Variable kann als temporäre lokale Variable gekennzeichnet werden.
