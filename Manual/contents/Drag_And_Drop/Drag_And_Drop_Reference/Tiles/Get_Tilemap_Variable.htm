Tilemap-Variable holen

Mit dieser Aktion können Sie eine beliebige aus einer vorgegebenen Anzahl von Variablen für das Kachel-Map-Element auf einer Ebene abrufen. Wenn Sie im Raum-Editor eine Kachel-Ebene erstellen, enthält diese Ebene ein Kachel-Map-Element, das dann mit Kacheln aus einem Kachel-Set bestückt wird. Diesem Tile-Map-Element sind bestimmte Werte zugeordnet, wie z. B. eine Offset-Position, das verwendete Tile-Set, die Breite, Höhe usw... Die vollständige Liste der Werte, die Sie erhalten können, ist:

  • X-Koordinate: Die Position entlang der X-Achse auf der Ebene, auf der das Kachelkartenelement platziert wurde.
  • Y-Koordinate: Die Position entlang der y-Achse auf der Ebene, auf der das Kachelkartenelement platziert wurde.
  • Spalten: Die Breite (definiert als die Anzahl der Spalten der Kachelzellen) des Kachelkartenelements.
  • Zeilen: Die Höhe (definiert als die Anzahl der Reihen von Kachelzellen) des Kachelkartenelements.
  • Kachelbreite: Die Breite einer einzelnen Kachelzelle.
  • Kachelhöhe: Die Höhe einer einzelnen Kachelzelle.
  • Kachelsatz-Ressource: Die Tile-Set-Ressource, die dem Tile-Map-Element zum Zeichnen zugewiesen wurde.
  • Aktueller Rahmen: Der aktuelle Rahmen, der für animierte Kacheln gezeichnet wird.
  • Maske: Die Maskendaten für das Tile-Map-Element.
  • Globale Maske: Die globalen Maskendaten für alle Tile-Map-Elemente.

Wenn Sie diese Aktion verwenden, geben Sie den Namen der Ebene (eine Zeichenkette, wie im Raum-Editor definiert) an, von der Sie die Daten des Kachelkartenelements abrufen möchten, und dann den Typ der Daten, die Sie abrufen möchten (wie in der Liste oben gezeigt). Der zurückgegebene Wert wird dann in der Zielvariable gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren). Beachten Sie, dass Sie weitere Werte abrufen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine andere Variable auswählen und eine weitere Variable zum Speichern des zurückgegebenen Wertes angeben.

Aktion Syntax:

Argumente:

Argument Beschreibung
Ebene Die Ebene, von der Daten abgerufen werden
Variabel Die Variable, deren Wert abgerufen werden soll (wie oben aufgeführt)
