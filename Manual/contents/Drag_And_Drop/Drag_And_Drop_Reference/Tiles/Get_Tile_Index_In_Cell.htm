Kachelindex in Zelle holen

Mit dieser Aktion können Sie einen Kachelindex auf einer bestimmten Ebene abrufen. Die Ebene wird aus dem angegebenen Ebenennamen (eine Zeichenkette, wie sie zur Benennung der Ebene im Raum-Editor verwendet wird) bestimmt, und dann geben Sie die Zeile und Spalte der Zelle innerhalb der Ebene an, aus der Sie den Kachel Index zu erhalten. Schließlich geben Sie die Zielvariable an, in der der zurückgegebene Kachelindex gespeichert werden soll, wobei 0 für keine Kachel und eine positive Ganzzahl für den Index der Kachel steht. Beachten Sie, dass die Zielvariable als temporäre (lokale) Variable gekennzeichnet werden kann, die bis zum Ende des Ereignisses verwendet werden kann. bis zum Ende des Ereignisses.

Aktion Syntax:

Argumente:

Argument Beschreibung
Ebene Die Ebene, aus der die Kachel geholt werden soll
Säule Die Spalte (von links nach rechts) der Zelle
Zeile Die Zeile (von oben nach unten) der Zelle
Ziel Die Zielvariable für den Rückgabewert

Beispiel:

Der obige Code des Aktionsblocks erzeugt eine Schleife, die die oberste Zeile der angegebenen Raumebene durchläuft. die oberste Zeile der angegebenen Raumebene zu durchlaufen, und prüft dabei, ob die Kachel, die mit der Zelle für die Spalte indiziert ist, auf 0 gesetzt ist. Ist dies nicht der Fall, wird der Kachelindex für diese Zelle auf 0 gesetzt, und die temporäre (lokale) Variable wird inkrementiert, bevor die Schleife einen weiteren Satz von Prüfungen durchführt.
