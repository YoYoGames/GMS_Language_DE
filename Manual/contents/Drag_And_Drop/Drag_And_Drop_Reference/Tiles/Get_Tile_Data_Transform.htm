Kacheldaten abrufen Transformieren

Mit dieser Aktion können Sie herausfinden, ob die Kacheldaten für eine Kachel auf eine von drei Arten transformiert wurden: gespiegelt, gespiegelt oder gedreht. Die Kacheldaten sind einfach ein Wert, der den Index der Kachel zusammen mit den angewendeten Transformationen wiedergibt, und Sie werden Normalerweise sollten Sie die Kacheldaten zuerst mit " Kacheldaten in Zelle holen " oder " Kacheldaten bei Pixel holen " abrufen, bevor Sie diese Aktion verwenden. Die Aktion gibt true zurück, wenn Die Aktion gibt true zurück, wenn die gewählte Transformation auf die Kacheldaten angewendet wurde, andernfalls false, und der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt worden sein kann oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" Kontrollkästchen aktivieren). Weitere Informationen zu Kacheldaten finden Sie im GML-Abschnitt über Kachelkartenfunktionen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Fliese Die Kacheldaten für die Kachel, von der die Transformation abgerufen werden soll
Transformieren Die zu prüfende Transformation

Beispiel:

Der obige Code des Aktionsblocks ruft die Kacheldaten an der Mausposition ab und prüft dann überprüft, ob auf die Kachel eine Rotationstransformation angewendet wurde. Wenn ja, wird die Transformation zurückgesetzt und die Kacheldaten setzen die Kachel wieder an die Position.
