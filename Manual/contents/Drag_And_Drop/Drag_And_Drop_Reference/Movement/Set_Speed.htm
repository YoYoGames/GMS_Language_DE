Geschwindigkeit einstellen

Mit dieser Aktion wird die Bewegungsgeschwindigkeit der Instanz festgelegt. Die Geschwindigkeit wird als die Anzahl der Pixel berechnet, die eine Instanz in einem einzelnen Spielframe (Schritt) bewegen muss. Standardmäßig wird der Geschwindigkeitswert zum Richtungsvektor addiert - wenn Sie also die Wenn Sie also die Geschwindigkeit auf 2 und die Richtung auf 45° einstellen, bewegt sich die Instanz in jedem Spielframe um 2 Pixel nach oben und rechts - Sie können aber auch auf das Richtungsfeld klicken und entweder die horizontale oder vertikale Geschwindigkeit auswählen.

Wenn Sie eine dieser Optionen wählen, wird die Geschwindigkeit nur für die horizontalen oder vertikalen Vektoren festgelegt. Beachten Sie dass Sie den Wert als relativ zur aktuellen Geschwindigkeit kennzeichnen können, wodurch der Wert zur aktuellen Geschwindigkeit der Instanz addiert oder subtrahiert wird, und Sie können auch negative Zahlen angeben, wodurch sich die Instanz in die entgegengesetzte Richtung zum aktuellen Richtungswert bewegt.
