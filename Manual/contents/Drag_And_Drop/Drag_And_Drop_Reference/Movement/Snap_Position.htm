Einrastposition

Diese Aktion wird verwendet, um eine Instanz an einer bestimmten Gitterkoordinatenposition "einzufangen". Sie geben die Größe der "Zelle" des Rasters an und die Instanz wird an diesem Raster gefangen, z. B. wenn Sie die Rasterfangwerte für horizontal und vertikal auf 8 setzen und die Instanz sich an der Position (4, 6) befindet, wird sie an der Position (0, 8) gefangen - im Wesentlichen wird die aktuelle Position auf den nächsten Wert gerundet, der durch die horizontalen und vertikalen Fangeinstellungen teilbar ist, wobei die Gitterzellen an der oberen linken Ecke des Raums (0, 0) beginnen. Die Standardwerte sind auf einen Fang von (32, 32) eingestellt, aber Sie können sie auf jeden gewünschten positiven ganzzahligen Wert ändern.

Aktion Syntax:

Argumente:

Argument Beschreibung
horizontal Der horizontale Zellenwert, an dem gefangen werden soll
vertikal Der Wert der vertikalen Zelle, an dem gefangen werden soll

Beispiel:

Der obige Code des Aktionsblocks prüft, ob die linke Maustaste Wenn dies der Fall ist, folgt die Instanz der Maus, und wenn die Taste losgelassen wird, wird die Instanz in einem 32 x 32-Raster gefangen.
