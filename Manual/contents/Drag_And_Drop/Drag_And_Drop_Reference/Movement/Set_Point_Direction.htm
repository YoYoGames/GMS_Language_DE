Sollwert Richtung

Mit dieser Aktion wird eine Richtung aus einem Vektor gesetzt, der aus der aktuellen Position der Instanz und der angegebenen Position gebildet wird. Sie geben die x- und y-Position an, auf die "gezeigt" werden soll - diese können als relativ zur Position der Instanz gekennzeichnet werden - und die Richtung von der Instanz zu diesem Punkt wird für die Instanz verwendet.

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die x-Komponente des Richtungsvektors.
y Die y-Komponente des Richtungsvektors.

Beispiel:

Der obige Code des Aktionsblocks prüft, ob eine Instanz des Objekts "obj_Player" im Raum existiert, und wenn dies der Fall ist, wird die Richtung der Instanz so eingestellt, dass sie auf die x/y-Position von "obj_Player" zeigt.
