Richtung einstellen Zufällig

Mit dieser Aktion wird die Bewegungsrichtung der Instanz auf eine oder mehrere zufällige Richtungen gesetzt, die aus den 8 Himmelsrichtungen ausgewählt werden. Sie können auf einen oder mehrere der 8 Richtungspfeile innerhalb der Aktion klicken und die Aktion wählt eine davon zufällig als Richtung für die Instanz zu verwenden. Wenn Sie keinen der Pfeile auswählen, hat die Aktion keinen Einfluss auf die aktuelle Richtung der Instanz.

Der obige Aktionsblockcode wählt entweder true oder false und speichert das Ergebnis in einer temporären (lokalen) Variablen. Diese Variable wird dann überprüft und wenn sie true ist, wird ein Satz zufälliger Richtungen gewählt und wenn sie false ist, wird ein anderer Satz gewählt.
