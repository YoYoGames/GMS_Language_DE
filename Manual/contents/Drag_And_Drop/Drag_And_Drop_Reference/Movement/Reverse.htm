Umgekehrt

Diese Aktion wird verwendet, um bestimmte Werte in Bezug auf Richtung und Bewegung umzukehren. Die verfügbaren Optionen sind:

Die Richtungsumkehrung addiert einfach 180° zur aktuellen Richtung (ggf. in einer Schleife um 360°). Wenn Sie die Richtung umkehren, addieren Sie einfach 180° zur aktuellen Richtung (ggf. in einer Schleife um 360°, d. h. wenn Sie z. B. 270° umkehren, erhalten Sie 90°). die Richtung der Schwerkraft.

Aktion Syntax:

Argumente:

Argument Beschreibung
Richtung Die Richtung zum Umkehren

Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision der Instanz mit "obj_Wall" relativ zur aktuellen Position nach links und nach rechts. Wenn eine gefunden wird, wird die horizontale Geschwindigkeit umgekehrt. Anschließend wird dasselbe für oberhalb und unterhalb der Instanz durchgeführt, wobei die vertikale Geschwindigkeit umgekehrt wird, wenn eine Kollision erkannt wird.
