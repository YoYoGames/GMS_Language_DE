Richtung der Schwerkraft einstellen

Mit dieser Aktion wird die Richtung der eingebauten Schwerkraftvariablen eingestellt. Jede Instanz kann ihre eigene Schwerkraftwirkung haben, und wenn der Schwerkraftwert auf etwas anderes als Null gesetzt wird, dann wird die Instanz in die Richtung gezogen, die durch diese Aktion festgelegt wurde. Sie können eine Richtung zwischen 0° und 360° einstellen, wobei rechts 0°, oben 90°, links 180° und unten 270° ist. Beachten Sie, dass Sie dies relativ zum aktuellen Schwerkraftwert einstellen können, d. h. was immer Sie eingeben, wird zu diesem Wert addiert bzw. von diesem subtrahiert. wird.
