Zum Punkt springen

Diese Aktion wird verwendet, um eine Instanz von einer Position zu einer anderen zu verschieben. Die Instanz bewegt sich nicht gleichmäßig zur neuen Position, sondern "springt" dorthin, d. h., wenn die Werte groß genug sind, "überspringt" sie alle anderen Instanzen auf dem Weg. Sie können absolute Werte für die Bewegung der Instanz festlegen, oder Sie können das Flag "relativ" ankreuzen, damit die Instanz relativ zur aktuellen Position bewegt wird. Wenn Sie zum Beispiel möchten, dass die Instanz nur 10 Pixel nach links springt, würden Sie den x-Wert auf -10 und den y-Wert auf 0 setzen und für beides "relativ" ankreuzen. Sie können negative Werte für absolute Koordinaten verwenden und die Instanz wird außerhalb des Basisraumbereichs bewegt.

Aktion Syntax:

Argumente:

Argument Beschreibung
x Die x-Position, zu der gesprungen werden soll
y Die y-Position, zu der gesprungen werden soll
