Map-Wert setzen

Mit dieser Aktion können Sie ein Schlüssel/Wert-Paar in die Map-Datenstruktur einfügen. Sie geben die Variable an, die den Map-Index enthält (wie von der Aktion " Map erstellen" zurückgegeben), und geben dann einen "Schlüssel" an (das ist der Bezeichner innerhalb der Map für einen Wert), und dann den "Wert", der mit diesem Schlüssel verknüpft werden soll. Beide Schlüssel können entweder Integer-Werte (wie 0, 200, 36712, etc...) oder Strings (wie "Name", "Schild" etc...) sein, aber der Wert kann jeder gültiger Datentyp sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Karte Der Index (gespeichert in einer Variablen) der zu setzenden Karte
Taste Der zu setzende Schlüssel (Integer oder String)
Wert Der Wert, der mit dem angegebenen Schlüssel gespeichert werden soll
