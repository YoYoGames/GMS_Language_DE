Pop Stack

Mit dieser Aktion können Sie einen Wert von der Spitze einer Stack-Datenstruktur abheben. Sie geben die Variable an, die den Stack-Index enthält (wie von der Aktion " Stack erzeugen" zurückgegeben), und dann die Zielvariable, die die die zurückgegebene Variable enthalten soll (die als temporäre lokale Variable gekennzeichnet sein kann).

Aktion Syntax:

Argumente:

Argument Beschreibung
Stapel Der Index (gespeichert in einer Variablen) des zu verwendenden Stacks
Ziel Die Zielvariable, die den zurückgegebenen Wert aufnehmen soll

Beispiel:

Der obige Code des Aktionsblocks prüft, ob der angegebene Stapel leer ist, und wenn nicht, wird der den obersten Wert ab und speichert ihn in einer temporären Variablen. Diese wird dann verwendet, um einen Alarm zu setzen, während, wenn der Stack leer ist, 10 zufällige Werte auf den Stack geschoben werden.
