Kartenwert holen

Mit dieser Aktion können Sie den Wert abrufen, der zu einem bestimmten Schlüssel in einer Map-Datenstruktur gehört. Sie übergeben die Variable, die den Map-Index enthält (wie von der Aktion " Map erstellen" zurückgegeben) und geben dann einen "Schlüssel" an (der ist der Bezeichner innerhalb der Map für einen Wert), und liefern auch eine Zielvariable, die den zurückgegebenen Wert enthält (die als temporäre lokale Variable gekennzeichnet werden kann).

HINWEIS: Wenn Sie einen Schlüssel angeben, der in der Map nicht vorhanden ist, gibt die Aktion undefiniert zurück. Sie können dies mit der Aktion "If Undefined " überprüfen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Karte Der Index (gespeichert in einer Variablen) der Map, aus der der Wert geholt werden soll
Taste Der zu erhaltende Schlüssel (real oder string)
Ziel Die Zielvariable, die den zurückgegebenen Map-Wert enthält

Beispiel:

Der obige Code des Aktionsblocks prüft auf eine Kollision an der Instanzposition und wenn eine gefunden wird, wird in einer Map nachgeschaut, um einen Wert aus dem Schlüssel "damage" zu erhalten, und dieser Wert wird dann verwendet, um die aufrufende Instanzvariable "hp" zu ändern.
