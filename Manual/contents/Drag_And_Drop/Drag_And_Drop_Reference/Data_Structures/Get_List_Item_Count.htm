Anzahl der Listenelemente abrufen

Mit dieser Aktion können Sie herausfinden, wie viele Einträge in einer Liste vorhanden sind. Sie übergeben die Variable, die den Listenindex enthält (wie von der Aktion " Liste erstellen" zurückgegeben), und die Zielvariable zum Speichern der zurückgegebenen Anzahl von Einträge in der Liste (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des Scripts oder Ereignisses verwendet wird).

Aktion Syntax:

Argumente:

Argument Beschreibung
Liste Der Index (gespeichert in einer Variablen), um die Elementanzahl zu erhalten von
Ziel Die Zielvariable zum Speichern der zurückgegebenen Artikelanzahl

Beispiel:

Der obige Aktionsblockcode ruft die Größe der Liste ab, auf die in der globalen referenziert wird, und verwendet dann eine for-Schleife, um sie durchzugehen und die Instanzen mit den in der Liste gespeicherten IDs zu zerstören. Nachdem die Schleife beendet ist, wird die Liste aus dem Speicher freigegeben.
