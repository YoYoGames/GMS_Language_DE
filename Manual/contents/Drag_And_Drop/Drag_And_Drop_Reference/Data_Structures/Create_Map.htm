Karte erstellen

Diese Aktion erstellt eine neue Map-Datenstruktur und gibt den Indexwert zurück, sodass Sie später über die anderen Datenstruktur-Aktionen auf die Map zugreifen können. Der Map-Index wird an die von Ihnen angegebene Zielvariable zurückgegeben, die zuvor mit Sie können auch das Kontrollkästchen " Temp" markieren, um eine temporäre lokale Variable zu benennen und zu erstellen, die den Wert bis zum Ende der den Wert bis zum Ende des Scripts oder Ereignisses zu speichern. Eine neu erstellte Map-Datenstruktur wird als "leer" betrachtet, d. h. sie enthält keine Map-Schlüssel/Werteinträge. Beachten Sie, dass Sie weitere DS-Maps erstellen können, indem Sie auf das Plus-Symbol neben der Aktion klicken und eine weitere Variable auswählen, die die Kennfeld-ID enthält.

WICHTIG! Das Erstellen von Datenstrukturen verbraucht Speicher auf der Zielplattform, und daher sollten alle Datenstrukturen mit der Aktion "Datenstruktur freigeben" freigegeben werden, wenn sie nicht mehr benötigt werden. Speicherleck, das die Leistung Ihres Spiels beeinträchtigen oder sogar zum Absturz führen kann. Dies ist besonders relevant, wenn Sie temporäre lokale Variablen zum Speichern von Datenstrukturindizes verwenden, da diese Variablen am Ende des Codes oder Ereignisses entfernt werden, aber Das bedeutet aber nicht, dass auch die Datenstruktur entfernt wird! Die Datenstruktur existiert weiterhin, Sie haben nur keine Möglichkeit, sie zu referenzieren, also verwenden Sie entweder eine Instanzvariable und geben Sie die Struktur zu einem späteren Zeitpunkt frei, oder geben Sie die Struktur vor dem Ende von des Ereignisses oder der Funktion, wenn ihr Index in einer temporären Variablen gespeichert ist.
