In Liste einfügen

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste wiederherstellen. Sie übergeben die Variable, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und den Wert, dessen Indexposition Sie von innerhalb der Liste suchen wollen, sowie eine Zielvariable, die die zurückgegebene Indexposition für das Element speichert (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des Scripts oder Ereignisses verwendet wird). Wenn Sie einen Wert in eine Liste einfügen, werden alle Indizes über dem angegebenen Index zum Einfügen um eins nach oben verschoben. Wenn Sie also eine Liste mit 4 Einträgen haben (Indexposition 0 bis 3) und einen neuen Eintrag an Indexposition 2 platzieren, wird der vorherige Eintrag 2 zu Eintrag 3 und der vorherige Eintrag 3 zu Eintrag 4. Beachten Sie, dass der Rückgabewert -1 ist, wenn der gesuchte Wert nicht in der Liste vorhanden ist und keine Indexposition der Liste zurückgegeben werden kann.

Aktion Syntax:

Argumente:

Argument Beschreibung
Liste Der Index (gespeichert in einer Variablen) der Liste, die eingefügt werden soll in
Index Die Indexposition innerhalb der Liste zum Einfügen eines Wertes
Wert Der zu speichernde Wert an der Position

Beispiel:

Der obige Code des Aktionsblocks verwendet eine Wiederholungsschleife, um die Größe der Liste zu ermitteln und dann eine eine zufällige Position innerhalb der Liste. Anschließend wird ein Zufallswert generiert und dieser Wert an der zuvor definierten Zufallsposition in die Liste eingefügt.
