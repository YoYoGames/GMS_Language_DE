Listenelement holen bei

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste wiederherstellen. Sie übergeben die Variable, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und die Indexposition innerhalb der Liste, von der Sie geben die Variable an, die den Listenindex speichert (wie von der Aktion "Liste erstellen" zurückgegeben), und die Indexposition innerhalb der Liste, von der der Elementwert abgerufen werden soll, sowie eine Zielvariable zum Speichern des zurückgegebenen Elementwerts (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des Scripts oder Ereignisses verwendet wird).

HINWEIS: Wenn Sie eine Position angeben, die außerhalb der vorgegebenen Listengröße liegt (z. B. Position 11 in einer Liste mit 10 Werten), dann gibt die Aktion undefiniert zurück. Sie können dies mit der Aktion "If Undefined " überprüfen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Liste Der (in einer Variablen gespeicherte) Index der Liste, aus der entfernt werden soll
Index Die Indexposition innerhalb der Liste, von der aus das Element geholt werden soll
Ziel Die Zielvariable zum Speichern des zurückgegebenen Wertes

Beispiel:

Der obige Aktionsblockcode ruft die Größe der in der globalen Variablen referenzierten Liste ab und verwendet dann eine for-Schleife um sie in einer Schleife zu durchlaufen und die Instanzen mit den in der Liste gespeicherten IDs zu zerstören. Nachdem die Schleife beendet ist, wird die Liste aus dem Speicher freigegeben.
