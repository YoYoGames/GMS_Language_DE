Index des Listenelements abrufen

Mit dieser Aktion können Sie die Indexposition eines Elements innerhalb der angegebenen Liste wiederherstellen. Sie übergeben die Variable, die den Listenindex speichert (wie von der Aktion Liste erstellen zurückgegeben) und den Wert, dessen Indexposition Sie von innerhalb der Liste suchen wollen, sowie eine Zielvariable, die die zurückgegebene Indexposition für das Element speichert (die als temporäre lokale Variable gekennzeichnet werden kann, die bis zum Ende des Scripts oder Ereignisses verwendet wird). Beachten Sie, dass, wenn der gesuchte Wert nicht in der Liste vorhanden ist, gibt es keine Indexposition in der Liste, die zurückgegeben werden kann, so dass der Rückgabewert -1 ist.

Aktion Syntax:

Argumente:

Argument Beschreibung
Liste Der (in einer Variablen gespeicherte) Index der Liste, aus der entfernt werden soll
Wert Der Wert, auf den die Liste geprüft werden soll
Ziel Die Zielvariable zum Speichern des zurückgegebenen Index

Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision an der Instanzposition und wenn eine gefunden wird, wird der eindeutige ID-Wert für die Instanz in einer temporären Variablen gespeichert und dann geprüft, ob er in der Listendatenstruktur existiert. Wenn dies der Fall ist, wird das Element aus der Liste entfernt und die Instanz zerstört.
