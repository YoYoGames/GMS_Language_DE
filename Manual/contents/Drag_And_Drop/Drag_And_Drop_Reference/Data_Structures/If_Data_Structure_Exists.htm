Wenn Datenstruktur vorhanden ist

Mit dieser Aktion können Sie eine Variable daraufhin überprüfen, ob sie eine Datenstruktur indiziert oder nicht. Sie müssen zuerst den Typ der Datenstruktur auswählen, auf die geprüft werden soll (entweder eine Liste, eine Karte, ein Gitter oder ein Stapel) und dann die Variable angeben, die den Datenstruktur Index (wie von den Aktionen Liste erstellen, Karte erstellen, Gitter erstellen und Stapel erstellen zurückgegeben). Die Aktion gibt true zurück, wenn die Datenstruktur existiert, und false, wenn sie nicht existiert. Beachten Sie, dass Sie "not" in der Aktion markieren können, um zu prüfen, ob eine Variable keinen Datenstrukturindex enthält.

Beachten Sie, dass zum Hinzufügen von Aktionen in den "if"-Block diese an der Seite der Aktion abgelegt werden sollten, wie im Bild unten gezeigt:

Diese Aktionen werden nun ausgeführt, wenn der "if"-Block als wahr ausgewertet wird, während alle Aktionen, die an anderer Stelle ausgelassen werden, werden nach dem "if"-Block ausgeführt.

Aktion Syntax:

Argumente:

Argument Beschreibung
Aufbau Der Typ der Datenstruktur, auf die geprüft werden soll (Liste, Map, Grid oder Stack)
Variabel Die Variable, die den Index der Datenstruktur enthält
