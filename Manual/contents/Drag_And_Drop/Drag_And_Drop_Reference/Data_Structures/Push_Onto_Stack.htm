Auf den Stapel schieben

Mit dieser Aktion können Sie einen Wert auf eine Stack-Datenstruktur schieben. Sie liefern die Variable, die den Stack-Index enthält (wie von der Aktion Stack erstellen zurückgegeben) und geben dann den Wert an, der nach oben geschoben werden soll. Bei Stack-Datenstrukturen Strukturen werden Werte immer an der "Spitze" des Stapels hinzugefügt und nie am Ende oder in der Mitte (wenn Sie diese Funktionalität benötigen, verwenden Sie eine Listendatenstruktur).

Aktion Syntax:

Argumente:

Argument Beschreibung
Stapel Der Index (gespeichert in einer Variablen) des zu verwendenden Stacks
Wert Der Wert, der auf den Stack geschoben werden soll

Beispiel:

Der obige Aktionsblockcode erstellt eine Instanzvariable und eine neue Stack-Datenstruktur. Der Index des Stapels wird in der neuen Variablen gespeichert, und dann wird eine Schleife ausgeführt, die 10 Instanzen erstellt und deren eindeutige ID-Werte auf den Stapel schiebt.
