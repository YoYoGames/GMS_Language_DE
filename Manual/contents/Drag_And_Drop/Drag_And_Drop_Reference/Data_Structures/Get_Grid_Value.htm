Rasterwert holen

Mit dieser Aktion können Sie eine bestimmte Zelle innerhalb der Grid-Datenstruktur auf einen Wert setzen. Sie übergeben die Variable, die den Grid-Index enthält (wie von der Aktion Create Grid zurückgegeben), und geben dann die x- und y-Position (die Spalte und Zeile) für die zu setzende Zelle, und schließlich den Wert für diese Zelle. Sie müssen auch eine Zielvariable setzen (die als temporäre lokale Variable gekennzeichnet werden kann), um den zurückgegebenen Zellwert zu speichern, und der Wert kann jeder gültige Datentyp sein. Beachten Sie, dass die x- und y-Position für die Zelle ganze Zahlen innerhalb der Grenzen des Gitters sein müssen.

Aktion Syntax:

Argumente:

Argument Beschreibung
Raster Der (in einer Variablen gespeicherte) Index des zu erhaltenden Gitters
x Die zu erhaltende Zellenposition entlang der x-Achse (die Spalte)
y Die zu erhaltende Zellenposition entlang der y-Achse (die Zeile)
Ziel Die Zielvariable, die zum Speichern des zurückgegebenen Wertes verwendet wird

Beispiel:

Der obige Code des Aktionsblocks ruft den Wert im Raster ab, der in einer globalen Variablen gespeichert ist, die die der Position der Instanz innerhalb des Raums entspricht. Dieser Wert wird dann überprüft und wenn es sich um einen Instanz-ID-Wert handelt, wird diese Instanz zerstört und die Rasterzelle auf das Schlüsselwort noone gesetzt.
