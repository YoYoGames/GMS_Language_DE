Rasterwert setzen

Mit dieser Aktion können Sie eine bestimmte Zelle innerhalb der Grid-Datenstruktur auf einen Wert setzen. Sie übergeben die Variable, die den Grid-Index enthält (wie von der Aktion Create Grid zurückgegeben) und geben dann die x- und y-Position (die Spalte und Zeile) für die zu setzende Zelle, und schließlich den Wert für diese Zelle. Der Wert kann ein beliebiger gültiger Datentyp sein, aber die x- und y-Position müssen ganze Ganzzahlen innerhalb der Grenzen des Gitters sein.

Aktion Syntax:

Argumente:

Argument Beschreibung
Raster Der Index (gespeichert in einer Variablen) des zu setzenden Gitters
x Die zu setzende Zellenposition entlang der x-Achse (die Spalte)
y Die zu setzende Zellenposition entlang der y-Achse (die Zeile)
Wert Der Wert, auf den die Gitterzelle gesetzt werden soll

Beispiel:

Der obige Code des Aktionsblocks erstellt eine neue Grid-Datenstruktur und hält ihren Indexwert in einer globalen Variablen. Er durchläuft dann alle Instanzen im Raum und fügt ihren eindeutigen ID-Wert in die Zelle ein, die ihrer Position entspricht.
