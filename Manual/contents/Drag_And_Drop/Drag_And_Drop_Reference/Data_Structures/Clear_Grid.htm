Raster löschen

Mit dieser Aktion können Sie eine Grid-Datenstruktur auf einen einzelnen Wert löschen (jede Zelle innerhalb des Grids wird auf den angegebenen Wert gesetzt). Sie geben die Variable an, die den Gitterindex enthält (wie von der Aktion " Gitter erstellen" zurückgegeben) zurückgegeben wird, und geben dann den Wert an, auf den das Gitter gesetzt werden soll.

Aktion Syntax:

Argumente:

Argument Beschreibung
Raster Der Index (gespeichert in einer Variablen) des zu löschenden Gitters
Wert Der Wert, auf den das Gitter gelöscht werden soll

Beispiel:

Der obige Aktionsblockcode löscht die angegebene Grid-Datenstruktur, um das Schlüsselwort noone in jeder Zelle, wenn eine globale Variable kleiner oder gleich 0 ist.
