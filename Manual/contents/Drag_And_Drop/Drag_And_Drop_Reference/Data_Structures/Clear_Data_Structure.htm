Datenstruktur löschen

Diese Aktion löscht alle Werte aus der angegebenen Datenstruktur. Sie müssen zuerst den Typ der zu löschenden Datenstruktur auswählen und dann den Index der Datenstruktur angeben (der Index ist der Wert, der zurückgegeben und in einer Variablen gespeichert wird, wenn Sie die Datenstruktur ursprünglich erstellt haben). Eine gelöschte Datenstruktur wird als "leer" betrachtet (d. h., sie hat keine Einträge), außer bei Gittern . Bei Gittern werden mit dieser Aktion einfach alle Gitterzellen auf 0 gelöscht.

HINWEIS: Dadurch wird die Datenstruktur nicht aus dem Speicher befreit, sondern es werden lediglich alle Informationen aus ihr entfernt. Um den der Datenstruktur zugewiesenen Speicher freizugeben, müssen Sie die Aktion Datenstruktur freigeben verwenden.

Aktion Syntax:

Argumente:

Argument Beschreibung
Aufbau Der Typ der Datenstruktur, die Sie löschen (Liste, Karte, Raster oder Stapel)
Variabel Die Variable, die den Index der zu löschenden Datenstruktur enthält

Beispiel:

Der obige Code des Aktionsblocks prüft, ob die in der globalen Variablen referenzierte Datenstruktur referenzierte Datenstruktur nicht leer ist, und wenn das der Fall ist, wird die Datenstruktur geleert.
