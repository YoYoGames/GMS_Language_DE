Ihre Marktplatz-Bibliothek

Das Bibliotheksfenster zeigt Ihnen die Pakete, die Sie gekauft haben, und gibt Ihnen die Möglichkeit, sie herunterzuladen und/oder zu Ihrem Projekt hinzuzufügen.Auf der linken Seite befinden sich die verschiedenen Kategoriefilter, die Sie verwenden können, um zu sehen, was Sie in jeder Kategorie haben, und in der Mitte haben Sie die Liste der Assets in Ihrer Bibliothek. Jedes Element kann ausgewählt werden, so dass Sie eine Übersicht über seine Funktionen sehen können (rechts). Rechts), und sie haben alle Symbole zum Anzeigen, Herunterladen oder Installieren des Asset-Pakets in einem Projekt. Beachten Sie, dass Sie oben im Hauptfenster zwei zusätzliche Filter haben, mit denen Sie auswählen können, dass nur die Assets angezeigt werden, die speziell für GameMaker Studio 2 erstellt wurden, sowie einen Filter, der nur die Assets anzeigt, die Sie bereits auf Ihren Rechner heruntergeladen haben. Sie können die Liste der angezeigten Assets auch jederzeit aktualisieren, indem Sie auf die Schaltfläche Aktualisieren klicken.

Um ein Anlagenpaket zu Ihrem aktuellen Projekt hinzuzufügen, müssen Sie es zunächst herunterladen, indem Sie auf die Schaltfläche " Herunterladen" klicken. Wenn der Download abgeschlossen ist, können Sie auf die Schaltfläche " Importieren" klicken, um es zu Ihrem Projekt hinzuzufügen. Das Hinzufügen eines Pakets auf diese Weise wird eine Erweiterung innerhalb Ihres aktuellen Projekts erstellt, und Sie werden aufgefordert, aus den verschiedenen Ressourcen des Anlagenpakets auszuwählen, welche Sie hinzufügen möchten (oder Sie können alle importieren). Sie können auch die Option Alle Ressourcen in ein neues Projekt importieren wählen, und statt die Assets zu Ihrem aktuellen Projekt hinzuzufügen, fordert GameMaker Studio 2 Sie auf, einen neuen Projektnamen zu vergeben und die Assets stattdessen in diesem Projekt zu speichern. Beachten Sie, dass Sie, wenn Sie direkt von der Startseite aus direkt zur Bibliothek gehen, werden Sie feststellen, dass die meisten Assets nicht einzeln hinzugefügt werden können und Sie stattdessen gezwungen sind, die Schaltfläche " Alle hinzufügen" zu verwenden. Bitte beachten Sie, dass vom Marketplace heruntergeladene Assets für den individuellen Gebrauch bestimmt sind und nicht an Dritte weitergegeben werden dürfen - obwohl Sie sie zur Sicherheit sichern und an andere Orte kopieren können - es sei denn, das Asset hat eine eigene EULA, die das Gegenteil besagt.

HINWEIS: Die Personen, die Asset-Pakete für den Marketplace erstellen, arbeiten sehr hart daran, Ihnen eine großartige Ressource zur Verfügung zu stellen. Bitte denken Sie daher daran, die Assets, die Sie kaufen, zu bewerten, wenn Sie sie getestet haben oder verwenden. Dies ermutigt die Entwickler und ist auch für andere Benutzer ein nützlicher Hinweis auf die Qualität der verfügbaren Assets. Sie können Ihre heruntergeladenen Assets ganz einfach über das Bibliotheksfenster bewerten.
