Erstellen eines Publisher-Kontos

Bevor Sie den YoYo Games Marketplace nutzen können, um Assets zu kaufen oder zu verkaufen, müssen Sie ein YoYo-Konto erstellt haben. Wenn Sie Asset-Pakete verkaufen möchten, müssen Sie außerdem eine GameMaker Studio 2 Lizenz (Creator, Desktop, Web, Mobile oder Konsole) erworben und sich für ein Marketplace Publisher-Konto registriert haben.

HINWEIS: Mit der 30-tägigen Testlizenz für GMS2 können Sie auf dem Marktplatz kostenlose oder kostenpflichtige Assets kaufen und herunterladen, aber Sie können damit kein Publisher-Konto erstellen.

Die Schritte, die Sie für die Registrierung als Publisher durchführen müssen, sind wie folgt:

  • Melden Sie sich bei Ihrem YoYo-Konto an.
  • Sobald Sie sich in Ihrem Konto-Dashboard befinden, besuchen Sie die Seite Publisher-Registrierung. Diese finden Sie in Ihrem YoYo-Konto-Dashboard im Publisher-Bereich, wo Sie auf den Abschnitt Sell Your Assets klicken müssen:
  • Klicken Sie auf der nächsten Seite auf die Schaltfläche " Registrieren" und geben Sie dann einige grundlegende Informationen zu Ihrer Publisher-Identität ein, z. B. Ihren Publisher-Namen, Ihre E-Mail-Adresse und die Arten von Assets, die Sie hochladen möchten.

Sie müssen die Marketplace-Publisher-Vereinbarung lesen und akzeptieren, bevor Sie die Registrierung Ihres Publisher-Kontos abschließen können. Wenn Sie dies tun, ist Ihre Anmeldung vollständig, aber beachten Sie, dass alle Registrierungen von YoYo Games verifiziert werden müssen, so dass Ihr neuer Publisher nicht sofort für Sie verfügbar sein wird. Wenn Ihre Registrierung verifiziert ist, werden Sie über die von Ihnen angegebene E-Mail-Adresse benachrichtigt. (Bitte beachten Sie, dass wir zu den üblichen Geschäftszeiten in Großbritannien arbeiten, so dass es bei Anmeldungen am Wochenende etc. zu einer Verzögerung der Genehmigung.)

Sobald Sie die Registrierung abgeschickt haben, werden Sie zu Ihrem Publisher-Konto weitergeleitet, wo Sie die bereits gemachten Angaben sehen und bearbeiten können, sowie sowie einen Avatar hochladen können:

Sie sollten nun in den Bereich Zahlungsdetails gehen und dort das Formular ausfüllen:

Bitte beachten Sie das:

  • Derzeit bietet der Marketplace nur die Bezahlung über PayPal an. Wenn Sie also ein Publisher-Konto anlegen, müssen Sie eine gültige E-Mail-Adresse für ein PayPal-Konto angeben, da Sie sonst nicht bezahlt werden können.
  • Für Benutzer aus Großbritannien/EU sollte eine gültige Umsatzsteuer-Identifikationsnummer eingestellt werden, wenn dies für Ihr Unternehmen zutrifft (für Benutzer außerhalb Großbritanniens sollten Sie Ihrer Umsatzsteuer-Identifikationsnummer den zweistelligen Ländercode für Ihr Land voranstellen).

Wenn Sie vorhaben, nur kostenlose Assets auf den Marketplace hochzuladen, können Sie die Felder PayPal und VAT ignorieren. Sie können dann auf die Schaltfläche Senden klicken, um fortzufahren.

Sobald Ihre Registrierung verifiziert wurde, können Sie die restlichen Optionen der Publisher-Konsole nutzen. Dies ist das "Zuhause" all Ihrer Asset-Publishing-Vorgänge, und Sie sehen ein Menü mit Elementen auf der linken Seite der Seite. Jedes zeigt Ihnen verschiedene Details zu Ihren Publishing-Aktivitäten an:

  • Assets: Auf dieser Seite erhalten Sie einen schnellen Überblick über Ihre Assets (Sprites, Audio, Projekte usw.) und können deren Details anzeigen und bearbeiten, Ihre Verkäufe überprüfen, Statistiken herunterladen und auch eine neue Version Ihres Assets veröffentlichen. Sie können auch neue Assets, die Sie erstellt haben, zur Veröffentlichung hinzufügen. Weitere Informationen zum Veröffentlichen von Assets finden Sie auf der Seite: Auflisten eines Assets.
  • Dienste: Hier können Sie die Dienste sehen, die Sie anderen Marketplace-Benutzern anbieten, sowie deren Details bearbeiten und neue Dienste aktualisieren oder hinzufügen. Weitere Informationen zu Diensten finden Sie auf der Seite: Einen Dienst auflisten.
  • Kontodetails: Aktualisieren Sie das öffentliche Profil Ihres Publishers im Shop und fügen Sie Zahlungsdetails über Kontodetails hinzu (siehe oben für weitere Informationen zum Einrichten eines Zahlungskontos).
  • Benutzerzugang: Wenn Sie in einem Team arbeiten, können Sie mehrere Benutzer zu Ihrem Publisher-Konto hinzufügen und festlegen, auf welche Teile der Publisher-Konsole sie zugreifen können. Das Konto, das das Publisher-Konto registriert, ist der Eigentümer und hat vollen Zugriff auf alle Teile der Developer Console. Der Besitzer kann dann Benutzerkonten hinzufügen und verwalten, auf welche Teile der Developer Console sie Zugriff haben. Geben Sie einfach die E-Mail-Adresse der Person an, die Sie hinzufügen möchten, und markieren Sie dann die Teile der Publisher Identität an, auf die sie Zugriff haben soll.
  • Analytik: Über die Entwicklerkonsole haben Sie Zugriff auf tägliche Verkaufs- und Download-Berichte und Diagramme, mit denen Sie die Leistung Ihrer Assets verfolgen können. Sie können Ihre Berichte nach mehreren Dimensionen filtern, darunter Paid oder Free, Kategorie, Land und Datum.
  • Käufe: Hier sehen Sie eine Liste aller Käufe, die von Ihren Assets getätigt wurden (kostenlose Käufe werden angezeigt, aber mit einem Preis von $0,00)
  • Autorenbedingungen: Hier finden Sie die aktuellen Autorenbedingungen für das Veröffentlichen auf dem Marketplace. Sie müssen diese akzeptieren, bevor Sie etwas veröffentlichen können, auch kostenlose Assets.

Beachten Sie, dass Sie, sobald Sie ein Publisher-Konto eingerichtet und verifiziert haben, zurück zu Ihrem YoYo Account Dashboard gehen und bei Bedarf weitere Konten anlegen können. Auf diese Weise können Sie zwei separate Konten anlegen, von denen eines Kunstwerke und das andere Code-Assets verkaufen (zum Beispiel).
