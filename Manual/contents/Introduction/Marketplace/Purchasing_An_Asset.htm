Kauf einer Anlage

Um ein Asset auf dem Marktplatz zu kaufen, müssen Sie zunächst in Ihrem YoYo-Konto eingeloggt sein. Sobald Sie Ihr Konto erstellt haben und auf dem Marktplatz angemeldet sind, können Sie die verfügbaren Asset-Pakete durchsuchen. Sie können offizielle YoYo Games-Inhalte oder von anderen Community-Mitgliedern erstellte Inhalte herunterladen, und die Inhalte können entweder bezahlt oder kostenlos sein.

Wenn Sie ein Asset-Paket für GameMaker Studio 2 kaufen möchten, müssen Sie es einfach aus den Marktlisten auswählen und auf die Schaltfläche "Kaufenfür [Preis]" klicken. Dadurch wird das Produkt in den Warenkorb gelegt, und Sie Sie können entweder zur Kasse gehen oder weiter in den Assets stöbern und weitere Einkäufe tätigen.

HINWEIS: Kostenlose Assets werden nicht in Ihren Warenkorb gelegt, sondern Sie werden lediglich aufgefordert, den Bedingungen der EULA zuzustimmen, und nach der Zustimmung wird das Asset zu Ihrer Bibliothek hinzugefügt.

Wenn Sie bereit sind, mit der Zahlung fortzufahren, können Sie die Option "Warenkorb" oben rechts auf dem Bildschirm auswählen und Sie werden zu einer Übersicht der Produkte in Ihrem Warenkorb geführt. Hier können Sie durch einen Kreditkartenkauf oder über PayPal bezahlen, oder Ihre Einkäufe ändern:

Wenn Sie die Zahlung abgeschlossen haben, wird Ihnen ein Bildschirm "Kauf abgeschlossen" angezeigt, auf dem Sie den Beleg ausdrucken können. Außerdem wird Ihnen Außerdem erhalten Sie eine E-Mail mit den Details zur Zahlungsbestätigung. Wenn Sie einen Kauf abgeschlossen haben (entweder ein kostenloses oder ein kostenpflichtiges Asset), können Sie zurück zum GameMaker Studio 2 Marketplace-Menü gehen und auf den Menüpunkt "Meine Bibliothek" klicken, wo Sie die gekauften Assets herunterladen und installieren können. Weitere Informationen zum Installieren von Assets finden Sie auf der Seite über die Bibliothek.
