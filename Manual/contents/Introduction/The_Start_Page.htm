Die Start-Seite

Wenn Sie GameMaker Studio 2 zum ersten Mal öffnen, werden Sie aufgefordert, sich bei Ihrem YoYo-Konto anzumelden (oder sich für ein Konto zu registrieren, falls Sie dies noch nicht getan haben - siehe hier ). Sobald Sie registriert und eingeloggt sind, wird die Haupt-IDE von GameMaker Studio 2 auf der Startseite geöffnet:

Oben finden Sie die allgemeinen IDE-Menüs, die in den folgenden Abschnitten des Handbuchs erläutert werden:

  • Das Menü "Datei
  • Das Menü Bearbeiten
  • Das Menü "Erstellen
  • Das Windows-Menü
  • Das Menü "Werkzeuge
  • Das Marktplatz-Menü
  • Das Menü Layouts
  • Das Hilfe-Menü

Beachten Sie, dass in der oberen Menüleiste auch kontextspezifische Menüoptionen angezeigt werden, je nachdem, auf welches Fenster Sie gerade fokussiert haben. Wenn Sie z. B. das Bildeditor-Fenster fokussiert haben, finden Sie hier zusätzliche Menüpunkte Menüpunkte für "Bilder", "Ansicht" und "Effekte". Diese zusätzlichen Menüpunkte werden in den entsprechenden Abschnitten des Handbuchs für den Arbeitsbereich oder das Fenster erklärt, das sie erzeugt hat. sie erzeugen.

Der Rest der Startseite sieht folgendermaßen aus:

  • Home - Wenn Sie durch die verschiedenen Bereiche der Startseite navigieren, können Sie mit einem Klick auf die linke Maustaste einfach zurück zur Hauptseite zurück.
  • Versionsdetails - Dieser Abschnitt enthält Details zur aktuellen IDE-Version, die verwendet wird, sowie die aktuelle Laufzeit Version. Außerdem werden Sie in diesem Abschnitt über alle verfügbaren Änderungen entweder an der IDE oder an der Runtime in diesem Abschnitt.
  • Letzte Projekte - Auf der linken Seite der Startseite sehen Sie eine Liste mit früheren Projekten, die Sie öffnen können. Wenn Sie mit der Maus darüberfahren, sehen Sie den vollständigen Projektnamen und -pfad, und wenn Sie mit der linken Maustaste auf klicken, wird das Projekt geöffnet.
  • Soziales - Die obere Leiste der Startseite enthält eine Reihe von Links, die das Navigieren erleichtern. Sie sind unten aufgelistet:
    • YoYo Games - bringt Sie zur Haupt-Websitevon YoYo Games
    • Social-Media-Symbole - diese führen Sie zu den verschiedenen Social-Media-Seiten von YoYo Games, auf denen Sie über Neuigkeiten zu GameMaker Studio 2auf dem Laufenden bleiben können
  • Projektoptionen - Im Bereich Projekte können Sie Projekte erstellen, öffnen oder importieren. Dies wird weiter unten auf dieser Seite ausführlicher erklärt.
  • Informationen - Der Informationsbereich der Startseite erlaubt es Ihnen, die YoYo Games Marktplatz sowie das Durchsuchen und Öffnen von Tutorials und Demos, die vom YoYo Games-Team erstellt wurden, um um bestimmte Aspekte von GameMaker Studio 2 vorzustellen (siehe unten für weitere Informationen).

Ein neues Projekt starten

Sie können auf die Schaltfläche Neu klicken, um ein neues Projekt zu erstellen, auf die Schaltfläche Öffnen , um ein bestehendes Projekt zu öffnen, oder auf die Schaltfläche Importieren , um eine GameMaker: Studio 1.4 GMX Projektdatei oder eine komprimierte YYZ GameMaker Studio 2-Projektdatei zu öffnen. Sowohl Öffnen als auch Importieren öffnen den Datei-Explorer in dem Sie die gewünschte Projektdatei suchen können.

HINWEIS: GameMaker Studio 2 ist nicht vollständig abwärtskompatibel zu GameMaker: Studio 1.4-Projekten. Importierte 1.4-Projekte sollten jedoch weiterhin laufen, da veraltete Funktionen mithilfe von Kompatibilitätsscripts automatisch für Sie neu erstellt wurden. Ausführliche Informationen zu den möglichen Problemen und den an GML vorgenommenen Änderungen finden Sie im Hilfe-Center-Artikel Portieren eines GMS 1.4-Projekts auf GameMaker Studio 2, sowie den Abschnitt des Handbuchs über veraltete Funktionen.

Um ein neues Projekt zu erstellen, klicken Sie einfach auf die Schaltfläche Neu, die Sie zum folgenden Bildschirm führt:

InGameMaker Studio 2 können Sie festlegen, welche Art von Projekt Sie erstellen möchten, und Sie können je nach Ihren Programmierkenntnissen oder -gewohnheiten eine Auswahl treffen. Die verfügbaren Optionen sind:

  • Neues Drag-and-Drop™-Projekt - Damit wird ein neues Projekt erstellt, in dem der Objekteditor (und alle anderen Editoren, die Code zulassen) so eingerichtet wird, dass er den GameMaker Studio 2 Ziehen und Ablegen™ Schnittstelle. Weitere Informationen zu Drag and Drop(DnD™) finden Sie hier. Beachten Sie, dass dies keine exklusive DnD™-Schnittstelleist und Sie zwischen DnD™und GML-Code mit der rechten Maustaste umschalten können Taste wechseln.
  • Neues GameMaker-Sprachprojekt: Damit wird ein Projekt erstellt, bei dem die Standardprogrammierung mit der GameMaker Language(GML) erfolgt. Weitere Informationen zu GML finden Sie hier. Beachten Sie, dass dies keine ausschließliche GML-Schnittstelle ist und Sie mit der rechten Maustaste zwischen GML-Code und DnD™wechseln können .

Unabhängig davon, was Sie auswählen, werden Sie anschließend aufgefordert, einen Speicherort für das Projekt zu wählen, und dann wird es in GameMaker Studio 2 geöffnet.

Tutorials und Demos

Wenn Sie auf die Schaltflächen Tutorials oder Demos klicken, gelangen Sie zu einem Bildschirm, der die Tutorials/Demos anzeigt, die Sie besitzen, sowie die offiziellen Tutorials/Demos, die zum Herunterladen verfügbar sind:

Um eines der offiziellen Tutorials oder Demos herunterzuladen, klicken Sie einfach auf das Symbol des gewünschten Tutorials, und es wird zu Ihrem Abschnitt "Eigene " oben hinzugefügt (beachten Sie, dass Sie sich möglicherweise beim Marketplace anmelden müssen bevor dies geschehen kann). Sobald sich das Tutorial oder die Demo in der "Owned"-Sektion befindet, können Sie mit der Maus darüberfahren und es entweder installieren oder die Informationen dazu auf dem Marketplace ansehen :

Sobald ein Lernprogramm oder eine Demo installiert wurde, müssen Sie nur noch darauf klicken, um es zu öffnen. Sie werden dann aufgefordert, einen Speicherort für das Lernprogramm-Projekt anzugeben, bevor sich die IDE öffnet und Sie beginnen können. Beachten Sie, dass installierte Tutorials und Demos ein zusätzliches Symbol haben, mit dem Sie das Projekt von Ihrem Computer deinstallieren können (dadurch wird das Tutorial oder die Demo nicht aus der Liste Ihrer "Besitzt"-Liste).

Der Bereich Tutorials/Demos auf der Startseite enthält außerdem einige Schaltflächen, die Ihnen helfen, Ihre Assets besser zu nutzen und zu navigieren:

Wenn Sie darauf klicken, wird die Projektliste im Marketplace aktualisiert, so dass Sie überprüfen können, ob etwas aktualisiert wurde und die neuesten Versionen erhalten.
Wenn Sie darauf klicken, wird die Ansicht zwischen den Tutorials oder Demos, die Sie besitzen, und denen, die Sie installiert haben, umgeschaltet.

Es ist auch erwähnenswert, dass Sie auch Tutorials und Demos von Drittanbietern auf dem Marktplatz finden können. Sobald diese gekauft und zu Ihrem Konto hinzugefügt wurden, erscheinen sie auch im Abschnitt "Eigene" auf der entsprechenden Seite.
