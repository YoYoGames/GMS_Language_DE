Das Inspektorfenster

Das Inspektor-Fenster ist ein Fenster, das alle Eigenschaften anzeigt, die mit einem ausgewählten Element in der IDE verbunden sind. Das Fenster zeigt Informationen Das Fenster zeigt Informationen über das zu prüfende Element sowie alle bearbeitbaren Eigenschaften für dieses Element an und erlaubt Ihnen, diese nach Bedarf zu ändern und das zu prüfende Asset entsprechend zu aktualisieren.

WICHTIG! Derzeit ist das Inspektorfenster nur für die Inspektion von Sequenzeigenschaften verfügbar, aber in zukünftigen Updates wird es auf andere Assets erweitert werden.

Sie können jederzeit mehrere Inspektor-Fenster öffnen, und jedes kann mit einem bestimmten Element in der IDE verknüpft werden. Um sicherzustellen, dass sich nicht jedes Mal, wenn Sie auf ein Element klicken, alle geöffneten Inspektoren ändern, um die neue Auswahl anzuzeigen, müssen Sie zuerst das Inspektor-Fenster sperren. Dazu öffnen Sie einen Inspektor, wählen ein zu inspizierendes IDE-Element aus und klicken dann auf die Schloss-Schaltfläche des Inspektors, um ihn für das Element zu sperren (siehe Bild unten). Sobald Sie den Inspektor gesperrt haben, können Sie einen anderen Inspektor öffnen und den Vorgang wiederholen.

Beachten Sie, dass ein gesperrtes Inspektorfenster nicht die Bearbeitung von Werten für das zu inspizierende Element sperrt. Es sperrt lediglich den Inspektor auf ein Element und Sie können es jederzeit weiter bearbeiten, auch wenn es nicht der Fokus der Haupt-IDE ist.
