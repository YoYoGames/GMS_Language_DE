Projekt Image Generator

Der Projektbild-Generator wird über das Menü "Werkzeuge" am oberen Rand der GameMaker Studio 2 IDE aufgerufen. am oberen Rand der GameMaker Studio 2 IDE aufgerufen. Dieses Werkzeug dient dazu, schnell alle verschiedenen Splash-Screens, Promo-Bilder und Icons, die für die verschiedenen Zielplattformen benötigt werden, aus einem einzigen Satz von Bildern zu erstellen. Sie liefern das Basisbild und das Basissymbol an und klicken dann auf die Schaltfläche " Generieren" , und Ihre Spieloptionen für jede Plattform werden mit den neuen Bildern aktualisiert.

Damit dies korrekt funktioniert, müssen Sie einen hochauflösenden Splash-Screen und ein hochauflösendes Symbolbild bereitstellen. Die empfohlenen Größen sind:

  • Splash Screen: mindestens 1920x1080px, oder 1080x1920px für Hochformat, und als 24bit *.png ohne Transparenzen verfasst.
  • Icons: mindestens 1024x1024px (das ist die Größe des größten Icons, das benötigt wird), und als 24bit *.png verfasst, das Transparenzen haben kann

Sobald Sie diese Bilder bereitgestellt haben, werden sie im Vorschaufenster angezeigt und Sie können dann die Art der "Einpassung" einstellen, die bei der Anwendung auf die verschiedenen Plattformen verwendet werden soll. Sie können aus den folgenden wählen:

  • Seitenverhältnis beibehalten - damit wird das Bild so skaliert, dass es passt, wobei das Seitenverhältnis beibehalten wird und eine Hintergrundfarbe hinzugefügt wird, um die Seiten auszufüllen, wenn die Anpassung nicht 1:1 ist
  • Auf Größe strecken - damit werden die Bilder so gestreckt, dass sie auf die volle Größe des Bildes/Icons passen.
  • Zuschneiden - dies nimmt das Basisbild und passt es am besten entlang der horizontalen oder vertikalen Achse an und schneidet dann den Rest zu. Wenn Sie z. B. ein 640x480-Bild haben, das als Splash-Screen für 480x800 verwendet werden soll, dann wird das Bild so skaliert, dass es auf die vertikale Achse passt (480 -> 800) und der Rest des Bildes wird weggeschnitten. Wenn Sie ein Bild mit einer Größe von 1080x1920 haben, das in 800x800 passen muss, wird das Bild verkleinert, damit es auf die horizontale Achse passt. Achse zu passen und der Rest wird weggeschnitten.

Sie können auch die Hintergrundfarbe sowohl für Symbole als auch für Bilder einstellen. Diese Farbe wird zum "Auffüllen" des Bildes bei der Skalierung mit dem Seitenverhältnis verwendet. Das folgende Bild zeigt, wie dies aussehen wird, wenn alles eingerichtet ist:

Jetzt müssen Sie nur noch auf " Generieren" klicken, und GameMaker Studio 2 füllt alle Einstellungen in den Spieloptionen "Bild" und "Symbol" mit Bildern für jede Plattform. Icon-Einstellungen mit Bildern für jede Plattform.

HINWEIS: Die Ausnahme von der obigen Aussage sind Android Adaptive Icons. Diese werden aufgrund ihrer Struktur nicht mit diesem Tool erzeugt.

WICHTIG! Dieses Tool generiert zwar alle Bilder, die Sie benötigen, aber das bedeutet nicht, dass es perfekt ist! Es wird sein Bestes tun, um die Bilder an die verschiedenen Größen und Auflösungen der einzelnen Plattformen anzupassen, aber manchmal sieht es vielleicht nicht so aus, wie Sie es sich wünschen. aber manchmal sieht es nicht so aus, wie Sie es sich wünschen, da es keine "Einheitslösung" gibt.
