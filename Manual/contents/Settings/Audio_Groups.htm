Audio-Gruppen

Der Audiogruppen-Manager ist über das Menü "Werkzeuge" in der IDE verfügbar. Hier können Sie Audiogruppen hinzufügen, löschen und umbenennen sowie ihre Plattform-Exportoptionen einstellen. GameMaker Studio 2 erlaubt es Ihnen, die einzelnen Audio-Ressourcen (Soundeffekte und Musik) verschiedenen Audiogruppen zuzuordnen zuzuordnen, um die Anzahl der gleichzeitig abgespielten Sounds zu optimieren, und Ihnen weitere Kontrolle darüber zu geben, auf welche Plattformen sie exportiert werden.

Um eine Audiogruppe zu definieren, müssen Sie auf auf die Schaltfläche Neu hinzufügen klicken, wodurch eine neue Gruppe erstellt wird, die Sie dann benennen können. Um eine Gruppe zu ändern, klicken Sie auf auf das Dropdown-Menü und wählen Sie die Gruppe aus, die Sie bearbeiten möchten.

HINWEIS: Wenn Sie im Sound-Editor Streaming-Audio für einen Sound auswählen, können Sie den Sound nicht einer Audiogruppe zuweisen, da Streaming-Sounds nicht auf die gleiche Weise verpackt werden wie andere Sounds verpackt sind und Sie die volle Kontrolle darüber haben, wann sie geladen, abgespielt und mittels Code entladen werden.

Um einen Sound zu einer Audiogruppe hinzuzufügen, müssen Sie den Sound Editor für diesen Sound öffnen und dann die Gruppe aus der dortigen Option auswählen, oder Sie können alternativ Assets zu einer Audiogruppe hinzufügen über das Menü der rechten Maustaste im Asset-Browser hinzufügen. Sie können ein oder mehrere Sound-Assets auswählen (oder einen ganzen Asset-Gruppenordner) auswählen und dann über das RMB-Menü die Sounds zu einer beliebigen Gruppe hinzufügen. Sie können auch auf die Schaltfläche " Ressource hinzufügen " am unteren Rand des Audiogruppen-Editors klicken, um auf diese Weise Assets hinzuzufügen.

Sobald Sie einen Sound zu einer benutzerdefinierten Audiogruppe hinzugefügt haben, können Sie in diesem Fenster selektiv auswählen, auf welche Plattform dieser Sound exportiert werden soll. Es kann sein, dass Sie standardmäßig alle Ihre Sounds in der höchsten Qualität haben, aber für HTML5 (zum Beispiel) (z. In diesem Fall fügen Sie die Dateien mit geringerer Qualität zu GameMaker Studio 2 hinzu und weisen sie dann einer neuen Audiogruppe zu. Sie würden dann diese Audiogruppe aus dem Dropdown-Menü auswählen und Sie würden dann diese Audiogruppe aus dem Dropdown-Menü auswählen und sie so einstellen, dass sie nur nach HTML5 exportiert, und den HTML5-Export aus den Exportoptionen für Audiogruppen höherer Qualität entfernen. Es ist wichtig zu beachten, dass Sie die Exportoptionen für die "Standard"-Audiogruppe nicht ändern können und dass sie immer in alle verfügbaren Plattformen exportiert wird, wenn Sie eine neue Gruppe erstellen. immer auf alle verfügbaren Plattformen exportiert wird, wenn Sie ein endgültiges Spielpaket erstellen.

Sobald Sie Audiogruppen definiert und ihnen Sounds zugewiesen haben, können Sie diese in der Liste links sehen, wenn Sie die Gruppe auswählen. Es wird immer eine "Standard"-Audiogruppe verfügbar sein, und alle Sounds, die sich in dieser Gruppe befinden, werden immer Alle Sounds dieser Gruppe sind immer im Spielpaket für alle Plattformen enthalten und werden beim Start in den Speicher geladen (es sei denn, sie sind in den Eigenschaften des Sound-Editors als "gestreamt" gekennzeichnet), aber wenn Sie eine benutzerdefinierte Audiogruppe erstellen, werden die zu dieser Gruppe hinzugefügten Dateien erst dann in den Speicher geladen, wenn Sie die Gruppe nicht in den Speicher geladen, bis Sie die Funktion audio_group_load(). Auf diese Weise können Sie kontrollieren Sie können auf diese Weise jederzeit kontrollieren, welche Audiodateien im Speicher vorhanden sind. Beachten Sie auch, dass Sie für alle Audiogruppenfunktionen den Wert für die Audiogruppen-ID angeben müssen. Dies ist einfach der Name, den Sie der Audiogruppe gegeben haben.

Audiogruppen sind auch mit den Einstellungen der Konfigurationen verknüpft. Wenn Sie eine Konfiguration auswählen, können Sie dann das Fenster "Audiogruppen" öffnen und die Exportziele auf der rechten Seite für diese Konfiguration auswählen, und Wenn Sie dann die Konfigurationen ändern, werden diese Ausgabeziele geändert. Beachten Sie, dass Sie Sound-Ressourcen nicht pro Konfiguration auf verschiedene Gruppen setzen können, sondern nur das Exportziel für die jeweilige Gruppe.
