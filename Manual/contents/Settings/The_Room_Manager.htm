Der Raum-Manager

Wenn Sie den Asset-Browser verwenden, können Sie Ihre Räume erstellen und sie zwischen den verschiedenen Ordnergruppen verschieben, um das Projekt so zu organisieren, wie es Ihnen am besten passt. Spiele in GameMaker Studio 2 haben jedoch in der Regel Räume, die einer bestimmten Reihenfolge folgen, und müssen zumindest einen bestimmten Startraum haben (nachdem Nachdem das Spiel in diesem Raum gestartet ist, können Sie mit Code oder Aktionen in jeden beliebigen Raum des Spiels wechseln, wenn Sie dies wünschen), und genau hier kommt der Raum-Manager ins Spiel.

Sie können den Raum-Manager öffnen, indem Sie oben im Asset-Browser auf die Schaltfläche "Extras" klicken und ihn aus dem sich öffnenden Menü auswählen, oder Sie können klicken Sie einfach auf das Symbol links neben einem Raum im Asset-Browser:

Sobald der Raum-Manager geöffnet ist, sehen Sie zwei verschiedene Registerkarten, eine für die Raumreihenfolge und eine für die Raumvererbung. Die Raumreihenfolge definiert die Reihenfolge, in der auf Räume zugegriffen wird, wenn Sie das Spiel buchstäblich durchqueren, indem Sie die Funktion room_goto_next() auf Der oberste Raum in der Liste ist immer durch das "Home"-Symbol gekennzeichnet, das Ihnen anzeigt, dass dieser Raum der Startraum für das Spiel ist. Startraum für das Spiel ist. Dieses Symbol wird auch im Asset-Browser angezeigt, so dass Sie deutlich sehen können, um welchen Raum es sich handelt, unabhängig davon, wie Sie Ihre Assets gefiltert oder eingerichtet haben.

Hinweis: Die IDs Ihrer Raum-Assets werden nicht in der Reihenfolge sein, wie sie in dieser Liste definiert sind, und daher sollten Sie nicht direkt einen Zahlenwert liefern, um den Raum zur Laufzeit zu ändern. Verwenden Sie immer die Raumkonstante für die Kühlstelle, auf die Sie verweisen möchten (die einfach ihr Name ist), oder rufen Sie sie über eine Funktion ab.

Sie können die Reihenfolge der Räume ändern, indem Sie einfach auf klicken und den Raum, den Sie nach oben oder unten in der Liste verschieben möchten, ziehen, und Sie können und/oder / verwenden, um mehrere Räume auszuwählen und sie dann durch Klicken und Ziehen insgesamt zu verschieben:

Das Register Raumvererbung dient zum Einrichten von "untergeordneten" Räumen. Ein untergeordneter Raum ist ein Raum, der von einem einem "Eltern"-Raum kopiert und so eingestellt wurde, dass er alle seine Eigenschaften erbt. Wenn Sie an diesem Raum nichts ändern, ist dieser Raum tatsächlich eine Kopie des ursprünglichen Raums, aber Sie können dann in den Raumeditor gehen und die Vererbung für bestimmte Dinge wie Ebenen oder Instanzen ein- oder ausschalten und dem Raum neue Ebenen, Instanzen und Eigenschaften hinzufügen.

Wie bei der Registerkarte "Room Order" können Sie auf klicken und ziehen, um die Räume neu anzuordnen. Beachten Sie jedoch, dass durch die Auswahl eines untergeordneten Raums und das Verschieben zu einem anderen übergeordneten Raum die Raumvererbung mit dem ursprünglichen übergeordneten Raum aufgehoben wird und er stattdessen die Eigenschaften des neuen Raums erbt. Beachten Sie auch, dass Sie einen untergeordneten Raum in einen "vollständigen" Raum ohne Vererbung umwandeln können, indem Sie ihn nach links aus der Liste ziehen (der Listenrand wird hervorgehoben), wodurch er einfach zu einem eigenständigen Raum wird.

Räume können mehrere Kinder haben, und jedes Kind kann auch ein Elternteil sein und z. B. eigene Kinder haben:

Hier haben wir einen Elternraum, "rm_Level_1_Day", der zwei Kinder hat "rm_Level_1_Day_AfterBattle" und "rm_Level_1_Night". Außerdem haben wir "rm_Level_1_Night" einen weiteren Kindraum namens " rm_Level_1_Night_AfterBattle ". Auf diese Weise können Sie die Ressourcen und das Ebenendesign eines übergeordneten Raums wiederverwenden und dann nur das ändern, was erforderlich ist, um es in dem/den untergeordneten Raum/Räumen etwas anders zu machen.

Schließlich verfügt der Raum-Manager über einige zusätzliche Werkzeuge, die Ihnen bei der Navigation und Organisation Ihrer Räume helfen. Oben befindet sich eine Suchleiste, mit der Sie einen Suchbegriff eingeben und dann schnell zu den Räumen gehen können, die den Suchbegriff in ihrem Namen haben Namen. Räume, in denen der Suchbegriff vorkommt, werden hervorgehoben, und Sie können mit den seitlichen Auf-/Ab-Tasten sofort zu ihnen navigieren:

Am unteren Rand des Raummanagers befinden sich außerdem einige hilfreiche Schaltflächen, mit denen Sie Räume in der Raumordnungsliste nach oben oder unten verschieben können:

Neben diesen Schaltflächen haben Sie auch eine Schaltfläche Gruppe erstellen, mit der Sie einen Gruppenordner erstellen können. Diese Ordner können verwendet werden, um die Räume in Ihrem Spiel besser zu organisieren (die Raumreihenfolge wird weiterhin beachtet), und Sie können Räume anklicken und auf den Ordner ziehen, um sie hinzuzufügen. Sie können auch mehrere Räume auswählen, indem Sie bei gedrückter Maustaste auf und/oder / klicken, und dann auf die Schaltfläche "Create Group" (Gruppe erstellen) klicken. Beachten Sie, dass sich dadurch die Reihenfolge der Räume in Ihrem Spiel ändert, und beachten Sie auch, dass, wenn Sie den Gruppenordner später löschen, alle Räume innerhalb des Ordners ebenfalls gelöscht werden (standardmäßig warnt Sie GameMaker vor diesem Vorgang). Standardmäßig warnt Sie GameMaker davor).

Beachten Sie auch, dass Sie mit der rechten Maustaste auf einen beliebigen Raum im Raum-Manager klicken können, um das folgende Menü aufzurufen:

Dieses Menü bietet Ihnen die folgenden Optionen:

  • Untergeordneten Raum erstellen: Dadurch wird ein neuer Raum als untergeordneter Raum des angeklickten Raums erstellt, wobei der neue Raum die Eigenschaften des übergeordneten Raums erbt.
  • Bearbeiten: Dadurch wird der Raumeditor für den Raum geöffnet.
  • Tags bearbeiten: Dadurch wird der Tag-Editor geöffnet, damit Sie dem Raum Tags hinzufügen können.
  • Umbenennen: Damit können Sie das Raum-Asset umbenennen (Sie können auch einen langsamen Doppelklick auf den Raum machen, um ihn umzubenennen).
  • Duplizieren: Damit wird der ausgewählte Raum dupliziert und der neue Raum unter dem ausgewählten Raum platziert.
  • Favorit: Damit wird der gewählte Raum zum Bereich Favoriten des Asset-Browsers hinzugefügt.
  • Löschen: Wenn Sie darauf klicken, wird der Raum (oder die Räume) gelöscht, der/die ausgewählt wurde(n). Bei Verwendung für einen Gruppenordner werden alle Räume innerhalb des Ordners zusammen mit dem Ordner gelöscht. Bei Verwendung für einen übergeordneten Raum wird nur der übergeordnete Raum gelöscht, und alle untergeordneten Räume bleiben erhalten, obwohl sie jetzt unabhängig sind und keine untergeordneten Räume mehr sind.
  • Kinder aufklappen: Wenn ein Gruppenordner ausgewählt ist, wird mit dieser Option der Inhalt des Ordners erweitert.
  • Alle erweitern: Mit dieser Option werden alle Ordnergruppen erweitert.
  • Alle einklappen: Mit dieser Option werden alle Ordnergruppen eingeklappt.
  • Anordnen: Mit dem Untermenü "Anordnen" können Sie Räume im Raum-Manager nach oben oder unten verschieben, ähnlich wie mit den Schaltflächen am unteren Rand des Managerfensters.
  • Im Explorer öffnen: Damit wird ein OS-Dateiexplorer für das Asset im Spielprojekt geöffnet.
  • Asset-Farbe: Hier können Sie eine Farbe für den Raum (oder die Räume) wählen, um sie im Raum-Manager zu identifizieren. Beachten Sie, dass diese Farbe auch im Asset-Browser verwendet wird und alle zuvor eingestellten Farben überschreibt.
