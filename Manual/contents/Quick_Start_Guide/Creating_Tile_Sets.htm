Erstellen von Kachelsätzen

Kachel-Sets werden aus Sprites erstellt, aber sie werden auch als separater Asset-Typ klassifiziert, da die Art und Weise, wie GameMaker Studio 2 sie behandelt, etwas anders ist als bei Sprites und Objekten. Ein Kachelset ist also eine Sammlung von Kacheln, die zusammen in einem Sprite, und eine Kachel ist einfach eine grafische Ressource, die zu einem GameMaker Studio 2-Raum als Teil eines Kachelsatz. Diese Kachel-Maps werden dann auf den Bildschirm gezeichnet Diese Kachelkarten werden dann mit sehr wenig CPU-/GPU-Verarbeitung auf den Bildschirm gezeichnet, was sie ideal für die Gestaltung beliebiger statischer Elemente in Levels macht.

Um die Verarbeitung von Kacheln so schnell zu machen, haben sie bestimmte Einschränkungen, wobei die grundlegendste darin besteht, dass sie nur rechteckig oder quadratisch sein können (wenn Sie etwas anderes als das wollen, müssen Sie ein reguläres Sprite erstellen erstellen und den Asset-Layer im Raum-Editor verwenden, aber das besprechen wir im späteren Abschnitt über Räume). Zu den weiteren Einschränkungen gehört die Tatsache, dass sie nur um 90° gedreht, gespiegelt oder umgedreht werden können, und sie können Sie können nur mit Frames animiert werden, die eine Potenz von 2 sind (d. h. 2, 4, 8, 16 usw...).

Wenn Sie also Ihre Kachelsätze erstellen, muss die Kachel in der linken oberen Ecke immer auf diese Weise leer sein. Beachten Sie auch, dass die obige Abbildung zwar ein komplettes Wandkachelset mit 47 Kacheln zeigt, Sie aber möglicherweise nicht so viele benötigen, da Sie die Fliesen später beim Platzieren im Raumeditor drehen und spiegeln können.

Nachdem das Sprite erstellt wurde, können wir fortfahren und das Kachelset selbst definieren, was bedeutet, dass Sie zuerst eine neue Kachelset-Ressource erstellen müssen, indem Sie mit der rechten Maustaste auf den Ordner Tile Set asset klicken und " Erstellen" wählen. Dadurch wird der Kachelsatz-Editor geöffnet, in dem Sie dann das Sprite hinzufügen und seine Eigenschaften festlegen können:

Wir haben die Größe des Kachelsatzes auf 64 px gesetzt und den Rest der Optionen auf ihren Standardwerten belassen, da wir keine Offsets oder ähnliches benötigen (die Offset-Werte sind für jedes Kachelsatz-Sprite nützlich, das Leerzeichen um jede der Kacheln hat). Sie sollten das Kachelset-Asset mit einem geeigneten Präfix wie " ts_" benennen, damit Sie es später leicht identifizieren können, und beachten Sie, dass die gleichen Einschränkungen wie für Sprites gelten, d. h. der Name darf nicht mit einer Zahl beginnen und nur mit einer Zahl beginnen und darf nur den Unterstrich "_" und alphanumerische Zeichen enthalten.

Sie können nun den Kachelsatz-Editor schließen, da Sie den Kachelsatz "so wie er ist" zum Erstellen von Kachelplan-Ebenen im Raum-Editor verwenden können.
