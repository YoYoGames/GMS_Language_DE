Sequenzen

Das letzte Asset, das wir in dieser Anleitung besprechen, bevor wir uns damit beschäftigen, wie alles im Raum-Editor zusammenkommt , ist das Asset "Sequenzen". Eine Sequenz ist im Wesentlichen eine Sammlung von anderen Assets, die an einem Ort gesammelt an einem Ort gesammelt und dann gemeinsam animiert werden. Die endgültige Animation (Sequenz) kann dann zu Räumen in Ihrem Spiel hinzugefügt werden, ähnlich wie Instanzen.

Sequenzen werden über den Editor Sequenz-Editor der geöffnet wird, wenn Sie ein neues Sequenz-Asset im Asset-Browser erstellen:

Um eine grundlegende Sequenz zu erstellen, fügen Sie entweder einen Sound, ein Sprite oder eine Instanz in den Sequenz-Canvas ein und positionieren es dann an der Stelle, an der Sie es am Anfang der Sequenz haben möchten. an der Stelle, an der es am Anfang der Sequenz stehen soll. Dadurch wird ein Asset-Spur im Track Panel und fügt außerdem fügt eine Keyframe in den Dope-Blatt um die Dauer anzuzeigen, für die das Asset sichtbar sein wird.

Die Zeit für die Sequenz wird in Frames angegeben (was dasselbe ist wie Schritte für Instanzen, d. h.: ein Frame ist dasselbe wie ein Spielschritt), und Sie können sehen, was bei einem bestimmten Frame passiert, indem Sie die Sequenz bewegen Abspielkopf in dem Abspielbogen. Die Abspielkopfposition wird auch für das Hinzufügen von Keyframes die die Frames sind, an denen ein bestimmter Zustand beginnen oder enden soll. Parameterschlüssel werden auf Parameterspuren für das angegebene Asset.

Sie würden also das Asset positionieren und seine Skalierung und Farbe sowie andere Eigenschaften bei Frame 0 der Sequenz einstellen. Sie würden dann die Parametertasten für das Asset aufzeichnen und den Abspielkopf zu dem Frame bewegen, an dem Sie den nächsten Asset-Zustand einstellen möchten (Position, Farbe, etc....). In diesem Frame würden Sie wieder die verschiedenen Parameter ändern und die Tasten für sie aufzeichnen und diesen Vorgang dann für die Länge der Sequenz beliebig wiederholen. Wenn Sie die Wiedergabe der Sequenz drücken, wird das Asset animiert, wobei die Die Sequenz interpoliert zwischen den verschiedenen Parameterzuständen, so dass das Endergebnis etwa so aussieht:

Wir werden hier nicht näher darauf eingehen, wie Sequenzen funktionieren, aber es genügt zu sagen, dass sie ein unglaublich mächtiges Werkzeug sind, um alles zu tun von Zwischensequenzen, über Menüs bis hin zu Charakterzuständen und vielem mehr.
