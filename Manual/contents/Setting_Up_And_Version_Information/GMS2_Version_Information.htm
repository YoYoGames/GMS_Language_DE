GMS2-Einstellung und Versionsinformationen

Es gibt verschiedene Versionen der Lizenzen, die Sie für GameMaker Studio 2 erwerben können. Die Wahl der Lizenzen hängt vor allem von der Zielgruppe ab, die Sie ansprechen wollen, und davon, ob Sie nur Spaß an der Entwicklung von Spielen haben oder diese auch verkaufen und vertreiben wollen. auch verkaufen und vertreiben möchten. Im Folgenden finden Sie Details zu den einzelnen Lizenztypen:

Versuch

Wenn Sie sich für einen YoYo-Account registrieren, erhalten Sie eine kostenlose 30-Tage-Testlizenz, mit der Sie GameMaker Studio 2 starten können, und können dann je nach Bedarf verschiedene Upgrades kaufen. Die Trial-Lizenz gibt Ihnen einen großen Teil der GameMaker Studio 2 Erfahrung und ist für den Betrieb auf einem macOS oder Windows PC, hat aber eine Begrenzung von 30 Tage Nutzung. Nach Ablauf der ersten 30 Tage müssen Sie auf eine der kostenpflichtigen Lizenzen upgraden, um das Produkt weiter nutzen zu können. Beachten Sie, dass die Testlizenz die folgenden Einschränkungen hat:

  • Sie erlaubt den Zugang zu GMS2 nur für 30 Tage ab dem Datum, an dem Sie Ihren YoYo Account über Ihre Bestätigungs-E-Mail aktiviert haben.
  • Sie können keine endgültige ausführbare Datei für eine beliebige Zielplattform kompilieren.
  • Sie haben keinen Zugriff, um die leistungsstärkeren "YYC"-Zielexporte zu testen.
  • Sie können die IDE-Benachrichtigungen über das baldige Auslaufen Ihrer Testlizenz nicht deaktivieren (aber andere Benachrichtigungen werden deaktiviert, wenn Sie die Benachrichtigungen in den Voreinstellungen ausschalten).
  • Sie können kein Marketplace-Publisher-Konto erstellen (aber Ihr YoYo-Konto kann auf den Marketplace zugreifen und bezahlte/freie Assets kaufen).


Ersteller

Mit der Creator-Version von GameMaker Studio 2 entfällt die 30-Tage-Beschränkung der Trial-Lizenz (siehe oben) und Sie können Ihre Spiele für die gewählte Plattform exportieren, die entweder macOS oder Windows. Welche Zielplattform verfügbar ist, hängt von der Version der IDE ab, die Sie gekauft haben ( Creator für macOS oder Creator für Windows) und verwendet die VM (ein Runner + interpretierter Code) zum Ausführen und Erstellen Ihrer Spiele. Informationen zur Einrichtung finden Sie im Helpdesk-Abschnitt über Erforderliche SDKs oder den Abschnitt über Desktop-Ziele. Beachten Sie, dass für die Creator-Lizenz zwei kleinere Einschränkungen gelten:

  • Sie können den Standard-Splash-Bildschirm für das Laden Ihres Spiels nicht ändern.
  • Sie haben keinen Zugang, um Ihre Projekte mit den leistungsfähigeren "YYC" (YoYo Compiler) Target-Exporten zu bauen.


Desktop

Mit der Desktop-Version von GameMaker Studio 2 werden alle Einschränkungen der Testlizenz (siehe oben) aufgehoben und Sie können Projekte zu den folgenden Desktop-Zielen kompilieren:

  • Windows
  • macOS
  • Ubuntu(Linux)

Alle Targets kommen mit zwei Kompilieroptionen, eine ist die VM , die einen Runner und interpretierten Code verwendet, und die andere ist der YYC (YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen finden Sie im Abschnitt über Erforderliche SDKs oder den Abschnitt über Desktop-Ziele.


Web

Mit der Web-Version von GameMaker Studio 2 sind alle Einschränkungen der Testlizenz (siehe oben) aufgehoben und Sie können Projekte zum HTML5-Ziel kompilieren. Spiele werden mit JavaScript kompiliert. Weitere Informationen erhalten Sie vom YoYo Games Helpdesk hier: Einrichten für Web (HTML5).


Mobil

Mit der Mobile-Version von GameMaker Studio 2 werden alle Einschränkungen der Trial-Lizenz (siehe oben) aufgehoben und Sie können Projekte zu den folgenden Zielen kompilieren:

  • Android
  • iOS
  • tvOS

Alle Targets kommen mit zwei Kompilieroptionen, eine ist die VM , die einen Runner und interpretierten Code verwendet, und die andere ist der YYC (YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt über Erforderliche SDKs oder im Helpdesk-Abschnitt über Mobile Targets.


UWP

Mit der UWP-Version von GameMaker Studio 2 sind alle Einschränkungen der Testlizenz (siehe oben) aufgehoben und Sie können Projekte als UWP-kompatible ausführbare Dateien kompilieren. Diese ausführbaren Dateien können für die Ausführung auf einer von zwei UWP-Plattformen konfiguriert werden:

  • Windows 10
  • XBox One

Das UWP-Ziel kommt mit zwei Kompilieroptionen, eine ist die VM , die einen Runner und interpretierten Code verwendet, und die andere ist der YYC (YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt über Erforderliche SDKs oder im Helpdesk-Abschnitt über UWP-Ziele.


Konsolen

Mit der Konsolen-Version von GameMaker Studio 2 werden alle Einschränkungen der Testlizenz (siehe oben) aufgehoben und Sie können Projekte für eine oder mehrere der folgenden Zielplattformen kompilieren:

  • XBox One
  • PlayStation 4
  • Nintendo Switch

Jedes der Konsolenziele kommt mit zwei Kompilieroptionen, eine ist die VM , die einen Runner und interpretierten Code verwendet, und die andere ist der YYC (YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt über Erforderliche SDKs.

WICHTIG! Aufgrund der Restriktionen der verschiedenen NDAs, die mit den Konsolenexporten verbunden sind, gibt es keine Informationen über das Einrichten oder Programmieren für diese Targets innerhalb des GameMaker Studio 2 Handbuchs. Sobald Sie ein registrierter Entwickler für die gewählte Plattform sind, kontaktieren Sie bitte den Helpdesk über Ihren YoYo-Konto und fordern Sie Zugang zur entsprechenden Dokumentation an.


Bildung

GameMaker Studio 2 kann auch in einer Schul- oder Hochschulumgebung eingesetzt werden. Für weitere Informationen hierzu kontaktieren Sie bitte YoYo Games über das YoYo Games Bildungsportal.
