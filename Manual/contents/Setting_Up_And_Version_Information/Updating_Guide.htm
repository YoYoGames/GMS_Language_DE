Aktualisierungsanleitung

Die GameMaker Studio 2 IDE sucht in regelmäßigen Abständen nach Updates und lädt diese dann automatisch herunter. Einige dieser Updates sind optional und Sie können sie nicht installieren, wenn Sie dies nicht möchten oder glauben, dass einige der Änderungen ein Projekt beschädigen könnten, das Sie gerade veröffentlichen wollen. Es gibt jedoch einige Updates, die wichtige Korrekturen oder Verbesserungen enthalten, die nicht übersprungen werden können, und GameMaker Studio 2 erzwingt in diesen Fällen ein Update.

GameMaker Studio 2 ist mehr als nur eine IDE und verfügt über eine Reihe von verschiedenen Laufzeit-Tools, die es im Hintergrund verwendet, um Ihre Spiele für die verschiedenen verfügbaren Zielplattformen zu erstellen. Diese Werkzeuge sind hauptsächlich die verschiedenen Runner und Compiler, die jede Plattform benötigt, und Sie können sogar mehrere davon gleichzeitig haben. Das bedeutet, dass Sie sich aussuchen können, welche Compiler- und Runner-Version Sie verwenden möchten, wenn Sie Probleme mit der aktuellen Version haben oder auf eine ganz bestimmte Zielversion oder Plattform abzielen. Die Tools werden automatisch aktualisiert, wenn eine neue Version verfügbar ist, aber Sie können dann in den Runtime Feeds Preferences auswählen, welche Sie tatsächlich verwenden möchten. Weitere Informationen zu Runtime-Updates finden Sie unter Voreinstellungen - Runtime Feeds.
