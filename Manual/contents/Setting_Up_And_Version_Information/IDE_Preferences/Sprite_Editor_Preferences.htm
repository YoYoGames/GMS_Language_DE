Sprite-Editor-Voreinstellungen

Mit den Sprite-Editor-Voreinstellungen legen Sie fest, wie der Sprite-Editor aussieht und funktioniert. Es gibt die folgenden Optionen:

  • Standard-Sprite-Ursprung: Mit dieser Option können Sie die Standard-Ursprungsposition für alle neuen Sprite-Ressourcen festlegen, die Sie erstellen. Der Standardwert ist "Oben links".
  • Standard-Sprite-Kollisionsmaskenart: Mit dieser Option können Sie die Standardart der Kollisionsmaske festlegen, die für neue Sprites verwendet werden soll. Zur Auswahl stehen Rechteck, gedrehtes Rechteck, Ellipse, Diamant oder präzise / präzise pro Frame. Die Voreinstellung ist Rechteck.
  • Standard-Sprite-Animationsmodus: Mit dieser Option können Sie den Standard-Animationsmodus für Sprites im Sprite-Editor festlegen. Sie können entweder " Frames pro Sekunde" oder" Frames pro Spielframe" wählen , wobei die Standardeinstellung "Bilder pro Sekunde" ist .
  • Standard-Sprite-Animationsgeschwindigkeit: Mit dieser Option können Sie den Standardwert für die Animationsgeschwindigkeit festlegen, wenn Sie ein neues Sprite-Asset erstellen. Beachten Sie, dass dieser Wert mit dem Animationsmodus verknüpft ist und - zum Beispiel - 3 Bilder pro Sekunde ist nicht dasselbe wie 30 Frames pro Spielframe. Stellen Sie also sicher, dass Sie diesen Wert korrekt in Bezug auf den bevorzugten Modus einstellen. Der Standardwert ist 30.
  • Standard-Sprite-Breite: Mit dieser Option können Sie die Standardbreite für alle neu erstellten Sprites festlegen. Der Standardwert ist 64.
  • Standard-Sprite-Höhe: Mit dieser Option können Sie die Standardhöhe für alle neuen Sprites festlegen. Der Standardwert ist 64.
  • Größenänderung und Skalierung auf Sprite-Ursprung anwenden: Wenn diese Option aktiviert ist, wird der Ursprung des Sprites geändert, wenn das Sprite in der Größe verändert oder skaliert wird, wobei der neue Ursprung so verschoben/skaliert wird, dass er mit den neuen Werten für das Sprite übereinstimmt. Beachten Sie, dass dies nur für benutzerdefinierte Sprite-Ursprünge gilt, die nicht im Sprite-Editor gesperrt sind. Wenn der Ursprung gesperrt ist oder das Sprite eine der Standardpositionen verwendet - z. B. "Mitte" oder "Oben links" usw. - dann wird diese Option ausgegraut und kann nicht geändert werden. Standardmäßig ist sie aktiviert.

Die folgende Unterkategorie existiert auch für den Sprite-Editor:

Dialoge bestätigen

Die hier verfügbaren Optionen steuern die Meldungen, die angezeigt werden, wenn Sie eine Aktion bestätigen müssen. Sie können diese Einstellung ändern, um die Meldung immer anzuzeigen oder sie mit einer bestimmten Aktion (entweder Ja oder Nein) zu verwerfen:

  • Automatische Reaktion auf den Datei-Import: Wenn Sie ein neues Bild importieren und bereits eines vorhanden ist, werden Sie standardmäßig aufgefordert, mit dem Import fortzufahren und das vorhandene Bild zu überschreiben oder den Vorgang abzubrechen. Mit dieser Option können Sie diese Meldung unterdrücken und immer entweder den Import abbrechen, indem Sie "Nein" wählen, oder das aktuelle Sprite überschreiben, indem Sie "Ja" wählen. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf Fehler beim Laden von Bildern: Wenn Sie ein neues Bild importieren und der Import fehlschlägt, wird standardmäßig eine Warnmeldung angezeigt, die Sie darüber informiert, dass ein Fehler aufgetreten ist. Mit dieser Option können Sie diese Meldung unterdrücken, indem Sie "Okay" wählen. Der Standardwert ist "Meldung anzeigen".
