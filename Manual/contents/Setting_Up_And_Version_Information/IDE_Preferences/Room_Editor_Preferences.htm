Raumeditor-Voreinstellungen

Mit den Raumeditor-Voreinstellungen legen Sie fest, wie der Raumeditor aussehen und sich verhalten soll. Die allgemeinen Einstellungen sind wie folgt:

  • Startabstand der Kettenausblendung (px): Legt den Abstand vom übergeordneten Fenster fest, ab dem das Kettenglied ausgeblendet werden soll. Standard ist 10px.
  • Kettenausblendungsstoppabstand (px): Legt den Abstand vom übergeordneten Fenster fest, bei dem die Ausblendung des Kettenglieds enden soll. Der Standardwert ist 100px.
  • Chain Minimum Alpha: Legt den endgültigen Alphawert nach dem Ausblendungsabfallpunkt für das Kettenalpha fest. Der Standardwert ist 0,3.
  • Standardabstand der Ebenentiefe: Wenn Sie mehrere Ebenen im Raum-Editor erstellen, wird ihnen automatisch eine Tiefe zugewiesen, die auf ihrer Position innerhalb der Ebenenliste basiert. Diese Tiefe wird um den hier angegebenen Wert erhöht. Der Standard Wert ist 100, d. h. jede Ebene hat die Tiefe der vorherigen Ebene + 100.
  • Standard-Raumbreite: Legt die anfängliche Breite fest, die als Standard für einen neuen Raum ausgewählt wird. Standard ist 1024px.
  • Standard-Raumhöhe: Legt die Anfangshöhe fest, die als Standard für einen neuen Raum ausgewählt wird. Standard ist 768px.
  • Hinzufügen von Standard-Raumebenen: Wenn Sie eine neue Raumressource erstellen, generiert GameMaker Studio 2 ein paar "Basis"-Ebenen für Sie - eine Hintergrundebene und eine Instanzebene. Sie können dieses Verhalten hier ausschalten, so dass neue Räume keine vorgefertigten Ebenen haben. Dies ist standardmäßig eingeschaltet.
  • Raumansichten anzeigen: Wenn in einem Raum Ansichten aktiv sind, wird dies durch einen Umriss an der Stelle angezeigt, an der die Ansichtsgrenzen im Raum liegen würden, und die Sichtbarkeit dieses Bereichs kann über die entsprechende Schaltfläche im Editor selbst umgeschaltet werden. Sie können jedoch festlegen, dass die Ansichtsgrenzen nicht sichtbar sind, wenn Sie einen Raum öffnen, indem Sie diese Option deaktivieren (die Schaltfläche kann immer noch verwendet werden, um sie wieder anzuzeigen). Der Standardwert ist eingeschaltet, so dass die Ansichtsgrenzen beim Öffnen eines Raums sichtbar sind.
  • Allgemeine Farboptionen: Diese beziehen sich auf die verschiedenen Farben, die in der gesamten Raumeditor-Oberfläche verwendet werden, und wenn Sie auf eines der Farbfelder doppelklicken, können Sie die Farbe für diese Einstellung bearbeiten.
  • Master-Lautstärke-Einstellung: Die Master-Lautstärke-Einstellung für die Vorschauwiedergabe im Raum-Editor, ein Wert zwischen 0 und 1. Standard ist 1.

In den unten aufgeführten Unterkategorien stehen weitere Optionen zur Bearbeitung zur Verfügung:

Dialoge bestätigen

Mit den Voreinstellungen für Bestätigungsdialoge können Sie die Meldungen ändern, die im Raumeditor angezeigt werden, wenn Sie bestimmte Aktionen durchführen. Die Meldungen, die Sie ändern können, sind:

  • Automatische Reaktion auf das Aktualisieren von Ebenentiefen: Wenn Sie Ebenentiefen ändern, wird eine Meldung angezeigt, um Sie zu warnen, dass dadurch auch alle anderen Ebenentiefen automatisch aktualisiert werden. Mit dieser Option können Sie festlegen, dass die Meldung einblenden, die Ebenen aktualisieren lassen, als ob Sie auf "Ok" geklickt hätten, oder nichts aktualisieren lassen, als ob Sie auf "Nein" geklickt hätten.
  • Automatische Reaktion auf Warnung bei gesperrter Ebene: Wenn Sie versuchen, eine Ebene im Raum zu löschen und diese zufällig gesperrt ist, wird Ihnen standardmäßig eine Meldung angezeigt. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können die Ebene einfach nicht löschen.
  • Automatische Reaktion auf Warnung bei gesperrtem Element: Wenn Sie versuchen, ein Element zu löschen, das zufällig gesperrt ist, wird standardmäßig eine Meldung angezeigt. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können das Objekt einfach nicht löschen.
  • Automatische Reaktion auf Warnung bei gesperrter Ebene des Elementbesitzers: Wenn Sie versuchen, ein Element auf einer Ebene zu löschen und die Ebene zufällig gesperrt ist (auch wenn das Element nicht gesperrt ist), wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht möglich ist. nicht durchgeführt werden kann. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können das Element einfach nicht löschen.
  • Neue Asset-Ebene erstellen, wenn Sequenzen/Sprites auf ungeeignete Ebenen gezogen werden: Wenn Sie versuchen, ein Sprite (oder eine Sequenz) zu einer Ebene hinzuzufügen, die keine Vermögensebene ist, wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht nicht möglich ist, und Sie werden gefragt, ob Sie eine neue Elementebene für das zu ziehende Element erstellen möchten. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und es wird automatisch eine neue Ebene für Sprites (und Sequenzen) erstellt.
  • Automatische Reaktion auf das Löschen von Elementen in übergeordneten Räumen: Hier können Sie wählen, was passiert, wenn Sie ein Element löschen, das Teil eines übergeordneten Raums (und somit in allen untergeordneten Räumen vorhanden) ist. Standardmäßig wird eine Warnmeldung angezeigt und Sie können Sie können aber auch "Delete All" wählen, um die Meldung zu unterdrücken und das Element aus dem übergeordneten Raum und allen untergeordneten Räumen zu löschen, oder Sie können "Keep All" wählen, um die Meldung zu unterdrücken und das Löschen abzubrechen.
  • Automatische Reaktion auf das Löschen von Ebenen im übergeordneten Raum: Hier können Sie wählen, was passiert, wenn Sie eine Ebene löschen, die Teil eines übergeordneten Raums ist (und somit in allen untergeordneten Räumen vorhanden ist). Standardmäßig wird eine Warnmeldung angezeigt und Sie können entscheiden, was zu tun ist. Sie können aber auch "Delete All" wählen, um die Meldung zu unterdrücken und die Ebene aus dem übergeordneten Raum und allen untergeordneten Räumen zu löschen, oder Sie können "Keep All" wählen, um die Meldung zu unterdrücken und das Löschen abzubrechen.
  • Automatische Reaktion auf Fehler beim Ziehen eines ungültigen Assets: Wenn Sie versuchen, ein Asset zu einer Ebene hinzuzufügen, die nicht für diesen Asset-Typ vorgesehen ist (z. B. Hinzufügen eines Pfads zu einer Instanz-Ebene), wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht möglich ist. die Sie darüber informiert, dass dies nicht möglich ist. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können das Kühlgut einfach nicht hinzufügen. Beachten Sie, dass diese Option nicht für Instanzen, Sprites oder Sequenzen gilt.
  • Neue Instanzebene erstellen, wenn Objekte auf ungeeignete Ebenen gezogen werden: Wenn Sie versuchen, eine Objektinstanz zu einem Layer hinzuzufügen, der kein Instanzlayer ist, wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht möglich ist und Sie werden gefragt, ob Sie eine neue Instanzebene für das gezogene Objekt erstellen möchten. Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und es wird automatisch ein neuer Layer für die Instanz erstellt.
  • Automatische Reaktion auf Fehler beim Ziehen einer ungültigen Ebenenauswahl: Hier können Sie wählen, was passiert, wenn Sie eine Ressource in den Raum-Editor ziehen, wenn Sie mehrere Ebenen im Ebenen-Editor ausgewählt haben (oder gar keine Ebene ausgewählt haben überhaupt nicht ausgewählt haben). Standardmäßig wird eine Warnmeldung angezeigt, aber Sie können diese Option auf "OK" setzen, um die Meldung zu unterdrücken und fortzufahren, ohne eine Ressource hinzuzufügen.
  • Automatische Reaktion auf Fehler beim Ziehen einer gesperrten oder unsichtbaren Ebene: Hier können Sie wählen, was passiert, wenn Sie eine Ressource auf eine Ebene ziehen, die gesperrt oder als unsichtbar gekennzeichnet ist. Standardmäßig wird eine Warnmeldung angezeigt, aber Sie Sie können diese Option jedoch auf "OK" setzen, um die Meldung zu unterdrücken und fortzufahren, ohne eine Ressource hinzuzufügen.

Rastereinstellungen

Die Gittereinstellungen werden verwendet, um die Funktionsweise der Gitterüberlagerung zu ändern, wobei die folgenden Optionen zur Verfügung stehen:

  • Horizontaler Abstand des Gitters (px): Hier wird der horizontale Abstand für Gürtelzellen des Raumgitters in Pixeln festgelegt. Der Standardwert ist 32px.
  • Vertikaler Abstand des Gitters (px): Hier wird der vertikale Abstand für Gitternetzzellen des Raumgitters in Pixeln festgelegt. Der Standardwert ist 32px.
  • Raster anzeigen: Mit dieser Option schalten Sie das Raumraster ein oder aus, wenn Sie einen Raum zum ersten Mal öffnen. Es ist standardmäßig eingeschaltet, aber auch wenn es ausgeschaltet ist, können Sie es über die entsprechende Schaltfläche im Raumeditor selbst wieder aktivieren.
  • Am Raster einrasten: Mit dieser Option können Sie einstellen, ob ein Raum standardmäßig Assets am Raumraster einrasten lässt oder nicht, wenn ein Raum erstmalig geöffnet wird. Sie ist standardmäßig eingeschaltet, aber auch wenn sie ausgeschaltet ist, können Sie sie wieder aktivieren, indem Sie die entsprechende Schaltfläche im Raumeditor selbst wieder aktivieren.
  • Rasterlinienfarbe: Mit dieser Option können Sie die Farbe (einschließlich der Alphakomponente) der Raumrasterlinien einstellen. Der Standardwert ist ( ARGB) #30FFFFFF.



Intelligente Führungen

Dieser Abschnitt beschreibt die Smart Guides-Voreinstellungen für den Raum-Editor. Die verfügbaren Optionen sind:

  • Smart-Guides aktivieren: Hiermit aktivieren/deaktivieren Sie die Verwendung von Smart Guides im Raum-Editor. Die Voreinstellung ist aktiviert.

  • Lineale einschalten: Mit dieser Option werden die Seitenlineale im Raum-Editor aktiviert/deaktiviert. Die Voreinstellung ist aktiviert.

  • Lineale an Kanten einrasten: Wenn diese Option aktiviert ist, werden Hilfslinien, die von den Linealen in den Raum-Editor gezogen werden, an allen im Raum platzierten Elementen gefangen. Standardmäßig ist diese Option aktiviert.

  • Fangabstand: Diese Einstellung teilt dem Raum-Editor mit, in welchem Abstand sich ein Guide von einem Asset befinden muss, bevor er an der Seite des Assets "einrastet". Der Standardwert ist 6px.
