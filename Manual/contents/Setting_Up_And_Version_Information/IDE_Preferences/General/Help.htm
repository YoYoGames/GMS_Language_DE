Hilfe-Voreinstellungen

Die Hilfedatei ist so konzipiert, dass sie wie eine Webseite funktioniert, und obwohl die Dateien lokal gespeichert sind, muss ein externer Browser verwendet werden, um sie anzuzeigen. Die folgenden Einstellungen in Bezug auf die Hilfedatei können hier geändert werden:

  • Manuelle Sprache: Hier wird die Sprache eingestellt, die die IDE verwenden soll. Die Vorgabe ist "Englisch".
  • Hilfe Manueller Port: Hier wird die Portnummer festgelegt, die der manuelle Micro-Webserver verwenden soll. Der Standardwert ist 51291.
