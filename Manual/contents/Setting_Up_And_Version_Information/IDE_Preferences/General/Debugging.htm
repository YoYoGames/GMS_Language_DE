Debugging-Voreinstellungen

Die Einstellungen hier beziehen sich auf den GameMaker Studio 2 Debugger und Sie haben die folgenden Optionen:

  • Automatische Reaktion auf das Schließen des Debuggers: Mit dieser Option können Sie die automatische Reaktion beim Schließen des Debuggers einstellen. Bei der Einstellung "Meldung anzeigen" wird Ihnen der Dialog angezeigt, wenn Sie den Debugger schließen wollen, während Wenn Sie die Option auf "Ja" oder "Nein" setzen, wird dieser Dialog nicht angezeigt und die Aufgabe wird wie angegeben ausgeführt.
  • Echtzeit-Debugging aktivieren: Wenn Sie diese Option aktivieren, können Sie Ihre Spielprojekte in Echtzeit debuggen, während Sie sie in der IDE ausführen. Dies ist standardmäßig aktiviert.
  • Debugger-Anschluss: Hier wird die Portnummer festgelegt, die das Debug-Modul verwenden soll. Der Standardwert ist 6509.
  • Ereignissymbole in der Debugger-Ressourcenansicht anzeigen: Legen Sie fest, ob in der Debugger-Ressourcenansicht Symbole angezeigt werden sollen. Dies ist standardmäßig aktiviert.
