Voreinstellungen für das Ausgabefenster

Die Ausgabefenster Voreinstellungen steuern die verschiedenen Farb- und Formatierungsoptionen für die verschiedenen Ausgabefenster in der IDE. Jedes der verschiedenen Fenster hat den gleichen Optionen, die unten aufgelistet sind:

  • Farbe: Die Standardfarbe für den Text, der in das Fenster geschrieben wird.
  • Schriftart-Name: Der Name der zu verwendenden Schriftart (die verfügbaren Schriftarten hängen von den auf Ihrem System installierten ab).
  • Stil: Der zu verwendende Schriftstil, wie z. B. fett oder kursiv (beachten Sie, dass nicht alle Fenster die gleichen Stile zulassen und es auch von der verwendeten Schriftart abhängt).
  • Größe: Die Größe, auf die die Schrift für das angegebene Ausgabefenster eingestellt werden soll.
  • Anti-Aliasing: Hier können Sie den Anti-Aliasing-Modus für das Rendern der Schriftart des Ausgabefensters einstellen. Sie können die IDE entscheiden lassen, indem Sie Standard verwenden wählen, oder Sie können es ein- oder Aus erzwingen, indem Sie eine der beiden Optionen auswählen.
