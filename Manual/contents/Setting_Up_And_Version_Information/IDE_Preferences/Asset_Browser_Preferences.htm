Asset-Browser-Voreinstellungen

Mit den Asset-Browser-Voreinstellungen können Sie bestimmte Eigenschaften für die Funktionsweise des Asset-Browsers und seiner Elemente festlegen. seine Elemente funktionieren. Die wichtigsten Optionen sind:

  • Schnellzugriff anzeigen: Wenn diese Option aktiviert ist, wird der Bereich "Schnellzugriff" am oberen Rand des Asset-Browsers angezeigt. Diese Option ist standardmäßig aktiviert.
  • Standardordner in neuem Projekt erstellen: Wenn diese Option aktiviert ist, werden alle neuen Projekte mit den bereits erstellten Standard-Asset-Ordnern gestartet, und wenn sie deaktiviert ist, wird nur ein Raumordner mit dem Standardraum erstellt, der für alle Projekte erstellt. Diese Option ist standardmäßig aktiviert.
  • Maximale Anzahl von Assets, die gleichzeitig ohne Warnung geöffnet werden können: Sie können mehrere Assets zum gleichzeitigen Öffnen auswählen, indem Sie sie mit / + auswählen und dann mit der rechten Maustaste auf klicken und Alle öffnen wählen. Wenn Sie mehr als die in dieser Option angegebene Anzahl ausgewählt haben, wird eine Warnung angezeigt, da das gleichzeitige Öffnen von zu vielen Assets zu Problemen mit der Leistung auf leistungsschwächeren Rechnern führen kann. auf leistungsschwächeren Rechnern führen kann. Der Standardwert ist 10.
  • Maximal kürzlich bearbeitete Elemente: Diese Einstellung regelt die Anzahl der Assets, die im Abschnitt "Zuletzt bearbeitet" des Asset-Browsers aufgeführt werden. Der Standardwert ist 10.
  • Farbpalettengröße: Hier können Sie die Anzahl der Farben einstellen, die Ihnen in der Asset-Browser-Palette für Asset-Gruppen zur Verfügung stehen. Der Standardwert ist 16, mit einem Mindestwert von 16 und einem Höchstwert von 100.
  • Asset-Farben im Schnellzugriff anwenden: Wenn diese Option aktiviert ist, werden die für einen bestimmten Asset-Typ eingestellten Farben verwendet, um das Asset im Schnellzugriffsbereich des Browsers einzufärben. Wenn diese Option nicht markiert ist, wird die Asset-Farbe nicht verwendet. Diese Option ist standardmäßig aktiviert.
  • Gruppenfarben haben Vorrang vor untergeordneten Farben: Wenn dieses Kontrollkästchen aktiviert ist, wird die Farbe der obersten Gruppenebene zur Einfärbung aller Kühlstellen innerhalb der Gruppe verwendet, wobei die für die einzelne Kühlstelle eingestellte Farbe ignoriert wird. Dies ist standardmäßig deaktiviert.
  • Asset-Dialogstil erstellen: Wenn Sie im Asset-Browser auf die Schaltfläche "Asset hinzufügen" klicken, wird ein Fenster angezeigt, in dem Sie die verschiedenen Assets sehen, die zu Ihrem Projekt hinzugefügt werden können. Dieses Fenster kann Dieses Fenster kann in einer Listen- oder einer Rasteransicht dargestellt werden, und diese Einstellung legt fest, was standardmäßig angezeigt wird. Der Standardwert ist Raster.
  • Standard-Sortierreihenfolge: Hier können Sie die Standardsortierreihenfolge für Assets im Browser festlegen, entweder alphabetisch oder von A bis Z oder von Z bis A. Standardmäßig ist von A bis Z eingestellt.
  • Standardmäßig Ordner zuerst sortieren: Wenn diese Option aktiviert ist, sortiert der Asset-Browser zuerst die Ordner und dann alle "losen" Assets, wenn sie deaktiviert ist, werden alle Assets und Ordner zusammen sortiert. Dies ist standardmäßig aktiviert.
  • Sprite-Doppelklick öffnet Bild-Editor: Standardmäßig öffnet ein Doppelklick auf auf ein beliebiges Asset dieses Asset zur Bearbeitung, aber oft, wenn Sie mit Sprites, nach der anfänglichen Einrichtung, wenn Sie das neue Asset erstellen, öffnen Sie es, um das Bild und nicht die Sprite-Eigenschaften zu bearbeiten. Wenn Sie diese Option auswählen, wird der Bildeditor für das Sprite-Asset anstelle des Sprite-Assets selbst geöffnet. Sie können trotzdem den den Sprite-Editor öffnen, indem Sie mit der rechten Maustaste auf klicken und " Edit" (Bearbeiten) wählen. Diese Option ist standardmäßig deaktiviert.
  • EinObjekt-Doppelklick öffnet alle Objekt-Events: Standardmäßig öffnet ein Doppelklick auf auf eine beliebige Kühlstelle diese Kühlstelle zur Bearbeitung, aber oft, wenn Sie mit Objekten arbeiten, nach der anfänglichen Einrichtung, wenn Sie die neue Kühlstelle erstellen, öffnen Sie sie nur, um die Ereignisse und nicht die Objekteigenschaften zu bearbeiten. Wenn Sie diese Option wählen, werden die verschiedenen Objektereignisse in einem neuen Arbeitsbereich geöffnet, wobei jedes Ereignis eine eigene Registerkarte innerhalb des neuen Arbeitsbereichs. Sie können den Objekteditor immer noch öffnen, indem Sie mit der rechten Maustaste auf klicken und Bearbeiten wählen. Diese Option ist standardmäßig ausgeschaltet.
  • Raumreihenfolge aufheben entfernt die leere Gruppe: Wenn aktiviert, wird beim Aufheben der Gruppierung aller Räume, die zu einem Gruppenordner gehören, auch der Gruppenordner entfernt. Wenn sie deaktiviert ist, hebt die gleiche Aktion die Gruppierung der Räume auf, lässt aber den leeren Gruppenordner Ordner im Asset-Browser. Dies ist standardmäßig aktiviert.
  • Standard-Zoom (%): Hier wird der Standard-Zoom-Prozentsatz für den Asset-Browser festgelegt, wenn Sie ein neues Projekt starten. Der Standardwert ist 100 %.
  • Minimaler Zoom (%): Hier wird der minimal zulässige Zoomwert für den Asset-Browser eingestellt. Der Standardwert ist 50 %.
  • Maximaler Zoom (%): Hier wird der maximal zulässige Zoomwert für den Asset-Browser eingestellt. Der Standardwert ist 250 %.
  • Baumlinien anzeigen: Hier können Sie die Baumlinien, die seitlich von Assets im Asset-Browser angezeigt werden, aktivieren/deaktivieren. Dies ist standardmäßig aktiviert, um die Baumlinien anzuzeigen.
  • Baum bei Filterwechsel aufklappen: Mit dieser Option wird der Asset-Browser gezwungen, alle kollabierten Baumknoten zu erweitern, wenn Sie die Filteroptionen ändern. Wenn Sie die Option deaktivieren, wird dies verhindert. Standardmäßig ist sie aktiviert.

Dialoge bestätigen

Hier können Sie das Verhalten der Bestätigungsdialoge beim Arbeiten mit einem Asset auswählen aus dem Asset-Browser. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf das Löschen von Assets: Wenn Sie ein Asset aus dem Asset-Browser löschen, erhalten Sie normalerweise eine Warnung, die Sie fragt, ob Sie sicher sind, dass Sie fortfahren wollen. Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Löschen von Raumordnungsgruppen: Wenn Sie einen Gruppenordner mit Räumen aus dem Asset-Browser löschen, erhalten Sie normalerweise eine Warnung, die Sie fragt, ob Sie sicher sind und den Vorgang fortsetzen möchten (die alle Räume in der Gruppe auflistet die gelöscht werden sollen). Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt, als ob Sie auf die Schaltfläche OK geklickt hätten, um fortzufahren und alle Räume in der Gruppe sowie den Gruppenordner selbst zu löschen. Der Standardwert ist "Show Meldung".
  • Automatische Reaktion auf das Öffnen mehrerer Assets: Wie oben in den allgemeinen Einstellungen für den Asset-Browser erwähnt, können Sie beim gleichzeitigen Öffnen mehrerer Assets im Asset-Browser eine Warnung erhalten, wenn die Anzahl der Assets die angegebene Anzahl überschreitet. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf die DnD™-Konvertierungin GameMaker Language: Wenn Sie die Option zum Konvertieren eines DnD™-Knotensin GML wählen, werden Sie aufgefordert, die Änderung zu bestätigen, da diese nicht rückgängig gemacht werden kann. Wenn Sie diese Option auf "OK" setzen wird die Warnung unterdrückt, als ob Sie zum Fortfahren auf die Schaltfläche "OK" geklickt hätten. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Setzen eines Raum-Elternteils über das Ziehen eines Knotens: Im Bereich "Vererbung" des Raum-Manager-Fensters können Sie einen Raum auf einen anderen Raum ziehen, um ihn als Kind des Raums zu setzen, auf den er gezogen wird. Wenn Sie dies erhalten Sie eine Warnung, die Sie fragt, ob Sie sicher sind und den Vorgang fortsetzen möchten. Wenn Sie diese Option auf "OK" setzen, wird die Meldung unterdrückt, als ob Sie zum Fortfahren auf die Schaltfläche "OK" geklickt hätten. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf Fehler bei doppeltem Gruppennamen: Wenn Sie einen Gruppenordner zum Organisieren Ihrer Assets erstellen und diesen gleich benennen wie einen anderen Ordner im Browser, wird eine Meldung angezeigt, in der Sie gefragt werden, ob Sie fortfahren möchten und zwei Ordner mit demselben Namen haben möchten. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Setzen eines Raums als Elternteil über das Ziehen eines Knotens: Wenn Sie im Raum-Manager einen Raum über einen anderen Raum ziehen und ihn dann loslassen, wird eine Meldung angezeigt, in der Sie gefragt werden, ob Sie den Raum zu einem "Kind" des Raums machen möchten, auf den Raumes machen wollen, auf den gezogen wurde. Wenn Sie diese Option auf "OK" setzen, wird die Warnung unterdrückt, als ob Sie auf die Schaltfläche "OK" geklickt hätten, um fortzufahren. Der Standardwert ist "Meldung anzeigen".
