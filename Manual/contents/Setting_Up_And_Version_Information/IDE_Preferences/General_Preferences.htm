Allgemeine Voreinstellungen

Die allgemeinen Voreinstellungen werden verwendet, um das allgemeine Verhalten der GameMaker Studio 2 IDE zu definieren. Die allgemeinsten sind auf der Hauptseite aufgeführt, während viele andere in mehrere Unterkategorien aufgeteilt sind, die weiter unten auf dieser Seite erläutert werden.

  • IDE-Sprache: Mit dieser Option können Sie die Sprache für die IDE aus der Liste der mitgelieferten Optionen auswählen. Die Voreinstellung ist Englisch.
  • IDE-Skin: Hier können Sie aus den verschiedenen Skins wählen, die die IDE verwenden kann. Standardmäßig wird GameMaker Studio 2 mit zwei Skins geliefert, einem hellen und einem dunklen, und die Ersteinrichtung verwendet den dunklen Skin.
  • Maximale Framerate des GameMaker Studio 2-Editors: Mit dieser Option können Sie die maximale Framerate einstellen, mit der die GameMaker Studio 2 IDE aktualisiert wird, so dass Sie die Framerate an die Aktualisierungsgeschwindigkeit Ihres Monitors anpassen können. Der Standard Wert ist 60.
  • Standard-Audiogerät: Hier können Sie das Standard-Audiogerät auswählen, das die GameMaker Studio 2 IDE verwenden soll. Sie können die Liste aktualisieren, indem Sie auf die Schaltfläche Aktualisieren klicken.
  • Automatische Reaktion auf Beenden-Bestätigung: Mit dieser Option können Sie einstellen, ob beim Beenden von GameMaker Studio 2 eine Bestätigungsmeldung angezeigt werden soll. Standardmäßig ist dies auf "Meldung anzeigen" eingestellt, kann aber auch so eingestellt werden, dass die Meldung übersprungen wird, als ob Sie auf OK geklickt hätten. Meldung zu überspringen, als ob Sie auf "OK" geklickt hätten.
  • Automatische Reaktion auf den Dialog "Systemschriftart geändert": Mit dieser Option können Sie einstellen, ob eine Bestätigungsmeldung angezeigt werden soll, wenn eine Systemschriftart hinzugefügt, entfernt oder bearbeitet wurde. Standardmäßig ist dies auf "Meldung anzeigen" eingestellt, kann aber so eingestellt werden, dass die Meldung übersprungen wird, als ob Sie auf "OK" geklickt hätten.
  • Benachrichtigungen aktivieren: Wenn diese Option aktiviert ist, zeigt die IDE Benachrichtigungen über neue Runtimes usw. an. Dies ist standardmäßig eingeschaltet.
  • Zeitüberschreitung für Benachrichtigung (ms): Hier wird die Zeitspanne (in Millisekunden) festgelegt, für die eine Benachrichtigung angezeigt wird. Der Standardwert ist 6000.
  • Beim Beenden abmelden: Wenn diese Option aktiviert ist, meldet die IDE die Benutzersitzung automatisch ab, wenn GameMaker Studio 2 geschlossen wird. Dies ist in erster Linie für Benutzer aus dem Bildungsbereich gedacht, da - wenn diese Option aktiviert ist - dies bedeutet, dass der Benutzer jedes Mal, wenn das Programm Wenn diese Option aktiviert ist, bedeutet dies, dass beim Beenden des Programms der Benutzer abgemeldet wird und somit der Sitzungsplatz für andere Benutzer frei wird. Diese Option ist standardmäßig deaktiviert.
  • DPI-Override aktivieren: Mit dieser Option können Sie die Standard-DPI-Einstellung mit Ihrer eigenen überschreiben. Sie müssen zuerst die Option aktivieren und dann entweder den Wert " Raw DPi Override" (der Standardwert ist 96) oder den Prozentsatz des nativen DPi (Standardwert 100 %) einstellen. Nach der Änderung werden Sie aufgefordert, die IDE neu zu starten, was möglicherweise erfordert, dass Sie die Arbeit am aktuell geöffneten Projekt speichern. Beachten Sie, dass der DPi einen minimalen Prozentsatz von 50 % und einen maximalen von 500 % hat, und wenn Sie Wenn Sie mit der Maus über das Informationssymbol fahren, können Sie die empfohlene Maximaleinstellung für den verwendeten Bildschirm sehen:

    Beachten Sie, dass, wenn Sie die empfohlene DPi-Einstellung für den Monitor überschreiten, eine eine Benachrichtigung angezeigt, die es Ihnen ermöglicht, den DPi zurückzusetzen.
  • Willkommensbildschirm beim Starten anzeigen: Diese Option ist standardmäßig aktiviert und weist GameMaker Studio 2 lediglich an, beim Starten des Programms einen kurzen Begrüßungsbildschirm anzuzeigen. Wenn Sie diese Option deaktivieren, wird der Begrüßungsbildschirm nicht angezeigt.
  • Schließen von leeren Tabs per Tastenkombination zulassen: Sie können alle offenen Tabs mit der Tastenkombination / + W schließen, aber wenn Sie diese Option deaktivieren, funktioniert diese Tastenkombination nicht mehr. Diese Option ist standardmäßig aktiviert.
  • Öffnen neuer Ressourcen zur Bearbeitung beim Ziehen von Dateien in die IDE: Bestimmte Ressourcen (wie Sprites oder Sounds) können auf den Hauptarbeitsbereich in der GameMaker Studio 2 IDE gezogen werden und werden als neue Ressource im Asset Browser hinzugefügt. Standardmäßig wird in diesem Fall auch der entsprechende Ressourcen-Editor zur Bearbeitung geöffnet, aber wenn Sie diese Option deaktivieren, werden die Editoren nicht geöffnet und Sie müssen sie manuell öffnen.
  • Neue Ressourcen bei der Erstellung zur Bearbeitung öffnen: Wenn dies aktiviert ist, wird jedes Mal, wenn Sie eine neue Ressource im Asset-Browser erstellen, automatisch der entsprechende Ressourcen-Editor geöffnet. Diese Option ist standardmäßig aktiviert, und wenn Sie sie deaktivieren bedeutet, dass Sie alle neu erstellten Ressourcen manuell öffnen müssen.
  • Geänderte Dateien automatisch neu laden: Wenn dies aktiviert ist, scannt und prüft die IDE automatisch, ob Dateien außerhalb von GameMaker Studio 2 geändert wurden, und lädt sie gegebenenfalls neu.
  • Deaktivieren Sie IDE-Übergangsanimationen: Die GameMaker Studio 2 IDE hat eine Reihe von animierten Übergängen für Dinge wie das Öffnen von Fenstern etc... allerdings sind sie nicht jedermanns Geschmack und Sie können einen Leistungsschub erleben, wenn sie auf älteren oder leistungsschwächeren Rechnern, wenn sie deaktiviert sind. Sie können sie hier deaktivieren, und sie sind standardmäßig eingeschaltet.
  • Opt-in für Feedback-Umfragen: Wenn diese Option aktiviert ist, teilen Sie GameMaker Studio 2 mit, dass Sie an Feedback-Umfragen teilnehmen möchten, d. h., dass gelegentlich während der Arbeit eine Meldung zu dem Bereich angezeigt wird, an dem Sie gerade arbeiten angezeigt und um eine Bewertung und ein Feedback gebeten wird. Diese Option ist standardmäßig aktiviert.

Die oben genannten Optionen sind die Hauptoptionen, die in den allgemeinen Einstellungen verfügbar sind, aber es gibt eine Reihe von zusätzlichen Optionen, die Ihnen in den unten aufgeführten Unterkategorien zur Verfügung stehen:

  • Hintergrund
  • Kompilieren
  • Dialoge bestätigen
  • Fehlersuche
  • Dateiwächter
  • GoTo
  • Hilfe
  • Eingang
  • Ausgabefenster
  • Pfade
  • Energieeinsparung
  • Aktuelle Fenster
  • Ressourcen
  • Arbeitsbereiche
