Bildeditor-Voreinstellungen

Mit den Voreinstellungen für den Bildeditor können Sie festlegen, wie das Bildeditor-Fenster aussieht und sich anfühlt. Im Folgenden sind die allgemeinen Optionen aufgeführt:

  • Sprite-Steuerungen standardmäßig überlagern erweitern: Wenn dies aktiviert ist, werden die Sprite-Steuerelemente und Frame-Informationen jedes Mal im Bild-Editor eingeblendet, wenn Sie ein Sprite zur Bearbeitung öffnen (dieses kann mit der Schaltfläche "Ausklappen" wieder geschlossen werden wieder geschlossen werden, öffnet sich aber später beim erneuten Bearbeiten des Sprites wieder). Wenn diese Option nicht aktiviert ist, werden die Sprite-Steuerelemente und Rahmeninformationen beim Öffnen eines Sprites ausgeblendet. eines Sprites ausgeblendet. Standardmäßig ist diese Option nicht aktiviert.
  • Standard-Schleifenmodus: Die Schaltfläche "Schleife" im Bildeditor kann so eingestellt werden, dass sie in zwei verschiedenen Modi funktioniert: entweder einmalige Wiedergabe, bei der die Animation beim letzten Bild stoppt, Schleife vom Anfang, bei der die Animation zum ersten Bild zurückgeht, wenn sie beendet ist, oder Ping-Pong, bei der die Animation bis zum Ende und dann wieder zurück zum Anfang läuft. oder Ping Pong, wobei die Animation von oben nach unten durch die Bilder bis zum Ende und dann wieder zurück zum Anfang läuft. Der Standardwert ist hier "Schleife".
  • Dünne Auswahlkontur zeichnen: Wenn diese Option aktiviert ist, hat der Umriss der Auswahl eine absolute Breite von 1px, unabhängig von der Skalierung des Bildes, während bei deaktivierter Option der Umriss der Auswahl eine relative Breite von 1px hat und skaliert mit dem Bild. Standardmäßig ist diese Option aktiviert.
  • Primärfarbe der Auswahlkontur: Die Schnittkontur ist eine gestrichelte Linie, die zwischen zwei Farben wechselt. Mit dieser Option können Sie die erste dieser Farben festlegen (einschließlich Alpha), und der Standardwert ist ARGB #FFFFFD0.
  • Sekundärfarbe der Auswahlkontur: Die Schnittkontur ist eine gestrichelte Linie, die zwischen zwei Farben wechselt. Mit dieser Option können Sie die zweite dieser Farben festlegen (einschließlich Alpha). Der Standardwert ist ARGB #0000D0.
  • Editor-Hintergrundfarbe: Während der Bearbeitung eines Sprite-Rahmens kann das Bild ein- oder ausgeskaliert werden oder nicht in das gesamte Zeichenfenster passen; in diesem Fall wird eine Hintergrundfarbe angezeigt, die Sie hier einstellen können. Der Standardwert ist ARGB #282828FF.

Die Bildeditor-Voreinstellungen haben außerdem die folgenden Unterkategorien:

Hintergrund

Hier können Sie die Optionen für den Hintergrund einstellen, vor dem Sie Ihre Sprite-Bilder bearbeiten. Standardmäßig ist dies ein Gitter aus grauen Quadraten, aber Sie können die folgenden Einstellungen verwenden, um es zu ändern:

  • Leinwand-Stil: Hier stellen Sie den "Stil" für den Hintergrund ein, der entweder " Block-Muster" oder " Einfarbig". Bei "Einfarbig" wird der Hintergrund einfach auf die von Ihnen gewählte Farbe eingestellt (unter Verwendung der Einstellung "Farbe 1", wie unten erläutert), während "Blockmuster" die Standardeinstellung ist (die auch mit den übrigen unten gezeigten Dialogoptionen bearbeitet werden kann). Optionen bearbeitet werden kann). Die Voreinstellung ist " Blockmuster".
  • Größe des Leinwandblocks: Wenn Sie das Blockmuster für den Hintergrund gewählt haben, können Sie hier die Größe des einzelnen Blocks einstellen. Der Standardwert ist 16px, wobei ein Minimalwert von 2px und ein Maximalwert von 256px zulässig sind.
  • Leinwand-Primärfarbe: Wenn Sie Single Colour für den Hintergrund gewählt haben, wird diese Farbe verwendet. Sie können auf das Farbfeld klicken, um den Farbeditor zu öffnen und die Farbe zu ändern. Wenn Sie die Option Blockmuster-Option wird hier die erste Farbe für abwechselnde Blöcke festgelegt. Die Standardfarbe ist ( ARGB) #FFC3C3C3.
  • Segeltuch-Sekundärfarbe (Blockmuster-Stil): Wenn Sie Blockmuster für den Hintergrund gewählt haben, dann wird diese Farbe im Wechsel mit Farbe 1 (oben) verwendet. Sie können auf das Farbfeld klicken, um den Farbeditor zu öffnen und die Farbe zu ändern. Die Standardfarbe ist (ARGB) #FF808080.

Dialoge bestätigen

Hier können Sie das Verhalten der Bestätigungsdialoge auswählen, wenn Sie verschiedene Aktionen am aktuellen Sprite durchführen. Sie sind:

  • Automatische Reaktion auf das Löschen von Rahmen: Wenn Sie einen Rahmen löschen, wird standardmäßig eine Meldung angezeigt, in der Sie gefragt werden, ob Sie sicher sind, dass Sie dies tun möchten. Wenn Sie diese Option auf "OK" setzen, können Sie diese Meldung entfernen und das Löschen von Rahmen ohne dass weitere Meldungen angezeigt werden. Die Voreinstellung ist "Meldung anzeigen".
  • Automatische Reaktion auf das Ersetzen von Bildern beim Importieren von Strips: Wenn Sie im Dropdown-Menü "Bild" die Option "Streifen importieren" auswählen, erhalten Sie eine Meldung, die Sie darauf hinweist, dass Sie bereits in der Sprite-Ressource vorhandene Bilder überschreiben werden, mit der Option, fortzufahren oder nicht. Sie können hier einstellen, ob diese Meldung angezeigt werden soll, oder die IDE einfach die Option "Ja" oder "Nein" wählen lässt. Der Standardwert ist "Meldung anzeigen".
  • Automatische Reaktion auf das Festschreiben von Änderungen beim Werkzeugwechsel: Wenn Sie nach der Bearbeitung eines Rahmens das Zeichenwerkzeug wechseln, wird standardmäßig eine Meldung angezeigt, in der Sie gefragt werden, ob Sie die vorgenommenen Änderungen übernehmen möchten. Durch Setzen dieser Option auf "OK" können Sie können Sie diese Meldung entfernen und zulassen, dass die Rahmenbearbeitung ohne weitere Meldungen beim Werkzeugwechsel übernommen wird. Wenn Sie "Nein" wählen, wird die Rahmenbearbeitung bei einem Werkzeugwechsel stillschweigend abgebrochen. Die Voreinstellung ist "Zeige Meldung".
  • Automatische Reaktion auf Warnung bei gesperrter Ebene: Wenn Sie versuchen, eine gesperrte Ebene zu bearbeiten, erhalten Sie standardmäßig eine Meldung, die Sie darüber informiert, dass Sie dies nicht tun können. Wenn Sie diese Option auf "OK" setzen, wird keine Meldung angezeigt und die Bearbeitung scheitert stillschweigend.
  • Automatische Reaktion auf Warnung bei Gruppenauswahl: Wenn Sie im Ebenen-Editor mehrere Ebenen markieren und dann versuchen, zu zeichnen oder zu bearbeiten, wird standardmäßig eine Meldung angezeigt, die Sie darüber informiert, dass dies nicht möglich ist. Sie können "OK" wählen wählen, um diese Meldung zu entfernen und die Bilder erst dann gemeinsam zu bearbeiten, wenn nur noch eine einzige Ebene ausgewählt ist. Die Voreinstellung ist "Meldung anzeigen".



Rastereinstellungen

Raster In den Voreinstellungen wird festgelegt, wie das Bildraster dargestellt wird, wobei folgende Optionen zur Verfügung stehen:

  • Horizontaler Abstand des Rasters (px): Dieser Wert legt den Abstand für die horizontalen Linien des Bildrasters fest. Der Standardwert ist 1px.
  • Vertikaler Abstand des Gitters (px): Dieser Wert legt den Abstand für die vertikalen Linien des Bildrasters fest. Der Standardwert ist 1px.
  • Raster anzeigen: Wenn dies aktiviert ist, wird der Bildeditor mit eingeschaltetem Bildraster geöffnet. Die Voreinstellung ist aus.
  • Am Raster ausrichten: Wenn diese Option aktiviert ist, rastet der Bildeditor bestimmte Werkzeuge automatisch am Bildraster ein. Die Voreinstellung ist aus.
  • Rasterlinienfarbe: Hier können Sie die Farbe (einschließlich Alphawert) für das Bildraster einstellen. Voreinstellung ist ( ARGB) #FF000000.

Werkzeuge

Hier können Sie die verschiedenen Optionen einstellen einstellen, wie einige der Zeichenwerkzeuge arbeiten und angezeigt werden.

Wenn Sie das Bogen-Werkzeug verwenden, können Sie hier können Sie die verschiedenen Farben einstellen, die zum Zeichnen der Knoteneigenschaften verwendet werden:

  • Ankerpunktfarbe: Legen Sie die Farbe für die verschiedenen Ankerpunkte auf dem Spline/Bogen fest. Standard ist ( ARGB) #FFFF0000.
  • Kontrollpunktfarbe: Legen Sie die Farbe für jeden der Kontrollpunkt-"Griffe" an einem Ankerpunkt fest. Standard ist (ARGB) #FFFF0000.
  • Farbe der Tangentenlinie: Legen Sie die Farbe für die Linien fest, die die Kontrollpunkte mit den Ankern verbinden. Standard ist (ARGB) #FF00FFFF.
  • Farbe des ausgewählten Punkts: Legen Sie die Farbe fest, die für alle ausgewählten Ankerpunkte auf dem Spline/Bogen verwendet werden soll. Standard ist (ARGB) #FF00FF00.
  • Punktgröße: Legen Sie die Größe der verschiedenen Anker- und Steuerpunkte fest, die verwendet werden sollen. Standard ist 10px.
