Sprachpräferenzen

Mit den Spracheinstellungen legen Sie fest, wie einige der Sprachfunktionen für die Programmierung in der IDE von GameMaker Studio 2 angezeigt werden sollen. Die Optionen für GML sind wie folgt:

  • Argumenttypen in der Statusleiste anzeigen: Diese Option bezieht sich darauf, wenn Sie Ihre eigenen benutzerdefinierten Skripte verwenden und die JSDoc-Notation verwendet haben, um die verschiedenen Argumente usw. zu identifizieren (siehe hier für weitere Informationen). Wenn diese Option aktiviert ist und Sie die Argumenttypen im Skript angegeben haben, werden sie in der Statusleiste am unteren Rand des Editors angezeigt, eingeschlossen in {}. In der Voreinstellung ist dies deaktiviert.
  • Argumentbeschreibungen in der Statusleiste anzeigen: Wenn dies aktiviert ist, werden alle Beschreibungen, die Sie Ihren benutzerdefinierten Skripten mit der JSDoc-Notation gegeben haben, in der Statusleiste am unteren Rand des Editorfensters angezeigt. Diese Option ist standardmäßig ausgeschaltet.
  • US-Schreibweise für Systemfunktionen anzeigen: Standardmäßig verwendet GameMaker Studio 2 die englische Schreibweise für Funktionen, z. B. draw_text_colour(), aber das ist vielleicht nicht genau das, was Sie gewohnt sind, und so können Sie Daher können Sie mit dieser Option auf die US-Schreibweise umschalten, so dass die Funktionen z. B. draw_text_color() heißen. Diese Option ist standardmäßig ausgeschaltet.
  • Maximale Anzahl der anzuzeigenden Syntaxfehler: Dieser Wert legt die maximale Anzahl von Syntaxfehlern fest, die zu einem beliebigen Zeitpunkt angezeigt werden. Wenn Sie feststellen, dass sich die IDE verzögert, können Sie versuchen, diesen Wert zu verringern. Dieser Wert ist standardmäßig auf 50 eingestellt.
  • Aktivieren Sie Variablenreferenzwarnungen: Wenn diese Option aktiviert ist, werden potenzielle Probleme mit Variablen angezeigt, z. B. wenn eine Variable irgendwo definiert wurde, aber im Code nicht tatsächlich verwendet wird. Wenn deaktiviert, werden diese Warnungen nicht mehr angezeigt. Standardmäßig ist dies aktiviert.
