Zeitleisten-Voreinstellungen

In den Zeitleisten-Voreinstellungen können Sie bestimmte Eigenschaften für die Arbeit mit dem Zeitleisten-Editor festlegen. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf DnD™-Konvertierungin GameMaker-Sprache: Wenn Sie die Menüoption der rechten Maustaste in einem Timeline-Moment verwenden verwenden, um Drag-and-Drop™-Aktionenin Script (Code) zu konvertieren, erhalten Sie eine Meldung, die Sie fragt, ob Sie fortfahren möchten, da es sich um eine einseitige Konvertierung handelt. Wenn Sie diese Option auf "Okay" ändern, wird die Meldung unterdrückt und die Konvertierung wird fortgesetzt als ob Sie im Dialogfeld auf "Okay" geklickt hätten. Die Standardeinstellung ist "Meldung anzeigen".
  • Eine automatische Reaktion auf eine permanente Aktionswarnung: Bestimmte Aktionen innerhalb einer Zeitleiste können nicht rückgängig gemacht werden, und wenn dies geschieht, erhalten Sie eine Warnmeldung, die Sie fragt, ob Sie fortfahren möchten. Ändern Sie diese Option auf "Okay" wird die Meldung unterdrückt und die Aktion wird fortgesetzt, als ob Sie im Dialog auf "Okay" geklickt hätten. Die Standardeinstellung ist "Meldung anzeigen".
  • Öffnen Sie neue Momente zum Bearbeiten bei der Erstellung: Normalerweise öffnet GameMaker Studio 2 beim Hinzufügen eines neuen Moments zu einer Zeitleiste automatisch den Editor, damit Sie Ihren Code hinzufügen oder ihn per Drag & Drop einfügen können. Wenn Sie diese Option jedoch deaktivieren, wird der Moment hinzugefügt, ohne den Editor zu öffnen (Sie können ihn öffnen, indem Sie auf auf den Moment doppelklicken). Standardmäßig ist diese Option aktiviert.
