Schriftarten-Editor-Voreinstellungen

Mit den Voreinstellungen für den Font-Editor können Sie festlegen, wie sich der Font-Editor verhält. Die verschiedenen verfügbaren Optionen sind unten aufgelistet, zusammen mit einer Beschreibung, was jede einzelne bewirkt.

  • Aus GameMaker Studio 1-Projekten importierte Schriftarten aktualisieren: Wenn diese Option aktiviert ist, versucht GameMaker Studio 2, Schriften zu aktualisieren und ihre Textur neu zu generieren. Dies geschieht nur, wenn:
    • Diese Voreinstellung ist aktiviert
    • Die Schriftart ist auf dem Rechner verfügbar
    • GameMaker Studio 2 kann eine passende Schrift in der Schriftfamilie finden (dies ist die wahrscheinlichste Ursache für Inkonsistenzen beim Import)
    Beachten Sie, dass dies auch geschieht, wenn die GameMaker Studio 1-Schrift keine zwischengespeicherte Schrifttextur hat. Wenn eine Legacy-Schrift nicht aktualisiert werden kann, verwendet GameMaker Studio 2 standardmäßig die zwischengespeicherte Schrifttextur zu verwenden, und der Schriftarten-Editor erlaubt es Ihnen nicht mehr, die Schriftart zu bearbeiten, und zeigt eine Option zum Sichern und Aktualisieren auf eine andere an:

    Wenn Sie darauf klicken, können Sie die Schriftart bearbeiten und entweder die fehlende Schriftart hinzufügen oder eine andere einstellen, und die gesicherte Schrift wird im Font-Asset-Ordner gespeichert, wie <font name>.yy.gms1 und <font name>.png.gms1. Um diese Dateien wiederherzustellen, löschen Sie manuell die Dateien <font name>.yy und ersetzen sie durch diese, indem Sie die Erweiterung .gms1 aus den Dateinamen entfernen. Wichtig! Alle Assets speichern ihre Namen in der Datei .yy, daher müssen Sie diese bearbeiten bearbeiten und den richtigen Namen der Ressource auch in das Feld <name> eintragen (falls anders).
  • Automatische Auswahl für fehlgeschlagene Inhaltsregenerierung Dialog: Wenn Sie eine Font-Asset-Textur regenerieren wollen, wird eine Warnung angezeigt. Wenn Sie diese Option auf "OK" setzen, wird die Schrift neu generiert, ohne dass die Warnung angezeigt wird. angezeigt wird.
  • Automatische Auswahl bei fehlgeschlagener Font-Regenerierung Dialog: Wenn die Regeneration einer Font-Asset-Textur fehlschlägt, wird eine Warnung angezeigt. Wenn Sie diese Option auf "OK" setzen, wird die Anzeige der Meldung übersprungen.
  • Automatische Auswahl für das Upgrade von importierten Legacy-Schriften: Wenn Sie ein Legacy-Projekt importieren, erhalten Sie möglicherweise eine Warnung über Legacy-Schriften und das Angebot, diese zu aktualisieren. Wenn Sie diese Option auf "OK" setzen, wird das Update automatisch genehmigt, ohne diese Meldung angezeigt wird.
