Sequenzen Voreinstellungen

Die Voreinstellungen für Sequenzen werden verwendet, um bestimmte Eigenschaften für die Funktionsweise des Sequenzeditors und seiner Elemente zu definieren. seine Elemente funktionieren. Die wichtigsten Optionen sind:

  • Canvas-Rahmen bei neu erstellten Sequenzen anzeigen: Wenn diese Option aktiviert ist, wird der Canvas-Rahmen-Smartguide im Canvas des Sequenzeditors für alle neuen Sequenzen angezeigt. Diese Option kann über die Canvas-Eigenschaften im Editor selbst aktiviert/deaktiviert werden. auch im Editor selbst aktiviert/deaktiviert werden. Der Standardwert ist aktiviert.
  • Canvas-Rahmenbreite bei neu erstellten Sequenzen: Hier können Sie die Standardrahmenbreite für alle neuen Sequenzen festlegen. Der Standardwert ist 1366px.
  • Rahmenhöhe derLeinwand bei neu erstellten Sequenzen: Hier können Sie die Standard-Rahmenhöhe für alle neuen Sequenzen festlegen. Der Standardwert ist 768px.
  • Pause beim Klicken auf die Leinwand: Wenn diese Option aktiviert ist, wird beim Klicken auf eine beliebige Stelle im Sequenzeditor-Canvas die Sequenzwiedergabe angehalten. Dies ist standardmäßig aktiviert.
  • Überlappende Auswahlverzögerung in Millisekunden: Hiermit wird gesteuert, wie lange Sie warten müssen, bevor die Element-Toolbox für überlappende Elemente auf der Arbeitsfläche des Sequenzeditors eingeblendet wird. Der Standardwert ist 1000 ms.
  • Einfügemarke für Translationspfadpunkte: Wenn Sie im Sequenzeditor-Canvas eine Kühlstelle hinzugefügt und dann eine Positionsparameterspur erstellt haben, werden gepunktete Linien gezeichnet, um den Pfad zu zeigen, den die Kühlstelle beim bewegt. Wenn Sie die Maus in der Nähe dieser Linie platzieren, können Sie einen neuen Parameter-Track-Punkt hinzufügen, und diese Option legt den Abstand von der Translationslinie fest, den die Maus haben muss, bevor die Punktmarkierung angezeigt wird. Der Standardwert ist 6px.
  • Standard-Parameterspuren automatisch erstellen: Wenn diese Option aktiviert ist, werden allen neuen Spuren, die im Spurenbedienfeld der Sequenz erstellt werden, automatisch Parameterspuren für Position, Skalierung, Drehung und Ursprung hinzugefügt. Die Voreinstellung ist deaktiviert.
  • Automatisch neu erstellte Assets dehnen: Mit dieser Option werden alle neu erstellten Asset-Schlüssel gestreckt, damit sie der Länge der Sequenz entsprechen. Die Voreinstellung ist deaktiviert.
  • Höhe der Animationskurvenspur im Dopw-Sheet: Mit dieser Option können Sie einstellen, wie viele Spurzeilen Parameterspuren, die Kurven verwenden, im Dopw-Sheet einnehmen, wenn sie sich in der Keyframe-Ansicht befinden. Die Voreinstellung ist 2, aber Sie können auch 1 oder 3 wählen.

Dialoge bestätigen

Die Voreinstellungen für Bestätigungsdialoge werden verwendet, um die Meldungen zu ändern, die im Sequenzeditor angezeigt werden, wenn Sie bestimmte Aktionen durchführen. Die Meldungen, die Sie ändern können, sind:

  • Automatische Auswahl zum Einbetten bearbeiteter Kurven: Wenn Sie versuchen, eine von Ihnen bearbeitete Animationskurve einzubetten, wird Ihnen standardmäßig eine Meldung angezeigt, um die Aktion zu bestätigen. Wenn Sie diese Option jedoch auf "Ja" setzen, wird die Meldung nicht angezeigt Meldung nicht angezeigt und jedes Mal eingebettet. Wenn Sie die Option auf "Nein" setzen, wird das Einbetten jedes Mal verhindert.
  • Automatische Auswahl zum Konvertieren von Kurven in Keyframes: Wenn Sie versuchen, eine Animationskurve in Keyframes zu konvertieren, wird standardmäßig eine Meldung angezeigt, die Sie fragt, ob Sie fortfahren möchten. Wenn Sie diese Option jedoch auf "Ok" setzen, wird keine Meldung angezeigt die Meldung nicht angezeigt und lässt Sie die Kurve einfach nicht konvertieren.
  • Automatische Auswahl der Anzeige einer Warnung bei der Bearbeitung externer Kurven: Wenn Sie versuchen, eine Kurve zu bearbeiten, die auf eine Parameterspur angewendet wird, und es sich um ein externes Animationskurven-Asset handelt, wird standardmäßig eine Warnmeldung angezeigt. Wenn Sie jedoch Wenn Sie diese Option jedoch auf "Ok" setzen, wird die Meldung nicht angezeigt und Sie können die Kurve trotzdem bearbeiten.


Intelligente Führungen

Hier können Sie das Verhalten der Smart-Guides beim Arbeiten im Sequenz-Editor auswählen. Die verfügbaren Optionen sind:

  • Aktivieren Sie Smart Guides: Hier können Sie die intelligenten Hilfslinien im Sequenz-Editor-Canvas aktivieren oder deaktivieren. Dies ist standardmäßig aktiviert.
  • Lineale aktivieren: Hier können Sie die intelligenten Hilfslinien im Sequences Editor Canvas aktivieren oder deaktivieren. Dies ist standardmäßig aktiviert.
  • Lineale an Kanten einrasten: Mit dieser Option können Lineale an den Kanten von Assets im Sequenz-Canvas "einrasten" (und umgekehrt). Die Voreinstellung ist aktiviert.
  • Fangabstand: Dies ist der Abstand (in Pixel), den ein Element zu einem Lineal (oder ein Lineal zu einem Element) haben muss, damit es an der Position "einrastet". Der Standardwert ist 6px.
