Voreinstellungen für den Animationskurven-Editor

In den Voreinstellungen für Animationskurven können Sie bestimmte Eigenschaften für die Arbeit mit dem Animationskurven-Editor festlegen. Die verfügbaren Optionen sind:

  • Automatische Reaktion auf das Löschen von Kurven: Mit dieser Option können Sie die Popup-Meldung steuern, die angezeigt wird, wenn Sie eine Kurve aus dem Animationskurven-Editor löschen. Standardmäßig wird hier eine Warnmeldung angezeigt, aber Sie können Sie können diese Option jedoch auf "Okay" setzen, dann wird der Dialog nicht angezeigt.
