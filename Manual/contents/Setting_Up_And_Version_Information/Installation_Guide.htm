Installationsanleitung

GameMaker Studio 2 ist konzipiert für den Einsatz auf macOS und Windows Desktop-Computern mit den hier genannten Mindestspezifikationen entwickelt.

Sie können den GameMaker Studio 2 Installer von Ihrer YoYo Account Seite auf der YoYo Games Website herunterladen. Nach der Installation werden Sie beim ersten Start von GameMaker Studio 2 möglicherweise aufgefordert, ein Update durchzuführen, bevor Sie das Produkt verwenden können. das Produkt zu verwenden. Nach der Installation eines Updates werden Sie aufgefordert, die Lizenzdaten für Ihr YoYo Games-Konto anzugeben (weitere Informationen finden Sie hier ), bevor Sie das Produkt verwenden können, Das heißt, wenn Sie GameMaker Studio 2 zum ersten Mal benutzen, müssen Sie eine Internetverbindung haben. Nach der ersten Anmeldung sollten Sie in der Lage sein, das Produkt offline zu verwenden, es wird jedoch in regelmäßigen Abständen verlangen, dass Sie eine Verbindung herstellen, um das Anmeldekontos.

Bitte beachten Sie, dass Sie - je nach Lizenztyp - möglicherweise zusätzliche Komponenten installieren oder zusätzliche Schritte ausführen müssen, um GameMaker Studio 2 für bestimmte Plattformen einzurichten und zu verwenden. Vollständige Informationen darüber, was jede Plattform benötigen, finden Sie in diesem Artikel auf dem YoYo Games Helpdesk: GMS2 Erforderliche SDKs.

Stabile und Beta-Versionen

Die Version, die Sie von der YoYo Games-Webseite herunterladen, sollte die aktuelle Stable-Version von GameMaker Studio 2 sein, aber es kann auch eine andere " Beta"-Version verfügbar sein, die dazu dient, neue Funktionen vorzustellen oder zu testen, bevor sie in die endgültige Stable-Version übernommen wird. Funktionalität zu testen, bevor sie in eine endgültige stabile Version übergeht. Die vollständigen Versionsdetails lauten wie folgt:

  • Stabil: Diese Version ist diejenige, die Sie für die allgemeine Entwicklung verwenden würden und wird für die Verwendung mit jedem kommerziellen Produkt empfohlen. Es kann sein, dass bestimmte Funktionen, die in den Beta-Versionen verfügbar sind, fehlen, aber das liegt daran, dass sie noch nicht nicht vollständig getestet wurden oder zusätzliche Arbeit erfordern.
  • Beta: Die Beta-Version von GameMaker Studio 2 ist für diejenigen gedacht, die neue Funktionen ausprobieren möchten, bevor sie zum Stable-Build hinzugefügt werden, oder für diejenigen, die einen kommenden Fix für ein Problem benötigen. Es wird nicht empfohlen, dass Sie die Beta Build für die endgültige Spielproduktion zu verwenden, es sei denn, sie enthält eine wichtige Korrektur für einen Fehler, der Ihr Projekt aufhält. Sie können den Beta-Build von GameMaker Studio 2 von der "Downloads"-Seite Ihres YoYo-Kontos herunterladen, und er kann parallel zum Stable-Build installiert und ausgeführt werden. mit dem Stable-Build. Bitte beachten Sie, dass Sie, wenn Sie GameMaker Studio 2 über Dampf verwenden, dann haben Sie keinen Zugriff auf den Beta-Build des Produkts über Steam selbst, obwohl Sie immer noch die Desktop-Version herunterladen und verwenden können.
