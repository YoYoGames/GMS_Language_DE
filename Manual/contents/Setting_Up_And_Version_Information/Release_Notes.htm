Freigabemitteilungen

Zu jedem Release von GameMaker Studio 2 gibt es einen eigenen Satz von Versionshinweisen. Diese enthalten detaillierte Informationen über die vorgenommenen Änderungen sowie eine vollständige Liste aller behobenen Fehler. Dies ist ein unschätzbares Hilfsmittel, um zu entscheiden ob Sie Ihre aktuelle Version aktualisieren sollten, oder wenn Sie nach einem Release einen Fehler finden, der möglicherweise ein Fehler ist.

Die Versionshinweise sind in zwei verschiedene Sätze aufgeteilt, da sowohl die IDE als auch die eigentlichen Runtimes unabhängig voneinander aktualisiert werden können. Die aktuellen Versionshinweise für die IDE finden Sie unter dem unten stehenden Link:

  • IDE-Versionshinweise

Die aktuellen Release Notes für die Runtimes finden Sie unter dem unten stehenden Link:

  • Runtime Release Notes
