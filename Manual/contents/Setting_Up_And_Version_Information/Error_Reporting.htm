Fehlerberichterstattung

Obwohl bei der Entwicklung von GameMaker Studio 2 große Sorgfalt darauf verwendet wurde, dass es ein stabiles und zuverlässiges Werkzeug ist, besteht aufgrund der Art der Software und der Betriebssysteme, auf denen sie läuft, immer die Möglichkeit, dass ein Problem auftritt. In den meisten In den meisten Fällen werden Fehler von GameMaker Studio 2 erkannt und Sie werden aufgefordert, Ihre Projekte zu speichern und das Programm anschließend zu schließen und neu zu starten. Sollte dies der Fall sein, bitten wir Sie, einen Fehler beim YoYo Games Helpdesk zu melden. Sie können dies tun über Wir bitten Sie, dabei so viele Informationen wie möglich anzugeben und auch einen Link zu der GameMaker Studio 2 ui.log Datei anzugeben, die Sie je nach Betriebssystem an folgenden Stellen finden können:

  • Unter Windows - C:\ProgramData\GameMakerStudio2
  • Unter macOS - /Users/Shared/GameMakerStudio2/ (Macintosh HD > Benutzer > Freigegeben > GameMakerStudio2 )

In sehr seltenen Fällen, wenn der Fehler schwerwiegend ist, kann es vorkommen, dass das Produkt abstürzt, ohne die oben erwähnte Warnung auszugeben, und wenn dies geschieht, wird das folgende Fenster angezeigt:

Dieses Fenster zeigt Ihnen einige der Absturzberichtsdaten, die von GameMaker Studio 2 zusammengestellt wurden, und gibt Ihnen die Möglichkeit, diese an YoYo Games zu senden, damit sie sehen können, was passiert ist, und das Problem, wenn möglich, in zukünftigen Updates beheben können. Wenn Sie genau wissen möchten, welche Daten gesendet werden, dann lesen Sie bitte diesen Artikel auf dem YoYo Games Helpdesk.
