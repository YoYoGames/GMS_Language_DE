tvOS-Einstellungen

Die tvOS-Einstellungen haben die folgenden Optionen:

  • Automatische Reaktion auf die Warnung zur Begrenzung des persistenten Speichers von tvOS: tvOS-Spiele haben einen begrenzten lokalen Speicher für Dateien, und wenn Sie für dieses Ziel kompilieren, wird standardmäßig eine Warnung angezeigt, um Sie an diese Tatsache zu erinnern. Sie können die Sie können die Einstellung hier ändern, um diese Warnung zu unterdrücken, indem Sie "OK" wählen. Die Standardeinstellung ist "Meldung anzeigen".
  • Build und Ausführung unterdrücken: Wenn Sie ein tvOS-Projekt erstellen und dann ein endgültiges ausführbares Paket erstellen, wird GameMaker Studio 2 standardmäßig das Paket erstellen und dann versuchen, es auf allen angeschlossenen Geräten auszuführen. Durch Aktivieren dieser Option wird unterdrückt das Erstellen der App und das Ausführen auf dem Gerät und erstellt einfach ein XCode-Projekt auf dem Host-Mac.
  • macOS-Installationspfad: Der Installationspfad für die Projektdateien auf dem Build-Mac.
  • Standard-Team-Kennung: Hier können Sie Ihre Standard-Team-Kennung hinzufügen, wie sie Ihnen von Apple zugewiesen wurde. Diese Team-ID wird verwendet, wenn Ihre Spieldateien an Xcode gesendet werden, um die App zu erstellen, und ermöglicht es Xcode, die erforderlichen Signierzertifikate zu generieren. Beachten Sie, dass diese Einstellung standardmäßig auf alle für tvOS erstellten Spiele angewendet wird, sie kann jedoch projektspezifisch in den allgemeinen tvOS tvOS.htm">Spiel-Optionen.
