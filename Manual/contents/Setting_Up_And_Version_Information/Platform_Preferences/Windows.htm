Windows-Voreinstellungen

Die Windows-Einstellungen haben die folgenden Optionen:

  • Visual Studio-Speicherort: Mit dieser Option verweisen Sie GameMaker Studio 2 auf die Microsoft Visual Studio-Installation auf dem Build-PC.
  • Standard-Paketierungstyp: Wenn Sie ein endgültiges ausführbares Windows-Paket erstellen, können Sie wählen, ob Sie ein Installationsprogramm (mit NSIS) oder eine komprimierte Datei ZIP die das Spiel und alle zusätzlichen Dateien enthält. Standardmäßig wird beim Kompilieren ein Dialogfeld angezeigt, in dem Sie zwischen diesen beiden Optionen wählen können, aber Sie können diese Voreinstellung so festlegen, dass immer die eine oder die andere Option verwendet wird. Beachten Sie, dass, wenn diese Option eingestellt ist auf Dialog anzeigen, im eigentlichen Dialogfenster, das beim Kompilieren erscheint, durch Aktivieren von "Verpackungsoption merken" diese Voreinstellung auf die von Ihnen gewählte Option gesetzt wird.
