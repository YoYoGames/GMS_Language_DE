HTML5-Voreinstellungen

Die HTML5-Einstellungen haben die folgenden Optionen:

  • Standard-Paketierungsart: Wenn Sie ein endgültiges ausführbares HTML5-Paket erstellen, können Sie zwischen "Package as loose files" (Paket als lose Dateien ) - was einfach alle erforderlichen Dateien in einem Ordner generiert - oder " Package as Zip" (Paket als Zip ) - was eine komprimierte ZIP Datei erstellt, die das Spiel und alle zusätzlichen Dateien enthält - wählen. Standardmäßig können Sie wählen, welche dieser beiden Optionen Sie beim Kompilieren wählen möchten, aber Sie können diese Sie können jedoch festlegen, dass immer die eine oder die andere Option verwendet wird. Beachten Sie, dass, wenn diese Option auf "Show Dialogue" (Dialog anzeigen) eingestellt ist, im eigentlichen Dialogfenster, das beim Kompilieren erscheint, die Auswahl von "Remember Packaging Option" diese Voreinstellung auf die von Ihnen gewählte Option die Sie auswählen.
  • Standard-Web-Server-Port: Hier können Sie den Standard-Port für alle über den Geräte-Manager erstellten Geräte ändern. Der Standardwert ist 51264.
  • Standard-Webserver-Adresse: Hier können Sie die Standard-Webserver-Adresse für Geräte ändern, die Sie dem Geräte-Manager hinzufügen. Der Standardwert ist 127.0.0.1.
  • Erlaubte Standard-IPs für Webserver: Hier können Sie beliebige erlaubte IPs für den Webserver hinzufügen, die dann als Standard für alle über den Target Manager hinzugefügten Geräte festgelegt werden.
