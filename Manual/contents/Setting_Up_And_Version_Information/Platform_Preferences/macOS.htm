macOS-Einstellungen

Die macOS-Einstellungen haben die folgenden Optionen:

  • Standard-Team-Kennung: Hier können Sie Ihre Standard-Team-Kennung hinzufügen, wie sie Ihnen von Apple zugewiesen wurde. Diese Team-Kennung wird verwendet, wenn Ihre Spieldateien an Xcode gesendet werden, um die App zu erstellen, und ermöglicht es Xcode, die erforderlichen Signierzertifikate zu generieren. Beachten Sie, dass diese Einstellung standardmäßig auf alle für macOS erstellten Spiele angewendet wird, aber sie kann projektspezifisch in den allgemeinen macOS Spiel-Optionen.
  • Wahl der automatischen Paketierung: Hier können Sie eine Standardpräferenz für die Erstellung einer ausführbaren Datei für das macOS-Ziel festlegen. Mit der Option "Dialog anzeigen" wird vor dem Erstellen einer ausführbaren Datei immer ein Dialog angezeigt, in dem Sie aufgefordert werden, zwischen DMG und Zip zu wählen, aber Sie können diese Option in den Voreinstellungen auf eine der beiden Varianten setzen, um diesen Dialog zu überspringen. Beachten Sie, dass das Aktivieren von "Paketoption merken" im Dialog auch diese Option auf den gewählten Pakettyp setzt.
