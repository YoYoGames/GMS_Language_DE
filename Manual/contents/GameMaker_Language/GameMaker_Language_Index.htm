GameMaker-Sprachindex

Die GameMaker Language (auch einfach GML genannt) ist die proprietäre ScriptspracheGameMaker Studio 2. Diese Sprache ist so strukturiert, dass der Benutzer seine Spiele auf intuitive und flexible Weise erstellen kann und gleichzeitig Dabei bietet sie alle Möglichkeiten, die auch jede andere große Programmiersprache bietet. Sie ist auch die Grundlage für Drag and Drop™ und kann bei Bedarf in Verbindung mit diesem verwendet werden.
