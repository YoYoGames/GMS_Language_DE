puffer_erzählen

Wenn Sie Daten in einen Puffer lesen oder schreiben, indem Sie die buffer_read() oder buffer_write() in einen Puffer schreiben oder lesen, wird die aktuelle "seek"-Position um die geschriebenen oder gelesenen Bytes vorverlegt, und mit Mit dieser Funktion können Sie die aktuelle "seek"-Position zur Verwendung in anderen Pufferfunktionen ermitteln. Wenn Ihre Pufferausrichtung z. B. auf 4 Byte eingestellt ist und Sie ein einzelnes Datenstück schreiben, das 1 Byte groß ist, führen Sie eine buffer_tell() aus, Wenn Sie jedoch ein weiteres Stück Daten schreiben, das ebenfalls 1 Byte groß ist, und dann eine buffer_tell() ausführen, erhalten Sie einen Rückgabewert von 5, da die Ausrichtung die Daten auf diese Position "aufgefüllt" hat.

Syntax:

buffer_tell(buffer);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.

Rückgabe:

Real

Beispiel:

var pos = buffer_tell(buff); buffer_seek(buff, buffer_seek_start, 0);
val[0] = buffer_read(buff, buffer_S16);
val[1] = buffer_read(buff, buffer_S16);
val[2] = buffer_read(buff, buffer_S16);
buffer_seek(buff, buffer_seek_start, pos);

Der obige Code speichert die aktuelle Suchposition innerhalb des in der Variablen "buff" indizierten Puffers in der lokalen Variablen "pos". Die Puffersuchposition wird dann auf den Anfang des Puffers gesetzt, und drei Daten werden in ein Array gelesen, bevor die Puffersuchposition wieder auf die vorherige Position gesetzt wird.
