buffer_get_alignment

Mit dieser Funktion können Sie das A-Byte-Alignment für die angegebene Puffer-ID ermitteln.

Syntax:

buffer_get_alignment(buffer);

Argument Beschreibung
Puffer Der Index des zu prüfenden Puffers.

Rückgabe:

Real

Beispiel:

alignment = buffer_get_alignment(buff);

Der obige Code holt sich die Ausrichtung des Puffers aus dem in der Variablen "buff" indizierten Wert und speichert sie in einer Variablen.
