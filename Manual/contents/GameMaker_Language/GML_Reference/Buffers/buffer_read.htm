puffer_lesen

Mit dieser Funktion können Sie Daten aus einem zuvor angelegten Puffer lesen. Der Rückgabewert hängt vom Typ der gelesenen Daten ab, der seinerseits durch die folgenden Konstanten definiert ist:

Konstant Beschreibung
Puffer_u8 Eine 8-Bit-Ganzzahl ohne Vorzeichen. Dies ist ein positiver Wert von 0 bis 255.
Puffer_s8 Eine vorzeichenbehaftete 8-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
Puffer_u16 Eine vorzeichenlose, 16-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 - 65.535.
Puffer_s16 Eine vorzeichenbehaftete, 16-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -32.768 bis 32.767 sein (0 wird als positiv eingestuft).
Puffer_u32 Eine vorzeichenlose, 32-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
Puffer_s32 Eine vorzeichenbehaftete, 32-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -2.147.483.648 bis 2.147.483.647 sein (0 wird als positiv eingestuft).
Puffer_u64 Eine 64-Bit-Ganzzahl ohne Vorzeichen.
Puffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb des Bereichs von +/- 65504 sein. (Wird derzeit nicht unterstützt!)
Puffer_f32 Ein 32-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb des Bereichs von +/-16777216 sein.
Puffer_f64 Ein 64-Bit-Float.
Puffer_Bool Ein boolescher Wert. Kann nur entweder 1 oder 0 sein(wahr oder falsch)
Puffer_Zeichenfolge Eine Zeichenkette mit beliebiger Größe.
puffer_text Eine Zeichenkette beliebiger Größe, ohne das abschließende Null-Abschlusszeichen.

Wenn die Funktion erfolgreich ist, gibt sie einen Wert des angegebenen Typs zurück, wenn sie jedoch fehlschlägt, verursacht sie einen Läuferfehler.

HINWEIS: Die Verwendung des falschen Datentyps für die zu lesenden Daten führt dazu, dass fehlerhafte Werte zurückgegeben werden.

Syntax:

buffer_read(buffer, type)

Argument Beschreibung
Puffer Der Index des Puffers, aus dem gelesen werden soll.
Typ Der Typ der Daten, die aus dem Puffer gelesen werden sollen (siehe die Liste der Konstanten oben).

Rückgabe:

Real/String

Beispiel:

var cmd = buffer_read(buff, buffer_s16);

Der obige Code liest aus dem Puffer mit der in der Variablen "buff" gespeicherten ID einen vorzeichenbehafteten 16-Bit-Wert in die lokale Variable "cmd".
