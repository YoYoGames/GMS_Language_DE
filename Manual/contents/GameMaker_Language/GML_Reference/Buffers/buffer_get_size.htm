buffer_get_size

Mit dieser Funktion können Sie die Größe des angegebenen Puffers in Bytes ermitteln.

Syntax:

buffer_get_size(index);

Argument Beschreibung
index Der Index des Puffers, dessen Größe ermittelt werden soll.

Rückgabe:

Real

Beispiel:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

Der obige Code erstellt einen neuen Puffer und speichert dessen Index in der lokalen Variablen "_temp", wobei die Größe dieses neuen Puffers mit der des in der Variablen "player_data" indizierten Puffers übereinstimmt.
