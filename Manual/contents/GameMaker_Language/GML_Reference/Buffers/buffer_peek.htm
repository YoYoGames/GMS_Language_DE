Pufferspeicher

Mit der Funktion buffer_read() Funktion können Sie Daten aus dem angegebenen Puffer an der aktuellen "seek"-Position lesen, wobei jedes zu lesende Datenstück diese Position um die zu lesenden oder zu schreibenden Bytes vorverlegt. Es kann jedoch notwendig sein, ein bestimmtes Stück Daten zu lesen, ohne die aktuelle "seek"-Position zu verändern, und in diesem Fall würden Sie diese Funktion verwenden. Sie übergeben der Funktion einfach eine Puffer-ID und dann eine Offset-Position (vom Sie geben der Funktion einfach eine Puffer-ID und eine Offset-Position (vom Pufferanfang) innerhalb des Puffers an, aus dem Sie lesen wollen, sowie den Datentyp, den Sie lesen wollen.

HINWEIS: Die Verwendung des falschen Datentyps für die zu lesenden Daten führt zu fehlerhaften Werten!

Bei Erfolg der Funktion wird ein Wert (Real, string, int, etc...) je nach Datentyp zurückgegeben, während bei einem Fehler einfach 0 zurückgegeben wird.

Syntax:

buffer_peek(buffer, offset, type);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.
Offset Die Offset-Position (in Bytes) innerhalb des Puffers, aus dem die angegebenen Daten gelesen werden sollen.
Typ Der Typ der Daten, die aus dem Puffer gelesen werden sollen (siehe die Liste der Konstanten hier).

Rückgabe:

Real/String oder 0, wenn es fehlschlägt.

Beispiel:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Der obige Code holt sich drei Werte von drei verschiedenen Positionen innerhalb des Puffers, die in der Variablen "buff" indiziert sind, und verwendet dann diese Werte, um die Bildüberblendung der Instanz einzustellen.
