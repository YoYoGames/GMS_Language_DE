Puffer_crc32

Diese Funktion nimmt Eingabedaten aus einem Puffer entgegen und gibt einen crc32-Prüfsummen-Hash zurück. Sie geben die Puffer-ID des zu verwendenden Puffers an, dann einen Offset-Wert (in Bytes) für die Stelle, an der begonnen werden soll, und dann eine Größe (wiederum in Bytes) für den zu hashenden Bereich, und die Funktion gibt einen 32-Bit-Ganzzahlwert für diesen Bereich zurück.

Syntax:

buffer_crc32(buffer, offset, size);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.
Offset Der Datenoffsetwert.
Größe Die Größe des Puffers.

Rückgabe:

Int 32

Beispiel:

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

Der obige Code erstellt einen crc32-Prüfsummen-Hash-Wert für die vollständigen Daten, die in dem durch die Variable "buff" indizierten Puffer gespeichert sind, und speichert den zurückgegebenen Integer-Hash-Wert in der Variablen "check_val".
