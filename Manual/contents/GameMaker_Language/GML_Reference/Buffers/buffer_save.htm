puffer_sichern

Mit dieser Funktion können Sie den Inhalt eines Puffers in einer Datei speichern, um ihn dann mit der puffer_laden() Funktion.

HINWEIS: Bei HTML5 wird der Inhalt des Puffers als base64-kodierte Zeichenkette gespeichert, wenn Sie diese Funktion verwenden.

Syntax:

buffer_save(buffer, Dateiname);

Argument Beschreibung
Puffer Der Index des zu speichernden Puffers.
Dateiname Der Name der Datei, unter dem gespeichert werden soll.

Rückgabe:

N/A

Beispiel:

buffer_save(buff, "Player_Save.sav");

Speichert den aktuellen Inhalt des Puffers mit der in der Variablen "buff" gespeicherten ID in eine Datei.
