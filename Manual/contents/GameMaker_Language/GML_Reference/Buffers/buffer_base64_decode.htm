puffer_base64_decode

Mit dieser Funktion können Sie einen base64-kodierten String (erstellt mit der Funktion buffer_base64_encode() Funktion) in einen Puffer. Diese Funktion erzeugt den Puffer (als 1-Byte-ausgerichteten "grow"-Puffer) und gibt den eindeutigen Index für den Puffer zurück, der in allen weiteren Funktionsaufrufen verwendet werden sollte.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erzeugten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden. Wenn Sie also mit dem erzeugten Puffer fertig sind, sollten Sie ihn wieder freigeben mit buffer_delete().

Syntax:

buffer_base64_decode(string);

Argument Beschreibung
String Die zu dekodierende base64-kodierte Zeichenfolge

Rückgabe:

Real

Beispiel:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

Der obige Code öffnet eine Ini-Datei und liest dann eine Zeichenkette daraus in die Decode-Funktion. Diese Funktion gibt einen Pufferindex zurück, der in der Variablen "buff" gespeichert ist und die zuvor kodierten und gespeicherten Daten enthält. Die ini-Datei wird dann geschlossen.
