puffer_speichern_ext

Mit dieser Funktion können Sie einen Teil des Inhalts eines Puffers in eine Datei speichern, um ihn dann mit der Funktion buffer_load() Funktion. Der "offset" definiert die Startposition innerhalb des Puffers zum Speichern (in Bytes), und die "Größe" ist die Größe des zu speichernden Pufferbereichs ab diesem Offset (ebenfalls in Bytes).

Syntax:

buffer_save_ext(buffer, Dateiname, offset, size);

Argument Beschreibung
Puffer Der Index des zu speichernden Puffers.
Dateiname Der Name der Datei, unter dem gespeichert werden soll.
Offset Der Offset innerhalb des Puffers, ab dem gespeichert werden soll (in Bytes).
Größe Die Größe des zu speichernden Pufferbereichs (in Bytes).

Rückgabe:

N/A

Beispiel:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Speichert einen Teil des aktuellen Inhalts des Puffers mit der in der Variablen "buff" gespeicherten ID in eine Datei.
