puffer_verkleinern

Mit dieser Funktion können Sie die Größe eines gegebenen Puffers auf die von Ihnen angegebene Größe (in Bytes) ändern.

Syntax:

buffer_resize(buffer, newsize);

Argument Beschreibung
Puffer Der Index des Puffers, dessen Größe geändert werden soll.
Nachrichtengröße Die neue Größe des Puffers (in Bytes).

Rückgabe:

N/A

Beispiel:

if (buffer_get_size(buff) < 16384)
{
buffer_resize(buff, 16384);
}

Der obige Code prüft die Größe des Puffers, der in der Variablen "buff" indiziert ist, und wenn sie kleiner als der angegebene Wert ist, wird die Größe des Puffers geändert.
