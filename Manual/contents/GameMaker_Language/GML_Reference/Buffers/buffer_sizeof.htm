puffer_groß

Diese Funktion gibt die Größe (in Bytes) einer der angegebenen Datenkonstanten ( hier aufgelistet) zurück.

Syntax:

buffer_sizeof(type);

Argument Beschreibung
Typ Der Typ der Daten, die geprüft werden sollen (siehe die Liste der Konstanten hier).

Rückgabe:

N/A

Beispiel:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Der obige Code berechnet zunächst die Größe des zu erstellenden Puffers, indem er den vorzeichenlosen 8-Bit-Datentyp mit 12 multipliziert (da wir den Puffer zur Aufnahme von 12 Daten verwenden werden), und verwendet dann diesen Wert, um einen festen Puffer auf die richtige Größe zu setzen.
