Puffer_vorhanden

Mit dieser Funktion kann eine Variable daraufhin überprüft werden, ob sie einen gültigen Puffer-ID-Wert enthält oder nicht. Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls wird false zurückgegeben.

Syntax:

buffer_exists(buffer)

Argument Beschreibung
Puffer Der Index des zu prüfenden Puffers.

Rückgabe:

Boolesche

Beispiel:

if buffer_exists(buff)
{
buffer_delete(buff);
}

Der obige Code prüft, ob die Variable " buff" eine Puffer-ID enthält, und wenn dies der Fall ist, wird der Puffer gelöscht.
