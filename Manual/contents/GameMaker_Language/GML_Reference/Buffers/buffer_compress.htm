puffer_komprimieren

Mit dieser Funktion können Sie einen Teil (oder den gesamten) Puffer mit der Zlib-Komprimierung komprimieren. Sie geben die ID des zu komprimierenden Puffers an (wie sie von puffer_erzeugen()), den zu verwendenden Offset innerhalb des Puffers in Bytes und die Größe der zu komprimierenden Pufferdaten (ebenfalls in Bytes). Die Funktion gibt einen neuen Wert für die Puffer-ID des komprimierten Puffers zurück oder einen Wert kleiner 0, wenn sie aus irgendeinem Grund fehlgeschlagen ist. Diese Funktion verändert den ursprünglichen Puffer nicht.

Syntax:

buffer_compress(buffer, offset, size);

Argument Beschreibung
Puffer Der Index des zu komprimierenden Puffers.
Offset Der Offset innerhalb des zu komprimierenden Puffers (in Bytes).
Größe Die Größe des zu komprimierenden Pufferbereichs (in Bytes).

Rückgabe:

Int (Puffer-ID-Wert)

Beispiel:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Der obige Code erstellt einen Puffer und füllt ihn dann mit den Daten aus einer Zeichenkette. Dieser Puffer wird dann komprimiert und gespeichert, und sowohl der Quellpuffer als auch der komprimierte Puffer werden gelöscht
