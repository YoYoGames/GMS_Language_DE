buffer_get_address

Mit dieser Funktion können Sie einen Zeiger auf die rohe, ausgerichtete Pufferadresse erhalten. Dies ist in erster Linie für die Verwendung mit Erweiterungen gedacht, da Sie diesen Wert an sie weitergeben können, um ihnen den Zugriff auf die Pufferdaten zu ermöglichen. Beachten Sie, dass es ein paar Dinge gibt die Sie mit dem Puffer nicht machen können, wenn Sie von einer Erweiterung aus zugreifen:

  • Sie können den Speicher nicht freigeben.
  • Sie können die Größe des Speichers nicht ändern.
  • Sie können nicht über die Pufferadresse plus die Puffergröße hinaus schreiben (Sie können die buffer_get_size() Funktion verwenden) oder Sie erhalten einen Out-of-Bounds-Fehler.

Syntax:

buffer_get_address(buffer);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.

Rückgabe:

Zeiger

Beispiel:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

Der obige Code ermittelt die Speicheradresse des Puffers, der in der Variablen "buff_model" indiziert ist, und ermittelt dann die Speicheradresse für das Ende des Puffers mit der Funktion buffer_get_size () und speichert beide Werte in lokalen Variablen für weitere Verwendung.
