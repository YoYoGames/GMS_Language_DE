puffer_laden_teilweise

Diese Funktion lädt einen Teil der Pufferdaten, die zuvor mit der Funktion buffer_save() Funktionen in einen bereits erstellten Puffer. Sie geben die ID des zuvor erstellten Puffers an, in den geladen werden soll, dann die gespeicherte Pufferdatei, die geladen werden soll, und dann den Offset vom Start des Puffers (in Bytes), von dem Sie die Daten laden möchten. Die folgenden Argumente dienen zum Festlegen der Länge der Pufferdaten (in Bytes) ab dem Offset-Anfangspunkt, den Sie laden möchten, und des Offset-Punkts, bis zu dem die Daten im Puffer geladen werden sollen (wiederum in Bytes).

Syntax:

buffer_load_partial(buffer, Dateiname, offset, src_len, dest_offset);

Argument Beschreibung
Puffer Der Index des zu ladenden Puffers.
Dateiname Der Name der zu ladenden Datei.
Offset Der Offset innerhalb des Puffers, in den geladen werden soll (in Bytes).
src_len Die Länge des zu ladenden Teils des Puffers (in Bytes).
dest_offset Der Offset, ab dem die Teildaten in den neuen Puffer eingefügt werden sollen (in Bytes).

Rückgabe:

Real

Beispiel:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

Der obige Code erzeugt einen neuen "grow"-Puffer und lädt dann einen Teil der in der Datei "save.dat" gespeicherten Daten in diesen.
