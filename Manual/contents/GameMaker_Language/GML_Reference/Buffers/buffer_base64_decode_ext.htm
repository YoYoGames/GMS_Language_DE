puffer_base64_decode_ext

Mit dieser Funktion können Sie einen base64-kodierten String (erstellt mit der Funktion buffer_base64_encode() Funktion) in einen Puffer. Anders als die Funktion buffer_base64_decode(), wird hier kein Puffer für Sie erstellt, sondern Sie sollten den Puffer bereits erstellt haben (siehe buffer_create()), dessen Id Sie dann mit dieser Funktion verwenden würden. Der "Offset" ist die Position innerhalb des Puffers zur Dekodierung der angegebenen Zeichenkette (in Bytes).

Syntax:

buffer_base64_decode_ext(buffer, string, offset);

Argument Beschreibung
Puffer Der Index des Puffers, in den der String dekodiert werden soll.
String Die base64-kodierte Zeichenfolge, die dekodiert werden soll.
Offset Der Datenoffsetwert.

Rückgabe:

N/A

Beispiel:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

Der obige Code erstellt einen Puffer und speichert die eindeutige ID dafür in der Variablen "buff", öffnet dann eine Ini-Datei und liest einen String daraus in die lokale Variable "str". Diese Zeichenkette wird dann in den neu angelegten Puffer dekodiert, bevor die Ini-Datei wieder geschlossen wird.
