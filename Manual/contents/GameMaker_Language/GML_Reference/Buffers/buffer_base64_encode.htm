buffer_base64_encode

Diese Funktion konvertiert die Daten aus dem angegebenen Puffer in eine im Base64-Format kodierte Zeichenfolge. Dies ist ein gängiges Kodierungsschema, das häufig für Medien verwendet wird, die als Text gespeichert oder über das Internet übertragen werden müssen, und macht die Ausgabe für das menschliche Auge unlesbar. Um dies zu verwenden, müssen Sie einen bereits erstellten Puffer, den Offset-Wert (das ist der Punkt innerhalb des Puffers, an dem Sie mit der Kodierung beginnen möchten) sowie die Größe des zu kodierenden Pufferspeichers in Bytes angeben.

Syntax:

buffer_base64_encode(buffer, offset, size);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.
Offset Der Datenoffsetwert.
Größe Die Größe des Puffers.

Rückgabe:

Zeichenfolge

Beispiel:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Der obige Code erzeugt eine Kodierung der vollständigen Daten, die in dem durch die Variable "buff" indizierten Puffer gespeichert sind, und speichert die zurückgegebene Zeichenfolge in der lokalen Variablen "b_str".
