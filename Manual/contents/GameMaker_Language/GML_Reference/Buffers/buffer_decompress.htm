puffer_dekomprimieren

Mit dieser Funktion können Sie einen zuvor komprimierten Puffer mit Zlib-Komprimierung dekomprimieren. Sie geben die Puffer-ID an, die dekomprimiert werden soll, und die Funktion gibt eine neue Puffer-ID zurück, die die unkomprimierten Daten enthält. Wenn die Dekomprimierung fehlgeschlagen ist (z. B. wenn Sie einen Puffer übergeben, der nicht komprimiert wurde), gibt die Funktion stattdessen einen Wert kleiner als 0 zurück.

Syntax:

buffer_decompress(buffer);

Argument Beschreibung
Puffer Der Index des Puffers, der dekomprimiert werden soll.

Rückgabe:

Puffer-ID

Beispiel:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Der obige Code lädt zuerst einen gespeicherten Puffer, dekomprimiert ihn dann und liest schließlich die String-Daten aus dem dekomprimierten Puffer in eine globale Variable.
