buffer_set_used_size

Diese Funktion ist in erster Linie für die Verwendung innerhalb von Erweiterungen gedacht und ermöglicht es Ihnen, die "verwendete" Größe des angegebenen Puffers einzustellen, d. h. die Anzahl der Bytes, die in ihn geschrieben wurden.

Wenn Sie Daten von einer Erweiterung in einen Puffer schreiben, weiß GameMaker nicht, wie viel des Puffers durch den Erweiterungscode gefüllt wurde und kann diese Daten nicht lesen. Diese Funktion kann von der Erweiterung aufgerufen werden, um der Engine mitzuteilen, wie viele Bytes an Daten in den Puffer geschrieben wurden, so dass die Engine diese Daten lesen kann.

Syntax:

buffer_set_used_size(buffer, size);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.
Größe Die Anzahl der Bytes, die als "verwendete" Größe eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

Die obigen Funktionen würden in einer Erweiterung aufgerufen werden, um einige Daten in einen GameMaker-Puffer zu schreiben (über seine Speicheradresse) und dann die Anzahl der Bytes zu setzen, die in ihn geschrieben wurden, damit die Engine diese Daten lesen kann.
