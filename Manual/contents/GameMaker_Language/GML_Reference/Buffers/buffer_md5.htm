Puffer_md5

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert und wurde in einer Vielzahl von Sicherheitsanwendungen eingesetzt. Sie wird auch häufig verwendet, um die Integrität von Daten zu prüfen. Diese Funktion nimmt die Eingabedaten aus einem gegebenen Puffer (oder einem Teil des Puffers) und gibt den 32-stelligen hexadezimalen MD5-Hashwert zurück, der für diese Daten eindeutig ist. Auf diese Weise können Sie einen einen sicheren Schlüssel, der gespeichert und verwendet werden kann, um die Integrität der Informationen zu prüfen, die z. B. an einen externen Server gesendet (oder von diesem empfangen) werden. Wenn Sie dies mit dieser Funktion auf Puffer anwenden, müssen Sie die Puffer-ID des zu verwendenden Puffers angeben. verwenden, dann einen Offset-Wert (in Bytes) für die Stelle, an der begonnen werden soll, und dann eine Größe (wieder in Bytes) für den zu hashenden Bereich.

HINWEIS: MD5 ist nicht vollständig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

buffer_md5(buffer, offset, size);

Argument Beschreibung
Puffer Der Index des zu verwendenden Puffers.
Offset Der Datenoffsetwert.
Größe Die Größe des Puffers.

Rückgabe:

Zeichenfolge

Beispiel:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

Der obige Code erstellt einen md5-Hash für die vollständigen Daten, die in dem durch die Variable "buff" indizierten Puffer gespeichert sind, und speichert den zurückgegebenen Hash in der Variablen "check_string".
