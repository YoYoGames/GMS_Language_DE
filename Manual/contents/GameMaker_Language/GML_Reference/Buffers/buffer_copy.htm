Puffer_kopieren

Diese Funktion kann verwendet werden, um ein Segment (oder alle) der in einem Puffer gespeicherten Daten in einen anderen zu kopieren. Wenn Sie zwei Puffer verwenden und von einem in den anderen kopieren, müssen beide Puffer zuvor mit der Funktion buffer_create() Funktion erstellt worden sein und Sie können einen Datenoffset (in Bytes) für den Startpunkt der zu kopierenden Daten aus dem Quellpuffer relativ zum Anfang des Puffers angeben, sowie einen weiteren Datenoffset, um die Position zu definieren, an die die Daten kopiert werden sollen im Zielpuffer.

HINWEIS: Sie können nicht in denselben Puffer kopieren.

Syntax:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

Argument Beschreibung
src_puffer Der Index des Puffers, aus dem kopiert werden soll.
src_offset Der Datenoffset, ab dem der Kopiervorgang beginnen soll (in Bytes).
Größe Die Größe der zu kopierenden Daten (in Bytes).
dest_buffer Der Index des Puffers, in den kopiert werden soll.
dest_offset Die Offsetposition, an die die Daten kopiert werden sollen (in Bytes).

Rückgabe:

N/A

Beispiel:

buffer_copy(buff1, 0, 2048, buff2, 2048);

Der obige Code kopiert die Daten, die in dem in der Variablen "buff1" indizierten Puffer gespeichert sind, und fügt sie dann in den in der Variablen "buff2" indizierten Puffer ein, jedoch um 2048 Bytes versetzt.
