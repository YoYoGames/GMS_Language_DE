puffer_kopieren_vom_vertex_puffer

Mit dieser Funktion können einige (oder alle) Scheitelpunktdaten, die in einem Scheitelpunktpuffer gespeichert sind, in einen zuvor erstellten regulären Puffer kopiert werden. Wenn Sie mit dieser Funktion aus einem Scheitelpunkt-Puffer in einen regulären Puffer kopieren, müssen beide Puffer zuvor erstellt worden sein (mit den Befehlen vertex_create_buffer() und buffer_create() Funktionen). Sie können den Bereich der Scheitelpunktdaten angeben, den Sie Sie können den Bereich der Scheitelpunktdaten angeben, den Sie in den Puffer kopieren möchten, wobei der Start-Scheitelpunkt irgendwo zwischen 0 und der Anzahl der Scheitelpunkte -1 liegen kann, und Sie können die Anzahl der Scheitelpunkte angeben, die von diesem Punkt an kopiert werden sollen. Sie können die Funktion vertex_get_number() auf den Scheitelpunktpuffer anwenden, um die Gesamtzahl der gespeicherten Scheitelpunkte zu erhalten. Schließlich geben Sie den Pufferindex an, in den die Scheitelpunktdaten kopiert werden sollen, sowie einen Datenoffset, um die Position zu definieren, an die die Scheitelpunktdaten im Zielpuffer kopiert werden sollen.

Syntax:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argument Beschreibung
vertex_buffer Der Index des Vertex-Puffers, aus dem kopiert werden soll.
start_vertex Der Startscheitelpunkt.
num_vertices Die Gesamtzahl der zu verwendenden Scheitelpunkte.
dest_buffer Der Index des Puffers, in den kopiert werden soll.
dest_offset Die Offsetposition, an die die Daten kopiert werden sollen (in Bytes).

Rückgabe:

N/A

Beispiel:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

Der obige Code kopiert die Scheitelpunktdaten, die in dem in der Variablen "model_buffer" indizierten Scheitelpunktpuffer gespeichert sind, und fügt sie dann in den in der Variablen "player_buffer" indizierten Puffer ein.
