Puffer_Löschen

Mit dieser Funktion können Sie einen Puffer, den Sie zuvor mit der Funktion puffer_erstellen() aus dem Speicher löschen, wobei die Ressourcen, die für seine Erstellung verwendet wurden, freigegeben und alle Daten, die er derzeit enthält, entfernt werden.

HINWEIS: Es ist wichtig, dass Sie dynamisch erstellte Ressourcen immer aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden.

WICHTIG! Wenn Sie einen Buffer erstellen, ist der Indexwert, mit dem er identifiziert wird, ein ganzzahliger Wert, der bei 0 beginnt. Diese Indizes werden vom GameMaker wiederverwendet, so dass ein zerstörter Buffer-Indexwert anschließend von einem neu erstellten verwendet werden kann, und wir empfehlen, dass Sie jede Variable, die einen Pufferindex enthält, auf -1 setzen, nachdem Sie den Puffer zerstört haben.

Syntax:

buffer_delete(buffer)

Argument Beschreibung
Puffer Der Index des zu löschenden Puffers.

Rückgabe:

N/A

Beispiel:

buffer_delete(player_buffer);
player_buffer = -1;

Der obige Code löscht den zuvor erstellten Puffer mit der in der Variablen "player_buffer" gespeicherten ID und setzt die Variable anschließend auf -1.
