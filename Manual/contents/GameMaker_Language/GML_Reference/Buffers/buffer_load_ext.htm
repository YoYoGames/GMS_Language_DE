puffer_laden_ext

Diese Funktion lädt die Pufferdaten, die zuvor mit den buffer_save() Funktionen in einen bereits erstellten Puffer. Sie geben die ID des bereits erstellten Puffers an, in den geladen werden soll, dann die gespeicherte Pufferdatei, die geladen werden soll, und schließlich den Offset vom Anfang des Puffers (in Bytes), in den Sie die Daten laden möchten.

Syntax:

buffer_load_ext(buffer, Dateiname, offset);

Argument Beschreibung
Puffer Der Index des Puffers, in den geladen werden soll.
Dateiname Der Name der Datei, aus der geladen werden soll.
Offset Der Offset innerhalb des Puffers, in den geladen werden soll (in Bytes).

Rückgabe:

N/A

Beispiel:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

Der obige Code ermittelt zunächst die Position des Endes des in der Variablen "player_buffer" indizierten Puffers und lädt dann die Daten aus dem angegebenen in diese Position (beachten Sie, dass dieses Beispiel nur mit den Puffertypen "grow" oder "wrap" funktioniert Puffertypen funktioniert).
