Debug_mode

Diese schreibgeschützte Variable gibt true zurück, wenn das Spiel im Debug-Modus gespielt wird, und false, wenn es als normales Spiel gespielt wird.

Syntax:

Debug_mode

Rückgabe:

Boolesche

Beispiel:

if debug_mode
{
ini_open("Cheats.ini");
}
sonst
{
ini_open("Spiel.ini");
}

Der obige Code öffnet je nach Wert der Nur-Lese-Variablen debug_mode eine andere Ini-Datei.
