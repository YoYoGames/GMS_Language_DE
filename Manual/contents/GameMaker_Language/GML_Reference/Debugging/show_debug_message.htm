show_debug_message

Diese Funktion erzeugt eine eigene Debug-Meldung, die zur Laufzeit im Compiler-Fenster angezeigt wird. Beachten Sie, dass die Meldung entweder eine Zeichenkette oder eine reelle Zahl sein kann, aber wenn Sie beides benötigen, muss die Zahl zuerst in eine Zeichenkette konvertiert werden, indem Sie der Funktion string() Funktion in eine Zeichenkette konvertiert werden muss (siehe das Beispiel unten) und dass, wenn die Zahl mehr als zwei Dezimalstellen hat, Sie die Funktion string_format() um sie anzuzeigen verwenden, da Dezimalzahlen standardmäßig auf die nächsten zwei Nachkommastellen gerundet werden (so würde "1,2468" als "1,25" im Ausgabefenster angezeigt).

Debug-Meldungen, die mit dieser Funktion angezeigt werden, werden sowohl im Compiler-Ausgabefenster unten in der IDE als auch in der Graph-Ansicht des des Debuggers angezeigt, wenn das Spiel im Debug-Modus läuft. Wenn Sie die Meldungen nur im Debug-Modus sehen wollen, sollten Sie wahrscheinlich die Funktion debug_event() stattdessen.

Syntax:

show_debug_message(string);

Argument Beschreibung
String Die benutzerdefinierte Debug-Meldung, die angezeigt werden soll.

Rückgabe:

N/A

Beispiel:

if !instance_exists(obj_Exit)
{
show_debug_message("Exit not created!");
show_debug_message("Error Num: " + string(global.error));
game_end();
}

Der obige Code prüft, ob eine Instanz vorhanden ist, und wenn nicht, wird eine Debug-Meldung im Kompilierfenster angezeigt und das Spiel wird beendet.
