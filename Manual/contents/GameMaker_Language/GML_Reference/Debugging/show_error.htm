show_error

Diese Funktion zeigt eine benutzerdefinierte Zeichenkette als Fehlermeldung an und gibt die Möglichkeit, das Spiel anschließend abzubrechen, falls erforderlich.

HINWEIS: Diese Funktion ist nur zur Fehlersuche geeignet.

Syntax:

show_error(str, abort);

Argument Beschreibung
str Die Zeichenfolge, die in der Popup-Meldung angezeigt werden soll.
Abbrechen Ob abgebrochen werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

if (room != rm_Dungeon)
{
show_error("Error: Went to wrong area. Aborting game.", true);
}

Der obige Code prüft, ob der Raum, in dem sich der Spieler befindet, "rm_Dungeon" ist, und wenn nicht, wird eine Fehlermeldung angezeigt und der Benutzer gezwungen, das Spiel abzubrechen.
