get_string

Dadurch wird ein Popup-Fenster mit einer Standardmeldung und einer Schaltfläche mit der Aufschrift "Ok" erstellt, die den Benutzer zur Eingabe einer Zeichenfolge auffordert. Die Funktion gibt die eingegebene Zeichenkette zurück, oder den Standardwert, wenn nichts eingegeben wurde.

HINWEIS: Diese Funktion ist nur für die Debug-Verwendung auf Desktop-Zielen vorgesehen, ist aber auf allen anderen Zielen veraltet.

Syntax:

get_string(str, def);

Argument Beschreibung
str Die Zeichenfolge, die in der Popup-Meldung angezeigt werden soll.
def Die Standardzeichenfolge im Textfeld.

Rückgabe:

Zeichenfolge

Beispiel:

global.test_name = get_string("Test-Highscore-Name:", "Anonym");

Der obige Code fordert den Benutzer auf, einen Namen einzugeben, der dann in der globalen Variablen "test_name" gespeichert wird. Wenn nichts eingegeben wird und der Benutzer nur "Ok" drückt, wird der Standardwert "Anonymous" zurückgegeben.
