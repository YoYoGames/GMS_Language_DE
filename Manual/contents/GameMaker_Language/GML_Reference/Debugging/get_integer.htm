get_integer

Dadurch wird ein Popup-Fenster mit einer benutzerdefinierten Meldung und einer Schaltfläche mit der Aufschrift "Ok" erstellt, in dem der Benutzer aufgefordert wird, einen Integer-Wert einzugeben. Die Funktion gibt den eingegebenen Integer-Wert zurück oder den Standardwert, wenn nichts eingegeben wurde.

HINWEIS: Diese Funktion ist nur für die Debug-Verwendung auf Desktop-Zielen vorgesehen, ist aber auf allen anderen Zielen veraltet.

Syntax:

get_integer(str, def);

Argument Beschreibung
str Die Zeichenfolge, die in der Popup-Meldung angezeigt werden soll.
def Der Standardwert im Textfeld.

Rückgabe:

Real

Beispiel:

global.level = get_integer("Level to test?", 1);

Der obige Code zeigt eine Meldung an, die den Benutzer auffordert, eine Ebene zum Testen auszuwählen. Der Rückgabewert wird in der globalen Variable "global.level" gespeichert.
