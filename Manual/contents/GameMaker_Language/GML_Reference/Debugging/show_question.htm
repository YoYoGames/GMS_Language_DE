show_question

Diese Funktion erzeugt ein Popup-Meldefeld mit zwei Schaltflächen für "Ja" und "Nein". Sie gibt true oder false zurück, je nachdem, welche der beiden Schaltflächen der Benutzer drückt.

HINWEIS: Diese Funktion ist nur für die Debug-Verwendung auf Desktop-Zielen vorgesehen, ist aber auf allen anderen Zielen veraltet.

Syntax:

show_question(str);

Argument Beschreibung
str Die Zeichenfolge, die in der Popup-Frage angezeigt werden soll.

Rückgabe:

Boolesche

Beispiel:

if (Punktestand > 500) && debug_mode
{
if show_question("Weiter zum nächsten Raum?")
{
room_goto(rm_Level2);
}
else game_end();
}

Der obige Code prüft den Punktestand, und wenn er über 500 liegt, wird der Benutzer gefragt, ob er fortfahren möchte oder nicht, und wenn die Schaltfläche "Ja" angeklickt wird, geht er zu einem anderen Raum, aber wenn die Schaltfläche "Nein" ausgewählt wird, wird das Spiel beendet.
