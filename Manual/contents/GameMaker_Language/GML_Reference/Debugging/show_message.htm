show_message

Diese Funktion erzeugt ein Popup-Fenster, in dem die angegebene Zeichenkette angezeigt wird, und eine Schaltfläche mit der Aufschrift "Ok", um es zu schließen.

HINWEIS: Diese Funktion ist nur für die Debug-Verwendung auf Desktop-Zielen und UWP vorgesehen, ist aber auf allen anderen Zielen veraltet.

Syntax:

show_message(str);

Argument Beschreibung
str Die Zeichenfolge, die in der Popup-Meldung angezeigt werden soll.

Rückgabe:

N/A

Beispiel:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
tot += inv[i];
}
show_message("Summe = " + string(tot));

Der obige Code durchläuft die im Array "inv" gespeicherten Werte in einer Schleife und addiert sie zur Variablen "tot", bevor eine Meldung mit der Gesamtsumme angezeigt wird.
