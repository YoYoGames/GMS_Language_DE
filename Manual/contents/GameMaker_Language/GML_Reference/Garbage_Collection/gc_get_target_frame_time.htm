gc_get_target_frame_time

Mit dieser Funktion können Sie den aktuellen Ziel-Frame-Wert für den Garbage Collector abrufen. Der zurückgegebene Wert ist in Mikrosekunden (wobei 1.000.000 Mikrosekunden einer Sekunde entsprechen) und die Standard-Zielbildzeit ist 100 Mikrosekunden. Wenn Sie diesen Wert ändern möchten, sollten Sie die Funktion gc_target_frame_time() verwenden.

Syntax:

gc_get_target_frame_time(time);

Rückgabe:

Real

Beispiel:

if (gc_get_target_frame_time() != 50)
{
gc_target_frame_time(50);
}

Der obige Code prüft das aktuelle Frame-Zeitziel für den Garbage Collector und wenn es nicht 50 Mikrosekunden beträgt, wird es auf diesen Wert gesetzt.
