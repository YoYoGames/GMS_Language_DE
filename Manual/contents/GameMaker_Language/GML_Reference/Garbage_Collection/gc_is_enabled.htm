gc_is_enabled

Mit dieser Funktion können Sie prüfen, ob der Garbage Collector aktiviert ist oder nicht. Die Funktion gibt true zurück, wenn er aktiviert ist, ansonsten false.

Syntax:

gc_is_enabled();

Rückgabe:

Bool

Beispiel:

if (!gc_is_enabled())
{
gc_enable(true);
}

Der obige Code prüft, ob der Garbage Collector aktiviert ist, und wenn nicht, aktiviert er ihn.
