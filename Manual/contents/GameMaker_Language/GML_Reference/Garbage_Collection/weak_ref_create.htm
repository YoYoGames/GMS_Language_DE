weak_ref_create

Mit dieser Funktion können Sie eine schwache Referenz auf eine struct erzeugen, die dann verwendet werden kann, um zu prüfen, ob die struct noch "lebendig" (referenziert) ist oder nicht im Spiel. Sie liefern die Referenz auf die Struktur, die Sie verfolgen wollen, und die Funktion gibt eine andere Struktur zurück, die eine schwache Referenz auf diese Struktur ist. Beachten Sie, dass Sie überprüfen können, ob eine Referenz "strong" oder "weak" ist, indem Sie die Funktion instanceof() verwenden, da eine starke Referenz entweder "struct" oder den Namen der Konstruktorfunktion zurückgibt, die die Struktur erzeugt hat, oder "weakref", wenn es sich um eine schwache Referenz handelt. Beachten Sie auch, dass die "weak reference"-Struktur eine Variable "ref" hat, auf die zugegriffen werden kann, um die String-Referenz auf die betreffende Struktur zu erhalten, es sei denn, es wurde Garabage gesammelt, in diesem Fall wird undefined zurückgegeben.

Syntax:

weak_ref_create(struct_to_track);

Argument Beschreibung
struct_to_track Die Struktur, für die Sie eine schwache Referenz erzeugen wollen

Rückgabe:

Struktur

Beispiel:

inventory_ref = weak_ref_create(inventory);

Der obige Code erzeugt eine schwache Referenz auf eine struct und speichert sie in einer Instanzvariablen zur späteren Verwendung.
