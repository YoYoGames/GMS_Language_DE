gc_freigeben

Mit dieser Funktion können Sie den Garbage Collector aktivieren oder deaktivieren. Der Aufruf der Funktion mit true als Argument aktiviert ihn und mit false wird er deaktiviert (nicht empfohlen). Er ist standardmäßig aktiviert.

Syntax:

gc_enable(enable);

Argument Beschreibung
aktivieren. aktivieren(true) oder deaktivieren(false) Sie den Garbage Collector.

Rückgabe:

N/A

Beispiel:

if (global.debug == true)
{
gc_enable(false);
}

Der obige Code deaktiviert die Garbage Collection, wenn die angegebene globale Variable true ist.
