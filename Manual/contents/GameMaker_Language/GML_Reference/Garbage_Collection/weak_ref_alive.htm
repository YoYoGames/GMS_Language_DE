weak_ref_create

Mit dieser Funktion können Sie die Weak-Referenz auf eine Struktur prüfen, um zu sehen, ob sie noch "am Leben" ist oder nicht. Sie übergeben die zu prüfende Weak-Referenz (wie sie von der Funktion weak_ref_create()), und die Funktion gibt true zurück, wenn die Struktur noch irgendwo referenziert wird, oder false, wenn dies nicht der Fall ist und sie bereits gelöscht wurde. Beachten Sie, dass die Funktion undefined zurückgibt, wenn Sie einen Wert übergeben, der keine schwache Referenz ist.

Syntax:

weak_ref_alive(weak_ref);

Argument Beschreibung
schwach_ref Der schwache Verweis auf die Struktur, die Sie prüfen möchten.

Rückgabe:

Boolesch (oder undefiniert)

Beispiel:

if weak_ref_alive(inventory_ref)
{
inventory = -1;
}

Der obige Code prüft eine schwache Referenz auf eine Struktur und wenn sie noch lebt, wird die Variable, die sie referenziert, auf -1 gesetzt, wodurch die Referenz auf die Struktur aufgehoben wird und sie garbage collected werden kann.
