Allgemeine Dampf-API

Die folgenden Funktionen dienen alle zur Überprüfung der Verfügbarkeit bestimmter Aspekte der Steam-Client- oder -Server-API und geben als solche entweder true oder false zurück, abhängig von Faktoren wie z. B. dass der Server beschäftigt ist oder der Benutzer angemeldet ist usw.. Das bedeutet, dass diese Funktionen vor jedem anderen Steam-API-Funktionsaufruf verwendet werden sollten, um sicherzustellen, dass die Client/Server-Einrichtung korrekt ist und mit Ihrem Spiel kommuniziert:

  • dampf_initialisiert
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
