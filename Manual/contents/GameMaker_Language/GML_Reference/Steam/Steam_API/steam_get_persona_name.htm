steam_get_persona_name

Mit dieser Funktion können Sie den Benutzernamen des Benutzers zurückgeben, der derzeit im Steam-Client angemeldet ist. Dies ist der sichtbare Bildschirmname und nicht die eindeutige Benutzer-ID (diese kann mit der Funktion steam_get_user_steam_id()).

Syntax:

steam_get_persona_name();

Rückgabe:

Zeichenfolge

Beispiel:

if steam_initialised()
{
global.p_name = steam_get_persona_name();
}

Der obige Code setzt eine globale Variable auf den Bildschirmnamen des aktuellen Benutzers, wenn die Steam-Client-API korrekt initialisiert ist.
