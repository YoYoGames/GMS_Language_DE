steam_get_app_id

Mit dieser Funktion wird die eindeutige App-ID abgerufen, die Steam für Ihr Spiel vergibt und die für die Nutzung einiger der Funktionen für benutzergenerierte Inhalte erforderlich ist.

Syntax:

steam_get_app_id();

Rückgabe:

Real

Beispiel:

global.app_id = steam_get_app_id();

Der obige Code ruft die eindeutige App-ID für Ihr Spiel auf Steam ab und speichert sie in einer globalen Variablen.
