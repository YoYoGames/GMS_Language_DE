steam_available_languages

Diese Funktion kann verwendet werden, um eine Liste aller verfügbaren Sprachen für Steam abzurufen. Der Rückgabewert ist einfach eine durch Komma getrennte Liste von Sprachen.

Syntax:

steam_available_languages();

Rückgabe:

Zeichenfolge

Beispiel:

language = steam_available_languages();

Das obige ruft die verfügbaren Sprachen für Steam als String ab und speichert ihn in einer Variablen.
