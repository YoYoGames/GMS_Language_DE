dampf_initialisiert

Wenn Sie die Steam-API verwenden, kann diese Funktion aufgerufen werden, um zu prüfen, ob die Steam-Client-API korrekt initialisiert wurde, bevor Sie weitere Aufrufe von Steam-spezifischen Funktionen in Ihrem Spiel durchführen.

Syntax:

steam_initialised();

Rückgabe:

Boolesche

Beispiel:

global.steam_api= false;
if steam_initialised()
{
if steam_stats_ready() && steam_is_overlay_enabled()
{
global.steam_api= true;
}
}

Der obige Code setzt eine globale Variable auf true, wenn die Steam-Client-API korrekt initialisiert ist, zusammen mit der Steam-Statistik- und Overlay-Funktionalität, oder er setzt die Variable andernfalls auf false.
