steam_is_user_logged_on

Diese Funktion gibt true zurück, wenn der Steam-Client derzeit eine aktive Verbindung zu den Steam-Servern hat. Wenn sie false zurückgibt, bedeutet dies, dass keine aktive Verbindung besteht, weil entweder ein Netzwerkproblem auf dem lokalen Rechner vorliegt oder der Steam Server ausgefallen oder beschäftigt ist.

Syntax:

steam_is_user_logged_on();

Rückgabe:

Zeichenfolge

Beispiel:

if steam_is_user_logged_on()
{
global.user_id = steam_get_user_account_id();
}

Der obige Code prüft, ob der Benutzer auf dem Steam-Server angemeldet ist, und speichert die Benutzer-ID in einer globalen Variablen.
