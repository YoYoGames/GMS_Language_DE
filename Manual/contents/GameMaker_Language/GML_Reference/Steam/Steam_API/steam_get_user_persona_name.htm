steam_get_user_persona_name

Diese Funktion kann verwendet werden, um den Benutzernamen (Bildschirmname) für eine bestimmte Benutzer-ID abzurufen. Sie geben die Benutzer-ID an, dann gibt die Funktion einen eindeutigen Async-ID-Wert zurück, der zur korrekten Überprüfung der Details im Steam-Async-Ereignis verwendet werden kann. Dieses Ereignis hat die async_load DS-Map, die mit den folgenden Schlüssel/Wert-Paaren bestückt wird:

  • "id" - Die asynchrone ID, die von der Funktion zurückgegeben wird, die das Ereignis ausgelöst hat
  • " event_type" - Die Zeichenkette " user_persona_name"
  • "steamid""- Der Steam-ID-Wert , der verwendet wird, um den Benutzernamen zu erhalten.
  • "persona_name" - Der Benutzername für die angegebene Steam-ID

Syntax:

steam_get_user_persona_name(steamID);

Argument Beschreibung
steamID Die eindeutige Steam-ID für einen Benutzer.

Rückgabe:

Real

Beispiel:

request = steam_get_user_persona_name(global.UGC_UserID);

Der obige Code fragt den Benutzernamen der in der globalen Variablen "UGC_UserID" gespeicherten Benutzer-ID ab und speichert den zurückgegebenen Wert in einer Variablen zum Parsen im Async-Event.
