steam_current_game_language

Mit dieser Funktion wird die aktuelle Sprache, die Steam verwendet, abgerufen (als String), zum Beispiel "english".

Syntax:

steam_current_game_language();

Rückgabe:

Zeichenfolge

Beispiel:

language = steam_current_game_language();

Der obige Code ruft die für das aktuelle Spiel verwendete Sprache ab.
