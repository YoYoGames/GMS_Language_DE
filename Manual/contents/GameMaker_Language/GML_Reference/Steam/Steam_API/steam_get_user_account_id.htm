steam_get_user_account_id

Mit dieser Funktion wird die eindeutige Benutzer-ID abgerufen, die Steam jedem Benutzer zuweist und die für die Verwendung einiger der Funktionen für benutzergenerierte Inhalte erforderlich ist.

Syntax:

steam_get_user_account_id();

Rückgabe:

Real

Beispiel:

global.user_id = steam_get_user_account_id();

Der obige Code ruft die eindeutige Benutzer-ID für die Person ab, die das Spiel besitzt, und speichert sie in einer globalen Variablen.
