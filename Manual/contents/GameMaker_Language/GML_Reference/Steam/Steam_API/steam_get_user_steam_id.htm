steam_get_user_steam_id

Mit dieser Funktion können Sie die eindeutige Steam-Benutzerkennung des Benutzers zurückgeben, der gerade im Steam-Client angemeldet ist. Dies ist ein eindeutiger 64-Bit-Wert, der verwendet werden kann, um einzelne Benutzer zu identifizieren und nicht ihren Benutzernamen auf dem Bildschirm (dieser kann mit der Funktion steam_get_persona_name()). Da es sich um einen int64-Wert handelt, müssen Sie ihn möglicherweise als solchen casten, wenn Sie ihn in nachfolgenden Funktionen mit der int64() Funktion.

Syntax:

steam_get_user_steam_id();

Rückgabe:

Ganzzahl (int64)

Beispiel:

if steam_initialised()
{
global.u_id = steam_get_user_steam_id();
}

Der obige Code setzt eine globale Variable auf die eindeutige Steam-ID des aktuellen Benutzers, wenn die Steam-Client-API korrekt initialisiert ist.
