Dampf

Das Steam-Overlay ist die visuelle Anzeige, die eingeblendet werden kann, um dem Benutzer die Steam-Oberfläche zu zeigen. Dies wird normalerweise vom Benutzer selbst mit einer Tastenkombination durchgeführt, aber Sie haben auch die Möglichkeit es von Ihrem Spiel aus zu tun, so dass Sie eine Schaltfläche oder ein Ereignis zuordnen können, um das Overlay anzuzeigen:

  • steam_is_overlay_enabled
  • steam_is_overlay_activated
  • steam_activate_overlay
  • steam_activate_overlay_browser
  • steam_activate_overlay_store
  • steam_activate_overlay_user
