steam_activate_overlay

Das Steam-Overlay ist ein Teil der Steam-Benutzeroberfläche, der über fast jedem Spiel aktiviert werden kann, das über Steam gestartet wird. Es ermöglicht dem Benutzer den Zugriff auf seine Freundesliste, den Webbrowser, den Chat und den Kauf von DLC im Spiel. Der Standardschlüssel für einen Benutzer um auf das Overlay zuzugreifen, während er sich in einem Spiel befindet, ist SHIFT+TAB, aber Sie können auch jede beliebige Seite des Overlays mit dieser Funktion aufrufen. Sie nimmt eine von sechs Konstanten, die unten aufgeführt sind:

Konstant Beschreibung
ov_friends Die Freunde-Seite für den aktuellen Benutzer
ov_community Die Community-Seite für Ihr Spiel
ov_players Die Seite, auf der andere Spieler angezeigt werden, die das gleiche Spiel spielen oder mit denen Sie kürzlich gespielt haben
ov_Einstellungen Die Overlay-Einstellungen des Steam-Clients
ov_gamegroup Öffnet den Webbrowser der Steam-Community zur offiziellen Spielgruppe für dieses Spiel
ov_Ergebnisse Die Seite mit den Erfolgen für Ihr Spiel

Syntax:

steam_activate_overlay(overlayindex);

Argument Beschreibung
overlayindex Der anzuzeigende Seitenindex der Steam-API-Benutzeroberfläche (siehe unten aufgeführte Konstanten).

Rückgabe:

N/A

Beispiel:

var key = keyboard_lastkey;
switch (key)
{
case vk_f1: steam_activate_overlay(ov_friends); break;
case vk_f2: steam_activate_overlay(ov_community); break;
case vk_f3: steam_activate_overlay(ov_players); break;
case vk_f4: steam_activate_overlay(ov_settings); break;
case vk_f5: steam_activate_overlay(ov_gamegroup); break;
case vk_f6: steam_activate_overlay(ov_achievements); break;
}

Der obige Code fragt die zuletzt gedrückte Tastaturtaste ab und wenn es eine der Funktionstasten von 1 bis 6 ist, wird die entsprechende Seite des Steam-Overlays geöffnet.
