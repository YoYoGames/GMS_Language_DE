steam_activate_overlay_user

Diese Funktion öffnet das Steam-Overlay zu einem der ausgewählten Dialoge, die sich auf die angegebene Benutzer-ID beziehen. Die möglichen Dialogstrings sind:

  • " steamid" - öffnet den Webbrowser der Steam-Community auf der Seite des Benutzers oder der Gruppe
  • " chat"- öffnet ein Chat-Fenster für den angegebenen Benutzer

Beachten Sie, dass Steam-IDs große Zahlen sein können und dass Sie daher Ihren ID-Wert möglicherweise als int64() kodieren, bevor Sie ihn an die Funktion übergeben.

Syntax:

steam_activate_overlay_user(dialog_name, steamid);

Argument Beschreibung
dialog_name Der Dialog, in dem das Overlay geöffnet werden soll (siehe unten).
steamid Die zu verwendende Steam-Benutzer-ID oder Gruppen-ID.

Rückgabe:

N/A

Beispiel:

var key = keyboard_lastkey;
switch (key)
{
case vk_f1: steam_activate_overlay_user(" steamid",global.GameGroupID); break;
case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

Der obige Code fragt die zuletzt gedrückte Tastaturtaste ab, und wenn es die Funktionstaste 1 oder die Funktionstaste 2 ist, wird das Steam-Overlay geöffnet, um entweder die in der globalen Variable "GamegroupID" gespeicherte Steam-Gruppe zu sehen, oder es wird das Chat-Fenster geöffnet, um mit dem Benutzer zu chatten, der in der globalen Variablen "FriendID" gespeichert ist.
