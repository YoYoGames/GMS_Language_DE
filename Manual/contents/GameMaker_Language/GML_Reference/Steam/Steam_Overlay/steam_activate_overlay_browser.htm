steam_activate_overlay_browser

Mit dieser Funktion können Sie das Steam-Spiel-Overlay in seinem Webbrowser öffnen und es dann die angegebene URL laden lassen. Sie müssen die vollständige URL als String verwenden, damit diese korrekt aufgelöst wird, zum Beispiel: "http://www. steamgames.com/".

Syntax:

steam_activate_overlay(url);

Argument Beschreibung
url Die (vollständige) URL für das zu öffnende Overlay.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_f1)
{
steam_activate_overlay_browser("http://www. steamgames.com/");
}

Der obige Code fragt die Tastatur nach der F1-Taste ab, und wenn dies der Fall ist, wird das Steam-Overlay geöffnet und zur angegebenen URL aufgelöst.
