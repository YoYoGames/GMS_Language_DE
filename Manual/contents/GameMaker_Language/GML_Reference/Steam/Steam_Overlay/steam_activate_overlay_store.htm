steam_activate_overlay_store

Mit dieser Funktion können Sie das Steam-Overlay auf der Store-Seite für ein Spiel öffnen, damit Benutzer z. B. DLC kaufen oder herunterladen können. Sie müssen die eindeutige App-ID für das Spiel oder den DLC bereitstellen, die Sie vom Steam-Dashboard erhalten, wenn Sie es einrichten einrichten.

Syntax:

steam_activate_overlay_store(app_id);

Argument Beschreibung
app_id Die eindeutige App-ID für Ihr Spiel.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_f1)
{
steam_activate_overlay_store(global.DLC_id);
}

Der obige Code fragt die Tastatur nach der F1-Taste ab, und wenn dies der Fall ist, wird das Steam-Overlay auf der Seite für den Spielinhalt geöffnet, wobei die in der globalen Variablen gespeicherte App-ID verwendet wird.
