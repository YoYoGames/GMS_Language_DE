steam_is_overlay_activated

Mit dieser Funktion können Sie herausfinden, ob der Benutzer das Steam-Overlay aktiv hat oder nicht. Wenn das Overlay aktiv und für den Benutzer sichtbar ist, gibt die Funktion true zurück, und wenn nicht, dann false. Ein Beispiel dafür, wofür diese Funktion verwendet werden kann, wäre die Abfrage der Steam-API für das Overlay, damit Sie Ihr Spiel anhalten können, während das Overlay angezeigt wird.

Syntax:

steam_is_overlay_activated();

Rückgabe:

Boolesche

Beispiel:

if steam_is_overlay_activated()
{
global.Pause = true;
}

Der obige Code prüft, ob das Steam-Overlay aktiv ist, und wenn ja, setzt er die globale Variable "Pause" auf true.
