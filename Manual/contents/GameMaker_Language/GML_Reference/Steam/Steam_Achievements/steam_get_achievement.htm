steam_get_achievement

Mit dieser Funktion können Sie in der Steam-API prüfen, ob ein bestimmtes Achievement vergeben wurde. Die Errungenschaft sollte zuvor auf der Kontenseite des Steamworks-Kontrollfeldsfür Ihr Spiel definiert worden sein und die Zeichenkette, die an die Funktion übergeben wird übergeben wird, sollte mit dem API-Namen im Bedienfeld übereinstimmen.

Syntax:

steam_get_achievement(ach_name);

Argument Beschreibung
ach_name Der Name der zu erhaltenden Leistung (String).

Rückgabe:

Boolesche

Beispiel:

if hp <= 0
{
global.Deaths += 1;
if global.Deaths == 10
{
if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
}
}

Der obige Code belohnt den Spieler mit einem Achievement, wenn die globale Variable "Deaths" gleich 10 ist und das Achievement noch nicht vergeben wurde.
