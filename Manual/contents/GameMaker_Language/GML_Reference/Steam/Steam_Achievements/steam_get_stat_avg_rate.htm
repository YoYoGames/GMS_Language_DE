steam_get_stat_avg_rate

Mit dieser Funktion können Sie den Wert einer bestimmten Durchschnittsstatistik abrufen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Bedienfeldsfür Ihr Spiel definiert worden sein, und der String, der an die Funktion übergeben wird, sollte mit der als API-Name auf dem Bedienfeld verwendet wird.

Syntax:

steam_get_stat_avg_rate(stat_name);

Argument Beschreibung
stat_name Der Name der abzurufenden Statistik (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

var avg = steam_get_stat_avg_rate("PointsPerHour");
draw_text(8, 8, "PPH = " + string(avg);

Der obige Code ruft den aktuellen Wert für die Durchschnittsstatistik "PointsPerHour" ab und zeichnet ihn auf den Bildschirm.
