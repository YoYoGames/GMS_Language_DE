steam_reset_all_stats_achievements

Mit dieser Funktion können Sie alle Statistiken und Erfolge für den aktuellen Benutzer auf die Standardwerte zurücksetzen (wie in der Steamworks-Systemsteuerungfür Ihr Spiel definiert). Es wird empfohlen, dass Sie diese Funktion nur nur als Debug-Tool bei der Entwicklung Ihres Spiels zu verwenden.

Syntax:

steam_reset_all_stats_achievements();

Rückgabe:

N/A

Beispiel:

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
{
ini_write_real("Daten", "Version", global.Version);
steam_reset_all_stats_achievements();
}
ini_close();

Der obige Code prüft einen gespeicherten Wert in einer Ini-Datei gegen den einer globalen Variablen und setzt, wenn sie unterschiedlich sind, die Statistiken und Erfolge für das Spiel zurück.
