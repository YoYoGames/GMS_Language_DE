Errungenschaften und Statistiken

Die Steam Stats and Achievements API bietet eine einfache Möglichkeit für Ihr Spiel, persistente, roamingfähige Achievement- und Statistikverfolgung für Ihre Benutzer bereitzustellen. Die Daten des Benutzers werden mit seinem Steam-Konto verknüpft, und die Erfolge und Statistiken jedes Benutzers können im Steam-Community-Profil formatiert und angezeigt werden. Statistiken können formatiert und in ihrem Steam-Community-Profil angezeigt werden.

Achievements sind nicht nur eine wertvolle Belohnung für die Spieler Ihrer Spiele, sondern auch nützlich, um Teamwork und Spielerinteraktion zu fördern und zu belohnen, den Spielzielen eine zusätzliche Dimension zu verleihen und die Benutzer dafür zu belohnen, dass sie mehr Zeit im Spiel verbringen. Es ist daher empfehlenswert, dass Ihr Spiel ein paar davon enthält. Sie lassen sich leicht über das Steam-Dashboard einrichten, erfordern aber, dass Sie spezielle Icons für sie erstellen.

Statistiken zeichnen feinkörnige Informationen auf, wie z. B. die Spielzeit, die Anzahl der verwendeten Power-Ups usw. Sie können sie auch nur für die Verfolgung interner Spieldaten verwenden - so können Sie z. B. eine Leistung auf der Basis von Spielstatistiken gewähren, die über mehrere Sitzungen hinweg vom Benutzer gesammelt wurden. Spielstatistiken, die vom Benutzer auf mehreren Computern gesammelt wurden. Oder Sie können interessante Spieldaten für die Anzeige auf der Steam-Community-Seite des Benutzers verfolgen, wo Benutzer ihre eigenen Statistiken mit denen ihrer Freunde vergleichen können.

Es gibt drei Arten von Statistiken - Ganzzahlen, Fließkommazahlen und Durchschnittswerte - und GameMaker Studio 2 erlaubt es Ihnen, jede von ihnen von Ihrem Spiel aus zu ändern, aber sie sollten zuvor Sie sollten jedoch zuvor über das Steamworks-Bedienfeldfür Ihr Spiel initialisiert worden sein.

HINWEIS: Sie müssen warten, bis steam_stats_ready() true zurückgegeben hat, bevor Sie versuchen, Statistiken und Erfolge zu lesen oder zu schreiben.

Wenn sich der Benutzer im Offline-Modus befindet, behält Steam einen lokalen Cache der Statistiken und Achievement-Daten, sodass die APIs wie gewohnt verwendet werden können. Alle Statistiken, die nicht übertragen werden können, werden für das nächste Mal gespeichert, wenn der Benutzer online ist. Für den Fall, dass es Änderungen auf mehr als einem Rechner vorgenommen wurden, führt Steam die Errungenschaften automatisch zusammen und wählt den Satz von Statistiken aus, der mehr Fortschritte gemacht hat. Da Steam einen lokalen Zwischenspeicher für Statistikdaten führt, ist es nicht notwendig, dass das Spiel auch einen lokalen Zwischenspeicher für die Daten auf dem Computer führt. Da Steam einen lokalen Cache der Statistikdaten führt, ist es nicht notwendig, dass das Spiel auch einen lokalen Cache der Daten auf der Festplatte führt, zumal solche Caches oft in Konflikt geraten und es dann für den Benutzer so aussieht, als ob sein Fortschritt rückgängig gemacht wurde, was eine frustrierende Erfahrung ist.
