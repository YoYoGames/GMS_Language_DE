steam_get_stat_float

Mit dieser Funktion können Sie den Wert einer bestimmten Fließkommastatistik abrufen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Bedienfeldsfür Ihr Spiel definiert worden sein, und der String, der an die Funktion übergeben wird, sollte mit dem API-Namen des Bedienfelds übereinstimmen.

Syntax:

steam_get_stat_float(stat_name);

Argument Beschreibung
stat_name Der Name der abzurufenden Statistik (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
steam_set_stat_int("Travelled", dist_pc);
}

Der obige Code berechnet einen Prozentsatz auf der Grundlage der Variable "dist" für die zurückgelegte Strecke und der maximalen Strecke, die Sie zurücklegen können, "dist_max". Er fragt dann den aktuellen Wert für die Statistik "Travelled" ab und wenn er kleiner ist als der berechnete Wert ist, setzt er die Statistik erneut.
