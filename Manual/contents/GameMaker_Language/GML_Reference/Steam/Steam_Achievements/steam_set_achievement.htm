steam_set_achievement

Mit dieser Funktion können Sie der Steam-API mitteilen, ein Achievement für den Spieler zu vergeben ("setzen"). Diese Achievements sollten auf der Kontenseite des Steamworks-Kontrollfeldsfür Ihr Spiel definiert worden sein und die Zeichenkette, die an die Funktion übergeben wird übergeben wird, sollte mit dem API-Namen auf dem Bedienfeld übereinstimmen. Das Steam Game Overlay zeigt dem Benutzer ein Benachrichtigungsfeld an, das ihn über die erhaltene Leistung informiert, es sei denn, die Leistung wurde bereits vergeben, in diesem Fall geschieht nichts. in diesem Fall passiert nichts.

Syntax:

steam_set_achievement(ach_name);

Argument Beschreibung
ach_name Der Name der zu setzenden Leistung (String).

Rückgabe:

N/A

Beispiel:

if hp <= 0
{
global.Deaths += 1;
if global.Deaths == 10
{
if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
}
}

Der obige Code belohnt den Spieler mit einem Achievement, wenn die globale Variable "Deaths" gleich 10 ist und das Achievement noch nicht vergeben wurde.
