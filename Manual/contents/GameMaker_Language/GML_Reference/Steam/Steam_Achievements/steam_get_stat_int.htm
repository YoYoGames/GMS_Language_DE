steam_get_stat_int

Mit dieser Funktion können Sie den Wert einer bestimmten vorzeichenbehafteten Integer-Statistik abrufen. Die Statistik sollte zuvor auf der Kontenseite des Steamworks-Bedienfeldsfür Ihr Spiel definiert worden sein und die Zeichenkette, die an die Funktion übergeben wird, sollte mit dem API-Namen des Bedienfelds übereinstimmen.

Syntax:

steam_get_stat_int(stat_name);

Argument Beschreibung
stat_name Der Name der abzurufenden Statistik (eine Zeichenkette).

Rückgabe:

Real

Beispiel:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int( "Total_XP") > 1000
{
if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

Der obige Code setzt eine Statistik und prüft dann den Endwert dafür, um zu entscheiden, ob eine Leistung vergeben wird oder nicht.
