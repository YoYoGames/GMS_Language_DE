steam_is_cloud_enabled_for_app

Mit dieser Funktion können Sie prüfen, ob der Steam-Cloud-Dienst für Ihr Spiel aktiviert ist. Sie gibt true zurück, wenn dies der Fall ist, und andernfalls false. Beachten Sie, dass dies nicht automatisch bedeutet, dass Sie die Cloud-Funktionen verwenden können verwenden können, da der Benutzer die Cloud-Synchronisierung in seinem Steam-Client ausschalten kann. Sie können dies mit der Funktion steam_is_cloud_enabled_for_account()überprüfen, aber selbst wenn sie für den Benutzer deaktiviert ist Benutzer deaktiviert (und für das Spiel aktiviert) ist, funktionieren die Funktionen zum Speichern und Abrufen von Daten aus einer lokalen Kopie aller Dateien trotzdem, sie werden nur nicht bei Spielende in die Cloud hochgeladen und beim Spielstart nicht synchronisiert.

Syntax:

steam_is_cloud_enabled_for_app();

Rückgabe:

Boolesche

Beispiel:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_total();
}

Der obige Code prüft, ob die Steam-Cloud für das Spiel aktiviert ist, und wenn ja, ermittelt er die Größe des Speicherkontingents und speichert sie in einer Variablen.
