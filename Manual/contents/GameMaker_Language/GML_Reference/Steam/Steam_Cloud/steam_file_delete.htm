dampf_datei_loeschen

Diese Funktion löscht die angegebene Datei aus der Steam Cloud. Die Funktion gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_delete(Dateiname);

Argument Beschreibung
Dateiname Der Name der Datei, die gelöscht werden soll.

Rückgabe:

Real

Beispiel:

if steam_file_exists("Save.txt")
{
steam_file_delete("Save.txt");
}

Der obige Code prüft, ob eine Datei vorhanden ist, und wenn dies der Fall ist, löscht er die Datei aus der Steam-Cloud.
