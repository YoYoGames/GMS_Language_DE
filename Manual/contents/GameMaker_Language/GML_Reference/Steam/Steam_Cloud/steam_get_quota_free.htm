steam_get_quota_free

Mit dieser Funktion können Sie herausfinden, wie viel freier Speicherplatz für den Benutzer der Steam-Cloud-Quota übrig ist. Der zurückgegebene Wert ist in Bytes.

Syntax:

steam_get_quota_free();

Rückgabe:

Real

Beispiel:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_free();
}

Der obige Code prüft, ob die Steam-Cloud für das Spiel aktiviert ist, und wenn ja, ermittelt er die Größe des freien Speicherplatzes und speichert sie in einer Variablen.
