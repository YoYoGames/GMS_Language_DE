steam_file_persisted

Mit dieser Funktion können Sie prüfen, ob die angegebene Datei mit der Steam-Cloud synchronisiert wurde. Ein Rückgabewert von true bedeutet, dass sie es ist, während false bedeutet, dass sie es nicht ist.

Syntax:

steam_file_persisted(Dateiname);

Argument Beschreibung
Dateiname Der Name der zu prüfenden Datei.

Rückgabe:

Boolesche

Beispiel:

if !steam_file_persisted("Save.txt")
{
steam_file_share("Speichern.txt");
}

Der obige Code prüft, ob eine Datei in der Steam-Cloud gespeichert wurde, und wenn nicht, wird sie anschließend synchronisiert.
