steam_file_write

Mit dieser Funktion können Sie Daten in eine Datei schreiben, die dann mit der Steam-Cloud synchronisiert wird, wenn der Benutzer das Spiel verlässt. Wenn die Datei nicht existiert, erstellt diese Funktion sie für Sie, und wenn sie bereits existiert, überschreibt sie alle Daten, die bereits in der Datei gespeichert sind, mit dem neuen Datenstring. Die Funktion gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_write(Dateiname, Daten, Größe);

Argument Beschreibung
Dateiname Der Name der Datei, in die geschrieben werden soll.
Daten Die zu schreibenden Daten (eine Zeichenkette).
Größe die Größe der zu schreibenden Daten.

Rückgabe:

Real

Beispiel:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

Der obige Code bereitet eine Reihe lokaler Variablen vor und verwendet diese dann, um in eine Datei zu schreiben (oder zu erstellen), die dann mit der Steam Cloud synchronisiert wird.
