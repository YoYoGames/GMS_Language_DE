steam_file_exists

Mit dieser Funktion können Sie prüfen, ob eine Datei aus der Steam Cloud existiert oder nicht, mit einem Rückgabewert von true, wenn sie existiert, oder false, wenn nicht.

Syntax:

steam_file_exists(Dateiname);

Argument Beschreibung
Dateiname Der Name der Datei, auf die geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

Der obige Code prüft, ob eine Datei in der Steam-Cloud vorhanden ist, und wenn ja, öffnet er sie und liest ihren Inhalt in die Variable "save_str".
