steam_file_write_file

Mit dieser Funktion können Sie den Inhalt einer lokal gespeicherten Datei in eine Datei kopieren, die mit der Steam-Cloud synchronisiert ist. Die lokale Datei muss vorhanden sein, bevor Sie diese Funktion verwenden, und sie gibt einen Wert von 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_file_write_file(steam_filename, local_filename);

Argument Beschreibung
dampf_dateiname Die zu kopierende Steam Cloud-Datei.
lokaler_dateiname Die lokale Datei, die zum Kopieren verwendet werden soll.

Rückgabe:

real

Beispiel:

steam_file_write_file("rm_koala.png", "Koala2.png");

Der obige Code kopiert den Inhalt der Datei "Koala2.png" in die Steam Cloud-Datei "rm_koala.png".
