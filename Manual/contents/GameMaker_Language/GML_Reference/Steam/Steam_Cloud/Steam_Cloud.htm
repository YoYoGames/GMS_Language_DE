Dampfwolke

Die Steam-Cloud bietet ein einfaches und transparentes Remote-Dateispeichersystem für Ihr Spiel. Alle Dateien, die mit den GameMaker Studio 2-Cloud-Funktionen (unten aufgeführt) auf die Festplatte geschrieben werden, werden nach dem Beenden des Spiels auf die Steam-Server repliziert. Wenn der Benutzer dann den Computer wechselt, werden die Dateien auf den neuen Computer heruntergeladen, bevor das Spiel gestartet wird, sodass das Spiel dann auf die Dateien zugreifen kann, indem es sie mit den entsprechenden Steam-Funktionen liest. Der Steam-Client übernimmt die Arbeit um sicherzustellen, dass die Dateien auf allen Computern, auf die der Benutzer zugreift, synchron gehalten werden.

HINWEIS: Standardmäßig ist die Cloud für ein Spiel auf Steamworks nicht aktiviert. Sie muss zuvor auf der Registerkarte "Cloud" in der Steamworks-Spielverwaltung aktiviert werden, wo Sie die Byte- und Dateiquote einstellen sollten. Wenn Sie das nächste Mal Ihre Spiele auf Steamworkskonfigurieren, ist der Cloud-Speicher bereit zur Verwendung.

Mit den folgenden Funktionen können Sie von GameMaker Studio 2aus auf die Steam Cloud zugreifen

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • dampf_datei_grösse
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • dampf_datei_lesen
  • steam_file_share
  • dampf_datei_loeschen
