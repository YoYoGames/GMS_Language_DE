dampf_datei_lesen

Diese Funktion liest den Inhalt der angegebenen Datei in eine Zeichenkette, die später in Ihrem Spiel geparst werden kann.

Syntax:

steam_file_read(Dateiname);

Argument Beschreibung
Dateiname Der Name der Datei, aus der gelesen werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

Der obige Code prüft, ob eine Datei in der Steam-Cloud vorhanden ist, und wenn ja, öffnet er sie und liest ihren Inhalt in die Variable "save_str".
