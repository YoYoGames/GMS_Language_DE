steam_ugc_query_set_ranked_by_trend_days

Diese Funktion kann verwendet werden, um eine UGC-Abfrage, die mit der Konstante ugc_query_RankedByTrend erstellt wurde, weiter zu filtern, insbesondere um über eine Anzahl von Tagen zu suchen. Das Abfrage-Handle ist der Wert, der zurückgegeben wird, wenn Sie die Abfrage erstellen (z. B. mit, steam_ugc_create_query_user()) und das zweite Argument ist die Anzahl der Tage, über die die Abfrage laufen soll. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, oder andernfalls false.

Syntax:

steam_ugc_query_set_ranked_by_trend_days(ugc_query_handle , days);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfrage-Handle.
Tage Die Anzahl der Tage, die abgefragt werden sollen.

Rückgabe:

Boolesche

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByTrend, ugc_match_Items, 1);
steam_ugc_query_set_ranked_by_trend_days(query_handle, 5);
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Der obige Code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variablen für die spätere Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
