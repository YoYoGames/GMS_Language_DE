steam_is_screenshot_requested

Diese Funktion fragt die Steam-API ab, um zu sehen, ob die Taste zum Erstellen eines Screenshots vom Spiel gedrückt wurde. Die Funktion gibt nur für einen Schritt (Spieltick) true zurück, wenn die Taste gedrückt wurde, und gibt zu allen anderen Zeiten false Zeiten.

HINWEIS: Damit wird kein Screenshot für Sie erstellt. Dies signalisiert nur, dass die Taste gedrückt wurde, und Sie müssen die GameMaker Studio 2-Funktionen screen_save() oder screen_save_part() verwenden, um eine lokale Kopie der hochzuladenden Datei zu speichern.

Syntax:

steam_is_screenshot_requested();

Rückgabe:

Boolesche

Beispiel:

if steam_is_screenshot_requested()
{
var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
screen_save(file)
steam_send_screenshot(file, window_get_width(), window_get_height());
global.scrn_num += 1;
}

Der obige Code fragt die Steam-API nach einer Screenshot-Anfrage ab und wenn dies der Fall ist, wird ein eindeutiger Name für die Bilddatei generiert, ein Screenshot erstellt und die Datei an die Steam-Community-Seite für den Benutzer gesendet.
