steam_ugc_start_item_update

Diese Funktion muss aufgerufen werden, bevor Sie Informationen zu einem UGC-Element hinzufügen oder aktualisieren. Sie müssen die eindeutige App-ID für Ihr Spiel auf Steam angeben, zusammen mit der eindeutigen ID der veröffentlichten Datei, die für das Element zurückgegeben wurde, als Sie es mit der Funktion steam_ugc_create_item(). Die Funktion gibt ein eindeutiges Update-Handle für den Artikel zurück, das Sie dann in den UGC-Artikelfunktionen zum Aktualisieren (oder Hinzufügen) von Informationen zum Hochladen verwenden können.

Syntax:

steam_ugc_start_item_update(consumer_app_id, published_file_id);

Argument Beschreibung
verbraucher_app_id Die eindeutige App-ID für Ihr Spiel auf Steam.
veröffentlichte_datei_id Der eindeutige Wert der veröffentlichten Datei-ID für das Element.

Rückgabe:

Handgriff

Beispiel:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "Mein Werkstattartikel(3)!");
steam_ugc_set_item_description( updateHandle, "Testen workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "Neu";
steam_ugc_set_item_tags( updateHandle, tagArray);
steam_ugc_set_item_preview (updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Der obige Code ruft die Spiel-ID ab und verwendet diese dann zusammen mit einer zuvor gespeicherten veröffentlichten Datei-ID, um ein Update-Handle für das Element zu generieren. Dieses Handle wird dann verwendet, um verschiedene Informationen zu aktualisieren, bevor die Aktualisierung an die Workshop Server übertragen wird.
