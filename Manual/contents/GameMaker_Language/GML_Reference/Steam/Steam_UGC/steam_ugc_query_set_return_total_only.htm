steam_ugc_query_set_return_total_only

Diese Funktion kann verwendet werden, um eine beliebige UGC-Abfrage weiter zu filtern, insbesondere um nur die Anzahl der Ergebnisse ohne weitere Informationen anzufordern (d. h., die von der Sendefunktion erzeugte DS-Map enthält den Schlüssel "num_results" ohne ohne weitere Map-Daten). Das Abfrage-Handle ist der Wert, der beim Erstellen der Abfrage zurückgegeben wurde (z. B. mit steam_ugc_create_query_user()) und das zweite Argument ist entweder true oder falsch. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, andernfalls false.

Syntax:

steam_ugc_query_set_return_total_only(ugc_query_handle , total_only);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfrage-Handle.
gesamt_nur Ob die Abfrage nur die Gesamtanzahl der Treffer zurückgeben soll oder nicht.

Rückgabe:

Boolesche

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_total_only(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Der obige Code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variablen für die spätere Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
