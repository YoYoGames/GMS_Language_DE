steam_ugc_get_subscribed_items

Diese Funktion füllt eine DS-Liste mit allen veröffentlichten Datei-IDs für die Elemente, die der Benutzer aktuell abonniert hat. Sie müssen zuerst die Liste erstellen und den Index in einer Variablen speichern und diese dann an die Funktion übergeben. Die Funktion gibt true zurück, wenn alles korrekt ist und die Steam-API initialisiert ist, oder false, wenn ein Fehler aufgetreten ist.

Syntax:

steam_ugc_get_subscribed_items(item_list);

Argument Beschreibung
Artikel_liste Ein (zuvor erstellter) DS-Listenindex.

Rückgabe:

Boolesche

Beispiel:

steam_list= ds_list_create();
steam_ugc_get_subscribed_items(steam_list);

Der obige Code erstellt eine leere DS-Liste und füllt sie dann mit den Datei-IDs für alle abonnierten Elemente des Benutzers.
