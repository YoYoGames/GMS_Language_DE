steam_ugc_set_item_title

Diese Funktion setzt den Titel, der für das angegebene Element verwendet werden soll. Der Titel sollte ein String sein und das Update-Handle ist der Wert, der beim Aufruf der Funktion steam_ugc_start_item_update(). Die Funktion gibt true zurück, wenn erfolgreich auf die API zugegriffen wurde, und false, wenn es ein Problem gab.

Syntax:

steam_ugc_set_item_title(ugc_update_handle, title);

Argument Beschreibung
ugc_update_handle Das eindeutige Handle für die zu aktualisierende UGC.
Titel Der Titel (eine Zeichenfolge - max. 128 Zeichen), der für das Element verwendet werden soll.

Rückgabe:

Boolesche

Beispiel:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "Mein Werkstattartikel(3)!");
steam_ugc_set_item_description( updateHandle, "Testen workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "Neu";
steam_ugc_set_item_tags( updateHandle, tagArray);
steam_ugc_set_item_preview (updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Der obige Code ruft die Spiel-ID ab und verwendet diese dann zusammen mit einer zuvor gespeicherten veröffentlichten Datei-ID, um ein Update-Handle für das Element zu generieren. Dieses Handle wird dann verwendet, um verschiedene Informationen zu aktualisieren, bevor die Aktualisierung an die Workshop Server übertragen wird.
