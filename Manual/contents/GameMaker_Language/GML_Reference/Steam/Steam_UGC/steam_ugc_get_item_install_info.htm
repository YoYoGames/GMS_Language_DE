steam_ugc_get_item_install_info

Diese Funktion kann verwendet werden, um Informationen über ein beliebiges veröffentlichtes Dateielement abzurufen, das abonniert und in den lokalen Speicherbereich von Steam für Ihr Spiel heruntergeladen wurde. Sie geben die Element-ID an und liefern den Index für eine leere DS-Map, die dann mit den folgenden Schlüssel/Wert-Paaren gefüllt wird, wenn das Element existiert:

  • "size_on_disk" - Die Dateigröße auf der Festplatte (in Bytes)
  • " legacy_item" - Wird true oder false sein , je nachdem, ob es sich um eine Legacy-Datei handelt oder nicht.
  • "folder" - Dies ist der vollständige Pfad zum installierten Inhalt (siehe "Item Installation" in den Steam SDK Docs, da "legacy " Items, die mit der alten Methode hochgeladen wurden, anders behandelt werden).

Wenn das Element existiert, gibt die Funktion true zurück und füllt die Karte auf, andernfalls gibt sie false zurück und die Karte bleibt leer.

Syntax:

steam_ugc_get_item_install_info(published_file_id, info_map);

Argument Beschreibung
veröffentlichte_datei_id Das eindeutige Handle für die zu aktualisierende UGC.
info_map Ein (zuvor erstellter) DS-Map-Index.

Rückgabe:

Boolesche

Beispiel:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

Der obige Code fragt den Installationsstatus des Elements ab, das in der globalen Variablen "fileID" indiziert ist, wobei eine DS-Map zum Speichern der Informationen verwendet wird.
