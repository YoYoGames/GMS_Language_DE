steam_ugc_create_query_all

Mit dieser Funktion können Sie die UGC-Datenbank mit einigen vordefinierten Abfragetypen abfragen. Die Funktion benötigt die folgenden Konstanten für den Typ der zu erstellenden Abfrage, den Typ des abzufragenden Elements und die abzufragende Seitenzahl - beachten Sie, dass eine Abfrage Folgendes zurückgibt eine maximale Anzahl von 50 Elementen zurückgibt.

Abfragetyp Konstante Beschreibung
ugc_query_RankedByVote Siehe die Steam-Dokumentation.
ugc_query_RankedByPublicationDate -
ugc_query_AcceptedForGameRankedByAcceptanceDate -
ugc_Abfrage_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_query_CreatedByFriendsRankedByPublicationDate -
ugc_query_RankedByNumTimesReported -
ugc_query_CreatedByFollowedUsersRankedByPublicationDate -
ugc_query_NotYetRated -
ugc_query_RankedByTotalVotesAsc -
ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch -

Spieltyp Konstante Beschreibung
ugc_match_Items Siehe die Steam-Dokumentation.
ugc_match_Items_Mtx -
ugc_match_Items_ReadyToUse -
ugc_match_Kollektionen -
ugc_match_Artwork -
ugc_match_Videos -
ugc_match_Screenshots -
ugc_match_AllGuides -
ugc_match_WebGuides -
ugc_match_IntegratedGuides -
ugc_match_UsableInGame -
ugc_match_ControllerBindings -



Die Funktion gibt einen eindeutigen Abfrage-Handle-Wert zurück, der in einer Variablen zur Verwendung in den anderen Abfragefunktionen gespeichert werden sollte. Beachten Sie, dass diese Funktion die Abfrage nur vorbereitet, aber nicht tatsächlich sendet - dafür müssen Sie die Funktion steam_ugc_send_query.htm">steam_ugc_send_query() - aufrufen und Sie können weitere steam_ugc_query_* ()-Funktionen verwenden, um die Suchanfrage zu verfeinern, bevor sie tatsächlich gesendet wird.

Syntax:

steam_ugc_create_query_all(query_type, match_type, page);

Argument Beschreibung
abfrage_typ Der Typ der zu erstellenden Abfrage (siehe unten aufgeführte Konstanten).
match_type Der Typ der abzufragenden UGC-Elemente (siehe unten aufgeführte Konstanten).
Seite Die abzufragende Seitennummer.

Rückgabe:

Abfrage Handle

Beispiel:

query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);

Der obige Code erstellt eine allgemeine Abfrage und speichert deren Handle in einer Variablen zur späteren Verwendung.
