steam_ugc_query_set_match_any_tag

Diese Funktion kann verwendet werden, um eine gegebene UGC-Abfrage weiter zu filtern, insbesondere um den Tag-Abgleich ein- oder auszuschalten. Das Abfrage-Handle ist der Wert, der zurückgegeben wird, wenn Sie die Abfrage erstellen (z. B. mit, steam_ugc_create_query_user()) und das zweite Argument ist entweder true oder false, je nachdem, ob Sie eine Prüfung auf übereinstimmende Tags wünschen. Die Funktion gibt true zurück, wenn der Abfragefilter korrekt gesetzt wurde, und andernfalls false.

Syntax:

steam_ugc_query_set_match_any_tag(ugc_query_handle , match_any_tag);

Argument Beschreibung
ugc_query_handle Das zu verwendende Abfrage-Handle.
match_any_tag Legt fest, ob die UGC-Element-Tags mit etwas übereinstimmen sollen oder nicht.

Rückgabe:

Boolesche

Beispiel:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_match_any_tag(query_handle, false);
steam_ugc_query_add_excluded_tag(query_handle, "walking simulator");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Der obige Code erstellt eine Abfrage und speichert deren Handle in einer lokalen Variablen für die spätere Verwendung in den übrigen Funktionen, die die Abfrage vor dem Senden der Abfrage weiter definieren.
