steam_send_screenshot

Mit dieser Funktion können Sie einen Screenshot auf die Steam Community-Profilseite des aktuell eingeloggten Benutzers hochladen. Der Dateiname, den Sie angeben, ist der Name der lokalen Datei, die erstellt wurde, als Sie den Screenshot mit den GameMaker Studio 2-Funktionen screen_save() oder screen_save_part(). Die Breite und Höhe definieren die Bildgröße, und die Funktion wird gibt den Wert 0 zurück, wenn sie aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn sie erfolgreich ist.

Syntax:

steam_send_screenshot(Dateiname, Breite, Höhe);

Argument Beschreibung
Dateiname Der Name der Bilddatei, die hochgeladen werden soll (eine Zeichenkette).
Breite Die Breite des Bildes (eine Ganzzahl).
Höhe Die Höhe des Bildes (eine Ganzzahl).

Rückgabe:

Real

Beispiel:

if steam_is_screenshot_requested()
{
var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
screen_save(file)
steam_send_screenshot(file, window_get_width(), window_get_height());
global.scrn_num += 1;
}

Der obige Code fragt die Steam-API nach einer Screenshot-Anfrage ab und wenn dies der Fall ist, wird ein eindeutiger Name für die Bilddatei generiert, ein Screenshot erstellt und die Datei an die Steam-Community-Seite für den Benutzer gesendet.
