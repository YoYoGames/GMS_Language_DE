steam_ugc_submit_item_update

Diese Funktion sendet den UGC-Artikel, der mit dem angegebenen Handle angegeben wurde, an die Steam Workshop-Server und fügt die Änderungsnotizen hinzu, die für den angegebenen Artikel verwendet werden sollen. Die Änderungsnotizen sollten ein String sein und das Update-Handle ist der Wert, der beim Aufruf der Funktion steam_ugc_start_item_update(). Bei Verwendung dieser Funktion wird ein asynchroner ID-Wert zurückgegeben, der dann geparst werden kann, wenn das asynchrone Ereignis von Steam ausgelöst wird, um die Erstellung des Gegenstands zu melden. Das Ereignis enthält die folgenden Schlüssel/Map-Werte in der async_load DS-Map:

  • "id" - Die asynchrone ID, die von der aufrufenden Funktion zurückgegeben wird
  • "result" - Das Ergebnis der Operation (ein reeller Wert). Dies wird entweder die GML-Konstante ugc_result_success oder eine andere reelle Zahl sein. Sie sollten also auf diese Konstante prüfen, um sicherzustellen, dass der Aufruf erfolgreich war, und wenn andernfalls irgendetwas nicht korrekt funktioniert hat. Der Rest der möglichen Werte, die zurückgegeben werden, wird als Ergebnis des Steam- "EResult"-Wertes angezeigt, und Sie sollten in der Datei steamclientpublic .h in den SDK-Headern alle 89 möglichen Werte finden.
  • " event_type" - Dieser Schlüssel enthält den Wert " ugc_update_item"
  • "legal_agreement_required" - Wird true oder false sein (siehe die Steam-Dokumente für weitere Details)

Syntax:

steam_ugc_submit_item_update(ugc_update_handle, change_note);

Argument Beschreibung
ugc_update_handle Das eindeutige Handle für die zu aktualisierende UGC.
change_note Die Änderungshinweise (eine Zeichenkette), die für das Element verwendet werden sollen.

Rückgabe:

Async-ID

Beispiel:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "Mein Werkstattartikel(3)!");
steam_ugc_set_item_description( updateHandle, "Testen workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "Neu";
steam_ugc_set_item_tags( updateHandle, tagArray);
steam_ugc_set_item_preview (updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Der obige Code ruft die Spiel-ID ab und verwendet diese dann zusammen mit einer zuvor gespeicherten veröffentlichten Datei-ID, um ein Update-Handle für das Element zu generieren. Dieses Handle wird dann verwendet, um verschiedene Informationen zu aktualisieren, bevor die Aktualisierung an die Workshop Server übertragen wird.
