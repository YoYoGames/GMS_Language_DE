steam_create_leaderboard

Mit dieser Funktion können Sie eine neue Rangliste für Ihr Spiel erstellen. Das erste Argument ist eine Zeichenkette, die den Namen Ihrer Bestenliste definiert. Dieser Name sollte in allen weiteren Funktionsaufrufen verwendet werden, die sich auf die zu erstellende Bestenliste beziehen. Sie können können Sie dann die Sortiermethode (siehe die unten aufgeführten Konstanten) sowie die Art der Anzeige der Informationen (wiederum eine der unten aufgeführten Konstanten) festlegen. Die Funktion gibt den Wert -1 zurück, wenn sie die Rangliste nicht erstellen kann, oder einen Wert größer als -1 zurück, wenn sie erfolgreich war.

HINWEIS: Wenn Sie zuvor eine Rangliste mit demselben Namen erstellt haben (entweder durch Code oder über Ihre Steam-Seite für das Spiel), wird diese Funktion ignoriert!

Konstante sortieren Beschreibung
lb_sort_none Keine Sortierung. Die Informationen werden "so wie sie sind" angezeigt.
lb_sort_ascending Sortiert die Rangliste in aufsteigender Reihenfolge.
lb_sort_absteigend Sortiert die Rangliste in absteigender Reihenfolge.

Anzeige Konstante Beschreibung
lb_disp_none Zeigt die Rangliste "wie sie ist".
lb_disp_numerisch Zeigt die Bestenliste als numerische Anzeige an.
lb_disp_zeit_sec Zeigt die Werte der Bestenliste als Zeiten an, wobei der Basiswert Sekunden sind.
lb_disp_zeit_ms Zeigt die Werte der Bestenliste als Zeiten an, wobei der Basiswert Millisekunden sind.

Syntax:

steam_create_leaderboard(lb_name, sort_method, display_type);

Argument Beschreibung
lb_name Der Name der Rangliste, die Sie erstellen (eine Zeichenkette).
sort_method Die Methode zum Sortieren der Ranglisteneinträge (siehe unten aufgeführte Konstanten).
display_type Die Art und Weise, wie die Bestenliste dem Benutzer angezeigt wird (siehe unten aufgeführte Konstanten).

Rückgabe:

Real

Beispiel:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

Der obige Code erstellt eine Bestenliste mit dem Namen "Game Times" und stellt sie so ein, dass die Ergebnisse in aufsteigender Reihenfolge und mit einer Anzeige in Sekunden angezeigt werden.
