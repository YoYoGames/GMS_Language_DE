klickbar_set_style

Mit dieser Funktion können Sie die CSS-Stileigenschaften für die angegebene Schaltfläche über die Schlüssel/Wertpaare in der übergebenen DS-Karte. Sie müssen zuvor sowohl das Button-Element (mit clickable_add()) und die DS-Map erstellt haben und die gespeicherten Indizes zu beiden als Argumente übergeben.

Syntax:

clickable_set_style(index, map)

Argument Beschreibung
index Der Index des anzuklickenden Symbols für den Stil.
Karte Der Index der DS Map, von der aus der Stil gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Der obige Code ändert den Stil der anklickbaren Schaltfläche mit dem Index, der in der Variablen "button" gespeichert ist.
