analyse_ereignis_ext

Diese Funktion sendet den angegebenen Text an den Analyseanbieter, den Sie über die HTML5-Spieloptionen eingerichtet haben. Diese Funktion kann verwendet werden, um eine benutzerdefinierte Analyse zu erstellen, um etwas zu verfolgen Diese Funktion kann verwendet werden, um eine benutzerdefinierte Analyse zu erstellen, um etwas außerhalb des Bereichs des verwendeten Anbieters zu verfolgen, und akzeptiert auch benutzerdefinierte Parameter/Wertpaare, bei denen der Parameter ein String und der Wert eine reale Zahl ist. Für Google Analytics können Sie nur ein zusätzliches Paar hinzufügen, während Flurry bis zu 7.

Syntax:

analytics_event_ext(string, string_param, value);

Argument Beschreibung
String Eine Zeichenkette, die an den Anbieter gesendet wird.
string_param[0 -9] Der zu sendende Parameter (eine Zeichenkette).
Wert[0 - 9] Der Wert des Parameters (eine reelle Zahl).

Rückgabe:

N/A

Beispiel:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Daten", "Spielstände", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Spielstände", num, "Zeit", time);
ini_write_real("Daten", "Spielstände", num);
ini_close();

Der obige Code holt sich Abspielinformationen aus einer Ini-Datei sowie die aktuelle Uhrzeit und sendet diese Angaben dann an den Analyseanbieter.
