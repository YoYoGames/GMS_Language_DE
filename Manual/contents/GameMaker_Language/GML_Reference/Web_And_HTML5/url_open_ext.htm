url_open_ext

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie das HTML5-Modul verwenden, im aktuell geöffneten Browser. Der Parameter "target", den Sie angeben, ist derselbe wie der Standard-JavaScript-Wert "name" Wert, wenn Sie die open() -Methode verwenden, und Sie sollten sich bewusst sein, dass alle außer " _self" dazu führen können, dass der Browser blockiert oder den Benutzer fragt, ob er es zulassen möchte.

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Pop-ups vom Benutzer blockiert werden; in diesem Fall können Sie stattdessen die anklickbar_* Funktionen stattdessen verwenden).
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (dies ist die Voreinstellung).
_Elternteil Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Syntax:

url_open_ext(url, target);

Argument Beschreibung
url Die URL (Website-Adresse), auf die verlinkt werden soll.
Ziel Dies ist der Zielbereich, in dem die URL geöffnet werden soll.

Rückgabe:

N/A

Beispiel:

url_open_ext("http://yoyogames.com", "_blank");

Dies würde die YoYo Games Homepage in einem neuen Fenster öffnen.
