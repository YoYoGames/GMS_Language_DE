webgl_enabled

Diese schreibgeschützte Variable gibt zurück, ob WebGL für Ihr Spiel aktiviert ist(true) oder nicht(false). Sie funktioniert nur für Spiele, die über einen Browser laufen (d. h. HTML5), und für alle anderen Plattformen wird true zurückgegeben.

Syntax:

webgl_enabled

Rückgabe:

Boolesche

Beispiel:

if webgl_enabled
{
global.quality = 1;
}
sonst global.quality = 0;

Der obige Code prüft das WebGL-Flag und setzt dann die globale Variable "quality" entsprechend.
