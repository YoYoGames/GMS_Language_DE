klickbar_ändern_ext

Mit dieser Funktion können Sie das Sprite und die Position eines klickbaren Icons ändern, das zuvor mit clickable_add(). Beachten Sie, dass sich die Position auf die Fenster-, nicht auf die Canvas-Position (0,0) bezieht und dass das Sprite direkt von der Textur-Seite referenziert werden muss (siehe: sprite_get_tpe()). Mit dieser Funktion können Sie auch den Maßstab des verwendeten Sprites ändern zu ändern (als Multiplikator, so dass 1 die Vorgabe ist, 0,5 die Hälfte und 2 das Doppelte wäre) und den Alpha-Wert von 0 (völlig transparent) bis 1 (völlig undurchsichtig) für das endgültige Symbol auf dem Bildschirm.

Syntax:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Argument Beschreibung
index Der Index des klickbaren Symbols, das geändert werden soll.
tpe Der Texturseiteneintrag für das zu verwendende Sprite
x Die neue x-Position innerhalb des Fensters.
y Die neue y-Position innerhalb des Fensters.
Skala Der Maßstab des Symbols (Standardwert 1).
alpha Das Bild-Alpha des Symbols (Standardwert 1).

Rückgabe:

N/A

Beispiel:

switch (room)
{
case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Der obige Code ändert den Bildindex und die Position des anklickbaren Symbols, das in der Variablen "global.Help" indiziert ist, in Abhängigkeit vom aktuellen Raum und ändert auch die Skalierung und das Alpha des Symbols.
