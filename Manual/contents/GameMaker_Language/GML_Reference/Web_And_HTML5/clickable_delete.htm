anklickbar_löschen

Diese Funktion muss verwendet werden, um ein klickbares Symbol, das zuvor mit clickable_add() erstellte klickbare Symbol aus dem Spielfenster zu entfernen.

Syntax:

clickable_delete(index);

Argument Beschreibung
index Index des klickbaren Symbols, das entfernt werden soll.

Rückgabe:

N/A

Beispiel:

if clickable_exists(global.Help_Icon)
{
clickable_delete(global.Help_Icon);
}

Der obige Code entfernt das anklickbare Symbol, das in der globalen Variable "Help_Icon" indiziert ist, aus dem Spielfenster, falls es existiert.
