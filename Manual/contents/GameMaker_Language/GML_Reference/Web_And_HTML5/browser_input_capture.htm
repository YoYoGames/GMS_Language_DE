browser_input_capture

Mit dieser Funktion können Sie einstellen, ob das Browserfenster alle Eingaben erfassen soll (setzen Sie sie auf false) oder ob das Spiel die Eingaben erfassen soll (setzen Sie sie auf true). Beachten Sie, dass diese Funktion nur für die Verwendung mit dem HTML5-Modul vorgesehen ist.

Syntax:

browser_input_capture(enable)

Rückgabe:

N/A

Beispiel:

browser_input_capture(true);

Der obige Code stellt das Spiel so ein, dass es alle Browsereingaben auf der HTML5-Zielplattform erfasst.
