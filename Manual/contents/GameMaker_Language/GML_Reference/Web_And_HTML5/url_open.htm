url_open

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie das HTML5-Modul verwenden, im aktuell geöffneten Browser.

Syntax:

url_open(url);

Argument Beschreibung
url Die URL (Website-Adresse), auf die verlinkt werden soll.

Rückgabe:

N/A

Beispiel:

url_open("http://yoyogames.com");

Dies würde die YoYo Games Homepage im aktuellen Fenster öffnen.
