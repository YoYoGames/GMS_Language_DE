anklickbar_hinzufügen

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spiel-Canvas, aber innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen id-Wert zurück, der in einer Variablen gespeichert werden muss, damit er in allen weiteren Schaltflächenfunktionen verwendet wird, die das neue Symbol referenzieren. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser ein Popup-Fenster erstellt, wenn Sie auf die Schaltfläche klicken, da dies alles durch GameMaker Studio 2 selbst geschieht.

Sie müssen zuerst die Position setzen, die relativ zur (0,0) des Browser-Fensters (linke obere Ecke) ist, und dann den Textur-Seiteneintrag für das Sprite angeben, das das Schaltflächen-Icon haben soll (siehe: sprite_get_tpe()). Sie können dann entweder einen URL angeben, der geöffnet werden soll, oder ein Script (aus dem Asset Browser von GameMaker Studio 2), das geöffnet oder ausgeführt werden soll. Der Parameter "target", den Sie angeben, ist derselbe wie der Standard-JavaScript-Wert "name", wenn Sie Methode open() verwenden (beachten Sie, dass alle Werte außer "_self" dazu führen können, dass der Browser das Öffnen blockiert oder den Benutzer fragt, ob er es zulassen will), und das Argument "params" ist dasselbe wie der JavaScript-Parameter "specs" für der steuert, welche Eigenschaften das neue Fenster/die neue Registerkarte anzeigen soll (nicht alle Browser unterstützen möglicherweise alle Funktionen). Wenn Sie das Symbol verwenden, um ein internes Skript auszuführen, können diese letzten beiden Argumente als leere Strings "" belassen werden.

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte.
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (dies ist die Voreinstellung).
_Elternteil Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Gültige Parameter sind:

Parameter Beschreibung
height=[px]' Die Höhe des Fensters, wobei der Mindestwert 100 ist.
width=[px]' Die Breite des Fensters, wobei der Mindestwert 100 ist.
links=[px]' Die linke Position des Fensters.
'top=[px]' Die obere Position des Fensters (nur IE).
location=[boolean]' Ob das Adressfeld angezeigt werden soll oder nicht (Standard ist 1).
menubar=[boolean]' Ob die Menüleiste angezeigt werden soll oder nicht (Standard ist 1).
resizable=[boolean]' Ob das Fenster in der Größe veränderbar ist oder nicht (Standard ist 1).
'scrollbars=[boolean]' Ob Bildlaufleisten angezeigt werden sollen oder nicht (Standard ist 1).
status=[boolean]' Ob eine Statusleiste hinzugefügt werden soll oder nicht (Standard ist 1).
'titlebar=[boolean]' Ob die Titelleiste angezeigt werden soll oder nicht. Dies wird ignoriert, es sei denn, die aufrufende Anwendung ist eine HTML-Anwendung oder ein vertrauenswürdiges Dialogfeld (Standard ist 1);
toolbar=[boolean]' Ob die Browser-Symbolleiste angezeigt werden soll oder nicht (Standard ist ja).


Es ist auch erwähnenswert, dass, wenn das URL-Argument, das Sie übergeben, der Name einer Skriptfunktion ist (d. h.: es beginnt nicht mit "http://") UND das Skript aufgerufen wird gmcallback_* aufruft, dann können Sie direkte Feeds in ein html5-Ereignis erhalten. GameMaker Studio 2 Scriptfunktionsnamen werden nicht verschleiert, wenn sie mit gmcallback_ beginnen.

Syntax:

clickable_add(x, y, tpe, url, target, params);

Argument Beschreibung
x Die x-Position innerhalb des Fensters.
y Die y-Position innerhalb des Fensters.
tpe Der Texturseiteneintrag für das zu verwendende Sprite.
url Die URL (Website-Adresse), auf die verlinkt werden soll, oder das Skript, das ausgeführt werden soll.
Ziel Dies ist der Zielbereich, in dem die URL geöffnet werden soll (siehe Beschreibung).
Parameter Die verschiedenen Parameter für das Symbol (siehe Beschreibung).

Rückgabe:

Real

Beispiel:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Der obige Code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der das Game Canvas läuft. Das Symbol verwendet das Sprite, das von der Texturseite als "spr_MS_Home" referenziert wird, und wenn das Symbol angeklickt wird, öffnet sich ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster.
