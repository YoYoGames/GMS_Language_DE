klickbar_vorhanden

Diese Funktion gibt zurück, ob ein klickbares DOM-Icon mit dem angegebenen Index existiert oder nicht. Bitte beachten Sie, dass der Wert, der zur Überprüfung verwendet wird, zuvor initialisiert worden sein muss, sonst erhalten Sie einen Fehler, der das GameMaker Studio 2 beendet.

Syntax:

clickable_exists(index);

Argument Beschreibung
index Der Index des zu prüfenden Clickables.

Rückgabe:

Boolesche

Beispiel:

if !clickable_exists(home_but)
{
home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

Der obige Code prüft die Variable "home_but", um zu sehen, ob sie bereits vorhanden ist, und wenn nicht, wird ein klickbares DOM-Symbol an der Position (32, 32) der Seite erstellt, auf der das Game Canvas läuft. Das Symbol verwendet das Sprite, das auf der Das Symbol verwendet das Sprite, das von der Texturseite als "spr_MS_Home" referenziert wird, und wenn es angeklickt wird, wird ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster geöffnet.
