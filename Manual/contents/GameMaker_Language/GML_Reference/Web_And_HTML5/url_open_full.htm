url_open_full

Dadurch wird die angegebene URL im Browser des gewählten Zielgeräts geöffnet, oder, wenn Sie das HTML5-Modul verwenden, im aktuell geöffneten Browser. Der Parameter "target", den Sie angeben, ist derselbe wie der Standard-JavaScript-"name" Wert, wenn Sie die open() -Methode verwenden (beachten Sie, dass alle außer " _self" dazu führen können, dass der Browser blockiert oder den Benutzer fragt, ob er es zulassen möchte) und der "options"-Parameter ist derselbe wie der JavaScript "specs" Parameter zur Steuerung der Eigenschaften, die das neue Fenster/der neue Reiter anzeigen soll (nicht alle Browser unterstützen möglicherweise alle Funktionen).

Gültige Ziele sind:

Ziel Beschreibung
_blank Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Pop-ups vom Benutzer blockiert werden; in diesem Fall können Sie stattdessen die anklickbar_* Funktionen stattdessen verwenden).
_self Öffnet das verknüpfte Dokument in demselben Rahmen, in dem es angeklickt wurde (dies ist die Voreinstellung).
_Elternteil Öffnet das verknüpfte Dokument im übergeordneten Rahmen.
_top Öffnet das verknüpfte Dokument im Volltext des Fensters.

Gültige Optionen sind:

Parameter Beschreibung
height=[px]' Die Höhe des Fensters, wobei der Mindestwert 100 ist.
width=[px]' Die Breite des Fensters, wobei der Mindestwert 100 ist.
links=[px]' Die linke Position des Fensters.
'top=[px]' Die obere Position des Fensters (nur IE).
location=[boolean]' Ob das Adressfeld angezeigt werden soll oder nicht (Standard ist 1).
menubar=[boolean]' Ob die Menüleiste angezeigt werden soll oder nicht (Standard ist 1).
resizable=[boolean]' Ob das Fenster in der Größe veränderbar ist oder nicht (Standard ist 1).
'scrollbars=[boolean]' Ob Bildlaufleisten angezeigt werden sollen oder nicht (Standard ist 1).
status=[boolean]' Ob eine Statusleiste hinzugefügt werden soll oder nicht (Standard ist 1).
'titlebar=[boolean]' Ob die Titelleiste angezeigt werden soll oder nicht. Dies wird ignoriert, es sei denn, die aufrufende Anwendung ist eine HTML-Anwendung oder ein vertrauenswürdiges Dialogfeld (Standard ist 1);
toolbar=[boolean]' Ob die Browser-Symbolleiste angezeigt werden soll oder nicht (Standard ist ja).

Syntax:

url_open_full(url, target, options);

Argument Beschreibung
url Die URL (Website-Adresse), auf die verlinkt werden soll.
Ziel Dies ist der Zielbereich, in dem die URL geöffnet werden soll (siehe Beschreibung).
Optionen Standard-Browser-Optionen (siehe Beschreibung).

Rückgabe:

N/A

Beispiel:

url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0");

Dies würde die YoYo Games-Homepage in einem neuen Fenster öffnen, das nicht in der Größe verändert werden kann, eine Höhe von 200 Pixeln hat und keine Scrollbalken aufweist.
