url_get_domain

Sie können dies verwenden, um die Domain zu erhalten, auf der Ihr HTML5-Spiel gespielt wird.

Syntax:

url_get_domain();

Rückgabe:

String

Beispiel:

dom_name=url_get_domain();

Dadurch wird die Domain Ihres Spiels in der Variablen "dom_name" gespeichert.
