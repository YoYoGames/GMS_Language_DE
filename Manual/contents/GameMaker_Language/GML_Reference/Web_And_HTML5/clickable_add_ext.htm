klickbar_hinzufügen_ext

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spiel-Canvas, aber innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen id-Wert zurück, der in einer Variablen gespeichert werden muss, damit er in allen weiteren Schaltflächenfunktionen verwendet wird, die das neue Symbol referenzieren. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser ein Popup-Fenster erstellt, wenn Sie auf die Schaltfläche klicken, da dies alles durch GameMaker Studio 2 selbst geschieht.

Diese Funktion arbeitet genau wie die clickable_add() Funktion, wobei Sie die Position relativ zur (0,0) des Browser-Fensters setzen müssen, den Textur-Seiteneintrag für das Sprite angeben müssen, das die Schaltfläche Icon haben soll (siehe: sprite_get_tpe()), entweder eine zu öffnende URL oder ein zu öffnendes bzw. auszuführendes Script (aus dem Asset Browser von GameMaker Studio 2) angeben und dann legen Sie das zu verwendende Ziel und die Parameter fest. Diese Funktion hat jedoch auch zwei zusätzliche Optionen, die es Ihnen erlauben, das Icon-Alpha von 0 (vollständig transparent) bis 1 (vollständig undurchsichtig) und die Skalierung des Icons einzustellen, die ein Multiplikatorwert ist, der auf das Sprite angewendet wird. Ein Wert von 1 ist also der Standardmaßstab, während (z. B.) 0,5 der halbe Maßstab wäre und 2 das Doppelte.

Es ist auch erwähnenswert, dass, wenn das URL-Argument, das Sie übergeben, der Name einer Skriptfunktion ist (d. h.: es beginnt nicht mit "http://") UND das Skript aufgerufen wird gmcallback_* dann können Sie direkte Einspeisungen in ein html5-Ereignis erhalten. GameMaker Studio 2 Scriptfunktionsnamen werden nicht verschleiert, wenn sie mit gmcallback_ beginnen.

Syntax:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argument Beschreibung
x Die x-Position innerhalb des Fensters.
y Die y-Position innerhalb des Fensters.
tpe Der Texturseiteneintrag für das zu verwendende Sprite.
url Die URL (Website-Adresse), auf die verlinkt werden soll, oder das Skript, das ausgeführt werden soll.
Ziel Dies ist der Zielbereich, in dem die URL geöffnet werden soll (siehe Beschreibung).
Parameter Die verschiedenen Parameter für das Symbol (siehe Beschreibung).
alpha Das Bild-Alpha des Symbols (Standardwert 1).
Skala Der Maßstab des Symbols (Standardwert 1).

Rückgabe:

Real

Beispiel:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

Der obige Code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der das Game Canvas läuft. Das Symbol verwendet das Sprite, das von der Texturseite als "spr_MS_Home" referenziert wird, und wenn es angeklickt wird, öffnet das Symbol ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster. Die Schaltfläche wird außerdem auf die doppelte Größe des Original-Sprites und ein halbes Bild-Alpha von 1 skaliert.
