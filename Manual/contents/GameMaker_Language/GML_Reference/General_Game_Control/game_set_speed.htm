game_set_speed

Mit dieser Funktion können Sie die Spielgeschwindigkeit einstellen. Sie können diese auf zwei Arten einstellen - entweder als Spielframes pro Sekunde (FPS) oder als Mikrosekunden pro Spielframe (MPF) - unter Verwendung einer der beiden folgenden Konstanten:

Konstant Beschreibung
gamespeed_fps Legt die Spielgeschwindigkeit in Bildern pro Sekunde fest.
gamespeed_microseconds Legt die Spielgeschwindigkeit in Mikrosekunden pro Frame fest.

So würde z. B. eine Spielgeschwindigkeit von 30 Bildern pro Sekunde 33333 Mikrosekunden pro Spielbild entsprechen, was dann durch diese Funktion als

game_set_speed(30, gamespeed_fps);

oder:

game_set_speed(33333, gamespeed_microseconds);

Syntax:

game_set_speed(speed, type);

Argument Beschreibung
Geschwindigkeit Die neue Spielgeschwindigkeit (entweder als FPS oder MPF).
Typ Der Typ der Methode, die zum Einstellen der Spielgeschwindigkeit verwendet wird (siehe die Konstanten oben).

Rückgabe:

N/A

Beispiel:

if os_browser == browser_not_a_browser
{
game_set_speed(60, gamespeed_fps);
}
sonst
{
game_set_speed(30, gamespeed_fps);
}

Der obige Code prüft, ob das Spiel in einem Browser läuft und setzt die Spielgeschwindigkeit entsprechend als FPS-Wert.
