spiel_projekt_name

Diese schreibgeschützte Variable gibt den Anzeigenamen Ihres Spiels für die Zielplattform in einem "speicherfreundlichen" Format für die Zielplattform zurück. Wenn der Anzeigename Zeichen enthält, die für einen Dateinamen nicht zulässig sind, werden diese werden diese automatisch durch "_" ersetzt. Der Anzeigename kann in den Spieloptionen eingestellt werden. Beachten Sie, dass, da es keine Einschränkungen für Dateinamen für HTML5 gibt, diese Zeichenkette wahrscheinlich die gleiche sein wird wie die von spiel_anzeige_name.

Syntax:

spiel_projekt_name

Rückgabe:

Zeichenfolge

Beispiel:

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

Der obige Code ruft den Projektnamen ab und verwendet diesen, um eine ini-Datei zu öffnen (oder zu erstellen), in die dann geschrieben wird.
