highscore_add

Mit dieser Funktion können Sie einen Namen und einen Punktestand in die interne globale Highscore-Liste aufnehmen. Sie brauchen nicht zu prüfen, ob der Wert hoch genug ist, um ihn in die Highscore-Liste aufzunehmen, da GameMaker Studio 2 nur nur die Werte speichert, die größer sind als die zehnte gespeicherte Position.

Syntax:

highscore_add(str, numb);

Argument Beschreibung
str Die Zeichenkette (Name), der die Partitur zugewiesen werden soll
taub Die zu addierende Zahl (Punktzahl).

Rückgabe:

N/A

Beispiel:

if lives = 0
{
highscore_add(global.Name, score);
score = 0;
room_goto(rm_MainMenu);
}

Der obige Code prüft die Leben und fügt, wenn sie auf 0 gesetzt sind, den aktuellen Punktestand und die in der globalen Variablen "Name" gespeicherte Zeichenkette in die Highscore-Liste ein, bevor er schließlich den Punktestand auf 0 setzt und den Spieler in den Raum der in "rm_MainMenu" indiziert ist.
