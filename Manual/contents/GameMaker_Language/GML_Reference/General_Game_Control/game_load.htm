Spiel_laden

Mit dieser Funktion können Sie ein Spiel laden, das zuvor mit game_save(). Beachten Sie, dass damit die Version des Spiels wiederhergestellt wird, die zum Erstellen des Speichers verwendet wurde, so dass spätere Aktualisierungen nicht mehr sichtbar sind. sichtbar. Für weitere Informationen lesen Sie bitte die Seite über game_save().

Syntax:

game_load(Dateiname);

Argument Beschreibung
Dateiname Der Name der zu ladenden Datei.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("L"))
{
if global.Save game_load("Save.dat");
}

Der obige Code lädt ein zuvor gespeichertes Spiel, wenn die globale Variable true ist, wenn der Spieler die Taste "L" drückt.
