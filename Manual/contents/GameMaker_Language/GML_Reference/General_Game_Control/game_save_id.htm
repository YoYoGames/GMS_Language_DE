game_save_id

Diese schreibgeschützte Variable gibt die vollständige Pfad-ID des Verzeichnisses zurück, das von Ihrem Spiel zum Speichern von Dateien verwendet wird. Dieses Verzeichnis kann je nach Plattform für andere Anwendungen sichtbar sein oder nicht, und auf dem HTML5-Ziel gibt sie eine leere Zeichenfolge zurück.

HINWEIS: Diese Variable dient nur zur Information und sollte nicht verwendet werden, um einen Dateipfad oder etwas Ähnliches zu erhalten, da sie nicht auf allen Plattformen korrekt funktioniert.

Syntax:

game_save_id

Rückgabe:

Zeichenfolge

Beispiel:

save_dir = game_save_id;

Dadurch wird das Verzeichnis zum Speichern von Dateien für die zukünftige Verwendung in einer Variablen gespeichert.
