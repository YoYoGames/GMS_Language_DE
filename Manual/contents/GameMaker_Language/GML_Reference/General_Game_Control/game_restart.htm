Spiel_Neustart

Mit dieser Funktion können Sie das Spiel neu starten. Dies ist im Wesentlichen dasselbe wie das erstmalige Starten des Spiels, so dass das Game Start Event und das Game End Event ausgelöst werden. Es ist jedoch zu beachten, dass bestimmte Dinge nicht zurückgesetzt werden, wenn diese Funktion aufgerufen wird:

  • Globale Variablen werden nicht neu initialisiert, es sei denn, sie sind explizit als solche kodiert - zum Beispiel wird die eingebaute globale Variable score nach einem Neustart des Spiels nicht bei Null beginnen, wenn sie im Spiel bereits geändert wurde.
  • Der GPU-Zustand wird nicht geändert (wenn Sie also z. B. die Zeichenfarbe oder Alpha eingestellt haben, bleibt es bei dem geänderten Wert).
  • Die Spielgeschwindigkeit bleibt auf dem Wert, den Sie in Ihrem Spielcode eingestellt haben (wenn Sie ihn geändert haben, wird diese Änderung beibehalten).
  • Jedes Asset aus dem Asset-Browser, das zur Laufzeit im Spiel geändert wurde - zum Beispiel, wenn Sie den Ursprung für eine Sprite-Ressource ändern oder die Position einer Pfad-Ressource verschieben - wird nicht zurückgesetzt.
  • Dynamische Ressourcen wie Puffer, Oberflächen, Datenstrukturen oder importierte Sprites werden ebenfalls nicht aufgeräumt oder entfernt (obwohl Sie möglicherweise Referenzen auf sie verlieren, achten Sie also bei der Verwendung dieser Funktion darauf, entweder globale Referenzen für die dynamische Ressource zu verwenden, oder sie vor dem Aufruf der Funktion aufzuräumen).

Syntax:

game_restart();

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("R")) game_restart();

Dies würde das Spiel neu starten, wenn der Spieler die Taste "R" drückt.
