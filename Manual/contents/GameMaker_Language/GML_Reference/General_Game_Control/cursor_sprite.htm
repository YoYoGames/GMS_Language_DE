cursor_sprite

Das Setzen dieser Variable weist GameMaker Studio 2 an, das angegebene Sprite als Cursor zu verwenden (im Grunde wird es bei jedem Schritt auf die aktuelle x/y-Position der Maus gesetzt). Der Standardwert ist -1, also kein Sprite für den Cursor ist, aber Sie können einen beliebigen Sprite-Index aus den Spiel-Assets zuweisen oder einen, der aus einer externen Ressource importiert wurde. Bitte beachten Sie, dass es keine Möglichkeit gibt, die Animationsgeschwindigkeit oder den image_index zu steuern; wenn das Sprite also Unterbilder hat, werden diese mit der gleichen Geschwindigkeit wie die Raumgeschwindigkeit durchlaufen. Um das Cursor-Sprite zu entfernen, können Sie diese Variable wieder auf -1 setzen.

Es ist auch erwähnenswert, dass diese Variable nicht den Cursor des Spielfensters ersetzt, sondern dass dieser weiterhin normal gezeichnet wird. Um dies zu vermeiden, können Sie die Funktion window_set_cursor() und verwenden. auf die Konstante cr_none setzen, wodurch der Standardcursor unsichtbar wird.

Syntax:

cursor_sprite;

Rückgabe:

Real

Beispiel:

cursor_sprite = spr_CustomCursor;

Der obige Code setzt das Sprite, das in der Variablen "spr_CustomCursor" indiziert ist, als Cursor-Sprite für das Spiel.
