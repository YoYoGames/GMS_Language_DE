highscore_name

Mit dieser Funktion können Sie die Namenszeichenfolge abrufen, die in der Highscore-Liste an der angegebenen Position gespeichert wurde. Wenn kein Name eingegeben wurde, wird die Zeichenfolge "Unbekannt" zurückgegeben.

Syntax:

highscore_name(Ort);

Argument Beschreibung
Ort Der Platz auf dem Tisch (1-10).

Rückgabe:

Zeichenfolge

Beispiel:

var i = 9;
repeat(10)
{
name[i] = highscore_name(i + 1);
i -= 1;
}

Der obige Code läuft in einer Schleife durch die Highscore-Liste und speichert alle Namen in einem Array.
