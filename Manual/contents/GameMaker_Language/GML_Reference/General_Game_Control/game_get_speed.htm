spiel_get_speed

Diese Funktion kann verwendet werden, um die Spielgeschwindigkeit entweder als Anzahl der Spielframes, die pro Sekunde ausgeführt werden, oder als Anzahl der Mikrosekunden pro Spielframe zu erhalten. Beachten Sie, dass dies nicht der tatsächliche FPS-Wert für die Spielgeschwindigkeit ist (verwenden Sie dafür die fps_real Variable) sondern vielmehr die Anzahl der Spielframes (FPS), die das Spiel pro Sekunde zu halten versucht, oder die Länge jedes Spielframes in Mikrosekunden, die das Spiel zu halten versucht (MPF). Wenn Sie diese Funktion verwenden, müssen Sie eine der folgenden Konstanten angeben, die den Typ des Rückgabewerts bestimmen:

Konstant Beschreibung
gamespeed_fps Ruft die Spielgeschwindigkeit in Bildern pro Sekunde ab.
gamespeed_microseconds Ruft die Spielgeschwindigkeit in Mikrosekunden pro Frame ab.

Wenn zum Beispiel die Spielgeschwindigkeit in den Spieloptionen auf 30 eingestellt ist und Sie den FPS-Typ verwenden, dann gibt die Funktion 30 zurück, aber wenn Sie den MPF-Typ verwenden, dann gibt die Funktion 33333 zurück.

Syntax:

game_get_speed(type);

Argument Beschreibung
Typ Der Typ der Methode, die verwendet wird, um die Spielgeschwindigkeit zu erhalten (siehe die Konstanten oben).

Rückgabe:

Real

Beispiel:

if game_get_speed(gamespeed_fps) != 60
{
game_set_speed(60, gamespeed_fps);
}

Der obige Code prüft, ob das Spiel mit einer Spielgeschwindigkeit von 60 FPS läuft und wenn nicht, wird es auf 60 FPS gesetzt.
