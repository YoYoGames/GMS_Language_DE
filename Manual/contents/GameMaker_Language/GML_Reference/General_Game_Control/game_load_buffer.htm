spiel_laden_puffer

Mit dieser Funktion können Sie einen Spielzustand laden, der zuvor gespeichert wurde. Das Spiel wird aus einem zuvor erstellten "grow"-Puffer (siehe Puffer) geladen und in diesem Puffer muss ein Spielzustand gespeichert worden sein mit game_save_buffer() Funktion.

Syntax:

game_load_buffer(buffer);

Argument Beschreibung
Puffer Die Puffer-ID, aus der geladen werden soll.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("L"))
{
if global.Checkpoint game_load_buffer(save_buff);
}

Der obige Code lädt einen zuvor gespeicherten Spielzustand aus dem Puffer, der in der Variablen "save_buff" indiziert ist, nur dann, wenn die globale Variable true ist, wenn der Spieler die Taste "L" drückt.
