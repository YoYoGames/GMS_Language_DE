spiel_id

Diese schreibgeschützte Variable gibt den eindeutigen Bezeichner für das von Ihnen erstellte Spiel zurück. Sie können dies verwenden, wenn Sie einen eindeutigen Dateinamen benötigen oder irgendetwas anderes, das nur zur Identifizierung Ihres Spiels dient. Dies kann in den Spieloptionen eingestellt werden.

Syntax:

spiel_id

Rückgabe:

Real

Beispiel:

ini_open("Score.ini");
ini_write_real("Scores", "0",score + game_id);
ini_close();

Der obige Code führt eine sehr einfache Verschlüsselung des Spielstands durch, indem er vor dem Speichern die game_id hinzufügt. Beim Zurücklesen in das Spiel würden Sie die game_id abziehen, um wieder den richtigen Wert zu erhalten.
