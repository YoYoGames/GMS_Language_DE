Spiel_Anzeige_name

Diese schreibgeschützte Variable gibt den Anzeigenamen Ihres Spiels für die Zielplattform zurück, wie in den Spieloptionen festgelegt.

Syntax:

Spiel_Anzeige_name

Rückgabe:

Zeichenfolge

Beispiel:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

Der obige Code ruft den Anzeigenamen und die Versionsnummer des Spiels ab und zeichnet sie ein.
