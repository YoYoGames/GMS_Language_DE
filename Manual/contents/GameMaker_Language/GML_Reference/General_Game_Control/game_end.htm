Spiel_Ende

Mit dieser Funktion können Sie das Spiel beenden (und das Game End Event wird ausgelöst). Dies geschieht nicht sofort, sondern erst am Ende des aktuellen Schritts, so dass jeder Code den Sie im selben Schritt haben, nachdem diese Funktion aufgerufen wurde, noch ausgeführt wird. Bitte beachten Sie, dass diese Funktion die folgenden Einschränkungen hat:

  • Auf Android-Geräten schiebt der Aufruf von game_end() die App in den Hintergrund, schließt sie aber nicht. Dies muss vom Benutzer durchgeführt werden.
  • Unter iOS wird es nichts tun und stillschweigend fehlschlagen.
  • Auf allen Konsolen (einschließlich UWP, wenn es auf einer Xbox läuft) kann es das Spiel zum Absturz bringen oder bestenfalls stillschweigend fehlschlagen, und es kann auch ein Übermittlungsfehler sein, daher darf diese Funktion nicht verwendet werden.
  • Auf HTML5 wird das Spiel beendet, aber der Benutzer starrt auf eine leere Zeichenfläche auf der Webseite, und als solches wird nicht empfohlen, dass es auf dieser Zielplattform verwendet wird.
  • Unter Windows (einschließlich UWP auf einem PC), Linux und macOS beendet die Funktion einfach das Spiel und schließt das Spielfenster (das Game End Event wird ebenfalls ausgelöst).

Syntax:

game_end();

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_escape) game_end();

Dies würde das Spiel beenden, wenn der Spieler die "Escape"-Taste drückt.
