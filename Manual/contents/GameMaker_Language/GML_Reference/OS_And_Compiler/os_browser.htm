os_browser

Diese schreibgeschützte Variable enthält eine von verschiedenen Konstanten (siehe unten), die GameMaker Studio 2 zur Verfügung stehen, um Ihnen mitzuteilen, in welchem Browser Sie das Spiel gerade ausführen (falls vorhanden).

Syntax:

os_browser

Rückgabe:

Konstant

Konstant Beschreibung
browser_not_a_browser Das Spiel wird nicht in einem Browser gespielt
browser_unbekannt Unbekannter Browser
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer auf einem mobilen Gerät
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari auf einem mobilen Gerät
browser_opera Oper
browser_tizen Browser für Tizen-Mobilgeräte
browser_windows_store Windows-App

Beispiel:

if (os_browser == browser_not_a_browser)
{
global.Config = 0;
}
sonst
{
global.Config = 1;
}

Der obige Code prüft, ob das Spiel in einem Browser läuft oder nicht, und setzt eine globale Variable auf einen Wert, der vom Ergebnis der Prüfung abhängt.
