umwelt_get_variable

Diese Funktion gibt den Wert (eine Zeichenkette) der Umgebungsvariablen mit dem angegebenen Namen (ebenfalls eine Zeichenkette) zurück. Sie können die verfügbaren Umgebungsvariablen unter macOS und Ubuntu(Linux) erhalten, indem Sie " env" in die Terminal-App eingeben, und für Informationen über Windows-Umgebungsvariablen erhalten Sie, wenn Sie die Eingabeaufforderung verwenden, indem Sie " echo%PATH%" eingeben, und mit PowerShell ist es " ls env". Beachten Sie, dass sowohl unter macOS als auch unter Ubuntu(Linux) die Umgebungsvariable "HOME" Umgebungsvariable den Pfad " ~/" zurückgibt, der unter macOS auf " /Users/<Benutzername>" und unter Ubuntu (Linux) auf" /home/<Benutzername> " abgebildet wird.

HINWEIS: Diese Funktion ist nur für Windows, macOS und Ubuntu(Linux) verfügbar.

Syntax:

environment_get_variable(name);

Argument Beschreibung
Name Der Name (eine Zeichenkette) der zu prüfenden Umgebungsvariablen.

Rückgabe:

Zeichenfolge

Beispiel:

e_str = environment_get_variable("APPDATA");

Der obige Code gibt den vollständigen Pfad für die Umgebungsvariable " %appdata%" zurück, der normalerweise " C:\Benutzer\{Benutzername}\AppData\Roaming" lautet.
