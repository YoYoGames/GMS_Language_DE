Os_Typ

Diese schreibgeschützte Variable enthält eine von verschiedenen Konstanten, die GameMaker Studio 2 zur Verfügung stehen, um Ihnen mitzuteilen, für welches Betriebssystem das Spiel erstellt wurde. Beachten Sie, dass dies nicht unbedingt mit dem Betriebssystem des Geräts übereinstimmt, auf dem da - zum Beispiel - Ihr Spiel auf einem Amazon Fire OS laufen könnte, aber für die Android-Plattform erstellt wurde (in diesem Fall ist os_type os_android).

Die folgenden Konstanten können zurückgegeben werden:

Konstant Beschreibung
os_windows Windows Betriebssystem
os_uwp Windows 10 Universelle Windows-Plattform
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos Apple tvOS
os_android Android
os_ps4 Sony PlayStation 4
os_xboxone Microsoft XBox One
Os_Schalter Nintendo Switch
os_unbekannt unbekanntes OS

Syntax:

os_type;

Rückgabe:

Konstant

Beispiel:

switch (os_type)
{
case os_windows: global.Config = 0; break;
case os_android: global.Config = 1; break;
case os_linux: global.Config = 2; break;
case os_macosx: global.Config = 3; break;
case os_ios: global.Config = 4; break;
case os_winphone: global.Config = 5; break;
}

Der obige Code überprüft das Betriebssystem, auf dem das Spiel läuft, und setzt eine globale Variable entsprechend.
