Parameter_Zeichenfolge

Befehlszeilenparameter sind die zusätzlichen Befehle, die Sie einer Exe hinzufügen können, um zu ändern, wie das Programm ausgeführt wird, und mit dieser Funktion können Sie den gewählten Befehlszeilenparameter als String erhalten. Sie können die Anzahl der Parameter für das aktuelle Spiel ermitteln mit der Funktion parameter_count()ermitteln, wobei der erste Parameter den Index 1 und der letzte den von der Funktion zurückgegebenen Index hat (ein Wert von 0 ist insofern besonders, als dass es sich um den Dateinamen der ausführbaren Datei des Spiels handelt, einschließlich des Pfades). Es sollte beachtet werden, dass diese Funktion für auf der HTML5-Plattform funktioniert und die url-Parameter abruft.

Syntax:

parameter_string(n);

Rückgabe:

Zeichenfolge

Beispiel:

var p_num;
p_num = parameter_count();
if p_num > 0
{
var i;
for (i = 0; i < p_num; i += 1)
{
p_string[i] = parameter_string(i + 1);
}
}

Der obige Code ermittelt die Anzahl der Befehlszeilenparameter, und wenn es 1 oder mehr sind, wird er sie in einer Schleife durchlaufen und als Zeichenketten in einem Array speichern.
