externer_Aufruf

Wenn Sie einen externen Funktionsaufruf zu einer dll oder dylib mit external_define()definiert haben, können Sie diese Funktion anschließend mit dieser Funktion aufrufen. Sie geben den Namen der zuvor definierten Funktion sowie sowie jedes der benötigten Argumente (jedes Argument muss vom richtigen Typ sein, entweder real oder string) und die Funktion gibt das Ergebnis des externen Aufrufs zurück.

Syntax:

external_call(id, args[0...15]);

Argument Beschreibung
id Der Name der Funktion, die Sie aufrufen möchten
args[0...10] Die verschiedenen Argumente, die Sie an die externe Funktion durchreichen wollen

Rückgabe:

Wert (der Typ des zurückgegebenen Wertes ist abhängig von der definierten Funktion)

Beispiel:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Der obige Beispielcode ruft eine zuvor definierte externe Funktion auf und speichert den zurückgegebenen Wert in einer lokalen Variablen.
