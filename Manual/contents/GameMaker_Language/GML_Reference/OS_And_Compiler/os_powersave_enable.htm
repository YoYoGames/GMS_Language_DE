os_powersave_enable

Mit dieser Funktion können Sie die Energiesparfunktionen des Geräts ein- oder ausschalten. Dies ist wichtig, da bestimmte Spiele (z. B. solche, die die Neigungsfunktionen verwenden) möglicherweise keine Ereignisse erzeugen, die das Betriebssystem als Benutzereingabe interpretieren kann und so den Bildschirm herunterfahren den Bildschirm abschalten oder das Spiel beenden. Indem Sie diese Funktion auf false setzen, können Sie die Energiesparfunktionen deaktivieren und sicherstellen, dass der Bildschirm (und das Spiel) immer funktionieren.

HINWEIS: Dies ist auf iOS- und Android-Ziele beschränkt.

Syntax:

os_powersave_enable(flag)

Argument Beschreibung
Flagge Wird auf true oder false gesetzt, um die Powersave-Funktionen zu aktivieren oder zu deaktivieren.

Rückgabe:

N/A

Beispiel:

if (os_type == os_android) || (os_type == os_ios)
{
os_powersave_enable(false);
}

Der obige Code prüft den OS-Typ und wenn es entweder Android oder ein iOS ist, werden die Energiesparfunktionen deaktiviert.
