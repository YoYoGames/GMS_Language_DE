GM_build_date

Diese Konstante enthält das Datum und die Uhrzeit, zu der die ausführbare Datei, die ausgeführt wird, von GameMaker Studio 2 erstellt wurde. Der Wert kann mit den Funktionen "Datum" und "Uhrzeit" geparst werden und wird dem UTC-Wert des Systems zur Kompilierungszeit entnommen.

Syntax:

GM_build_date;

Rückgabe:

Real

Beispiel:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

Der obige Code zeichnet die Versionsnummer für das Spiel und das Datum und die Uhrzeit, zu der es kompiliert wurde.
