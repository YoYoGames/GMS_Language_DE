GM_Laufzeit_Version

Diese Konstante enthält die Versionsnummer der Runtime, wie sie in den Voreinstellungen als die Runtime definiert ist, die zum Erstellen des Projekts verwendet wird. Der Wert wird als eine Zeichenkette.

Syntax:

GM_Laufzeit_Version;

Rückgabe:

Zeichenfolge

Beispiel:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, " Runtime " + GM_runtime_version);

Der obige Code nimmt die GM-Datum-, Versions- und Laufzeitkonstanten und zeichnet sie auf den Bildschirm.
