os_get_language

Diese Funktion gibt eine Zeichenkette mit dem zweibuchstabigen Sprachcode für das Betriebssystem zurück, auf dem das Spiel läuft, wie im ISO639-Standard festgelegt. Wenn die Information nicht verfügbar ist, enthält sie einfach eine leere Zeichenkette "", oder "en" für die Sprache "Englisch". Beachten Sie, dass einige Sprachen auch einen entsprechenden Regionalcode haben. Um also zwischen verschiedenen Regionen desselben Landes zu unterscheiden, verwenden Sie die Funktion os_get_region().

Die folgende Tabelle zeigt Beispiele für einige der wichtigsten zweibuchstabigen Sprachcodes, wie sie in ISO 639 definiert sind:

Sprache Code
Arabisch ar
Chinesisch zh
Dänisch da
Englisch en
Französisch fr
Deutsch de
Griechisch el
Italienisch es
Japanisch ja
Norwegisch keine
Polnisch pl
Portugiesisch pt
Russisch ru
Spanisch es
Schwedisch sv

HINWEIS: Dies ist nicht der Ländercode des Standorts, der zurückgegeben wird, sondern der Sprachcode des Betriebssystems.

Syntax:

os_get_language()

Rückgabe:

Zeichenfolge

Beispiel:

switch (os_get_language())
{
case "es": ini_open("spanisch.ini"); break;
case "fr": ini_open("französisch.ini"); break;
case "it": ini_open("italienisch.ini"); break;
default: ini_open("englisch.ini"); break;
;

Der obige Code prüft die Betriebssystemsprache und öffnet je nach dem zurückgegebenen Wert eine andere *.ini-Datei.
