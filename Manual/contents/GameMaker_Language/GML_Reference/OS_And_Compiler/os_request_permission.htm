os_check_permission

Mit dieser Funktion können Sie eine bestimmte Berechtigung anfordern. Sie geben die anzufordernde Berechtigung als String im Format " android.permission.<Permission> " an, so dass Sie zum Beispiel die Berechtigung RECORD_AUDIO anfordern würden Sie aufrufen

os_request_permission("android.permission.RECORD_AUDIO");

Die Funktion löst ein asynchrones Systemereignis aus, wobei die eingebaute async_load DS-Map die folgenden Schlüssel/Wertpaare enthalten wird:

  • " Typ": Dies ist der Ereignistyp, der ausgelöst wurde, und der Wert ist die Zeichenfolge "permission_request_result".
  • " <Erlaubnis>": Hier wird die von der Funktion angeforderte Berechtigung als String angegeben, der eine der folgenden Konstanten enthält:
    Konstante Beschreibung
    os_erlaubnis_erteilt Zeigt an, dass die Berechtigung erteilt wurde
    os_permission_denied Zeigt an, dass die Berechtigung nicht erteilt wurde

Es ist erwähnenswert, dass die folgenden Berechtigungen von GameMaker Studio 2 nativ unterstützt werden, aber von Google als "gefährlich" eingestuft werden und als solche explizit angefordert werden müssen (beachten Sie auch, dass einige Berechtigungen über den Android-Spieloptionen angefordert werden können, ohne dass diese Funktion erforderlich ist):

  • android.permission.WRITE_EXTERNAL_STORAGE
  • android.permission.READ_PHONE_STATE
  • android.permission.RECORD_AUDIO

Weitere Informationen zu App-Berechtigungen finden Sie in der Android-Dokumentation.

WICHTIG! Diese Funktion ist nur für das Android nur für das Ziel.

Syntax:

os_request_permission(erlaubnis)

Argument Beschreibung
Genehmigung Die anzufordernde Berechtigung (eine Zeichenkette)

Rückgabe:

N/A

Beispiel:

if (os_type == os_android)
{
if (os_check_permission("android.permission.INTERNET") == os_permission_denied)
{
os_request_permission("android.permission.INTERNET");
}
}

Der obige Code prüft den Betriebssystemtyp und wenn es sich um Android handelt, führt er eine Überprüfung der Berechtigungen durch und fordert die Berechtigung "INTERNET" an, wenn sie noch nicht erteilt wurde.
