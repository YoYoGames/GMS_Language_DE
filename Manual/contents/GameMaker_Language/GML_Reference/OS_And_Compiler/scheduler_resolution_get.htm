scheduler_resolution_get

Mit dieser Funktion wird die Auflösung des Windows-Thread-Schedulers in Millisekunden abgerufen. Wenn die Auflösung des Schedulers auf den Standardwert gesetzt ist (wie von Windows eingestellt ), gibt die Funktion -1 zurück.

Informationen über den Thread-Scheduler finden Sie auf der Seite für scheduler_resolution_set().

HINWEIS: Diese Funktion ist für Windows nur.

Syntax:

scheduler_resolution_get();

Rückgabe:

Real (oder -1 für Standard)

Beispiel:

scheduler_resolution = scheduler_resolution_get();

Dieses Beispiel ruft die Auflösung des Windows-Thread-Schedulers ab und speichert sie in der Variablen "scheduler_resolution".
