Parameter_Zahl

Befehlszeilenparameter sind die zusätzlichen Befehle, die Sie einer Exe hinzufügen können, um zu ändern, wie das Programm ausgeführt wird. Sie können die Anzahl der Parameter für das aktuelle Spiel mit dieser Funktion ermitteln, wobei der erste Parameter den Index 1 hat und der letzte der von der Funktion zurückgegebene Index ist (ein Wert von 0 ist insofern besonders, als dass es sich um den Dateinamen der ausführbaren Datei des Spiels handelt, einschließlich des Pfads). Es sollte beachtet werden, dass diese Funktion für auf der HTML5-Plattform arbeiten wird, Abrufen der url-Parameter.

Syntax:

parameter_count();

Rückgabe:

Real

Beispiel:

p_num = parameter_count();

Der obige Code speichert die Anzahl der Befehlszeilenparameter, die für das Spiel verwendet wurden, in der Variablen "p_num".
