os_lock_orientation

Mit dieser Funktion können Sie Ihr Gerät in der aktuellen Ausrichtung "sperren", bis Sie es "freigeben", um alle in den Spieloptionen aktivierten Ausrichtungen für diese Zielplattform wieder zuzulassen. Beachten Sie, dass Sie vor dem Sperren wahrscheinlich bestätigen möchten, dass die Ausrichtung wie gewünscht ist.

Syntax:

os_lock_orientation(flag)

Argument Beschreibung
Flagge Wird auf true oder false gesetzt, um die Orientierungssperre zu aktivieren oder zu deaktivieren.

Rückgabe:

N/A

Beispiel:

if (os_type == os_android) || (os_type == os_ios)
{
os_lock_orientation(true);
}

Der obige Code prüft den Betriebssystemtyp und wenn es entweder Android oder ein iOS ist, dann wird die Orientierungssperre als wahr gekennzeichnet.
