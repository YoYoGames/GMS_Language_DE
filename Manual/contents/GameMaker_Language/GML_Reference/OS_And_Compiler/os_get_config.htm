os_get_config

Diese Funktion gibt den Namen (als String) der aktuell ausgewählten Konfiguration für Ihr Spiel zurück. Weitere Informationen zu Konfigurationen finden Sie im Abschnitt Konfigurationen.

Syntax:

os_get_config()

Rückgabe:

Zeichenfolge

Beispiel:

if os_get_config() = "Free_Version"
{
global.Ads = true;
}
sonst global.Ads = false;

Der obige Code prüft, welche Konfiguration verwendet wird, und wenn es die mit dem Namen "Free_Version" ist, wird die Werbung im Spiel aktiviert.
