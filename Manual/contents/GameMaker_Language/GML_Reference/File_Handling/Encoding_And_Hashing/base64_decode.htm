base64_decode

Diese Funktion konvertiert einen String, der zuvor im Base64-Format kodiert wurde, in Standardtext. Base64 ist ein gängiges Kodierungsschema, das häufig für alle Medien verwendet wird, die als Text gespeichert oder über das Internet übertragen werden müssen, und macht die Ausgabe für das menschliche Auge unlesbar.

Syntax:

base64_decode(zeichenkette)

Argument Beschreibung
String Die zu dekodierende Zeichenkette.

Rückgabe:

Zeichenfolge

Beispiel:

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

Der obige Code öffnet eine Textdatei und liest einen String daraus in die lokale Variable "str". Dieser String wird dann dekodiert und das Ergebnis in der Instanzvariablen "level_data" gespeichert.
