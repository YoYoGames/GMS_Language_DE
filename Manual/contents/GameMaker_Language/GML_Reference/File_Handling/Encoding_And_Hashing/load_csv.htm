laden_csv

Diese Funktion lädt eine Datei im CSV-Format und konvertiert sie in ein DS-Gitter und gibt den eindeutigen ID-Wert für das erstellte Gitter zurück.

Syntax:

load_csv(Dateiname)

Argument Beschreibung
Dateiname Der Name der zu öffnenden Datei (als String)

Rückgabe:

DS-Gitterindex (real)

Beispiel:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
for (var j = 0; j < hh; j++;)
{
draw_text(xx, yy, string(file_grid[# i, j]));
yy += 32;
}
yy = 32;
xx += 32;
}

Der obige Code öffnet die angegebene CSV-Datei und speichert das zurückgegebene DS-Gitter in der Variablen "file_grid". Dieses Raster wird dann in ein paar for lops geparst und der Inhalt auf den Bildschirm gezeichnet.
