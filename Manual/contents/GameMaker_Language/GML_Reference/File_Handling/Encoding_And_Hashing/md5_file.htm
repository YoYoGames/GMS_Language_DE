md5_datei

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert und wurde in einer Vielzahl von Sicherheitsanwendungen eingesetzt. Sie wird auch häufig verwendet, um die Integrität von Dateien und Zeichenketten zu prüfen. Diese Funktion nimmt die gegebene Datei und generiert einen eindeutigen MD5 für diese Datei, der dann zur späteren Verwendung gespeichert werden kann.

HINWEIS: MD5 ist nicht vollständig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

md5_file(Dateiname)

Argument Beschreibung
Dateiname Die Datei, für die der MD5-Hash generiert werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

hash = md5_file(arbeits_verzeichnis + "game_data.ini")

Der obige Code erzeugt einen MD5-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "hash".
