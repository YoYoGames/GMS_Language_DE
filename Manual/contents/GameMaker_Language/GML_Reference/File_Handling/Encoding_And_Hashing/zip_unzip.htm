zip_unzip

Diese Funktion öffnet eine gespeicherte ZIP-Datei und entpackt ihren Inhalt in das angegebene Verzeichnis. Beachten Sie, dass, wenn Sie keinen vollständigen Pfad zum ZIP-Verzeichnis angeben, das aktuelle Stammverzeichnis des Laufwerks verwendet wird, und wenn Sie es in einem relativen Pfad zum Arbeitsverzeichnis des Spiele-Bundles setzen wollen, sollten Sie die Variable arbeits_verzeichnis Variable als Teil des Pfades verwenden (relative Pfade mit "." oder ".." funktionieren nicht wie erwartet und sollten daher vermieden werden). und sollten daher vermieden werden). Beachten Sie auch, dass die Zip-Datei entweder Teil des Spiele-Bundles sein muss (d. h. eine eingeschlossene Datei) oder mit http_get_file() in den Speicherbereich heruntergeladen worden sein muss.

Die Funktion gibt einen Wert zurück, der die Anzahl der extrahierten Dateien angibt, oder sie gibt 0 oder weniger zurück, wenn die Extraktion fehlgeschlagen ist.

Syntax:

zip_unzip(zip_file, ziel_verzeichnis)

Argument Beschreibung
zip_file Die zu öffnende Zip-Datei
ziel_verzeichnis Das Zielverzeichnis, in das die Dateien entpackt werden sollen

Rückgabe:

Real

Beispiel:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
show_debug_message("Extraktion fehlgeschlagen!");
}

Der obige Code öffnet die Zip-Datei, die im Verzeichnis "downloads" gespeichert ist, und extrahiert ihren Inhalt in das Verzeichnis "extracted" (indem er dieses Verzeichnis anlegt, falls es noch nicht existiert) und prüft dann, ob das Extrahieren korrekt durchgeführt wurde und zeigt eine Debug-Meldung an, falls dies fehlschlägt.
