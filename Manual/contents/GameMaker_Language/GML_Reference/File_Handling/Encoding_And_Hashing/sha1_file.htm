sha1_datei

In der Kryptografie ist SHA-1 eine kryptografische Hash-Funktion, die von der United States National Security Agency entwickelt wurde und in mehreren weit verbreiteten Anwendungen eingesetzt wird, z. B. Git wo sie zur Überprüfung von Dateiänderungen verwendet wird, sowie in den Protokollen TLS und SSL, PGP, SSH, S/MIME und IPsec. Diese Funktion nimmt eine Eingabedatei und gibt einen 160-Bit-Message-Digest im ASCII-Format zurück, der eindeutig für diese Datei ist und zu einem späteren Zeitpunkt zur Integritätsprüfung verwendet werden kann.

HINWEIS: SHA-1 ist nicht vollständig sicher und kann gebrochen werden. Siehe diese Seite für weitere Informationen.

Syntax:

sha1_file(Dateiname)

Argument Beschreibung
Dateiname Die Datei, für die der sha1-Hash generiert werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

hash = sha1_file(arbeits_verzeichnis + "game_data.ini")

Der obige Code erzeugt einen sha1-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "hash".
