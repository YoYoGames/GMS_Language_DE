base64_encode

Diese Funktion wandelt eine Zeichenkette in eine im Base64-Format kodierte Zeichenkette um. Dies ist ein gängiges Kodierungsschema, das häufig für Medien verwendet wird, die als Text gespeichert oder über das Internet übertragen werden müssen, und das die Ausgabe für das menschliche Auge unlesbar macht das menschliche Auge unlesbar.

Syntax:

base64_encode(zeichenkette)

Argument Beschreibung
String Die zu kodierende Zeichenkette.

Rückgabe:

Zeichenfolge

Beispiel:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

Der obige Code wandelt die in "game_data" gespeicherte Zeichenfolge in eine base64-kodierte Zeichenfolge um, die dann in einer externen Textdatei gespeichert wird.
