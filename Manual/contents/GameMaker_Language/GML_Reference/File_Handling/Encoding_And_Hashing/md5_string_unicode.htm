md5_string_unicode

In der Kryptografie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptografische Hash-Funktion mit einem 128-Bit-Hash-Wert und wurde in einer Vielzahl von Sicherheitsanwendungen eingesetzt. Sie wird auch häufig verwendet, um die Integrität von Dateien und Zeichenketten zu prüfen. Diese Funktion nimmt eine Unicode-Eingabezeichenfolge (mit 16 Bit für jedes Zeichen) und gibt den 32-stelligen hexadezimalen MD5-Hashwert zurück, der für diese Zeichenfolge eindeutig ist. Auf diese Weise können Sie einen sicheren Schlüssel erzeugen, der gespeichert und verwendet werden kann, um die Integrität der Informationen, die z. B. an einen externen Server gesendet (oder von diesem empfangen) werden, zu überprüfen.

HINWEIS: Es gibt zwei Formate für die MD5-Kodierung, UTF-8 und Unicode. Beide sind vorgesehen, um die Kommunikation mit verschiedenen Serverkonfigurationen zu erleichtern, aber am häufigsten wird Unicode verwendet.

HINWEIS: MD5 ist nicht vollständig sicher und kann geknackt werden. Siehe diese Seite für weitere Informationen.

Syntax:

md5_string_unicode(zeichenkette)

Argument Beschreibung
String Die zu hashende Zeichenfolge.

Rückgabe:

Zeichenfolge

Beispiel:

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

Der obige Code kodiert eine Zeichenkette mit base64 und erzeugt dann einen MD5-Hash. Schließlich werden sowohl der Hash als auch die kodierte Zeichenfolge an einen Server gesendet.
