verzeichnis_zerstören

Mit dieser Funktion wird ein Verzeichnis mit dem angegebenen Namen im Speicherbereich entfernt.

HINWEIS: Diese Funktion funktioniert nicht bei HTML5, da Sie im lokalen Speicher des Browsers keine Verzeichnisse erstellen oder entfernen können.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

directory_destroy(dname)

Argument Beschreibung
dname Der Name des zu entfernenden Verzeichnisses.

Rückgabe:

N/A

Beispiel:

if directory_exists("DLC")
{
directory_destroy("DLC");
}

Damit wird geprüft, ob das angegebene Verzeichnis im lokalen Datenordner vorhanden ist, und wenn ja, wird es entfernt.
