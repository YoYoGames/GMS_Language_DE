verzeichnis_existiert

Diese Funktion gibt true zurück, wenn das angegebene Verzeichnis existiert, oder false, wenn es nicht existiert. Der angegebene Name muss den vollständigen Pfad enthalten, nicht einen relativen Pfad, und standardmäßig können Sie nicht auf Verzeichnisse außerhalb des Spiele-Bundles zugreifen, da alle Spiele in einer Sandbox liegen (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Syntax:

directory_exists(dname)

Argument Beschreibung
dname Der Name des Verzeichnisses, nach dem gesucht werden soll.

Rückgabe:

Boolesche

Beispiel:

if directory_exists(arbeits_verzeichnis + "Saves/")
{
file = file_find_first(arbeits_verzeichnis + "Saves/*.doc", fa_readonly);
}

Dies prüft, ob das angegebene Verzeichnis existiert, und wenn ja, geht es dorthin und gibt die erste gefundene "read only" doc-Datei zurück.
