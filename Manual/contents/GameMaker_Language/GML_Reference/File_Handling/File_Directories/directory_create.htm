verzeichnis_erzeugen

Mit dieser Funktion wird ein Verzeichnis mit dem angegebenen Namen im Speicherbereich angelegt.

HINWEIS: Diese Funktion funktioniert nicht bei HTML5, da Sie im lokalen Speicher des Browsers keine Verzeichnisse erstellen oder entfernen können.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

directory_create(dname)

Argument Beschreibung
dname Der Name des zu erstellenden Verzeichnisses.

Rückgabe:

N/A

Beispiel:

if !directory_exists("Spiele")
{
directory_create("Spiele");
}

Dadurch wird geprüft, ob das angegebene Verzeichnis im lokalen Datenordner vorhanden ist, und wenn nicht, wird es für Sie erstellt.
