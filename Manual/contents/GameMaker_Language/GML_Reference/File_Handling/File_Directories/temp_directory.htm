temp_Verzeichnis

Dies kann verwendet werden, um das temporäre Verzeichnis zurückzugeben, das für Ihr Spiel bei jedem Start erstellt wird (das Stammverzeichnis enthält nicht das endgültige "\"). Dieses Verzeichnis enthält Dateien und kann aufgerufen werden, während das Spiel läuft, aber es wird (zusammen mit allen darin enthaltenen Dateien) entfernt, wenn das Spiel geschlossen wird. mit allen Dateien, die es enthält), wenn das Spiel geschlossen wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

temp_Verzeichnis

Rückgabe:

Zeichenfolge

Beispiel:

ini_open(temp_directory + "\temp_ini.ini");

Dadurch wird eine ini-Datei im temporären Verzeichnis des Spiels geöffnet (und erstellt, falls sie noch nicht vorhanden ist).
