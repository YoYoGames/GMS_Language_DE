Dateiverzeichnisse

Wenn Sie mit Dateien arbeiten, wird es Momente geben, in denen Sie auf das Verzeichnis verweisen müssen, in dem die Dateien gespeichert sind, und GameMaker Studio 2 verfügt über einige spezielle Funktionen und Variablen, die sich auf die verschiedenen möglichen Verzeichnisse beziehen, die Sie eventuell benötigen:

  • verzeichnis_existiert
  • verzeichnis_erzeugen
  • verzeichnis_zerstören
  • temp_Verzeichnis
  • Arbeits_Verzeichnis
  • Programm_Verzeichnis

HINWEIS: Sie können auf HTML5 keine Verzeichnisse erstellen oder zerstören.
