file_bin_position

Diese Funktion gibt die aktuelle Position in Bytes, wobei 0 die erste Position ist, der Datei mit der angegebenen Datei-ID zurück. Sie übergeben den Wert der Datei-ID, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.

Syntax:

file_bin_position(binfile);

Argument Beschreibung
binfile Die ID der Datei, in der die Position ermittelt werden soll.

Rückgabe:

Real

Beispiel:

pos = file_bin_position(file);

Dadurch würde die aktuelle Position in der Variablen "pos" gespeichert.
