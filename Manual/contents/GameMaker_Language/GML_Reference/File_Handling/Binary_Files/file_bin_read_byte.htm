file_bin_read_byte

Diese Funktion gibt ein Datenbyte von der aktuellen Position innerhalb der Datei mit der angegebenen Datei-ID zurück. Sie übergeben den Wert der Datei-ID, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.

Syntax:

file_bin_read_byte(binfile);

Argument Beschreibung
binfile Die ID der Datei, aus der die Daten geholt werden sollen.

Rückgabe:

Real

Beispiel:

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ein Byte an Daten aus dieser Datei holen, bevor sie wieder geschlossen wird.
