file_bin_seek

Diese Funktion verschiebt die aktuelle Leseposition innerhalb der Datei an die angegebene Position. Sie übergeben den Wert der Datei-ID, wie er von der Funktion file_bin_open()zurückgegeben wird, und um eine Datei anzuhängen, verschieben Sie die Position um die Größe der Datei vor dem Schreiben.

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.

Syntax:

file_bin_seek(binfile, pos);

Argument Beschreibung
binfile Die ID der Datei, aus der gelesen werden soll.
pos Die Position in der Datei, zu der verschoben werden soll.

Rückgabe:

N/A

Beispiel:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen, dann die Größe der Datei ermitteln und die nächste Schreibposition auf diese Größe setzen.
