file_bin_close

Schließt eine zuvor geöffnete Binärdatei. Sie liefern den Wert der Datei-ID, wie er von der Funktion file_bin_open().

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.

Syntax:

file_bin_close(binfile);

Argument Beschreibung
binfile Die ID der zu schließenden Datei.

Rückgabe:

N/A

Beispiel:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen. Dann wird die Datei neu geschrieben (die Daten werden gelöscht) und wieder geschlossen.
