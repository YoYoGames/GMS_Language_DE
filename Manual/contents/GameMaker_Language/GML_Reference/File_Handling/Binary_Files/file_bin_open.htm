file_bin_open

Mit dieser Funktion wird die Binärdatei mit dem angegebenen Namen geöffnet. Der Modus gibt an, was mit der Datei gemacht werden kann:

  • 0 = Lesen
  • 1 = Schreiben
  • 2 = sowohl Lesen als auch Schreiben

Wenn die Datei nicht existiert, wird sie erstellt, und die Funktion gibt die ID der Datei zurück, die in den anderen Funktionen verwendet werden muss. Sie können mehrere Dateien gleichzeitig öffnen (max. 32), aber vergessen Sie nicht, sie zu schließen, wenn Sie mit ihnen fertig sind.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

HINWEIS: Diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.

Syntax:

file_bin_open(fname, mode);

Argument Beschreibung
fname Der Name der Datei, aus der gelesen werden soll.
Modus die Anzeige, was mit der Datei gemacht werden kann.

Rückgabe:

Real

Beispiel:

file = file_bin_open("myfile.bin", 2);

Dies würde eine Datei aus dem gleichen Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "file" zuweisen.
