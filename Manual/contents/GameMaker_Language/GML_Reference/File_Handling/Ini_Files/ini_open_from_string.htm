ini_open_from_string

Diese Funktion erzeugt eine ini-Datei aus einer Zeichenkette und öffnet sie zum Lesen/Schreiben. Die Zeichenkette sollte korrekt als ini-Datei formatiert sein (d. h.: mit Abschnitten, Zeilenumbrüchen, Schlüsseln und Werten), andernfalls wird die ini-Datei nicht korrekt erstellt. Beachten Sie dass diese ini-Datei temporär ist und aus dem Speicher entfernt wird, sobald sie geschlossen wird, wodurch alle darin gespeicherten Informationen verloren gehen. ini_close() Funktion gibt jedoch eine Zeichenkette der vollständigen Ini-Datei zurück, die dann auf einem Server oder auf der Festplatte gespeichert werden kann.

Syntax:

ini_open_from_string(string);

Argument Beschreibung
String Die Zeichenkette, die alle ini-Informationen enthält.

Rückgabe:

N/A

Beispiel:

ini_open_from_string(str)
global.sound = ini_read_string("Optionen", "Sound", true);
ini_close();

Der obige Code setzt eine temporäre Ini, um die Informationen aus der Zeichenkette "str" zu halten, liest dann aus der Ini, bevor er sie wieder schließt.
