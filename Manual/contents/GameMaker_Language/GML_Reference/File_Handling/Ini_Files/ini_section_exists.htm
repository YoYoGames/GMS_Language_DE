ini_section_exists

Diese Funktion prüft, ob ein Abschnitt in der aktuell geöffneten ini existiert und gibt true zurück, wenn dies der Fall ist, andernfalls false. Diese Prüfung ist nicht notwendig, um Fehler zu vermeiden, denn wenn ein Abschnitt nicht existiert, gibt das Lesen aus einem nicht existierenden Diese Prüfung ist nicht notwendig, um Fehler zu vermeiden, denn wenn ein Abschnitt nicht existiert, wird beim Lesen aus einem nicht existierenden Abschnitt nur ein Standardwert zurückgegeben, aber sie kann nützlich sein, um zu sehen, ob eine Ini-Datei bestimmte Daten gespeichert hat.

Syntax:

ini_section_exists(section);

Argument Beschreibung
Abschnitt Der Abschnitt in der geöffneten .ini-Datei, auf den geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
global.savegame = "1";
}
ini_close();

Dadurch wird die Variable global.savegame auf "1" gesetzt, wenn "save1" NICHT gefunden wird. Mit einer for-Schleife könnte dies als grundlegende Methode verwendet werden, um herauszufinden, wie viele Savegames es für ein Spiel gibt (d. h.: Erstellen eines Systems für mehrere gleichzeitige Savegames Spiele für ein Spiel).
