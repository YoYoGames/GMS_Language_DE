ini_read_string

Mit dieser Funktion können Sie einen String (Text) aus einer Ini-Datendatei lesen. Ini-Dateien werden in Abschnitte aufgeteilt und jeder Abschnitt wird anschließend in Schlüssel - Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Syntax:

ini_read_string(Abschnitt, Schlüssel, Standard);

Argument Beschreibung
Abschnitt Der Abschnitt der .ini, aus dem gelesen werden soll.
Taste Der Schlüssel innerhalb des entsprechenden Abschnitts der .ini, aus dem gelesen werden soll.
Standard Die Zeichenkette, die zurückgegeben werden soll, wenn eine Zeichenkette nicht an der definierten Stelle gefunden wird (oder die .ini-Datei nicht existiert). Muss eine Zeichenkette sein.

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

Dadurch wird die "savedata.ini" geöffnet und global.name auf die Zeichenkette unter "save1" > "Name" darin gesetzt, dann wird die .ini wieder geschlossen. Sollte keine Zeichenkette unter "save1" > "Name", oder keine "savedata.ini" vorhanden sein, wird global.name auf "Player" gesetzt. Datei vorhanden sein, wird global.name auf "Player1" gesetzt.
