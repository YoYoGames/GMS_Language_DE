ini_write_string

Mit dieser Funktion können Sie einen String (Text) in eine Ini-Datendatei schreiben. Ini-Dateien werden in Abschnitte aufgeteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische ini-Datei würde also etwa so aussehen:

Es ist zu beachten, dass die Verwendung von Anführungszeichen und Escape-Zeichen beim Schreiben von Ini-Strings zu Problemen führen kann, wenn später beim späteren Zurücklesen der Daten zu Problemen führen kann, da die Daten möglicherweise vorzeitig abgeschnitten werden. Dies ist zum Beispiel beim Schreiben ein Problem:

ini_write_string("test2", "section", "hallo \"Fritz\"");

Syntax:

ini_write_string(Abschnitt, Schlüssel, Wert);

Argument Beschreibung
Abschnitt Der Abschnitt der .ini, in den geschrieben werden soll.
Taste Der Schlüssel innerhalb des entsprechenden Abschnitts der .ini, in den geschrieben werden soll.
Wert Die Zeichenkette, die in das entsprechende Ziel geschrieben werden soll.

Rückgabe:

N/A

Beispiel:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Der obige Code öffnet eine ini-Datei zum Lesen und Schreiben, schreibt dann die in der globalen Variable "Name" gespeicherte Zeichenkette in den Abschnitt "Save" mit dem Schlüssel "Player", bevor er die Datei wieder schließt.
