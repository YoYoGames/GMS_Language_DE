ini_schliessen

Diese Funktion sollte in dem Moment aufgerufen werden, in dem Sie mit dem Lesen oder Schreiben einer geöffneten ini-Datei fertig sind. Wenn Sie die Funktion nicht verwenden, nachdem Sie eine der Ini-Schreibfunktionen verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher gehalten werden, bis diese Funktion im Speicher gehalten werden, bis diese Funktion aufgerufen wird, die das Schreiben erzwingt. Wenn Sie versuchen, eine ini zu öffnen, ohne vorher eine andere (oder dieselbe) geschlossen zu haben, erhalten Sie ebenfalls einen Fehler.

Die Funktion gibt auch einen String zurück, in dem die ini-Datei kodiert ist. Diese Zeichenkette kann dann auf einem Server gespeichert werden und/oder zusammen mit der Funktion ini_open_from_string() verwendet werden, um die ini neu zu erstellen.

Syntax:

ini_close();

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Dadurch wird die 'savedata.ini' geöffnet und score auf den Wert unter dem Abschnitt "save1" mit dem Schlüssel "score" darin gesetzt, dann wird die .ini wieder geschlossen. Sollte unter "save1", "score" kein Wert stehen oder keine "savedata.ini"-Datei vorhanden sein, wird score auf 0 gesetzt und eine neue ini_datei erstellt.
