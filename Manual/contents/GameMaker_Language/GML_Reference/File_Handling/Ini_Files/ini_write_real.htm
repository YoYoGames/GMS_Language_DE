ini_write_real

Mit dieser Funktion können Sie einen Wert (numerisch) in eine Ini-Datendatei schreiben. Ini-Dateien werden in Abschnitte aufgeteilt und jeder Abschnitt wird anschließend in Schlüssel - Wert-Paare aufgeteilt. Eine typische ini-Datei würde also etwa so aussehen:

Syntax:

ini_write_real(Abschnitt, Schlüssel, Wert);

Argument Beschreibung
Abschnitt Der Abschnitt der .ini, in den geschrieben werden soll.
Taste Der Schlüssel innerhalb des entsprechenden Abschnitts der .ini, in den geschrieben werden soll.
Wert Der tatsächliche Wert, der in das entsprechende Ziel geschrieben werden soll.

Rückgabe:

N/A

Beispiel:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Dies setzt score auf 1000, öffnet dann "savedata.ini" und schreibt diesen Wert in "save1" > "score". Dann wird score2 auf den tatsächlichen Wert unter "save1" > "score" gesetzt, 1000, und die .ini geschlossen wieder. Hätte es diesen Wert nicht gefunden, wäre score2 am Ende auf 0 gesetzt worden.
