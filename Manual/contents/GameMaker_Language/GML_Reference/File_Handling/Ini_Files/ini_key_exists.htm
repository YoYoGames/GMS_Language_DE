ini_key_exists

Diese Funktion prüft, ob ein Schlüssel in der aktuell geöffneten Ini existiert und gibt true zurück, wenn dies der Fall ist, andernfalls false. Diese Prüfung ist nicht notwendig, um Fehler zu vermeiden, denn wenn ein Schlüssel nicht existiert, wird beim Lesen aus einem nicht existierenden Schlüssel einfach einen Standardwert zurückgeben wird. Es kann jedoch nützlich sein, um zu sehen, ob eine Ini-Datei bestimmte Daten und ein paar andere Dinge gespeichert hat.

Syntax:

ini_key_exists(Abschnitt, Schlüssel);

Argument Beschreibung
Abschnitt Der Abschnitt in der geöffneten .ini-Datei, in dem ein Schlüssel überprüft werden soll.
Taste Die zu prüfende Taste.

Rückgabe:

Boolesche

Beispiel:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
global.name = "Spieler1";
}
ini_close();

Dadurch wird die Variable global.name auf "Player1" gesetzt, wenn kein solcher Schlüssel als "name" existiert.
