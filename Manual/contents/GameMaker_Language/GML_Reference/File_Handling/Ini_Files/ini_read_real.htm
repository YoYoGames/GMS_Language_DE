ini_lesen_real

Mit dieser Funktion können Sie eine Zahl aus einer Ini-Datendatei lesen. Ini-Dateien werden in Abschnitte aufgeteilt und jeder Abschnitt wird anschließend in Schlüssel-Wert-Paare aufgeteilt. Eine typische Ini-Datei würde also etwa so aussehen:

Syntax:

ini_read_real(Abschnitt, Schlüssel, Standard);

Argument Beschreibung
Abschnitt Der Abschnitt der .ini, aus dem gelesen werden soll.
Taste Der Schlüssel innerhalb des entsprechenden Abschnitts der .ini, aus dem gelesen werden soll.
Standard Der Wert, der zurückgegeben werden soll, wenn ein Wert nicht an der definierten Stelle gefunden wird (oder die .ini-Datei nicht existiert). Muss eine reelle Zahl sein.

Rückgabe:

Real

Beispiel:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Dadurch wird die "savedata.ini" geöffnet und score auf den Wert unter "save1" > "score" darin gesetzt, dann wird die .ini wieder geschlossen. Sollte unter "save1" > "score" kein Wert stehen, oder keine "savedata.ini" Datei vorhanden sein, wird score auf 0 gesetzt.
