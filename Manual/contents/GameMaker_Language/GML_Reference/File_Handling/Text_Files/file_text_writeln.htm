file_text_writeln

Mit dieser Funktion können Sie eine neue Zeile in eine geöffnete Textdatei schreiben. Auf diese Weise können Sie Zeilen überspringen oder Informationen zeilenweise schreiben (siehe Beispielcode unten).

Syntax:

file_text_writeln(fileid);

Argument Beschreibung
fileid Die ID der zu bearbeitenden Datei.

Rückgabe:

Real

Beispiel:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file);
file_text_write_string(datei, scr_name[i]);
file_text_writeln(datei);
}
file_text_close(file);

Der obige Code öffnet eine Datei zum Schreiben und läuft dann in einer Schleife durch zwei Arrays, wobei jeder Array-Wert in eine neue Zeile der Datei geschrieben wird. Die Datei wird dann geschlossen, wenn die Schleife beendet ist.
