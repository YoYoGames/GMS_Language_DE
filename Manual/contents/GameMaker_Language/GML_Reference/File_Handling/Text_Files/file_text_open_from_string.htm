file_text_open_from_string

Diese Funktion erstellt eine Textdatei aus einer Zeichenkette und öffnet sie zum Lesen. Sie gibt das Datei-"Handle" zurück, das in allen weiteren Dateifunktionsaufrufen zum Lesen aus dieser Datei verwendet werden soll. Beachten Sie, dass diese Datei temporär und nur lesbar ist, und als solche wird sie aus dem Speicher entfernt, sobald sie geschlossen wird.

HINWEIS: Sie können immer nur maximal 32 Dateien gleichzeitig geöffnet haben. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_from_string(string);

Argument Beschreibung
String Die Zeichenfolge, aus der die Datei erstellt werden soll.

Rückgabe:

Real

Beispiel:

file = file_text_open_from_string(reset_str);

Der obige Code nimmt die in der Variablen "reset_str" gespeicherte Zeichenkette und verwendet sie, um eine schreibgeschützte Textdatei zu erstellen. Das "Handle" für diese Datei wird dann in der Variablen "file" gespeichert, damit alle weiteren Dateifunktionen sie verwenden können.
