file_text_write_string

Mit dieser Funktion können Sie eine Zeichenkette in eine zuvor geöffnete Textdatei schreiben. Wenn die Datei bereits Informationen enthält, werden diese Informationen gelöscht und die Zeichenkette wird an den Anfang der Datei geschrieben, es sei denn, Sie haben die Datei mit der Funktion file_text_open_append(). Sie können dies auch vermeiden, indem Sie die file_text_readln() Funktion zusammen mit der file_text_eof() Funktion den Inhalt der Datei in einer Schleife durchlaufen, bis Sie am Ende angelangt sind, und dann mit dem Schreiben beginnen.

Syntax:

file_text_write_string(fileid, str);

Argument Beschreibung
fileid Die ID der zu bearbeitenden Datei.
str Die Zeichenkette, die in die Datei geschrieben werden soll.

Rückgabe:

Real

Beispiel:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file);
file_text_write_string(datei, scr_name[i]);
file_text_writeln(datei);
}
file_text_close(file);

Der obige Code öffnet eine Datei zum Schreiben und läuft dann in einer Schleife durch zwei Arrays, wobei jeder Array-Wert in eine neue Zeile der Datei geschrieben wird. Die Datei wird dann geschlossen, wenn die Schleife beendet ist.
