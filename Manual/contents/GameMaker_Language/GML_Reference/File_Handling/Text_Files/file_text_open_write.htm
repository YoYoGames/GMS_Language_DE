file_text_open_write

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt) und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden sollte, da sie für alle weiteren Aktionen verwendet wird, die mit dieser Datei zu tun haben. Beachten Sie, dass die Funktion -1 zurückgibt, wenn die Datei nicht erstellt werden kann (z. B. wegen eines illegalen Dateinamens).

HINWEIS: Sie können immer nur maximal 32 Dateien gleichzeitig geöffnet haben. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_write(fname);

Argument Beschreibung
fname Der Name der Datei, in die geschrieben werden soll.

Rückgabe:

Real

Beispiel:

var file;
file = file_text_open_write(arbeits_verzeichnis + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Der obige Code öffnet die Datei "level.txt" zum Schreiben und schreibt dann die in der Variablen "level_data" gespeicherte Zeichenkette, bevor er die Datei schließlich wieder schließt.
