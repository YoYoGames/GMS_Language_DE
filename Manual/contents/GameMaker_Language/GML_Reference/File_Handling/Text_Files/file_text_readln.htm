file_text_readln

Mit dieser Funktion können Sie den Rest der aktuellen Zeile aus einer gegebenen geöffneten Textdatei überspringen und zum Anfang der nächsten Zeile springen. Die Funktion gibt auch die komplette Zeile als Zeichenkette zurück, so dass es eine einfache Möglichkeit ist, komplette "Brocken" von Daten zu lesen von Daten zu lesen und später zu parsen.

Syntax:

file_text_readln(fileid);

Argument Beschreibung
fileid Die ID der Datei, aus der gelesen werden soll.

Rückgabe:

String

Beispiel:

var file = file_text_open_read(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
scr[i] = file_text_read_real(file);
file_text_readln(file);
scr_name[i] = file_text_read_string(file);
file_text_readln(file);
}
file_text_close(file);

Der obige Code öffnet eine Datei zum Lesen und liest dann in einer Schleife durch die Zeilen der Datei abwechselnd einen reellen Zahlenwert und eine Zeichenkette in zwei verschiedene Arrays zur späteren Verwendung. Die Datei wird dann geschlossen, wenn die Schleife beendet ist.
