file_text_open_read

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Lesen und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden sollte, da sie für alle weiteren Aktionen im Zusammenhang mit dieser Datei verwendet wird. Wenn die Datei nicht existiert, gibt die Funktion den Wert -1 zurück.

HINWEIS: Sie können immer nur maximal 32 Dateien gleichzeitig geöffnet haben. Außerdem sollten Sie Dateien immer schließen, wenn Sie fertig sind, da dadurch die Informationen geschrieben werden und der mit der Datei verbundene Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_text_open_read(fname);

Argument Beschreibung
fname Der Name der Datei, aus der gelesen werden soll.

Rückgabe:

Real

Beispiel:

file = file_text_open_read(arbeits_verzeichnis + "level.txt");

Dies würde "level.txt" aus dem gleichen Verzeichnis wie das Spiel öffnen und die Datei-ID in der Variablen "file" speichern.
