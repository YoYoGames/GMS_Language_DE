file_text_eof

Diese Funktion gibt true zurück, wenn das Ende einer gegebenen geöffneten Textdatei erreicht wurde oder false, wenn nicht.

Syntax:

file_text_eof(fileid);

Argument Beschreibung
fileid Die ID der zu prüfenden Datei.

Rückgabe:

Boolesche

Beispiel:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
str[num++] = file_text_readln(file);
}
file_text_close(file);

Der obige Code öffnet eine Datei zum Schreiben und läuft dann in einer Schleife durch die bereits in die Datei geschriebenen Textzeilen, bis er das Ende erreicht hat, wobei jede Zeile im Array "str" gespeichert wird.
