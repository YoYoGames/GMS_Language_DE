file_find_close

Muss aufgerufen werden, nachdem Dateien behandelt wurden, die mit file_find_first() und file_find_next() Funktionen geöffnet wurden, um Speicher freizugeben. Die Dateisuchfunktionen öffnen Handles im Dateiverzeichnis Verzeichnis und diese belegen eine minimale Menge an Speicher, der sich (mit der Zeit) aufaddiert. Daher sollten Sie diese Funktion immer aufrufen, nachdem Sie die gewünschten Dateien gefunden haben, um diese Handles zu "schließen".

Syntax:

file_find_close();

Rückgabe

N/A

Beispiel:

file_find_close();

Dadurch wird der mit den Handles der Dateisuchfunktion verbundene Speicher geschlossen und freigegeben.
