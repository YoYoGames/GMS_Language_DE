Dateisystem

Es gibt in GameMaker Studio 2 eine Reihe von Funktionen, mit denen Sie Dateien aller Art auf unterschiedliche Weise manipulieren können:

  • file_exists
  • datei_löschen
  • datei_umbenennen
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • datei_attribute

Die folgenden Funktionen können verwendet werden, um Dateinamen zu ändern:

HINWEIS: Diese Funktionen arbeiten nicht mit den eigentlichen Dateien! Sie befassen sich nur mit den Zeichenketten, die sich auf die verschiedenen Pfadelemente beziehen, und sollten in Verbindung mit den anderen Dateifunktionen verwendet werden.

  • Dateiname_name
  • Dateiname_pfad
  • filename_dir
  • Dateiname_Laufwerk
  • Dateiname_ext
  • Dateiname_ändern_ext

Schließlich gibt es noch einige spezielle Funktionen zur Ermittlung von Dateinamen, die zum Laden und Speichern von Dateien mit Dateityp-Filtern verwendet werden können. Diese Funktionen öffnen ein Explorer-Fenster und führen dazu, dass das Spiel den Fokus verliert. Dies bedeutet, dass alle Eingaben, auf die das Das bedeutet, dass alle Eingaben, von denen das Spiel abhängt, möglicherweise nicht korrekt registriert werden. Daher empfehlen wir, dass Sie (wenn Sie diese Funktionen in einem Tastaturereignis verwenden) das Ereignis "Taste oben" und nicht die Ereignisse "Gedrückt" oder "Unten" verwenden.

HINWEIS: Diese Funktionen sind Windows und macOS nur.

WARNUNG! Diese Funktionen funktionieren möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für mehr Informationen.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
