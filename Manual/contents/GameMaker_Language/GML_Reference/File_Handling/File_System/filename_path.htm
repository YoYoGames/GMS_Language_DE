Dateiname_pfad

Mit dieser Funktion wird der Pfadteil des angegebenen Dateipfads einschließlich des abschließenden Backslashs zurückgegeben.

Syntax:

filename_path(fname);

Argument Beschreibung
fname Der zu verwendende Dateiname und Pfad.

Rückgabe:

Zeichenfolge

Beispiel:

path = filename_path(arbeits_verzeichnis + "Test.ini");

Der obige Code ruft den Pfad (als String) der angegebenen Datei ab.
