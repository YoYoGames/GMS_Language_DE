datei_löschen

Diese Funktion löscht die angegebene Datei aus dem System. Es ist zu beachten, dass diese Funktion nur die Dateien löscht, die GameMaker Studio 2 erstellen und parsen kann: ini-Dateien, Textdateien und Binärdateien, oder solche Dateien, die um im Spiel erstellte Ressourcen wie Sprites oder Oberflächen zu speichern. Alle anderen Dateien werden jedoch nicht gelöscht. Die Funktion gibt außerdem true zurück, wenn die Datei erfolgreich entfernt wurde, oder false in allen anderen Fällen.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Weitere Informationen finden Sie im Abschnitt über die Begrenzung des Dateisystems.

Syntax:

file_delete(fname);

Argument Beschreibung
fname Der Name der zu löschenden Datei.

Rückgabe:

Boolesche

Beispiel:

if file_exists("level.txt")
{
file_delete("level.txt");
}

Dies würde nach einer Datei suchen und wenn sie existiert, wird sie gelöscht.
