get_open_filename

Diese Funktion öffnet einen Dialog und fragt den Spieler nach einem Dateinamen, der mit dem angegebenen Filter geöffnet werden soll. Der Filter hat die Form " name1|mask1|name2|mask2|... ", wobei die Maske die verschiedenen Optionen mit einem Semikolon dazwischen enthalten kann und Sie können auch einen "*" verwenden, um eine beliebige Zeichenfolge zu bezeichnen. Zum Beispiel: " bitmaps|*.bmp;*.wmf", würde den Benutzer auffordern, den Namen einer Bitmap-Datei aus dem System auszuwählen, aber nur solche, die die beiden angegebenen Erweiterungen haben (bmp und png). Diese Funktion öffnet nicht die Datei selbst! Sie gibt nur eine Zeichenkette mit dem vollständigen Namen und Pfad der Datei zurück. Wenn der Benutzer "Abbrechen" drückt, wirdeine leere Zeichenkette "" zurückgegeben.

Wichtig bei der Verwendung dieser Funktion ist, dass sie Ihnen bestimmte Rechte für diese Datei gewährt, und zwar für die Dauer Ihres Spiels. Wenn Sie also einmal den Dateipfad vom Benutzer erhalten haben, können Sie immer wieder darauf zugreifen, ohne fragen zu müssen. Wie auch immer, da es sich bei dieser Funktion um das Lesen einer Datei handelt, sind die gewährten Berechtigungen nur zum Lesen und Speichern vom Betriebssystem nicht erlaubt. Sollten Sie die Datei lesen und schreiben müssen, können Sie die Berechtigung mit der Funktion get_save_filename().

HINWEIS: Diese Funktion ist nur für die Ziele Windows und macOS gültig.

Syntax:

get_open_filename(filter, fname);

Argument Beschreibung
Filter Der Dateitypfilter und/oder der Name.
fname Der vorgeschlagene Dateiname, der verwendet werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

var file;
file = get_open_filename("text file|*.txt", "");
if file != ""
{
file_text_open_read(file);
}

Der obige Code fordert den Benutzer auf, eine Textdatei zum Öffnen auszuwählen, und prüft dann den zurückgegebenen Dateinamen und -pfad, und öffnet die Datei zum Lesen, falls sie existiert.
