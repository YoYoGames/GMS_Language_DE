get_save_filename_ext

Diese Funktion öffnet einen Dialog und fragt den Spieler nach einem Dateinamen, unter dem mit dem angegebenen Filter gespeichert werden soll. Der Filter hat die Form " name1|mask1|name2|mask2|... ", wobei die Maske die verschiedenen Optionen mit einem Semikolon dazwischen enthalten kann und Sie können auch einen "*" verwenden, um eine beliebige Zeichenfolge zu bezeichnen. Zum Beispiel: " bitmaps|*.bmp;*.wmf", würde den Benutzer auffordern, den Namen einer Bitmap-Datei aus dem System auszuwählen, aber nur solche, die die beiden angegebenen Erweiterungen (bmp undpng ) haben. Diese Funktion speichert die Datei selbst nicht! Sie gibt nur eine Zeichenkette mit dem vollständigen Namen und Pfad der Datei zurück. Wenn der Benutzer "Abbrechen" drückt, wirdeine leere Zeichenkette "" zurückgegeben. Sie können auch ein Verzeichnis angeben, von dem aus die Suche gestartet werden soll, und eine Beschriftung, die am oberen Rand des Fensters anstelle der Standardbeschriftung angezeigt werden soll.

Ein wichtiger Hinweis bei der Verwendung dieser Funktion ist, dass sie Ihnen bestimmte Rechte für diese Datei gewährt, und zwar für die Dauer Ihres Spiels. Sobald Sie also den Dateipfad vom Benutzer erhalten haben, können Sie immer wieder darauf zugreifen, ohne fragen zu müssen, und die Berechtigung gilt für das Lesen und Schreiben gleichermaßen.

HINWEIS: Diese Funktion ist nur für die Ziele Windows und macOS gültig.

Syntax:

get_save_filename_ext(filter, fname, directory, caption);

Argument Beschreibung
Filter Der Dateitypfilter und/oder der Name.
fname Der vorgeschlagene Dateiname, der verwendet werden soll.
Verzeichnis Das Verzeichnis, von dem aus die Suche gestartet werden soll.
Beschriftung Die Beschriftung für das Fenster.

Rückgabe:

Zeichenfolge

Beispiel:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Hier einen Level speichern");
if file != ""
{
screen_save(file);
}

Der obige Code fordert den Benutzer auf, eine Bilddatei zum Speichern im Arbeitsverzeichnis auszuwählen und prüft dann den zurückgegebenen Dateinamen und Pfad. Wenn es sich nicht um eine leere Zeichenfolge handelt, wird ein Bildschirmfoto erstellt und unter dem angegebenen Namen gespeichert.
