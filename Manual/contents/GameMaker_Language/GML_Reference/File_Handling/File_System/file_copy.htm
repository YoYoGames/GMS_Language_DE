file_copy

Mit dieser Funktion wird die angegebene Datei kopiert, wobei sie den von Ihnen gewählten neuen Namen erhält. Wenn Sie eine Datei auf Android aus dem Bereich " Enthaltene Dateien" (der sich in der APK befindet) kopieren, wird zuerst in den Speicher kopiert, daher sollten Sie beachten dass das Kopieren von sehr großen Dateien aus der APK dazu führen kann, dass die Geräte keinen Speicher mehr haben. Auf dem HTML5-Ziel verwenden wir die HTML5 Local Storage API für das Speichern auf der Client-Seite, und die Funktionen unterstützen keine binären Datei-Operationen, so dass angenommen wird, dass Sie Kopieren einer Textdatei.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Weitere Informationen finden Sie im Abschnitt über die Begrenzung des Dateisystems.

Syntax:

file_copy(fname, newname);

Argument Beschreibung
fname Der Name der zu kopierenden Datei.
neuer Name Der Name der neuen, kopierten, Datei.

Rückgabe:

N/A

Beispiel:

if file_exists("level1.txt")
{
file_copy("level1.txt", "level2.txt");
}

Dies würde nach einer Datei suchen und wenn sie existiert, wird sie kopiert.
