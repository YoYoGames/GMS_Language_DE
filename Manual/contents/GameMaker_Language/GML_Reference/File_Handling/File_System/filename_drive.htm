Dateiname_Laufwerk

Diese Funktion gibt die Laufwerksinformationen des Dateinamens zurück.

Syntax:

filename_drive(fname);

Argument Beschreibung
fname Die zu verwendende Datei.

Rückgabe:

Zeichenfolge

Beispiel:

Laufwerk = filename_drive(file_find_first(working_directory + "*.doc", 0));

Der obige Code ruft die Laufwerksinformationen (als String) der angegebenen Datei ab.
