file_find_next

Diese Funktion gibt den Namen der nächsten Datei zurück, die der zuvor angegebenen Maske und den Attributen (definiert durch file_find_first()). Wenn keine solche Datei existiert, wird die leere Zeichenkette zurückgegeben.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

HINWEIS: Diese Funktion funktioniert auf allen C++ Zielplattformen(Windows, Mac, iOS, Android und WinPhone), ABER die Filterflags funktionieren nur unter Windows.

Syntax:

file_find_next();

Rückgabe:

Zeichenfolge

Beispiel:

file = file_find_next();

Dadurch wird die nächste Datei zurückgegeben, die mit den zuvor angegebenen Attributen übereinstimmt.
