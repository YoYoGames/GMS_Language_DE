datei_umbenennen

Diese Funktion benennt die angegebene Datei mit dem angegebenen Namen um. Die Funktion gibt true zurück, wenn die Datei erfolgreich umbenannt wurde, oder false in allen anderen Fällen.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht so, wie Sie es erwarten, da GameMaker Studio 2 sandboxed ist! Bitte lesen Sie den Abschnitt über das Dateisystem für weitere Informationen.

Syntax:

file_rename(altername, neuername);

Argument Beschreibung
alterName Der Name der zu ändernden Datei.
neuer Name Der neue Name, den die Datei erhalten soll.

Rückgabe:

Boolesche

Beispiel:

if file_exists("level1.txt")
{
file_rename("level1.txt", "level.txt");
}

Dies würde nach einer Datei suchen und wenn sie existiert, wird sie umbenannt.
