Dateiname_name

Mit dieser Funktion wird der Namensteil der angegebenen Datei zurückgegeben, mit der Erweiterung, aber ohne den Pfad

Syntax:

Dateiname_name(fname);

Argument Beschreibung
fname Die zu verwendende Datei.

Rückgabe:

Zeichenfolge

Beispiel:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

Der obige Code ruft den Namen (als Zeichenkette) der ersten im angegebenen Verzeichnis gefundenen Datei vom Typ "doc" ab.
