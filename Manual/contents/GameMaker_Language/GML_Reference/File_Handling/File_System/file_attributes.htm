datei_attribute

Sie können diese Funktion verwenden, um die Attribute einer bestimmten Datei zu prüfen. Sie können die folgenden Konstanten addieren, um den gewünschten Dateityp zu ermitteln:

Konstant Beschreibung
fa_readonly Schreibgeschützte Dateien
fa_hidden Versteckte Dateien
fa_sysfile System-Dateien
fa_volumeid Volume-id-Dateien
fa_directory Verzeichnisse
fa_archive Archivierte Dateien

HINWEIS: Dies ist eine Funktion nur für Windows.

Syntax:

file_attributes(fname, attr);

Argument Beschreibung
fname Der Name der zu prüfenden Datei.
attr Die Attribute, auf die geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if !file_attributes(file, fa_hidden)
{
file_delete(file);
}

Dies würde eine Datei daraufhin überprüfen, ob sie versteckt ist oder nicht, und wenn nicht, wird sie gelöscht.
