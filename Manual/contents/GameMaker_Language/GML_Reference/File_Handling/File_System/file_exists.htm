file_exists

Diese Funktion gibt true zurück, wenn die angegebene Datei existiert, und false, wenn sie nicht existiert. Beachten Sie, dass die Funktion nur zum Überprüfen lokaler Dateien verwendet werden kann, nicht aber von Dateien, die auf einem Remote-Server gespeichert sind.

Syntax:

file_exists(fname);

Argument Beschreibung
fname Der Name der Datei, auf die geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if file_exists("level.txt")
{
file = file_text_open_read("level.txt");
}

Dies würde nach einer Datei suchen und wenn sie existiert, wird sie zum Lesen geöffnet.
