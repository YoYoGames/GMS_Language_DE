Dateiname_ext

Diese Funktion gibt den Erweiterungsteil des angegebenen Dateinamens zurück, einschließlich des führenden Punktes.

Syntax:

Dateiname_ext(fname);

Argument Beschreibung
fname Die zu verwendende Datei.

Rückgabe:

Zeichenfolge

Beispiel:

ext = filename_ext(file_find_first("*.*", 0));

Der obige Code ruft die Erweiterung (als String) der angegebenen Datei ab.
