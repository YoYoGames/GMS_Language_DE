dcos

In einem rechtwinkligen Dreieck ist cos(val) = Anliegend / Hypotenuse, wobei val einer der drei Winkel ist. Diese Funktion wird immer eine Zahl zwischen 1 und -1 zurückgeben.

HINWEIS: Diese Funktion nimmt einen Wert in Grad, nicht im Bogenmaß an.

Syntax:

dcos(val);

Argument Beschreibung
val Der Winkel (in Grad), von dem der Kosinus zurückgegeben werden soll.

Rückgabe:

Real

Beispiel:

val = dcos(45);

Dadurch wird "val" auf 0,71 gesetzt.
