lengthdir_x

Diese Funktion wird verwendet, um die x-Komponente einer Position "len" Pixel vom Startpunkt und in Richtung "dir" zu erhalten. Wenn Sie sich einen Kreis um Ihre Instanz vorstellen, und sich dann einen Punkt irgendwo auf diesem Kreis vorstellen, müssen wir, um uns zu diesem Punkt zu bewegen zu diesem Punkt zu bewegen, müssen wir das Objekt um so viele Pixel in diese Richtung verschieben... also liefert diese Funktion (wenn sie mit lengthdir_y()) die Position dieses Punktes auf dem Kreis, um sie im Code der Instanz zu verwenden. Siehe das Bild unten für Details:

Syntax:

lengthdir_x(len, dir);

Argument Beschreibung
len Die Länge der Entfernung vom Punkt zur Rückkehr.
dir Die Richtung des Umkehrpunktes.

Rückgabe:

Real

Beispiel:

var _xx = x + lengthdir_x(64, image_angle);
var _yy = y + lengthdir_y(64, image_angle);
instance_create_layer(_xx, _yy, "Bullets", obj_bullet);

Dadurch wird eine Aufzählungszeicheninstanz an(_xx, _yy) erzeugt, die 64 Pixel von der übergeordneten Instanz in Richtung des Bildwinkels entfernt ist.
