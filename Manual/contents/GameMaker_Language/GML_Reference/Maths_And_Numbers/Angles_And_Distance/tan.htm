tan

In einem rechtwinkligen Dreieck ist tan(val) = Opposite / Adjacent, wobei val einer der drei Winkel ist.

HINWEIS: Diese Funktion nimmt einen Wert in Bogenmaß und nicht in Grad an.

HINWEIS: Pi/2, (3 pi/2), -pi/2 und eine große Anzahl anderer Werte werden bei dieser Funktion Fehler machen, da sie Unendlichkeit zurückgeben, eine grafische Darstellung würde bei diesen Werten Asymptoten erzeugen.

Syntax:

tan(val);

Argument Beschreibung
val Der Winkel (im Bogenmaß), der den Tangens von zurückgeben soll.

Rückgabe:

Real

Beispiel:

val = tan( pi );

Dadurch wird val auf 0 gesetzt.
