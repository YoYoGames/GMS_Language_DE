Punkt_Abstand

Diese Funktion gibt die Länge eines Vektors zurück, der aus den angegebenen Komponenten [x1,y1] und [x2,y2] gebildet wird. Wenn wir z. B. in der Abbildung unten den Abstand zwischen der Position des Spielerschiffs und der Position des Feindes ermitteln wollen, um zu berechnen, ob der Feind nahe genug ist, um auf ihn zu schießen, dann würden wir diese Funktion verwenden (den genauen Code finden Sie im Beispiel unter dem Bild):

Syntax:

point_distance(x1, y1, x2, y2);

Argument Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
x2 Die x-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors

Rückgabe:

Real

Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}

Der obige Code ermittelt die x- und y-Koordinaten des nächstgelegenen Gegners und prüft damit den Abstand (die Länge) des von ihnen und den Spielerkoordinaten gebildeten Vektors. Wenn der Wert kleiner als 200 ist, erzeugt das Player-Objekt eine Instanz von "obj_Missile".
