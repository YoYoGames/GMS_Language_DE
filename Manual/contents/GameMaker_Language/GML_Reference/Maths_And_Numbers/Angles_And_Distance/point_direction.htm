Punkt_Richtung

Diese Funktion gibt die Richtung eines Vektors zurück, der aus den angegebenen Komponenten [x1,y1] und [x2,y2] in Bezug auf die festen x/y-Koordinaten des Raums gebildet wird. Wenn wir zum Beispiel im folgenden Bild die Richtung von der Position des Spielerschiffs zur gegnerischen Position ermitteln wollen, damit wir eine Rakete auf den Feind abfeuern können, dann würden wir diese Funktion verwenden (den genauen Code finden Sie im Beispiel unter dem Bild):

Syntax:

Punkt_Richtung(x1, y1, x2, y2)

Argument Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
x2 Die x-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors

Rückgabe:

Real

Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
direction = point_direction(x, y, ex, ey);
}

Der obige Code ermittelt die x- und y-Koordinaten des nächstgelegenen Gegners und übergibt sie dann an ein Bullet-Objekt, das in der Funktion point_direction verwendet wird, um seine Bewegungsrichtung korrekt einzustellen.
