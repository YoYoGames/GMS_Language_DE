Punkt_Abstand_3d

Diese Funktion nimmt die gelieferten Komponenten des Vektors und gibt die Länge (Abstand) des Vektors zurück. Sie arbeitet genau wie die Funktion point_distance() aber mit dem Zusatz, dass der z-Wert mit einbezogen wird (Tiefe) für die Verwendung im 3D-Raum.

Syntax:

point_distance_3d(x1, y1, z1, x2, y2, z2);

Argument Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
z1 Die z-Koordinate der ersten Komponente des Vektors
x2 Die x-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors
z2 Die z-Koordinate der zweiten Komponente des Vektors

Rückgabe:

Real

Beispiel:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
ex = inst.x;
ey = inst.y;
ez = inst.z;
if point_distance_3d(x, y, z, ex, ey, ez) < 200
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}
}

Der obige Code ermittelt die x- und y- und z-Koordinaten des nächstgelegenen Gegners und prüft damit den Abstand (Länge) des Vektors, der durch diesen und die Spielerkoordinaten gebildet wird. Wenn der Wert kleiner als 200 ist, erzeugt das Player-Objekt eine Instanz von "obj_Missile".
