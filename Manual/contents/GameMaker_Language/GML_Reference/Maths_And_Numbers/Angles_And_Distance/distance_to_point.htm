abstand_zum_punkt

Diese Funktion berechnet den Abstand vom Rand der Bounding Box der aufrufenden Instanz zur angegebenen x/y-Position im Raum, wobei der Rückgabewert in Pixel angegeben wird. Beachten Sie, dass die Ergebnisse falsch sind, wenn das aufrufende Objekt kein Sprite oder keine Maske definiert hat. Ergebnisse falsch sein werden.

Syntax:

abstand_zum_punkt(x, y);

Argument Beschreibung
x Die zu prüfende x-Position.
y Die zu prüfende y-Position.

Rückgabe:

Real

Beispiel:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
canshoot = true;
}

Der obige Code prüft den Abstand zur x/y-Position des Player-Objekts, und wenn dieser kleiner ist als der in der Variablen "range" gespeicherte Wert, wird die Variable "canshoot" auf true gesetzt.
