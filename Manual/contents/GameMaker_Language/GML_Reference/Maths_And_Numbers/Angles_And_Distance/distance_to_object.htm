Abstand_zum_Objekt

Diese Funktion berechnet den Abstand von der Kante der Bounding Box der aufrufenden Instanz zur nächstgelegenen Kante der nächstgelegenen Instanz des angegebenen Objekts. Das Objekt kann ein Objektindex oder eine bestimmte Instanz-ID sein, aber auch die Schlüsselwort andere sein, und der Abstand wird in Pixel zurückgegeben. Beachten Sie, dass die Ergebnisse falsch sind, wenn für eines der Objekte kein Sprite oder keine Maske definiert ist.

Syntax:

distance_to_object(obj);

Argument Beschreibung
obj Das Objekt, auf das geprüft werden soll.

Rückgabe:

Real.

Beispiel:

if (distance_to_object(obj_Player) < range)
{
canshoot = true;
}

Der obige Code prüft den Abstand zum Player-Objekt und wenn dieser kleiner ist als der in der Variable "range" gespeicherte Wert, wird die Variable "canshoot" auf true gesetzt.
