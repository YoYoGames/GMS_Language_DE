date_datetime_string

Mit dieser Funktion können Sie eine Zeichenkette erstellen, die die angegebene Datumszeit enthält, formatiert für das System oder Gerät, auf dem das Spiel läuft, wenn die Funktion aufgerufen wird.

Syntax:

date_datetime_string(date);

Argument Beschreibung
Datum Das zu verwendende Datum.

Rückgabe:

Zeichenfolge

Beispiel:

str = date_datetime_string(date_current_datetime());

Dies würde "str" auf etwas wie "15. September 2011, 11:33.00" setzen, abhängig von den Systemeinstellungen für die Anzeige von Datum/Uhrzeit und dem aktuellen Datum und der Uhrzeit.
