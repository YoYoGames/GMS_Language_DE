aktueller_Tag

Diese schreibgeschützte Variable gibt den Tag als einen Wert von 1 bis 31 zurück, abhängig vom Monat.

Syntax:

aktueller_tag;

Rückgabe:

Real

Beispiel:

draw_text(32, 32, "Heute ist " + string(aktueller_tag) + "/" + string (aktueller_monat) + "/" + string(aktuelles_jahr) +".");

Der obige Code zeichnet den Tag, den Monat und das Jahr.
