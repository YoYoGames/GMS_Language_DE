aktuelle_zeit

Diese nur lesbare Variable gibt die Anzahl der Millisekunden zurück, die seit dem Start des Spiels vergangen sind.

Syntax:

aktuelle_zeit;

Rückgabe:

Real

Beispiel:

if (aktuelle_zeit > 600000)
{
msg = show_question_async("Möchten Sie bewerten?");
}

Der obige Code prüft, ob mehr als 10 Minuten vergangen sind, bevor er dem Benutzer eine Frage stellt.
