date_is_today

Diese Funktion gibt true zurück, wenn der angegebene Datumswert der Tag ist, an dem sie geprüft wird (z. B. heute), oder false, wenn nicht. Dies kann eine praktische Funktion für Dinge wie Easter Eggs in Ihren Spielen oder zum Freischalten von saisonalen Inhalten sein. Beachten Sie, dass diese Funktion von der eingestellten Zeitzone beeinflusst wird (Standard ist die lokale Zeit), die Sie mit der Funktion date_set_timezone() Funktion ändern können.

Syntax:

date_is_today(date);

Argument Beschreibung
Datum Die zu verwendende Datumszeit.

Rückgabe:

Boolesche

Beispiel:

if date_is_today(global.Halloween)
{
global.Max_Levels = 200;
}

Der obige Code überprüft die in der globalen Variable "Halloween" gespeicherte Datumszeit, um zu sehen, ob es heute ist, und wenn ja, setzt er eine andere globale Variable auf einen neuen Wert.
