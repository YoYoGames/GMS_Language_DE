delta_zeit

Der Zweck des Delta-Timings ist es, die Auswirkungen der Verzögerung oder Langsamkeit von Computern zu beseitigen, die versuchen, komplexe Grafiken oder eine Menge Code zu verarbeiten. Es ist ein Wert, der zur Geschwindigkeit von Objekten addiert werden kann, so dass sie sich schließlich mit der gleichen Geschwindigkeit bewegen, unabhängig vom Lag. Dies wird in GameMaker Studio 2 durch die Verwendung der schreibgeschützten Variable delta_time bei jedem Schritt erreicht, da sie die Zeit, die zwischen einem Schritt und dem nächsten vergangen ist, in Mikrosekunden misst (1Mikrosekunde ist 1.000.000stel einer Sekunde). Daher kann die Variable delta_time verwendet werden, um zu berechnen, wie viel schneller (z. B.) sich eine Spielfigur bewegen muss, um eine Verzögerungsspitze im Spiel auszugleichen.

Syntax:

delta_zeit

Rückgabe:

Real (Ganzzahl)

Beispiel:

var _dt = delta_time / 1000000;
speed = spd * dt;

Der obige Code setzt die Geschwindigkeit der Instanz unter Verwendung der Delta-Zeit, um die Verzögerung zu korrigieren. Die Variable "_dt" wird verwendet, um den zuvor berechneten Deltazeitwert zu halten, so dass das Verhältnis verwendet werden kann, um den Basisgeschwindigkeitswert (gehalten in der Variablen "spd") zu multiplizieren. in der Variablen "spd") multipliziert werden kann und so eine konsistente Geschwindigkeit für die Instanz erhält.
