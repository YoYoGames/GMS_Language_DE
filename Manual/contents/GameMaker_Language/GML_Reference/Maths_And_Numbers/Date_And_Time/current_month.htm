aktueller_Monat

Diese schreibgeschützte Variable gibt den aktuellen Monat als numerischen Wert zurück, wobei 1 für Januar und 12 für Dezember steht.

Syntax:

aktueller_Monat;

Rückgabe:

Real

Beispiel:

draw_text((32, 32, "Heute ist " + string(aktueller_tag) + "/" + string (aktueller_monat) + "/" + string(aktuelles_jahr) +".");

Der obige Code zeichnet den Tag, den Monat und das Jahr.
