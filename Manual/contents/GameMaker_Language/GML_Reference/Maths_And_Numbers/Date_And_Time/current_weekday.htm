aktueller_Wochentag

Diese schreibgeschützte Variable gibt den Wochentag als Wert zurück, wobei Sonntag gleich 0 und Samstag gleich 6 ist.

Syntax:

aktueller_Wochentag;

Rückgabe:

Real

Beispiel:

var day;
switch(aktueller_Wochentag)
{
case 0: Tag = "Sonntag"; break;
case 1: Tag = "Montag"; break;
case 2: Tag = "Dienstag"; break;
case 3: Tag = "Mittwoch"; break;
case 4: Tag = "Donnerstag"; break;
case 5: Tag = "Freitag"; break;
case 6: Tag = "Samstag"; break;
}
draw_text(32, 32, "Heute ist " + Tag +".");

Der obige Code verwendet den Wert von current_weekday, um eine Variable mit dem richtigen Tag in Text zu setzen, und zeichnet diesen dann für den Benutzer.
