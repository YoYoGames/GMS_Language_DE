date_compare_date

Mit dieser Funktion können Sie zwei Daten daraufhin überprüfen, welches der beiden Daten früher oder später als das andere ist. Die Funktion gibt -1 zurück, wenn Datum1 früher ist, 0, wenn beide Daten gleich sind, und 1, wenn Datum1 später ist.

Syntax:

date_compare_date( date1, date2 );

Argument Beschreibung
Datum1 Das erste Date.
Datum2 Das Datum, mit dem es verglichen wird.

Rückgabe:

Real

Beispiel:

d = date_compare_date(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime());

Dies würde "d" auf den entsprechenden Wert setzen, je nachdem, welches der Daten das früheste war, wahrscheinlich 1, da das aktuelle Datum weiter vorne liegen würde als der 15. September 2011.
