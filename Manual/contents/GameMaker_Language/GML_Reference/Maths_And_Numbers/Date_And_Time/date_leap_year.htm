Datum_Schaltjahr

Diese Funktion gibt true zurück, wenn die Jahreskomponente des angegebenen datetime-Werts ein Schaltjahr ist, andernfalls false. Dies kann eine praktische Funktion für Dinge wie Easter Eggs in Ihren Spielen oder zum Freischalten von speziellen Inhalten sein.

Syntax:

date_leap_year(date);

Argument Beschreibung
Datum Die zu verwendende Datumszeit.

Rückgabe:

Boolesche

Beispiel:

if date_leap_year(date_current_datetime())
{
if !global.ExtraContent
{
global.ExtraContent = true;
}
}

Der obige Code prüft die aktuelle Datumszeit, um zu sehen, ob das Jahr ein Schaltjahr ist oder nicht. Wenn ja, setzt er eine globale Variable.
