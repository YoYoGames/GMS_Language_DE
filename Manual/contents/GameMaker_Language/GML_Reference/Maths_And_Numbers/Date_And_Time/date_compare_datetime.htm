date_compare_datetime

Mit dieser Funktion können Sie zwei Datums- und Zeitangaben daraufhin überprüfen, welches Datum früher oder später als das andere ist. Die Funktion gibt -1 zurück, wenn Datum1 früher ist, 1, wenn Datum1 später ist und 0, wenn sie gleich sind, obwohl, da dies auch die Zeit berücksichtigt auch die Zeit berücksichtigt, ist eine 0 für die Übereinstimmung sehr selten (es muss buchstäblich die gleiche Sekunde des gleichen Tages des gleichen Monats des gleichen Jahres sein, um eine 0 zurückzugeben).

Syntax:

date_compare_datetime(date1, date2);

Argument Beschreibung
Datum1 Der erste Zeitpunkt.
Datum2 Der Zeitpunkt, mit dem er verglichen wird.

Rückgabe:

Real

Beispiel:

d = date_compare_datetime(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime());

Dies würde d auf den entsprechenden Wert setzen, je nachdem, welcher der Zeitpunkte der früheste war, wahrscheinlich -1, da das aktuelle Datum weiter in der Zukunft liegen würde als der 15. September 2011.
