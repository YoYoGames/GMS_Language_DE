datum_ink_minute

Mit dieser Funktion können Sie einen gegebenen datetime-Wert um eine bestimmte Anzahl von Minuten inkrementieren, und es wird der neue datetime-Wert zurückgegeben.

Syntax:

date_inc_minute( date, amount );

Argument Beschreibung
Datum Der Zeitpunkt, zu dem addiert werden soll.
Betrag Die Anzahl der zu addierenden Minuten (muss eine ganze Zahl sein).

Rückgabe:

Real

Beispiel:

mynewdatetime = date_inc_minute(date_current_datetime(), 60);

Dies würde "mynewdatetime" auf das aktuelle Datum setzen, aber mit 60 Minuten hinzugefügt.
