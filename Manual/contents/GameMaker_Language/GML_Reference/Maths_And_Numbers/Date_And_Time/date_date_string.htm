date_date_string

Mit dieser Funktion können Sie eine Zeichenkette erzeugen, die das angegebene Datum enthält, formatiert als Tag/Monat/Jahr.

Syntax:

date_date_string(date);

Argument Beschreibung
Datum Das zu verwendende Datum.

Rückgabe:

Zeichenfolge

Beispiel:

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

Dies würde "str" so einstellen, dass es eine formatierte Zeichenfolge des aktuellen Datums und der Uhrzeit enthält, wie sie vom System angezeigt wird.
