aktuelle_minute

Diese schreibgeschützte Variable gibt die Minuten zurück, die dem aktuellen Zeitpunkt entsprechen.

Syntax:

aktuelle_minute;

Rückgabe:

Real

Beispiel:

draw_text(32, 32, "Die Zeit ist " + string(aktuelle_Stunde) + ":" + string(aktuelle_Minute) + "." + string(aktuelle_Sekunde));

Der obige Code würde die aktuelle internationale Zeit auf den Bildschirm zeichnen.
