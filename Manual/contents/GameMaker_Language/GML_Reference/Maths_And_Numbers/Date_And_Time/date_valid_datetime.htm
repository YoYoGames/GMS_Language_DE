date_valid_datetime

Mit dieser Funktion können Sie einen datetime-Wert daraufhin überprüfen, ob er gültig ist (gibt true zurück) oder nicht (gibt false zurück). Beachten Sie, dass diese Funktion nur Datumswerte nach dem 1.1.1970 als gültig betrachtet und alles davor als falsch zurückgibt. Daher können Sie frühestens ab dem 1.1.1970 prüfen:

date_valid_datetime(1970, 01, 01, 0, 0, 0);

Syntax:

date_valid_datetime(Jahr, Monat, Tag, Stunde, Minute, Sekunde);

Argument Beschreibung
Jahr Das Jahr zum Prüfen.
Monat Der zu prüfende Monat.
Tag Der Tag zum Prüfen.
Stunde Die Stunde zum Prüfen.
Minute Die Minute zum Prüfen.
zweite Die zweite zur Überprüfung.

Rückgabe:

Boolesche

Beispiel:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

Dies setzt "mydatetime" auf den 15. September 2011, 10:03.30, wenn es ein gültiger Wert ist (was es ist).
