date_set_timezone

Legt die Basiszeitzone fest, die für alle übrigen Datums- und Zeitfunktionen verwendet wird. Diese Zeitzone kann entweder lokal (wie vom System eingestellt) oder UTC sein, und Sie würden eine der folgenden Konstanten verwenden, um zu definieren, welche verwendet wird (standardmäßig ist dies die lokale Zeit):

Konstant Beschreibung
zeitzone_lokal die lokale Zeitzone verwenden, wie vom System eingestellt
zeitzone_utc Koordinierte Weltzeit verwenden

Syntax:

date_set_timezone(timezone);

Argument Beschreibung
Zeitzone Die Zeitzone, die für die Basiszeit verwendet werden soll.

Rückgabe:

N/A

Beispiel:

if (date_get_timezone() != timezone_utc)
{
date_set_timezone(timezone_utc);
}

Dieser Code prüft die Basiszeitzoneneinstellung für das Spiel und ändert sie, wenn sie nicht UTC ist.
