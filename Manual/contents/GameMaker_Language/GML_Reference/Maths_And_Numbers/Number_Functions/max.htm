max

Diese Funktion gibt das Maximum der Eingangswerte zurück, von denen sie bis zu 16 haben kann. Zum Beispiel wird max(12, 96, 32, 75) 96 zurückgeben, da dies der höchste aller Eingangswerte ist.

Syntax:

max(val1, val2, ... max_val);

Argument Beschreibung
val0 ... max_val Die zu vergleichenden Werte.

Rückgabe:

Real

Beispiel:

x = max(x, 0);

Dadurch wird der Spieler daran gehindert, den linken Teil des Raums zu verlassen. Dies funktioniert, indem sein x ständig entweder auf sich selbst oder auf 0 gesetzt wird, je nachdem, welcher Wert größer ist. Wenn der Spieler den Raum nach links verlässt, wäre sein x kleiner als 0 (d. h. negativ), also wird er direkt zurückgesetzt. zurück.
