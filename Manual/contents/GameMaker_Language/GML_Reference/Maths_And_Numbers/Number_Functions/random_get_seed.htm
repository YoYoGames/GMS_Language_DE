random_get_seed

Um eine Zufallszahl zu erzeugen, startet GameMaker Studio 2 mit einer zufälligen Seed-Zahl. Mit dieser Funktion können Sie diesen Seed abrufen, der dann gespeichert werden kann, um eine bestimmte Serie von Zufallsereignissen zu reproduzieren. Nehmen wir zum Beispiel an, Sie haben ein prozedural generiertes Spiel, aber Sie wollen, dass der Spieler zu vorher entdeckten Gebieten zurückkehren kann. Nun, am Anfang eines jeden Gebiets können Sie den Zufallswert mit dieser Funktion speichern und jedes Mal, wenn Sie dorthin zurückkehren müssen, können Sie mit random_set_seed() verwenden, um den Seed auf den vorherigen Wert zu setzen und so GameMaker Studio 2 zu zwingen, denselben Satz von "Zufallsfunktionen" zu reproduzieren, den es zuvor verwendet hat.

Syntax:

random_get_seed();

Rückgabe:

Real

Beispiel:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

Der obige Code ermittelt den Zufallswert und speichert ihn in einer Ini-Datei.
