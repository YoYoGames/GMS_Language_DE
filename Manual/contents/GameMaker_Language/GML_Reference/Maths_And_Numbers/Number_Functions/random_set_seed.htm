random_set_seed

Um eine Zufallszahl zu erzeugen, startet GameMaker Studio 2 mit einer zufälligen Seed-Zahl. Mit dieser Funktion können Sie diesen Seed auf einen bekannten Wert setzen und so "erzwingen", dass das Ergebnis aller nachfolgenden Zufallsereignisse bei jedem Programmdurchlauf das gleiche ist laufen. Diese Funktion kann zum Beispiel in Verbindung mit random_get_seed() verwendet werden, um prozedural generierte Inhalte zu erstellen und die Ergebnisse zu speichern, ohne große Savegames zu haben (Sie speichern nur den Seed, kein für irgendetwas anderes). Sollten Sie wirklich zufällige Ergebnisse für alles benötigen, sollten Sie die randomise() Funktion verwenden.

HINWEIS: Während dieser Seed auf jeder Zielplattform konsistente Ergebnisse liefert, können die Ergebnisse zwischen den Plattformen aufgrund der unterschiedlichen Arbeitsweise der einzelnen Ziele variieren.

Syntax:

random_set_seed(val);

Argument Beschreibung
val Das Saatgut zum Setzen.

Rückgabe:

N/A

Beispiel:

if debug
{
random_set_seed(1);
}

Der obige Code setzt die Zufallsauswahl nur dann auf 1, wenn die Variable "debug" wahr ist.
