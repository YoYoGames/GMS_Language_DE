rund

Wie der Name schon sagt, nimmt round() eine reelle Zahl und rundet sie auf die nächste ganze Zahl auf oder ab. In dem speziellen Fall, dass die übergebene Zahl genau eine halbe Ganzzahl ist (1,5, 17,5, -2,5 usw.), wird die Zahl auf den nächsten geraden Wert gerundet, z. B. würde 2,5 auf 2 gerundet werden, während 3,5 auf 4 gerundet wird. Diese Art der Rundung wird Banker-Rundung genannt und ergibt bei einer großen Anzahl von Iterationen oder bei Verwendung von Fließkomma-Mathematik eine statistisch bessere Rundung als der traditionellere Ansatz "aufrunden, wenn über 0,5 und sonst abrunden".

Das bedeutet, wenn der Bruchteil eines Wertes 0,5 ist, dann ist das gerundete Ergebnis die gerade ganze Zahl, die dem Eingabewert am nächsten liegt. So wird z. B. 23,5 zu 24, ebenso wie 24,5, während -23,5 zu -24 wird, ebenso wie -24,5. Diese Methode behandelt positive Diese Methode behandelt positive und negative Werte symmetrisch, ist also frei von Vorzeichenverzerrungen, und, was noch wichtiger ist, für vernünftige Verteilungen von Werten ist der erwartete (durchschnittliche) Wert der gerundeten Zahlen derselbe wie der der ursprünglichen Zahlen.

Syntax:

rund(n);

Argument Beschreibung
n Die zu rundende Zahl.

Rückgabe:

Integer

Beispiel:

score += round(hp / 5);

Der obige Code fügt eine gerundete Ganzzahl zum Score-Wert hinzu.
