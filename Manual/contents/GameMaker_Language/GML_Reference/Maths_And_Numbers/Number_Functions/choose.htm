Wählen Sie

Manchmal möchten Sie etwas anderes als Zahlen für eine Zufallsauswahl angeben, oder die gewünschten Zahlen befinden sich nicht in einer echten Reihenfolge oder innerhalb eines festgelegten Bereichs. In diesen Fällen würden Sie choose() verwenden, um ein zufälliges Ergebnis zu erzeugen. Nehmen wir zum Beispiel an Sie wollen ein Objekt mit einem zufälligen Sprite am Anfang erzeugen, dann könnten Sie diese Funktion verwenden, um den Sprite-Index auf eines aus einer Menge von gegebenen Sprites zu setzen. Beachten Sie, dass Sie so viele Argumente haben können, wie Sie benötigen (beachten Sie, dass mehr Argumente bedeuten, dass die Funktion beim Parsen langsamer sein wird).

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels den gleichen Wert zurück, da GameMaker Studio 2 jedes Mal den gleichen anfänglichen Zufallswert erzeugt, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start Ihres Spiels.

Syntax:

choose(val0, val1, val2... max_val);

Argument Beschreibung
val0... max_val Ein Eingangswert, der eine Zeichenkette, eine Ganzzahl, eine Variable oder eine Konstante sein kann.

Rückgabe:

Eines der angegebenen Argumente

Beispiel:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

Der obige Code verwendet choose, um eine Reihe von Eigenschaften für die Instanz zu setzen.
