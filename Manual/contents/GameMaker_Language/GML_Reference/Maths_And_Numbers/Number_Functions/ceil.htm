Decke

Diese Funktion nimmt eine beliebige reelle Zahl und rundet sie auf die nächste Ganzzahl auf. Seien Sie vorsichtig mit dieser Funktion, da ein häufiger Fehler darin besteht, einen zufälligen Wert aufzurunden und zu erwarten, dass er immer größer als 1 ist, d. h.:

int = ceil(random(5));

Nun würden Sie erwarten, dass dieser Code immer eine ganze Zahl zwischen 1 und 5 ergibt, aber das ist nicht immer der Fall, da es eine sehr kleine Möglichkeit gibt, dass die Zufallsfunktion 0 zurückgibt, und das Aufrunden von 0 ergibt immer noch 0. Dies ist eine geringe Möglichkeit, sollte aber bei der Verwendung dieser Funktion in Betracht gezogen werden.

Syntax:

ceil(val);

Argument Beschreibung
x Die zu ändernde Nummer.

Rückgabe:

Integer

Beispiel:

val = ceil( 3.4 );

Dadurch wird val auf 4 gesetzt.
