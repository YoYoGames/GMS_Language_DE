randomisieren

Diese Funktion setzt den Seed auf einen Zufallswert. Sollten Sie einen konstanten Wert über mehrere Durchläufe eines Spiels benötigen, sollten Sie die Funktion random_set_seed(). Bitte beachten Sie, dass bei der Verwendung der Zufallszahlen Funktionen im GameMaker Studio 2 immer derselbe Anfangswert verwendet wird, da dies die Fehlersuche und das Debugging erheblich erleichtert. Wenn Sie mit echtem Zufall testen wollen, sollten Sie diese Funktion beim Start Ihres Spiels aufrufen. Die Funktion wird gibt den neuen randomisierten Seed-Wert zurück (eine vorzeichenlose 32-Bit-Ganzzahl).

Syntax:

randomise();
// oder
randomize();

Rückgabe:

Integer (32-Bit-Wert ohne Vorzeichen)

Beispiel:

randomise();

Der obige Code wird den Seed randomisieren.
