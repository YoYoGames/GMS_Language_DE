irregulär

Diese sehr nützliche Funktion gibt nur ganze Zahlen zurück. Wenn Sie also zum Beispiel eine Zufallszahl von 0 bis 9 erhalten möchten, können Sie irandom(9) verwenden, und es wird eine Zahl von 0 bis einschließlich 9 zurückgegeben. Beachten Sie, dass auch Fließkommazahlen verwendet werden können, aber der Der obere Wert nach dem Punkt wird jedoch ausgeschlossen. irandom(9.7) gibt also nur einen Wert von 0 bis 9 zurück. Die Funktion hat eine Obergrenze von $7fffffffffffffffLL, daher ist Vorsicht geboten, wenn Sie sehr große Zahlen verwenden.

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start Ihres Spiels.

Syntax:

irandom(n);

Argument Beschreibung
n Der obere Bereich, aus dem die Zufallszahl ausgewählt wird.

Rückgabe:

Integer

Beispiel:

if (irandom(9) == 1)
{
score += 100;
}

Dadurch ergibt sich eine Chance von eins zu zehn (da 0 enthalten ist), dass die Punktzahl um 100 erhöht wird.
