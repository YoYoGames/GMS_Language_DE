Zufallsbereich

Diese Funktion gibt eine Zufallszahl zwischen dem angegebenen Bereich zurück, wobei dieser Rückgabewert keine ganze Zahl sein muss. Zum Beispiel gibt random_range(20,50) eine Zufallszahl zwischen 20 und 50 zurück, aber der Wert kann eine reelle Zahl wie 38,65265 sein. Als Eingabeargumente können auch reelle Zahlen verwendet werden.

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() am Anfang Ihres Spiels.

Syntax:

random_range(n1, n2);

Argument Beschreibung
n1 Das untere Ende des Bereichs, aus dem die Zufallszahl ausgewählt wird.
n2 Das obere Ende des Bereichs, aus dem die Zufallszahl ausgewählt wird.

Rückgabe:

Real

Beispiel:

score += random_range(500, 600);

Dies trägt zwischen 500 und 600 zur Gesamtpunktzahl bei.
