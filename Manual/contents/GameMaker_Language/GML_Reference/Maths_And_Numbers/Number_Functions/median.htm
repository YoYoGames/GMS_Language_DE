Median

Diese Funktion gibt den Median der Eingabewerte zurück, d. h. den mittleren Wert. Wenn die Anzahl der Argumente gerade ist, wird der kleinere der beiden mittleren Werte zurückgegeben und die Funktion kann so viele Argumente wie nötig haben (beachten Sie, dass mehr Argumente bedeutet, dass die Funktion langsamer geparst wird), die alle reelle Werte sein müssen. Das bedeutet, dass z. B. median(43, 12, 25, 19, 6) 19 zurückgeben würde, da dies der Mittelwert zwischen allen anderen Werten ist.

Syntax:

median(val1, val2, ... max_val);

Argument Beschreibung
val0 ... max_val Die zu vergleichenden Werte.

Rückgabe:

Real

Beispiel:

x = Median( 0, x, Raum_Breite );
y = Median( 0, y, Raum_Höhe );

Dadurch wird verhindert, dass der Spieler eine beliebige Seite des Raums verlässt, indem der Median als Klammer verwendet wird. Wenn sich der Spieler z. B. nach links von der Raumgrenze bewegt, wird sein x kleiner als 0 sein. wird 0 sein, so dass der Spieler auf (0,y) gesprungen wird.
