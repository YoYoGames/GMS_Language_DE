math_set_epsilon

Epsilon ist ein Wert, der verwendet wird, um zu bestimmen, ob zwei Zahlen, die Rundungsfehlern unterliegen, nahe genug sind, um als "gleich" zu gelten. Es ist nützlich, wenn Sie mit Fließkomma-Mathematik arbeiten, da es die "Rundungsfehler" reduzieren kann, die dazu führen, dass bestimmte Operationen Werte zurückgeben, die falsch oder anders als erwartet erscheinen. Wir haben z. B. einen Wert zum Bildindex eines Sprites addiert und erwarten, dass das Ergebnis eine einzelne Ganzzahl ist, aber aufgrund der Natur der Fließkommamathematik ist der tatsächliche Wert am Ende etwa 5,0000002, so dass die folgende Prüfung durchgeführt werden muss:

if image_index == 5
{
//do something
}

Der Code verhält sich nicht wie erwartet und gibt false zurück. Wenn wir jedoch den Epsilon-Wert auf 0,000001 setzen, wird der Wert von image_index auf die nächste reelle Zahl gerundet, die +/- 0,000001 des ursprünglichen Wertes ist, wodurch der obige Vergleich "true" zurückgibt.

Der Epsilon-Wert wird für alle folgenden Ganzzahloperatoren verwendet:

  • < : Weniger als
  • > : Größer als
  • == : Gleichbedeutend mit
  • <= : Kleiner als oder gleich
  • >= : Größer als oder gleich
  • != : Nicht gleich

Beachten Sie, dass bei einem Epsilon-Wert von 0 alle Rundungen deaktiviert werden und bei einem Wert von 1 ein Fehler auftritt.

Syntax:

math_set_epsilon(epsilon);

Argument Beschreibung
epsilon Der neue Epsilon-Wert (von 0 bis 0,999999999).

Rückgabe:

Real

Beispiel:

math_set_epsilon(0,0001);

Damit wird der Epsilon-Wert für alle weiteren Fließkommaoperationen festgelegt.
