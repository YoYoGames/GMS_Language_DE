min

Diese Funktion gibt das Minimum der Eingabewerte zurück, von denen sie so viele haben kann, wie Sie benötigen (beachten Sie, dass mehr Argumente bedeuten, dass die Funktion langsamer analysiert wird). Zum Beispiel gibt min(12, 96, 32, 75) 12 zurück, da dies der der niedrigste aller Eingabewerte ist.

Syntax:

min(val1, val2, ... max_val);

Argument Beschreibung
val0 ... max_val Die zu vergleichenden Werte.

Rückgabe:

Real

Beispiel:

x = min(x, Raum_Breite);

Dadurch wird verhindert, dass der Spieler den rechten Teil des Raums verlässt. Dies funktioniert, indem sein x ständig entweder auf sich selbst oder auf die Variable room_width gesetzt wird, je nachdem, was kleiner ist. Wenn der Spieler den rechten Teil des Raums verlässt, wäre sein x größer als die Variable room_width, also wird er direkt zurückgesetzt werden.
