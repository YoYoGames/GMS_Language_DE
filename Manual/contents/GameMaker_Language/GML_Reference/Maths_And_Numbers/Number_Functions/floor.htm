Boden

Gibt die Untergrenze von n zurück, d. h. n abgerundet auf eine ganze Zahl. Dies ist ähnlich wie die round() Funktion, aber sie rundet nur ab, unabhängig vom Dezimalwert, so dass floor(5,99999) 5 zurückgibt, ebenso wie floor(5.2), floor(5.6457) usw...

Syntax:

Boden(n);

Argument Beschreibung
n Die Nummer zum Stockwerk.

Rückgabe:

Real

Beispiel:

val = floor( 3,9 );

Dadurch wird val auf 3 gesetzt.
