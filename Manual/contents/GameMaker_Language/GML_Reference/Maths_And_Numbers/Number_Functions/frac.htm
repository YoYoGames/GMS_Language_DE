frac

Diese Funktion gibt den Nachkommateil von n zurück, d. h. den Teil hinter dem Dezimalpunkt. Sie gibt nur die Dezimalstellen hinter dem Punkt eines Wertes zurück, also gibt frac(3,125) 0,125 zurück, frac(6,921) 0,921, usw...

Syntax:

frac(n);

Argument Beschreibung
n Die zu ändernde Nummer.

Rückgabe:

Real

Beispiel:

val = frac(3.4);

Dadurch wird val auf 0,4 gesetzt.
