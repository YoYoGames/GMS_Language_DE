Zeichen

Diese Funktion gibt zurück, ob eine Zahl positiv, negativ oder keines von beiden ist und gibt jeweils 1, -1, 0 zurück. Zum Beispiel - sign(458) gibt 1 zurück, sign(-5) gibt -1 zurück und sign(0) gibt 0 zurück.

Syntax:

vorzeichen(n);

Argument Beschreibung
n Die Zahl, für die das Vorzeichen ermittelt werden soll.

Rückgabe:

Integer

Beispiel:

y += sign(y - Maus_y);

Der obige Code addiert 1, -1 oder 0 auf y, je nach dem Ergebnis von y - mouse_y.
