zufällig

Diese Funktion ist gut für Wahrscheinlichkeiten geeignet, bei denen die Rückgabe einer ganzen Zahl nicht erforderlich ist. Zum Beispiel gibt random(100) einen Wert von 0 bis 99 zurück, aber dieser Wert kann 22,56473 sein! Sie können in dieser Funktion auch reelle Zahlen und keine ganzen Zahlen verwenden Funktion verwenden, z. B. random(0,5), das einen Wert zwischen 0 und 0,4999999 zurückgibt.

HINWEIS: Diese Funktion gibt bei jedem neuen Durchlauf des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise() am Anfang Ihres Spiels.

Syntax:

random(n);

Argument Beschreibung
n Der obere Bereich, aus dem die Zufallszahl ausgewählt wird.

Rückgabe:

Real

Beispiel:

if (random(10) >= 9)
{
score += 100;
}

Dadurch ergibt sich eine Wahrscheinlichkeit von etwa eins zu zehn, dass die Punktzahl um 100 erhöht wird.
