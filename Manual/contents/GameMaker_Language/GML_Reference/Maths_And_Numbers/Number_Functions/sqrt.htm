sqrt

Wenn Sie eine Zahl mit sich selbst multiplizieren, erhalten Sie das Quadrat dieser Zahl, aber manchmal möchten Sie das Gegenteil tun und die Quadratwurzel einer Zahl erhalten. Um also herauszufinden, welche Zahl mit sich selbst multipliziert wurde, um einen beliebigen positiven Wert zu erhalten, verwenden wir diese Funktion. Zum Beispiel: sqrt(9) gibt 3 zurück, da 3*3=9.

Syntax:

sqrt(val);

Argument Beschreibung
val Die Zahl, aus der die Quadratwurzel gezogen werden soll.

Rückgabe:

Real

Beispiel:

num = sqrt(exp);

Der obige Code setzt die Variable "num" so, dass sie die Quadratwurzel des in "exp" enthaltenen Wertes enthält.
