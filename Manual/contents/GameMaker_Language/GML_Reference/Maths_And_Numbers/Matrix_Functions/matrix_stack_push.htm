matrix_stack_push

Diese Funktion schiebt die angegebene Matrix auf den Matrix-Stack.

Syntax:

matrix_stack_push(matrix);

Argument Beschreibung
Matrix Der zu verwendende Matrixindex.

Rückgabe:

N/A

Beispiel:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Der obige Code erstellt zwei neue Matrizen und schiebt sie dann auf den Matrix-Stack. Die Weltmatrix wird dann an die Spitze des Stapels gesetzt, ein Sprite wird gezeichnet und die Transformationen werden dann vom Stapel gepoppt.
