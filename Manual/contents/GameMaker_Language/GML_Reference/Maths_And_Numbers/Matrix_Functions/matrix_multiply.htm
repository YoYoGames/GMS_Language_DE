matrix_multiplizieren

Mit dieser Funktion können Sie zwei Matrix-Arrays miteinander multiplizieren, um eine neue Transformationsmatrix zu erstellen. Die Funktion gibt den Index der neuen Matrix zurück, der zur späteren Verwendung in einer Variablen gespeichert werden sollte.

HINWEIS: Sie können mit dieser Funktion keine Matrixkonstante als Argument verwenden. Wenn Sie also die (z. B.) View-Matrix mit einer von Ihnen erstellten benutzerdefinierten Matrix multiplizieren wollen, müssen Sie zuerst matrix_get() aufrufen und die Werte der Ansichtsmatrix einer Array-Variablen zuweisen, diese dann mit Ihrer benutzerdefinierten Matrix multiplizieren und dann die gewählte Matrix (Ansicht, Projektion oder Welt) setzen.

Syntax:

matrix_multiply(matrix1, matrix2);

Argument Beschreibung
Matrix1 Der erste zu verwendende Matrixindex.
Matrix2 Der zweite zu verwendende Matrixindex.

Rückgabe:

Matrix-Index (4x4-Matrix)

Beispiel:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

Der obige Code ermittelt die aktuelle Ansichtsmatrix, multipliziert diese dann mit einer benutzerdefinierten Matrix und verwendet den resultierenden Matrixindex, um die Ansichtsmatrix erneut zu setzen.
