matrix_get

Diese Funktion gibt ein Array mit 16 Werten zurück, die dem angegebenen 4x4-Matrixtyp entsprechen, wobei Zeile 1 die Elemente [0 - 3], Zeile 2 die Elemente [4 -7], usw. sind. (siehe das Bild auf der Hauptseite). Die verfügbaren Matrizen sind view, projection und world, für die Sie eine der folgenden Konstanten verwenden würden:

Konstant Beschreibung
matrix_view Die aktuelle Ansichtsmatrix
matrix_projektion Die aktuelle Projektionsmatrix
matrix_welt Die aktuelle Weltmatrix

Syntax:

matrix_get(type);

Argument Beschreibung
Typ Der Typ der Matrix, für die die Werte ermittelt werden sollen (siehe die oben aufgeführten Konstanten )

Rückgabe:

Array (16 Elemente)

Beispiel:

v_array = matrix_get(matrix_view);

Der obige Code holt die Werte der aktuellen Ansichtsmatrix und füllt das Array "v_array" damit.
