matrix_transform_vertex

Diese Funktion transformiert einen Vektor durch eine Matrix. Sie übergeben die ID der Transformationsmatrix (wie sie von der Funktion matrix build()zurückgegeben wird), sowie die x-, y- und z-Werte für den zu transformierenden Vektor. Die Funktion gibt zurück ein Array mit 3 Elementen zurück, wobei:

  • array[0] = x
  • array[1] = y
  • array[2] = z.

Syntax:

matrix_transform_vertex(matrix, x, y, z);

Argument Beschreibung
Matrix Die zu verwendende Matrix
x Die x-Komponente des Transformationsvektors
y Die y-Komponente des Transformationsvektors
z Die z-Komponente des Transformationsvektors

Rückgabe:

Array (3 Elemente)

Beispiel:

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

Der obige Code transformiert die angegebenen Werte mit Hilfe der in der Variablen "t_matrix" gespeicherten Matrix und speichert sie im Array "verts".
