matrix_build_projection_perspective_fov

Diese Funktion erstellt eine perspektivische Projektionsmatrix auf der Grundlage des Sichtfelds unter Verwendung der unten aufgeführten Parameter.

Syntax:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);

Argument Beschreibung
fov Der Winkel des Sichtfelds.
Aspekt Das Seitenverhältnis des Sichtfelds.
znear Die nahe Clipping-Ebene.
zfar Die entfernte Clipping-Ebene.

Rückgabe:

Matrix-Index

Beispiel:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erstellt eine Sichtfeld-Projektionsmatrix, die dann in einer Variablen gespeichert wird. Diese Matrix wird dann verwendet, um die Kamera einzurichten, die dem View Port[0] zugewiesen ist.
