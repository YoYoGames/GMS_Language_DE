matrix_set

Diese Funktion nimmt ein Array mit 16 Werten entgegen, die einem gegebenen 4x4-Matrixtyp entsprechen, wobei die Elemente [0 - 3] die Zeile 1, die Elemente [4 -7] die Zeile 2 usw. wären... (siehe das Bild auf der Hauptseite). Sie können eine solche Matrix mit den Befehlen matrix_build() oder matrix_get() Funktionen erstellen oder einfach das Array selbst aufbauen und dieses an die Funktion übergeben. Die verfügbaren Matrix Typen sind view, projection und world, für die Sie eine der folgenden Konstanten verwenden würden:

Konstant Beschreibung
matrix_view Die aktuelle Ansichtsmatrix
matrix_projektion Die aktuelle Projektionsmatrix
matrix_welt Die aktuelle Weltmatrix

Syntax:

matrix_set(typ, matrix);

Argument Beschreibung
Typ Der Typ der Matrix, für die die Werte ermittelt werden sollen (siehe die oben aufgeführten Konstanten )
Matrix Die Matrixdaten als Array

Rückgabe:

NA

Beispiel:

matrix_set(matrix_world, m_array);

Der obige Code setzt die Werte der aktuellen Weltmatrix auf diejenigen, die in der Array-Matrix "m_array" gespeichert sind.
