matrix_build_projektion_perspektive

Diese Funktion erstellt eine perspektivische Projektionsmatrix basierend auf den Dimensionen der nahen Ebene unter Verwendung der unten aufgeführten Parameter.

Syntax:

matrix_build_projection_perspective(width, height, znear, zfar);

Argument Beschreibung
w Die Breite der Projektion.
h Die Höhe der Projektion.
znear Die nahe Clipping-Ebene.
zfar Die entfernte Clipping-Ebene.

Rückgabe:

Matrix-Index

Beispiel:

var projmat = matrix_build_projection_perspective(640, 480, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erstellt eine perspektivische Projektion und verwendet dann die zurückgegebene Matrix, um die dem Viewport[0] zugewiesene Kamera einzustellen.
