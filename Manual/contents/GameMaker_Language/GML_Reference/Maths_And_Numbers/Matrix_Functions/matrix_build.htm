matrix_build

Diese Funktion kann verwendet werden, um Ihre eigene benutzerdefinierte Matrix zu erstellen und gibt einen Indexwert für diese neue Matrix zurück, der in einer Variablen für zukünftige Referenz und Verwendung gespeichert werden sollte. Die Matrix selbst wird als Array mit 16 Werten erstellt, wobei die ersten 4 Elemente die Zeile 1, die zweiten 4 Elemente die Zeile 2 usw... einer 4x4-Matrix sind.

HINWEIS: Wenn Sie eine neue Matrix auf diese Weise aufbauen, ist die Reihenfolge der Bedienung YXZ.

Syntax:

matrix_build(x, y, z, xRotation, yRotation, zRotation, xSkala, ySkala, zSkala);

Argument Beschreibung
x Die x-Komponente des Translationsvektors.
y Die y-Komponente des Translationsvektors.
z Die z-Komponente des Translationsvektors.
xRotation Der Winkel, der um die x-Achse gedreht werden soll (in Grad °).
yrotation Der Winkel, der um die y-Achse gedreht werden soll (in Grad °).
xzotation Der Winkel, der um die z-Achse gedreht werden soll (in Grad °).
x-Skala Der Betrag der x-Skala.
y-Skala Der y-Skalenbetrag.
zskala Der Betrag der z-Skala.

Rückgabe:

Matrix-Index (4x4 Matrix)

Beispiel:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Der obige Code erstellt eine neue Matrixtransformation und speichert den resultierenden Matrixindex in der Variablen "t_matrix".
