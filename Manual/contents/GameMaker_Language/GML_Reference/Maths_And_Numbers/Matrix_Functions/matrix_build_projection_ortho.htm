matrix_build_projection_ortho

Diese Funktion erstellt eine orthografische Projektionsmatrix basierend auf den unten angegebenen Parametern (dies ist die Standard-Projektionsmethode, die verwendet wird, wenn Sie einen Raum in GameMaker Studio 2 erstellen, ohne etwas zu ändern). Manchmal müssen Sie umschalten von einer perspektivischen Projektion zu einer orthografischen Projektion wechseln, wofür diese Funktion gedacht ist. Sie wird typischerweise verwendet, um ein Overlay zu zeichnen, z. B. um den Spielstand oder andere Aspekte zu zeigen, da dies eine "flache" Ansicht der gezeichneten Elemente ergibt (d. h.: keine Perspektive) in einem 3D-Spiel. Sehen Sie sich das Bild unten an, um eine Vorstellung vom Unterschied zwischen orthografischen und perspektivischen Ansichten zu bekommen.

HINWEIS: Möglicherweise müssen Sie auch die Entfernung verdeckter Flächen vorübergehend ausschalten, wenn Sie möchten, dass die Informationen unabhängig vom aktuellen Tiefenwert gezeichnet werden.

Syntax:

matrix_build_projection_ortho(width, height, znear, zfar);

Argument Beschreibung
w Die Breite der Projektion.
h Die Höhe der Projektion.
znear Die nahe Clipping-Ebene.
zfar Die entfernte Clipping-Ebene.

Rückgabe:

Matrix-Index

Beispiel:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erstellt eine neue Look-at-Matrix und orthografische Matrix, speichert ihre IDs in lokalen Variablen und verwendet sie dann, um die Ansichts- und Projektionsmatrizen für die Kamera zu setzen, die dem Viewport[0] zugewiesen ist.
