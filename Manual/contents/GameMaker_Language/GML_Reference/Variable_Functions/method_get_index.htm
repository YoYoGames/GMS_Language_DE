methode_get_index

Mit dieser Funktion können Sie den Skriptindex für das Skript abrufen, in dem die Methode definiert wurde. Wenn die Methode nicht in einem Skript definiert wurde, gibt die Funktion -1, andernfalls gibt sie den Indexwert für das Skript zurück.

Syntax:

method_get_index(method);

Argument Beschreibung
Methode Die zu prüfende Methodenvariable

Rückgabe:

Integer

Beispiel:

var _index = method_get_index(light_setup);
if _index != -1
{
show_debug_message(script_get_name(_index));
}

Der obige Code prüft den Skriptindex auf die Methode "light_setup" und gibt dann, wenn er nicht -1 ist, den Skriptnamen auf der Konsole aus.
