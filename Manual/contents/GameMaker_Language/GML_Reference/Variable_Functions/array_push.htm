array_push

Mit dieser Funktion können Sie einen Wert (oder Werte) an das Ende eines Arrays schieben, ohne die Länge des Arrays kennen zu müssen. Die Funktion erfordert die Angabe einer Variablen, die das Array enthält, sowie mindestens einen zu verschiebenden Wert, obwohl Sie können optional weitere Argumente angeben, die dann alle in fortlaufender Reihenfolge an das Array angehängt werden.

Syntax:

array_push(Variable, Wert, [Wert], [Wert], [usw...]);

Argument Beschreibung
variabel Die Variable, die das Array enthält.
Wert Der Wert, der an das Ende des Arrays geschoben werden soll
[Wert], [Wert], [etc...] OPTIONAL! Weitere Werte, die auf das Array geschoben werden sollen

Rückgabe:

N/A

Beispiel:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

Der obige Code schiebt vier Werte an das Ende des angegebenen Arrays.
