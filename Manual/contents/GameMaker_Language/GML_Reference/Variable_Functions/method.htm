Methode

Mit dieser Funktion können Sie eine beliebige Funktion, die zuvor definiert wurde, an eine bestimmte Instanz oder Struktur binden und so eine Methodenvariable erzeugen, die später verwendet werden kann. Sie geben die zu verwendende Instanz-ID (es muss eine Instanz sein, die aktiv und im Raum ist, und kann keine Objekt-ID sein) oder eine struct-Referenz an, sowie die ID der Funktion, die Sie binden möchten. Die Funktion wird eine Methode zurückgeben, die von der Variablen, der sie zugewiesen ist, aufgerufen werden kann (siehe das Codebeispiel unten). Es ist erwähnenswert, dass Sie sowohl eingebaute Funktionen als auch benutzerdefinierte Funktionen binden können. definierte Funktionen binden können, und Sie können auch undefiniert als Instanz/Strukt-Argument angeben, was bedeutet, dass der aktuelle selbst Bereich für die Bindung verwendet wird.

Syntax:

method(struct_ref_oder_instance_id, function);

Argument Beschreibung
struct_ref_oder_instanz_id Die eindeutige Referenz oder der ID-Wert der zu verwendenden Struktur oder Instanz (oder undefiniert für self)
Funktion Die ID der zu verwendenden Funktion

Rückgabe:

Methode

Beispiel:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
feind_func = method(_inst, feind_ai);
}

Der obige Code prüft, ob an der angegebenen Position eine Instanz existiert, und wenn ja, dann wird eine Funktion an die Instanz gebunden und als Methodenvariable "enemy_func" zurückgegeben.
