array_set

Mit dieser Funktion können Sie den Wert eines Indexes in einem Array auf einen Wert setzen. Die Funktion erfordert, dass Sie eine Variable bereitstellen, die das Array sowie den zu setzenden Index und den Wert, auf den er gesetzt werden soll, enthält. Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden Arrays verwendet werden, solange Sie bei der Übergabe des Array-Indexes angeben, welche Dimension Sie setzen wollen, indem Sie diesem Muster folgen:

// 1D-Array
array_set(mein_Array, 0, 100);
// 2D-Array
array_set(mein_Array[0], 0, 100);
// 3D-Array
array_set(mein_Array[0][0], 0, 100);
// usw...

Syntax:

array_set(Variable, Index, Wert);

Argument Beschreibung
variabel Die Variable, die das Array enthält.
index Der Index des Arrays, für den der Wert gesetzt werden soll.
Wert Der einzustellende Wert.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < 10; ++i;)
{
array_set(score_array, i, i*100));
}

Der obige Code setzt die ersten 10 Elemente im angegebenen Array auf einen Wert.
