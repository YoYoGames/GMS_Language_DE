ist_ptr

Diese Funktion gibt zurück, ob eine gegebene Variable ein Zeiger ist oder nicht. In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker Studio 2 hat, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem je nachdem, ob der Wert ein Zeiger ist oder nicht.

Syntax:

is_ptr(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

if !is_ptr(val)
{
show_debug_message("Not a valid texture!");
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie einen Zeiger enthält, und wenn dies nicht der Fall ist, wird eine Meldung in der Debug-Konsole angezeigt.
