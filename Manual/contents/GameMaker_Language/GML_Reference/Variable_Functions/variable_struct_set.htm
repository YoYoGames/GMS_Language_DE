variable_struct_set

Mit dieser Funktion können Sie den Wert einer gegebenen Variablen in einer struct setzen. Sie übergeben die struct-Referenz sowie den Namen der Variablen, deren Wert gesetzt werden soll, als Zeichenkette (siehe Beispielcode unten), und schließlich den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die Variable nicht bereits in der Struktur vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

variable_struct_set(struct, name, val);

Argument Beschreibung
struct Die zu setzende Strukturreferenz
Name Der Name der zu setzenden Variablen (als String)
val Der Wert, auf den die Variable gesetzt werden soll

Rückgabe:

N/A

Beispiel:

if !variable_struct_exists(mystruct, "shields")
{
variable_struct_set(mystruct, "shields", 0);
}

Der obige Code prüft, ob die angegebene Variable in der angegebenen Struktur existiert, und wenn nicht, wird sie erstellt und auf 0 gesetzt.
