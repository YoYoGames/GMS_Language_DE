array_einfügen

Mit dieser Funktion können Sie einen Wert (oder mehrere Werte) an einer beliebigen Position in ein Array einfügen. Die Funktion benötigt eine Variable, die das Array enthält, den Index (die Position) im Array, an der eingefügt werden soll, sowie mindestens einen einzufügenden Wert. Sie können jedoch optional weitere Argumente angeben, die dann alle in fortlaufender Reihenfolge ab dem angegebenen Index in das Array eingefügt werden.

Beachten Sie, dass, wenn der angegebene Index außerhalb der Länge des Arrays liegt, die Werte am angegebenen Index hinzugefügt werden und alle leeren Stellen im Array zwischen dem letzten Wert und den eingefügten Werten auf einen Standardwert von 0 gesetzt werden.

Syntax:

array_insert(variable, index, wert, [wert], [wert], [usw...]);

Argument Beschreibung
variabel Die Variable, die das Array enthält.
index Der Index (Position) im Array zum Einfügen des/der Werte(s)
Wert Der einzufügende Wert
[Wert], [Wert], [etc...] (Optional) Weitere Werte zum Einfügen in das Array

Rückgabe:

N/A

Beispiel:

array_insert(score_array, score_rank, current_score);

Der obige Code fügt den aktuellen Punktestand in das Punkte-Array ein, und zwar an dem durch die Variable score_rank definierten Index.
