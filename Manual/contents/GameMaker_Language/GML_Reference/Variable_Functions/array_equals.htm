array_equals

Mit dieser Funktion können Sie prüfen, ob zwei Arrays gleich (äquivalent oder gleich) sind. Sie geben die beiden zu prüfenden Arrays an, und die Funktion gibt true zurück, wenn sie gleich sind (entweder äquivalent oder gleich), oder false, wenn sie es nicht sind. Beachten Sie, dass dies nicht dasselbe ist wie die Prüfung, ob zwei Arrays gleich sind, indem Sie == verwenden, das nicht prüft, ob die beiden Arrays gleichwertige Werte enthalten, sondern nur, ob die Arrays auf das gleichen Ausgangs-Array verweisen. Zum Beispiel:

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
{
show_debug_message( "Dies wird niemals ausgelöst, da a und b nicht das GLEICHE Array referenzieren" );
}
if (array_equals(a, b))
{
show_debug_message( "Das wird jetzt ausgelöst" );
}

Syntax:

array_equals(var1, var2);

Argument Beschreibung
var1 Der Index des ersten Arrays.
var2 Der Index des zweiten Arrays.

Rückgabe:

Boolesche

Beispiel:

if !array_equals(inventar_array, item_array)
{
var len = array_length(inventar_array);
array_copy(item_array, 0, inventar_array, 0, len);
}

Der obige Code prüft zwei Arrays, um zu sehen, ob sie gleichwertige Werte enthalten, und wenn nicht, dann kopiert der Code den gesamten Inhalt des Arrays "inventory_array" in das Array "item_array".
