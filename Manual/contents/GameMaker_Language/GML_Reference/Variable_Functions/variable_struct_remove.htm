variable_struct_remove

Mit dieser Funktion können Sie eine Variable aus einer struct entfernen. Sie übergeben die struct-ID, aus der die Variable entfernt werden soll, und den String-Namen der zu entfernenden Variablen.

Syntax:

variable_struct_set(struct, name);

Argument Beschreibung
struct Die Strukturreferenz, aus der die Variable entfernt werden soll
Name Der Name der zu entfernenden Variablen (als String)

Rückgabe:

N/A

Beispiel:

if variable_struct_exists(mystruct, "shields")
{
variable_struct_remove(mystruct, "shields");
}

Der obige Code prüft, ob die angegebene Variable in der angegebenen Struktur vorhanden ist, und wenn ja, dann wird sie entfernt.
