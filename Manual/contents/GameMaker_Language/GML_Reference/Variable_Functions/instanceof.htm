instanceof

Diese Funktion kann verwendet werden, um den Namen der Funktion zu erhalten, die verwendet wurde, um eine struct zu erzeugen, wenn die struct mit einer Konstruktorfunktion und dem neu Operator erzeugt wurde. Sie übergeben die Variable mit der zu prüfenden struct-Referenz und die Funktion gibt entweder einen String mit dem Funktionsnamen oder undefiniert zurück. Beachten Sie, dass, wenn Sie Wenn Sie der Funktion ein struct-Literal übergeben (d. h. ein struct, das ohne Verwendung einer Konstruktorfunktion erstellt wurde), gibt sie einfach die Zeichenfolge "struct" zurück. Diese Funktion kann auch verwendet werden, um zu prüfen, ob eine struct-Referenz ein schwache Referenz oder in diesem Fall gibt die Funktion die Zeichenkette "weakref" anstelle des Namens der Funktion zurück, die das struct erstellt hat (weitere Informationen finden Sie in der Funktion weak_ref_create()).

Syntax:

instanceof(struct_id);

Argument Beschreibung
struct Die zu verwendende Strukturreferenz.

Rückgabe:

String oder undefiniert

Erweitertes Beispiel:

In diesem Beispiel müssen wir zunächst die Funktion definieren, die als Konstruktor für die struct verwendet wird:

function init_struct(_a, _b, _c) constructor
{
a = _a;
b = _b;
c = _c;
}

Diese Funktion kann dann zusammen mit dem new-Operator verwendet werden, um eine Struktur zu erstellen und sie mit den Variablen zu füllen, die auf die Werte der in der Funktion verwendeten Argumente gesetzt sind:

mystruct = new init_struct(10, 100, "Hello World");

Wir können dann den Namen der Funktion, die verwendet wurde, wie folgt erhalten:

var _name = instanceof(mystruct);
if is_string(_name)
{
show_debug_message(_name);
}
