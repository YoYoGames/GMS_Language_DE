variable_instanz_existiert

Mit dieser Funktion können Sie prüfen, ob eine Instanzbereichsvariable existiert oder nicht. Sie liefern den eindeutigen Wert der Instanz-ID (den Sie in den Instanz-Eigenschaften im Raum-Editor finden oder der zurückgegeben wird, wenn Sie die Funktion instance_create_layer()) sowie den Variablennamen, auf den geprüft werden soll, als String (siehe Beispielcode unten). Die Funktion gibt true zurück, wenn eine Variable mit dem angegebenen Namen für die Instanz existiert, und sonst false.

Syntax:

variable_instance_exists(instance_id, name);

Argument Beschreibung
instance_id Der eindeutige ID-Wert der zu prüfenden Instanz
Name Der Name der Variable, auf die geprüft werden soll (als String)

Rückgabe:

Boolesche

Beispiel:

if !variable_instance_exists(id, "shields"))
{
shields = 0;
}

Der obige Code prüft, ob die Variable "shields" in der Instanz, die den Code ausführt, vorhanden ist, und wenn nicht, wird sie erstellt und auf 0 initialisiert.
