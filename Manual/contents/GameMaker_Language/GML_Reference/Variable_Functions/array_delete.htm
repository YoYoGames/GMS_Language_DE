array_delete

Mit dieser Funktion können Sie einen Wert (oder mehrere Werte) an einer beliebigen Position aus einem Array löschen. Die Funktion erfordert, dass Sie eine Variable bereitstellen, die das Array, den Index (Position) im Array, von dem aus gelöscht werden soll, sowie die Anzahl der zu löschenden Werte enthält.

Syntax:

array_delete(variable, index, number);

Argument Beschreibung
variabel Die Variable, die das Array enthält.
index Der Index (Position) im Array, aus dem der oder die Werte gelöscht werden sollen.
Nummer Die Anzahl der zu löschenden Werte.

Rückgabe:

N/A

Beispiel:

array_delete(score_array, 0, 10);

Der obige Code löscht die ersten 10 Werte im angegebenen Array, beginnend bei Index 0 (die Indizes 0 bis 9 werden also aus dem Array entfernt).
