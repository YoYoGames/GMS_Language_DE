is_vec3

In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker Studio 2 enthält, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem je nachdem, ob der Wert ein 3-Werte-Vektor ist oder nicht.

Syntax:

is_vec3(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

if !is_vec3(val)
{
show_debug_message("Not a 3 value vector!");
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie einen vec3 enthält, und wenn nicht, dann zeigt er eine Meldung in der Debug-Konsole an.
