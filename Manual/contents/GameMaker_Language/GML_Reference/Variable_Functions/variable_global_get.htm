variable_global_get

Mit dieser Funktion können Sie den Wert einer gegebenen benannten globalen Variablen abrufen. Sie geben den Namen der globalen Variable, deren Wert Sie abrufen möchten, als String an (siehe Beispielcode unten) und die Funktion gibt den Wert zurück, den die globale Variable enthält, oder undefined, wenn die Variable nicht existiert.

WICHTIG! Wenn die globale Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefiniert zurück und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, also verwenden Sie im Zweifelsfall die Funktion variable_global_exists zuerst.

Syntax:

variable_global_get(name);

Argument Beschreibung
Name Der Name der globalen Variable, die abgerufen werden soll (als String)

Rückgabe:

Wert (beliebiger Datentyp) oder undefiniert (wenn die benannte Variable nicht existiert)

Beispiel:

if variable_global_exists("feind_num")
{
show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

Der obige Code prüft, ob eine globale Variable existiert, und wenn ja, wird sie auf der Konsole ausgegeben.
