variabel_global_existiert

Mit dieser Funktion können Sie prüfen, ob eine globale Bereichsvariable existiert oder nicht. Sie geben den Namen der globalen Variable, auf die geprüft werden soll, als String an (siehe Beispielcode unten) und die Funktion gibt true zurück, wenn eine globale Variable mit dem angegebenen Namen existiert oder andernfalls false.

Syntax:

variable_global_exists(name);

Argument Beschreibung
Name Der Name der globalen Variable, auf die geprüft werden soll (als String)

Rückgabe:

Boolesche

Beispiel:

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

Der obige Code prüft, ob die globale Variable mit dem Namen "enemy_num" existiert, und wenn nicht, wird sie erstellt und ein Wert gesetzt.
