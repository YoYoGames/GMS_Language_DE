variable_struct_names_count

Mit dieser Funktion können Sie die Gesamtanzahl der für eine struct definierten Variablen ermitteln. Sie geben die zu prüfende struct-ID an, und die Funktion gibt einen ganzzahligen Wert für die Anzahl der gefundenen Variablen zurück, oder (falls kein struct mit der angegebenen ID existiert) -1.

Syntax:

variable_struct_names_count(struct_id);

Argument Beschreibung
struct_id Der eindeutige ID-Wert der zu prüfenden Struktur.

Rückgabe:

Integer

Beispiel:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

Der obige Code ruft die Anzahl der Variablen in der angegebenen Struktur ab und zeigt in der Konsolenausgabe eine Debug-Meldung mit diesem Wert an.
