bool

Diese Funktion versucht, einen gegebenen Wert in einen booleschen Datentyp zu konvertieren, wobei der Wert als wahr zurückgegeben wird, wenn er größer als 0,5 ist, und andernfalls als falsch.

Syntax:

bool(n);

Argument Beschreibung
n Der zu konvertierende Wert.

Rückgabe:

Boolesche

Beispiel:

if !is_bool(val)
{
val = bool(val);
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie ein Boolean ist, und wenn nicht, wird sie in eines umgewandelt.
