ptr

Diese Funktion versucht, einen gegebenen Wert in einen Zeigerdatentyp zu konvertieren, wobei der Wert entweder ein real, ein string, ein int64, ein int32 sein muss, oder ein ptr sein muss. Alles andere führt zu einem Absturz des Spiels mit einer Fehlermeldung.

Syntax:

ptr(n);

Argument Beschreibung
n Der zu konvertierende Wert.

Rückgabe:

Zeiger

Beispiel:

if !is_ptr(val)
{
val = ptr(application_surface);
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie einen Zeiger enthält, und wenn nicht, wird ihr einer zugewiesen.
