variable_struct_get

Mit dieser Funktion können Sie den Wert einer gegebenen benannten Variablen innerhalb einer struct abrufen. Sie übergeben die struct-Referenz sowie den Namen der Variablen, deren Wert Sie abrufen möchten , als String (siehe Beispielcode unten). Die Funktion gibt den Wert zurück oder undefiniert, wenn die benannte Variable nicht existiert.

WICHTIG! Wenn die Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefiniert zurück und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, verwenden Sie also im Zweifelsfall zuerst die Funktion variable_struct_exists() zuerst.

Syntax:

variable_struct_get(struct, name);

Argument Beschreibung
struct Die zu verwendende Strukturreferenz
Name Der Name der zu erhaltenden Variablen (als String)

Rückgabe:

Wert (beliebiger Datentyp) oder undefiniert

Beispiel:

if variable_struct_exists(mystruct, "shields")
{
var ss = variable_struct_get(mystruct, "shields");
}
sonst
}
var ss = -1;
}

Der obige Code prüft, ob eine Variable in der angegebenen Struktur vorhanden ist. Wenn dies der Fall ist, wird der Wert, den sie enthält, abgerufen und in einer lokalen Variablen gespeichert. Wenn sie nicht vorhanden ist, wird die lokale Variable auf -1 gesetzt.
