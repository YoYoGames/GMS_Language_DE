is_bool

Diese Funktion gibt zurück, ob eine gegebene Variable ein Boolescher Wert ist(true ioder false) oder nicht. In manchen Fällen möchten Sie prüfen, ob eine Variable in GameMaker Studio 2 einen booleschen Wert hat, und in diesem Fall würden Sie diese Funktion verwenden.

Syntax:

is_bool(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

if is_bool(val)
{
global.Sound = val;
}
sonst
{
global.Sound = true;
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie eine reelle Zahl ist, und wenn ja, verwendet er sie, um eine globale Variable zu setzen.
