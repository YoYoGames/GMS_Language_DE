variable_global_set

Mit dieser Funktion können Sie den Wert einer bestimmten globalen Variablen setzen. Sie geben den Namen der globalen Variable, deren Wert gesetzt werden soll, als String an (siehe Beispielcode unten), und dann den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die globale Variable nicht bereits im Spiel vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

variable_global_set(name, val);

Argument Beschreibung
Name Der Name der zu setzenden globalen Variable (als String)
val Der Wert, auf den die globale Variable gesetzt werden soll

Rückgabe:

N/A

Beispiel:

if !variable_global_exists("enemy_num")
{
variable_global_set("feind_zahl", 0);
}

Der obige Code prüft, ob eine globale Variable existiert, und wenn nicht, wird sie erstellt und auf 0 gesetzt.
