variable_struct_get_names

Mit dieser Funktion können Sie ein Array abrufen, das mit den Variablennamen aus einer Struktur gefüllt ist. Sie übergeben die zu prüfende struct-Referenz, und jeder Eintrag im Array ist eine Zeichenkette mit den Variablennamen, die die struct enthält.

Syntax:

variable_struct_get_names(struct);

Argument Beschreibung
struct Die zu prüfende Strukturreferenz.

Rückgabe:

Array (jeder Eintrag ist ein String)

Beispiel:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variablen für struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
show_debug_message(str);
}

Der obige Code ruft ein Array mit den Variablennamen für die angegebene Struktur ab und zeigt diese dann zusammen mit ihren Werten in der Debug-Ausgabe an.
