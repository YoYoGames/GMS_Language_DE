array_erstellen

Mit dieser Funktion können Sie ein 1D-Array einer bestimmten Größe erzeugen. Sie teilen der Funktion die Länge des zu erstellenden Arrays mit, und sie gibt das "Handle" für das Array zurück, das Sie dann einer Variablen zuweisen können. Bei Arrays, die auf diese Weise erstellt werden, wird jeden Eintrag auf 0 initialisiert, es sei denn, Sie geben einen (optionalen) Initialisierungswert an. Wenn Sie den zusätzlichen Wert für die Initialisierung des Arrays angeben, werden alle Indizes innerhalb des neuen Arrays auf diesen Wert anstelle von 0 gesetzt, aber beachten Sie, dass die Funktion in diesem Fall einen größeren Performance-Overhead hat.

Syntax:

array_create(Größe, [Wert]);

Argument Beschreibung
Größe Die Größe des zu erstellenden Arrays.
Wert Der Wert, der für die Initialisierung aller Array-Indizes verwendet werden soll (optional).

Rückgabe:

Array-Referenz

Beispiel:

instance_array = array_create(100, noone);

Der obige Code erzeugt ein neues Array mit 100 Einträgen, die jeweils auf das Schlüsselwort " noone" gesetzt werden, und weist es dann der Variablen "instance_array" zu.
