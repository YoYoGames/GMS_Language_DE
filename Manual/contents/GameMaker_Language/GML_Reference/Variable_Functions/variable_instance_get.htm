variable_instanz_holen

Mit dieser Funktion können Sie den Wert einer gegebenen benannten Variablen abrufen. Sie geben den eindeutigen Wert der Instanz-ID an (den Sie in den Instanz-Eigenschaften im Raum Editor gefunden werden kann oder beim Aufruf der Funktion zurückgegeben wird instance_create_layer()) sowie den Namen der Variable, deren Wert Sie abrufen wollen, als String (siehe Beispielcode unten). Die Funktion gibt den Wert zurück, den die Variable enthält, oder undefined, wenn die Variable nicht existiert.

WICHTIG! Wenn die Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort undefiniert zurück und Sie können Fehler erhalten, die das Spiel zum Stillstand bringen, verwenden Sie also im Zweifelsfall zuerst die Funktion variable_instanz_existiert zuerst.

Syntax:

variable_instance_get(instance_id, name);

Argument Beschreibung
instance_id Der eindeutige ID-Wert der zu verwendenden Instanz
Name Der Name der zu erhaltenden Variablen (als String)

Rückgabe:

Wert (beliebiger Datentyp) oder undefiniert (wenn die benannte Variable nicht existiert)

Beispiel:

if variable_instance_exists(id, "shields")
{
var ss = variable_instance_get(id, "shields");
}
sonst
}
var ss = -1;
}

Der obige Code prüft, ob eine Variable existiert und wenn ja, dann wird der Wert, den sie enthält, abgerufen und in einer lokalen Variable gespeichert. Wenn sie nicht vorhanden ist, wird die lokale Variable auf -1 gesetzt.
