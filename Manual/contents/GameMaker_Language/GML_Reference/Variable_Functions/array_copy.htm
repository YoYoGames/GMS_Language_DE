array_copy

Mit dieser Funktion können Sie ein Array ganz oder teilweise an einer beliebigen Position in ein anderes Array kopieren. Sie müssen sowohl das Quell- als auch das Ziel-Array angeben (beide müssen zuvor erstellt worden sein), sowie eine Position innerhalb des Quell-Arrays, von der kopiert werden soll von und eine Position innerhalb des Ziel-Arrays, in das kopiert werden soll. Schließlich müssen Sie die Länge des zu kopierenden Arrays (oder die Länge des gewünschten Teils) angeben. Wenn die zu kopierenden Daten die Länge des Ziel-Arrays überschreiten, wird das Array erweitert, um die Daten aufzunehmen.

Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, solange Sie bei der Angabe des Array-Indexes angeben, welche Dimension Sie kopieren wollen, und zwar nach diesem Muster:

// Kopieren von der ersten Dimension eines Arrays
// in die zweite Dimension eines Arrays
array_copy(item_array, 0, inventory_array[0], 0, len);

// Kopieren von der dritten Dimension eines Arrays
// in die erste Dimension eines Arrays
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// usw..

Syntax:

array_copy(dest, dest_index, src, src_index, length);

Argument Beschreibung
dest Die ID des Arrays, in das kopiert werden soll.
dest_index Der Index innerhalb des Arrays, in den kopiert werden soll.
src Die ID des Arrays, aus dem kopiert werden soll.
src_index Der Index mit dem Array, von dem aus kopiert werden soll.
Länge Die zu kopierende Länge (Anzahl der Array-Indizes).

Rückgabe:

N/A

Beispiel:

if !array_equals(inventar_array, item_array)
{
var len = array_length(inventar_array);
array_copy(item_array, 0, inventar_array, 0, len);
}

Der obige Code prüft zwei Arrays, um zu sehen, ob sie gleichwertige Werte enthalten, und wenn nicht, dann kopiert der Code den gesamten Inhalt des Arrays "inventory_array" in das Array "item_array".
