is_string

Diese Funktion gibt zurück, ob eine gegebene Variable eine String-Zahl ist oder nicht. In manchen Fällen möchten Sie überprüfen, ob eine Variable in GameMaker Studio 2 eine Zeichenkette und nicht eine reelle Zahl enthält, und dann würden Sie diese Funktion. Sie gibt nicht die Zeichenkette zurück, sondern true oder false, so dass ein Wert von z. B. "fish" für n true zurückgibt, aber ein Wert von 200 für n gibt false zurück.

Syntax:

is_string(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

if is_string(val)
{
name = "Spieler: " + val;
}

Der obige Code prüft, ob die Variable "val" eine Zeichenkette enthält, und wenn ja, fügt er diese in eine andere Zeichenkette ein, die dann der Variablen "name" zugewiesen wird.
