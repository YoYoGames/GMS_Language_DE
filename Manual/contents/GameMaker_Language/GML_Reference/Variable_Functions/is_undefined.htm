ist_unbestimmt

Diese Funktion gibt zurück, ob eine bestimmte Variable definiert ist oder nicht. In manchen Fällen möchten Sie überprüfen, welchen Datentyp eine Variable in GameMaker Studio 2 hat, und dann würden Sie diese Funktion verwenden. Sie gibt true oder false zurück, je nachdem je nachdem, ob der Wert definiert ist oder nicht.

HINWEIS: Diese Funktion kann nicht verwendet werden, um die Existenz einer Variablen zu überprüfen. Verwenden Sie variable_instanz_existiert() oder variable_global_exists() stattdessen.

Syntax:

is_undefined(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
show_debug_message("Karteneintrag existiert nicht!");
}

Der obige Code prüft die Variable "val", ob sie undefiniert ist oder nicht und zeigt eine Debug-Meldung an, wenn dies der Fall ist.
