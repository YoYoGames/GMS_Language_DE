array_get

Mit dieser Funktion können Sie den Wert von einem Index in einem Array abrufen. Die Funktion erfordert, dass Sie eine Variable bereitstellen, die das Array und den Index enthält, um den Wert aus diesem Array zu erhalten. Beachten Sie, dass das Spiel mit einem Fehler abstürzt, wenn der angegebene Array-Index außerhalb der Grenzen liegt. dann stürzt das Spiel mit einem Fehler ab. Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, solange Sie bei der Angabe des Array-Indexes angeben, welche Dimension Sie abrufen wollen, indem Sie diesem Muster folgen:

// 1D-Array
array_get(mein_Array, 0);
// 2D-Array
array_get(mein_Array[0], 0);
// 3D-Array
array_get(mein_Array[0][0], 0);
// usw...

Syntax:

array_get(variable, index);

Argument Beschreibung
variabel Die Variable, die das Array enthält.
index Der Index des Arrays, aus dem der Wert geholt werden soll.

Rückgabe:

Jeder gültige Datentyp, den ein Array aufnehmen kann.

Beispiel:

for (var i = 0; i < 10; ++i;)
{
show_debug_message(array_get(my_array, i));
}

Der obige Code gibt die ersten 10 Elemente des angegebenen auf der Konsole aus.
