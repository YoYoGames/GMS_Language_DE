variable_instanz_satz

Mit dieser Funktion können Sie den Wert einer bestimmten Variablen in einer Instanz setzen. Sie geben den eindeutigen Wert der Instanz-ID an (den Sie in den Instanz-Eigenschaften im Raum-Editor gefunden werden kann oder beim Aufruf der Funktion zurückgegeben wird instance_create_layer()) sowie den Namen der Variablen, deren Wert gesetzt werden soll, als String (siehe Beispiel Code unten), und schließlich den zu setzenden Wert (kann jeder gültige Datentyp sein). Wenn die Variable nicht bereits in der Instanz vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

Syntax:

variable_instance_set(instance_id, name, val);

Argument Beschreibung
instance_id Der eindeutige ID-Wert der zu verwendenden Instanz
Name Der Name der zu setzenden Variablen (als String)
val Der Wert, auf den die Variable gesetzt werden soll

Rückgabe:

N/A

Beispiel:

if !variable_instance_exists(id, "shields")
{
variable_instance_set(id, "shields", 0);
}

Der obige Code prüft, ob eine Instanzvariable in der aufrufenden Instanz existiert, und wenn nicht, wird sie erstellt und auf 0 gesetzt.
