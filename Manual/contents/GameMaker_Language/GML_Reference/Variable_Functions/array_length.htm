array_length

Mit dieser Funktion können Sie die Länge (Anzahl der Einträge) einer Array-Dimension ermitteln. Sie geben den Array-Indexwert an und die Funktion gibt einen Integer-Wert zurück, der die Anzahl der Einträge im Array angibt. Diese Funktion kann auch verwendet werden für mehrdimensionale Arrays verwendet werden, solange Sie bei der Übergabe des Array-Index angeben, für welche Dimension Sie die Länge ermitteln wollen, und zwar nach diesem Muster:

// Liefert die erste Dimension des Arrays
val = array_length(mein_array);

// Liefert die zweite Dimension des Arrays
val = array_length(mein_array[0]);

// Liefert die dritte Dimension des Arrays
val = array_length(mein_array[0][0]);

// usw...

Syntax:

array_length(array_index);

Argument Beschreibung
array_index Der Index des zu prüfenden Arrays.

Rückgabe:

Integer

Beispiel:

for (var i = 0; i < array_length(a); ++i;)
{
a[i] = -1;
}

Der obige Code führt eine Schleife durch ein Array und setzt jeden Eintrag auf -1.
