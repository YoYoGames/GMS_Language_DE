ist_wirklich

Diese Funktion gibt zurück, ob eine gegebene Variable eine reelle Zahl (Single, Double oder Integer) ist oder nicht. In manchen Fällen möchten Sie prüfen, ob eine Variable eine reelle Zahl enthält, und dann würden Sie diese Funktion verwenden. Sie gibt nicht Die Funktion gibt nicht die reelle Zahl zurück, sondern wahr oder falsch, d. h. ein Wert von z. B. "Fisch" würde falsch zurückgeben, ein Wert von 200 jedoch wahr.

HINWEIS: Diese Funktion gibt false für alle Enum-Werte zurück, da diese als int64 gespeichert sind. Die korrekte Funktion zur Überprüfung dieses Datentyps ist is_int64().

Syntax:

is_real(n);

Argument Beschreibung
n Das zu prüfende Argument.

Rückgabe:

Boolesche

Beispiel:

if is_real(val)
{
score += val;
}

Der obige Code prüft die Variable "val", um zu sehen, ob sie eine reelle Zahl ist, und wenn sie es ist, wird sie zur Punktzahl addiert.
