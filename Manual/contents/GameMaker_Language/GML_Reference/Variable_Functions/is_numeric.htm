ist_numerisch

Diese Funktion gibt zurück, ob eine gegebene Variable ein numerischer Wert (real, int32, int64 oder boolean) ist oder nicht. In manchen Fällen möchten Sie prüfen, ob eine Variable in GameMaker Studio 2 einen numerischen Wert enthält, und dann dann würden Sie diese Funktion verwenden. Die Funktion gibt true zurück, wenn die angegebene Eingabe numerisch ist, und andernfalls false.

Syntax:

is_numeric(n);

Argument Beschreibung
n Der zu prüfende Eingang.

Rückgabe:

Boolesche

Beispiel:

if is_numeric(val)
{
current_val += val;
}

Der obige Code prüft die Variable "val", um zu sehen, ob es sich um einen numerischen Wert handelt, und wenn dies der Fall ist, fügt er ihn der angegebenen Variablen hinzu.
