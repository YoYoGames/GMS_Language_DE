Variable Funktionen

Die Funktionen auf dieser Seite sind für den Umgang mit all den verschiedenen Variablen und Variablentypen bei der Verwendung der GameMaker-Sprache in Ihren Spielen gedacht. Weitere Informationen zu Variablen finden Sie auf der Seite zu Variablen und Variablenumfang.

Die folgenden Variablenfunktionen sind verfügbar:

  • variable_instanz_existiert
  • variable_instanz_get_names
  • variable_instanz_namen_anzahl
  • variable_instanz_holen
  • variable_instanz_satz
  • variabel_global_existiert
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Es gibt auch ein paar Funktionen, die mit der Verwendung von Arrays verbunden sind. Diese wurden entwickelt, um Ihnen Flexibilität in Ihrem Code zu geben und erlauben Ihnen, funktionellere und dynamischere Arrays bei der Erstellung Ihrer Spiele zu erstellen. Diese Funktionen sind:

  • array_erstellen
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_einfügen
  • array_delete
  • array_sortieren
  • array_length
  • array_resize
  • array_length_1d(veraltet)
  • array_length_2d(veraltet)
  • array_height_2d(veraltet)

Diese nächsten Funktionen beziehen sich alle auf die Verwendung von Methodenvariablen:

  • Methode
  • methode_get_self
  • methode_get_index

Schließlich stehen Ihnen noch folgende Funktionen zur Bestimmung des Variablentyps zur Verfügung (weitere Informationen zu Datentypen finden Sie hier, zu Typentabellen siehe hier):

  • is_string
  • ist_wirklich
  • ist_numerisch
  • is_bool
  • is_array
  • is_struct
  • is_method
  • ist_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • ist_unbestimmt
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
