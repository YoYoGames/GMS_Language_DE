array_length_2d(Veraltet)

WARNUNG! Diese Funktion ist veraltet, da Arrays nicht mehr auf nur 1 oder 2 Dimensionen beschränkt sind, und als solche wird diese Funktion nur zur Unterstützung von Legacy-Projekten bereitgestellt. Alle neuen Projekte sollten die aktuelle Methode zum Erstellen und Zugreifen auf mehrdimensionale Arrays verwenden (siehe hier für weitere Informationen).

Mit dieser Funktion können Sie die Länge (Anzahl der Einträge) der zweiten Dimension eines Arrays ermitteln. Sie geben die Eintragsnummer für die erste Dimension an und die Funktion gibt die Anzahl der Einträge der zweiten Dimension zurück, die das Array hat (um die Länge der ersten Dimension zu ermitteln, verwenden Sie die Funktion die Länge der ersten Dimension zu ermitteln, verwenden Sie die Funktion array_height_2D()). Die Funktion gibt 0 zurück, wenn die angegebene Variable kein Array ist, oder 1, wenn die Variable ein 1D-Array ist (da es noch 1 Zeile gibt).

Syntax:

array_length_2d(array_index, n);

Argument Beschreibung
array_index Der Index des zu prüfenden Arrays.
n Der Eintrag des Arrays, dessen Länge ermittelt werden soll.

Rückgabe:

Real

Beispiel:

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

Der obige Code führt eine Schleife durch ein 2D-Array und setzt jeden Eintrag auf -1.
