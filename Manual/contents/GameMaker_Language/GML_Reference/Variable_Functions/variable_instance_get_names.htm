variable_instanz_get_names

Mit dieser Funktion können Sie ein Array abrufen, das mit den Namen der Instanzvariablen für eine Instanz oder den globalen Variablen für ein Spiel gefüllt ist. Wenn Sie einen Instanz-ID-Wert übergeben, ist jeder Eintrag im Array eine Zeichenkette mit dem Variablennamen sein, der einer in der Instanz erstellten Instanzbereichsvariablen entspricht. Wenn Sie jedoch übergeben, ist jeder Eintrag im Array eine Zeichenkette mit dem Variablennamen, der einer Variablen im globalen Bereich entspricht.

Syntax:

variable_instance_get_names(instance_id/global);

Argument Beschreibung
instanz_id/global Der eindeutige ID-Wert der zu prüfenden Instanz oder das Schlüsselwort global

Rückgabe:

Array (jeder Eintrag ist ein String)

Beispiel:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variablen für " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
str = array[i] + ":" + string(variable_instance_get(id, array[i]));
show_debug_message(str);
}

Der obige Code ruft ein Array aller Instanzbereichsvariablen für die Instanz ab, die den Codeblock ausführt, und zeigt diese dann zusammen mit ihren Werten in der Debug-Ausgabe an.
