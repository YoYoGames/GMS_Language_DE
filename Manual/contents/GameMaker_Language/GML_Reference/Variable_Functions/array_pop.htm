array_pop

Mit dieser Funktion können Sie einen Wert vom Ende eines Arrays abrufen, ohne dass Sie die Länge des Arrays kennen müssen. Sie übergeben die Variable, die das Array enthält, und die Funktion gibt den Inhalt des letzten Eintrags im Array zurück. Wenn das Array leer, gibt die Funktion undefined zurück.

Syntax:

array_pop(variable);

Argument Beschreibung
variabel Die Variable, die das Array enthält.

Rückgabe:

Wert (beliebiger Datentyp) oder undefiniert

Beispiel:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

Der obige Code holt den letzten Wert aus dem angegebenen Array und zeichnet ihn dann zusammen mit einem Text auf den Bildschirm.
