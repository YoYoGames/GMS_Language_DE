variable_struct_exists

Mit dieser Funktion können Sie prüfen, ob eine Variable innerhalb der angegebenen struct existiert oder nicht. Sie übergeben die zu verwendende struct-Referenz sowie den zu prüfenden Variablennamen als String (siehe Beispielcode unten). Die Funktion gibt true zurück, wenn eine Variable mit dem angegebenen Namen für das struct existiert, und andernfalls false.

Syntax:

variable_struct_exists(struct, name);

Argument Beschreibung
struct Die zu prüfende Strukturreferenz
Name Der Name der struct-Variable, auf die geprüft werden soll (als String)

Rückgabe:

Boolesche

Beispiel:

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

Der obige Code prüft, ob die Variable "shields" in der angegebenen Struktur vorhanden ist, und wenn nicht, wird sie erstellt und auf 0 initialisiert.
