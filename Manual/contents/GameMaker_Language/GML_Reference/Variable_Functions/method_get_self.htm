methode_get_self

Mit dieser Funktion können Sie die Instanz-ID oder die Strukturreferenz abrufen, die der self Kontext, der beim Aufruf der Methode verwendet wird. Wenn die Variable keine Methode ist, gibt die Funktion undefiniert zurück. Bitte beachten Sie, dass die Funktion auch die Konstante pointer_null zurückgeben kann. In diesem Fall wird das aktuelle self zum Zeitpunkt des Aufrufs verwendet.

Syntax:

method_get_self(method);

Argument Beschreibung
Methode Die zu prüfende Methodenvariable.

Rückgabe:

Integer, undefiniert, pointer_null

Beispiel:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

Der obige Code ruft den Selbstkontext für die angegebene Methodenvariable ab und gibt ihn auf der Konsole aus.
