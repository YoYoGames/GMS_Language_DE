array_resize

Mit dieser Funktion können Sie die Größe eines bestehenden Arrays auf eine neue Größe ändern. Sie geben das Array an, dessen Größe geändert werden soll, sowie die neue Anzahl der Indizes für das Array, und die Funktion ändert die Größe dieses Arrays. Beachten Sie, dass diese Funktion für die Größenänderung ein Array auf eine kleinere Länge zu verkleinern, da Sie die Größe durch einfaches Setzen eines neuen Index im Array ändern können. Wenn Sie die Funktion zum Vergrößern eines Arrays verwenden, werden alle neuen Indizes auf den Standardwert 0 gesetzt.

Diese Funktion kann auch für mehrdimensionale Arrays verwendet werden, solange Sie bei der Übergabe des Array-Indexes angeben, welche Dimension Sie in der Größe verändern wollen, und zwar nach diesem Muster:

// Größenänderung der ersten Dimension des Arrays
array_resize(mein_Array, 10);

// Größenänderung der zweiten Dimension des Arrays
array_resize(mein_Array[0], 10);

// Größenänderung der dritten Dimension des Arrays
array_resize(mein_Array[0][0], 10);

// usw...

Syntax:

array_resize(array_index, new_size);

Argument Beschreibung
array_index Der Index des Arrays, dessen Größe geändert werden soll.
neue_Größe Die neue Größe für das Array (eine Ganzzahl, beginnend bei 0).

Rückgabe:

N/A

Beispiel:

if array_length(a) > 10
{
array_resize(a, 10);
}

Der obige Code prüft die Länge eines Arrays und wenn es mehr als 10 Indizes hat, dann wird es in der Größe angepasst.
