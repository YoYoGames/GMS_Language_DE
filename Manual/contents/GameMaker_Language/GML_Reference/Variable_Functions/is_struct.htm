is_struct

Diese Funktion prüft, ob der übergebene Wert ein struct ist. Ist dies der Fall, gibt sie true zurück, andernfalls false.

Beachten Sie, dass Methodenvariablen ebenfalls true und Objektinstanzen false zurückgeben.

Syntax:

is_struct(val);

Argument Beschreibung
val Der zu prüfende Wert.

Rückgabe:

Boolesche

Beispiel:

if is_struct(a)
{
delete(a);
}

Der obige Code prüft eine Variable, um zu sehen, ob sie ein struct ist, und wenn die Funktion true zurückgibt, wird das struct gelöscht.
