is_array

Diese Funktion kann verwendet werden, um zu prüfen, ob eine Variable ein Array enthält (es wird true zurückgegeben) oder nicht (in diesem Fall wird false zurückgegeben).

Syntax:

is_array(n);

Argument Beschreibung
n Die zu prüfende Variable.

Rückgabe:

Boolesche

Beispiel:

if is_array(a)
{
a = -1;
}

Der obige Code prüft eine Variable, um zu sehen, ob sie ein Array ist, und wenn die Funktion true zurückgibt, wird das Array gelöscht, indem die Variable auf -1 gesetzt wird.
