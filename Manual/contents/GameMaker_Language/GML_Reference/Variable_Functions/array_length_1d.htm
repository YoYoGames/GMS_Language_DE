array_length_1d(Veraltet)

WARNUNG! Diese Funktion ist veraltet, da Arrays nicht mehr auf nur 1 oder 2 Dimensionen beschränkt sind, und als solche wird diese Funktion nur zur Unterstützung von Legacy-Projekten bereitgestellt. Alle neuen Projekte sollten die aktuelle Methode zum Erstellen und Zugreifen auf mehrdimensionale Arrays verwenden (siehe hier für weitere Informationen).

Mit dieser Funktion können Sie die Länge (Anzahl der Einträge) eines 1D-Arrays ermitteln. Für 2D-Arrays sollten Sie die Funktionen array_height_2d und array_Länge_2d Funktionen verwenden.

WICHTIG! Wenn das Array mehr als 32.000 Einträge hat, gibt diese Funktion einen fehlerhaften Wert zurück und sollte nicht verwendet werden.

Syntax:

array_length_1d(array_index);

Argument Beschreibung
array_index Der Index des zu prüfenden Arrays.

Rückgabe:

Real

Beispiel:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
a[i] = -1;
}

Der obige Code führt eine Schleife durch ein Array und setzt jeden Eintrag auf -1.
