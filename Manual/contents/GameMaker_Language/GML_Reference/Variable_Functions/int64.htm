int64

Diese Funktion versucht, einen gegebenen Wert in eine 64-Bit-Ganzzahl zu konvertieren, wobei der Wert entweder ein Real, ein String, ein int64, ein int32 oder ein ptr sein muss. Alles, was alles andere führt zu einem Absturz des Spiels mit einer Fehlermeldung. Sie können prüfen, ob eine Variable einen int64 enthält, indem Sie die Funktion is_int64().

Syntax:

int64(val);

Argument Beschreibung
val Der zu konvertierende Wert.

Rückgabe:

64bit Ganzzahl

Beispiel:

steam_handle= int64(global.fileReadString);

Der obige Code wandelt den in der globalen Variablen gehaltenen Wert in eine 64-Bit-Ganzzahl um.
