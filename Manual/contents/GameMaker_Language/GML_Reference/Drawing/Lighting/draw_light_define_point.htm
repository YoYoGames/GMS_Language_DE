licht_define_punkt_zeichnen

Diese Funktion dient zur Definition eines Positionslichts, wobei Sie die x-, y- und z-Position des Lichts, den Lichtbereich und seine Farbe definieren können (was auch die wahrgenommene Intensität des Lichts beeinflusst, da bestimmte Farben "dunkler" erscheinen als andere andere). Sie müssen dem Licht auch eine Indexnummer geben, die in anderen Funktionen verwendet wird, um es zu referenzieren.

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_define_point(ind, x, y, z, range, col);

Argument Beschreibung
ind Die Indexnummer der Leuchte (beliebig)
x Die x-Position der Leuchte
y Die y-Position der Leuchte
z Die z-Position der Leuchte
Bereich Die Leuchtweite (in Pixel)
col Die Farbe, die für das Licht verwendet werden soll (entweder eine Konstante, ein Real- oder ein Hex-Wert).

Rückgabe:

N/A

Beispiel:

draw_set_lighting(true);
draw_light_define_point(1, 200, 123, 50, 2000, c_white);
draw_light_enable(1, true);

Der obige Code aktiviert die Beleuchtung für die gesamte Szene, definiert dann ein weißes Licht an einem bestimmten Punkt im Raum und schaltet schließlich dieses Licht ein.
