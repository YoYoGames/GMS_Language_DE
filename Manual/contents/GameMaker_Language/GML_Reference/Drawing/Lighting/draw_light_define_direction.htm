draw_light_define_direction

Diese Funktion dient zur Definition eines gerichteten Lichts, wobei die Richtung und die Intensität durch die eingegebenen Werte für den x-, y- und z-Einheitsvektor festgelegt werden (d. h. sie müssen zwischen -1 und 1 liegen, wobei 1 oder -1 die volle Intensität und 0 keine Intensität bedeutet). Die Lichtern kann auch eine Farbe gegeben werden, die ebenfalls die wahrgenommene Intensität des Lichts beeinflusst, da bestimmte Farben "dunkler" erscheinen als andere. Sie müssen dem Licht auch eine Indexnummer geben, die in anderen Funktionen verwendet wird, um es zu referenzieren. es.

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_define_direction(ind, x, y, z, col);

Argument Beschreibung
ind Die Indexnummer der Leuchte (beliebig)
x Die x-Komponente des Lichtvektors
y Die y-Komponente des Lichtvektors
z Die z-Komponente des Lichtvektors
col Die Farbe, die für das Licht verwendet werden soll (entweder eine Konstante, ein Real- oder ein Hex-Wert).

Rückgabe:

N/A

Beispiel:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Der obige Code aktiviert die Beleuchtung für die gesamte Szene, definiert dann ein weißes, gerichtetes Licht im Raum und schaltet schließlich dieses Licht ein.
