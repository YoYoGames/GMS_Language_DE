draw_light_enable

Diese Funktion dient dazu, ein definiertes Licht zu aktivieren. Wenn Sie ein Positions- oder Richtungslicht definieren, müssen Sie ihm eine Indexnummer zuweisen, die dann von dieser Funktion verwendet wird, um das Licht ein- oder auszuschalten. Die Voreinstellung ist deaktiviert(false).

HINWEIS: Es stehen nur 8 Hardware-Lichter zur Verfügung, so dass nur 8 definierte Lichter gleichzeitig aktiviert werden können (obwohl mehr definiert werden können).

Syntax:

draw_light_enable(ind, enable);

Argument Beschreibung
ind Die Indexnummer der Leuchte (von 0 bis 7)
aktivieren. Aktivieren oder Deaktivieren der gesamten Beleuchtung(true oder false)

Rückgabe:

N/A

Beispiel:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Der obige Code aktiviert die Beleuchtung für die gesamte Szene, definiert dann ein weißes, gerichtetes Licht im Raum und schaltet schließlich dieses Licht ein.
