draw_light_define_ambient

Diese Funktion dient zur Steuerung des Umgebungslichts einer Szene, d. h. des Lichts, das Sie in einer Szene auch dann haben, wenn Sie keine punktförmigen oder gerichteten Lichtquellen definiert haben. Es ist quasi die Gesamtfarbe und Helligkeit (oder Dunkelheit) einer Szene. Die Standardfarbe ist c_black.

Syntax:

draw_light_define_ambient(col);

Argument Beschreibung
col Die zu verwendende Farbe (entweder eine Konstante, ein Real- oder ein Hex-Wert).

Rückgabe:

N/A

Beispiel:

draw_light_define_ambient(c_white);

Mit dem obigen Code wird die Umgebungsbeleuchtung als weiß definiert.
