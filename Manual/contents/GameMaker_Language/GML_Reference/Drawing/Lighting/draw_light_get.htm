zeichnen_licht_holen

Diese Funktion liefert die angegebenen Lichtparameter als Array mit den folgenden 6 Elementen -

  • [0] = aktiviert / deaktiviert(true / false)
  • [1] = Lichttyp (siehe Konstanten unten)
  • [2] = x-Position
  • [3] = y-Position
  • [4] = z-Position
  • 5] = Lichtradius (nur für Punktlichter, wird bei Richtungslichtern 0 sein)
  • [6] = helle Farbe (a real)

Der Lichttyp kann eine der beiden folgenden Konstanten sein:

Konstant Beschreibung
lighttype_dir Das Licht ist ein gerichtetes Licht
Lichtart_Punkt Das Licht ist ein Punktlicht

Syntax:

draw_light_get(ind);

Argument Beschreibung
ind Die Indexnummer der Leuchte (von 0 bis 7)

Rückgabe:

Real

Beispiel:

light_a = draw_light_get(1);
if light_a[5] < 200
{
light_a[5] += 5;
draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

Der obige Code holt sich die Werte, die zur Definition des als "1" indizierten Lichts verwendet werden, prüft dann den Radius und wenn dieser kleiner als 200 ist, wird er erhöht und der Lichtradius auf den neuen Wert gesetzt.
