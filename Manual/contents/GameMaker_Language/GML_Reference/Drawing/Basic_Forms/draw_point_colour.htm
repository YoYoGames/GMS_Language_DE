draw_point_colour

Mit dieser Funktion können Sie einen einzelnen Pixel an beliebiger Stelle auf dem Bildschirm mit einer von Ihnen definierten Farbe zeichnen. Die Farbeinstellungen überschreiben die Grundfarbe, die mit der Funktion draw_set_colour().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_point_colour(x, y, col1);

Argument Beschreibung
x Die x-Koordinate des Punktes.
y Die y-Koordinate des Punktes.
col1 Die Farbe des Punktes.

Rückgabe:

N/A

Beispiel:

draw_point_colour(50, 50, c_red);

Dies würde einen roten Pixel bei (50,50) zeichnen.
