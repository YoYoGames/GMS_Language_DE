draw_healthbar

Mit dieser Funktion können Sie einen farbigen Balken zeichnen, um einen konstanten Wert anzuzeigen. Obwohl die Funktion das Wort "Gesundheitsbalken" verwendet, können Sie damit jeden beliebigen Wert anzeigen, solange der anzuzeigende Betrag ein Prozentwert zwischen 0 und 100 ist (ein Prozentsatz eines beliebigen Wertes kann mit der Formel (AKTUELLER_WERT / MAXIMALER_WERT) * 100 berechnet werden), so dass Sie diese Funktion z. B. zur Anzeige von Energie, Gesundheit, Mana, Zeit oder anderen Werten verwenden können, die von einer Balkenanzeige profitieren würden.

Sie können verschiedene Dinge einstellen, die den visuellen Aspekt des Balkens verändern, und diese sind im folgenden Bild dargestellt (beachten Sie, dass die hier verwendeten Farben c_green für den Maximalwert und c_red für den Minimalwert sind, aber Sie können die gewünschten Farben verwenden) Sie können aber die Farben verwenden, die Sie wünschen):

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Argument Beschreibung
x1 Die x-Koordinate der linken Seite der Gesundheitsleiste
y1 Die y-Koordinate des oberen Bereichs der Gesundheitsleiste
x2 Die x-Koordinate der rechten Seite der Gesundheitsleiste
y2 Die y-Koordinate des unteren Bereichs der Gesundheitsleiste
Betrag Die Variable, die den gesamten Gesundheitszustand definiert (zwischen 0 und 100)
backcol Die Farbe des Hintergrunds für den Balken
mincol Die Farbe des Balkens, wenn er keine Gesundheit hat (0)
maxcol Die Farbe des Balkens bei voller Gesundheit (100)
Richtung Wo der Balken "verankert" ist (0 = links, 1 = rechts, 2 = oben, 3 = unten)
showback Ob der Hintergrund des Balkens gezeichnet wird (true) oder nicht (false). Wenn false, wird das backcol-Argument ignoriert.
showborder Ob die Elemente der Leiste einen 1px breiten schwarzen Rand haben (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Der obige Code verwendet den Prozentwert, den Sie aus den Variablen "hp" und "hp_max" erhalten, um einen standardmäßigen rot/grünen Gesundheitsbalken zu zeichnen.
