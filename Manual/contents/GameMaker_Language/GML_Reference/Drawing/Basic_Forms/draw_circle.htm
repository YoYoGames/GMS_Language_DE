kreis_zeichnen

Mit dieser Funktion können Sie entweder einen Umriss eines Kreises oder einen gefüllten Kreis zeichnen. Wie genau die Zeichnung ist, können Sie mit der Funktion draw_set_circle_precision().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_circle(x, y, r, outline);

Argument Beschreibung
x Die x-Koordinate des Mittelpunkts des Kreises.
y Die y-Koordinate des Mittelpunkts des Kreises.
r Der Radius des Kreises (Länge vom Mittelpunkt zum Rand)
Umriss Ob der Kreis gefüllt(false) oder als ein Pixel breiter Umriss(true) gezeichnet wird.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
draw_circle(100, 100, 50, true);

Dadurch wird ein ein Pixel breiter weißer Kreisumriss mit einem Radius von 50 Pixeln gezeichnet.
