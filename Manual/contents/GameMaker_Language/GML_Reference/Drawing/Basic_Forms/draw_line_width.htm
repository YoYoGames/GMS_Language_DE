draw_line_width

Mit dieser Funktion können Sie eine Linie mit einer bestimmten Breite zwischen zwei beliebigen Punkten im Spielraum zeichnen. Bitte beachten Sie, dass die zu zeichnende Linie aufgrund von Unterschieden zwischen den verschiedenen unterstützten Plattformen möglicherweise unterschiedliche Werte (+/-1 auf x, y) benötigt, um mit den gewünschten Abmessungen gezeichnet zu werden. den verschiedenen unterstützten Plattformen.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_line_width(x1, y1, x2, y2, w);

Argument Beschreibung
x1 Die x-Koordinate des Beginns der Linie.
y1 Die y-Koordinate des Beginns der Linie.
x2 Die x-Koordinate des Endes der Linie.
y2 Die y-Koordinate des Endes der Linie.
w Die Breite der Linie in Pixel.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_red);
draw_line_width(100, 100, 200, 200, 6);

Dies zeichnet eine rote diagonale Linie, 6 Pixel breit, von Punkt (100,100) zu Punkt (200,200).
