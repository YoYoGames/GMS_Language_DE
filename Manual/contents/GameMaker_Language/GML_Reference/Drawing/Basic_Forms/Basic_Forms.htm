Grundformulare

GameMaker Studio 2 verfügt über eine ganze Sammlung von Funktionen zum Zeichnen verschiedener Formen und Gestalten. Diese Funktionen sollten immer nur im Draw-Event einer Instanz verwendet werden (Sie können sie anderweitig verwenden, aber Sie werden nichts sehen!) und erzeugen rein grafische Effekte, d. h. sie erzeugen keine Kollisionsereignisse, da diese nur über das Sprite oder die Kollisionsmaske, die einer Instanz zugewiesen wurde, erfolgen können. Alle diese Funktionen werden von der aktuellen Zeichenfarbe, dem Alpha- und dem Überblendungsmodus beeinflusst (siehe den Abschnitt über Farbe).

WICHTIG! Linien, Punkte und Formen werden immer im Verhältnis 1:1 mit der Bildschirmauflösung gezeichnet, unabhängig davon, in welchem Event sie gezeichnet werden.

WICHTIG! Diese Funktionen unterbrechen das Vertex-Batching und erhöhen die Anzahl der Textur-Swaps in Ihrem Spiel, so dass mehrere Aufrufe einer dieser Funktionen in einem einzelnen Draw-Frame zu Performance-Problemen führen können.

Zum Zeichnen von Grundformen gibt es die folgenden Zeichenfunktionen. Beachten Sie, dass diese Funktionen langsam aufzurufen sind und nur als grundlegende Zeichenwerkzeuge geliefert werden:

  • zeichnen_pfeil
  • kreis_zeichnen
  • draw_circle_colour
  • zeichnen_ellipse
  • draw_ellipse_colour
  • Linie_zeichnen
  • draw_line_colour
  • draw_line_width
  • draw_line_width_colour
  • Punkt_zeichnen
  • draw_point_colour
  • rechtwinklig_zeichnen
  • Rechteck_Farbe_zeichnen
  • draw_roundrect
  • draw_roundrect_colour
  • draw_roundrect_ext
  • draw_roundrect_colour_ext
  • zeichnen_dreieck
  • draw_triangle_colour

Die folgenden Funktionen sind auch in GameMaker Studio 2 enthalten, um die Zeichengenauigkeit einzustellen und einige sehr spezielle Dinge zu zeichnen:

  • draw_set_circle_precision
  • Zeichnen_Schaltfläche
  • draw_healthbar
  • Pfad_zeichnen
