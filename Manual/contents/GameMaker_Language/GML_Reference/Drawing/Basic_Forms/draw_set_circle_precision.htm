draw_set_circle_precision

Um die Leistung beim Zeichnen von Kreisen zu optimieren, zeichnet GameMaker Studio 2 grundsätzlich eine Polygonform mit genügend Seiten, um sie kreisförmig erscheinen zu lassen. Je nachdem, wie groß oder klein Sie Ihre Kreise benötigen, können Sie jedoch feststellen, dass eine Änderung dieses Werts die Leistung Ihres Spiels erhöhen oder die Kreise besser aussehen lassen kann. diesen Wert ändern, um die Leistung Ihres Spiels zu erhöhen oder die Kreise besser aussehen zu lassen. Der Präzisionswert, den Sie in diese Funktion eingeben können, ist die Anzahl der Seiten, die das Kreispolygon hat, und diese Zahl kann zwischen 4 und 64 liegen, Diese Zahl kann zwischen 4 und 64 liegen, muss aber durch 4 teilbar sein, wobei der Standardwert 24 ist.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_set_circle_precision(precision);

Argument Beschreibung
Präzision Die Genauigkeit aller Kreise, die nach dem Aufruf dieser Funktion gezeichnet werden.

Rückgabe:

N/A

Beispiel:

draw_set_circle_precision(12);

Der obige Code zeichnet alle Kreise unter Verwendung eines 12-seitigen Polygons.
