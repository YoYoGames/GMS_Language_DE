zeichnen_ellipse

Mit dieser Funktion können Sie entweder einen Umriss einer Ellipse oder eine gefüllte Ellipse zeichnen, indem Sie einen rechteckigen Bereich definieren, in den dann die Ellipse passend gezeichnet wird. Wie genau die Zeichnung ist, können Sie mit der Funktion draw_set_circle_precision().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_ellipse(x1, y1, x2, y2, outline);

Argument Beschreibung
x1 Die x-Koordinate der linken Seite der Ellipse.
y1 Die y-Koordinate des oberen Randes der Ellipse.
x2 Die x-Koordinate der rechten Seite der Ellipse.
y2 Die y-Koordinate der Unterseite der Ellipse.
Umriss Ob die Ellipse gefüllt (false) oder als ein Pixel breiter Umriss (true) gezeichnet wird.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
draw_ellipse(100, 100, 300, 200, false);

Dadurch wird eine gefüllte Ellipse innerhalb des definierten rechteckigen Bereichs gezeichnet.
