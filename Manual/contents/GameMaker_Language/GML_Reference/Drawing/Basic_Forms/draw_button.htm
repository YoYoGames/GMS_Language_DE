Zeichnen_Schaltfläche

Diese Funktion zeichnet einen sehr einfachen, rechteckigen "Knopf" unter Verwendung der aktuell gewählten Zeichenfarbe und Alpha, wobei das Argument up definiert, wie der abgeschrägte Kanteneffekt aussieht, wie im Bild unten gezeigt:

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader folgende Eingaben erwarten folgende Eingaben erwarten: Vertex, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie auf diese Weise Formen zeichnen müssen, dann sollte der Shader in diesem Sinne angepasst werden.

Syntax:

draw_button(x1, y1, x2, y2, up);

Argument Beschreibung
x1 Die x-Koordinate der linken Seite der Schaltfläche
y1 Die y-Koordinate der Oberseite der Schaltfläche
x2 Die x-Koordinate der rechten Seite der Schaltfläche
y2 Die y-Koordinate der Unterseite der Schaltfläche
auf Ob die Taste oben(true) oder unten(false) ist

Rückgabe:

N/A

Beispiel:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Damit wird eine Schaltfläche gezeichnet, die gedrückt erscheint, wenn die linke Maustaste gedrückt gehalten wird.
