draw_ellipse_colour

Mit dieser Funktion können Sie entweder einen Umriss einer Ellipse oder eine gefüllte Ellipse zeichnen, indem Sie einen rechteckigen Bereich definieren, in den dann die Ellipse passend gezeichnet wird. Bei einer gefüllten Ellipse können Sie die inneren und äußeren Füllfarben definieren. Wenn diese Farben nicht gleich sind, erhalten Sie einen Verlaufseffekt von der einen zur anderen (die Farbeinstellungen übersteuern die mit der Funktion draw_set_colour()). Sie können festlegen, wie präzise die Zeichnung ist, können Sie mit der Funktion draw_set_circle_precision().

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen wollen, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Wenn Sie diese Funktion verwenden, werden jedoch nur Scheitelpunkt- und Farbdaten übergeben, so dass der Shader möglicherweise nichts zeichnen (oder etwas zeichnen, aber nicht korrekt). Wenn Sie Formen auf diese Weise zeichnen müssen, sollte der Shader entsprechend angepasst werden.

Syntax:

draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline);

Argument Beschreibung
x1 Die x-Koordinate der linken Seite der Ellipse.
y1 Die y-Koordinate des oberen Randes der Ellipse.
x2 Die x-Koordinate der rechten Seite der Ellipse.
y2 Die y-Koordinate der Unterseite der Ellipse.
col1 Die Farbe der Mitte.
col2 Die Farbe der Kante.
Umriss Ob die Ellipse ein Umriss (true) oder ausgefüllt (false) ist.

Rückgabe:

N/A

Beispiel:

draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false);

Dies würde eine ausgefüllte, horizontal gestreckte Ellipse mit schwarzer Mitte und weißem Rand zeichnen.
