draw_enable_drawevent

Mit dieser Funktion können Sie wählen, ob Sie das Draw-Ereignis für alle Instanzen im Spiel aktivieren(true) oder deaktivieren(false) möchten, wodurch Sie die Kontrolle darüber erhalten, wie und wann Dinge gezeichnet werden. "Frame-Sprung"-Technik implementieren möchten. Dies ist nützlich, wenn Sie eine "Frame-Skip"-Technik implementieren möchten. Beachten Sie, dass dies nicht nur das Zeichnen von Instanzen auf dem Bildschirm verhindert, sondern das Draw-Ereignis komplett unterdrückt, was bedeutet, dass Sie vorsichtig sein sollten, da jegliche Spiellogik, die in diesem Ereignis vorhanden ist, nicht ausgeführt wird auch nicht. Eine wichtige Sache, die man über diese Funktion verstehen sollte, ist, dass, wenn man sie gleich zu Beginn des Spiels aufruft, bevor der erste Frame gerendert wird (d.h.: das Create-Event des ersten Objekts im ersten Raum des Spiels), dann wird das Spielfenster nicht gerendert. Dies könnte für Online-Projekte nützlich sein, die einen Server benötigen, der nichts rendern muss.

Syntax:

draw_enable_drawevent(enable);

Argument Beschreibung
Aktivieren Sie Wird auf true oder false gesetzt.

Rückgabe:

N/A

Beispiel:

frame_skip ++;
if (frame_skip mod 5) == 0
{
draw_enable_drawevent(true);
}
sonst
{
draw_enable_drawevent(false);
}

Der obige Code prüft eine Variable und wenn er true zurückgibt, aktiviert er das Draw-Ereignis, andernfalls ist das Draw-Ereignis deaktiviert.
