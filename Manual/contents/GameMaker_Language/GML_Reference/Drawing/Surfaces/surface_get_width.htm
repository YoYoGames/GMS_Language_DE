oberfläche_get_width

Diese Funktion gibt einfach die Breite der indizierten Oberfläche in Pixel zurück. Es ist zu beachten, dass, wenn Sie diese Funktion aufrufen, um die application_surface zu überprüfen, unmittelbar nachdem Sie ihre Größe mit surface_resize() geändert haben, wird sie nicht den neuen Wert zurückgeben, da die Änderung ein oder zwei Schritte benötigt, um vollständig verarbeitet zu werden. Nach einer Wartezeit von einem Schritt sollte es die neue Größe korrekt zurückgeben.

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_get_width(surface_id);

Argument Beschreibung
oberfläche_id Die ID der Fläche, deren Breite ermittelt werden soll.

Rückgabe:

Real

Beispiel:

sw = surface_get_width(surf);

Der obige Code speichert die Breite der in der Variablen "surf" indizierten Fläche in der Variablen "sw".
