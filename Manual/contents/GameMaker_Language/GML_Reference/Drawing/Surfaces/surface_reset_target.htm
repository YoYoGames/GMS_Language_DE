oberfläche_zurueck_ziel

Mit dieser Funktion setzen Sie alle weiteren Zeichnungen von der Zielfläche zurück auf den Bildschirm. Bitte beachten Sie, dass Sie, um das Zeichnen auf eine Oberfläche zu starten, zuerst die Funktion surface_set_target() und und danach diese Funktion für jede Zielfläche, die Sie gesetzt haben, aufgerufen haben müssen, da sonst nichts auf den Bildschirm gezeichnet wird, da alle weiteren Zeichnungen (auch in anderen Instanzen) auf die Fläche erfolgen. Sie sollten auch wissen, dass nichts nichts zu sehen sein wird, wenn die Oberfläche selbst nicht im Draw-Ereignis einer Instanz auf den Bildschirm gezeichnet wird.

HINWEIS: Wenn Sie zuvor kein Rendering-Ziel mit der Funktion surface_set_target()gesetzt haben, wird die Verwendung dieser Funktion stillschweigend (d. h. ohne ohne Fehlermeldung) alle weitere Codeausführung für das Ereignis beenden.

Syntax:

surface_reset_target();

Rückgabe:

N/A

Beispiel:

if view_current = 0
{
surface_set_target(surf);
mit (obj_Effect)
{
draw_self();
}
surface_reset_target();
}
sonst
{
draw_surface(surf, 0, 0);
}

Der obige Code prüft, welche Ansicht gerade gezeichnet wird, und wenn es view[0] ist, setzt er das Zeichenziel auf eine Fläche und zeichnet alle Instanzen des Objekts "obj_Effect", bevor er das Zeichenziel wieder zurücksetzt. Wenn die Ansicht nicht view[0] ist ist, wird die Oberfläche auf den Bildschirm gezeichnet.
