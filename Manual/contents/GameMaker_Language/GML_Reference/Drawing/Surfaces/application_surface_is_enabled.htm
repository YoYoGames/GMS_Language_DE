application_surface_is_enabled

Diese Funktion gibt true zurück, wenn die Anwendungsoberfläche zum Zeichnen verwendet wird, oder false, wenn der Bildschirmpuffer verwendet wird.

Syntax:

application_surface_is_enabled();

Rückgabe:

Boolesche

Beispiel:

if keyboard_check_pressed(vk_space)
{
if application_surface_is_enabled()
{
application_surface_enable(false);
}
sonst
{
application_surface_enable(true);
}
}

Der obige Code prüft auf einen Tastendruck und schaltet die Anwendungsoberfläche je nach Zustand ein oder aus (wie in einem Optionsmenü).
