oberfläche_ziel_ext_setzen

Diese Funktion ist für die Verwendung mit den Shader-Funktionen vorgesehen und setzt die MRT (0 - 3) für Shader auf nativer Ebene (DX9, DX11, OpenGL).

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_set_target_ext(index, surface_id);

Argument Beschreibung
index Der zu verwendende Renderzielindex (von 0 bis 3).
oberfläche_id Die ID der zu verwendenden Oberfläche.

Rückgabe:

Boolesche

Beispiel:

surface_set_target_ext(0, surf);

Der obige Code setzt das Renderziel 0 auf die in der Variablen "surf" indizierte Oberflächen-ID.
