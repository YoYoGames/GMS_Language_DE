flaeche_teil_ext_zeichnen

Diese Funktion zeichnet einen Teil der gewählten Fläche an der angegebenen Position nach den gleichen Regeln wie bei draw_surface_part()nur dass Sie jetzt den Teil skalieren, mit einer Farbe überblenden oder sein Alpha verändern können beim Zeichnen auf dem Bildschirm ändern (genau wie beim Zeichnen einer Fläche mit draw_surface_ext()).

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_part_ext(id, left, top, w, h, x, y, xscale, yscale, color, alpha);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
links Die linke Position in der Oberfläche des zu zeichnenden Teils.
oben Die obere Position in der Oberfläche des zu zeichnenden Teils.
w Die Breite des zu zeichnenden Teils, von links.
h Die Höhe des zu zeichnenden Teils, von oben.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.
x-Skala Die horizontale Skalierung, mit der das Teil gezeichnet werden soll.
y-Skala Die vertikale Skalierung, mit der das Teil gezeichnet werden soll.
Farbe Die Farbmischung, mit der das Teil gezeichnet werden soll.
alpha Die Alpha-Transparenz, mit der das Teil gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_surface_part_ext(surf, 8, 8, 32, 32, x, y, 2, 0.5, c_black, 1);

Damit wird ein 32x32 Pixel großer Bereich von 8x8 Pixeln in die in der Variablen "surf" indizierte Fläche gezeichnet. Er wird auf die doppelte Breite, aber die halbe Höhe gestreckt. Sie ist undurchsichtig und wird mit Schwarz überblendet (wodurch sie zu einer eine Silhouette).
