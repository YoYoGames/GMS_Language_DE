surface_get_target

Mit dieser Funktion können Sie die ID der aktuellen Oberfläche ermitteln, auf die gezeichnet werden soll. Wenn Sie diese Oberfläche mit surface_create()erzeugt haben, dann ist dies die positive Ganzzahl, die die eindeutige ID für diese Oberfläche

Wenn keine benutzerdefinierte Oberfläche eingestellt ist, aber die Standard-Anwendungsoberfläche noch aktiviert ist, dann gibt die Funktion normalerweise 0 zurück. Wenn Sie jedoch zuvor die application_surface manipuliert haben, sollten Sie prüfen, ob die erhaltene ID nicht mit dem aktuellen Wert der Variable "application_surface" übereinstimmt, falls dieser nicht mehr 0 ist.

Wenn keine benutzerdefinierte Oberfläche eingestellt ist und die standardmäßige application_surface ebenfalls deaktiviert wurde, gibt die Funktion -1 zurück.

Syntax:

surface_get_target();

Rückgabe:

Integer (-1, wenn kein Oberflächenziel existiert, sonst die ID der Zieloberfläche)

Beispiel:

surface_get_target() != -1
{
surface_reset_target();
}
draw_surface(surf, 0, 0);

Der obige Code prüft, ob das aktuelle Rendering-Ziel eine Fläche ist oder nicht, und wenn ja, setzt er das Ziel zurück und zeichnet dann eine Fläche.
