application_surface_draw_enable

Mit dieser Funktion können Sie das automatische Zeichnen der Anwendungsoberfläche aktivieren oder deaktivieren. Standardmäßig ist dies aktiviert, aber in vielen Fällen werden Sie selbst bestimmen wollen, wann und wie die Oberfläche gezeichnet wird (z. B. bei Verwendung von Shadern ). In diesem Fall können Sie diese Funktion auf false setzen, so dass Sie die Oberfläche selbst zeichnen können, wann und wie Sie es wünschen. Es ist wichtig zu beachten, dass, wenn Sie das automatische Zeichnen ausschalten und die Anwendungsoberfläche selbst zeichnen, bestimmte Probleme auftreten können mit der Alpha-Komponente von Sprites und der Oberfläche selbst auftreten können. Das liegt daran, dass GameMaker Studio 2 die Programmoberfläche ohne Alpha-Blending zeichnet, wenn das automatische Zeichnen eingeschaltet ist. Wenn Sie das automatische Zeichnen ausschalten, müssen Sie müssen Sie dies selbst handhaben, z. B. mit dem folgenden Code:

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Syntax:

application_surface_draw_enable(flag);

Argument Beschreibung
Flagge Das Flag wird entweder true (aktiviert, der Standardwert) oder false (deaktiviert) sein.

Rückgabe:

N/A

Beispiel:

application_surface_draw_enable(false);

Der obige Code schaltet das automatische Zeichnen der Applikationsoberfläche aus.
