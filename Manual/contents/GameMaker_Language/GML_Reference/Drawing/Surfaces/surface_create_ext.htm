oberfläche_erstellen_ext

Mit dieser Funktion können Sie eine Fläche an ein Canvas-Element anhängen, das bereits in Ihrer Webseite vorhanden ist, was bedeutet, dass Sie Teile Ihres Spiels effektiv aufteilen können, um sie an verschiedenen Stellen innerhalb der Seite zu zeichnen. Zu diesem Zweck müssen Sie das Canvas-Element in der *.html-Seite Ihres Spiels korrekt definiert haben, indem Sie die richtigen Größen und Namen verwenden, die den Flächen entsprechen, die Sie erstellen möchten. Sie haben also Ihre "Haupt"-Leinwand und dann Ihre sekundären Oberflächen Canvas-Elemente, die mit dieser Funktion den richtigen Oberflächen zugewiesen werden. Das folgende Bild ist ein Beispiel dafür, wie eine Seite mit drei Canvas-Elementen eingerichtet werden würde:

Das Seitenlayout kann knifflig sein, vor allem, wenn Sie möchten, dass alle Elemente richtig ausgerichtet sind, aber sobald Sobald die schwierige Aufgabe der Erstellung des Layouts abgeschlossen ist, können Sie diese HTML-Datei als Standardseitendatei für das Spiel hinzufügen, indem Sie die Registerkarte HTML5 der Spieloptionen verwenden. Als Nächstes sollten Sie den Raum und die Ansichten einrichten, da jede Oberfläche mit einer bestimmten Ansicht verknüpft werden muss, um die Spielbilder zu "erfassen" (siehe die Ansichtsvariable view_surface_id[0...7]). Das folgende Bild zeigt, wie der Spielraum für das obige Canvas-Beispiel eingerichtet werden würde:

Schließlich würden Sie dann diese Funktion verwenden, um die Flächen zu erstellen, wobei der Name derselbe ist wie bei den der für die Canvas-Elemente verwendet wird, und die Größe entspricht der Größe desselben Canvas. Die Funktion gibt den Index der Fläche zurück, der in einer Variablen für zukünftige Funktionsaufrufe gespeichert werden sollte. Wenn die Oberfläche zum ersten Mal erstellt wird, kann sie folgendes enthalten "Rauschen" enthalten, da es sich im Grunde nur um einen Speicherbereich handelt, der für diesen Zweck beiseite gelegt wird (und dieser Speicher kann noch Informationen enthalten), daher sollten Sie die Oberfläche vor der Verwendung mit einer Funktion wie draw_clear_alpha().

HINWEIS: Diese Funktion ist nur für die Verwendung mit dem HTML5-Modul verfügbar.

Syntax:

surface_create_ext(name, w, h);

Argument Beschreibung
Name Der Name des Canvas-Elements, mit dem die Fläche verknüpft werden soll.
w Die Breite der zu erstellenden Fläche.
h Die Höhe der zu erstellenden Fläche.

Rückgabe:

Real

Beispiel:

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

Der obige Code erzeugt zwei unterschiedlich große Flächen, die jeweils einem anderen Canvas-Element zugewiesen werden, und dann werden diese Flächen zwei Ansichten zugewiesen, damit der richtige Teil des Raums erfasst wird.
