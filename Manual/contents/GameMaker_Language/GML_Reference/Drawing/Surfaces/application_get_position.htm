application_get_position

Wenn Sie in den Spieloptionen für eine Zielplattform das Häkchen bei "Seitenverhältnis beibehalten" gesetzt haben, stellt GameMaker Studio 2 die Zeichenposition für die Programmoberfläche automatisch so ein, dass sie auf dem gegebenen Display korrekt zentriert und skaliert dargestellt wird dem gegebenen Display angezeigt wird. Wenn Sie jedoch diese Oberfläche manipulieren und sie selbst zeichnen möchten, bietet Ihnen diese Funktion eine einfache Möglichkeit, genau herauszufinden, wo innerhalb der Anzeige oder des Fensters die Oberfläche gezeichnet wurde, so dass Sie sie dann selbst zeichnen oder GUI-Bilder oder Post-Draw-Bilder an ihr ausrichten können.

Die Funktion gibt ein Array mit vier Schlüsseln zurück, wobei die Schlüssel 0 und 1 die x- und y-Position der linken oberen Ecke der Oberfläche und die Schlüssel 2 und 3 die x- und y-Position der rechten unteren Ecke der Oberfläche sind. der Oberfläche sind, alle relativ zur Größe des Displays oder Fensters.

Syntax:

application_get_position();

Rückgabe:

Array

Beispiel:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

Der obige Code ermittelt die Position der Anwendungsoberfläche sowie die absolute Breite und Höhe in Bezug auf das Anzeigefenster und speichert sie in vier Variablen zur späteren Verwendung.
