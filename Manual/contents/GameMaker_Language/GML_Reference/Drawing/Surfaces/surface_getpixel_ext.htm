oberfläche_getpixel_ext

Mit dieser Funktion können Sie den vollen abgr 32bit Wert eines beliebigen Pixels einer (zuvor erstellten) Oberfläche erhalten.

HINWEIS: Diese Funktion hat einen enormen Leistungseinbruch zur Folge und sollte daher nur verwendet werden, wenn es unbedingt notwendig ist.

Syntax:

surface_getpixel_ext(surface_id, x, y);

Argument Beschreibung
oberfläche_id Die ID der zu verwendenden Oberfläche.
x Die x-Koordinate des zu prüfenden Pixels
y Die y-Koordinate des zu prüfenden Pixels

Rückgabe:

Real

Beispiel:

col = surface_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Der obige Code holt sich den 32-Bit-Farbwert an der Position der Maus und teilt ihn dann in seine Komponentenwerte auf und speichert sie in Variablen.
