oberfläche_ziel_ext_holen

Diese Funktion ruft die Oberflächen-ID ab, die einem der 4 für Oberflächen verfügbaren Render-Targets zugeordnet ist. Sie geben den Index des zu prüfenden Rendering-Ziels an und die Funktion gibt -1 zurück, wenn keine Oberfläche zugeordnet ist, oder einen Integer-Wert >= 0, der die ID der zugewiesenen Oberfläche (wie von der Funktion surface_create()).

Syntax:

surface_get_target_ext(index);

Argument Beschreibung
index Der zu prüfende Renderzielindex (von 0 bis 3).

Rückgabe:

Integer (-1 für kein Oberflächenziel, oder ein Oberflächen-ID-Wert)

Beispiel:

if surface_get_target_ext(0) == -1
{
surface_set_target_ext(0, global.Surf);
}

Der obige Code prüft zunächst, ob das Shader-Rendering-Target 0 auf eine Oberfläche gesetzt wurde, und wenn nicht, dann wird eine zugewiesen.
