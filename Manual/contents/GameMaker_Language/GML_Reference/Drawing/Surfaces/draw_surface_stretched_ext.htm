flaeche_gestreckt_ext zeichnen

Diese Funktion macht genau dasselbe wie die draw_surface_stretched() Funktion mit der zusätzlichen Möglichkeit, die Farbmischung und den Alphawert für die Oberfläche zu setzen, wenn sie gezeichnet wird (ähnlich wie bei der Funktion draw_surface_ext()).

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass diese jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten daher IMMER prüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.
w Die Breite, in der die Fläche gezeichnet werden soll.
h Die Höhe, in der die Fläche gezeichnet werden soll.
Farbe Die Farbe, mit der die Oberfläche eingefärbt werden soll.
alpha Das Alpha, mit dem die Oberfläche überblendet werden soll.

Rückgabe:

N/A

Beispiel:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Dadurch wird die angegebene Fläche mit der linken Ecke an der x/y-Position der Instanz gezeichnet und so gedehnt, dass sie einen Bereich von 200x200 Pixeln einnimmt, ohne Überblendung, aber mit partieller Transparenz.
