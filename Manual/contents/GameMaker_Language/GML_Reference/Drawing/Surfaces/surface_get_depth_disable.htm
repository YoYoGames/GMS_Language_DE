surface_get_depth_disable

Diese Funktion prüft, ob die automatische Tiefenpuffer-Generierung für Oberflächen aktiviert ist. Normalerweise haben alle Oberflächen Tiefenpuffer, so dass, wenn Sie 3D-Objekte auf sie zeichnen, diese korrekt nach Tiefe sortiert werden, jedoch verdoppelt die Zuweisung von Tiefenpuffern im Wesentlichen verdoppelt jedoch die Größe der Oberflächen, was ein übermäßiger und unnötiger Overhead sein kann, besonders wenn Ihr Spiel sehr speicherintensiv oder überwiegend 2D ist. In diesen Fällen können Sie dies mit dieser Funktion überprüfen und den Tiefenpuffer für Oberflächen deaktivieren bei Bedarf mit der Funktion surface_depth_disable().

Syntax:

surface_get_depth_disable();

Rückgabe:

Boolesche

Beispiel:

if (!surface_exists(surf))
{
if surface_get_depth_disable() == false
{
surface_depth_disable(true);
}
surf = surface_create(room_width, room_height);
}

Der obige Code prüft, ob die angegebene Oberfläche existiert, und wenn nicht, dann prüft er den aktuellen Status des Oberflächentiefenpuffers und deaktiviert ihn stattdessen, bevor er schließlich die Oberfläche erzeugt.
