oberfläche_kopieren_teil

Diese Funktion nimmt einfach das Bild von einer Fläche und kopiert es auf eine andere an der angegebenen lokalen Position innerhalb dieser Fläche (wobei die (0,0)-Position die linke obere Ecke der Zielfläche ist). Sie können eine lokale x- und y-Position angeben angeben, von der aus kopiert werden soll, sowie die Breite und Höhe des Ausschnitts. Bitte beachten Sie, dass es sich hierbei um Koordinaten handelt, die sich auf die Größe der Fläche beziehen und nicht auf die Position, an der die Fläche im Raum gezeichnet wird. Wenn die Zielfläche bereits Informationen wird diese durch das Kopieren überschrieben, und die Funktion verändert die Quellfläche in keiner Weise.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass diese jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten daher IMMER prüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_copy_part(Ziel, x, y, Quelle, xs, ys, ws, hs);

Argument Beschreibung
Ziel Die ID der Oberfläche, in die die andere Oberfläche kopiert werden soll.
x Die x-Position, auf die kopiert werden soll.
y Die y-Position, auf die kopiert werden soll.
Quelle Die ID der zu kopierenden Oberfläche.
xs Die x-Position in der Quellfläche, von der aus kopiert werden soll.
ys Die y-Position in der Quellfläche, von der aus kopiert werden soll.
ws Die Breite des Bereichs in der Quellfläche, von der kopiert werden soll.
hs Die Höhe des Bereichs in der Quellfläche, von der kopiert werden soll.

Rückgabe:

N/A

Beispiel:

if view_current == 0
{
surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
}
sonst
{
draw_surface(surf, 0, 0);
}

Der obige Code prüft die aktuell gezeichnete Ansicht und kopiert, wenn es view[0] ist, die in der Variablen "temp_surf" indizierte Fläche auf die in der Variablen "surf" indizierte Fläche. Die kopierte Fläche entspricht einem Rechteck, das der relativen Position der Maus innerhalb der Fläche, wie sie in view[1] gezeichnet werden würde. Wenn die aktuelle Ansicht eine andere als view[0] ist, wird die Oberfläche "surf" auf den Bildschirm gezeichnet.
