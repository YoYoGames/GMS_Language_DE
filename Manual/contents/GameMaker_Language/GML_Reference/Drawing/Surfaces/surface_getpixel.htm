surface_getpixel

Mit dieser Funktion kann die Farbe eines bestimmten Pixels aus einer Fläche ermittelt werden, wobei die lokalen Koordinaten der Fläche verwendet werden, wobei (0,0) die linke obere Ecke ist. Diese Funktion sollte nicht sehr oft verwendet werden, da sie extrem langsam ist und zu einer Pause in Ihrem Spiel verursachen kann.

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_getpixel(surface_id, x, y);

Argument Beschreibung
oberfläche_id Die ID der Oberfläche.
x Die x-Position auf der Oberfläche, von der aus das Pixel ermittelt werden soll.
y Die y-Position auf der Oberfläche, von der aus das Pixel geholt werden soll.

Rückgabe:

Real

Beispiel:

col = surface_getpixel(surf, 56, 78 );

Dies liefert die Farbe des Pixels an den Koordinaten (56,78) der in der Variablen "surf" indizierten Fläche.
