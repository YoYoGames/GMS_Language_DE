application_surface

Diese eingebaute Variable mit globalem Umfang kann verwendet werden, um mit einer der Oberflächenfunktionen auf die Programmoberfläche zuzugreifen. Diese Oberfläche steht permanent zur Verfügung und ist der Ort, an dem der Großteil des Zeichnens in GameMaker Studio 2 stattfindet.

Syntax:

application_surface;

Rückgabe:

Zeiger

Beispiel:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Der obige Code ändert die Größe der Anwendungsoberfläche so, dass sie ein 1:1-Verhältnis mit der GUI-Ebene hat.
