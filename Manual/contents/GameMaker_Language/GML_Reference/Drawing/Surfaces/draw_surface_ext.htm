fläche_zeichnen_ext

Diese Funktion zeichnet die angegebene Oberfläche wie in der Funktion zeichne_Fläche() gezeichnet, jedoch mit zusätzlichen Optionen zum Ändern von Skalierung, Blending, Rotation und Alpha der zu zeichnenden Oberfläche. Das Ändern dieser Werte ändert die Ressource in keiner Weise (nur wie sie gezeichnet wird).

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.
x-Skala Die horizontale Skala.
y-Skala Die vertikale Skala.
rot Die Drehung oder der Winkel zum Zeichnen der Oberfläche.
col Die Farbe, mit der die Oberfläche überblendet werden soll.
alpha Die Alpha-Transparenz für das Zeichnen der Oberfläche.

Rückgabe:

N/A

Beispiel:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

Der obige Code zeichnet eine in der Variablen "surf" indizierte Fläche an der Position (0,0) im Raum, mit doppeltem Originalmaßstab, rot überblendet und halbtransparent.
