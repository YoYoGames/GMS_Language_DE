oberfläche_sichern

Mit dieser Funktion wird eine Oberfläche unter dem angegebenen Dateinamen auf der Festplatte gespeichert. Die Oberfläche muss als Datei im *.png-Format gespeichert werden.

Syntax:

surface_save(surface_id, fname);

Argument Beschreibung
oberfläche_id Die ID der Oberfläche, die als Zeichenziel festgelegt werden soll.
fname Der Name der gespeicherten Bilddatei.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("S")
{
surface_save(surf, "test.png");
}

Der obige Code prüft, ob der Benutzer die Taste "S" auf der Tastatur drückt, und wenn dies der Fall ist, wird die in der Variablen "surf" indizierte Oberfläche auf der Festplatte gespeichert.
