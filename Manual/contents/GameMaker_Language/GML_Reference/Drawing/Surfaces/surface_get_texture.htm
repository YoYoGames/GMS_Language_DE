oberfläche_holen_textur

Diese Funktion gibt einen speziellen Zeiger für die Oberflächentextur zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in allgemeinen 3D- und einigen der 2D-Primitivfunktionen.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass diese jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten daher IMMER prüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_get_texture(surface_id);

Argument Beschreibung
oberfläche_id Die ID der Fläche, deren Breite ermittelt werden soll.

Rückgabe:

Zeiger

Beispiel:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

Der obige Code zeichnet einen Dreiecksstreifen mit 4 Eckpunkten, der mit der Textur in der Variablen "tex" texturiert ist, die wiederum von einer zuvor erstellten Oberfläche stammt.
