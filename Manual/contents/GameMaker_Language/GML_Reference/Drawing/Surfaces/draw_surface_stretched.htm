Fläche_gestreckt_zeichnen

Diese Funktion nimmt einfach eine Fläche und spannt sie über die angegebene Breite und Höhe auf, so dass sie den Bereich einnimmt. Wie bei draw_surface() können Sie eine Fläche angeben und dann die x/y-Position im Raum für die Fläche gezeichnet werden soll, und schließlich eine Breite und eine Höhe (die Pixelwerte sein müssen).

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_stretched(id, x, y, w, h);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.
w Die Breite, in der die Fläche gezeichnet werden soll.
h Die Höhe, in der die Fläche gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_surface_stretched(surf, 10, 10, 100, 100);

Dadurch wird die in der Variablen "surf" indizierte Fläche mit ihrer linken Ecke bei (10,10) gezeichnet. Die Breite und die Höhe werden beide auf 100 gesetzt, da sie unabhängig von der tatsächlichen Breite und Höhe der Fläche so viel Platz einnehmen.
