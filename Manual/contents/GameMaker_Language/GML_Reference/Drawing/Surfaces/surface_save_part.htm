oberfläche_speichern_teil

Diese Funktion speichert einen Teil einer Oberfläche unter dem angegebenen Dateinamen auf der Festplatte. Die Oberfläche muss als Datei im *.png-Format gespeichert werden, und die (x,y)-Position muss als lokale Koordinaten der Oberfläche angegeben werden, wobei zu beachten ist, dass die obere linke Ecke der Oberfläche immer (0,0) ist.

Syntax:

surface_save_part(surface_id, fname, x, y, width, height);

Argument Beschreibung
oberfläche_id Die ID der Oberfläche, die als Zeichenziel festgelegt werden soll.
fname Der Name der gespeicherten Bilddatei.
x Die Start-X-Position innerhalb der Fläche.
y Die y-Startposition innerhalb der Fläche.
Breite Die Breite des zu speichernden Teils.
Höhe Die Höhe des zu speichernden Teils.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("S")
{
surface_save_part(surf, "test.png", 0, 0, 100, 100);
}

Der obige Code prüft, ob der Benutzer die Taste "S" auf der Tastatur drückt, und wenn dies der Fall ist, wird ein Teil der in der Variablen "surf" indizierten Oberfläche auf der Festplatte gespeichert.
