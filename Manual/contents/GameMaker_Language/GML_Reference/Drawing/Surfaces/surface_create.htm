Oberfläche_erstellen

Diese Funktion wird verwendet, um eine Fläche zu erzeugen und gibt den Index der Fläche zurück, der in einer Variablen für zukünftige Funktionsaufrufe gespeichert werden sollte. Wenn die Oberfläche zum ersten Mal erstellt wird, kann sie "Rauschen" enthalten, da es sich im Grunde nur um einen Bereich von Speicherbereich ist, der für diesen Zweck zur Seite gelegt wird (und dieser Speicher kann noch Informationen enthalten), daher sollten Sie die Oberfläche vor der Verwendung mit einer Funktion wie draw_clear_alpha().

Es wird dringend empfohlen, alle Oberflächen mit einer Größe zu erstellen, die eine Potenz von 2 ist, d. h. z. B. 16, 128, 512 oder 1024 Pixel groß ist. Dies ist auf bestimmten Plattformen (wie Windows und MacOS) nicht unbedingt notwendig, aber es erhöht sicherlich die Kompatibilität auf diesen Zielen, während es für HTML5 und Geräte unerlässlich ist und es ist sehr wichtig, dass Sie sich daran erinnern, sonst könnten Sie später Probleme bekommen.

Syntax:

surface_create(w, h);

Argument Beschreibung
w Die Breite der zu erstellenden Fläche.
h Die Höhe der zu erstellenden Fläche.

Rückgabe:

Real

Beispiel:

if !surface_exists(surf)
{
surf = surface_create(1024, 1024);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
surface_reset_target();
view_surface_id[0] = surf;
}

Der obige Code prüft, ob eine Oberfläche existiert, und wenn nicht, wird eine Oberfläche mit einer Breite von 1024 Pixeln und einer Höhe von 1024 Pixeln erstellt, wobei der Index der Variablen "surf" zugewiesen wird. Das Zeichenziel wird dann auf die neue Fläche gesetzt, die wird gelöscht und transparent gemacht, bevor das Zeichenziel auf die Anzeige zurückgesetzt wird. Abschließend wird die Fläche einer Ansicht zugewiesen.
