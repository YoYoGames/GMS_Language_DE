oberfläche_kopie

Diese Funktion nimmt einfach das Bild von einer Oberfläche und kopiert es auf eine andere an der angegebenen lokalen Position innerhalb dieser Oberfläche (wobei die (0,0)-Position die linke obere Ecke der Zieloberfläche ist). Wenn die Zielfläche bereits Informationen hat, werden diese durch die Kopie überschrieben, und die Funktion verändert die Quellfläche in keiner Weise.

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_copy(Ziel, x, y, Quelle);

Argument Beschreibung
Ziel Die ID der Oberfläche, in die die andere Oberfläche kopiert werden soll.
x Die x-Position, auf die kopiert werden soll.
y Die y-Position, auf die kopiert werden soll.
Quelle Die ID der zu kopierenden Oberfläche.

Rückgabe:

N/A

Beispiel:

if view_current == 0
{
surface_copy(surf, 0, 0, temp_surf);
}
sonst
{
draw_surface(surf, 0, 0);
}

Der obige Code prüft die aktuelle Ansicht, die gezeichnet wird, und wenn es view[0] ist, kopiert er die in der Variablen "temp_surf" indizierte Fläche auf die in der Variablen "surf" indizierte Fläche. Ist die aktuelle Ansicht eine andere als view[0] ist, wird die Oberfläche "surf" auf den Bildschirm gezeichnet.
