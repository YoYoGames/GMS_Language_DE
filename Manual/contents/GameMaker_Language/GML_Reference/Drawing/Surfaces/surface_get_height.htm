oberfläche_holen_höhe

Diese Funktion gibt einfach die Höhe der indizierten Oberfläche in Pixel zurück. Es ist zu beachten, dass, wenn Sie diese Funktion aufrufen, um die application_surface unmittelbar nach einer Änderung ihrer Größe mit surface_resize() geändert haben, wird sie nicht den neuen Wert zurückgeben, da die Änderung ein oder zwei Schritte benötigt, um vollständig verarbeitet zu werden. Nach einer Wartezeit von einem Schritt sollte es die neue Größe korrekt zurückgeben.

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

surface_get_height(surface_id);

Argument Beschreibung
oberfläche_id Die ID der Oberfläche, von der die Höhe ermittelt werden soll.

Rückgabe:

Real

Beispiel:

sh = surface_get_height(surf);

Der obige Code speichert die Höhe der in der Variablen "surf" indizierten Fläche in der Variablen "sh".
