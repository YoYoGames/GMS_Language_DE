flaeche_teilen_zeichnen

Mit dieser Funktion können Sie einen Teil einer beliebigen Fläche an einer bestimmten Position im Raum zeichnen. Wie bei draw_surface() können Sie eine Fläche angeben, aber Sie müssen dann die relativen Koordinaten innerhalb der Oberfläche eines Bereichs angeben, der zum Zeichnen ausgewählt werden soll. Das bedeutet, dass eine linke Position von 0 und eine obere Position von 0 die linke obere Ecke der Fläche wäre und alle weiteren Koordinaten von dieser Position aus genommen werden sollten.

HINWEIS: Bei der Arbeit mit Flächen besteht die Möglichkeit, dass diese jederzeit nicht mehr existieren, da sie im Texturspeicher abgelegt sind. Sie sollten daher IMMER prüfen, ob eine Fläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_part(id, left, top, w, h, x, y);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
links Die linke Position in der Oberfläche des zu zeichnenden Teils.
oben Die obere Position in der Oberfläche des zu zeichnenden Teils.
w Die Breite des zu zeichnenden Teils, von links.
h Die Höhe des zu zeichnenden Teils, von oben.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_surface_part(surf, 8, 8, 32, 32, x, y);

Dies zeichnet einen 32x32 Bereich 8px mal 8px von der oberen linken Seite der in "surf" indizierten Oberfläche, an der Position der Instanz (x,y).
