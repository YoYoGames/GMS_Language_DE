fläche_gekippt_zeichnen

Diese Funktion nimmt eine Fläche und kachelt sie dann wiederholt über den gesamten Raum, ausgehend von den Koordinaten, die Sie in der Funktion angeben.

HINWEIS: Bei der Arbeit mit Oberflächen besteht die Möglichkeit, dass diese jederzeit aufhören können zu existieren, da sie im Texturspeicher abgelegt sind. Sie sollten IMMER prüfen, ob eine Oberfläche existiert, indem Sie surface_exists() überprüfen, bevor Sie sie direkt referenzieren.

Syntax:

draw_surface_tiled(id, x, y);

Argument Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
x Die x-Position, an der die Fläche gezeichnet werden soll.
y Die y-Position, an der die Fläche gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_surface_tiled(surf, x, y);

Dadurch wird die in "surf" indizierte Fläche an der eigenen x- und y-Position der Instanz gezeichnet und in jede Richtung im Raum gekachelt.
