draw_getpixel_ext

Mit dieser Funktion können Sie den vollen abgr 32bit-Wert jedes Pixels erhalten, das auf das aktuelle Rendering-Ziel gezeichnet wird. Das bedeutet, dass die Ergebnisse vom Ereignis abhängen, in dem die Funktion aufgerufen wird, und auch von der verwendeten Zieloberfläche verwendet wird.

HINWEIS: Diese Funktion verursacht einen enormen Leistungseinbruch und sollte daher nur verwendet werden, wenn dies unbedingt erforderlich ist.

Syntax:

draw_getpixel_ext(x, y);

Argument Beschreibung
x Die x-Koordinate des zu prüfenden Pixels
y Die y-Koordinate des zu prüfenden Pixels

Rückgabe:

Real

Beispiel:

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Der obige Code holt sich den 32-Bit-Farbwert an der Position der Maus und teilt ihn dann in seine Komponentenwerte auf und speichert sie in Variablen.
