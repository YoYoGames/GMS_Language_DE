draw_set_alpha

Mit dieser Funktion können Sie das Basis-Alpha für das Spiel einstellen. Dieser Wert kann von 0 bis 1 eingestellt werden, wobei 0 vollständig transparent und 1 vollständig undurchsichtig ist (der Standardwert), und wirkt sich auf alle weiteren Zeichnungen aus, einschließlich Hintergründe, Sprites, Schriftarten, Primitive und 3D.

Beachten Sie, dass, wenn Sie eines der Assets zeichnen und ein Alpha-Argument als Teil der Funktion übergeben (zum Beispiel bei der Verwendung von draw_sprite_ext()), dann wird der hier eingestellte Alpha-Wert von überschrieben. Wenn Sie also ein Sprite mit draw_sprite_ext() mit einem Alpha-Argument von 1 zeichnen, wird das Sprite vollständig undurchsichtig gezeichnet, unabhängig von dem mit draw_set_alpha() gesetzten Alpha-Wert (dies funktioniert genauso, wenn Sie draw_self() und Setzen des bild_alpha der Instanz). Beachten Sie auch, dass, wenn eine Instanz standardmäßig gezeichnet wird (d.h.: sie hat ein Sprite, aber kein Draw-Ereignis), dann wird die image_alpha-Variable auf das mit draw_set_alpha() gesetzte Alpha geklemmt wird, so dass ein image_alpha von 1 und ein draw alpha von 0.5 das Sprite bei Alpha 0.5 zeichnet, während ein image_alpha von 0.25 und ein draw alpha von 0,5 das Sprite mit 0,25 Alpha zeichnet.

Syntax:

draw_set_alpha(alpha);

Argument Beschreibung
alpha Das einzustellende Alpha (zwischen 0 und 1)

Rückgabe:

N/A

Beispiel:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "STUFE 1");

Der obige Code zeichnet einen Text an der angegebenen Position mit einem Schatteneffekt, der durch eine Änderung von Alpha und Farbe beim Zeichnen erzeugt wird.
