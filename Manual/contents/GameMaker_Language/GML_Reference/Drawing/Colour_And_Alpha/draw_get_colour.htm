draw_get_colour

Diese Funktion gibt die aktuelle Zeichenfarbe zurück, die zum Zeichnen von Formen, Text, Primitiven und untexturierten 3D-Modellen verwendet wird. Diese kann mit der Funktion draw_set_colour() Funktion gesetzt werden.

Syntax:

draw_get_colour()

Rückgabe:

Real

Beispiel:

if draw_get_colour() != c_black
{
draw_set_colour(c_black);
}

Der obige Code prüft zunächst die aktuelle Zeichenfarbe und setzt sie dann, wenn sie nicht auf c_black gesetzt ist, auf c_black.
