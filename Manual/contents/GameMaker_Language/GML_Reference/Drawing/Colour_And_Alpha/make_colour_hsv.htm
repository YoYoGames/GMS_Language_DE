make_colour_hsv

GameMaker Studio 2 stellt diese Funktion (sowie weitere) zur Verfügung, um dem Benutzer die Möglichkeit zu geben, seine eigenen Farben zu erstellen. Diese spezielle Funktion nimmt drei Komponenten, den Farbton, die Sättigung und den Wert (auch als "Leuchtkraft" bekannt) um die gewünschte Farbe zu erzeugen. Diese Werte werden als zwischen 0 und 255 liegend angenommen, so dass Sie damit 16.777.216 (256*256*256) Farben erzeugen können! Unten sehen Sie ein Bild, wie diese Komponenten aussehen, wenn sie getrennt sind:

Das Bild auf der linken Seite ist eine Aufschlüsselung der einzelnen Komponenten der Funktion, und dann auf der rechten Seite ist eine Illustration wie sich die Änderung dieser Komponenten auf die Endfarbe auswirkt. Wie Sie sehen, können wir die Farbe auch ohne Änderung des Farbtons (er bleibt bei 0, rot) mit den beiden anderen Einstellungen drastisch verändern, wobei die Sättigung den Farbton von grau (0) zu einem reinen Ton (255), und mit dem Wert von dunkler (0) zu heller (255).

Syntax:

make_colour_hsv(hue, sat, val);

Argument Beschreibung
Farbton Der Farbton der Farbe
satt Wie gesättigt die Farbe ist
val Wie dunkel die Farbe ist

Rückgabe:

Integer

Beispiel:

col = make_colour_hsv(100, 145, 255);

Der obige Code verwendet die Funktion, um eine Farbe zu erzeugen und ihren Wert in der Variablen "col" zur späteren Verwendung zu speichern.
