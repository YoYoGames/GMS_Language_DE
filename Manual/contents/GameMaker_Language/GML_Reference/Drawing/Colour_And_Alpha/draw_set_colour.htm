Farbe_setzen_zeichnen

Mit dieser Funktion können Sie die Basis-Zeichenfarbe für das Spiel festlegen. Dieser Wert wirkt sich gegebenenfalls auf alle weiteren Zeichnungen aus, einschließlich Schriftarten, Formen, Primitive und 3D. Wenn eines dieser Elemente mit einem eigenen, geänderten Farbwert gezeichnet wird, wird dieser Wert ignoriert werden.

Syntax:

draw_set_colour(col);

Argument Beschreibung
col Die Farbe, die für das Zeichnen eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "STUFE 1");

Der obige Code zeichnet einen Text an der angegebenen Position mit einem Schatteneffekt, der durch eine Änderung von Alpha und Farbe beim Zeichnen erzeugt wird.
