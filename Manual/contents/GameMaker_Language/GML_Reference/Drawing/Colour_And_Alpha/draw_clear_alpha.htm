draw_clear_alpha

Diese Funktion kann verwendet werden, um den gesamten Bildschirm mit einer bestimmten Farbe zu löschen, wobei die Alphakomponente des Ziels auf den von Ihnen eingestellten Wert gesetzt wird - diese Funktion führt keine Überblendung durch, während sie arbeitet, aber alle nachfolgenden Überblendungsoperationen können Diese Funktion führt keine Überblendung durch, aber alle nachfolgenden Überblendungsoperationen können so eingerichtet werden, dass sie das von Ihnen eingestellte Ziel-Alpha verwenden. Diese Funktion ist nur für die Verwendung im Draw-Ereignis einer Instanz vorgesehen (sie wird nicht angezeigt, wenn sie in einem anderen Ereignis verwendet wird), und sie kann auch sehr nützlich sein, um Oberflächen zu löschen, wenn sie neu erstellt werden.

Es ist erwähnenswert, dass Sie bei Verwendung des HTML5-Moduls diese Funktion verwenden können, um den Canvas-Hintergrund transparent zu machen, was dann erlaubt, dass alle Bilder, die im Host-HTML verwendet werden, zu sehen sind, wenn das Spiel darüber läuft. Um diesen Effekt zu erreichen müssen Sie zunächst in den Raum-Editor gehen und dort alle Hintergrund-Ebenen entfernen und dann die Option Clear View Background in den View Properties deaktivieren. Schließlich müssen Sie ein neues Objekt erstellen, es im ersten Raum des Spiels platzieren und ihm eine sehr hohe Tiefe zuweisen (z. B. 10000). Verwenden Sie dann im Pre Draw-Ereignis diese Funktion, um das Canvas-Alpha auf die gewünschte Farbe und Transparenz zu setzen. Sie es haben möchten.

Syntax:

draw_clear_alpha(col, alpha);

Argument Beschreibung
col Die Farbe, mit der der Bildschirm geleert wird
alpha Die Transparenz der Farbe, mit der der Bildschirm geleert wird

Rückgabe:

N/A

Beispiel:

draw_clear_alpha(c_red, 0.5);

Dadurch wird der Bildschirm mit 50 % (0,5) Transparenz mit der Farbe Rot gelöscht.
