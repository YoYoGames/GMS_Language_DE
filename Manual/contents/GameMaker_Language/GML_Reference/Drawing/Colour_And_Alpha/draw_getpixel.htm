draw_getpixel

Mit dieser Funktion können Sie den Farbwert eines beliebigen Pixels abrufen, das auf das aktuelle Rendering-Ziel gezeichnet wird. Das bedeutet, dass die Ergebnisse vom Ereignis abhängen, in dem die Funktion aufgerufen wird, und auch von der verwendeten Zieloberfläche. Beachten Sie, dass diese Funktion keine Alphawerte zurückgibt, dafür sollten Sie die Funktion draw_getpixel_ext().

HINWEIS: Diese Funktion verursacht einen enormen Leistungseinbruch und sollte daher nur verwendet werden, wenn dies unbedingt erforderlich ist.

Syntax:

draw_getpixel(x, y);

Argument Beschreibung
x Die x-Koordinate des zu prüfenden Pixels
y Die y-Koordinate des zu prüfenden Pixels

Rückgabe:

Real

Beispiel:

draw_set_colour(draw_getpixel(500,200));

Der obige Code setzt die Zeichenfarbe für alle Texte, Formen und Primitive auf die Farbe des Pixels bei (500,200) auf dem Spielbildschirm.
