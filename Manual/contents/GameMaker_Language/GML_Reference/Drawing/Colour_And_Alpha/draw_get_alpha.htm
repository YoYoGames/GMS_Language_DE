draw_get_alpha

Diese Funktion gibt den aktuellen Wert von "draw alpha" zurück, der zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) liegt. Das Zeichen-Alpha beeinflusst die Transparenz aller Zeichenfunktionen und kann mit der draw_set_alpha() Funktion.

Syntax:

draw_get_alpha()

Rückgabe:

Real

Beispiel:

if draw_get_alpha() != 1
{
draw_set_alpha(1);
}

Der obige Code prüft zunächst den aktuellen Draw-Alpha-Wert und setzt ihn dann, wenn er nicht auf 1 gesetzt ist, auf 1.
