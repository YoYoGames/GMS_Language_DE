make_colour_rgb

GameMaker Studio 2 stellt diese Funktion (sowie weitere) zur Verfügung, um dem Benutzer die Möglichkeit zu geben, seine eigenen Farben zu erstellen. Diese spezielle Funktion benötigt drei Komponenten, die rote, die grüne und die blaue Komponente der Farbe, die die Sie erzeugen möchten. Diese Werte werden als zwischen 0 und 255 liegend angenommen, so dass Sie damit 16.777.216 (256*256*256) Farben erzeugen können! Unten sehen Sie ein Bild, wie diese Komponenten aussehen, wenn sie getrennt sind:

Das Bild auf der linken Seite ist eine Aufschlüsselung der einzelnen Komponenten der Funktion, und dann auf der rechten Seite ist eine Illustration wie sich eine Änderung dieser Komponenten auf die Endfarbe auswirkt.

Syntax:

make_colour_rgb(rot, grün, blau);

Argument Beschreibung
rot Der Rotanteil der Farbe
grün Der Grünanteil der Farbe
blau Der Blauanteil der Farbe

Rückgabe:

Int

Beispiel:

col = make_colour_rgb(100, 145, 255);

Der obige Code verwendet die Funktion, um eine Farbe zu erzeugen und ihren Wert in der Variablen "col" zur späteren Verwendung zu speichern.
