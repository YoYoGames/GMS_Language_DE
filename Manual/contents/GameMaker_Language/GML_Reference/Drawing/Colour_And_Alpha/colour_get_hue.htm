farbe_holen_farbton

Diese Funktion gibt den Farbton der angegebenen Farbe zurück. Dies ist der "reine" Farbton, der Teil der Farbton-, Sättigungs- und Wertmethode (Luminanz) zur Definition einer Farbe ist. Das folgende Bild veranschaulicht, wie dieser Wert mit der HSV-Skala der Farbe entspricht:

Syntax:

colour_get_hue(col);

Argument Beschreibung
col Die zu prüfende Farbe

Rückgabe:

Int

Beispiel:

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

Der obige Code ruft den Farbton ab, der zur Erstellung der Farbkonstante "c_teal" verwendet wird, und mischt ihn dann mit Weiß im Verhältnis 50:50, wobei die resultierende Farbe in der Variablen "col" gespeichert wird.
