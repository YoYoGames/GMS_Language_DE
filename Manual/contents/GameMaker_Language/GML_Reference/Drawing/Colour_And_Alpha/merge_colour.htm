merge_colour

Mit dieser Funktion können Sie zwei Farben nehmen und sie dann zu einer neuen Farbe zusammenfügen. Die Menge jeder der Komponentenfarben kann durch Ändern des Arguments "amount" definiert werden, wobei ein Wert von 0 die erste Farbe (col1) zurückgibt, Ein Wert von 1 ergibt die zweite Farbe (col2) und ein Wert dazwischen ergibt die entsprechende Mischung. Ein Wert von 0,5 zum Beispiel mischt die beiden Farben gleichmäßig. Das folgende Bild veranschaulicht, wie dies funktioniert, indem die Farben Rot und Blau zusammen:

Syntax:

merge_colour(col1, col2, amount);

Argument Beschreibung
col1 Die erste Farbe, die verschmilzt
col2 Die zweite Farbe zum Verschmelzen
Betrag Wie viel von jeder Farbe gemischt werden soll. Beispiel: 0 gibt Farbe 1 zurück, 1 gibt Farbe 2 zurück, und 0,5 würde eine Mischung beider Farben zu gleichen Teilen ergeben

Rückgabe:

Int

Beispiel:

col = merge_colour(c_lime, c_orange, 0.5);

Der obige Code verwendet die Funktion, um eine Farbe zu erzeugen, indem er Limette und Orange 50/50 mischt, und speichert dann ihren Wert in der Variablen "col" zur späteren Verwendung.
