spalt_teil_zeichnen

Mit dieser Funktion können Sie einen Teil eines beliebigen Sprites an einer bestimmten Position im Raum zeichnen. Wie bei draw_sprite() können Sie ein Sprite und ein Teilbild zum Zeichnen angeben, dann müssen Sie die relativen Koordinaten innerhalb des Sprites für den Bereich angeben, der zum Zeichnen ausgewählt werden soll. Das bedeutet, dass eine linke Position von 0 und eine obere Position von 0 die linke obere Ecke des Sprites wäre und alle weiteren Koordinaten von dieser Position aus genommen werden sollten. Das folgende Bild zeigt ein Beispiel wie dies funktioniert:

Beachten Sie, dass diese Funktion möglicherweise nicht wie erwartet funktioniert, wenn die Texturseite das automatische Beschneiden zulässt, da der zusätzliche "leere" Platz um das Sprite für die Erstellung der Texturseite entfernt wurde. Um dieses Problem zu beheben, müssen Sie in den Einstellungen der Texturseite (im Texturgruppen-Editor) die Option Automatisches Zuschneiden deaktivieren.

HINWEIS: Beim Zeichnen mit dieser Funktion werden der Sprite-X-Offset und der Sprite-Y-Offset ignoriert und der Sprite-Teil wird mit der linken oberen Ecke an der angegebenen x-/y-Position im Raum gezeichnet.

HINWEIS: Diese Funktion ist nur für Bitmap-Sprites nützlich und funktioniert nicht mit SWF- oder JSON (Spine)-Sprites.

Syntax:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

Argument Beschreibung
Sprite Der Index des zu zeichnenden Sprites.
subimg Das Subimg (Frame) des zu zeichnenden Sprites (image_index oder -1 entsprechen dem aktuellen Frame der Animation im Objekt).
links Die x-Position auf dem Sprite der oberen linken Ecke des zu zeichnenden Bereichs.
oben Die y-Position auf dem Sprite der oberen linken Ecke des zu zeichnenden Bereichs.
Breite Die Breite des zu zeichnenden Bereichs.
Höhe Die Höhe des zu zeichnenden Bereichs.
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Dies zeichnet das der Instanz zugewiesene Sprite (sprite_index) und sein aktuelles Animationsbild (image_index), rasiert jedoch einen 4-Pixel-Rand von der Breite auf beiden Seiten und einen 8-Pixel-Rand von der Höhe vom unteren Rand des ursprünglichen 24x24 Pixel-Sprite.
