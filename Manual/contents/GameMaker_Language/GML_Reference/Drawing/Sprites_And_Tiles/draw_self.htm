zeichnen_selbst

Diese Funktion zeichnet das der Instanz zugewiesene Sprite genau so, wie es gezeichnet werden würde, wenn das Draw-Ereignis keinen Code oder Aktionen enthielte, und reflektiert und Änderungen, die in anderen Ereignissen an den Sprite-Variablen vorgenommen wurden.

HINWEIS: Wenn der Instanz kein Sprite zugewiesen ist, wird diese Funktion einen Fehler auslösen!

Syntax:

draw_self();

Rückgabe:

N/A

Beispiel:

draw_self();

Dies bewirkt, dass die Instanz sich selbst mit den Eigenschaften zeichnet, die durch die eingebauten Sprite-Variablen definiert sind.
