kachel_zeichnen

Mit dieser Funktion können Sie eine beliebige Kachel aus einem Kachelsatz zeichnen - komplett mit Drehungen, Flips und Spiegelungen - und dabei den Rahmen (falls animiert) und die Position innerhalb des Raums festlegen. Der Kachelsatz-ID-Wert ist der Kachelsatz-Index, wie er in der IDE eingestellt ist, und die Kachel data ist der Daten-"Blob", der alle Informationen über den Kachelindex und die vorgenommenen Transformationen enthält. Sie können mit der Funktion tilemap_get() verwenden, um die Kacheldaten zu erhalten, die dann mit den Kachelfunktionen manipuliert werden können, oder Sie können sie selbst konstruieren, indem Sie die folgenden Kachelkonstanten zusammen mit dem Indexwert der zu erstellenden Kachel:

Konstant Beschreibung
kachel_rotieren Wird verwendet, um das Rotationsbit eines Kacheldatenblob zu setzen/zu holen.
tile_mirror Wird verwendet, um das Spiegelbit eines Kacheldatenblob zu setzen/zu holen.
kachel_flip Wird verwendet, um das Flip-Bit eines Kacheldatenblob zu setzen/zu holen.
tile_index_mask Eine spezielle Konstante für die "und"-Verknüpfung mit dem Kacheldaten-Blob, um den Kachelindex zu extrahieren.


Sie würden diese Konstanten mit dem Kachelindexwert bitweise "oder", um die endgültigen Kacheldaten zu erhalten:

var tiledata = 8 | tile_rotate | tile_flip;

Der obige Code erstellt einfach eine lokale Variable, um die Kacheldaten für die Kachel mit dem Index 8 auf dem Kachelsatz zu speichern, und sie wird gedreht und gespiegelt. Diese kann dann verwendet werden, um die Kachel an diesem Index aus dem von Ihnen angegebenen Kachelsatz zu zeichnen. Die Kachel wird in der Ebenentiefe der Instanz gezeichnet, die das Zeichnen durchführt, und zwar an der x/y-Position, die Sie im Raum angeben. Beachten Sie, dass Sie auch den Rahmen zum Zeichnen festlegen können. Wenn die indizierte Kachel Teil einer Animation ist Wenn die indizierte Kachel Teil eines Animations-Sets ist, können Sie hier die Frame-Nummer einstellen (von 0 bis zur Anzahl der Frames - 1) und die indizierte Kachel wird einfach der Indikator dafür sein, welches animierte Kachel-Set Sie verwenden, während der angegebene Frame gezeichnet wird. Wenn die Kachel nicht Teil eines einem animierten Satz ist, sollte frame auf 0 gesetzt werden.

Syntax:

draw_tile(Kachelsatz, Kacheldaten, Frame, x, y)

Argument Beschreibung
Fliesensatz Die ID des zu verwendenden Kachelsatzes
Kacheldaten Die zu verwendenden Kacheldaten
Rahmen Die Frame-Nummer für die zu verwendende animierte Kachel (Standardwert 0 für nicht-animierte Kacheln)
x Die x-Position innerhalb des Raums, an der gezeichnet werden soll
y Die y-Position innerhalb des Raums, an der gezeichnet werden soll

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

Der obige Code holt sich die Daten einer Kachel aus der angegebenen Tilemap, setzt dann das Flag "flip" und verwendet diese Daten, um die Kachel an der Instanzposition gespiegelt zu zeichnen.
