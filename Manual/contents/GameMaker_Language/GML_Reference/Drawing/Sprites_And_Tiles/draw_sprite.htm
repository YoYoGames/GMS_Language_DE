zeichnen_sprite

Diese Funktion zeichnet das angegebene Sprite und Sub-Image an einer Position innerhalb des Spielraums. Für das Sprite können Sie die Instanzvariable sprite_index verwenden. um das aktuelle Sprite zu erhalten, das der Instanz zugewiesen ist, die den Code ausführt, oder Sie können ein beliebiges anderes Sprite-Asset verwenden. Dasselbe gilt für das Sub-Image, das ebenfalls auf die Instanzvariable bild_index gesetzt werden, wodurch das Sub-Image auf das für das aktuelle Instanz-Sprite ausgewählte gesetzt wird (beachten Sie, dass Sie ein anderes Sprite zeichnen und trotzdem den Sub-Image-Wert für die aktuelle Instanz verwenden können), oder Sie können einen beliebigen anderen Wert dafür verwenden, um ein bestimmtes Sub-Image des gewählten Sprites zu zeichnen. Wenn der Wert größer ist als die Anzahl der Sub-Images, durchläuft GameMaker Studio 2 automatisch eine Schleife, um das entsprechende Bild auszuwählen (z. B. wenn das gezeichnete Sprite 5 Sub-Images mit den Nummern 0 bis 4 hat und wir den Indexwert auf 7 setzen, dann wird die Funktion das Unterbild 3 mit der Nummer 0 zeichnen). Schließlich ist die x- und y-Position die Position innerhalb des Raums, an der das Sprite gezeichnet werden soll, und sie ist auf den x- und y-Offset des Sprites zentriert.

HINWEIS: Diese Funktion funktioniert möglicherweise nicht wie erwartet, wenn Sie Skelett-Animationssprites verwenden, und Sie stellen möglicherweise fest, dass die Funktion nur das erste Bild der Standardpose zeichnet. Sie sollten stattdessen die Funktionen draw_skeleton_* verwenden.

Syntax:

draw_sprite(sprite, subimg, x, y);

Argument Beschreibung
Sprite Der Index des zu zeichnenden Sprites.
subimg Das zu zeichnende Teilbild (Frame) des Sprites (image_index oder -1 entsprechen dem aktuellen Frame der Animation im Objekt).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.

Rückgabe:

N/A

Beispiel:

draw_sprite(sprite_index, image_index, x, y);
draw_sprite(spr_Halo, 0, x, y-32);

Dadurch wird das der Instanz zugewiesene Sprite (sprite_index) mit dem aktuellen Teilbild an der x- und y-Position der Instanz innerhalb des Raums gezeichnet. Anschließend wird das erste Bild des durch "spr_Halo" indizierten Sprites an der gleichen x- und y-Position gezeichnet aber 32 Pixel "darüber".
