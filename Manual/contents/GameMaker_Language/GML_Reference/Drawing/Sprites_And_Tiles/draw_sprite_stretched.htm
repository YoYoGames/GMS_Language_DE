draw_sprite_stretched

Diese Funktion nimmt einfach eine Sprite-Ressource und streckt sie über die angegebene Breite und Höhe, so dass sie diesen Bereich einnimmt. Wie bei draw_sprite() können Sie ein Sprite und ein Unterbild zum Zeichnen angeben, dann die x / y-Position im Raum, an der das Sprite gezeichnet werden soll, und schließlich eine Breite und eine Höhe (die Pixelwerte sein müssen). Das folgende Bild zeigt das Ergebnis dieser Funktion mit verschiedenen Argumenten:

HINWEIS: Beim Zeichnen mit dieser Funktion werden der x-Offset und y-Offset des Sprites ignoriert und das Sprite wird mit der linken oberen Ecke an der angegebenen x-/y-Position im Raum gezeichnet.

Syntax:

draw_sprite_stretched(sprite, subimg, x, y, w, h);

Argument Beschreibung
Sprite Der Index des zu zeichnenden Sprites.
subimg Das Subimg (Frame) des zu zeichnenden Sprites (image_index oder -1 entsprechen dem aktuellen Frame der Animation im Objekt).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.
w Die Breite des Bereichs, den das gestreckte Sprite einnimmt.
h Die Höhe des Bereichs, den das gestreckte Sprite einnimmt.

Rückgabe:

N/A

Beispiel:

draw_sprite_stretched(sprite_index, image_index, x, y, sprite_width, sprite_height / 2);

Dies zeichnet das der Instanz zugewiesene Sprite und sein Unterbild mit der linken Ecke an der x/y-Position der Instanz. Die Breite wird auf die gleiche Breite wie das Sprite gesetzt, und die Höhe ist die Sprite-Höhe geteilt durch zwei.
