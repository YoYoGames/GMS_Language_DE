draw_enable_swf_aa

Mit dieser Funktion können Sie Anti-Aliasing (AA) für Vektorsprites im SWF-Format aktivieren oder deaktivieren. AA glättet einfach die Kanten von Vektorbildern, um ihnen ein schöneres Aussehen zu geben. Wie viel AA verwendet wird, hängt von dem Wert ab, den Sie mit der Funktion draw_set_swf_aa_level(). Standardmäßig ist dies deaktiviert.

Syntax:

draw_enable_swf_aa(enable);

Argument Beschreibung
aktivieren. Aktivieren(true) oder deaktivieren(false) Sie AA für alle SWF-Sprites.

Rückgabe:

N/A

Beispiel:

if draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

Der obige Code überprüft den AA-Wert für Sprites im SWF-Format, und wenn er 0 ist, aktiviert er AA und setzt den Wert auf 0,5.
