draw_set_swf_aa_level

Mit dieser Funktion können Sie die Anti-Aliasing-Stufe (AA) für Vektorsprites im SWF-Format einstellen. Dies kann ein realer Wert von 0 bis 1 sein und wird die Kanten dieser Sprites "glätten". Beachten Sie, dass Sie, um diesen Effekt zu sehen, zuerst AA mit der Funktion die Funktion draw_enable_swf_aa().

Syntax:

draw_set_swf_aa_level(AA);

Argument Beschreibung
AA Der zu verwendende Anti-Aliasing-Wert von 0 bis 1.

Rückgabe:

N/A

Beispiel:

if draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

Der obige Code prüft den AA-Wert für Sprites im SWF-Format, und wenn er 0 ist, aktiviert er AA und setzt den Wert auf 0,5.
