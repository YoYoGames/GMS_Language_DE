draw_sprite_stretched_ext

Diese Funktion macht genau das Gleiche wie die draw_sprite_stretched() Funktion mit der zusätzlichen Möglichkeit, die Farbmischung und den Alphawert für das Sprite zu setzen, wenn es gezeichnet wird (ähnlich wie bei der Funktion draw_sprite_ext()).

HINWEIS: Farbüberblendung wird nur für das HTML5-Ziel empfohlen, wenn WebGL aktiviert ist, obwohl Sie die Überblendungsfarbe auch dann setzen können, wenn sie nicht aktiviert ist, und das Sprite wird normal überblendet. Alle Überblendungen auf diese Weise erzeugen jedoch ein Duplikat Sprite, das dann im Cache gespeichert und bei Bedarf verwendet wird. Dies ist bei weitem nicht optimal, und wenn Sie mehrere Farbwechsel verwenden, wird die Leistung Ihres Spiels verlangsamt, es sei denn, Sie aktivieren WebGL. Wenn Sie WebGL nicht verwenden möchten, können Sie die die Schrift-Cache-Größe einstellen, um dies bei Bedarf zu begrenzen, indem Sie die Funktion sprite_set_cache_sizee().

Syntax:

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, color, alpha);

Argument Beschreibung
Sprite Der Index des zu zeichnenden Sprites.
subimg Das Subimg (Frame) des zu zeichnenden Sprites (image_index oder -1 entsprechen dem aktuellen Frame der Animation im Objekt).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.
w Die Breite des Bereichs, den das gestreckte Sprite einnimmt.
h Die Höhe des Bereichs, den das gestreckte Sprite einnimmt.
Farbe Die Farbe, mit der das Sprite überblendet werden soll. c_white bedeutet, dass es normal angezeigt wird.
alpha Das Alpha des Sprites (von 0 bis 1, wobei 0 transparent und 1 undurchsichtig ist).

Rückgabe:

N/A

Beispiel:

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0.5);

Dies zeichnet das der Instanz zugewiesene Sprite und sein Unterbild mit der linken Ecke an der x-/y-Position der Instanz. Die Breite wird auf die gleiche Breite wie das Sprite gesetzt, und die Höhe ist die Sprite-Höhe geteilt durch zwei. Es wird auch normal überblendet, hat aber einen teilweise transparenten Alpha-Wert.
