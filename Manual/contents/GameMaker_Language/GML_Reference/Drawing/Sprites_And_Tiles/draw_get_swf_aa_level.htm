draw_get_swf_aa_level

Mit dieser Funktion können Sie den Anti-Aliasing (AA)-Level für Vektorsprites im SWF-Format ermitteln. Der Rückgabewert liegt zwischen 0 und 1 und zeigt an, wie "glatt" die Kanten dieser Sprites gezeichnet werden. Sie können den AA-Level mit der Funktion draw_set_swf_aa_level().

Syntax:

draw_get_swf_aa_level();

Rückgabe:

Real

Beispiel:

if draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

Der obige Code überprüft den AA-Wert für Sprites im SWF-Format, und wenn er 0 ist, aktiviert er AA und setzt den Wert auf 0,5.
