kachelplan_zeichnen

Diese Funktion kann verwendet werden, um eine gegebene Tilemap an einer beliebigen Stelle im Raum zu zeichnen. Sie geben die ID des Tilemap-Elements an (die Sie erhalten, wenn Sie ein Tilemap-Element mit layer_tilemap_erzeugen() oder wenn Sie die Funktion layer_tilemap_get_id()), dann geben Sie die x/y-Position zum Zeichnen an (im Raum). Beachten Sie, dass dies die Tilemap einfach am dem angegebenen Punkt gezeichnet wird, wobei die Ebenentiefe der Instanz verwendet wird, die die Funktion aufruft. Die Tilemap wird nicht verschoben - und auch nicht in irgendeiner Weise verändert - und es spielt keine Rolle, ob die Tilemap als sichtbar markiert ist oder nicht.

Syntax:

draw_tilemap(tilemap_element_id, x, y);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tilemap-Elements, um die y-Position der Zelle zu erhalten
x Die x-Position innerhalb des Raums, an der gezeichnet werden soll
y Die y-Position innerhalb des Raums, an der gezeichnet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

Der obige Code ruft die Layer-ID ab und verwendet diese dann, um eine bestimmte Tilemap-ID zu erhalten, die wiederum verwendet wird, um die Tilemap an der Mausposition zu zeichnen.
