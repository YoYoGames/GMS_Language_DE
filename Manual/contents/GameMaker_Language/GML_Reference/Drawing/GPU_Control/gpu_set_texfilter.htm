gpu_set_texfilter

Mit dieser Funktion können Sie die Texturfilterung (lineare Interpolation) aller auf dem Spielbildschirm gezeichneten Bilder einstellen. Wenn dies aktiviert ist(true), werden alle Texturen beim Zeichnen geglättet (dies schließt Sprites ein, da auch sie als Texturen gelten), Das bedeutet, dass es beim Skalieren oder Verschieben, wenn das Pixelverhältnis nicht 1:1 ist, zu einem "Verschmieren" über verschiedene Pixel kommt, wodurch die Bilder je nach verwendetem Zeichenstil unscharf erscheinen können. Wenn dies deaktiviert ist(false), werden Bilder beim Skalieren oder Verschieben auf Basis des nächstgelegenen Pixels gezeichnet, was zu "blockigen" Bildern führen kann. Der Standardwert ist false, und dies kann auch in den globalen Spieloptionen für einzelne Zielplattformen geändert werden.

HINWEIS:Diese Einstellung überschreibt jede Texturstufen-Interpolation, die für Shader mit der Funktion gpu_set_texfilter_ext().

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_set_texfilter(enable);

Argument Beschreibung
aktivieren. Aktivieren oder Deaktivieren der Texturfilterung(true / false)

Rückgabe:

N/A

Beispiel:

if gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
sonst
{
gpu_set_texfilter(true);
}

Der obige Code prüft, ob die Texturfilterung ein- oder ausgeschaltet ist und schaltet sie entsprechend.
