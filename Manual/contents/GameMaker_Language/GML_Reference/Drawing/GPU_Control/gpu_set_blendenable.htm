gpu_set_blendenable

Diese Funktion kann verwendet werden, um das Alpha-Blending ein- und auszuschalten. Wenn Sie diese Funktion auf "false" setzen, werden alle Bilder, die gezeichnet werden, zu 100 % undurchsichtig gezeichnet, was bedeutet, dass alle transparenten oder halbtransparenten Bereiche eines Sprites oder Hintergrunds als einfarbig sichtbar sind. Es ist eine gute Idee, Alpha-Blending auszuschalten, wann immer dies möglich ist (insbesondere bei der Entwicklung für mobile Geräte), da dies die Zeichengeschwindigkeit erheblich erhöht.

Syntax:

gpu_set_blendenable(enable);

Argument Beschreibung
aktivieren. Aktivieren oder deaktivieren Sie den Alpha-Blending-Wert(true oder false).

Rückgabe:

N/A

Beispiel:

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

Der obige Code schaltet das Alpha-Blending aus, um ein Hintergrund-Sprite zu zeichnen, und schaltet es dann wieder ein, um weiter zu zeichnen.
