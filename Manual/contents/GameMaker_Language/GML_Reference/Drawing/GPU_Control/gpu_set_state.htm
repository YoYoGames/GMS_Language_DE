gpu_set_state

Diese Funktion setzt den aktuellen GPU-Status mit Hilfe der übergebenen DS-Map. Die übergebene Map kann mit der Funktion gpu_get_state()erstellt werden .

Syntax:

gpu_set_state(ds_map);

Argument Beschreibung
ds_map Der GPU-Zustand, der als DS-Map eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

gpu_set_state(gpu_map);

Der obige Code setzt den GPU-Zustand anhand der in der Variablen "gpu_map" bereitgestellten Karte.
