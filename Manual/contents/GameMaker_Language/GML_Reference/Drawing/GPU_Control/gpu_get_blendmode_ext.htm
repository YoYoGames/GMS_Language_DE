gpu_get_blendmode_ext

Mit dieser Funktion kann der aktuelle erweiterte Überblendungsmodus abgerufen werden, der zum Zeichnen verwendet wird. Die Funktion gibt ein 2-elementiges 1D-Array mit den folgenden Elementen darin zurück:

  • 0] = Quell-Mischmodus (Standard ist bm_src_alpha)
  • 1] = Ziel-Mischmodus (Standard ist bm_inv_src_alpha)

Die Werte des Arrays sind eine der folgenden Konstanten ("s" steht für einen Wert aus der Quelle, während ein "d" für einen Wert aus dem Ziel steht):

Konstant Überblendfaktor (Rot, Grün, Blau, Alpha)
bm_null (0, 0, 0, 0)
bm_eins (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min(As, 1-Ad)

Beachten Sie, dass Sie diese Werte ändern und das vollständige Array an die gpu_set_blendmode_ext() Funktion als ein einziges Argument übergeben.

Syntax:

gpu_get_blendmode_ext();

Rückgabe:

2 Element 1D Array (Konstanten siehe oben)

Beispiel:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

Der obige Code ruft den aktuellen erweiterten Überblendungsmodus ab, ändert die Quelle und setzt dann den erweiterten Überblendungsmodus erneut.
