gpu_get_zwriteenable

Mit dieser Funktion kann abgefragt werden, ob das Z-Schreiben aktiviert ist (die Funktion gibt true zurück) oder nicht (die Funktion gibt false zurück). Der Standardwert ist, dass das Z-Schreiben aktiviert ist, daher gibt die Funktion true zurück.

Syntax:

gpu_get_zwriteenable();

Rückgabe:

Boolesche

Beispiel:

if gpu_get_zwriteenable() == false
{
gpu_set_zwriteenable(true);
}

Der obige Code prüft, ob das Z-Schreiben aktiviert ist oder nicht, und wenn es deaktiviert ist, wird es dann wieder aktiviert.
