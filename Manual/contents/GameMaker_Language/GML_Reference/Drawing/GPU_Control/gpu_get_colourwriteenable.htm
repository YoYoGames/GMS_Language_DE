gpu_get_colourwriteenable

Mit dieser Funktion können Sie die aktuellen Werte für die Schreibfreigabe der Farben abrufen. Die Funktion gibt ein 4-elementiges 1D-Array mit den folgenden Elementen darin zurück, die entweder true (aktiviert) oder false ( deaktiviert) sein werden. Standardmäßig sind alle Farb Schreiben auf true gesetzt:

  • [0] = Rotkanal aktiviert/deaktiviert
  • [1] = Grüner Kanal aktiviert/deaktiviert
  • [2] = Blauer Kanal aktiviert/deaktiviert
  • [3] = Alphakanal aktiviert/deaktiviert

Syntax:

gpu_get_colorwriteenable();

Rückgabe:

4-Element-1D-Array (Details siehe oben)

Beispiel:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Der obige Code ruft die aktuellen Farbschreibwerte ab und setzt dann die Alphakomponente auf false.
