gpu_set_texrepeat

Mit dieser Funktion können Sie dem GameMaker Studio 2 mitteilen, ob sich Texturen bei der Verwendung für Modelle und Primitive wiederholen(true) oder nicht(false) sollen. Normalerweise liegen Texturbreiten und -höhen im Bereich 0 - 1, aber wenn Sie einen Wert größer als 1 angeben, wird der Rest standardmäßig nicht gezeichnet (die gesamte Textur wird mit einem "leeren" Abschnitt gezeichnet, der dem verwendeten Prozentsatz über 1 entspricht. Ein Wert von 2 für eine Texturbreite oder -höhe würde also die Textur in Hälfte des Raums zeichnen und die andere Hälfte leer lassen). Wenn Sie jedoch diese Funktion verwenden, um repeat auf true zu setzen, wird die Textur automatisch über alle Modelle und Primitive wiederholt. Der wahrscheinliche Anwendungsfall für diese Funktionen ist das Wiederholen einer Textur in 3D, aber Damit es funktioniert und keine Bilder vom Rest der Texturseite abzieht, muss das verwendete Sprite im Sprite-Editor als "Separate Texturseite" markiert werden.

HINWEIS: Sprites und Hintergründe werden immer ohne Wiederholung gezeichnet. Sobald Sie also ein Sprite oder einen Hintergrund zeichnen, wird dieser Wert auf false zurückgesetzt. Dieses Standardverhalten wirkt sich nicht auf die Einstellungen der Texturstufe aus (d. h.: der Aufruf der Funktion wird, aber die interne Einstellung auf false nicht).

HINWEIS: Diese Einstellung überschreibt alle Texturstufen-Wiederholungen, die für Shader gesetzt wurden, wenn Sie die Funktion gpu_set_texrepeat_ext().

Syntax:

gpu_set_texrepeat(enable);

Argument Beschreibung
aktivieren. Aktivieren oder Deaktivieren der Texturfilterung(true / false)

Rückgabe:

N/A

Beispiel:

if gpu_get_texrepeat()
{
gpu_set_texrepeat(false);
}
sonst
{
gpu_set_texrepeat(true);
}

Der obige Code prüft, ob die Texturwiederholung ein- oder ausgeschaltet ist und schaltet sie entsprechend.
