gpu_set_alphatestenable

Mit dieser Funktion können Sie den Alphatest für Ihr Spiel aktivieren oder deaktivieren (standardmäßig ist er deaktiviert). Wenn Sie den Alphatest einschalten, können Sie dann mit der Begleitfunktion gpu_set_alphatestref() den "Cut-off"-Wert einstellen, bei dem alle Alpha-Werte auf 0 gesetzt werden. Wert setzen, bei dem alle Alphawerte auf 0 gesetzt werden. Das Bild unten zeigt den Unterschied, der zu sehen ist, wenn der Alphatest ein- oder ausgeschaltet ist:

HINWEIS: Diese Funktion kann die Leistung auf iOS- und Android-Geräten beeinträchtigen.

Syntax:

gpu_set_alphatestenable(enable);

Argument Beschreibung
aktivieren. Aktivieren oder Deaktivieren des Alpha-Tests(true / false)

Rückgabe:

N/A

Beispiel:

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

Der obige Code prüft, ob der Alphatest aktiviert ist, und wenn nicht, schaltet er den Alphatest ein und setzt den Testschwellenwert auf 128 (nur Pixel mit einem Alpha-Wert über 0,5 werden gezeichnet).
