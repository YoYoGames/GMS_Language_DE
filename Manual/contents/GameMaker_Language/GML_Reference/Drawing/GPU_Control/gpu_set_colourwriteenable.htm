gpu_set_colourwriteenable

Mit dieser Funktion können Sie die Farbkanäle und den Alphakanal für alle weiteren Zeichnungen ein- oder ausschalten. Sie können diese Funktion z. B. verwenden, um reine Alphakanal-Flächen zu erstellen (schalten Sie die RGB-Schrift aus, bevor Sie die Fläche verwenden), oder um spezielle Effekte beim Zeichnen auf dem Bildschirm zu erzeugen. Der Standardwert für jede der Komponenten ist true und kann entweder als vier eindeutige Argumente oder als 4-Element-1D-Array mit den folgenden Elementen darin übergeben werden, die dann entweder true (aktiviert) oder false (deaktiviert):

  • [0] = Rotkanal aktiviert/deaktiviert
  • [1] = Grüner Kanal aktiviert/deaktiviert
  • [2] = Blauer Kanal aktiviert/deaktiviert
  • [3] = Alphakanal aktiviert/deaktiviert

Syntax:

gpu_set_colorwriteenable(rot, grün, blau, alpha);

Argument Beschreibung
rot Aktivieren/Deaktivieren des Rotkanals
grün Aktivieren/Deaktivieren des grünen Kanals
blau Aktivieren/Deaktivieren des blauen Kanals
alpha Aktivieren/Deaktivieren des Alphakanals

Rückgabe:

N/A

Beispiel:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

Der obige Code ruft die aktuellen Farbschreibwerte ab und setzt dann die Alphakomponente auf false.
