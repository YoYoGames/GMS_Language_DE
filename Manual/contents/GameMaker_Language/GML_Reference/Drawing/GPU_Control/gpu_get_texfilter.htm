gpu_get_texfilter

Mit dieser Funktion können Sie prüfen, ob die Texturfilterung (lineare Interpolation) aktiviert ist (gibt true zurück) oder nicht (gibt false zurück). Weitere Informationen zur Texturfilterung finden Sie in der Funktion gpu_set_texfilter().

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_get_texfilter();

Rückgabe:

Boolesche

Beispiel:

if gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
sonst
{
gpu_set_texfilter(true);
}

Der obige Code prüft, ob die Texturfilterung ein- oder ausgeschaltet ist und schaltet sie entsprechend.
