GPU-Steuerung

Wenn Sie Dinge in einem Raum zeichnen, benötigen Sie oft eine genauere Kontrolle darüber, wie das Zeichnen durchgeführt werden soll. Vielleicht möchten Sie, dass kein Alpha-Blending stattfindet, oder Sie benötigen Culling, oder Sie müssen die Texturinterpolation ein- oder ausschalten. ausschalten. All dies ist möglich - und noch viel mehr! - mit den unten aufgeführten GPU-Controller-Funktionen. Beachten Sie, dass die meisten dieser Funktionen global sind und sich nach dem Setzen oder Ändern darauf auswirken, wie alles von da an gezeichnet wird.

Die folgenden Funktionen können verwendet werden, um die Werte verschiedener GPU-Eigenschaften zu erhalten:

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_destalpha
  • gpu_get_colourwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphatestref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

Die folgenden Funktionen können verwendet werden, um die Werte verschiedener GPU-Eigenschaften einzustellen:

  • gpu_set_blendenable
  • gpu_set_ztestenable
  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colourwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphatestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

Bitte beachten Sie, dass es vor der Verwendung der Mischmodus-Funktionen wichtig ist, deren Funktionsweise zu verstehen, damit Sie bei der Anwendung die richtigen Ergebnisse erzielen. Dazu haben wir die folgende Anleitung erstellt:

  • Anleitung zur Verwendung von Blendmodes

Schließlich gibt es noch die folgenden Funktionen, die zur Manipulation des GPU-Stacks verwendet werden können:

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
