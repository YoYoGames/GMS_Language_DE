gpu_get_fog

Mit dieser Funktion können Sie die Nebeleinstellungen abrufen. Die Funktion gibt ein 4-Element-1D-Array mit den folgenden Informationen zurück:

  • 0] = aktivierter Toggle (ein Boolean, entweder true oder false), Voreinstellung false
  • [1] = Farbe (real), Voreinstellung c_black
  • [2] = Startabstand (real), Voreinstellung 0
  • [3] = Endabstand (real), Voreinstellung 1

Beachten Sie, dass Sie diese Werte ändern können und das gesamte Array an die gpu_set_fog() Funktion als ein einziges Argument übergeben.

Syntax:

gpu_get_fog();

Rückgabe:

4 Element 1D Array (Arrayelemente siehe oben)

Beispiel:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Der obige Code ruft die aktuellen Nebeleinstellungen ab und setzt dann das Farbelement des Arrays auf c_red, bevor der Nebel mit dem geänderten Array erneut eingestellt wird.
