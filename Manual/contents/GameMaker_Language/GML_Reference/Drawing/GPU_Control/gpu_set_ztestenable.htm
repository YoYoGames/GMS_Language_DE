gpu_set_ztestenable

Diese Funktion kann verwendet werden, um den Z-Puffer-Test ein- oder auszuschalten, was sich auf die Art und Weise auswirkt, wie Dinge gezeichnet werden (im Allgemeinen nur von Nutzen, wenn Sie mit 3D-Projekten arbeiten). Wenn die Z-Prüfung standardmäßig ausgeschaltet ist und Sie zwei Objekte haben, die in denselben Raum zeichnen, werden beide Objekte gerendert, werden beide Objekte gerendert, unabhängig davon, ob das eine das andere überzeichnet, was zu unnötigen Zeichenaufrufen führt. Wenn Sie dies einschalten, wird der Z-Puffer getestet, um festzustellen, ob ein Objekt "sichtbar" ist, und nicht gezeichnet, wenn es nicht. Beachten Sie, dass dies das Standardverhalten ist, aber Sie können dies ändern, indem Sie die Art des Vergleichs ändern, der für den Z-Puffer-Test verwendet wird (siehe die Funktion gpu_set_zfunc().

Standardmäßig ist der z-Puffer-Test ausgeschaltet(false).

Syntax:

gpu_set_ztestenable(enable);

Argument Beschreibung
aktivieren. Aktivieren oder deaktivieren Sie den Z-Puffer-Test(true oder false).

Rückgabe:

N/A

Beispiel:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Der obige Code schaltet den Z-Puffer-Test ein, um ein Hintergrund-Sprite zu zeichnen, und schaltet ihn dann wieder aus, um mit dem Zeichnen fortzufahren.
