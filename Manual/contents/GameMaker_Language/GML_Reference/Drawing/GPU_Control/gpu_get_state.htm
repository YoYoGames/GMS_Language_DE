gpu_get_state

Diese Funktion ermittelt den aktuellen Zustand der GPU und gibt ihn als DS-Map zurück. Diese kann dann manipuliert oder sogar gespeichert werden, und Sie können diese Map an die GPU zurückgeben, indem Sie die Funktion gpu_set_state().

Syntax:

gpu_get_state();

Rückgabe:

Integer (DS-Karten-ID)

Beispiel:

gpu_map = gpu_get_state();

Der obige Code speichert den aktuellen Zustand der GPU in einer Variablen.
