gpu_get_blendmode_ext_sepalpha

Mit dieser Funktion kann der aktuelle erweiterte Überblendungsmodus, der zum Zeichnen verwendet wird, abgerufen werden, wobei die Überblendungsmodi in zwei Teile - die RGB-Komponente und die Alpha-Komponente - aufgeteilt werden. Die Funktion gibt ein 4-elementiges 1D-Array mit den folgenden Elementen zurück darin:

  • 0] = Quell-Mischmodus abzüglich der Alphakomponente (Standard ist bm_src_alpha (As, As, As))
  • 1] = Ziel-Mischmodus abzüglich der Alpha-Komponente (Standard ist bm_inv_src_alpha (1-As, 1-As, 1-As))
  • 2] = Quell-Alpha-Komponente (Voreinstellung ist bm_src_alpha (As))
  • [3] = Ziel-Alpha-Komponente (Voreinstellung ist bm_inv_src_alpha (1-As))

Die Werte des Arrays sind eine der folgenden Konstanten ("s" steht für einen Wert aus der Quelle, während ein "d" für einen Wert aus dem Ziel steht):

Konstant Überblendfaktor (Rot, Grün, Blau, Alpha)
bm_null (0, 0, 0, 0)
bm_eins (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min(As, 1-Ad)

Wenn Sie die erweiterten Überblendungsmodi auf diese Weise erstellen, werden die RGB-Komponenten von einem Überblendungsmodus und die Alphakomponente von einem anderen übernommen, so dass Sie - wenn Sie z. B. bm_zero (0, 0, 0, 0) als RGB-Überblendungsmodus und bm_one als Alphamodus verwenden - eine zusammengesetzte Überblendung von (0, 0, 0, 1) erhalten.

Beachten Sie, dass Sie diese Werte direkt innerhalb des Arrays ändern und das gesamte Array als einzelnes Argument an die Funktion übergeben können (wie im Beispiel unten gezeigt).

Syntax:

gpu_get_blendmode_ext_sepalpha();

Rückgabe:

4 Element 1D Array (Konstanten siehe oben)

Beispiel:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

Der obige Code erstellt ein lokales Array und ruft den aktuellen erweiterten Mischmodus mit separaten Alphawerten ab. Dieses Array wird dann manipuliert und zurückgegeben, um den Mischmodus einzustellen.
