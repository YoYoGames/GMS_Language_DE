gpu_set_texfilter_ext

Diese Funktion kann verwendet werden, um die lineare Interpolation für einen einzelnen Sampler-"Slot" bei der Verwendung von Shader in GameMaker Studio 2. Wenn dies aktiviert ist(true), wird die Sampler Textur geglättet und wenn dies deaktiviert ist(false), werden die Bilder basierend auf dem nächstgelegenen Pixel gezeichnet. Der Standardwert ist derjenige, der in den globalen Spieloptionen für Ihr Spiel eingestellt ist, oder derjenige, der mit der Funktion gpu_set_texfilter().

HINWEIS: Diese Einstellung wird durch den Wert überschrieben, der beim Aufruf der Funktion gpu_set_texfilter().

HINWEIS: Auf dem HTML5-Ziel funktioniert diese Funktion nur, wenn WebGL aktiviert ist.

Syntax:

gpu_set_texfilter_ext(sampler_id, enable);

Argument Beschreibung
sampler_id Die Sampler-ID aus dem Shader.
aktivieren. Aktivieren oder Deaktivieren der Texturfilterung(true / false)

Rückgabe:

N/A

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
sonst
{
gpu_set_texfilter_ext(s_tex, true);
}

Der obige Code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (gespeichert in einer lokalen Variablen) ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
