gpu_set_fog

Mit dieser Funktion können Sie das Zeichnen von Nebel aktivieren oder deaktivieren. Nebel kann in 3D-Spielen verwendet werden, um Instanzen in der Ferne unscharf aussehen zu lassen oder sogar verschwinden zu lassen, was dabei hilft, Atmosphäre zu schaffen sowie die Tatsache zu maskieren, dass Sie Instanzen nicht zeichnen die weit entfernt sind. Sie stellen ein, ob er aktiviert(true) oder deaktiviert(false) ist, die Farbe, die der Nebel zum Überblenden verwenden soll, sowie die Start- und End-Zeichendistanz. Beachten Sie, dass der Start des Nebels den Radius relativ zur Ansichtskamera angibt, wo der Nebel beginnt, und das Nebelende gibt an, bei welchem Radius (ebenfalls relativ) die Instanzen vollständig unsichtbar werden.

Die Funktion kann vier einzelne Argumente (wie oben angegeben) oder ein Array mit der folgenden Struktur annehmen (der Beispielcode unten zeigt diese Methode):

  • 0] = aktivierter Toggle (ein Boolean, entweder true oder false), Voreinstellung false
  • [1] = Farbe (real), Voreinstellung c_black
  • [2] = Startabstand (real), Voreinstellung 0
  • [3] = Endabstand (real), Voreinstellung 1

Syntax:

gpu_set_fog(enable, color, start, end);

Argument Beschreibung
aktivieren. Aktivieren oder Deaktivieren des Nebels
Farbe Die Farbe des Nebels
Start Der Abstand zum Beginn der Nebelaufbringung von
Ende Der Abstand zum Ende des Nebels

Rückgabe:

N/A

Beispiel:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Der obige Code ruft die aktuellen Nebeleinstellungen ab und setzt dann das Farbelement des Arrays auf c_red, bevor der Nebel mit dem geänderten Array erneut eingestellt wird.
