gpu_set_texrepeat_ext

Mit dieser Funktion kann eingestellt werden, ob ein einzelner Sampler-"Slot" die angegebene Textur wiederholt, wenn man Sampler in GameMaker Studio 2. Die Einstellung true wiederholt die Textur wiederholt, wenn die UV-Koordinaten außerhalb des Bereichs 0-1 liegen, während die Einstellung false keine Wiederholung bedeutet. Der wahrscheinliche Anwendungsfall für diese Funktionen ist das Wiederholen einer Textur in 3D, aber damit es funktioniert und nicht Bilder vom Rest der Texturseite der Texturseite zu ziehen, muss das verwendete Sprite im Sprite-Editor als "Separate Texture Page" markiert werden.

HINWEIS: Diese Einstellung wird durch den Wert überschrieben, der beim Aufruf der Funktion gpu_set_texrepeat().

Syntax:

gpu_set_texrepeat_ext(sampler_id, enable);

Argument Beschreibung
sampler_id Die Sampler-ID aus dem Shader.
aktivieren. Aktivieren oder Deaktivieren der Texturfilterung(true / false)

Rückgabe:

N/A

Beispiel:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
sonst
{
gpu_set_texfilter_ext(s_tex, true);
}

Der obige Code prüft, ob die Texturfilterung für eine bestimmte Sampler-ID (gespeichert in einer lokalen Variablen) ein- oder ausgeschaltet ist und schaltet sie entsprechend um.
