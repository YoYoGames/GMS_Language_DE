gpu_get_alphatestenable

Mit dieser Funktion können Sie prüfen, ob Alpha-Testing aktiviert ist (gibt true zurück) oder nicht (gibt false zurück). Weitere Informationen zum Alphatest finden Sie in der Funktion gpu_set_alphatestref().

Syntax:

gpu_get_alphatestenable();

Rückgabe:

Boolesche

Beispiel:

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

Der obige Code prüft, ob der Alphatest aktiviert ist, und wenn nicht, schaltet er den Alphatest ein und setzt den Testschwellenwert auf 128 (nur Pixel mit einem Alpha-Wert über 0,5 werden gezeichnet).
