gpu_get_zfunc

Mit dieser Funktion können Sie den z-Vergleichsmodus abrufen. Der zurückgegebene Wert ist eine der folgenden Konstanten (der Standardwert ist cmpfunc_lessequal):

  • cmpfunc_never
  • cmpfunc_less
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_greater
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

Syntax:

gpu_get_zfunc();

Rückgabe:

Konstant (siehe Tabelle oben)

Beispiel:

if gpu_get_zfunc() != cmpfunc_greater
{
gpu_set_zfunc(cmpfunc_greater);
}

Der obige Code prüft, ob die z-Testmethode auf cmpfunc_greater gesetzt ist, und wenn nicht, wird sie auf diese Konstante gesetzt.
