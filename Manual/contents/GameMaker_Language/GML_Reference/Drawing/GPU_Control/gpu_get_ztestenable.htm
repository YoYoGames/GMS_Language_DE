gpu_get_ztestenable

Mit dieser Funktion kann abgerufen werden, ob die Z-Prüfung aktiviert ist (die Funktion gibt true zurück) oder nicht (die Funktion gibt false zurück). Der Standardwert ist, dass die Z-Prüfung deaktiviert ist, daher gibt die Funktion false zurück.

Syntax:

gpu_get_ztestenable();

Rückgabe:

Boolesche

Beispiel:

if gpu_get_ztestenable() == false
{
gpu_set_ztestenable(true);
}

Der obige Code prüft, ob die z-Prüfung aktiviert ist oder nicht, und wenn sie deaktiviert ist, wird sie anschließend wieder aktiviert.
