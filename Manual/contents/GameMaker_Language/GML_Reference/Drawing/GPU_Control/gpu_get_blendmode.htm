gpu_get_blendmode

Mit dieser Funktion können Sie den aktuellen Überblendungsmodus abrufen, der zum Zeichnen verwendet wird. Der zurückgegebene Wert ist eine der folgenden Konstanten (der Standardwert ist bm_normal):

Konstant Beschreibung Erweiterter Blend-Modus
bm_normal Normales Überblenden (der Standardüberblendungsmodus). (bm_src_alpha, bm_inv_src_alpha)
bm_add Additives Überblenden. Leuchtkraftwerte von hellen Bereichen werden addiert. (bm_src_alpha, bm_one)
bm_subtrahieren Subtraktives Überblenden. Helligkeitswerte von hellen Bereichen werden subtrahiert. (bm_zero, bm_inv_src_colour)
bm_max Max Blending. Ähnlich wie beim additiven Blending. (bm_src_alpha, bm_inv_src_colour)

Syntax:

gpu_get_blendmode();

Rückgabe:

Konstante (siehe oben für Konstanten)

Beispiel:

if gpu_get_blendmode() != bm_normal
{
gpu_set_blendmode(bm_normal);
}

Der obige Code ruft den aktuellen Mischmodus ab und wenn er nicht bm_normal ist, wird er auf diese Konstante gesetzt.
