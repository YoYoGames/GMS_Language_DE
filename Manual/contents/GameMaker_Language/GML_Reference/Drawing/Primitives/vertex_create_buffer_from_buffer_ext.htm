vertex_create_buffer_from_buffer_ext

Wie bei der Funktion vertex_create_buffer()erzeugt diese Funktion einen neuen Scheitelpunkt-Puffer, nur dass jetzt die darin gespeicherten Scheitelpunkt-Daten aus dem regulären Puffer kopiert werden, der als Quelle angegeben ist. Der Puffer ist mit dem Vertex-Format für die Erstellung von Primitiven zur Verwendung mit (z. B.) Shadern vorformatiert, und Sie können auch einen Offset innerhalb des Quellpuffers angeben, von dem kopiert werden soll, sowie die Anzahl der Scheitelpunkte, die der endgültige Puffer haben soll. Beachten Sie, dass wenn die Anzahl der Scheitelpunkte nicht mit denen übereinstimmt, die kopiert werden, können Sie beschädigte Scheitelpunktdaten erhalten.

Syntax:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Argument Beschreibung
Puffer Der Puffer, aus dem der Vertex-Puffer erstellt wird.
Format Das zu verwendende primitive Scheitelpunktformat.
src_offset Der Offset innerhalb des Quellpuffers, von dem aus kopiert werden soll.
vert_num Die Anzahl der Scheitelpunkte, die der Puffer haben soll.

Rückgabe:

Zeiger

Beispiel:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

Der obige Code erstellt ein neues Scheitelpunktformat und erzeugt dann einen neuen Scheitelpunktpuffer aus einem zuvor erstellten regulären Puffer, wobei das benutzerdefinierte Scheitelpunktformat mit 0 Offset darauf angewendet wird. Die Funktion teilt dem neuen Vertex-Puffer mit, dass er 512 Scheitelpunkte erwarten soll.
