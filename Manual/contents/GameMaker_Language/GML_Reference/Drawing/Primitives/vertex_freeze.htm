vertex_freeze

Mit dieser Funktion kann ein Vertex-Puffer "eingefroren" werden. Dieser Puffer wird schreibgeschützt, d. h., wenn Sie ihn ändern möchten, müssen Sie den gesamten Puffer löschen und neu erstellen. Ein eingefrorener Puffer kann an den Shaderübergeben werden schneller an den Shader übergeben werden als ein normaler, dynamischer Puffer und wird für solche Effekte empfohlen, die ein unveränderliches benutzerdefiniertes Primitiv für die Dauer eines Levels oder des Spiels benötigen.

Syntax:

vertex_freeze(buffer);

Argument Beschreibung
Puffer Der Puffer zum Einfrieren.

Rückgabe:

N/A

Beispiel:

vertex_freeze(buff);

Der obige Code friert den angegebenen Vertex-Puffer ein.
