vertex_position

Diese Funktion fügt 2D-Positionsdaten zum Scheitelpunkt hinzu, der gerade für das benutzerdefinierte Primitiv definiert wird. Sie geben den Puffer an, in den die Daten geschrieben werden sollen, sowie die x- und y-Position zum Zeichnen.

Syntax:

vertex_position(buffer, x, y);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
x Die X-Position.
y Die y-Position.

Rückgabe:

N/A

Beispiel:

vertex_position(buff, x - 100, y - 125);

Der obige Code setzt die Position des zu definierenden Scheitelpunkts.
