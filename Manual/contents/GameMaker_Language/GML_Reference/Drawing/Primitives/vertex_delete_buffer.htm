vertex_delete_buffer

Mit dieser Funktion können Sie einen zuvor angelegten Scheitelpunktpuffer (siehe vertex_create_buffer()) aus dem Systemspeicher zu entfernen. Dabei werden auch alle Scheitelpunktdaten für alle benutzerdefinierten Primitive, die er enthielt, entfernt.

Syntax:

vertex_delete_buffer(buffer);

Argument Beschreibung
Puffer Der zu entfernende Puffer.

Rückgabe:

N/A

Beispiel:

vertex_delete_buffer(v_buff);

Der obige Code löscht den in der Variablen "v_buff" indizierten Puffer aus dem Speicher.
