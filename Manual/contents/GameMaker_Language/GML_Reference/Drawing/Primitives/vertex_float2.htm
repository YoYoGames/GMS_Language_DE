vertex_float2

Diese Funktion fügt zwei Fließkommawerte zu den Scheitelpunktdaten hinzu. Der Scheitelpunkt muss korrekt formatiert sein, um dies mit der vertex_format_add_custom() Funktion.

Syntax:

vertex_float2(buffer, float, float);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
Schwimmer Der erste Eingangswert.
Schwimmer Der zweite Eingangswert.

Rückgabe:

N/A

Beispiel:

vertex_float2(buff, 0.05, 0.01);

Der obige Code fügt zwei Fließkommawerte zu den definierten Scheitelpunktdaten hinzu.
