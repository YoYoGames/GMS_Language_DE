draw_primitive_begin_texture

Diese Funktion muss aufgerufen werden, bevor Sie die Scheitelpunkte eines texturierten Primitivs definieren. Sie müssen die Art des zu verwendenden Primitivs angeben (siehe draw_primitive_begin() für weitere Informationen) und die ID einer Textur angeben, bei der es sich um ein Sprite- oder Hintergrundbild-Asset handeln kann. Diese Asset-ID können Sie mit den Funktionen sprite_get_texture()erhalten, zum Beispiel (verwenden Sie -1 für keine Textur).

HINWEIS: Damit eine Textur wiederholt werden kann, muss sie eine Zweierpotenz in der Größe sein, d. h.: 32x32, 128x128, usw.

Syntax:

draw_primitive_begin_texture(kind, tex)

Argument Beschreibung
freundlich Die Art des Primitivs, das Sie zeichnen werden.
tex Die Textur, die mit dem Primitiv verwendet werden soll.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Der obige Code zeichnet einen Dreiecksstreifen mit 4 Eckpunkten (wodurch ein Rechteck entsteht), der mit der in der Variablen "tex" gespeicherten Textur texturiert ist, und die gesamte Textur wird verwendet, um das fertige Primitiv abzudecken.
