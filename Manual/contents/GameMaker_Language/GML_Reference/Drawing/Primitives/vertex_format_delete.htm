vertex_format_delete

Diese Funktion muss immer dann aufgerufen werden, wenn Sie die Verwendung der erstellten Vektorformate beendet haben. Sie geben den Format-ID-Wert (wie von der Funktion vector_format_end()), und diese Funktion gibt den Speicher, der damit verbunden ist. Beachten Sie, dass Sie einen Fehler erhalten, wenn Sie versuchen, dieses Format nach dem Aufruf dieser Funktion erneut zu verwenden.

Syntax:

vertex_format_delete(formatID);

Rückgabe:

N/A

Beispiel:

vertex_format_delete(myFormat);

Der obige Code entfernt das in der Variablen "myFormat" angelegte Vertex-Format aus dem Speicher.
