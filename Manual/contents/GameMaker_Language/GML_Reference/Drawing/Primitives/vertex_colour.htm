vertex_colour

Wenn Ihr definiertes Scheitelpunktformat einen Farbwert annimmt, können Sie diese Funktion verwenden, um diese Daten zum Scheitelpunkt hinzuzufügen, der für das aktuelle Primitiv definiert ist. Die Funktion benötigt einen Puffer zum Speichern der Daten und nimmt entweder eine Farbkonstante, oder einen Hex-Wert (unter Verwendung des GameMaker Studio 2-Standardformats BGR, z. B.: $FF0000 für Blau) sowie einen Alpha-Wert von 0 (transparent) bis 1 (völlig undurchsichtig).

Syntax:

vertex_colour(buffer, color, alpha);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
Farbe Die Farbe für diesen Scheitelpunkt (kann eine Konstante oder ein Hex-Wert sein).
alpha Der Alphawert für den Scheitelpunkt (von 0 bis 1).

Rückgabe:

N/A

Beispiel:

vertex_colour(b, c_white, 1);

Der obige Code setzt den zu definierenden Scheitelpunkt auf Weiß mit einem Alphawert von 1.
