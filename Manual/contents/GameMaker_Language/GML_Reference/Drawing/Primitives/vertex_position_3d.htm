scheitel_lage_3d

Diese Funktion fügt dem Scheitelpunkt, der gerade für das benutzerdefinierte Primitiv definiert wird, 3D-Positionsdaten hinzu. Sie geben den Puffer an, in den die Daten geschrieben werden sollen, sowie die x-, y- und z-Position zum Zeichnen.

Syntax:

vertex_position_3d(buffer, x, y, z);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
x Die X-Position.
y Die y-Position.
z Die z-Position.

Rückgabe:

N/A

Beispiel:

vertex_position_3d(buff, x - 100, y - 125, 0);

Der obige Code setzt die Position des zu definierenden Scheitelpunkts.
