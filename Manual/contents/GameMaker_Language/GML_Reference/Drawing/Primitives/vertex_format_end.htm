vertex_format_end

Diese Funktion muss nach der Definition eines neuen Scheitelpunktformats aufgerufen werden. Sie gibt das neue Format "Handle" (Index) zurück, das in allen weiteren Scheitelpunktfunktionen verwendet werden muss, die sich auf dieses neue Format beziehen.

Syntax:

vertex_format_end();

Rückgabe:

Real

Beispiel:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
mein_format = vertex_format_end();

Der obige Code erzeugt ein neues Scheitelpunktformat mit nur Farb- und Positionswerten und speichert dann die Format-ID in der Variablen "my_format".
