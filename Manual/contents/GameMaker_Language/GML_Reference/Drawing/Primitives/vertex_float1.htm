vertex_float1

Diese Funktion fügt einen Fließkommawert zu den Vertexdaten hinzu. Der Scheitelpunkt muss korrekt formatiert worden sein, um dies mit der vertex_format_add_custom() Funktion.

Syntax:

vertex_float1(buffer, float);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
Schwimmer Der Eingangswert.

Rückgabe:

N/A

Beispiel:

vertex_float1(buff, 0.05);

Der obige Code fügt einen Fließkommawert zu den definierten Scheitelpunktdaten hinzu.
