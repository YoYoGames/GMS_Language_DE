vertex_create_buffer

Mit dieser Funktion können Sie einen neuen Vertex-Puffer erstellen. Dabei handelt es sich um einen speziellen, von GameMaker Studio 2 erzeugten Wachstumspuffer, der für die Verwendung bei der Erstellung von Primitiven (z. B. für die Verwendung mit Shadern) vorformatiert ist. Die Funktion gibt einen "Pointer" (Index) für den Puffer zurück, der dann bei allen weiteren Aufrufen verwendet werden sollte.

Wenn Sie einen mit dieser Funktion erstellten Scheitelpunktpuffer verwenden, rufen Sie einfach vertex_begin() auf, um damit zu beginnen, ihm Scheitelpunktdaten zuzuweisen, um mit der Definition Ihres benutzerdefinierten Primitivs zu beginnen, das dann im Puffer bereitgehalten wird für Übergabe an den Shader. Der Puffer kann bei Bedarf wiederverwendet werden (es sei denn, Sie haben die vertex_freeze() Funktion), wobei jeder Aufruf von vertex_begin() löscht die vorherigen Pufferdaten bereithält, um die neuen Daten aufzunehmen.

Syntax:

vertex_create_buffer();

Rückgabe:

Zeiger

Beispiel:

v_buff = vertex_create_buffer();

Der obige Code erzeugt einen neuen Scheitelpunktpuffer und speichert dessen Handle in der Variablen "v_buff".
