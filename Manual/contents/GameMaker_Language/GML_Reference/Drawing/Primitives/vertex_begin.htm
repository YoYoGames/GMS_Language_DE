vertex_begin

Mit dieser Funktion beginnen Sie die Definition eines benutzerdefinierten Primitivs. Sie weisen einen Puffer zu, in den das Primitiv geschrieben wird, und das zu verwendende Scheitelpunktformat (das zuvor mit den Scheitelpunktformat-Funktionen definiert wurde). Anschließend definieren Sie die erforderlichen Punkte für jeden Scheitelpunkt des Primitivs, bevor Sie vertex_end() aufrufen, um die Erstellung des Primitivs abzuschließen.

Syntax:

vertex_begin(buffer, format);

Argument Beschreibung
Puffer Der Puffer, in den geschrieben werden soll.
Format Das zu verwendende Scheitelpunktformat.

Rückgabe:

N/A

Beispiel:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

Der obige Code definiert ein neues Scheitelpunktformat, erstellt einen neuen Puffer und startet den Definitionsprozess eines neuen Primitivs.
