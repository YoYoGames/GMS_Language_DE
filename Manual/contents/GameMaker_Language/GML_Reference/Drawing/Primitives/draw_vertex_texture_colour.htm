draw_vertex_texture_colour

Diese Funktion definiert die Position eines texturierten Scheitelpunktes für ein Primitiv. Das endgültige Aussehen des Primitivs hängt von dem zum Zeichnen gewählten Primitivtyp ab (siehe draw_primitive_begin() für weitere Informationen), der Reihenfolge, in der Sie die Scheitelpunkte hinzufügen, der Position der Start- und Endpunkte, die Sie für das Texturmuster angeben, und den Farb- und Alphawerten, die Sie eingestellt haben. Um das Aussehen der Textur beizubehalten und nur den Alphawert zu ändern, kann ein Wert von -1 (oder c_white) für das Farbargument verwendet werden. Zum Beenden und Zeichnen des Primitivs müssen Sie draw_primitive_end().

HINWEIS: Damit eine Textur wiederholt werden kann, muss sie eine Zweierpotenz in der Größe sein, d. h.: 32x32, 128x128, usw.

Syntax:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

Argument Beschreibung
x Die x-Koordinate des Scheitelpunkts.
y Die y-Koordinate des Scheitelpunkts.
xtex Die x-Koordinate innerhalb der Textur.
ytex Die y-Koordinate innerhalb der Textur.
col Die Farbe, die mit der Textur an diesem Scheitelpunkt überblendet werden soll (-1 oder c_white für keine Überblendung).
alpha Das Alpha, mit dem dieser Scheitelpunkt gezeichnet werden soll (0-1).

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

Der obige Code zeichnet einen Dreiecksstreifen mit 4 Eckpunkten (wodurch ein Rechteck entsteht), der mit der in der Variablen "tex" gespeicherten Textur texturiert ist, und die gesamte Textur wird verwendet, um das fertige Primitiv abzudecken, und es wird mit vier verschiedenen Farben gemischt.
