zeichnen_vertex

Diese Funktion definiert die Position eines Scheitelpunktes für ein Primitiv. Das endgültige Aussehen des Primitivs hängt von dem zum Zeichnen gewählten Primitiv-Typ und der Reihenfolge ab, in der Sie die Scheitelpunkte dazu hinzufügen. Siehe . draw_primitive_begin() für weitere Informationen. Um das Primitiv zu beenden und zu zeichnen, müssen Sie draw_primitive_end().

Syntax:

draw_vertex(x, y)

Argument Beschreibung
x Die x-Koordinate des Scheitelpunkts.
y Die y-Koordinate des Scheitelpunkts.

Rückgabe:

N/A

Beispiel:

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150);
draw_primitive_end();

Der obige Code zeichnet ein einfaches Dreiecksprimitiv.
