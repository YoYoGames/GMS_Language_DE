draw_primitive_begin

Diese Funktion muss aufgerufen werden, bevor Sie irgendwelche Primitive definieren können. Es gibt 6 Arten von Primitiven, die Sie mit den folgenden Konstanten definieren können:

  • pr_punktliste
  • pr_linelist
  • pr_linestrip
  • pr_dreiecksliste
  • pr_trianglestrip
  • pr_trianglefan

Das folgende Bild veranschaulicht, wie diese grundsätzlich aussehen sollten und auch die Reihenfolge, in der Sie die Eckpunkte definieren sollten:

Bitte beachten Sie, dass auf einigen Plattformen(Windows, UWP, XBox) der Typ pr_trianglefan nicht nativ nicht nativ unterstützt wird und GameMaker Studio 2 daher eine Konvertierung vornimmt, wenn das Spiel kompiliert wird, damit es funktioniert. Das bedeutet, dass auf diesen Plattformen der pr_trianglefan-Typ viel langsamer ist als die anderen. Beachten Sie außerdem, dass Sie zur Verwendung dieser Funktion auf HTML5 zu verwenden, müssen Sie WebGL in den Spieloptionen aktivieren.

Syntax:

draw_primitive_begin(kind)

Argument Beschreibung
freundlich Die Art des Primitivs, das Sie zeichnen werden.

Rückgabe:

N/A

Beispiel:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

Der obige Code zeichnet drei Viertel eines Kreises, der aus Primitiven besteht.
