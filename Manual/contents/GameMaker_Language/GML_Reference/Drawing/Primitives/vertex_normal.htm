scheitel_normal

Diese Funktion fügt dem Scheitelpunkt, der gerade für das benutzerdefinierte Primitiv definiert wird, Oberflächennormaldaten hinzu. Sie geben den Puffer an, in den die Daten geschrieben werden sollen, sowie die x-, y- und z-Komponententeile der Normalen.

Syntax:

vertex_normal(buffer, nx, ny, nz);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
nx Die x-Komponente der Oberflächennormale.
ny Die y-Komponente der Oberflächennormale.
nz Die z-Komponente der Oberflächennormale.

Rückgabe:

N/A

Beispiel:

vertex_normal(buff, 0, 1, 1);

Der obige Code setzt die Oberflächennormale des zu definierenden Scheitelpunkts.
