vertex_erzeugen_puffer_ext

Wie bei der Funktion vertex_create_buffer()wird mit dieser Funktion ein neuer Vertex-Puffer erzeugt. Dies ist ein spezieller, von GameMaker Studio 2 erstellter, wachsender Puffer, der vorformatiert ist für die Verwendung beim Erstellung von Primitiven für die Verwendung mit Shadern vorformatiert ist. Sie können eine anfängliche Startgröße für den Puffer angeben (in Bytes) und es wird ein "Handle" (Index) für den Puffer zurückgegeben, der dann bei allen weiteren Aufrufen des Puffers verwendet werden sollte.

Syntax:

vertex_create_buffer_ext(size);

Argument Beschreibung
Größe Die anfängliche Größe des Puffers (in Bytes).

Rückgabe:

Zeiger

Beispiel:

v_buff = vertex_create_buffer_ext(1024 * 1024);

Der obige Code erzeugt einen neuen Vertex-Puffer, der zunächst 1 MB groß ist, und speichert dessen Handle in der Variablen "v_buff".
