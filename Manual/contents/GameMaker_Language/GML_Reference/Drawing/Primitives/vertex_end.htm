vertex_end

Mit dieser Funktion beenden Sie den Aufbau des benutzerdefinierten Primitivs. Sobald Sie diesen Befehl aufrufen, kann das Primitiv in der vertex_submit() Funktion zur Verwendung in einem Shader verwendet werden oder Sie können den Puffer einfrieren (wodurch der Vertex-Puffer schreibgeschützt und viel schneller verwendet) mit vertex_freeze().

Syntax:

vertex_end(buffer);

Argument Beschreibung
Puffer Der Puffer, in dem das Schreiben beendet werden soll.

Rückgabe:

N/A

Beispiel:

vertex_end(buff);

Mit dem obigen Code wird die Definition eines benutzerdefinierten Primitivs beendet.
