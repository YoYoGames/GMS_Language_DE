vertex_get_number

Mit dieser Funktion können Sie die Anzahl der einzelnen Scheitelpunkte herausfinden, die in einem beliebigen Scheitelpunktpuffer definiert sind.

Syntax:

vertex_get_number(buffer);

Argument Beschreibung
Puffer Der zu prüfende Vertex-Puffer.

Rückgabe:

Real

Beispiel:

v_num = vertex_get_number(v_buffer);

Der obige Code speichert die Anzahl der Scheitelpunkte, die im angegebenen Scheitelpunktpuffer gespeichert sind, in einer Variablen.
