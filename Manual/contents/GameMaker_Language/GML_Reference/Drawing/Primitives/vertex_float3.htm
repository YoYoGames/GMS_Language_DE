vertex_float3

Diese Funktion fügt drei Fließkommawerte zu den Scheitelpunktdaten hinzu. Der Scheitelpunkt muss korrekt formatiert sein, um dies mit der vertex_format_add_custom() Funktion.

Syntax:

vertex_float3(buffer, float, float, float);

Argument Beschreibung
Puffer Der Puffer, in den die Informationen geschrieben werden.
Schwimmer Der erste Eingangswert.
Schwimmer Der zweite Eingangswert.
Schwimmer Der dritte Eingangswert.

Rückgabe:

N/A

Beispiel:

vertex_float3(buff, 0.05, 0.01, room_width / x);

Der obige Code fügt drei Fließkommawerte zu den definierten Scheitelpunktdaten hinzu.
