teil_system_zerstören

Mit dieser Funktion können Sie ein gegebenes Partikelsystem zerstören und aus dem Speicher entfernen. Diese Funktion sollte immer dann aufgerufen werden, wenn das System nicht mehr benötigt wird, z. B. am Ende eines Raums oder im Zerstörungsereignis einer Instanz, da sonst sonst könnte es passieren, dass Partikel in späteren Räumen auftauchen und nicht mehr entfernt werden können, und dass ein Speicherleck entsteht, das schließlich zum Absturz des Spiels führt.

HINWEIS: Diese Funktion zerstört und entfernt auch alle Emitter, die möglicherweise erstellt wurden und mit dem zu zerstörenden System verbunden sind.

Syntax:

part_system_destroy(ind);

Argument Beschreibung
ind Der Index des zu zerstörenden Partikelsystems.

Rückgabe:

N/A

Beispiel:

if (global.player_hp <= 0)
{
part_system_destroy(p_sys);
room_goto_next();
}

Der obige Code prüft, ob der Wert einer globalen Variablen kleiner oder gleich Null ist. Wenn dies der Fall ist, wird das Partikelsystem, auf das in der angegebenen Variablen verwiesen wird, zerstört und dann zum nächsten Raum gewechselt.
