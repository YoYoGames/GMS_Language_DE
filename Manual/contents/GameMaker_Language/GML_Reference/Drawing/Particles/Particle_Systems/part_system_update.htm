teil_system_aktualisierung

Diese Funktion rückt das angegebene Partikelsystem um einen Schritt in der Spielzeit vor und kann verwendet werden, um das Verhalten des Partikelsystems zu simulieren, wenn die automatische Aktualisierung ausgeschaltet ist, oder sie kann verwendet werden, um ein Partikelsystem zu einem bestimmten Punkt in einem Schritt vorzurücken. Wenn Sie z. B. ein Partikelsystem für Schnee haben und möchten, dass es so aussieht, als würde der Schnee gleich in dem Moment fallen, in dem der Spieler den Raum betritt, dann möchten Sie vielleicht Ihre Schneepartikel im Erstellungsereignis einer Instanz um einige Sekunden vorziehen so dass beim ersten Zeichnen die Partikel über den ganzen Bildschirm verteilt sind (das Codebeispiel unten zeigt dies). Sie können diese Funktion auch verwenden, wenn das automatische Aktualisieren des Systems durch die Funktion part_system_automatic_update() um das System weiter zu bewegen.

Syntax:

part_system_update(ind);

Argument Beschreibung
ind Der Index des zu aktualisierenden Partikelsystems.

Rückgabe:

N/A

Beispiel:

repeat (room_speed * 3)
{
part_system_update(global.RainSys);
}

Der obige Code schaltet das in der globalen Variable "RainSys" indizierte Partikelsystem um 3 Sekunden weiter.
