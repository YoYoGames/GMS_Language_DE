teil_system_automatisch_ziehen

Mit dieser Funktion kann das Zeichnen eines Partikelsystems ausgeschaltet werden, so dass alle Aktualisierungen, die am System vorgenommen werden (automatisch oder anderweitig), nicht gesehen werden. Dies ist eine rein visuelle Option, und wenn sie auf "false" gesetzt ist, werden Sie nicht in der Lage sein, die Partikel nicht sehen, da sie nicht gezeichnet werden, aber sie sind immer noch vorhanden und ändern ihre Position, Farbe etc... wie sie es normalerweise tun würden. Wenn das automatische Zeichnen ausgeschaltet ist, können Sie GameMaker Studio 2 explizit anweisen, den aktuellen Zustand des Partikelsystems zu zeichnen Systems zu zeichnen, indem Sie die Funktion part_system_drawit()setzen, und wenn Sie diese Funktion wieder auf true setzen, können Sie das automatische Zeichnen wieder einschalten.

Eine Sache, die Sie beachten sollten, ist, dass Sie, wenn Sie die einfachen Effekte verwenden, die mit den Funktionen effect_create_above() oder effect_create_below() erzeugen, dann können Sie mit die Werte 0 (für Effekte unterhalb) oder 1 (für Effekte oberhalb) als Partikelsystem-Index verwenden und so das automatische Zeichnen auch für diese umschalten (dies funktioniert auch, um das Zeichnen für die DnD™-Partikel-Effekteumzuschalten).

Syntax:

part_system_automatic_draw(ind, automatic);

Argument Beschreibung
ind Der Index des zu ändernden Partikelsystems.
automatisch Ob das automatische Zeichnen eingeschaltet ist (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

part_system_automatic_draw(global.Sname, false);

Der obige Code schaltet das automatische Zeichnen für das in der globalen Variable "Sname" indizierte Partikelsystem aus.
