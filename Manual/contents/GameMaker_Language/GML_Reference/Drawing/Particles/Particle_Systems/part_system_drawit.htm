teil_system_aktualisierung

Mit dieser Funktion können Sie ein Partikelsystem dazu zwingen, alle aktuellen Partikel auf den Bildschirm zu zeichnen. Wenn part_system_automatic_draw() ausgeschaltet ist, zeigt diese Funktion die Partikel, wenn im Zeichen-Ereignis einer Instanz verwendet wird, oder sie kann verwendet werden, wenn das Zeichen-Ziel auf eine Oberfläche gesetzt ist (mit surface_set_target()), um die Partikel im System auf diese Fläche zu zeichnen.

Syntax:

part_system_drawit(ind);

Argument Beschreibung
ind Der Index des Partikelsystems, von dem die Partikel gezeichnet werden sollen.

Rückgabe:

N/A

Beispiel:

if surface_exists(surf)
{
surface_set_target(surf);
part_system_drawit(global.psys);
surface_reset_target();
}

Der obige Code prüft, ob die in der Variablen "surf" indizierte Oberfläche existiert, und wenn dies der Fall ist, setzt er das Zeichenziel auf die Oberfläche, zeichnet das Partikelsystem mit der in der globalen Variablen gespeicherten ID und setzt dann das Zeichenziel zurück Ziel zurück, so dass alle normalen Zeichnungen wieder auf dem Bildschirm erscheinen.
