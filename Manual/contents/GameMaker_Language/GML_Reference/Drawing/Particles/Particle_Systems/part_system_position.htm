teil_system_position

Mit dieser Funktion können Sie die Basisposition für das Partikelsystem relativ zur (0,0)-Position des Raumes setzen, d. h. alle weiteren Partikelfunktionen, die sich auf dieses System beziehen, werden nun relativ zur neuen Position gezeichnet. Standardmäßig ist diese Standardmäßig ist diese Position immer (0,0), aber unter besonderen Umständen können Sie diese Position auch auf etwas anderes ändern.

HINWEIS: Diese Funktion ändert alles innerhalb des Partikelsystems. Wenn Sie also einen Emitter an der Position (100,100) haben und dann die Position des Partikelsystems auf (0,100) setzen, wird der Emitter jetzt an (100,200) gezeichnet. Das Gleiche gilt, wenn Sie Das Gleiche gilt, wenn Sie das System verschieben und dann den Emitter erstellen, da er, obwohl Sie ihn an (100,100) erstellen, an (100,200) gezeichnet wird.

Syntax:

part_system_position(ind, x, y);

Argument Beschreibung
ind Der Index des zu ändernden Partikelsystems.
x Die neue x-Koordinate des Partikelsystems.
y Die neue y-Koordinate des Partikelsystems.

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
part_system_position(global.Sname, mouse_x, mouse_y);
}

Der obige Code prüft auf das Drücken der Maustaste und wenn er eine erkennt, wird das in der globalen Variable "Sname" indizierte Partikelsystem an die x/y-Position der Maus verschoben
