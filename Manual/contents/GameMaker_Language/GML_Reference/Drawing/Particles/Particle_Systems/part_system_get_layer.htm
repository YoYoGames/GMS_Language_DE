part_system_get_layer

Diese Funktion kann verwendet werden, um den eindeutigen Layer-ID-Wert für das angegebene Partikelsystem abzurufen. Sie übergeben den Partikelsystem-ID-Wert (wie er von der Funktion part_system_create_layer()) und die Funktion gibt den ID-Wert des Layers zurück.

Syntax:

part_system_get_layer(ind);

Argument Beschreibung
ind Der Partikelsystem-ID-Wert, um die Layer-ID von

Rückgabe:

Real (eindeutiger Ebenen-ID-Wert oder 0, wenn die Ebene eine intern verwaltete ist)

Beispiel:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer";
}

Der obige Code überprüft ein Partikelsystem, um zu sehen, auf welcher Ebene es sich befindet, und wenn es sich nicht auf der angegebenen Ebene befindet, wird es geändert.
