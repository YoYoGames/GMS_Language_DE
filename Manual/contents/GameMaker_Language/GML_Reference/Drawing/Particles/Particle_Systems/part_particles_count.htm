part_particles_count

Mit dieser Funktion können Sie prüfen, ob in einem Partikelsystem gerade Partikel erzeugt werden, und sie gibt auch die Anzahl der Partikel zurück.

Syntax:

part_particles_count(ind);

Argument Beschreibung
ind Der Index des Partikelsystems.

Rückgabe:

Real

Beispiel:

if (part_particles_count(Sname) == 0)
{
part_system_destroy(Sname);
instance_destroy();
}

Der obige Code prüft die Anzahl der Partikel im lokalen Partikelsystem, das in der Variablen "Sname" indiziert ist, und wenn es keine gibt, wird das System und anschließend die Instanz zerstört.
