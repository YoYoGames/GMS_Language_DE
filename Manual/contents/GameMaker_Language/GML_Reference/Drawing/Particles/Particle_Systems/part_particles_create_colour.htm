teil_partikel_erzeugen

Diese Funktion ist ideal für Effekte, die keine der von Partikelemittern angebotenen Funktionen benötigen (z. B. um Rauch aus einer Rakete oder einen einfachen Explosionseffekt zu erzeugen), da Sie ermöglicht es Ihnen, schnell und einfach Partikel an jeder beliebigen Stelle im Spielraum zu erzeugen. Sie können die Partikel auch "on the fly" einfärben, während sie mit dieser Funktion erstellt werden, und diese Farbe überlagert die vordefinierte Farbe des Partikels, allerdings wird diese Farbe nicht über die Lebensdauer des Partikels gemischt. Beachten Sie, dass Sie das Partikelsystem und den Partikeltyp zuvor erstellt haben müssen, damit diese Funktion verwendet werden kann.

Syntax:

part_particles_create_colour(ind, x, y, parttype, color, number);

Argument Beschreibung
ind Der Index des Partikelsystems.
x Die x-Koordinate, an der die Partikel erstellt werden sollen.
y Die y-Koordinate, an der die Partikel erstellt werden sollen.
parttype Der Index (Typ) der zu erstellenden Partikel.
Farbe Die Farbüberblendung für die Partikel.
Nummer Die Anzahl der zu erstellenden Partikel.

Rückgabe:

N/A

Beispiel:

if (speed > 1)
{
var _c = choose(c_aqua, c_lime, c_fuschia, c_orange);
part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

Der obige Code erzeugt bei jedem Schritt, bei dem die Instanz mit dem Code eine Geschwindigkeit größer als 1 hat, ein einzelnes Partikel. Diese Partikel werden eine zufällige Farbe haben.
