teil_partikel_klar

Mit dieser Funktion können Sie alle aktuell vom System erzeugten Partikel aus dem Raum löschen. Es werden nicht die Partikeltypen selbst zurückgesetzt oder entfernt, sondern nur ihre visuelle Darstellung, und wenn Sie ein Objekt haben, das Partikel von Wenn Sie ein Objekt haben, das Partikel von einem Emitter abgibt, verschwinden diese Partikel, werden aber im nächsten Schritt nach dem Aufruf dieses Codes wieder erscheinen.

Syntax:

part_particles_clear(ind);

Argument Beschreibung
ind Der Index des Partikelsystems.

Rückgabe:

N/A

Beispiel:

if (Leben <= 0)
{
part_particles_clear(global.Sname);
room_goto(rm_intro);
}

Der obige Code prüft den Wert der Variable "lives" und wenn er gleich 0 ist, löscht er alle Partikel aus dem System und wechselt dann den Raum.
