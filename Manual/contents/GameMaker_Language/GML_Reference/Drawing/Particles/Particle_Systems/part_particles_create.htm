teil_partikel_erzeugen

Diese Funktion ist ideal für Effekte, die keine der von Partikelemittern angebotenen Funktionen benötigen (z. B. um Rauch aus einer Rakete oder einen einfachen Explosionseffekt zu erzeugen), da sie Ihnen erlaubt, schnell und einfach Partikel an jeder beliebigen Position im Spielraum zu erzeugen. Beachten Sie, dass Sie das Partikelsystem und den Partikeltyp zuvor erstellt haben müssen, damit diese Funktion verwendet werden kann.

Syntax:

part_particles_create(ind, x, y, parttype, number);

Argument Beschreibung
ind Der Index des Partikelsystems.
x Die x-Koordinate, an der die Partikel erstellt werden sollen.
y Die y-Koordinate, an der die Partikel erstellt werden sollen.
parttype Der Index (Typ) der zu erstellenden Partikel.
Nummer Die Anzahl der zu erstellenden Partikel.

Rückgabe:

N/A

Beispiel:

if mouse_check_button(mb_left)
{
part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

Der obige Code prüft, ob die Maustaste gedrückt wird, und wenn er true zurückgibt, erzeugt er 5 Partikel an der Mausposition.
