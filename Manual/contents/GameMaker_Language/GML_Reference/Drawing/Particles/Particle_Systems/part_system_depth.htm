teil_system_tiefe

Mit dieser Funktion können Sie die Zeichnungstiefe für das Partikelsystem einstellen, ähnlich wie Sie die Rendertiefe der verschiedenen Ebenen im Raum einstellen können, wobei eine niedrige Zeichnungstiefe bedeutet, dass es über allen Dingen erscheint, die mit einer höheren Tiefe gezeichnet wird, und eine hohe Zeichentiefe bedeutet, dass es unter allem erscheint, was eine niedrigere Zeichentiefe hat.

Syntax:

part_system_depth( ind, depth );

Argument Beschreibung
ind Der Index des zu ändernden Partikelsystems.
Tiefe Die Tiefe, auf die das Partikelsystem gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

Der obige Code erzeugt ein Partikelsystem und speichert dessen Index in der globalen Variable "Sname". Diesem System wird dann eine niedrige Tiefe von -1000 gegeben, was bedeutet, dass es über allem erscheint, was eine höhere Zeichentiefe hat.
