teil_system_existiert

Mit dieser Funktion können Sie prüfen, ob das angegebene indizierte Partikelsystem im Spiel existiert oder nicht. Beachten Sie, dass diese Funktion einen Fehler auslöst, wenn die zu prüfende Variable eine nicht initialisierte Variable ist (der sonst ein Partikelsystem seinen Index zugewiesen hätte). einen Fehler auslösen.

Syntax:

part_system_exists(ind);

Argument Beschreibung
ind Der Index des Partikelsystems, auf das geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if !part_system_exists(global.Sname)
{
global.Sname = part_system_create();
}

Der obige Code prüft, ob das in der globalen Variablen referenzierte Partikelsystem existiert, und wenn nicht, wird es erstellt.
