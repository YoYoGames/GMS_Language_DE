part_system_layer

Diese Funktion kann verwendet werden, um ein Partikelsystem von seinem aktuellen Layer auf einen neuen zu schalten. Sie geben den Wert der Partikelsystem-ID an (wie er von der Funktion part_system_erzeugen_schicht()) und dann die eindeutige Layer-ID (wie von der Funktion zurückgegeben layer_create() oder den Namen der zu verwendenden Schicht als String - zum Beispiel "instance_layer" - wie im Raumeditor definiert), und das System wird in die neue Schicht verschoben.

Syntax:

part_system_layer(ps, layer);

Argument Beschreibung
ps Die Partikel-System-ID
Ebene Die neue Layer-ID

Rückgabe:

N/A

Beispiel:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer";
}

Der obige Code überprüft ein Partikelsystem, um zu sehen, auf welcher Ebene es sich befindet, und wenn es sich nicht auf der angegebenen Ebene befindet, wird es geändert.
