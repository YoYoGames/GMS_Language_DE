part_system_draw_order

Mit dieser Funktion können Sie die Art und Weise einstellen, in der Partikel beim Erstellen auf dem Bildschirm gezeichnet werden. Das Standardsystem verwendet ein "alt > neu"-Look (die Funktion ist auf "true" gesetzt ), bei dem alte Partikel in einer höheren Tiefe gezeichnet werden als neuere und somit "unter" den neuen Partikeln erscheinen. Wenn Sie diese Funktion auf false setzen, können Sie diese Reihenfolge umkehren und die neuen Partikel höher zeichnen lassen, so dass sie "unter" den älteren Partikeln erscheinen. Die folgenden Bilder veranschaulichen dies, wobei das linke Bild den Standardwert true und das rechte Bild false darstellt:

HINWEIS: Wenn die Partikel mit einem additiven Überblendungsmodus gezeichnet werden, ist die Wirkung dieser Funktion möglicherweise nicht immer offensichtlich.

Syntax:

part_system_draw_order(ind, oldtonew);

Argument Beschreibung
ind Der Index des zu ändernden Partikelsystems.
oldtonew Ob alte Partikel hinter neuere gezeichnet werden sollen (true) oder umgekehrt (false).

Rückgabe:

N/A

Beispiel:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Dadurch wird ein neues Partikelsystem mit dem Index " mysystem" erstellt, und dann werden die Partikel so eingestellt, dass neuere Partikel über ältere gezeichnet werden.
