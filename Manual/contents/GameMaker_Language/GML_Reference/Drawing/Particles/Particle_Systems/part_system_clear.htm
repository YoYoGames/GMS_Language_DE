teil_system_klar

Mit dieser Funktion können Sie das indizierte System in den Grundzustand zurückversetzen, indem Sie alle Emitter entfernen und die Tiefe und Position zurücksetzen (falls sie geändert wurden). Seien Sie vorsichtig bei der Verwendung dieser Funktion, denn wenn Sie eine Instanz setzen oder ausbrechen oder eine andere Aktion, die einen Emitter verwendet, der mit diesem System verbunden ist, erhalten Sie einen Fehler, es sei denn, Sie verwenden die part_emitter_exists() Funktion zu überprüfen. Es ist auch nicht notwendig, die part_emitter_destroy() Funktion aufzurufen, da auch dies automatisch erledigt wird.

HINWEIS: diese Funktion löscht die sichtbaren Partikel im Raum, aber sie löscht nicht die Partikeleigenschaften und entfernt sie auch nicht aus dem Speicher. Hierfür sollten Sie die Funktionen part_type_clear() und verwenden. part_type_destroy().

Syntax:

part_system_clear(ind);

Argument Beschreibung
ind Der Index des zu löschenden Partikelsystems.

Rückgabe:

N/A

Beispiel:

part_system_clear(global.Sname);

Der obige Code setzt das in der globalen Variable "Sname" indizierte Partikelsystem auf den Standardzustand zurück.
