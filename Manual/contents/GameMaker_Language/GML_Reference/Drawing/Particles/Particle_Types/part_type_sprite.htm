part_type_sprite

Mit dieser Funktion können Sie einen Partikeltyp so einstellen, dass er ein eigenes Sprite aus den Spiel-Assets verwendet. Sie können das Sprite auswählen und dann GameMaker Studio 2 mitteilen, ob es das Sprite verwenden und animieren soll oder nicht, und wenn Sie animiert wählen, können Sie auch Wenn Sie animiert wählen, können Sie auch festlegen, dass die Teilbilder in einem solchen Intervall geändert werden, dass die Animation am Anfang und am Ende der Lebensdauer jedes erstellten Partikels beginnt und endet (wenn Sie also eine Lebensdauer mit einem Minimum von 15 Schritten und einem Maximum von 30 Schritten eingestellt haben, wird die Animation unabhängig vom Wert der Lebensdauer (wenn Sie also eine Lebensdauer von mindestens 15 und höchstens 30 Schritten eingestellt haben, wird die Animation unabhängig vom Wert der Lebensdauer angepasst, wobei sie bei kürzeren Lebenszeiten schneller und bei längeren langsamer abläuft). Sie können auch ein zufälliges Subbild auswählen, so dass, wenn das Sprite nicht animiert ist, ein anderes Bild zufällig ausgewählt wird, um das Sprite zu erstellen. wenn das Sprite nicht animiert ist, ein anderes Bild zufällig ausgewählt wird, um das Sprite zu erstellen, oder wenn das Partikel animiert ist, die Animation an einem zufälligen Punkt beginnt.

Syntax:

part_type_sprite(ind, sprite, animate, stretch, random);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Sprite Der Index des Sprites, auf den der Partikeltyp gesetzt werden soll.
beleben Ob der Animation des Sprites normal (true) oder nicht (false) gefolgt werden soll.
dehnen Ob die Animation des Sprites an die Lebensdauer der Partikel angepasst werden soll (true) oder nicht (false).
zufällig Ob ein zufälliges Teilbild gewählt werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Der obige Code setzt den in der globalen Variablen "p1" indizierten Partikeltyp auf die Form des in "spr_Coins" indizierten Sprites und setzt dann verschiedene andere Partikeleigenschaften.
