part_type_clear

Mit dieser Funktion können Sie einen Partikel "zurücksetzen", wobei alle Werte für jede der Funktionen, die sich auf den Partikel beziehen (Leben, Farbe, Alpha, Ausrichtung usw...) auf ihre Standardwerte zurückgesetzt werden. Beachten Sie, dass diese Funktion keine Partikel entfernt, die die gerade im Raum sichtbar sind, vom Bildschirm, dafür sollten Sie part_particles_clear().

Syntax:

part_type_clear(ind);

Argument Beschreibung
ind Der Index des zu löschenden Partikeltyps.

Rückgabe:

N/A

Beispiel:

part_type_clear(global.explode_part);

Der obige Code setzt den in der globalen Variablen "explode_part" indizierten Partikeltyp auf seine Standardwerte zurück.
