part_type_death

Mit dieser Funktion können Sie Ihren Partikel am Ende seiner Lebensdauer einen anderen Partikeltyp platzen lassen.

HINWEIS: Verwenden Sie NIEMALS denselben Partikeltyp, da dies zu einer Endlosschleife führt und das Spiel innerhalb von Sekunden zum Absturz bringen kann! Seien Sie außerdem sehr vorsichtig, wenn Sie diese Funktion verwenden, da sie die Anzahl der Partikel auf dem Bildschirm stark erhöhen kann und Ihr Spiel dadurch merklich verlangsamen.

Syntax:

part_type_death(ind, death_number, death_type);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
todesfall_number Die Anzahl der neuen Partikel, die beim Tod des Partikels erstellt werden sollen. Wenn ein negativer Wert, wird es stattdessen einen Bruchteil Chance ein erstellt wird (zB -3 würde bedeuten, dass ein alle 3 oder so Schritte erstellt wird).
Tod_Art Der Index des zu erstellenden Partikeltyps.

Rückgabe:

N/A

Beispiel:

particle1 = part_type_create();
part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 0.20, 0.30, 0.50, 0);
part_type_scale(particle1, 0.10, 0.20);
part_type_colour3(particle1, 33023, 8454143, 33023);
part_type_alpha3(particle1, 0.30, 0.50, 0);
part_type_speed(particle1, 0.40, 1, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_gravity(particle1, 0.20, 90);
part_type_orientation(particle1, 90, 90, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 1, 40);
particle2 = part_type_create();
part_type_shape(particle2, pt_shape_smoke);
part_type_size(particle2, 1, 1, 0, 0);
part_type_scale(particle2, 0.50, 0.50);
part_type_colour1(particle2, 12632256);
part_type_alpha3(particle2, 0.10, 0.40, 0);
part_type_speed(particle2, 0.50, 0.50, 0, 0);
part_type_direction(particle2, 0, 359, 0, 0);
part_type_gravity(particle2, 0.20, 90);
part_type_orientation(particle2, 0, 359, 0, 1, 1);
part_type_blend(particle2, 0);
part_type_life(particle2, 60, 60);
part_type_death(particle1, 1, particle2);

Der obige Code erzeugt und definiert zwei Partikel und setzt den in der Variablen "particle1" indizierten Partikel so, dass er am Ende seiner Lebensdauer einen Partikel von "particle2" erzeugt.
