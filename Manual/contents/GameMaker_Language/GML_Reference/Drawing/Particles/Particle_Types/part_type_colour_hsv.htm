teil_art_farbe_hsv

Mit dieser Funktion können Sie einen Farbton, eine Sättigung und einen Wertebereich für alle Partikel des angegebenen Typs festlegen. Sie geben für jede der drei Komponenten einen Minimal- und einen Maximalwert an und die erzeugten Partikel haben eine zufällige Farbe, die auf dem angegebenen Bereich der Parameter. Auf diese Weise können Sie Partikel mit gleichem Farbton, aber unterschiedlicher Sättigung, oder mit unterschiedlichem Farbton, aber gleichem Wert (Leuchtkraft) usw. erzeugen. Alle Werte müssen zwischen 0 und 255 liegen.

Syntax:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
hmin Das Minimum, das der Farbtonanteil der endgültigen Farbe sein kann.
hmax Das Maximum, das der Farbtonanteil der endgültigen Farbe sein kann.
smin Das Minimum, das die Sättigungskomponente der endgültigen Farbe sein kann.
smax Das Maximum, das die Sättigungskomponente der endgültigen Farbe sein kann.
vmin Das Minimum, das die Wertkomponente der endgültigen Farbe sein kann.
vmax Das Maximum, das die Wertkomponente der endgültigen Farbe sein kann.

Rückgabe:

N/A

Beispiel:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Der obige Code setzt jedes emittierte Partikel des in der globalen Variablen "Stars" indizierten Partikeltyps so, dass es unterschiedliche Farben und Sättigungen, aber denselben Wert (Leuchtkraft) hat.
