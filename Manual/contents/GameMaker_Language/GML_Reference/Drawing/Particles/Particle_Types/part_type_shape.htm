teil_art_form

Mit dieser Funktion können Sie die Sprite-Form einstellen, die für den Partikeltyp verwendet werden soll, wobei die folgenden Konstanten zulässig sind:

Konstant Beschreibung
pt_shape_pixel Ein 1x1-Pixel. (Dies ist die Standardeinstellung.)
pt_shape_disk Ein gefüllter Kreis.
pt_form_quadrat Ein gefülltes Quadrat.
pt_form_linie Eine 8px breite horizontale Linie.
pt_form_stern Ein fünfzackiger gefüllter Stern.
pt_shape_circle Ein 3px umrandeter Kreis.
pt_form_ring Ein Kreis mit einem Glühen nach innen (sieht aus wie eine Blase).
pt_form_kugel Ein nach außen leuchtender Kreis ' fest in der Mitte, nach außen leuchtend.
pt_shape_flare Ein grell leuchtender Punkt (sieht aus wie ein echter Stern in der Nacht).
pt_form_funke Ein Funken-Effekt ' wie ein Stern mit mehreren ausgeblendeten Punkten.
pt_form_explosion Eine viereckige Rauchwolke ' benötigt mehrere Farben, um einer Explosion zu ähneln.
pt_form_wolke Eine dünne Wolke, erfordert eine Hochskalierung und mehrere Partikel, um einer Wolke zu ähneln.
pt_form_rauch Eine glatte Version des Explosionseffekts. Verwenden Sie mehrere, um eine Rauchwolke zu erzeugen.
pt_shape_snow Eine generische Schneeflockenform.

Syntax:

part_type_shape(ind, shape);

Argument Beschreibung
ind Der Index des einzustellenden Partikeltyps.
Form Die zu wählende Form.

Rückgabe:

N/A

Beispiel:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Der obige Code setzt den in der globalen Variablen "p1" indizierten Partikeltyp auf die Form eines Quadrats und setzt dann verschiedene andere Partikeleigenschaften.
