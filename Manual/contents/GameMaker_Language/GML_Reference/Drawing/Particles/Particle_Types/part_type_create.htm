part_type_create

Mit dieser Funktion können Sie einen neuen Partikeltyp erstellen und der Rückgabewert sollte in einer Variablen zur Verwendung in allen nachfolgenden Partikelfunktionen gespeichert werden.

Syntax:

part_type_create();

Rückgabe:

Integer (Partikel-ID)

Beispiel:

mypart = part_type_create();

Dadurch wird ein neuer Partikeltyp erstellt und sein Index in der Variablen "mypart" gespeichert.
