part_type_alpha2

Mit dieser Funktion kann für jedes erzeugte Partikel des angegebenen Typs ein Alphawert (Transparenz)-Gradient eingestellt werden. Der erste Alpha-Wert ist derjenige, mit dem alle Partikel beginnen, und der zweite Alpha-Wert ist derjenige, mit dem die Partikel enden werden, und eine sanfte Gradientenänderung des Alphas über die Lebensdauer der Partikel von einem zum anderen erfolgt. Dieser kann zwischen 0 (transparent) und 1 (undurchsichtig) liegen.

Syntax:

part_type_alpha2(ind, alpha1, alpha2);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
alpha1 Das Start-Alpha des Teilchens.
alpha2 Das End-Alpha des Teilchens.

Rückgabe:

N/A

Beispiel:

part_type_alpha2(global.Snow_Part, 0, 1);

Der obige Code setzt alle erzeugten Partikel des in der globalen Variable "Snow_Part" indizierten Partikeltyps auf einen Alphawert von 0 (transparent) und blendet dann am Ende ihrer Lebensdauer auf einen Alphawert von 1 (undurchsichtig) ein.
