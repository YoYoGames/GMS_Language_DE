part_type_gravity

Diese Funktion stellt die Schwerkraft ein, die auf jedes erzeugte Partikel des angegebenen Typs wirken soll. Der Wert für die Schwerkraftstärke wird bei jedem Schritt zur Partikelgeschwindigkeit addiert und ist normalerweise ein kleiner Wert wie 0,5, während die Richtung die Richtung der Die Richtung ist die Richtung der Anziehungskraft und folgt den Standardrichtungen von GameMaker Studio 2: 0° ist rechts, 90° ist oben, 180° ist links und 270° ist unten.

Syntax:

part_type_gravity(ind, grav_amount, grav_direction);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
grav_menge Stärke der Schwerkraft.
grav_direction Die Richtung der Schwerkraft.

Rückgabe:

N/A

Beispiel:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Der obige Code stellt verschiedene Partikelwerte ein, einschließlich der Schwerkraft, die bei jedem Schritt 0,2 zur Geschwindigkeit hinzufügt, und zwar in einer Richtung von 270º, so dass das Partikel "nach unten" zum unteren Rand des Bildschirms gezogen wird.
