teil_art_grösse

Diese Funktion wird verwendet, um die Größe des Partikels zu bestimmen, wenn er erstellt wird, und kann auch verwendet werden, um den Partikel während seiner Lebensdauer größer oder kleiner werden zu lassen. Die Werte für die minimale und die maximale Größe sind standardmäßig auf 1 gesetzt (was bedeutet, dass die Größe das gleiche Verhältnis wie der Basispartikel ist, nachdem die part_type_scale() Skalierung eingestellt wurde) und wenn sie auf denselben Wert eingestellt sind, ist die Anfangsgröße für alle Partikel gleich, aber wenn sie auf unterschiedliche Werte eingestellt sind, wird jeder Partikel mit einer zufälligen Größe aus dem Bereich beginnen, z. B. bei einem Minimum von 0,5 und einem Maximum von 2 werden alle Partikel mit einer Größe zwischen der Hälfte und dem Doppelten der Basisskala erzeugt.

Sie können auch einen Inkrementwert festlegen, der das Partikelsprite über die Lebensdauer des Partikels wachsen (wenn es sich um eine positive Zahl handelt) oder schrumpfen (wenn es sich um eine negative Zahl handelt) lässt, wobei der angegebene Betrag bei jedem Schritt zur Partikelgröße hinzugefügt wird. Dieser Wert kann sein ein Minimum von (+/-)0,01 sein.

Schließlich können Sie den "Wiggle"-Faktor einstellen. Dies ist ein Wert, der bei jedem Schritt der Partikellebensdauer zufällig zur Größe addiert oder subtrahiert wird. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Größe_min Die Mindestgröße, mit der das Partikel beginnen kann.
größe_max Die maximale Größe, mit der das Partikel starten kann.
größe_incr Wie stark der Partikel pro Schritt zu- oder abnehmen soll.
Größe_Wackeln Wie viel soll pro Schritt zufällig von der Partikelgröße addiert oder subtrahiert werden.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

Der obige Code stellt verschiedene Partikelwerte ein, einschließlich der Größe, die bei der Erstellung jedes Partikels auf die doppelte Größe gesetzt wird, und erhöht dann die Größe bei jedem Schritt um 0,01 mit einem Wackelschritt von 0,1 zu.
