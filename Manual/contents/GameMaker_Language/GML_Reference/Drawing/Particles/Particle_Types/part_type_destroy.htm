teil_art_zerstören

Mit dieser Funktion können Sie den angegebenen Partikeltyp aus dem Spiel entfernen. Wenn Sie diese Funktion verwenden, verschwinden alle Partikel des angegebenen Typs aus dem Raum und der Partikel selbst wird aus dem Speicher entfernt, daher sollte diese Funktion nur verwendet werden wenn Sie den Partikel nicht mehr benötigen.

Syntax:

part_type_destroy(ind);

Argument Beschreibung
ind Der Index des zu zerstörenden Partikeltyps.

Rückgabe:

N/A

Beispiel:

if part_particles_count(smoke_sys) = 0
{
part_type_destroy(smoke_part);
part_system_destroy(smoke_sys);
instance_destroy();
}
else alarm[0] = 1;

Der obige Code prüft ein Partikelsystem, um zu sehen, ob derzeit Partikel im Raum sichtbar sind, und wenn nicht, werden die Partikel, das System und die Instanz zerstört.
