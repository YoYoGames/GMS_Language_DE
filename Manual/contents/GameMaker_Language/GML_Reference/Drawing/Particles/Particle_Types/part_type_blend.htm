part_type_blend

Mit dieser Funktion können Sie bewirken, dass Ihre Partikel mit einem additiven Überblendmodus gezeichnet werden(true) oder nicht(false). Additive Überblendung ist ein spezieller Überblendungsmodus, der die Helligkeitswerte der einzelnen Partikel addiert, wenn sie sich überlappen, so dass helle Farben allmählich heller werden (bis sie weiß erscheinen), wenn sie sich überlappen, und dunkle Farben immer transparenter werden, bis Schwarz fast unsichtbar wird. Diese Funktion kann einige schöne Partikeleffekte erzeugen, ist aber nicht immer notwendig. Für Bei einer Rauchfahne wäre die additive Überblendung beispielsweise ausgeschaltet, um den Effekt grau zu halten, aber bei einem Flammeneffekt wäre sie wahrscheinlich eingeschaltet, um die Partikel durchsichtiger und heller zu machen.

Syntax:

part_type_blend(ind, additiv);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Zusatzstoff Ob die Partikel additiv (true) oder normal (false) gemischt werden sollen.

Rückgabe:

N/A

Beispiel:

part_type_blend(part_Fire, true);

Der obige Code setzt alle erzeugten Partikel des in der Variablen "part_Fire" indizierten Partikeltyps auf einen additiven Überblendmodus.
