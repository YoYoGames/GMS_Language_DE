part_type_colour_rgb

Mit dieser Funktion können Sie die Mischung der Farben Rot, Grün und Blau für alle erzeugten Partikel des angegebenen Typs festlegen. Sie geben einen Minimal- und einen Maximalwert für jede der drei Komponenten an und die erzeugten Partikel haben eine zufällige Farbe, die auf dem angegebenen Bereich basierend auf dem angegebenen Parameterbereich. Alle Werte müssen zwischen 0 und 255 liegen.

Syntax:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
rmin Das Minimum, das der Rotanteil der endgültigen Farbe sein kann.
rmax Das Maximum, das der Rotanteil der endgültigen Farbe sein kann.
gmin Das Minimum, das der Grünanteil der endgültigen Farbe sein kann.
gmax Das Maximum, das der Grünanteil der endgültigen Farbe sein kann.
bmin Das Minimum, das der Blauanteil der endgültigen Farbe sein kann.
bmax Das Maximum, das der Blauanteil der endgültigen Farbe sein kann.

Rückgabe:

N/A

Beispiel:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Der obige Code setzt jedes emittierte Partikel des in der globalen Variablen "Blood_Part" indizierten Partikeltyps nur auf unterschiedliche Rottöne.
