teil_art_vorhanden

Mit dieser Funktion können Sie prüfen, ob der angegebene indizierte Partikeltyp im Spiel existiert oder nicht. Beachten Sie, dass diese Funktion einen Fehler auslöst, wenn die zu prüfende Variable eine nicht initialisierte Variable ist (der sonst ein Partikeltyp seinen Index zugewiesen hätte). einen Fehler.

Syntax:

part_type_exists(ind);

Argument Beschreibung
ind Der Index des Partikeltyps, auf den geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if !part_type_exists(global.p1)
{
global.p1 = part_type_create();
}

Der obige Code prüft, ob die globale Variable "p1" den Index eines Partikeltyps speichert, und wenn nicht, wird einer erstellt.
