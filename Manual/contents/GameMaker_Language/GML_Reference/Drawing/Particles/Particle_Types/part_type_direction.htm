part_type_direction

Diese Funktion wird verwendet, um die Richtung des Partikels zu bestimmen, wenn er erstellt wird, und kann auch verwendet werden, um den Partikel dazu zu bringen, seine Richtung während seiner Lebensdauer zu erhöhen oder zu verringern. Die minimalen und maximalen Richtungswerte sind standardmäßig auf 0 eingestellt, was bedeutet, dass Das bedeutet, dass sich der Partikel ohne Variation nach rechts bewegt. Diese Werte können jedoch geändert werden, um die Bewegungsrichtung entsprechend den Standardrichtungen von GameMaker Studio 2 zu begrenzen: 0 ist rechts, 90 ist oben, 180 ist links und 270 ist unten. Wenn Sie sie auf denselben Wert setzen, bewegen sich die Partikel nur in eine Richtung.

Sie können auch einen Inkrementwert einstellen, der über die Lebensdauer einen Gradbetrag zur Richtung addiert (wenn eine positive Zahl) oder subtrahiert (wenn eine negative Zahl). Dieser Wert kann ein Minimum von (+/-) 0,01 sein.

Schließlich können Sie den "Wiggle"-Faktor einstellen. Dies ist ein Wert, der bei jedem Schritt der Partikellebensdauer zufällig zur Richtung addiert oder subtrahiert wird. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar ein negativer Wert sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
dir_min Die minimale Richtung, aus der der Partikel starten kann.
dir_max Die maximale Richtung, aus der der Partikel starten kann.
dir_incr Wie stark die Partikelrichtung pro Schritt zu- oder abnehmen soll.
dir_wiggle Wie viel soll pro Schritt zufällig zur Richtung des Partikels addiert oder subtrahiert werden.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Der obige Code stellt verschiedene Partikelwerte ein, einschließlich der Richtung, die ein Zufallswert zwischen 0 und 359 Grad ist. Es wird auch um 1 Grad jeden Schritt und hat eine zufällige Menge hinzugefügt, um es von irgendwo zwischen 0 und 20 jeden Schritt zu gehen.
