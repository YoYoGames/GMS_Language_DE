teil_art_leben

Dies ist die Funktion, die bestimmt, wie lange jedes einzelne Partikel des angegebenen Typs auf dem Bildschirm bleibt. Wie bei anderen Partikelfunktionen geben Sie einen Minimal- und einen Maximalwert (in Schritten) an, und jede Partikel-Lebensdauer wird eine zufällige Anzahl von Schritten innerhalb des angegebenen Bereichs. Um alle Partikel mit der gleichen Lebensdauer zu haben, stellen Sie die beiden Werte gleich ein.

Syntax:

part_type_life(ind, life_min, life_max);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Lebensdauer_min Die Mindestlebensdauer der Partikel.
Lebensdauer_max Die maximale Lebensdauer der Partikel.

Rückgabe:

N/A

Beispiel:

part_type_life(part_Flare, room_speed, room_speed * 2);

Der obige Code setzt alle erzeugten Partikel des in der Variablen "part_Flare" indizierten Partikeltyps auf eine Lebensdauer zwischen 1 und 2 Sekunden, basierend auf der aktuellen Raumgeschwindigkeit.
