part_type_colour1

Mit dieser Funktion wird ein Partikeltyp so eingestellt, dass er für die Gesamtdauer der Lebensdauer jedes einzelnen Partikels eine einzige Farbe hat.

Syntax:

part_type_colour1(ind, color1);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Farbe1 Die einzelne Farbe, die den Partikeltyp bildet.

Rückgabe:

N/A

Beispiel:

part_type_colour1(global.Snow_Part, c_white);

Der obige Code setzt alle erzeugten Partikel des in der globalen Variablen "Snow_Part" indizierten Partikeltyps ausschließlich auf Weiß.
