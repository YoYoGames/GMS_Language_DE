teil_art_drehzahl

Diese Funktion wird verwendet, um die Geschwindigkeit des Partikels zu bestimmen, wenn er erstellt wird, und kann auch dazu verwendet werden, dass der Partikel seine Geschwindigkeit während seiner Lebensdauer erhöht oder verringert. Die Werte für die minimale und maximale Geschwindigkeit sind standardmäßig auf 1 gesetzt, was bedeutet, dass der Partikel sich bei jedem Schritt um 1 Pixel bewegt. Wenn sie auf denselben Wert eingestellt sind, ist die Anfangsgeschwindigkeit für alle Partikel gleich, aber wenn sie auf unterschiedliche Werte eingestellt sind, startet jeder Partikel mit einer zufälligen Geschwindigkeit innerhalb des angegebenen Bereichs. Wenn Sie zum Beispiel einen Mindestwert von 3 und einen Höchstwert von 6 angeben, werden alle Partikel mit einer Geschwindigkeit zwischen drei und sechs Pixeln pro Schritt erzeugt.

Sie können auch einen Inkrementwert festlegen, der den Partikel während seiner Lebensdauer schneller (wenn es sich um eine positive Zahl handelt) oder langsamer (wenn es sich um eine negative Zahl handelt) werden lässt, wobei der angegebene Betrag bei jedem Schritt zur Partikelgeschwindigkeit hinzugefügt wird. Dieser Wert kann ein Minimum sein von (+/-)0,01 sein.

Schließlich können Sie den "Wiggle"-Faktor einstellen. Dies ist ein Wert, der bei jedem Schritt der Partikellebensdauer zufällig zur Geschwindigkeit addiert oder subtrahiert wird. Offensichtlich sind größere Werte stärker ausgeprägt als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.

Syntax:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
geschwindigkeit_min Die minimale Geschwindigkeit, mit der der Partikel starten kann.
geschwindigkeit_max Die maximale Geschwindigkeit, mit der der Partikel starten kann.
geschwindigkeit_incr Wie stark die Partikelgeschwindigkeit pro Schritt zu- oder abnehmen soll.
speed_wiggle Wie viel soll pro Schritt zufällig von der Geschwindigkeit des Partikels addiert oder subtrahiert werden.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

Der obige Code stellt verschiedene Partikelwerte ein, darunter die Geschwindigkeit, die eine minimale Geschwindigkeit von fünf und eine maximale Geschwindigkeit von zehn festlegt, wobei die Geschwindigkeit bei jedem Schritt um einen Wert von -0,2 verlangsamt wird. Die Geschwindigkeit wackelt auch um einen Zufallswert zwischen 0 und 20 bei jedem Schritt.
