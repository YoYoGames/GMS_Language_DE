part_type_colour_mix

Mit dieser Funktion können Sie den angegebenen Partikeltyp auf eine zufällige Mischung aus zwei Farben einstellen.

Syntax:

part_type_colour_mix(ind, Farbe1, Farbe2);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Farbe1 Die erste Farbe, von der die Mischung ausgeht.
Farbe2 Die zweite Farbe, von der die Mischung übernommen wird.

Rückgabe:

N/A

Beispiel:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

Der obige Code setzt die Farbe für jedes emittierte Partikel des in der globalen Variable "P_Damage" indizierten Partikeltyps auf eine zufällige Mischung aus den Farben Rot und Gelb.
