teil_art_skala

Mit dieser Funktion wird die horizontale und vertikale Skalierung des Partikels festgelegt, bevor andere Größenänderungen vorgenommen werden. Diese Funktion arbeitet mit dem Verhältnis der Sprite-Größe, also ist ein Wert von 1 (der Standardwert) ein 1:1-Verhältnis, was bedeutet, dass 0,5 die Hälfte wäre und 2 das Doppelte.

Syntax:

part_type_scale(ind, xscale, yscale);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
x-Skala Die horizontale Skalierung des Basispartikels.
y-Skala Die vertikale Skalierung des Basispartikels.

Rückgabe:

N/A

Beispiel:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

Der obige Code stellt verschiedene Partikelwerte ein, einschließlich der Skalierung, die die x-Skala auf die achtfache Größe und die y-Skala auf die Hälfte der Größe setzt.
