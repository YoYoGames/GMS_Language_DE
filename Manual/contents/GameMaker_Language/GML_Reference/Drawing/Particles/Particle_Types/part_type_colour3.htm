teil_art_farbe3

Mit dieser Funktion können Sie einen dreifachen Farbverlauf für jedes erzeugte Partikel des angegebenen Typs einstellen. Die erste Farbe ist diejenige, mit der alle Partikel beginnen, die zweite Farbe ist diejenige, die nach der Hälfte der Lebensdauer überblendet wird und die dritte Farbe ist diejenige, mit der das Partikel enden wird. Während der Lebensdauer des Partikels wird ein sanfter Farbübergang von einer Farbe zur anderen stattfinden.

Syntax:

part_type_colour3( ind, Farbe1, Farbe2, Farbe3 );

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
Farbe1 Die Farbe, mit der das Partikel startet.
Farbe2 Die Farbe, die das Teilchen nach der Hälfte seiner Lebensdauer haben wird.
Farbe3 Die Farbe, bei der der Partikel endet.

Rückgabe:

N/A

Beispiel:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Der obige Code bewirkt, dass alle Partikel, die von dem in der Variablen "part_Health" indizierten Typ erzeugt werden, im Laufe ihrer Lebensdauer eine Farbmischung von rot über weiß nach kastanienbraun aufweisen.
