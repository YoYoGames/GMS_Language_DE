teil_art_alpha1

Mit dieser Funktion wird ein Partikeltyp so eingestellt, dass er für die gesamte Lebensdauer jedes einzelnen Partikels einen einzelnen Alpha-Wert (Transparenz) hat, der von 0 (transparent) bis 1 (undurchsichtig) reichen kann.

Syntax:

part_type_alpha1(ind, alpha1);

Argument Beschreibung
ind Der Index des zu ändernden Partikeltyps.
alpha1 Das Alpha des Teilchens.

Rückgabe:

N/A

Beispiel:

part_type_alpha1(global.Snow_Part, 0.5);

Der obige Code setzt alle erzeugten Partikel des in der globalen Variable "Snow_Part" indizierten Partikeltyps auf einen Alpha-Wert von 0,5 (halbtransparent).
