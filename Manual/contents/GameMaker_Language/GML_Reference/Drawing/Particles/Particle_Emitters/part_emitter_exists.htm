part_emitter_exists

Mit dieser Funktion können Sie prüfen, ob der angegebene Partikelemitter mit Index im angegebenen System existiert oder nicht. Beachten Sie, dass diese Funktion einen Fehler auslöst, wenn die zu prüfende Variable eine nicht initialisierte Variable ist (der sonst ein Partikelemitter seinen Index zugewiesen hätte). wird dies einen Fehler auslösen.

Syntax:

part_emitter_exists(ps, ind);

Argument Beschreibung
ps Das Partikelsystem, das auf einen Emitter geprüft werden soll.
ind Der Index des Emitters, nach dem gesucht werden soll.

Rückgabe:

Boolesche

Beispiel:

if part_emitter_exists(sname, p_emit1)
{
part_emitter_burst(sname, p_emit1, part_1, 30);
}

Der obige Code sucht nach dem Emitter, der in der Variable "permit" indiziert ist, und wenn er existiert, werden 30 Partikel von ihm ausgestoßen.
