part_emitter_clear

Mit dieser Funktion können Sie den angegebenen Emitter aus dem angegebenen Partikelsystem in seinen Standardzustand zurückversetzen. Dadurch werden auch alle Partikel gestoppt, die zu diesem Zeitpunkt vom Emitter gestreamt werden, und wenn Sie den Emitter wieder verwenden wollen, müssen Sie müssen Sie die Position der Region und den Partikeltyp mit der part_emitter_region() Funktion.

Syntax:

part_emitter_clear(ps, ind);

Argument Beschreibung
ps Das Partikelsystem, in dem sich der Emitter befindet.
ind Der Index des zu löschenden Emitters.

Rückgabe:

N/A

Beispiel:

part_emitter_clear(Sname, p_emit1 );

Der obige Code löscht den Partikelemitter, der in der Variablen "p_emit1" indiziert ist.
