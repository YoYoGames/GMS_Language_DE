part_emitter_destroy_all

Diese Funktion entfernt alle definierten Emitter aus dem gegebenen System und löscht sie aus dem Speicher (dadurch werden auch keine Partikel mehr von dem gegebenen Emitter produziert, aber sie werden NICHT aus dem Raum entfernt). Diese Funktion sollte immer aufgerufen werden, wenn die Emitter nicht mehr für das System benötigt werden, um Speicherlecks und Fehler zu vermeiden.

Syntax:

part_emitter_destroy_all( ps );

Argument Beschreibung
ps Das Partikelsystem, von dem alle Emitter zerstört werden sollen.

Rückgabe:

N/A

Beispiel:

if lives = 0
{
part_emitter_destroy_all(global.Sname);
room_goto(rm_Menu);
}

Der obige Code prüft die eingebaute globale Variable "lives" und wenn sie 0 ist, zerstört er alle Partikelemitter und wechselt dann den Raum.
