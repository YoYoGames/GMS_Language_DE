effect_create_above

Mit dieser Funktion können Sie einen einfachen Effekt über allen Instanzen Ihres Raums erzeugen (er wird tatsächlich in einer Tiefe von -100000 erzeugt). Wenn der Effekt etwas anderes als ef_rain oder ef_snow ist, können Sie eine x/y-Position definieren, um den Effekt zu erzeugen, und die Größe kann ein Wert von 0, 1 oder 2 sein, wobei 0 für klein, 1 für mittel und 2 für groß steht.

Es ist erwähnenswert, dass das Zeichnen dieser Effekte ein- und ausgeschaltet sowie pausiert werden kann, indem die Funktionen part_system_automatisches_Zeichnen() und part_system_automatic_update() mit dem entsprechenden Wert für den Partikelsystem-Index (wobei 0 für Effekte unterhalb und 1 für Effekte oberhalb steht).

Die verfügbaren Konstanten für die verschiedenen Partikelarten sind:

Konstant Beispiel Beschreibung
ef_cloud Ein Effekt, der zufällige Wolkenpartikel unterschiedlicher Größe erzeugt
ef_ellipse Ein Effekt, der expandierende Ellipsen erzeugt
ef_explosion Ein Effekt, der expandierende, ausklingende Explosionen erzeugt
ef_feuerwerk Ein Effekt, der mehrere kleine Partikel erzeugt, um eine Feuerwerksexplosion zu erzeugen
ef_flare Ein Effekt, der einen brillanten Punkt erzeugt, der aufflackert und ausklingt
ef_regen Ein Effekt, der Regenpartikel erzeugt, die vom oberen Rand des Bildschirms herunterkommen
ef_ring Ein Effekt, der expandierende und verblassende Kreise erzeugt
ef_smoke Ein Effekt, der kleine Rauchschwaden erzeugt
ef_smokeup Ein Effekt, der eine Rauchfahne erzeugt, die auf dem Bildschirm aufsteigt
ef_snow Ein Effekt, der mehrere Schneepartikel erzeugt, die auf den Bildschirm fallen
ef_spark Ein Effekt, der einen kleinen Funken erzeugt
ef_star Ein Effekt, der Sternteilchen erzeugt

Syntax:

effect_create_above(kind, x, y, size, color);

Argument Beschreibung
freundlich Die Art des Effekts (verwenden Sie eine der oben aufgeführten Konstanten).
x Die x-Positionierung des Effekts, falls relevant.
y Die y-Positionierung des Effekts, falls relevant.
Größe Die Größe des Effekts.
Farbe Die Farbe des Effekts.

Rückgabe:

N/A

Beispiel:

if health <= 0
{
effect_create_above(ef_explosion, x, y, 1, c_yellow);
instance_destroy();
}

Der obige Code erzeugt eine mittlere, gelbe Explosion über der Instanz und zerstört sie, wenn die Variable "Gesundheit" kleiner oder gleich 0 ist.
