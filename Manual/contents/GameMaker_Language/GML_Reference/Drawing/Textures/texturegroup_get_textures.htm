texturegroup_get_textures

Mit dieser Funktion können Sie die Texturseiten-IDs der einzelnen Seiten abrufen, aus denen eine Texturgruppe besteht. Sie geben den Texturgruppen-ID-String an (wie im Texturgruppen-Editor definiert), und die Funktion gibt ein 1D-Array zurück, wobei jeder Eintrag in dem Array eine einzelne Texturseiten-ID ist. Wenn die Funktion fehlschlägt - d. h. eine ungültige Gruppe wird angegeben oder der Gruppe ist keine Textur zugewiesen -, ist das Array leer (0 Länge).

Syntax:

texturegroup_get_textures(tex_id);

Argument Beschreibung
tex_id Der Name der zu prüfenden Texturgruppe (eine Zeichenkette)

Rückgabe:

1D-Array

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
if texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i]);
}
}

Der obige Code ruft die Texturseiten-IDs für die Texturgruppe "MainMenu" ab, prüft dann, ob sie ausgepackt sind, und wenn ja, werden sie im VRAM abgelegt.
