textur_get_texel_width

Ein Texel oder Texturelement ist die grundlegende Einheit des Texturraums, der in der Computergrafik verwendet wird. Texturen werden durch Arrays von Texels dargestellt, so wie Bilder durch Arrays von Pixeln dargestellt werden, und diese Funktion gibt die Breite eines einzelnen Texels aus der Texturseite des verwendeten Bild-Assets zurück.

Syntax:

texture_get_texel_width(tex);

Argument Beschreibung
tex Der zu verwendende Asset-Zeiger der Texturseite

Rückgabe:

Real

Beispiel:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Der obige Code ermittelt die Texelbreite und -höhe der Textur aus einem Sprite-Asset.
