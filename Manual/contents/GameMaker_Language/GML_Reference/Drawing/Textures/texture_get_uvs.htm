textur_get_uvs

Diese Funktion gibt ein 1D-Array mit 4 Elementen zurück, die die UV-Koordinaten für das Bild auf der Texturseite darstellen, wobei das Array mit den folgenden Werten gefüllt wird:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten
  • [4] = Anzahl der Pixel, die der Asset-Compiler von der linken Seite des Sprites abgeschnitten hat (nur Sprite-Assets)
  • [5] = Anzahl der Pixel, die der Asset-Compiler von der Oberseite des Sprites abgeschnitten hat (nur Sprite-Assets)
  • 6] = normalisierter Prozentsatz der Pixeldaten von der ursprünglichen Sprite-Breite, die in der Texturseite gespeichert wurden (nur Sprite-Assets)
  • 7] = normalisierter Prozentsatz der Pixeldaten aus der Höhe des Original-Sprites, der in der Texturseite gespeichert wurde (nur Sprite-Assets)

Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in allgemeinen 3D- und einigen der 2D-Primitivfunktionen sowie in den Shader-Funktionen. Wenn Sie die UVS für ein Sprite benötigen, können Sie die sprite_get_uvs() und für eine Schriftart verwenden, font_get_uvs().

Syntax:

textur_get_uvs(texid)

Argument Beschreibung
texid Das Bild-Asset, von dem die Textur abgerufen werden soll

Rückgabe:

1D-Array (4 - 8 Elemente)

Beispiel:

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Der obige Code speichert die UV-Koordinaten für die angegebene Oberflächentextur in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
