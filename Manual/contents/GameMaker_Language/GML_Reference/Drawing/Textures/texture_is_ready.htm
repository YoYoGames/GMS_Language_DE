textur_ist_bereit

Mit dieser Funktion kann geprüft werden, ob eine bestimmte Texturseite ausgepackt wurde und zur Verwendung bereit ist, oder ob eine Gruppe von Texturseiten ausgepackt wurde und zur Verwendung bereit ist. Sie geben die eindeutige Texturseiten-ID (wie von den texturegroup_*-Funktionen zurückgegeben) oder die Texturgruppen-ID-Zeichenkette (wie im Texturgruppen-Editor definiert) an, und die Funktion gibt true zurück, wenn sie ausgepackt wurden, oder false andernfalls.

Syntax:

texture_is_ready(tex_id);

Argument Beschreibung
tex_id Die Texturseiten-ID (eine Ganzzahl) oder eine Texturgruppen-ID (eine Zeichenkette)

Rückgabe:

Boolesche

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
if !texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i]);
}
}

Der obige Code ruft die Texturseiten-IDs für die Texturgruppe "MainMenu" ab, prüft dann, ob sie ausgepackt sind, und wenn nicht, werden sie in den VRAM vorgeholt.
