textur_get_width

Gibt die Breite der Textur mit der angegebenen ID zurück, die immer ein Wert im Bereich von 0 - 1 ist. Dies kann dann beim Zuordnen von Texturen zu Modellen oder Primitiven verwendet werden.

Syntax:

texture_get_width(tex);

Argument Beschreibung
tex Der zu verwendende Asset-Zeiger der Texturseite

Rückgabe:

Real

Beispiel:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Der obige Code ermittelt die Breite der Textur aus einem Sprite-Asset.
