textur_prefetch

Diese Funktion kann verwendet werden, um eine Texturseite oder eine Gruppe von Texturseiten "vorzuholen", d. h. sie bei Bedarf in den VRAM zu laden. Sie geben die eindeutige Texturseiten-ID an (wie sie von den texturegroup_*-Funktionen zurückgegeben wird), um eine einzelne Seite vorzuholen. Seite zu holen, oder Sie können einen Texturgruppennamen angeben (wie im Texturgruppen-Editor definiert), um alle Texturseiten der Gruppe zu holen.

Syntax:

texture_prefetch(tex_id);

Argument Beschreibung
tex_id Die Texturseiten-ID (eine Ganzzahl) oder eine Texturgruppen-ID (eine Zeichenkette)

Rückgabe:

N/A

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
texture_prefetch(_tex_array[i]);
}

Mit dem obigen Code werden alle Texturseiten unter der Texturgruppe "MainMenu" vorgeholt.
