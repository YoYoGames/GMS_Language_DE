texture_debug_messages

Mit dieser Funktion können Sie Textur-Debug-Meldungen aktivieren oder deaktivieren. Wenn sie aktiviert ist (auf true gesetzt), werden zusätzliche Informationen über die Verwendung von Texturseiten an das Ausgabefenster gesendet. Setzen Sie sie auf false, um diese Ausgabe wieder zu deaktivieren.

Syntax:

texture_debug_messages(enable);

Argument Beschreibung
aktivieren. Aktivieren oder Deaktivieren der Textur-Debug-Meldungen

Rückgabe:

N/A

Beispiel:

if global.debug
{
texture_debug_messages(true);
}

Der obige Code prüft den Wert einer globalen Variablen, und wenn er auf "true" gesetzt ist, werden Textur-Debug-Meldungen aktiviert.
