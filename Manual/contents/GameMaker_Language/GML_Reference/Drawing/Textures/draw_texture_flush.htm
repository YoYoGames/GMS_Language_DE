zeichnen_textur_bündig

Mit dieser Funktion können Sie alle Texturen aus dem Videospeicher entfernen, und sie werden dann bei der ersten Verwendung neu geladen. Dies ist die einzige effektive Möglichkeit, den Videospeicher zu verwalten, wenn Sie mehrere Texturseiten für ein Spiel haben, und Sie sollten den Texturspeicher zwischen den Ebenen Ihres Spiels leeren und die Grafiken mit der Texturgruppenfunktion organisieren, um sicherzustellen, dass möglichst wenige Texturen verwendet werden.

Es ist erwähnenswert, dass standardmäßig auf allen Zielen Textur-Seiten nur geladen werden, wenn sie benötigt werden, und Sie können die verschiedenen Prefetch- und Flush-Funktionen für Sprites (zu finden hier) verwenden verwenden, um die Dinge zu verwalten, ebenso wie diese Funktion.

Syntax:

draw_texture_flush();

Rückgabe:

N/A

Beispiel:

draw_texture_flush();

Der obige Code löscht den Videospeicher von Texturdaten und würde wahrscheinlich im Ereignis "create" der ersten Instanz eines im Raum platzierten Objekts platziert werden.
