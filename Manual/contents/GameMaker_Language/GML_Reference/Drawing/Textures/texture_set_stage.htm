textur_set_stage

Diese Funktion legt für den angegebenen Bühnen-"Slot" eine zu verwendende Textur fest. Die Anzahl der verfügbaren Bühnen-"Slots" hängt von der Plattform ab, für die Sie kompilieren, wobei für Windows, Mac und Linux maximal 8 zur Verfügung stehen . Es ist auch erwähnenswert, dass der erste Stage-"Slot" von GameMaker Studio 2 immer automatisch verwendet wird .

HINWEIS: Diese Funktion bewirkt nichts außerhalb des Kontextes eines laufenden Shaders! Siehe Shader für weitere Informationen.

Syntax:

texture_set_stage(stage, tex);

Argument Beschreibung
Bühne Der zu verwendende Textur-"Steckplatz".
tex Die zu verwendende Textur.

Rückgabe:

N/A

Beispiel:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set (GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Der obige Code holt sich das Handle für den Sampler innerhalb des als "GlassShader" indizierten Shadersund setzt dann diese Shader-Konstante auf die angegebene Sprite-Textur.
