textur_bündig

Diese Funktion kann verwendet werden, um eine Texturseite oder eine Gruppe von Texturseiten zu "flushen", d. h. sie aus dem VRAM zu entfernen, wenn sie nicht mehr benötigt werden. Sie geben die eindeutige Texturseiten-ID an (wie von den texturegroup_*-Funktionen zurückgegeben), um eine einzelne Seite zu flushen. Sie können aber auch einen Texturgruppennamen (wie im Texturgruppen-Editor definiert) angeben, um alle Texturseiten in der Gruppe zu flushen.

Syntax:

texture_flush(tex_id);

Argument Beschreibung
tex_id Die Texturseiten-ID (eine Ganzzahl) oder eine Texturgruppen-ID (eine Zeichenkette)

Rückgabe:

N/A

Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
texture_flush(_tex_array[i]);
}

Der obige Code spült alle Texturseiten unter der Texturgruppe "MainMenu".
