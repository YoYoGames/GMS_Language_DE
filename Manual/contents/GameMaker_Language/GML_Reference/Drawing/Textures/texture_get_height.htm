textur_get_height

Gibt die Höhe der Textur mit der angegebenen ID zurück, die immer ein Wert im Bereich von 0 - 1 ist. Dies kann dann beim Zuordnen von Texturen zu Modellen oder Primitiven verwendet werden.

Syntax:

texture_get_height(tex);

Argument Beschreibung
tex Der zu verwendende Asset-Zeiger der Texturseite

Rückgabe:

Real

Beispiel:

tex_h = texture_get_height(surface_get_texture(global.Surf));

Mit dem obigen Code wird die Höhe der Textur aus einer zuvor erstellten Oberfläche entnommen.
