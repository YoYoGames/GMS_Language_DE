texturegroup_get_fonts

Mit dieser Funktion können Sie den Font-Index jeder der Fonts abrufen, die den Texturseiten innerhalb der angegebenen Texturgruppe zugeordnet sind. Sie geben den Texturgruppen-ID-String an (wie im Texturgruppen-Editor definiert) und die Funktion gibt ein 1D-Array zurück, in dem jeder Eintrag den Font-Index für eine Font-Ressource enthält. Wenn die Funktion fehlschlägt - d. h. eine ungültige Gruppe angegeben wird oder der Gruppe keine Textur zugewiesen ist -, ist das Array leer (0 Länge).

Syntax:

texturegroup_get_fonts(tex_id);

Argument Beschreibung
tex Der Name der zu prüfenden Texturgruppe (eine Zeichenkette)

Rückgabe:

1D-Array

Beispiel:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
show_debug_message("Schriftart " + string(i) + " Index:" + string(tex_array[i]));
}

Der obige Code ruft die Font-Indizes für die Texturgruppe "MainMenu" ab und zeigt diese IDs dann im Konsolenausgabefenster an.
