zeichnen_holen_ausrichten

Diese Funktion wird verwendet, um die Einstellung der Textausrichtung entlang der horizontalen Achse zu erhalten, und gibt eine der unten aufgeführten Konstanten zurück.

Syntax:

draw_get_halign();

Rückgabe:

Konstant

Konstant Ausrichtung
fa_links
fa_center
fa_rechts

Beispiel:

if draw_get_halign() != fa_left
{
draw_set_halign(fa_left);
}
draw_text(100, 32, "Ergebnis: " + string(Ergebnis));

Der obige Code prüft, ob die aktuelle Textausrichtung linksbündig ist, und wenn nicht, wird sie gesetzt und dann wird ein Text gezeichnet.
