zeichnen_ausrichten

Diese Funktion wird verwendet, um Text entlang der horizontalen Achse auszurichten. Wenn Sie die horizontale Ausrichtung ändern, ändert sich die Position und Richtung, in der alle weiteren Texte gezeichnet werden, wobei der Standardwert fa_left ist. Die folgenden Konstanten werden akzeptiert:

Konstant Ausrichtung
fa_links
fa_center
fa_rechts

Syntax:

draw_set_halign(halign);

Argument Beschreibung
halign Horizontale Ausrichtung.

Rückgabe:

N/A

Beispiel:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Gesundheit: " + string(health));

Der obige Code zeichnet zwei Zeichenketten in dieselbe Zeile, wobei die Partitur linksbündig und die Gesundheit rechtsbündig ist.
