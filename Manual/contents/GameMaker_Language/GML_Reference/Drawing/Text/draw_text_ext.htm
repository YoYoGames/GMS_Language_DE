zeichen_text_ext

Diese Funktion zeichnet Text auf ähnliche Weise wie zeichnen_text() nur dass Sie jetzt den Abstand zwischen den einzelnen Textzeilen festlegen können - falls der Text mehr als eine Zeile einnimmt - und die Breite (in Pixeln) der Zeichenkette pro Zeile begrenzen können. des Strings pro Zeile begrenzen, so dass, sollte eine Zeile diesen Wert überschreiten, GameMaker Studio 2 den Text automatisch auf die nächste Zeile am nächsten verfügbaren Leerraum aufteilt (wenn der Text keine Leerräume hat, überschreitet er diesen maximalen Breitenwert). Beachten Sie, dass dadurch jedes Leerzeichen am Anfang der Zeichenkette entfernt wird, bevor es zum Zeichnen geparst wird. Beachten Sie auch, dass ein Wert von -1 für das Argument "Zeilentrennung" standardmäßig zu einer Trennung führt, die auf der Höhe des Zeichens "M" in der gewählten Schriftart basiert.

Syntax:

draw_text_ext(x, y, string, sep, w);

Argument Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
String Die Schnur zum Ziehen.
sep Der Abstand in Pixeln zwischen den Textzeilen.
w Die maximale Breite der Zeichenkette in Pixel vor einem Zeilenumbruch.

Rückgabe:

N/A

Beispiel:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Der obige Code zeichnet den Text, den der Benutzer in die Tastatur eingibt, und teilt ihn jedes Mal auf neue Zeilen auf, wenn die Länge der Zeichenkette für diese Zeile 300 Pixel überschreitet. Der Code behält auch einen Abstand von 3 Pixeln zwischen den Zeilen bei, wenn dies der Fall ist.
