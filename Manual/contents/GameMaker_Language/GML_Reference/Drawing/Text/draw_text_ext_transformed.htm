draw_text_ext_transformed

Diese Funktion ist eine Kombination aus der Basis zeichnen_text() Funktion mit der zeichne_text_ext() und draw_text_transformiert() Funktionen. Sie ermöglicht das Skalieren und Drehen von Text unter Beibehaltung eines bestimmten Zeilenabstands und einer maximalen Breite pro Zeile. Beachten Sie, dass das Argument "Breite" auf einer Skalierung von 1 basiert; wenn die Skalierung also anders ist, sollte dieser Wert proportional geändert werden. Wenn z. B. die Basisbreite für einen Zeilenumbruch 300 ist und Sie die Skala auf 2 setzen, dann wird der Text falsch angezeigt, da er die angegebene Breite überschreitet. Stattdessen hätten Sie die Breite auf 150 setzen sollen, um die Skalierung auszugleichen.

Syntax:

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

Argument Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
String Die Schnur zum Ziehen.
sep Der Abstand in Pixeln zwischen den Textzeilen.
w Die maximale Breite der Zeichenkette in Pixel vor einem Zeilenumbruch.
x-Skala Die horizontale Skala.
y-Skala Die vertikale Skala.
Winkel Der Winkel des Textes.

Rückgabe:

N/A

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

Der obige Code zeichnet den angegebenen Text in der Mitte des Raums, mit einer maximalen Zeichenfolgelänge von 300 Pixeln, einem Abstand zwischen den einzelnen Zeilen von 10 Pixeln, umlaufend und auf die doppelte Größe skaliert.
