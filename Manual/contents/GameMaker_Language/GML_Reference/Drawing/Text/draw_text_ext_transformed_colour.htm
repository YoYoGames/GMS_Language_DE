draw_text_ext_transformed_colour

Diese Funktion ist eine Kombination aus der Basis zeichnen_text() Funktion mit der zeichnen_text_ext(), draw_text_transformiert() und draw_text_colour() Funktionen. Als solche ist sie die allgemeinste aller Funktionen zum Zeichnen von Text und bietet ein hohes Maß an Flexibilität beim Zeichnen. Beachten Sie, dass das Argument "Breite" Argument auf einem Maßstab von 1 basiert. Wenn der Maßstab also anders ist, sollte dieser Wert proportional geändert werden. Wenn z. B. die Basisbreite für einen Zeilenumbruch 300 ist und Sie den Maßstab auf 2 setzen, dann wird der Text falsch dargestellt, da er die angegebene Breite. Stattdessen hätten Sie die Breite auf 150 setzen sollen, um die Skalierung auszugleichen.

HINWEIS: Das Überblenden von Farbverläufen ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, obwohl Sie trotzdem die Überblendungsfarben festlegen können und die Schrift mit der ersten angegebenen Farbe überblendet wird. Alle Überblendungen auf diese Weise erzeugen jedoch ein Duplikat Schriftart, die dann im Cache gespeichert und bei Bedarf verwendet wird, was alles andere als optimal ist, und wenn Sie mehrere Farbwechsel verwenden, verlangsamt dies die Leistung Ihres Spiels. Sie können die Größe des Schrift-Caches einstellen, um dies zu begrenzen, falls es notwendig sein sollte mit der Funktion font_set_cache_size().

Syntax:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Argument Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
String Die Schnur zum Ziehen.
sep Der Abstand in Pixeln zwischen den Textzeilen.
w Die maximale Breite der Zeichenkette in Pixel vor einem Zeilenumbruch.
x-Skala Die horizontale Skala.
y-Skala Die vertikale Skala.
Winkel Der Winkel des Textes.
c1 Die Farbe für den linken oberen Rand des gezeichneten Textes.
c2 Die Farbe für den oberen rechten Rand des gezeichneten Textes.
c3 Die Farbe für den unteren rechten Rand des gezeichneten Textes.
c4 Die Farbe für den unteren linken Teil des gezeichneten Textes.
alpha Das Alpha für den Text.

Rückgabe:

N/A

Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "Mein Name ist" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Damit wird der angegebene Text mit einem blauen bis dunkelblauen Verlauf nach unten gezeichnet, an der Position (200,200) im Raum, auf den Kopf gedreht und mit der Hälfte der normalen Punktgröße der Schrift. Zwischen jeder Zeile sollen 20 Pixel liegen, wenn mehr als eine Zeile gezeichnet wird. eine Linie gezeichnet werden und jede Linie hat eine maximale Breite von 200 Pixeln.
