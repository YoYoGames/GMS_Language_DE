zeichnen_get_font

Diese Funktion ermittelt die aktuell zugewiesene Schriftart zum Zeichnen von Text. Die Funktion gibt -1 zurück, wenn keine Schriftart eingestellt ist, oder den ID-Wert (eine positive Ganzzahl) für die zugewiesene Schriftartressource.

Syntax:

draw_get_font();

Rückgabe:

Integer (-1 oder Font-ID-Wert)

Beispiel:

if draw_get_font() != fnt_MainMenu
{
draw_set_colour(c_blue);
draw_set_font(fnt_MainMenu);
}
draw_text(room_width / 2, 200, "MENU" );

Der obige Code prüft, welche Schriftart zum Zeichnen verwendet wird, und wenn es nicht fnt_MainMenu ist , dann werden die Schriftart und die Zeichenfarbe eingestellt, bevor ein Text auf den Bildschirm gezeichnet wird.
