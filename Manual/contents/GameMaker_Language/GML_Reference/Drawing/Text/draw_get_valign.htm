draw_get_valign

Diese Funktion wird verwendet, um die Einstellung der Textausrichtung entlang der vertikalen Achse zu erhalten, und gibt eine der unten aufgeführten Konstanten zurück.

Syntax:

draw_get_valign();

Rückgabe:

Konstant

Konstant Ausrichtung
fa_top
fa_middle
fa_bottom

Beispiel:

if draw_get_valign() != fa_top
{
draw_set_valign(fa_top);
}
draw_text(100, 32, "Ergebnis: " + string(Ergebnis));

Der obige Code prüft, ob die aktuelle Textausrichtung oben ausgerichtet ist, und wenn nicht, wird sie gesetzt und dann wird ein Text gezeichnet.
