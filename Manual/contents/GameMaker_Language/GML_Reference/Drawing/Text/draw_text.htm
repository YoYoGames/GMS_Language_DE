zeichnen_text

Mit dieser Funktion können Sie eine beliebige Zeichenkette an einer beliebigen Stelle im Raum zeichnen (um reelle Zahlen zu zeichnen, sollten Sie die string() Funktion verwenden, um sie in Text umzuwandeln). Um Zeichenketten zu kombinieren, können Sie verwenden (siehe Beispiel unten) und Sie können auch \n innerhalb einer Zeichenkette verwenden, um einen Zeilenumbruch hinzuzufügen, so dass sie über mehrere Zeilen gezeichnet wird (Informationen darüber, wie Sie eine Zeichenkette richtig formatieren und welche Escape-Zeichen Sie verwenden können, finden Sie hier). Die Farbe des Textes und der Alpha-Wert richten sich nach den aktuellen Basis-Alpha- und Farbwerten, die mit draw_set_alpha() und . draw_set_colour().

HINWEIS:Die tatsächliche Position des Textes wird von den Ausrichtungswerten beeinflusst, die durch draw_set_halign() und draw_set_valign().

Syntax:

draw_text(x, y, string);

Argument Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
String Die Schnur zum Ziehen.

Rückgabe:

N/A

Beispiel:

draw_text(x, y, "Hallo, " + global.Name + "!\nIch hoffe, es geht Ihnen gut!");

Der obige Code zeichnet an der x/y-Position der Instanz eine Zeichenkette, die die in der globalen Variable "Name" gespeicherte Zeichenkette verwendet und diese auf zwei Zeilen aufteilt.
