zeichen_setzen_schrift

Mit dieser Funktion wird die Schriftart festgelegt, die für alle weiteren Textzeichnungen verwendet werden soll. Diese Schriftart muss zu den Schriftarten-Assets des Spiels hinzugefügt worden sein oder entweder mit der Funktion Schriftart_hinzufügen() oder font_add_sprite().

Syntax:

draw_set_font(font);

Argument Beschreibung
Schriftart Der Name der zu verwendenden Schriftart.

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World" );

Der obige Code zeichnet den angegebenen Text mit der Schriftart, die in der Variablen "fnt_Game" indiziert und blau eingefärbt ist.
