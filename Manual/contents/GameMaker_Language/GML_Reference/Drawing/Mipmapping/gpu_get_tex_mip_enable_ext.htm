gpu_get_tex_mip_enable_ext

Mit dieser Funktion können Sie abfragen, ob Mipmapping auf einem Shader-Sampler ausgeschaltet, für alles eingeschaltet oder nur für im Texturgruppen-Manager ausgewählte Texturgruppen eingeschaltet ist. Sie geben den Indexwert für den Shader-Sampler an (wie er von der Funktion shader_get_sampler_index()zurückgegeben wird, und die Funktion gibt eine der unten aufgeführten Konstanten zurück.

Syntax:

gpu_get_tex_mip_enable_ext(sampler_index);

Argument Beschreibung
sampler_index Der Index des zu erhaltenden Shader-Samplers

Rückgabe:

Konstant:

Konstant Beschreibung
mip_aus Mipmapping ist deaktiviert.
mip_on Mipmapping für alle Texturen ist aktiviert.
mip_markedonly Mipmapping ist für Texturen aktiviert, die es in den Optionen der Texturgruppe aktiviert haben.

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
{
gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Der obige Code aktiviert Mipmapping für den angegebenen Shader-Textursampler, wenn es nicht bereits aktiviert ist.
