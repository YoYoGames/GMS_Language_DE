gpu_get_tex_mip_enable

Mit dieser Funktion können Sie ermitteln, ob das Mipmapping ausgeschaltet, für alles eingeschaltet oder nur für die im Texturgruppen-Manager ausgewählten Texturgruppen eingeschaltet ist. Die Funktion gibt zurück eine der unten aufgeführten Konstanten zurück, wobei die Voreinstellung mip_markedonly ist.

Syntax:

gpu_get_tex_mip_enable();

Rückgabe:

Konstant:

Konstant Beschreibung
mip_aus Mipmapping ist deaktiviert.
mip_on Mipmapping für alle Texturen ist aktiviert.
mip_markedonly Mipmapping ist für Texturen aktiviert, die es in den Texturgruppenoptionen aktiviert haben (Standard).

Beispiel:

if gpu_get_tex_mip_enable != mip_on
{
gpu_set_tex_mip_enable(mip_on);
}

Der obige Code prüft, ob mipmapping aktiviert ist, und wenn nicht, wird es aktiviert.
