gpu_get_tex_mip_bias_ext

Mit dieser Funktion können Sie den Mipmap-Bias-Wert für einen bestimmten Shader-Sampler abrufen. Sie liefern den Indexwert für den Shader-Sampler (wie er von der Funktion shader_get_sampler_index(), zurückgegeben wird) und die Funktion gibt einen Wert von 0,0 für keinen Bias oder einen größeren Wert zurück, wobei 1 der ersten Mipmap entspricht, 2 der zweiten Mipmap usw.. Dies steuert die Rate, mit der die Mipmap ausgetauscht wird, und macht die Shader-Texturen im Allgemeinen unschärfer je höher der Wert und je größer der "Abstand", der betrachtet wird. Beachten Sie, dass dies auch negative Werte zurückgeben kann, in diesem Fall werden die Shader-Texturen über eine größere Entfernung schärfer, je niedriger der Wert ist.

Syntax:

gpu_get_tex_mip_bias_ext(sampler_index);

Argument Beschreibung
sampler_index Der Index des zu erhaltenden Shader-Samplers

Rückgabe:

Real (Voreinstellung: 0)

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Der obige Code setzt den Mip-Filter-Bias für den angegebenen Shader-Textursampler auf 0, wenn er nicht bereits auf 0 gesetzt wurde.
