gpu_set_tex_min_mip_ext

Mit dieser Funktion können Sie die minimale Mipmap-Stufe setzen, die aktuell für einen bestimmten Shader-Sampler verwendet wird. Sie liefern den Indexwert für den Shader-Sampler (wie er von der Funktion shader_get_sampler_index(), zurückgegeben wird) und geben dann einen Wert an, wobei 0 für volle Auflösung steht, 1 für die erste Mipmap, 2 für die zweite usw...

Syntax:

gpu_set_tex_min_mip_ext(sampler_index, minmip);

Argument Beschreibung
sampler_index Der Index des zu erhaltenden Shader-Samplers
minmip Die minimal zu verwendende Mipmap-Ebene

Rückgabe:

N/A

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
if gpu_get_tex_min_mip_ext(_sampleIndex) != 0
{
gpu_set_tex_min_mip_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Der obige Code setzt den minimalen Mipmap-Level für den angegebenen Shader-Textursampler auf 0, wenn er nicht bereits auf 0 gesetzt wurde.
