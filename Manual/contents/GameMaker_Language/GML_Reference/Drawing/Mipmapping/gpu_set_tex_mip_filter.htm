gpu_set_tex_mip_filter

Mit dieser Funktion können Sie den aktuellen mip-Filtermodus auf einen der drei unterstützten Typen einstellen. Sie geben die Konstante an, die sich auf den gewünschten mip-Filtermodus bezieht, wobei Sie zwischen Punktfilterung (Voreinstellung, d. h. keine Filterung), lineare Filterung (dies wird auch als trilineare Filterung bezeichnet) oder aniostrope Filterung (siehe die Konstantentabelle unten).

Konstant Beschreibung
tf_punkt Dadurch wird die Filterung zwischen den Mipmap-Ebenen deaktiviert, was zu sichtbaren Texturübergängen führen kann, aber die beste Leistung bietet.
tf_linear Dadurch wird die lineare Filterung zwischen den Mipmap-Ebenen aktiviert (dies wird auch als trilineare Filterung bezeichnet), wodurch die Texturübergänge geglättet werden, was jedoch zu einer geringfügigen Leistungseinbuße führt.
tf_anisotrop Dadurch wird die anisotrope Filterung zwischen den Mipmap-Ebenen aktiviert, was die Qualität der Texturübergänge erheblich verbessert und die bei anderen Filterungsmodi sichtbare Unschärfe reduzieren kann, aber den größten Einfluss auf die Leistung hat.

Syntax:

gpu_get_tex_mip_filter(filter);

Argument Beschreibung
Filter Der zu verwendende mip-Filtermodus (eine Konstante, Voreinstellung: tf_point).

Rückgabe:

N/A

Beispiel:

if keyboard_check(ck_enter)
{
switch(gpu_get_tex_mip_filter())
{
case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
}
}

Der obige Code prüft die Tastatur und wenn die angegebene Taste gedrückt wird, wird der aktuelle mip-Filter abgerufen und der Wert auf den nächsten umgeschaltet, wobei die verschiedenen Modi durchlaufen werden.
