gpu_set_tex_max_aniso

Mit dieser Funktion können Sie den aktuellen Maximalwert der Anisotropie bei Verwendung des Filtermodus tf_anisotropic setzen (siehe gpu_get_tex_mip_filter() für weitere Informationen). Der Eingabewert muss zwischen zwischen 1 und 16 liegen.

Syntax:

gpu_set_tex_max_aniso(maxaniso);

Argument Beschreibung
maxaniso Der maximale Pegel für die anisotrope Filterung (Voreinstellung: 16)

Rückgabe:

N/A

Beispiel:

if gpu_get_tex_max_aniso() != 8
{
gpu_set_tex_max_aniso(8);
}

Der obige Code prüft den aktuellen maximalen anisotropen Filterungsgrad und wenn dieser nicht 8 ist, wird er auf 8 gesetzt.
