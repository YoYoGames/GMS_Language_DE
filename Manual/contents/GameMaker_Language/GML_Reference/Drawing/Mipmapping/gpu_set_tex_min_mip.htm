gpu_set_tex_min_mip

Mit dieser Funktion können Sie die minimale Mipmap-Stufe einstellen, die aktuell verwendet wird, wobei ein Wert von 0 die höchste Auflösung ist, 1 bedeutet, die erste Mipmap zu verwenden, 2 die zweite usw..

Syntax:

gpu_set_tex_min_mip(minmip);

Argument Beschreibung
minmip Die minimal zu verwendende Mipmap-Ebene

Rückgabe:

N/A

Beispiel:

if gpu_get_tex_min_mip() != 0
{
gpu_set_tex_min_mip(0);
}

Der obige Code prüft den aktuellen minimalen Mipmap-Pegel und wenn dieser nicht 0 ist, wird er auf 0 gesetzt.
