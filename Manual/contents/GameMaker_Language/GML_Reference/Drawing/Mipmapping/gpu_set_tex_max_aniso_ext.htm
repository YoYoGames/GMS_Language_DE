gpu_set_tex_max_aniso_ext

Mit dieser Funktion können Sie den maximalen Grad der Anisotropie einstellen, wenn Sie den Filtermodus tf_anisotropic (siehe gpu_get_tex_mip_filter() für weitere Informationen) auf einen Shader-Sampler anwenden. Sie liefern den Indexwert für den Shader-Sampler (wie er von der Funktion shader_get_sampler_index()zurückgegeben wird, und dann einen Wert im Bereich von 1 bis 16, um den Pegel einzustellen.

Syntax:

gpu_set_tex_max_aniso_ext(sampler_index, maxaniso);

Argument Beschreibung
sampler_index Der Index des zu erhaltenden Shader-Samplers
maxaniso Die maximal zu verwendende anisotrope Stufe (Standard: 16).

Rückgabe:

N/A

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set (shd_Glass);
if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8
{
gpu_set_tex_max_aniso_ext(_sampleIndex, 8);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Der obige Code setzt den maximalen Grad der Anisotropie für den angegebenen Shader-Textursampler auf 8, wenn er nicht bereits auf 8 gesetzt wurde.
