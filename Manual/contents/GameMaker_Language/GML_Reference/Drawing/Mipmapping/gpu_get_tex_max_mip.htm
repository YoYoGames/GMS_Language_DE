gpu_get_tex_max_mip

Mit dieser Funktion können Sie die aktuell eingestellte maximale Mipmap-Stufe ermitteln, die verwendet werden soll, wobei ein Wert von 0 die höchste Auflösung bedeutet, 1 bedeutet, dass die erste Mipmap verwendet werden soll, 2 die zweite usw.. Beachten Sie, dass dies sehr nützlich sein kann, um Bleeding-Artefakte zu vermeiden beim Rendern von Texturen, z. B. wenn Sie den Seitenrand der Textur auf 8px setzen und dann den maximalen Mipmap-Level auf 3 setzen, wird sichergestellt, dass Sie keine Bleeding-Probleme bei größeren Renderabständen bekommen.

Syntax:

gpu_get_tex_max_mip();

Rückgabe:

Real (Voreinstellung: 16)

Beispiel:

if gpu_get_tex_max_mip() != 4
{
gpu_set_tex_max_mip(4);
}

Der obige Code prüft den aktuellen maximalen Mipmap-Pegel und wenn dieser nicht 4 ist, wird er auf 4 gesetzt.
