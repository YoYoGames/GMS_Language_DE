Mipmapping

Dieser Abschnitt enthält alle Funktionen im Zusammenhang mit der Verwendung der Mipmapping-Funktionen. Bevor Sie diese Funktionen verwenden, sollten Sie das Mipmapping für die benötigten Textur-Seiten im Texturgruppen-Editor aktiviert haben und/oder Mipmapping für die Sprites aktivieren, die in den allgemeinen Spieloptionen so eingestellt wurden, dass sie nur eine einzige Textur-Seite verwenden. Das folgende Bild zeigt den Unterschied, den Mipmapping beim Rendern Ihres Projekts machen kann:

Sobald Sie Mipmapping für das Projekt oder eine Texturseite aktiviert haben, können Sie die folgenden Funktionen verwenden verwenden, um Werte zu erhalten und zu setzen, die das Aussehen Ihres Projekts bei der Ausführung verändern.

Die unten aufgeführten Funktionen können verwendet werden, um die verschiedenen mit Mipmapping verbundenen Werte in GameMaker Studio 2 zu erhalten:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

Mit den unten aufgeführten Funktionen können Sie die verschiedenen Werte für das Mipmapping in GameMaker Studio 2 einstellen:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
