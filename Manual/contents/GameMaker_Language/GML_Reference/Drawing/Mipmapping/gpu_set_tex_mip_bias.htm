gpu_set_tex_mip_bias

Mit dieser Funktion können Sie den Mipmap-Bias-Wert einstellen, wobei ein Wert von 0 keinem Bias entspricht, 1 der ersten Mipmap, 2 der zweiten Mipmap usw.. Dies steuert die Rate, mit der die Mipmap ausgetauscht wird, und macht gerenderte Texturen im Allgemeinen unschärfer je höher der Wert und je größer die "Entfernung", die betrachtet wird. Beachten Sie, dass diese Funktion auch negative Werte annehmen kann. In diesem Fall werden die gerenderten Texturen über eine größere Entfernung schärfer, je niedriger der Wert ist.

Syntax:

gpu_set_tex_mip_bias(bias);

Argument Beschreibung
Vorspannung Der zu verwendende Mipmap-Bias-Wert (Voreinstellung: 0)

Rückgabe:

N/A

Beispiel:

if gpu_get_tex_mip_bias() != 0
{
gpu_set_tex_mip_bias(0);
}

Der obige Code prüft den aktuellen Mipmap-Bias und wenn er nicht 0 ist, wird er auf 0 gesetzt.
