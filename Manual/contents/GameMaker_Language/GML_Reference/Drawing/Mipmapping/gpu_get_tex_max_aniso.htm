gpu_get_tex_max_aniso

Mit dieser Funktion können Sie den aktuellen Maximalwert der Anisotropie ermitteln, wenn Sie den Filtermodus tf_anisotropic verwenden (siehe gpu_get_tex_mip_filter() für weitere Informationen). Der zurückgegebene Wert liegt liegt zwischen 1 und 16.

Syntax:

gpu_get_tex_max_aniso();

Rückgabe:

Real (Voreinstellung: 16)

Beispiel:

if gpu_get_tex_max_aniso() != 8
{
gpu_set_tex_max_aniso(8);
}

Der obige Code prüft den aktuellen maximalen anisotropen Filterungsgrad und wenn dieser nicht 8 ist, wird er auf 8 gesetzt.
