platz_frei

Sie können diese Funktion verwenden, um zu prüfen, ob die aufrufende Instanz mit einer Instanz kollidieren würde, die in Ihrem Spiel als solide gekennzeichnet ist. Damit dies funktioniert, muss die Instanz, die den Code ausführt, eine gültige Kollisions Maske haben muss (entweder für das Sprite selbst, oder durch die mask_index) und es werden nur Kollisionen mit soliden Instanzen registriert, die auch eine gültige Maske haben.

Die Funktion selbst arbeitet grundsätzlich so, dass sie die Instanz nimmt und auf Kollisionen mit nur soliden markierten Instanzen prüft, wenn sie an der durch die x/y-Argumente angegebenen Position platziert wird. Die Kollisionsprüfung kann entweder präzise sein oder auf dem Begrenzungsrahmen Box der Instanz basieren, je nachdem, welche Art von Kollisionsmaske ausgewählt wurde.

Beachten Sie, dass die angegebenen x/y-Koordinaten vor der Prüfung auf die nächstliegende Ganzzahl heruntergerechnet werden.

Syntax:

place_free(x, y);

Argument Beschreibung
x Die zu prüfende x-Position.
y Die zu prüfende y-Position.

Rückgabe:

Boolesche

Beispiel:

if place_free(mouse_x, mouse_y)
{
x = mouse_x;
y = mouse_y;
}

Der obige Code prüft auf eine Kollision mit einer beliebigen Solid-gekennzeichneten Instanz, wenn die aufrufende Instanz an der gleichen Position wie die Maus platziert werden würde. Wenn keine Kollision erkannt wird, werden die x/y-Koordinaten der Instanz auf die der Maus gesetzt.
