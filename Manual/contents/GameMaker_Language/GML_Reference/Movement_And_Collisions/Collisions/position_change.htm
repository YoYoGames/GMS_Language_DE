position_change

Diese Funktion prüft eine Position auf eine Kollision mit irgendwelchen Instanzen am angegebenen Punkt, und wenn es eine gibt, ändert sie alle Instanzen in Kollision zu Instanzen des gewählten Objekts. Sie können das Argument "perf" auf true setzen, was GameMaker Studio 2 zwingt, die Ereignisse " Zerstören" und "Aufräumen " der gefundenen Instanz sowie das Ereignis " Erzeugen" der neuen Instanz auszuführen, oder false, um keine dieser Ereignisse auszuführen. keines dieser Ereignisse auszuführen. Bitte beachten Sie, dass, wenn Sie die Ereignisse "Zerstören", "Aufräumen" und "Erzeugen" nicht ausführen, jede erzeugte Instanz, die eine normalerweise im Ereignis "Erzeugen" definierte Variable verwendet, das Spiel zum Absturz bringt, da diese Variable nicht mehr existiert.

Syntax:

position_change(x, y, obj, perf);

Argument Beschreibung
x Die x-Koordinate, an der kollidierende Instanzen geändert werden sollen.
y Die y-Koordinate, an der kollidierende Instanzen geändert werden sollen.
obj Das neue Objekt, in das sich das aufrufende Objekt verwandeln soll.
perf Ob das Ereignis "Create" für dieses neue Objekt ausgeführt werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

position_change(200, 200, obj_Bird, true);

Dadurch werden alle Instanzen, die am Punkt (200,200) kollidieren, in eine Instanz von "obj_Bird" umgewandelt, wobei das "obj_Bird"-Ereignis für jede dieser Instanzen ausgeführt wird.
