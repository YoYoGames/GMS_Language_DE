ort_besprechung

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die den Code für die Prüfung ausführt. Wenn Sie dies verwenden, fordern Sie GameMaker Studio 2 effektiv auf, die Instanz an die neue Position zu bewegen, auf eine Kollision zu prüfen, zurückzufahren und Ihnen mitzuteilen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl für die Instanz als auch für das zu prüfende Objekt präzise Kollisionsmasken ausgewählt sind, da sonst nur Bounding-Box-Kollisionen angewendet werden. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other. Bitte beachten Sie, dass Sie, wenn Sie die eindeutige Instanz-ID des Objekts, mit dem kollidiert wird, benötigen, die Funktion verwenden sollten instance_place().

Beachten Sie, dass die angegebenen x/y-Koordinaten auf die nächste Ganzzahl heruntergerechnet werden, bevor die Prüfung durchgeführt wird durchgeführt wird.

Syntax:

place_meeting(x, y, obj);

Argument Beschreibung
x Die zu prüfende x-Position.
y Die zu prüfende y-Position.
obj Die Instanz oder das Objekt, auf das geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if keyboard_check(vk_left)
{
if !place_meeting(x - 5, y, obj_wall) x -=5;
}

Der obige Code prüft, ob es links von der Instanz keine Kollision gibt und verschiebt die Instanz, wenn es keine gibt.
