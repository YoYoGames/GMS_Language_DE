Punkt_im_Rechteck

Wenn Sie diese Funktion verwenden, definieren Sie einen rechteckigen Bereich und GameMaker Studio 2 ermittelt, ob der angegebene Punkt innerhalb seiner Grenzen liegt oder nicht. Wenn der Punkt innerhalb des definierten Rechtecks liegt, gibt die Funktion true zurück, andernfalls gibt die Funktion false zurück.

Syntax:

point_in_rectangle(px, py, x1, y1, x2, y2);

Argument Beschreibung
px Die x-Koordinate des zu prüfenden Punktes.
py Die y-Koordinate des zu prüfenden Punktes.
x1 Die x-Koordinate der linken Seite des zu prüfenden Rechtecks.
y1 Die y-Koordinate der oberen Seite des zu prüfenden Rechtecks.
x2 Die x-Koordinate der rechten Seite des zu prüfenden Rechtecks.
y2 Die y-Koordinate der unteren Seite des zu prüfenden Rechtecks.

Rückgabe:

Boolesche

Beispiel:

if point_in_rectangle(mouse_x, mouse_y, x -10, y - 10, x + 10, y + 10)
{
audio_play_sound(snd_click, 0, false);
}

Dieser kurze Code prüft die Mausposition gegen den definierten rechteckigen Bereich und spielt einen Ton ab, wenn sie innerhalb der Grenzen liegt.
