kollision_kreis

Collision_circle erzeugt eine kreisförmige Zone mit dem vom Benutzer angegebenen Radius um die Punkte x1,y1 und prüft dann diese Zone auf eine Kollision mit einer beliebigen Instanz des durch das Argument "obj" angegebenen Objekts. Diese Prüfung kann entweder präzise oder nicht präzise sein, aber damit präzise Kollisionen aktiviert sind, muss das Objekt oder die Instanz, auf die geprüft wird, auch präzise Kollisionen für ihr Sprite aktiviert haben. Wenn dies nicht der Fall ist, basiert die Standardprüfung auf Bounding Boxes. Das folgende Bild veranschaulicht, wie dies funktioniert:

Hier verwendet die Instanz in der Mitte einen Kollisionskreis, um auf Ball Objekten. Die blauen Objekte haben keine präzise Bounding Box, und wie Sie sehen können, kann die Kollision auch dann stattfinden, wenn das Sprite den Kreis nicht berührt (selbst wenn Sie die Option precise in der Funktion auf true setzen), da die Bounding Box des Sprites den durch collision_circle definierten kreisförmigen Bereich überlappt. Box dieses Sprites den durch collision_circle definierten Kreisbereich überlappt. Andererseits werden die grünen Kugeln nur dann als Kollisionen betrachtet, wenn das aktuelle Sprite den definierten Kreis überlappt. Denken Sie daran, dass für die Berücksichtigung von präzisen Kollisionen müssen sowohl das Objekt-Sprite als auch die Kollisionsfunktion präzise als ein markiert sein. Es sollte auch beachtet werden, dass der Rückgabewert der Funktion die ID einer beliebigen der Instanzen sein kann, die als in Kollision befindlich betrachtet werden.

Beachten Sie, dass Sie anstelle eines Objektindexes auch eine Instanz id angeben können, um nach einer bestimmten Instanz zu suchen, oder die Instanzschlüsselwörter all oder other (je nach Ereignis und aktuellem Codeumfang).

Syntax:

collision_circle(x1, y1, rad, obj, prec, notme);

Argument Beschreibung
x1 Die x-Koordinate des Mittelpunkts des zu prüfenden Kreises.
y1 Die y-Koordinate des Mittelpunkts des zu prüfenden Kreises.
rad Der Radius (Abstand in Pixeln vom Mittelpunkt zum Rand).
obj Das Objekt, das auf Instanzkollisionen geprüft werden soll.
prec Ob die Prüfung auf pixelgenauen Kollisionen (true = langsam) oder allgemein auf der Bounding Box basiert (false = schnell).
notme Ob die aufrufende Instanz, falls relevant, ausgeschlossen werden soll(true) oder nicht(false).

Rückgabe:

Instanz-ID oder keine

Beispiel:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
image_index = 1;
}
sonst image_index = 0;

Der obige Code prüft einen kreisförmigen Bereich mit einem Radius von 20 Pixeln auf eine Kollision mit "obj_Cursor", und wenn es eine gibt, setzt er den image_index des Objekts, das den Code ausführt, auf 1, aber wenn es keine gibt, setzt er den image_index des Objekts auf 0 gesetzt.
