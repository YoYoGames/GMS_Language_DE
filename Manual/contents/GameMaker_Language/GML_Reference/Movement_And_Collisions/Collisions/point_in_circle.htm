Punkt_im_Kreis

Wenn Sie diese Funktion verwenden, definieren Sie einen kreisförmigen Bereich und GameMaker Studio 2 ermittelt, ob der angegebene Punkt innerhalb seiner Grenzen liegt oder nicht. Wenn der Punkt innerhalb des definierten Kreises liegt, gibt die Funktion true zurück, andernfalls gibt die Funktion false zurück.

Syntax:

point_in_circle(px, py, x1, y1, rad);

Argument Beschreibung
px Die x-Koordinate des zu prüfenden Punktes.
py Die y-Koordinate des zu prüfenden Punktes.
x1 Die x-Koordinate des Kreismittelpunkts.
y1 Die y-Koordinate des Kreismittelpunkts.
rad Der Radius des Kreises.

Rückgabe:

Boolesche

Beispiel:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
over = true;
}
sonst
{
over = false;
}

Der obige Code verwendet die Funktion point_in_circle, um zu prüfen, ob die Mausposition in den definierten kreisförmigen Bereich fällt, und setzt die Variable "over" auf true, wenn dies der Fall ist, oder andernfalls auf false.
