platz_leer

Sie können diese Funktion verwenden, um zu prüfen, ob die aufrufende Instanz mit einer anderen Instanz eines Objekts oder mit allen Instanzen in Ihrem Spiel kollidieren würde. Damit diese Kollision korrekt aufgelöst wird, muss die Instanz, die den Code ausführt, eine gültige Kollisions Maske haben (entweder für das Sprite selbst, oder durch die maske_index) und es werden nur Kollisionen mit solchen Instanzen registriert, die ebenfalls eine gültige Maske haben.

Die Funktion prüft, ob es keine Kollisionen gibt, wenn die aufrufende Instanz an einer bestimmten Position platziert wird, und Sie können ein optionales Argument angeben, um die Prüfung zu verfeinern und nur zu prüfen, ob eine Position frei von Kollisionen mit Instanzen des angegebenen Typs ist. Beachten Sie, dass die Prüfung für alle Instanzen innerhalb des Raums durchgeführt wird, wenn keine optionale Objekt-ID angegeben wird. Die Kollisionsprüfung erfolgt entweder präzise oder basierend auf dem Begrenzungsrahmen der Instanz, je nachdem, welche Art von Kollisionsmaske ausgewählt wurde. Maske ausgewählt wurde. Damit präzise Kollisionen korrekt funktionieren, sollten alle Instanzen in der Prüfung präzise Kollisionsmasken haben.

Beachten Sie, dass die angegebenen x/y-Koordinaten vor der Prüfung auf die nächste Ganzzahl heruntergerechnet werden.

Syntax:

place_empty(x, y, [object_id]);

Argument Beschreibung
x Die zu prüfende x-Position.
y Die zu prüfende y-Position.
[object_id] Optional! Das Objekt, gegen das geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
x = mouse_x;
y = mouse_y;
}

Der obige Code prüft auf eine Kollision mit einer beliebigen anderen Instanz des Objekts "obj_Enemy", als ob die aufrufende Instanz an der gleichen Position wie die Maus stehen würde. Wenn keine Kollision festgestellt wird, werden die x/y-Koordinaten der Instanz auf die der Maus gesetzt.
