kollision_ellipse_liste

Diese Funktion ist die gleiche wie die kollision_ellipse() Funktion, nur dass sie nicht nur eine Instanz in Kollision erkennt, sondern mehrere Instanzen auf einmal. Sie geben die x/y-Positionen der oberen Sie geben die x/y-Positionen oben links und unten rechts des zu prüfenden elliptischen Bereichs sowie das zu prüfende Objekt an und können festlegen, dass die Prüfung präzise sein soll (in diesem Fall müssen alle geprüften Instanzen präzise Kollisionsmasken haben) und ob die Prüfung die aufrufende Instanz die aufrufende Instanz einschließen soll oder nicht.

Sie liefern auch eine DS-Liste, so dass die id Werte aller Instanzen, die mit der aufrufenden Instanz kollidieren an das Ende der angegebenen Liste angehängt werden. Sie können die Liste vor dem Aufruf dieser Funktion löschen, so dass sie nur noch Ergebnisse aus diesem Funktionsaufruf enthält. Sie haben auch die Möglichkeit, die Instanzen basierend auf ihren Abständen vom Ursprung des elliptischen Bereichs zu ihren Ursprüngen zu ordnen. Die Funktion gibt die Anzahl der gefundenen Instanzen zurück, oder 0, wenn keine gefunden wurden.

Beachten Sie, dass Sie anstelle eines Objektindex das Instanz-Schlüsselwort all angeben können, um nach allen Instanzen im aktuellen Raum zu suchen, was auch die Instanz umfassen kann, die den Code ausführt (abhängig vom Argument notme ).

Syntax:

collision_ellipse_list(x1, y1, x2, y2, obj, prec, notme, list, ordered);

Argument Beschreibung
x1 Die x-Koordinate der linken Seite der zu prüfenden Ellipse.
y1 Die y-Koordinate der oberen Seite der zu prüfenden Ellipse.
x2 Die x-Koordinate der rechten Seite der zu prüfenden Ellipse.
y2 Die y-Koordinate der unteren Seite der zu prüfenden Ellipse.
obj Das Objekt, das auf Instanzkollisionen geprüft werden soll.
prec Ob die Prüfung auf präzisen Kollisionen(true, was langsamer ist) oder auf der Bounding Box im Allgemeinen(false, schneller) basiert.
notme Ob die aufrufende Instanz, falls relevant, ausgeschlossen werden soll(true) oder nicht(false).
Liste Die DS-Liste, die zum Speichern der IDs kollidierender Instanzen verwendet wird.
bestellt Ob die Liste nach Abstand sortiert werden soll(true) oder nicht(false).

Rückgabe:

Int (Die Anzahl der gefundenen Instanzen, die sich in Kollision befinden)

Beispiel:

var _list = ds_list_create();
var _num = collision_ellipse_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list);

Der obige Code prüft einen elliptischen Bereich 100 Pixel um die aufrufende Instanzposition auf Kollisionen mit Instanzen von "obj_Enemy". Wenn es irgendwelche Kollisionen gibt, wird die zuvor erstellte Liste durchlaufen und jede Instanz, die die an der Kollision beteiligt war, wird zerstört.
