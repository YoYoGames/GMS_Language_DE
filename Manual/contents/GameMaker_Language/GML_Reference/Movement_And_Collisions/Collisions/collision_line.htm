kollisions_linie

Collision_line prüft entlang einer Linie vom Punkt x1,y1 zum Punkt x2,y2 auf eine Kollision mit einer beliebigen Instanz des durch das Argument "obj" angegebenen Objekts. Diese Prüfung kann entweder präzise oder nicht präzise sein, aber damit präzise Kollisionen aktiviert werden, muss für das Objekt oder die Instanz, auf die Sie prüfen, auch präzise Kollisionen für ihr Sprite aktiviert sein. Wenn nicht, basiert die Standardprüfung auf Bounding Boxes. Das folgende Bild veranschaulicht, wie dies funktioniert:

Denken Sie daran, dass für die Berücksichtigung präziser Kollisionen sowohl das Objekt-Sprite als auch die Kollisionsfunktion als präzise markiert sein müssen. Es sollte auch beachtet werden, dass der Rückgabewert der Funktion die ID einer beliebigen Instanz sein kann, die als kollidierend entlang der Linie betrachtet wird, wenn also drei Instanzen die Wenn also drei Instanzen die definierte Linie überlappen, könnte jede ihrer IDs der Rückgabewert der Funktion sein.

Beachten Sie, dass Sie anstelle eines Objektindexes auch eine Instanz id angeben können, um nach einer bestimmten Instanz zu suchen, oder die Instanz-Schlüsselwörter all bzw. other (je nach Ereignis und aktuellem Codeumfang).

Syntax:

collision_line(x1, y1, x2, y2, obj, prec, notme);

Argument Beschreibung
x1 Die x-Koordinate des Beginns der Linie.
y1 Die y-Koordinate des Beginns der Linie.
x2 Die x-Koordinate des Endes der Linie.
y2 Die y-Koordinate des Endes der Linie.
obj Das Objekt, das auf Instanzkollisionen geprüft werden soll.
prec Ob die Prüfung auf präzisen Kollisionen(true, was langsamer ist) oder allgemein auf der Bounding Box basiert(false, schneller).
notme Ob die aufrufende Instanz, falls relevant, ausgeschlossen werden soll(true) oder nicht(false).

Rückgabe:

Instanz-ID oder keine

Beispiel:

with (obj_Enemy)
{
if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

Dieser Code veranlasst alle Instanzen von "obj_Enemy", entlang einer Linie von 100x, 400y bis 100x, 600y auf eine Kollision mit sich selbst zu prüfen, und wenn es eine gibt, werden sie zerstört.
