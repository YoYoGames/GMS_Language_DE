Punkt_im_Dreieck

Wenn Sie diese Funktion verwenden, definieren Sie einen dreieckigen Bereich und GameMaker Studio 2 ermittelt, ob der angegebene Punkt in dessen Grenzen fällt oder nicht. Wenn der Punkt innerhalb des definierten Dreiecks liegt, gibt die Funktion true zurück, ansonsten gibt die Funktion false zurück.

Syntax:

point_in_triangle(px, py, x1, y1, x2, y2, x3, y3);

Argument Beschreibung
px Die x-Koordinate des zu prüfenden Punktes.
py Die y-Koordinate des zu prüfenden Punktes.
x1 Die x-Koordinate der ersten Ecke des zu prüfenden Dreiecks.
y1 Die y-Koordinate der ersten Ecke des zu prüfenden Dreiecks.
x2 Die x-Koordinate der zweiten Ecke des zu prüfenden Dreiecks.
y2 Die y-Koordinate der zweiten Ecke des zu prüfenden Dreiecks.
x3 Die x-Koordinate der dritten Ecke des zu prüfenden Dreiecks.
y3 Die y-Koordinate der dritten Ecke des zu prüfenden Dreiecks.

Rückgabe:

Boolesche

Beispiel:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
{
var x1 = x + lengthdir_x(100, image_angle - 45);
var y1 = y + lengthdir_y(100, image_angle - 45);
var x2 = x + lengthdir_x(100, image_angle + 45);
var y2 = y + lengthdir_y(100, image_angle + 45);
if point_in_triangle(inst.x, inst.y, x, y, x1, y1, x2, y2)
{
can_see = true;
}
}

Der obige Code verwendet die Funktion point_in_triangle als "Sichtkegel", um nach einer Instanz von "obj_player" zu suchen und setzt eine Variable auf true, wenn die x/y-Position des Objekts innerhalb des definierten Dreiecks liegt.
