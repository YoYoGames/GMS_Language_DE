kollisionspunkt_liste

Diese Funktion ist die gleiche wie die kollision_punkt() Funktion, nur dass sie nicht jeweils nur eine Instanz in Kollision erkennt, sondern mehrere Instanzen. Sie geben die x/y-Position des zu prüfenden Punktes zu prüfenden Punktes und Sie können einstellen, dass die Prüfung präzise sein soll (in diesem Fall müssen alle geprüften Instanzen präzise Kollisionsmasken haben) und ob die Prüfung die aufrufende Instanz einschließen soll oder nicht.

Sie liefern auch eine DS-Liste, so dass die id Werte aller Instanzen, die mit der aufrufenden Instanz kollidieren an das Ende der angegebenen Liste angehängt werden. Sie können die Liste vor dem Aufruf dieser Funktion löschen, so dass sie nur noch Ergebnisse aus diesem Funktionsaufruf enthält. Sie haben auch die Möglichkeit, die Instanzen basierend auf ihren Entfernungen vom angegebenen Punkt zu ihren Ursprüngen zu ordnen. Die Funktion gibt die Anzahl der gefundenen Instanzen zurück, oder 0, wenn keine gefunden wurden.

Beachten Sie, dass Sie anstelle eines Objektindex das Instanz-Schlüsselwort all angeben können, um nach allen Instanzen im aktuellen Raum zu suchen, was auch die Instanz umfassen kann, die den Code ausführt (abhängig vom Argument notme ).

Syntax:

collision_point_list(x, y, obj, prec, notme, list, ordered);

Argument Beschreibung
x Die x-Koordinate des zu prüfenden Punktes.
y Die y-Koordinate des zu prüfenden Punktes.
obj Das Objekt, das auf Instanzkollisionen geprüft werden soll.
prec Ob die Prüfung auf präzisen Kollisionen(true, was langsamer ist) oder allgemein auf der Bounding Box basiert(false, schneller).
notme Ob die aufrufende Instanz, falls relevant, ausgeschlossen werden soll(true) oder nicht(false).
Liste Die DS-Liste, die zum Speichern der IDs kollidierender Instanzen verwendet wird.
bestellt Ob die Liste nach Abstand sortiert werden soll(true) oder nicht(false).

Rückgabe:

Integer (Die Anzahl der gefundenen Instanzen, die sich in Kollision befinden)

Beispiel:

var _list = ds_list_create();
var _num = collision_point_list(x, y, obj_Enemy, false, true, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list);

Der obige Code prüft die Position der aufrufenden Instanz auf Kollisionen mit Instanzen von "obj_Enemy". Wenn es irgendwelche Kollisionen gibt, dann wird die vorher erstellte Liste in einer Schleife durchlaufen und jede Instanz, die in der Kollision war, wird zerstört.
