position_meeting

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen. Wenn Sie dies verwenden, prüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Prüfung wird gegen die Bounding Box der Instanz oder gegen die Maske der Instanz durchgeführt, wenn diese Instanz präzise Kollisionen geprüft hat. Wenn Sie die eindeutige Instanz erhalten müssen id wenn das Objekt, mit dem kollidiert, sollten Sie instance_position(). Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und andere.

Syntax:

position_meeting(x, y, obj);

Argument Beschreibung
x Die zu prüfende x-Position.
y Die zu prüfende y-Position.
obj Das Objekt (oder die Instanz-ID, oder die Schlüsselwörter "all" oder "other"), mit dem auf eine Kollision geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if mouse_check_button(mb_left)
{
if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
}

Der obige Code prüft auf die linke Maustaste, und wenn sie gedrückt ist, prüft er die x/y-Position der Maus auf eine Kollision mit einer beliebigen Instanz. Wenn es keine gibt, dann wird eine Instanz von "obj_Wall" erzeugt.
