bewegen_kontakt_fest

Diese Funktion verschiebt die Instanz, die den Code ausführt, um eine festgelegte Anzahl von Pixeln in die angegebene Richtung, bis sie auf eine Instanz mit solid als wahr markiert ist. Sie können Sie können -1 oder 0 für die maxdist verwenden, die standardmäßig 1000px beträgt, d. h.: GameMaker Studio 2 bewegt die Instanz kontinuierlich um 1000 Pixel nach oben, bis sie nicht mehr kollidiert.

Syntax:

move_contact_solid( dir, maxdist );

Argument Beschreibung
dir Die Richtung, in die Sie sich bewegen müssen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ein Standardwert von 1000 Pixel).

Rückgabe:

N/A

Beispiel:

if !place_meeting(x, y + 1, obj_Floor)
{
move_contact_solid(270, -1);
}

Der obige Code prüft unterhalb der Instanz auf eine Kollision mit "obj_Floor", und wenn es keine gibt, dann wird sie nach unten bewegt, bis es eine Kollision gibt oder 1000 Pixel abgedeckt wurden.
