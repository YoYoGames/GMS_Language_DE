move_bounce_solid

Mit dieser Funktion können Sie einer Instanz befehlen, nur an den Instanzen abzuprallen, die innerhalb des Raums als solide markiert sind. Sie können ihr auch sagen, dass sie eine präzise Kollisionsprüfung verwenden soll, wenn diese Funktion aktiviert ist, aber beachten Sie, dass dafür alle Instanzen über präzise Masken aktiviert sein müssen und das Spiel bei vielen Instanzen aufgrund der erforderlichen Verarbeitung stark verlangsamt wird. Dies sollte normalerweise im Step-Ereignis einer Instanz erfolgen, kann aber auch selektiv in einem Kollisionsereignis verwendet werden.

Syntax:

move_bounce_solid(adv);

Argument Beschreibung
adv Ob das erweiterte Bouncing aktiviert werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

move_bounce_solid(false);

Dies ermöglicht ein unpräzises Abprallen von als "solid" gekennzeichneten Instanzen.
