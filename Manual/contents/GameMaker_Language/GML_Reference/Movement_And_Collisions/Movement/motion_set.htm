motion_set

Diese Funktion setzt eine neue Bewegungsrichtung und eine neue Geschwindigkeit für die Instanz, die den Code ausführt. Beachten Sie, dass dies nicht zur aktuellen Geschwindigkeit und Richtung der Instanz hinzugefügt wird (dafür würden Sie motion_add()) sondern erzwingt vielmehr die neuen Einstellungen.

Syntax:

motion_set(dir, speed);

Argument Beschreibung
dir Die neue Richtung.
Geschwindigkeit Die neue Geschwindigkeit.`

Rückgabe:

N/A

Beispiel:

if irandom(9) = 1
{
motion_set(random(360), 1 + random(3));
}

Der obige Code bewirkt, dass die Instanz in zufälligen Intervallen Geschwindigkeit und Richtung ändert.
