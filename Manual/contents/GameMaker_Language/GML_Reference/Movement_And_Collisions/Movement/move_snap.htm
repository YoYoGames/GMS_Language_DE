move_snap

Diese Funktion wird verwendet, um die Instanz an einem Raster einer bestimmten Größe "einzufangen". Sie wird an der nächstgelegenen entsprechenden Position auf dem "unsichtbaren" Raster eingerastet, das die Werte hsnap und vsnap definieren.

Syntax:

move_snap( hsnap, vsnap );

Argument Beschreibung
hsnap Das horizontale Einrasten (die Größe in Pixeln zwischen "Zellen").
vsnap Das vertikale Einrasten (die Größe in Pixeln zwischen "Zellen").

Rückgabe:

N/A

Beispiel:

mit (obj_Stücke)
{
if !place_snapped(32, 32)
{
move_snap(32, 32);
}
}

Der obige Code überprüft alle Instanzen von "obj_Pieces", um zu sehen, ob sie an einem Raster von 32x32 Pixeln gefangen sind, und wenn nicht, fängt er sie an der nächstgelegenen Position in diesem Raster ein.
