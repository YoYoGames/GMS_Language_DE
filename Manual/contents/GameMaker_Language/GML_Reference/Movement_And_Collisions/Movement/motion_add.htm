Bewegung_hinzufügen

Diese Funktion ändert die aktuellen direction und speed der der Instanz, die den Code ausführt, indem sie die angegebenen Werte mit den aktuellen Werten kombiniert. Wenn Sie diese Werte einfach nur ändern möchten, sollten Sie die Funktion motion_set() verwenden.

Syntax:

motion_add(dir, speed);

Argument Beschreibung
dir Die zusätzliche Richtung.
Geschwindigkeit Die zusätzliche Geschwindigkeit.

Rückgabe:

N/A

Beispiel:

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

der obige Code würde im Kollisionsereignis mit einem anderen Objekt aufgerufen werden. Er fügt der Bewegungsrichtung und der Geschwindigkeit der Instanz einen Vektor hinzu, der auf der Position und Geschwindigkeit der anderen an der Kollision beteiligten Instanz basiert. Er begrenzt dann die Geschwindigkeit, wenn sie über 8 (Pixel pro Schritt) liegt.
