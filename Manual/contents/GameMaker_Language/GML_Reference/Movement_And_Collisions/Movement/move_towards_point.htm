bewege_nach_Punkt

Diese Funktion sagt der Instanz, die den Code ausführt, einfach, dass sie sich mit einer bestimmten Geschwindigkeit auf einen bestimmten Punkt zubewegen soll. Beachten Sie, dass es an Ihnen liegt, ihr zu sagen, was sie tun soll, wenn sie am Ziel angekommen ist, da sie nicht einfach von selbst anhält. Das bedeutet, dass Sie eine "vibrierende" Instanz erhalten, wenn sie sich (z. B.) 3 Pixel vorwärts bewegt, die x/y-Zielposition um 2 überschreitet, sich dann wieder 3 Pixel zurück bewegt und den Zielpunkt um eins überschreitet und so weiter... Es gibt eine Reihe von Möglichkeiten, dies zu vermeiden, z. B. mit distance_to_point() um zu sehen, wie weit die Instanz von der angegebenen Position entfernt ist, bevor sie sich bewegt.

HINWEIS: Diese Funktion legt die Geschwindigkeit der Instanz fest. Auch wenn Sie mit diesem Code anhalten, bewegt sich die Instanz weiter in die vorherige Richtung, daher ist es notwendig, die Instanz Geschwindigkeit auf 0 zu setzen, wenn Sie sie anhalten möchten. Siehe das Codebeispiel unten.

Syntax:

move_towards_point( x, y, sp );

Argument Beschreibung
x Die x-Position des Punktes, der angefahren werden soll.
y Die y-Position des Punktes, der angefahren werden soll.
sp Die Geschwindigkeit, mit der die Bewegung erfolgen soll, in Pixeln pro Schritt.

Rückgabe:

N/A

Beispiel:

if point_distance(x, y, ziel.x, ziel.y) > 5
{
move_towards_point(ziel.x, ziel.y, 5);
}
else speed = 0;

Der obige Code bewegt die Instanz mit einer Geschwindigkeit von 5 Pixeln pro Schritt in Richtung der in der Variablen "target" indizierten Position der Instanz. Sobald sie diese Position erreicht hat, wird die Geschwindigkeit auf 0 gesetzt.
