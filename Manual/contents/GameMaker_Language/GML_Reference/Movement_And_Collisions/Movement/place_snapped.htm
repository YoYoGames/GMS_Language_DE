platz_snapped

Mit dieser Funktion können Sie überprüfen, ob der Ursprung einer Instanz (ihre x- und y-Position) an einem Raster mit den von Ihnen angegebenen hsnap- und vsnap-Werten ausgerichtet ist.

Syntax:

place_snapped(hsnap, vsnap);

Argument Beschreibung
hsnap Das horizontale Einrasten zur Kontrolle.
vsnap Das vertikale Einrasten zur Kontrolle.

Rückgabe:

Boolesche

Beispiel:

mit (obj_Stücke)
{
if !place_snapped(32, 32)
{
move_snap(32, 32);
}
}

Der obige Code überprüft alle Instanzen von "obj_Pieces", um zu sehen, ob sie an einem Raster von 32x32 Pixeln gefangen sind, und wenn nicht, fängt er sie an der nächstgelegenen Position in diesem Raster ein.
