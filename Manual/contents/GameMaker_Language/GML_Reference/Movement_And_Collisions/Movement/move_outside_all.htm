verschieben_außen_alle

Mit dieser Funktion können Sie einer Instanz sagen, dass sie sich in eine beliebige Richtung und eine beliebige Anzahl von Pixeln pro Schritt aus einer Kollision herausbewegen soll, wobei ein Wert von -1 oder 0 für maxdist ein Standardwert von 1000px ist, d. h.: GameMaker Studio 2 bewegt die Instanz kontinuierlich um 1000 Pixel nach oben, bis sie sich außerhalb der Kollision befindet.

Syntax:

move_outside_all(dir, maxdist);

Argument Beschreibung
dir Die Richtung, in die Sie sich bewegen müssen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ein Standardwert von 1000 Pixel).

Rückgabe:

N/A

Beispiel:

if place_meeting(x, y, all)
{
move_outside_all(90, 1);
}

Der obige Code verschiebt die Instanz bei jedem Schritt, bei dem sie mit einer anderen Instanz kollidiert, um 1 Pixel nach oben.
