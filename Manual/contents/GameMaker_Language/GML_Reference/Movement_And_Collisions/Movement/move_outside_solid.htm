move_outside_solid

Mit dieser Funktion können Sie einer Instanz sagen, dass sie sich aus einer Kollision mit einer beliebigen Instanz, die als solid in eine beliebige Richtung und eine beliebige Anzahl von Pixeln pro Schritt bewegen, wobei ein Wert von -1 oder 0 für die maxdist ist ein Standardwert von 1000px, d.h.: GameMaker Studio 2 wird die Instanz kontinuierlich um 1000 Pixel nach oben bewegen, bis sie aus der Kollision heraus ist.

Syntax:

move_outside_solid(dir, maxdist);

Argument Beschreibung
dir Die Richtung, in die Sie sich bewegen müssen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ein Standardwert von 1000 Pixel).

Rückgabe:

N/A

Beispiel:

if other.solid
{
var pdir;
pdir = point_direction(other.x, other.y, x, y);
move_outside_solid(pdir, -1);
}

Der obige Code würde in ein Kollisionsereignis gehen und prüfen, ob die "andere" Instanz in der Kollision als "solid" gekennzeichnet ist. Wenn ja, wird die Instanz aus der Kollision herausgeschoben.
