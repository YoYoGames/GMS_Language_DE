move_bounce_all

Mit dieser Funktion können Sie einer Instanz befehlen, an allen Instanzen innerhalb des Raums abzuprallen, mit der einzigen Ausnahme der Instanzen, denen kein Sprite- oder Maskenindex zugewiesen ist. Sie können ihr auch sagen, dass sie eine präzise Kollisionsprüfung verwenden soll, wenn diese aktiviert ist, Beachten Sie jedoch, dass dafür alle Instanzen präzise Masken aktiviert haben müssen und dass dies Ihr Spiel bei vielen Instanzen aufgrund der erforderlichen Verarbeitung stark verlangsamen wird. Dies sollte normalerweise im Step-Event einer Instanz verwendet werden, kann aber auch selektiv in einem Kollisionsereignis verwendet werden, wie im folgenden Codebeispiel dargestellt.

Syntax:

move_bounce_all( adv );

Argument Beschreibung
adv Ob das erweiterte Bouncing aktiviert werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

if other.visible
{
move_bounce_all(false);
}

Der obige Code würde in einem Kollisionsereignis mit einem anderen Objekt platziert werden. Er prüft zunächst, ob das Objekt sichtbar ist, und wenn ja, führt er die Aktion move_bounce_all() aus. Beachten Sie, dass in diesem Fall der Abprall selektiv ist und nur für diese Kollision berechnet wird und nicht für jede Instanz, jeden Schritt.
