move_random

Mit dieser Funktion können Sie die Instanz auf eine beliebige Position im Raum setzen, die jedoch an einem "unsichtbaren" Raster ausgerichtet ist. Ein Wert von 32 für hsnap und vsnap setzt die Instanz also auf eine zufällige Position, die an einem Raster von 32x32 Quadraten ausgerichtet ist. Sie können diese Werte auf 1 setzen, um eine beliebige Position im Raum zu erhalten.

HINWEIS: Diese Funktion rastet bei jedem neuen Durchlauf des Spiels an denselben Positionen ein, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert erzeugt, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden verwenden Sie randomise() beim Start Ihres Spiels. Dies gilt nur für das Testen und Debuggen des Spiels, da das endgültige ausführbare Paket wird dieses Verhalten nicht zeigen und wird bei jedem Spiel zufällig sein.

Syntax:

move_random(hsnap, vsnap);

Argument Beschreibung
hsnap Das horizontale Einrasten (die Größe in Pixeln zwischen "Zellen").
vsnap Das vertikale Einrasten (die Größe in Pixeln zwischen "Zellen").

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_space)
{
move_random(1,1);
}

Dadurch wird die Instanz an eine beliebige Position im Raum bewegt, wenn die Leertaste gedrückt wird.
