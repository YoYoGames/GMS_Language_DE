mp_grid_draw

Diese Funktion zeichnet das angegebene MP-Gitter (wie durch mp_grid_create()), wobei freie Zellen als grün und verbotene Zellen als rot markiert werden. Diese Funktion ist als Debug-Werkzeug unerlässlich, aber es sollte Diese Funktion ist ein wichtiges Debug-Werkzeug, aber es sollte beachtet werden, dass sie sehr langsam ist und nur funktioniert, wenn sie im Draw-Ereignis der Instanz verwendet wird, und dass Sie das Draw-Alpha setzen können, um die Deckkraft des Gitters zu ändern, was es Ihnen erlaubt, es als Overlay zu zeichnen und gleichzeitig zu sehen, was tatsächlich im Raum gleichzeitig zu sehen.

Syntax:

mp_grid_draw(id);

Argument Beschreibung
id Index des mp_grid, das gezeichnet werden soll

Rückgabe:

Boolesche

Beispiel:

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

Der obige Code zeichnet das in der Variablen "grid" indizierte mp_grid als halbtransparentes Overlay (aber nur, wenn die Instanz, die den Code ausführt, eine geringere Tiefe als alle anderen hat).
