mp_potential_step_object

Diese Funktion lässt die Instanz einen Schritt in Richtung einer bestimmten, durch xgoal/ygoal definierten Position machen, während sie gleichzeitig versucht, Hindernissen auszuweichen. Wenn die Instanz auf eine Instanz des durch das Argument "obj" angegebenen Objekts stößt, ändert sie die Bewegungsrichtung, um zu versuchen, diese Instanz zu umgehen und sich um sie herum zu bewegen. Es ist nicht garantiert, dass dieser Ansatz funktioniert, aber in den meisten einfachen Fällen wird er die Instanz effektiv in Richtung des Ziels bewegen. Die Funktion gibt zurück, ob das Ziel erreicht wurde oder nicht.

Syntax:

mp_potential_step_object(xZiel, yZiel, Schrittweite, obj)

Argument Beschreibung
xgoal Die Ziel-X-Position.
yZiel Die Ziel-Y-Position.
Schrittweite Die Geschwindigkeit, mit der sich die Instanz in Pixeln pro Schritt bewegt.
obj Das Objekt, das den Pfad der Instanz, die die Funktion ausführt, blockieren soll. Kann ein Objektindex, eine Instanz-ID oder das spezielle Schlüsselwort,all sein.

Rückgabe:

Boolesche

Beispiel:

if instance_exists(obj_Enemy)
{
var inst;
inst = instance_nearest(x, y, obj_Enemy);
mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

Der obige Code prüft zunächst, ob sich Instanzen von "obj_Enemy" im Raum befinden. Wenn dies der Fall ist, wird die nächstgelegene Instanz gefunden und ihre ID in einer Variablen gespeichert. Diese Variable wird dann verwendet, um mp_potential_step_object anzuweisen, die Instanz die Instanz mit dem Code in Richtung der x/y-Position des gefundenen Objekts mit einer Geschwindigkeit von 5 Pixeln pro Schritt zu bewegen, wobei nur Instanzen des Objekts "obj_Wall" vermieden werden.
