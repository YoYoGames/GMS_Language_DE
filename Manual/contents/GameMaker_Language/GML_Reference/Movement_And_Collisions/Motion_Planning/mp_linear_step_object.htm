mp_linear_step_object

Mit dieser Funktion weisen Sie eine Instanz an, einen "Schritt" zu einem bestimmten Punkt zu machen, der durch die Werte xgoal und ygoal angegeben wird. Die Größe des Schritts (d. h. wie viele Pixel sich die Instanz pro Schritt bewegen soll) wird durch die Schrittgröße angegeben, und wenn die Instanz bereits an der Position ist, bleibt sie stehen und bewegt sich nicht weiter, im Gegensatz zu anderen, einfacheren Funktionen wie move_towards_point(). Die Schrittweite ist auch der Abstand "nach vorne" dass das Objekt bei jedem Schritt auf eine Kollision prüft, und Sie können wählen, ob die Instanz bei einer Kollision mit einem Objekt oder einer Instanz Ihrer Wahl anhält.

Syntax:

mp_linear_step_object(xgoal, ygoal, Schrittweite, obj);

Argument Beschreibung
xgoal Die Ziel-X-Position.
yZiel Die Ziel-Y-Position.
Schrittweite Die Geschwindigkeit, mit der sich die Instanz in Pixeln pro Schritt bewegt.
obj Das Objekt, das den Pfad sperren soll. Kann ein Objektindex, eine Instanz-ID oder das spezielle Schlüsselwort sein, alle

Rückgabe:

Boolesche

Beispiel:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
instance_create_layer(x, y, "Effekte", obj_Explosion);
instance_destroy();
}

Der obige Code lässt das Objekt mit einer Geschwindigkeit von 5 Pixeln pro Schritt auf die Maus zugehen und prüft dabei nur auf Kollisionen mit dem Objekt "obj_Wall". Sobald es die Mausposition erreicht hat, wird es ein Objekt "obj_Explosion" erzeugen und sich selbst zerstören.
