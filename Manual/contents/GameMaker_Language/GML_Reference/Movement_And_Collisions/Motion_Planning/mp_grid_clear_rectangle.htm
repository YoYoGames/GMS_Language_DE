mp_grid_clear_rectangle

Mit dieser Funktion können Sie einen Bereich in Raumkoordinaten definieren, der dann die entsprechenden Zellen im angegebenen MP-Raster löscht. Auch wenn eine Zelle teilweise innerhalb des definierten rechteckigen Bereichs liegt, wird sie gelöscht.

Syntax:

mp_grid_clear_rectangle(id, x1, y1, x2, y2);

Argument Beschreibung
id Index des mp_grid, das verwendet werden soll
x1 Die x-Koordinate der linken Seite des zu prüfenden Rechtecks.
y1 Die y-Koordinate der oberen Seite des zu prüfenden Rechtecks.
x2 Die x-Koordinate der rechten Seite des zu prüfenden Rechtecks.
y2 Die x-Koordinate der unteren Seite des zu prüfenden Rechtecks.

Rückgabe:

Boolesche

Beispiel:

mp_grid_clear_rectangle(grid, 0, 0, 100, 200);

Der obige Code markiert alle Zellen des in der Variablen "grid" indizierten mp_grid als frei, die in den Bereich 0,0 bis 100,200 fallen.
