mp_linearer_Weg_Objekt

Diese Funktion berechnet einen geradlinigen Weg von der aktuellen Instanzposition zu dem durch die Werte "xgoal" und "ygoal" angegebenen Punkt. Sie verwendet die angegebene Schrittweite, die gleiche wie bei der Funktion mp_linear_step(). Der angegebene Pfad muss bereits existieren und wird durch den neuen Pfad überschrieben und die Funktion gibt zurück, ob ein vollständiger Pfad gefunden wurde (true) oder nicht (false). Ein vollständiger Pfad wird nur gefunden, wenn es keine Kollision mit dem angegebenen Objekt oder der Instanz gab und wenn false zurückgegeben wird, dann wird trotzdem ein Pfad erzeugt, der aber nur bis zu der Position läuft, an der der Pfad blockiert wurde.

HINWEIS: Mit dieser Funktion wird die Instanz nicht bewegt. Sie legt nur einen Pfad fest, und Sie müssen die Pfadfunktionen für die Bewegung verwenden.

Syntax:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

Argument Beschreibung
Pfad Der Index des zu verwendenden Pfades
xgoal Die Ziel-X-Position.
yZiel Die Ziel-Y-Position.
Schrittweite Die Geschwindigkeit, mit der sich die Instanz in Pixeln pro Schritt bewegt.
obj Das Objekt, das den Pfad sperren soll. Kann ein Objektindex, eine Instanz-ID oder das spezielle Schlüsselwort,all' sein

Rückgabe:

Boolesche

Beispiel:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
path_start(path, 4, 0, 0);
}

Der obige Code prüft auf eine Kollision mit "obj_Wall" entlang des Pfades zwischen dem Objekt, das den Code ausführt, und der x/y-Position der Maus. Wenn es keine Kollision gibt und der komplette Pfad generiert ist, wird das Objekt entlang des Pfad generiert.
