mp_grid_get_cell

Mit dieser Funktion können Sie jede beliebige Zelle des mp_grid daraufhin überprüfen, ob sie als belegt markiert wurde oder nicht. Dazu geben Sie den Index des (zuvor erstellten) mp_grid und die x- und y-Koordinaten der zu überprüfenden Zelle an. Wenn sie belegt ist oder die zu prüfende Position außerhalb des Rasters liegt, gibt die Funktion den Wert -1 zurück, andernfalls den Wert 0.

Syntax:

mp_grid_get_cell(id, x , y);

Argument Beschreibung
id Index des mp_grid, das verwendet werden soll
x1 Die x-Koordinate des zu prüfenden Gitters.
y1 Die y-Koordinate des zu prüfenden Gitters.

Rückgabe:

Real

Beispiel:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
image_blend = c_red;
}
sonst
{
image_blend = c_lime;
}

Der obige Code prüft die mp_grid-Zelle, die der Mausposition entspricht, und setzt, wenn sie belegt ist, die Variable image_blend auf rot, und wenn sie nicht belegt ist, auf grün.
