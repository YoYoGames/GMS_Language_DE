mp_grid_destroy

Mit dieser Funktion können Sie das angegebene mp_grid zerstören (wie durch die Funktion mp_grid_create()) und geben den ihm zugewiesenen Speicher frei. Es ist wichtig, dass Sie diese Funktion immer dann aufrufen, wenn Sie müssen diese Funktion unbedingt immer dann aufrufen, wenn das MP-Grid fertig ist, da es sonst zu einem Speicherleck kommen kann, was dazu führt, dass Ihr Spiel mit der Zeit langsamer wird und schließlich abstürzt.

HINWEIS: Die Verwendung von mp_grid_*-Funktionen auf ein Grid, nachdem es zerstört wurde, führt zu einem Fehler.

Syntax:

mp_grid_destroy(id);

Argument Beschreibung
id Index des mp_grid, das zerstört werden soll

Rückgabe:

N/A

Beispiel:

if timer = 0
{
mp_grid_destroy(grid);
room_goto(rm_Menu);
}

Der obige Code zerstört das in der Variablen "grid" indizierte mp_grid, wenn die Variable "timer" gleich 0 ist, und geht dann in einen anderen Raum.
