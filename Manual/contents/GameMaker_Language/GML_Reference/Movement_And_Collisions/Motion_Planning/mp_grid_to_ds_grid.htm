mp_grid_to_ds_grid

Mit dieser Funktion können Sie das angegebene MP-Gitter in ein DS-Gitter kopieren. Das DS-Gitter sollte die gleiche Größe haben wie das MP-Gitter, muss es aber nicht (wenn es kleiner ist, gehen Daten verloren, und wenn es größer ist, werden alle zusätzlichen Gitterzellen 0 sein). Die DS-Gitterzellen enthalten den Wert -1, wenn die MP-Gitterzelle als belegt markiert wurde, oder 0, wenn nicht.

Syntax:

mp_grid_to_ds_grid(Quelle, Ziel);

Argument Beschreibung
Quelle Index des mp_grid, das verwendet werden soll
Ziel Index des ds_grid, in das die Grid-Daten kopiert werden sollen.

Rückgabe:

Boolesche

Beispiel:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

Der obige Code erstellt ein neues DS-Grid und kopiert dann die in der Variablen "mp_grid" enthaltenen MP-Grid-Daten in das neue DS-Grid.
