xboxlive_set_savedata_user

Diese Funktion legt fest, dass künftige Dateioperationen, die im Bereich "Save Game" ausgeführt werden (d. h. alle Dateischreiboperationen und Leseoperationen von Dateien, die sich nicht im Bundle-Bereich befinden), mit dem angegebenen Benutzer-ID-Zeiger verknüpft werden. Dies kann so oft aufgerufen werden aufgerufen werden, um Speicherdaten an den entsprechenden Benutzer umzuleiten, oder Sie können die Konstante pointer_null verwenden, um im allgemeinen Maschinenspeicherbereich zu speichern.

Syntax:

xboxlive_set_savedata_user(user_id);

Argument Beschreibung
user_id Die Benutzer-ID (ein Zeiger), die zum Speichern eingestellt werden soll, oder pointer_null

Rückgabe:

N/A

Beispiel:

if (xboxlive_get_savedata_user() != user_id[0])
{
xboxlive_set_savedata_user(user_id[0]);
}

Der obige Code prüft, ob ein Benutzer aktuell als Speicherziel zugewiesen ist, und wenn nicht, wird er zugewiesen.
