xboxlive_get_savedata_user

Diese Funktion gibt den Zeiger für die Benutzer-ID zurück, die derzeit mit dem Speichern von Dateien verbunden ist (oder die Konstante pointer_null, wenn derzeit keine Benutzer-ID verwendet wird). Siehe . xboxlive_set_savedata_user() für weitere Details.

Syntax:

xboxlive_get_savedata_user();

Rückgabe:

Zeiger oder pointer_null

Beispiel:

if (xboxlive_get_savedata_user() != user_id[0])
{
xboxlive_set_savedata_user(user_id[0]);
}

Der obige Code prüft, ob ein Benutzer aktuell als Speicherziel zugewiesen ist, und wenn nicht, wird er zugewiesen.
