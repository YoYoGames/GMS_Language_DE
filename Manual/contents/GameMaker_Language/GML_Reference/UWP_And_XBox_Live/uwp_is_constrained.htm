uwp_is_constrained

Diese Funktion gibt true zurück, wenn das Spiel in einem ressourcenbeschränkten Modus läuft, oder false, wenn es normal läuft. Der ressourcenbeschränkte Modus kann aktiviert werden, wenn das Spiel keinen Fokus hat, und in diesem Modus hat das Spiel reduzierte GPU-Ressourcen. Die Reaktion darauf sollte normalerweise das Anhalten des Spiels sein (da der Benutzer keine Kontrolle mehr hat). Auch das Deaktivieren von besonders belastenden Effekten kann eine gute Idee sein, wenn in diesem Modus eine Verlangsamung beobachtet wird. Beachten Sie, dass im Gegensatz zur uwp_is_suspending() Funktion diese Funktion kontinuierlich true zurückgibt, solange sich das Spiel im eingeschränkten Modus befindet (und nicht nur einmal, wenn es zum ersten Mal passiert).

Syntax:

uwp_is_constrained();

Rückgabe:

Boolesche

Beispiel:

if xboxliveis_constrained()
{
if !instance_exists(obj_PauseMenu)
{
instance_create(0,0,obj_PauseMenu);
}
}

Der obige Code prüft, ob die App eingeschränkt ist, und wenn ja, erstellt er eine Instanz eines Objekts (das zum Anhalten des Spiels verwendet wird).
