xboxlive_get_user

Mit dieser Funktion können Sie den Zeiger der Benutzer-ID für den indizierten Benutzer abrufen. Wenn der Benutzer nicht existiert, gibt die Funktion stattdessen die Konstante pointer_null zurück. Die Anzahl der aktuell angemeldeten Benutzer können Sie mit der Funktion xboxlive_get_user_count().

WICHTIG: Diese Funktion sollte nur mit xboxlive_get_user_count() verwendet werden - verwenden Sie sie nicht mit Gamepads. Verwenden Sie xboxlive_user_for_pad() stattdessen.

Syntax:

xboxlive_get_user(index);

Argument Beschreibung
index Der Index (eine Ganzzahl), von dem die Benutzer-ID abgerufen werden soll.

Rückgabe:

Zeiger

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
}

Die obige Schleife durchläuft alle angemeldeten Benutzer und speichert deren eindeutigen ID-Zeiger in einem Array.
