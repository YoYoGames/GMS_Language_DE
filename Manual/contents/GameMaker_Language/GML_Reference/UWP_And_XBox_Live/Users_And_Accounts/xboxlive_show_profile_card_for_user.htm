xboxlive_profil_karte_zeigen_für_benutzer

Mit dieser Funktion können Sie anfordern, dass die XBox die Profilkarte für den Ziel-Benutzer-ID-Zeiger anzeigt. Die Funktion benötigt sowohl den Benutzer-ID-Zeiger für den Benutzer, der die Informationen anfordert, als auch den Benutzer-ID-Zeiger des Benutzers des Zielbenutzers, dessen Profilkarte sie abrufen möchte.

Syntax:

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

Argument Beschreibung
anfrage_benutzer_id Die Benutzer-ID (ein Zeiger) des anfragenden Benutzers
ziel_benutzer_id Die Benutzer-ID (ein Zeiger) des Benutzers, von dem die Profilkarte abgerufen werden soll

Rückgabe:

N/A

Beispiel:

if (gamepad(0, gp_start))
{
xboxlive_show_profile_card_for_user(user[0], user[1]);
}

Der obige Code prüft auf einen Tastendruck des Gamepads und zeigt, wenn einer erkannt wird, die Profilkarte für den angegebenen Benutzer an.
