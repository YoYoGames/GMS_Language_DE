xboxlive_pad_für_benutzer

Xbox-Benutzer können ihrer Benutzer-ID verschiedene Pads zuweisen, wobei jedes Pad einen "Steckplatz" füllt. Diese Steckplätze sind von 0 an nummeriert und können einen beliebigen Pad-Index enthalten, so dass Steckplatz 0 dem Pad 3 und Steckplatz 1 dem Pad 2 zugewiesen werden könnte. Diese Funktion kann verwendet werden, um das Pad zu ermitteln, das einem bestimmten Steckplatz zugewiesen ist, und ist besonders nützlich, wenn ein Benutzer ein Gamepad für eine bestimmte Zeit ohne Eingabe lässt, da es sich selbst ausschaltet, so dass Sie jede beliebige Benutzer-ID mit dieser Funktion überprüfen, ob der Benutzer ein aktives Gamepad hat, und wenn nicht, entsprechend reagieren, indem Sie z. B. das Spiel pausieren. Wenn das Pad aktiv ist, wird sein Indexwert zurückgegeben, wenn nicht, gibt die Funktion -1 zurück.

Beachten Sie, dass, wenn ein Pad in einem Slot inaktiv wird, spätere Slots in die freie Position "rutschen". Wenn Sie also drei aktive Pads den Slots 0, 1 und 2 zugewiesen haben und das Pad in Slot 0 inaktiv wird, werden die Slots 1 und to nun zu Slots 0 und 1, so dass Steckplatz 0 immer ein aktives Pad enthalten sollte, es sei denn, alle Pads sind ausgeschaltet (in diesem Fall wird er -1 zurückgeben).

Syntax:

xboxlive_pad_for_user(user_id, slot);

Argument Beschreibung
user_id Die zu prüfende Benutzer-ID (ein Zeiger)
user_id Der Index (eine Ganzzahl) des zu prüfenden Gamepad-Steckplatzes

Rückgabe:

Zeiger

Beispiel:

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
for(var i = 0; i < gamepad_get_device_count(); i++;)
{
var u_id = xboxlive_user_for_pad(i);
if (u_id == -1)
{
async_ID = xboxlive_pad_for_user(i, u_id);
}
}
}

Der obige Code prüft den Steckplatz 0 für den gegebenen Benutzer-ID-Zeiger, um zu sehen, ob irgendwelche Gamepads zugewiesen wurden, und wenn nicht, wird eines zugewiesen.
