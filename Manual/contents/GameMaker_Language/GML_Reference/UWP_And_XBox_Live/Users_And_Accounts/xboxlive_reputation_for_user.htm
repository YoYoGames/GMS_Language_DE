xboxlive_reputation_für_benutzer

Mit dieser Funktion können Sie den XBox Live-Reputationswert für den angegebenen Benutzer-ID-Zeiger abrufen.

Syntax:

xboxlive_reputation_for_user(requesting_user_id);

Argument Beschreibung
anfrage_benutzer_id Die Benutzer-ID (ein Zeiger) des zu prüfenden Benutzers

Rückgabe:

Real

Beispiel:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i;)
{
var _uid = xboxlive_get_user(i);
if _uid != pointer_null
{
global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
++a;
}
}

Der obige Code durchläuft die angemeldeten Benutzer in einer Schleife und speichert deren Spieldaten in verschiedenen globalen Arrays.
