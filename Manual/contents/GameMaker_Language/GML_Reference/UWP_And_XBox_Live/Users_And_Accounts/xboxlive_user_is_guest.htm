xboxlive_user_is_guest

Diese Funktion gibt true zurück, wenn der angegebene Benutzer-ID-Zeiger ein Gastbenutzer ist, und false, wenn er es nicht ist. Sie können einen Benutzer-ID-Zeiger mit der Funktion xboxlive_get_user().

Syntax:

xboxlive_user_is_guest(user_id);

Argument Beschreibung
user_id Der ID-Zeiger des zu prüfenden Benutzers.

Rückgabe:

Boolesche

Beispiel:

if xboxlive_user_is_guest(user_id[0])
{
global.user_name[0] = "GAST";
}
sonst
{
global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
}

Das obige speichert den aktivierenden Benutzer-ID-Zeiger in einer globalen Variablen.
