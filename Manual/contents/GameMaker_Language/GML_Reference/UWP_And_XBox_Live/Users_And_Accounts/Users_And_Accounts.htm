Benutzer und Konten

Dieser Abschnitt richtet sich an Benutzer der UWP- oder Xbox One-Plattformen und beschreibt die verfügbaren XBox Live-Funktionen für den Umgang mit Benutzern und Benutzerkonten.

Die Xbox One erlaubt es mehreren Benutzern, sich gleichzeitig an einer Konsole anzumelden. Jedem Benutzer ist ein bestimmter Controller zugeordnet und auch alle Spielstände sind benutzerspezifisch. Um dieses System optimal zu nutzen, führt GameMaker Studio 2 eine Reihe von von Funktionen, die nur für die XBox One verfügbar sind und sich auf die Zuordnung von Controllern und Benutzern beziehen. So können Sie Informationen über den Benutzer abfragen und auch steuern, wie Speicherdaten pro Benutzer behandelt werden. Sie können weitere Informationen für Entwickler über dieses System finden Sie im offiziellen Whitepaper auf der Microsoft-Entwicklerseite.

HINWEIS: Beachten Sie, dass die Benutzer-ID für jede einzelne Person eindeutig ist und über verschiedene Konsolen hinweg beibehalten wird, d. h., wenn sich derselbe Benutzer auf zwei verschiedenen Geräten anmeldet, ist die Benutzer-ID dieselbe.

Die folgenden Funktionen können Sie mit XBox Live-Konten verwenden, sowohl auf der XBox One als auch mit dem regulären UWP-Export (beachten Sie, dass dies die Mindestfunktionalität ist, die für UWP im Rahmen des Creators Program erlaubt ist, wie in diesem Helpdesk-Artikel beschrieben):

  • xboxlive_user_is_signed_in
  • xboxlive_user_is_signing_in
  • xboxlive_gamertag_für_benutzer
  • xboxlive_show_account_picker

Die folgenden Funktionen sind nur gültig, wenn Sie die XBox One über UWP mit einem gültigen ID@XBox-Konto ansteuern:

  • xboxlive_get_user_count
  • xboxlive_get_user
  • xboxlive_get_activating_user
  • xboxlive_user_is_guest
  • xboxlive_benutzer_ist_aktiv
  • xboxlive_user_is_remote
  • xboxlive_user_id_for_user
  • xboxlive_sponsor_für_benutzer
  • xboxlive_set_rich_presence
  • xboxlive_gamedisplayname_für_benutzer
  • xboxlive_benutzer_für_pad
  • xboxlive_pad_für_benutzer
  • xboxlive_pad_count_for_user
  • xboxlive_agegroup_for_user
  • xboxlive_gamerscore_für_benutzer
  • xboxlive_profil_karte_zeigen_für_benutzer
  • xboxlive_reputation_für_benutzer
  • xboxlive_sprite_add_from_gamerpicture
  • xboxlive_generate_player_session_id
