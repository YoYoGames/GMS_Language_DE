xboxlive_benutzer_für_pad

Diese Funktion gibt den Zeiger für die Benutzer-ID zurück, der mit dem angegebenen Gamepad-Indexwert verbunden ist, oder pointer_null, wenn kein Benutzer damit verbunden ist. Beachten Sie, dass diese Funktion nur mit Gamepad-Indizes verwendet werden sollte, und nicht mit xboxlive_get_user_count().

Syntax:

xboxlive_user_for_pad(pad_index);

Argument Beschreibung
pad_index Der Index (eine Ganzzahl) des zu prüfenden Gamepad-Steckplatzes

Rückgabe:

Zeiger

Beispiel:

for(var i = 0; i < 11; ++i;)
{
var u_id = xboxlive_user_for_pad(i);
if u_id != pointer_null
{
pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
}
sonst
{
pad_name[i] = "Press Play";
}
}

Der obige Code durchläuft alle verfügbaren Pad-Slots und prüft, ob ein Benutzer-ID-Zeiger damit verbunden ist. Wenn ja, wird ein Array mit dem Anzeigenamen des Benutzers gesetzt, andernfalls wird das Array auf einen Standardtext gesetzt.
