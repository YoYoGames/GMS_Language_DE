xboxlive_benutzer_ist_aktiv

Mit dieser Funktion können Sie prüfen, ob die angegebene Benutzer-ID in der Liste der Benutzer enthalten ist, die derzeit die Konsole verwenden. Die Funktion gibt true zurück, wenn dies der Fall ist, oder false, wenn nicht. Sie können einen Zeiger auf die Benutzer-ID mit der Funktion xboxlive_get_user()erhalten .

Syntax:

xboxlive_user_is_active(user_id);

Argument Beschreibung
user_id Der ID-Zeiger des zu prüfenden Benutzers.

Rückgabe:

Boolesche

Beispiel:

global.PlayerNum = 0;
global.Player_ID = array_create();
for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
var _uid = xboxlive_get_user(i);
if xboxlive_user_is_active(_uid)
{
global.Player_ID[global.PlayerNum++] = _uid;
}
}

Der obige Code durchläuft die Benutzerkonten in einer Schleife und prüft dann, ob eines von ihnen aktiv ist. Wenn ja, wird ihre Benutzer-ID in ein globales Array eingefügt.
