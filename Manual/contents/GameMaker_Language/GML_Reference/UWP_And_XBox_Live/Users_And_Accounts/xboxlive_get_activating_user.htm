xboxlive_get_activating_user

Mit dieser Funktion können Sie den Benutzer-ID-Zeiger für den Benutzer abrufen, der das Spiel von der Konsole aus gestartet hat.

HINWEIS: Wie Sie diesen Zeiger verwenden, hängt davon ab, was Sie im Spiel tun möchten, aber Sie sollten sich nicht darauf verlassen, dass es sich um die aktuell spielende Benutzer-ID handelt, da sich der aktivierende Benutzer abmelden kann, während das Spiel läuft.

Syntax:

xboxlive_get_activating_user();

Rückgabe:

Zeiger

Beispiel:

global.activUser = xboxlive_get_activating_user();

Das obige speichert den aktivierenden Benutzer-ID-Zeiger in einer globalen Variablen
