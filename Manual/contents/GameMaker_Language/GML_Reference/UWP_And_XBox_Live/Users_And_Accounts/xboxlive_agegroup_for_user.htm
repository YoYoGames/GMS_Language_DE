xboxlive_agegroup_for_user

Diese Funktion gibt eine von vier Konstanten (siehe unten) zurück, um den Altersbereich anzugeben, der dem angegebenen Benutzer-ID-Zeiger zugeordnet ist.

Syntax:

xboxlive_agegroup_for_user(user_id);

Argument Beschreibung
user_id Die zu prüfende Benutzer-ID (ein Zeiger)

Rückgabe:

Konstant

Konstant Beschreibung
xboxlive_agegroup_unbekannt Die Altersgruppe des Benutzers ist unbekannt
xboxlive_agegroup_child Die Altersgruppe der Benutzer liegt zwischen 8 und 12 Jahren (einschließlich)
xboxlive_agegroup_teen Die Altersgruppe der Benutzer liegt zwischen 13 und 17 Jahren (einschließlich, und zwischen 13 und 19 Jahren in Südkorea)
xboxlive_agegroup_adult Die Altersgruppe der Benutzer ist 18 oder älter (bzw. 20 und älter in Südkorea

Beispiel:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
global.ContentControl = true;
}
else global.ContentControl = false;

Der obige Code prüft die Altersgruppe des aktivierenden Benutzers und setzt eine globale Variable abhängig von der zurückgegebenen Konstante auf true oder false.
