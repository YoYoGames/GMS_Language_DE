xboxlive_set_rich_presence

Mit dieser Funktion wird die Rich Presence-Zeichenkette für den angegebenen Benutzer festgelegt. Eine Rich Presence-Zeichenkette zeigt die In-Game-Aktivität eines Benutzers nach dem Namen des Spiels, das der Benutzer gerade spielt, getrennt durch einen Bindestrich. Diese Zeichenfolge wird unter dem Gamertag eines Spielers Gamertag in der Liste "Freunde & Clubs" sowie im Xbox Live-Benutzerprofil des Spielers angezeigt.

Wenn Sie diese Funktion verwenden, müssen Sie den Zeiger für die Benutzer-ID des Benutzers angeben, und dann können Sie den Benutzer als derzeit im Spiel aktiv oder nicht aktiv kennzeichnen (mit true/false). Wenn auf false gesetzt, wird der Rich Presence String mit "/afk" oder etwas entsprechendem angehängt. Das nächste Argument ist die ID der Rich Presence-Zeichenkette, die angezeigt werden soll, und schließlich können Sie (optional) eine service_config_id-Zeichenkette angeben. Beachten Sie, dass dies ein optionales Argument ist, denn wenn Sie aufgerufen haben xboxlive_stats_setup() aufgerufen haben, brauchen Sie die service_config_id hier nicht zu übergeben.

Weitere Informationen zu Rich Presence und zum Einrichten der Strings für die Verwendung im Partnercenter finden Sie in der folgenden Microsoft-Dokumentation:

  • Übersicht der Anwesenheitsstrings

Syntax:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id])

Argument Beschreibung
user_id Der ID-Zeiger des zu prüfenden Benutzers.
ist_benutzer_aktiv Kennzeichnen Sie den Benutzer als aktiv oder nicht.
reich_präsenz_string Die zu verwendende Rich Presence String ID (wie im Partner Center definiert - max. 50 Zeichen)
service_config_id [OPTIONAL] Dies ist die config_id-Zeichenfolge für das Spiel.

Rückgabe:

N/A

Beispiel:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

Der obige Code ruft den Benutzer-ID-Zeiger für den Benutzer ab, der dem Gamepad [0]-Steckplatz zugewiesen ist, und setzt dann die Rich-Present-String für diesen Benutzer.
