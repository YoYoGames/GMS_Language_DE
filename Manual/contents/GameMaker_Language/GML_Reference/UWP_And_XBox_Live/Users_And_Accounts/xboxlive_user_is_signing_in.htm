xboxlive_user_is_signing_in

Die Funktion prüft, ob ein Benutzer dabei ist, sich anzumelden und gibt true zurück, wenn dies der Fall ist, und false, wenn nicht.

Syntax:

xboxlive_user_is_signing_in();

Rückgabe:

Boolesche

Beispiel:

if !xboxlive_user_is_signed_in()
{
if !xboxlive_user_is_signing_in()
{
xboxlive_show_account_picker();
}
}
sonst
{
global.GamerTag = xboxlive_gamertag_for_user();
}

Der obige Code prüft, ob ein Benutzer bei XBox Live angemeldet ist. Wenn dies der Fall ist, ermittelt der Code den Gamertag des Benutzers und speichert ihn in einer globalen Variablen für die zukünftige Verwendung. Wenn er nicht angemeldet ist, prüft der Code, ob er gerade Wenn dies nicht der Fall ist, öffnet der Code die Kontoauswahl und fordert den Benutzer auf, sich anzumelden.
