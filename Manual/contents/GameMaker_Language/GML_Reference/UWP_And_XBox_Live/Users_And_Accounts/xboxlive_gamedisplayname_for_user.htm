xboxlive_gamedisplayname_für_benutzer

Mit dieser Funktion können Sie den Anzeigenamen - als String - aus dem angegebenen Benutzer-ID-Zeiger abrufen. Beachten Sie, dass, wenn der Benutzer lokal ist, der zurückgegebene Wert einfach eine Zeichenkette mit dem Anzeigenamen ist, aber wenn es sich um einen entfernten Benutzer handelt, ist die Zeichenkette sein Gamertag, der dann z. B. zum Stummschalten/Aufheben der Stummschaltung im Voice-Chat verwendet werden kann.

Syntax:

xboxlive_gamedisplayname_for_user(user_id);

Argument Beschreibung
user_id Der ID-Zeiger des zu prüfenden Benutzers.

Rückgabe:

Zeichenfolge

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

Der obige Code ruft den Zeiger der Benutzer-ID für jedes verfügbare Benutzerkonto ab und speichert den Zeiger dann zusammen mit dem Anzeigenamen in zwei Arrays.
