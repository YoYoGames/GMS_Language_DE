uwp_license_trial_user

Diese Funktion gibt den Zeiger der Benutzer-ID zurück, deren Trial-Lizenz gerade verwendet wird, oder sie gibt pointer_null zurück, wenn der Benutzer gerade nicht aktiv ist oder wenn sich das Spiel gerade nicht im Trial-Modus befindet.

Syntax:

uwp_license_trial_user();

Rückgabe:

Zeiger

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
if uwp_license_trial_version()
{
var _uid = uwp_license_trial_user();
uwp_show_help(_uid);
}
}

Der obige Code prüft, ob eine Schaltfläche gedrückt wird und prüft dann, ob die App mit einer Testlizenz läuft. Wenn dies der Fall ist, wird die Benutzer-ID für die Lizenz abgerufen und die Hilfedatei für diesen Benutzer geöffnet.
