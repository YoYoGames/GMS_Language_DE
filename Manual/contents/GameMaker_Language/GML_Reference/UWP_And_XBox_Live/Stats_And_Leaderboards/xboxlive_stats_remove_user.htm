xboxlive_stats_remove_user

Mit dieser Funktion kann ein bestimmter Benutzer aus dem Statistik-Manager entfernt (deregistriert) werden, wobei ein Flush der Statistikdaten an den Live-Server durchgeführt wird. Laut der XBox-Dokumentation muss das Spiel den Benutzer nicht aus dem Statistik-Manager entfernen, da das XBox-Betriebssystem die Statistiken ohnehin regelmäßig flushen wird. Sie geben die Benutzer-ID an, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

HINWEIS: Das Entfernen des Benutzers kann zu einem Fehler führen, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. wenn die Statistiknamen nicht-alphanumerische Zeichen enthalten).

Die Funktion erzeugt einen Callback, der ein asynchrones Systemereignis auslöst. Dieses Ereignis hat die spezielle DS-Map async_load die dann für die folgenden Schlüssel geparst werden sollte:

  • " id" - Enthält die Konstante achievement_stat_event
  • " eventname" - enthält die Zeichenkette " LocalUserRemoved"
  • "userid" - Die mit der Anfrage verbundene Benutzer-ID
  • " error" - 0 bei Erfolg, ein anderer Wert, wenn ein Fehler aufgetreten ist
  • "errormessage" - Eine Zeichenkette mit einer Fehlermeldung, falls eine vorhanden ist

Beachten Sie, dass Sie, wenn Sie die Statistikdaten jederzeit auf den Live-Server spülen wollen, ohne den Benutzer zu entfernen, die Funktion xboxlive_stats_flush_user().

Syntax:

xboxlive_stats_remove_user(user_id);

Argument Beschreibung
user_id Die zu entfernende Benutzer-ID (ein Zeiger)

Rückgabe:

Real

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_remove_user(user_id[i]);
}

Der obige Code durchläuft die gespeicherten Benutzerkonto-IDs und entfernt (hebt die Registrierung auf) jeden Benutzer aus dem Statistikmanager.
