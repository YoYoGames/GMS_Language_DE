xboxlive_achievements_set_progress

Diese Funktion kann verwendet werden, um den Fortschritt eines Achievements zu aktualisieren. Sie liefern die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann den Achievement-String (das ist die Achievement-ID, wie sie im XDP/Windows Dev Center bei der Erstellung vergeben wurde). Schließlich setzen Sie den Fortschrittswert auf einen Wert von 0 bis 100. Beachten Sie, dass das Achievement-System Aktualisierungen ablehnt, die niedriger sind als der aktuelle Fortschrittswert Wert liegen.

Syntax:

xboxlive_achievements_set_progress(user_id, achievement, progress);

Argument Beschreibung
user_id Die Benutzer-ID des Benutzers, für den die Leistung festgelegt werden soll
Leistung Die zu setzende Leistung (eine Zeichenkette)
Fortschritt Der einzustellende Fortschrittswert (von 0 bis 100)

Rückgabe:

N/A

Erweitertes Beispiel:

Im Folgenden finden Sie ein erweitertes Beispiel, wie diese Funktion verwendet werden kann. Zunächst würden Sie sie in einem Ereignis wie Room Start oder Create aufrufen :

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Der obige Code würde aufgerufen werden, um eine Liste aller globalen Leaderboard-Positionen für das Spiel zu erhalten, und erzeugt einen asynchronen Ereignisrückruf, den wir wie folgt behandeln würden:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Der obige Code erstellt einen Prozentwert, der auf einigen globalen Variablen basiert, und verwendet ihn dann, um den Leistungsfortschritt für die Leistung "Game_Completed" festzulegen.
