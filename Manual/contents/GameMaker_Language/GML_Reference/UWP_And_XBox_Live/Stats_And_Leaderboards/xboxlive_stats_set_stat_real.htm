xboxlive_stats_set_stat_real

Diese Funktion kann verwendet werden, um den Wert einer Statistik für die angegebene Benutzer-ID zu setzen. Sie übergeben die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die Statistik-Zeichenkette die gesetzt werden soll(wenn die Statistik-Zeichenkette nicht bereits existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und einen Wert (einen Realwert), auf den die Statistik gesetzt werden soll. Beachten Sie, dass der Statistikname nur alphanumerisch sein darf, ohne Symbole oder Leerzeichen.

Beim Einstellen des Statistikwerts wird jeder vorherige Wert überschrieben, daher liegt es an Ihnen, zu bestimmen, ob der Statistikwert aktualisiert werden soll oder nicht (d. h. zu überprüfen, ob der Highscore tatsächlich der höchste ist), indem Sie den aktuellen Statistikwert mit dem neuen Wert, bevor Sie ihn setzen.

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_real(user_id, stat, value);

Argument Beschreibung
user_id Der Zeiger für die Benutzer-ID, für die der Status gesetzt werden soll
stat Die zu setzende Statistik (eine Zeichenkette)
Wert Der Wert, auf den die Statistik gesetzt werden soll (ein Realwert)

Rückgabe:

Real

Beispiel:

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

Der obige Code prüft eine Variable und wenn diese wahr ist , dann wird der Wert der Statistik "PercentDone" geprüft. Wenn dieser Wert kleiner als 100 ist, wird ein Wert erzeugt und die Statistik auf diesen Wert gesetzt.
