xboxlive_stats_get_stat

Diese Funktion kann verwendet werden, um einen einzelnen Statistikwert aus dem Statistikmanager für einen bestimmten Benutzer abzurufen. Sie übergeben die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user(), zurückgegeben wird, und dann die Statistik-Zeichenkette, wie sie beim Anlegen des Benutzers mit einer der Funktionen xboxlive_stats_set_stat_* definiert wurde. Der Rückgabewert kann entweder ein String oder ein Real sein (abhängig von der zu prüfenden Statistik) oder die GML-Konstante undefined, wenn die Statistik nicht existiert oder ein Fehler aufgetreten ist.

Syntax:

xboxlive_stats_get_stat(user_id, stat);

Argument Beschreibung
user_id Der Benutzer-ID-Zeiger zum Abrufen der Statistiknamen von
stat Die zu setzende Statistik (eine Zeichenkette)

Rückgabe:

Real, String oder undefiniert

Beispiel:

if (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

Der obige Code prüft eine Variable und wenn diese wahr ist , dann wird der Wert der Statistik "PercentDone" geprüft. Wenn dieser Wert kleiner als 100 ist, wird ein Wert erzeugt und die Statistik auf diesen Wert gesetzt.
