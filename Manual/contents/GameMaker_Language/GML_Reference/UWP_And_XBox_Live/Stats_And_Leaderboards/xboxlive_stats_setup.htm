xboxlive_stats_setup

Diese Funktion muss aufgerufen werden, bevor Sie eine der anderen Xbox Live-Statistikfunktionen verwenden können, und initialisiert einfach die erforderlichen Bibliotheken auf dem System. Das Argument "user_id" ist die rohe Benutzer-ID, wie sie von der Funktion xboxlive_get_user(), während "service_config" und "title_id" die eindeutige ID für Ihr Spiel im Xbox Live Dev Center ist.

Syntax:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Argument Beschreibung
user_id Der ID-Zeiger des zu prüfenden Benutzers.
service_config_id Dies ist die config_id-Zeichenkette für das Spiel.
titel_id Die eindeutige ID für Ihr Spiel im Xbox Dev Center.

Rückgabe:

N/A

Beispiel:

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Der obige Code initialisiert das Statistiksystem für die angegebene Benutzer-ID.
