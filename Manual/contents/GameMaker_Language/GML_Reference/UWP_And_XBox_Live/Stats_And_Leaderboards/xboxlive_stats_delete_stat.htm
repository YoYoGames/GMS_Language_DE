xboxlive_stats_delete_stat

Mit dieser Funktion können Sie eine Statistik für die angegebene Benutzer-ID aus dem Statistik-Manager löschen. Sie geben die Benutzer-ID an, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die stat-Zeichenfolge, die gelöscht werden soll. Dadurch wird der Statistikwert gelöscht und aus dem Statistikmanager entfernt, d. h. er wird nicht mehr von den Funktionen xboxlive_stats_get_stat_names() oder zurückgegeben wird. xboxlive_stats_get_stat().

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde

Syntax:

xboxlive_stats_delete_stat(user_id, stat);

Argument Beschreibung
user_id Der Benutzer-ID-Zeiger zum Löschen des Status von
stat Die zu löschende Statistik (eine Zeichenkette)

Rückgabe:

Real

Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

Der obige Code läuft in einer Schleife durch die verbundenen Benutzer und löscht für jeden die angegebene Statistik aus dem Statistikmanager.
