xboxlive_stats_set_stat_string

Diese Funktion kann verwendet werden, um eine Statistik für die angegebene Benutzer-ID auf eine Zeichenkette zu setzen. Sie übergeben die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und dann die Zeichenkette die gesetzt werden soll(wenn die Statistikzeichenkette nicht bereits existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und dann die Zeichenkette, auf die die Statistik gesetzt werden soll. Beachten Sie, dass der Name der Statistik nur alphanumerisch sein darf, ohne Symbole oder Leerzeichen, aber der String Wert, den Sie setzen, kann Leerzeichen und Sonderzeichen enthalten.

Beim Einstellen des Statistikstrings wird jeder vorherige String überschrieben, daher liegt es an Ihnen, zu bestimmen, ob der Statistikstring aktualisiert werden soll oder nicht, indem Sie den aktuellen Statistikstring mit dem neuen vergleichen, bevor Sie ihn einstellen.

Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argument Beschreibung
user_id Der Zeiger für die Benutzer-ID, für die der Status gesetzt werden soll
stat Die zu setzende Statistik (eine Zeichenkette)
Wert Der Wert, auf den die Statistik gesetzt werden soll (eine Zeichenkette)

Rückgabe:

Real

Beispiel:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Der obige Code setzt den Status "LevelName" auf die Zeichenkette, die in der angegebenen globalen Variablen gespeichert ist.
