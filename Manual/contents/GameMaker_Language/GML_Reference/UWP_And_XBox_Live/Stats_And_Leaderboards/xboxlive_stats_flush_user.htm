xboxlive_stats_flush_user

Diese Funktion kann verwendet werden, um die Statistikdaten für einen bestimmten Benutzer aus dem Statistik-Manager auf den Live-Server zu übertragen, um sicherzustellen, dass der Server mit den aktuellen Werten auf dem neuesten Stand ist. Laut XBox-Dokumentation sollten Entwickler darauf achten, diese Funktion nicht Laut XBox-Dokumentation sollten Entwickler darauf achten, diese Funktion nicht zu oft aufzurufen, da die XBox die Anfragen begrenzt und das XBox-Betriebssystem die Statistiken ohnehin automatisch etwa alle 5 Minuten flusht.

HINWEIS: Das Entfernen des Benutzers kann zu einem Fehler führen, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. wenn die Statistiknamen nicht-alphanumerische Zeichen enthalten).

Um die Funktion zu verwenden, geben Sie die Benutzer-ID an, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, und einen Prioritätswert (der 1 für hohe Priorität und 0 für niedrige Priorität ist). Die Funktion gibt -1 zurück, wenn ein Fehler auftrat oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde. Die Funktion erzeugt außerdem einen Callback, der ein asynchrones Systemereignis auslöst. Dieses Ereignis wird die spezielle DS-Map async_load haben, die dann für die folgenden Schlüssel geparst werden sollte:

  • " id" - Enthält die Konstante achievement_stat_event
  • " eventname" - enthält die Zeichenkette " StatisticUpdateComplete"
  • "userid" - Die mit der Anfrage verbundene Benutzer-ID
  • " error" - 0 bei Erfolg, ein anderer Wert, wenn ein Fehler aufgetreten ist
  • "errormessage" - Eine Zeichenkette mit einer Fehlermeldung, falls eine vorhanden ist

Beachten Sie, dass die Statistiken automatisch für Sie geleert werden, wenn Sie einen Benutzer aus dem Statistikmanager entfernen. Siehe xboxlive_stats_remove_user() für weitere Informationen.

Syntax:

xboxlive_stats_flush_user(user_id, priority);

Argument Beschreibung
user_id Die zu verwendende Benutzer-ID (ein Zeiger)
Priorität Der Prioritätswert (0 oder 1)

Rückgabe:

Real

Beispiel:

for(var i = 0; i < array_length(user_id); ++i;)
{
xboxlive_stats_flush_user(user_id[i], 0);
}

Der obige Code durchläuft die gespeicherten Benutzerkonto-IDs in einer Schleife und spült dann die Statistikmanagerdaten für jeden Benutzer.
