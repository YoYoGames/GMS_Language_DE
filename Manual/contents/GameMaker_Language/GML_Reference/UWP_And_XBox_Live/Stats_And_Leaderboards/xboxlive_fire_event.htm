xboxlive_player_leaderboard_lesen

Diese Funktion kann verwendet werden, um ein Statusereignis auszulösen. Das Argument "event_name" ist der Name des auszulösenden Ereignisses, wie er in der XDP-Konsole für Ihr Spiel definiert ist, und die folgenden zusätzlichen Parameter hängen ebenfalls davon ab, was Sie für die Statistik eingerichtet haben für die Statistik. Die Funktion gibt 0 zurück, wenn das Ereignis gesendet wurde (und vom Server empfangen/verarbeitet werden sollte), oder -1, wenn es einen Fehler gab (d. h.: Ihr Ereignis wurde nicht eingerichtet, da die im Projekt enthaltene Ereignis-Manifestdatei eine andere Zahl angibt).

Syntax:

xboxlive_fire_event(event_name, [stat_parameters]);

Argument Beschreibung
Ereignis_name Der Name des Ereignisses, das ausgelöst werden soll.
stat_parameters [OPTIONAL] Zusätzliche Parameter, die dem Ereignis hinzugefügt werden sollen.

Rückgabe:

Zeichenfolge

Beispiel:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

Der obige Code holt sich einige Informationen über den Benutzer und feuert dann ein Stat-Ereignis zusammen mit diesen (und zusätzlichen) Daten ab.
