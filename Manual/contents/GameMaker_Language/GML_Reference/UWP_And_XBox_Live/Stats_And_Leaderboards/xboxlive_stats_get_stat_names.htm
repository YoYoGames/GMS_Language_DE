xboxlive_stats_get_stat_names

Diese Funktion kann verwendet werden, um alle definierten Statistiken aus dem Statistik-Manager für einen bestimmten Benutzer abzurufen. Sie übergeben die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user(), zurückgegeben wird, und die Funktion gibt ein Array von Strings zurück, das die Statistiken für den Benutzer enthält. Wenn ein Fehler auftritt oder der Benutzer keine Statistiken hat, wird das Array trotzdem zurückgegeben, hat aber eine Elementanzahl von 0.

Syntax:

xboxlive_stats_get_stat_names(user_id);

Argument Beschreibung
user_id Der Benutzer-ID-Zeiger zum Abrufen der Statistiknamen von

Rückgabe:

Array

Beispiel:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Der obige Code ruft alle definierten Statistiken für den in der angegebenen Variable gespeicherten Benutzer-ID-Zeiger ab, durchläuft dann das Array und löscht die angegebene Statistik aus dem Statistikmanager.
