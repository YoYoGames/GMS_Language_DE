xboxlive_stats_set_stat_int

Diese Funktion kann verwendet werden, um den Wert einer Statistik für die angegebene Benutzer-ID zu setzen. Sie übergeben die Benutzer-ID, wie sie z. B. von der Funktion xboxlive_get_user()zurückgegeben wird, dann die Statistik-Zeichenkette die gesetzt werden soll(wenn die Statistikzeichenkette nicht bereits existiert, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt) und einen Wert (eine ganze Zahl), auf den die Statistik gesetzt werden soll. Beachten Sie, dass der Statistikname nur alphanumerisch sein darf, ohne Symbole oder Leerzeichen.

Beim Einstellen des Statistikwerts wird jeder vorherige Wert überschrieben, daher liegt es an Ihnen, zu bestimmen, ob der Statistikwert aktualisiert werden soll oder nicht (d. h. zu prüfen, ob der Highscore tatsächlich der höchste ist), indem Sie den aktuellen Statistikwert mit dem neuen Wert, bevor Sie ihn setzen.

Die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen beliebigen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Syntax:

xboxlive_stats_set_stat_int(user_id, stat, value);

Argument Beschreibung
user_id Der Zeiger für die Benutzer-ID, für die der Status gesetzt werden soll
stat Die zu setzende Statistik (eine Zeichenkette)
Wert Der Wert, auf den die Statistik gesetzt werden soll (eine Ganzzahl)

Rückgabe:

Real

Beispiel:

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

Der obige Code ruft den Wert für die Statistik "GamesPlayed" ab und setzt dann dieselbe Statistik auf diesen Wert plus 1.
