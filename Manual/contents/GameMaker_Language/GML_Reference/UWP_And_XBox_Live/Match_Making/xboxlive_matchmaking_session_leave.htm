xboxlive_matchmaking_session_leave

Diese Funktion veranlasst den Benutzer, der die angegebene Sitzung erstellt oder gefunden hat, diese zu verlassen. Sie können den Zeiger auf die Sitzungs-ID aus der async_load DS-Map, die im asynchronen sozialen Ereignis zurückgegeben wird, wenn Sie eine Sitzung erstellt oder gefunden haben (siehe xboxlive_matchmaking_create() für weitere Details).

Syntax:

xboxlive_matchmaking_session_leave(session_id);

Argument Beschreibung
session_id Der Sitzungs-ID-Zeiger der zu verlassenden Sitzung.

Rückgabe:

N/A

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
xboxlive_matchmaking_session_leave(global.SessionID);
}

Der obige Code prüft, ob eine Taste des Gamepads gedrückt wurde, und wenn dies der Fall ist, wird der Benutzer von der jeweiligen Matchmaking-Sitzung abgemeldet.
