xboxlive_matchmaking_create

Diese Funktion erstellt eine Multiplayer-Sitzung unter Verwendung der Sitzungsvorlage, des Matchmaking-Hoppers und der Vorlage für die sichere Gerätezuordnung, die Sie zuvor im XDP-Dashboard erstellt haben sollten. Die Benutzer-ID, die Sie für ein bestimmtes Spiel abrufen können Pad durch Aufruf der xboxlive_user_for_pad() Funktion und Sichtbarkeit eine der folgenden Konstanten sein:

Konstant Beschreibung
xboxlive_match_visibility_open gibt an, dass die Sitzung von den anderen Teilnehmern betreten werden kann.
xboxlive_match_visibility_private legt fest, dass die Sitzung für Benutzer, die keine Sitzungsmitglieder sind, nicht sichtbar ist.
xboxlive_match_visibility_usetemplate gibt an, dass der in der Sitzungsvorlage verwendete Wert verwendet werden soll (dies sollte normalerweise verwendet werden, da Sie den Wert einer Vorlage nicht überschreiben können, wenn einer in XDP festgelegt wurde).

Die Funktion gibt einen eindeutigen Request-ID-Wert zurück, der dann zur Identifizierung des richtigen sozialen asynchronen Ereignisses für diese Funktion verwendet werden kann. Dieses Ereignis wird ausgelöst, wenn die Sitzung erstellt wurde und enthält eine DS-Map in der Variablen async_load mit den folgenden Schlüssel/Wertpaaren:

  • "requestid" - enthält die Anfrage-ID, wie sie von der aufrufenden Funktion zurückgegeben wird
  • "status" - enthält die Zeichenfolge "session_created", um Sie darüber zu informieren, welche Art von Ereignis ausgelöst wurde.
  • "sessionid" - enthält die Sitzungs-ID, oder -1 im Falle eines Fehlers
  • "error" - enthält 0 bei Erfolg, oder -1 im Falle eines Fehlers
  • "correlationid" - das Sitzungshandle für die Korrelation mit der Funktion xboxlive_matchmaking_join_session()

Syntax:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Argument Beschreibung
user_id Der zu verwendende Zeiger für die Benutzer-ID
Sichtbarkeit Eine der oben aufgeführten Konstanten
Vorlage Der Name der Sitzungsvorlage
Trichter Der Name des Matchmaking-Trichters
sdatemplate Der Name der Vorlage für die sichere Gerätezuordnung
matchattributes [OPTIONAL]

Rückgabe:

Anforderungs-ID (Ganzzahl)

Beispiel:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Die obige Funktion ruft die Benutzer-ID für den Benutzer auf dem angegebenen Gamepad ab und initialisiert dann eine Matchmaking-Sitzung für ihn.
