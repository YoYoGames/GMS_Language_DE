xboxlive_matchmaking_find

Diese Funktion sucht nach einer Multiplayer-Sitzung unter Verwendung der Sitzungsvorlage, des Matchmaking-Trichters und der Vorlage für die sichere Gerätezuordnung, die Sie zuvor im XDP-Dashboard erstellt haben sollten. Die Benutzer-ID können Sie für ein bestimmtes Spielfeld durch Aufruf der xboxlive_user_for_pad() Funktion. Die Funktion gibt einen eindeutigen Wert für die Request ID zurück, der dann verwendet werden kann, um das richtige soziale asynchrone Ereignis für diese Funktion zu identifizieren. Dieses Ereignis wird ausgelöst, wenn die Sitzung erstellt wurde, und enthält eine DS-Map in der Variablen async_load mit den folgenden Schlüssel/Wert-Paaren:

  • "requestid" - enthält die Anfrage-ID, wie sie von der aufrufenden Funktion zurückgegeben wird
  • "status" - enthält die Zeichenfolge "session_created", um Sie darüber zu informieren, welche Art von Ereignis ausgelöst wurde.
  • "num_results" - enthält 1 oder mehr, wenn eine Sitzung gefunden wurde, sonst enthält es 0.
  • "sessionid<index>" - enthält die Sitzungs-ID, oder -1 im Falle eines Fehlers (<index/gt; ist ein Wert von 0 bis "num_results" - 1).
  • "sessionOwner<index>" - enthält die ID des Sitzungsleiters, wenn eine Sitzung gefunden wurde (<index> ist ein Wert von 0 bis "num_results" - 1).
  • "error" - enthält 0 bei Erfolg, oder -1 im Falle eines Fehlers

Syntax:

xboxlive_matchmaking_find(user_id, template, hopper, sdatemplate, [matchattributes]);

Argument Beschreibung
user_id Der zu verwendende Zeiger für die Benutzer-ID
Vorlage Der Name der Sitzungsvorlage
Trichter Der Name des Matchmaking-Trichters
sdatemplate Der Name der Vorlage für die sichere Gerätezuordnung
matchattributes [OPTIONAL]

Rückgabe:

Anforderungs-ID (Ganzzahl)

Beispiel:

var userID = xboxone_user_for_pad(global.PadIndex);
result = xboxone_matchmaking_find(userID, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Die obige Funktion ruft die Benutzer-ID für den Benutzer auf dem angegebenen Gamepad ab und versucht dann, eine Matchmaking-Sitzung für ihn zu finden.
