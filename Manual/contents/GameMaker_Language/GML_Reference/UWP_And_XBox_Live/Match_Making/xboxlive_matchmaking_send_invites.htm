xboxlive_matchmaking_send_invites

Diese Funktion ruft den Xbox Live-Systemdialog auf, um einem Benutzer zu ermöglichen, Einladungen zu seinem Spiel zu senden. Sie geben den lokalen Benutzer an, der den Dialog steuern soll (und dessen Freundesliste vom Dialog verwendet werden soll), den eindeutigen Sitzungs-ID-Wert für die Sitzung, zu der Sie Leute einladen möchten, und einen Standardtext, der im Dialogfeld angezeigt werden soll. Sie erhalten die Sitzungs-ID aus der async_load DS-Map, die von der Funktion xboxlive_matchmaking_create() erzeugt wird.

Die Funktion gibt -1 zurück, wenn die Argumente ungültig sind oder der Benutzer nicht bei Xbox Live angemeldet ist, oder andernfalls einen Wert größer als 0.

Syntax:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

Argument Beschreibung
von_benutzer Die lokale Benutzerkennung für die Steuerung des Dialogs.
sitzung_zur_invite_into Die eindeutige Sitzungs-ID für die beizutretende Sitzung.
text_zur_einladung_hinzufügen Ein kurzer Text, der der Einladung hinzugefügt werden kann.

Rückgabe:

Integer

Beispiel:

if gamepad_button_check_pressed(0, gp_face1)
{
var _uid = xboxlive_user_for_pad(0);
xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

Der obige Code prüft auf einen Gamepad-Tastendruck und wenn einer erkannt wird, wird der Xbox Live-Dialog für Sitzungseinladungen geöffnet.
