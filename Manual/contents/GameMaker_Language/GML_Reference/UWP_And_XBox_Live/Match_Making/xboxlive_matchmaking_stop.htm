xboxlive_matchmaking_stop

Diese Funktion kann verwendet werden, um eine Matchmaking-Sitzung für den angegebenen Benutzer zu beenden.

Syntax:

xboxlive_matchmaking_stop(user_id);

Argument Beschreibung
user_id Der zu verwendende Zeiger für die Benutzer-ID

Rückgabe:

N/A

Beispiel:

if !xboxlive_user_is_signed_in(user_id[0])
{
xboxlive_matchmaking_stop(user_id[0]);
}

Mit dem obigen Code wird die Matchmaking-Sitzung für den angegebenen Benutzer beendet.
