uwp_show_help

Diese Funktion startet die Xbox One-Hilfe-App, die wiederum die Hilfedatei Ihres Spiels anzeigt (die eine Datei im HTML-Format ist, die Sie zuvor in das XBox Developer Portal hochgeladen haben). Beachten Sie, dass beim Aufruf dieser Funktion eine Benutzer-ID angegeben wird angegeben wird, damit die Gebietsschema-Informationen des Benutzers bei der Anzeige verwendet werden können. Sie können die Benutzer-ID mit den Funktionen xboxlive_get_user(), xboxlive_get_activating_user(), oder uwp_license_trial_user().

Syntax:

uwp_show_help(user_id);

Argument Beschreibung
user_id Der Zeiger der Benutzer-ID, für den die Hilfedatei geöffnet werden soll.

Rückgabe:

Boolesche

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
if uwp_license_trial_version()
{
var _uid = uwp_license_trial_user;
uwp_show_help(_uid);
}
}

Der obige Code prüft, ob eine Schaltfläche gedrückt wird und prüft dann, ob die App mit einer Testlizenz läuft. Wenn dies der Fall ist, wird die Benutzer-ID für die Lizenz abgerufen und die Hilfedatei für diesen Benutzer geöffnet.
