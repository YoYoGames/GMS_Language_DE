uwp_wurde_beendet

Mit dieser Funktion können Sie herausfinden, ob das Spiel zuvor vom System heruntergefahren wurde (d.h. es ist nicht der erste Durchlauf des Spiels). Wenn dies der Fall ist, gibt die Funktion true zurück, ansonsten false. Ein Beispiel für wäre, zu prüfen, ob das Spiel zuvor vom System beendet wurde, und dann (wenn sie true zurückgibt) die vorherige Phase zu laden, in der sich der Spieler im Spiel befand. Wenn es auf die XBox One abzielt, dann ist das erforderliche Verhalten von Spielen wenn dies erkannt wird, zwischen den verschiedenen Versionen der Zertifizierungsdokumente unterschiedlich sein, daher lohnt es sich, die neueste Version zu überprüfen, um zu sehen, was das Spiel tun sollte.

Syntax:

uwp_was_terminated();

Rückgabe:

Boolesche

Beispiel:

if xboxlivewas_terminated()
{
buffer_load("Player_Save.sav");
global.GameLoaded = true;
}

Der obige Code überprüft, ob die App zuvor vom System beendet wurde, und wenn ja, lädt er einen Puffer und setzt eine globale Variable.
