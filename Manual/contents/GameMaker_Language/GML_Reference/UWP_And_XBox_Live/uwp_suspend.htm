uwp_suspend

Diese Funktion zeigt dem System an, dass die App das Speichern von Daten nach der Erkennung einer Aussetzungsanforderung beendet hat. Der Aufruf dieser Funktion ist eine Voraussetzung für die Übermittlung an Xbox One und sollte immer in Verbindung mit der Funktion uwp_is_suspending()verwendet werden, wie im folgenden Beispiel gezeigt.

Syntax:

uwp_suspend();

Rückgabe:

N/A

Beispiel:

if uwp_is_suspending()
{
scr_Save_Game_Data();
uwp_suspend();
}

Der obige Code prüft, ob die App in den Schwebezustand geht, und wenn ja, ruft er ein Skript auf, um die Spieldaten zu speichern, bevor das Spiel in den Schwebezustand versetzt wird.
