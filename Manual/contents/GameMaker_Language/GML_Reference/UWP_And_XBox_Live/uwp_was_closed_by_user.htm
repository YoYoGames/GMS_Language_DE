uwp_wurde_vom_Benutzer_geschlossen

Mit dieser Funktion können Sie herausfinden, ob das Spiel zuvor vom Benutzer während des Spielens beendet wurde (d. h., es wurde korrekt und nicht vom System beendet). Wenn dies der Fall ist, gibt die Funktion true zurück, ansonsten false.

Syntax:

uwp_was_closed_by_user();

Rückgabe:

Boolesche

Beispiel:

if uwp_was_closed_by_user()
{
global.GameLoaded = false;
}

Der obige Code prüft, ob die App zuvor vom Benutzer beendet wurde, und wenn ja, setzt er eine globale Variable.
