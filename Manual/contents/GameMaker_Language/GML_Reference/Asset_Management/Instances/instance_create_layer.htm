Instanz_erzeugen_Schicht

Mit dieser Funktion können Sie eine neue Instanz des angegebenen Objekts an einem beliebigen Punkt innerhalb des Raums und auf der angegebenen Ebene erzeugen. Die Ebene kann über den Wert der Ebenen-ID identifiziert werden (wie von der Funktion layer_create()) oder durch den Namen der Schicht (als String, z. B. "instance_layer"), wie im Raumeditor definiert .

Diese Funktion gibt die id der neuen Instanz zurück, die dann in einer Variablen gespeichert oder für den Zugriff auf diese Instanz verwendet werden kann. Beachten Sie, dass diese Funktion auch das Ereignis "Create" der zu erstellenden Instanz aufruft , bevor sie mit dem Code oder den Aktionen für das Ereignis fortfährt, das die Funktion aufgerufen hat.

WICHTIG! Es gibt eine minimale und maximale Ebenentiefe von -16000 bis 16000. Alles, was auf einer Ebene außerhalb dieses Bereichs platziert wird , wird nicht gezeichnet, obwohl alle Ereignisse weiterhin normal ablaufen.

Syntax:

instance_create_layer(x, y, layer_id, obj);

Argument Beschreibung
x Die x-Position, an der das Objekt erstellt werden soll
y Die y-Position, an der das Objekt erstellt werden soll
layer_id Die Ebenen-ID (oder der Name), der die erstellte Instanz zugewiesen werden soll
obj Der Objektindex des Objekts, von dem eine Instanz erzeugt werden soll

Rückgabe:

Real (Wert der Instanz-ID)

Beispiel:

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
{
speed = other.shoot_speed;
direction = other.image_angle;
}

Der obige Code erzeugt eine neue Instanz des Objekts "obj_Bullet" und speichert die Instanz-ID in einer Variablen. Diese Variable wird dann verwendet, um der neuen Instanz Geschwindigkeit und Richtung zuzuweisen.
