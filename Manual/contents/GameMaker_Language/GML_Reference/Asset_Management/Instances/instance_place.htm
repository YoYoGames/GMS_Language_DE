Instanz_Ort

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die den Code für die Prüfung ausführt. Wenn Sie dies verwenden, bitten Sie GameMaker Studio 2 effektiv, die Instanz an die neue Position zu verschieben, auf eine Kollision zu prüfen, zurückzufahren und Ihnen mitzuteilen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl für die Instanz als auch für das zu prüfende Objekt präzise Kollisionsmasken sonst werden nur Bounding-Box-Kollisionen angewendet. Diese Funktion gibt die eindeutige Instanz id des kollidierten Objekts zurück, aber wenn das nicht benötigt wird, ist es etwas schneller, die Funktion Funktion place_meeting(). Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und other und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt, oder den eindeutigen Instanz-ID-Wert der gefundenen Instanz, wenn eine Kollision auftritt.

Beachten Sie, dass die angegebenen x/y-Koordinaten auf die nächste Ganzzahl gerundet werden, bevor die Prüfung durchgeführt wird. Wenn Sie dies also nicht benötigen (oder ein älteres GameMaker-Produkt verwenden), sollten Sie die Koordinaten in der Option prüfen:

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Syntax:

instance_place(x, y, obj);

Argument Beschreibung
x Die x-Position, an der nach Instanzen gesucht werden soll.
y Die y-Position, an der nach Instanzen gesucht werden soll.
obj Das Objekt, das auf Instanzen von geprüft werden soll.

Rückgabe:

Real (entweder ein Instanz-ID-Wert oder das Schlüsselwort noone)

Beispiel:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
{
hp -= _inst.dmg;
instance_destroy(_inst);
}

Der obige Code prüft, ob eine Kollision mit Instanzen von "obj_Enemy" vorliegt, und wenn dies der Fall ist, wird die Variable "hp" um den Betrag reduziert, der in der Variable "dmg" der kollidierenden Instanz gespeichert ist, und dann wird die kollidierende Instanz.
