instance_id_get

Mit dieser Funktion können Sie den eindeutigen ID-Wert einer beliebigen Instanz aus der aktuell aktiven Instanzliste abrufen. Sie geben den Index in der Instanzliste an, von dem Sie die ID erhalten möchten, und die Funktion gibt den Wert zum Speichern in einer Variablen zurück.

Syntax:

instance_id_get(index);

Argument Beschreibung
index Der Index innerhalb der Instanzliste von 0 - (Instanzanzahl - 1).

Rückgabe:

Instanz-ID-Wert (real)

Beispiel:

for (var i = 0; i < instance_count; ++i;)
{
var temp_id = instance_id_get(i);
with (temp_id)
{
speed += 0.1;
}
}

Der obige Code geht in einer Schleife durch alle Instanzen innerhalb des Raums und addiert 0,1 zu ihrer Geschwindigkeit.
