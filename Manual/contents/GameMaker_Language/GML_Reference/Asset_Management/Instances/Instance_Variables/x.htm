x

Der x-Wert einer Instanz ist die horizontale Position im aktuellen Raum, gemessen in Pixel. Dieser Wert kann entweder 0, positiv oder minus sein, wobei 0 die linke Seite des Raums ist und eine Bewegung nach rechts x erhöht, eine Bewegung nach links x verringert (ein negativer Wert für x bedeutet, dass die Instanz über die linke Seite des Raums hinausgegangen ist). Sie können x sogar einen realen Wert wie 12,345 geben, aber da Sie 0,345 Pixel nicht erreichen können, scheint sich die Instanz möglicherweise nicht bewegt zu haben, obwohl diese Werte beibehalten werden und intern von GameMaker Studio 2 verwendet werden und vollkommen gültig sind.

Durch Einstellen der x- (und y-) Werte können Sie das Objekt im Raum an die von Ihnen gewünschte Position springen lassen, oder Sie können zu diesen Werten kleinere Beträge addieren und subtrahieren, um die Illusion einer Bewegung zu erzeugen, ohne die eingebauten Geschwindigkeit und Richtung Funktionen. Dies sollte bei der Erstellung Ihrer Spiele berücksichtigt werden, da sich oft eine Art der Bewegung für einen bestimmten Spielstil besser eignet als eine andere.

Syntax:

x;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

Der obige Code verwendet die Pfeiltasten, um zur x-Position zu addieren oder zu subtrahieren und so die Illusion einer Bewegung zu erzeugen.
