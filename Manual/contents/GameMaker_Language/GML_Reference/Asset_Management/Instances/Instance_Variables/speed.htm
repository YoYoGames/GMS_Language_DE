Geschwindigkeit

Alle Instanzen in GameMaker Studio 2 haben bestimmte "eingebaute" Eigenschaften, die Sie verwenden und einstellen können, um zu bestimmen, wie sie aussehen und sich verhalten. speed ist eine dieser Eigenschaften und definiert, wie viele Pixel sich die Instanz pro Schritt bewegt. Anders als hspeed und vGeschwindigkeitist der Geschwindigkeit keine Richtung zugeordnet, da diese durch den Richtung Wert der Instanz bestimmt wird, aber sie kann einen negativen Wert haben. In diesem Fall bewegt sich die Instanz in die entgegengesetzte Richtung zu der mit der Funktion direction eingestellten Richtung (d. h.: direction - 180°). Beachten Sie, dass das Setzen von speed und/oder direction, auch die Werte der Variablen hspeed und vspeed ändern, und dass gravity, gravity_direction und friction alle den Wert dieser Variablen ändern können, wenn sie in Ihren Spielen verwendet werden.

Syntax:

Geschwindigkeit;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Der obige Code verwendet die Pfeiltasten, um die Geschwindigkeit und Richtung der Instanz einzustellen.
