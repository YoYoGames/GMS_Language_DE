yvorherige

Diese eingebaute Variable gibt die vorherige y-Position für die Instanz zurück. Diese Variable wird kurz vor dem Beginn des Beginn-Schritt-Ereignisses auf die aktuelle x-Position gesetzt, kann aber auch jederzeit durch Code gesetzt werden, d. h. Sie können das heißt, Sie können ihr einen eigenen Wert geben, falls dies notwendig sein sollte.

Syntax:

yVorheriges;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if x != xprevious || y != yprevious
{
bewegt = true;
}

Der obige Code prüft, ob eine Differenz zwischen den Werten xprevious und yprevious und den aktuellen x- und y-Werten besteht. Wenn ja, setzt er eine Variable auf true.
