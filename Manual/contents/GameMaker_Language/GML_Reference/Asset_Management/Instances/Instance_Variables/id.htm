id

Diese schreibgeschützte Variable enthält die eindeutige Identifikationsnummer für die Instanz. Jede Instanz, die Sie erstellen - ob durch Code oder durch Hinzufügen zu einem Raum im Raum-Editor - erhält eine Nummer, die intern zur Identifizierung dieser Instanz verwendet wird und die Variable id ist das, was Sie verwenden können, um sie zu referenzieren. Die id wird auch zurückgegeben (und kann in einer Variablen gespeichert werden), wenn eine Instanz mit instance_create_layer() oder erstellt wird. instance_create_depth() erstellt wird, sowie bei anderen Instanzfunktionen.

HINWEIS: Der Wert dieser Variablen ist nicht derselbe wie der Bezeichner, der der Instanz im Raumeditor zugewiesen wird, und unterscheidet sich auch von dem instanz_id Array das die IDs aller derzeit aktiven Instanzen enthält.

Syntax:

id;

Rückgabe:

Real (Ganzzahl - Instanz-ID-Wert)

Beispiel:

for(var i = 0; i < instance_count; i++;)
{
if instance_id[i] != id
{
instance_id[i].scr += 5;
}
}

Der obige Code fügt der Variablen "scr" für jede aktive Instanz im Raum außer der, die den Code ausführt, 5 hinzu. Er tut dies, indem er eine Schleife durch ALLE aktiven Instanzen durchläuft (mit dem Array "instance_id", um jede aktive Instanz zurückzugeben ID) und vergleicht sie mit der eingebauten "id"-Variablen, die die ID der ursprünglichen Instanz ist, die den Code ausführt.
