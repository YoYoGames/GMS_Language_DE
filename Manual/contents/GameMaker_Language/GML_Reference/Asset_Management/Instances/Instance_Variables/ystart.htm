ystart

Diese Variable speichert die anfängliche y-Position der Instanz, wenn sie zum ersten Mal im Raum angelegt wird. Dies ist keine schreibgeschützte Variable und kann sowohl gesetzt als auch gelesen werden.

Syntax:

ystart;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if place_meeting(x, y, obj_spike)
{
score -= 100;
x = xstart;
y = ystart;
}

Der obige Code prüft auf eine Kollision mit einer Instanz von "obj_spike" und wenn es eine gibt, zieht er 100 von der Punktzahl ab und verschiebt die Instanz zurück an ihre Ausgangsposition.
