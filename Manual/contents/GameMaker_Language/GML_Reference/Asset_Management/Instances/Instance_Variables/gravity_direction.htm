Schwerkraft_Richtung

gravity_direction ist eine der "eingebauten" Eigenschaften, die alle Instanzen haben, und kann verwendet werden, um die Bewegungsrichtung festzulegen, wenn die Instanz Schwerkraft größer als 0 ist. Beachten Sie, dass Richtungen in GameMaker Studio 2 normalerweise so berechnet werden, dass 0° rechts, 90° oben, 180° links und 270° unten ist.

Syntax:

Schwerkraft_Richtung;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if place_meeting(x, y, obj_switch)
{
gravity_direction += 180;
}

Der obige Code ändert die gravity_direction, wenn das Player-Objekt auf eine Instanz von "obj_switch" trifft.
