Schwerkraft

DieSchwerkraft ist eine der "eingebauten" Variablen, die alle Instanzen haben und die, wenn sie gesetzt ist, eine konstante Kraft in der Schwerkraft_Richtung der Instanz und beeinflusst sowohl die Instanz Geschwindigkeit und direction . Beachten Sie, dass die Schwerkraft eine kumulative Kraft ist und das Objekt beschleunigt, wenn Sie die Endgeschwindigkeit nicht begrenzen, und es ist üblich, diese Variable auf kleine Dezimalwerte wie 0,01 Werte wie 0,01 setzen. Wenn Sie gravity auf 0 setzen, wird keine Schwerkraft auf die Instanz angewendet (dies ist der Standardwert).

Syntax:

Schwerkraft;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if !place_meeting(x, y + 1, obj_Ground)
{
gravity = 0.01;
}
sonst
{
gravity = 0;
}

Der obige Code wendet die Schwerkraft nur dann an, wenn die Instanz keine Instanzen von "obj_Ground" unter ihr findet.
