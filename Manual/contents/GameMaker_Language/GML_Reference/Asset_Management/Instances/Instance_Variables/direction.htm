Richtung

Alle Instanzen in GameMaker Studio 2 haben bestimmte "eingebaute" Eigenschaften, die Sie verwenden und einstellen können, um zu bestimmen, wie sie aussehen und sich verhalten. Richtung ist eine dieser Eigenschaften und kann verwendet werden, um die Bewegungsrichtung der Instanz festzulegen Instanz festzulegen, wenn die Instanz einen Geschwindigkeit Beachten Sie, dass Richtungen in GameMaker Studio 2 normalerweise so berechnet werden, dass 0° rechts, 90° oben, 180° links und 270° unten ist, und dass die gravity und gravity_direction Variablen den Wert direction ändern können, wenn sie in Ihren Spielen verwendet werden.

Syntax:

Richtung;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Der obige Code ändert die Bewegungsrichtung der Instanz je nachdem, welche Taste (links oder rechts) gedrückt wird.
