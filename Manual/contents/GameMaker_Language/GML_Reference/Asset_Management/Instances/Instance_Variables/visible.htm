sichtbar

Eine Instanz kann als sichtbar oder nicht sichtbar gekennzeichnet werden, indem diese Variable auf true (sichtbar) oder false (unsichtbar) gesetzt wird. Dies funktioniert, indem GameMaker Studio 2 angewiesen wird, das Zeichenereignis für diese Instanz zu überspringen. wenn Sie dies verwenden, da es bedeutet, dass kein Code, der im Draw-Ereignis platziert wird, ausgeführt wird, während die Instanz nicht sichtbar ist. Beachten Sie auch, dass, wenn die Ebene, der die Instanz zugewiesen ist, als unsichtbar gekennzeichnet ist, das Setzen dieser Variablen auf true so lange keine Auswirkung hat, bis die Ebene selbst auch als sichtbar gekennzeichnet ist.

Syntax:

sichtbar;

Rückgabe:

Boolesche

Beispiel:

if other.visible = true
{
x = xprevious;
y = yprevious;
}

Der obige Code prüft bei einem Kollisionsereignis das Sichtbarkeitsflag der "anderen" Instanz und verschiebt , wenn es auf true gesetzt ist , diese Instanz zurück an ihre vorherige Position.
