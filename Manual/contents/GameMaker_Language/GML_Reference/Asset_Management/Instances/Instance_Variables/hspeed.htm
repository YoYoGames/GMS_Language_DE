hspeed

hspeed ist eine der "eingebauten" Eigenschaften, die alle Instanzen haben und definiert die horizontale Bewegungsgeschwindigkeit (entlang der x-Achse) der Instanz in Pixel pro Schritt. Eine hspeed von 3 bedeutet also eine Bewegung von 3 Pixeln nach rechts (+x) pro Schritt, und eine hspeed von -3 würde eine Bewegung um 3 Pixel nach links (-x) pro Schritt bedeuten.

Beachten Sie, dass, wenn Sie die Variablen speed und/oder direction einstellen, der Wert hspeed automatisch aktualisiert wird, um diese Änderungen widerzuspiegeln. Wenn Sie die Variablen hspeed und/oder einstellen, wird der Wert speed automatisch aktualisiert, um diese Änderungen widerzuspiegeln, und eine Änderung des Werts wirkt sich auch auf die Werte und direction aus. Beachten Sie auch, dass gravity, gravity_direction und friction alle den Wert dieser Variablen ändern können, wenn sie in Ihren Spielen verwendet werden.

Syntax:

hGeschwindigkeit;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

Der obige Code ändert die horizontale Geschwindigkeit in Abhängigkeit davon, welche Tasten gedrückt werden.
