vspeed

vspeed ist eine der "eingebauten" Eigenschaften, die alle Instanzen haben und definiert die vertikale Bewegungsgeschwindigkeit (entlang der y-Achse) der Instanz in Pixel pro Schritt. Eine vspeed von 3 bedeutet also eine Bewegung von 3 Pixeln nach unten (+y) pro Schritt, und eine vspeed von -3 würde eine Bewegung von 3 Pixeln nach oben (-y) pro Schritt bedeuten.

Beachten Sie, dass, wenn Sie die Variablen speed und/oder direction einstellen, der Wert vspeed automatisch aktualisiert wird, um diese Änderungen widerzuspiegeln. Wenn Sie die Variablen speed und/oder einstellen, wird der Wert automatisch aktualisiert, um diese Änderungen widerzuspiegeln, und eine Änderung des Werts vspeed wirkt sich auch auf die Werte und direction aus. Beachten Sie auch, dass gravity, gravity_direction und friction alle den Wert dieser Variablen ändern können, wenn sie in Ihren Spielen verwendet werden.

Syntax:

vGeschwindigkeit;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

Der obige Code ändert die vertikale Geschwindigkeit in Abhängigkeit davon, welche Tasten gedrückt werden.
