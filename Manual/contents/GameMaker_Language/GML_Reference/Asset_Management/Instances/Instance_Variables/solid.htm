solide

Eine Instanz kann über die Objekteigenschaften im Objekteditor oder durch Ändern des Wertes dieser integrierten Variablen als solide gekennzeichnet werden. Wenn solid auf true gesetzt ist, dann, Wenn eine Kollision erkannt wird, wird die kollidierende Instanz automatisch an die Position zurückgebracht, an der sie sich im Schritt vor der Kollision befand (und dann werden alle Codes oder Aktionen im Kollisionsereignis ausgeführt). Wenn es auf false gesetzt ist, müssen alle Positionierungen über das Kollisionsereignis abgewickelt werden.

Syntax:

fest;

Rückgabe:

Boolesche

Beispiel:

if other.solid = true
{
x = xprevious;
y = yprevious;
}

Der obige Code prüft das Solid-Flag der "anderen" Instanz bei einer Kollision und wenn es auf true gesetzt ist, wird diese Instanz zurück an ihre vorherige Position bewegt.
