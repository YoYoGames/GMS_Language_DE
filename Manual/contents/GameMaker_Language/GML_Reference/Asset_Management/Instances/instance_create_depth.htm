instance_create_depth

Mit dieser Funktion können Sie eine neue Instanz des angegebenen Objekts an einem beliebigen Punkt innerhalb des Raums und in der angegebenen Tiefe erzeugen. Die Tiefe kann ein beliebiger Wert sein, wobei je niedriger die Tiefe, desto "näher" an der Kamera gezeichnet wird und je höher die Tiefe, desto weiter weg, d. h. eine Instanz mit Tiefe -200 wird über eine Instanz mit Tiefe +300 gezeichnet (zum Beispiel).

Beachten Sie, dass diese Funktion tatsächlich eine Raumebene für die Instanz erzeugt, da alle Instanzen auf einer Ebene im Raum liegen müssen, aber da dies eine verwaltete Ebene ist (d. h. keine, die Sie durch Code oder im Raum erzeugt haben, sondern eine, die GameMaker Studio 2 automatisch angelegt hat), können Sie nicht auf sie zugreifen, und die Schicht Instanzvariable wird -1 zurückgeben.

WICHTIG! Es gibt eine minimale und maximale Ebenentiefe von -16000 bis 16000. Alles, was auf einer Tiefe außerhalb dieses Bereichs platziert wird , wird nicht gezeichnet, obwohl alle Ereignisse weiterhin normal ablaufen.

Diese Funktion liefert die id der neuen Instanz zurück, die dann in einer Variablen gespeichert oder für den Zugriff auf diese Instanz verwendet werden kann. Beachten Sie, dass diese Funktion auch das Ereignis "Create" der zu erstellenden Instanz aufruft , bevor Sie mit dem Code oder den Aktionen für das Ereignis fortfahren, das die Funktion aufgerufen hat.

Syntax:

instance_create_depth(x, y, depth, obj);

Argument Beschreibung
x Die x-Position, an der das Objekt erstellt werden soll
y Die y-Position, an der das Objekt erstellt werden soll
Tiefe Die Tiefe, der die erstellte Instanz zugewiesen werden soll
obj Der Objektindex des Objekts, von dem eine Instanz erzeugt werden soll

Rückgabe:

Real (Wert der Instanz-ID)

Beispiel:

var inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
{
speed = other.shoot_speed;
direction = other.image_angle;
}

Der obige Code erzeugt eine neue Instanz des Objekts "obj_Bullet" und speichert die Instanz-ID in einer Variablen. Diese Variable wird dann verwendet, um der neuen Instanz Geschwindigkeit und Richtung zuzuweisen.
