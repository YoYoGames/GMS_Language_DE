Instanz_aktivieren_Schicht

Mit dieser Funktion können Sie einen Layer aktivieren, der zuvor deaktiviert wurde. Sie müssen die Layer-ID angeben, die entweder der Name des Layers sein kann, wie er im Code-Editor geschrieben wurde (als String), oder der tatsächliche Layer-ID-Wert, wie er von der Funktion von der schicht_erzeugen() und alle deaktivierten Instanzen auf dieser Ebene werden wieder aktiviert. Beachten Sie, dass Sie, wenn Sie eine Ebene deaktiviert haben, die persistente Instanzen hat, die Ebene Wenn Sie eine Ebene deaktiviert haben, die persistente Instanzen enthält, müssen Sie die Ebene mit dieser Funktion erneut aktivieren, bevor Sie den Raum wechseln, da sonst alle persistenten Instanzen nicht übernommen werden und verworfen werden. Beachten Sie auch, dass die Aktivierung nicht sofort erfolgt, und eine Instanz, die auf diese Weise aktiviert wurde Eine Instanz, die auf diese Weise aktiviert wurde, wird erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als aktiv betrachtet.

Syntax:

instance_activate_layer(layer_id);

Argument Beschreibung
layer_id Der zu verwendende Layer-ID-String (oder Wert)

Rückgabe:

N/A

Beispiel:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

Der obige Code aktiviert alle Instanzen innerhalb des Raums und deaktiviert dann diejenigen, die sich außerhalb der Grenzen der aktuellen Kameraansicht befinden, mit Ausnahme der Instanzen auf dem Layer "Player_Layer", die am Ende wieder aktiviert werden.
