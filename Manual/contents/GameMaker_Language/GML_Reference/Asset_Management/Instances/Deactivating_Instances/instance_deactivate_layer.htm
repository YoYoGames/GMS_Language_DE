instance_deactivate_layer

Mit dieser Funktion können Sie alle Instanzen deaktivieren, die einem bestimmten Layer zugeordnet sind. Sie müssen die Layer-ID angeben, die entweder der Name des Layers sein kann, wie er im Code-Editor geschrieben wurde (als String), oder der tatsächliche Wert der Layer-ID, wie er von von der layer_create()und beachten Sie, dass Sie mit dieser Funktion nur Instanzlayer deaktivieren können. Beachten Sie, dass Sie, wenn Sie eine Ebene deaktiviert haben, deren Instanzen von Objekten als persistent gekennzeichnet sind, müssen Sie die Ebene vor dem Raumwechsel mit der Funktion instance_activate_layer() reaktivieren müssen, bevor Sie den Raum wechseln, da sonst alle persistenten Instanzen auf der Ebene nicht übertragen und werden verworfen. Beachten Sie auch, dass die Deaktivierung nicht sofort erfolgt und dass eine Instanz, die auf diese Weise deaktiviert wurde, erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als inaktiv gilt.

HINWEIS: Wenn Sie eine Instanz beim Raumstart deaktivieren (d. h.: über den Raumerstellungscode oder über ein Instanzerstellungsereignis einer Instanz innerhalb des Raums), führen alle Instanzen, die über den Raumeditor im Raum platziert werden, weiterhin ihr Erstellungsereignis aus, bevor sie deaktiviert werden.

WARNUNG: Das Deaktivieren von Instanzen, bei denen die Physik aktiviert ist, stoppt NICHT die Interaktion ihrer Fixtures innerhalb der Physiksimulation. Dafür sollten Sie ihre phy_active Variable auf true oder false setzen, wenn Sie die Instanzen aktivieren/deaktivieren.

Syntax:

instance_deactivate_layer(obj);

Argument Beschreibung
Ebene Der zu verwendende Layer-ID-String (oder Wert)

Rückgabe:

N/A

Beispiel:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Der obige Code deaktiviert alle Instanzen, die dem Layer "Enemy_Layer" zugeordnet sind, und aktiviert dann einen Bereich innerhalb des Raums.
