instance_deactivate_region

Mit dieser Funktion können Sie einen Bereich innerhalb des Raums definieren, um zuvor aktivierte Instanzen zu deaktivieren. Dieser Bereich kann entweder als "innen" oder "außen" gekennzeichnet werden, wie im folgenden Bild gezeigt:

Sie können im obigen Bild sehen, dass die Instanz "Apfel" immer inaktiv ist, denn auch wenn das Sprite selbst die Region nicht überlappt, tut es die Bounding Box. Es wird also davon ausgegangen, dass sich Instanzen innerhalb der angegebenen Region befinden, wenn sich ihr Begrenzungsrahmen mit ihr überschneidet, und der Zustand der Kollisionsmaske (präzise oder nicht) wird nicht berücksichtigt. Beachten Sie, dass die Deaktivierung nicht sofort erfolgt und eine Instanz, die auf diese Weise deaktiviert wurde, erst nach dem Ende des Ereignisses, in dem die Funktion aufgerufen wurde, als inaktiv betrachtet wird aufgerufen wurde.

HINWEIS: Wenn Sie eine Instanz beim Raumstart deaktivieren (d. h.: über den Raumerstellungscode oder über ein Instanzerstellungsereignis einer Instanz innerhalb des Raums), führen alle Instanzen, die über den Raumeditor im Raum platziert werden, weiterhin ihr Erstellungsereignis aus, bevor sie deaktiviert werden.

ACHTUNG: Das Deaktivieren von Instanzen, bei denen die Physik aktiviert ist, verhindert NICHT, dass deren Einbauten in der Physiksimulation interagieren. Dafür sollten Sie ihre phy_active Variable auf true oder false setzen, wenn Sie die Instanzen aktivieren/deaktivieren.

Syntax:

instance_deactivate_region(left, top, width, height, inside, notme);

Argument Beschreibung
links Die x-Koordinate der linken Seite des rechteckigen Bereichs, der deaktiviert werden soll.
oben Die y-Koordinate des oberen Teils des rechteckigen Bereichs, der deaktiviert werden soll.
Breite Die Breite des Bereichs, der deaktiviert werden soll.
Höhe Die Höhe des Bereichs, der deaktiviert werden soll.
innerhalb Ob Instanzen auf der Innenseite der Region (true) oder auf der Außenseite (false) deaktiviert werden sollen.
notme Ob die aufrufende Instanz von der Deaktivierung ausgeschlossen werden soll (true) oder nicht (false).

Rückgabe:

N/A

Beispiel:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Der obige Code aktiviert alle Instanzen und deaktiviert dann einen Bereich innerhalb des Raums.
