Instanz_am-weitesten

Diese Funktion prüft alle Instanzen des angegebenen Objekts, um zu sehen, welche am weitesten vom angegebenen x/y-Punkt entfernt ist. Alle Prüfungen erfolgen von der angegebenen x/y-Position zum Ursprung (der x/y-Position) der Instanzen des angegebenen Objekts. Wenn keine Instanzen des Objekts existieren, gibt die Funktion das Schlüsselwort noone zurück, aber wenn es Instanzen gibt, gibt sie die id der gefundenen Instanz zurück. Bitte beachten Sie, dass die Instanz, die den Code ausführt, in die Prüfung einbezogen wird, wenn sie denselben Objektindex hat wie das zu prüfende Objekt (dies schließt Prüfungen für übergeordnete Objekte ein, wenn die aufrufende Instanz auch ein Kind des übergeordneten Objekts ist).

Syntax:

instance_furthest(x, y, obj);

Argument Beschreibung
x Die x-Position, ab der nach Instanzen gesucht werden soll.
y Die y-Position, an der nach Instanzen fern von gesucht werden soll.
obj Das Objekt, das auf Instanzen von geprüft werden soll.

Rückgabe:

Real (entweder ein Instanz-ID-Wert oder das Schlüsselwort noone)

Beispiel:

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
draw_line(x, y, inst.x, inst.y);
}

Der obige Code findet die am weitesten entfernte Instanz, die den gleichen Objektindex hat wie die Instanz, die den Code ausführt, und speichert die ID in einer Variablen. Diese Variable wird dann daraufhin überprüft, ob sie mit der ID der aufrufenden Instanz übereinstimmt, und wenn nicht nicht, wird eine Linie zwischen den beiden Instanzen gezogen.
