instance_id

Dieses schreibgeschützte Array enthält alle IDs aller aktiven Instanzen innerhalb des Raums. Das bedeutet, dass, wenn Sie eine der Funktionen " Instance Deactivate" verwendet haben, die Instanzen, die deaktiviert wurden, nicht in diesem Array enthalten sind (wenn Sie zuvor einen Wert aus diesem Array verwendet haben, gibt es jetzt das Schlüsselwort "noone" zurück).

Syntax:

instance_id[num]);

Rückgabe:

Real (Wert der Instanz-ID)

Beispiel:

for (var i = 0; i < instance_count; i ++;)
{
with (instance_id[i]) speed += 0.1;
}

Der obige Code geht in einer Schleife durch alle Instanzen innerhalb des Raums und addiert 0,1 zu ihrer Geschwindigkeit.
