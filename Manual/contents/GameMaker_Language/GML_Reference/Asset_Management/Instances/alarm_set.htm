alarm_setzen

Mit dieser Funktion können Sie einen Alarm einstellen. Sie geben die Alarmnummer von 0 bis 11 und dann den Wert an, auf den der Alarm gesetzt werden soll. Der Wert muss ein ganzzahliger Wert sein, und Sie können ihn auf -1 setzen, um den Alarm zu stoppen (nicht ganzzahlige Werte werden auf die nächstliegende Ganzzahl gerundet). Dies ist eine alternative Methode zum direkten Setzen des Alarm-Arrays.

Syntax:

alarm_set(index, wert);

Argument Beschreibung
index Der einzustellende Alarmindex, von 0 bis 11.
Wert Der Wert (eine ganze Zahl), auf den der Alarm eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < 12; i++)
{
if alarm_get(i) > 0 alarm_set(i, -1);
}

Der obige Code prüft alle Alarme in der aufrufenden Instanz, und wenn sie größer als 0 sind, setzt er sie auf -1 und stoppt sie damit, weiter herunterzuzählen.
