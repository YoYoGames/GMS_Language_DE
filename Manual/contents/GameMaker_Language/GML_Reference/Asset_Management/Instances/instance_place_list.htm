instanz_platz_liste

Mit dieser Funktion können Sie eine Position auf eine Kollision mit allen Instanzen eines Objekts prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die den Code für die Prüfung ausführt. Wenn Sie dies verwenden, bitten Sie GameMaker Studio 2, die Instanz an die neue Position zu bewegen, auf eine Kollision zu prüfen, sich dann zurückzubewegen und Ihnen mitzuteilen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert bei präzisen Kollisionen, aber nur, wenn sowohl die Instanz als auch die Instanzen des Objekts, auf das geprüft wird, präzise Kollisionsmasken ausgewählt haben. Masken ausgewählt sind, ansonsten werden nur Bounding-Box-Kollisionen angewendet. Die Funktion füllt eine (zuvor erstellte) DS-Liste mit den eindeutigen id Werten aller Instanzen des Objekts, die als kollidierend mit der aufrufenden Instanz angesehen werden, und Sie haben die Möglichkeit, die Liste basierend auf dem Abstand zwischen dem Ursprung der überprüfenden Instanz und dem Ursprung der Instanzen die als kollidierend angesehen werden. Beachten Sie, dass die Funktion auch das spezielle Schlüsselwort alleakzeptiert. In diesem Fall werden alle Instanzen aufgelistet, bei denen eine Kollision festgestellt wurde. Die Funktion gibt zurück die Anzahl der gefundenen Instanzen, oder 0, wenn keine gefunden wurden.

Beachten Sie, dass die angegebenen x/y-Koordinaten auf die nächste Ganzzahl gerundet werden, bevor die Prüfung durchgeführt wird. Wenn Sie dies also nicht benötigen (oder ein älteres GameMaker-Produkt verwenden), sollten Sie die Koordinaten in der Option prüfen:

instance_place_list(floor(x), floor(y), obj_Enemy, list, true).

Syntax:

instance_place_list(x, y, obj, list, ordered);

Argument Beschreibung
x Die x-Position, an der nach Instanzen gesucht werden soll.
y Die y-Position, an der nach Instanzen gesucht werden soll.
obj Das Objekt, das auf Instanzen von geprüft werden soll.
Liste Die DS-Liste, die zum Speichern der IDs kollidierender Instanzen verwendet wird.
bestellt Ob die Liste nach Abstand sortiert werden soll(true) oder nicht(false).

Rückgabe:

Int (Die Anzahl der gefundenen Instanzen, die sich in Kollision befinden)

Beispiel:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list);

Der obige Code prüft auf eine Kollision mit allen Instanzen, die an der Position der aufrufenden Instanz gefunden werden. Diese werden zu einer DS-Liste hinzugefügt, die dann in einer Schleife durchlaufen wird, um jede der kollidierenden Instanzen zu zerstören.
