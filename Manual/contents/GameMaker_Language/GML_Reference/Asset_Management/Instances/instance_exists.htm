instance_exists

Diese Funktion kann auf zwei Arten verwendet werden, je nachdem, was Sie prüfen möchten. Sie können ihr einen object_index übergeben, auf den sie prüfen soll. In diesem Fall gibt diese Funktion true zurück, wenn aktive Instanzen des angegebenen Objekts im aktuellen Raum vorhanden sind, In diesem Fall gibt die Funktion true zurück, wenn diese bestimmte Instanz im aktuellen Raum existiert und aktiv ist.

Bitte beachten Sie, dass diese Funktion nicht die Instanzen berücksichtigt, die mit den Funktionen zum Deaktivieren von Instanzen deaktiviert wurden.

Syntax:

instance_exists(obj);

Argument Beschreibung
obj Das Objekt oder die Instanz, auf deren Vorhandensein geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if !instance_exists(obj_Enemy)
{
score += 200;
room_goto(rm_hiscores);
}

Der obige Code prüft, ob Instanzen des Objekts "obj_Enemy" vorhanden sind, und wenn nicht, addiert er zur Variablen "score" und ändert den Raum.
