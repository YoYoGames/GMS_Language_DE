Instanz_Position

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder allen Instanzen eines Objekts prüfen. Wenn Sie dies verwenden, prüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Prüfung wird gegen die Bounding Box der Instanz oder gegen die Maske der Instanz durchgeführt, wenn diese Instanz präzise Kollisionen geprüft hat, und gibt die eindeutige Instanz id. Wenn Sie die ID der kollidierenden Instanz nicht benötigen, sollten Sie sollten Sie die Verwendung von position_meeting() stattdessen verwenden. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter alle und other und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt, oder den eindeutigen ID-Wert der gefundenen Instanz, wenn eine Kollision auftritt.

Syntax:

instance_position( x, y, obj );

Argument Beschreibung
x Die x-Position, an der nach Instanzen gesucht werden soll.
y Die y-Position, an der nach Instanzen gesucht werden soll.
obj Das Objekt, das auf Instanzen von geprüft werden soll.

Rückgabe:

Real (entweder ein Instanz-ID-Wert oder das Schlüsselwort noone)

Beispiel:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
with (inst) image_index=1;
instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

Der obige Code prüft auf eine Kollision mit einer Instanz von "obj_Pause_Button" an der Mausposition, und wenn es eine gibt, verwendet er die zurückgegebene id, um seinen image_index auf einen neuen Wert zu setzen, bevor er eine neue Instanz des Objekts "obj_Menu".
