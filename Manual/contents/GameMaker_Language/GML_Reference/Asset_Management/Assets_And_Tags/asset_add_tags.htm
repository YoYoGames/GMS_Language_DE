asset_add_tags

Mit dieser Funktion können Sie eine oder mehrere Tag-Zeichenketten zu jedem Asset aus dem Asset-Browser hinzufügen. Sie geben entweder den Asset-Namen (als Zeichenkette) oder seinen Asset-Index an, sowie entweder eine einzelne Tag-Zeichenkette oder ein Array, in dem jedes Element eine einzelne Tag-Zeichenkette ist. Wenn Sie einen Asset-Indexwert angeben, müssen Sie das optionale Asset-Typ-Argument (eine Konstante) angeben, da Assets unterschiedlichen Typs denselben Index haben können, auch wenn sie nicht denselben Namen haben. Die verfügbaren Kühlstellentypen sind in der folgenden Tabelle aufgeführt in der folgenden Tabelle aufgeführt:

Konstant Beschreibung
Asset_Objekt Der angegebene Name bezieht sich auf ein Objekt.
asset_sprite Der angegebene Name bezieht sich auf ein Sprite.
asset_klang Der Vorname bezieht sich auf einen Klang.
asset_raum Der angegebene Name bezieht sich auf einen Raum.
asset_tiles Der angegebene Name bezieht sich auf einen Kachelsatz.
asset_pfad Der angegebene Name verweist auf einen Pfad.
asset_script Der angegebene Name verweist auf ein Skript.
asset_font Der angegebene Name bezieht sich auf eine Schriftart.
asset_Zeitleiste Der angegebene Name bezieht sich auf eine Zeitlinie.
asset_shader Der Vorname bezieht sich auf einen Shader.
asset_animationcurve Der angegebene Name bezieht sich auf eine Animationskurve.
asset_sequenz Der angegebene Name bezieht sich auf eine Sequenz.

Wenn die Funktion den/die Tag(s) erfolgreich hinzufügen konnte, gibt sie true zurück, andernfalls false.

Syntax:

asset_add_tags(name_oder_index, tags, [asset_type]);

Argument Beschreibung
name_oder_index Der Name des Assets (eine Zeichenfolge) oder sein Indexwert (eine ganze Zahl).
tags Eine einzelne Asset-Tag-Zeichenkette oder ein Array mit verschiedenen Asset-Tags.
[asset_type] OPTIONAL! Der Typ des Assets, auf den die Tags gesetzt werden sollen, wird nur verwendet, wenn ein Index für das erste Argument angegeben wird.

Rückgabe:

Boolesche

Beispiel:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_add_tags(obj_Enemy_Boss_Parent, _a, asset_object);

Der obige Code erstellt ein Array von Tags und fügt sie dann dem angegebenen Objekt hinzu.
