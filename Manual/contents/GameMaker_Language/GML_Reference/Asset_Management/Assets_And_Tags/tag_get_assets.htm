tag_get_assets

Mit dieser Funktion können Sie die Namen aller Assets abrufen, denen das angegebene Tag oder die angegebenen Tags zugewiesen wurden. Sie geben entweder eine einzelne Tag-Zeichenkette oder ein Array an, wobei jedes Element im Array eine Tag-Zeichenkette ist. Die Funktion gibt ein Array zurück, in dem jeder Eintrag der Name (als String) des Assets mit dem angegebenen Tag ist. Wenn Sie den eindeutigen Index für das Asset benötigen, dann können Sie die Funktion asset_get_index() zusammen mit mit dem zurückgegebenen Namen verwenden. Wenn es keine Assets mit dem/den angegebenen Tag(s) gibt, oder wenn ein Fehler auftritt (z. B.: die angegebenen Tags existieren nicht), wird ein leeres Array zurückgegeben.

Syntax:

tag_get_assets(tags);

Argument Beschreibung
tags Eine einzelne Asset-Tag-Zeichenkette oder ein Array mit verschiedenen Asset-Tags.

Rückgabe:

Array

Beispiel:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
if asset_get_type(_assets[i]) == asset_sprite
{
ds_list_add(backgrounds, asset_get_index(_assets[i]));
}
}

Der obige Code erstellt eine Liste und ruft dann die Namen aller Elemente mit dem Tag "background" ab. Er durchläuft das Array der zurückgegebenen Namen in einer Schleife und prüft, ob es sich um Sprite-Assets handelt, und wenn ja, wird der eindeutige Indexwert für das Element zur zukünftigen Verwendung zur Liste hinzugefügt.
