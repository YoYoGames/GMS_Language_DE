asset_get_index

Mit dieser Funktion können Sie den eindeutigen identifizierenden Index für ein Spiel-Asset aus seinem Namen ermitteln. Wenn das Asset nicht gefunden wird, gibt die Funktion den Wert -1 zurück, andernfalls gibt sie die eindeutige Index-ID für das überprüfte Asset zurück. Diese ID kann dann in anderen Funktionen wie jeder andere Indexwert verwendet werden, z. B. der sprite_index oder die pfad_index, zum Beispiel. Bitte beachten Sie, dass Sie, obwohl diese Funktion verwendet werden kann, um Assets aus Strings zu referenzieren (siehe Beispiel unten), immer sicherstellen sollten, dass das Asset existiert, bevor Sie es verwenden, da Sie sonst Fehler erhalten können, die Ihr Spiel zum Absturz bringen.

Syntax:

asset_get_index(name);

Argument Beschreibung
Name Der Name des Spiel-Assets, von dem der Index abgerufen werden soll (eine Zeichenkette).

Rückgabe:

Asset-Index (Integer)

Beispiel:

var obj = asset_get_index("obj_Feind_" + string(global.Level));
if obj > -1
{
instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

Der obige Code ermittelt einen Objektindex aus einer Zeichenkette, und wenn dieser Index existiert, wird eine Instanz des Objekts im Spiel erstellt.
