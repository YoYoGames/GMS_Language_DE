font_replace_sprite_ext

Diese Funktion ersetzt einen Sprite-Font, den Sie zuvor mit der Funktion font_add_sprite_ext(). Sie verwendet einen "Sprite-Streifen", wobei jedes Teilbild ein einzelnes Symbol oder ein Buchstabe ist, und die Bilder werden auf der Grundlage des Arguments "string_map" der Funktion auf Buchstaben abgebildet. Dieses Argument ist eine Zeichenkette, mit der Sie GameMaker Studio 2 mitteilen können, in welcher Reihenfolge die Teilbilder des Sprite-Fonts angeordnet sind, und es wird diese automatisch zuordnen beim Schreiben von Text. Wenn Sie also z. B. eine String-Map von "AaBbCcDdEeFfGgHh" haben, muss Ihr Sprite-Font die Unterbilder in dieser Reihenfolge haben.

Sie können auch den Abstand für die Schrift proportional(true) oder nicht proportional(false) einstellen, wobei eine proportionale Schrift den Abstand auf der Basis der tatsächlichen Breite der Buchstaben hat (so nimmt der Buchstabe "i" weniger Platz ein als der Buchstabe "w", ), während eine nicht-proportionale Schriftart auf der Basis der Breite des Teilbildes platziert wird, wie eine monospaced Schriftart. Schließlich können Sie den Abstand zwischen den einzelnen Buchstaben beim Schreiben definieren, und dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Abstand ist (die Buchstaben werden sich "berühren", wenn sie proportional sind). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dieser in allen weiteren Codes, die sich auf diese Schriftart beziehen, benötigt wird.

Wenn die Funktion fehlschlägt, dann schlägt sie stillschweigend fehl, ohne die angegebene Sprite-Schriftart zu ersetzen.

HINWEIS: Diese Funktion setzt voraus, dass Sie zuvor einen Sprite-Font mit font_add_sprite()hinzugefügt haben, denn sie ersetzt diese und kann nicht verwendet werden auf Schriftarten, die als Teil der Spielressourcen enthalten sind.

Syntax:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argument Beschreibung
Schriftart Die zu ersetzende Schriftart.
spr Das Sprite, auf dessen Basis eine Schrift hinzugefügt werden soll.
string_map String, aus dem die Reihenfolge der Sprite-Subbilder entnommen wird.
Stütze Als Proportionalschrift einstellen oder nicht.
sep Der Abstand, der zwischen den einzelnen Buchstaben gelassen werden soll.

Rückgabe:

N/A

Beispiel:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Der obige Code ersetzt ein Sprite-Font-Asset, das in der Variablen "global.Font" indiziert ist, durch Sprite-Bilder, die in der Variablen "spr_CalcFont" indiziert sind.
