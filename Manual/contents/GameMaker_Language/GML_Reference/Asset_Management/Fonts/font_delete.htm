schrift_löschen

Mit dieser Funktion können Sie ein Schrift-Asset aus dem Spiel löschen. Dies ist eine dauerhafte Entfernung, und ein Raumwechsel oder ein Neustart des Spiels bringt die entfernte Schriftart nicht zurück. Dazu müsste der Spieler das Spiel verlassen und das Spiel neu starten. Seien Sie also vorsichtig, wenn Sie diese Funktion verwenden. Im Allgemeinen wird diese Funktion nur benötigt, um Speicher freizugeben, der von einer Schriftart belegt wurde, die dem Spiel über die Funktionen font_add() oder font_add_sprite().

Syntax:

font_delete(ind);

Argument Beschreibung
ind Der Index der zu löschenden Schriftart.

Rückgabe:

N/A

Beispiel:

font_delete(global.Font);

Der obige Code löscht die in der globalen Variable "Font" indizierte Schriftart.
