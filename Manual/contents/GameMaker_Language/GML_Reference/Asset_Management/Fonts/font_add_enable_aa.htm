font_add_enable_aa

Mit dieser Funktion können Sie Anti-Aliasing (AA) für Schriften aktivieren oder deaktivieren, die mit font_add(). Diese Funktion muss aufgerufen werden , bevor Sie Schriftarten hinzufügen. Wenn Sie sie auf true setzen, wird AA aktiviert, und wenn Sie sie auf false setzen, wird sie deaktiviert. Standardmäßig ist AA aktiviert. Beachten Sie, dass dies keine Auswirkungen auf Schriftarten hat, die vor dem Aufruf der Funktion hinzugefügt wurden. aufgerufen wurde, und die Funktion nur einmal aufgerufen werden muss, wenn die Schriftart hinzugefügt wird, und nicht bei jedem Zeichnen/Schritt, bei dem die Schriftart verwendet wird.

Syntax:

font_add_enable_aa(enable);

Argument Beschreibung
aktivieren. Ob AA für hinzugefügte Schriftarten aktiviert(true) oder deaktiviert(false) werden soll.

Rückgabe:

N/A

Beispiel:

if !font_add_get_enable_aa()
{
font_add_enable_aa(true);
}

Der obige Code prüft den Status von Anti-Aliasing für hinzugefügte Schriftarten und wenn es nicht aktiviert ist, dann aktivieren wir es.
