schrift_get_textur

Diese Funktion gibt einen speziellen Zeiger für die Schrifttextur-Seite zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere im allgemeinen Zeichnen bei Verwendung von Primitiven sowie als auch die Shader Funktionen. Weitere Informationen über die zurückgegebene Texturseite können Sie mit den verschiedenen texture_-Funktionen erhalten, die Sie hier finden.

Syntax:

font_get_texture(Schriftart);

Argument Beschreibung
Schriftart Der Index der zu verwendenden Schriftart.

Rückgabe:

Zeiger (Texturseiten-ID)

Beispiel:

tex = font_get_texture(fnt_Main);

Der obige Code erhält den Texturzeiger für die als "fnt_Main" indizierte Schriftart.
