Schriftarten

GameMaker Studio 2 verfügt über eine Reihe von Funktionen, die verwendet werden können, um Informationen über die enthaltenen Schriftarten-Assets zu erhalten, sowie über eine weitere Reihe von Funktionen, mit denen Sie zur Laufzeit Ihres Spiels Schriftarten ändern, hinzufügen oder entfernen können. Insbesondere die Funktion font_add() kann verwendet werden, um Nicht-Standard-Schriften, wie z. B. asiatische Glyphen-Schriften, aus einer *.ttf- oder *.otf-Datei hinzuzufügen.

Es ist erwähnenswert, dass, wenn Sie eine Schriftart zu Ihrem Spiel hinzufügen, die nicht über die Glyphen verfügt, die für den zu schreibenden Text erforderlich sind, das Unicode-Zeichen 9647 (▯) verwendet wird, um diese fehlenden Glyphen beim Rendern im Draw-Event zu ersetzen. Also Wenn Ihre Schriftart z. B. nicht über das Symbol º verfügt, wird beim Schreiben von 90º tatsächlich 90▯ erzeugt.

Die folgenden Funktionen können verwendet werden, um Informationen über Schriftartenressourcen zu erhalten:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • schrift_get_textur
  • schrift_get_uvs
  • font_get_info

Die folgenden Funktionen können verwendet werden, um Schriftarten während eines Spiels zu ändern und zu manipulieren:

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • Schrift_hinzufügen
  • font_add_sprite
  • font_add_sprite_ext
  • schrift_ersetzen
  • font_replace_sprite
  • font_replace_sprite_ext
  • schrift_löschen
  • font_texture_page_size
