font_get_info

Diese Funktion wird verwendet, um Informationen für die angegebene Schriftart abzurufen. Sie geben einen Schriftindex an (der ein über den Asset-Browser hinzugefügtes Asset oder eine zur Laufzeit hinzugefügte Schrift sein kann) und die Funktion gibt eine Struktur mit den folgenden Variablen zurück:

Variable Name Daten Typ Beschreibung
ascenderOffset real Der maximale Versatz von der Grundlinie zum oberen Rand der Schrift (in Pixel)
Größe real Die ungefähre Größe der Schrift (in Pixel)
spriteIndex Vermögenswert (real) Der Sprite-Index für die Schrift, wenn sie aus einem Sprite erstellt wurde, sonst -1
Textur Vermögenswert (real) -1, wenn der Font aus einem Sprite erstellt wurde, sonst die Textur-ID des Fonts
Name String Der Name der Schriftart
fett boolean true, wenn die Schrift fett ist, sonst false (siehe HINWEIS unten)
kursiv boolean true, wenn die Schrift kursiv ist, sonst false (siehe HINWEIS unten)
Glyphen struct Eine Struktur, die Informationen für jede Glyphe im Font enthält (weitere Informationen finden Sie unten)

HINWEIS: Die fett unditalic-Variablen spiegeln nur die Einstellungen des Benutzers für die Schriftart wider, so dass sie für Schriftarten, die standardmäßig fett oder kursiv sind, möglicherweise nicht genau sind.

Wenn die übergebene Schriftart nicht existiert, gibt die Funktion undefiniert zurück .

Beachten Sie auch, dass das Ändern der Werte einer dieser Variablen oder einer in der Glyphs-Struktur enthaltenen Variablen die Darstellung der Schriftart nicht ändert, sodass diese Informationen als schreibgeschützt betrachtet werden sollten.

Glyphen Struktur

Die Variable " glyphs" in der zurückgegebenen Struktur ist eine eigenständige Struktur, die Informationen für jede im Font enthaltene Glyphe enthält. Jede Variable in dieser Struktur ist der Zeichenname einer Glyphe, die eine Struktur enthält, die Informationen für diese Glyphe enthält. Sie können sich diese verschachtelten Structs in folgendem Format vorstellen (ausgehend von der Haupt-Struct):

{
glyphs:
{
A: {},
B: {},
C: {},
// ...andere Glyphen
},
// ...andere Fontinformationen
}

Sie können den Zeichennamen einer Glyphe verwenden, um ihre Informationen aus der Struktur abzurufen, wie z. B. im folgenden Code:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ ist ein Struktur-Accessor

show_debug_message(_info_A);

Der obige Code ruft die Glyphenstruktur für das Zeichen "A" ab und druckt sie in das Ausgabeprotokoll. Sie können auch die Zeichenkette ("A") durch eine Variable ersetzen, die eine Zeichenkette enthält, wodurch Sie Informationen für jede Glyphe über diese Variable zur Laufzeit abrufen können.

Die Struktur, die für eine Glyphe zurückgegeben wird, enthält die folgenden Variablen:

HINWEIS: Bei HTML5 ist die Glyphen-Struktur für alle geladenen Dateischriften (z. B. *.ttf oder *.otf) leer, da die Runtime keine Informationen über diese Glyphen hat.

Variable Name Daten Typ Beschreibung
Zeichen real Wenn der Font aus einem Sprite erstellt wurde, ist dies der Bildindex der Glyphe aus diesem Sprite, andernfalls ist es seine Unicode-Zeichennummer

Hinweis: Alle Variablen darunter sind in der Struktur nicht vorhanden, wenn der Font aus einem Sprite erstellt wurde
x real Die X-Position der Glyphe auf der Texturseite (in Texel)
y real Die Y-Position der Glyphe auf der Texturseite (in Texel)
w real Die Breite der Glyphe auf der Texturseite (in Texel)
h real Die Höhe der Glyphe auf der Texturseite (in Texel)
Schicht real Die Anzahl der Pixel, die beim Weiterschalten zum nächsten Zeichen nach rechts verschoben werden (kann negativ sein, um nach links zu verschieben)
Offset real Die Anzahl der Pixel, um die das Rendering dieser Glyphe horizontal versetzt werden soll (kann positiv oder negativ sein)
Unterschneiden Array Ein Array von Ganzzahlen, das Kerning-Informationen in Paaren (oder Gruppen von 2) enthält. Die erste Ganzzahl in einem Paar ist der Unicode-Wert für ein Zeichen, und die zweite Ganzzahl ist der Betrag, der zum Verschiebungswert dieses Zeichens addiert wird (kann positiv oder negativ sein kann), wenn das Zeichen dieser Glyphe vorangestellt ist.

HINWEIS: Bei Dateischriften, die zur Laufzeit geladen werden, können die x- und y-Werte für eine Glyphe -1 sein, wenn sie noch nicht gerendert wurde, da sie keine Position auf der für die Zwischenspeicherung verwendeten Texturseite hat.

Syntax:

font_get_info(Schriftart);

Argument Beschreibung
Schriftart Die Schriftart, für die Informationen abgerufen werden sollen.

Rückgabe:

Struktur (oder undefiniert)

Beispiel:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
image_xscale *= 2;
image_yscale *= 2;
}

Dieser Code ruft Informationen für die Schriftart ab, die in der Variablen my_font gespeichert ist. Wenn der zurückgegebene Wert nicht undefiniert ist und die angegebene Schriftart fett ist, wird der Maßstab der Instanz verdoppelt.
