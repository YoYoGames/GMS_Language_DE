schrift_ersetzen

Mit dieser Funktion können Sie eine Schriftart in Ihrem Spiel aus den Schriftarten ersetzen, die auf dem System installiert sind, auf dem es läuft. Beachten Sie, dass die zu ersetzende Schriftart keine Schriftart aus dem Asset-Browser sein kann, sondern eine Schriftart sein muss, die zuvor hinzugefügt wurde mit der Funktion font_add().

Wenn Sie eine Schrift auf diese Weise ersetzen, können Sie die Größe der Schrift (in Punkt) festlegen, sowie ob die Schrift fett oder kursiv sein soll, und Sie können auch den Bereich der einzuschließenden Zeichen festlegen. Wenn die Funktion fehlschlägt, wird sie fehl und die angegebene Schriftart wird nicht ersetzt.

Für das HTML5-Zielmodul kann diese Funktion nur zum Ersetzen von Web Fonts verwendet werden und funktioniert nur, wenn WebGL nicht aktiviert ist - wenn WebGL aktiviert ist, müssen Sie eine allgemeine Font-Ressource verwenden (d. h.: mit draw_set_font() zusammen mit einer Schrift aus dem Asset-Browser) oder Bitmap-Schriften (siehe font_add_sprite()).

WICHTIG! Bei der Ausrichtung auf HTML5 wird kursiver und fetter Text nur unterstützt, wenn das Font-Asset oder die Webschrift selbst diese Zeichen enthält. Sie werden nicht aus der Basisschriftart synthetisiert.

Auf allen anderen (nicht Web-)Targets können Sie diese Funktion verwenden, um eine Schrift aus einer Datei zu ersetzen. Die Datei muss mit der Funktion " Enthaltene Dateien" von GameMaker®Studio 2 in das Spiel-Bundle aufgenommen werden und muss entweder eine Schriftdatei im Format *.ttf oder *.otf sein. Studio 2 eingebunden werden und muss entweder eine Schriftartdatei im Format *.ttf oder *.otf sein, was für das Hinzufügen von Nicht-Standard-Schriftarten wie asiatisch oder arabisch nützlich ist.

WARNUNG! Wenn Sie eine solche Schriftartdatei in ein Spiel einbinden, muss sie für den Vertrieb mit dem Spiel lizenziert werden.

Wenn Sie eine Schriftart auf diese Weise aus einer Datei laden, wird die Größe der Schriftart in Pixeln angegeben und der erste und letzte Wert werden ignoriert, d. h. alle Glyphen der Schriftart werden hinzugefügt. Auf diese Weise hinzugefügte Schriften werden einer eigenen Textur zugewiesen Seite zugewiesen, daher ist bei der Verwendung dieser Funktion Vorsicht geboten, da sie beim Zeichnen die Anzahl der Texturwechsel erhöht. Es ist auch zu beachten, dass Schriften beim Zeichnen etwas größer erscheinen können, da Glyphen Teile haben können, die außerhalb des für die Schrift definierten Begrenzungsrahmens, der für die Schriftart definiert ist, gezeichnet werden. Sie sollten auch beachten, dass es eine Begrenzung von ca. 200 Glyphen gibt, die in einem einzelnen Frame für eine bestimmte Schriftart gerendert werden können.

HINWEIS: Wenn Sie eine Schrift in GameMaker Studio 2: Studio laden, müssen Sie daran denken, sie wieder zu entfernen (mit font_delete()), wenn Sie sie nicht mehr benötigen, sonst besteht die Gefahr eines Speicherlecks, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

font_replace(ind, name, size, bold, italic, first, last);

Argument Beschreibung
ind Der Index der (zuvor hinzugefügten) Schriftart, die ersetzt werden soll.
Name Der Name der zu verwendenden Schriftart (z. B. "Arial") oder der Dateipfad, wenn die Schriftart eine enthaltene *. ttf- oder *. otf-Datei ist .
Größe Die Größe der Schrift - Punkte für Webfonts, Pixel für Dateifonts.
fett Ob die Schrift fett ist (true) oder nicht (false).
Kursivschrift Ob die Schrift kursiv ist (true) oder nicht (false).
erste Das erste einzuschließende Zeichen (wenn Sie unsicher sind, nehmen Sie 32).
zuletzt Das letzte einzuschließende Zeichen (wenn Sie unsicher sind, nehmen Sie 128).

Rückgabe:

Real

Beispiel:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Dadurch wird die in der Variablen "myfont" indizierte Schriftart durch eine neue Schriftart ersetzt, die 24pt groß ist, "Arial" verwendet und fett und kursiv ist. Der Schriftbereich umfasst Groß- und Kleinbuchstaben, Zahlen und alle gängigen Interpunktionszeichen.
