font_get_fontname

Mit dieser Funktion können Sie den tatsächlichen Systemnamen des ausgewählten Schrift-Assets abrufen. Diese Funktion gibt eine Zeichenfolge und keinen Index zurück, und der zurückgegebene Name hängt von der verwendeten Schriftart ab. Ein Beispiel, Sie haben z. B. eine Schriftart-Asset namens "fnt_Main" im Asset-Browser von GameMaker Studio 2, und die Schriftart selbst kann die Windows-Systemschriftart Arial sein. In diesem Fall wird die Funktion "Arial" zurückgeben zurück, da dies der Systemname der Schriftart ist. Sollten Sie den Namen benötigen, wie er im Asset-Browser erscheint, sollten Sie font_get_name().

Syntax:

font_get_fontname(ind);

Argument Beschreibung
ind Index der zu prüfenden Schriftart.

Rückgabe:

Zeichenfolge

Beispiel:

fnt_Name = font_get_fontname(font0);

Der obige Code ermittelt den Systemnamen einer Font-Ressource und speichert ihn als String in der Variablen "fnt_Name".
