font_get_italic

Mit dieser Funktion können Sie jedes Schrift-Asset daraufhin überprüfen, ob es das Kursiv-Flag hat oder nicht. Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

font_get_italic(ind);

Argument Beschreibung
ind Index der zu prüfenden Schriftart.

Rückgabe:

Boolesche

Beispiel:

if font_get_italic(fnt_Main)
{
draw_set_font(fnt_Main);
}

Dadurch wird die aktive Zeichenschrift auf fnt_Main gesetzt, wenn sie in ihren Schrifteigenschaften als kursiv eingestellt ist.
