font_replace_sprite

Mit dieser Funktion können Sie einen "Sprite-Streifen" verwenden, um ein zuvor erstelltes Sprite-Font-Asset zu ersetzen, wobei jedes Teilbild ein einzelnes Symbol oder einen Buchstaben darstellen würde. Der Index ist die UTF8-Map-Nummer für das erste zu verwendende Zeichen und die Bereiche sind die gleichen wie bei einer normalen Schriftart, beginnend mit "!", aber denken Sie daran, dass Sie nur das einschließen sollten, was Sie brauchen, und dass Sie mit der ord() Funktion verwenden, um den korrekten UTF8-Wert für den Anfangsbuchstaben zu erhalten.

Sie können auch den Abstand für die Schrift proportional(true) oder nicht proportional(false) einstellen, wobei eine proportionale Schrift den Abstand auf der Basis der tatsächlichen Breite der Buchstaben hat (so nimmt der Buchstabe "i" weniger Platz ein als der Buchstabe "w", ), während eine nicht-proportionale Schriftart auf der Basis der Breite des Teilbildes platziert wird, wie eine monospaced Schriftart. Schließlich können Sie den Abstand zwischen den einzelnen Buchstaben beim Schreiben definieren, und dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Abstand ist (die Buchstaben werden sich "berühren", wenn sie proportional sind).

Wenn die Funktion fehlschlägt, dann schlägt sie stillschweigend fehl, ohne die angegebene Sprite-Schriftart zu ersetzen.

ußerdem ist zu beachten, dass in der Zeichentabelle keine Leerzeichen vorkommen dürfen, d. h. wenn Sie z. B. '.' und '0', aber nicht '/' verwenden, müssen Sie entweder trotzdem '/' einfügen oder ein leeres Teilbild zwischen den zwei verwendeten Zeichen. Die folgende Abbildung zeigt ein typisches Sprite-Font-Layout:

Syntax:

font_replace_sprite(ind, spr, first, prop, sep);

Argument Beschreibung
ind Der Index der Schrift, die ersetzt werden soll (d. h. alle Verweise auf diese Ressource nach dieser Funktion beziehen sich auf die neue Schrift, nicht auf die ursprüngliche).
spr Das Sprite, auf dessen Basis eine Schrift hinzugefügt werden soll.
erste Der Index des ersten verfügbaren Zeichens.
Stütze Ob der Leerraum um jeden Buchstaben entfernt werden soll (wenn false, wird die erstellte Schrift monospaced, andernfalls wird sie wie die meisten regulären Schriftarten aussehen).
sep Der Abstand zwischen den einzelnen Buchstaben, 0 bedeutet, dass die rechte Seite eines Buchstabens die linke Seite des nächsten Buchstabens gerade berührt.

Rückgabe:

N/A

Beispiel:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Der obige Code ersetzt das in der globalen Variable "Font" indizierte Font-Asset durch ein neues Font-Asset aus einer externen Bilddatei.
