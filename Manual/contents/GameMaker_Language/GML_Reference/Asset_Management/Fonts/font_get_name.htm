font_get_name

Diese Funktion kann verwendet werden, um den Namen (als String) zu erhalten, der der Schriftart gegeben wurde, als sie als Asset zum GameMaker Studio 2 Asset Browser hinzugefügt wurde. Bitte beachten Sie, dass es sich hierbei nur um einen String ist und nicht verwendet werden kann, um die Schrift direkt zu referenzieren - dazu bräuchten Sie den Schriftindex. Sie können diese Zeichenkette jedoch verwenden, um den Font-Index zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

font_get_name(ind);

Argument Beschreibung
ind Index der zu prüfenden Schriftart.

Rückgabe:

Zeichenfolge

Beispiel:

fnt_Name = font_get_name(font0);

Der obige Code ermittelt den Namen einer Schriftressource, wie er im Asset-Browser erscheint, und speichert ihn als String in der Variablen "fnt_Name".
