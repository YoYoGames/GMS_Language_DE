font_add_get_enable_aa

Mit dieser Funktion kann geprüft werden, ob Anti-Aliasing (AA) für Fonts aktiviert ist, die mit font_add(). Die Funktion gibt true zurück, wenn AA aktiviert ist, und false, wenn es nicht ist. Beachten Sie, dass AA standardmäßig aktiviert ist, aber Sie können den AA-Status für hinzugefügte Schriftarten mit der Funktion font_add_enable_aa()ändern, sofern diese Funktion vor dem Hinzufügen der Schriftart aufgerufen wird.

Syntax:

font_add_get_enable_aa();

Rückgabe:

Boolesche

Beispiel:

if !font_add_get_enable_aa()
{
font_add_enable_aa(true);
}

Der obige Code prüft den Status von Anti-Aliasing für hinzugefügte Schriftarten und wenn es nicht aktiviert ist, dann aktivieren wir es.
