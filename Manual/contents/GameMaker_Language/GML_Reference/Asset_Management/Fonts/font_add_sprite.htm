font_add_sprite

Mit dieser Funktion können Sie einen "Sprite-Strip" verwenden, um ein neues Font-Asset zu erstellen, bei dem jedes Unterbild ein einzelnes Symbol oder einen Buchstaben darstellt. Der Index ist die UTF8-Map-Nummer für das erste zu verwendende Zeichen und die zu verwendenden Bereiche sind die gleichen wie Die zu verwendenden Bereiche sind die gleichen wie bei einer normalen Schriftart, beginnend mit "!", aber denken Sie daran, dass Sie nur das einfügen sollten, was Sie benötigen und dass Sie die ord() Funktion verwenden, um den richtigen UTF8-Wert für den Anfangsbuchstaben Buchstaben zu erhalten. Die erstellte Sprite-Schriftart hat dann die Unterbilder, die auf eine einzelne Glyphe aus der Schriftart gesetzt sind, wobei die "erste" Glyphe den image_index 0 hat. Sie können "space" als ein beliebiges Zeichen definieren, zum Beispiel eine einzelne Zeile in der Größe des Leerzeichens, und alle Leerzeichen im Text werden in dieser Breite gerendert (aber das gewählte Bild wird nie gerendert), oder wenn Sie kein Sprite für Leerzeichen angeben, wird die Breite des breitesten Zeichens in der Sprite-Schriftart stattdessen verwendet werden.

Sie können auch den Abstand für die Schrift proportional(true) oder nicht proportional(false) einstellen, wobei eine proportionale Schrift den Abstand auf der Basis der tatsächlichen Breite der Buchstaben hat (so nimmt der Buchstabe "i" weniger Platz ein als der Buchstabe "w", ), während eine nicht-proportionale Schriftart auf der Basis der Breite des Teilbildes platziert wird, wie eine monospaced Schriftart. Schließlich können Sie den Abstand zwischen den einzelnen Buchstaben beim Schreiben definieren, und dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Abstand ist (die Buchstaben werden sich "berühren", wenn sie proportional sind). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dieser in allen weiteren Codes, die sich auf diese Schriftart beziehen, benötigt wird, oder sie gibt -1 zurück, wenn die Funktion aus irgendeinem Grund fehlschlägt. Es ist auch erwähnenswert, dass es keine Leerzeichen in der Zeichentabelle geben kann, so dass Sie, wenn Sie z. B. '.' und '0', aber nicht '/' verwenden, entweder trotzdem '/' einfügen müssen oder ein leeres Teilbild zwischen den zwei verwendeten Zeichen.

Das Sprite selbst kann ein Sprite-Asset aus dem Asset-Browser sein oder eines, das aus einer Datei geladen wird, indem zuerst sprite_add verwendet wird, um den Index des geladenen Sprites zu erhalten (der dann in der Funktion verwendet wird). Es ist wichtig zu beachten, dass für die Schriftausrichtung Funktionen (wie draw_set_halign()) bei der Verwendung eines Sprite-Fonts wie erwartet funktionieren, sollte der Ursprung des Sprites auf die linke obere Ecke gesetzt werden. HINWEIS: Wenn Sie eine Schriftart in GameMaker Studio 2 laden, müssen Sie daran denken, sie wieder zu entfernen (mit font_delete()), wenn Sie sie nicht mehr benötigen, sonst besteht besteht die Gefahr eines Speicherlecks, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

Syntax:

font_add_sprite(spr, first, prop, sep);

Argument Beschreibung
spr Das Sprite, auf dessen Basis eine Schrift hinzugefügt werden soll.
erste Der Index des ersten verfügbaren Zeichens.
Stütze Als Proportionalschrift einstellen oder nicht.
sep Der Abstand, der zwischen den einzelnen Buchstaben gelassen werden soll.

Rückgabe:

Real

Beispiel:

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

Der obige Code erstellt ein neues Font-Asset aus einem Sprite-Asset und speichert den Index des neuen Fonts in der Variablen "global.Font" zur späteren Verwendung.
