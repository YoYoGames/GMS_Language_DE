schrift_get_uvs

Diese Funktion gibt ein Array mit den UV-Koordinaten für die Schrifttextur auf der Texturseite zurück und füllt das Array mit den folgenden Werten auf:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten

Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere im allgemeinen Zeichnen bei Verwendung von Primitiven sowie in den Shader Funktionen.

Syntax:

font_get_uvs(Schriftart);

Argument Beschreibung
Schriftart Der Index der zu verwendenden Schriftart.

Rückgabe:

Array

Beispiel:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Der obige Code speichert die UV-Koordinaten für den gegebenen Hintergrund in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
