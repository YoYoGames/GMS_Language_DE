Die Neun-Scheiben-Struktur

Eine Nine-Slice-Struktur kann mit sprite_nineslice_create()erzeugt oder abgerufen werden aus einem Sprite mit sprite_get_nineslice(). Diese Struktur enthält Eigenschaften, die geändert und auf ein Sprite angewendet werden können mit sprite_set_nineslice().

Wenn es von einem Sprite abgerufen wird, wirkt sich eine Änderung der Werte in dieser Struktur auf das ursprüngliche Sprite aus .

Die folgenden Eigenschaften werden in dieser Struktur verfügbar sein:

  • aktiviert: Dieser boolesche Wert ist true , wenn Nine Slicing für das Sprite aktiviert ist, und false , wenn es deaktiviert. Sie können dies ändern, um Nine Slicing für das Sprite zu aktivieren oder zu deaktivieren.
  • links: Dies ist der Offset für die linke Führung auf der X-Achse, relativ zum linken Rand des Sprites.
  • rechts: Dies ist der Offset für die rechte Führung auf der X-Achse, relativ zum rechten Rand des Sprites.
  • oben: Dies ist der Offset für die obere Führung auf der y-Achse, relativ zur oberen Kante des Sprites.
  • unten: Dies ist der Offset für die untere Führung auf der y-Achse, relativ zur unteren Kante des Sprites.
  • tilemode: Dies gibt ein Array zurück, das die Kachelmodi für die Rand- und Mittelscheiben enthält.

Der Kachelmodus jedes Slice kann mit diesen Konstanten als Array-Index gelesen oder geändert werden:

Konstant Beschreibung
Neunerscheibe_links Die linke Randscheibe
neunschneidig_oben Die obere Randscheibe
neunschneidig_rechts Die rechte Randscheibe
neuner_Unterteil Die untere Randscheibe
Neunmalklug_Zentrum Die mittlere Scheibe

Der Kachelmodus für eine Scheibe ist eine der folgenden Konstanten (oder kann auf eine solche eingestellt werden):

Konstant Beschreibung
neuner_stretch Die Scheibe wird gestreckt
neunfache_Wiederholung Der Slice wird wiederholt
neunschneidiger_Spiegel Der Slice wird durch Spiegelung wiederholt
neuner_blank Der Slice wird nicht gedehnt oder wiederholt, so dass nach ihm ein leerer Bereich entsteht
neunschneidig_ausblenden Der Slice wird überhaupt nicht angezeigt

Da tilemode ein Array ist, müssen Sie die Array-Syntax verwenden, um den Kachelmodus eines Slice zu ändern. Das folgende Codebeispiel setzt den Kachelmodus des oberen Slice auf " Repeat":

Beispiel:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
