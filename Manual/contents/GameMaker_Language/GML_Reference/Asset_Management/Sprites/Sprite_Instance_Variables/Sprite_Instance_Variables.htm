Sprite-Instanz-Variablen

Diese Variablen sind standardmäßig in GameMaker Studio 2-Objekten enthalten und beziehen sich alle auf das Sprite, das der Instanz im Raum zugewiesen ist. Auf diese Variablen kann jederzeit innerhalb des Codes oder der Aktionen für eine Instanz zugegriffen und verwendet werden um zu ändern, was und wie gezeichnet wird.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • Bild_Winkel
  • image_blend
  • image_index
  • Bild_Nummer
  • image_speed
  • Bild_x-Skala
  • image_yscale
  • maske_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
