image_speed

Diese Variable bestimmt die Geschwindigkeit, mit der GameMaker Studio 2 die Teilbilder für das aktuelle Instanz-Sprite durchläuft. Der angegebene Geschwindigkeitswert ist ein Multiplikator, wobei 1 der Standardwert ist und die Einstellung 0,5 die Hälfte der Die Einstellung auf 0,5 halbiert die Animationsgeschwindigkeit - wie im Sprite-Editor oder Bild-Editor eingestellt -, während die Einstellung auf 2 sie verdoppelt. Wenn das verwendete Sprite keine Sub-Images hat, hat diese Variable keine Auswirkung.

Syntax:

image_speed;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

with (instance_create_layer(x, y, "Effekte", obj_Explosion))
{
image_speed = 0.5;
}

Der obige Code erzeugt eine Instanz des Objekts "obj_Explosion" und setzt dessen image_speed auf 0,5 (was effektiv eine Halbierung der Laufgeschwindigkeit bedeutet).
