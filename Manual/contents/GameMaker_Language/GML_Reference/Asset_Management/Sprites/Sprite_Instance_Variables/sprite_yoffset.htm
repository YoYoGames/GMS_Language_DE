sprite_yoffset

Diese schreibgeschützte Variable gibt den lokalen yOffset (die y-Komponente des Ursprungs, wie im Sprite-Editor definiert) des Sprites zurück, das der Instanz zugewiesen wurde. Dieser yOffset wird in Pixel zurückgegeben und ist abhängig von der image_yscale, so dass (zum Beispiel), wenn ein quadratisches 32x32-Sprite so definiert ist, dass der y-Ursprung bei 16 liegt, wenn dieses Sprite um 2 skaliert wird, diese Variable 32 zurückgeben wird. Wenn Sie den nicht skalierten y-Offset benötigen, sollten Sie sprite_get_yoffset().

Syntax:

sprite_yoffset;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if y - sprite_yoffset < 0
{
y = sprite_yoffset;
}

Der obige Code prüft, ob die obere Kante des Sprites der Instanz außerhalb des Raumes liegt, und wenn dies der Fall ist, bewegt er die Instanz so, dass das gesamte Sprite sichtbar ist.
