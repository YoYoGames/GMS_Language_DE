image_blend

Diese Variable steuert die "Einfärbung" des Instanz-Sprites und der Standardwert ist -1 (kann aber auch c_white sein). Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red oder c_aqua) mischt die angegebene Farbe mit dem Original-Sprite. Bitte beachten Sie, dass Änderungen in dieser Variablen nur dann sichtbar sind, wenn die Instanz entweder kein Draw-Ereignis hat (und GameMaker Studio 2 das Sprite daher standardmäßig zeichnet) oder mit einer der erweiterten Zeichenfunktionen Funktionen wie draw_self() oder draw_sprite_ext().

Bitte beachten Sie, dass Sie versuchen sollten, das Überblenden auf der HTML5-Plattform zu begrenzen (es sei denn, Sie verwenden WebGL), da jedes überblendete Sprite separat zwischengespeichert werden muss und daher viele überblendete Sprites die Leistung beeinträchtigen können (Sie können die Cache-Größe auch mit der die Funktion sprite_set_cache_size()).

Syntax:

image_blend;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

image_blend = make_colour_hsv(255, 255, random(255));

Der obige Code setzt das image_blend des für die Instanz definierten Sprites auf eine zufällige Farbe.
