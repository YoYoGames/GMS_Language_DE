Bild_Nummer

Diese schreibgeschützte Variable kann verwendet werden, um die Anzahl der Sub-Images in einem Sprite zu ermitteln, das einer Instanz zugewiesen wurde (wenn Sie die Anzahl der Sub-Images für ein anderes Sprite als das der Instanz zugewiesene benötigen, sollten Sie verwenden sprite_get_number()). Bitte beachten Sie, dass diese Variable "1" zurückgibt, wenn es (z. B.) 1 Sub-Image gibt, aber der image_index dieses Sub-Images 0 ist.

Syntax:

image_number;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if image_speed > 0
{
if image_index > image_number - 1 instance_destroy();
}

Der obige Code prüft, ob das Sprite animiert ist, und wenn ja, dann prüft er, ob der aktuelle image_index größer als die Anzahl der Sub-Images ist, und wenn ja, zerstört er die Instanz.
