sprite_height

Diese schreibgeschützte Variable gibt die Höhe des Sprites zurück, das der Instanz zugewiesen wurde. Diese Höhe wird in Pixel zurückgegeben und ist abhängig von der image_yscale. Wenn Sie die nicht skalierte Höhe benötigen, sollten Sie sprite_get_height().

Syntax:

sprite_height;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if sprite_height != sprite_get_height(sprite_index)
{
image_yscale = 1;
}

Der obige Code prüft die Höhe des Sprites, das der Instanz zugewiesen ist, die den Code ausführt, gegen die Höhe der Sprite-Ressource und setzt, wenn sie nicht übereinstimmt, die image_yscale auf 1 zurück,
