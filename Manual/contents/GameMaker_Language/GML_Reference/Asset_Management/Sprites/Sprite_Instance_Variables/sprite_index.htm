sprite_index

Diese Variable gibt den Index des aktuellen Sprites für die Instanz zurück, oder -1, wenn der Instanz kein Sprite zugeordnet ist. Sie können sie ändern, um der Instanz ein anderes Sprite zu geben, indem Sie ihr den Namen eines Sprites aus dem Ressourcenbaum geben oder indem Sie eine Variable verwenden, in der ein extern geladenes Sprite indiziert ist. Das Ändern des Sprites ändert nicht den Index des aktuell sichtbaren Frames, wenn Sie also das Sprite auf Frame Nummer 3 ändern, wird das neue Sprite mit diesem sichtbaren Frame gezeichnet (vorausgesetzt, es hat die gleiche Anzahl von Frames).

Syntax:

sprite_index;

Rückgabe:

Real (Ganzzahl - Ressourcenindexwert)

Beispiel:

with (obj_Check)
{
if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
{
sprite_index = spr_spotted;
}
sonst
{
sprite_index = spr_clear;
}
}

Der obige Code läuft in einer Schleife durch alle Instanzen von "obj_Check" und prüft, ob eine Kollisionslinie zwischen ihnen und der Instanz, die den Code ausführt, besteht. Das Sprite dieser Instanzen wird in Abhängigkeit vom Rückgabewert (true oder false) für die Kollisionslinie.
