bbox_bottom

Diese schreibgeschützte Variable gibt die y-Position (innerhalb des Raums) des unteren Teils des Begrenzungsrahmens für die Instanz zurück, wobei der Begrenzungsrahmen durch die maximale Breite und Höhe der Maske für die Instanz definiert ist (wie durch die sprite_index oder durch die mask_index). Auch wenn ein Sprite eine präzise Kollisionsmaske hat, existiert der Begrenzungsrahmen und wird für bestimmte Dinge, und so können Sie diese Variable verwenden, um sie zu finden. Bitte beachten Sie, dass der zurückgegebene Wert der Y-Position der Instanz entspricht, wenn der Instanz kein Sprite zugewiesen ist.

Syntax:

bbox_bottom;

Rückgabe:

Real (Ganzzahl)

Beispiel:

if bbox_bottom > room_height
{
y = room_height - sprite_yoffset;
}

Der obige Code prüft, ob der Begrenzungsrahmen der Instanz außerhalb des Raums liegt, und wenn ja, setzt er die Instanz an eine neue Position.
