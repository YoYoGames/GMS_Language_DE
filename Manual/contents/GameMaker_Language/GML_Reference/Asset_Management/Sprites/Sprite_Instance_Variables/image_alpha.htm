image_alpha

Diese Variable wird verwendet, um den Alpha-Wert für das Sprite zu erhalten oder zu setzen. Alpha wird immer als ein Wert zwischen 0 und 1 berechnet, wobei 0 komplett transparent und 1 komplett undurchsichtig ist. Bitte beachten Sie, dass Änderungen in dieser Variablen nur dann sichtbar sind, wenn die Instanz entweder kein Draw-Ereignis haben sollte (und damit GameMaker Studio 2 das Sprite standardmäßig zeichnet) oder mit einer der erweiterten Zeichenfunktionen wie draw_self() oder draw_sprite_ext().

Syntax:

image_alpha;

Rückgabe:

Real (zwischen 0 und 1 - Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

image_alpha = clamp(image_alpha - 0,01, 0, 1);

Der obige Code reduziert langsam das image_alpha, bis es 0 erreicht.
