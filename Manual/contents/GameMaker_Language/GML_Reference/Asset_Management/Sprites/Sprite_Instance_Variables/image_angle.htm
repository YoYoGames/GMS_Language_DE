Bild_Winkel

Dieser Wert legt den Winkel (Drehung) des Sprites fest und wird in Grad gemessen, wobei rechts 0º, oben 90º, links 180º und unten 270º ist. Setzen Sie diese Variable auf 0, damit das Sprite wieder so gezeichnet wird, wie es im Sprite-Editor definiert wurde. Bitte beachten Sie, dass Änderungen in dieser Variablen nur dann sichtbar sind, wenn die Instanz entweder kein Draw-Ereignis hat (so dass GameMaker Studio 2 das Sprite standardmäßig zeichnet) oder wenn sie mit einer der erweiterten Zeichenfunktionen wie draw_self() oder draw_sprite_ext().

Syntax:

image_angle;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Der obige Code wird das Sprite der Instanz so drehen, dass es immer auf die Mausposition zeigt.
