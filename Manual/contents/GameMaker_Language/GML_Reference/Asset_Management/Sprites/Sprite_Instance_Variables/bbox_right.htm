bbox_right

Diese schreibgeschützte Variable gibt die Position innerhalb des Raums (entlang der x-Achse) der rechten Seite des Begrenzungsrahmens für die Instanz zurück, wobei der Begrenzungsrahmen durch die maximale Breite und Höhe der Maske definiert ist für die Instanz (wie von sprite_index oder von mask_index festgelegt). Auch wenn ein Sprite eine präzise Kollisionsmaske hat, existiert der Begrenzungsrahmen existiert und wird für bestimmte Dinge verwendet, und so können Sie diese Variable verwenden, um sie zu finden. Beachten Sie, dass, wenn der Instanz kein Sprite zugewiesen ist, der zurückgegebene Wert mit der X-Position der Instanz identisch ist.

Syntax:

bbox_right;

Rückgabe:

Real (Ganzzahl)

Beispiel:

if bbox_right > room_width
{
x = room_width - sprite_xoffset;
}

Der obige Code prüft, ob der Begrenzungsrahmen der Instanz außerhalb des Raums liegt, und wenn ja, setzt er die Instanz an eine neue Position.
