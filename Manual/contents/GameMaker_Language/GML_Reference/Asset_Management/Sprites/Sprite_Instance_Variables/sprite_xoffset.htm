sprite_xoffset

Diese schreibgeschützte Variable gibt den lokalen X-Offset (die X-Komponente des Ursprungs wie im Sprite-Editor definiert) des Sprites zurück, das der Instanz zugewiesen wurde. Dieser xOffset wird in Pixel zurückgegeben und ist abhängig von der bild_xskalieren, so dass (zum Beispiel), wenn ein quadratisches 32x32-Sprite so definiert ist, dass der x-Ursprung bei 16 liegt, wenn dieses Sprite um 2 skaliert wird, diese Variable 32 zurückgeben wird. Wenn Sie den nicht skalierten x-Offset benötigen, sollten Sie sprite_get_xoffset().

Syntax:

sprite_xoffset;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if x - sprite_xoffset < 0
{
x = sprite_xoffset;
}

Der obige Code prüft, ob der linke Rand des Sprites der Instanz außerhalb des Raumes liegt, und wenn dies der Fall ist, bewegt er die Instanz so, dass das gesamte Sprite sichtbar ist.
