maske_index

Wenn Sie ein Objekt in GameMaker Studio 2 definieren, können Sie ihm eine Maske zuweisen, die für Kollisionen verwendet wird und nicht die, die dem definierten Sprite entspricht. Diese Variable kann verwendet werden, um die sprite_index dieser Maske zu finden (oder sie gibt -1 zurück, wenn kein Sprite zugewiesen wurde) oder um die Maske für eine Instanz auf das gewählte Sprite zu setzen. Das Setzen des Maskenindexes bedeutet, dass Sie z. B. ein Sprite für die Instanz mit einer unregelmäßigen Form haben können, ihm aber eine kreisförmige Kollisionsmaske geben, die von einem anderen Sprite bezogen wird.

Syntax:

mask_index;

Rückgabe:

Real (Ganzzahl - Ressourcenindexwert)

Beispiel:

mask_index = spr_Round;

Der obige Code setzt die Maske der Instanz auf die des Sprites "spr_Round".
