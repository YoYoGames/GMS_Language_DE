skelett_attachment_create

Wenn Sie Ihr Skelett-Animations-Sprite erstellen, können Sie Anhänge-Slots und Anhänge zuweisen, die in diese gehen. Dabei handelt es sich einfach um Bilder, die nicht zur Animation gehören, aber wenn sie angehängt sind wenn sie angehängt sind, zusammen mit dem Bone bewegt werden, an dem sie angehängt sind. Normalerweise würden Sie diese Anhänge in Ihrem Animationsprogramm (Spine) zuweisen, aber mit dieser Funktion können Sie Ihre eigenen zur Laufzeit erstellen, indem Sie ein Sprite-Asset aus Ihrem Spiel verwenden. die Funktion erfordert dass Sie dem Attachment einen Namen geben (als String) und dann den zu verwendenden sprite_index und image_index sowie den x- und y-Ursprung (der sich von dem in den sprite-Eigenschaften definierten unterscheiden kann) festlegen. setzen, die beim Anhängen auf das Bild angewendet werden sollen. Wenn das Attachment erfolgreich erstellt wurde, gibt die Funktion 1 zurück, wenn nicht (Sie haben einen ungültigen Sprite-Index angegeben, oder das Basis-Sprite ist kein Spine Sprite), dann gibt sie -1 zurück.

Syntax:

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

Argument Beschreibung
Name Der Name (als String) des zu erstellenden Anhangs.
Sprite Der sprite_index, von dem das Anhangsbild abgerufen werden soll.
ind Der image_index, von dem das Anhangsbild abgerufen werden soll.
xOrigin Der x-Ursprung für das verwendete Bild.
yHerkunft Der y-Ursprung für das verwendete Bild.
x-Skala Die horizontale Skalierung des Bildes, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
y-Skala Die vertikale Skalierung des Bildes, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
rot Die Drehung des Bildes. 0=normal, 90=um 90 Grad gegen den Uhrzeigersinn gedreht usw.

Rückgabe:

1 wenn erfolgreich, -1 wenn nicht

Beispiel:

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

Der obige Code würde den aktuell zugewiesenen Anlagennamen für den Steckplatz mit dem Namen "slot_leftHand" prüfen und wenn eine leere Zeichenfolge zurückgegeben wird, wird ein neues Sprite angehängt.
