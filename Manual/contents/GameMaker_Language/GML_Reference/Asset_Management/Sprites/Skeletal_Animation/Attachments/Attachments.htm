Anhänge

Skelettanimations-Sprites können "Slots" haben, in denen Sie "Anhänge" platzieren können. Ein Anhang ist einfach ein Bild, das vom Sprite als Requisite verwendet werden kann und in einem Slot platziert wird, so dass es sich zusammen mit der Animation des Teils wo der Slot positioniert ist. Wenn Sie sich das Sprite eines Kriegers mit einem Attachment-Slot an seiner Hand vorstellen - Sie können dann verschiedene Waffen-Sprites auf diesem Slot platzieren und den Krieger so aussehen lassen, als würde er ein Schwert oder eine Pike oder ein Schild oder was auch immer benutzen.

Dieser Abschnitt befasst sich speziell mit Anbaugeräten und ihren Sprites, Sie können jedoch auch Slots einrichten und Informationen über sie von den hier aufgeführten Funktionen erhalten.

Die verfügbaren Anschlussfunktionen sind:

  • skelett_anhaengung_holen
  • skelett_anhang_set
  • skelett_attachment_create
  • skeleton_attachment_create_colour
