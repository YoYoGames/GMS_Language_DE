skelett_anhaengung_holen

Ein Skelett-Animationssprite kann andere Sprites als Attachments hinzufügen, wobei diese Sprites einem benannten Slot hinzugefügt werden (der Name wird beim Erstellen des Attachment-Slots in Ihrem Animationsprogramm vergeben) und sie werden zusammen mit der Animation des aktuellen Sprites gezeichnet. Mit dieser Funktion können Sie den Namen (als String) des Attachments für den angegebenen Slot des aktuell zugewiesenen Sprites abrufen. Beachten Sie, dass angehängte Sprites über ihren Namensstring referenziert werden, wie er in Spine zugewiesen wurde, oder wenn Sie die Funktion skeleton_attachment_create().

Syntax:

skeleton_attachment_get(Steckplatz);

Argument Beschreibung
Steckplatz Der Name des Steckplatzes (eine Zeichenkette), von dem der Anhang abgerufen werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Der obige Code würde den aktuell zugewiesenen Anlagennamen für den Steckplatz mit dem Namen "slot_leftHand" prüfen und wenn eine leere Zeichenfolge zurückgegeben wird, wird ein neues Sprite angehängt.
