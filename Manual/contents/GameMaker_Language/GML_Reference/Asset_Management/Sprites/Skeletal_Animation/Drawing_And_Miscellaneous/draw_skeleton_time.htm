zeichnen_skelett_zeit

Diese Funktion zeichnet die angegebene Animation unter Verwendung des angegebenen Skins zu einem bestimmten Zeitpunkt in der Animation. Der Zeitwert sollte zwischen 0 (dem Anfang) und der Enddauer der Animation liegen, die Sie mit der Funktion skeleton_animation_get_duration(). Sie können den Zeitwert auf Werte setzen, die höher sind als die Gesamtdauer der Animation, und die Animation wird in einer Schleife zum Anfang zurückgeführt, aber Sie laufen Gefahr, die Fließkommagenauigkeit zu verlieren, wenn die kumulierte Zeit größer wird.

Syntax:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, color)

Argument Beschreibung
Sprite Der Index des zu zeichnenden Sprites.
animname Der Name der Animation, aus der das Bild geholt werden soll (eine Zeichenkette).
Skinname Der Name des Skins, der für das Zeichnen verwendet werden soll (eine Zeichenkette).
Zeit Die Zeitspanne, in der die Animation gezeichnet werden soll (von 0 bis zur Enddauer, in Sekunden).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.
x-Skala Die horizontale Skalierung des Sprites, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
y-Skala Die vertikale Skalierung des Sprites, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
rot Die Drehung des Sprites. 0=normal, 90=um 90 Grad gegen den Uhrzeigersinn gedreht usw.
Farbe Die Farbe, mit der das Sprite überblendet werden soll.

Rückgabe:

N/A

Beispiel:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Der obige Code zeichnet das angegebene Skelett-Animationssprite unter Verwendung der Delta-Zeit, um das zu zeichnende Bild festzulegen.
