skelett_kollision_zeichnen_setzen

Mit dieser Funktion können Sie das Zeichnen der Kollisionsdaten für das aktuelle Skelettanimationssprite, das von der Instanz verwendet wird, ein-(true) oder ausschalten(false). Wenn dies eingeschaltet ist, werden der Begrenzungsrahmen und die genaue Kollisionsmaske als Umrisse um das Sprite gezeichnet.

Syntax:

skeleton_collision_draw_set(flag);

Argument Beschreibung
Flagge Setzen Sie diesen Wert auf true, um das Zeichnen einzuschalten, und auf false, um es auszuschalten.

Rückgabe:

N/A

Beispiel:

if debug_mode == true
{
skeleton_collision_draw_set(true);
}

Der obige Code prüft, ob das Spiel im Debug-Modus ausgeführt wird, und wenn dies der Fall ist, schaltet er die Skelettkollisionsdaten für die anzuzeigende Instanz um.
