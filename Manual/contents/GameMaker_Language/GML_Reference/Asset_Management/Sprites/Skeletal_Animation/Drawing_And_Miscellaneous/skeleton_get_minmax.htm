skelett_get_minmax

Diese Funktion gibt ein Array mit dem Gesamtwert der Boundingboxen für alle einzelnen Boundingboxen zurück, die einem Skelett-Animationssprite zugewiesen sind. Das zurückgegebene Array hat 4 Elemente:

  • 0] - die minimale x-Position für alle Bounding Boxes
  • 1] - die minimale y-Position für alle Bounding Boxes
  • 2] - die maximale x-Position für alle Bounding Boxes
  • 3] - die maximale y-Position für alle Bounding Boxes

Syntax:

skeleton_get_minmax();

Rückgabe:

Array (4 Elemente: xMin, yMin, xMax, yMax)

Beispiel:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
var box = skeleton_get_minmax();
var ww = (box[2] - box[0]) / 2;
var hh = (box[3] - box[1]) / 2;
part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Der obige Code prüft das aktuelle Bild der Animation, die der Spur 0 zugewiesen ist, und lässt dann einige Partikel von einem Punkt in der Mitte des gesamten Begrenzungsrahmens für das Sprite platzen.
