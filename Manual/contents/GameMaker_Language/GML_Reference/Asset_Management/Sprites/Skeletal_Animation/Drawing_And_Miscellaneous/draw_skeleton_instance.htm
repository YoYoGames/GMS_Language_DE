zeichnen_skelett_instanz

Diese Funktion ist nur für die Verwendung mit Instanzen gedacht, denen Sprites zugewiesen sind, die mit dem Skelett-Animationsprogramm Spine erstellt wurden. Sie können diese Sprites zwar mit den normalen Zeichenfunktionen (wie draw_self()), können Sie zwar diese Sprites mit den normalen Zeichenfunktionen (z. B. ) zeichnen, aber es kann vorkommen, dass Sie ein einzelnes Bild zeichnen oder eine Änderung der Hauttextur anzeigen möchten, ohne die Sprite-Eigenschaften für eine Instanz zu ändern. In diesen Fällen können Sie das Sprite, das der Instanz zugewiesen wurde, mit dieser Funktion zeichnen, wobei Sie die eindeutige Instanz-ID und dann den Namen der Animation angeben, von der Sie das Einzelbild abrufen möchten (eine Zeichenkette, wie sie in dem Programm festgelegt wurde, mit dem das Sprite erstellt wurde). Als Nächstes geben Sie den Namen des zu verwendenden Skins an (wieder so, wie er bei der Erstellung des Sprites festgelegt wurde), und einen Frame-Wert. Der Frame-Wert dient dazu, GameMaker Studio 2 mitzuteilen, welcher Teil der Animation gezeigt werden soll, und ist ein Wert zwischen 0 und dem image_number. Der Rest der Argumente ist Die übrigen Argumente sind die gleichen wie beim normalen Zeichnen von Sprites und ändern die Skalierung, den Winkel, die Überblendungsfarbe und das Alpha des endgültigen Animationsframes, das gezeichnet wird.

WARNUNG! Aufgrund der Art und Weise, wie Skelett-Animationen zwischen verschiedenen "Schlüssel"-Frames interpoliert werden, ist mit diesem Aufruf ein zusätzlicher Overhead verbunden, und es wird daher empfohlen, dass Sie es vermeiden, skelettbasierte Sprites mit dieser Methode zu vermeiden, es sei denn, es ist absolut notwendig.

Syntax:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, color, alpha);

Argument Beschreibung
Instanz Der Index der Instanz, von der das Sprite gezeichnet werden soll.
animname Der Name der Animation, aus der das Bild geholt werden soll (eine Zeichenkette).
Skinname Der Name des zu verwendenden Skins (eine Zeichenkette).
Rahmen Das zu zeichnende Animationsbild (von 0 bis image_number - 1).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.
x-Skala Die horizontale Skalierung des Sprites, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
y-Skala Die vertikale Skalierung des Sprites, als Multiplikator: 1 = normale Skalierung, 0,5 ist die Hälfte usw...
rot Die Drehung des Sprites. 0=normal, 90=um 90 Grad gegen den Uhrzeigersinn gedreht usw.
Farbe Die Farbe, mit der das Sprite überblendet werden soll.
alpha Das Alpha des Sprites (von 0 bis 1, wobei 0 transparent und 1 undurchsichtig ist).

Rückgabe:

N/A

Beispiel:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

Der obige Code zeichnet das erste Bild des Skelett-Animationssprites, das der angegebenen Instanz zugewiesen ist, unter Verwendung des Animationssets "jump" und des als "skin1" definierten Skins.
