skelett_get_num_bounds

Diese Funktion gibt die Anzahl der Bounding Boxes zurück, die mit dem Skelett-Animationssprite verbunden sind, das der Instanz zugeordnet ist, die den Code ausführt. Dies kann dann zusammen mit der Funktion skeleton_get_bounds() verwendet werden, um Daten über jede der Bounding Boxen abrufen.

Syntax:

skeleton_get_num_bounds();

Rückgabe:

Integer

Beispiel:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
{
var b_info = skeleton_get_bounds(i);
if b_info[0] > 0
{
var data = b_info[1] + ":";
for(var j = 0; j < b_info[0]; j++;)
{
data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
}
draw_text(20, yy, data);
yy += 20;
}
}

Der obige Code läuft in einer Schleife durch alle Boundingboxen, die mit dem aktuell zugewiesenen Sprite verbunden sind, und zeichnet diese Daten dann als String auf den Bildschirm.
