Skelett_Haut_holen

Bei Skelettanimations-Sprites können Sie der Animation separate Texturen (sogenannte "Skins") zuweisen und so eine Animation für mehrere verschiedene Dinge verwenden. Diese Funktion gibt den Namen des Skins (als String) zurück, der aktuell dem Skelettanimations-Sprite Ihrer Instanz zugewiesen ist. Der zurückgegebene Name ist der, den Sie beim Erstellen des Sprites in Ihrem Animationsprogramm festgelegt haben.

Syntax:

skeleton_skin_get();

Rückgabe:

Zeichenfolge

Beispiel:

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Der obige Code würde den aktuell zugewiesenen Skin für die Animation prüfen und, wenn es "skin_Enemy1" ist, einen neuen Skin aus einer der drei Optionen auswählen und einstellen.
