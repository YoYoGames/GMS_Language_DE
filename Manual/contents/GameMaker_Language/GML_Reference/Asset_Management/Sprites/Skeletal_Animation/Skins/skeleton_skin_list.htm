skelett_haut_liste

Mit dieser Funktion können Sie eine (vorher erstellte) DS-Liste mit allen Namen der Häute füllen, die als Teil des Skelett-Animationssprites enthalten sind. Die Namen sind Strings und können dann in den anderen Skelettanimations-Skin-Funktionen für diese Sprite-Typen verwendet werden.

Syntax:

skeleton_skin_list(sprite, list);

Argument Beschreibung
Sprite Der Sprite-Index der Wirbelsäulen-Skelettanimation, von der die Liste abgerufen werden soll.
Liste Die ID der DS-Liste, die mit den Animationsnamen gefüllt werden soll.

Rückgabe:

N/A

Beispiel:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Der obige Code erstellt eine DS-Liste und füllt sie mit den Skin-Namen. Anschließend wird ein zufälliger Name ausgewählt und auf die Instanz angewendet, bevor die Liste zerstört wird.
