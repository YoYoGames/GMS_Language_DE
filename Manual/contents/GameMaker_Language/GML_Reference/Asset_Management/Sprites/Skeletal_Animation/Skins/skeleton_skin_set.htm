skelett_haut_set

Ein Skelett-Animations-Sprite kann mit einer Vielzahl von Texturen (sogenannten "Skins") gezeichnet werden, so dass Sie eine einzige Animation für mehrere verschiedene Zwecke verwenden können. Sie können z. B. eine generische Laufanimation haben, die von Ihren NPC-Charaktere, Feinde usw. in einem Spiel verwendet werden, indem Sie einfach den Skin ändern, mit dem er gezeichnet wird. Diese Skins müssen zuvor in Ihrem Animationsprogramm erstellt und benannt worden sein, und dieser Name wird an die Funktion (als String) übergeben, um das Sprite so einzustellen, dass es beim Zeichnen diesen Skin verwendet.

Syntax:

skeleton_skin_set(skinname);

Argument Beschreibung
Skinname Der Name (eine Zeichenkette) des zu verwendenden Skins.

Rückgabe:

N/A

Beispiel:

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Der obige Code würde den aktuell zugewiesenen Skin für die Animation prüfen und, wenn es "skin_Enemy1" ist, einen neuen Skin aus einer der drei Optionen auswählen und einstellen.
