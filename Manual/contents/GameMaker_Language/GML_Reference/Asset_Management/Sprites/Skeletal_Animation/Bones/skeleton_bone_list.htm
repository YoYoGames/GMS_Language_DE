skelett_knochen_liste

Mit dieser Funktion können Sie eine (vorher erstellte) DS-Liste mit allen Namen der Bones füllen, die als Teil des Skelettanimations-Sprites verwendet werden. Die Namen sind Strings und können dann in den anderen Skelettanimations-Bone-Funktionen für diese Art von Sprite verwendet werden.

Syntax:

skeleton_bone_list(sprite, list);

Argument Beschreibung
Sprite Der Sprite-Index der Wirbelsäulen-Skelettanimation, von der die Liste abgerufen werden soll.
Liste Die ID der DS-Liste, die mit den Bone-Namen gefüllt werden soll.

Rückgabe:

N/A

Beispiel:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

Der obige Code erstellt eine DS-Liste und füllt sie mit den Bone-Namen für die spätere Verwendung.
