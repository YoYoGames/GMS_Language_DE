skelett_schlitz_farbe_set

Mit dieser Funktion können Sie die Farbe und das Alpha eines Attachment-Slots setzen, so dass alle daran angehängten Sprites mit diesen Blend-Werten gezeichnet werden. Beachten Sie, dass das Instanz-Sprite eine Blend-Farbe und Alpha-Einstellung haben kann (image_blend und bild_alpha), ebenso wie das Attachment (siehe die Funktion skeleton_attachment_create_colour()), und somit die endgültige Farbe und Alpha, die das zugewiesene Attachment-Sprite für den Slot haben wird, eine Mischung aus all diesen Werten sein wird.

Syntax:

skeleton_slot_colour_set(slot, color, alpha);

Argument Beschreibung
Steckplatz Der zu setzende Steckplatzname, eine Zeichenkette
Farbe Die einzustellende Farbe, entweder eine Ganzzahl, eine Konstante oder ein Hex-Wert.
alpha Das einzustellende Alpha von 0 bis 1.

Rückgabe:

N/A

Beispiel:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
for (var i = 0; i < ds_list_size(slot_list); ++i;)
{
if skeleton_slot_colour_get(slot_list[| i]) != c_white
{
skeleton_slot_colour_set(slot_list[| i], c_white, 1);
}
}
}

Der obige Code erstellt eine DS-Liste und füllt sie dann mit den Slot-Namen, die an der Position der Maus im Raum gefunden wurden. Anschließend durchläuft er die Slot-Liste und setzt die Farbe für die gefundenen Slots auf Weiß mit einem Alpha-Wert von 1 zurück, wenn die Farbe nicht bereits weiß ist.
