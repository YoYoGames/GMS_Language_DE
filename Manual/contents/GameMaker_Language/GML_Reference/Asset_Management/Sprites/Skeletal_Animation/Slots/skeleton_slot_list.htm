skelett_schlitz_liste

Mit dieser Funktion können Sie eine (vorab erstellte) DS-Liste mit allen Namen der Slots füllen, die als Teil des Skelettanimations-Sprites erstellt wurden. Die Namen sind Strings und können dann in den anderen Skelettanimations-Slot-Funktionen verwendet werden für diese Typen von Sprites verwendet werden.

Syntax:

skeleton_slot_list(sprite, list);

Argument Beschreibung
Sprite Der Sprite-Index der Wirbelsäulen-Skelettanimation, von der die Liste abgerufen werden soll.
Liste Die ID der DS-Liste, die mit den Bone-Namen gefüllt werden soll.

Rückgabe:

N/A

Beispiel:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

Der obige Code erstellt eine DS-Liste und füllt sie mit den Steckplatznamen für die spätere Verwendung.
