skelett_find_schlitz

Mit dieser Funktion können Sie herausfinden, welche Slots sich an einer bestimmten Raum-Raum-Position in dem mit der aktuellen Instanz verbundenen Spine-Sprite befinden. Sie erstellen eine DS-Liste und geben ihre ID zusammen mit einer mit einer zu prüfenden x/y-Position an und die Liste wird mit Namensstrings für jeden der verfügbaren Attachment-Slots für das Sprite (einschließlich aller Attachment-Modifikationen) gefüllt. Beachten Sie, dass die Liste immer in absteigender Reihenfolge sortiert ist, beginnend mit dem obersten Steckplatz.

Syntax:

skeleton_find_slot(x, y, list);

Argument Beschreibung
x Die zu prüfende x-Position im Raum.
y Die zu prüfende y-Position im Raum.
Liste Die ID der DS-Liste, die mit den DS-Maps gefüllt werden soll.

Rückgabe:

N/A

Beispiel:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Der obige Code erstellt eine DS-Liste und füllt sie dann mit den Slot-Namen, die an der Position der Maus im Raum gefunden wurden.
