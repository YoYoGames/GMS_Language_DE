skelett_animation_set

Sobald Sie Ihrer Instanz ein Skelett-Animationssprite zugewiesen haben, können Sie mit dieser Funktion festlegen, welches Animationsset es zu einem bestimmten Zeitpunkt verwenden soll. Als Sie Ihr Sprite (in Spine) erstellt haben, haben Sie mehrere Animations Sets für verschiedene Zustände definiert und diese entsprechend benannt. Diese Namen werden Sie verwenden, um GameMaker Studio 2 mitzuteilen, welches Animationsset zu einem bestimmten Zeitpunkt in Ihrem Spiel verwendet werden soll. Wenn Ihr Sprite z. B. für ein Plattformspiel gedacht ist, können Sie ein "Sprung"-Set, ein "Lauf"-Set und ein "Leerlauf"-Set, die alle in einem Sprite enthalten sind, und Sie können mit dieser Funktion in Ihrem Spiel zwischen ihnen umschalten.

Beachten Sie, dass das Setzen einer neuen Animation auf diese Weise den bild_index auf 0 zurückgesetzt wird und das neue Animationsset am Anfang seiner Animationsschleife beginnt. Wenn dies nicht das gewünschte Verhalten ist, dann können Sie einen "Mix"-Wert mit skelett_animation_mix() einen "Mix"-Wert setzen, und wenn Sie eine neue Animation setzen, wird diese nahtlos mit dem vorherigen Satz interpoliert. Sie sollten auch darauf achten Sie sollten auch darauf achten, dass am Anfang jeder Animation ein Keyframe steht, um zu verhindern, dass Daten aus vorherigen Animationen übernommen werden, es sei denn, Sie planen, separate Animationsspuren zu verwenden.

Syntax:

skeleton_animation_set(animname);

Argument Beschreibung
animname Der Name (eine Zeichenkette) des zu verwendenden Animationssets.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("springen");
}
}

Der obige Code ändert das verwendete Animations-Set in das "Sprung"-Set, wenn die Leertaste gedrückt wird, aber nur, wenn das aktuell verwendete Set nicht bereits "Sprung" ist.
