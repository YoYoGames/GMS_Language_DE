skelett_animation_get

Mit dieser Funktion können Sie das aktuelle Animationsset ermitteln, das von Ihrem Skelettanimations-Sprite verwendet wird. Der Rückgabewert ist eine Zeichenkette, die der Name des Sets ist, wie Sie es in Ihrem Skelettanimationsprogramm definiert haben.

Syntax:

skeleton_animation_get();

Rückgabe:

Zeichenfolge

Beispiel:

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("springen");
}
}

Der obige Code ändert das verwendete Animations-Set in das "Sprung"-Set, wenn die Leertaste gedrückt wird, aber nur, wenn das aktuell verwendete Set nicht bereits "Sprung" ist.
