skelett_animation_get_duration

Diese Funktion gibt die Zeit zurück, die das angegebene Animationsset benötigt, um zu laufen, bevor eine Schleife zum Anfang zurückkehrt. Der Rückgabewert wird in Sekunden angegeben.

Syntax:

skeleton_animation_get_duration(animname);

Argument Beschreibung
animname Der Name (eine Zeichenkette) des zu verwendenden Animationssets.

Rückgabe:

Real

Beispiel1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Der obige Code setzt den image_index auf den richtigen Wert für das aktuell zugewiesene Skelettanimations-Sprite.

Beispiel2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Der obige Code zeichnet das angegebene Skelett-Animationssprite unter Verwendung der Delta-Zeit, um das zu zeichnende Bild festzulegen.
