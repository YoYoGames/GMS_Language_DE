skelett_animation_set_ext

Wenn Sie ein Animationsset für Ihr Skelettanimationssprite einstellen, wird es intern der Spur 0 zugewiesen. Sie können jedoch weitere Sets weiteren Spuren (1 und höher) zuweisen und auf diese Weise Ihr Sprite zwei (oder (oder mehr) Animationen gleichzeitig abspielen. Beachten Sie, dass das Sprite in Ihrem Animationsprogramm so gestaltet sein sollte, dass die "Basis"-Animation in Spur 0 nur Keyframes für bestimmte Bones hat, während die nächsten Spuren Keyframes für die anderen. Wenn Sie z. B. eine "Geh"-Animation haben und deren Keyframes nur die Beine des Sprites betreffen, können Sie dann einen Satz von Kopf- und Arm-Animationen haben, die Sie in Spur 1 bzw. 2 setzen können, so dass so dass Sie eine Basisbewegung haben und diese in Echtzeit anpassen können, während das Spiel gespielt wird.

Syntax:

skeleton_animation_set_ext(animname, track);

Argument Beschreibung
animname Der Name (eine Zeichenkette) des zu verwendenden Animationssets.
Spur Die zu verwendende Spur für das Animationsset.

Rückgabe:

N/A

Beispiel:

if mouse_check_button(mb_left)
{
skeleton_animation_set_ext("bodyfight", 1);
}
sonst
{
skeleton_animation_set_ext("bodywalk", 1);
}

Der obige Code ändert das Animations-Set, das von Spur 1 verwendet wird, in das "bodyfight"-Set oder das "bodywalk"-Set, wenn die linke Maustaste gedrückt wird.
