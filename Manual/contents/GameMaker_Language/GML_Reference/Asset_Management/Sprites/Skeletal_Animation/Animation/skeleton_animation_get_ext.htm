skelett_animation_get_ext

Ein einzelnes Skelett-Animationssprite kann verschiedene Animationssets haben, und diesen Sets können verschiedene Spuren zugewiesen werden, so dass Sie Animationssets "mischen und anpassen" können. Diese Funktion gibt den Namen des Animationssets zurück, das aktuell von der angegebenen Spurnummer (wie von der Funktion skelett_animation_set_ext).

Syntax:

skeleton_animation_get_ext(track);

Argument Beschreibung
Spur Die Spurnummer, von der der Name des Animationssets abgerufen werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

if skeleton_animation_get_ext(1) != "bodyfight"
{
skeleton_animation_set_ext("bodyfight", 1);
}

Der obige Code ändert das Animations-Set, das von Spur 1 verwendet wird, in "bodyfight", wenn es das nicht schon ist.
