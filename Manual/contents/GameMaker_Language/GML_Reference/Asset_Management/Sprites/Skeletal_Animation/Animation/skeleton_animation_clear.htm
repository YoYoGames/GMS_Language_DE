skelett_animation_klar

Mit dieser Funktion wird die angegebene Animationsspur von allen Animationen befreit, so dass sie neu zugewiesen werden kann.

Syntax:

skeleton_animation_clear(track);

Argument Beschreibung
Spur Die Animationsspur zu löschen.

Rückgabe:

N/A

Beispiel:

if mouse_check_button(mb_right)
{
skeleton_animation_clear(1);
}

Mit dem obigen Code wird die Animationsspur 1 gelöscht, wenn die rechte Maustaste gedrückt wird.
