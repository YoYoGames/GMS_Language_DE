skelett_animation_set_frame

Diese Funktion setzt die Animation, die der angegebenen Spur zugeordnet ist, auf das von Ihnen angegebene Bild. Der Frame-Index sollte ein reeller Wert zwischen 0 und der Anzahl der Frames für die Animation sein (die Sie mit der Funktion skelett_animation_get_frames). Wenn Sie einen Wert außerhalb dieses Bereichs angeben, wird die Animation in einer Schleife ausgeführt, um die Differenz auszugleichen, z. B. wenn die Animation 5 Frames hat und Sie den Frame-Index auf 8 setzen, wird das tatsächlich angezeigte Frame 3 sein.

Syntax:

skeleton_animation_set_frame(track, index);

Argument Beschreibung
Spur Die Animationsspur, für die das Bild eingestellt werden soll.
index Der neue Index für den Animationsrahmen.

Rückgabe:

N/A

Beispiel:

if global.pause
{
skeleton_animation_set_frame(0, -image_speed);
}

Der obige Code setzt den Frame-Index für die Animation in Spur 0 auf die negative Bildgeschwindigkeit, wenn die globale "pause"-Variable wahr ist (im Wesentlichen pausiert die Animation).
