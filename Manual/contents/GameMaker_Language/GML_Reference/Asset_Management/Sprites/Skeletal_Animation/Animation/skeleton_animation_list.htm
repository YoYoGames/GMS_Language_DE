skelett_animation_liste

Mit dieser Funktion können Sie eine (vorher erstellte) DS-Liste mit allen Namen der Animationen füllen, die als Teil des Skelettanimations-Sprites enthalten sind. Die Namen sind Strings und können dann in den anderen Animationsfunktionen für diese Typen von Sprites verwendet werden.

Syntax:

skeleton_animation_list(sprite, list);

Argument Beschreibung
Sprite Der Sprite-Index der Wirbelsäulen-Skelettanimation, von der die Liste abgerufen werden soll.
Liste Die ID der DS-Liste, die mit den Animationsnamen gefüllt werden soll.

Rückgabe:

N/A

Beispiel:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);
for (var i = 1; i < ds_list_size(list); i++;)
{
skeleton_animation_mix(list[| 0], list[| i], 0.5);
}
ds_list_destroy(list);

Der obige Code erstellt eine DS-Liste mit allen Animationsnamen für das Sprite, das von der Instanz verwendet wird. Er geht dann in einer Schleife durch diese und setzt den Mischwert für alle mit der ersten Animation auf 0,5.
