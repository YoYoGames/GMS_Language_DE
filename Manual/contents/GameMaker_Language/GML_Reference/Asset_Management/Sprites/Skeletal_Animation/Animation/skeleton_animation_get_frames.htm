skelett_animation_get_frames

Mit dieser Funktion können Sie die Anzahl der Bilder einer bestimmten Skelettanimation abrufen. Sie geben den Namen der Skelett-Animation an (als String, wie er im Programm definiert ist, mit dem die Animation erstellt wurde, oder wie er mit Hilfe der Funktion skelett_animation_holenund die Funktion gibt die Anzahl der Frames als Integer-Wert zurück. Die Funktion gibt 0 zurück, wenn die angegebene Animation nicht existiert.

Syntax:

skeleton_animation_get_frames(anim_name);

Argument Beschreibung
anim_name Der Name der Animation, von der die Bilder abgerufen werden sollen.

Rückgabe:

Integer

Beispiel:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Der obige Code ermittelt die Anzahl der Bilder in der Animation und setzt dann das Sprite auf das letzte Bild und stoppt die Animation.
