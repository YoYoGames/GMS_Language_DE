skelett_animation_mix

Sie können die Animationssätze einfach mit der Funktion skelett_animation_set() Funktion wechseln, aber dies kann zu einem Sprung oder Stottern führen, wenn eine Animation gegen eine andere ausgetauscht wird. Um dies zu verhindern, können Sie den Mix-Wert Um dies zu verhindern, können Sie den Mix-Wert zwischen zwei Animations-Sets setzen, und das Sprite wird zwischen ihnen interpolieren. Normalerweise würden Sie dies im Create-Ereignis der Instanz mit der Skelett-Animation tun wollen, da er nur einmal gesetzt werden muss, und GameMaker Studio 2 wird alle weiteren Alle weiteren Änderungen am Sprite werden von GameMaker Studio 2 unter Verwendung der Animationssätze in dieser Instanz interpoliert. Beachten Sie, dass der Wert für die Dauer von 0 bis 1 reicht, wobei ein Wert von 0,5 eine "halb und halb"-Interpolation von einem Set zum anderen zur Folge hätte.

Syntax:

skeleton_animation_mix(animfrom, animto, duration);

Argument Beschreibung
animfrom Der Name (eine Zeichenkette) des ersten Animationssatzes, aus dem interpoliert werden soll.
animto Der Name (eine Zeichenkette) des zweiten Animationssatzes, auf den interpoliert werden soll.
Dauer Die Dauer der Interpolation (von 0 bis 1)

Rückgabe:

N/A

Beispiel:

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

Der obige Code würde in das Create-Ereignis einer Instanz mit einem Skelett-Animationssprite gehen und setzt die Animations-Mix-Dauer für die Interpolation zwischen den beiden Animations-Sets "walk" und "jump".
