Sprites

Sprites sind im Allgemeinen die visuellen Darstellungen von Objekten innerhalb des Spiels, das Sie erstellen, obwohl sie auch für viele andere Zwecke verwendet werden können. Ein Sprite ist entweder ein einzelnes Bild oder eine Reihe von Bildern, die, wenn sie nacheinander abgespielt werden, wie eine Schleifenanimation aussieht. Sprites können in vielen Formaten erstellt werden, wobei das "traditionelle" Sprite aus Einzelbildern von Pixelbildern besteht, die in GameMaker Studio 2 als *. png-Dateien importiert werden . Sie können aber auch Vektorsprites importieren importieren, die im *.swf-Format gespeichert wurden, oder Skelett-Animationssprites, die als *.json-Dateien mit zugehörigen *.atlas- und *.png-Dateien gespeichert wurden (für weitere Informationen zu diesen Arten von Bilddateien und wie man sie in GameMaker Studio 2 importiert, finden Sie im Abschnitt Importieren von Nicht-Bitmap-Sprites).

Da Sprites ein integraler Bestandteil eines jeden Spiels sind, verfügt GameMaker Studio 2 über eine große Anzahl von Funktionen und Variablen, die verwendet werden können, um Informationen über bestimmte Sprites zu erhalten oder um die Sprites in Ihrem Spiel zu ändern oder sogar um eigene Sprites zur Laufzeit aus einer Quellbilddatei zu laden.
