sprite_get_nineslice

Diese Funktion gibt die Nine-Slice-Struktur für ein Sprite zurück, die alle seine Nine-Slice-Eigenschaften enthält, wie sie im Sprite-Editor oder mit sprite_set_nineslice() eingestellt wurden. Der Inhalt dieser Struktur wird auf dieser Seite detailliert beschrieben.

Wenn dem übergebenen Sprite keine Nine-Slice-Struktur zugewiesen ist, wird eine neue Struktur mit Standard-Nine-Slice-Eigenschaften für das Sprite erstellt und zurückgegeben.

Wenn Sie die Werte in dieser Struktur ändern , werden die Nine Slice-Eigenschaften des ursprünglichen Sprites geändert, was sich auf alle zukünftigen Zeichenaufrufe mit diesem Sprite auswirkt.

Syntax:

sprite_get_nineslice(ind);

Argument Beschreibung
ind Der Index des Sprites, von dem die Nine-Slice-Struktur abgerufen werden soll

Rückgabe:

Struktur (oder -1, wenn das Sprite nicht existiert)

Beispiel:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

Der obige Code ruft die Nine-Slice-Struktur von einem Sprite ab, aktiviert das Nine-Slicing für dieses und setzt die Führungsoffsets.
