sprite_get_speed_type

Mit dieser Funktion kann der Sprite-Animationstyp abgerufen werden, wie er für die Sprite-Ressource im Sprite-Editor definiert ist. Der zurückgegebene Wert ist eine der folgenden Konstanten:

Konstant Beschreibung
spritespeed_framespersecond Das Sprite wurde mit einer Animation in Bildern pro Sekunde definiert.
spritespeed_framespergameframe Das Sprite wurde mit in Animation in Frames pro Spielframe definiert.


Die folgenden zwei Beispiele verdeutlichen den Unterschied, den dies bei der Berechnung der Animationsgeschwindigkeiten für das Sprite machen kann:

  • Wenn Sie ein Sprite haben, das 1 Bild pro Sekunde zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es mit 0,5 Bildern pro Sekunde gezeichnet. Wenn Ihre Spielbildrate 60 Bilder pro Sekunde beträgt, zeichnet das Sprite 1 Bild pro 120 Spielbilder.
  • Wenn Sie ein Sprite haben, das 1 Bild pro Spielbild zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es 0,5 Bilder pro Spielbild zeichnen. Wenn Ihre Spielbildrate 60 Bilder pro Sekunde beträgt, zeichnet das Sprite 30 Bilder pro 60 Spielbilder.

Den Geschwindigkeitswert für die Sprite-Animation können Sie mit der Funktion sprite_get_speed()herausfinden und die Animationsgeschwindigkeit und den Animationstyp mit der Funktion sprite_set_speed().

Syntax:

sprite_get_speed_type(index)

Argument Beschreibung
index Der Index des Sprites, um den Animationstyp von

Rückgabe:

Konstant (siehe oben)

Beispiel:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Der obige Code ruft die Sprite-Geschwindigkeit und den Sprite-Animationstyp ab und speichert sie in Instanzvariablen für die zukünftige Verwendung.
