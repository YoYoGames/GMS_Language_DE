sprite_get_bbox_mode

Diese Funktion gibt den aktuellen "Modus" für die Bounding-Box-Berechnungen zurück. Sie geben den Sprite-Index des zu prüfenden Sprites an, und die Funktion gibt eine der unten aufgeführten Konstanten zurück.

Syntax:

sprite_get_bbox_mode(ind);

Argument Beschreibung
ind Der Index des zu prüfenden Sprites.

Rückgabe

Konstant

Konstant Beschreibung
bboxmode_automatic Automatisch - Der Begrenzungsrahmen wird automatisch berechnet, basierend auf der Toleranzeinstellung für das Sprite
bboxmode_fullimage Volles Bild - Der Begrenzungsrahmen wird so eingestellt, dass er die volle Breite und Höhe des Sprites verwendet, unabhängig von der Toleranz und den "leeren" Pixeln
bboxmode_manual Manuell - Der Begrenzungsrahmen wurde manuell auf benutzerdefinierte Werte gesetzt (entweder im Sprite-Editor oder mit der Funktion sprite_set_bbox())

Beispiel:

if sprite_get_bbox_mode(sprite_index) != 0
{
sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Das obige prüft den bbox-Modus für das aktuelle Sprite und wenn er nicht automatisch ist , dann wird er auf diesen Wert gesetzt.
