Sprite-Informationen

Die folgenden Funktionen können verwendet werden, um Informationen über ein Sprite-Asset zu erhalten. Bitte beachten Sie, dass diese Funktionen die Informationen aus dem Basis-Sprite (d. h. dem Sprite-Asset, wie es im Asset-Browser erstellt wurde) abrufen und nicht von dem Sprite, wie es in einer Instanz verwendet wird. Sie können zum Beispiel die sprite Instanz-Variable sprite_width verwenden, um die Breite des Sprites zu ermitteln, das in einer Instanz verwendet wird, und diese Breite berücksichtigt alle Skalierungen, die möglicherweise vorgenommen wurden. Wenn Sie jedoch die Breite eines Sprites wissen müssen, wie sie normalerweise wäre, ohne angewandte Transformationen, müssen Sie die folgende Funktion sprite_get_width() verwenden.

Weitere Informationen zu den Sprite-Instanzvariablen finden Sie im Abschnitt über Instanzvariablen.
