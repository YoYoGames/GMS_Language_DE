sprite_get_width

Mit dieser Funktion können Sie die Breite des Basis-Ressource-Sprites, ohne Transformationen, in Pixel ermitteln.

Syntax:

sprite_get_width(index);

Argument Beschreibung
index Der Index des Sprites, für das die Breite ermittelt werden soll.

Rückgabe

Real

Beispiel:

if sprite_width != sprite_get_width(sprite_index)
{
image_xscale = 1;
}

Der obige Code prüft die Breite des Sprites, wie sie in der aktuellen Instanz ist, und wenn es einen Unterschied zwischen dem und dem ursprünglichen Basis-Sprite gibt, setzt er die Skalierung der X-Achse zurück.
