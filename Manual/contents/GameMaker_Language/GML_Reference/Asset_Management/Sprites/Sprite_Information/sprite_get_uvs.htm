sprite_get_uvs

Diese Funktion gibt ein Array mit den UV-Koordinaten und anderen Daten für die Textur des Sprite-Unterbildes auf der Textur-Seite zurück. Die Funktion gibt ein Array mit den folgenden 8 Elementen zurück:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten
  • [4] = Anzahl der Pixel, die der Asset-Compiler von der linken Seite des Sprites abgeschnitten hat
  • [5] = Anzahl der Pixel, die der Asset-Compiler von der Oberseite des Sprites abgeschnitten hat
  • 6] = normalisierter Prozentsatz der Pixeldaten aus der ursprünglichen Sprite-Breite, die in der Texturseite gespeichert wurden
  • 7] = normalisierter Prozentsatz der Pixeldaten von der ursprünglichen Höhe des Sprites, der in der Texturseite gespeichert wurde

Dieses Array kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere bei der Arbeit in 3D oder bei der Verwendung der 2D-Primitivfunktionen, sowie bei der Arbeit mit den Shader Funktionen.

HINWEIS: Diese Funktion funktioniert nicht mit Vektor-Sprites oder Skelett-Animations-Sprites.

Syntax:

sprite_get_uvs(sprite, subimage);

Argument Beschreibung
Sprite Der Index des zu verwendenden Sprites.
Unterbild Das Unterbild des zu verwendenden Sprites.

Rückgabe

Array (1D, 8 Elemente)

Beispiel:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Der obige Code speichert die UV-Koordinaten für das angegebene Sprite in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
