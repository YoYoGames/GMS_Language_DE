sprite_get_tpe

Diese Funktion gibt einen speziellen Wert zurück, der sich auf die Position des angegebenen Sprites und Sub-Images auf der von Ihrem Spiel verwendeten Textur-Seite bezieht. Auf diese Weise können Sie den Wert an Erweiterungen zur Verwendung außerhalb des kompilierten Spiels übergeben und effektiv die die GameMaker Studio 2-Zeichenfunktionen zu umgehen und das Sprite in DOM-Inhalten zu verwenden, die dann an einer beliebigen Stelle innerhalb des Fensters, das den Spiel-Canvas enthält, gezeichnet werden können. Diese Funktion ist besonders interessant für diejenigen, die Schaltflächen und andere interaktive Medien außerhalb des GameMaker Studio 2 Canvas-Elements auf ihrer Host-Seite mit der Funktion clickable_add().

HINWEIS: Diese Funktion ist nur für HTML5.

Syntax:

sprite_get_tpe(sprite, index);

Argument Beschreibung
Sprite Der Index des Sprites, von dem der Textur-Seiteneintrag gefunden werden soll.
index Der Bildindex des Sprites.

Rückgabe

Real

Beispiel:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Der obige Code erstellt eine DOM-Schaltfläche an der Position (32, 32) der Seite, auf der das Game Canvas läuft. Die Schaltfläche verwendet das Sprite, das von der Texturseite als "spr_MS_Home" referenziert wird, und wenn die Schaltfläche angeklickt wird, öffnet sich ein neues Fenster für der angegebenen URL und mit den definierten Eigenschaften für dieses Fenster.
