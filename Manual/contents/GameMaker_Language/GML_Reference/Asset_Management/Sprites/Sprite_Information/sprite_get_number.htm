sprite_get_number

Ein Sprite muss mindestens ein Sub-Image haben und diese Funktion gibt nur zurück, wie viele es hat. Denken Sie daran, dass ein Sprite-Sub-Image von 0 an nummeriert wird. Wenn diese Funktion also 3 zurückgibt (das geprüfte Sprite hat 3 Sub-Images), werden diese nummeriert 0, 1 und 2.

Syntax:

sprite_get_number(index);

Argument Beschreibung
index Der Index des Sprites.

Rückgabe

Integer

Beispiel:

image_index = floor(random(sprite_get_number(sprite_index)));

Der obige Code setzt den Bildindex auf ein zufälliges Bild, basierend auf der Anzahl der Unterbilder, die das Sprite hat.
