sprite_get_xoffset

Wenn Sie ein Sprite im Sprite-Editor definieren, müssen Sie den Ursprung für dieses Sprite festlegen. Dies ist der Punkt, an dem das Sprite "angehängt" oder "gezeichnet" wird, wenn es mit einer Instanz verwendet wird. Diese Funktion gibt den relativen Offset für die x-Achse des Ursprungs zurück, basierend auf der oberen linken Ecke, die die Position 0,0 ist, wobei ein +x rechts und ein -x links davon ist. Das folgende Bild veranschaulicht dies:

Syntax:

sprite_get_xoffset(index);

Argument Beschreibung
index Der Index des Sprites, dessen X-Offset gesucht werden soll.

Rückgabe

Real

Beispiel:

if x - sprite_get_xoffset(sprite_index) < 0
{
x = sprite_get_xoffset(sprite_index);
}

Der obige Code stellt sicher, dass eine Instanz innerhalb des Raums basierend auf dem Sprite beibehalten wird (so dass das Sprite immer sichtbar ist).
