sprite_get_info

Diese Funktion wird verwendet, um Informationen für das angegebene Sprite abzurufen. Sie geben einen Sprite-Index an (der ein über den Asset-Browser hinzugefügtes Asset oder ein zur Laufzeit hinzugefügtes Sprite sein kann) und die Funktion gibt eine Struktur mit den folgenden Variablen zurück:

Variable Name Daten Typ Beschreibung
Breite real Die Breite des Sprites (in Pixel)
Höhe real Die Höhe des Sprites (in Pixel)
xOffset real Der X-Offset/Ursprung des Sprites (in Pixel)
yOffset real Y-Offset/Ursprung des Sprites (in Pixel)
transparent boolean true, wenn das Sprite als transparent markiert ist, sonst false.
(Dies kann nur über sprite_add() oder ähnliche Funktionen angegeben werden und ist für Sprites, die in der IDE erstellt wurden, false)
glatt boolean true, wenn das Sprite als glatt markiert ist, sonst false
(Dies kann nur durch sprite_add() oder ähnliche Funktionen angegeben werden und wird für Sprites, die in der IDE erstellt wurden, false sein)
Typ real Der Typ des Sprites:
0 - Bitmap (Reguläre Sprites)
1 - SWF
2 - Wirbelsäule
bbox_left real Position des linken Randes der Bounding Box (in Pixel)
bbox_top real Position der oberen Kante des Begrenzungsrahmens (in Pixel)
bbox_right real Position des rechten Randes der Bounding Box (in Pixel)
bbox_bottom real Position der unteren Kante des Begrenzungsrahmens (in Pixel)
Name String Der Name des Sprites
num_subimages real Die Anzahl der Sub-Images (oder Frames) im Sprite
verwenden_maske boolean true, wenn dieses Sprite eine Kollisionsmaske verwendet, sonst false
num_masks real Die Anzahl der Masken in diesem Sprite (wird größer als 1 sein, wenn das Sprite Per-Frame-Masken verwendet)
Neuner-Scheibe struct Die Nine-Slice-Struktur für dieses Sprite, oder undefiniert, wenn es keine Nine-Slice-Daten hat
Meldungen Array Array von Broadcast-Nachrichten für dieses Sprite, wobei jede Broadcast-Nachricht ein struct ist, das Informationen über die Nachricht enthält (weitere Informationen unter "Allgemeine Sprite-Daten")
frame_info Array Array von Frames für dieses Sprite, wobei jedes Frame ein struct ist, das Informationen zu seinem Timing enthält (weitere Informationen unter "Allgemeine Sprite-Daten")

Diese zusätzliche Variable ist nur für Bitmap (reguläre) Sprites verfügbar:

Variable Name Daten Typ Beschreibung
Rahmen Array Array von Frames für dieses Sprite, wobei jedes Frame ein struct ist, das Informationen über seine Textur enthält (weitere Informationen unter "Bitmap-Sprite-Daten")

Diese zusätzlichen Variablen sind nur für Spine-Sprites verfügbar:

Variable Name Daten Typ Beschreibung
num_atlas real Die Anzahl der verwendeten Atlastexturen
atlas_textur Array Array mit Textur-IDs, die für den Atlas verwendet werden
vormultipliziert boolean true, wenn dieses Sprite als vormultipliziert markiert ist, sonst false
animation_names Array Array mit den Namen der einzelnen Animationen in diesem Sprite
skin_names Array Array mit den Namen der einzelnen Skins in diesem Sprite
Knochen Array Array mit Strukturen für jeden Bone in diesem Sprite (weitere Informationen unter "Spine-Sprite-Daten")
Steckplätze Array Array mit Strukturen für jeden Slot in diesem Sprite (weitere Informationen unter "Spine Sprite Data")

Die Funktion gibt einen undefinierten Wert zurück, wenn das angegebene Sprite nicht vorhanden ist. Beachten Sie auch, dass die in dieser Struktur zurückgegebenen Informationen als schreibgeschützt betrachtet werden sollten, da das Ändern einer dieser Variablen keine Auswirkungen auf das Sprite hat.

Die folgenden Abschnitte enthalten Informationen zu den Arrays und Strukturen, die in der zurückgegebenen Struktur enthalten sind, basierend auf dem Sprite-Typ:

Allgemeine Sprite-Daten

Dieser Abschnitt enthält Informationen zu Variablen, die in der Struktur für alle Arten von Sprites enthalten sind.

Die Variable messages ist ein Array, das Informationen über die Broadcast-Nachrichten enthält, die im angegebenen Sprite vorhanden sind. Jede Broadcast-Nachricht in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Daten Typ Beschreibung
Rahmen real Das Timing dieser Broadcast-Nachricht ab dem Start der Animation (in Frames)
Nachricht String Der String der Broadcast-Nachricht

Die Variable "frame_info" ist ein Array, das Informationen über die Timings der Frames des Sprites enthält. Jedes Frame in diesem Array ist ein struct, das die folgenden Variablen enthält:

Variable Name Daten Typ Beschreibung
Rahmen real Das Timing für den Start dieses Frames (in Frames)
image_index real Der Bildindex dieses Rahmens

Bitmap-Sprite-Daten

Dieser Abschnitt enthält Informationen zu Variablen, die in der Struktur für Bitmap-Sprites enthalten sind (Sprites, die nicht vom Typ Spine oder SWF sind).

Die Variable frames ist ein Array, das Informationen über die Texturen der Frames des Sprites enthält. Jeder Rahmen in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Daten Typ Beschreibung
x real Die X-Position dieses Rahmens auf seiner Texturseite (in Pixel)
y real Die Y-Position dieses Rahmens auf seiner Texturseite (in Pixel)
w real Die logische Breite des Rahmens (in Pixel), die intern verwendet wird
h real Die logische Höhe des Rahmens (in Pixel), die intern verwendet wird
Textur real Die Texturseiten-ID für diesen Rahmen
ursprüngliche_Breite real Die ursprüngliche Breite des Rahmens (in Pixel)
ursprüngliche_Höhe real Die ursprüngliche Höhe des Rahmens (in Pixel)
beschneiden_breite real Die tatsächliche Breite des Rahmens auf der Texturseite nach dem Beschneiden und Skalieren (da GMS2 automatisch den leeren Raum um ein Bild herum beschneidet und es auch verkleinert, wenn es nicht passt)
crop_height real Die tatsächliche Höhe des Rahmens auf der Texturseite nach dem Zuschneiden und Skalieren
x_Offset real Der X-Versatz von der linken Kante des Originalbildes zur linken Kante des ausgeschnittenen Ausschnitts
y_Offset real Der Y-Versatz von der Oberkante des Originalbildes zur Oberkante des ausgeschnittenen Ausschnitts

Wirbelsäulen-Sprite-Daten

Dieser Abschnitt enthält Informationen zu Variablen, die in der Struktur für Spine-Sprites enthalten sind.

Die Bones-Variable ist ein Array, das Informationen über alle Bones im angegebenen Spine-Sprite enthält. Jeder Bone in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Daten Typ Beschreibung
Elternteil String Der Name des übergeordneten Bones, oder undefiniert, wenn dieser Bone keinen übergeordneten Bone hat
Name String Der Name dieses Knochens
index real Der Index dieses Knochens
Länge real Die Länge dieses Knochens
x real Die X-Position dieses Knochens
y real Die Y-Position dieses Knochens
Rotation real Die Rotation dieses Knochens
skala_x real (Intern zur Wirbelsäule) Skalenwert an X
skala_y real (Intern zur Wirbelsäule) Skalenwert auf Y
scher_x real (Intern zur Wirbelsäule) Scherwert an X
scheren_y real (Intern zur Wirbelsäule) Scherwert an Y
transform_mode real (Intern zu Spine) Der Transformationsmodus

HINWEIS: Weitere Informationen zu den internen Variablen von Spine entnehmen Sie bitte der Spine-Dokumentation.

Die Variable " slots" ist ein Array, das Informationen über alle Slots im angegebenen Spine-Sprite enthält. Jeder Slot in diesem Array ist eine Struktur, die die folgenden Variablen enthält:

Variable Name Daten Typ Beschreibung
Name String Der Name des Steckplatzes
index real Der Index des Steckplatzes
Knochen String Der Name des Bones des Steckplatzes, oder "( none)", wenn es keinen Bone für diesen Steckplatz gibt
Anhang String Name des Anhängers
rot real Rotanteil der Farbe des Steckplatzes (0-1)
grün real Grünanteil der Farbe des Steckplatzes (0-1)
blau real Blauanteil der Farbe des Steckplatzes (0-1)
alpha real Alpha-Komponente der Farbe des Slots (0-1)
blendMode real (Intern zu Spine) Überblendmodus für den Steckplatz
dunkel_rot* real Rotanteil der dunklen Farbe des Steckplatzes (0-1)
dunkel_grün* real Grünanteil der dunklen Farbe des Steckplatzes (0-1)
dunkel_blau* real Blauanteil der dunklen Farbe des Steckplatzes (0-1)
dunkel_alpha* String Alpha-Komponente der dunklen Farbe des Slots (0-1)

* HINWEIS: Die dunklen Farbvariablen sind nur verfügbar, wenn der Steckplatz Tint Black aktiviert hat.

Syntax:

sprite_get_info(index);

Argument Beschreibung
index Der Index des Sprites, für das die Informationen abgerufen werden sollen.

Rückgabe

Struktur (oder undefiniert)

Beispiel:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
var _messages = _info.messages;
var _messageCount = array_length(_messages);

for (var i = 0; i < _messageCount; i ++)
{
var _message = _messages[i];
show_debug_message("Meldung bei Frame " + string(_message.frame) + ": " + _message.message);
}
}

Der obige Code ruft die Info-Struktur für das Sprite der Instanz ab und prüft dann, ob es nicht undefiniert ist und ob sein Typ 0 ist (was bedeutet, dass es sich um ein Bitmap-Sprite handelt). In diesem Fall wird das Broadcast-Message-Array für dieses Sprite abgerufen und führt dann eine for-Schleife aus, um jede Broadcast-Nachricht (zusammen mit ihrem Frame) in das Ausgabeprotokoll zu drucken.
