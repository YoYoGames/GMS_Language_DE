sprite_get_texture

Diese Funktion gibt einen speziellen Zeiger für die Sprite-Texturseite zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in den 2D-Primitivfunktionen, als auch in den Shader Funktionen. Weitere Informationen über die zurückgegebene Texturseite erhalten Sie über die verschiedenen texture_-Funktionen, die Sie hier finden.

Syntax:

sprite_get_texture(spr, subimg);

Argument Beschreibung
spr Der Index des zu verwendenden Sprites.
subimg Das Unterbild des zu verwendenden Sprites.

Rückgabe

Zeiger (Texturseiten-ID)

Beispiel:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Der obige Code zeichnet einen Dreiecksstreifen mit 4 Eckpunkten, der mit der Textur in der Variablen "tex" texturiert ist.
