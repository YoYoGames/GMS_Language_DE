sprite_get_yoffset

Wenn Sie ein Sprite im Sprite-Editor definieren, müssen Sie den Ursprung für dieses Sprite festlegen. Dies ist der Punkt, an dem das Sprite "angehängt" oder "gezeichnet" wird, wenn es mit einer Instanz verwendet wird. Diese Funktion gibt den relativen Offset für die y-Achse des Ursprungs zurück, basierend auf der oberen linken Ecke, die die Position 0,0 ist, wobei ein +y nach unten und ein -y nach oben von dieser Position aus bedeutet. Das folgende Bild veranschaulicht dies:

Syntax:

sprite_get_yoffset(index);

Argument Beschreibung
index Der Index des Sprites, von dem der y-Offset gesucht werden soll.

Rückgabe

Real

Beispiel:

if y - sprite_get_yoffset(sprite_index) < 0
{
y = sprite_get_yoffset(sprite_index);
}

Der obige Code stellt sicher, dass eine Instanz innerhalb des Raums basierend auf dem Sprite beibehalten wird (so dass das Sprite immer sichtbar ist).
