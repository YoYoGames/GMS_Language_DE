sprite_get_name

Diese Funktion gibt den Namen des angegebenen Sprites als String zurück. Dieser Name ist derjenige, der für das Sprite im Ressourcenbaum des Hauptfensters von GameMaker Studio 2 angegeben wurde. Bitte beachten Sie, dass dies nur ein String ist ist und nicht verwendet werden kann, um das Sprite direkt zu referenzieren - dazu würden Sie den Sprite-Index benötigen. Sie können diesen String jedoch verwenden, um den Sprite-Index zu erhalten, indem Sie den zurückgegebenen String zusammen mit der Funktion asset_get_index().

Syntax:

sprite_get_name(index);

Argument Beschreibung
index Der Index des Sprites, von dem der Name abgerufen werden soll.

Rückgabe

Zeichenfolge

Beispiel:

str = sprite_get_name(sprite_index);

Der obige Code ermittelt den Namen des Sprite-Index für die Instanz, die den Code ausführt, und speichert den Rückgabewert in der Variablen "str".
