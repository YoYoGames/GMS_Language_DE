Neun Slice-Funktionen

Zusätzlich zum Nine Slice-Bereich im Sprite-Editor, GameMaker Studio 2 bietet Funktionen zum Setzen von Nine-Slice-Eigenschaften für Sprites zur Laufzeit. Bevor Sie sich mit diesen Funktionen beschäftigen, sollten Sie diese Seite lesen, um ein Verständnis für die Nine-Slice-Technik zu bekommen.

Die folgenden Funktionen können verwendet werden, um Nine Slice-Eigenschaften für ein Sprite-Asset zur Laufzeit abzurufen und zu ändern:

  • sprite_nineslice_create
  • sprite_get_nineslice
  • sprite_set_nineslice

Diese Funktionen verwenden eine Nine Slice-Struktur, deren Inhalt Sie auf dieser Seite nachlesen können.
