sprite_bündig

Mit dieser Funktion können Sie die angegebene Texturseite für das angegebene Sprite aus dem Texturspeicher entfernen. Sie geben den Sprite-Index an (wie beim Erstellen der Sprite-Ressource definiert) und die Textur-Seite, der sie zugewiesen ist, wird aus dem Textur-Speicher entfernt. Beachten Sie, dass, wenn die Texturseite an anderer Stelle im Raum verwendet wird (durch ein anderes Instanz-Sprite oder einen Hintergrund, etc...), Sie einen kleinen Performance-Schlag bekommen können, da die Seite sofort neu geladen wird, daher sollte man bei der Verwendung dieser Funktion vorsichtig sein. Beachten Sie, dass die Funktion -1 zurückgibt, wenn flush für die gewählte Ressource nicht unterstützt wird, oder sie gibt 0 zurück, wenn alles korrekt funktioniert hat.

Syntax:

sprite_flush(ind)

Argument Beschreibung
ind Der Index (Ressourcenname) des zu spülenden Sprite-Assets

Rückgabe:

Int (-1 oder 0)

Beispiel:

sprite_flush(spr_Player_Aura);

Der obige Code löscht das Sprite "spr_Player_Aura" aus dem Speicher.
