sprite_add_from_surface

Diese Funktion arbeitet genau wie die Funktion sprite_erzeugen_aus_Fläche() nur dass sie, anstatt ein neues Sprite aus dem von Ihnen gewählten Bereich der indizierten Oberfläche zu erzeugen, den fügt den definierten Bereich der Oberfläche als neues Sub-Image zu einem zuvor erstellten Sprite hinzu (das bedeutet, dass Sie es nicht direkt zu einem Sprite aus dem Ressourcenbaum hinzufügen können, sondern nur zu einem, das zuvor aus einer Oberfläche erstellt wurde, oder zu einem, das dupliziert wurde aus einem Ressource-Sprite mit sprite_duplicate()).

Es ist zu beachten, dass der Speicherbedarf beim Hinzufügen eines Sprites auf diese Weise größer ist, als Sie vielleicht erwarten. Das liegt daran, dass GameMaker Studio 2 das Sprite als Texturseite speichert und es Das liegt daran, dass GameMaker Studio 2 das Sprite als Texturseite speichert und es auch im GPU-Speicher abgelegt wird, so dass der Gesamtspeicher größer ist als bei einer Bilddatei desselben Sprites zu erwarten wäre.

Syntax:

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

Argument Beschreibung
index Der Index des Sprites, zu dem das neue Bild hinzugefügt werden soll.
Oberfläche Der Index der Oberfläche, von der Sie das Bild erhalten.
x Die x-Position, ab der kopiert werden soll.
y Die y-Position, ab der kopiert werden soll.
w Die Breite des zu kopierenden Bereichs (ab der x-Position).
h Die Höhe des zu kopierenden Bereichs (ab der y-Position).
zurück entfernen Gibt an, ob alle Pixel mit der Hintergrundfarbe (linkes unteres Pixel) transparent gemacht werden sollen.
glatt Gibt an, ob die Kanten geglättet werden sollen.

Rückgabe:

N/A

Beispiel:

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr_Custom, surf, i, 0, 32, 32, true, true, 16, 16);
}

Der obige Code erzeugt ein Sprite aus der in der Variablen "surf" indizierten Oberfläche, wobei sein Index der Variablen "spr_Custom" zugewiesen wird, und verwendet dann eine for-Schleife, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte, die als Teilbilder in das Sprite eingefügt werden.
