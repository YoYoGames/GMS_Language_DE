sprite_delete

Diese Funktion löscht ein Sprite aus dem Spiel und gibt den dafür reservierten Speicher frei. Dies ist eine sehr wichtige Funktion für die Momente, in denen Sie Sprites aus externen Quellen erstellen und ändern müssen (wie das Laden eines Sprites aus einer Datei mit sprite_add() , oder das Duplizieren eines Sprites mit sprite_duplicate()) und sollte immer verwendet werden, um Assets zu entfernen, die nicht mehr im Spiel benötigt werden, oder um den Speicher zu löschen. nicht mehr benötigt werden, oder um ein indiziertes Element aus einer Variablen zu löschen, bevor ein anderes Element dieser Variablen neu zugewiesen wird. Dies kann nicht verwendet werden, um Sprites zu löschen, die im Spiel als Teil der Assets im Asset-Browser enthalten sind.

Syntax:

sprite_delete(index);

Argument Beschreibung
index Der Index des zu löschenden Sprites.

Rückgabe

N/A

Beispiel:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

Der obige Code erstellt eine lokale Variable und speichert dann den Index des Sprites, das von der Anwendungsoberfläche erstellt wurde. Dieses Sprite wird dann mit dem in der Variablen "spr_Player" indizierten Asset zusammengeführt, bevor es wieder aus dem Speicher entfernt wird.
