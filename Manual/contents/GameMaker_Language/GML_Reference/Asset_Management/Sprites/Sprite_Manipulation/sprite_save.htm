sprite_save

Mit dieser Funktion können Sie ein beliebiges Teilbild eines Sprites unter dem angegebenen Dateinamen auf der Festplatte speichern. Das Sprite muss zur Laufzeit hinzugefügt worden sein (Sie können keine Sprites speichern, die über die IDE hinzugefügt wurden) und die Datei muss mit einer *.png-Erweiterung gespeichert werden.

HINWEIS: Je nach gewählter Zielplattform sind Sie eingeschränkt, woher Sie Dateien speichern und laden können. Weitere Informationen finden Sie unter Dateihandhabung.

Syntax:

sprite_save(ind, subimg, fname);

Argument Beschreibung
ind Der Index des zu speichernden Sprites.
subimg Das Teilbild des zu speichernden Sprites.
fname Der Dateiname für das gespeicherte Sprite.

Rückgabe

N/A

Beispiel:

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

Der obige Code erzeugt eine Oberfläche und speichert ihren Index in der lokalen Variablen "surf". Er zielt dann auf diese Oberfläche, löscht sie und zeichnet mehrere Sprites übereinander. Er erstellt ein neues Sprite aus dem zusammengesetzten Bild, das auf der Oberfläche gezeichnet wurde und weist seinen Index der lokalen Variablen "spr_Custom" zu, bevor es den von der Oberfläche verwendeten Speicher freigibt. Schließlich wird das resultierende Sprite in einer Datei gespeichert und dann ebenfalls aus dem Speicher entfernt.
