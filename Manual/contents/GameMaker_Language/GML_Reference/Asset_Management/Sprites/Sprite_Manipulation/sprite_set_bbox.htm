sprite_set_bbox

Mit dieser Funktion können Sie die Bounding-Box-Werte für ein Sprite festlegen. Sie geben den zu verwendenden Sprite-Index und dann die linken, oberen, rechten und unteren Werte für die Positionen des Begrenzungsrahmens an. Die Positionen sind absolute Werte, wobei die Position (0, 0) der linken oberen Ecke des Sprites entspricht, unabhängig vom Offset für das Sprite, von "leeren" Pixeln, die das Sprite haben kann, oder davon, wo es im Raum gezeichnet wird.

HINWEIS: Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle weiteren Instanzen mit diesem Sprite denselben Begrenzungsrahmen haben.

Syntax:

sprite_set_bbox(ind, left, top, right, bottom);

Argument Beschreibung
ind Der Index des Sprites, auf das die Bounding Box gesetzt werden soll.
links Die linke Seite des Begrenzungsrahmens
oben Der obere Rand des Begrenzungsrahmens.
rechts Die rechte Seite des Begrenzungsrahmens
unten Der untere Rand des Begrenzungsrahmens.

Rückgabe

N/A

Beispiel:

if sprite_get_bbox_mode(sprite_index) == 2
{
left = irandom(sprite_width / 2);
right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
top = irandom(sprite_height / 2);
bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
sprite_set_bbox(sprite_index, left, top, right, bottom);
}

Der obige Code prüft den Begrenzungsrahmenmodus des Sprites, das dem sprite_index zugewiesen ist, und wenn er auf manuell eingestellt ist, wird sein Begrenzungsrahmen geändert.
