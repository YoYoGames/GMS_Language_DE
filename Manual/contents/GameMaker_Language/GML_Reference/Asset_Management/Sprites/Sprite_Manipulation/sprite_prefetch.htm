sprite_prefetch

Diese Funktion kann verwendet werden, um eine Texturseite mit dem angegebenen Sprite vorzuholen (im Texturspeicher abzulegen). Sie geben den Sprite-Index an, wie er beim Erstellen des Sprite-Assets definiert wurde, und die Textur-Seite, auf der es sich befindet, wird in den Speicher geladen. Beachten Sie, dass die Funktion -1 zurückgibt, wenn Prefetch für die gewählte Ressource nicht unterstützt wird oder die Zielplattform HTML5 ist, oder sie gibt 0 zurück, wenn alles korrekt funktioniert hat.

HINWEIS: Es gibt einen Leistungseinbruch, da die Textur auf den meisten Geräten in den Texturspeicher hochgeladen wird, daher wird empfohlen, dass Sie sprite_prefetch auf am Anfang eines Levels aufzurufen, um einen Stillstand zu vermeiden.

Syntax:

sprite_prefetch(ind)

Argument Beschreibung
ind Der Sprite-Index zum Abrufen

Rückgabe:

Int (-1 oder 0)

Beispiel:

sprite_prefetch(spr_Player_Aura);

Der obige Code platziert das referenzierte Sprite in den Texturspeicher und macht es zur Verwendung bereit.
