sprite_assign

Diese Funktion nimmt zwei zuvor erstellte (oder eingebundene) Sprite-Indizes und kopiert das Bild von einem in den anderen. Auf diese Weise können Sie ein Sprite in einen anderen Index kopieren (oder "klonen"). Beachten Sie, dass Sie nicht in eine Spielressource kopieren können. Sie müssen das Sprite, in das kopiert werden soll, zuvor mit den Befehlen sprite_add() oder sprite_duplicate() Funktionen.

Syntax:

sprite_assign(index, sprite);

Argument Beschreibung
index Der Index des Sprites, in das kopiert (d. h. überschrieben) werden soll.
Sprite Das Sprite, von dem kopiert werden soll.

Rückgabe

N/A

Beispiel:

if sprite_exists(global.Player_Sprite)
{
var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
sprite_assign(global.Player_Sprite, t_spr);
sprite_delete(t_spr);
}
sonst
{
global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Der obige Code prüft, ob die globale Variable "Player_Sprite" ein Sprite enthält, und wenn ja, verwendet er sprite_assign, um es gegen ein Sprite auszutauschen, das aus einer externen Datei geladen wurde. Wenn sie kein Sprite enthält, wird eines geladen und sein Index wird in dieser Variablen gespeichert.
