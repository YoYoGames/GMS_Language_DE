sprite_set_offset

Diese Funktion kann verwendet werden, um den x- und y-Ursprung eines Sprites zu setzen, und nimmt relative Werte basierend auf der (0,0)-Position, die die obere linke Ecke des Sprites ist. Das folgende Bild veranschaulicht dies:

HINWEIS: Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle weiteren Instanzen mit diesem Sprite denselben Offset haben werden.

Syntax:

sprite_set_offset(ind, xoff, yoff);

Argument Beschreibung
ind Der Index des Sprites, dessen Offset geändert werden soll.
xoff Die x-Position des Ursprungs.
yoff Die y-Position des Ursprungs.

Rückgabe

N/A

Beispiel:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

Der obige Code weist das in "spr_Base" indizierte Sprite dem in "spr_Custom" indizierten Sprite zu und verwendet dann die x- und y-Offsetwerte von "spr_Base", um den Ursprung des neuen Sprites festzulegen.
