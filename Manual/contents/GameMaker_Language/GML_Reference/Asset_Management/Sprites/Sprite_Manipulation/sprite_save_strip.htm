sprite_save_strip

Diese Funktion erstellt ein Streifenbild aus allen Teilbildern eines Sprites und speichert es unter dem angegebenen Dateinamen auf der Festplatte. Das Sprite muss zur Laufzeit hinzugefügt worden sein (Sie können keine Sprites speichern, die über die IDE hinzugefügt wurden) und die Datei muss mit einer *.png-Erweiterung gespeichert werden.

HINWEIS: Je nach gewählter Zielplattform sind Sie eingeschränkt, woher Sie Dateien speichern und laden können. Weitere Informationen finden Sie unter Dateihandhabung.

Syntax:

sprite_save_strip(ind, filename);

Argument Beschreibung
ind Der Index des Sprites, dessen Transparenz-Map geändert werden soll.
Dateiname Der Dateiname für den gespeicherten Sprite-Streifen.

Rückgabe

N/A

Beispiel:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Der obige Code erstellt ein Sprite aus der in der Variablen "surf" indizierten Oberfläche, wobei sein Index der lokalen Variablen "spr_Custom" zugewiesen wird, und verwendet dann eine for-Schleife, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte zu erfassen, die Dieses neue Sprite wird dann als png-Streifen gespeichert, bevor es aus dem Speicher entfernt wird.
