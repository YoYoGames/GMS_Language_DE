sprite_exists

Diese Funktion gibt zurück, ob ein Sprite mit dem angegebenen Index im ausgeführten Projekt existiert oder nicht.

Syntax:

sprite_exists(index);

Argument Beschreibung
index Der Index des zu prüfenden Sprites.

Rückgabe

N/A

Beispiel:

if sprite_exists(spr_array[0])
{
sprite_index = spr_array[0];
}
sonst
{
sprite_index = spr_BaseSprite;
}

Der obige Code prüft ein Array, um zu sehen, ob es einen gültigen Sprite-Index enthält, und wenn ja, weist er dieses Sprite der Instanz zu, aber wenn nicht, weist er ein Sprite aus den enthaltenen Ressourcen zu.
