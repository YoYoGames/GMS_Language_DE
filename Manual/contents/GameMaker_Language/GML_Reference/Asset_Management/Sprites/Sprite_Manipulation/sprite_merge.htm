sprite_merge

Diese Funktion fügt das in Argument 1 ("ind2") indizierte Sprite mit dem in Argument 0 ("ind1") indizierten zusammen. Die Bilder selbst werden NICHT zusammengeführt, sondern die Bildindizes werden Die Bilder selbst werden NICHT zusammengeführt, sondern die Bildindizes werden zusammengeführt, wobei die Unterbilder von Sprite "ind2" an die von Sprite "ind1" angehängt werden, d. h., sie werden am Ende hinzugefügt. Beachten Sie, dass, wenn die Sprites unterschiedlich groß sind, die angehängten Sprites gestreckt werden, damit sie in die Bildgröße für "ind1".

Diese Änderung ist permanent, und von dem Moment an, in dem Sie diese Funktion verwenden, bis das Spiel geschlossen oder das Sprite gelöscht wird, wird das Sprite, in das es zusammengeführt wird, geändert, das Sprite, mit dem es zusammengeführt wird, bleibt jedoch gleich.

HINWEIS: Weder das Bild-Asset, in das zusammengeführt werden soll, noch das Bild-Asset, das zusammengeführt werden soll, kann eine der ursprünglichen Spielressourcen sein. Sie müssen sprite_duplicate() zuerst zwei neue Ressourcen erstellen (wenn Sie enthaltene Spiel-Assets zusammenführen) und diese dann zusammenführen.

HINWEIS: Diese Funktion ist nur für Bitmap-Sprites nützlich und funktioniert nicht mit SWF- oder JSON (Spine)-Sprites.

Syntax:

sprite_merge(ind1, ind2);

Argument Beschreibung
ind1 Der Index des zu verschmelzenden Sprites.
ind2 Der Index des Sprites, mit dem ind1 zusammengeführt werden soll.

Rückgabe

N/A

Beispiel:

var tspr;
tspr[0] = spr_add(working_directory + "Sprite Assets\explode1.png", 16, true, true, 0, 0);
tspr[1] = sprite_duplicate(spr_Explosion);
sprite_merge(tspr[0], tspr[1]);
sprite_index = t_spr[0];
sprite_delete(tspr[1]);

Der obige Code lädt ein Sprite in eine lokale Variable und verschmilzt es dann mit einem anderen Sprite, das dupliziert wurde. Schließlich wird das zusammengeführte Sprite der Instanz sprite_index zugewiesen, während das geladene Sprite entfernt wird um den von ihm verwendeten Speicher freizugeben. Beachten Sie, dass Sie irgendwann nach dieser Operation, z. B. bei den Ereignissen "Destroy" oder "Room End", auch die Funktion sprite_delete() um das zusammengeführte Sprite aus dem Speicher zu entfernen.
