sprite_flush_multi

Mit dieser Funktion können Sie die beliebige Anzahl von Texturseiten für die angegebenen Sprites aus dem Texturspeicher entfernen. Sie geben die Sprite-Indizes als Array an und die Texturseiten, denen sie zugeordnet sind, werden aus dem Texturspeicher entfernt. Beachten Sie, dass wenn eine der Wenn eine der Textur-Seiten an anderer Stelle im Raum verwendet wird (von einem anderen Instanz-Sprite oder einem Hintergrund usw.), kann es zu einem kleinen Leistungseinbruch kommen, da die Seite sofort wieder in den Speicher zurückgeladen wird, daher ist bei der Verwendung dieser Funktion Vorsicht geboten. Beachten Sie, dass die Funktion -1 zurückgibt, wenn flush für die gewählten Ressourcen nicht unterstützt wird, oder sie gibt 0 zurück, wenn alles korrekt funktioniert hat.

Syntax:

sprite_flush_multi(array);

Argument Beschreibung
Array Array mit den zu spülenden Sprite-Indizes

Rückgabe:

Int (-1 oder 0)

Beispiel:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Der obige Code erstellt ein Array, in dem jedes Element einen Sprite-Index enthält. Dieses Array wird dann verwendet, um diese Sprite-Texturen aus dem Speicher zu löschen.
