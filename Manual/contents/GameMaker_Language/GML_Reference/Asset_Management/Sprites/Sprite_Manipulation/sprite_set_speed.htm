sprite_set_speed

Mit dieser Funktion können Sie die Basis-Animationsgeschwindigkeit und den Typ wie für die Sprite-Ressource einstellen. Normalerweise wird dies im Sprite-Editor eingestellt, aber es kann Momente geben, in denen es erforderlich ist dies manuell für eine bestimmte Sprite-Ressource einstellen müssen. Sie müssen den einzustellenden Sprite-Index, die Animationsgeschwindigkeit und den Typ der anzuwendenden Animationsgeschwindigkeit angeben. Der Typ der Animationsgeschwindigkeit kann eine der folgenden Konstanten sein:

Konstant Beschreibung
spritespeed_framespersecond Das Sprite wurde mit einer Animation in Bildern pro Sekunde definiert.
spritespeed_framespergameframe Das Sprite wurde mit in Animation in Frames pro Spielframe definiert.


Die Auswahl des Geschwindigkeitstyps, der zur Berechnung von Animationen verwendet werden soll, ist sehr wichtig, da es einen großen Unterschied zwischen den beiden Optionen gibt. Die folgenden zwei Beispiele veranschaulichen diesen Unterschied:

  • Wenn Sie ein Sprite haben, das 1 Bild pro Sekunde zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, wird es mit 0,5 Bildern pro Sekunde gezeichnet. Wenn die Bildrate Ihres Spiels 60 Bilder pro Sekunde beträgt, zeichnet das Sprite 1 Bild alle 120 Spiel-Frames.
  • Wenn Sie ein Sprite haben, das 1 Bild pro Spielbild zeichnet, und die Bildgeschwindigkeit auf 0,5 einstellen, zeichnet es 0,5 Bilder pro Spielbild. Wenn Ihre Spielbildrate 60 Bilder pro Sekunde beträgt, zeichnet das Sprite 30 Bilder pro alle 60 Spiel-Frames.

Sie können herausfinden, welcher Geschwindigkeitswert für die Sprite-Animation verwendet wurde, indem Sie die Funktion sprite_get_speed()und den Geschwindigkeitstyp der Animation erhalten Sie mit sprite_get_speed_type().

WICHTIG! Diese Funktion wirkt sich auf die Sprite-Ressource aus, so dass alle Änderungen, die Sie mit dieser Funktion vornehmen, sich auf alle Instanzen auswirken, die das Sprite verwenden, sowie auf alle zukünftigen Instanzen auch.

Syntax:

sprite_set_speed(index, geschwindigkeit, typ)

Argument Beschreibung
index Der Index des Sprites, um den Animationstyp von

Rückgabe:

Konstant (siehe oben)

Beispiel:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Der obige Code setzt die Basis-Animationsgeschwindigkeit für das der Instanz zugewiesene Sprite auf 10 Bilder pro Sekunde.
