sprite_set_nineslice

Diese Funktion wird verwendet, um eine Nine-Slice-Struktur auf ein Sprite anzuwenden. Sie geben den Sprite-Index und die anzuwendende Nine-Slice-Struktur an, die Sie mit sprite_nineslice_create() oder sprite_get_nineslice().

HINWEIS: Diese Funktion wirkt sich auf das Sprite-Asset aus, sodass sich alle Änderungen, die Sie mit dieser Funktion vornehmen, auf alle Instanzen auswirken, die dieses Sprite verwenden, und auch auf alle zukünftigen Instanzen.

Syntax:

sprite_set_nineslice(ind, nineslice);

Argument Beschreibung
ind Der Index des zu ändernden Sprites
Neuner-Scheibe Die Neun-Scheiben-Struktur zum Anwenden

Rückgabe:

N/A

Beispiel:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

Der obige Code erstellt eine neue Nine-Slice-Struktur, aktiviert sie und setzt die Offsets für die Hilfslinien. Die Struktur wird dann auf ein Sprite angewendet und ändert dessen Nine-Slice-Eigenschaften.
