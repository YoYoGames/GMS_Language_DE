sprite_set_bbox_mode

Mit dieser Funktion können Sie den Bounding-Box-Modus für ein Sprite einstellen. Sie geben den Sprite-Index und den zu verwendenden Modus an, der eine der folgenden Konstanten sein sollte:

Konstant Beschreibung
bboxmode_automatic Automatisch - Der Begrenzungsrahmen wird automatisch berechnet, basierend auf der Toleranzeinstellung für das Sprite
bboxmode_fullimage Volles Bild - Der Begrenzungsrahmen wird so eingestellt, dass er die volle Breite und Höhe des Sprites verwendet, unabhängig von der Toleranz und den "leeren" Pixeln
bboxmode_manual Manuell - Die Bounding-Box wird manuell auf benutzerdefinierte Werte gesetzt, was die Verwendung der Funktion sprite_set_bbox()

HINWEIS: Diese Funktion wirkt sich auf das Sprite-Asset aus, so dass alle weiteren Instanzen mit diesem Sprite denselben Bounding-Box-Modus haben werden.

Syntax:

sprite_set_bbox_mode(ind, mode);

Argument Beschreibung
ind Der Index des Sprites, für das der Modus geändert werden soll.
Modus Der einzustellende Modus (eine Konstante).

Rückgabe

N/A

Beispiel:

if sprite_get_bbox_mode(sprite_index) != 0
{
sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

Das obige prüft den bbox-Modus für das aktuelle Sprite und wenn er nicht automatisch ist , dann wird er auf diesen Wert gesetzt.
