tileset_get_uvs

Diese Funktion gibt ein Array mit den UV-Koordinaten und anderen Daten für die Textur der angegebenen Kachel zurück, die auf der Texturseite eingestellt ist. Die Funktion gibt ein Array mit den folgenden 8 Elementen zurück:

  • [0] = links
  • [1] = oben
  • [2] = rechts
  • [3] = unten
  • [4] = Anzahl der Pixel, die der Asset-Compiler von der linken Seite des Kachelsatzes abgeschnitten hat
  • [5] = Anzahl der Pixel, die der Asset-Compiler von der Oberseite des Kachelsatzes abgeschnitten hat
  • 6] = normalisierter Prozentsatz der Pixeldaten aus der ursprünglichen Kachelsatzbreite, die in der Texturseite gespeichert wurden
  • 7] = normalisierter Prozentsatz der Pixeldaten aus der ursprünglichen Kachelsatzhöhe, die in der Texturseite gespeichert wurden

Dieses Array kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere bei der Arbeit in 3D oder bei der Verwendung der 2D-Primitivfunktionen, sowie bei der Arbeit mit den Shader Funktionen.

Syntax:

tileset_get_uvs(tileset);

Argument Beschreibung
Kachelsatz Der Index des zu verwendenden Kachelsatzes.

Rückgabe

Array (1D, 8 Elemente)

Beispiel:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Der obige Code speichert die UV-Koordinaten für den gegebenen Kachelsatz in einem lokalen Array und weist die Werte dann den Instanzvariablen zu.
