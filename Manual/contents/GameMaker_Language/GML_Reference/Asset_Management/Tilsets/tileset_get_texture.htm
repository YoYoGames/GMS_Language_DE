tileset_get_texture

Diese Funktion gibt einen speziellen Zeiger für die Kachelsatz-Texturseite zurück. Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in den 2D-Primitivfunktionen, sowie als auch in den Shader Funktionen. Weitere Informationen über die zurückgegebene Texturseite können Sie mit den verschiedenen texture_-Funktionen erhalten, die Sie hier finden.

Syntax:

tileset_get_texture(tileset);

Argument Beschreibung
Kachelsatz Der Index des zu verwendenden Kachelsatzes.

Rückgabe

Zeiger (Texturseiten-ID)

Beispiel:

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

Der obige Code zeichnet einen Dreiecksstreifen mit 4 Eckpunkten, der mit der Textur in der Variablen "tex" texturiert ist.
