kachelset_get_name

Diese Funktion gibt den Namen des angegebenen Kachelsatzes als String zurück. Dieser Name ist derjenige, der für das Kachelset im Asset-Browser des Hauptfensters von GameMaker Studio 2 angegeben wurde. Bitte beachten Sie, dass dies nur ein String ist ist und nicht verwendet werden kann, um das Kachelset direkt zu referenzieren - dazu bräuchten Sie den Kachelset-Index. Sie können diese Zeichenkette jedoch verwenden, um den Kachelsatzindex zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

tileset_get_name(index);

Argument Beschreibung
index Der Index des Kachelsatzes, von dem der Name abgerufen werden soll.

Rückgabe

Zeichenfolge

Beispiel:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_tileset(_m);
tileset_name = tileset_get_name(_t);

Der obige Code ermittelt den Namen des Kachelsatzindexes für die angegebene Ebene und speichert die Rückgabezeichenfolge in der Variablen tileset_name.
