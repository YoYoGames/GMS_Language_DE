audio_sound_get_track_position

Diese Funktion ermittelt die Position (in Sekunden) innerhalb der Sounddatei, ab der der Sound abgespielt werden soll. Der Ton kann nur eine einzelne Instanz eines Tons sein (der Index für einzelne abzuspielende Töne kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert werden).

Syntax:

audio_sound_get_track_position(index);

Argument Beschreibung
index Der Index des Sounds, dessen Abspielposition ermittelt werden soll.

Rückgabe:

Real

Beispiel:

if audio_sound_get_track_position(global.Music) != 0
{
audio_sound_set_track_position(global.Music, 0);
}

Der obige Code prüft eine Spur, um ihre Startposition zu ermitteln, und wenn sie nicht 0 Sekunden beträgt, wird sie auf 0 Sekunden gesetzt.
