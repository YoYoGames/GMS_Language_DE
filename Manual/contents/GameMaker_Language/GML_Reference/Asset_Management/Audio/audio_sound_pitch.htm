audio_sound_pitch

Diese Funktion kann verwendet werden, um die Tonhöhe eines bestimmten Sounds zu ändern. Der Sound kann entweder aus einem Index für einen einzelnen abgespielten Sound referenziert werden, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser. Wenn es sich um einen Index eines abgespielten Sounds handelt, wird nur diese Instanz geändert, bei Verwendung eines Sound-Assets aus dem Asset-Browser werden jedoch alle werden jedoch alle weiteren Instanzen dieses Sounds geändert, die gerade abgespielt werden.

Das Argument "Tonhöhe" ist ein Tonhöhenmultiplikator, d. h., der Eingabewert multipliziert die aktuelle Tonhöhe mit diesem Wert, so dass der Standardwert 1 keine Tonhöhenänderung bedeutet, während ein Wert von weniger als 1 die Tonhöhe senkt und ein Wert von mehr als 1 die die Tonhöhe erhöhen. Es ist am besten, kleine Schritte für diese Funktion zu verwenden, da ein Wert unter 0 oder über 5 möglicherweise ohnehin nicht hörbar ist. Es ist zu beachten, dass die zulässige Gesamttonhöhenänderung auf (1/256) - 256 Oktaven begrenzt ist, so dass jeder Wert über oder unter nicht registriert wird.

HINWEIS: Der oben angegebene geklammerte Wert ist der, auf den GameMaker Studio 2 versucht, den Bereich zu klammern, aber dieser Wert ist nicht auf allen Zielplattformen garantiert. iOS, zum Beispiel, auf (1/256) - 8 geklemmt, so dass Sie möglicherweise auf jeder Zielplattform experimentieren und verschiedene Versionen einer Sound-Ressource haben müssen, die jeweils vorverschiebt sind, falls Sie höhere oder niedrigere Oktavwerte benötigen.

Syntax:

audio_sound_pitch(index, pitch);

Argument Beschreibung
index Der Index des zu ändernden Tons.
Stellplatz Der Pitch-Multiplikator (Voreinstellung 1).

Rückgabe:

N/A

Beispiel:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
case 1: audio_sound_pitch(s_engine, 0.8); break;
case 2: audio_sound_pitch(s_engine, 0.9); break;
case 3: audio_sound_pitch(s_engine, 0.95); break;
case 4: audio_sound_pitch(s_engine, 1); break;
case 5: audio_sound_pitch(s_engine, 1.2); break;
}

Der obige Code ändert die Tonhöhe des abgespielten Tons, der in der Variablen "s_engine" indiziert ist, basierend auf dem Wert der Variablen "gear".
