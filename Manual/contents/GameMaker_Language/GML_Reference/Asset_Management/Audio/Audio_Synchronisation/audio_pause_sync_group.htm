audio_pause_sync_gruppe

Diese Funktion pausiert die angegebene Sync-Gruppe, wenn sie gerade abgespielt wird, wobei der Gruppenindex der Wert ist, der zurückgegeben wurde, als Sie die Gruppe mit der Funktion audio_create_sync_group(). Dies stoppt nicht Der Sound wird nicht gestoppt, und der Aufruf von audio_resume_sync_group()wird die Wiedergabe an der gleichen Stelle, an der sie angehalten wurde, wieder gestartet.

HINWEIS: Diese Funktionalität ist für die HTML5-Zielplattform nicht verfügbar.

Syntax:

audio_pause_sync_group(group_index);

Argument Beschreibung
gruppen_index Der Gruppenindex zum Innehalten.

Rückgabe:

N/A

Beispiel:

if (keyboard_check_pressed((ord) "P"))
{
global.Pause = !global.Pause
if global.Pause
{
audio_pause_sync_group(sg);
}
sonst
{
audio_resume_sync_group(sg);
}
}

Der obige Code prüft auf einen Tastendruck der Taste "P", und wenn einer erkannt wird, schaltet er die Variable "global.Pause" um und prüft sie dann, um die in der Variable "sg" indizierte Sync-Gruppe anzuhalten oder fortzusetzen.
