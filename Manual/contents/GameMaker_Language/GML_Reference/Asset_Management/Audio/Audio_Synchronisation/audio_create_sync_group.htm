audio_create_sync_group

Erzeugt eine Sync-Gruppe und gibt einen eindeutigen ID-Wert für sie zurück, der dann in allen weiteren Audio-Funktionsaufrufen für diese Gruppe verwendet werden sollte. Wenn die Gruppe eine Schleife bilden soll, geben Sie true ein, andernfalls false, aber beachten Sie, dass, wenn Sie Beachten Sie jedoch, dass alle später hinzugefügten Spuren die gleiche Länge haben müssen, wenn Sie sie in einer Schleife laufen lassen wollen. Beachten Sie, dass Sie beim Erstellen einer Sync-Gruppe den Speicher und die Sounds, die mit ihr verbunden sind, freigeben müssen, wenn sie nicht verwendet werden, indem Sie die audio_destroy_sync_group() Funktion freigeben müssen - zum Beispiel bei den Ereignissen " Raumende" oder "Zerstören".

HINWEIS: Diese Funktionalität ist für die HTML5-Zielplattform nicht verfügbar.

Syntax:

audio_create_sync_group(loop);

Argument Beschreibung
Schleife Ob die Spuren in der Gruppe eine Schleife bilden sollen(true) oder nicht(false).

Rückgabe:

index

Beispiel:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Das obige erzeugt eine neue Sync-Gruppe und weist den Index der Gruppe der Variablen "sg" zu. Anschließend werden der Gruppe vier Sounds hinzugefügt, wobei die Verstärkung für drei von ihnen auf 0 gesetzt wird. Zum Schluss wird die Sync-Gruppe abgespielt.
