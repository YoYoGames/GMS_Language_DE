audio_sync_group_get_track_pos

Diese Funktion gibt die aktuelle Abspielposition der angegebenen Sync-Gruppe zurück. Der Gruppenindex ist der Wert, der zurückgegeben wird, wenn Sie die Gruppe mit der Funktion audio_create_sync_group()erstellt haben, und der Rückgabewert Wert ist die Zeit in Sekunden, die die Tracks abgespielt wurden.

HINWEIS: Diese Funktionalität ist für die HTML5-Zielplattform nicht verfügbar.

Syntax:

audio_sync_group_get_track_pos(group_index);

Argument Beschreibung
gruppen_index Der Gruppenindex, um die Position von zu erhalten.

Rückgabe:

Real

Beispiel:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
secs = string(secs);
}
sonst
{
secs = "0" + string(secs);
}
draw_text(32, 32, "Zeit = " + mins + ":" + secs);

Der obige Code holt sich die Zeitposition für die Sync-Gruppe, die in der Variablen "sg" indiziert ist, und verwendet dann diesen Wert, um die gespielte Zeit auf dem Bildschirm zu zeichnen.
