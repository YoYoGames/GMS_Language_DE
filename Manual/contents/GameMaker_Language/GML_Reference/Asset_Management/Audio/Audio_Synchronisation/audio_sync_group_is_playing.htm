audio_sync_group_is_playing

Mit dieser Funktion können Sie prüfen, ob Audiodaten in einer synchronisierten Gruppe abgespielt werden. Sie müssen die Sync-Gruppen-ID angeben, wie sie von der Funktion audio_create_sync_group().

HINWEIS: Diese Funktionalität ist für die HTML5-Zielplattform nicht verfügbar.

Syntax:

audio_sync_group_is_playing(group_index);

Argument Beschreibung
gruppen_index Der zu prüfende Gruppenindex.

Rückgabe:

N/A

Beispiel:

if audio_sync_group_is_playing(group_one)
{
audio_stop_sync_group(group_one);
}

Der obige Code prüft, ob die Sync-Gruppe group_one gerade abgespielt wird, und wenn ja, wird die Gruppe gestoppt.
