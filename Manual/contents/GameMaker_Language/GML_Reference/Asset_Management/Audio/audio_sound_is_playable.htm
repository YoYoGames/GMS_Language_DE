audio_klang_ist_abspielbar

Mit dieser Funktion kann geprüft werden, ob der angegebene Tonindex aktuell abgespielt werden kann. Dies ist aufgrund der unterschiedlichen Handhabung von gestreamter und nicht gestreamter Soundwiedergabe auf der HTML5-Zielplattform erforderlich und gibt gibt true zurück, wenn der Sound abgespielt werden kann, und false, wenn er nicht abgespielt werden kann. Beachten Sie, dass die Funktion auf allen anderen Plattformen außer HTML5 immer true zurückgibt.

Syntax:

audio_sound_is_playable(index);

Argument Beschreibung
index Der Index des zu prüfenden Tons.

Rückgabe:

Real

Beispiel:

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

Der obige Code ändert die Tonhöhe des abgespielten Tons, der als "snd_Explode" indiziert ist, wenn seine Tonhöhe ungleich 1 ist.
