audio_exists

Diese Funktion gibt zurück, ob ein Sound existiert(true) oder nicht(false). Beachten Sie, dass diese Funktion einen Fehler verursacht, wenn der gesuchte Index zuvor nicht initialisiert wurde, da sie nach nicht existierenden Asset-Indizes sucht. Der Sound, der überprüft werden soll, kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne Sounds, die abgespielt werden, kann in einer Variablen gespeichert werden, wenn Sie die audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein.

Syntax:

audio_exists(index);

Argument Beschreibung
index Der Index des Tons, dessen Vorhandensein geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if audio_exists(global.Music)
{
audio_play_sound(global.Music, 0, true);
}

Der obige Code prüft, ob ein Ton vorhanden ist. Wenn dies der Fall ist, wird er in einer Schleife abgespielt.
