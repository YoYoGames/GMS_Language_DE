audio_pause_ton

Mit dieser Funktion können Sie einen beliebigen Sound, der gerade abgespielt wird, pausieren. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn Sie die audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein, wobei in diesem Fall alle Instanzen des angegebenen Sounds pausiert werden.

Syntax:

audio_pause_sound(index);

Argument Beschreibung
index Der Index des zu pausierenden Tons.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
if global.Pause
{
audio_pause_sound(snd_Wasserfall);
}
sonst
{
audio_resume_sound(snd_Wasserfall);
}
}

Der obige Code prüft, ob die Taste "P" auf der Tastatur gedrückt wird, und wenn er dies erkennt, setzt er die globale Variable "Pause" auf true oder false und hält dann entweder den in der Variablen "snd_Waterfall" indizierten Ton an indiziert ist, oder er setzt den Ton aus dem Pausenzustand fort.
