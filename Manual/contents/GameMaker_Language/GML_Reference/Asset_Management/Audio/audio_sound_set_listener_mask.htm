audio_sound_set_listener_mask

Mit dieser Funktion können Sie die Bit-Maske für einen Sound setzen, so dass er nur von den angegebenen Hörern abgespielt wird. Sie können eine Bit-Maske mit den Befehlen audio_get_listener_info() und verwenden und dann mit dem bitweisen Oder ("|") eine benutzerdefinierte Maske mit den Hörern erstellen, von denen der Sound abgespielt werden soll, und diese benutzerdefinierte Maske dann nach dem Abspielen auf den Sound anwenden. Diese Maske wird überschreibt die Standardmaske oder die Maske, die Sie möglicherweise mit der Funktion audio_set_listener_mask().

HINWEIS: Diese Funktion kann nur auf einen Sound angewendet werden, nachdem er zum Abspielen eingestellt wurde, und benötigt die SoundID, die von der Funktion zum Abspielen des Sounds zurückgegeben wird.

Syntax:

audio_sound_set_listener_mask(soundID, mask);

Argument Beschreibung
soundID Die eindeutige ID des Klangs, für den die Maske eingestellt werden soll
Maske Die zu setzenden Bitmasken-Daten für den Sound

Rückgabe:

N/A

Beispiel:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Der obige Code spielt einen Ton ab und stellt dann den/die Hörer ein, von dem/denen er abgespielt werden soll, indem er die in einer globalen Variablen gespeicherten Maskendaten verwendet.
