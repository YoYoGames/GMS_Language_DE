audio_klang_get_pitch

Diese Funktion kann verwendet werden, um die Tonhöhe eines bestimmten Sounds zu ermitteln. Der Sound kann entweder einer sein, der aus einem Index für einen einzelnen abgespielten Sound referenziert wird, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser.

Syntax:

audio_sound_get_pitch(index);

Argument Beschreibung
index Der Index des Tons, dessen Tonhöhe ermittelt werden soll.

Rückgabe:

Real

Beispiel:

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

Der obige Code ändert die Tonhöhe des abgespielten Tons, der als "snd_Explode" indiziert ist, wenn seine Tonhöhe ungleich 1 ist.
