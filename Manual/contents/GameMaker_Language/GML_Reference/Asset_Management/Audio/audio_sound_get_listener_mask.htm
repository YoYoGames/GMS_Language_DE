audio_sound_get_listener_mask

Diese Funktion gibt die Bit-Masken-Daten zurück, die definieren, von welchen Audio-Listenern ein Sound abgespielt werden soll. Weitere Informationen finden Sie im Abschnitt über Audio-Listener.

Syntax:

audio_sound_get_listener_mask(soundID);

Argument Beschreibung
soundID Die eindeutige ID des Tons, von dem die Maske abgerufen werden soll

Rückgabe:

Real

Beispiel:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
audio_sound_set_listener_mask(snd, global.PlayerMask);
}

Der obige Code spielt einen Sound ab und prüft dann die Hörermasken-Daten für den Sound, und wenn diese nicht mit denen übereinstimmen, die in einer globalen Variable gespeichert sind, werden die Hörer zum Abspielen anhand der in der globalen Variable gespeicherten Masken-Daten eingestellt.
