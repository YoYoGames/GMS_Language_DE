audio_get_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die globale Lautstärke aller Sounds und Musik für einen bestimmten Hörer abrufen. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen für die Zielplattform verfügbaren Indizes abrufen. Der zurückgegebene Verstärkungswert basiert auf einer linearen Skala von 0 (leise) bis 1 (volle Lautstärke). Beachten Sie, dass Sie auf einigen Plattformen eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als Allerdings gilt ein Wert von 1 als "volle Lautstärke" und alles, was darüber liegt, kann zu Audio-Clipping führen.

Syntax:

audio_get_master_gain(listenerIndex);

Argument Beschreibung
listenerIndex Der Index des Hörers, von dem die Verstärkung abgerufen werden soll.

Rückgabe:

Real

Beispiel:

var num = audio_get_listener_count();
for( var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var ind = info[? "index"];
if audio_get_master_gain(ind) != 1
{
audio_set_master_gain(info[? "index"], 1);
}
ds_map_destroy(info);
}

Der obige Code durchläuft die verfügbaren Zuhörer in einer Schleife und prüft, ob ihr Gain 1 ist oder nicht, und wenn nicht, setzt er ihn für jeden von ihnen auf 1.
