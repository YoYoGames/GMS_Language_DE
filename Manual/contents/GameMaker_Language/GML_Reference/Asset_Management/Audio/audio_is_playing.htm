audio_ist_abspielend

Diese Funktion prüft den angegebenen Sound, um zu sehen, ob er gerade abgespielt wird. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert werden) oder ein Sound-Asset sein. In diesem Fall werden alle Instanzen des angegebenen Sounds überprüft, und wenn einer von ihnen abgespielt wird, gibt die Funktion true zurück, andernfalls wird sie false zurückgeben. Beachten Sie, dass diese Funktion auch dann true zurückgibt, wenn der überprüfte Sound zuvor mit der Funktion audio_pause_klang() Funktion angehalten wurde.

Syntax:

audio_is_playing(index);

Argument Beschreibung
index Der Index des zu prüfenden Tons.

Rückgabe:

Boolesche

Beispiel:

if !audio_is_playing(snd_Wasserfall)
{
audio_play_sound_at(snd_Wasserfall, x, y, 0, 300, true, 1);
}

Der obige Code prüft, ob der in der Variablen "snd_Waterfall" indizierte Sound gerade abgespielt wird, und wenn er false zurückgibt, wird der Sound an seiner Raumposition geloopt, mit einer Fall-off-Distanz von 300 und einer niedrigen Priorität.
