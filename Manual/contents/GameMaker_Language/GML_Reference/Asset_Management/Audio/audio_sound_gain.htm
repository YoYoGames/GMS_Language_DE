audio_sound_gain

Mit dieser Funktion können Sie einen Sound über eine bestimmte Zeitspanne ein- oder ausblenden, oder sie kann verwendet werden, um die Soundverstärkung sofort einzustellen. Die Zeit wird in Millisekunden gemessen, und die Funktion erfordert die Eingabe eines Endpegels, den der Sound am Ende dieser Zeit erreicht haben soll. am Ende dieser Zeit erreicht haben soll. Diese Verstärkung kann zwischen 0 (stumm) und einem beliebigen Wert größer als 0 liegen, obwohl man normalerweise die maximale Lautstärke als 1 betrachtet. Alles über 1 kann verwendet werden, aber je nach verwendetem Sound und der Plattform, auf die kompiliert wird, kann es zu Verzerrungen oder kann es zu Verzerrungen oder Übersteuerungen kommen, wenn der Ton wiedergegeben wird. Beachten Sie, dass die Verstärkungsskala linear ist, und um die Verstärkung sofort zu ändern, setzen Sie einfach das Zeitargument auf 0.

Diese Funktion wirkt auf alle Instanzen des Sounds, die gerade im Raum abgespielt werden, wenn der Index eine Sound-Ressource ist, und die Endlautstärke ist die Lautstärke, mit der alle weiteren Instanzen des Sounds abgespielt werden gespielt werden. Wenn Sie jedoch den Index verwendet haben, der von einer Funktion wie audio_play_sound() zurückgegeben haben, wirkt sich dies nur auf diese eine Instanz des Klangs aus.

Syntax:

audio_sound_gain(index, volume, time);

Argument Beschreibung
index Der Index des Klangs, für den die Verstärkung eingestellt werden soll.
Lautstärke Wert für die Musiklautstärke.
Zeit Die Länge für die Änderung der Verstärkung in Millisekunden.

Rückgabe:

N/A

if val
{
var snd = audio_play_sound(snd_fountain);
audio_sound_gain(snd, 0, 0);
audio_sound_gain(snd, 1, 5000);
}

Der obige Code prüft eine Variable und weist dann, wenn er true zurückgibt, den Index eines abzuspielenden Tons der lokalen Variable "snd" zu. Diese Variable wird dann verwendet, um die Lautstärke dieses spezifischen Tons auf 0 zu reduzieren und über 5 Sekunden auf volle Lautstärke zu erhöhen.
