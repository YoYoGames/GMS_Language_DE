audio_sound_get_gain

Diese Funktion gibt den aktuellen Verstärkungswert für den angegebenen Sound zurück. Der Sound kann entweder einer sein, der von einem Index für einen einzelnen abgespielten Sound referenziert wird, der in einer Variablen gespeichert wurde, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert wurde, oder ein tatsächliches Sound-Asset aus dem Asset-Browser. Die Verstärkung wird normalerweise als Wert von 0 bis 1 berechnet, aber auf einigen Plattformen können Sie eine Verstärkung von mehr als 1 haben, obwohl ein Wert von 1 als Allerdings gilt ein Wert von 1 als "volle Lautstärke", und alles, was darüber liegt, kann zu Audio-Clipping führen.

Syntax:

audio_sound_get_gain(index);

Argument Beschreibung
index Der Index des Klangs, für den die Verstärkung ermittelt werden soll.

Rückgabe:

Real

Beispiel:

if audio_sound_get_gain(snd_Music) != 1
{
audio_sound_gain(snd_Music, 1, 0);
}

Der obige Code ändert die Verstärkung des Tons, der von dem als "snd_Music" indizierten Sound abgespielt wird, wenn seine Verstärkung ungleich 1 ist.
