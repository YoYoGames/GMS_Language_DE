audio_channel_num

Mit dieser Funktion können Sie einstellen, wie viele Audiokanäle für die Wiedergabe von Audio in GameMaker Studio 2 zur Verfügung stehen. Das bedeutet im Wesentlichen, dass Sie die Anzahl der gleichzeitig abspielbaren Sounds angeben, und wenn die Anzahl Wenn diese Anzahl überschritten wird, werden die Sounds mit einer niedrigeren Priorität gestoppt, um einen Kanal für die Sounds mit einer höheren Priorität freizugeben. Sie können diese Funktion nutzen, um Ihr Spiel für Geräte zu optimieren denn je geringer die Anzahl der Audiokanäle ist, desto besser ist die Leistung, aber bedenken Sie, dass dies auch zu einem merklichen Abschneiden bestimmter Sounds führen kann, wenn viele auf einmal abgespielt werden.

Syntax:

audio_channel_num(num);

Argument Beschreibung
num Anzahl der verfügbaren Audiokanäle (Standard ist 128).

Rückgabe:

N/A

Beispiel:

switch (os_browser)
{
case browser_not_a_browser:
switch (os_type)
{
case os_windows:
case os_macos:
audio_channel_num(200);
break;
default:
audio_channel_num(64);
break;
}
break;
default:
audio_channel_num(16);
break;
}

Der obige Code prüft die Plattform, auf der das Spiel läuft, und ändert die Anzahl der verfügbaren Tonkanäle, um die Leistung zu erhöhen.
