audio_get_name

Diese Funktion gibt den Namen einer gegebenen Audio-Ressource als String zurück. Der "Index"-Wert kann der der Ressource selbst sein (wie im Asset-Browser zu sehen) oder der eindeutige ID-Wert, der angegeben wird, wenn Sie den Ton z. B. mit abspielen, audio_play_sound(). Beachten Sie, dass die zurückgegebene Zeichenkette nicht mit der ID der Ressource übereinstimmt und nicht zum Zugriff auf die Ressource selbst verwendet werden kann; sie sollte also nur zur Anzeige oder zur Fehlerprüfung verwendet werden.

Syntax:

audio_get_name(index);

Argument Beschreibung
index Der Index des zu prüfenden Tons.

Rückgabe:

Zeichenfolge

Beispiel:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

Der obige Code spielt einen zufälligen Sound ab, der aus drei verschiedenen Sound-Ressourcen ausgewählt wird, und zeigt dann eine Debug-Meldung mit seinem Namen an.
