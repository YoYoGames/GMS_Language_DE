audio_ist_pausiert

Diese Funktion prüft den angegebenen Sound, um festzustellen, ob er gerade pausiert. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen gespeichert werden) oder ein Sound-Asset sein. In diesem Fall werden alle Instanzen des angegebenen Sounds überprüft, und wenn eine von ihnen pausiert, gibt die Funktion true zurück, ansonsten wird sie false zurückgeben.

Syntax:

audio_is_paused(index);

Argument Beschreibung
index Der Index des zu prüfenden Tons.

Rückgabe:

Boolesche

Beispiel:

if audio_is_paused(snd_Wasserfall)
{
audio_resume_sound(snd_Wasserfall);
}

Der obige Code prüft, ob der in der Variablen "snd_Waterfall" indizierte Ton gerade angehalten ist, und wenn er true zurückgibt, wird die Wiedergabe des Tons fortgesetzt.
