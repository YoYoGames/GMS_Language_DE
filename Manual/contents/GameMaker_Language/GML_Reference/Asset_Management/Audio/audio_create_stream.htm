audio_erzeugen_strom

Mit dieser Funktion können Sie einen neuen Tonindex erstellen, der dann in den regulären Audiofunktionen verwendet werden kann, um Audio direkt von einer externen OGG-Dateiquelle zu streamen. Die Funktion benötigt den Dateinamen (der z. B. eine eingebundene Datei sein kann) und gibt den neuen Sound-Index zur Verwendung zurück. Beachten Sie, dass Sie, nachdem Sie den Sound nicht mehr benötigen, die Funktion audio_destroy_stream() mit dem Sound-Index aufrufen, um ihn aus dem Speicher zu entfernen, da sonst ein Speicherleck entstehen kann, das Ihr Spiel verlangsamt und schließlich zum Absturz bringt.

HINWEIS: Diese Funktionalität ist für die HTML5-Zielplattform nicht verfügbar.

Syntax:

audio_create_stream(filename);

Argument Beschreibung
Dateiname Die Datei (nur OGG), von der das Audio gestreamt werden soll.

Rückgabe:

Ton-Index

Beispiel:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Der obige Code erzeugt einen neuen Sound-Index in der Variablen "snd" aus der angegebenen Datei und spielt dann diesen Sound ab.
