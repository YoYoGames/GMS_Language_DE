audio_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die globale Lautstärke aller Sounds und Musik einstellen. Sie basiert auf einer linearen Skala von 0 (stumm) bis zu einem Wert größer als 0, obwohl Sie normalerweise die maximale Lautstärke als 1 betrachten würden. Alles über 1 kann verwendet werden, aber je nach verwendetem Sound und der Plattform, für die kompiliert wird, kann es zu Verzerrungen oder Clipping kommen, wenn der Sound wiedergegeben wird. Diese Funktion beeinflusst die relative Lautstärke aller Klänge und Musik, die in Ihrem Spiel abgespielt werden.

Syntax:

audio_master_gain(gain);

Argument Beschreibung
gewinnen Wert für das globale Volumen (0 bis 1).

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_up)
{
if vol < 1 vol += 0.05;
audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
if vol > 0 vol -= 0.05;
audio_master_gain(vol);
}

Der obige Code prüft auf Tastendrucke der Pfeiltasten nach oben und unten, die dann die Variable "vol" erhöhen oder verringern. Diese Variable wird dann verwendet, um die globale Verstärkung aller Sounds und Musik im Spiel einzustellen.
