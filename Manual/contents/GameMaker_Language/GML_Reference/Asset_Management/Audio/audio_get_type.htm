audio_get_type

Wenn Sie Audio zu GameMaker Studio 2 hinzufügen, kann es entweder gestreamt oder im Speicher sein. Wenn Sie wissen müssen, ob ein gegebener Sound-Index für gestreamtes Audio ist oder nicht, können Sie diese Funktion verwenden, die 1 für gestreamten Sound, 0 für Sound im Speicher und -1 zurückgibt, wenn ein Fehler vorliegt oder der Index nicht auf eine gültige Soundressource zeigt.

Syntax:

audio_get_type(index);

Argument Beschreibung
index Der Index des zu prüfenden Tons.

Rückgabe:

Real

Beispiel:

type = audio_get_type(snd_Music_1);

Der obige Code prüft den Typ des in der Variablen "snd_Music_1" indizierten Audios und speichert den zurückgegebenen Wert in der Variablen "type".
