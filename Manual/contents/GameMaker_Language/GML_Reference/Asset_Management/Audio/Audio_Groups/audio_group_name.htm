audio_group_name

Diese Funktion gibt eine Zeichenkette zurück, die den Namen der angegebenen Audiogruppe zur Anzeige oder Überprüfung enthält. Wenn Sie eine Audiogruppe in den Spieloptionen definieren, geben Sie ihr einen eindeutigen "Namen", der eigentlich eine Konstante ist, die als ID-Wert für die Gruppe. Alles, was diese Funktion tut, ist, die ID zu nehmen und eine Zeichenkette mit dem von Ihnen angegebenen ID-Namen zurückzugeben.

Syntax:

audio_group_name(groupID);

Argument Beschreibung
groupID Die Indexwertkonstante der zu prüfenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

Zeichenfolge

Beispiel:

var name= audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

Der obige Code ruft den Namen der angegebenen Audiogruppen-Konstante ab und zeigt ihn auf dem Bildschirm an.
