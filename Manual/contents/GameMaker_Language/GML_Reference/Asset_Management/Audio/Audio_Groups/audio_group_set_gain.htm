audio_group_set_gain

Mit dieser Funktion können Sie eine Gruppe von Klängen über eine bestimmte Zeit ein- oder ausblenden, oder sie kann verwendet werden, um die Gruppenverstärkung sofort einzustellen. Die Zeit wird in Millisekunden gemessen, und die Funktion erfordert die Eingabe eines Endpegels, den die Gruppe am Ende dieser Zeit erreicht haben soll. Diese Verstärkung kann zwischen 0 (leise) und 1 (volle Lautstärke) liegen, und die Skala ist linear, so dass ein Wert von 0,5 der halben Lautstärke entsprechen würde. Um die Verstärkung sofort zu ändern, setzen Sie einfach das Argument time auf 0. Beachten Sie, dass auf einigen Plattformen eine Verstärkung von mehr als 1 möglich ist, obwohl ein Wert von 1 als "volle Lautstärke" gilt und alles, was darüber hinausgeht, zu Audio-Clipping führen kann.

Syntax:

audio_group_set_gain(groupID, volume, time);

Argument Beschreibung
groupID Der Index der zu stoppenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)
Lautstärke Der endgültige Wert für das Gruppenvolumen.
Zeit Die Länge der Änderung der Verstärkung in Millisekunden.

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
audio_group_set_gain(audiogroup1, 0, 5000);
}

Der obige Code prüft auf die "Leertaste" und blendet dann den gesamten Ton für "audiogroup1" über 5 Sekunden auf 0 aus.
