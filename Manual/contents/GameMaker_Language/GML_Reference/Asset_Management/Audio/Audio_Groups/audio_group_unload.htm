audio_group_unload

Diese Funktion entlädt alle Sounds, die als zur angegebenen Audiogruppe gehörend gekennzeichnet sind, in den Speicher. Die Funktion gibt true zurück, wenn das Entladen eingeleitet wurde, und false, wenn die Gruppen-ID ungültig ist oder sie bereits zum Entladen markiert wurde. Beachten Sie, dass jegliches Audio, das gerade abgespielt wird, wenn diese Funktion aufgerufen wird, gestoppt wird.

Syntax:

audio_group_unload(groupID);

Argument Beschreibung
groupID Der Index der zu entladenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

Boolesche

Beispiel:

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unload(audiogroup_level1);
}

Der obige Code prüft, ob eine Audiogruppe geladen wurde, und wenn ja, entlädt er sie.
