audio_group_is_loaded

Mit dieser Funktion wird geprüft, ob eine bestimmte Audiogruppe in den Speicher geladen wurde und zur Verwendung bereit ist.

Syntax:

audio_group_is_loaded(groupID);

Argument Beschreibung
groupID Der Index der zu prüfenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

Boolesche

Beispiel:

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unload(audiogroup_level1);
}

Der obige Code prüft, ob eine Audiogruppe geladen wurde, und wenn ja, entlädt er sie.
