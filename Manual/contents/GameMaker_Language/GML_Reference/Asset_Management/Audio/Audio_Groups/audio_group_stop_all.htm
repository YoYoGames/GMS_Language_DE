audio_group_stop_all

Diese Funktion stoppt alle Sounds der angegebenen Audiogruppe, die gerade abgespielt werden.

Syntax:

audio_group_stop_all(groupID);

Argument Beschreibung
groupID Der Index der zu stoppenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_space)
{
audio_group_stop_all(audiogroup_level1);
}

Der obige Code prüft auf die "Leertaste" und stoppt dann die gesamte Audiowiedergabe der angegebenen Gruppe.
