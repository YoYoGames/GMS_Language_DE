audio_group_load

Diese Funktion lädt alle Sounds, die als zur angegebenen Audiogruppe gehörend gekennzeichnet sind, in den Speicher. Die Funktion gibt true zurück, wenn das Laden eingeleitet wurde, und false, wenn die Gruppen ID ungültig ist oder sie bereits zum Laden markiert wurde. Die Funktion ist asynchron, so dass Ihr Spiel weiterläuft, während das Audio im Hintergrund geladen wird. Das bedeutet, dass sie auch ein Asynchronous Load/Save Event auslösen wird, um Sie zu informieren, dass die gesamte Gruppe in den Speicher geladen wurde und die Sounds nun verwendet werden können.

Syntax:

audio_group_load(groupID);

Argument Beschreibung
groupID Der Index der zu ladenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

Boolesche

Beispiel:

if !audio_group_is_loaded(audiogroup_level1)
{
audio_group_load(audiogroup_level1);
}

Der obige Code prüft, ob eine Audiogruppe geladen wurde, und wenn nicht, lädt er sie.
