audio_group_load_progress

Diese Funktion prüft den Ladefortschritt für eine Audiogruppe und gibt einen (ungefähren) Wert zwischen 0 und 100 zurück.

Syntax:

audio_group_load_progress(groupID);

Argument Beschreibung
groupID Der Index der zu prüfenden Audiogruppe (wie im Fenster "Audiogruppen" definiert)

Rückgabe:

Real

Beispiel:

if load
{
var pc = audio_group_load_progress(audiogroup_level1);
draw_text(32, 32, "Loading: " + string(pc));
}

Der obige Code prüft eine Variable, und wenn sie true zurückgibt, wird ein Text gezeichnet, der den Fortschritt beim Laden des Audios anzeigt.
