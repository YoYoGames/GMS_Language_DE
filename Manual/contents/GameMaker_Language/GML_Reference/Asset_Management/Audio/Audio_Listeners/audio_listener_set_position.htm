audio_listener_set_position

Mit dieser Funktion können Sie die Position eines bestimmten Hörers innerhalb des 3D-Audio-Raums ändern. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen für die Zielplattform verfügbaren Indizes abrufen. Das Beispielbild unten zeigt die Standardposition für den Listener im Audio-Raum:

Wie Sie sehen können, ist die Standardposition (0, 0, 0), aber Sie würden diese Funktion normalerweise verwenden verwenden, um den Listener mit dem Player-Objekt in Ihrem Spiel zu verschieben und so die Art und Weise zu ändern, in der die von Emittern erzeugten Audiosignale vom Spieler gehört werden, z. B. in der folgenden Abbildung eines Top-Down-Spiels setzt die Player-Instanz den Listener, der bewirkt, dass die Audiosignale der verschiedenen Emitter "verändern", wenn sich der Spieler im Level bewegt:

Syntax:

audio_listener_set_position(index, x, y, z);

Argument Beschreibung
index Der Listener, für den die Daten abgerufen werden sollen (Standardwert 0).
x Die x-Position des Hörers (Voreinstellung 0).
y Die y-Position des Hörers (Voreinstellung 0).
z Die z-Position des Hörers (Voreinstellung 0).

Rückgabe:

N/A

Beispiel:

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

Der obige Code ruft die Ansichtsmatrix für die Kameraansicht [0] ab und verwendet sie dann, um die Position und Ausrichtung des Audio-Hörers für den Hörer mit der in der globalen Variablen "Player_Listener" gespeicherten ID festzulegen.
