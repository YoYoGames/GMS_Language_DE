audio_get_listener_info

Diese Funktion erstellt eine DS-Map und füllt sie mit Informationen für den angegebenen Hörer.

HINWEIS: Sie sind für die Zerstörung der zurückgegebenen DS Map mit der entsprechenden Funktion verantwortlich.

Die DS-Map enthält die folgenden Schlüssel:

  • "name"- Der Name des Hörers, als String, wobei "default" der Standard-Hörername auf den meisten Zielplattformen ist
  • "mask"- Die Bit-Maske für den Hörer
  • "index" - Der eindeutige Indexwert des Hörers

Der Maskenwert kann verwendet werden, um einen Sound oder Emitter so einzustellen, dass er von mehreren Zuhörern gleichzeitig abgespielt wird, indem man einfach die bitweise oder "|"-Funktion verwendet, um eine Maske für den Sound zu erzeugen (siehe den Beispielcode unten), während der Index verwendet wird, um die Eigenschaften wie Position oder Geschwindigkeit für einen bestimmten Hörer mit Funktionen wie audio_listener_set_position().

Syntax:

audio_get_listener_info(num);

Argument Beschreibung
num Die Hörernummer, für die die Daten abgerufen werden sollen.

Rückgabe:

ds_map

Beispiel:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
mask = mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Der obige Code prüft die Anzahl der verfügbaren Listener und ruft dann in einer Schleife deren Maskenbits ab, die dann zu einer einzigen Bitmaske kombiniert werden, die auf den globalen Listener angewendet wird.
