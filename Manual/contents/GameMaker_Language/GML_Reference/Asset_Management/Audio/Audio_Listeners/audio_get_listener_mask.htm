audio_get_listener_mask

Diese Funktion gibt die Bitmasken-Daten zurück, die die aktuelle (globale) Standardmaske für die Audio-Hörer definieren.

Syntax:

audio_get_listener_mask();

Rückgabe:

Real

Beispiel:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
var num = audio_get_listener_count();
global.Audio_Mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
global.Audio_Mask = global.Audio_Mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask);
}

Der obige Code ruft die aktuellen Listener-Maskendaten ab und vergleicht sie mit den Daten, die in einer globalen Variablen gespeichert sind. Wenn sie nicht übereinstimmen, prüft der Code die Anzahl der verfügbaren Hörer, durchläuft sie in einer Schleife und ermittelt ihre Maskenbits, die dann kombiniert werden, um eine einzige Maske zu erstellen, die auf das Audiosystem angewendet wird, um die globalen Hörer zu definieren.
