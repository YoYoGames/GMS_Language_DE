audio_listener_get_data

Diese Funktion erstellt eine DS-Map und füllt sie mit den Positions-, Geschwindigkeits- und Orientierungswerten für den angegebenen Hörer. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen für die Zielplattform verfügbaren Indizes zu erhalten. Wenn Sie einen falschen Listener-Index angeben, gibt die Funktion -1 zurück.

HINWEIS: Sie sind für die Zerstörung der zurückgegebenen DS-Karte mit der entsprechenden Funktion verantwortlich.

Die DS-Map enthält die folgenden Schlüssel:

  • " x" - Die x-Position des Hörers
  • " y" - Die y-Position des Hörers
  • " z" - Die z-Position des Hörers
  • "vx" - Die Geschwindigkeit der x-Achse des Zuhörers
  • "vy" - Die y-Achsen-Geschwindigkeit des Hörers
  • "vz" - Die z-Achsen-Geschwindigkeit des Hörers
  • " lookat_x" - Die x-Komponente des Look-At-Vektors des Hörers
  • " lookat_y" - Die y-Komponente des Look-At-Vektors des Listeners
  • " lookat_z" - Die z-Komponente des Look-At-Vektors des Hörers
  • " up_x"- Die x-Komponente des Aufwärtsvektors des Hörers
  • "up_y"- Die y-Komponente des Aufwärtsvektors des Hörers
  • "up_z"- Die z-Komponente des Aufwärtsvektors des Hörers

Syntax:

audio_listener_get_data(index);

Argument Beschreibung
index Der Listener, für den die Daten abgerufen werden sollen (Standardwert 0).

Rückgabe:

ds_map

Beispiel:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var data = audio_listener_get_data(info[? "index"]);
if data[? "x"] != 0
{
audio_listener_set_position(info[? "index"], 0, 0, 0);
}
ds_map_destroy(info);
ds_map_destroy(data);
}

Der obige Code prüft die Anzahl der verfügbaren Hörer und durchläuft diese in einer Schleife, und wenn ihre x-Position nicht 0 ist, setzt er ihre Position auf 0, 0, 0.
