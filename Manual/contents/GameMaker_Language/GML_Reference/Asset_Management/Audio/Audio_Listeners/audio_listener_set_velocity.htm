audio_listener_set_velocity

Diese Funktion kann verwendet werden, um dem angegebenen Hörer Doppler-Effekte zu geben und die Audio-Bewegung basierend auf dem Vektor zu simulieren, der aus den angegebenen relativen x-, y- und z-Positionen aufgelöst wird. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen für die Zielplattform verfügbaren Indizes zu erhalten.

Wenn der angegebene Listener sich nie bewegt oder die Bewegung keine konstante Bewegung ist, müssen Sie diese Werte normalerweise nicht einstellen, aber wenn Sie z. B. ein scrollendes Spiel machen, bei dem der Spieler eine konstante Bewegung von unten nach oben und die Feinde eine konstante Bewegung von oben nach unten haben, würden Sie die Geschwindigkeiten für Listener und Emitter einstellen. die Feinde eine konstante Bewegung von oben nach unten haben, würden Sie die Hörer- und Emitter-Geschwindigkeiten (für Emitter würden Sie audio_emitter_velocity()) auf die entsprechenden Vektoren setzen, um den richtigen Dopplereffekt zu simulieren, wenn sie sich an der Player-Instanz vorbeibewegen.

Das folgende Bild zeigt, wie dies für das oben angegebene Beispielspiel eingerichtet werden könnte:

Syntax:

audio_listener_set_velocity(index, x, y, z);

Argument Beschreibung
index Der Listener, für den die Daten abgerufen werden sollen (Standardwert 0).
x Die x-Geschwindigkeit des Hörers (Voreinstellung 0).
y Die y-Geschwindigkeit des Hörers (Voreinstellung 0).
z Die z-Geschwindigkeit des Hörers (Voreinstellung 0).

Rückgabe:

N/A

Beispiel:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var data = audio_listener_get_data(info[? "index"]);
if data[? "vx"] != 0
{
audio_listener_set_velocity(info[? "index"], 0, 0, 0);
}
ds_map_destroy(info);
ds_map_destroy(data);
}

Der obige Code prüft die Anzahl der verfügbaren Zuhörer und durchläuft sie in einer Schleife. Wenn ihre x-Geschwindigkeit nicht 0 ist, setzt er ihre Geschwindigkeitswerte auf 0, 0, 0.
