audio_set_listener_mask

Wenn Sie mehrere Hörer auf einem System verwenden, können Sie die Bit-Maske für einen Sound einstellen und ihn nur von dem markierten Hörer hören lassen. Sie können aber auch die globale Maske mit dieser Funktion einstellen und alle normal gespielten Sounds werden von den Sie können aber auch die globale Maske mit dieser Funktion einstellen, und alle normal abgespielten Sounds werden von den durch diese Maske gekennzeichneten Zuhörern gehört, ohne dass Sie die Maske für jeden Sound einzeln einstellen müssen.

Syntax:

audio_set_listener_mask(mask);

Argument Beschreibung
Maske Die zu setzenden Bit-Masken-Daten für die Hörer

Rückgabe:

N/A

Beispiel:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
mask = mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Der obige Code prüft die Anzahl der verfügbaren Listener und ruft dann in einer Schleife deren Maskenbits ab, die dann zu einer einzigen Bitmaske kombiniert werden, die auf den globalen Listener angewendet wird.
