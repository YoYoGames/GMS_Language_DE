audio_stop_ton

Diese Funktion hält den angegebenen Sound an, wenn er gerade abgespielt wird. Der Sound kann entweder eine einzelne Instanz eines Sounds sein (der Index für einzelne abgespielte Sounds kann in einer Variablen gespeichert werden, wenn die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein, wobei in diesem Fall alle Instanzen des angegebenen Sounds gestoppt werden.

Syntax:

audio_stop_sound(index);

Argument Beschreibung
index Der Index des zu stoppenden Tons.

Rückgabe:

N/A

Beispiel:

if !global.SFX
{
audio_stop_sound(snd_Wasserfall);
}
sonst
{
audio_play_sound_at(snd_Wasserfall, x, y, 0, 100, 300, 1, true, 1);
}

Der obige Code prüft die globale Variable "SFX" und wenn er false zurückgibt, wird der in der Variable "snd_Waterfall" indizierte Sound, der gerade abgespielt wird, gestoppt, und wenn er true zurückgibt, wird der Ton in einer Schleife abgespielt.
