audio_pause_alle

Mit dieser Funktion können Sie alle Töne, die gerade abgespielt werden, pausieren.

Syntax:

audio_pause_all();

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
if global.Pause
{
audio_pause_all();
}
sonst
{
audio_resume_all();
}
}

Der obige Code prüft, ob die Taste "P" auf der Tastatur gedrückt wird, und wenn er dies erkennt, setzt er die globale Variable "Pause" auf true oder false und pausiert dann entweder alle Sounds oder startet alle zuvor pausierten Klänge.
