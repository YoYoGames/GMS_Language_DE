audio_set_master_gain

Mit dieser Funktion können Sie den absoluten Wert für die globale Lautstärke aller Töne und Musik für einen bestimmten Hörer einstellen. Der Standard-Hörerindex ist 0, aber Sie können die Funktion audio_get_listener_info() verwenden, um die verschiedenen für die Zielplattform verfügbaren Indizes abrufen. Der Verstärkungswert basiert auf einer linearen Skala von 0 (stumm) bis 1 (volle Lautstärke) und wirkt sich auf die relative Lautstärke aller Sounds und Musik aus, die von Ihrem Spiel über diesen Hörer abgespielt werden.

Syntax:

audio_set_master_gain(listenerIndex, gain);

Argument Beschreibung
listenerIndex Der Index des Hörers, für den die Verstärkung eingestellt werden soll.
gewinnen Wert für das globale Volumen (0 bis 1).

Rückgabe:

N/A

Beispiel:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
var info = audio_get_listener_info(i);
audio_set_master_gain(info[? "index"], 0.75);
ds_map_destroy(info);
}

Der obige Code führt eine Schleife durch die verfügbaren Hörer und setzt dann deren Master-Gain auf 0,75.
