audio_sound_set_track_position

Mit dieser Funktion wird die Position (in Sekunden) innerhalb der Sounddatei festgelegt, ab der der Sound abgespielt werden soll. Der Ton kann entweder eine einzelne Instanz eines Tons sein (der Index für einzelne abzuspielende Töne kann in einer Variablen gespeichert werden, wenn Sie die Funktion audio_play_sound() oder audio_play_sound_at() Funktionen) oder ein Sound-Asset sein. Wenn es eine eindeutige Sound-ID ist, die Sie verwenden, dann wird die Position sofort auf die angegebene neue Zeit, und wenn es sich um ein Asset-Browser-Asset handelt, dann beginnen alle weiteren Wiedergaben des angegebenen Sounds zur neuen Zeit. Die Funktion ignoriert jeden Wert, der länger ist als die Länge des Tons, der für den "index" Wertes.

Syntax:

audio_sound_set_track_position(index, time);

Argument Beschreibung
index Der Index des zu ändernden Tons.
Zeit Die Zeit (in Sekunden), auf die der Startpunkt gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

Der obige Code spielt einen Ton ab und verwendet dann den zurückgegebenen ID-Wert, um die Startposition für den Ton auf 32 Sekunden zu setzen.
