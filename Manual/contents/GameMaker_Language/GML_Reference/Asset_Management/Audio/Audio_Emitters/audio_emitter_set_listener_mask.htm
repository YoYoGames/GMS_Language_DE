audio_emitter_set_listener_mask

Mit dieser Funktion kann die Bit-Maske für einen Emitter gesetzt werden, so dass alle Sounds, die über den Emitter abgespielt werden, nur von den angegebenen Hörern abgespielt werden. Sie können eine Bit-Maske mit den Befehlen audio_get_listener_info() und und dann das bitweise Oder ("|") verwenden, um eine benutzerdefinierte Maske mit den Hörern zu erstellen, von denen der Sound abgespielt werden soll, und diese benutzerdefinierte Maske dann auf den Emitter anwenden. Diese Maske überschreibt die Standard oder die Maske, die Sie mit der Funktion audio_set_listener_mask().

Syntax:

audio_emitter_set_listener_mask(emitterID, mask);

Argument Beschreibung
emitterID Die eindeutige ID des Emitters, für den die Maske eingestellt werden soll
Maske Die zu setzenden Bitmasken-Daten für den Sound

Rückgabe:

N/A

Beispiel:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Der obige Code erstellt einen Audio-Emitter und legt dann den/die Hörer fest, von dem/denen er abgespielt werden soll, indem er die in einer globalen Variablen gespeicherten Maskendaten verwendet.
