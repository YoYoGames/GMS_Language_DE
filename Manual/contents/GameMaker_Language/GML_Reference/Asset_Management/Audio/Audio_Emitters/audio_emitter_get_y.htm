audio_emitter_get_y

Diese Funktion liefert die aktuelle y-Position des angegebenen Audio-Emitters.

Syntax:

audio_emitter_get_y(emitter);

Argument Beschreibung
Emitter Der Index des zu verwendenden Emitters.

Rückgabe:

Real

Beispiel:

if audio_emitter_get_y(emitter_player) != x
{
audio_emitter_position(emitter_player, x, y, 0);
}

Der obige Code prüft die aktuelle y-Position eines bestimmten Emitters und wenn diese nicht gleich der y-Position der Instanz ist, wird sie auf die Instanzposition gesetzt.
