audio_emitter_exists

Diese Funktion gibt zurück, ob ein Audioemitter existiert(true) oder nicht(false). Beachten Sie, dass diese Funktion einen Fehler verursacht, wenn der gesuchte Index zuvor nicht initialisiert wurde, da sie nach nicht existierenden Indizes sucht.

Syntax:

audio_emitter_exists(index);

Argument Beschreibung
index Der Index des Emitters, dessen Vorhandensein geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if audio_emitter_exists(s_emit)
{
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
sonst
{
s_emit = audio_emitter_create();
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

Der obige Code prüft, ob ein Emitter existiert, der in der (zuvor initialisierten) Variablen "s_emit" indiziert ist. Wenn dies der Fall ist, wird ein Ton über ihn abgespielt. Wenn nicht, wird er erstellt und dann der Ton abgespielt.
