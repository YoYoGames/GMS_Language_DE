audio_emitter_erzeugen

Diese Funktion erzeugt einen neuen Audioemitter und gibt den Index für diesen zurück. Dieser Index sollte für alle weiteren Funktionen, die sich auf diesen Emitter beziehen, in einer Variablen gespeichert werden und dann, wenn er nicht mehr benötigt wird, mit der Funktion audio_emitter_free() aus dem Speicher entfernt werden, um Speicherlecks zu vermeiden, die eventuell zum Absturz des Spiels führen können.

Syntax:

audio_emitter_create();

Rückgabe:

Integer (Emitter-ID)

Beispiel:

s_emit = audio_emitter_create();

Der obige Code erzeugt einen neuen Audio-Emitter und weist dessen Index der Variablen "s_emit" zu.
