audio_emitter_get_gain

Diese Funktion gibt die aktuell eingestellte Verstärkung (Lautstärke) für den angegebenen Audioemitter zurück, normalerweise zwischen 0 und 1, wobei 0 stumm und 1 die volle Lautstärke ist. Beachten Sie, dass Sie auf einigen Plattformen eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als Allerdings gilt ein Wert von 1 als "volle Lautstärke", und alles, was darüber liegt, kann zu Audio-Clipping führen.

Syntax:

audio_emitter_get_gain(emitter);

Argument Beschreibung
Emitter Der Index des zu verwendenden Emitters.

Rückgabe:

Real

Beispiel:

if audio_emitter_get_gain(emitter_player) < 1
{
audio_emitter_gain(emitter_player, 1);
}

Der obige Code prüft die aktuelle Verstärkung eines bestimmten Emitters und wenn diese kleiner als 1 ist, wird sie auf 1 gesetzt.
