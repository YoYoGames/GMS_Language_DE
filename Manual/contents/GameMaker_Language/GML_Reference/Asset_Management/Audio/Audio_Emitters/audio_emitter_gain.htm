audio_emitter_gain

Diese Funktion stellt die maximale Verstärkung (Lautstärke) für den Ton ein. Die wahrgenommene Lautstärke für einen Klang kann sich abhängig vom Abklingwert und der Position, die er relativ zum Zuhörer hat, ändern, Durch die Einstellung der Verstärkung mit dieser Funktion wird die volle Lautstärke jedoch nie den angegebenen Verstärkungswert überschreiten. Das folgende Bild zeigt, wie sich die Verstärkung auf die Lautstärke des Emitters auswirkt, wenn der Fall-Off-Wert größer als 0 ist:

Diese Funktion ändert die Lautstärke des Tons, während er abgespielt wird, sowie aller nachfolgenden Töne, die durch den angegebenen Emitter. Beachten Sie, dass Sie auf einigen Plattformen eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als "volle Lautstärke" angesehen wird und alles, was größer ist, zu Audio-Clipping oder Verzerrungen führen kann.

HINWEIS: die endgültige Lautstärke wird auch von der globalen Audioverstärkung beeinflusst, die mit der Funktion audio_master_gain().

Syntax:

audio_emitter_gain(emitter, gain);

Argument Beschreibung
Emitter Der Index des zu ändernden Emitters.
gewinnen Die maximale Verstärkung (Voreinstellung 1).

Rückgabe:

N/A

Beispiel:

if up
{
gain += 0,05;
if gain >= 1 up = false;
}
sonst
{
gain += 0.05;
if gain <= 0 up = true;
}
audio_emitter_gain(s_emit, gain);

Der obige Code setzt die Variable "gain" auf verschiedene Werte und verwendet dann dieselbe Variable, um die Verstärkung des in der Variable "s_emit" indizierten Emitters einzustellen.
