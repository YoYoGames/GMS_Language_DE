audio_emitter_position

Mit dieser Funktion können Sie die Position eines Audio-Emitters innerhalb des 3D-Audio-Raums ändern. Die Position wirkt sich unterschiedlich auf den Klang aus, je nachdem, wo der Hörer im Audio-Raum positioniert ist zu (Standardposition ist (0, 0, 0). Siehe audio_listener_position() für weitere Informationen). Wenn die Position des Emitters also beispielsweise auf (100, 0, 0) eingestellt ist und der aktuelle Hörer sich bei (200, 0, 0) befindet, erscheint das vom Emitter gestreamte Audio auf der linken Seite des Audiofelds. Das folgende Bild zeigt eine visuelle Darstellung der Emitter und ihrer relativen Positionen zum Hörer:

Syntax:

audio_emitter_position(emitter, x, y, z);

Argument Beschreibung
Emitter Der Index des zu ändernden Emitters.
x Die X-Position.
y Die y-Position.
z Die z-Position.

Rückgabe:

N/A

Beispiel:

wenn Geschwindigkeit > 0
{
audio_emitter_position(s_emit, x, y, 0);
}

Der obige Code prüft, ob die Geschwindigkeit der Instanz über 0 liegt, und wenn dies der Fall ist, aktualisiert er den in der Variablen "s_emit" indizierten Audioemitter auf die aktuelle x/y-Position.
