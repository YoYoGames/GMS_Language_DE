audio_get_recorder_count

Diese Funktion gibt die Anzahl der Audioaufnahmequellen (wie Mikrofone, etc...) zurück, die derzeit für Ihr Spiel verfügbar sind. Wenn Sie also einen Rückgabewert von z. B. vier haben, dann haben Sie Audioeingang an den Quellen 0, 1, 2 und 3, wobei einer dieser Werte ist derjenige, von dem aus Sie die Aufnahme mit der Funktion audio_start_aufzeichnung(). Dieser Wert kann sich jederzeit ändern, wenn Sie Mikrofone oder andere Eingangsquellen an das Gerät anschließen/abziehen. Eingangsquellen an das Gerät angeschlossen werden. Beachten Sie, dass Sie mit der Funktion audio_get_recorder_info() verwenden können, um Informationen über jedes angeschlossene Gerät zu erhalten.

WICHTIG! Diese Funktion gibt immer 0 zurück, wenn sie auf dem HTML5-Ziel verwendet wird.

Syntax:

audio_get_recorder_count();

Rückgabe:

Integer

Beispiel:

if audio_get_recorder_count() > 0
{
channel_index = audio_start_recording(0);
}

Der obige Code prüft die verfügbaren Audiorecorder und wenn es 1 oder mehrere gibt, beginnt er die Aufnahme vom ersten indizierten (Quelle 0).
