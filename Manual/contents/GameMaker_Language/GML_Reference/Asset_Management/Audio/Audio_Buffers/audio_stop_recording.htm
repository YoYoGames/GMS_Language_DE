audio_stop_aufzeichnung

Diese Funktion stoppt die Aufnahme auf dem angegebenen Rekorderkanal (der Kanalindex wird beim Aufruf der Funktion audio_start_aufzeichnung()). Wenn Sie die Aufzeichnung stoppen, werden keine weiteren asynchronen Audio-Aufzeichnungsereignisse für den angegebenen Recorder-Kanal ausgelöst, so dass Sie diese Funktion normalerweise im eigentlichen asynchronen Ereignis verwenden, um sicherzustellen, dass Sie alle Daten aufgezeichnet haben.

HINWEIS: Die meisten Plattformen außer HTML5 unterstützen die Aufnahme von Audio in irgendeiner Form, aber das bedeutet nicht, dass alle Geräte dies zulassen, selbst wenn die Plattform dies tut. audio_get_recorder_count() Funktion einen Wert größer als 0 zurückgibt, um zu prüfen, ob Aufnahmegeräte verfügbar sind, bevor Sie die restlichen den Rest der Aufnahmefunktionen.

Syntax:

audio_stop_recording(channel_index);

Argument Beschreibung
Kanal_Index Der Index des Rekorderkanals, der angehalten werden soll.

Rückgabe:

N/A

Beispiel:

audio_stop_recording(audio_channel);

Der obige Code weist GameMaker Studio 2 an, die Aufnahme auf dem angegebenen Audiokanalindex zu stoppen.
