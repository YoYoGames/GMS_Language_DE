skript_existiert

Diese Funktion gibt true oder false zurück, je nachdem, ob das Skript oder die Skriptfunktion mit dem angegebenen Index existiert. Beachten Sie, dass es sich hierbei nicht um eine Zeichenkette handelt, sondern um den Asset-Namen, der den eindeutigen Index für jedes Skript enthält (wie er in der IDE erscheinen würde), oder um die Variable "Skriptfunktionsname", wie sie im Skript-Asset definiert ist (beachten Sie, dass dies nicht für Methodenvariablen funktioniert). Weitere Informationen zu Skripten finden Sie unter Der Skript-Editor.

Syntax:

script_exists(scr);

Argument Beschreibung
scr Der Name des Skripts, das Sie prüfen möchten.

Rückgabe:

Boolesche

Beispiel:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
script_execute(script[script_num]);
}

Das obige Beispiel fügt zwei Skriptfunktionen und einen Wert in ein Array ein, ermittelt dann eine Zufallszahl und verwendet diese, um eine Skriptfunktion auszuwählen, die ausgeführt werden soll, es sei denn, es wird -1 gewählt, in diesem Fall passiert nichts.
