script_get_name

Diese Funktion gibt den Namen als String des angegebenen Scripts zurück. Dieser Name ist derjenige, der für das Script im Asset-Browser des Hauptfensters von GameMaker Studio 2 angegeben wurde. Weitere Informationen über Scripts finden Sie unter Der Script-Editor.

Syntax:

script_get_name(scr);

Argument Beschreibung
scr Der Index des Skripts, dessen Namen Sie abrufen möchten.

Rückgabe:

Zeichenfolge

Beispiel:

scr_name = script_get_name(Help_File);

Der obige Beispielcode speichert den Namen des angegebenen Skriptindex in der Variablen "scr_name".
