skript_ausfuehren_ext

Diese Funktion arbeitet ähnlich wie die Funktion script_execute(), nur dass Sie ein Array übergeben können, das die für die aufgerufene Funktion/das aufgerufene Skript erforderlichen Argumente enthält. Sie können auch zwei optionale Argumente an die Funktion übergeben, um einen Offset im Array anzugeben, aus dem die Argumente geholt werden sollen, sowie die Anzahl der Argumente, die aus dem Array verwendet werden sollen (diese darf maximal array_length - offset betragen).

Syntax:

script_execute(scr, array_args, [offset], [num_args]);

Argument Beschreibung
scr Der Name der Funktion/des Skripts, das Sie aufrufen möchten
array_args Das Array, das die Argumente für die Funktion/das Skript enthält
[Offset] OPTIONAL! Der Offset in das Argument-Array
[num_args] OPTIONAL! Die Anzahl der Argumente zu (ab dem Offset)

Rückgabe:

Hängt vom Rückgabewert der aufgerufenen Funktion/des aufgerufenen Skripts ab

Beispiel:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

Der obige Beispielcode verwendet script_execute_ext, um die angegebene Funktion aufzurufen und liefert ein Array mit vier Argumenten, verwendet aber nur eines davon nach dem Zufallsprinzip.

HINWEIS: Diese Funktion kann nicht mit eingebauten Funktionen verwendet werden.
