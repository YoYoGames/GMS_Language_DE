sequence_instance_override_object

Mit dieser Funktion können Sie alle Instanzen eines in einer Sequenz verwendeten Objekts durch ein anderes überschreiben (ersetzen). Sie geben die Struktur-ID der Sequenzinstanz an (wie sie beim Anlegen der Sequenzinstanz im Raum oder durch Verwendung einer der Raumschicht Funktionen - siehe hier), sowie den Objektindex (wie im Asset-Browser definiert) für das Objekt, das Sie überschreiben wollen. Schließlich geben Sie einen Objektindex oder eine Instanz-ID an, die als das Objekt, das die Sequenz überschreiben soll (bei Angabe einer Instanz-ID wird einfach das Objekt, von dem die Instanz erzeugt wurde, als Überschreibung verwendet). Beachten Sie, dass dies nur bei Sequenzinstanzen (nicht bei Sequenzobjekten) möglich ist und durchgeführt werden muss bevor die Sequenz abgespielt wird, sonst funktioniert es nicht.

Syntax:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argument Beschreibung
sequenz_instanz_strukt Die zu ändernde Sequenzinstanzstruktur.
objekt_id Der Objektindex des zu überschreibenden Objekts innerhalb der Sequenz.
instanz_oder_objekt_id Der Objektindex oder die Instanz-ID, die zum Überschreiben der Sequenzobjekte verwendet werden soll.

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

Der obige Code erzeugt eine neue Sequenzinstanz auf der angegebenen Ebene und modifiziert sie dann so, dass alle Instanzen des Objekts "obj_Bäume_Winter" durch Instanzen des Objekts "obj_Bäume_Sommer" ersetzt werden.
