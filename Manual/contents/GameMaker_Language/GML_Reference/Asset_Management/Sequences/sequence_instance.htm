Reihenfolge_Instanz

Dies ist eine eingebaute Variable, die Teil der Instanzvariablen ist, die für jede Objektinstanz in Ihrem Spiel erstellt werden. Wenn die Instanz von einer Sequenz gesteuert wird, Wenn die Instanz von einer Sequenz gesteuert wird, enthält diese Variable die Sequenzinstanzstruktur für die Sequenz, die die Instanz steuert, andernfalls ist sie undefiniert. Dies ist eine schreibgeschützte Variable und kann nicht geändert werden.

Beachten Sie, dass diese Variable nach Beendigung der steuernden Sequenz undefiniert wird, auch wenn ihr Sequenzelement noch existiert, und wieder die Struktur erhält, wenn dieses Sequenzelement erneut abgespielt wird.

Syntax:

Reihenfolge_Instanz

Rückgabe:

Boolesche

Beispiel:

if (in_sequence)
{
sequence_instance.speedScale = 2;
}

Der obige Code prüft die Variable in_sequence Variable, und wenn sie true ist (was bedeutet, dass die Instanz von einer Sequenz gesteuert wird), dann ändert er die Geschwindigkeitsskala dieser Sequenz auf 2.
