sequence_keyframe_new

Mit dieser Funktion können Sie eine neue Spur-Keyframe-Struktur erstellen, indem Sie den Typ der Spur angeben, auf die der Keyframe angewendet wird, der eine der folgenden Konstanten ist:

Konstant Beschreibung Wert
seqtracktype_graphic Dies ist eine Grafik- (Sprite-) Asset-Spur. 1
seqtracktype_audio Dies ist eine Audio-Asset-Spur. 2
seqtracktype_instance Dies ist eine Instance-Asset-Spur. 14
seqtracktype_sequence Dies ist ein Sequenz-Asset-Track. 7
seqtracktype_clipmask Dies ist eine Clip-Masken-Gruppen-Asset-Spur. 8
seqtracktype_clipmask_mask Dies ist eine Clip-Masken-Sprite-Asset-Spur, die zum Erzeugen der Clip-Maske verwendet wird. 9
seqtracktype_clipmask_subject Dies ist eine Clip-Masken-Sprite-Asset-Spur, die maskiert wird. 10
seqtracktype_group Dieser Gruppenordner Asset-Track. 11
seqtracktype_colour Dies ist eine Farbdaten-Parameterspur. 4
seqtracktype_real Dies ist eine reale Zahlenwert-Parameterspur. 3
seqtracktype_message Dies ist eine Spur für Broadcast-Nachrichten. 15
seqtracktype_moment Dies ist ein Ereignis-/Moment-Track. 16
seqtracktype_bool Derzeit nicht verwendet. 5
seqtracktype_string Derzeit nicht verwendet. 6
seqtracktype_spriteframes Derzeit nicht verwendet. 13
seqtracktype_empty Derzeit nicht verwendet. 12

Die Funktion gibt eine Track-Keyframe-Struktur zurück, zu der dann Daten hinzugefügt werden können, bevor sie einer Track-Struktur zugewiesen wird.

Syntax:

sequence_keyframe_new(type);

Argument Beschreibung
Typ Der Typ des zu erstellenden Keyframes, eine Konstante, die oben aufgeführt ist.

Rückgabe:

Struktur

Beispiel:

myseq = sequence_create();
var mytracks = array_create(1);
mytracks[0] = sequence_track_new(seqtracktype_graphic);
var graphickeys = array_create(1);
graphickeys[0] = sequence_keyframe_new(seqtracktype_graphic);
graphickeys[0].frame = 0;
graphickeys[0].length = 1;
graphickeys[0].stretch = true;
graphickeys[0].disabled = false;
var graphickeydata = array_create(1);
graphickeydata[0] = sequence_keyframedata_new(seqtracktype_graphic);
graphickeydata[0].spriteIndex = spr_Platform;
graphickeydata[0].channel = 0;
graphickeys[0].channels = graphickeydata;
mytracks[0].name = "TestGraphicTrack";
mytracks[0]. keyframes= graphickeys;
myseq.tracks = mytracks;

Der obige Code erstellt eine neue Sequenz und erstellt dann eine Grafik-Asset-Spur und setzt einige Keyframe-Daten auf die Spur. Diese Spur wird dann der Instanz zugewiesen.
