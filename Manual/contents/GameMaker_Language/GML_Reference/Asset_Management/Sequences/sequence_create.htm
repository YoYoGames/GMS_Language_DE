sequenz_erstellen

Mit dieser Funktion können Sie ein neues Sequenz-Objekt erzeugen, dem Sie dann Spuren hinzufügen können. Die Funktion gibt eine struct zurück, auf die Sie dann zugreifen können, um die neu erstellte Sequenz einzurichten. Der Inhalt dieser Struktur ist auf dieser Seite beschrieben. Das Sequenz-Objekt struct kann dann verwendet werden, um Instanzen der Sequenz auf einer Raumebene mit der Funktion layer_sequence_create(). Beachten Sie, dass Sie Sequenz-Objekte, die Sie auf diese Weise anlegen, durch Aufruf der Funktion sequence_destroy() entfernen, wenn sie nicht mehr benötigt werden, da sonst ein Speicherleck entsteht, das Ihr Spiel verlangsamen und schließlich zum Absturz bringen kann.

Syntax:

sequence_create();

Rückgabe:

Struktur

Beispiel:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

Der obige Code erzeugt eine neue Sequenz-Objektstruktur und legt deren Wiedergabelänge und Schleifenmodus fest.
