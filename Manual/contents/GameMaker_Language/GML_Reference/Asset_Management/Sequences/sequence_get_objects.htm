sequenz_get_objects

Mit dieser Funktion können Sie ein Array mit allen Objektindizes abrufen, die Instanzen haben, die innerhalb der angegebenen Sequenz erstellt werden. Sie liefern entweder die Sequenz-Objektstruktur (wie von der Funktion sequence_create() oder sequence_get()) oder die Sequenz-ID (zurückgegeben von der Funktion layer_sequence_get_sequence() oder aus der Sequenzinstanz struct-Eigenschaft sequence) und die Funktion gibt ein Array zurück, wobei jedes Element im Array ein objekt_index für die verschiedenen Objekte ist, die von der Sequenz zum Erzeugen von Instanzen.

Syntax:

sequence_get_objects(sequence_struct_or_id);

Argument Beschreibung
sequenz_strukt_oder_id Die Sequenzobjekt-Struktur oder ID, von der die Objekte abgerufen werden sollen.

Rückgabe:

Array

Beispiel:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

Der obige Code ruft die Struktur für ein Sequenzobjekt ab und ruft dann die verwendeten Objekte ab und speichert das Array in einer Variablen.
