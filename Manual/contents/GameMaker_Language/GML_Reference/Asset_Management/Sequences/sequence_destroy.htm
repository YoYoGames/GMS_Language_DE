sequenz_zerstören

Mit dieser Funktion können Sie ein Sequenz-Objekt zerstören, das dynamisch erzeugt wurde. Sie übergeben entweder die Sequenz-Objekt-Struktur (wie sie von der Funktion sequenz_erstellen()) oder die Sequenz-ID (zurückgegeben von der Funktion von der Funktion layer_sequence_get_sequence() oder aus der Sequenzinstanz-Struktur-Eigenschaft sequence). Diese Funktion sollte immer dann verwendet werden, wenn eine dynamisch erzeugte Sequenz nicht mehr benötigt wird, um den mit ihr verbundenen Speicher freizugeben.

Syntax:

sequence_destroy(sequence_struct_or_id);

Argument Beschreibung
sequenz_strukt_oder_id Die zu zerstörende Sequenzobjekt-Struktur oder ID

Rückgabe:

N/A

Beispiel:

if sequence_exists(my_seq)
{
sequence_destroy(meine_seq);
}

Der obige Code prüft, ob das angegebene Sequenz-Objekt existiert und wenn ja, wird es zerstört.
