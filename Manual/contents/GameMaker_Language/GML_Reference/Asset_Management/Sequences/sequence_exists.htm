Reihenfolge_vorhanden

Mit dieser Funktion können Sie prüfen, ob ein Sequenz-Objekt existiert oder nicht. Sie übergeben entweder die Sequenz-Objekt-Struktur (wie sie von der Funktion sequence_create() oder sequenz_holen()) oder die Sequenz-ID (wie von der Funktion layer_sequence_get_sequence() oder von der Sequenzinstanz-Struktur-Eigenschaft sequence, oder den Index aus dem Asset Browser) und die Funktion gibt true zurück, wenn das Sequenzobjekt existiert, oder false, wenn es nicht existiert.

Syntax:

sequence_exists(sequence_struct_or_id);

Argument Beschreibung
sequenz_strukt_oder_id Die Sequenzobjekt-Struktur oder ID, auf die geprüft werden soll

Rückgabe:

Bool

Beispiel:

if sequence_exists(my_seq)
{
sequence_destroy(meine_seq);
}

Der obige Code prüft, ob das angegebene Sequenz-Objekt existiert und wenn ja, wird es zerstört.
