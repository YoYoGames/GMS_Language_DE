sequenz_holen

Mit dieser Funktion können Sie die Sequenzobjektstruktur aus einem Sequenz-Asset-Index abrufen. Sie geben den Asset-Index für die Sequenz an, wie im Asset-Browser definiert, und die Funktion Funktion gibt die Sequenz-Objektstruktur zurück, auf die dann zugegriffen werden kann, oder -1, wenn die Sequenz nicht existiert oder der angegebene Asset-Index keine Sequenz ist.

Syntax:

sequence_get(sequence_index);

Argument Beschreibung
Reihenfolge_index Der Sequenzindex, wie im Asset-Browser definiert.

Rückgabe:

Struktur oder -1

Beispiel:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

Der obige Code ruft die Sequenzobjekt-Struktur aus dem Sequenzobjekt "seq_Loop" ab und setzt dann den Schleifenmodus für die Sequenz (alle nachfolgend erzeugten Sequenzinstanzen verwenden nun diesen Schleifenmodus).
