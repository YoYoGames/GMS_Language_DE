Die Keyframe-Struktur

Jeder Spur (wie auf der Seite über Spurstrukturen definiert) werden eine oder mehrere Keyframe-Strukturen zugewiesen (die Sie über die keyframes Eigenschaft der Spur-Struktur ablesen können), und jede davon hat die folgenden Eigenschaften:

  • Rahmen: Die Position (in Frames) entlang der Zeitleiste für den Keyframe. Der Standardwert ist 0.
  • Länge: Die Länge des Keyframes. Der Standardwert ist 1. Wenn er auf größere Werte gesetzt wird, wird die Spureigenschaft, auf die sich der Keyframe bezieht, für die Dauer der angegebenen Länge auf dem Anfangswert gehalten. Beachten Sie, dass die Eigenschaft stretch dies außer Kraft setzt, wenn sie auf "true" gesetzt ist.
  • strecken: Wenn diese Eigenschaft auf true gesetzt ist, wird der Keyframe entweder bis zum nächsten Keyframe für die Spur oder bis zum Ende der Spur gedehnt, wenn es der letzte Keyframe ist . Sie können diesen Wert abrufen oder setzen, und der Standardwert ist false.
  • Kanäle: Diese Eigenschaft ermöglicht den Zugriff auf die Liste der Keyframe-Datenstrukturen für die Kanäle der Spur. Wenn Sie diese Eigenschaft erhalten, wird ein Array von Keyframe-Datenstrukturen zurückgegeben, und beim Setzen dieser Eigenschaft sollte ein Array von Keyframe-Datenstrukturen angegeben werden.
