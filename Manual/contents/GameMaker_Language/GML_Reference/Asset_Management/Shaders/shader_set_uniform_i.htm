shader_set_uniform_i

Mit dieser Funktion können Sie den Wert (oder die Werte) einer Shader-Konstante setzen. Sie müssen zuvor das "Handle" der Konstante mit der Funktion shader_get_uniform()erhalten haben, und Sie müssen wissen, um welchen Typ von Konstante es sich handelt, um die richtige Anzahl von Integer-Werten an die Funktion zu übergeben, d.h.: wenn Sie einen vec2 haben, müssen Sie zwei Werte an die Funktion übergeben.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion shader_set()und vor dem Aufruf von shader_reset().

Syntax:

shader_set_uniform_i(handle, wert1 [, wert2, wert3, wert4]);

Argument Beschreibung
Griff Das Handle der zu setzenden Shader-Konstante.
Wert1 ... Wert4 Der ganzzahlige Wert (oder Werte), auf den die Shader-Konstante gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
shader_params= shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

Der obige Code holt sich das Handle der Shader-Konstante "u_vParams" (ein vec3) und setzt diese Konstante dann auf die angegebenen Integer-Werte.
