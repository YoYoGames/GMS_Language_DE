shader_current

Diese Funktion gibt den Index-ID-Wert des Shaders zurück, der gerade zum Rendern verwendet wird, oder sie gibt -1 zurück, wenn kein Shader verwendet wird.

Syntax:

shader_current();

Rückgabe:

Real (ID-Wert des Shaders, oder -1, wenn kein Shader)

Beispiel:

if(shader_current() == -1)
{
shader_set(sh_warp)
}

Der obige Code prüft, was der aktuelle Shader ist, und wenn er -1 zurückgibt (kein Shader wird verwendet), wird ein Shader gesetzt.
