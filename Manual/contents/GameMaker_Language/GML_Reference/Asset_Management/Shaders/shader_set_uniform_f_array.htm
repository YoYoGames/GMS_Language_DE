shader_set_uniform_f_array

Mit dieser Funktion können Sie eine Shader-Konstante setzen, um ein Array von Werten zu halten. Sie müssen zuvor das "Handle" der Konstante mit der Funktion shader_get_uniform()erhalten haben, und Sie müssen müssen Sie das Array zuvor initialisiert haben.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion shader_set(), und vor dem Aufruf von shader_reset().

Syntax:

shader_set_uniform_f_array(handle, array);

Argument Beschreibung
Griff Das Handle der zu setzenden Shader-Konstante.
Array Ein zuvor initialisiertes Array mit Fließkommawerten.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params= shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f_array(shader_params, tex_array);
draw_self();
shader_reset();

Der obige Code holt sich das Handle der Shader-Konstante "u_vParams" und setzt diese Konstante dann auf das angegebene Array.
