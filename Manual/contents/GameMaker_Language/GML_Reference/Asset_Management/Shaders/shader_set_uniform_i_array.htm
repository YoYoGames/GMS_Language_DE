shader_set_uniform_i

Mit dieser Funktion können Sie eine Shader-Konstante setzen, die ein Array von Werten enthält. Sie müssen sich zuvor das "Handle" der Konstante mit der Funktion shader_get_uniform()erhalten haben, und Sie müssen zuvor das Array initialisiert haben.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion shader_set()und vor dem Aufruf von shader_reset().

Syntax:

shader_set_uniform_i_array(handle, array);

Argument Beschreibung
Griff Das Handle der zu setzenden Shader-Konstante.
Array Ein zuvor initialisiertes Array mit Ganzzahlwerten.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
col_array[0] = 255;
col_array[2] = 255;
col_array[3] = 64;
col_array[4] = 128;
shader_params= shader_get_uniform(shader_tint, "cColourArray");
shader_set_uniform_i_array(shader_params, col_array);
draw_self();
shader_reset();

Der obige Code erhält das Handle der Shader-Konstante "cColourArray" und setzt diese Konstante dann auf das angegebene Array.
