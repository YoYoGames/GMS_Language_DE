shader_enable_corner_id

Mit dieser Funktion können Sie einen globalen Zustand für alle verwendeten Shader einstellen, bei dem der Shader, wenn er aktiviert ist, 2 Bits aus den Eingangsfarbwerten "stiehlt". Das erste ist das untere Bit des roten Farbwertes, das zweite das untere Bit des blauen Farbwerts. Diese Werte können dann im Shader zurückgewonnen werden, um herauszufinden, mit welchem Vertex Sie es zu tun haben (d. h. mit welcher Ecke).

Syntax:

shader_enable_corner_id(enable);

Argument Beschreibung
aktivieren. Aktivieren(true) oder deaktivieren(false) Sie diese Funktion.

Rückgabe:

N/A

Beispiel:

shader_enable_corner_id(true);

Mit dem obigen Code wird die Verwendung von Farbbits für die Eck-ID für alle Shader aktiviert.
