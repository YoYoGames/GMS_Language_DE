shader_set

Mit dieser Funktion können Sie das Zeichnungsziel auf den angegebenen Shader setzen und alle weiteren Zeichnungen werden mit diesem durchgeführt. Sie können die Verwendung des Shaders mit der Funktion shader_reset().

HINWEIS: Sie können einen Shader nur im Draw Event einer Instanz setzen.

Syntax:

shader_set(shader);

Argument Beschreibung
Shader Der Index des zu verwendenden Shaders.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
draw_self();
shader_reset();

Der obige Code legt einen Shader fest, der zum Zeichnen verwendet wird, und zeichnet dann das aktuelle Sprite, das für die Instanz verwendet wird, mit diesem Shader.
