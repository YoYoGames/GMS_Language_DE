shader_set_uniform_matrix_array

Diese Funktion setzt das Zeichen-Ziel zurück und sollte aufgerufen werden, wenn Sie den aktuellen Shader nicht mehr verwenden wollen (gesetzt mit shader_set()).

Syntax:

shader_reset();

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
draw_self();
shader_reset();

Der obige Code legt einen Shader fest, der zum Zeichnen verwendet wird, und zeichnet dann das aktuelle Sprite, das für die Instanz verwendet wird, mit diesem Shader.
