shader_set_uniform_matrix_array

Mit dieser Funktion können Sie eine Shader-Konstante setzen, die ein Array von Matrixwerten enthält. Sie müssen zuvor das "Handle" der Konstante mit der Funktion shader_get_uniform(), erhalten haben und Sie müssen das Array zuvor als Array von Fließkommawerten initialisiert haben, wobei jede aufeinanderfolgende Gruppe von 16 Floats eine 4x4-Matrix darstellt.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion shader_set()und vor dem Aufruf von shader_reset().

Syntax:

shader_set_uniform_matrix_array(handle, array);

Argument Beschreibung
Griff Das Handle der zu setzenden Shader-Konstante.
Array Ein zuvor initialisiertes Array mit Fließkommawerten.

Rückgabe:

N/A

Beispiel:

shader_set(shader_Glass);
shader_params= shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

Der obige Code erhält das Handle der Shader-Konstante "cMatrixArray" und setzt diese Konstante dann auf das angegebene Array.
