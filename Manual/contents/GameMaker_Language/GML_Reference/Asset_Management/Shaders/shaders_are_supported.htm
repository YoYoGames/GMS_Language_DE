shader_werden_unterstützt

Diese Funktion prüft, ob die gewählte Zielplattform Shader unterstützt und gibt true zurück, wenn dies der Fall ist, und false, wenn dies nicht der Fall ist. Es ist wichtig zu beachten, dass bei Android, wenn das Projekt keine Shader-Ressourcen definiert hat, die Funktion immer false zurückgibt, unabhängig davon, ob das Gerät Shader unterstützt oder nicht.

Syntax:

shaders_are_supported();

Rückgabe:

Boolesche

Beispiel:

global.GFX = false;
if shaders_are_supported()
{
if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
{
global.GFX = true;
}
}

Der obige Code setzt eine globale Variable auf false, und wenn die Plattform Shader unterstützt und die beiden geprüften Shader korrekt kompiliert wurden, wird sie auf true gesetzt.
