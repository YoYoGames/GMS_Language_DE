shader_ist_kompiliert

Diese Funktion prüft einen Shader zur Laufzeit, um sicherzustellen, dass er erfolgreich kompiliert wurde. Wenn dies der Fall ist, gibt die Funktion true zurück, andernfalls gibt sie false zurück. Diese Funktion sollte zu Beginn des Spiels verwendet werden, um sicherzustellen, dass die Plattform, auf der Ihr Spiel läuft, alle verwendeten Shader erfolgreich kompiliert hat (insbesondere unter Windows, wo einige Computer möglicherweise DX9 mit Shader Level 2.0 verwenden und nicht eine spätere Version mit Shader Level 3.0).

Wenn Ihr Shader NICHT kompiliert wurde und Sie rufen shader_set() aufruft, stürzt das Spiel ab. Es lohnt sich also, eine Art Prüfung vorzunehmen, wenn Sie etwas anderes als einfache GLSL ES-Shader verwenden.

Syntax:

shader_is_compiled(shader);

Argument Beschreibung
Shader Der zu prüfende Shader.

Rückgabe:

N/A

Beispiel:

global.GFX = false;
if(shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
global.GFX = true;
}

Der obige Code setzt eine globale Variable auf false und dann, wenn beide zu prüfenden Shader korrekt kompiliert wurden, wird sie auf true gesetzt.
