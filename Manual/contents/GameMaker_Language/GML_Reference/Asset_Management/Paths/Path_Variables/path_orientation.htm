Pfad_Orientierung

Diese Variable enthält die aktuelle Ausrichtung des Pfades, die der Instanz beim Aufruf der Funktion path_start() zugewiesen wurde. Wenn ein Pfad erstellt wird, ist seine Ausrichtung der Standardwert 0 Grad, aber Sie können diesen Wert damit auf jeden beliebigen Wert setzen. Denken Sie daran, dass in GameMaker Studio 2 (sofern Sie nicht mit Physik arbeiten) die Winkel gegen den Uhrzeigersinn berechnet werden, so dass das Setzen der Pfadausrichtung auf 90° setzen, würde den Pfad nach links drehen.

Syntax:

Pfad_Orientierung;

Rückgabe:

Real

Beispiel:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

Der obige Code dupliziert ein zufälliges, vorgefertigtes Pfad-Asset in die Variable "mypath". Dieser neue Pfad wird dann gestartet und um 90° gedreht.
