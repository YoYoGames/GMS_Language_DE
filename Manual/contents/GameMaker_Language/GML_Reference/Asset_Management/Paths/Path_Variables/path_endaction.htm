Pfad_Aktion

Diese Variable kann verwendet werden, um die Reaktion einer Instanz zu erhalten oder zu ändern, wenn sie das Ende des aktuellen Pfades erreicht. Normalerweise würden Sie diese setzen, wenn Sie den Pfad mit pfad_start() aber können Sie dieses Verhalten in Abhängigkeit von einer beliebigen Anzahl von Ereignissen in Ihrem Spiel ändern. Die verfügbaren Werte werden durch die folgenden Konstanten ausgedrückt:

Konstant Beschreibung
pfad_action_stop den Weg anhalten
pfad_aktion_neustart von der Startposition aus fortfahren und zum Start springen, wenn der Pfad nicht geschlossen ist
Pfad_Aktion_Weiter den Weg ab der aktuellen Position neu beginnen
Pfad_Aktion_Umkehr die Geschwindigkeit des Pfades umkehren (den Pfad in umgekehrter Richtung fahren)

Syntax:

path_endaction;

Rückgabe:

Konstante (siehe Beschreibung)

Beispiel:

if (pfad_endaction == pfad_action_stop)
{
path_endaction = path_action_reverse;
}

Der obige Code prüft die Endaktion des Pfades und wenn diese auf "Stop" gesetzt ist, wird die Endaktion in "Reverse" geändert.
