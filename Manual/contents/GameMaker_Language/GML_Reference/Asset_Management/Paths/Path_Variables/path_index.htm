Pfad_Index

Alle Ressourcen in GameMaker Studio 2 haben eine eindeutige Identifikationsnummer, und der path_index ist eine schreibgeschützte Variable, die den Indexwert für eine bestimmte Pfad-Ressource enthält, die einer Instanz mit der Funktion path_start() zugewiesen wurde. Wenn der Instanz kein Pfad zugewiesen ist, wird die Variable auf -1 gesetzt.

Syntax:

Pfad_Index;

Rückgabe:

Ganzzahl (Pfad-ID)

Beispiel:

if (path_index == -1)
{
path_start(pth_enemy3, 4, path_action_reverse, 0);
}

Der obige Code prüft, ob der Instanz ein Pfad zugewiesen wurde, und wenn nicht, beginnt er einen neuen Pfad (und weist ihn gleichzeitig dem path_index zu).
