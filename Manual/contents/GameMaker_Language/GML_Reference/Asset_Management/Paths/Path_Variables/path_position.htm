weg_position

Mit dieser Funktion können Sie die Position einer Instanz entlang eines Pfades ermitteln oder festlegen. Der Wert ist von 0 - 1 normalisiert, wenn Sie ihn also z. B. auf 0,5 setzen, wird die Instanz genau in die Mitte des Pfades verschoben.

Syntax:

pfad_position;

Rückgabe:

Real

Beispiel:

path_position = random(1);

Der obige Code setzt die Instanz an eine beliebige Position auf dem aktuellen Pfad.
