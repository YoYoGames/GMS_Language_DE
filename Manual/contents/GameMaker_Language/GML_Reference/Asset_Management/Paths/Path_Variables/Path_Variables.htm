Pfad-Variablen

Die folgenden Variablen sind bei der Erstellung in alle Instanzen eingebaut und können verwendet werden, um Informationen über einen Pfad zu erhalten oder um bestimmte Eigenschaften zu ändern:

  • Pfad_Index
  • weg_position
  • weg_positionvorher
  • Weg_Geschwindigkeit
  • Pfad_Skala
  • Pfad_Orientierung
  • Pfad_Aktion

Beachten Sie, dass Sie hier weitere Informationen zu Instanzvariablen finden können: Instanz-Variablen.
