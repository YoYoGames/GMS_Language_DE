weg_positionvorher

Diese Variable kann verwendet werden, um die Position einer Instanz entlang ihres aktuellen Pfades im vorherigen Schritt zu erhalten oder zu setzen, und ist ein normalisierter Wert zwischen 0 und 1, d. h.: 0 ist die Startposition des Pfades und 1 wäre die Endposition. Er ist ähnlich wie der xprevious und yvorherige Variablen, nur ist sie spezifisch für Pfade. Sie kann nützlich sein für Dinge wie das vorübergehende Anhalten eines Pfadverfolgers, wenn etwas im Weg ist (siehe den Beispielcode unten).

Syntax:

path_positionprevious;

Rückgabe:

Real

Beispiel:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if collision_circle(xx, yy, 16, obj_Player, false,true)
{
path_position = path_positionprevious;
}

Der obige Code prüft einen Bereich vor der Instanz auf eine Kollision mit dem Objekt "obj_Player" und wenn es eine gibt, setzt er die Instanz zurück auf die vorherige Position, die sie auf dem aktuellen Pfad eingenommen hat, wie von der Funktion path_start().
