Pfad_existiert

iese Funktion gibt zurück, ob ein Pfad mit dem angegebenen Index existiert oder nicht. Beachten Sie, dass diese Funktion einen Fehler auslöst, wenn Sie die Existenz eines Pfades über eine Variable prüfen, die noch nicht deklariert wurde.

Syntax:

path_exists(index);

Argument Beschreibung
index Der Index des Pfades, auf den geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if (path_exists(mein_pfad))
{
path_delete(mein_pfad);
}

Dieser Code prüft, ob die angegebene Variable einen Pfadindex speichert und wenn ja, dann wird der Pfad aus dem Speicher gelöscht.
