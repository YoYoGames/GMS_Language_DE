Pfad_anhängen

Mit diesem Code können Sie einen Pfad an einen anderen anhängen, sie also effektiv zusammenfügen, und damit das funktioniert, müssen beide Pfade vorher existieren. Im Allgemeinen sollten Sie diese Funktion für Pfade verwenden, die mit path_add() erstellt wurden, denn wenn Sie sie auf ein Pfad-Asset anwenden, wirkt sie sich ab dem Moment, in dem die Funktion verwendet wird, bis zum Ende des Spiels permanent auf den Pfad für Instanzen im Spiel aus.

HINWEIS: Dadurch wird der angehängte Pfad nicht aus dem Spiel entfernt. Er ist immer noch da, nur hat er jetzt keine Punkte mehr und wenn Sie ihn nicht mehr verwenden möchten, sollten Sie ihn mit der Funktion path_delete().

Syntax:

path_append(index, path);

Argument Beschreibung
index Der neue Index des Pfades.
Pfad Der Index des Pfades, der an 'index' angehängt wird.

Rückgabe:

N/A

Beispiel:

path_append(mypath, mppath);
path_delete(mppath);

Der obige Code hängt die in der Variablen "mppath" indizierten Pfaddaten an den in der Variablen "mypath" indizierten Pfad an und löscht dann "mppath" aus dem Speicher.
