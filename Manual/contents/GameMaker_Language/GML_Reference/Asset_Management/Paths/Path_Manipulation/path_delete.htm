Pfad_Löschen

Sie können diesen Code verwenden, um einen Pfad aus dem Speicher zu entfernen. Wurde dieser Pfad dynamisch mit path_add()erstellt wurde, ist die Variable, die den Pfadindex enthält, für den Zugriff auf den Pfad nicht mehr gültig, da sie nicht mehr Wenn der Pfad mit dem Pfad-Editor erstellt wurde, kann auf diesen Pfad im gesamten Spiel nicht mehr zugegriffen werden, da er permanent gelöscht wird.

Syntax:

path_delete(index);

Argument Beschreibung
index Der Index des zu löschenden Pfades.

Rückgabe:

N/A

Beispiel:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
path_assign(mypath, t_path);
}
path_delete(t_path);

Der obige Code erzeugt einen Pfad und speichert dessen Index in einer lokalen Variablen. Dieser Pfad wird dann verwendet, um einen mp_grid_path() generierten Pfad, der, wenn er erfolgreich den Weg zum Ziel zu finden, dann dem in "mypath" indizierten Pfad zugewiesen wird. Abschließend wird der "t_path" gelöscht.
