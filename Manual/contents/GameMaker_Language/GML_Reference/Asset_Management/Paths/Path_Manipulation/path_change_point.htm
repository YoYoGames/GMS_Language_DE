pfad_wechsel_punkt

Mit dieser Funktion können Sie die Position und/oder den Geschwindigkeitsfaktor eines beliebigen Punktes ändern, der zuvor für einen Pfad definiert wurde (der Pfad kann im Pfadeditor oder durch Code mit path_add()). Wenn Sie die Funktion auf einen Pfad aus dem Asset-Browser verwendet wird, beachten Sie, dass die Funktion das eigentliche Asset ändert und sich somit dauerhaft darauf auswirkt, wie der Pfad von allen Instanzen im Spiel ab dem Zeitpunkt der Verwendung der Funktion bis zum Ende des Spiels verwendet wird. Wenn dies nicht gewünscht ist, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfads zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, für das Asset aufzurufen, wenn es Vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_change_point(index, n, x, y, speed);

Argument Beschreibung
index Der Index des Pfades, von dem ein Punkt geändert werden soll.
n Der definierende Punkt zum Ändern der Attribute von.
x Die neue x-Koordinate (relativ zum Pfad) des Punktes.
y Die neue y-Koordinate (relativ zum Pfad) des Punktes.
Geschwindigkeit Der neue Geschwindigkeitsfaktor des Punktes.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < pfad_get_number(pfad0); ++i;)
{
var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
path_change_point(pth_Patrol, i, px, py, 100);
}

Der obige Code durchläuft alle Punkte in dem als "path0" indizierten Pfad und positioniert alle Punkte an eine zufällige Position innerhalb eines Bereichs von 128x128 Pixeln.
