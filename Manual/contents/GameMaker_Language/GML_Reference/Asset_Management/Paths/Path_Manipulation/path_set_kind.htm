pfad_set_kind

Mit dieser Funktion können Sie die Art des Pfades einstellen, den die angegebene Ressource haben soll. Dies kann entweder ein geradliniger Pfad sein (auf 0 gesetzt) oder ein geglätteter Pfad (auf 1 gesetzt), wobei in diesem Fall auch die Pfadgenauigkeit berücksichtigt werden muss (die Genauigkeit kann ebenfalls eingestellt werden mit path_set_precision()). Diese Funktion ändert die tatsächliche Pfad-Ressource und wirkt sich somit permanent darauf aus, wie der Pfad von allen Instanzen im Spiel verwendet wird, von dem Moment Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zuerst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset auf (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_set_kind(index, val);

Argument Beschreibung
index Der Index des zu ändernden Pfades.
val Die Art des Pfades, 0 für gerade oder 1 für glatt.

Rückgabe:

N/A

Beispiel:

path_set_kind(pth_Patrol, true);

Dadurch wird der angegebene Pfad auf einen "glatten" Pfad eingestellt.
