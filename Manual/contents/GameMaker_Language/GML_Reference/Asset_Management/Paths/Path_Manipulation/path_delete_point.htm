pfad_loeschen_punkt

Mit dieser Funktion können Sie einen Punkt aus dem angegebenen Pfad entfernen. Pfadpunkte werden von 0 an nummeriert, so dass der erste Punkt immer der 0. und der letzte Pfadpunkt der 1. ist. path_get_number() - 1. Wenn Sie Wenn Sie diese Funktion für eine Pfadanlage aufrufen, ist das Entfernen des Punkts dauerhaft, und allen Instanzen, die dem Pfad in Zukunft zugewiesen werden, fehlt der entfernte Punkt. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zuerst eine Kopie des Pfades zu erstellen, und dann diese Funktion für das duplizierte Asset aufrufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_delete_point(ind, n);

Argument Beschreibung
index Der Index des Pfades, aus dem ein Punkt gelöscht werden soll.
n Der definierende Punkt zum Löschen.

Rückgabe:

N/A

Beispiel:

var _col = true;
while (_col)
{
var _num = path_get_number(my_path) - 1;
if (_num > 2)
{
var _x1 = path_get_point_x(my_path, _num);
var _y1 = path_get_point_y(my_path, _num);
var _x2 = path_get_point_x(my_path, _num -1);
var _y2 = path_get_point_y(mein_pfad, _num -1);
if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
{
path_delete_point(mein_pfad, _num);
_num -= 1;
}
sonst _col = false;
}
sonst _col = false;
}

Der obige Code erzeugt eine Schleife, die die Punkte auf einer Rückwärtsstrecke vom Ende aus auf eine Kollision mit dem angegebenen Objekt prüft. Wenn eine Kollision gefunden wird, wird ein Pfadpunkt gelöscht, bis es keine Kollision mehr gibt oder der Pfad nur noch 2 Punkte lang ist.
