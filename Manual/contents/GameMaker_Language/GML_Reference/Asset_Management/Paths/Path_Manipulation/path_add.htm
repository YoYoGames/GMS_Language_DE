Pfad_hinzufügen

Mit dieser Funktion können Sie in GameMaker Studio 2 einen Pfad erstellen, ohne den Pfad-Editor zu verwenden. Diese Funktion gibt den Index des Pfades zurück, der in einer Variablen gespeichert und fortan als Referenz für diesen Pfad verwendet werden soll. Bitte beachten Sie dass der erzeugte Pfad leer ist, d. h. er hat keine definierten Punkte, so dass Sie dann die anderen verfügbaren Funktionen verwenden müssen, um dem Pfad Punkte hinzuzufügen oder MP-Gitter zu verwenden, um den Pfad den Pfad zu erzeugen. Wenn Sie den Pfad nicht mehr verwenden oder einen neuen Pfad erstellen und seinen Index in derselben Variablen speichern möchten, sollten Sie zunächst den alten Pfad mit path_delete löschen, um Speicherlecks zu vermeiden, die Ihr Spiel zum Absturz bringen können.

Syntax:

path_add();

Rückgabe:

Real

Beispiel:

global.newpath = path_add();

Dadurch wird ein neuer Pfad erstellt und sein Index global.newpath zugewiesen.
