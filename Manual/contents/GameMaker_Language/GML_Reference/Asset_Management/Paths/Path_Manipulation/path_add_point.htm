pfad_hinzuf_punkt

Mit dieser Funktion können Sie dem angegebenen Pfad einen Punkt hinzufügen und seinen Geschwindigkeitsfaktor festlegen. Dieser Punkt wird am Ende des Pfades hinzugefügt, unabhängig von der Position des Punktes, und der Faktor ist gleich dem Prozentsatz von Pfad_Geschwindigkeit der die folgende Instanz tatsächlich fährt, wenn sie diesen Punkt auf dem Pfad erreicht. Wenn Sie einen Pfadpunkt an einer anderen Position platzieren möchten, die nicht das Ende ist, sollten Sie pfad_einfuegen_punkt().

Wenn Sie die Funktion auf einen Pfad aus dem Asset-Browser anwenden, beachten Sie, dass die Funktion das eigentliche Asset ändert und sich somit dauerhaft darauf auswirkt, wie der Pfad von allen Instanzen im Spiel verwendet wird, und zwar von dem Moment an, in dem die Funktion verwendet wird, bis zum Ende des Spiels. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfads zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, für das Asset aufzurufen). Vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_add_point(index, x, y, speed);

Argument Beschreibung
index Der Index des Pfades, zu dem der Punkt hinzugefügt werden soll.
x Die x-Koordinate des neuen Punktes.
y Die y-Koordinate des neuen Punktes.
Geschwindigkeit Der Geschwindigkeitsfaktor des Punktes (Standardwert ist 100).

Rückgabe:

N/A

Beispiel:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Der obige Code erzeugt einen kreisförmigen Pfad um die x/y-Position der Instanz, die den Code ausführt.
