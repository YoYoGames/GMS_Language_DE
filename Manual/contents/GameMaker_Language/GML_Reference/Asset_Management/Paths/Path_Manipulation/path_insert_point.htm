pfad_einfuegen_punkt

Mit dieser Funktion können Sie einen neuen Punkt in einen Pfad einfügen (der Pfad kann im Pfadeditor oder durch Code mit path_add()). Der Punkt wird vor dem Punkt "n" in den Pfad eingefügt der in der Funktion angegeben ist.

Syntax:

path_insert_point(index, n, x, y, speed);

Argument Beschreibung
index Der Index des Pfades, in den der Punkt eingefügt werden soll.
n Der definierende Punkt, vor dem der neue Punkt eingefügt werden soll.
x Die x-Koordinate (relativ zum Pfad) des neuen Punkts.
y Die y-Koordinate (relativ zum Pfad) des neuen Punkts.
Geschwindigkeit Der Geschwindigkeitsfaktor des Punktes.

Rückgabe:

N/A

Beispiel:

path_insert_point(mypath, 0, 50, 50, 100);

Dies fügt einen Punkt ganz am Anfang des Pfades ein, der in der Variablen "mypath" indiziert ist, bei (50,50), mit einem Geschwindigkeitsfaktor von 100 %.
