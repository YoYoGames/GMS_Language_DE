Pfad_Umskalierung

Mit dieser Funktion kann der angegebene Pfad sowohl entlang der vertikalen als auch der horizontalen Achse neu skaliert werden, wobei im Grunde jeder der Pfadpunkte an eine neue Position verschoben wird, die diesem Maßstab um die Mitte des Pfades entspricht. Diese Funktion ändert den eigentlichen Pfadwert, und wirkt sich somit dauerhaft darauf aus, wie der Pfad von allen Instanzen im Spiel ab dem Zeitpunkt der Verwendung der Funktion bis zum Ende des Spiels verwendet wird. Wenn Sie dies nicht wünschen, sollten Sie eine Funktion wie path_duplicate() verwenden, um zunächst eine Kopie des Pfades zu erstellen und dann diese Funktion für das duplizierte Asset aufzurufen (vergessen Sie nicht, path_delete() für das Asset aufzurufen, wenn es nicht mehr benötigt wird).

Syntax:

path_rescale(index, xscale, yscale);

Argument Beschreibung
index Der Index des zu skalierenden Pfades.
x-Skala Womit die aktuelle horizontale Skala multipliziert werden soll. Die Standardskala ist 1.
y-Skala Womit die aktuelle vertikale Skala multipliziert werden soll. Die Standardskala ist 1.

Rückgabe:

N/A

Beispiel:

path_rescale(pth_AI, 3, 3);

Dadurch wird der angegebene Pfad auf das Dreifache seiner ursprünglichen Größe skaliert.
