Pfadmanipulation

In GameMaker Studio 2 sind Sie nicht nur auf die Pfade beschränkt, die Sie mit dem Pfad-Editor erstellen. Es gibt eine Reihe von Funktionen, die verwendet werden können Es gibt eine Reihe von Funktionen, mit denen Sie diese Pfade zur Laufzeit manipulieren und ändern können, oder sogar neue Pfade erstellen und diese dann ändern und manipulieren können.

Die folgenden Funktionen werden zum Ändern von Pfaden verwendet:

  • Pfad_existiert
  • Pfad_hinzufügen
  • pfad_hinzuf_punkt
  • pfad_wechsel_punkt
  • pfad_einfuegen_punkt
  • pfad_loeschen_punkt
  • pfad_klar_punkte
  • Pfad_anhängen
  • Pfad_zuweisen
  • Pfad_Löschen
  • Pfad_duplizieren
  • Pfad_Umklappen
  • Pfad_Spiegel
  • Pfad_umgekehrt
  • Pfad_rotieren
  • Pfad_Umskalierung
  • pfad_gesetzt_geschlossen
  • pfad_set_kind
  • pfad_set_präzision
  • Pfad_Verschiebung
