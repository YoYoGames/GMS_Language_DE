pfad_get_name

Diese Funktion gibt den Namen des Pfades, auf den verwiesen wird, als String zurück. Der Name ist das, was auch immer verwendet wurde, um den Pfad im Editor zu definieren, oder (wenn der Pfad durch eine Codefunktion erstellt wurde) gibt sie eine Zeichenkette mit dem Format " _newpathXX" zurück, wobei "XX" die Nummer des erzeugten Pfades ist, beginnend bei 0 und jedes Mal um eins erhöht, wenn ein neuer Pfad erzeugt wird. Bitte beachten Sie, dass es sich hierbei nur um eine Zeichenkette ist und nicht verwendet werden kann, um den Pfad direkt zu referenzieren - dazu bräuchten Sie die Pfadindex-ID. Sie können diese Zeichenkette jedoch verwenden, um die Pfadindex-ID zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit mit der Funktion asset_get_index().

Syntax:

path_get_name(index);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.

Rückgabe:

Zeichenfolge

Beispiel:

path_name = path_get_name(path_array[0]);

Dadurch wird "path_name" auf den Namen des Pfads gesetzt, der im angegebenen Array an Position 0 indiziert ist.
