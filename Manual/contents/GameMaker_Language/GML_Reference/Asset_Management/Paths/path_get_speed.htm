Pfad_get_speed

Diese Funktion gibt den Geschwindigkeitsfaktor einer beliebigen Position auf einem Pfad zurück. Die Position sollte ein Wert zwischen 0 und 1 sein (Sie können weg_positionverwenden, wenn Sie zum Beispiel die aktuelle Position der Instanz), wobei 0 die Startposition und 1 das Ende ist. Der zurückgegebene Wert ist der Faktor, um den die aktuelle Weg_Geschwindigkeit gesetzt wird. Zum Beispiel, auf einem Pfad mit nur zwei Punkten, 0 und 1, bei dem der Geschwindigkeitsfaktor von Punkt 0 100 und der von Punkt 1 0 ist, gibt die Funktion path_get_speed(path, 0.5) den Wert 50 zurück.

Syntax:

path_get_speed(ind, pos);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.
pos Wie weit durch den Pfad geprüft werden soll. Zwischen 0 (Anfang) und 1 (Ende), kann dezimal sein.

Rückgabe:

Real

Beispiel:

Geschwindigkeit = path_get_speed(pth_Patrol, 0.5) / 20;

Dadurch wird die Geschwindigkeit der aufrufenden Instanz auf den Geschwindigkeitsfaktor des Punktes auf genau der Hälfte des angegebenen Pfades, geteilt durch 20, gesetzt. Wenn der gefundene Geschwindigkeitsfaktor also der Standardwert 100 ist, wird die Geschwindigkeit der aufrufenden Instanz auf 5 gesetzt.
