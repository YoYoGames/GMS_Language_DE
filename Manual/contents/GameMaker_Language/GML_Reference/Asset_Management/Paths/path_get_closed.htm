pfad_get_geschlossen

Mit dieser Funktion kann zurückgegeben werden, ob der Pfad als geschlossen (true) oder offen (false) gekennzeichnet ist, d. h. ob der Pfad eine Schleife bildet oder ob er einen definitiven Anfang und ein definitives Ende hat.

Syntax:

path_get_closed(index);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.

Rückgabe:

Boolesche

Beispiel:

state = path_get_closed(pth_Patrol);

Dadurch wird "state" entweder auf true oder false gesetzt , je nach dem geschlossenen Zustand des in "pth_Patrol" indizierten Pfades.
