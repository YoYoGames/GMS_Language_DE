Pfade

GameMaker Studio 2 verfügt über eine umfangreiche Sammlung von Funktionen und eingebauten Variablen zur Verwendung von Pfaden innerhalb Ihres Spiels. Dieser Abschnitt befasst sich mit den grundlegenden, die beim Umgang mit Pfaden verwendet werden, die zuvor im Pfad-Editor erstellt wurden, aber es gibt noch weitere Funktionen, die es Ihnen erlauben, eigene Pfade mit Hilfe von Code zu erstellen und sie auf verschiedene Weise zu manipulieren.

Es gibt die folgenden Grundfunktionen, die sich mit Pfaden befassen:

  • Pfad_Start
  • Pfad_Ende
  • pfad_get_geschlossen
  • Pfad_get_kind
  • path_get_length
  • pfad_get_name
  • Pfad_get_number
  • pfad_get_punkt_geschw.
  • pfad_get_punkt_x
  • pfad_get_punkt_y
  • pfad_get_präzision
  • Pfad_get_speed
  • Pfad_get_x
  • pfad_get_y

Um das Debuggen von Pfaden zu erleichtern, gibt es auch eine Funktion, um sie auf den Bildschirm zu zeichnen:

  • Pfad_zeichnen

Jede Instanz eines Objekts verfügt außerdem über eine Reihe von eingebauten Variablen, die die Verwendung von Pfaden erleichtern sollen und im folgenden Abschnitt aufgelistet sind:

  • Pfad-Variablen

Schließlich ist es möglich, den Inhalt eines Pfades zu manipulieren sowie neue Pfade zu erstellen, während das Spiel läuft. Die Funktionen, die diese Dinge und mehr ermöglichen, finden Sie im folgenden Abschnitt:

  • Pfadmanipulation
