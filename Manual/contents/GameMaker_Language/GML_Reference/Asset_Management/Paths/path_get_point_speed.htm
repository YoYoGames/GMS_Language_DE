pfad_get_punkt_geschw.

Mit dieser Funktion können Sie die Geschwindigkeit des Punktes (wie im Pfad-Editor definiert oder wenn Sie einen Pfadpunkt dynamisch hinzufügen mit path_add_point()) in Prozent ausgedrückt. Wenn Sie also einen Pfadpunkt im Pfad-Editor auf 50 gesetzt haben, würde diese Funktion bei Verwendung 50 zurückgeben.

Syntax:

path_get_point_speed(index, n);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.
n Der Punkt Identität zu prüfen.

Rückgabe:

Real

Beispiel:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
var _px = path_get_point_x(pth_Patrol, 0);
var _py = path_get_point_x(pth_Patrol, 0);
path_change_point(pth_Patrol, 0, _px, _py, 100);
}

Der obige Code prüft den Pfadpunkt "0" des durch "pth_Patrol" indizierten Pfades, um zu sehen, ob die Geschwindigkeit ungleich 100 ist. Wenn dies nicht der Fall ist, wird die Geschwindigkeit dieses Punkts auf 100 gesetzt.
