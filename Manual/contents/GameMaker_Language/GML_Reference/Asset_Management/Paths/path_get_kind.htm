Pfad_get_kind

Pfade können entweder glatt oder gerade sein (ein glatter Pfad berechnet einen gekrümmten Pfad um die definierten Punkte, während ein gerader Pfad einfach nur von einem Punkt zum anderen führt). Diese Funktion kann verwendet werden, um herauszufinden, ob der gegebene Pfad glatt ist(wahr) oder nicht(falsch).

Syntax:

path_get_kind(index);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.

Rückgabe:

Boolesche

Beispiel:

path_kind = path_get_kind(pth_Patrol);

Dadurch wird die Variable "path_kind" auf true oder false gesetzt , je nachdem, welche Art von Pfad der angegebene Pfadindex ist.
