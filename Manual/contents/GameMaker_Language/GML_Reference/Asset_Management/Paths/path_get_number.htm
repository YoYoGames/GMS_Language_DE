Pfad_get_number

Diese Funktion kann verwendet werden, um die Anzahl der Punkte auf einem Pfad zurückzugeben.

Syntax:

path_get_number(index);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.

Rückgabe:

Real

Beispiel:

if (path_get_number(pth_AI) > 1)
{
path_start( pth_AI, 4, 3, 0);
}

Der obige Code prüft, ob ein Pfad mehr als einen Punkt enthält, und wenn ja, beginnt die Instanz, sich entlang dieses Pfades zu bewegen.
