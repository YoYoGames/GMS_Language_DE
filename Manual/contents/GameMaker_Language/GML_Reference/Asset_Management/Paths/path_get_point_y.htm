pfad_get_punkt_y

Diese Funktion gibt die y-Position (in Raumkoordinaten) des Punktes zurück, den Sie für den Pfad, den Sie indizieren, eingegeben haben. Wenn der Punkt außerhalb des Bereichs des Pfads liegt (z. B.: ein Pfad hat 8 Punkte und Sie fragen nach der y-Position von Punkt 10), wird ein Wert von 0 zurückgegeben werden.

Syntax:

path_get_point_y(index, n);

Argument Beschreibung
index Der Index des zu prüfenden Pfades.
n Der Punkt Identität zu prüfen.

Rückgabe:

Real

Beispiel:

if (pfad_position == 1)
{
var _num = path_get_number(pth_Patrol);
var _pos = floor(random(_num));
x = path_get_point_x(pth_Patrol, _pos);
y = path_get_point_y(pth_Patrol, _pos);
path_position = (1 / _num) * _pos;
}

Der obige Code prüft, ob sich eine Instanz am Ende eines Pfades befindet. Wenn dies der Fall ist, wird ein zufälliger Punkt auf dem Pfad ausgewählt und die Instanz zu diesem Punkt verschoben.
