animcurve_get_channel

Diese Funktion gibt die Struktur zurück, die die Kanaldaten für den in einem Animationskurven-Asset angegebenen Kanal enthält. Sie geben die Animationskurven-ID oder struct sowie den Kanalnamen oder Index an, und die Funktion gibt eine Struktur mit dem folgenden Format zurück:

Variable Name Daten Typ Beschreibung
Name String Dies ist der Name des Kanals.
Typ Konstante Dies wird eine der Konstanten sein animcurvetype_linear für eine lineare Interpolation zwischen den Punkten, oder animcurvetype_catmullrom für eine "glatte" Interpolation zwischen den Punkten mittels Katmull-Rom-Interpolation.
Iterationen Ganzzahl Wenn der Kanal die Katmull-rom ("glatte") Interpolation verwendet, wird hier festgehalten, wie viele Punkte für jedes Segment der Kurve generiert wurden (beachten Sie, dass diese zusätzlichen Punkte intern in der Funktion sind und nur für die Laufzeitberechnungen verwendet werden). Wenn der Kanal eine lineare Interpolation verwendet, ist dieser Wert immer noch vorhanden, kann aber ignoriert werden, da er keinen Einfluss darauf hat, wie die Kurve interpoliert wird.
Punkte Array-Zeiger Dies ist ein Array, wobei jedes Element im Array eine Punktstruktur ist.


Die Animationskurven-ID ist der Name der Animationskurve, wie sie im Asset-Browser definiert wurde, oder der Strukturzeiger, wie er von der Funktion animcurve_create(). Der Kanalname ist eine Zeichenkette, die Der Kanalname ist eine Zeichenkette, die sich auf den Kanal bezieht, wie er im Animationskurven-Asset definiert wurde, oder Sie können einen Indexwert angeben, der von 0 bis n reicht, wobei n der letzte Kanal im Kurven-Asset ist (z. B.: Wenn eine Animationskurve 4 Kanäle hat, werden sie indiziert von 0 bis 3 indiziert). Beachten Sie, dass die Übergabe eines Indexwerts weniger Verarbeitung erfordert als die Übergabe eines Kanalnamens. Wenn die Funktion fehlschlägt (d. h.: es existiert kein Kanal mit dem angegebenen Namen oder Index), erhalten Sie eine Fehlermeldung.

Die Punkte auf einem einzelnen Kanal werden als Structs in einem Array gespeichert, wobei jedes Element im Array ein einzelnes Punkt-Struct ist. Die Punktstruktur hat die folgenden Variablen:

Variable Name Daten Typ Beschreibung
posx real Die Position in der Zeit (normiert von 0 bis 1) des Punktes.
Wert real Der Wert des Punktes.

Syntax:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Argument Beschreibung
kurven_strukt_oder_id Die ID oder der Strukturzeiger der Animationskurve, die angepeilt werden soll
kanal_name_oder_index Der Kanalname (eine Zeichenfolge) oder der Kanalindex (eine ganze Zahl).

Rückgabe:

Struktur

Beispiel:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

Der obige Code ruft die Kanalstruktur für Kanal 0 im Kurven-Asset "ac_ButtonTween" ab, läuft dann in einer Schleife durch die Punkte auf der Kanalkurve und fügt zu ihrem Wert einen hinzu.
