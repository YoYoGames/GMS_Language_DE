animcurve_destroy

Mit dieser Funktion können alle Animationskurven zerstört werden, die zuvor mit der Funktion animcurve_create(). Durch den Aufruf dieser Funktion wird die Animationskurve aus dem Speicher entfernt und bereinigt auch alle Kanäle oder Punkte, die sie enthält (diese müssen nie manuell bereinigt werden). Beachten Sie, dass Sie keine Animationskurven zerstören können, die im Asset-Browser erstellt wurden, sondern nur solche, die zur Laufzeit erstellt wurden.

Syntax:

animcurve_destroy(curve_struct);

Argument Beschreibung
Kurven_Konstrukt Der Zeiger auf die zu zerstörende Kurvenstruktur.

Rückgabe:

N/A

Beispiel:

animcurve_destroy(my_curve);

Der obige Code zerstört die (zuvor erstellte) Animationskurvenstruktur, die in der Variablen "my_curve" indiziert ist.
