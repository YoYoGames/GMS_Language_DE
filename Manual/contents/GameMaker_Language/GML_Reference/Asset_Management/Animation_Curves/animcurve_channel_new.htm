animcurve_channel_new

Mit dieser Funktion können Sie eine neue Animationskurven-Kanalstruktur erstellen. Eine Kanal-Struktur hat die folgenden Variablen:

Variable Name Daten Typ Beschreibung
Name String Dies ist der Name des Kanals.
Typ Konstante Dies wird eine der Konstanten animcurvetype_linear - für lineare Interpolation zwischen den Punkten - oder animcurvetype_catmullrom - für eine "glatte" Interpolation zwischen den Punkten mittels Catmull-Rom-Interpolation.
Iterationen Ganzzahl Wenn der Kanal die Katmull-rom ("glatte") Interpolation verwendet, wird hier festgehalten, wie viele Punkte für jedes Segment der Kurve generiert wurden (beachten Sie, dass diese zusätzlichen Punkte intern in der Funktion sind und nur für die Laufzeitberechnungen verwendet werden). Wenn der Kanal eine lineare Interpolation verwendet, ist dieser Wert immer noch vorhanden, kann aber ignoriert werden, da er keinen Einfluss darauf hat, wie die Kurve interpoliert wird.
Punkte Array-Zeiger Dies ist ein Array, wobei jedes Element im Array eine Punktstruktur ist.


Wenn Sie eine neue Kanalstruktur erstellen, ist standardmäßig die Variable "name" eine leere Zeichenkette, der "type" ist animcurvetype_linear und die "iterations " ist 16. Alle diese Variablen können auf die gewünschten Werte gesetzt werden. Werte gesetzt werden (beachten Sie, dass der Wert "iterations" bei linearen Kurventypen keine Wirkung hat). Nach der Erstellung müssen Sie der Kurve Punkte hinzufügen, was durch Hinzufügen verschiedener Punktstrukturen zum Array "points" geschieht. Diese Punkt-Strukturen können mit der Funktion animcurve_point_new()erstellt werden und enthalten die folgenden Variablen:

Variable Name Daten Typ Beschreibung
posx real Die Position in der Zeit (normiert von 0 bis 1) des Punktes.
Wert real Der Wert des Punktes.

Syntax:

animcurve_channel_new();

Rückgabe:

Struktur-Referenz

Beispiel:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Der obige Code erzeugt eine neue Animationskurven-Struktur und speichert sie in der Variablen "my_curve". Diese Struktur wird dann mit einem Namen und einem Kanal-Array befüllt. Das Kanal-Array enthält einen einzelnen Kanal mit drei Punkten.
