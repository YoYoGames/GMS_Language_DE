animcurve_point_new

Mit dieser Funktion kann eine neue Punktstruktur erstellt werden, die zu einem Animationskurvenkanal hinzugefügt wird. Bei der Erstellung ist eine Punkte-Struktur leer und Sie müssen die folgenden Variablen setzen, um den Punkt zu erzeugen Daten:

Variable Name Daten Typ Beschreibung
posx real Die Position in der Zeit (normiert von 0 bis 1) des Punktes.
Wert real Der Wert des Punktes.

Jede Punktstruktur, die Sie erstellen, sollte zu einem Array hinzugefügt werden, und dies ist das Array, das der Animationskurven-Kanalstruktur-Variable "channels" hinzugefügt wird (wie im Beispiel unten gezeigt).

Syntax:

animcurve_point_new();

Rückgabe:

Struktur-Referenz

Beispiel:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

Der obige Code erzeugt eine neue Animationskurven-Struktur und speichert sie in der Variablen "my_curve". Diese Struktur wird dann mit einem Namen und einem Kanal-Array befüllt. Das Kanal-Array enthält einen einzelnen Kanal mit drei Punkten.
