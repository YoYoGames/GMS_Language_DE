animcurve_get_channel_index

Diese Funktion gibt den Indexwert für einen beliebigen Animationskurvenkanal zurück. Sie übergeben die Animationskurven-ID oder struct, wobei die Kurven-ID der Name der Animationskurve ist, wie sie im Asset-Browser definiert wurde, oder den struct-Zeiger, wie er von der Funktion von der Funktion animcurve_create(). Dann geben Sie den Namen des Kanals als String an, und die Funktion gibt den Indexwert zurück, der diesem Kanal zugeordnet ist. Beachten Sie, dass, wenn die Kurve oder der Kanal nicht existiert, erhalten Sie eine Fehlermeldung.

Syntax:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argument Beschreibung
kurven_strukt_oder_id Die ID oder der Strukturzeiger der Animationskurve, die angepeilt werden soll
Kanal_Name Der Kanalname (eine Zeichenkette).

Rückgabe:

Struktur

Beispiel:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

Der obige Code ruft die Kanalstruktur für den Kanal mit dem Namen "x_pos" im Kurven-Asset "ac_ButtonTween" ab, läuft dann in einer Schleife durch die Punkte auf der Kanalkurve und fügt ihrem Wert einen hinzu.
