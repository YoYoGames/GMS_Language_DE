animcurve_get

Diese Funktion gibt eine Struktur zurück, die alle Daten für die angegebene Animationskurve enthält. Sie geben die Asset-ID der Animationskurve an (wie im Asset-Browser definiert), und die Funktion gibt eine struct mit den folgenden Variablen zurück:

Variable Name Daten Typ Beschreibung
Name String Dies ist der Name der Animationskurve.
Kanäle Array-Zeiger Dies ist ein Array, wobei jedes Element im Array eine Kanalstruktur ist.


Die Kanalvariable in der Struktur ist ein Array, wobei jedes Array-Element eine Struktur mit Daten ist, die sich auf einen Kanal innerhalb der Kurve beziehen. Die Kanal-Struktur hat die folgenden Variablen:

Variable Name Daten Typ Beschreibung
Name String Dies ist der Name des Kanals.
Typ Konstante Dies wird eine der Konstanten animcurvetype_linear für eine lineare Interpolation zwischen den Punkten, oder animcurvetype_catmullrom für eine "glatte" Interpolation zwischen den Punkten mittels Catmull-Rom-Interpolation.
Iterationen Ganzzahl Wenn der Kanal die Katmull-rom ("glatte") Interpolation verwendet, wird hier festgehalten, wie viele Punkte für jedes Segment der Kurve generiert wurden (beachten Sie, dass diese zusätzlichen Punkte intern in der Funktion sind und nur für die Laufzeitberechnungen verwendet werden). Wenn der Kanal eine lineare Interpolation verwendet, ist dieser Wert immer noch vorhanden, kann aber ignoriert werden, da er keinen Einfluss darauf hat, wie die Kurve interpoliert wird.
Punkte Array-Zeiger Dies ist ein Array, wobei jedes Element im Array eine Punktstruktur ist.


Wie bei Kanälen werden die Punkte eines einzelnen Kanals als Strukturen in einem Array gespeichert, wobei jedes Element im Array eine einzelne Punktstruktur ist. Die Punktstruktur hat die folgenden Variablen:

Variable Name Daten Typ Beschreibung
posx real Die Position in der Zeit (normiert von 0 bis 1) des Punktes.
Wert real Der Wert des Punktes.


Beachten Sie, dass die Funktion -1 zurückgibt, wenn sie fehlschlägt (d. h.: die angegebene Animationskurven-ID existiert nicht).

Syntax:

animcurve_get(curve_id);

Argument Beschreibung
Kurven_id Die Asset-Browser-ID (Index) der abzurufenden Animationskurve.

Rückgabe:

Struktur oder -1

Beispiel:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
_channel.type = animcurvetype_linear;
}

Der obige Code ruft die Animationskurvenstruktur für das Kurven-Asset "ac_ButtonTween" ab, überprüft dann den Kurventyp für den ersten Kanal und setzt ihn, wenn er nicht linear ist, auf linear.
