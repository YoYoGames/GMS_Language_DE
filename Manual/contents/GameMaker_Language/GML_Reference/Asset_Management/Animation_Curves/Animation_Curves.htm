Animation Kurven

Animationskurven können im Asset-Browser mit dem Animationskurven-Editor als Assets eingerichtet werden und können dann im Sequenzen-Editor verwendet werden, um Spuren in einer Sequenz zu steuern. Sie können aber auch in Ihrem Code verwendet werden, um Werte im Laufe der Zeit in Ihren Spielen zu ändern, z. B. für Tweening-Effekte auf Schaltflächen, für Bewegungen, zu Farbänderungen und mehr. Sie können mit den hier aufgeführten Funktionen auch Ihre eigenen Animationskurven zur Laufzeit erstellen, was sie zu einem sehr flexiblen Werkzeug für Ihre Spiele macht.

Die verfügbaren Funktionen für Animationskurven sind wie folgt:

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
