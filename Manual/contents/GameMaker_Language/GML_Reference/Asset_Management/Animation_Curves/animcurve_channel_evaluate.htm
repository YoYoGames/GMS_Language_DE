animcurve_channel_evaluate

Diese Funktion kann verwendet werden, um den Wert zu einem bestimmten Zeitpunkt aus einer Kanal-Struktur zu erhalten. Sie liefern den struct-Zeiger für den Kanal (wie er von der Funktion animcurve_get_channel(), oder wie in der Animationskurven-Struktur von der Funktion animcurve_get()) und die "x"-Position (Zeit) entlang des auszuwertenden Kurvenkanals. Diese Position sollte zwischen 0 und 1 liegen, und die Funktion gibt den Kurvenwert an dieser Position zurück, oder sie gibt 0 zurück, wenn die übergebene Kanalstruktur ungültig ist.

Syntax:

animcurve_channel_evaluate(channel_struct, posx);

Argument Beschreibung
Kanal_Struktur Der Strukturzeiger für den auszuwertenden Kanal.
posx Die zu prüfende Position in der Zeit (von 0 bis 1).

Rückgabe:

Real

Beispiel:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

Der obige Code ruft die Kanalstruktur für das Animationskurven-Asset "ac_AlphaCurve" ab. Er verwendet dann den zurückgegebenen Bewertungswert, um das Bild-Alpha für die Instanz einzustellen.
