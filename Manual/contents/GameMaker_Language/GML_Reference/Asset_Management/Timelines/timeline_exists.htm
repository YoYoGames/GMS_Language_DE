timeline_exists

Mit dieser Funktion können Sie prüfen und sehen, ob eine Zeitleiste existiert (gibt true zurück) oder nicht (gibt false zurück). Dies ist besonders nützlich, wenn Sie Zeitleisten dynamisch mit der timeline_add() Funktion, Sie sollten jedoch beachten, dass bei der Suche nach einer nicht initialisierten Variablen (die sonst dem Index einer Zeitleiste zugewiesen wäre) ein Fehler ausgelöst wird.

Syntax:

timeline_exists(ind);

Argument Beschreibung
ind Der Index der Zeitleiste, auf den geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if timeline_exists(global.tl)
{
timeline_delete(global.tl);
}

Der obige Code prüft, ob eine Zeitleiste existiert und in der globalen Variablen "tl" indiziert ist, und wenn ja, dann wird diese Zeitleiste gelöscht.
