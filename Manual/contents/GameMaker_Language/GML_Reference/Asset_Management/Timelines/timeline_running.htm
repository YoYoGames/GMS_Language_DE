timeline_running

Diese Variable enthält den aktuellen Status der zugewiesenen Zeitleiste und gibt true zurück, wenn sie läuft, und false, wenn sie nicht läuft. Sie können diese Variable auch entweder auf true oder false setzen, um Sie können diese Variable auch auf true oder false setzen, um die Zeitleiste jederzeit zu starten und zu stoppen. Es ist zu beachten, dass eine gestoppte Zeitleiste nicht zurückgesetzt wird, so dass sie beim erneuten Starten zu einem späteren Zeitpunkt genau an dem Punkt beginnt, an dem sie gestoppt wurde.

Syntax:

timeline_running;

Rückgabe:

Boolesche

Beispiel:

if !timeline_running
{
timeline_position = 0;
timeline_running = true;
}

Der obige Code prüft, ob die Instanz eine Zeitleiste ausführt, und wenn dies nicht der Fall ist, setzt er die zugewiesene Zeitleiste auf den ersten Moment zurück und startet sie dann.
