zeitleiste_position

Diese Variable enthält die aktuelle Position (Moment), an der sich eine Zeitleiste gerade befindet. Sie können diesen Wert ändern, um Teile der Zeitleiste zu überspringen, Teile zu wiederholen oder die Zeitleiste wieder von vorne zu beginnen.

Syntax:

timeline_position;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel:

if !timeline_running
{
timeline_position = 0;
timeline_running = true;
}

Der obige Code prüft, ob die Instanz eine Zeitleiste ausführt, und wenn dies nicht der Fall ist, setzt er die zugewiesene Zeitleiste auf den ersten Moment zurück und startet sie dann.
