zeitleiste_moment_klar

Mit dieser Funktion können Sie einen bestimmten Moment einer beliebigen, zuvor definierten Zeitleiste von allen Codes und Aktionen löschen.

Syntax:

timeline_moment_clear(ind, step);

Argument Beschreibung
ind Der Index der zu löschenden Zeitleiste.
Schritt Der Moment zum Löschen.

Rückgabe:

N/A

Beispiel:

timeline_moment_clear(global.tl, room_speed * 30);

Der obige Code löscht den angegebenen Zeitpunkt der Zeitleiste, die durch die Variable "global.tl" indiziert ist.
