zeitleiste_max_moment

Diese Funktion gibt den Wert des Moments zurück, in dem die Zeitleiste ihre letzte Aktion ausführt. Wenn Sie also eine Zeitleiste mit 8 verschiedenen Aktionen haben, die 20 Momente auseinander liegen, würde diese Funktion den Momentwert der letzten Aktion in dieser Zeitleiste zurückgeben, das wäre dann 160. Diese Funktion ist eine gute Möglichkeit, um zu testen, ob eine Zeitleiste den letzten aktiven Moment überschritten hat, wenn sie läuft, da die Position der Zeitleiste bei jedem Schritt des Spiels weiter inkrementiert wird, unabhängig davon, ob es weitere Aktionen gibt oder nicht.

Syntax:

timeline_max_moment(ind);

Argument Beschreibung
ind Der Index der Zeitleiste, von der der letzte Moment abgerufen werden soll.

Rückgabe:

N/A

Beispiel:

if timeline_position > timeline_max_moment(timeline_index)
{
timeline_running = false;
}

Der obige Code prüft die aktuelle Position der Zeitleiste mit dem maximalen aktiven Moment, und wenn dieser größer ist, wird die Zeitleiste angehalten.
