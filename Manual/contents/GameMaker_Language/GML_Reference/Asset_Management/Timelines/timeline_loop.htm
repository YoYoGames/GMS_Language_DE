timeline_loop

Diese Variable gibt zurück, ob die Zeitleiste in einer Schleife läuft (true) oder nicht (false). Sie können diese Variable ändern, um die Schleifenbildung ein- oder auszuschalten, und sie funktioniert mit einer negativen Zeitliniengeschwindigkeit (wenn die Position der Zeitlinie unter 0 fällt, beginnt sie wieder am letzten definierten Moment).

Syntax:

timeline_loop;

Rückgabe:

Boolesche

Beispiel:

if !timeline_loop
{
timeline_loop = true;
}

Der obige Code prüft, ob die Zeitleiste, die aktuell einer Instanz zugewiesen ist, eine Schleife durchläuft oder nicht, und wenn nicht, setzt er sie auf true.
