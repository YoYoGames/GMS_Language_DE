zeitleiste_löschen

Mit dieser Funktion können Sie eine beliebige Zeitleiste aus Ihrem Spiel löschen. Wenn diese Zeitleiste im Asset Browser von GameMaker Studio 2 erstellt wurde, beachten Sie bitte, dass sie vollständig aus dem Spiel entfernt wird und der Versuch, die indizierte Zeitleiste erneut aufzurufen, nachdem Sie diese Funktion verwendet haben, führt zu einem Fehler. Wurde die Zeitleiste dynamisch mit der timeline_add() Funktion dynamisch erstellt wurde, dann sollte diese Funktion in dem Moment verwendet werden wenn die Zeitleiste nicht mehr benötigt wird, um Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen (und schließlich zum Absturz bringen) könnten.

Syntax:

timeline_delete(ind);

Argument Beschreibung
ind Der Index der zu löschenden Zeitleiste.

Rückgabe:

N/A

Beispiel:

if timeline_exists(global.tl)
{
timeline_delete(global.tl);
}

Der obige Code prüft, ob eine Zeitleiste existiert und in der globalen Variablen "tl" indiziert ist, und wenn ja, dann wird diese Zeitleiste gelöscht.
