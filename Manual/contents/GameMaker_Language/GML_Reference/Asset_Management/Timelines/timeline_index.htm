zeitleiste_index

Diese Variable enthält den Index der Zeitleiste, die derzeit mit der Instanz verknüpft ist. Sie können sie auf eine bestimmte Zeitleiste setzen, um diese zu verwenden, oder sie auf -1 setzen, um die Verwendung einer Zeitleiste für die Instanz zu beenden (wenn keine Zeitleiste für die Instanz definiert ist, wird auch -1 zurückgegeben). Beachten Sie, dass damit die Zeitleiste nicht gestartet wird - verwenden Sie dafür die Variable timeline_running.

Syntax:

timeline_index;

Rückgabe:

Real (Ganzzahl - Ressourcenindexwert)

Beispiel:

if timeline_exists(global.tl)
{
timeline_index = global.tl;
}

Der obige Code weist der Instanz, die den Code ausführt, eine Zeitleiste zu, die in der Variablen "global.tl" indiziert ist, wenn diese Zeitleiste existiert.
