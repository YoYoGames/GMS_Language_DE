timeline_size

Mit dieser Funktion können Sie die Gesamtzahl der aktiven Momente für eine Zeitleiste ermitteln (ein "aktiver" Moment ist ein Moment, zu dem Code oder DnD™ hinzugefügtwurde). Dies kann praktisch sein, wenn Sie dynamische Zeitleisten mit den Funktionen timeline_moment_add_script() und timeline_moment_clear() Funktionen, da Sie prüfen können, ob eine bestimmte Zeitleiste die richtige Anzahl aktiver Momente oder gar keine hat.

Syntax:

timeline_size(ind);

Argument Beschreibung
ind Der Index der Zeitleiste, aus der die Informationen stammen.

Rückgabe:

N/A

Beispiel:

if timeline_size(global.tl) == 0
{
timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Der obige Code prüft die angegebene Größe der Zeitleiste, und wenn er 0 zurückgibt (d. h.: die Zeitleiste hat keine aktiven Momente), fügt er eine Skriptfunktion hinzu, die zu einem zufälligen Moment innerhalb der Zeitleiste verwendet wird, der in der globalen Variablen "tl" indiziert ist.
