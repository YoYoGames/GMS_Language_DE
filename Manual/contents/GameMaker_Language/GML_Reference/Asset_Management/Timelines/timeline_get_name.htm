timeline_get_name

Diese Funktion kann verwendet werden, um den Namen einer Zeitleiste als String zu erhalten. Wenn die Zeitleiste dynamisch mit der timeline_add() Funktion erstellt wurde, hat der zurückgegebene Name das Format "__newtimelineN", wobei " N" dieNummer der Zeitleiste ist (beginnend bei 0). Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um die Zeitleiste direkt zu referenzieren - dafür Dazu bräuchten Sie den Zeitleistenindex. Sie können diese Zeichenkette jedoch verwenden, um den Zeitleistenindex zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

timeline_get_name(ind);

Argument Beschreibung
ind Der Index der Zeitleiste, deren Name geprüft werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

var _str = timeline_get_name(timeline_index);
show_debug_message("Aktuelle Zeitleiste = " + _str);

Der obige Code ermittelt den Namen der aktuell zugewiesenen Zeitleiste und gibt ihn auf der Konsole aus.
