timeline_moment_add_script

Mit dieser Funktion können Sie dynamisch eine Skriptfunktion zu einem beliebigen "Moment" innerhalb dieser Zeitleiste hinzufügen, wobei ein "Moment" einem Spieltick (oder Schritt). Auf diese Weise können Sie eine neue Zeitleiste mit der timeline_add() Funktion eine neue Zeitleiste erstellen und an jedem beliebigen Punkt verschiedene Verhaltensweisen hinzufügen oder einfach eine zuvor erstellte Zeitleistenressource mit neuen Verhaltensweisen ändern. Beachten Sie dass die Funktion keine zusätzlichen Argumente benötigt, wenn Sie diese Funktion verwenden, und wenn Sie sie verwenden, um ein im Asset-Browser vorhandenes Zeitleisten-Asset zu ändern, dann sind alle Instanzen betroffen, die diese Zeitleiste verwenden, und die Änderung dauert bis das Ende des Spiels (der Aufruf von game_restart() setzt die Änderung auch nicht zurück).

Syntax:

timeline_moment_add_script(ind, step, script);

Argument Beschreibung
ind Der Index der Zeitleiste, zu der ein Moment hinzugefügt werden soll.
Schritt Der Moment (Schritt), zu dem hinzugefügt werden soll.
Skript Der Index der Skriptfunktion, die in den Moment eingefügt werden soll.

Rückgabe:

N/A

Beispiel:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
i += room_speed * 60;
}

Der obige Code erstellt eine neue Zeitleiste und speichert ihren Index in der Variablen "global.tl". Er fügt dann drei zufällig ausgewählte Skripte im Abstand von einer Minute in die Zeitleiste ein.
