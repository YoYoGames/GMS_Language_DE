Zeitleisten

Zeitleisten sind ein leistungsstarker Mechanismus zur Steuerung von Ereignissen in Ihrem Spiel und basieren auf "Momenten", wobei ein "Moment" einfach ein einzelner Spielschritt ist. Diese Zeitleisten werden in der Regel über den Asset-Browser mit dem Zeitleisten-Editor ähnlich wie ein Objekt erstellt, d. h.: Sie erstellen eine Zeitleiste und fügen dann in jedem Moment, in dem sie eine Aktion ausführen soll, etwas Code oder ein DnD™-Symbolhinzu, das sie ausführen soll. Einmal erstellt, können Sie dann Zeitleisten durch Code anhalten, starten und manipulieren. Sie können sogar einzelne Momente modifizieren, indem Sie eine Skriptfunktion definieren und diese dynamisch in die Zeitleiste einfügen (z. B. von einem Controller-Objekt aus), was dies zu einem sehr mächtigen und vielseitigen Werkzeug bei der Erstellung Ihrer Spiele.

Die folgenden Funktionen dienen zum Hinzufügen und Manipulieren von Zeitleisten aus Ihrem Spiel:

  • timeline_exists
  • timeline_get_name
  • timeline_add
  • zeitleiste_löschen
  • timeline_moment_add_script
  • zeitleiste_moment_klar
  • zeitleiste_klar
  • timeline_size
  • zeitleiste_max_moment

Die folgenden Variablen sind in allen Instanzen "eingebaut", um die Arbeit mit Zeitleisten zu erleichtern und Ihnen die Kontrolle darüber zu geben:

  • zeitleiste_index
  • timeline_running
  • timeline_speed
  • zeitleiste_position
  • timeline_loop

Beachten Sie, dass Sie hier weitere Informationen zu Instanzvariablen finden können: Instanz-Variablen.
