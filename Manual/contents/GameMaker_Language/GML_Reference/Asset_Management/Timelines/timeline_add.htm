timeline_add

Mit dieser Funktion können Sie eine neue (leere) Zeitleiste in Ihr Spiel einfügen. Die Funktion gibt den Index dieser neuen Zeitleiste zurück, der in einer Variablen zur Verwendung in allen weiteren Funktionsaufrufen, die diese neue Zeitleiste betreffen, gespeichert werden sollte. Sie sollten auch darauf achten Sie sollten die Funktion timeline_delete() immer dann verwenden, wenn Sie die Zeitleiste nicht mehr verwenden möchten (z. B. am Ende des Raums), um mögliche Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen oder zum Absturz bringen könnten. verlangsamen oder Ihr Spiel zum Absturz bringen könnten. Um einer auf diese Weise erstellten Zeitleiste Momente hinzuzufügen, siehe die Funktion timeline_moment_add_script().

Syntax:

timeline_add();

Rückgabe:

Real

Beispiel:

global.tl = timeline_add();

Der obige Code erstellt eine neue Zeitleiste und speichert deren Index in der Variablen "global.tl".
