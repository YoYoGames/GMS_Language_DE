zeitleiste_klar

Mit dieser Funktion können Sie eine bestimmte Zeitleiste von "Momenten" löschen, wobei alle Codes und Aktionen für diese Zeitleiste entfernt werden und sie leer bleibt.

Syntax:

timeline_clear(ind);

Argument Beschreibung
ind Der Index der zu löschenden Zeitleiste.

Rückgabe:

N/A

Beispiel:

if timeline_position > 200
{
timeline_clear(global.tl);
timeline_index = -1;
}

Der obige Code löscht die angegebene Zeitleiste, die durch die Variable "global.tl" indiziert wird, von allen Momenten, wenn ein bestimmter Moment überschritten wurde.
