raum_vorher

Mit dieser Funktion können Sie den Index des Raums vor der Raumeingabe in die Funktion abrufen. Zum Beispiel können Sie die Variable Raum verwenden, um den Index des aktuellen Raums zu erhalten, und dann diese Funktion verwenden, um den den davor liegenden Raum, wie er im Raum-Manager aufgelistet ist. Wenn es keinen Raum vor dem von Ihnen eingegebenen gibt, wird -1 zurückgegeben. Beachten Sie, dass diese Funktion nicht erkennt oder berücksichtigt Räume, die mit room_add() oder room_duplicate() dynamisch hinzugefügt wurden.

Syntax:

room_previous(numb);

Argument Beschreibung
taub Der Index des Raums, aus dem der vorherige zu holen ist.

Rückgabe:

Index

Beispiel:

if room_previous(room) != -1
{
room_goto_previous());
}

Der obige Code prüft, ob der vorherige Raum existiert, und wenn ja, geht er dorthin.
