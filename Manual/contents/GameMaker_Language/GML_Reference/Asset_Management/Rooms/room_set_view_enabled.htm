room_set_view_enabled

Mit dieser Funktion können Sie die Ansicht eines beliebigen Raums innerhalb Ihres Spiels mit Ausnahme des aktuellen Raums aktivieren(true) oder deaktivieren(false).

Syntax:

room_set_view_enabled(index, val);

Argument Beschreibung
index Der Index des einzustellenden Raums.
val Ob Ansichten im angegebenen Raum aktiviert(true) oder deaktiviert(false) werden sollen.

Rückgabe:

N/A

Beispiel:

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

Dadurch werden Ansichten in dem in "global.myroom" indizierten Raum aktiviert.
