layer_sprite_exists

Mit dieser Funktion können Sie prüfen, ob ein Sprite-Element auf einer bestimmten Ebene existiert. Sie übergeben die ID der Ebene (die Sie beim Erstellen der Ebene mit schicht_erzeugen()) oder den Ebenenname (als String - dies hat Auswirkungen auf die Performance) und die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()) und die Funktion gibt entweder true zurück, wenn das Element existiert, oder false, wenn es nicht existiert.

HINWEIS: Diese Funktion arbeitet im Rahmen des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird - der mit der Funktion layer_set_target_room().

Syntax:

layer_sprite_exists(layer_id, sprite_element_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
sprite_element_id Der eindeutige ID-Wert des zu prüfenden Sprite-Elements

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
layer_sprite_destroy(lay_id, global.TreesSprites);
}

Der obige Code prüft die Ebene "Asset_trees", um zu sehen, ob das angegebene Sprite-Element existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
