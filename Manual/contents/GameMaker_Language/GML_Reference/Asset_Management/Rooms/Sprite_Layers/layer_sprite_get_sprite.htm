layer_sprite_get_sprite

Diese Funktion kann verwendet werden, um den aktuellen Sprite-Index des Sprite-Elements zu erhalten. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit schicht_sprite_erzeugen() oder wenn Sie mit die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert zurück, der den angezeigten Sprite-Index darstellt. Wenn dem Element kein Sprite zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

layer_sprite_get_sprite(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Sprite-Index (Real) oder -1

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
layer_sprite_change(spr_id, spr_Clouds);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das dem Element zugewiesene Sprite zu prüfen und es auf das Sprite "spr_Clouds" zu setzen gesetzt, wenn es nicht bereits vorhanden ist.
