layer_sprite_destroy

Diese Funktion zerstört das angegebene Sprite-Element. Sie übergeben die Sprite-ID (die Sie erhalten, wenn Sie das Sprite mit layer_sprite_create() oder wenn Sie die Ebenen-ID zusammen mit layer_get_sprite_id()) und dies wird es entfernen. Beachten Sie, dass dadurch nicht die Ebene entfernt wird, sondern nur das Sprite, und wenn das Sprite eines ist, das im Raum-Editor hinzugefügt wurde, dann wird das Sprite beim nächsten Verlassen des Raums und bei der Rückkehr neu erstellt neu erstellt. Wenn der Raum jedoch persistent ist, wird das Sprite entfernt, bis die Raumpersistenz wieder ausgeschaltet wird.

Syntax:

layer_sprite_destroy(sprite_element_id)

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu zerstörenden Sprites

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
layer_sprite_destroy(lay_id);
}

Der obige Code prüft die Ebene "Asset_Trees", um zu sehen, ob das angegebene Sprite-Element existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
