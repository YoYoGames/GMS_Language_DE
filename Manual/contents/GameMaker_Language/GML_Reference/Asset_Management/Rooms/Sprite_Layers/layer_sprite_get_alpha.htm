layer_sprite_get_alpha

Mit dieser Funktion können Sie den Alphawert des Sprite-Elements abrufen. Sie geben die Sprite-Element-ID an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen Wert zwischen 0 (völlig transparent) und 1 (völlig undurchsichtig) zurück.

Syntax:

layer_sprite_get_alpha(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real (von 0 bis 1)

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha(spr_id) < 0.1
{
layer_sprite_destroy(spr_id);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das Element-Alpha zu prüfen, und wenn es kleiner als 0,1 ist, dann wird das Ebenenelement zerstört.
