layer_sprite_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert eines Sprite-Elements auf einer Ebene abzurufen. Sie übergeben die ID der Ebene (die Sie erhalten, wenn Sie die Ebene mit layer_create() oder wenn Sie den Ebenennamen zusammen mit layer_get_id()) und den Namen des Sprite-Elements, wie er im Raum-Editor definiert ist. Die Funktion gibt den ID-Wert zurück, der mit diesem Sprite Element auf der Ebene zugeordnet ist.

Beachten Sie, dass diese Funktion speziell für die Verwendung mit Sprites gedacht ist, die in der IDE auf einer Asset-Ebene hinzugefügt wurden. Wenn Sie ein Sprite einer Ebene durch Code mit der Funktion layer_sprite_create(), ein Sprite zu einer Ebene hinzugefügt haben, dann sollte die von dieser Funktion zurückgegebene ID für alle zukünftigen Referenzen verwendet werden (da dieses Sprite-Element keinen Namen hat, der an diese Funktion übergeben werden kann).

Wenn die angegebene Ebene nicht existiert oder das angegebene Sprite-Element nicht gefunden werden kann, gibt die Funktion -1 zurück.

Syntax:

layer_sprite_get_id(layer_id, sprite_element_name)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene
sprite_element_name Der eindeutige Name des Sprite-Elements auf der Ebene, wie im Raum-Editor definiert

Rückgabe:

Sprite-Element-ID-Wert (Real)

Beispiel:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

Der obige Code ruft die Ebenen-ID für die Ebene "Assets_trees" ab und verwendet diese dann, um die ID des Sprite-Elements "graphic_254367CB" auf dieser Ebene abzurufen. Die abgefragte Sprite-Element-ID wird dann verwendet, um das Sprite des Elements Sprite.
