layer_sprite_alpha

Diese Funktion steuert das Alpha (die Transparenz) des Sprites auf der Asset-Ebene. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie mit die Funktion layer_sprite_get_id()), und legen dann den zu verwendenden Alpha-Wert fest. Alpha kann zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) liegen, wobei der Standard-Alpha-Wert für das Sprite-Element 1 ist. Beachten Sie wenn die Ebene, der das Sprite-Element zugewiesen wurde, nicht sichtbar ist - oder das Element selbst unsichtbar gemacht wurde - werden Sie mit dieser Funktion keinen Unterschied sehen, bis die Ebene oder das Element wieder sichtbar gemacht wurde.

Syntax:

layer_sprite_alpha(sprite_element_id, alpha);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
alpha Das Alpha für das Sprite, von 0 bis 1 (Standard ist 1)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

Der obige Code ruft den ID-Wert des Sprite-Assets mit dem Namen "Clouds" ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann dessen Alpha auf einen Zufallswert zwischen 0 und 1.
