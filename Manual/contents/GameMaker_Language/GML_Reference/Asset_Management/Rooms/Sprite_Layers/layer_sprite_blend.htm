layer_sprite_blend

Diese Funktion steuert die Überblendung (oder "Färbung") des Sprites. Der Standardwert ist -1 (was die Konstante c_white darstellt, die auch verwendet werden kann). Jeder andere Wert (einschließlich interner Farbkonstanten wie c_red, oder c_aqua) wird die angegebene Farbe mit dem Original-Sprite überblenden. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie mit die Funktion layer_sprite_get_id()) und legen dann die zu verwendende Überblendfarbe fest. Unten sehen Sie ein Beispiel für ein Sprite, das mit verschiedenen Farben überblendet wurde:

Bitte beachten Sie, dass Sie versuchen sollten, das Überblenden auf der HTML5-Plattform zu begrenzen (es sei denn, Sie verwenden WebGL), da jedes überblendete Sprite separat zwischengespeichert werden muss und daher viele überblendete Sprites die Leistung beeinträchtigen können (Sie können die Cache-Größe auch mit der Funktion sprite_set_cache_size()).

Syntax:

layer_sprite_blend(sprite_element_id, blend);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
mischen Die Farbe, die mit dem Sprite überblendet werden soll (Standard ist c_white)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

Der obige Code holt sich den ID-Wert des Sprites namens "Clouds", das der Ebene "Asset_sky" zugewiesen ist, und färbt es dann in einer Farbe ein.
