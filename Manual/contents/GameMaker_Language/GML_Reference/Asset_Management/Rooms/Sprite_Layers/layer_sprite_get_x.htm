layer_sprite_get_x

Diese Funktion kann verwendet werden, um die x-Position des Sprite-Elements im Raum zu erhalten. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit schicht_sprite_erzeugen() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt den Wert der x-Position zurück.

Syntax:

layer_sprite_get_x(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_x(spr_id) < 0
{
layer_sprite_x(spr_id, room_width);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die x-Position des Elements zu prüfen, und wenn sie kleiner als 0 ist, wird das Ebenenelement an eine andere x-Position verschoben.
