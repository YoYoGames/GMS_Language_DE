layer_sprite_get_yscale

Mit dieser Funktion können Sie den aktuellen Skalierungsmultiplikator entlang der y-Achse für das Sprite-Element ermitteln. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert für die y-Skalierung zurück, wobei 1 keine Skalierung bedeutet.

Syntax:

layer_sprite_get_yscale(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_yscale(spr_id) != 1
{
layer_sprite_yscale(spr_id, 1);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den y-Skala-Wert zu prüfen, und wenn er etwas anderes als 1 ist, wird er auf 1 gesetzt.
