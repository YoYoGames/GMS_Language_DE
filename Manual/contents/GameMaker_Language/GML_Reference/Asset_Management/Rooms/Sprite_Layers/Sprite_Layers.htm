Sprite-Ebenen

Der GameMaker Studio 2 Raum-Editor erlaubt es Ihnen, Sprites mit Hilfe von Ebenen in jeden beliebigen Raum einzufügen. Wie der Name schon sagt, ist eine Ebene Wie der Name schon sagt, ist eine Ebene einfach eine Sammlung von Elementen, die sich alle in der gleichen Tiefe innerhalb des Raums befinden, und durch das Stapeln von Ebenen können Sie einige Dinge über oder unter anderen zeichnen lassen. Sie können auch bestimmte Aspekte von Ebenen über den Code steuern, indem Sie Dinge hinzufügen oder entfernen oder bestimmte Eigenschaften der Ebene ändern. Sie können auch bestimmte Aspekte von Ebenen vom Code aus steuern, indem Sie Dinge hinzufügen oder entfernen oder bestimmte Eigenschaften der Ebene oder ihrer Inhalte zur Laufzeit ändern.
