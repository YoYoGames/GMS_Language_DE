layer_sprite_yscale

Mit dieser Funktion können Sie ändern, ob das angegebene Sprite-Element auf einer Ebene entlang der y-Achse skaliert werden soll oder nicht. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den Skalierungswert fest. Ein Skalierungswert von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 z. B. halbiert die Höhe des Sprites), größere Werte skalieren nach oben, und negative Werte spiegeln das Sprite und skalieren es, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das verwendete Sprite einfach von oben nach unten an seiner (0, 0)-Position ohne Skalierung gespiegelt).

Syntax:

layer_sprite_yscale(sprite_element_id, yscale);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
y-Skala Der y-Skalenwert (Standard ist 1)

Rückgabe:

N/A

Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

Der obige Code prüft das Sprite-Element, das der Ebene zugewiesen ist, auf der sich die Instanz befindet, die den Code ausführt, und wenn es in eine der beiden Richtungen skaliert ist, werden sowohl die x-Achsen-Skala als auch die y-Achsen-Skala auf 1 gesetzt.
