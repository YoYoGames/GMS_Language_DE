layer_sprite_xscale

Mit dieser Funktion können Sie ändern, ob das angegebene Sprite-Element auf einer Ebene entlang der X-Achse skaliert werden soll oder nicht. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den Skalierungswert fest. Ein Skalierungswert von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 z. B. wird die Hälfte der Breite des Sprites), größere Werte skalieren nach oben, und negative Werte spiegeln das Sprite und skalieren es, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das verwendete Sprite einfach von rechts nach links um seine Ursprungsposition gespiegelt, ohne Skalierung).

Syntax:

layer_sprite_xscale(sprite_element_id, xscale);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
x-Skala Der x-Skalenwert (Voreinstellung ist 1)

Rückgabe:

N/A

Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

Der obige Code prüft das Sprite-Element, das der Ebene zugewiesen ist, auf der sich die Instanz befindet, die den Code ausführt, und wenn es in eine der beiden Richtungen skaliert ist, werden sowohl die x-Achsen-Skala als auch die y-Achsen-Skala auf 1 gesetzt.
