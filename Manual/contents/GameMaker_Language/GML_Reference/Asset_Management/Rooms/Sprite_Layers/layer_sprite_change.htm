layer_sprite_change

Mit dieser Funktion können Sie die Sprite-Ressource ändern, die einem bestimmten Sprite-Element auf einer Ebene zugewiesen ist. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit schicht_sprite_erzeugen() oder wenn Sie die Funktion schicht_sprite_get_id()), und geben Sie dann eine Sprite-Indexelement-Ressource an, die geändert werden soll. Beachten Sie, dass das Sprite, wenn es Unterbilder hat, auch animiert wird (dies kann gesteuert werden unter Verwendung der schicht_sprite_index() und layer_sprite_speed() Funktionen gesteuert werden). Beachten Sie, dass Sie einen Wert von -1 als neuen Sprite-Index zuweisen können und kein Sprite angezeigt wird, obwohl das Element noch existiert und später wieder geändert werden kann.

Syntax:

layer_sprite_change(sprite_element_id, sprite_index)

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
sprite_index Der neue zu verwendende Sprite-Index

Rückgabe:

N/A

Beispiel:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
layer_sprite_change(s, spr_nighttime);
}

Der obige Code prüft den Sprite-Index des Elements mit der ID, die in der globalen Variablen global.Asset_sprite gespeichert ist, und wenn er nicht "spr_Nighttime" ist, wird dieses Sprite dem Element zugewiesen.
