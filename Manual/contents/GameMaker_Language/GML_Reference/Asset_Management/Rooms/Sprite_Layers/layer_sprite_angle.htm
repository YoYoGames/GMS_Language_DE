layer_sprite_angle

Mit dieser Funktion können Sie den Winkel für das angegebene Sprite-Element auf einer Ebene ändern. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und setzen dann den Winkelwert, von 0 bis 359 gegen den Uhrzeigersinn, wobei 0 nach rechts, 90 nach oben, 180 nach links und 270 nach unten zeigt. Wenn Sie einen Wert einen Wert größer als 360 einstellen, wird dieser in einer Schleife in den Bereich von 0 bis 359 gebracht.

Syntax:

layer_sprite_angle(sprite_element_id, angle);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
Winkel Der Winkel des Sprites (Standard ist 0)

Rückgabe:

N/A

Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0);
}

Der obige Code prüft das Sprite-Element, das der Ebene zugewiesen ist, auf der sich die Instanz befindet, die den Code ausführt, und wenn sein Winkel nicht 0 ist, wird er auf 0 gesetzt.
