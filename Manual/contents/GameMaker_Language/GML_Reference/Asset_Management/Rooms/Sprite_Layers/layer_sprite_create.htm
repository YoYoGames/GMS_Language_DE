layer_sprite_create

Mit dieser Funktion können Sie einer Ebene eine Sprite-Ressource zuweisen, die in Ihrem Projekt verwendet werden soll. Sie geben die Ebenen-ID an (die Sie erhalten, wenn Sie die Ebene mit layer_create() oder wenn Sie den Ebenennamen zusammen mit layer_get_id()), eine Position innerhalb des Raums und einen Sprite-Index (das wäre der Name des Sprites, wie er im Asset-Browser angezeigt wird), und es wird zu der Ebene hinzugefügt werden. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Layer-Funktionen für Sprites verwendet werden kann.

Syntax:

layer_sprite_create(layer_id, x, y, sprite)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene
x Die zu verwendende x-Position
y Die zu verwendende y-Position
Sprite Der zu verwendende Sprite-Index

Rückgabe:

Sprite-Element-ID (Real)

Beispiel:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
var _x = random(room_width);
var _y = room_height - 100;
global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

Der obige Code erstellt eine neue Ebene und fügt ihr dann 10 neue Sprite-Elemente hinzu, wobei die ID jedes Elements in einem Array gespeichert wird.
