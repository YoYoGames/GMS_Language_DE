schicht_sprite_get_speed

Diese Funktion kann verwendet werden, um den aktuellen Geschwindigkeitsmultiplikatorwert des Sprite-Elements zu erhalten. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert zurück, der den Geschwindigkeitsmultiplikator darstellt, der zur Animation des Sprites verwendet wird. Der Standardwert ist 1.

Syntax:

layer_sprite_get_speed(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_speed(spr_id) > 0
{
layer_sprite_speed(spr_id, 0);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Animationsgeschwindigkeit für das Element zu prüfen, und wenn sie größer als 0 ist, wird sie auf 0 gesetzt.
