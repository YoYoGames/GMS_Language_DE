layer_sprite_get_index

Mit dieser Funktion können Sie den aktuellen Bildindexwert des Sprite-Elements ermitteln. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit schicht_sprite_erzeugen() oder wenn Sie die Funktion layer_sprite_get_id()), und die Funktion gibt einen realen Wert zurück, der den angezeigten Bildindex für das Sprite-Element darstellt. Die Funktion gibt -1 zurück, wenn entweder das Sprite-Element nicht existiert oder dem Element kein gültiges Sprite zugewiesen ist.

Syntax:

layer_sprite_get_index(sprite_element_id);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprite-Elements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real (der aktuelle Sprite-Bildindex oder -1)

Beispiel:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_index(spr_id) < 4
{
layer_sprite_index(spr_id, 4);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_sky" und verwendet diese dann, um die ID des Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um zu prüfen, ob der Bildindex für das Element kleiner als 4 ist, und wenn ja, wird er auf 4 gesetzt.
