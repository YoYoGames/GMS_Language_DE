layer_sprite_index

Diese Funktion kann verwendet werden, um den Bildindex eines Sprite-Assets zu setzen, das mehrere Unterbilder auf einer Ebene hat. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den zu verwendenden Bildindex fest. Wenn Sie einen Wert außerhalb des Bereichs der Sub-Images setzen, dann wird der Bildindex in einer Schleife umlaufen.

Syntax:

layer_sprite_index(sprite_element_id, image_index);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu setzenden Sprite-Elements
index Der Bildindex, der für das Sprite verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "sprite_trees" und verwendet diese dann, um die ID des angegebenen Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den Bildindex des Elements zu ändern.
