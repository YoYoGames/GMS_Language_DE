schicht_sprung_geschwindigkeit

Mit dieser Funktion können Sie die Geschwindigkeit eines animierten Bildes, das einem Sprite-Element zugewiesen wurde, einstellen. Sie geben die ID des Sprite-Elements an (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und setzen dann die Animationsgeschwindigkeit. Wenn die IDE die Animationsgeschwindigkeit für das verwendete Sprite so eingestellt hat, dass sie als Frames Per Game Frame berechnet wird, dann ist der angegebene Geschwindigkeits Wert ein Multiplikator, wobei 1 der Standardwert ist und eine Einstellung von 0,5 die Animationsgeschwindigkeit halbiert, während eine Einstellung von 1,5 sie verdoppelt. Wenn die IDE jedoch die Animationsgeschwindigkeit für das verwendete Sprite auf die Berechnung von Frames Per Second (Bilder pro Sekunde) festgelegt hat, dann ist der angegebene Geschwindigkeitswert ein buchstäblicher Wert für Bilder pro Sekunde, so dass das Setzen dieses Wertes auf z. B. 5 dazu führt, dass die Animation in jeder Sekunde, in der das Spiel läuft, 5 Bilder wiederholt. Wenn das verwendete Sprite keine Sub-Images hat, hat diese Funktion keine Auswirkung haben.

Syntax:

layer_sprite_speed(sprite_element_id, speed);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
Geschwindigkeit Der Geschwindigkeitsmultiplikator/-wert, der für das Element verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Der obige Code ermittelt die Ebenen-ID für die Ebene "Assets_water" und verwendet diese dann, um die ID des angegebenen Sprite-Elements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Bildgeschwindigkeit des Elements zu ändern.
