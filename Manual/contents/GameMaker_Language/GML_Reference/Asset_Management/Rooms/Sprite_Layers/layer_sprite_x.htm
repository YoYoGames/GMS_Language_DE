layer_sprite_x

Diese Funktion steuert die Position entlang der x-Achse des Raums des Asset-Sprite-Elements auf der Ebene. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit layer_sprite_create() oder wenn Sie die Funktion layer_sprite_get_id()), und legen dann den zu verwendenden x-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

layer_sprite_x(sprite_element_id, x);

Argument Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
x Die x-Position für das Sprite

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

Der obige Code ruft den ID-Wert des Sprite-Assets "Clouds" ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine x-Position auf einen Zufallswert zwischen 0 und der Breite des Raums.
