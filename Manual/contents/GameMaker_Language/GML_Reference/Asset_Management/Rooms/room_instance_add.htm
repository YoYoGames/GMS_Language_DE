room_instance_add

Mit dieser Funktion können Sie eine Instanz in einen beliebigen anderen Raum als den aktuellen und an einer beliebigen Position innerhalb dieses Raums hinzufügen. Die Funktion gibt die eindeutige id der Instanz zurück, die dann in weiteren Funktionsaufrufen verwendet werden kann, um Eigenschaften usw. dieser Instanz zu setzen, aber erst, wenn das Spiel den angegebenen Raum betreten hat. Wenn Sie eine Instanz im aktuellen Raum erstellen möchten, sollten Sie die Funktion instance_create_layer(). Beachten Sie dass der Aufruf dieser Funktion bei einem Raum-Asset, das im Asset-Browser erstellt wurde , die Instanz dauerhaft zum Raum hinzufügt, und dass auch der Aufruf von game_restart() den Raum nicht in seinen ursprünglichen Zustand zurückversetzt (nur wenn Sie das Spiel beenden und es erneut öffnen, wird der Raum wieder in seinen ursprünglichen Zustand versetzt).

Syntax:

room_instance_add(index, x, y, obj);

Argument Beschreibung
index Der Index des Raums, dem eine Objektinstanz hinzugefügt werden soll.
x Die x-Position der neuen Instanz.
y Die y-Position der neuen Instanz.
obj Das Objekt, von dem eine Instanz hinzugefügt werden soll.

Rückgabe:

Instanz-ID

Beispiel:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

Der obige Code fügt dem Spiel einen neuen Raum hinzu und kopiert dann den Inhalt des als "rm_Base" indizierten Raums in diesen. Er fügt dann eine Instanz des Objekts "obj_player" an der Position 100,100 dieses neuen Raums hinzu.
