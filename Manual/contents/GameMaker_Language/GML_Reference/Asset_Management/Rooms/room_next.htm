raum_next

Mit dieser Funktion können Sie den Index des Raums nach der Eingabe des Raums in die Funktion abrufen. Zum Beispiel können Sie die Variable Raum verwenden, um den Index des aktuellen Raums zu erhalten, und dann diese Funktion verwenden, um den den Raum, der darauf folgt, wie im Raum-Manager aufgelistet. Wenn es keinen Raum nach dem von Ihnen eingegebenen Raum gibt, wird -1 zurückgegeben. Beachten Sie, dass diese Funktion nicht erkennt oder berücksichtigt Räume, die mit room_add() oder room_duplicate() dynamisch hinzugefügt wurden.

Syntax:

room_next(numb);

Argument Beschreibung
taub Der Index des Raums, um den übernächsten zu erhalten.

Rückgabe:

Index

Beispiel:

if room_next(room) != -1
{
room_goto_next());
}

Der obige Code prüft, ob der nächste Raum existiert, und wenn ja, geht er dorthin.
