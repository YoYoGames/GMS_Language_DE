Raum_hinzufügen

Diese Funktion erstellt einen neuen, leeren Raum und fügt ihn zu Ihrem Spiel hinzu. Sie gibt seinen Index zurück, der in einer Variablen für alle weiteren Codes gespeichert wird, die sich mit diesem Raum befassen. Beachten Sie, dass jeder Raum dem Spiel dauerhaft hinzugefügt wird, bis die ausführbare Datei geschlossen wird, d. h.: Räume, die durch Code hinzugefügt wurden, können nicht wieder gelöscht werden. Dies hat wichtige Auswirkungen auf die Speichernutzung und daher sollten Sie diese Funktion mit Vorsicht verwenden.

HINWEIS: Neue Räume sind nicht Teil der üblichen Raumordnung, sie haben also keinen "vorherigen" oder "nächsten" Raum (was bedeutet, dass die Funktionen room_next() und room_previous() nicht funktionieren). Um von einem hinzugefügten Raum zu einem anderen zu springen, müssen Sie den Index des Raums selbst verwenden.

Syntax:

room_add();

Rückgabe:

Index

Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Dadurch wird ein neuer Raum erstellt und sein Index in der Variablen "global.myroom" gespeichert. Dann wird seine Breite auf 640 Pixel, seine Höhe auf 480 Pixel und seine Persistenz auf false gesetzt.
