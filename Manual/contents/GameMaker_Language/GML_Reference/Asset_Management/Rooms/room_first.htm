Zimmer_erste

Diese schreibgeschützte Variable gibt den Index des allerersten Raums im Spiel zurück (dieser wird durch die Reihenfolge definiert, in der die Räume im Raum-Manager erscheinen, und nicht durch die Reihenfolge, in der sie erstellt wurden).

Syntax:

room_first;

Rückgabe:

Index

Beispiel:

wenn lebt < 1
{
room_goto(room_first);
}

Der obige Code prüft die Variable "lives" und geht, wenn sie kleiner als 1 ist, in den ersten Raum des Spiels.
