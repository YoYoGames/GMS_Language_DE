raum_neustart

Mit dieser Funktion wird der aktuelle Raum neu gestartet, als ob er gerade betreten worden wäre. Beachten Sie, dass der Raum erst am Ende des Ereignisses, in dem die Funktion aufgerufen wurde, neu gestartet wird, so dass jeglicher Code nach dem Aufruf dieser Funktion weiterhin ausgeführt wird, wenn er sich im selben Ereignis befindet. Die Einschränkung hierbei ist, dass jeglicher Code, der Instanzen erzeugt, der nach dieser Funktion aufgerufen wird, die Instanz nicht erzeugt und daher vor dem Raumwechsel aufgerufen werden sollte. Diese Funktion löst auch das Ereignis " Room End" aus.

Syntax:

room_restart();

Rückgabe:

N/A

Beispiel:

wenn lebt < 1 room_restart();

Der obige Code prüft, ob die Variable "lives" kleiner als 1 ist, und wenn dies der Fall ist, wird der Raum neu gestartet.
