Zimmer

Diese Variable enthält den Raumindex für den aktuellen Raum, in dem Ihr Spiel läuft. Dies ist keine schreibgeschützte Variable, aber wenn Sie diese Variable ändern, wird der Indexwert für den aktuellen Raum nicht geändert, sondern der Raum wird an den Index angepasst, auf den Sie die Variable auf den Sie die Variable gesetzt haben. Seien Sie vorsichtig, wenn Sie dies tun, denn wenn der Index, auf den Sie die Variable ändern, nicht gültig ist, wird das Spiel einen Fehler ausgeben und geschlossen. Im Allgemeinen ist es viel besser, die Funktion room_goto() um Räume zu ändern.

Hinweis: Die Raum-IDs basieren nicht auf ihrer Reihenfolge im Asset-Browser oder im Raum-Manager, daher sollten Sie es vermeiden, direkt einen Zahlenwert anzugeben. Verwenden Sie stattdessen die Raumkonstante für die Kühlstelle, auf die Sie verweisen wollen (das ist einfach ihr Name), oder rufen Sie sie über eine Funktion ab.

Syntax:

Zimmer;

Rückgabe:

Real (eindeutiger Indexwert)

Beispiel:

if (room == rm_level1)
{
audio_play_sound(snd_level1, 1, 1);
}

Der obige Code prüft, ob der aktuelle Raum die erste Ebene ist, und spielt in diesem Fall die Musik dieser Ebene ab.
