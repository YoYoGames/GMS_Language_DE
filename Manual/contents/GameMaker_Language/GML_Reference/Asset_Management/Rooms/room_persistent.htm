raum_persistent

Diese Variable kann verwendet werden, um das Persistent-Flag für den aktuellen Raum zu erhalten und zu setzen. Wenn sie auf true gesetzt ist, wird der Raum als persistent betrachtet. In diesem Fall wird jedes Mal, wenn Sie den Raum verlassen und wieder zurückkommen, der Zustand der Instanzen innerhalb dieses Raums beibehalten werden. Wenn es jedoch auf false gesetzt ist, wird der Raum jedes Mal, wenn Sie zurückkehren, auf seinen Ausgangszustand zurückgesetzt. Beachten Sie, dass ein persistenter Raum wesentlich mehr Speicherplatz benötigt als ein normaler Raum und es wird nicht empfohlen zu viele von ihnen in Ihrem Spiel zu haben.

Syntax:

raum_persistent;

Rückgabe:

Boolesche

Beispiel:

if lives < 1
{
room_persistent = false;
room_goto(rm_start);
}

Der obige Code prüft die Variable "lives" und wenn sie kleiner als 1 ist, setzt er die Raumpersistenz auf false und wechselt dann die Räume.
