raum_holen_name

Diese Funktion kann verwendet werden, um den Namen des angegebenen Raums als String zurückzugeben. Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um den Raum direkt zu referenzieren - dafür würden Sie den Raumindex benötigen. Sie können jedoch mit dieser String verwenden, um den Raumindex zu erhalten, indem Sie den zurückgegebenen String zusammen mit der Funktion asset_get_index().

Syntax:

room_get_name(index);

Argument Beschreibung
index Der Index des Raums, dessen Name geprüft werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

Der obige Code ermittelt den Namen des aktuellen Raums und zeichnet ihn auf den Bildschirm.
