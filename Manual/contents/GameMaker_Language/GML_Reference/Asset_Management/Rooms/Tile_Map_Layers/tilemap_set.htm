tilemap_set

Mit dieser Funktion kann eine beliebige Zelle (Rasterquadrat) innerhalb des Tile-Map-Elements auf dem Layer auf eine neue Kachel gesetzt werden. Sie geben die ID des Tilemap-Elements an (die Sie erhalten, wenn Sie ein Tilemap-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und dann die zu setzenden Kacheldaten sowie die Position innerhalb der Kachelübersicht. Sie können Kacheldaten mit der Funktion tilemap_get() abrufen und sie dann mit den tile_-Funktionen ändern, bevor Sie die Zelle mit dieser Funktion setzen. Die Funktion gibt true zurück, wenn die Kachel erfolgreich gesetzt wurde, und false, wenn es ein Problem gab und sie nicht gesetzt wurde.

Syntax:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
Kacheldaten Die zu setzenden Kacheldaten
xcell Die Position der Zelle (Gitter), die entlang der x-Achse gesetzt werden soll
ycell Die Position der Zelle (Gitter), die entlang der y-Achse gesetzt werden soll

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Der obige Code ruft die ID für die Kachelübersicht auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel in Zelle (0, 0) abzurufen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte erneut zu setzen.
