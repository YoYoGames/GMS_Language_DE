tile_get_rotate

Mit dieser Funktion können Sie prüfen, ob in einem gegebenen Satz von Kacheldaten die Kachel um 90 Grad gedreht wurde oder nicht. Sie geben die Kacheldaten an, die mit der Funktion tilemap_get()abgerufen werden können, und die Funktion gibt true zurück, wenn die Kachel gedreht ist, oder false, wenn nicht.

Syntax:

tile_get_rotate(tiledata)

Argument Beschreibung
Kacheldaten die zu prüfenden Kacheldaten

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Der obige Code holt sich die Kachel-Map-ID aus der angegebenen Ebene und ermittelt dann die x- und y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die Kachel-Daten zu erhalten, die gedreht werden und dann verwendet werden, um die Kachel wieder zu setzen.
