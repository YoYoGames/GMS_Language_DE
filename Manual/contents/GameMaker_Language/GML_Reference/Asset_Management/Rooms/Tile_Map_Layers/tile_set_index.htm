tile_set_index

Mit dieser Funktion können Sie den Kachelindex (die Position der Kachel innerhalb des Kachelsatzbildes) für einen Satz von Kacheldaten setzen. Sie geben die Kacheldaten, die mit der Funktion tilemap_get()abgerufen werden können, und dann den ganzzahligen Wert, der die Position der Kachel (ihren Index) innerhalb des Kachelsatzes darstellt. Die Funktion gibt einen modifizierten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() Funktion.

Syntax:

tile_set_index(tiledata, index)

Argument Beschreibung
Kacheldaten die zu setzenden Kacheldaten
index Der ganzzahlige Indexwert des zu verwendenden Kachelsatzes

Rückgabe:

Real (Kacheldaten)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Der obige Code holt sich die Kachel-Map-ID aus der angegebenen Ebene und verwendet diese dann, um die Kacheldaten für die Zelle unter der Mausposition zu erhalten. Diese Daten werden dann verwendet, um den Kachelindex auf eine Zufallszahl zu setzen.
