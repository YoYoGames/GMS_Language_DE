tilemap_set_height

Mit dieser Funktion können Sie die Größe eines Tile-Map-Elements ändern. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die neue Höhe der Kachelkarte in Kachelzellen.

Syntax:

tilemap_set_height(tilemap_element_id, height)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, für das die Höhe festgelegt werden soll
Höhe Der Höhenwert (in "Zellen")

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_height(map_id) != room_height div 16
{
tilemap_set_height(map_id, room_height div 16);
}

Der obige Code prüft die Breite einer bestimmten Kachelkarte und wenn sie nicht die richtige Größe hat, wird die Breite eingestellt.
