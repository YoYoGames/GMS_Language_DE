tilemap_x

Diese Funktion steuert die Position entlang der X-Achse des Raums des Asset-Kachelkartenelements auf der Ebene. Sie geben die Kachel-Map-Element-ID an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und legen dann den zu verwendenden x-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

tilemap_x(tilemap_element_id, x);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
x Die x-Position für die Kachelkarte

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

Der obige Code ruft den ID-Wert des Kachelkarten-Assets ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine x-Position auf einen Zufallswert zwischen 0 und der Breite des Raums.
