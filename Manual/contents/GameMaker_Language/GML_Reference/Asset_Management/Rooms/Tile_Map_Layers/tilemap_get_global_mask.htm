tilemap_get_global_mask

Mit dieser Funktion kann der Bitmaskenwert für alle Kachelmaps abgerufen werden. Zurückgegeben wird der aktuelle Maskenwert oder -1, wenn ein Fehler vorliegt, oder 0, wenn keine Maske angegeben ist. Weitere Informationen zu globalen Kachel-Map-Bitmasken finden Sie in der Funktion tilemap_set_global_mask().

Syntax:

tilemap_get_global_mask();

Rückgabe:

Real (0 für keine Maske, -1 für einen Fehler, sonst Maskenwert)

Beispiel:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_global_mask(new_mask);
}

Der obige Code ruft den globalen Maskenwert ab, der mit allen Kachelmaps verknüpft ist. Wenn er nicht mit dem in der Variablen "new_mask" definierten Wert übereinstimmt, wird er auf diesen Wert gesetzt.
