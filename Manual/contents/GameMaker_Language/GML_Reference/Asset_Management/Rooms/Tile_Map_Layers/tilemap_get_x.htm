tilemap_get_x

Mit dieser Funktion können Sie die x-Position (innerhalb des Raumes) des Tile-Map-Elements abfragen. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt die x-Achsenposition zurück.

Syntax:

tilemap_get_x(tilemap_element_id);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die x-Position von

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

Der obige Code verwendet die abgefragte Kachel-Map-ID, um die Kachel-X- und -Y-Position der Kachel-Map zu ermitteln, und verwendet diese Werte dann zum Verschieben der Kachel.
