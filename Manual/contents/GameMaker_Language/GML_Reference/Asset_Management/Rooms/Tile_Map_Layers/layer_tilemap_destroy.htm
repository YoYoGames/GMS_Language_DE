layer_tilemap_destroy

Diese Funktion zerstört das angegebene Tile-Map-Element. Sie übergeben die ID der Kachelkarte (die Sie erhalten, wenn Sie die Kachelkarte mit layer_tilemap_create() oder wenn Sie die Ebenen-ID zusammen mit layer_get_tilemap_id()) und diese Funktion wird sie entfernen. Beachten Sie, dass dadurch nicht die Ebene entfernt wird, sondern nur die Kachelübersicht, und wenn die Kachelübersicht eine ist, die im Raum-Editor hinzugefügt wurde, dann wird die Kachelübersicht beim nächsten Verlassen des Raums und bei der Rückkehr neu erstellt neu erstellt. Wenn der Raum jedoch persistent ist, wird die Tilemap entfernt, es sei denn, die Raumpersistenz wird wieder ausgeschaltet.

Syntax:

layer_tilemap_destroy(tilemap_element_id)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert der Kachel-Map, die zerstört werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
{
layer_tilemap_destroy(tile_id);
}

Der obige Code prüft die Ebene "Tiles_trees", um zu sehen, ob das angegebene Kachel-Map-Element existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
