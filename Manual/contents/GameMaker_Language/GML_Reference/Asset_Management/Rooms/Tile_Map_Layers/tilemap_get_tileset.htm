tilemap_get_tileset

Mit dieser Funktion können Sie den Indexwert der Kachel-Set-Ressource abrufen, die einem bestimmten Kachel-Map-Element auf einer Ebene zugeordnet ist. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_erstellen() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt den Tile-Set-Indexwert zurück.

Syntax:

tilemap_get_tileset(tilemap_element_id);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, von dem der Kachelsatz abgerufen werden soll

Rückgabe:

Kachel Indexwert setzen (Real)

Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
tilemap_tileset(map_id, ts_Nighttime);
}

Der obige Code prüft den aktuellen Kachelsatz, der der Ebene "Tiles_trees" zugewiesen ist, und wenn es nicht "ts_Nighttime" ist, wird dieser Kachelsatz der Kachelkarte zugewiesen.
