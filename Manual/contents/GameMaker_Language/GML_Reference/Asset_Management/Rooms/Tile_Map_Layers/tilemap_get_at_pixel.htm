tilemap_get_at_pixel

Mit dieser Funktion können Sie die Kacheldaten von einer Position (innerhalb des Raums) des Kachelkartenelements abrufen. Sie geben die Kachel-Map-Element-ID an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()) sowie die x- und y-Position im Raum, von der die Kacheldaten abgeholt werden sollen, und die Funktion gibt den Kacheldaten-"Blob" zurück. Diese Daten sind im Wesentlichen eine Bitmaske, die den Kachelindex, die Booleschen Werte für Spiegeln/Drehen/Spiegeln und alle angewendeten Maskendaten enthält (siehe tilemap_set_mask() für Details), und der resultierende Datenwert kann dann Der resultierende Datenwert kann dann in den Funktionen von tile_* verwendet werden, um die Eigenschaften einer Kachel zu ändern. Wenn Sie die Kacheldaten von einer bestimmten Kachelzelle benötigen, sollten Sie die Funktion tilemap_get() stattdessen verwenden.

WICHTIG! Wenn die Kacheln in der Tile-Map unverändert geblieben sind (d. h.: sie sind nicht gedreht oder gespiegelt usw...), dann ist der Rückgabewert der Kachelsatzdaten genau gleich dem Index der Kachel auf dem Kachelsatz. Sie können also "Kollisions Maps" von Kacheln erstellen, indem Sie z. B. eine Kachel mit dem Index 1 im Kachelsatz verwenden - und dann diese Funktion verwenden, um auf 1 oder 0 (eine leere Kachel) zu prüfen, um Kollisionen zu berechnen.

Syntax:

tilemap_get_at_pixel(tilemap_element_id, x, y);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die Kacheldaten von
x Die Position entlang der x-Achse, von der die Kacheldaten abgerufen werden sollen (in Raumpixeln)
y Die Position entlang der y-Achse, von der die Kacheldaten abgerufen werden sollen (in Raumpixeln)

Rückgabe:

Real (-1, wenn ein Fehler vorliegt)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

Der obige Code holt sich die ID für die Kachelkarte "Clouds" auf der Ebene "Tiles_Sky" und verwendet diese dann, um die Daten der Kachel an der Position (64, 128) zu holen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte wieder.
