layer_tilemap_get_id

Mit dieser Funktion können Sie den eindeutigen ID-Wert des Kachelkartenelements auf einer Ebene abrufen. Sie liefern die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt den ID-Wert zurück, der mit dem Kachelzuordnungselement auf der Ebene verknüpft ist. Beachten Sie, dass diese Funktion speziell für die Verwendung mit Kachelmaps gedacht ist, die Wenn Sie eine Tile-Map zu einer Ebene durch Code hinzufügen, der die Funktion layer_tilemap_create()hinzufügen, gibt sie die eindeutige ID für das hinzugefügte Tilemap-Element zurück. Wenn die angegebene Tilemap-ID falsch ist oder die Tilemap nicht existiert, gibt die Funktion -1 zurück.

Syntax:

layer_tilemap_get_id(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)

Rückgabe:

Kachelkartenelement-ID-Wert (Real) oder -1

Beispiel:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "tilemap_trees" und verwendet diese dann, um die ID des Kachelkartenelements auf dieser Ebene zu ermitteln. Diese ID wird dann zum Entfernen der Kachelkarte aus der Ebene verwendet.
