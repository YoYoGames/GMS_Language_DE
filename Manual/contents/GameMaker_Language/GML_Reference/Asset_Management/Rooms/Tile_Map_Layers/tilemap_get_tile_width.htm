tilemap_get_tile_width

Mit dieser Funktion können Sie die Breite (in Pixel) jeder Kachelzelle des Tile-Map-Elements abrufen. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt die Kachelzellenbreite zurück.

Syntax:

tilemap_get_tile_width(tilemap_element_id);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die Kachelzellenbreite zu erhalten von

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Der obige Code verwendet die abgerufene Kachel-Map-ID, um die Kachelzellenbreite und -höhe zu erhalten und verwendet sie, um zwei globale Variablen zu setzen.
