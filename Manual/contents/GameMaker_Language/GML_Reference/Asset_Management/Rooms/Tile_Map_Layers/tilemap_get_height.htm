tilemap_get_height

Mit dieser Funktion können Sie die Höhe (in Zellen) des Tile-Map-Elements abrufen. Sie geben die ID des Kachel-Map-Elements an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt die Höhe (Anzahl der Zellen in der Breite) zurück.

Syntax:

tilemap_get_height(tilemap_element_id);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, dessen Höhe ermittelt werden soll

Rückgabe:

Integer

Beispiel:

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

Der obige Code verwendet die abgerufene Kachel-Map-ID, um die Kachelbreite und -höhe der Kachel-Map zu ermitteln, und verwendet diese Werte dann zum Setzen einer Instanzvariablen.
