tile_get_empty

Mit dieser Funktion kann geprüft werden, ob ein gegebener Satz von Kacheldaten zu einer leeren Kachel gehört oder nicht. Sie geben die Kacheldaten an, die mit der Funktion tilemap_get()abgerufen werden können, und die Funktion gibt true zurück, wenn die Kachel leer ist, oder false, wenn es einen Kachelindex gibt.

Syntax:

tile_get_empty(tiledata)

Argument Beschreibung
Kacheldaten die zu prüfenden Kacheldaten

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++;)
{
for (var j = 0; j < tilemap_get_height(map_id); j++;)
{
var data = tilemap_get(map_id, i, j);
if !tile_get_empty(data)
{
data = tile_set_empty(data)
tilemap_set(map_id, data, i, j);
}
}
}

Der obige Code ruft die Kachelkarten-ID aus der angegebenen Ebene ab und prüft dann jede Kachelzelle auf der Karte, um zu sehen, ob sie Daten enthält oder nicht. Wenn dies der Fall ist, wird die Kachel auf leer gesetzt.
