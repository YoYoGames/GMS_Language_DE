tile_set_mirror

Diese Funktion kann verwendet werden, um einen gegebenen Satz von Kachel-Daten so zu setzen, dass die Kachel gespiegelt wird oder nicht. Sie geben die Kachel-Daten an, die mit der Funktion tilemap_set() abgerufen werden können. tilemap_get()abgerufen werden können, und setzen dann das Argument "Spiegeln" entweder auf "true", wenn die Kachel gespiegelt werden soll, oder auf "false", wenn die Kachel in ihrem standardmäßigen, nicht gespiegelten Zustand bleiben soll. Die Funktion gibt einen geänderten Kacheldatensatz zurück, der dann mit der Funktion tilemap_set() Funktion.

Syntax:

tile_set_mirror(tiledata, mirror)

Argument Beschreibung
Kacheldaten die zu setzenden Kacheldaten
Spiegel Auf true setzen, um zu spiegeln, und false, um es so zu lassen, wie es ist

Rückgabe:

Real (Kacheldaten)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

Der obige Code holt sich die Kachel-Map-ID aus der angegebenen Ebene und ermittelt dann die x- und y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die Kacheldaten zu erhalten, die gespiegelt werden und dann verwendet werden, um die Kachel wieder zu setzen.
