tilemap_get

Mit dieser Funktion können Sie die Kacheldaten aus einer Zelle des Kachelmappenelements abrufen. Sie geben die ID des Kachel-Map-Elements an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), sowie die x- und y-Position der Zelle an und die Funktion gibt den Kacheldaten-"Blob" zurück. Diese Daten sind im Wesentlichen eine Bitmaske, die Folgendes enthält den Kachelindex, die Booleschen Werte für Spiegeln/Drehen/Spiegeln und alle angewendeten Maskendaten (siehe tilemap_set_mask() für Details), und der resultierende Datenwert kann dann in den tile_* Funktionen verwendet werden verwendet werden, um die Eigenschaften einer Kachel zu ändern. Beachten Sie, dass wir die x/y-Zellenpositionen auf der Grundlage der Größe des Kachelplanrasters und nicht als Position im Raum verwenden, daher ist "cell_x" ein Wert von 0 bis zur Kachelplanbreite, und "cell_y" ist ein Wert von 0 bis zur Kachelplanhöhe. Wenn Sie die Daten für eine Kachel an einer bestimmten Raumposition benötigen, müssen Sie die Funktion tilemap_get_at_pixel().

WICHTIG! Wenn die Kacheln in der Tile-Map unverändert geblieben sind (d. h.: sie sind nicht gedreht oder gespiegelt usw...), dann ist der Rückgabewert der Kachelsatzdaten genau gleich dem Index der Kachel auf dem Kachelsatz. Sie können also "Kollisions Maps" von Kacheln erstellen, indem Sie z. B. eine Kachel mit dem Index 1 im Kachelsatz verwenden - und dann diese Funktion verwenden, um auf 1 oder 0 (eine leere Kachel) zu prüfen, um Kollisionen zu berechnen.

Syntax:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die Kacheldaten von
x_Zelle Die Zelle entlang der x-Achse, aus der die Kacheldaten geholt werden sollen
y_Zelle Die Zelle entlang der y-Achse, aus der die Kacheldaten geholt werden sollen

Rückgabe:

Real (-1, wenn ein Fehler vorliegt)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Der obige Code ruft die ID für die Kachelübersicht auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel in Zelle (0, 0) abzurufen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelkarte erneut zu setzen.
