tilemap_set_at_pixel

Mit dieser Funktion kann eine Zelle innerhalb des Tile-Map-Elements auf der Ebene auf eine neue Kachel gesetzt werden, wobei die aktuelle Position der Kachel im Raum verändert wird. Sie geben die ID des Kachel-Map-Elements (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und dann die zu setzenden Kacheldaten sowie die Position innerhalb des Raums. Sie können Kacheldaten mit der Funktion tilemap_get_at_pixel() abrufen und sie dann mit den tile_-Funktionen ändern, bevor Sie sie mit dieser Funktion erneut setzen. Die Funktion gibt true zurück, wenn die Kachel erfolgreich gesetzt wurde, und false, wenn es ein Problem gab und sie nicht gesetzt wurde.

Syntax:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
Kacheldaten Die zu setzenden Kacheldaten
xcell Die x-Position (im Raum)
ycell Die y-Position (im Raum)

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

Der obige Code ruft die ID für die Kachelübersicht auf der Ebene "Tiles_Sky" ab und verwendet diese dann, um die Daten der Kachel an der Position (4, 4) abzurufen. Diese Kacheldaten werden dann gespiegelt, bevor sie verwendet werden, um die Kachel auf der Kachelübersicht erneut zu setzen.
