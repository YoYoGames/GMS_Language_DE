tilemap_y

Diese Funktion steuert die Position entlang der y-Achse des Raums des Asset-Kachel-Map-Elements auf der Ebene. Sie geben die ID des Kachel-Map-Elements an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und legen dann den zu verwendenden y-Wert fest (basierend auf den Raumkoordinaten).

Syntax:

tilemap_y(tilemap_element_id, y);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
y Die y-Position für die Kachelkarte

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Der obige Code ruft den ID-Wert des Kachelkarten-Assets ab, das der Ebene "Asset_sky" zugewiesen ist, und setzt dann seine y-Position auf einen Zufallswert zwischen 0 und der Höhe des Raums.
