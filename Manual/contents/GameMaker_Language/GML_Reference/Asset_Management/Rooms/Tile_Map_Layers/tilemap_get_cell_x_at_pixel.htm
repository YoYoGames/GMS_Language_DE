tilemap_get_cell_x_at_pixel

Mit dieser Funktion können Sie die X-Achsen-Position einer einzelnen Kachelplan-Zelle abrufen, indem Sie die relative X-Achsen-Position innerhalb des Raums angeben. Sie geben die Kachelplan-Element-ID an (die Sie erhalten, wenn Sie ein Kachelplan-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()) sowie die x- und y-Position innerhalb des Raums an und die Funktion gibt die x-Position der Zelle innerhalb der Tile Map für diesen Punkt zurück. Beachten Sie, dass die Funktion -1 zurückgibt, wenn der Wert außerhalb des Tilemap-Bereichs liegt und keine Zelle verfügbar ist.

Syntax:

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die x-Position der Zelle zu erhalten
x Die x-Position innerhalb des Raums, die für das Erhalten der Zelle verwendet werden soll
y Die y-Position innerhalb des Raums, die für das Abrufen der Zelle verwendet werden soll

Rückgabe:

Integer (Position der x-Achsen-Zelle oder -1, wenn ein Fehler vorliegt)

Beispiel:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Der obige Code verwendet die abgerufene Kachel-Map-ID, um die Zellenposition aus einem Satz von Raumkoordinaten zu ermitteln und dann die Daten für jede dort gefundene Kachel in einer Instanzvariablen zu speichern.
