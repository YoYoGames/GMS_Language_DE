tilemap_get_mask

Mit dieser Funktion können Sie den Bitmaskenwert für die angegebene Tilemap abrufen. Sie geben die ID des Tilemap-Elements an (die Sie erhalten, wenn Sie ein Tilemap-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und die Funktion gibt den aktuellen Maskenwert zurück oder -1, wenn ein Fehler vorliegt, oder 0, wenn keine Maske angegeben ist. Für weitere Informationen über Kachel-Map-Bitmasken siehe die Funktion tilemap_set_mask().

Syntax:

tilemap_get_mask(tilemap_element_id);

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tile-Map-Elements, um die Maske von

Rückgabe:

Real (0 für keine Maske, -1 für einen Fehler)

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_mask(map_id, new_mask);
}

Der obige Code holt sich die Tile-Map-ID aus der angegebenen Ebene und überprüft dann den damit verbundenen Maskenwert. Wenn er nicht mit dem in der Variablen "new_mask" definierten Wert übereinstimmt, wird er auf diesen Wert gesetzt.
