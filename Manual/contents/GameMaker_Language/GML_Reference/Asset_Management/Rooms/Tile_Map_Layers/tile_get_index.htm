tile_get_index

Diese Funktion kann verwendet werden, um den Kachelindex (die Position der Kachel innerhalb des Kachelsatzbildes) aus einem Satz von Kacheldaten zu erhalten. Sie geben die Kacheldaten, die mit der Funktion tilemap_get()abgerufen werden, und die Funktion gibt einen ganzzahligen Wert für den Index zurück oder -1, wenn ein Fehler vorliegt.

Syntax:

tile_get_index(tiledata)

Argument Beschreibung
Kacheldaten die zu prüfenden Kacheldaten

Rückgabe:

Integer

Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

Der obige Code holt sich die Kachel-Map-ID aus der angegebenen Ebene und verwendet diese dann, um die Kacheldaten für die Zelle unter der Mausposition zu erhalten. Diese Daten werden dann verwendet, um den Kachelindex auf eine Zufallszahl zu setzen.
