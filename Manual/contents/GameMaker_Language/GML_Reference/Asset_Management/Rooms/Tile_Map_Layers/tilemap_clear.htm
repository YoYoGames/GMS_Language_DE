tilemap_clear

Mit dieser Funktion können Sie alle Kacheln auf einer gegebenen Tile-Map löschen/einstellen. Sie geben die ID des Kachel-Map-Elements an (die Sie erhalten, wenn Sie ein Kachel-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion die Funktion layer_tilemap_get_id()), und geben dann die Kacheldaten an, mit denen Sie die Ebene löschen möchten. Der Standardwert 0 löscht alle Kacheln aus der Ebene (wodurch alle Kacheln im Wesentlichen "leer" werden), Sie können aber auch die speziellen Funktionen von tile_* verwenden, um Ihre eigenen Kacheldaten zu erstellen, mit denen Sie die Kachelübersicht löschen möchten.

Syntax:

tilemap_clear(tilemap_element_id, tiledata)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
Kacheldaten Die Kacheldaten, die zum Löschen der Ebene verwendet werden sollen

Rückgabe:

N/A

Beispiel:

var layer_id = layer_get_id("Wald");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

Der obige Code ruft den ID-Wert einer im Raum-Editor erstellten Kachel-Map ab und löscht sie dann mit "leeren" Kacheln.
