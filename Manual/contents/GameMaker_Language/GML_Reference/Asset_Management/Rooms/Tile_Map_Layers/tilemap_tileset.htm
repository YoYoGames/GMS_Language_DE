tilemap_tileset

Mit dieser Funktion können Sie die einem bestimmten Tile-Map-Element zugewiesene Tile-Set-Ressource auf einer Ebene ändern. Sie geben die ID des Tile-Map-Elements an (die Sie erhalten, wenn Sie ein Tile-Map-Element mit layer_tilemap_create() oder wenn Sie die Funktion layer_tilemap_get_id()), und geben dann einen Tile-Set-Index an, und die Tile Map erhält das neue Sprite.

Syntax:

tilemap_tileset(tilemap_element_id, tileset_index)

Argument Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Tile-Map-Elements
kachelsatz_index Der zu verwendende neue Kachelsatzindex

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
{
tilemap_tileset(tile_id, ts_Nighttime);
}

Der obige Code prüft den aktuellen Kachelsatz, der der Kachelübersicht auf der Ebene "Tiles_trees" zugewiesen ist, und wenn es nicht "ts_Nighttime" ist, wird dieser Kachelsatz der Kachelübersicht zugewiesen.
