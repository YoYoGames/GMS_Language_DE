Raum_Höhe

Diese Variable enthält die Höhe des aktuellen Raums in Pixel. Sie können diese Variable jederzeit ändern, um die Höhe des Raums zu ändern. Änderungen werden auf den unteren Teil des Raums angewendet, da der Ursprung als die linke obere Ecke betrachtet wird. Beispiel, Wenn der Raum z. B. 480px hoch ist und Sie ihn auf 640px einstellen, wird der Raum nach unten erweitert und am unteren Rand um 180px vergrößert.

Syntax:

Raum_Höhe;

Rückgabe:

Real

Beispiel:

if bbox_bottom > room_height
{
y += room_height - bbox_bottom;
}

Der obige Code prüft, ob die Sprite Bounding Box der aktuellen Instanz größer ist als die Höhe des Raums, und wenn dies der Fall ist, wird die Instanz nach oben verschoben.
