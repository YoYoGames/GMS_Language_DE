room_get_viewport

Mit dieser Funktion können Sie die Details eines View Ports in einem anderen als dem aktuellen Raum abrufen. Sie geben die Raum-ID und den Index des abzurufenden View Ports an (von 0 bis 7) und die Funktion gibt ein Array mit 5 Indizes zurück, wobei:

  • [0] = sichtbar (Boolean: true / false)
  • [1] = x-Position (real)
  • [2] = y-Position (real)
  • [3] = Breite (real)
  • [4] = Höhe (real)

Syntax:

room_get_viewport(rm, vind);

Argument Beschreibung
rm Der Index des Raums, von dem Ansichtsfensterdaten abgerufen werden sollen
vind Der Index des abzurufenden Ansichtsports

Rückgabe:

Array (5 Elemente: sichtbar, x, y, Breite, Höhe)

Beispiel:

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
{
room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

Der obige Code ruft die Daten des Ansichtsanschlusses für den angegebenen Raum ab und prüft dann, ob der Anschluss als sichtbar gekennzeichnet ist. Wenn dies nicht der Fall ist, werden die Daten des Ansichtsanschlusses so eingestellt, dass er sichtbar wird.
