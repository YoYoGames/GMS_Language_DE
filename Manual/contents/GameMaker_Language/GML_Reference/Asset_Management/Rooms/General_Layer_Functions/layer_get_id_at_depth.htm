layer_get_id_at_depth

Mit dieser Funktion können Sie die IDs aller Ebenen abrufen, denen eine bestimmte Tiefe zugeordnet ist. Sie geben die zu prüfende Tiefe an und die Funktion gibt ein Array mit 1 oder mehreren Einträgen zurück, je nachdem, ob es Ebenen in der angegebenen Tiefe gibt oder nicht. Wenn es keine Ebenen in der angegebenen Tiefe gibt, enthält das Array einen einzelnen Eintrag an der Position [0] mit dem Wert -1. Wenn es jedoch Ebenen in der Tiefe gibt, wird für jede gefundene Ebene ein Eintrag im Array erstellt. Der Eintragswert ist der eindeutige ID-Wert für eine Schicht.

Syntax:

layer_get_id_at_depth(depth)

Argument Beschreibung
Tiefe Die Tiefe, aus der die Ebenen-IDs geprüft und abgerufen werden sollen.

Rückgabe:

Array

Beispiel:

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
for (var i = 0; i < array_length(a); i++;)
{
layer_destroy(a[i]);
}
}

Der obige Code ruft Daten über die Ebenen mit der Tiefe 0 ab. Es wird geprüft, ob Ebenen in dieser Tiefe vorhanden sind, und wenn dies der Fall ist, wird das zurückgegebene Array geparst und jede der gefundenen Ebenen wird zerstört.
