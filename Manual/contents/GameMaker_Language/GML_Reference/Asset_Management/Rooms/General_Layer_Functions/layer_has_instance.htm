schicht_hat_instanz

Mit dieser Funktion kann geprüft werden, ob eine gegebene Instanz aktuell dem gegebenen Layer zugeordnet ist. Sie übergeben die Layer-ID (die Sie beim Erstellen des Layers mit ebene_erstellen()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Instanz-ID der Instanz, auf die geprüft werden soll. Sie können auch einen object_index (d. h. den Namen des Objekts im Asset-Browser) angeben, und die Funktion prüft, ob sich Instanzen dieses Objekts auf der angegebenen Ebene befinden. Die Funktion gibt true zurück, wenn sich die Instanz auf der Ebene befindet, und false, wenn dies nicht der Fall ist.

Syntax:

layer_has_instance(layer_id, instance_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
instance_id Die eindeutige Instanz-ID oder der Objektindex der Instanz, auf die geprüft werden soll

Rückgabe:

Boolesche

Beispiel:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
layer_destroy(global.Bullet_Layer);
}

Der obige Code prüft, ob die angegebene Ebene irgendwelche Instanzen des Objekts "obj_Bullet_Parent" enthält und wenn nicht, wird die Ebene zerstört.
