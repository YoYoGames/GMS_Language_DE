layer_destroy_instances

Mit dieser Funktion können Sie alle Instanzen zerstören, die der angegebenen Ebene zugeordnet sind. Sie übergeben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung), und dann werden alle Instanzen, die sich auf der Ebene befinden, aus dem Spiel entfernt, wobei ihre Destroy- und Clean Up-Ereignisse ausgelöst werden.

Syntax:

layer_destroy_instances(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, für die die Tiefe eingestellt werden soll (oder der Name der Ebene als String)

Rückgabe:

Real (Eindeutiger ID-Wert für eine Instanz)

Beispiel:

if global.game_over
{
layer_destroy_instances(layer);
}

Der obige Code prüft eine globale Variable und wenn sie wahr ist, werden alle Instanzen, die sich auf der Ebene der aufrufenden Instanz befinden, zerstört (einschließlich der aufrufenden Instanz).
