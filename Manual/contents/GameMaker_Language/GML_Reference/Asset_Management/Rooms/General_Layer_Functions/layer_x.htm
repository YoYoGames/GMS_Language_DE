Schicht_x

Mit dieser Funktion können Sie die x-Position der Ebene innerhalb des aktuell skopierten Raums festlegen. Sie übergeben die Ebenen-ID (die Sie beim Anlegen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Funktion verschiebt die Ebene um die angegebene Anzahl von Pixeln entlang der horizontalen Achse des Raums.

Syntax:

layer_x(layer_id, x)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, für die die x-Position festgelegt werden soll
x Die x-Position im Raum, auf die die Ebene gesetzt werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
layer_x(lay_id, 0);
layer_y(lay_id, 0);
}

Der obige Code prüft die angegebene Schichtposition und wenn sie nicht auf (0, 0) gesetzt ist, wird sie auf diese Position gesetzt.
