schicht_shader

Mit dieser Funktion können Sie einer beliebigen Ebene eine Shader-Ressource zuweisen und die Ebene wird dann mit diesem Shader gerendert. Sie geben entweder den eindeutigen ID-Wert der Ebene oder den Namen der Ebene (als String - dies hat Auswirkungen auf die Leistung) zusammen mit der ID des zu verwendenden Shaders. Der Shader muss zuvor im Asset-Browser erstellt worden sein und der Shader-Index (der Name der Shader-Ressource ) wird dann an diese Funktion übergeben. Wenn der zugewiesenen Ebene Instanzen hinzugefügt wurden, dann wird der Shader auf alle Zeichenereignisse angewendet, die die Instanz verwendet - wenn die Instanz z. B. ein "Draw GUI Begin"-Ereignis hat, wird der Shader automatisch auf dieses Ereignis angewendet. Der Shader wirkt sich auch auf alle anderen grafischen Elemente aus, die auf dieser Ebene gezeichnet werden, wie z. B. Sprite-Assets oder Tile-Maps.

Beachten Sie, dass die Funktion nicht in irgendwelchen Draw-Events oder Step-Events aufgerufen werden soll, sondern nur beim Start des Raums im Room Creation Code oder im Create Event / Room Start Event einer Instanz aufgerufen werden muss.

Syntax:

layer_shader(layer_id, shader)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
Shader Der Shader-Index, der der Ebene zugewiesen werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

Der obige Code weist der angegebenen Ebene für alle Zeichnungen die Shader-Ressource "shd_Sepia" zu.
