layer_add_instance

Diese Funktion kann verwendet werden, um eine bestimmte Instanz von der Ebene, auf der sie sich gerade befindet, auf eine andere Ebene zu verschieben. Sie geben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als eine Zeichenkette - dies hat Auswirkungen auf die Leistung) und die Instanz-ID der Instanz, die zwischen den Ebenen verschoben werden soll. Angenommen, Ihr Player befindet sich auf einer Ebene, die eine geringere Tiefe als eine andere Ebene hat, und Sie möchten, dass er hinter den Ebenen zwischen den beiden. Sie können diese Funktion verwenden, um den Player von der Vordergrundebene zur Hintergrundebene zu wechseln und ihn hinter den anderen gezeichneten Ebenen erscheinen zu lassen.

HINWEIS: Diese Funktion kann nicht verwendet werden, um eine neue Instanz zu einer Ebene hinzuzufügen. Sie müssen die Instanz zuvor erstellt und ihre ID in einer Variablen gespeichert haben.

HINWEIS: Diese Funktion ist nur innerhalb des Bereichs des aktuellen Raums gültig und kann nicht verwendet werden, wenn der Bereich mit der Funktion layer_set_target_room() geändert wurde.

Syntax:

layer_add_instance(layer_id, instance_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Instanzschicht (oder der Schichtname als String)
instance_id Der eindeutige Instanz-ID-Wert der Instanz, die in die Zielebene verschoben werden soll

Rückgabe:

N/A

Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Der obige Code ermittelt zunächst den Index der nächstgelegenen Instanz zur angegebenen x/y-Position und speichert ihn in einer lokalen Variablen. Dann wird die eindeutige Instanz-Layer-ID für den Layer mit dem Namen "Instances Front" ermittelt und die gefundene Instanz auf diese Ebene.
