layer_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert für eine bestimmte Ebene zu erhalten. In der IDE haben alle Ebenen einen Namen und einen Typ, und um sie durch Code bearbeiten oder ändern zu können, müssen Sie den ID-Wert der Ebene angeben. Diese Funktion dient zum Abrufen Diese Funktion dient zum Abrufen dieser ID unter Verwendung des Namens (einer Zeichenkette) der Ebene (wie in der IDE geschrieben). Wenn Sie eine neue Ebene durch Code mit der Funktion layer_create() erstellen, wird diese Funktion stattdessen den eindeutigen ID-Wert zurückgeben (dynamisch erstellte Ebenen erhalten keine Namen). Beachten Sie, dass Sie einen Fehler erhalten und das Projekt abstürzt, wenn Sie den Namen einer Ebene angeben, die im aktuellen Raum nicht vorhanden ist.

Syntax:

layer_get_id(layer_name)

Argument Beschreibung
layer_name Der Name der Ebene (eine Zeichenkette)

Rückgabe:

Ebenen-ID Wert (Real), oder -1, wenn die angegebene Ebene nicht existiert

Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Der obige Code ermittelt zunächst den Index der nächstgelegenen Instanz zur angegebenen x/y-Position und speichert ihn in einer lokalen Variablen. Dann wird die eindeutige Instanz-Layer-ID für den Layer mit dem Namen "Instances Front" ermittelt und die gefundene Instanz auf diese Ebene.
