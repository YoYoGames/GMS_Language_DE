layer_get_element_type

Sie können diese Funktion verwenden, um den Elementtyp für das angegebene Element zu erhalten. Sie geben den eindeutigen Element-ID-Wert an (z. B. wie er von der Funktion, die das Element erstellt hat, oder vom Raum-Editor zurückgegeben wurde) und die Funktion gibt eine der folgenden Konstanten zurück (oder -1, wenn das Element nicht existiert oder der ID-Wert fehlerhaft ist):

Konstant Beschreibung
layerelementtyp_hintergrund Das Element ist ein Hintergrund.
layerelementtyp_instanz Das Element ist eine Instanz.
layerelementtype_sprite Das Element ist ein Sprite-Asset.
layerelementtyp_tilemap Das Element ist eine Tilemap.
layerelementtyp_partikelsystem Das Element ist ein Partikelsystem.
layerelementtyp_kachel Das Element ist eine Legacy-Hintergrundkachel (dies gilt nur für Projekte, die aus früheren Versionen von GameMaker).
layerelementtyp_sequenz Das Element ist ein Sequenz-Asset.

Beachten Sie, dass diese Funktion am nützlichsten ist, wenn Sie mehrere verschiedene Elementtypen einer Ebene zugewiesen haben, und dass Sie eine Liste aller Elemente auf einer bestimmten Ebene mit der Funktion layer_get_all_elements().

Syntax:

layer_get_element_type(element_id)

Argument Beschreibung
Ebene Der eindeutige ID-Wert des Elements, um den Typ von

Rückgabe:

Konstante (siehe Tabelle, oben) oder -1, wenn Element nicht existiert oder ungültig ist

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

Der obige Code ruft die IDs für alle Instanzelemente ab, die der Ebene der Instanz zugewiesen sind, die den Code ausführt. Der Code prüft dann, ob eines der zurückgegebenen Elemente Sprite-Assets sind, und wenn ja, werden sie zerstört.
