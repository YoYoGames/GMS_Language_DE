layer_instance_get_instance

Diese Funktion kann verwendet werden, um die eindeutige Instanz-ID des angegebenen Instanzelements zu erhalten. Sie geben die ID des Instanzelements an (siehe das Code-Beispiel unten), und die Funktion gibt einen realen Wert zurück, der die eindeutige Instanz-ID für das Element darstellt. Wenn das Element keine Instanz ist, gibt die Funktion -1 zurück.

Syntax:

layer_instance_get_instance(element_id)

Argument Beschreibung
element_id Der eindeutige ID-Wert des Instanzelements, um die ID von

Rückgabe:

Real (Eindeutiger ID-Wert für eine Instanz)

Beispiel:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
if (layer_get_element_type(elements[i]) == layerelementtype_instance)
{
var layerelement = elements[i];
var inst = layer_instance_get_instance(layerelement);
inst.x = inst.x + 10;
}
}

Der obige Code überprüft alle Instanz-Elemente auf einer Ebene, ermittelt dann ihren eindeutigen ID-Wert und verwendet diesen, um sie um 10px nach rechts zu verschieben.
