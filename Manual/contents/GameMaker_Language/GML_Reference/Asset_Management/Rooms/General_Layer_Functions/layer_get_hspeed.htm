layer_get_hspeed

Mit dieser Funktion können Sie die horizontale Geschwindigkeit (in Pixeln pro Spielframe) der Ebene innerhalb des aktuell bearbeiteten Raums abfragen. Sie übergeben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Performance) und die Funktion gibt eine reelle Zahl für die horizontale Geschwindigkeit zurück, wobei ein positiver Wert nach rechts und ein negativer Wert nach links zeigt. Die Voreinstellung ist 0 (sofern nicht im Raumeditor eingestellt).

Syntax:

layer_get_hspeed(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, von der die horizontale Geschwindigkeit abgerufen werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

Der obige Code prüft die angegebenen horizontalen und vertikalen Geschwindigkeiten der Schicht und setzt sie auf 0, wenn sie nicht beide auf 0 gesetzt sind.
