layer_depth

Diese Funktion kann verwendet werden, um die Tiefe der angegebenen Ebene zu ändern und damit die Reihenfolge zu ändern, in der ihr Inhalt auf dem Bildschirm gerendert wird. Sie geben die ID der Ebene an (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und geben dann die neue Tiefe an, auf die sie gesetzt werden soll (ein ganzzahliger Wert von -16000 bis 16000). Die Ebenentiefe wird so definiert, dass sie höher ist, wenn sie "weiter weg" von der Kamera ist, und niedriger Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (d. h. die Ebene, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 ist.

Beachten Sie, dass Sie die Tiefe einer Ebene jederzeit mit der Funktion layer_get_depth(). Beachten Sie auch, dass die minimale und maximale Ebenentiefe -16000 bis 16000 beträgt und alles, was außerhalb dieser Tiefen liegt , nicht gerendert wird. Wenn Sie eine Tiefe außerhalb dieses Bereichs benötigen, müssen Sie die Funktion layer_force_draw_depth().

Syntax:

layer_depth(layer_id, depth)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, für die die Tiefe eingestellt werden soll (oder der Name der Ebene als String)
Tiefe Die neue Tiefe für die Ebene

Rückgabe:

N/A

Beispiel:

if layer_get_depth(layer) != -100
{
layer_depth(layer, -100);
}

Der obige Code ermittelt die Tiefe der Ebene, auf der sich die Instanz befindet, die den Code ausführt, und wenn diese nicht -100 ist, wird die Tiefe auf -100 gesetzt.
