layer_get_target_room

Mit dieser Funktion setzen Sie das Ebenenziel auf den aktuellen Raum zurück. Siehe die Funktion layer_set_target_room() für weitere Informationen.

Syntax:

layer_reset_target_room()

Rückgabe:

N/A

Beispiel:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees);
}
layer_reset_target_room();

Der obige Code setzt den Zielraum auf den Raum "rm_Game" und ruft dann die Ebenen-ID für die Ebene namens "SpriteAssets" in diesem Raum ab. Diese Ebenen-ID wird dann verwendet, um der Ebene 50 zufällige Sprite-Assets hinzuzufügen, bevor das Ziel der Ebene auf den aktuellen Raum zurückgesetzt wird.
