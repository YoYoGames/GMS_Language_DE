Schicht_Geschwindigkeit

Mit dieser Funktion können Sie die horizontale Geschwindigkeit (in Pixel pro Spielframe) der Ebene innerhalb des aktuell bearbeiteten Raums einstellen. Sie geben die Ebenen-ID an (die Sie erhalten, wenn Sie die Ebene mit ebene_erzeugen()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Performance) und den zu setzenden Geschwindigkeitswert, wobei ein positiver Wert nach rechts und ein negativer Wert nach links zeigt.

Syntax:

layer_hspeed(layer_id, hspd)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene zum Einstellen der horizontalen Geschwindigkeit von
hspd Die einzustellende horizontale Geschwindigkeit (in Pixel pro Spielframe)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

Der obige Code prüft die angegebenen horizontalen und vertikalen Geschwindigkeiten der Schicht und setzt sie auf 0, wenn sie nicht beide auf 0 gesetzt sind.
