layer_get_element_layer

Mit dieser Funktion können Sie die Layer-ID ermitteln, auf der sich das angegebene Element befindet. Sie geben den eindeutigen Element-ID-Wert an (z. B. wie er von der Funktion, die das Element erstellt hat, oder vom Raum-Editor zurückgegeben wird) und die Funktion gibt die eindeutige ID der Ebene zurück, auf der sich das Element befindet. Wenn die angegebene Element-ID nicht gültig ist, gibt die Funktion -1 zurück.

Syntax:

layer_get_element_layer(element_id)

Argument Beschreibung
element_id Der eindeutige ID-Wert des Elements, um die Ebene von

Rückgabe:

Ebenen-ID oder -1, wenn das Element ungültig ist (d. h.: nicht existiert)

Beispiel:

element = layer_get_element_layer(asset_1);

Der obige Code ruft die Ebenen-ID für das Element mit der in der Variablen "asset_1" gespeicherten ID ab und speichert sie in einer Variablen.
