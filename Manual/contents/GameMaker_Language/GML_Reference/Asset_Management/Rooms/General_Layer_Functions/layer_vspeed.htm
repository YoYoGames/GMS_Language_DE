Schicht_Geschwindigkeit

Mit dieser Funktion können Sie die vertikale Geschwindigkeit (in Pixeln pro Spielframe) der Ebene innerhalb des aktuell bearbeiteten Raums einstellen. Sie übergeben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und den zu setzenden Geschwindigkeitswert, wobei ein positiver Wert nach unten und ein negativer Wert nach oben zeigt.

Syntax:

layer_vspeed(layer_id, vspd)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene zum Einstellen der vertikalen Geschwindigkeit von
vspd Die einzustellende vertikale Geschwindigkeit (in Pixel pro Spielframe)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

Der obige Code prüft die angegebenen horizontalen und vertikalen Geschwindigkeiten der Schicht und setzt sie auf 0, wenn sie nicht beide auf 0 gesetzt sind.
