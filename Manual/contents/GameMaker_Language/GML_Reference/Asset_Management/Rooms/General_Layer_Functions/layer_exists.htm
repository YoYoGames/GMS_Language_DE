layer_exists

Mit dieser Funktion können Sie prüfen, ob die angegebene Ebene existiert. Sie übergeben die Ebenen-ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Performance Dies hat Auswirkungen auf die Leistung) und die Funktion gibt den booleschen Wert true zurück, wenn die Ebene existiert, oder false, wenn sie nicht existiert.

HINWEIS: Diese Funktion arbeitet im Rahmen des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird - der mit der Funktion layer_set_target_room().

Syntax:

layer_exists(layer_name)

Argument Beschreibung
layer_name Der Name der Ebene (eine Zeichenkette oder ein ID-Wert)

Rückgabe:

Boolesche

Beispiel:

if !layer_exists(global.tileLayer)
{
global.tileLayer = layer_create(1000);
}

Der obige Code prüft, ob die in der globalen Variable gespeicherte Ebene tatsächlich existiert, und wenn nicht, wird sie erstellt.
