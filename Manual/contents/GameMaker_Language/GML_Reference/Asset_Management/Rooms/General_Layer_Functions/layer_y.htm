Schicht_y

Mit dieser Funktion können Sie die y-Position der Ebene innerhalb des aktuell bearbeiteten Raums festlegen. Sie übergeben die Ebenen-ID (die Sie beim Anlegen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Performance) und die Funktion verschiebt die Ebene um die angegebene Anzahl von Pixeln entlang der vertikalen Achse des Raums.

Syntax:

layer_y(layer_id, y)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, für die die y-Position festgelegt werden soll
y Die y-Position im Raum, auf die die Ebene gesetzt werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
layer_x(lay_id, 0);
layer_y(lay_id, 0);
}

Der obige Code prüft die angegebene Schichtposition und wenn sie nicht auf (0, 0) gesetzt ist, wird sie auf diese Position gesetzt.
