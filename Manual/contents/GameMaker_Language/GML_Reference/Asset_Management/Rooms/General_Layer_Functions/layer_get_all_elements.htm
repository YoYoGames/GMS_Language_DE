layer_get_all_elements

Mit dieser Funktion können Sie die Element-IDs der angegebenen Ebene ermitteln. Sie übergeben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies hat Die Funktion gibt ein Array von IDs zurück, wobei jeder Eintrag im Array eine eindeutige ID für ein Element auf dieser Ebene ist. Wenn die Ebene beispielsweise eine Vermögensebene ist, wird das Array mit den ID-Werten für jedes Sprite-Asset gefüllt, das der Ebene zugewiesen ist. Beachten Sie, dass die Verwendung von Code für die Arbeit mit Ebenen bedeutet, dass Sie der gleichen Ebene verschiedene Elementtypen zuweisen können - Sie können also Sprite-Assets zusammen mit Instanzen haben, zum Beispiel - in diesem Fall können Sie dann die Funktion layer_get_element_type() verwenden, um den Typ des Elements zu ermitteln, auf den sich die ID bezieht.

Syntax:

layer_get_all_elements(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, von der die Elemente abgerufen werden sollen (oder der Ebenenname als String)

Rückgabe:

Array (1D, bestückt mit Element-IDs)

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

Der obige Code ruft die IDs für alle Instanzelemente ab, die der Ebene der Instanz zugewiesen sind, die den Code ausführt. Der Code prüft dann, ob eines der zurückgegebenen Elemente Sprite-Assets sind, und wenn ja, werden sie zerstört.
