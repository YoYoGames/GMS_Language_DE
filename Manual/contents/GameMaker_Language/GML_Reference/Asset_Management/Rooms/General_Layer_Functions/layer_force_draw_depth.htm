layer_force_draw_depth

Diese Funktion erzwingt, dass alle Ebenen mit der angegebenen z-Tiefe gezeichnet werden. Dies ändert nicht die Reihenfolge, in der die Ebenen gerendert werden und sie werden immer noch in der Tiefenreihenfolge gezeichnet, es wird lediglich geändert, welcher z-Wert verwendet wird. Im Allgemeinen müssen Sie sich nicht Wenn Sie jedoch Ebenen haben, deren Tiefe außerhalb des zulässigen Bereichs (-16000 bis 16000) liegt, werden diese nicht gerendert, so dass Sie die Z-Tiefe auf einen vernünftigen Wert setzen können - z. B. 0 - und alle Ebenen werden problemlos gerendert. Beachten Sie, dass dies im Allgemeinen nur für die Verwendung mit älteren Projekten aus früheren Versionen von GameMaker verwendet werden, bei denen die Zeichnungstiefen höher oder niedriger als der zulässige Ebenenbereich sein können.

Syntax:

layer_force_draw_depth(force, depth)

Argument Beschreibung
Kraft Ob das Z-Tiefen-Forcing aktiviert (auf true gesetzt) oder deaktiviert (auf false gesetzt) werden soll
Tiefe Die neue Z-Tiefe

Rückgabe:

N/A

Beispiel:

if !layer_is_draw_depth_forced()
{
layer_force_draw_depth(true, 0);
}

Der obige Code prüft, ob die Z-Tiefe der Ebene erzwungen ist oder nicht, und wenn nicht, setzt er die Z-Tiefe auf 0 und aktiviert sie.
