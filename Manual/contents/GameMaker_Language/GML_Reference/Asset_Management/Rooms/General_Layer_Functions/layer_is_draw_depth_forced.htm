layer_is_draw_depth_forced

Mit dieser Funktion können Sie prüfen, ob die erzwungene Z-Tiefe für das Rendern der Ebenen im Projekt aktiviert ist. Siehe layer_force_draw_depth() für weitere Informationen.

Syntax:

layer_is_draw_depth_forced()

Rückgabe:

Boolesche

Beispiel:

if !layer_is_draw_depth_forced()
{
layer_force_draw_depth(true, 0);
}

Der obige Code prüft, ob die Z-Tiefe der Ebene erzwungen ist oder nicht, und wenn nicht, setzt er die Z-Tiefe auf 0 und aktiviert sie.
