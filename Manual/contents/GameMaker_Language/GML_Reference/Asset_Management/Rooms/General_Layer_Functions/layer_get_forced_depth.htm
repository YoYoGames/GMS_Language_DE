layer_is_draw_depth_forced

Mit dieser Funktion kann die eingestellte Z-Tiefe für das Rendern von Ebenen innerhalb des Raums abgerufen werden. Siehe layer_force_draw_depth() für weitere Informationen.

Syntax:

layer_get_forced_depth()

Rückgabe:

Real

Beispiel:

if layer_get_forced_depth() != 0
{
layer_force_draw_depth(true, 0);
}

Der obige Code prüft, ob die Ebene Z-Tiefe auf einen Wert von 0 gezwungen wird oder nicht, und wenn dies nicht der Fall ist, setzt er die Ebene Z-Tiefe auf 0.
