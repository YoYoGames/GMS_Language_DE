layer_get_shader

Mit dieser Funktion können Sie prüfen, ob der angegebenen Ebene ein Shader zugewiesen ist. Sie übergeben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies wird Die Funktion gibt entweder den Shader-Index des zugewiesenen Shaders zurück oder -1, wenn kein Shader zugewiesen ist.

Syntax:

layer_get_shader(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)

Rückgabe:

Shader-Index (oder -1, wenn kein Shader zugewiesen ist)

Beispiel:

if layer_get_shader(layer) == -1
{
layer_shader(layer, shd_Sepia);
}

Der obige Code prüft, ob der Ebene, auf der die Instanz den Code ausführt, ein Shader zugewiesen ist, und wenn nicht, wird einer zugewiesen.
