layer_get_script_end

Diese Funktion gibt den Skriptfunktionsindex der Funktion zurück, die am Ende des Renderns der angegebenen Ebene ausgeführt werden soll, oder sie gibt -1 zurück, wenn keine Funktion zugewiesen ist. Sie übergeben die Layer ID (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung). Sie können einer Ebene Skriptfunktionen zuweisen mit layer_script_begin() und layer_script_end() ZUWEISEN.

Syntax:

layer_get_script_end(layer_id);

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)

Rückgabe:

Skriptfunktionsindex (Real) oder -1, wenn kein Skript zugewiesen ist

Beispiel:

if layer_get_script_end(layer) == -1
{
layer_script_end(layer, scr_ResetShaderValues);
}

Der obige Code prüft, ob der Ebene, auf der die Instanz den Code ausführt, eine Skriptfunktion zugewiesen ist, und wenn nicht, wird eine zugewiesen.
