layer_get_name

Sie können diese Funktion verwenden, um den Namen der angegebenen Ebene zu erhalten. Sie geben den eindeutigen Wert der Ebenen-ID an und wenn die Ebene eine der benannten Ebenen ist, die im Raum-Editor erstellt wurden, dann gibt die Funktion eine Zeichenkette mit dem Namen der Ebene zurück. Wenn die Ebene keine der im Raum-Editor angelegten Ebenen ist (d. h.: sie wurde mit layer_create()), wird eine leere Zeichenkette zurückgegeben.

Syntax:

layer_get_name(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, deren Name ermittelt werden soll

Rückgabe:

Zeichenfolge

Beispiel:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
if layer_get_name(a[i]) != ""
{
ds_list_add(layer_list, a[i])
}
}

Der obige Code ruft die IDs für alle Ebenen im Raum ab und durchläuft dann eine Schleife, um zu prüfen, ob es sich um benannte Ebenen handelt. Wenn dies der Fall ist, wird ihre ID zu einer Liste hinzugefügt.
