layer_set_visible

Mit dieser Funktion können Sie die Sichtbarkeit eines Layers umschalten. Sie übergeben die Ebenen-ID (die Sie beim Erstellen der Ebene mit layer_create()) oder den Ebenennamen (als String - das hat Auswirkungen auf die Performance auswirken) sowie den Umschaltwert für die Ebene, wobei visible für true und invisible für false steht.

Wenn eine Ebene unsichtbar ist, wird nichts, was der Ebene zugewiesen ist, gezeichnet, und wenn der Ebene Instanzen zugewiesen sind, werden diese nicht einmal ihr Zeichnungsereignis ausführen (ähnlich wie wenn Sie die Instanzvariable visible auf false setzen). Beachten Sie, dass Instanzen, die bereits als unsichtbar gekennzeichnet sind, nicht als sichtbar gekennzeichnet werden, wenn die Ebene, auf der sie sich befinden, auf sichtbar gesetzt ist, da es sich um zwei unabhängige Einstellungen handelt, obwohl ihr Verhalten das gleiche ist.

Syntax:

layer_set_visible(layer_id, visible)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
sichtbar Ob die Ebene sichtbar sein soll(true) oder nicht(false)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
layer_set_visible(lay_id, false);
}
sonst
{
layer_set_visible(lay_id, true);
}

Der obige Code ruft den ID-Wert für die Ebene mit dem Namen "Instances" im Raumeditor ab und verwendet dann die ID, um zu prüfen, ob die Ebene sichtbar ist oder nicht, wobei die Sichtbarkeit der Ebene abhängig vom zurückgegebenen Wert umgeschaltet wird.
