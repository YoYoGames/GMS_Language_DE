layer_set_target_room

Wenn Sie diese Funktion aufrufen, teilen Sie dem GameMaker Studio 2 mit, dass alle weiteren Ebenenfunktionen auf den angegebenen Raum angewendet werden sollen. Auf diese Weise können Sie prozedural Ebenen und Ebeneninhalte in einem Raum ändern oder erzeugen, der nicht der aktuellen Raum ist. Wenn Sie mit dem Hinzufügen von Ebenen oder Ebenenelementen zu einem Raum fertig sind, rufen Sie die Funktion layer_reset_target_room() auf, um das Raumziel zurückzusetzen (oder rufen Sie diese Funktion erneut mit einem Raumargument von -1). Beachten Sie, dass diese Funktion nur für Räume verwendet werden kann , die nicht der aktuelle Raum sind, und dass sie so konzipiert ist, dass Sie Schichten und Schichtelemente zu anderen Räumen als dem aktuellen Raum hinzufügen/entfernen können.

WARNUNG! Während Sie einen anderen Raum anvisieren, können Sie alle regulären Ebenenfunktionen verwenden , außer dass Sie keine Instanzen mit instance_create_layer() oder instance_create_depth()erstellen, noch ist die Ebenenfunktion layer_add_instance() verfügbar sein.

Syntax:

layer_set_target_room(room)

Argument Beschreibung
Zimmer Der Zielraum für alle weiteren Schichtfunktionen

Rückgabe:

N/A

Beispiel:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
var _x = random(room_get_width(rm_Game));
var _y = random(room_get_height(rm_Game));
layer_sprite_create(l, _x, _y, spr_Trees);
}
layer_reset_target_room();

Der obige Code setzt den Zielraum auf den Raum "rm_Game" und ruft dann die Ebenen-ID für die Ebene namens "SpriteAssets" in diesem Raum ab. Diese Ebenen-ID wird dann verwendet, um der Ebene 50 zufällige Sprite-Assets hinzuzufügen, bevor das Ziel der Ebene auf den aktuellen Raum zurückgesetzt wird.
