schicht_element_verschieben

Mit dieser Funktion können Sie ein Element von einer Schicht in eine andere verschieben. Sie geben die Element-ID an, wie sie von der Funktion zum Anlegen des Elements oder des Raum-Editors oder der Funktion layer_get_all_elements(), und dann geben Sie die Ebenen-ID (die Sie beim Anlegen der Ebene mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung).

Syntax:

layer_element_move(element_id, layer_id)

Argument Beschreibung
element_id Der eindeutige ID-Wert des zu verschiebenden Elements
layer_id Der eindeutige ID-Wert der Ebene, in die das Element verschoben werden soll (oder der Ebenenname als String)

Rückgabe:

N/A

Beispiel:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_element_move(a[i], asset_layer)
}
}

Der obige Code ruft die Elemente auf der Ebene ab, der die Instanz, die den Code ausführt, zugewiesen ist, und prüft dann, ob es sich um Sprite-Assets handelt. Wenn dies der Fall ist, werden sie auf die Ebene mit der in der Variablen "asset_layer" gespeicherten ID verschoben.
