Ebene_erstellen

Diese Funktion wird verwendet, um eine neue Schicht innerhalb des aktuellen Raums zu erstellen. Sie geben die Tiefe an, die die neue Schicht einnehmen soll, und die Funktion gibt den eindeutigen Wert der Schicht-ID zurück, der dann für den Zugriff auf die Schicht in weiteren Funktionen verwendet werden kann. Sie können auch ein optionales "Name"-Argument angeben, mit dem Sie der Ebene einen bestimmten Namen (als Zeichenkette) geben können, und anstelle der Ebenen-ID können Sie diese Namenszeichenkette für den Zugriff auf die Ebene verwenden, wobei zu beachten ist, dass die Verwendung von Namensstrings eine größere Auswirkung auf die Leistung hat als die Verwendung nur des ID-Werts. Wenn Sie den Namen der Ebene nicht angeben, wird der erstellten Ebene ein Name mit dem Format " _layer_XXX" zugewiesen, wobei "XXX" ein Hexadezimalwert ist, um der Ebene einen eindeutigen Namen zu geben.

Wenn Sie einen Tiefenwert angeben, ist dieser umso höher, je weiter die Ebene von der Kamera entfernt ist, und umso niedriger, je näher die Ebene an der Kamera liegt; wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100, so dass die "oberste" Ebene (d. h. die Ebene, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 ist.

Das folgende Bild zeigt ein Schema, wie die Tiefe für Ebenen funktioniert:

Es ist zu beachten, dass Sie zwar mehrere Ebenen einer einzelnen Tiefe zuweisen können, aber nicht garantieren können, dass die Ebenen auf derselben Tiefe in einer bestimmten Reihenfolge gezeichnet werden. Beachten Sie auch, dass die minimale und maximale Ebenentiefe -16000 bis 16000 beträgt und alles außerhalb dieser Tiefen nicht gerendert wird. Wenn Sie eine Tiefe außerhalb dieses Bereichs benötigen dann müssen Sie die Funktion layer_force_draw_depth().

Syntax:

layer_create(depth, [name])

Argument Beschreibung
Tiefe Die Tiefe, in der die Ebene erstellt werden soll.
Name Der Name, den die Ebene erhalten soll (eine Zeichenkette - optional)

Rückgabe:

Ebenen-ID (Real)

Beispiel:

global.Bullet_Layer = layer_create(-100);

Der obige Code erstellt eine neue Ebene mit einem Tiefenwert von -100 und weist ihren ID-Wert einer globalen Variablen zu.
