layer_get_depth

Mit dieser Funktion können Sie den Tiefenwert abrufen, der zu einer bestimmten Ebene gehört. Sie übergeben die Layer-ID (die Sie beim Erstellen des Layers mit layer_create()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Leistung) und die Funktion gibt die Ebenentiefe als reelle Zahl zurück. Beachten Sie, dass die Tiefe umso höher definiert ist, je weiter die Ebene von der Kamera entfernt ist, und umso niedriger, je näher sie an der Kamera liegt, wenn Sie also Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (d. h. die Ebene, die der Kameraansicht am nächsten ist und daher über alles andere gezeichnet wird) die Ebene mit der Tiefe -100 ist.

Das folgende Bild zeigt ein Schema, wie die Tiefe für Ebenen funktioniert:

Beachten Sie, dass Sie einen Fehler erhalten, wenn Sie einen ungültigen Wert für die Ebenen-ID angeben.

Syntax:

layer_get_depth(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, von der die Tiefe abgerufen werden soll (oder der Ebenenname als String)

Rückgabe:

Real

Beispiel:

if layer_get_depth(global.temp_layer) != -100
{
layer_destroy(global.temp_layer);
global.temp_layer = layer_create(-100);
}

Der obige Code prüft die Tiefe einer in einer globalen Variablen gespeicherten Ebenen-ID und wenn diese nicht -100 ist, wird die Ebene zerstört und mit der Tiefe von -100 neu erstellt.
