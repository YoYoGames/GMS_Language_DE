room_get_camera

Mit dieser Funktion können Sie die eindeutige Index-ID der Kamera abrufen, die einer bestimmten Ansicht in einem anderen als dem aktuellen Raum zugeordnet ist. Sie geben den zu verwendenden Raum und den zu verwendenden Ansichtsanschluss (von 0 bis 7) an und die Funktion gibt einen Kameraindex zurück.

Syntax:

room_get_camera(rm, vind);

Argument Beschreibung
rm Der Index des Raums, von dem die Fachkamera abgerufen werden soll
vind Der Index des Ansichtsports, um die Kamera von

Rückgabe:

Real (Kamera-Index)

Beispiel:

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
room_set_camera(rm_Game, 0, global.MainCam);
}

Der obige Code weist eine Kamera in einem neu angelegten Raum dem View Port [0] zu.
