schicht_hintergrund_erzeugen

Mit dieser Funktion können Sie einer Ebene eine Sprite-Ressource zuweisen, die als Hintergrund in Ihrem Projekt verwendet werden soll. Sie geben die Ebenen-ID an (die Sie erhalten, wenn Sie die Ebene mit layer_create()) oder den Ebenennamen (als String - dies wirkt sich auf die Leistung aus) und einen Sprite-Index (das wäre der Name des Sprites, wie er im Asset-Browser angezeigt wird), und schon wird es der Ebene hinzugefügt. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Ebenenfunktionen für Hintergründe verwendet werden kann.

Syntax:

layer_background_create(layer_id, sprite)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
Sprite Der zu verwendende Sprite-Index

Rückgabe:

Hintergrund-Element ID (Real)

Beispiel:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Der obige Code erstellt eine neue Ebene und fügt ihr dann ein neues Hintergrundelement hinzu, wobei ein Sprite als Hintergrundbild verwendet wird.
