layer_hintergrund_get_id

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert des Hintergrundelements auf einer Ebene abzurufen. Sie übergeben die Layer-ID (die Sie erhalten, wenn Sie den Layernamen zusammen mit layer_get_id()) und die Funktion gibt den ID-Wert zurück, der mit dem Hintergrundelement auf der Ebene verbunden ist. Beachten Sie, dass diese Funktion speziell für die Verwendung mit Hintergründen gedacht ist, die in der IDE hinzugefügt wurden, denn wenn Sie einen Hintergrund zu einer Ebene durch Code hinzufügen, indem Sie die Funktion layer_background_create()einen Hintergrund hinzufügen, wird die eindeutige ID für das hinzugefügte Hintergrundelement zurückgegeben.

Syntax:

layer_background_get_id(layer_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene

Rückgabe:

ID-Wert des Hintergrundelements (Real)

Beispiel:

var lay_id = layer_get_id("Hintergrund_Bäume");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Der obige Code ermittelt die Ebenen-ID für die Ebene "Background_trees" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das Element-Sprite zu ändern.
