layer_background_get_blend

Mit dieser Funktion können Sie die Mischfarbe des Hintergrundelements ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen realen Wert zurück, der die zugewiesene Farbe darstellt.

Syntax:

layer_background_get_blend(background_element_id);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Farbwert (Real)

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_blend(back_id) == c_white
{
layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

Der obige Code ermittelt die Ebenen-ID für die Ebene "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Farbe der Elementüberblendung zu prüfen, und wenn sie der Konstante c_white entspricht, dann wird die Ebenenüberblendung auf eine zufällige Farbe gesetzt.
