layer_background_yscale

Mit dieser Funktion können Sie die Skalierung entlang der y-Achse eines Hintergrundelements festlegen. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn Sie die Funktion layer_background_get_id()), und legen dann den Skalenwert fest. Der Skalenwert, den Sie angeben, ist ein Multiplikator, der verwendet wird, um die Art und Weise zu ändern, wie das Hintergrundelement angezeigt wird, Ein Wert von 0,5 würde das Element im halben Maßstab anzeigen, ein Wert von 2 im doppelten Maßstab. Beachten Sie, dass negative Werte gültig sind und das Element um die Position (0,0) "spiegeln", d. h. eine x-Skala von -1 würde das Bild seitenverkehrt anzeigen.

Syntax:

layer_background_yscale(background_element_id, yscale);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
Geschwindigkeit Der Geschwindigkeitsmultiplikator/-wert, der für das Element verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Hintergrund_Wasser" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann zum Ändern der y-Skala des Elements verwendet.
