layer_background_destroy

Diese Funktion zerstört das angegebene Hintergrundelement. Sie übergeben die Hintergrund-ID (die Sie erhalten, wenn Sie den Hintergrund mit layer_background_create() oder wenn Sie die Ebenen-ID zusammen mit mit layer_get_background_id()) und diese Funktion entfernt ihn. Beachten Sie, dass dadurch nicht die Ebene, sondern nur der Hintergrund entfernt wird, und wenn der Hintergrund einer ist, der im Raum Editor hinzugefügt wurde, dann wird der Hintergrund beim nächsten Verlassen des Raums und bei der Rückkehr wieder neu erstellt. Wenn der Raum jedoch persistent ist, wird der Hintergrund entfernt, es sei denn, die Raumpersistenz wird wieder ausgeschaltet.

Syntax:

layer_background_destroy(background_element_id)

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu zerstörenden Hintergrunds

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Hintergrund_Bäume");
var bck_id = layer_background_get_id("Wald");
if layer_background_exists(lay_id, bck_id)
{
layer_hintergrund_destroy(bck_id);
}

Der obige Code prüft die Ebene "Background_trees", um zu sehen, ob das angegebene Hintergrundelement existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
