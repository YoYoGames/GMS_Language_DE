layer_hintergrund_get_xscale

Mit dieser Funktion können Sie den aktuellen Skalierungsmultiplikatorwert des Hintergrundelements ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen realen Wert zurück, der den Skalierungsmultiplikator darstellt, der zum Zeichnen des Sprites verwendet wird. Der Standardwert ist 1.

Syntax:

layer_background_get_xscale(background_element_id);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_xscale(back_id) != -1
{
layer_background_xscale(back_id, -1);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Background_sky" und verwendet diese, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die x-Skala für das Element zu prüfen, und wenn sie nicht -1 ist, wird sie auf -1.
