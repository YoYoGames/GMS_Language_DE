layer_background_stretch

Mit dieser Funktion können Sie ein Hintergrundelement-Sprite so einstellen, dass es sich an den Raum anpasst oder bei der Auflösung 1:1 bleibt. Sie übergeben die ID des Hintergrundelements (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann das Argument stretch auf true oder false. Wenn es auf true gesetzt ist, wird das Element-Sprite gestreckt, damit es in den Wenn Sie das Argument auf true setzen, wird das Element-Sprite gestreckt, damit es in den Raum passt (entweder nach oben oder nach unten skaliert, je nach Größe des Sprites und des Raums), und wenn Sie es auf false setzen, wird das Element-Sprite in seiner ursprünglichen Auflösung angezeigt.

Syntax:

layer_background_stretch(background_element_id, stretch)

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
dehnen Der Umschalter für die Streckung, der true oder falsesein kann

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_stretch(back)
{
layer_background_stretch(back, false);
}
sonst
{
layer_background_stretch(back, true);
}
}

Der obige Code prüft, ob eine Maustaste gedrückt wurde, und wenn dies der Fall ist, wird die Dehnung des Hintergrundelementsprites umgeschaltet, das der Ebene zugewiesen ist, auf der sich die Instanz befindet, die den Code ausführt.
