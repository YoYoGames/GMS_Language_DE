Schicht_Hintergrund_Geschwindigkeit

Mit dieser Funktion können Sie die Geschwindigkeit eines animierenden Sprites einstellen, das einem Hintergrundelement zugewiesen wurde. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erzeugen() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann die Animationsgeschwindigkeit. Wenn die IDE die Animationsgeschwindigkeit für das verwendete Sprite so eingestellt hat, dass sie als Frames per Game Frame berechnet wird, dann der angegebene Geschwindigkeitswert ein Multiplikator, wobei 1 der Standardwert ist und die Einstellung auf 0,5 die Animationsgeschwindigkeit halbiert, während die Einstellung auf 2 sie verdoppelt. Wenn die IDE jedoch die Animationsgeschwindigkeit für das verwendete Sprite so eingestellt hat, dass sie als Frames Per Second (Bilder pro Sekunde) festgelegt hat, dann ist der angegebene Geschwindigkeitswert ein buchstäblicher Wert für Bilder pro Sekunde, so dass das Setzen dieses Wertes auf z. B. 5 dazu führt, dass die Animation in jeder Sekunde, in der das Spiel läuft, 5 Bilder wiederholt. Wenn dem Hintergrundelement kein Sprite zugeordnet ist oder das verwendete Sprite keine Sub-Images hat, hat diese Funktion keine Wirkung.

Syntax:

layer_background_speed(background_element_id, speed);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
Geschwindigkeit Der Geschwindigkeitsmultiplikator/-wert, der für das Element verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Hintergrund_Wasser");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Background_water" und verwendet diese, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Bildgeschwindigkeit des Elements zu ändern.
