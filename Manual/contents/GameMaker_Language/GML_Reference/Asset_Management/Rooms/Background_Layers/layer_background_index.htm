schicht_hintergrund_index

Mit dieser Funktion können Sie den Bildindex des Hintergrund-Sprites setzen, das mehrere Unterbilder hat. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann den zu verwendenden Bildindex. Wenn Sie einen Wert außerhalb des Bereichs von Unterbildern festlegen, wird der Bildindex in einer Schleife angezeigt.

Syntax:

layer_background_index(background_element_id, image_index);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
index Der Bildindex, der für den Hintergrund verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Background_trees" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um den Bildindex des Elements zu ändern.
