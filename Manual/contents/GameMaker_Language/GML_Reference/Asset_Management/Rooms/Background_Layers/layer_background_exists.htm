layer_background_exists

Mit dieser Funktion können Sie prüfen, ob ein Hintergrundelement auf einer bestimmten Ebene existiert. Sie übergeben die ID der Ebene (die Sie beim Erstellen der Ebene mit ebene_erzeugen()) oder den Ebenennamen (als String - dies hat Auswirkungen auf die Performance) und die ID des Hintergrundelements (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion die Funktion layer_background_get_id()) und die Funktion gibt entweder true zurück, wenn das Element existiert, oder false, wenn es nicht existiert.

HINWEIS: Diese Funktion arbeitet im Rahmen des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird - der mit der Funktion layer_set_target_room().

Syntax:

layer_background_exists(layer_id, background_element_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Ebene (oder der Ebenenname als String)
hintergrund_element_id Der eindeutige ID-Wert des zu prüfenden Hintergrundelements

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
layer_background_destroy(lay_id, global.TreesBackground);
}

Der obige Code prüft die Ebene "Background_trees", um zu sehen, ob das angegebene Hintergrundelement existiert, und wenn ja, dann wird es zerstört (aber nicht die Ebene).
