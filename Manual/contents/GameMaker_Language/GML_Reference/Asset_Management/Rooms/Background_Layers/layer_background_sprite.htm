layer_background_sprite

Mit dieser Funktion können Sie den Sprite-Index des Hintergrundelements setzen. Sie liefern die ID des Hintergrundelements (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn Sie die Funktion layer_background_get_id()), und geben dann einen zu verwendenden Sprite-Index an. Das Bild des Hintergrundelements wird durch das neue Sprite ersetzt. Wenn Sie einen Wert von -1 angeben, wird dem Element kein Sprite zugewiesen (es ist aber noch vorhanden und kann später wieder ein Sprite zugewiesen bekommen).

Syntax:

layer_background_sprite(background_element_id, sprite_index)

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
sprite_index Der Sprite-Index des Sprites, das für das Hintergrundelement verwendet werden soll

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das dem Element zugewiesene Sprite zu prüfen und es auf das Sprite "spr_Clouds" zu setzen gesetzt, wenn es das nicht schon ist.
