layer_background_visible

Mit dieser Funktion können Sie die Sichtbarkeit eines Hintergrundelements umschalten. Sie übergeben die ID des Hintergrundelements (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn verwenden Sie die Funktion layer_background_get_id()), und setzen dann das Argument visible auf true oder false. Wenn es auf true gesetzt ist, wird das Element angezeigt, und wenn es auf false gesetzt ist, wird das Element nicht angezeigt. Beachten Sie, dass dies von der Ebenensichtbarkeit abhängig ist. Selbst wenn das Hintergrundelement als sichtbar gekennzeichnet ist, wird es nicht gezeichnet, wenn die Ebene, auf der es sich befindet, als nicht sichtbar gekennzeichnet ist.

Syntax:

layer_background_visible(background_element_id, visible)

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
sichtbar Die Sichtbarkeitsumschaltung, die true oder falsesein kann

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_visible(back)
{
layer_background_visible(back, false);
}
sonst
{
layer_background_visible(back, true);
}
}

Der obige Code prüft, ob eine Maustaste gedrückt wurde, und wenn dies erkannt wird, schaltet er die Hintergrundsichtbarkeit des Hintergrundelements um, das der Ebene zugeordnet ist, auf der sich die Instanz befindet, die den Code ausführt.
