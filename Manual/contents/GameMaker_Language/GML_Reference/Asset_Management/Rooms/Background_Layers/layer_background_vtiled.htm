layer_background_vtiled

Mit dieser Funktion können Sie festlegen, ob das angegebene Hintergrundelement auf einer Ebene vertikal gekachelt werden soll oder nicht. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn Sie die Funktion layer_background_get_id()), und setzen dann den Wert für die Kachelung. Wenn der Wert auf true gesetzt ist, wird das Element vertikal im Raum gekachelt dargestellt, und wenn er auf false gesetzt ist, wird das Element nicht gekachelt. Diese Funktion ist nur für 2D-Projekte geeignet und funktioniert nicht korrekt, wenn eine 3D-Kameraprojektion verwendet wird.

Syntax:

layer_background_vtiled(background_element_id, tiled);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
sichtbar Die vertikale Kachelungsumschaltung, die true oder falsesein kann

Rückgabe:

N/A

Beispiel:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
layer_background_htiled(back, true);
layer_background_vtiled(back, true);
}

Der obige Code prüft das Hintergrundelement, das der Ebene zugewiesen ist, auf der sich die Instanz befindet, die den Code ausführt, und wenn es nicht in beide Richtungen gekachelt ist, wird es so eingestellt, dass es vertikal und horizontal gekachelt wird.
