layer_hintergrund_alpha

Diese Funktion steuert das Alpha (die Transparenz) des Hintergrund-Sprites. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_hintergrund_erstellen() oder wenn verwenden Sie die Funktion layer_background_get_id()), und setzen dann den zu verwendenden Alpha-Wert. Alpha kann zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) liegen, wobei der Standard-Alpha-Wert für das Hintergrund Element ist 1. Beachten Sie, dass, wenn die Ebene, der das Hintergrundelement zugewiesen wurde, nicht sichtbar ist - oder das Element selbst unsichtbar gemacht wurde - Sie mit dieser Funktion keinen Unterschied sehen werden, bis die Ebene oder das Element wieder sichtbar gemacht wurde wieder sichtbar gemacht wird.

Syntax:

layer_background_alpha(background_element_id, alpha);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
alpha Das Alpha für das Hintergrund-Sprite, von 0 bis 1 (Standard ist 1)

Rückgabe:

N/A

Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Der obige Code ruft den ID-Wert des Hintergrunds ab, der der Ebene "Background_sky" zugewiesen ist, und setzt dann dessen Alpha auf einen Zufallswert zwischen 0 und 1.
