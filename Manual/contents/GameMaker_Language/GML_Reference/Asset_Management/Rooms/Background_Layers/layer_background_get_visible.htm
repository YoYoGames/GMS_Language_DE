layer_background_get_visible

Mit dieser Funktion können Sie den sichtbaren Zustand des Hintergrundelements abfragen. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt entweder true zurück, wenn das Element gerade sichtbar ist, oder false, wenn es nicht sichtbar ist. Beachten Sie, dass dieser Rückgabewert nicht davon beeinflusst wird, ob die Ebene, auf der sich das Element befindet, sichtbar ist oder nicht.

Syntax:

layer_background_get_visible(background_element_id);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
{
layer_background_visible(back_id, false);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Background_sky" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um die Sichtbarkeit des Elements zu prüfen, und wenn es sichtbar ist, wird diese Eigenschaft auf false gesetzt.
