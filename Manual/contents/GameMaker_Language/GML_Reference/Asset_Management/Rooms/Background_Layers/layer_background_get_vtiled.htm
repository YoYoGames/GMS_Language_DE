layer_background_get_vtiled

Mit dieser Funktion können Sie herausfinden, ob das Hintergrundelement vertikal gekachelt ist oder nicht. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt entweder true zurück, wenn das Element gerade gekachelt ist, oder false, wenn es nicht gekachelt ist.

Syntax:

layer_background_get_vtiled(background_element_id);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Boolesche

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_vtiled(back_id)
{
layer_background_vtiled(back_id, true);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um zu prüfen, ob das Element über die vertikale Achse kachelt, und wenn nicht nicht, dann wird es so eingestellt, dass es dies tut.
