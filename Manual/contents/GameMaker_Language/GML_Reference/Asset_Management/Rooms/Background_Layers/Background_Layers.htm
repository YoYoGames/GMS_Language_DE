Hintergrund-Ebenen

Der GameMaker Studio 2 Raum-Editor erlaubt es Ihnen, Hintergründe in jeden beliebigen Raum mithilfe von Ebenen einzufügen. Wie der Name schon sagt, ist eine Hintergrundebene einfach ein Sprite-Asset, das als Hintergrund auf einer Ebene in einer bestimmten Tiefe im Raum verwendet wird. Wie der Name schon sagt, ist eine Hintergrundebene einfach ein Sprite-Asset, das als Hintergrund auf einer Ebene in einer bestimmten Tiefe innerhalb des Raums verwendet wird, und durch das Stapeln von Ebenen können Sie einige Dinge über oder unter anderen zeichnen. Sie können auch bestimmte Aspekte von Ebenen über den Code steuern, indem Sie Dinge hinzufügen oder entfernen oder bestimmte Eigenschaften der Ebene oder der darin enthaltenen Elemente ändern. der Ebene oder deren Inhalt zur Laufzeit ändern. Im Folgenden finden Sie eine Liste aller Funktionen, die für die Bearbeitung verwendet werden können Hintergrundebenen:

  • layer_hintergrund_get_id
  • layer_background_exists
  • schicht_hintergrund_erzeugen
  • layer_background_destroy
  • layer_background_visible
  • layer_background_sprite
  • layer_background_htiled
  • layer_background_vtiled
  • layer_background_stretch
  • layer_background_blend
  • layer_hintergrund_alpha
  • schicht_hintergrund_index
  • Schicht_Hintergrund_Geschwindigkeit
  • layer_hintergrund_xscale
  • layer_background_yscale
  • layer_background_get_visible
  • layer_background_get_sprite
  • layer_background_get_htiled
  • layer_background_get_vtiled
  • layer_background_get_stretch
  • layer_background_get_blend
  • layer_hintergrund_get_alpha
  • layer_hintergrund_get_index
  • layer_hintergrund_get_speed
  • layer_hintergrund_get_xscale
  • layer_background_get_yscale

Beachten Sie, dass Sie die folgenden allgemeinen Ebenenfunktionen verwenden müssen, wenn Sie die Hintergrundposition oder die horizontale oder vertikale Bildlaufgeschwindigkeit des Hintergrunds einstellen möchten:

  • Schicht_x
  • Schicht_y
  • Schicht_Geschwindigkeit
