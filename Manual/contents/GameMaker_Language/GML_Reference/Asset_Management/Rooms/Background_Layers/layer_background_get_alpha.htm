layer_hintergrund_get_alpha

Mit dieser Funktion können Sie den Alphawert des Hintergrundelements ermitteln. Sie geben die ID des Hintergrundelements an (die Sie erhalten, wenn Sie ein Hintergrundelement mit layer_background_create() oder wenn Sie die Funktion layer_background_get_id()), und die Funktion gibt einen Wert zwischen 0 (völlig transparent) und 1 (völlig undurchsichtig) zurück.

Syntax:

layer_background_get_alpha(background_element_id);

Argument Beschreibung
hintergrund_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden sollen

Rückgabe:

Real (von 0 bis 1)

Beispiel:

var lay_id = layer_get_id("Hintergrund_Himmel");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_alpha(back_id) < 0.1
{
layer_background_destroy(back_id);
}

Der obige Code ermittelt die Ebenen-ID für die Ebene mit dem Namen "Hintergrund_Himmel" und verwendet diese dann, um die ID des Hintergrundelements auf dieser Ebene zu ermitteln. Diese ID wird dann verwendet, um das Element Alpha zu prüfen, und wenn es kleiner als 0,1 ist, wird das Ebenenelement zerstört.
