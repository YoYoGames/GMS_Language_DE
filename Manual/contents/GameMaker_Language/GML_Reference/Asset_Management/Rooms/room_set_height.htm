raum_setzen_höhe

Mit dieser Funktion können Sie die Höhe eines beliebigen Raums in Ihrem Spiel ändern (oder einstellen) , mit Ausnahme des aktuellen Raums.

Syntax:

room_set_height(index, h);

Argument Beschreibung
index Der Index des Raums, für den die Höhe eingestellt werden soll.
h Die neue Höhe des Raums in Pixel.

Rückgabe:

N/A

Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Dadurch wird ein neuer Raum erstellt und sein Index in der Variablen "global.myroom" gespeichert. Dann wird seine Breite auf 640 Pixel, seine Höhe auf 480 Pixel, seine Beschriftung auf "Game Room" und seine Persistenz auf "false" gesetzt.
