Raum_Springen

Mit dieser Funktion können Sie zu einem beliebigen Raum in Ihrem Spielprojekt wechseln, unabhängig davon, ob dieser mit Code oder im Asset-Browser erstellt wurde. Sie geben den Raumindex an (gespeichert in der Variablen für den Raumnamen oder als Variable, die von der Funktion room_add()). Beachten Sie, dass sich der Raum bis zum Ende des Ereignisses, in dem die Funktion aufgerufen wurde, nicht ändert. Die Einschränkung hierbei ist, dass jeder Code, der Instanzen erzeugt und nach dieser Funktion aufgerufen wird aufgerufen wird, keine Instanz erzeugt und daher vor dem Ändern des Raums aufgerufen werden sollte. Diese Funktion löst auch das Ereignis " Room End" aus.

Hinweis: Die Raum-IDs basieren nicht auf ihrer Reihenfolge im Asset-Browser oder im Raum-Manager, daher sollten Sie es vermeiden, direkt einen Zahlenwert anzugeben. Verwenden Sie stattdessen die Raumkonstante für die Kühlstelle, auf die Sie verweisen wollen (das ist einfach ihr Name), oder rufen Sie sie über eine Funktion ab.

Syntax:

room_goto(index);

Argument Beschreibung
index Der Index des Raums, zu dem Sie gehen sollen.

Rückgabe:

N/A

Beispiel:

switch (global.level)
{
case 0: room_goto(rm_level1); break;
case 1: room_goto(rm_level2); break;
case 2: room_goto(rm_level3); break;
}

Der obige Code überprüft eine globale Variable und ändert den Raum basierend auf dem Wert, den sie enthält.
