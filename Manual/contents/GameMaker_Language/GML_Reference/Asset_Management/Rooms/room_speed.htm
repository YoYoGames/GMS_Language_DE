room_speed (veraltet)

Diese Variable enthält die Laufgeschwindigkeit aller Räume (und des Spiels) in Spielbildern pro Sekunde. Beachten Sie, dass dies NICHT die FPS (Bilder pro Sekunde) ist, sondern die Anzahl der Spielschritte, die GameMaker Studio 2 versucht, jede Sekunde.

WICHTIG! Diese Variable wird nur für den Legacy-Support gepflegt und sollte nicht verwendet werden, da sie nicht mehr die Geschwindigkeit für einen einzelnen Raum, sondern für ALLE Räume im Spiel setzt. Um die Spielgeschwindigkeit zu ändern, sollten Sie stattdessen die Funktion game_set_speed().

Syntax:

room_speed;

Rückgabe:

Real

Beispiel:

alarm[0] = room_speed * 10;

Der obige Code verwendet room_speed, um einen Alarm so einzustellen, dass er 10 Sekunden lang läuft. Wenn Sie die Variable auf diese Weise verwenden, wird sichergestellt, dass der Alarm 10 Sekunden lang läuft, unabhängig davon, welche Geschwindigkeit für den Raum eingestellt ist.
