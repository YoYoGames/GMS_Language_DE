layer_sequence_get_headdir

Mit dieser Funktion übergeben Sie die ID des Sequenzelements - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die aktuelle Abspielkopfrichtung zurück, die eine der unten aufgeführten Konstanten sein wird.

Syntax:

layer_sequence_get_headdir(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Konstant

Konstant Beschreibung Wert
seq_dir_right Die Sequenz spielt Frames in einer inkrementellen Reihenfolge von links nach rechts ab 1
seq_dir_left Die Sequenz spielt Frames in einer absteigenden Reihenfolge von rechts nach links ab -1

Beispiel:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
layer_sequence_headdir(title_sequence, seq_dir_left);
}

Der obige Code prüft die aktuelle Abspielkopfrichtung der Sequenzelement-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf seq_dir_left gesetzt ist, wird sie auf diesen Wert gesetzt.
