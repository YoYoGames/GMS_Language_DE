Schicht_Reihenfolge_y

Mit dieser Funktion können Sie die Position entlang der Y (vertikalen) Achse des Raumes für das angegebene Sequenzelement festlegen. Sie liefern die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit der zu setzenden Y-Position und die Sequenz wird an die neue Position verschoben.

Syntax:

layer_sequence_y(sequence_element_id, pos_y)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
pos_y Die Y-Position, an die das Sequenzelement verschoben werden soll

Rückgabe:

N/A

Beispiel:

if layer_sequence_exists(mein_seq)
{
layer_sequence_x(my_seq, x);
layer_sequence_y(my_seq, y);
}

Der obige Code prüft, ob das in der Variablen "my_seq" referenzierte Sequenzelement existiert, und wenn ja, setzt er die x/y-Position auf die der Instanz, die den Code ausführt.
