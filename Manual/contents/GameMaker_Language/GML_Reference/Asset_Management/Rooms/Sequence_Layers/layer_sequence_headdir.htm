layer_sequence_headdir

Mit dieser Funktion können Sie die Richtung für den angegebenen Sequenzabspielkopf einstellen. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, und geben dann die Abspielkopfrichtung an, die eine der folgenden Konstanten sein sollte:

Konstant Beschreibung Wert
seq_dir_right Die Sequenz spielt die Bilder in einer inkrementellen Reihenfolge von links nach rechts ab 1
seq_dir_left Die Sequenz spielt die Frames in einer absteigenden Reihenfolge von rechts nach links ab -1

Syntax:

layer_sequence_headdir(sequence_element_id, direction)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
Richtung Die Abspielkopfrichtung, eine Konstante, die oben aufgeführt ist

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

Der obige Code erstellt eine neue Sequenz und speichert ihre ID in einer lokalen Variablen. Diese ID wird dann zum Einstellen der Abspielkopfrichtung verwendet, um Frames zu dekrementieren (Wiedergabe von rechts nach links).
