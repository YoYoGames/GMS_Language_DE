layer_sequence_get_headpos

Mit dieser Funktion liefern Sie die ID des Sequenzelements - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die aktuelle Abspielkopfposition zurück (das aktuelle Bild, auf dem sich der Abspielkopf befindet).

Syntax:

layer_sequence_get_headpos(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Real

Beispiel:

if layer_sequence_get_headpos(title_sequence) != 0
{
layer_sequence_headpos(title_sequence, 0);
}

Der obige Code prüft die aktuelle Abspielkopfposition der Sequenzelement-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf 0 gesetzt ist, wird sie auf diesen Wert gesetzt.
