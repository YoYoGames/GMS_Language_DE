layer_sequence_get_instance

Mit dieser Funktion liefern Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die Sequenzinstanz-Struktur zurück. Mehr über das Format der Sequenzinstanz-Struktur erfahren Sie auf dieser Seite.

Syntax:

layer_sequence_get_instance(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Struktur

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

Der obige Code erstellt ein neues Sequenzelement und ruft dann dessen Sequenzinstanz-ID ab. Diese wird dann verwendet, um die Eigenschaft Skalierung der Wiedergabegeschwindigkeit der Sequenzinstanz zu ändern.
