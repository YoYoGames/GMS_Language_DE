layer_sequence_play

Mit dieser Funktion können Sie die Wiedergabe der angegebenen Sequenz starten. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Layer-Element-Funktionen zurückgegeben wird, und die Funktion spielt die Sequenz ab, die Sie dann bei Bedarf mit der Funktion layer_sequence_pause().

WICHTIG! Wenn Ihre Sequenz irgendwelche Instanzen enthält, sollten diese Instanzen nicht ihre image_xscale / image_yscale / Bild_Winkel / x / y Variablen nicht ändern, da sie überschrieben werden, wenn die Sequenz jeden Schritt nach dem Start der Wiedergabe aktualisiert.

Syntax:

layer_sequence_play(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sequence
{
if global.Pause
{
layer_sequence_pause(a[i]);
}
sonst
{
layer_sequence_play(a[i]);
}
}
}
}

Der obige Code prüft, ob das Spiel angehalten wurde oder nicht, wenn eine Taste gedrückt wird. Wenn das Spiel angehalten wurde, wird eine Schleife über alle Sequenzelemente auf der aktuellen Ebene (der Ebene der aufrufenden Instanz) ausgeführt und deren Wiedergabe angehalten, und wenn das Spiel nicht pausiert ist, dann startet die Schleife deren Wiedergabe erneut.
