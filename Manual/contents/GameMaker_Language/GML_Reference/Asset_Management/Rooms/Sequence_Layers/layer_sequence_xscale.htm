layer_sequence_xscale

Mit dieser Funktion können Sie die X-Skala für das angegebene Sequenzelement einstellen. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit dem neu zu setzenden Maßstab auf der X-Achse und die Sequenz wird um diesen Betrag skaliert. Eine Skalierung von 1 bedeutet keine Skalierung (1:1), kleinere Werte skalieren nach unten (0,5 z. B. halbiert die Breite der Sequenz), größere Werte skalieren und negative Werte spiegeln die Sequenz um ihren Ursprung und skalieren sie, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird die Sequenz nur um ihren Ursprung gespiegelt und nicht skaliert).

Es ist sehr wichtig zu beachten, dass die Anwendung einer ungleichmäßigen Skalierung (z. B. Skalierung der X-Achse um 3 und der Y-Achse um 2) auf Sequenzelemente, die eine Instanz enthalten, die eine Drehung verwendet, zu Problemen mit dem Zeichnen der Instanz, Kollisionen, Culling und vielen anderen Dingen führen kann. Grundsätzlich gilt: Wenn Ihre Sequenz auf Instanzeigenschaften angewiesen ist, empfehlen wir Ihnen nicht, eine ungleichmäßige Skalierung mit einer Instanzdrehung zu kombinieren.

Beachten Sie auch, dass Sie die Variablen image_xscale \ image_yscale \ image_angle \ x \ y nicht für alle Instanzen in einer Sequenz, die diese Funktion verwendet, ändern sollten, da sie überschrieben werden, sobald die Sequenz nach dem Aufruf der Funktion aktualisiert wird.

Syntax:

layer_sequence_xscale(sequence_element_id, xscale)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
x-Skala Der neue Skalenwert der X-Achse, der auf das Sequenzelement angewendet werden soll

Rückgabe:

N/A

Beispiel:

if seq_scale < 2
{
seq_scale += 0.01;
layer_sequence_xscale(my_seq, seq_scale);
layer_sequence_yscale(my_seq, seq_scale);
}

Der obige Code prüft den Wert in der Variablen seq_scale, und wenn er kleiner als 2 ist, addiert er ihn und verwendet den Wert, um die X- und Y-Skala des in der Variablen my_seq referenzierten Sequenzelements einzustellen.
