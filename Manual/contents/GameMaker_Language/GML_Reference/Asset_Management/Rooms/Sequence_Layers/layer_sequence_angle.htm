layer_sequence_angle

Mit dieser Funktion können Sie den Drehwinkel für das angegebene Sequenzelement setzen. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit dem neu zu setzenden Winkel und die Sequenz wird um ihren Ursprung an die neue Position gedreht. Winkel werden im GameMaker mit 0º nach rechts berechnet und verlaufen gegen den Uhrzeigersinn - 90º ist also oben, 180º ist links und 270º ist unten - und der Standardwinkel für eine Sequenz wäre 0º.

Beachten Sie, dass Sie die Variablen image_xscale \ image_yscale \ image_angle \ x \ y für alle Instanzen nicht ändern sollten in einer Sequenz, die diese Funktion verwendet, ändern sollten, da sie überschrieben werden, sobald die Sequenz nach dem Aufruf der Funktion aktualisiert wird.

Syntax:

layer_sequence_angle(sequence_element_id, angle)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
Winkel Der neue Winkel, in den das Sequenzelement gedreht werden soll

Rückgabe:

N/A

Beispiel:

if current_angle < 90
{
current_angle += 1;
layer_sequence_angle(my_seq, current_angle);
}

Der obige Code prüft den Wert, der in der Variablen current_angle gehalten wird, und wenn er kleiner als 90 ist, addiert er ihn und verwendet den Wert, um den Winkel des Sequenzelements einzustellen, auf das in der Variablen my_seq verwiesen wird.
