layer_sequence_is_finished

Mit dieser Funktion übergeben Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie prüft, ob die Sequenz zu Ende abgespielt wird oder nicht, und gibt true zurück, wenn dies der Fall ist, und false, wenn dies nicht der Fall ist. Beachten Sie, dass dies nur anwendbar ist, wenn die Sequenz nicht auf Schleife oder Ping-Pong im Wiedergabemodus eingestellt ist.

Syntax:

layer_sequence_is_finished(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Boolesche

Beispiel:

if layer_sequence_is_finished(title_sequence) != 0
{
alarm[0] = room_speed * 3;
layer_sequence_play(title_sequence);
}

Der obige Code prüft, ob die in der Variablen "title_sequence" gespeicherte Sequenzelement-ID beendet ist, und wenn ja, startet er die Wiedergabe erneut und setzt einen Alarm.
