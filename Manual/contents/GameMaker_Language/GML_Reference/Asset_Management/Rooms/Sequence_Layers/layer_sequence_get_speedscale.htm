layer_sequence_get_speedscale

Mit dieser Funktion liefern Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die aktuelle Skala für die Wiedergabegeschwindigkeit zurück. Dies ist der Multiplikatorwert, der zum Verlangsamen oder Beschleunigen der Wiedergabegeschwindigkeit verwendet wird. Ein Wert von 1 ist der Standardwert, und Werte unter 1 bedeuten, dass die Wiedergabe verlangsamt wird, und Werte über 1 bedeuten, dass die Wiedergabe beschleunigt wird. Werte größer als 1 bedeuten, dass die Wiedergabe beschleunigt wird.

Syntax:

layer_sequence_get_speedscale(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Real

Beispiel:

if layer_sequence_get_speedscale(title_sequence) != 1
{
layer_sequence_speedscale(title_sequence, 1);
}

Der obige Code prüft die aktuelle Abspielgeschwindigkeitsskala der Sequenzelement-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf 1 gesetzt ist, wird sie auf diesen Wert gesetzt.
