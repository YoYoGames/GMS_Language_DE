layer_sequence_get_sequence

Mit dieser Funktion liefern Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die Sequenzobjekt-Struktur zurück. Diese Funktion umgeht die Notwendigkeit, zuerst die Sequenzinstanz-Struktur zu erhalten, und ermöglicht Ihnen den direkten Zugriff auf die Sequenzdaten. Sie können mehr erfahren über das Format der Sequenzobjekt-Struktur erfahren Sie auf dieser Seite.

Syntax:

layer_sequence_get_sequence(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Struktur

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

Der obige Code erstellt ein neues Sequenzelement und ruft dann dessen Sequenzdatenstruktur ab. Diese wird dann verwendet, um die Abspielgeschwindigkeit und den Typ der Sequenz zu ändern.
