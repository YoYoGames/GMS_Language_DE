layer_sequence_exists

Mit dieser Funktion können Sie prüfen, ob ein Sequenzelement auf der angegebenen Ebene existiert. Sie geben die Ebenen-ID an, die eine Zeichenkette des Ebenennamens sein kann - wie im Raumeditor definiert - oder die eindeutige Ebenen-ID - wie von der Funktion layer_get_id(), zurückgegeben wird, sowie die ID des Sequenzelements - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und es wird true zurück, wenn das angegebene Element existiert, ansonsten false.

Syntax:

layer_sequence_exists(layer_id, sequence_element_id)

Argument Beschreibung
Schicht_ID Die eindeutige ID oder der Name der zu prüfenden Ebene
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Boolesche

Beispiel:

if !layer_sequence_exists("Hintergrund", my_seq)
{
my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(my_seq);
}

Der obige Code prüft, ob das angegebene Sequenzelement vorhanden ist, und wenn nicht, wird eine neue Sequenz auf der angegebenen Ebene erstellt und dann pausiert.
