layer_sequence_is_paused

Mit dieser Funktion übergeben Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenen-Element-Funktionen zurückgegeben wird - und sie prüft, ob die Sequenz gerade angehalten ist oder nicht, und gibt true zurück, wenn sie angehalten ist, und false, wenn sie nicht angehalten ist.

Syntax:

layer_sequence_is_paused(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Boolesche

Beispiel:

if layer_sequence_is_paused(title_sequence) != 0
{
layer_sequence_play(title_sequence);
}

Der obige Code prüft, ob die in der Variablen "title_sequence" gespeicherte Sequenzelement-ID pausiert ist, und wenn ja, startet er die Wiedergabe.
