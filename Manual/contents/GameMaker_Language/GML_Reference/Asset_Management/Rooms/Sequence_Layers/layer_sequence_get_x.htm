layer_sequence_get_x

Mit dieser Funktion übergeben Sie die ID des Sequenzelements - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt die aktuelle X-Position im Spielraum für die Sequenz zurück.

Syntax:

layer_sequence_get_x(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Real

Beispiel:

if layer_sequence_get_x(title_sequence) != room_width / 2
{
layer_sequence_x(title_sequence, room_width / 2);
}

Der obige Code prüft die X-Position der Sequenzelement-ID, die in der Variablen "title_sequence" gespeichert ist, und wenn sie nicht auf die halbe Raumbreite gesetzt ist, wird sie auf diesen Wert gesetzt.
