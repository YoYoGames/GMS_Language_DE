layer_sequence_create

Mit dieser Funktion können Sie eine Instanz eines Sequenz-Assets auf der angegebenen Ebene erzeugen. Sie geben die Ebenen-ID an, die eine Zeichenkette des Ebenennamens sein kann - wie im Raumeditor definiert - oder die eindeutige Ebenen-ID - wie sie von der Funktion layer_get_id(), sowie die X- und Y-Position im Raum, an der die Sequenz erstellt werden soll, und schließlich die ID der zu erstellenden Sequenz. Die Sequenz-ID ist die Namenskonstante, die Sie im Asset-Browser für die Sequenz definiert haben. Die Funktion gibt die eindeutige ID des Sequenzelements zurück, die dann in allen weiteren Ebenenfunktionen für Sequenzen verwendet werden kann, oder sie kann zum Abrufen der Sequenzinstanzstruktur mit der Funktion layer_sequence_get_instance().

Syntax:

layer_sequence_create(layer_id, x, y, sequence_id)

Argument Beschreibung
layer_id Der eindeutige ID-Wert der anzuvisierenden Instanzschicht
x Die x-Position im Raum, an der die Sequenz erstellt werden soll
y Die y-Position im Raum, an der die Sequenz erstellt werden soll
sequenz_id Die zu erstellende Sequenz-ID, wie im Asset-Browser definiert

Rückgabe:

Integer (Element-ID)

Beispiel:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

Der obige Code erstellt eine neue Sequenz auf der Ebene "Hintergrund" und hält sie dann an.
