layer_sequence_get_xscale

Mit dieser Funktion liefern Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird - und sie gibt den aktuellen Maßstab entlang der X-Achse des Sequenzelements im Spielraum zurück.

Syntax:

layer_sequence_get_xscale(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Real

Beispiel:

var _xs = layer_sequence_get_xscale(title_sequence)
if _xs < 1
{
_xs += 0.01;
layer_sequence_xscale(title_sequence, _xs);
}

Der obige Code ruft die aktuelle Skala entlang der X-Achse des Sequenzelements mit der in der Variablen "title_sequence" gespeicherten ID ab, und wenn diese kleiner als 1 ist, dann wird 0,01 zur aktuellen X-Skala addiert.
