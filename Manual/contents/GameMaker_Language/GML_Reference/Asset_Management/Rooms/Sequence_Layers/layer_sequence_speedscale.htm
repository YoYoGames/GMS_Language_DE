layer_sequence_speedscale

Mit dieser Funktion können Sie die Abspielgeschwindigkeit der angegebenen Sequenz ändern. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelementfunktionen zurückgegeben wird, und dann einen Geschwindigkeitsskalierungswert. Dieser Wert ist ein Multiplikator, wobei 1 die Standard-Wiedergabegeschwindigkeit ist und Werte kleiner als 1 die Wiedergabe verlangsamen und Werte größer als 1 die Wiedergabe beschleunigen, z. B.: ein Wert von 0,5 wäre die halbe Wiedergabegeschwindigkeit, während ein Wert von 2 würde die doppelte Wiedergabegeschwindigkeit bedeuten.

Syntax:

layer_sequence_speedscale(sequence_element_id, speedscale)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
Geschwindigkeitsskala Die zu verwendende Geschwindigkeitsskala (ein Multiplikator), Standard ist 1

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

Der obige Code erstellt eine neue Sequenz auf der Ebene der aufrufenden Instanz und setzt dann ihren Geschwindigkeitsskalierungswert auf 0,75 (drei Viertel der Standard-Wiedergabegeschwindigkeit).
