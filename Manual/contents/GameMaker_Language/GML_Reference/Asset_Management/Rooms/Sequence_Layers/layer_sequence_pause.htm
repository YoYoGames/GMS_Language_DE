layer_sequence_pause

Mit dieser Funktion können Sie die Wiedergabe der angegebenen Sequenz pausieren. Sie übergeben die Sequenzelement-ID, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, und die Funktion hält die Sequenz an, bis Sie die Wiedergabe erneut mit der Funktion layer_sequence_play().

Syntax:

layer_sequence_pause(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

N/A

Beispiel:

var _seq = layer_sequence_create("Hintergrund", 0, 0, seq_AnimatedBackground);
if global.Pause
{
layer_sequence_pause(_seq);
}

Der obige Code erstellt eine neue Sequenz und speichert ihre ID in einer lokalen Variablen. Anschließend wird geprüft, ob das Spiel angehalten ist, und wenn ja, wird die Wiedergabe der Sequenz angehalten.
