layer_sequence_headpos

Mit dieser Funktion können Sie die Abspielposition eines Sequenzelements auf einen bestimmten Frame setzen. Sie übergeben die ID des Sequenzelements, wie sie von layer_sequence_create() oder von einer der Ebenenelement-Funktionen zurückgegeben wird, zusammen mit der neu zu setzenden Position. Beachten Sie, dass die Position in Frames angegeben wird. Wenn Sie einen Wert eingeben, der größer als die Gesamtzahl der Frames (oder weniger als 0), dann hängt die tatsächliche endgültige Abspielkopfposition von der Art der Sequenzwiedergabe ab, die ausgewählt wurde, und folgt diesen Regeln:

  • Keine Schleifenbildung: Der Abspielkopf wird an das Ende der Schleife geklemmt (oder an den Anfang, wenn der angegebene Wert negativ ist)
  • Schleifen: Der Abspielkopf springt zurück zum Anfang der Sequenz plus die zusätzlichen Bilder, z. B.: Wenn Sie den Abspielkopf bei einer Sequenz mit 30 Bildern auf eine Position von Bild 45 setzen, ist die tatsächliche Position des Abspielkopfs am Ende 15. Wenn Sie einen negativen Wert angeben, wird der Abspielkopf der Sequenz in umgekehrter Richtung umlaufen.
  • Ping-Pong-Schleife: Der Abspielkopf bewegt sich bis zum Ende und geht dann um die zusätzliche Anzahl von Bildern zurück, die über der Gesamtbildlänge liegen, wobei die Wiedergaberichtung umgekehrt wird, z. B.: Wenn die Sequenz 30 Bilder umfasst und Sie den Abspielkopf auf 40 gesetzt haben, ist die endgültige Abspielkopfposition 20 und die Richtung wird umgekehrt (nach unten auf 0). Beachten Sie, dass Sie niemals einen negativen Wert für die Ping-Pong-Wiedergabe einer Sequenz angeben sollten, da dies zu undefinierten Ergebnissen führen kann.

Wenn Sie die Kopfposition auf diese Weise einstellen, wird die Wiedergabe nicht gestoppt, sondern die Sequenz wird einfach an der neuen Abspielkopfposition fortgesetzt, sofern sie nicht angehalten wird.

Syntax:

layer_sequence_headpos(sequence_element_id, position)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll
Position Die Position innerhalb der Sequenz (in Frames), auf die die Abspielkopfposition gesetzt werden soll (kann ein dezimaler Wert sein)

Rückgabe:

N/A

Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sequence
{
layer_sequence_headpos(a[i], 0)
}
}

Der obige Code ruft die IDs für alle Elemente ab, die der Ebene der Instanz zugewiesen sind, die den Code ausführt. Der Code prüft dann, ob es sich bei den zurückgegebenen Elementen um Sequenz-Assets handelt, und wenn ja, wird ihre Abspielkopfposition auf 0 gesetzt.
