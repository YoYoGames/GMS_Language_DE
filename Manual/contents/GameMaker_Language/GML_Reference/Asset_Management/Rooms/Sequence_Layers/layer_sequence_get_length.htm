layer_sequence_get_length

Mit dieser Funktion liefern Sie die Sequenzelement-ID - wie sie von layer_sequence_create() oder von einer der Ebenenelementfunktionen zurückgegeben wird - und sie gibt die Länge der Sequenz zurück. Dies ist die Anzahl der Frames, für die die Sequenz laufen wird.

Syntax:

layer_sequence_get_length(sequence_element_id)

Argument Beschreibung
sequenz_element_id Der eindeutige ID-Wert des Sequenzelements, auf das abgezielt werden soll

Rückgabe:

Integer

Beispiel:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

Der obige Code ruft die Anzahl der Frames ab, die eine Sequenz durchläuft, und verwendet dann diesen Wert, um einen Alarm zu setzen.
