event_type

Diese schreibgeschützte Variable gibt den Typ des aktuell ausgeführten Ereignisses zurück, der eine der folgenden Konstanten sein kann:

  • ev_erstellen
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_maus
  • ev_kollision
  • ev_andere
  • ev_gesture
  • ev_draw


Für eine vollständige Liste der Konstanten, die sich auf die verschiedenen Ereignisse beziehen, siehe event_perform()und wenn Sie die Ereignisnummer (d. h. das Unterereignis) benötigen, sollten Sie die event_number.

Syntax:

event_type;

Rückgabe:

Real

Beispiel:

show_debug_message("Aktuelles Ereignis = " + string(event_type));

Der obige Code zeigt eine Debug-Meldung mit dem Ereignistyp an, der gerade ausgeführt wird.
