Ereignis_Objekt

Diese schreibgeschützte Variable gibt den Objektindex der Instanz zurück, die das zu prüfende Ereignis ausführt.

Syntax:

event_object;

Rückgabe:

Real

Beispiel:

global.obj = event_object;

Der obige Code speichert den Objektindex der Instanz, die das Ereignis ausführt, in einer globalen Variablen.
