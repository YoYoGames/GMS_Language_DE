event_action (veraltet)

Diese schreibgeschützte Variable gab den Index der gerade ausgeführten Aktion zurück, beginnend bei 0 in früheren Versionen von GameMaker. In GameMaker Studio 2 ist diese Variable jedoch obsolet geworden . Sie wurde nur noch als Legacy-Unterstützung verwendet und gibt immer 0 zurück, da alle Aktionen miteinander verkettet werden, um die Ausführungsgeschwindigkeit zu verbessern.

Syntax:

Ereignis_Aktion;

Rückgabe:

Real

Beispiel:

num = event_action;

Der obige Code speichert die aktuelle Aktionsnummer in der Variablen "num".
