Erzeugen von Objektereignissen

GameMaker Studio 2 ist vollständig ereignisgesteuert, d. h. alle Aktionen erfolgen als Folge bestimmter Ereignisse, die in einer bestimmten Reihenfolge oder durch ein bestimmtes Ereignis im Spiel ausgelöst werden. Es gibt eine Reihe von verschiedenen Ereignissen, jedes mit einer bestimmten Aufgabe oder einem bestimmten Timing und einige von ihnen sind sogar in separate "Sub"-Events aufgeteilt. Normalerweise werden diese Ereignisse vom GameMaker Studio 2 ausgeführt, wenn es einen Auslöser im Spiel erkennt, oder bei jedem Spieltick (wie bei den Ereignissen "Step" und "Draw"), aber Sie können ein Ereignis auch aus einem Stück Code heraus auf eine Instanz im aktuellen Raum anwenden.

Es gibt folgende Funktionen, die sich mit der Erzeugung von Ereignissen befassen:

  • event_inherited
  • event_perform
  • event_perform_object
  • event_user

Es gibt auch eine Reihe von Nur-Lese-Variablen, die mit den oben aufgeführten Funktionen verwendet werden können, die besonders für die Fehlersuche nützlich sind:

  • Ereignis_Aktion
  • Ereignis_Nummer
  • Ereignis_Objekt
  • event_type

Weitere Informationen zu Ereignissen finden Sie im Abschnitt Objekt-Ereignisse.
