event_perform

Diese Funktion führt das angegebene Ereignis mit dem angegebenen Argument für die Instanz aus, die den Code ausführt. Es gibt hier viele Optionen, die eine vollständige Simulation aller möglichen Ereignisse ermöglichen, aber beachten Sie, dass dies buchstäblich nur den gesamten Code ausführt im Ereignis ausführt und das Spiel nichts verändert, um es selbst manuell auszulösen. Wenn Sie z. B. ein Tastendruck-Ereignis auswählen, wird das Ereignis zwar ausgelöst, aber die entsprechende Taste wird nicht als gedrückt erkannt. Oder wenn Sie ein Alarmereignis ausführen, wird der Alarmzähler nicht auf -1 gesetzt, sondern zählt weiter abwärts.

Die folgende Tabelle zeigt alle Konstanten, die in GameMaker Studio 2 zur Referenzierung von Ereignistypen enthalten sind, sowie deren Zahlenwert. Die Ereignisse selbst sind fett markiert und entsprechen den Rückgabewerten der event_type Variablen, während ihre Zahlenwerte mit kursivem Text gekennzeichnet sind und diese mit den Rückgabewerten der event_number Variable sind:

ev_draw Ereignis "Zeichnen". HINWEIS: Dieses Ereignis kann nicht außerhalb eines Draw-Ereignisses erzwungen werden und die Konstanten und die Konstanten dienen nur zur Identifizierung des Ereignisses, wenn es in diesen Fällen ausgeführt wird.

---- ev_gui_end Das Ereignis "Draw Gui End".

Syntax:

event_perform(type, numb);

Argument Beschreibung
Typ Der Typ des auszuführenden Ereignisses. Siehe unten.
taub Die spezifische Ereigniskonstante oder der Wert, falls einer erforderlich ist (ansonsten einfach 0 verwenden).

Rückgabe:

N/A

Erweitertes Beispiel


Um zu verstehen, wie diese Funktion verwendet werden kann, sehen Sie sich bitte die folgenden Beispiele an:

event_perform(ev_keypress, ord("W"));

Dies würde das Ereignis ausführen, das mit Tastaturprüfung Gedrückte Taste "W" verbunden ist (ohne tatsächlich einen Tastendruck zu erzeugen).

event_perform(ev_step, ev_step_begin);

Dies würde das Ereignis "Begin Step" ausführen (wenn es von einem der Step-Ereignisse aus aufgerufen wird, würde es dazu führen, dass der Code des Ereignisses "Begin Step" zweimal ausgeführt wird).

event_perform(ev_create, 0);

Dies würde das Ereignis" Create" der Instanzausführen .
