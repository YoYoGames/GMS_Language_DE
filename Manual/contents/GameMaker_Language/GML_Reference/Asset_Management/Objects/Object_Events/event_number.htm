Ereignis_Nummer

Diese schreibgeschützte Variable gibt die Nummer des aktuell aufgerufenen Ereignisses zurück, wobei sich die Nummer eigentlich auf das "Unterereignis" des Ereignisses bezieht, d. h.: für das Schritt-Ereignis könnte die Ereignisnummer eine der Konstanten ev_step_normal , ev_step_begin, oder ev_step_end sein. Eine vollständige Liste der Konstanten, die für die spezifischen Unterereignisse verfügbar sind, finden Sie unter event_perform(), und wenn Sie das Hauptereignis selbst kennen müssen, sollten Sie überprüfen Sie die event_type.

Syntax:

event_number;

Rückgabe:

Real

Beispiel:

switch (event_number)
{
case ev_step_normal: show_debug_message("Step event!"); break;
case ev_game_start: show_debug_message("Game Start!"); break;
case ev_room_start: show_debug_message("Room Start!"); break;
}

Der obige Code könnte von einem Skript aus aufgerufen und verwendet werden, um Debug-Meldungen anzuzeigen, die Sie darüber informieren, welches Ereignis gerade ausgelöst wird.
