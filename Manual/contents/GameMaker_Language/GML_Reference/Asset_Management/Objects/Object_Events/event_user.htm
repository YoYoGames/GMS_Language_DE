event_user

Mit dieser Funktion weisen Sie die Instanz an, die Aktionen oder den Code auszuführen, der in einem der 16 benutzerdefinierten Ereignisse platziert wurde. Diese Ereignisse können nur auf diese Weise oder mit der event_perform Funktion.

Syntax:

event_user(numb);

Argument Beschreibung
taub Die Nummer des aufzurufenden Ereignisses, zwischen 0 und 15.

Rückgabe:

N/A

Beispiel:

event_user(4);

Dies würde das benutzerdefinierte Ereignis 4 aufrufen.
