objekt_get_persistent

Diese Funktion teilt Ihnen mit, ob das Objekt, das Sie prüfen, als "persistent" gekennzeichnet wurde oder nicht. Ein persistentes Objekt ist eines, dessen Instanzen von Raum zu Raum weitergeführt werden, wenn sie nicht explizit zerstört werden. Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können also ein persistentes Objekt und eine nicht-persistente Instanz desselben Objekts haben und umgekehrt. Sie können das Persistent-Flag einer einzelnen Instanz mit der Funktion persistent Instanz Variable setzen.

Syntax:

object_get_persistent(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts.

Rückgabe:

Boolesche

Beispiel:

if (!persistent) && (object_get_persistent(object_index))
{
persistent = true;
}

Der obige Code prüft die ausgeführte Instanz, um festzustellen, ob sie persistent ist oder nicht, sowie den Objektindex der Instanz, um festzustellen, ob sie als persistent gekennzeichnet ist oder nicht. Wenn die Instanz nicht persistent ist, aber der Objektindex als persistent gekennzeichnet ist als persistent gekennzeichnet ist, wird "persistent" für diese Instanz auf true gesetzt.
