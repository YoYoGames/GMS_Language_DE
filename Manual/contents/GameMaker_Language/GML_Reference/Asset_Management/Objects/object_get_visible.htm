object_get_visible

Diese Funktion teilt Ihnen mit, ob das Objekt, das Sie prüfen, als "sichtbar" gekennzeichnet wurde (führt sein Draw-Ereignis aus) oder nicht (führt sein Draw-Ereignis nicht aus). Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können also ein sichtbares Objekt und eine unsichtbare Instanz desselben Objekts haben und umgekehrt. Sie können die Sichtbarkeit einer einzelnen Instanz mit der sichtbar Instanz-Variable einstellen.

Syntax:

object_get_visible(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts.

Rückgabe:

Boolesche

Beispiel:

if (!visible) && (object_get_visible(object_index))
{
visible = true;
}

Der obige Code prüft die laufende Instanz, um zu sehen, ob sie sichtbar ist oder nicht, sowie den Objektindex der Instanz, um zu sehen, ob sie als sichtbar gekennzeichnet ist oder nicht. Wenn die Instanz noch nicht sichtbar ist, wird der Objektindex als "on" gekennzeichnet, setzt es "visible" für diese Instanz auf true.
