Objekt_Index

Diese schreibgeschützte Variable gibt den Index des Objekts zurück, von dem die Instanz erstellt wurde. Dies ist nicht dasselbe wie der Objektname, der eine Zeichenkette ist und mit object_get_name(), gefunden werden kann, da diese Funktion die Indexnummer zurückgibt, die ein eindeutiger Wert ist, den GameMaker Studio 2 jedem Objekt zum Zeitpunkt der Erstellung zuweist.

Syntax:

Objekt_Index;

Rückgabe:

Real (Ganzzahl - Ressourcenindexwert)

Beispiel:

obj_name = object_get_name(object_index);

Der obige Code verwendet den object_index, um den Namen des Objekts zu finden, von dem die aktuelle Instanz erstellt wurde.
