objekt_get_mask

Diese Funktion sagt Ihnen, ob das zu prüfende Objekt einen Maskenindex hat oder nicht, und wenn ja, dann gibt sie den Index dieser Maske zurück, oder -1, wenn das nicht der Fall ist. Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können ein Objekt mit keine Maske haben, während eine Instanz desselben Objekts eine haben kann und umgekehrt, oder sie können sogar unterschiedliche Masken haben. Sie können den Maskenindex einer einzelnen Instanz mit der mask_index Instanz-Variable setzen.

Syntax:

object_get_mask(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts

Rückgabe:

real

Beispiel:

if (mask_index != object_get_mask(object_index))
{
mask_index = object_get_mask(object_index);
}

Das obige Beispiel prüft den Maskenindex der Instanz gegen die Maske des Objekt_Index der Instanz. Wenn sie nicht gleich sind, dann wird der Instanz die gleiche Maske zugewiesen, die der Objektindex hat.
