objekt_get_name

Diese Funktion gibt den Namen des angegebenen Objekts als String zurück. Dieser Name ist derjenige, der für das Objekt im Asset-Browser des Hauptfensters von GameMaker Studio 2 angegeben wurde. Bitte beachten Sie, dass dies nur ein String ist ist und nicht verwendet werden kann, um das Objekt direkt zu referenzieren - dazu würden Sie den Objektindex benötigen. Sie können diese Zeichenkette jedoch verwenden, um den Objektindex zu erhalten, indem Sie die zurückgegebene Zeichenkette zusammen mit der Funktion asset_get_index().

Syntax:

object_get_name(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts.

Rückgabe:

Zeichenfolge

Beispiel:

str = object_get_name(object_index);

Der obige Code ermittelt den Namen des Objektindex für die Instanz, die den Code ausführt, und speichert den Rückgabewert in der Variablen "str".
