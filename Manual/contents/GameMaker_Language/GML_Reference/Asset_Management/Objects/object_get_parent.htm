object_get_parent

Diese Funktion liefert Ihnen den object_index jedes übergeordneten Objekts, das dem angegebenen Objekt zugewiesen wurde, oder gibt -100 zurück, um anzuzeigen, dass dem Objekt kein übergeordnetes Objekt zugewiesen ist, oder -1, wenn das überprüfte Objekt nicht existiert. Für weitere Informationen über Eltern finden Sie im Abschnitt über den Objekteditor.

Syntax:

object_get_parent(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts

Rückgabe:

Real

Beispiel:

par = object_get_parent(object_index);

Der obige Code holt das Parent des Objektindex für die Instanz, die den Code ausführt, und speichert den Rückgabewert in der Variablen "par".
