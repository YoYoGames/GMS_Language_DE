objekt_get_physik

Diese Funktion teilt Ihnen mit, ob das zu prüfende Objekt als "physics enabled" gekennzeichnet wurde - in diesem Fall gibt sie true zurück, - oder nicht - in diesem Fall gibt sie false zurück.

Syntax:

object_get_physics(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts.

Rückgabe:

Boolesche

Beispiel:

if object_get_physics(object_index)
{
phy_active = true;
}

Der obige Code prüft die Instanz, die ihn ausführt, um zu sehen, ob das Objekt, aus dem er erstellt wird, physikfähig ist, und wenn ja, aktiviert er die Physiksimulation für die Instanz.
