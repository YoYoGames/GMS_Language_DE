Objekt_existiert

Diese Funktion gibt zurück, ob ein Objekt mit dem angegebenen Index vorhanden ist oder nicht. Beachten Sie, dass damit geprüft wird, ob ein Objekt im Asset-Browser vorhanden ist, und nicht tatsächlich im Spielraum. Hierfür sollten Sie die Funktion instance_exists().

Syntax:

object_exists(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts.

Rückgabe:

Boolesche

Beispiel:

if object_exists(obj_temp)
{
mit (obj_Temp) scr_add_event();
}

Der obige Code prüft die Variable "obj_temp", um sicherzustellen, dass sie den Index eines Objekts speichert, und lässt dann, wenn dies der Fall ist, das Objekt ein Skript ausführen.
