object_set_solid

Mit dieser Funktion können Sie ein bestimmtes Objekt als solide oder nicht solide kennzeichnen (weitere Informationen zum Flag "solid" finden Sie im Abschnitt über den Objekteditor). Das bedeutet, dass alle Instanzen dieses Objekts, die nach dem Ändern von "solid" erzeugt werden, mit diesem neuen Zustand erzeugt werden, während Instanzen, die sich bereits im Raum befinden, möglicherweise nicht betroffen sind.

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können das Solid-Flag einzelner Instanzen mit der solid Variable setzen und so zehn Instanzen alle als solid kennzeichnen, obwohl das betreffende Objekt das Flag in der IDE auf false gesetzt hat, und selbst wenn Sie das Solid-Flag des Objekts mit dieser Funktion auf true ändern, bleiben alle Instanzen, die sich derzeit im Raum befinden, wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion der Funktion erstellt werden, beginnen mit dem Solid-Flag auf true.

Syntax:

object_set_solid( index, solid );

Argument Beschreibung
index Der Index des zu ändernden Objekts.
solide Die neue Solidität des Objekts (true=solide, false=nicht solide).

Rückgabe:

N/A

Beispiel:

if !object_get_solid(obj_Block)
{
object_set_solid(obj_Block, true);
}

Der obige Code prüft das Solid-Flag des Objekts "obj_Block" und wenn es falsch ist, dann wird es in das Flag "true" geändert.
