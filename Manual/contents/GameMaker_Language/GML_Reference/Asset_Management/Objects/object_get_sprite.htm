object_get_sprite

Diese Funktion sagt Ihnen, ob das zu prüfende Objekt ein Sprite hat oder nicht, und wenn ja, gibt sie den Index dieses Sprites zurück, oder -1, wenn es kein Sprite hat. Bitte beachten Sie, dass dies keine Instanzfunktion ist! Sie können ein Objekt haben, das kein Sprite haben, während eine Instanz desselben Objekts eines haben kann und umgekehrt, oder sie können sogar unterschiedliche Sprites haben. Sie können das Sprite einer einzelnen Instanz mit der sprite_index Instanz-Variable setzen.

Syntax:

object_get_sprite(obj);

Argument Beschreibung
obj Der Index des zu prüfenden Objekts

Rückgabe:

Real (Sprite-Asset-ID)

Beispiel:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
sprite_index = _spr;
}

Im obigen Beispiel wird das sprite_index der Instanz mit dem Sprite der object_index der Instanz verglichen. Wenn sie nicht übereinstimmen, wird der Instanz das gleiche Sprite zugewiesen wie das des Objektindex der Instanz zu.
