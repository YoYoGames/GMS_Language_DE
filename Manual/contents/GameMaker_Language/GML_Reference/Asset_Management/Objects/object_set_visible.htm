object_set_visible

Mit dieser Funktion können Sie ein bestimmtes Objekt als sichtbar oder nicht sichtbar kennzeichnen (weitere Informationen zum Flag "sichtbar" finden Sie im Abschnitt über den Objekteditor). Das bedeutet, dass alle Instanzen dieses Objekts, die nach der Änderung von "visible" erzeugt werden, mit diesem neuen Zustand erzeugt werden, während Instanzen, die sich bereits im Raum befinden, davon nicht betroffen sein dürfen.

Bitte beachten Sie, dass es sich hierbei nicht um eine Instanzfunktion handelt! Sie können das visible-Flag einzelner Instanzen mit der Variable sichtbar Variable setzen und so zehn Instanzen alle als sichtbar kennzeichnen, obwohl obwohl das betreffende Objekt das Flag in der IDE auf false gesetzt hat, und selbst wenn Sie das visible-Flag des Objekts mit dieser Funktion auf true ändern, bleiben alle Instanzen, die sich derzeit im Raum befinden, so wie sie waren, und nur Instanzen, die nach dem die nach dem Aufruf der Funktion erstellt werden, beginnen mit dem Kennzeichen visible auf true.

Syntax:

object_set_visible( index, vis );

Argument Beschreibung
index Der Index des zu ändernden Objekts.
vis Die neue Sichtbarkeit des Objekts (true=sichtbar, false=unsichtbar).

Rückgabe:

N/A

Beispiel:

if !object_get_visible(obj_Block)
{
object_set_visible(obj_Block, true);
}

Der obige Code prüft das Sichtbarkeitsflag des Objekts "obj_Block" und wenn es falsch ist, dann wird es auf wahr geändert.
