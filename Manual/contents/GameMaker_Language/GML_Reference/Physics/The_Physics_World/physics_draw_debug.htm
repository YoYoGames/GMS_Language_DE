physics_draw_debug

Beim Erstellen eines Physiksystems in GameMaker Studio 2 ist es oft wichtig, eine visuelle Darstellung dessen zu haben, was tatsächlich mit einer Instanz passiert. Dies kann mit der Funktion physics_draw_debug() erreicht werden, die einen schematischen Umriss der physikalischen Eigenschaften zeichnet, die mit der Instanz verbunden sind, auf der der Code läuft. Hier ist ein Bild davon, wie es aussieht:

Es ist zu beachten, dass dies nur funktioniert , wenn es im Draw-Ereignis der Instanz sein muss, und es wird mit der aktuell definierten Zeichenfarbe gezeichnet.

Syntax:

physics_draw_debug()

Rückgabe:

N/A

Beispiel:

draw_set_colour(c_red);
physics_draw_debug();

Der obige Code zeichnet eine Darstellung der physikalischen Eigenschaften, die mit dieser Instanz verbunden sind, unter Verwendung der Farbe Rot.
