Die Welt der Physik

Bevor in GameMaker Studio 2 physikalische Reaktionen auftreten können, müssen Sie zunächst die Physikwelt definieren, in der sie ablaufen sollen. Wenn Sie dies nicht tun, wird GameMaker Studio 2 standardmäßig die grundlegenden Kollisionsfunktionen, und alle anderen physikbezogenen Codes oder Einstellungen werden ignoriert. Sie können die Physikwelt über den Raum-Editor einrichten, aber Sie können die gleichen Eigenschaften auch über Code mit den folgenden Funktionen einstellen:

  • physik_welt_erschaffen
  • physik_welt_gravitation
  • physik_welt_aktualisieren_wiederholungen
  • physik_welt_aktualisieren_Geschwindigkeit
  • physics_world_draw_debug
  • physics_draw_debug
  • physik_pause_aktivieren
