physik_welt_gravitation

Wie bei allen Physik-Funktionen funktioniert physics_world_gravity() nur dann auf einer Instanz, wenn für den Raum eine Physik-Welt im Raum-Editor oder mittels physics_world_create(). Sie wird Sie verwendet dann den 0,0 definierenden Punkt der Physikwelt, um die Richtung und Stärke der Schwerkraft basierend auf dem Vektor zu berechnen, der aus den Eingaben xg und yg berechnet wird. Hier sind einige Bilder zur Veranschaulichung...

Wie Sie sehen können, sind die Koordinaten xg,yg relativ zur Position 0,0, wie sie für die definiert ist, und die Schwerkraft selbst wird als Vektor dieser beiden Komponenten berechnet. Die Standardwerte von 0,10 ergeben also eine Gravitationsrichtung von unten (270 Grad) mit einer Beschleunigung von 10 Metern pro Sekunde zum Quadrat. Beachten Sie die Meter pro Sekunde zum Quadrat! Wie alle Physikfunktionen wird die Schwerkraft auf der Basis von Metern berechnet, daher ist der Pixel-zu-Meter-Maßstab, den Sie in der Funktion physics_world_create einstellen, hier wichtig. Die Schwerkraft wirkt auch kumulativ auf das Objekt (aber die genaue Menge hängt davon ab, was Sie für die Funktion eingestellt haben physics_fixture_set_linear_damping()), was bedeutet, dass in einem Raum mit einer room_speed von 60, ohne Dämpfung, die Instanz in jeder Sekunde um 10 Meter pro Sekunde beschleunigen wird - was bei einer Skalierung von 0,1 Pixel zu Meter dasselbe ist, als würde sie in 60 Schritten auf eine Geschwindigkeit von 100 Pixel pro Sekunde beschleunigen.

Syntax:

physics_world_gravity(xg, yg)

Argument Beschreibung
xg Die x-Komponente des Gravitationsvektors
yg Die y-Komponente des Gravitationsvektors

Rückgabe:

N/A

Beispiel:

physics_world_gravity(-5, 0);

Der obige Code setzt die Schwerkraft (unter der Annahme, dass die Raumphysik 0,0 auf links oben eingestellt ist) nach rechts - 0 Grad - mit einer Beschleunigung von 5 Metern pro Sekunde zum Quadrat.
