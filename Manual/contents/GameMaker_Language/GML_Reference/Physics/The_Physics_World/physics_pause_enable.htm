physik_pause_aktivieren

Damit eine Physiksimulation funktioniert, muss sie normalerweise kontinuierlich ablaufen und kann nicht angehalten und gestartet werden oder Instanzen plötzlich von einem Ort zu einem anderen im Raum bewegt werden. Es gibt jedoch Momente, in denen Sie die Simulation anhalten müssen, weil (z. B. z. B.) das Gerät os angehalten hat, und so würden Sie diese Funktion verwenden. Sie pausiert die Simulation, wenn das Flag auf true gesetzt ist, und es werden keine weiteren Physikberechnungen durchgeführt, bis das Flag wieder auf false gesetzt wird.

HINWEIS: Dies ist besonders nützlich, wenn Sie alle Instanzen in einem Raum deaktivieren möchten, da ein physischer Körper auch bei Deaktivierung weiterhin in der Physikwelt berechnet und simuliert wird.

Syntax:

physics_pause_enable(flag)

Argument Beschreibung
Flagge Dies kann auf true gesetzt werden, um die Simulation anzuhalten, oder false, um sie wieder zu starten.

Rückgabe:

N/A

Beispiel:

if os_is_paused()
{
physics_pause_enable(true);
instance_deactivate_all(true);
instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

Der obige Code prüft, ob das Betriebssystem angehalten wurde, und wenn dies der Fall ist, hält er die Physikwelt an, deaktiviert alles außer sich selbst und erstellt dann eine Instanz des Pausenmenüs.
