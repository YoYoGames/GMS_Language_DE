physics_apply_angular_impulse

Diese Funktion gibt einen Winkelimpuls an eine physikalisch aktivierte Instanz. Dieser Impuls setzt die Winkeldrehung um den angegebenen Betrag, wobei das aktuelle Drehmoment ignoriert wird, wodurch im Wesentlichen der "Spin" einer Vorrichtung festgelegt wird. Wenn Sie eine Winkelkraft auf eine Instanz mit Drehmoment anwenden wollen, dann sollten Sie die Funktion physics_apply_torque().

Syntax:

physics_apply_angular_impulse(impulse)

Argument Beschreibung
impuls Der anzulegende Impuls (in Newtonmeter)

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_up)
{
physics_apply_angular_impulse(10);
}

Der obige Code gibt dem physikfähigen Gerät einen Winkelimpuls, wenn eine Taste gedrückt wird.
