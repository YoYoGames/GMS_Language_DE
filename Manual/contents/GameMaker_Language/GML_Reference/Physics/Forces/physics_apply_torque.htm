physics_apply_torque

Wenn wir von Drehmoment sprechen, meinen wir in der Regel die Tendenz einer Kraft oder eines Impulses, eine Instanz um ihren Massenschwerpunkt zu drehen, aber in GameMaker Studio 2 können Sie auch ein Drehmoment unabhängig von einer Instanz anwenden, wodurch sie sich in eine bestimmte Richtung drehen. Wie viel Drehmoment Sie aufbringen, um die gewünschte Drehung zu erreichen, hängt von der Größe und der Dichte des Fixtures ab, das Sie Ihrer Instanz zuweisen, da diese beiden Eigenschaften zur Definition der Masse beitragen, die wiederum die Trägheit definiert, die sich auf das Drehmoment! Als Faustregel gilt: Wenn eine Instanz eine große Trägheit hat, benötigen Sie ein großes Drehmoment, um sie in Drehung zu versetzen. Es sollte auch beachtet werden, dass andere Faktoren wie Schwerkraft und Kollisionen das Drehmoment einer Instanz ebenfalls beeinflussen können. Hier ist ein Bild zur Erläuterung:

Wie Sie sehen können, hängt die Höhe des Drehmoments, das Sie benötigen, um das Objekt zu drehen, von der zugrundeliegenden Masse und der Trägheit ab, die es aufweist. Dies alles wird von GameMaker Studio 2 für Sie berechnet, basierend auf der Größe und Dichte des Fixtures, das der Instanz beim Erstellen zugewiesen wird. Beachten Sie auch, dass für eine Drehung gegen den Uhrzeigersinn der Wert negativ sein sollte.

Syntax:

physics_apply_torque(Drehmoment)

Argument Beschreibung
Drehmoment Das aufzubringende Drehmoment (in Newtonmeter)

Rückgabe:

N/A

Beispiel:

if keyboard_check(vk_left)
{
physics_apply_torque(-100);
}

Der obige Code dreht die Instanz gegen den Uhrzeigersinn (nach links) mit einem Drehmoment von 100Nm, wenn die linke Pfeiltaste gedrückt wird.
