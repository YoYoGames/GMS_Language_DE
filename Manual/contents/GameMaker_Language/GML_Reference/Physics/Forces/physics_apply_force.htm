physics_apply_force

Wenn Sie mit Physik arbeiten, müssen Sie nicht nur die Schwerkraft auf Objekte anwenden, sondern auch Kräfte. Eine Kraft ist jeder Einfluss, der eine Geschwindigkeits-, Richtungs- oder Formänderung eines Objekts bewirkt, und wird in der GameMaker Studio 2-Physikwelt genauso definiert wie die Schwerkraft, indem man den Vektor aus den beiden xforce/yforce-Komponenten erhält. Es ist zu beachten, dass Kräfte nicht auf den Massenschwerpunkt des Objekts wirken, sondern auf einen Punkt im Raum und sie verändern die Richtung und Geschwindigkeit sowie das Drehmoment (Spin) in Abhängigkeit von anderen Kräften, die auf das Objekt wirken (z. B. Schwerkraft). Hier ist ein Beispielbild zur Veranschaulichung:

Im obigen Beispiel hat das Federobjekt unten eine Kraft auf die Kugel Objekt eine Kraft erzeugt, die es dazu bringt, sich in eine andere Richtung zu bewegen. Die genaue Kraft wird durch den Vektor definiert, den wir aus den Komponenten xforce/yforce in Bezug auf die Koordinaten xpos/ypos erhalten - was einfach bedeutet, dass die Kraft als der Abstand von xpos/ypos zu xforce/yforce in Newton (ein Newton ist definiert als die Nettokraft, die erforderlich ist, um eine Masse von einem Kilogramm mit einer Geschwindigkeit von einem Meter pro Sekunde im Quadrat zu beschleunigen) und die Richtung ist der Winkel, den wir von xpos/ypos zu xforce/yforce erhalten. Der Code für eine Kollision wie die obige würde in das Kollisionsereignis des Federobjekts gehen und würde in etwa so aussehen wie in dem unten stehenden Beispiel.

HINWEIS: Die Physikwelt verwendet Meter für alle Berechnungen, und das Verhältnis von Pixeln pro Meter wird im Raumeditor oder mit physics_world_create().

Syntax:

physics_apply_force(xpos, ypos, xforce, yforce)

Argument Beschreibung
xpos Die x-Koordinate im Raum, an der die Kraft angreifen soll
ypos Die y-Koordinate im Raum, an der die Kraft angreifen soll
xforce die x-Komponente des Kraftvektors
yforce die y-Komponente des Kraftvektors

Rückgabe:

N/A

Beispiel:

mit (andere)
{
physics_apply_force(x, y, 0, -30);
}

Der obige Code wird auf das andere Objekt in der Kollision mit einer Kraft von 30 Newton und einer Richtung angewendet.
