Kräfte

In der GameMaker Studio 2-Physikwelt werden Instanzen nicht nur von der Schwerkraft und von Kollisionen beeinflusst, sondern auch von den Kräften, die Sie auf sie anwenden, und das Wissen um die richtige Kraft ist sehr wichtig, wenn Sie eine Instanz innerhalb der Physiksimulation korrekt steuern wollen. Die Kenntnis der richtigen Kraft ist sehr wichtig, wenn Sie eine Instanz in der Physiksimulation korrekt steuern wollen. Es gibt zwei Kräfte, die Sie verwenden und auf Ihre Einbauten anwenden können:

  • Impulse: Ein Impuls setzt die Richtung und die Geschwindigkeit sofort, nachdem er angewendet wird, wodurch er ideal für Dinge wie Federn oder schießende Kugeln usw. ist.
  • Kräfte: Eine Kraft wirkt auf die vorhandene Geschwindigkeit und Richtung und ändert diese im Laufe der Zeit, wodurch sie ideal für Dinge wie Wind oder einen Magneten usw. ist.
  • Drehmoment: Das Drehmoment bestimmt die Wahrscheinlichkeit, dass sich eine Instanz weiterhin um ihre Achse dreht, nachdem eine Kraft aufgebracht wurde, und die Geschwindigkeit, bei der sie aufhört, sich zu drehen. Das Hinzufügen eines Drehmoments ist ein allmählicher Prozess, ähnlich wie das Hinzufügen einer Bewegung durch eine Kraft.
  • Winkelimpuls: Ein Winkelimpuls kann verwendet werden, um ein Gerät sofort in Drehung zu versetzen, ähnlich wie ein normaler Impuls verwendet werden kann, um ein Gerät sofort in Bewegung zu versetzen.

Die folgenden Funktionen existieren, um verschiedene Arten von Kräften auf physikfähige Instanzen anzuwenden, und jede Funktion gibt eine ausführlichere Erklärung, wie die Kraft angewandt wird, als die oben angegebenen Skizzen:

  • physics_apply_force
  • physics_apply_impulse
  • physics_apply_local_force
  • physics_apply_local_impulse
  • physics_apply_torque
  • physics_apply_angular_impulse
