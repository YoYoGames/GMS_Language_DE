phy_kinematisch

Diese schreibgeschützte Variable gibt true zurück, wenn die Instanz als kinematisches Objekt klassifiziert ist, oder false, wenn sie es nicht ist. Eine kinematische Instanz ist eine Instanz mit unendlicher Masse (einer Dichte von 0), die sich jedoch bewegen kann. Um also eine Instanz kinematisch zu machen, erstellen Sie zunächst eine statische Instanz und setzen dann eine oder mehrere der Instanzvariablen, die sich auf die Bewegung beziehen (z. B: phy_Geschwindigkeit_x, phy_Geschwindigkeit_y, oder phy_angular_velocity)

Syntax:

phy_kinematic;

Rückgabe:

Boolean (oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if !phy_kinematic
{
phy_speed_x = 5;
}

Der obige Code prüft, ob die Instanz kinematisch ist, und wenn nicht, setzt er die horizontale Geschwindigkeit auf 5.
