phy_fixed_rotation

Mit dieser Variablen kann eingestellt werden, ob die Instanz von Rotationskräften beeinflusst werden kann oder nicht (Standard ist false). Wenn dies auf true gesetzt ist, wirkt keine externe Kraft (weder von kodierten Impulsen oder Kräften, noch von Kollisionen) auf den den Rotationswert der Instanz und dieser müsste manuell mit der phy_rotation Variable

Syntax:

phy_fixed_rotation;

Rückgabe:

Boolean (oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

phy_rotation = 0;
phy_fixed_rotation = true;

Der obige Code schaltet die Instanz so, dass sie eine feste Drehung hat, und setzt dann den Drehwinkel.
