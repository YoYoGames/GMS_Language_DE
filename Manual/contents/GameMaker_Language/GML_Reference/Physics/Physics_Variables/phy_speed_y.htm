phy_speed_y

Diese Variable kann verwendet werden, um die y-Komponente des linearen Geschwindigkeitsvektors der Instanz zu erhalten oder zu ändern, und wird in Pixeln pro Schritt definiert (für Pixel pro Sekunde, siehe phy_linear_velocity_y). Das Ändern dieser für eine statische Instanz (d. h. eine Instanz mit 0 Dichte) wird sie zu einer kinematischen Instanz.

Syntax:

phy_speed_y;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if phy_speed_y != 0
{
phy_speed_y = 0;
}

Der obige Code prüft die y-Komponente des linearen Geschwindigkeitsvektors und setzt sie auf 0, wenn sie einen anderen Wert als 0 hat.
