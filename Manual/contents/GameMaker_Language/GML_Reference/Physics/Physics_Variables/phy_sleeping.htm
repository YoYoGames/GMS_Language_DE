phy_schlafend

Diese schreibgeschützte Variable gibt zurück, ob die Instanz gerade "schläft" (true) odernicht (false). Eine"schlafende" Instanz ist eine, die nicht aktiv an einer physikalischen Simulation beteiligt ist. GameMaker Studio 2 versetzt Objekte in den Ruhezustand, um Simulationszyklen zu sparen, wenn eine Instanz in Ruhe ist und nicht mit einer anderen Instanz kollidiert

Syntax:

phy_sleeping;

Rückgabe:

Boolean (oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if phy_sleeping
{
instance_destroy();
}

Der obige Code prüft, ob das Objekt aktiv simuliert wird oder nicht, und wenn nicht, wird es zerstört.
