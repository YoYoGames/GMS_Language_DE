phy_angular_damping

Diese Variable kann verwendet werden, um die Winkeldämpfung der Instanz einzustellen, oder sie kann verwendet werden, um die aktuelle Winkeldämpfung zu erhalten. Die Dämpfung ist der "Widerstand", den die physikalisch aktivierte Instanz gegenüber einer Winkelrotation hat, wobei ein niedriger Wert Bei einem niedrigeren Wert kann sich die Instanz nach einer Kollision (z. B.) schneller drehen, bei einem höheren Wert benötigt sie einen kräftigeren Stoß und dreht sich langsamer.

Syntax:

phy_angular_damping;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_angular_damping = 10;
}
sonst
{
phy_angular_damping = 3;
}

Der obige Code prüft auf eine Kollision zwischen der aufrufenden Instanz und Instanzen von "obj_Water" und ändert die Winkeldämpfung entsprechend.
