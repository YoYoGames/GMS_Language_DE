phy_position_xprevious

Diese Variable kann verwendet werden, um die vorherige X-Position der Instanz innerhalb der Spielraum-Physikwelt zu erhalten (oder zu setzen). Dies ist die Position der Instanz innerhalb der Physikwelt im vorherigen Schritt zum aktuellen Schritt.

Syntax:

phy_position_xprevious;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

xx = phy_position_xprevious;
yy = phy_position_yprevious;

Der obige Code speichert die vorherige x- und y-Position für die physikaktivierte Instanz in zwei Variablen.
