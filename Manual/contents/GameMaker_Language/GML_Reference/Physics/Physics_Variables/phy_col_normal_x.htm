phy_col_normal_x

Diese schreibgeschützte Variable gibt die x-Komponente der Kollisionsnormale zurück, die dem phy_kollision_x Array-Wert entspricht. Für jeden Kontaktpunkt gibt es eine zugehörige Kontaktnormale (die normalerweise die gleiche Normale für alle Kontaktpunkte in der Kollision). Diese Kontaktnormale ist ein Einheitsvektor, der von einer Instanz in der Kollision zu einer anderen zeigt, und kann z. B. verwendet werden, um die korrekte "Push"-Richtung zu berechnen, um die Kollisionen

HINWEIS: Diese Variable ist nur im Kollisionsereignis einer physikalisch aktivierten Instanz verfügbar.

Syntax:

phy_col_normal_x;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

repeat(5 + irandom(5))
{
mit (instance_create_layer(x, y, "Effects", obj_Debris)
{
physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
}
}

Der obige Code verwendet die Kontaktnormale, um die Bewegungsrichtung für eine Instanz festzulegen, die im Kollisionsereignis zwischen zwei physikfähigen Instanzen erstellt wurde.
