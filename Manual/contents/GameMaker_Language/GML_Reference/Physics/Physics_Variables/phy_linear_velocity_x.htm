phy_linear_velocity_x

Diese Variable kann verwendet werden, um die x-Komponente des linearen Geschwindigkeitsvektors der Instanz zu erhalten oder zu ändern, und ist in Pixeln pro Sekunde definiert (für Pixel pro Schritt, siehe phy_Geschwindigkeit_x). Das Ändern dieser Variable für eine statische Instanz (d. h. eine Instanz mit einer Dichte von 0), wird sie zu einer kinematischen Instanz

Syntax:

phy_linear_velocity_x;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if (phy_linear_velocity_x != 0)
{
phy_linear_velocity_x = 0;
}

Der obige Code prüft die x-Komponente des linearen Geschwindigkeitsvektors und setzt sie auf 0, wenn sie einen anderen Wert als 0 hat.
