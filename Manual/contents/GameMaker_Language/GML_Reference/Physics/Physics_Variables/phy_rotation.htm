phy_rotation

Diese Variable kann verwendet werden, um den Winkel des Instanzen-Fixtures in Grad zu erhalten (oder zu setzen), ähnlich wie das Setzen oder Erhalten der Bild_Winkel. Beachten Sie jedoch, dass in der Physikwelt Drehungen auf die entgegengesetzte Weise berechnet werden wie in der normalen GameMaker Studio 2-Spielwelt, was bedeutet, dass Vektor-Funktionen wie punkt_richtung()sollten ihre Rückgabewerte ändern müssen (einfach von positiv auf negativ ändern sollte dies beheben).

Syntax:

phy_rotation;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

Der obige Code prüft die lineare Geschwindigkeit und wenn einer der beiden Vektoren nicht 0 ist, berechnet er die tatsächliche Geschwindigkeit und verwendet diese, um die Rotation einzustellen.
