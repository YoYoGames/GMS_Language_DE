phy_aktiv

Diese Variable steuert, ob die Instanz gerade "aktiv" ist oder nicht. Wenn sie auf false gesetzt wird, nimmt die Instanz nicht mehr an der Physikwelt teil, und wenn sie auf true gesetzt wird, nimmt sie wieder teil. Bitte Bitte beachten Sie, dass dies nicht dasselbe ist wie das Deaktivieren der Instanz, da die Instanz immer noch auf dem Bildschirm sichtbar ist und immer noch durch Code verändert werden kann, sondern diese Funktion verhindert nur, dass sie an der Physiksimulation teilnimmt

Syntax:

phy_active;

Rückgabe:

Boolean (oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if keyboard_check_pressed(ord "P")
{
global.Pause = !global.Pause
with (obj_Parent)
{
phy_active = !global.Pause;
}
}

Der obige Code erkennt einen Tastendruck des Buchstaben "P" und schaltet dann die globale Variable "Pause" von true auf false und wieder zurück . Diese Variable wird dann verwendet, um in den Kindinstanzen des in der Variable "obj_Parent" indizierten Objekts festzulegen, ob die Physik aktiv ist oder nicht. Instanzen des Objekts, das in der Variablen "obj_Parent" indiziert ist.
