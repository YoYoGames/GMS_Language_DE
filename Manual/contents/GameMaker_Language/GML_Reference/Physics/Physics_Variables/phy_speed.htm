phy_Drehzahl

Diese schreibgeschützte Variable gibt die aktuelle Geschwindigkeit der physikalisch aktivierten Instanz zurück, definiert in Pixel pro Schritt. Sollten Sie diesen Wert ändern wollen, müssen Sie dies tun, indem Sie die x- und y-Vektoren mit den Variablen phy_Geschwindigkeit_x und phy_speed_y.

Syntax:

phy_speed;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if phy_speed > 10
{
phy_linear_damping += 0,01;
}
sonst
{
phy_linear_damping = 2;
}

Der obige Code prüft die Geschwindigkeit der physikalisch aktivierten Instanz und ändert dann die lineare Dämpfung basierend auf dem zurückgegebenen Wert.
