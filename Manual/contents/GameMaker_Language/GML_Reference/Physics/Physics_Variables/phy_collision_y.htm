phy_kollision_y

Dieses schreibgeschützte Array gibt die y-Position aller Punkte zurück, die bei einer Kollision zwischen zwei physikfähigen Instanzen erkannt wurden.

HINWEIS: Diese Variable ist nur im Kollisionsereignis einer physikalisch aktivierten Instanz verfügbar.

Syntax:

phy_kollision_y;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

for(var i = 0; i < phy_collision_points; i += 1;)
{
part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);
}

Der obige Code erzeugt Partikel an allen definierten Punkten einer Kollision zwischen zwei physikfähigen Instanzen.
