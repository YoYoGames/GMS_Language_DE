phy_mass

Diese schreibgeschützte Variable gibt die Masse der Instanz in Kilogramm zurück. Dieser Wert wird automatisch auf Basis der Oberfläche der zugewiesenen Fixtures und ihrer Dichtewerte berechnet, kann aber mit der Funktion physics_mass_properties()geändert werden.

Syntax:

phy_mass;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if (phy_mass < other.phy_mass)
{
instance_destroy();
}

Der obige Code stammt aus dem Kollisionsereignis der Instanz mit einer anderen und vergleicht die Masse der einzelnen Instanzen und zerstört diejenige, die weniger Masse hat.
