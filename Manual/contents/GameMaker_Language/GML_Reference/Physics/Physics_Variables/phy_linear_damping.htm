phy_linear_dämpfung

Diese Variable kann verwendet werden, um die lineare Dämpfung der Instanz einzustellen, oder sie kann verwendet werden, um die aktuelle lineare Dämpfung zu erhalten. Die Dämpfung ist der "Widerstand", den die physikalisch aktivierte Instanz der Vorwärtsbewegung entgegensetzt, wobei ein niedriger Wert Bei einem niedrigeren Wert kann sich die Instanz schneller bewegen und beschleunigen, während sie bei einem höheren Wert einen stärkeren Schub benötigt.

Syntax:

phy_linear_damping;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_linear_damping = 10;
}
sonst
{
phy_linear_damping = 3;
}

Der obige Code prüft auf eine Kollision zwischen der aufrufenden Instanz und Instanzen von "obj_Water" und ändert die lineare Dämpfung entsprechend.
