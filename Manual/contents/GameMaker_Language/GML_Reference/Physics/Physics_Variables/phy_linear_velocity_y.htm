phy_linear_velocity_y

Diese Variable kann verwendet werden, um die y-Komponente des linearen Geschwindigkeitsvektors der Instanz abzurufen oder zu ändern, und ist in Pixel pro Sekunde definiert (für Pixel pro Schritt, siehe phy_Geschwindigkeit_y). Das Ändern dieser Variable für eine statische Instanz (d. h. eine Instanz mit einer Dichte von 0), wird sie zu einer kinematischen Instanz

Syntax:

phy_linear_velocity_y;

Rückgabe:

Real (Fließkommawert mit einfacher Genauigkeit, oder undefiniert, wenn die Instanz nicht physikaktiviert ist)

Beispiel:

if (phy_linear_velocity_y != 0)
{
phy_linear_velocity_y = 0;
}

Der obige Code prüft die y-Komponente des linearen Geschwindigkeitsvektors und setzt sie auf 0, wenn sie einen anderen Wert als 0 hat.
