physics_particle_get_density

Mit dieser Funktion können Sie herausfinden, wie hoch die Stromdichte für Partikel in der Physiksimulation ist (Sie können diesen Wert mit physics_particle_set_density()).

Syntax:

physics_particle_get_damping()

Rückgabe:

Real

Beispiel:

if (physics_particle_get_density() < 1)
{
physics_particle_set_density(physics_particle_get_density() + 0.01);
}

Der obige Code prüft den aktuellen Dichtewert für alle Partikel im System, und wenn er kleiner als 1 ist, wird er 0,01 dazu addieren.
