physics_particle_group_delete

Mit dieser Funktion können Sie eine Partikelgruppe aus der Physiksimulation im aktuellen Raum löschen (entfernen). Die Funktion nimmt die eindeutige Gruppen-ID der zu löschenden Gruppe, wie sie von der Funktion physics_particle_group_end().

Syntax:

physics_particle_group_delete(ind)

Argument Beschreibung
ind Der Index (ID) der zu löschenden Partikelgruppe.

Rückgabe:

N/A

Beispiel:

physics_particle_group_delete(gp1);

Der obige Code löscht alle Partikel, die die Gruppe mit der in der Variablen "gp1" gespeicherten ID bilden, aus der Simulation.
