physics_particle_set_category_flags

Mit dieser Funktion können Sie die Partikel-Flags für eine Kategorie von Partikeln ändern. Der Wert der Kategorie ist derjenige, den Sie mit der Funktion physik_teilchen_erzeugen() oder (wenn Sie eine Gruppe) physics_particle_group_begin(), während die Flags der Rückgabewert einer Kombination der folgenden Konstanten sind:

Konstant Beschreibung
phy_particle_flag_water Die Standardeigenschaften für einen Weichkörperpartikel.
phy_particle_flag_zombie Ein Zombie-Partikel ist ein Partikel, der in einem einzigen Schritt zerstört wird, während alle anderen auf diese Weise gekennzeichnet sind.
phy_particle_flag_wall Damit wird das Partikel als statisch definiert, was es im Wesentlichen zu einem unbeweglichen Objekt in der Physiksimulation macht, da es in einer festen Position bleibt, egal was mit ihm kollidiert. Sie sollten dieses Flag verwenden, anstatt die Dichte auf 0 zu setzen.
phy_particle_flag_spring Federpartikel erzeugen den Effekt, als seien sie wie durch eine Feder miteinander verbunden. Teilchen, die mit dieser Flagge erstellt werden, sind paarweise "verbunden", wobei jedes Teilchen mit demjenigen verbunden ist, das ihm zum Zeitpunkt der Erstellung am nächsten war. Einmal gepaart, wechseln die Teilchen ihre "Partner" nicht mehr, und je weiter eine äußere Kraft sie voneinander wegzieht, desto größer ist die Kraft, mit der sie kollidieren, wenn diese äußere Kraft entfernt wird. Beachten Sie, dass, egal wie weit gepaart Teilchen voneinander entfernt sind, die Verbindung zwischen ihnen nicht reißen wird.
phy_particle_flag_elastic Elastische Partikel verformen sich und können auch abprallen, wenn sie mit anderen starren Körpern in der Physiksimulation zusammenstoßen.
phy_particle_flag_viscous Ein viskoser Partikel ist ein Partikel, der eine "Klebrigkeit" oder "Klebrigkeit" aufweist, wie Öl. Viskose Partikel verklumpen und kleben stärker zusammen.
phy_particle_flag_powder Pulverpartikel erzeugen einen Streueffekt, wie man ihn bei Sand oder Staub sehen kann.
phy_particle_flag_tensile Zugpartikel werden verwendet, um den Effekt der Oberflächenspannung oder die gespannte Krümmung auf der Oberfläche eines Flüssigkeitskörpers zu erzeugen. Sie können z. B. verwendet werden, um die Oberflächenspannung zu erzeugen, die Sie an einem Wassertropfen sehen würden. Sobald die Spannung Sobald die Spannung gebrochen ist, prallen die Teilchen ab, als ob sie elastisch wären, ziehen sich aber auch weiterhin gegenseitig an. Infolgedessen neigen die Teilchen dazu, Cluster zu bilden, während sie hüpfen.
phy_particle_flag_colourmixing Farbmischende Partikel nehmen einen Teil der Farbe anderer Partikel an, mit denen sie kollidieren. Beachten Sie, dass, wenn nur eines der beiden kollidierenden Teilchen ein farbmischendes Teilchen ist, das andere Teilchen seine Farbe vor der Kollision beibehält.

Diese Flags verwenden Bit-Maskierung, um einen endgültigen Ausgabewert zu erzeugen, der dann geprüft wird, um die verschiedenen Basiseigenschaften der Partikelkategorie einzustellen (wobei die Basiseigenschaft immer die von phy_particle_flag_water ist). Zum Beispiel, wenn Sie eine viskose Flüssigkeit mit Oberflächenspannung simulieren möchten, würden Sie die entsprechenden Bits bitweise oder mit "|" ausblenden, wie im folgenden Beispiel gezeigt. Auf diese Weise können Sie können Sie auf diese Weise für jede erstellte Partikelkategorie unterschiedliche Eigenschaften (mit Ausnahme der globalen Eigenschaften) festlegen.

Syntax:

physics_particle_set_category_flags(category, flags)

Argument Beschreibung
Kategorie Die einzustellende Partikelkategorie.
Flaggen Die Flags, die auf dem Partikel gesetzt werden sollen.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_category_flags(1, flags);

Der obige Code erstellt eine Variable zum Speichern des Flags-Wertes und verwendet sie dann zum Setzen der Flags aller Partikel, die mit dem Kategoriewert 1 erstellt wurden.
