physics_particle_set_max_count

Mit dieser Funktion wird die zulässige Gesamtzahl von Partikeln in einer Physiksimulation festgelegt. Wenn Sie diesen Wert auf 0 setzen, gibt es keine Obergrenze für die erzeugten Partikel, und wenn Sie ihn auf einen anderen Wert setzen, werden keine weiteren Partikel mehr erzeugt, wenn die Gesamtzahl der Partikel gleich dem Wert ist. Beachten Sie, dass die minimale Anzahl von Partikeln, die Sie in einer Simulation erstellen können, 128 beträgt, so dass das Einstellen dieses Werts auf einen niedrigeren Wert keine Auswirkungen hat (es sei denn, er wird auf 0 gesetzt).

Syntax:

physics_particle_set_max_count(count)

Argument Beschreibung
zählen Die maximale Anzahl der zuzulassenden Partikel.

Rückgabe:

N/A

Beispiel:

physics_particle_set_max_count(500);

Mit dem obigen Code wird die Partikelobergrenze der Physiksimulation auf 500 gesetzt.
