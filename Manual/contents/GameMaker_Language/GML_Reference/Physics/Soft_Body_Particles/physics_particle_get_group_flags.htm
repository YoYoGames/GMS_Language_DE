physics_particle_get_group_flags

Mit dieser Funktion können Sie die Gruppenflags für eine Gruppe von Partikeln abrufen. Der Gruppenwert ist der, der zurückgegeben wurde, als Sie die Gruppe von Partikeln mit der Funktion physics_particle_group_end(), und die Funktion gibt einen Wert zurück, der dem kombinierten Wert der aktuell gesetzten Flags entspricht.

Syntax:

physics_particle_get_group_flags(group)

Argument Beschreibung
Gruppe Die zu erhaltende Partikelgruppe.

Rückgabe:

Real

Beispiel:

var flags = physics_particle_get_group_flags(group, flags);
if flags != phy_particle_group_flag_solid | phy_particle_group_flag_rigid
{
flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);
}

Der obige Code erstellt eine Variable, um den Wert der Flags zu speichern, und verwendet diese dann, um die Flags der in der Variablen "group" indizierten Gruppe zu prüfen. Wenn sie nicht übereinstimmen, wird die Gruppe mit diesen Flags gesetzt.
