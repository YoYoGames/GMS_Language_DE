physics_particle_group_get_vel_x

Mit dieser Funktion können Sie die horizontale Geschwindigkeit einer Gruppe von Partikeln abrufen. Der Gruppenwert ist derjenige, der zurückgegeben wurde, als Sie die Gruppe von Partikeln mit der Funktion physics_particle_group_end(), und die Funktion gibt einen Wert zurück, der dem kombinierten Wert der aktuell gesetzten Flags entspricht.

Syntax:

physics_particle_group_get_vel_x(group)

Argument Beschreibung
Gruppe Die zu erhaltende Partikelgruppe.

Rückgabe:

Real

Beispiel:

vx = physics_particle_group_get_vel_x(group1);
vy = physics_particle_group_get_vel_y(group1);

Der obige Code ermittelt die horizontale und vertikale Geschwindigkeit der in der Variablen "group1" indizierten Partikelgruppe und speichert sie in Variablen.
