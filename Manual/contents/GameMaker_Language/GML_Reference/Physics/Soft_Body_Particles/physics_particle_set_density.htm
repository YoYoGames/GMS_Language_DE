physics_particle_set_density

Mit dieser Funktion können Sie die Dichte der Partikel in einer Physiksimulation einstellen. Das Einstellen der Dichte der Partikel hat einen direkten Einfluss darauf, wie viel Trägheit sie haben und wie sie auf Kollisionen reagieren. Wenn Sie also ein kleines Teilchen mit einer hohen Dichte definieren, wird es eine sehr große Masse haben, aber wenn Sie ein großes Teilchen mit einer niedrigen Dichte definieren, wird es eine viel kleinere Masse haben. Diese Funktion ist global, d. h. sie ändert die Dichte nicht nur für neue Partikel, die nach der der Änderung erzeugt werden, sondern auch für diejenigen, die bereits in der Simulation vorhanden sind.

Syntax:

physics_particle_set_density(Dichte)

Argument Beschreibung
Dichte Die Dichte der Partikelbefestigung.

Rückgabe:

N/A

Beispiel:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Der obige Code setzt die Basiseigenschaften für alle Partikel in der Simulation.
