physics_particle_group_end

Diese Funktion wird verwendet, um die Definition einer Partikelgruppenform zu beenden. Der Aufruf dieser Funktion erzeugt die Partikel innerhalb der angegebenen Formparameter und gibt außerdem einen ID-Wert zurück, der gespeichert und in weiteren Funktionen für Interaktionen mit der Partikelgruppe verwendet werden kann.

Syntax:

physics_particle_group_end()

Rückgabe:

Real

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Der obige Code speichert die Flags für den Partikeltyp und die Eigenschaften der Partikelgruppe in Variablen und verwendet diese dann, um eine kreisförmige Partikelgruppe mit einem Radius von 100px an der Mausposition zu erstellen. Die ID für die erstellte Gruppe wird in der Variablen in der Variable "mLastGroup" gespeichert.
