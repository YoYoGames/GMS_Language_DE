physics_particle_set_damping

Mit dieser Funktion können Sie die lineare Dämpfung von Partikeln in der Simulation einstellen. Die Dämpfung wird verwendet, um die Geschwindigkeit der Instanzen in der Physiksimulation im Laufe der Zeit zu verringern, ähnlich wie der Luftwiderstand in der realen Welt. Diese Funktion ist global gültig, da sie die Dämpfung nicht nur für neue Partikel ändert, die nach der Änderung erzeugt werden, sondern auch für solche, die bereits in der Simulation vorhanden sind.

Syntax:

physics_particle_set_damping(damping)

Argument Beschreibung
Dämpfung Die lineare Dämpfung, die auf die Partikelhalterung angewendet werden soll.

Rückgabe:

N/A

Beispiel:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Der obige Code setzt die Basiseigenschaften für alle Partikel in der Simulation.
