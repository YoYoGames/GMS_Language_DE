physik_teilchen_holen_radius

Mit dieser Funktion können Sie herausfinden, wie groß der aktuelle Radius (in Pixeln) für Partikel in der Physiksimulation ist (Sie können diesen Wert mit physics_particle_set_radius()).

Syntax:

physics_particle_get_radius()

Rückgabe:

Real

Beispiel:

if (physics_particle_get_radius() < 32)
{
physics_particle_set_radius(physics_particle_get_radius() + 1);
}

Der obige Code prüft den aktuellen Radius der Partikel in der Simulation und wenn dieser kleiner als 32 ist, dann wird er ihre Größe um 1 erhöhen.
