physics_particle_group_circle

Mit dieser Funktion wird die Form der zu erstellenden Partikelgruppe festgelegt. Sie müssen zuerst die Gruppendefinition mit der Funktion physics_particle_group_begin()begonnen haben, und dann würden Sie mit dieser Funktion den ungefähren Radius der Gruppe in Pixeln einstellen - ungefähr, weil der genaue Radius von der Größe der Basispartikel abhängt, wie sie mit der Funktion physics_particle_set_radius() Funktion definiert wurde, da die Physiksimulation versucht, so viele Partikel wie möglich in die definierte Form "einzupassen". Schließlich müssen Sie die Funktion physics_particle_group_end() aufrufen, um die Gruppe von Partikeln im Raum zu erstellen.

Syntax:

physics_particle_group_circle(radius)

Argument Beschreibung
Radius Der Radius (in Pixel) des zu erstellenden Kreises.

Rückgabe:

N/A

Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Der obige Code speichert die Flags für den Partikeltyp und die Eigenschaften der Partikelgruppe in Variablen und verwendet diese dann, um eine kreisförmige Partikelgruppe mit einem Radius von 100px an der Mausposition zu erstellen.
