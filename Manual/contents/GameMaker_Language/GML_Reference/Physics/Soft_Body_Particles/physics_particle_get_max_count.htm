physics_particle_get_max_count

Mit dieser Funktion können Sie herausfinden, wie hoch die aktuelle Obergrenze für die in der Physiksimulation erlaubten Partikel ist (Sie können diesen Wert mit physics_particle_set_max_count()).

Syntax:

physics_particle_get_max_count()

Rückgabe:

Real

Beispiel:

if (physics_particle_count() < physics_particle_get_max_count())
{
physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

Der obige Code prüft, ob weniger als die maximal zulässige Anzahl von Partikeln im Raum vorhanden ist, und erzeugt in diesem Fall ein weiteres.
