Weiche Körperpartikel

GameMaker Studio 2 verwendet die Starrkörper-Physikbibliothek Box2D und erlaubt daher keine Weichkörperphysik. Allerdings fügt die Flüssiger Spaß Erweiterung fügt jedoch Partikel zum dem Physiksystem Partikel hinzu, die die Erzeugung von simulierten Weichkörpern unter Verwendung der Starrkörperphysik erlauben. Grundsätzlich können Sie eine Form aus Partikeln erzeugen, die den Zusammenhalt beibehalten und so einen vollen Körper simulieren, der aufgrund der Interaktion der Partikel den Eindruck eines Weichkörpers vermittelt. Sie können die Physikpartikel auch zur Simulation von Flüssigkeiten und anderen partikelbasierten Physiksystemen verwenden.

Bevor Sie mit der Erstellung Ihrer Physikpartikel beginnen, müssen Sie die grundlegenden Physikeigenschaften für diese Partikel einstellen, so wie Sie es mit einem Einbauteil in der regulären Physik tun würden. Beachten Sie jedoch, dass sich das Einstellen dieser Eigenschaften auf alle zuvor erstellten Partikel sowie auf alle neu erstellten Partikel auswirkt. Beachten Sie jedoch, dass sich das Einstellen dieser Eigenschaften auf alle zuvor erstellten Partikel sowie auf alle neu erstellten Partikel auswirkt, so dass Sie diese Funktionen normalerweise nur zu Beginn des Spiels oder Levels verwenden. Es gibt jedoch spezielle Funktionen zum Setzen der Partikel-Flags für einzelne Partikel (und Gruppen von Partikeln) zu setzen, die ihre Eigenschaften wie "Abprallverhalten" oder "Viskosität" ändern, so dass Sie verschiedene Arten von Partikeln innerhalb der Bereiche dieser globalen Einstellungen erstellen können.

Die folgenden Funktionen stehen zum Einstellen und Abrufen von globalen Partikelsystemdaten zur Verfügung:

  • physik_teilchen_zahl
  • physik_teilchen_gruppen_zahl
  • physics_particle_set_max_count
  • physik_teilchen_setzen_radius
  • physics_particle_set_density
  • physics_particle_set_damping
  • physics_particle_set_gravity_scale
  • physics_particle_get_max_count
  • physik_teilchen_holen_radius
  • physics_particle_get_density
  • physics_particle_get_damping
  • physics_particle_get_gravity_scale

Außerdem stehen Ihnen die folgenden Funktionen zur Verfügung, um Flags für einzelne Partikel (oder Gruppen von Partikeln) zu setzen, sowie um Informationen, wie Position oder Farbe, von einem einzelnen Partikel oder einer einzelnen Gruppe zu erhalten:

  • physik_teilchen_setzen_flags
  • physics_particle_set_category_flags
  • physics_particle_set_group_flags
  • physics_particle_get_group_flags
  • physics_particle_group_get_inertia
  • physics_particle_group_get_centre_x
  • physics_particle_group_get_centre_y
  • physics_particle_group_get_vel_x
  • physics_particle_group_get_vel_y
  • physics_particle_group_get_ang_vel
  • physics_particle_group_get_x
  • physics_particle_group_get_y
  • physics_particle_group_get_angle
  • physics_particle_get_data
  • physics_particle_get_data_particle
  • physics_particle_group_get_data

Sie können Partikel auf zwei Arten erstellen - einzeln oder in Gruppen. Einzelne Partikel verhalten sich genau so, wie es durch die globalen Einstellungen und die bei ihrer Erstellung gesetzten Flags definiert ist, aber Gruppen von Partikeln können ein unterschiedliches Verhalten zeigen abhängig von ihren Gruppen-Flags. Diese Gruppen-Flags können auf unterschiedliche Werte gesetzt werden, um der Gruppe unterschiedliche Eigenschaften zu verleihen, wie z. B. Penetration und Steifigkeit.

Zum Erstellen von Partikeln stehen die folgenden Funktionen zur Verfügung:

  • physik_teilchen_erzeugen
  • physics_particle_group_begin
  • physics_particle_group_circle
  • physik_teilchen_gruppe_box
  • physics_particle_group_polygon
  • physics_particle_group_add_point
  • physics_particle_group_end
  • physics_particle_group_join
  • physics_particle_group_delete

Sie können nicht nur Partikel erstellen, sondern auch löschen, entweder einzeln oder in einer Region. Zum Löschen von Partikeln aus dem System stehen die folgenden Funktionen zur Verfügung:

  • physik_teilchen_loeschen
  • physics_particle_delete_region_circle
  • physics_particle_delete_region_box
  • physics_particle_delete_region_poly

Sofern Sie nicht das Debug-Zeichnen mit physics_draw_debug()aktiviert haben, werden Sie keine Partikel auf dem Bildschirm sehen, wenn Sie sie erzeugen. Um die Partikel also tatsächlich zu sehen Partikel zu sehen, müssen Sie sie daher im Draw-Event einer Instanz mit den folgenden Funktionen auf den Bildschirm zeichnen (beachten Sie, dass Sie sie auch mit den regulären draw_sprite-Funktionen zeichnen können zeichnen können, wenn Sie die Daten für jeden Partikel mit der Funktion physics_particle_get_data()):

  • physics_particle_draw
  • physics_particle_draw_ext
