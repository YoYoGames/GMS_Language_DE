physics_particle_get_damping

Mit dieser Funktion können Sie herausfinden, wie hoch die aktuelle lineare Dämpfung für Partikel in der Physiksimulation ist (Sie können diesen Wert mit physics_particle_set_damping()).

Syntax:

physics_particle_get_damping()

Rückgabe:

Real

Beispiel:

if (physics_particle_get_damping() < 1)
{
physics_particle_set_damping(physics_particle_get_damping() + 0.01);
}

Der obige Code prüft den aktuellen Dämpfungswert für alle Partikel im System, und wenn er kleiner als 1 ist, wird er 0,01 dazu addieren.
