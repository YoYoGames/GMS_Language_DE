Physik_Gelenk_aktivieren_Motor

Wenn Sie ein Gelenk mit einem Motor(prismatisch oder drehbar) haben, möchten Sie vielleicht in der Lage sein, den Motor ein- oder auszuschalten, abhängig von Variablen und Bedingungen innerhalb des Spiels. Dazu müssen Sie den Index des Gelenks zuvor in einer Variablen gespeichert haben und können dann den Motor ein- oder ausschalten, indem Sie diese Funktion verwenden und das Argument "motor" auf true oder false setzen .

Syntax:

physics_joint_enable_motor(joint, motor)

Argument Beschreibung
gemeinsame Das Gelenk, an dem Sie den Motor aktivieren oder deaktivieren möchten
Motor Ob Sie den Motor ein- (true) oder ausschalten (false) möchten

Rückgabe:

N/A

Beispiel:

var t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, sprite_get_width(sprite_index) / 2);
var o_id=instance_create_layer(x+300, y, "Instances", obj_Door);
physics_fixture_bind(t_fix, id);
physics_fixture_bind(t_fix, o_id);
perma_joint = physics_joint_revolute_create(id, o_id, x+25, y, -90, 90, 1, 10, 2, 0, 0);
physics_joint_enable(perma_joint, 1);
physics_fixture_delete(t_fix);

Der obige Code erstellt und definiert ein neues Fixture und erstellt dann eine Instanz von "obj_Door" und bindet das erstellte Fixture an die beiden Instanzen. Die beiden Instanzen werden dann durch ein Drehgelenk ohne Motor verbunden, wobei die Winkel auf +/- 90 Grad begrenzt sind. und wir speichern den Gelenkindex in der Variablen "perma_joint". Anschließend schalten wir den Motor über diese Variable ein, bevor wir die Halterung schließlich aus dem Speicher löschen, da sie nicht mehr benötigt wird.
