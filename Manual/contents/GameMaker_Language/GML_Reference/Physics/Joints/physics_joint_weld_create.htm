physik_verbindung_schweißen_erzeugen

Die Schweißverbindung wurde entwickelt, um zwei Vorrichtungen in einer starken, aber flexiblen Verbindung miteinander zu verbinden. Die Schweißverbindung erlaubt eine Biegung zwischen den beiden verbundenen Vorrichtungen, jedoch ohne die Dehnung, die z. B. bei einer Abstandsverbindung auftritt, und wird immer versuchen Sie wird immer versuchen, in den Bezugswinkel zurückzuspringen, wenn sie unter Spannung oder Last steht. Sie definieren den Punkt im Raum, an dem die Verbindung erzeugt werden soll, sowie den Winkel, den die Verbindung stets beizubehalten versuchen soll, wie in Bild unten gezeigt:

Wie Sie sehen können, werden die Ankerpunkte als Raumkoordinaten angegeben, so dass bei ihrer Definition Vorsicht geboten ist Besonders dann, wenn die Instanzen gleichzeitig mit den Joints erstellt werden und nicht über den Raumeditor im Raum platziert werden. Sie sollten auch wissen, dass die Joints unabhängig von der Größe des Sprites der Instanzen oder der Einbauten, die sie angehängt haben, erstellt werden. Wenn Sie also eine Schweißnaht an einer anderen Stelle als dem Ursprung der Instanz erstellen, ist sie immer noch gültig und schränkt die beiden Instanzen relativ zu der Position ein, an der sie erstellt wurde.

Da die Schweißnaht flexibel ist und sich bei Belastung verbiegt und durchbiegt, können Sie die Schwingungsfrequenz (in Hz) sowie das Dämpfungsverhältnis für die Naht einstellen, um verschiedene Effekte zu erzielen - möglicherweise müssen Sie mit diesen Werten spielen, um sie fein abzustimmen Es wird empfohlen, mit kleineren Werten zu beginnen und diese zu erhöhen, bis Sie den gewünschten Effekt erzielen.

Wenn Sie den "col"-Wert auf "true" setzen, können die beiden Instanzen miteinander interagieren und kollidieren, aber nur, wenn sie Kollisionsereignisse haben; wenn er jedoch auf "false" gesetzt ist, werden sie nicht kollidieren, egal was passiert.

Syntax:

physics_joint_weld_create(inst1, inst2, anchor_x, anchor_y, ref_angle, freq_hz, damping_ratio, col)

Argument Beschreibung
inst1 Die erste Instanz zum Verbinden mit dem Gelenk
inst2 Die zweite Instanz zum Verbinden mit dem Gelenk
Anker_x Die x-Koordinate für das Gelenk, innerhalb der Spielwelt
Anker_y Die y-Koordinate für das Gelenk, innerhalb der Spielwelt
ref_angle Der zu erhaltende Gelenkwinkel
freq_hz Dies ist die Schwingungsfrequenz für das Gelenk, in Hertz
Dämpfungsverhältnis Dieses Dämpfungsverhältnis für das Gelenk
col Ob die beiden Instanzen kollidieren können (true) oder nicht (false)

Rückgabe:

Integer (Index des Gelenks)

Beispiel:

var i, fix, o_id, p_id;
p_id = noone;
o_id = noone;
fix = physics_fixture_create();
physics_fixture_set_box_shape(fix, 64, 32);
for (i = 0; i < 5; i++;)
{
o_id = instance_create_layer(x + (128 * i), y, "Instances", obj_BridgePart);
physics_fixture_bind(fix, o_id);
if i > 0 && i < 4
{
physics_joint_weld_create(p_id, o_id, x + (128 * i) - 64, y, 0, 10, 12, true);
}
p_id = o_id;
}
physics-fixture_delete(fix);

Der obige Code erzeugt eine Einspannvorrichtung, erstellt dann in einer Schleife eine Reihe von Instanzen, bindet die Einspannvorrichtung an jede neue Instanz und verbindet sie dann alle mit einer Schweißverbindung. Zum Schluss wird das Fixture aus dem Speicher gelöscht.
