physics_joint_delete

Sobald zwei Instanzen mit Physikdarstellungen durch eine Verbindung gebunden sind, kann diese jederzeit wieder gelöscht werden. Normalerweise geschieht dies automatisch, wenn eine der beiden Instanzen zerstört wird, oder wenn der Raum endet, aber es gibt Zeiten, in denen Sie aber es kann vorkommen, dass Sie dies manuell tun möchten. In diesen Fällen würden Sie diese Funktion verwenden.

Es sollte auch beachtet werden, dass immer, wenn eine Instanz, die Teil eines Zahnradgelenks ist, zerstört wird, das Zahnradgelenk mit dieser Funktion gelöscht werden sollte , bevor eine der Instanzen, die an der Bildung des (die verbleibenden Gelenke werden jedoch automatisch gelöscht), z. B. im Ereignis "destroy" der Instanz.

Syntax:

physics_joint_delete(joint)

Argument Beschreibung
gemeinsame Der Index der Verbindung, die Sie löschen möchten

Rückgabe:

N/A

Beispiel:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
{
reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
if reactionForce > 2
{
physics_joint_delete(shipJoint);
shipJoint = -1;
}
}

Der obige Code prüft, ob die Variable "shipJoint" einen Gelenkindex enthält. Wenn dies der Fall ist, wird die Kraft, die auf dieses Gelenk wirkt, mit Hilfe der beiden Konstanten berechnet. Wenn schließlich die Gesamtkraft größer als 2 ist, wird das Gelenk gelöscht.
