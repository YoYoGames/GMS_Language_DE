physics_joint_get_value

Durch die Verwendung einer Reihe von vordefinierten Konstanten können Sie GameMaker Studio 2 bitten, Ihnen eine Reihe von Dingen über den Zustand eines beliebigen Gelenks mitzuteilen. Dies ist sehr nützlich, da es Ihnen die Möglichkeit gibt, Gelenke zu löschen oder das Verhalten einer Instanz zu ändern, je nachdem, was Sie gerade brauchen. je nachdem, was Sie gerade brauchen. Es gibt eine Reihe von Konstanten, die in dieser Funktion verwendet werden können und die Sie hier finden: Physics Joint Constants, aber beachten Sie, dass komplexe Berechnungen durchgeführt werden Beachten Sie jedoch, dass beim Aufrufen dieser Konstanten komplexe Berechnungen durchgeführt werden. Sie sollten daher mit Vorsicht und nur bei Bedarf verwendet werden, und beachten Sie, dass viele Konstanten nur für einen bestimmten Gelenktyp gelten.

Wenn die Eigenschaft nicht vorhanden ist (wenn Sie z. B. ein Riemenscheibengelenk auf prismatische Verschiebung prüfen), ist der Rückgabewert 0.

Syntax:

physics_joint_get_value(joint, wert)

Argument Beschreibung
gemeinsame Der Index des Gelenks, das Sie testen möchten
Wert Die Konstante für die gemeinsame Eigenschaft, die Sie testen möchten

Rückgabe:

Real

Beispiel:

if shipJoint
{
var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
if reactionForce > 2
{
physics_joint_delete(shipJoint);
shipJoint = -1;
}
}

Der obige Code prüft, ob die Variable "shipJoint" einen Gelenkindex enthält. Wenn dies der Fall ist, wird die Kraft, die auf dieses Gelenk wirkt, mit Hilfe der beiden Konstanten berechnet. Wenn schließlich die Gesamtkraft größer als 2 ist, wird das Gelenk gelöscht.
