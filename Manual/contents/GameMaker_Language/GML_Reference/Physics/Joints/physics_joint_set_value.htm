physics_joint_set_value

Bestimmte Gelenkeigenschaften können auch nach der Erstellung des Gelenks geändert und eingestellt werden. Es gibt eine Reihe von Konstanten, die in dieser Funktion verwendet werden können und die Sie hier finden: Physik Gelenkkonstanten.

Syntax:

physics_joint_set_value(Gelenk, Feld, Wert)

Argument Beschreibung
gemeinsame Der Index der Verbindung, die Sie ändern möchten
Feld Die Konstante für die gemeinsame Eigenschaft, die Sie ändern möchten
Wert Der neue Wert für die gemeinsame Eigenschaft

Rückgabe:

N/A

Beispiel:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

Der obige Code prüft, ob das maximale Motordrehmoment der Gelenke auf weniger als 2 eingestellt ist, und wenn dies der Fall ist, setzt er es auf 2.
