physics_fixture_set_awake

Wenn Sie ein Fixture an eine Instanz binden, indem Sie physics_fixture_bind() binden, gibt dies eine "id" für das gebundene Fixture zurück. Sie können diese ID verwenden, um den Reibungswert des gebundenen Fixtures(nicht des "Basis"-Fixtures) jederzeit mit dieser Funktion zu ermitteln.

Syntax:

physics_get_friction(Einrichtungsgegenstand)

Argument Beschreibung
Halterung die ID des gebundenen Fixtures

Rückgabe:

Real

Beispiel:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Der obige Code ruft den aktuellen Reibungswert für die gebundenen Physikeigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
