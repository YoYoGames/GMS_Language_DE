physics_set_friction

Wenn Sie ein Fixture an eine Instanz binden, indem Sie physics_fixture_bind() binden, gibt dies eine "id" für das gebundene Fixture zurück. Sie können diese Id verwenden, um den Reibungswert des gebundenen Fixtures, nicht des "Basis"-Fixtures, jederzeit mit dieser Funktion zu setzen. Beachten Sie, dass die Reibung normalerweise auf einen Wert zwischen 0 und 1 gesetzt wird, Sie können aber auch einen beliebigen nicht-negativen Wert verwenden, falls erforderlich.

Syntax:

physics_set_friction(fixture, friction)

Argument Beschreibung
Halterung die ID des gebundenen Fixtures
Reibung der neue anzuwendende Reibungswert

Rückgabe:

N/A

Beispiel:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Der obige Code ruft den aktuellen Reibungswert für die gebundenen Physikeigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
