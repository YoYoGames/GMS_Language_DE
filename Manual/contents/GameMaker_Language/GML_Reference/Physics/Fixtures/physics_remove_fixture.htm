physics_remove_fixture

Diese Funktion entfernt (oder "entbindet") ein Fixture von einer Instanz oder mehreren Instanzen. Sie benötigt die eindeutige "id" der gebundenen Fixture (wie sie von der Funktion physics_fixture_bind() und Es werden alle aktuell definierten Physikeigenschaften für die Instanz entfernt, so dass Sie ein neues Fixture definieren und dieses an die Instanz binden können. Auf diese Weise können Sie die physikalischen Eigenschaften der Instanz ändern, ohne sie zerstören und neu erstellen zu müssen erstellen.

Syntax:

physics_remove_fixture(id, fixture)

Argument Beschreibung
id Die ID der Instanz, aus der das Fixture entfernt werden soll
Halterung Die ID des Fixtures, das aus der Instanz entfernt werden soll

Rückgabe:

N/A

Beispiel:

physics_remove_fixture(id, my_fix);

Der obige Code entfernt das Fixture mit der "id", die in der Variablen "my_fix" gespeichert ist, aus der Instanz.
