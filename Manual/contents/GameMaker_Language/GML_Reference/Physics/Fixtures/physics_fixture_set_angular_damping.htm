physics_fixture_set_angular_damping

Wenn Sie an jedes rotierende Objekt in der "realen Welt" denken, sofern es nicht über einen Motor verfügt oder sich im Weltraum befindet, wird es durch den Einfluss äußerer Kräfte (wie Reibung mit der umgebenden Luft) mit der Zeit langsamer. Mit der Funktion physics_fixture_set_angular_damping() im GameMaker Studio 2 können wir diesen Effekt simulieren und die Rotationsgeschwindigkeit von Instanzen in der Physikwelt reduzieren, denn ohne sie würde sich jede rotierende Instanz unendlich weiterdrehen. Die Dämpfungsparameter sollten zwischen 0 und unendlich liegen, wobei 0 für keine Dämpfung und unendlich für volle Dämpfung steht. Normalerweise werden Sie einen Dämpfungswert zwischen 0 und 1 verwenden, aber Sie können bei Bedarf auch einen beliebigen nicht-negativen Wert verwenden.

Syntax:

physics_fixture_set_angular_damping(fixture, damping)

Argument Beschreibung
Halterung der Index der Halterung
Dämpfung die Winkeldämpfung der Halterung, normalerweise zwischen 0 und 1

Rückgabe:

N/A

Beispiel:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Der obige Code setzt die Winkeldämpfung des in "fix_ball" indizierten Fixtures auf 0,1.
