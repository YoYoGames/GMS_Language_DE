physics_get_density

Wenn Sie ein Fixture an eine Instanz binden, indem Sie physics_fixture_bind() binden, gibt dies eine "id" für das gebundene Gerät zurück. Sie können diese ID verwenden, um den Dichtewert des gebundenen Fixtures(nicht des "Basis"-Fixture) jederzeit mit dieser Funktion abrufen.

Syntax:

physics_get_density(Einrichtungsgegenstand)

Argument Beschreibung
Halterung die ID des gebundenen Fixtures

Rückgabe:

Real

Beispiel:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

Der obige Code ruft den Stromdichtewert für die gebundenen Physikeigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
