physics_fixture_add_point

Diese Funktion definiert die Punkte eines Polygons, das zuvor entweder mit physics_fixture_set_polygon_shape() oder sie kann verwendet werden, um Punkte zu einer Kette von Rand Fixtures mit der Funktion physics_fixture_set_chain_shape(). Bei einer Polygonform sind die Punkte relativ zum Ursprung des Fixtures, wobei die Position 0,0 der x/y-Achse das Zentrum ist, und die nachfolgenden Koordinaten werden wie in GameMaker Studio 2 üblich berechnet, wobei unten/rechts +x,+y und oben/links -x,-y ist. Zur Veranschaulichung dieses Vorgangs finden Sie hier eine Illustration:

Wie Sie sehen können, kann jedes einfache konvexe Polygon auf diese Weise modelliert werden, aber konkave Polygone werden im Moment nicht unterstützt. Um das Polygon schließlich zu schließen, müssen Sie das Fixture an eine Instanz binden. Sie sollten auch beachten, dass Sie mindestens drei Punkte für Ihr Polygon definiert haben müssen, bevor Sie es an eine Instanz binden Sie sollten auch beachten, dass Sie mindestens drei Punkte für Ihr Polygon definiert haben müssen, bevor Sie es an eine Instanz binden, sonst erhalten Sie einen Fehler, und dass das Hinzufügen eines Punktes zu einem Fixture, für das zuvor keine Polygonform festgelegt wurde, ebenfalls einen Fehler verursacht. Beachten Sie auch, dass die maximal zulässige Anzahl von Punkten für ein Fixture acht ist.

Bei Kettenformen (die einfach eine Reihe verbundener Kantenformen sind) setzt diese Funktion einen Punkt als "Verbindung", an dem die Kanten miteinander verkettet werden. Daher müssen Sie mindestens zwei Punkte definiert haben, bevor Sie die Halterung an die Instanz sonst erhalten Sie einen Fehler. Anders als bei Polygonen sind Sie jedoch nicht auf ein Maximum von acht Punkten beschränkt und können so viele Punkte hinzufügen, wie Sie für die Kette benötigen. Instanz.

Syntax:

physics_fixture_add_point(fixture, xpos, ypos)

Argument Beschreibung
Halterung der Index der Halterung
xpos die x-Position des Punktes(relativ zum Fixture-Ursprung bei Polygonformen)
ypos die y-Position des Punktes(relativ zum Fixture-Ursprung bei Polygonformen)

Rückgabe:

N/A

Beispiel:

physics_fixture_set_polygon_shape(fix_schiff);
physics_fixture_add_point(fix_schiff, 0,0);
physics_fixture_add_point(fix_schiff, -40, 100);
physics_fixture_add_point(fix_schiff, 40, 100);

Der obige Code wendet eine Polygonform auf das in der Variablen "fix_Ship" indizierte Gerät an und definiert dann drei Punkte, um eine dreieckige Form zu erzeugen.
