physics_fixture_set_circle_shape

Diese Funktion definiert eine Kreisform für Ihr Gerät mit einem Radius, der durch das Argument "rad" definiert wird.

Syntax:

physics_fixture_set_circle_shape(fixture, rad)

Argument Beschreibung
Halterung der Index der Halterung
rad Radius des Kreises

Rückgabe:

N/A

Beispiel:

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2);

Der obige Code wendet auf das in der Variablen "fix_Ball" indizierte Fixture eine Kreisform mit einem Radius an, der der Breite des Sprites "spr_Ball" geteilt durch 2 entspricht.
