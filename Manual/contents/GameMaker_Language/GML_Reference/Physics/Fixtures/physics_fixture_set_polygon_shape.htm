physics_fixture_set_polygon_shape

Diese Funktion setzt eine Polygonform für Ihr Fixture, aber Sie müssen mit physics_fixture_add_point() verwenden, um die Form dieses Polygons relativ zum Ursprung des Fixtures zu definieren. Das Polygon wird geschlossen, wenn das Fixture an eine Instanz gebunden wird. Beachten Sie auch, dass diese Funktion aufgerufen werden muss, bevor Sie Punkte definieren, und dass Sie mindestens drei Punkte für Ihr Polygon definiert haben müssen, bevor Sie es an eine Instanz binden, sonst erhalten Sie einen Fehler.

Syntax:

physics_fixture_set_polygon_shape(fixture)

Argument Beschreibung
Halterung der Index der Halterung

Rückgabe:

N/A

Beispiel:

physics_fixture_set_polygon_shape(fix_schiff);
physics_fixture_add_point(fix_schiff, 0,0);
physics_fixture_add_point(fix_schiff, -40, 100);
physics_fixture_add_point(fix_schiff, 40, 100);

Der obige Code wendet eine Polygonform auf das in der Variablen "fix_Ship" indizierte Gerät an und definiert dann drei Punkte, um eine dreieckige Form zu erzeugen.
