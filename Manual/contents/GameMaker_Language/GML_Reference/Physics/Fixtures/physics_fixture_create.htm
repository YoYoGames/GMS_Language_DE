physics_fixture_create

Der erste Schritt bei der Einrichtung eines Fixtures ist das Anlegen mit dieser Funktion. Der zurückgegebene Index sollte in einer Variablen gespeichert werden, um in allen weiteren Funktionen verwendet zu werden, die zur Definition und Verwendung dieses Fixtures dienen.

Syntax:

physics_fixture_create()

Rückgabe:

Index des Geräts

Beispiel:

fix_Ball = physics_fixture_create();

Der obige Code erzeugt ein Fixture und speichert dessen Index in der Variablen "fix_Ball".
