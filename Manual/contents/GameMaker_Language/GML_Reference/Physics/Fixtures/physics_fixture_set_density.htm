physics_fixture_set_density

Die Dichte von etwas ist definiert als seine Masse pro Volumeneinheit, was im Grunde bedeutet, wie viel Masse in den Raum gezwängt ist, den es in der Welt einnimmt. So würde ein Luftballon eine niedrige Dichte haben, da er wenig Masse hat, die einen großen Raum einnimmt, während ein Bleistab eine hohe Dichte haben würde, da er eine große Masse hat, aber wenig Raum einnimmt.

In der 2D-Physikwelt von GameMaker Studio 2 wird die Masse automatisch berechnet aus den Werten, die Sie für die Dichte eingeben, und der Oberfläche der Form, die Sie für das Fixture definieren. Dies hat einen direkten Einfluss darauf, wie viel Trägheit ein Fixture hat und wie es auf Kollisionen und Kräfte reagiert, also Wenn Sie eine kleine Form mit einer hohen Dichte definieren, hat sie eine sehr große Masse (wie ein Stab aus Blei), aber wenn Sie eine große Form mit einer niedrigen Dichte definieren, hat sie eine viel kleinere Masse (wie ein Luftballon).

HINWEIS: Wenn Sie möchten, dass die Instanz, die das Fixture erhält, statisch ist (d. h. unbeweglich in der Spielwelt), dann sollte die Dichte auf 0 gesetzt werden, was im Wesentlichen eine unendliche Dichte ergibt.

HINWEIS: Ein statisches Fixture kann kinematisch gemacht werden (d.h. es kann sich bewegen, aber ohne die Auswirkungen von Kraft oder Schwerkraft), indem es eine Dichte von 0 erhält und dann die phy_speed_x , phy_speed_y , oder phy_angular_velocity Variablen setzen.

Syntax:

physics_fixture_set_density(fixture, density)

Argument Beschreibung
Halterung der Index der Halterung
Dichte die Dichte des Einbauteils, berechnet in kg/m²

Rückgabe:

N/A

Beispiel:

physics_fixture_set_density(fix_Ballon, 0.1);

Der obige Code setzt die Dichte des in "fix_Ballon" indizierten Geräts auf 0,1.
