physics_fixture_set_chain_shape

Diese Funktion definiert eine "Ketten"-Einbauteilform. Eine Ketteneinspannung besteht aus einer Anzahl von Punkten, die dann mit Hilfe von Kantenformen miteinander verbunden werden, um sie zu verbinden. Die Funktion selbst nimmt den Index (ID) des Einbauteils und kann dem Einbauteil sagen eine Schleife bilden soll oder nicht. Wenn Sie diese Funktion auf true setzen, wird der letzte Punkt mit dem ersten Punkt durch eine Kante verbunden, wenn Sie sie auf false setzen, nicht. Beachten Sie, dass diese Funktion das Spannzeug für die Aufnahme der Punkte vorbereitet, die zur Bildung der Kette benötigt werden, und diese sollten nach dem Aufruf dieser Funktion mit physics_fixture_add_point()hinzugefügt werden, ähnlich wie bei der Erstellung einer Polygon-Fixture.

Wenn Sie eine Kettenbefestigung erstellen, müssen Sie ihr mindestens zwei Punkte geben, aber Sie sind in der Anzahl der nachfolgenden Punkte, die Sie hinzufügen können, um die fertige Befestigung zu bilden, nicht begrenzt.

Syntax:

physics_fixture_set_chain_shape(fixture, loop)

Argument Beschreibung
Halterung Der Index der Halterung
Schleife Ob die Kette geschlossen werden soll(true) oder nicht(false).

Rückgabe:

N/A

Beispiel:

var xx = -100;
var yy = raum_höhe / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
physics_fixture_add_point(fix, xx, yy);
xx += 50 + random(150);
yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

Der obige Code erzeugt eine Reihe von "Rand"-Einbauten mit unterschiedlichen Höhen über die Länge des Raums.
