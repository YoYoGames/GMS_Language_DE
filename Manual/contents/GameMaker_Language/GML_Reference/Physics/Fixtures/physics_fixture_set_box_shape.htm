physics_fixture_set_box_shape

Diese Funktion definiert eine Kastenform für Ihr Gerät. Sie nimmt die halbe Breite und Höhe, da die Physik diesen Wert viel mehr als ganze Breiten-/Höhenwerte verwendet, um Dinge wie Kollisionen zu bestimmen.

Syntax:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

Argument Beschreibung
Halterung der Index der Halterung
halbeBreite die halbe Breite der Box
halbeHöhe die halbe Höhe der Box

Rückgabe:

N/A

Beispiel:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

Der obige Code wendet auf das in der Variablen "fix_Border" indizierte Gerät eine Kastenform mit einer Breite des Raums und einer Höhe von 20 Pixeln an.
