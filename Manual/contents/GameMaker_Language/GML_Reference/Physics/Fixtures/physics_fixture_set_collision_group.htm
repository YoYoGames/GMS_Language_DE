physics_fixture_set_collision_group

Sobald Sie Ihr Fixture erstellt haben, können Sie es als Teil einer Kollisionsgruppe definieren. Das bedeutet, dass Sie Instanzen mit diesem Fixture anweisen können, immer oder nie mit anderen Instanzen zu kollidieren, an die Fixtures mit derselben Kollisionsgruppe gebunden sind. der gleichen Kollisionsgruppe gebunden sind. Die Werte, die zugewiesen werden können, basieren auf positiven oder negativen Zahlen, wobei 0 der Standardwert ist (was bedeutet, dass die Instanz mit dieser Fixture-Einstellung ignoriert wird, es sei denn, es wird ein Kollisionsereignis für sie erzeugt). Hier ist ein Bild zur Erläuterung:

Wie Sie sehen können, kann das Einstellen von Kollisionsgruppen sehr nützlich sein, um komplexe physikalische Körper zu erstellen, die mit der physikalischen Welt um sie herum, aber nicht untereinander interagieren. Es sollte jedoch beachtet werden, dass aufgrund der Komplexität der Berechnung von Kollisionen, die auf solchen Gruppen basieren, empfohlen wird, die Anzahl der in einem Raum verwendeten der in einem Raum verwendeten Gruppen auf ein absolutes Minimum zu beschränken.

Syntax:

physics_fixture_set_collision_group(fixture, group)

Argument Beschreibung
Halterung das Gerät, das gebunden werden soll
Gruppe die Gruppe, der dieses Gerät zugeordnet ist (entweder ein positiver Wert, 0, oder ein negativer Wert)

Rückgabe:

N/A

Beispiel:

physics_fixture_set_collision_group(fix_Wheel, -1);

Der obige Code setzt das in der Variable "fix_Wheel" indizierte Gerät in die Gruppe "-1".
