physics_get_restitution

Wenn Sie ein Fixture an eine Instanz binden, indem Sie physics_fixture_bind() binden, gibt dies eine "id" für das gebundene Fixture zurück. Sie können diese ID verwenden, um die Restitution (die Eigenschaft "Bounciness") Wert des gebundenen Fixtures, nicht des "Basis"-Fixtures, jederzeit mit dieser Funktion ermitteln.

Syntax:

physics_get_restitution(Einbauteil)

Argument Beschreibung
Halterung die ID des gebundenen Fixtures

Rückgabe:

Real

Beispiel:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

Der obige Code ruft den aktuellen Restitutionswert für die gebundenen Physikeigenschaften der Instanz ab und setzt sie dann auf einen anderen Wert.
