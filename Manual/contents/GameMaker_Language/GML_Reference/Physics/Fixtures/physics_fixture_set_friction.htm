physics_fixture_set_friction

Reibung ist die Kraft, die der relativen Bewegung von Materialelementen, die gegeneinander gleiten, entgegenwirkt, was in der GameMaker Studio 2-Physikwelt als Impulsverlust durch die Kollision von zwei Instanzen mit daran gebundenen Fixtures mit ihnen. Wenn also zwei Instanzen kollidieren, wird ihre Bewegung von diesem Wert beeinflusst, wobei eine hohe Reibung einen größeren Impulsverlust verursacht als ein niedrigerer Wert. Beachten Sie, dass die Reibung in der Regel auf einen Wert zwischen 0 und 1 gesetzt wird, Sie können aber auch einen beliebigen nicht-negativen Wert verwenden, falls erforderlich.

Syntax:

physics_fixture_set_friction(fixture, friction)

Argument Beschreibung
Halterung der Index der Halterung
Reibung die Reibung der Halterung

Rückgabe:

N/A

Beispiel:

physics_fixture_set_friction(fix_Brick, 0.1);

Der obige Code setzt die Reibung des in "fix_brick" indizierten Fixtures auf 0,1.
