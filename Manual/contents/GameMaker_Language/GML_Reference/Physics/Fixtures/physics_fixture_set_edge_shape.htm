physics_fixture_set_edge_shape

Diese Funktion definiert eine "Rand"-Fixture-Form. Eine Kantenform ist einfach eine Linie, die eine Kollision erzeugt, wenn andere Fixtures sie überlappen, und kann sehr nützlich sein, um (z. B.) Terrain zu erzeugen oder um Grenzen um einen Raum herum zu schaffen. Die Position der Kante wird mit Hilfe des lokalen Raums definiert, d. h. die x/y-Position der Instanz wird als (0,0) betrachtet, daher sollte dies bei der Erstellung berücksichtigt werden (im Codebeispiel unten wäre die Instanz an (0,0) im Raum platziert, um Komplikationen zu vermeiden).

Syntax:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

Argument Beschreibung
Halterung der Index der Halterung
lokal_x1 Start-X-Position für die Kante
lokal_y1 Start-Y-Position für die Kante
lokal_x2 x-Endposition für die Kante
lokal_y2 y-Endposition für die Kante

Rückgabe:

N/A

Beispiel:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
var fix = physics_fixture_create();
physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
physics_fixture_bind(fix, id);
physics_fixture_delete(fix);
xx += 50;
y1 = y2;
y2 = room_height - 50 - irandom(100);
}

Der obige Code erzeugt eine Reihe von "Rand"-Einbauten mit unterschiedlichen Höhen über die Länge des Raums.
