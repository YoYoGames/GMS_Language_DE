physik_test_überlappung

Mit dieser Funktion kann geprüft werden, ob ein physischer Körper (d. h. die Halterung einer Instanz) überlappt oder überlappen wird, wenn er gedreht und an einer bestimmten Position im Raum platziert wird. das Argument "angle" ist der Drehwinkel, den die aufrufende Instanz an der zu prüfenden Position hat (oder haben wird), und das Argument "obj" kann entweder eine einzelne Instanz-ID und ein Objektindex sein oder die Schlüsselwörter alle oder andere sein.

Syntax:

physics_test_overlap(xpos, ypos, angle, obj);

Argument Beschreibung
xpos Die zu prüfende x-Position im Raum
ypos Die zu prüfende y-Position im Raum
Winkel Der zu prüfende Winkel (der aufrufenden Instanz)
obj Das Objekt, auf das geprüft werden soll

Rückgabe:

Boolesche

Beispiel:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
alarm[0] = room_speed;
ignited = true;
}

Der obige Code prüft eine Position auf eine Überlappung der physikalischen Vorrichtungen, und wenn es eine gibt, setzt er eine Variable und einen Alarm.
