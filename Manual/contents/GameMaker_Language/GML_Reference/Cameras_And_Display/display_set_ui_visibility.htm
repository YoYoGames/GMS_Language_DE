display_set_ui_visibility

Diese Funktion kann verwendet werden, um die System-Benutzeroberfläche ein- oder auszublenden auf Android nur. Die Funktion erfordert die Übergabe eines oder mehrerer Systemflags als Ganzzahlwert. Wenn Sie mehr als ein Flag verwenden, müssen Sie diese mit dem bitweisen "oder" verknüpft werden, wie im folgenden Beispiel gezeigt. Eine Liste der Android-Systemflags finden Sie hier.

Syntax:

display_set_ui_visibility(flags);

Argument Flaggen
Flaggen Die zu verwendenden Android-SYSTEM-Flags (als Ganzzahlwert).

Rückgabe:

N/A

Beispiel:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Der obige Code verwendet die Android-Systemflags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" und "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN", um die Sichtbarkeit des Displays einzustellen.
