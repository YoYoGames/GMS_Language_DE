display_mouse_set

Mit dieser Funktion können Sie die Position der Maus innerhalb der Spieldarstellung ändern oder einstellen, was z. B. bei FPS-Spielen nützlich sein kann. Die Funktion funktioniert nur, solange das Spiel im Fokus ist. Mit ALT + Tab wird die Maus entsperrt.

Syntax:

display_mouse_set(x, y);

Argument Beschreibung
x Die x-Koordinate in der Anzeige.
y Die y-Koordinate in der Anzeige.

Rückgabe:

N/A

Beispiel:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Der obige Code würde die Maus in der Spielanzeige zentrieren.
