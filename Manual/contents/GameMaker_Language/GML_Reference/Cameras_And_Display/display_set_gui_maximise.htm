display_set_gui_maximise

Mit dieser Funktion kann die GUI-Ebene maximiert und so eingestellt werden, dass sie in Bezug auf die Bildschirmabmessungen skaliert und versetzt wird, anstatt die Standardposition und -abmessungen der Anwendungsoberfläche zu verwenden. Standardmäßig ist die GUI-Ebene 1:1 mit der Anwendungsoberfläche Auflösung und wird ebenfalls an der Position (0,0) der Oberfläche gezeichnet. Das ist jedoch nicht immer das, was Sie wollen, und deshalb können Sie mit dieser Funktion einstellen, dass sie relativ zur absoluten (0,0)-Position des Displays oder des Spielfensters gezeichnet wird. Welche Argumente Sie dieser Funktion übergeben, hängt davon ab, welchen Effekt sie auf die GUI-Ebene haben soll. Wenn Sie die Funktion ohne Argumente aufrufen, wird die GUI-Ebene an der Position (0,0) des Bildschirms oder des Spielfensters gezeichnet, wobei die Breite und Höhe werden so skaliert, dass sie in den gesamten Bereich passen:

display_set_gui_maximise();

Sie können jedoch den Skalierungsfaktor für die GUI-Ebene festlegen, und die Breite und Höhe werden um diesen Betrag skaliert. Denken Sie daran, dass die GUI-Ebene immer an die Größe des Anzeige- oder Spielfensters bzw. der Anwendungsoberfläche angepasst wird, so dass die Einstellung Wenn Sie diesen Wert auf einen anderen Wert als 1 setzen, wird die Anzahl der Pixel entlang der Breite und Höhe skaliert. Wenn Ihr Bildschirm eine Größe von 1024x768 hat und Sie den Skalierungsfaktor auf 0,5 einstellen, dann wird Ihre GUI-Ebene halb so groß wie der Bildschirm, was effektiv eine Verdoppelung der Pixelgröße bedeutet die Pixelgröße:

display_set_gui_maximise(0.5, 0.5);

Wenn Sie die Werte auf diese Weise einstellen, wird auch die Zeichenposition auf die (0,0) des Anzeige- oder Spielfensters gesetzt, sodass Sie Offset-Werte angeben können, um die (0,0)-Position zu "verschieben":

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

Schließlich können Sie mit dieser Funktion die GUI-Ebene zurücksetzen, indem Sie die Skalierungsfaktoren auf -1 setzen. Dadurch wird die GUI-Ebene wieder auf einen Maßstab von 1:1 gesetzt und die Zeichenposition auf die (0,0)-Position der Anwendungsoberfläche und nicht auf das Display oder Fensters.

display_set_gui_maximise(-1, -1);

Syntax:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

Argument Beschreibung
x-Skala <Optional> Der horizontale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
y-Skala <Optional> Der vertikale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
xOffset <Optional> Die x-Offset-Position für das Zeichnen.
yOffset <Optional> Die y-Versatzposition für das Zeichnen.

Rückgabe:

N/A

Beispiel:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

Der obige Code sperrt das GUI-Ereignis "Draw" auf die angegebene Breite und Höhe und skaliert alle Komponenten so, dass sie in die Anzeige passen, wobei diese Proportionen verwendet werden.
