window_get_width

Mit dieser Funktion können Sie die aktuelle Breite (in Pixel) des Spielfensters ermitteln.

Syntax:

window_get_width();

Rückgabe:

Real

Beispiel:

ww = window_get_width();

Der obige Code speichert die aktuelle Fensterbreite in der Variablen "ww".
