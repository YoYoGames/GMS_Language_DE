window_mouse_set

Mit dieser Funktion können Sie die Position der Maus innerhalb des Spielfensters ändern oder festlegen, was z. B. für FPS-Spiele nützlich sein kann. Die Funktion funktioniert nur, wenn das Spiel fokussiert ist, und mit alt + tab wird die Maus entsperrt.

HINWEIS: Für die regulären Mausfunktionen siehe den Abschnitt über die Mauseingabe

Syntax:

window_mouse_set(x, y);

Argument Beschreibung
x Die x-Koordinate im Fenster.
y Die y-Koordinate im Fenster.

Rückgabe:

N/A

Beispiel:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

Der obige Code würde die Maus im Spielfenster zentrieren.
