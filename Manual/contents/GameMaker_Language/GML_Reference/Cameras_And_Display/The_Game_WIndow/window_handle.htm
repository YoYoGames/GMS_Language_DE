fenster_handle

Mit dieser Funktion können Sie den internen Windows-Id-Wert (den HWND, einen Zeiger) erhalten. Diese Funktion ist wirklich nur für Erweiterungsschreiber nützlich, die das Fenster-Handle benötigen, um Windows-APIs im DLL-Code aufzurufen (der zurückgegebene Zeiger sollte (der zurückgegebene Zeiger sollte in einen String und dann in C++ einfach in ein HWND umgewandelt werden). Die folgende Tabelle zeigt die unterstützten Plattformen und was sie zurückgeben:

Plattform Rückgabe
Windows Fenster HWND
macOS NSWindow-Klasse
Ubuntu(Linux) XWindow-Handle
HTML5 Leinwand-ID

Syntax:

window_handle();

Rückgabe:

Zeiger

Beispiel:

win_id = windows_handle();

Der obige Code speichert die Spielfenster-ID in einer Variablen.
