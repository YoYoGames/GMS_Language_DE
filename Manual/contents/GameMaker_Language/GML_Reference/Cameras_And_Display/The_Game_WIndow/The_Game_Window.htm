Das Spielfenster

Das Spiel, das Sie erstellen, findet in einem Fenster statt (auch im Vollbildmodus), und dieses Fenster hat eine Reihe von Eigenschaften, wie Position, Größe, ob es im Vollbildmodus ist, usw.. Diese Details werden normalerweise automatisch für Sie eingestellt, basierend auf der Raumgröße und den View Ports in Kombination mit den Einstellungen der Spieloptionen für die Zielplattform aktiviert, aber Sie können sie während des Spiels mit den auf dieser Seite aufgeführten Funktionen ändern.

HINWEIS: Diese Funktionen sind für Windows, Ubuntu(Linux), macOS und HTML5-Zielmodule beschränkt und funktionieren möglicherweise nicht auf anderen Geräten.

Beachten Sie, dass das Bild oben das Spielfenster zeigt, das in einem Browser gezeichnet wird, aber die für die anderen Ziele, können Sie den Browser durch das Display ersetzen, sodass eine Funktion wie window_get_y() die Position des oberen Teils des Spielfensters relativ zum Display zurückgibt.
