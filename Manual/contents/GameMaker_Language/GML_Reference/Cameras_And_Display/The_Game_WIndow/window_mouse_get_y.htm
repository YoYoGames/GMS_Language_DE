window_mouse_get_y

Mit dieser Funktion können Sie die y-Position des Mauszeigers (in Pixeln) innerhalb des Browsers ermitteln, wenn es sich um ein HTML5-Spiel handelt, oder innerhalb der Anzeige, wenn es sich um ein Windows-, Ubuntu(Linux)- oder macOS-Spiel handelt.

HINWEIS: Für die regulären Mausfunktionen siehe den Abschnitt über die Mauseingabe

Syntax:

window_mouse_get_y();

Rückgabe:

Real

Beispiel:

wy = window_mouse_get_y();

Der obige Code speichert die aktuelle y-Achsen-Fensterposition der Maus in der Variablen "wy".
