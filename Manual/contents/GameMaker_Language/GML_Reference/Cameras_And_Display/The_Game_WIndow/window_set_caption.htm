window_set_caption

Mit dieser Funktion können Sie die Fensterbeschriftung für den Raum, in dem Sie sich gerade befinden, ändern oder einstellen. Diese Beschriftung erscheint oben im Fenster neben dem Spielsymbol, wenn sich das Spiel nicht im Vollbildmodus befindet.

Syntax:

window_set_caption(beschriftung);

Argument Beschreibung
Beschriftung Die neue Beschriftung.

Rückgabe:

N/A

Beispiel:

if window_get_caption() != ""
{
window_set_caption("");
}

Der obige Code prüft, ob die Beschriftung des Fensters Text enthält oder nicht, und wenn ja, setzt er sie auf eine leere Zeichenfolge, so dass keine Beschriftung angezeigt wird.
