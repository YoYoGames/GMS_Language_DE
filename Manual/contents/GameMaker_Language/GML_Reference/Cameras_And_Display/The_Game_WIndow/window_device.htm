window_device

Diese Funktion gibt den aktuellen d3d-Gerätezeiger zurück, den Sie dann (zum Beispiel) an eine DLL oder Dylib unter Windows und macOS weitergeben können.

HINWEIS: Diese Funktion wurde in GameMaker Studio 2 veraltet zugunsten von os_get_info().

Syntax:

window_device();

Rückgabe:

Zeiger

Beispiel:

gfx_pointer = window_device()

Der obige Code speichert den d3d-Gerätezeiger in einer Variablen.
