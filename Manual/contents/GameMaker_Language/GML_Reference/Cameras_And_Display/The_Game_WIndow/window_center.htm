fenster_mitte

Mit dieser Funktion können Sie das Spielfenster im Display zentrieren, wenn das Zielmodul Windows, Ubuntu(Linux) oder macOS ist, oder Sie können es im Browser zentrieren, wenn das Zielmodul HTML5 ist. Diese Funktion hat keine Auswirkung auf ein anderes Gerät.

HINWEIS: Wenn Sie die Größe des Spielfensters geändert haben (z. B. durch Umschalten von Vollbild auf Fenster) und das neue Fenster zentrieren möchten, sollte diese Funktion in einem Alarm[0] mindestens einen Schritt später als die Änderung der Fenstergröße aufgerufen werden, sonst wird sie nicht korrekt funktionieren.

Syntax:

window_center();

Rückgabe:

N/A

Beispiel:

if !window_get_fullscreen()
{
window_center();
}

Der obige Code zentriert das Spielfenster innerhalb der Anzeige, wenn es sich nicht im Vollbildmodus befindet.
