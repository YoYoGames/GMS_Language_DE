window_set_fullscreen

Mit dieser Funktion können Sie das Spielfenster als Vollbild(true) oder nicht(false) einstellen. Bitte beachten Sie, dass für das macOS Ziel die Option "Im Vollbildmodus starten" deaktiviert und die Option "Dem Spieler das Ändern der Größe des Spielfensters erlauben" in den Spieloptionen aktiviert sein muss. die Option "Allow the player to resize the game window" in den Spieloptionen aktiviert haben müssen, da diese Funktion sonst fehlschlagen wird. Beachten Sie auch, dass diese Funktion nicht mit HTML5 funktioniert, es sei denn, sie wird als "klickbarer" Callback (siehe hier für weitere Details).

Syntax:

window_set_fullscreen(full);

Argument Beschreibung
vollständig Ob der Bildschirm auf Vollbild (true) oder nicht (false) eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

if mouse_check_button_pressed(mb_left)
{
if window_get_fullscreen()
{
window_set_fullscreen(false);
}
sonst
{
window_set_fullscreen(true);
}
}

Der obige Code prüft, ob eine Maustaste gedrückt wird, und setzt dann das Fenster auf Vollbild, wenn es das nicht schon ist, oder setzt es auf Fenster, wenn es das schon ist.
