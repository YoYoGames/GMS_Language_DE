fenster_setzen_position

Mit dieser Funktion können Sie das Spielfenster auf eine bestimmte Position innerhalb der Anzeige (unter macOS, Linux(Ubuntu) und Windows) oder innerhalb des Browsers (HTML5) setzen.

HINWEIS: Wenn Ihr HTML5-Spiel eine benutzerdefinierte index.html verwendet und diese die Leinwand auf eine feste Position setzt, hat diese Funktion keine Auswirkungen auf die Fensterposition.

Syntax:

window_set_position(x, y);

Argument Beschreibung
x Die x-Koordinate, an der das Fenster positioniert werden soll.
y Die y-Koordinate, an der das Fenster positioniert werden soll.

Rückgabe:

N/A

Beispiel:

window_set_position(0, 0);

Der obige Code positioniert das Spielfenster in der oberen linken Ecke des Browsers oder des Displays (je nach verwendetem Zielmodul).
