fenster_setzen_rechteck

Mit dieser Funktion können Sie die Position des Spielfensters innerhalb des Browsers (HTML5) oder der Anzeige(Windows, Ubuntu(Linux) oder macOS) festlegen und auch die Skalierung des Fensters einstellen. Weitere Informationen zur Fensterposition und Fenstergröße finden Sie unter window_set_position() und . window_set_size().

Syntax:

window_set_rectangle(x, y, w, h);

Argument Beschreibung
x Die neue x-Koordinate des Fensters.
y Die neue y-Koordinate des Fensters.
w Die neue Breite des Fensters.
h Die neue Höhe des Fensters.

Rückgabe:

N/A

Beispiel:

window_set_rectangle(0, 0, display_get_width(), display_get_height());

Mit dem obigen Code wird das Spielfenster so eingestellt, dass es den gesamten Anzeigebereich einnimmt (entweder den Browser oder den Bildschirm, je nach Zielplattform).
