window_get_colour

Diese Funktion gibt die Hintergrundfarbe des Spielfensters zurück. Diese Farbe stellt die Farbe dar, die für die Bereiche des Spielfensters verwendet wird, die nicht von Ansichten belegt sind. Das folgende Bild veranschaulicht dies:

Das obige Bild hat zwei Ansichten mit zwei Ansichtsports, die jeweils an unterschiedlichen Positionen gezeichnet sind. Dadurch wird das Spielfenster gestreckt, um beide Ports unterzubringen, und die Fensterfarbe wird verwendet, um den Hintergrund zu färben, wo keine Ansicht angezeigt wird.

Syntax:

window_get_colour();

Rückgabe:

N/A

Beispiel:

if window_get_colour() != c_black
{
window_set_colour(c_black);
}

Der obige Code prüft, ob die Farbe des Fensters als schwarz eingestellt ist oder nicht, und wenn nicht, setzt er sie auf schwarz.
