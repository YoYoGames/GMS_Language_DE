fenster_hat_fokus

Mit dieser Funktion können Sie den Zustand des Fensters (oder der Registerkarte) abfragen, und wenn es den Fokus verliert, gibt die Funktion false zurück, andernfalls wird sie true zurückgeben. In den meisten Fällen können Sie einfach die os_is_paused() Funktion verwenden, um dies zu testen, aber in einigen sehr speziellen Fällen (z. B. Spiele in Chrome-Apps) wird diese Funktion nicht ausgelöst; in diesem Fall sollten Sie stattdessen diese Funktion verwenden.

HINWEIS: Diese Funktion ist nur für HTML5, Windows und MacOS gültig.

Syntax:

window_has_focus();

Rückgabe:

Boolesche

Beispiel:

if !window_has_focus()
{
scr_PauseSpiel();
}

Der obige Code prüft, ob das Spielfenster im Fokus ist oder nicht, und wenn die Funktion false zurückgibt, wird ein Skript aufgerufen.
