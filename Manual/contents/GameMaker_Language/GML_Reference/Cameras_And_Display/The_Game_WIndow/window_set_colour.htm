window_set_colour

Mit dieser Funktion kann die Hintergrundfarbe des Spielfensters eingestellt werden. Diese Farbe stellt die Farbe dar, die für die Bereiche des Spielfensters verwendet wird, die nicht von Ansichten belegt sind. Das folgende Bild veranschaulicht dies:

Das obige Bild hat zwei Ansichten mit zwei View Ports, die jeweils an unterschiedlichen Positionen gezeichnet sind. Dieses streckt das Spielfenster, um beide Ports unterzubringen und verwendet die Fensterfarbe, um den Hintergrund zu färben, wo keine Ansicht angezeigt wird.

Syntax:

window_set_colour(Farbe);

Argument Beschreibung
Farbe Die Farbe, mit der der Bereich festgelegt wird.

Rückgabe:

N/A

Beispiel:

if window_get_colour() != c_black
{
window_set_colour(c_black);
}

Der obige Code prüft, ob die Farbe des Fensters als schwarz eingestellt ist oder nicht, und wenn nicht, setzt er sie auf schwarz.
