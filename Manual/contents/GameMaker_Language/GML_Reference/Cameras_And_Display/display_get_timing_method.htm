display_get_timing_method

Mit dieser Funktion können Sie die Timing-Methode abrufen, die für das Rendern Ihres Spiels verwendet werden soll. Die Methode kann eine der unten aufgeführten Konstanten sein.

Weitere Informationen zu den verschiedenen Timing-Methoden finden Sie unter display_set_timing_method().

Syntax:

display_get_timing_method();

Rückgabe:

Konstant

Konstant Beschreibung
tm_sleep Der Sleep-Margin-Wert ist die wichtigste Timing-Methode
tm_countvsyncs Vsync-Timing ist die Haupt-Timing-Methode (Standard für alle unterstützten Plattformen)

Beispiel:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
{
display_set_sleep_margin(20);
}
}

Der obige Code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diesen Wert gesetzt und die Sleep-Marge auf 20 gesetzt.
