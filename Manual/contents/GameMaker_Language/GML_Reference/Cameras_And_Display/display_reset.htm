display_reset

Diese Funktion setzt die Anzeigeeinstellungen auf die beim Start des Spiels eingestellten zurück, ermöglicht es Ihnen aber auch, den aktuellen Grad des verwendeten Vollbild-Anti-Aliasings zu ändern und zu entscheiden, ob die vertikale Synchronisation verwendet werden soll. Die verfügbaren Anti-Aliasing Stufen sind 0, 2, 4 und 8, wobei der Standardwert beim Start auf 0 gesetzt ist und die Standardeinstellung für v-sync falsch (aus) ist. Das Einschalten vonv-sync kann zu einem flüssigeren Spielerlebnis führen, benötigt aber auch mehr Rechenleistung, so dass die Auswirkungen sorgfältig abgewogen werden müssen Das Gleiche gilt für das Anti-Aliasing: Je höher die Zahl, desto mehr Rechenleistung wird benötigt.

Da nicht alle Zielgeräte gleich sind und manche z. B. kein 8x oder 4x Anti-Aliasing unterstützen, gibt es eine schreibgeschützte Variable, um die verschiedenen Stufen von AA zu erhalten, die das Gerät, auf dem das Spiel läuft, anzeigen kann:

anzeigen_aa

Diese Variable gibt einen ganzzahligen Wert zurück, der auf der Einstellung der Bits für die verschiedenen Stufen basiert. Für nur 2xAA wird also 2 gemeldet, für 2x und 4x Verfügbarkeit wird 6 gemeldet. Für 8 und 4 wird sie 12 melden. Für alle 3 Stufen (2, 4 und 8) wird es 14 melden.

Syntax:

display_reset(aa, vsync);

Argument Beschreibung
aa Der Grad der Anti-Aliasing-Filterung (0, 2, 4 oder 8).
vsync Schaltet die vertikale Synchronisation ein (true) oder aus (false).

Rückgabe:

Real

Beispiel:

if (display_aa > 12)
{
display_reset(8, true);
}

Der obige Code setzt die Anti-Aliasing-Stufe auf 8, falls unterstützt, und schaltet v-sync auf ein.
