display_set_sleep_margin

Mit dieser Funktion können Sie den für das Rendering-Timing Ihres Spiels verwendeten Wert für den Ruhezustand einstellen und benötigen einen Millisekundenwert. Die Standardwerte sind wie folgt:

Plattform Millisekunden
Windows 10
macOS 10
Ubuntu 10
HTML5 10
Android 4
iOS 4
Windows UWP 10
Xbox 10
PS4 10
Nintendo Switch 10

Beachten Sie, dass selbst wenn die Timing-Methode auf die Verwendung von Vsync-Timing gesetzt ist, der Sleep-Margin eine gewisse Auswirkung auf das Rendering des Spiels hat und Sie daher beim Setzen dieses Wertes vorsichtig sein sollten. Weitere Informationen zum Anzeige-Timing finden Sie unter display_set_timing_method().

HINWEIS: Zusätzlich zum Ruhebereich können Sie die Leistung und den Stromverbrauch unter Windows weiter kontrollieren, indem Sie die Auflösung des Thread-Schedulers zur Laufzeit anpassen .

Syntax:

display_set_sleep_margin(milliseconds);

Argument Beschreibung
Millisekunden Die Anzahl der Millisekunden für den Schlafabstand

Rückgabe:

N/A

Beispiel:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
{
display_set_sleep_margin(20);
}
}

Der obige Code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diesen Wert gesetzt und die Sleep-Marge auf 20 gesetzt.
