gif_save_buffer

Mit dieser Funktion können Sie eine GIF-Animation ausspeichern. Sie geben den GIF-Index an (wie er von der Funktion gif_open()) und die Funktion gibt einen 1-Byte-alignierten Grow-Buffer mit den GIF-Daten zurück. Beachten Sie, dass die endgültigen GIF-Daten mit Hilfe der Universal 884 Palette palettiert wurden (siehe hier für weitere Informationen).

Syntax:

gif_save_buffer(gif_index);

Argument Beschreibung
gif_index Die ID des zu speichernden Gifs

Rückgabe:

Integer (Puffer-ID)

Beispiel:

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
sonst
{
global.capture_buff = gif_save_buffer(gif_image);
count = 0;
save_gif = false;
}
count++;
}

Der obige Code erstellt eine GIF-Bilddatei mit 30 Einzelbildern aus der Anwendungsoberfläche und speichert sie dann in einem Puffer.
