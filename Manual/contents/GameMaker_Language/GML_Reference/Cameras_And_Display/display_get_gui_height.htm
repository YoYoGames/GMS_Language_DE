display_get_gui_width

Mit dieser Funktion können Sie die Höhe (in Pixel) der GUI ermitteln, wie sie im Draw GUI Event verwendet wird.

Syntax:

display_get_gui_height();

Rückgabe:

Real

Beispiel:

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

Mit dem obigen Code wird eine Anzeige in der linken unteren Ecke des Displays eingestellt.
