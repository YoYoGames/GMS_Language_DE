display_get_sleep_margin

Diese Funktion kann verwendet werden, um den aktuellen Wert des Ruhezustands zu ermitteln, der für das Render-Timing Ihres Spiels verwendet wird, und gibt einen Wert in Millisekunden zurück. Weitere Informationen zum Anzeige-Timing finden Sie unter display_set_timing_method().

Syntax:

display_get_sleep_margin();

Rückgabe:

Real

Beispiel:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
{
display_set_sleep_margin(20);
}
}

Der obige Code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diesen Wert gesetzt und die Sleep-Marge auf 20 gesetzt.
