camera_get_proj_mat

Mit dieser Funktion kann die Projektionsmatrix abgerufen werden. Die Funktion gibt den Matrix-ID-Wert zurück, der dann in anderen Matrixfunktionen verwendet werden kann oder um die Projektions Matrix einer anderen Kamera (mit camera_set_proj_mat()).

Syntax:

camera_get_proj_mat(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Matrix-ID (real)

Beispiel:

mat = camera_get_proj_mat(camera_view[0]);

Der obige Code ruft die Projektionsmatrix für die Kamera ab, die dem Viewport[0] zugewiesen ist.
