ansicht_setzen_kamera

Diese Funktion setzt eine Kamera auf einen bestimmten Ansichtsanschluss. Sie geben den zu setzenden Ansichtsanschluss an (von 0 bis 7) und liefern den eindeutigen Kamera-ID-Wert (wie er von den camera_create() Funktionen oder bei Verwendung von view_get_camera()). Wenn Sie als Kamera-ID den Wert -1 angeben, entfernen Sie eine Kamera aus dem Ansichtsanschluss und beachten Sie, dass, wenn dieser Ansichtsanschluss aktiviert und sichtbar ist, Sie möglicherweise unvorhersehbare Ergebnisse erhalten.

Syntax:

view_set_camera(view_port, camera_id)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

N/A

Beispiel:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

Der obige Code erstellt eine neue Kamera und weist ihr dann view port[0] zu.
