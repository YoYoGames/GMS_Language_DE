camera_set_view_angle

Mit dieser Funktion können Sie den Winkel der Fachkamera innerhalb des Raums aktualisieren. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben dann den Winkel an, den Sie Sie den Winkel angeben, auf den Sie die Kameraansicht einstellen möchten. Der Standardwert ist 0°, wobei positive Werte die Kamera gegen den Uhrzeigersinn drehen, d. h.: wenn Sie den Wert auf 90 setzen, wird die Kamera um 90° nach links gedreht.

Syntax:

camera_set_view_angle(camera_id, angle)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde
Winkel Der neue Winkel, auf den die Kameraansicht eingestellt werden soll

Rückgabe:

N/A

Beispiel:

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

Der obige Code stellt den Winkel der Fachkamera für die Kamera ein, die dem Anschluss view[0] zugewiesen ist.
