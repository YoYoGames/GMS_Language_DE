camera_set_view_mat

Mit dieser Funktion wird die Ansichtsmatrix für eine bestimmte Kamera eingestellt. Sie geben den eindeutigen Kamera-ID-Wert (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und eine anzuwendende Ansichtsmatrix. Sie können sich mehr über das Erstellen von Ansichtsmatrizen aus dem Abschnitt Matrixfunktionen, insbesondere matrix_build_lookat(). Beachten Sie, dass, wenn Ihre Kamera eine automatische Objektverfolgung durchführt - d. h., sie wurde mit kamera_erzeugen_ansicht() mit einem Objektindex / einer Instanz-ID, die nicht -1 ist, oder Sie setzen eine Kamera, die im Raum-Editor definiert ist, oder Sie die Standard-Kamera einstellen - dann wird diese Matrix im nächsten Spiel-Frame überschrieben.

Syntax:

camera_set_view_mat(camera_id, matrix)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
Matrix Die eindeutige Matrix-ID, die beim Erstellen der Matrix zurückgegeben wurde.

Rückgabe:

N/A

Beispiel:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erstellt eine neue Kamera und weist sie dem View-Port[0] zu. Dieser Kamera werden dann ihre Ansichts- und Projektionsmatrizen zugewiesen.
