camera_set_end_script

Mit dieser Funktion kann eine Skriptfunktion festgelegt werden, die am Ende jedes Spielframes aufgerufen wird, wenn die Kamera einem sichtbaren und aktiven Viewport zugewiesen ist, nachdem alles für diese Kamera gerendert wurde. Sie geben den eindeutigen Kamera-ID-Wert (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und die ID der aufzurufenden Funktion an. Die Reihenfolge, in der die an die Kameras angehängten Funktionen und dem eigentlichen Rendering der Kameraansicht aufgerufen werden, ist wie folgt:

  • Die Kameras für alle sichtbaren und aktiven Ansichtsports haben ihre Aktualisierungsfunktion mit
  • Dann für jede Ansicht:
    • Die begin-Funktion für die Kamera für diese Ansicht wird aufgerufen
    • Die Zeichenereignisse werden für diese Ansicht ausgeführt
    • Die Endfunktion für die Kamera wird aufgerufen
  • Zur nächsten aktiven sichtbaren Ansicht wechseln und wiederholen...

Syntax:

camera_set_end_script(camera_id, script)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
Skript Das Skript zum Ausführen jedes Spiel-Frames

Rückgabe:

N/A

Beispiel:

camera_set_end_script(view_camera[0], endCamera);

Der obige Code setzt das Endskript für die Kamera, die dem Ansichtsport [0] zugewiesen ist.
