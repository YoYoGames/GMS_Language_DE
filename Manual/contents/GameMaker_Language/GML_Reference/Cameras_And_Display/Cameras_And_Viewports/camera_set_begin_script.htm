camera_set_begin_script

Mit dieser Funktion können Sie eine Skriptfunktion festlegen, die zu Beginn jedes Spielframes aufgerufen wird, in dem die Kamera einem sichtbaren und aktiven Viewport zugewiesen ist. Sie geben den eindeutigen Kamera ID-Wert (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und die ID der aufzurufenden Funktion. Die Reihenfolge, in der Skripte an Kameras angehängt werden, und das tatsächliche Rendern der Kameraansicht ist wie folgt:

  • Die Kameras für alle sichtbaren und aktiven Ansichtsports haben ihre Aktualisierungsfunktion mit
  • Dann für jede Ansicht:
    • Die begin-Funktion für die Kamera für diese Ansicht wird aufgerufen
    • Die Zeichenereignisse werden für diese Ansicht ausgeführt
    • Die Endfunktion für die Kamera wird aufgerufen
  • Zur nächsten aktiven sichtbaren Ansicht wechseln und wiederholen...

Syntax:

camera_set_begin_script(camera_id, script)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
Skript Die Skriptfunktion zum Ausführen jedes Spielframes

Rückgabe:

N/A

Beispiel:

camera_set_begin_script(view_camera[0], beginCamera);

Der obige Code setzt die Skriptfunktion begin für die Kamera, die dem Ansichtsanschluss [0] zugewiesen ist.
