view_get_yport

Mit dieser Funktion können Sie die y-Position des angegebenen View Ports abrufen.

Syntax:

view_get_yport(view_port)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)

Rückgabe:

Real

Beispiel:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

Der obige Code prüft die y-Position des View Ports[0] und wenn sie nicht an der gewünschten Stelle ist, wird sie auf diese Position gesetzt.
