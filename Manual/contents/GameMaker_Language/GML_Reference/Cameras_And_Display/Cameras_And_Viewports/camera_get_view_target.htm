camera_get_view_target

Mit dieser Funktion kann das Verfolgungsziel der angegebenen Kamera abgerufen werden. Der Rückgabewert kann ein Objektindex, eine Instanz-ID oder -1 sein, wenn kein Verfolgungsziel gesetzt wurde.

Syntax:

camera_get_view_target(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Objektindex, Instanz-ID oder -1, wenn kein Ziel festgelegt ist.

Beispiel:

var target = camera_get_view_target(view_camera[0]);
if target != obj_Player
{
camera_set_view_target(view_camera[0], obj_Player);
}

Der obige Code ruft das Verfolgungsziel der Kamera ab, die dem Ansichtsport[0] zugewiesen ist, und prüft dann, ob es mit dem Objektindex "obj_Player" übereinstimmt. Wenn dies nicht der Fall ist, wird die Ansichtskamera so eingestellt, dass sie einer Instanz dieses Objekts folgt.
