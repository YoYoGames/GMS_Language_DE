camera_get_update_script

Mit dieser Funktion kann die ID der Skriptfunktion abgefragt werden, die als Update-Skript für die angegebene Kamera zugewiesen ist. Wenn kein Skript zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

camera_get_update_script(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Skriptfunktions-ID oder -1, wenn keine Funktion zugewiesen ist

Beispiel:

var scr = camera_get_update_script(camera_view[0]);
if scr != scr_cutscene
{
camera_set_update_script(camera_view[0], scr_cutscene);
}

Der obige Code prüft die Skriptfunktion, die als Aktualisierungsfunktion für die dem Ansichtsport[0] zugewiesene Kamera zugewiesen ist, und wenn sie nicht "cutScene" ist, wird sie auf dieses Skript gesetzt.
