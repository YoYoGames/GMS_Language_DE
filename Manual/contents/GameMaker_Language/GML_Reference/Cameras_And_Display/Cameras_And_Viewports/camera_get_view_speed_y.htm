camera_get_view_speed_y

Mit dieser Funktion können Sie die Bewegungsgeschwindigkeit der angegebenen Kamera entlang der y-Achse (vertikale Bewegung) abfragen. Der Rückgabewert wird in Pixel pro Spielframe angegeben.

Syntax:

camera_get_view_speed_y(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Real

Beispiel:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
camera_set_view_speed(view_camera[0], 5, 5);
}

Der obige Code ruft die x- und y-Geschwindigkeit der Kamera ab, die dem Ansichtsport[0] zugewiesen ist, und prüft dann, ob diese mit dem angegebenen Wert übereinstimmt. Wenn dies nicht der Fall ist, wird die Fachkamera auf diese Geschwindigkeit eingestellt.
