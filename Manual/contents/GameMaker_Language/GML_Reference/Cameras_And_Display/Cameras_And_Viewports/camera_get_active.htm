camera_get_active

Mit dieser Funktion kann der eindeutige Kamera-ID-Wert der aktuell aktiven Kamera abgefragt werden.

Syntax:

camera_get_active()

Rückgabe:

Kamera-Index (real).

Beispiel:

var active = camera_get_active();
if active != view_camera[0]
{
view_camera[0] = active;
}

Der obige Code ruft die Kamera-ID für die aktive Kamera ab und setzt die Fachkamera für port[0], um sie zu verwenden.
