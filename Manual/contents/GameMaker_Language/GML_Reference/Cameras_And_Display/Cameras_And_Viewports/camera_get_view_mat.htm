camera_get_view_mat

Mit dieser Funktion kann die Ansichtsmatrix abgefragt werden. Die Funktion gibt den Matrix-ID-Wert zurück, der dann in anderen Matrix-Funktionen verwendet werden kann oder zum Setzen der Ansichtsmatrix einer einer anderen Kamera (mit camera_set_view_mat()). Beachten Sie, dass diese Funktion in der Regel für Kameras verwendet wird, die mit kamera_erzeugen()erstellt wurden, aber sie kann auch verwendet werden auf Kameras angewendet werden, die im Raum-Editor (oder mit camera_create_view()) - in diesem Fall ist die zurückgegebene Matrix erst gültig, wenn die angegebene Kamera zum ersten Mal in einem Ansichtsfenster verwendet wird.

Syntax:

camera_get_view_mat(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Matrix-ID (real)

Beispiel:

mat = camera_get_view_mat(camera_view[0]);

Der obige Code ruft die Ansichtsmatrix für die Kamera ab, die dem Ansichtsanschluss[0] zugewiesen ist.
