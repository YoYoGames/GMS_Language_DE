view_set_wport

Mit dieser Funktion können Sie die Breite des angegebenen Ansichtsports einstellen. Sie geben den Index des Ansichtsports (von 0 bis 7) und die zu verwendende neue Breite an.

Syntax:

view_set_wport(view_port, w)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)
w Die neue Breite (in Pixel)

Rückgabe:

Real

Beispiel:

if view_get_wport(0) != (display_get_width () / 2)
{
view_set_wport(0, display_get_width() / 2);
}

Der obige Code prüft die Breite des View Ports[0] und wenn sie nicht die Hälfte der Anzeigebreite beträgt, wird sie auf diesen Wert gesetzt.
