camera_set_view_target

Mit dieser Funktion können Sie das Verfolgungsziel der Fachkamera innerhalb des Raumes einstellen. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben Sie dann die Zielinstanz oder die Objekt-ID an, auf die Sie die Kameraansicht einstellen möchten. Beachten Sie, dass, wenn Sie eine Objekt-ID wählen und es mehr als eine Instanz dieses Objekts im Raum gibt, GameMaker Studio 2 keine Möglichkeit hat, zu wissen, welcher Instanz Sie folgen wollen. also könnte es eine beliebige von ihnen sein.

Syntax:

camera_set_view_target(camera_id, instance_id/object_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde
Instanz_id/Objekt_id Instanz oder Objekt, auf das die Kamera bei der Verfolgung zielen soll

Rückgabe:

N/A

Beispiel:

camera_set_view_target(view_camera[0], id);

Der obige Code setzt das Ziel der Ansichtskamera für die Kamera, die dem Ansichtsanschluss[0] zugewiesen ist, auf das der Instanz, die den Code ausführt.
