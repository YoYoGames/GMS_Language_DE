Ansicht_Kamera

Dieses Array enthält die eindeutige Kamera-ID, die dem angegebenen Ansichtsanschluss zugewiesen ist, und kann auf eine neue Kamera gesetzt oder gelesen werden, um die aktuelle Kamera zu erhalten, wobei -1 zurückgegeben wird, wenn keine Kamera zugewiesen ist. Sie können bis zu 8 Ansichtsfenster in einem Raum aktiv haben (Array-Werte 0 bis 7), und Sie können jedem von ihnen eine Kamera zuweisen, indem Sie diese Variable einfach auf den Wert der Kamera-ID setzen (wie von den Funktionen kamera_erzeugen() oder camera_create_view()). Wenn Sie die Kamera über den Raumeditor einem Ansichtsanschluss zugewiesen haben, enthält diese Variable die Kamera-ID für diesen Anschluss.

Syntax:

view_camera[0 ... 7];

Rückgabe:

Kamera-Index oder -1, wenn keine Kamera zugeordnet ist

Beispiel:

if view_camera[0] == -1
{
view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

Der obige Code prüft, ob an den Ansichtsanschluss [0] eine Kamera angeschlossen ist, und wenn nicht, wird eine erstellt und auf diesen eingestellt.
