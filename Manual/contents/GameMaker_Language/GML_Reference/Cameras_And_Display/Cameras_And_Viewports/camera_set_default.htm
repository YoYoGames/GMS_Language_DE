camera_set_default

Mit dieser Funktion können Sie die Standardkamera so einstellen, dass eine benutzerdefinierte Kamera verwendet wird, die Sie zuvor mit einer der camera_create() Funktionen erstellt haben. Wenn Sie einen Raum ohne aktive Ansichtsports oder Ansichtskameras erstellen, verwendet GameMaker Studio 2 dennoch eine Kamera, um das Geschehen im Spiel zu zeigen. Diese Kamera wird als Standardkamera bezeichnet und kann wie jede andere Kamera gesetzt und manipuliert (und sogar zerstört) werden.

Syntax:

camera_set_default(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

N/A

Beispiel:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

Der obige Code erstellt eine neue Kamera und weist sie dann als zu verwendende Standardkamera zu.
