kamera_zerstören

Mit dieser Funktion können Sie eine beliebige Kamera zerstören. Beim Aufruf der Funktion übergeben Sie den eindeutigen Kamera-ID-Wert, den Sie aus den camera_create_*-Funktionen oder aus dem Ansicht_Kamera Array, wenn Sie den Raum-Editor verwenden, um den Ansichtsanschluss und die Ansicht einzurichten. Sie sollten niemals eine Kamera zerstören, die gerade einer sichtbaren Ansicht zugewiesen ist, es sei denn, Sie weisen dieser Ansicht im selben Schritt eine neue Kamera zu, und Sie sollten immer alle Kameras zerstören Sie können auch die Standardkameras zerstören, wenn Sie welche im Raum-Editor zugewiesen haben, aber Sie sollten dem Ansichtsanschluss eine neue Kamera zuweisen (oder sie deaktivieren), da Sie sonst seltsame Ergebnisse erhalten. deaktivieren), sonst erhalten Sie merkwürdige Ergebnisse.

Syntax:

camera_destroy(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.

Rückgabe:

Real (eindeutiger Kamera-ID-Wert)

Beispiel:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

Der obige Code zerstört die Kamera, die derzeit dem Ansichtsanschluss [0] zugewiesen ist, erstellt dann eine neue Kamera und weist ihre ID diesem Anschluss zu.
