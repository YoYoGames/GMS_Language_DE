anzeigen_sichtbar

Diese Variable kann verwendet werden, um herauszufinden, ob ein bestimmter Ansichtsanschluss derzeit sichtbar ist oder nicht. Sie können diese Variable auch setzen, um eine Ansicht effektiv "ein-" oder "auszuschalten", indem Sie den Wert auf true (sichtbar) oder false (unsichtbar) setzen . Beachten Sie, dass selbst wenn Sie einen Ansichtsanschluss auf "sichtbar" gesetzt haben, wenn die Ansichtsanschlüsse nicht aktiviert sind (mithilfe der integrierten Variable view_enabled oder im Raum-Editor aktivieren), werden sie nicht auf dem Bildschirm dem Bildschirm angezeigt.

Syntax:

view_visible[0 ... 7];

Rückgabe:

Boolean(true: aktiviert, false: deaktiviert)

Beispiel:

if !view_visible[0]
{
view_visible[0] = true;
}

Der obige Code prüft, ob view[0] sichtbar ist oder nicht, und wenn nicht, dann wird es auf sichtbar gesetzt.
