sicht_setzen_sichtbar

Diese Funktion kann verwendet werden, um die Sichtbarkeit des angegebenen Ansichtsports einzustellen. Die Funktion nimmt den Index des Ansichtsports (von 0 bis 7) und ein boolesches true, wenn er sichtbar ist, und false, wenn er nicht sichtbar ist.

Syntax:

view_set_visible(view_port, visible)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)
sichtbar Sichtbarkeitsumschaltung(true ist sichtbar und false ist unsichtbar)

Rückgabe:

N/A

Beispiel:

if !view_get_visible(0)
{
view_set_visible(0, true);
}

Der obige Code prüft, ob der View Port[0] sichtbar ist, und wenn nicht, wird er auf sichtbar gesetzt.
