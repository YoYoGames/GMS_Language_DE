camera_get_view_width

Mit dieser Funktion können Sie die Breite (in Pixel) der angegebenen Kamera abfragen. Beachten Sie, dass diese Funktion nur für Kameras gültig ist, die mit camera_create_view() erstellt wurden oder für solche, die im Raum Editor.

Syntax:

camera_get_view_width(camera_id)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde

Rückgabe:

Real

Beispiel:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Der obige Code ruft die Breite und Höhe der Kamera ab, die dem Ansichtsport[0] zugewiesen ist, und setzt dann ihre Position relativ zur Mitte.
