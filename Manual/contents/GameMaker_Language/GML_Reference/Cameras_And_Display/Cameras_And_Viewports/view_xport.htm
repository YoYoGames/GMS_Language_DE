Ansicht_Export

Mit diesem eingebauten Array können Sie die x-Position des angegebenen Ansichtsfensters abrufen oder einstellen. Das Ansichtsfenster ist der Bereich auf dem Bildschirm, in dem die Ansicht gezeichnet wird, und es können bis zu 8 gleichzeitig aktiv sein (das Array enthält Werte von 0 bis einschließlich 7, so dass 8 Ports). Die Vorgabe für GameMaker Studio 2 ist, dass das Spielfenster (oder die Hintergrundleinwand) die gleiche Größe wie der Raum hat. Wenn Sie jedoch View-Ports und Kameras aktivieren, ändert sich dieses Verhalten und es wird die Gesamtgröße der Bounding Box für alle View-Ports verwendet. Wenn Sie also zwei verschiedene Ansichtsfenster an zwei verschiedenen Positionen haben, definiert die Gesamtfläche, die sie abdecken, die Größe des Spielfensters. Das folgende Bild veranschaulicht dies:

Wenn Sie nur einen Ansichtsanschluss haben, kann das Einstellen des x- oder y-Wertes des Anschlusses einige interessante Effekte haben, ist aber normalerweise nicht praktikabel (siehe das Bild unten) und daher ist es besser, die x- und y-Position als (0,0) beizubehalten.

Syntax:

view_xport[0 .. 7];

Rückgabe:

Real

Beispiel:

view_xport[0] = 0;
view_yport[0] = 0;

Der obige Code setzt die Position von view port[0] auf die Position (0,0) (linke obere Ecke) des Displays zurück.
