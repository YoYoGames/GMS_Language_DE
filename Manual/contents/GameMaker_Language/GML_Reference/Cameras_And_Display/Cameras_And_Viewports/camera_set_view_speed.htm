camera_set_view_speed

Mit dieser Funktion können Sie die Geschwindigkeit der Fachkamera innerhalb des Raums aktualisieren. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben Sie dann die x- und y-Geschwindigkeit (horizontal und vertikale) Geschwindigkeit, mit der sie sich bewegen soll, wenn sie so eingestellt ist, dass sie einer bestimmten Instanz folgt. Die Geschwindigkeit wird als Pixel pro Schritt berechnet und kann auf "-1" gesetzt werden, damit sich die Kamera sofort bewegt, aber wenn die Kamera nicht so eingestellt ist, dass sie einer Instanz folgt, haben die Wenn die Kamera jedoch keiner Instanz folgt, haben die hier eingestellten Werte keinen sichtbaren Effekt.

Syntax:

camera_set_view_speed(camera_id, xspeed, yspeed)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
xspeed Die Geschwindigkeit (Anzahl der Pixel pro Spielframe), mit der sich die Ansicht auf der horizontalen (x) Achse bewegen soll
yspeed Die Geschwindigkeit (Anzahl der Pixel pro Spielframe), mit der sich die Ansicht auf der vertikalen (y) Achse bewegen soll

Rückgabe:

N/A

Beispiel:

camera_set_view_speed(view_camera[0], 5, 5);

Der obige Code stellt die Geschwindigkeit der Fachkamera für die Kamera ein, die dem Anschluss view[0] zugewiesen ist.
