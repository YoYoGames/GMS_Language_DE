Ansicht_aktuell

Diese nur lesbare Variable ist nur im Ereignis Zeichnen und gibt den aktuell gerenderten Ansichtsanschluss zurück. Der Rückgabewert ändert sich während des Draw-Ereignisses wenn Sie verschiedene Ansichten haben, da das Draw-Ereignis einmal für jeden View-Port nacheinander aufgerufen wird. Wenn Sie also (zum Beispiel) view port[0] und view port[1] in Ihrem Spielraum verwenden, wird das draw-Ereignis für ALLE Instanzen zweimal ausgeführt, einmal für jeden Port, und mit dieser Variablen können Sie überprüfen, welche Ansicht gerade gezeichnet wird. Im Allgemeinen wird dies verwendet, um nur bestimmte Details an einen einzelnen Port zu rendern, wenn mehrere View-Ports gleichzeitig im Raum sichtbar sind. Siehe das Beispiel Code unten.

Syntax:

view_current;

Rückgabe:

Integer (Ansicht Port-Index von 0 bis 7)

Beispiel:

if view_current == 0
{
var xx = camera_get_view_x(view_camera[0]);
var yy = camera_get_view_y(view_camera[0]);
draw_text(xx + 32, yy + 32, "Spieler 1");
}
sonst
{
var xx = camera_get_view_x(view_camera[1]);
var yy = camera_get_view_y(view_camera[1]);
draw_text(xx + 32, yy + 32, "Spieler 2");
}

Der obige Code prüft, welche Ansicht gerade gezeichnet wird, und zeichnet dann auf der Grundlage des Rückgabewerts für jede Ansicht einen anderen Text.
