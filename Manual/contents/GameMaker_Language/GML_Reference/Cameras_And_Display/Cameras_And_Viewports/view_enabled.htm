view_enabled

Diese Variable steuert, ob alle View Ports, die innerhalb des Raums sichtbar sind, aktiviert sind oder nicht. Wenn Sie die Ansichtsfenster auf "sichtbar" eingestellt haben und diese Option dann deaktivieren, wird der gesamte Raum auf den Bildschirm gezeichnet, der auf die Fenstergröße skaliert ist, anstatt die die verschiedenen Kameras durch die Ansichtsfenster gezeichnet werden.

Syntax:

view_enabled;

Rückgabe:

Boolean(true: aktiviert, false: deaktiviert)

Beispiel:

if !view_enabled
{
view_visible[0] = true;
view_enabled = true;
}

Der obige Code prüft, ob Ansichten für den Raum aktiviert sind, und wenn dies nicht der Fall ist, stellt er sicher, dass view port[0] sichtbar ist und aktiviert dann Ansichten für den Raum.
