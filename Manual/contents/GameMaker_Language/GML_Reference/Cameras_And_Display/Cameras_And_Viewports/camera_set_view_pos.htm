camera_set_view_pos

Mit dieser Funktion können Sie die Position der Kameraansicht innerhalb des Raums aktualisieren. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben Sie dann die x- und y-Positionen auf die die Kamera gesetzt werden soll.

Syntax:

camera_set_view_pos(camera_id, x, y)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
x Die x-Position, an der die Ansicht platziert werden soll (im Raum)
y Die y-Position, an der die Ansicht platziert werden soll (im Raum)

Rückgabe:

N/A

Beispiel:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

Der obige Code setzt die Position der Fachkamera für die Kamera, die dem Anschluss view[0] zugewiesen ist.
