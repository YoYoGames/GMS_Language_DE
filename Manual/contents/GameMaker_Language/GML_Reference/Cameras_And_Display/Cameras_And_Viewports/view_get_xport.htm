view_get_xport

Mit dieser Funktion können Sie die x-Position des angegebenen View Ports abrufen.

Syntax:

view_get_xport(view_port)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)

Rückgabe:

Real

Beispiel:

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

Der obige Code prüft die x-Position des View Ports[0] und wenn er nicht an der gewünschten Position ist, wird er auf diese Position gesetzt.
