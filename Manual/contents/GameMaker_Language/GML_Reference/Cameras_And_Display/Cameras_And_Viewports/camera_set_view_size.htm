camera_set_view_size

Sie können diese Funktion verwenden, um die Größe der Fachkamera innerhalb des Raums zu aktualisieren. Sie geben den eindeutigen Kamera-ID-Wert an (wie er von den verschiedenen camera_create() Funktionen zurückgegeben wird) und geben Sie dann die Breite und Höhe (in Pixel) an, auf die die Kamera eingestellt werden soll.

Syntax:

camera_set_view_size(camera_id, width, height)

Argument Beschreibung
Kamera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
Breite Die Breite der Kameraansicht in Pixel
Höhe Die Höhe der Kameraansicht in Pixel

Rückgabe:

N/A

Beispiel:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

Der obige Code stellt die Größe der Fachkamera für die Kamera ein, die dem Anschluss view[0] zugewiesen ist.
