view_get_camera

Mit dieser Funktion kann der eindeutige Kamera-ID-Wert für die Kamera abgerufen werden, die dem angegebenen Ansichtsanschluss zugewiesen ist (von 0 - 7). Wenn keine Kamera zugewiesen ist, gibt die Funktion -1 zurück.

Syntax:

view_get_camera(view_port)

Argument Beschreibung
Ansicht_Port Der anzuvisierende Ansichtsanschluss (0 - 7)

Rückgabe:

Kamera-Index (Real)

Beispiel:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Der obige Code ruft die Kamera-ID für die Kamera ab, die dem Ansichtsanschluss[0] zugewiesen ist, und ändert dann ihre Position.
