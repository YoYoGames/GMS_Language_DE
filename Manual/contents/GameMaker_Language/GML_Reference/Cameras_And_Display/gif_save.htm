gif_sichern

Mit dieser Funktion können Sie eine GIF-Animation ausspeichern. Sie übergeben den GIF-Index (wie er von der Funktion gif_open()zurückgegeben wird) sowie einen Dateinamen, unter dem sie gespeichert werden soll. Beachten Sie, dass GameMaker Studio 2 nicht automatisch die gif-Dateierweiterung anhängt, so dass Sie diese als Teil des Dateinamens angeben sollten, wenn die gespeicherte Datei als GIF identifiziert werden soll. Das erzeugte GIF wird mit der Universal 884 Palette palettiert (weitere Informationen finden Sie hier ).

Beachten Sie, dass die Funktion 0 zurückgibt, wenn sie erfolgreich ist und das GIF korrekt gespeichert wurde, andernfalls gibt sie -1 zurück.

Syntax:

gif_save(gif_index, fname);

Argument Beschreibung
gif_index Die ID des zu speichernden Gifs
fname Der Dateiname, der für das gif verwendet werden soll

Rückgabe:

Integer

Beispiel:

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
sonst
{
gif_save(gif_image, "GameCapture.gif");
count = 0;
save_gif = false;
}
count++;
}

Der obige Code erstellt eine GIF-Bilddatei mit 30 Bildern aus der Anwendungsoberfläche und speichert sie anschließend.
