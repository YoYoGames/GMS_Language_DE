display_mouse_get_x

Diese Funktion gibt die x-Position der Maus auf dem Bildschirm zurück. Es ist zu beachten, dass diese Funktion nur auf dem Windows-Ziel richtig funktioniert. Sie kann auch für HTML5 verwendet werden, gibt aber nur einen Wert relativ zu 0,0 des Canvas selbst zurück und gibt keinen Wert zurück, wenn sich die Maus außerhalb des Canvas befindet. Für andere Geräte wird der Wert 0 zurückgegeben, und Sie sollten die device_mouse_raw_x()und verwenden. device_mouse_raw_y() Funktionen verwenden.

Syntax:

display_mouse_get_x();

Rückgabe:

Real

Beispiel:

my_x = display_mouse_get_x();

Dies würde my_x auf die x-Koordinate der Maus relativ zur Anzeige setzen.
