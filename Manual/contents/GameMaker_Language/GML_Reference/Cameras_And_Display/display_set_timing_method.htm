display_set_timing_method

Mit dieser Funktion können Sie die Timing-Methode einstellen, die für das Rendern Ihres Spiels verwendet werden soll. Die Methode kann eine der folgenden Konstanten sein:

Konstant Beschreibung
tm_sleep Verwenden Sie den Einschlafspannenwert als Haupt-Timing-Methode
tm_countvsyncs vsync-Timing als Haupt-Timing-Methode verwenden (Standard für alle unterstützten Plattformen)

Die vsync-Timing-Methode verwendet die Unterstützung der Zielplattform für die vertikale Synchronisation, um einen Anker für die Rendering-Timing-Berechnungen des Spiels bereitzustellen, während die Einstellung des Modus auf "Sleep Margin" nur versucht, sicherzustellen, dass jedes Bild für die korrekte Zeitspanne dauert (z. B. 1/30 oder 1/60 Sekunde), indem gewartet oder geschlafen wird. Im Allgemeinen liefert das voreingestellte vsync-Timing die gleichmäßigsten Ergebnisse, aber beachten Sie, dass bei Verwendung der vsync-Methode die Werte für die Schlafspanne immer noch relevant sind, obwohl sie einen geringeren Einfluss haben. Wir empfehlen daher, den Standardwert beizubehalten, wenn Sie diese Methode verwenden.

Standardmäßig verwendet GameMaker Studio 2 auf allen Plattformen außer PS4, Ubuntu und HTML5 die vsync-Timing-Methode, während auf den nicht unterstützten Plattformen nur das Sleep-Margin-Timing verwendet werden kann.

Wenn Sie den Schlafabstand einstellen möchten, können Sie dies mit der Funktion display_set_sleep_margin() und welche Timing-Methode gerade verwendet wird, können Sie mit der Funktion display_get_timing_method().

Syntax:

display_set_timing_method(method);

Argument Beschreibung
Methode Die zu verwendende Timing-Methode (siehe die Liste der Konstanten, oben)

Rückgabe:

N/A

Beispiel:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
{
display_set_sleep_margin(20);
}
}

Der obige Code prüft die Timing-Methode, und wenn sie nicht auf tm_sleep eingestellt ist, wird sie auf diesen Wert gesetzt und die Sleep-Marge auf 20 gesetzt.
