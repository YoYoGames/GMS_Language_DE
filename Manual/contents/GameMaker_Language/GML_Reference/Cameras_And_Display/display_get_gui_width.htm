display_get_gui_width

Mit dieser Funktion können Sie die Breite (in Pixel) der GUI ermitteln, wie sie im Draw GUI Event verwendet wird.

Syntax:

display_get_gui_width();

Rückgabe:

Real

Beispiel:

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

Der obige Code legt fest, dass eine Anzeige oben rechts auf dem Display erscheint.
