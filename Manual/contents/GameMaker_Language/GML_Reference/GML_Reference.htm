GameMaker-Sprachreferenz

Dieser Teil des Handbuchs ist ein Referenzhandbuch für das GameMaker Studio 2 Sprache(GML). Sie finden hier alle verfügbaren Funktionen mit den erforderlichen Argumenten und Code-Beispielen dokumentiert, die zeigen, wie sie verwendet werden können. Informationen zur Verwendung von GML finden Sie im Abschnitt GML-Übersicht Abschnitt des Handbuchs.

HINWEIS: Überall dort, wo im Handbuch "N/A" als Rückgabewert für eine Funktion angegeben ist, bedeutet dies, dass die Funktion keinen Wert als Teil ihrer Operation zurückgeben soll. Wenn Sie versuchen, nach einem Rückgabewert dieser Funktionen zu suchen, erhalten Sie erhalten Sie den Wert undefined.
