string_last_pos_ext

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, indem sie die Zeichenkette von der als Startposition angegebenen Position aus rückwärts durchsucht. Die Funktion gibt 0 zurück, wenn die Suchzeichenfolge nicht gefunden wird, oder die Position des ersten Zeichens der Suchzeichenfolge, wenn sie gefunden wird. Beachten Sie, dass für die Legacy-Unterstützung Zeichenketten ab 1 indiziert werden, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_last_pos_ext(substr, str, start_pos);

Argument Beschreibung
substr Die Teilzeichenkette, nach der in der Zeichenkette gesucht werden soll.
str Die Zeichenfolge.
start_pos Die Startposition, von der aus gesucht werden soll.

Rückgabe:

Integer

Beispiel:

if (string_last_pos_ext(",", text, 20) != 0)
{
string_insert(name, text, string_last_pos_ext(",", text, 20));
}

Der obige Code sucht in der Zeichenkette, die in der Variablen "text" gespeichert ist, nach einem Komma vor dem 20. Zeichen, und wenn er eines findet, fügt er die Teilzeichenkette "name" an dieser Position ein.
