string_ord_at

Mit dieser Funktion können Sie einen bestimmten Zeichencode an einer bestimmten Position innerhalb einer Zeichenkette zurückgeben, wobei der Index bei 1 für das erste Zeichen beginnt. Wenn kein Zeichen gefunden wird oder die Zeichenkette kürzer ist als der für index angegebene Wert, wird -1 zurückgegeben.

Syntax:

string_ord_at(str, index);

Argument Beschreibung
str Die zu prüfende Zeichenkette.
index Die Position, von der der Zeichencode abgerufen werden soll.

Rückgabe:

Real

Beispiel:

str = "Hello World";
char_code = string_ord_at(str, 7);

Damit wird der Zeichencode für das siebte Zeichen (wobei "H" als erstes Zeichen zählt) in der Zeichenkette "str" ermittelt und in der Variablen "char_code" gespeichert.
