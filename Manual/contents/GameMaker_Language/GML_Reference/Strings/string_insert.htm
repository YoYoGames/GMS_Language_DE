string_einfügen

Mit dieser Funktion können Sie einen neuen String erstellen, der aus zwei Strings besteht, wobei der eine an einer bestimmten Position in den anderen eingefügt wurde. Dies kann z. B. nützlich sein, um einen Benutzernamen in einen vordefinierten Text einzufügen und so dem Spieler Ihres Spiels das Gefühl zu geben mehr in das Geschehen einzubeziehen. Beachten Sie bei der Berechnung der Position, an der eingefügt werden soll, dass Strings von 1 an indiziert werden, so dass das erste Zeichen in einem String 1 ist und nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_insert(substr, str, index);

Argument Beschreibung
substr Die Teilzeichenkette, die eingefügt werden soll.
str Die zu kopierende Zeichenkette.
index Die Position in Zeichen der Zeichenkette, an der die Teilzeichenkette eingefügt werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

str2 = string_insert(username, "Hallo, , wie geht es Ihnen?", 8);

Dies fügt die Zeichenkette in der Variablen "username" in die angegebene Phrase ein, wobei die resultierende Zeichenkette wie folgt aussieht: "Hallo, NAME, wie geht es Ihnen?".
