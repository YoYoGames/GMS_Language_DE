String_Buchstaben

Mit dieser Funktion können Sie alle Zeichen entfernen, die nicht als Buchstaben klassifiziert sind. Dies ist praktisch, um zu verhindern, dass Personen unerwünschte Zeichen in eine Texteingabe eingeben (z. B. die Eingabe von "#", um eine neue Zeile zu erzwingen).

HINWEIS: Diese Funktion erkennt und gibt nur das englische Alphabet mit 26 Buchstaben von A - Z (Groß- oder Kleinschreibung) wieder.

Syntax:

string_letters(string);

Argument Beschreibung
String Die in Buchstaben zu konvertierende Zeichenfolge.

Rückgabe:

Zeichenfolge

Beispiel:

username = string_letters(username);

Der obige Code setzt die Variable "username" so, dass sie nur die Buchstaben der ursprünglichen Zeichenkette enthält (wenn die ursprüngliche Zeichenkette "MacSw33ny" wäre, würde die zurückgegebene Zeichenkette "MacSwny" sein).
