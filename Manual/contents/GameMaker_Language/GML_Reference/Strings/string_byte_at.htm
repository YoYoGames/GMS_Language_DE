string_byte_at

Gibt den rohen Byte-Wert als realen Wert an einer bestimmten Position in der angegebenen Zeichenkette zurück.

Syntax:

string_byte_at(str, index);

Argument Beschreibung
str Die zu prüfende Zeichenkette.
index Die Position, von der das Byte geholt werden soll.

Rückgabe:

Real

Beispiel:

newbyte = string_byte_at("Hello World", 5);

Dadurch wird newbyte auf den Rohbyte-Wert des sechsten Buchstabens von "Hello World" gesetzt.
