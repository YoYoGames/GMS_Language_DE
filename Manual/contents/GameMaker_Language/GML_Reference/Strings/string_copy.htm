string_copy

Mit dieser Funktion können Sie ganz einfach eine Anzahl von Zeichen aus einer Zeichenkette auswählen, die in eine andere kopiert werden sollen. Das erste Zeichen in einer Zeichenkette wird immer als 1 indiziert und nicht als 0, wie Sie vielleicht erwarten, so dass Sie zum Kopieren (zum Beispiel) der ersten fünf Zeichen einer Zeichenkette zu kopieren, würden Sie string_copy(str, 1, 5) verwenden. Ein weiteres Beispiel finden Sie weiter unten.

Syntax:

string_copy(str, index, count);

Argument Beschreibung
str Die Zeichenkette, aus der kopiert werden soll.
index Die Position des ersten Zeichens in der Zeichenkette, aus der kopiert werden soll (nummeriert ab 1)
zählen Die Anzahl der zu kopierenden Zeichen, ausgehend von der Position des ersten Zeichens

Rückgabe:

Zeichenfolge

Beispiel:

name = keyboard_string;
if (string_length(name) > 15)
{
keyboard_string = string_copy(name, 1, 15);
}

Mit dem obigen Code kann der Spieler eine Zeichenfolge über die Tastatur eingeben, die dann in der Variablen name gespeichert wird. Diese Variable wird dann daraufhin überprüft, ob sie die maximale Länge von 15 Zeichen überschreitet, und wenn dies der Fall ist, wird die Zeichenkette keyboard_string durch eine 15-stellige Kopie der Zeichenkette name ersetzt. Dadurch wird die Spielereingabe effektiv auf 15 Zeichen begrenzt.
