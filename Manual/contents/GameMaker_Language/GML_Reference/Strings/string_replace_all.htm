string_replace_all

Mit dieser Funktion können Sie eine Zeichenkette parsen und nach bestimmten Teilen suchen, die dann an allen Stellen, an denen sie vorkommen, durch die neue, von Ihnen angegebene Zeichenkette ersetzt werden können.

Syntax:

string_replace_all(str, substr, newstr);

Argument Beschreibung
str Die zu kopierende Zeichenkette.
substr Die Teilzeichenkette innerhalb der Zeichenkette, die in allen Instanzen ersetzt werden soll.
newstr Die neue Teilzeichenkette, die die vorherige(n) Teilzeichenkette(n) ersetzen soll.

Rückgabe:

Zeichenfolge

Beispiel:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Dadurch wird str2 auf str1 gesetzt, wobei jedoch die Instanzen von "x" durch "l" ersetzt werden, so dass str2 "Hello World" ist.
