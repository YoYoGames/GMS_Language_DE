string_height_ext

Diese Funktion gibt die Höhe (in Pixeln) der eingegebenen Zeichenkette zurück, unter Berücksichtigung des Zeilenabstands und der Zeilenumbruchsbreite (die als die Anzahl der Pixel definiert ist, die die Zeichenkette einnehmen kann, bevor ein Zeilenumbruch eingefügt wird). Es ist sehr praktisch ist sehr praktisch, um Abstände zwischen Textelementen zu berechnen, die auf dem höchsten der Buchstaben basieren, aus denen die Zeichenkette besteht, sowie die tatsächliche Länge der Zeichenkette selbst, wenn sie sich über mehrere Zeilen erstreckt, wie sie mit draw_text_ext() mit der aktuell definierten Schriftart gezeichnet würde. Abstand und Breite können auf -1 gesetzt werden, um die Standardabstände zu erhalten.

Syntax:

string_height_ext(string, sep, w);

Argument Beschreibung
String Der String, dessen Höhe gemessen werden soll.
sep Der Abstand in Pixeln zwischen den Textzeilen, als ob die Zeichenkette gezeichnet würde.
w Die maximale Breite (in Pixel) der Zeichenfolge vor einem Zeilenumbruch, als ob die Zeichenfolge gezeichnet würde.

Rückgabe:

Real

Beispiel:

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100);

Der obige Code ermittelt die Höhe der angegebenen Zeichenkette unter Berücksichtigung des Zeilenabstands und der Zeilenumbruchbreite und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Gesamthöhe der Zeichenkette als Trennzeichen verwendet wird.
