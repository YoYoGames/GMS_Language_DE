string_replace

Mit dieser Funktion können Sie eine Zeichenkette parsen und nach einem bestimmten Teil suchen, der dann durch die neue, von Ihnen angegebene Zeichenkette ersetzt werden kann.

Syntax:

string_replace(str, substr, newstr);

Argument Beschreibung
str Die zu kopierende Zeichenkette.
substr Die Teilzeichenkette innerhalb der zu ersetzenden Zeichenfolge.
newstr Die neue Teilzeichenkette, die die vorherige ersetzen soll.

Rückgabe:

Zeichenfolge

Beispiel:

str1 = "Hallo Erde";
str2 = string_replace(str1, "Erde", "Welt");

Dadurch wird str2 auf str1 gesetzt, wobei jedoch die Instanz von "Earth" durch "World" ersetzt wird, so dass str2 "Hello World" ist.
