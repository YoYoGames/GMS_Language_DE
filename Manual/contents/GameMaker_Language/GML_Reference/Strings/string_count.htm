string_count

Diese Funktion gibt zurück, wie oft die angegebene Teilzeichenkette innerhalb einer bestimmten Zeichenkette vorkommt. Auf diese Weise können Sie prüfen, wie oft ein einzelner Buchstabe oder eine Phrase in einem Abschnitt des gespeicherten Textes wiederholt wird.

Syntax:

string_count(substr, str);

Argument Beschreibung
substr Die Teilzeichenkette, auf die die Zeichenfolge geprüft werden soll.
str Die zu prüfende Zeichenkette.

Rückgabe:

Real

Beispiel:

str1 = "Hello World";
ocount = string_count( "o", str1 );

Dadurch wird die Variable ocount auf die Anzahl der "o "s in str1 gesetzt, in diesem Fall 2.
