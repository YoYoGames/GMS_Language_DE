string_height

Diese Funktion gibt die Höhe (in Pixel) der eingegebenen Zeichenkette zurück, wobei der Zeilenabstand und eventuelle Zeilenumbrüche des Textes berücksichtigt werden. Sie ist sehr praktisch, um Abstände zwischen Textelementen zu berechnen, basierend auf dem höchsten der Buchstaben die die Zeichenkette bilden, wie sie mit draw_text() unter Verwendung der aktuell definierten Schriftart gezeichnet würde.

Syntax:

string_height(string);

Argument Beschreibung
String Der String, dessen Höhe gemessen werden soll.

Rückgabe:

Real

Beispiel:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

Der obige Code ermittelt die Höhe der angegebenen Zeichenkette und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Höhe der Zeichenkette als Trennzeichen verwendet wird.
