string_width

Diese Funktion gibt die Breite (in Pixeln) der eingegebenen Zeichenkette zurück, wobei eventuelle Zeilenumbrüche im Text berücksichtigt werden. Sie ist sehr praktisch, um Abstände zwischen Textelementen zu berechnen, basierend auf der Gesamtbreite der Buchstaben, aus denen die Zeichenkette besteht wie sie mit draw_text() mit der aktuell definierten Schriftart gezeichnet würde.

Syntax:

string_width(string);

Argument Beschreibung
String Die Zeichenfolge, deren Breite gemessen werden soll.

Rückgabe:

Real

Beispiel:

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "hat das Spiel gewonnen!");

Der obige Code ermittelt die Breite der angegebenen Zeichenfolge und zeichnet dann zwei Textzeilen, wobei die zurückgegebene Zeichenfolgenbreite als Trennzeichen verwendet wird.
