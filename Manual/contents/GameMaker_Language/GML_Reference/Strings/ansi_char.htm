ansi_char

Diese Funktion gibt eine Zeichenkette zurück, die das Zeichen mit dem eingestellten rohen BYTE-Wert enthält. Dies wird nicht angezeigt und sollte auch nicht angezeigt werden, aber es wird für die Verwendung bei der Codierung korrekt auf der Festplatte gespeichert.

Syntax:

ansi_char(val);

Argument Beschreibung
val Der rohe Byte-Wert.

Rückgabe:

String (Einzelzeichen)

Beispiel:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

Der obige Code erstellt eine Zeichenkette aus Byte-Rohdaten und schreibt sie in eine (zuvor geöffnete) Datei.
