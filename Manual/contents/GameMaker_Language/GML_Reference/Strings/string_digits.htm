string_digits

Sie können diese Funktion verwenden, um eine gegebene Zeichenkette zu analysieren und beliebige Zahlen aus ihr zu erhalten. Nehmen wir zum Beispiel an, Sie haben diesen Text - "Ich bin 81 Jahre alt". Mit dieser Funktion würden Sie die Zeichenkette "81" zurückerhalten.

Syntax:

string_digits(string);

Argument Beschreibung
String Die Zeichenkette, aus der die Ziffern geholt werden.

Rückgabe:

Zeichenfolge

Beispiel:

var t_str = string_digits(input_str);
age = real(t_str);

Der obige Code nimmt die Eingabezeichenkette, entfernt alle Zeichen außer Zahlen und setzt dann die Variable age so, dass sie den realen Zahlenwert der Rückgabezeichenkette enthält (wenn die Eingabezeichenkette also zum Beispiel "Ich bin 18" wäre, würde die Funktion "18" zurückgeben).
