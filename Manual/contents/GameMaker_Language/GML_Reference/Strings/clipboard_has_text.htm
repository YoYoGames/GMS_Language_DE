zwischenablage_hat_text

Diese Funktion gibt true zurück, wenn die Zwischenablage Text enthält, oder false, wenn sie keinen Text enthält.

HINWEIS: Diese Funktion gilt nur für das Windows Ziel.

Syntax:

clipboard_has_text();

Rückgabe:

Boolesche

Beispiel:

if clipboard_has_text()
{
str = clipboard_get_text();
clipboard_set_text("");
}

Der obige Code prüft die Zwischenablage auf Text und wenn sie welchen enthält, wird dieser als String in die Variable "str" eingelesen. Abschließend wird die Zwischenablage geleert, indem sie auf einen leeren String gesetzt wird.
