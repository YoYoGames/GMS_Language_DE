string_char_at

Mit dieser Funktion können Sie ein bestimmtes Zeichen an einer bestimmten Position innerhalb einer Zeichenkette zurückgeben, wobei der Index bei 1 für das erste Zeichen beginnt. Wenn kein Zeichen gefunden wird oder die Zeichenkette kürzer ist als der für index angegebene Wert, wird eine leere Zeichenkette "" zurückgegeben.

Syntax:

string_char_at(str, index);

Argument Beschreibung
str Die zu prüfende Zeichenkette.
index Die Position, aus der das Zeichen geholt wird.

Rückgabe:

Zeichenfolge

Beispiel:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Dadurch wird str2 auf das siebte Zeichen ("H" zählt als 1.) in Zeichenkette str1 gesetzt, in diesem Fall "W".
