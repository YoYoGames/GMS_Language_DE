String

Mit dieser Funktion können Sie eine beliebige reelle Zahl in eine Zeichenkette umwandeln. Wenn die reelle Zahl eine ganze Zahl ist, wird sie ohne Nachkommastellen gespeichert, andernfalls wird sie mit zwei Nachkommastellen gespeichert. Wenn Sie mehr Nachkommastellen benötigen, dann verwenden Sie die Funktion string_format(). Beachten Sie auch, dass die Verwendung dieser Funktion auf eine Variable, die ein Array, eine Datenstruktur, oder eine Struktur speichert, den Inhalt dieser Variablen in eine Zeichenkette umwandelt, die dann auf der Konsole ausgegeben oder zur Fehlersuche in einer Datei gespeichert werden kann.

Syntax:

string(val);

Argument Beschreibung
String Die reelle Zahl, die in eine Zeichenkette umgewandelt werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

Der obige Code verwendet die String-Funktion, um sowohl reelle Zahlen als auch Strings zusammen zu zeichnen (draw akzeptiert nur entweder einen String oder einen Real, aber nicht beides).
