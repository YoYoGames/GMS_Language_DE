string_last_pos

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, wobei vom Ende der Zeichenkette zum Anfang gesucht wird (also die Umkehrung von string_pos(). Die Funktion gibt 0 zurück zurück, wenn die Suchzeichenkette nicht gefunden wird, oder die Position des ersten Zeichens der Suchzeichenkette, wenn sie gefunden wird. Beachten Sie, dass für die Legacy-Unterstützung Zeichenketten von 1 an indiziert werden, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten.

Syntax:

string_last_pos(substr, str);

Argument Beschreibung
substr Die Teilzeichenkette, nach der in der Zeichenkette gesucht werden soll.
str Die Zeichenfolge.

Rückgabe:

Integer

Beispiel:

if (string_last_pos(",", text) != 0)
{
string_insert(name, text, string_last_pos(",", text));
}

Der obige Code sucht in der in der Variablen "text" gespeicherten Zeichenkette nach einem Komma und beginnt die Suche am Ende der Zeichenkette. Wenn er eines findet, fügt er die Teilzeichenkette "name" an dieser Position ein.
