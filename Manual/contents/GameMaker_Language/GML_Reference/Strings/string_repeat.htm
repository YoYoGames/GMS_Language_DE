string_repeat

Diese Funktion gibt einfach die gleiche Zeichenkette zurück, die eine bestimmte Anzahl von Malen über sich selbst wiederholt wird.

Syntax:

string_repeat(str, count);

Argument Beschreibung
str Die zu wiederholende Zeichenfolge.
zählen Die Anzahl, wie oft die Zeichenfolge wiederholt werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Der obige Code setzt str2 auf "Hello WorldHello WorldHello World".
