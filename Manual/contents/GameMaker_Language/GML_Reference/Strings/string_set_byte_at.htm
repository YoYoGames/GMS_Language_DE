string_set_byte_at

Diese Funktion setzt ein Byte direkt in einem String (basierend auf dem UTF8-Format) und gibt eine Kopie des Strings mit den Änderungen zurück.

HINWEIS: Diese Funktion ist unglaublich langsam, überlegen Sie also genau, ob sie notwendig ist und wo Sie sie verwenden.

Syntax:

string_set_byte_at(str, pos, byte);

Argument Beschreibung
str Die Zeichenkette, deren Byte geändert werden soll.
pos Die Position innerhalb des Strings (beginnend bei 1), deren Byte geändert werden soll.
Byte Der neue Byte-Wert.

Rückgabe:

Zeichenfolge

Beispiel:

str = string_set_byte_at("hallo", 2, 97);

Der obige Code würde den Byte-Wert des zweiten Buchstabens in der Zeichenkette ändern und so die Variable str so setzen, dass sie "hallo" enthält.
