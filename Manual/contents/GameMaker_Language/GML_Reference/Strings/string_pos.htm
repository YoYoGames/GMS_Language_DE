string_pos

Diese Funktion gibt die Zeichenposition einer Instanz einer Teilzeichenkette innerhalb einer Zeichenkette zurück, wobei vom Anfang der Zeichenkette aus gesucht wird (um vom Ende her zu suchen, verwenden Sie die Funktion string_last_pos()). Die Funktion Funktion gibt 0 zurück, wenn sie überhaupt nicht gefunden wird, oder die Position des ersten Zeichens der Teilzeichenkette, wenn sie gefunden wird. Beachten Sie, dass für die Legacy-Unterstützung Zeichenketten von 1 an indiziert werden, so dass 1 die erste Position in der Zeichenkette ist, nicht 0, wie Sie vielleicht erwarten. Eine Verwendung hierfür ist das Filtern von Wörtern, die als anstößig angesehen werden könnten, oder das Finden der richtigen Stelle zum Einfügen von Text in eine andere Zeichenfolge.

Syntax:

string_pos(substr, str);

Argument Beschreibung
substr Die Teilzeichenkette, nach der in der Zeichenkette gesucht werden soll.
str Die Zeichenfolge.

Rückgabe:

Integer

Beispiel:

if (string_pos(",", text) != 0)
{
string_insert(name, text, string_pos(",", text));
}

Der obige Code sucht in der Zeichenkette, die in der Variablen "text" gespeichert ist, nach einem Komma, und wenn er eines findet, fügt er die Teilzeichenkette "name" an dieser Position ein.
