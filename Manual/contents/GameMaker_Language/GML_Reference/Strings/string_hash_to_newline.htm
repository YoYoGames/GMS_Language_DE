string_hash_to_newline

Diese Funktion gibt eine Zeichenkette zurück, bei der das Symbol "#" in eine neue Zeile umgewandelt wurde. Also eine Zeichenkette, die als formatiert ist:

"Hallo#Welt"

Würde auf dem Bildschirm ausgedruckt werden als:

Hallo
Welt

WICHTIG! Diese Funktion dient der Importkompatibilität zwischen GameMaker Studio 2 und früheren Versionen der Software und sollte daher nicht verwendet werden, es sei denn, es ist notwendig, veraltete Funktionen zu replizieren. Stattdessen sollten Sie die Escape-Zeichen verwenden, um eine neue Zeile zu markieren. Weitere Informationen zur Importkompatibilität finden Sie hier.

Syntax:

string_hash_to_newline(string);

Argument Beschreibung
String Die zu konvertierende Zeichenkette über mehrere Zeilen

Rückgabe:

Zeichenfolge

Beispiel:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Der obige Code wandelt die Zeichenkette mit dem Raute-Symbol in eine auf zwei Zeilen aufgeteilte Zeichenkette um und gibt sie dann auf dem Bildschirm aus.
