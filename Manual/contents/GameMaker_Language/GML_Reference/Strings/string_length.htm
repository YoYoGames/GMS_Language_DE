string_length

Diese Funktion gibt die Anzahl der Zeichen zurück, aus denen eine gegebene Zeichenkette besteht. Sie kann z. B. nützlich sein, um herauszufinden, wann die Zeichenlänge eines benutzerdefinierten Texteintrags begrenzt werden sollte (z. B.: Begrenzung des Spielernamens auf 10 Zeichen). Beachten Sie, dass diese Funktion anders als string_width() insofern unterscheidet, als dass die Anzahl der Zeichen in der Zeichenfolge gemessen wird und nicht ihre Breite, wie sie auf dem Bildschirm in Pixeln dargestellt wird.

Syntax:

string_length(string);

Argument Beschreibung
String Die Zeichenkette, deren Anzahl an Zeichen gemessen werden soll.

Rückgabe:

Real

Beispiel:

if (string_length(name) > 10)
{
name = string_copy(name, 1, 10);
}

Dies prüft, ob die Zeichenfolge des Namens größer als zehn Zeichen ist, und wenn ja, werden nur die ersten zehn Zeichen kopiert und verwendet.
