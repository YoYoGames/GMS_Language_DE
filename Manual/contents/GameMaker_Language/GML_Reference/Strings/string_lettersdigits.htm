string_lettersdigits

Diese Funktion gibt eine Kopie einer gegebenen Zeichenkette zurück, bei der alles außer den Buchstaben und Ziffern entfernt wurde, was bedeutet, dass sie verwendet werden kann, um unerwünschte Zeichen (wie "#" oder "?") z. B. aus einem Anmeldenamen oder einem Passwort zu entfernen.

HINWEIS: Diese Funktion gibt nur das englische Alphabet mit 26 Buchstaben von A - Z (Groß- oder Kleinschreibung) und die Zahlen 0 - 9 zurück.

Syntax:

string_lettersdigits(string);

Argument Beschreibung
String Die in Buchstaben und Ziffern zu konvertierende Zeichenfolge.

Rückgabe:

Zeichenfolge

Beispiel:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
draw_text(32,32, "Ungültiges Passwort! Nur Buchstaben und Zahlen bitte!");
}

Der obige Code prüft die Länge einer Zeichenkette gegen die Länge der gleichen Zeichenkette, bei der jedoch alle Symbole entfernt wurden. Wenn sie nicht gleich sind, wird eine Meldung auf den Bildschirm gezeichnet.
