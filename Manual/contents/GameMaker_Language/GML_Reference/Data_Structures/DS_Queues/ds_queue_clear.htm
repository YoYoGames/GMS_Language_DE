ds_queue_clear

Mit dieser Funktion können Sie alle Daten aus der angegebenen Queue-Datenstruktur löschen. Dabei wird die Datenstruktur NICHT zerstört (dafür sollten Sie die Funktion ds_queue_destroy()), sondern löscht nur alle Daten aus ihr und gibt eine leere Warteschlange zurück.

Syntax:

ds_queue_clear(id);

Argument Beschreibung
id Die ID der zu löschenden Datenstruktur.

Rückgabe:

N/A

Beispiel:

if (count = 15) && (!ds_queue_empty(command_queue))
{
ds_queue_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

Der obige Code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat, und wenn dies der Fall ist, löscht er die DS-Warteschlange, die in der Variablen "command_queue" indiziert ist, setzt einen Alarm und setzt die Variable auf 0 zurück.
