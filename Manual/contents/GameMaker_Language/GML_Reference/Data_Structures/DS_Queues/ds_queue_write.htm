ds_queue_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der ds_queue_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenfolge ist keine von Menschen lesbare Zeichenfolge, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_queue_write(id);

Argument Beschreibung
id Die ID der zu schreibenden Datenstruktur.

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

Der obige Code öffnet eine Ini-Datei und schreibt dann einen String, der die in der DS-Warteschlange gespeicherten Informationen enthält, die in der Variablen "queue" indiziert sind. Die Warteschlange wird dann geleert und die Ini-Datei geschlossen.
