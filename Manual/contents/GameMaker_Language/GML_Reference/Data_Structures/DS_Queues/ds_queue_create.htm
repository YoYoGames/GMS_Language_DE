ds_queue_create

Diese Funktion erzeugt eine neue Queue-Datenstruktur und gibt den Indexwert zurück. Dieser Wert sollte in einer Variablen gespeichert werden und in allen weiteren Funktionsaufrufen, die sich auf die Warteschlange beziehen, verwendet werden.

WICHTIG! Wenn Sie eine Datenstruktur erstellen, ist der Indexwert zu ihrer Identifizierung ein Integer-Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, daher sollten Sie im Zweifelsfall den ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein zerstörter Indexwert einer Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_queue_create();

Rückgabe:

Real

Beispiel:

Warteschlange = ds_queue_create();

Dadurch wird eine neue Warteschlange erstellt und ihre Index-ID der Instanzvariablen "queue" zugewiesen.
