ds_queue_size

Diese Funktion gibt die "Größe" der Warteschlange zurück, d. h. die Anzahl der Elemente, die in die Warteschlange gestellt wurden.

Syntax:

ds_queue_size(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Real

Beispiel:

if !ds_queue_empty(control_queue)
{
num = ds_queue_size(control_queue);
}

Der obige Code prüft eine DS-Warteschlange, um zu sehen, ob sie leer ist oder nicht, und wenn nicht, wird die Anzahl der darin enthaltenen Elemente abgerufen und der Wert in einer Variablen gespeichert.
