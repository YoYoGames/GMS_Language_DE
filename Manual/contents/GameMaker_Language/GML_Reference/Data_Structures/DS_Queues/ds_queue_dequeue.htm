ds_queue_dequeue

Diese Funktion entfernt den Head-Wert aus der DS-Warteschlange, entfernt ihn aus der Warteschlange und gibt den Wert zurück, der in einer Variablen gespeichert werden soll. Wenn die Warteschlange leer ist, gibt die Funktion die Konstante undefiniert zurück, andernfalls gibt sie den in der Warteschlange enthaltenen Real- oder String-Wert zurück.

Syntax:

ds_queue_dequeue(id);

Argument Beschreibung
id Die ID der Warteschlange, aus der dequeued werden soll.

Rückgabe:

In der Warteschlange gespeicherter Datentypwert oder undefiniert

Beispiel:

if !ds_queue_empty(move_queue)
{
var xx = ds_queue_dequeue(move_queue);
var yy = ds_queue_dequeue(move_queue);
move_towards_point(xx, yy, 4);
}

Der obige Code prüft die DS-Warteschlange, die in der Variablen "move_queue" indiziert ist, um zu sehen, ob sie leer ist. Ist dies nicht der Fall, werden die beiden Werte aus dem Kopf der Warteschlange entnommen und verwendet, um eine Richtung für die Bewegung festzulegen.
