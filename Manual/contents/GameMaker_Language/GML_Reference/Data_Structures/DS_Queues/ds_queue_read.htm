ds_queue_read

Mit dieser Funktion können Sie eine gespeicherte DS-Warteschlange wiederherstellen (eine, die zuvor als String mit ds_queue_write()). Sie müssen zuerst eine neue DS-Warteschlange erstellen, um die Zeichenkette einzulesen, und wenn die DS-Warteschlange bereits bereits existiert und Informationen in ihr gespeichert sind, wird diese vor dem Lesen geleert. Diese Funktion ist von entscheidender Bedeutung, wenn Sie Speicher-/Lademechanismen für Ihr Spiel erstellen. Beachten Sie, dass wenn die Datenstruktur mit früheren Versionen von GameMaker erstellt wurde, sollten Sie das optionale Argument "legacy" hinzufügen, indem Sie es auf true setzen, da sich das String-Format mit dieser Version geändert hat.

Syntax:

ds_queue_read(id, str [, legacy]);

Argument Beschreibung
id Die ID der Datenstruktur, in die eingelesen werden soll.
str Die Zeichenkette, aus der gelesen werden soll.
Vermächtnis(optional) Kann entweder wahr oder falsch sein oder komplett weggelassen werden.

Rückgabe:

N/A

Beispiel:

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Warteschlangen", "0", "");
if (str != "")
{
ds_queue_read(warteschlange, str);
}
ini_close();

Der obige Code erstellt eine Warteschlange und speichert den Index in der Variablen "queue". Dann wird eine ini-Datei geöffnet und eine Zeichenkette daraus gelesen, wobei zuerst überprüft wird, dass die Zeichenkette nicht als leer zurückgegeben wird. Diese Zeichenkette wird dann in die die neu erstellte DS-Warteschlange eingelesen.
