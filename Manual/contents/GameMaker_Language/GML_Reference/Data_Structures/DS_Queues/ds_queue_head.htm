ds_queue_head

Diese Funktion liest nur den ersten Wert der Warteschlange (den, der "am Kopf" steht). Der Wert wird nicht aus der Warteschlange entfernt, d. h. er kann auch in Zukunft noch von dieser Funktion oder der Funktion ds_queue_dequeue(). Wenn die Warteschlange leer ist, gibt die Funktion die Konstante undefiniert zurück, andernfalls gibt sie den in der Warteschlange enthaltenen Real- oder String-Wert zurück.

Syntax:

ds_queue_head(id);

Argument Beschreibung
id Die ID der Datenstruktur, aus der gelesen werden soll.

Rückgabe:

In der Warteschlange gespeicherter Wert vom Datentyp

Beispiel:

num = ds_queue_head(control_queue);

Der obige Code liest den Wert aus der in der Variablen "control_queue" indizierten Warteschlange und speichert den Rückgabewert in der Variablen "num".
