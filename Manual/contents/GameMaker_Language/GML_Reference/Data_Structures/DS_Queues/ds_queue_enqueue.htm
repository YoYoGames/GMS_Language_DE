ds_queue_enqueue

Diese Funktion fügt einen Wert (Real oder String) an das Ende der DS-Warteschlange an. Die Funktion kann weitere 14 optionale Argumente annehmen (insgesamt 15 mögliche Hinzufügungen), so dass Sie mehrere Werte in einem einzigen Aufruf mehrere Werte nacheinander an das Ende der Warteschlange anhängen.

Syntax:

ds_queue_enqueue(id, val [, val2, ... val15]);

Argument Beschreibung
id Die ID der Warteschlange, zu der hinzugefügt werden soll.
val Der Wert, der der Warteschlange hinzugefügt werden soll.
[val2, ... val15] Optionale Werte, die der Warteschlange hinzugefügt werden sollen.

Rückgabe:

N/A

Beispiel:

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

Der obige Code erstellt eine neue DS-Warteschlange und speichert ihren Index in der Variablen "move_queue". Er schiebt dann eine Anzahl von Werten zur zukünftigen Verwendung in die Warteschlange.
