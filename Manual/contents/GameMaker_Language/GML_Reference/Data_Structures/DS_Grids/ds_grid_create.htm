ds_gitter_erstellen

Mit dieser Funktion können Sie eine neue DS-Grid-Datenstruktur mit der angegebenen Zellenbreite und -höhe erzeugen. Diese Funktion gibt eine ID zurück, die in allen weiteren Funktionen, die sich mit diesem DS-Grid beschäftigen, verwendet werden muss.

WICHTIG! Wenn Sie eine Datenstruktur anlegen, ist der Indexwert zu ihrer Identifizierung ein Integer-Wert, der bei 0 beginnt. Das bedeutet, dass Datenstrukturen unterschiedlichen Typs denselben Indexwert haben können, daher sollten Sie im Zweifelsfall die ds_exists() Funktion verwenden, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, so dass ein zerstörter Indexwert einer Datenstruktur anschließend von einer neu erstellten verwendet werden kann.

Syntax:

ds_grid_create(w, h);

Argument Beschreibung
w Die Breite des zu erstellenden Rasters.
h Die Höhe des zu erstellenden Rasters.

Rückgabe:

Real

Beispiel:

mygrid = ds_grid_create(10, 10)

Dadurch wird ein 10 Zellen hohes und 10 Zellen breites Gitter erzeugt.
