ds_gitter_wert_x

Mit dieser Funktion erhalten Sie die x-Koordinate (innerhalb des angegebenen rechteckigen Gitterbereichs) des gesuchten Wertes. Sie geben den DS-Gitterindex (wie er von ds_grid_create()) zusammen mit den x/y-Positionen für die linke obere und rechte untere Ecke des zu durchsuchenden Rasterbereichs an, bevor Sie den Wert angeben, nach dem gesucht werden soll. Wenn der gesuchte Wert nicht vorhanden ist, gibt die Funktion -1 zurück.

Syntax:

ds_grid_value_x(index, x1, y1, x2, y2, val);

Argument Beschreibung
index Der Index des Gitters.
x1 Die x-Position der linken Seite des Bereichs im Raster, von 0 bis (Rasterbreite - 1).
y1 Die y-Position der Oberkante des Bereichs im Raster, von 0 bis (Rasterhöhe - 1).
x2 Die x-Position der rechten Seite des Bereichs im Raster, von 0 bis (Rasterbreite - 1).
y2 Die y-Position des Bodens des Bereichs im Raster, von 0 bis (Rasterhöhe - 1).
val Der zu findende Wert.

Rückgabe:

Real

Beispiel:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

Der obige Code prüft ein ds_grid auf einen bestimmten Wert innerhalb eines rechteckigen Bereichs. Wenn er gefunden wird, speichert er anschließend die x- und y-Position des Werts in zwei Variablen zur späteren Verwendung.
