ds_grid_sort

Diese Funktion kann verwendet werden, um ein DS-Gitter basierend auf den Werten einer bestimmten Spalte zu sortieren (ähnlich wie Sie Dateien im OS-Dateiexplorer nach Datum, Größe usw. sortieren würden). Das folgende Bild zeigt ein Beispiel:

Syntax:

ds_grid_sort(index, Spalte, aufsteigend);

Argument Beschreibung
index Der Index des zu sortierenden Gitters.
Spalte Die Spalte, die für die Sortierung der Zeilen verwendet werden soll
aufsteigend Ob die Sortierung von unten nach oben(true) oder von oben nach unten(false) erfolgen soll.

Rückgabe:

N/A

Beispiel:

ds_grid_sort(grid, 3, false)

Dies würde alle Werte im DS-Gitter nehmen, die in der Variablen "grid" indiziert sind, und sie nach den Werten in der 3. Spalte des Gitters sortieren (wie im obigen Bild gezeigt).
