ds_grid_resize

Mit dieser Funktion können Sie die Größe des angegebenen DS-Gitters ändern, um eine andere Breite und/oder Höhe zu erhalten. Wenn die Größe des Gitters größer ist als das aktuelle Gitter, haben die neuen Zellen den Basiswert 0, und wenn die Größe kleiner ist, gehen die Werte in den Zellen, die nicht mehr innerhalb der neuen Größe liegen, verloren. die nicht mehr innerhalb der neuen Größe liegen, verloren gehen. Alle anderen Zellen bleiben unberührt.

Syntax:

ds_grid_resize(index, w, h);

Argument Beschreibung
index Dieser Index des Rasters, dessen Größe geändert werden soll.
w Die neue Breite des Gitters.
h Die neue Höhe des Gitters.

Rückgabe:

N/A

Beispiel:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

Der obige Code ändert die Größe des DS-Gitters, das in der globalen Variablen "Grid" indiziert ist, und löscht es dann, so dass jede Zelle den Wert -1 enthält.
