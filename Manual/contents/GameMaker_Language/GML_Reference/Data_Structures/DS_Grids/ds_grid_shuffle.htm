ds_grid_shuffle

Diese Funktion kann verwendet werden, um die Positionen aller Werte in allen Zellen innerhalb eines Gitters zu randomisieren.

HINWEIS: Diese Funktion mischt die Gitterzellen bei jedem neuen Durchlauf des Spiels an dieselben Positionen, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert generiert, um das Debuggen von Code zu erleichtern. Zur Vermeidung von dieses Verhalten zu vermeiden, verwenden Sie randomisieren() beim Start Ihres Spiels. Dies gilt nur für das Testen und Debuggen des Spiels, da das endgültige ausführbare Paket wird dieses Verhalten nicht zeigen und bei jedem Spiel zufällig sein.

Syntax:

ds_grid_shuffle(index);

Argument Beschreibung
index Der Index des Rasters, das gemischt werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_shuffle(raster)

Dies würde alle Werte im DS-Gitter nehmen, die in der Variablen "grid" indiziert sind, und sie umherschieben, um eine andere Reihenfolge zu erhalten.
