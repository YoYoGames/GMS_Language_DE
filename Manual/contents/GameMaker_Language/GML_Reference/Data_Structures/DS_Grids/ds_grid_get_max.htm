ds_grid_get_max

Diese Funktion kann verwendet werden, um den Maximalwert für alle Zellen zu finden, die sich innerhalb des definierten Bereichs eines Gitters befinden, wie im folgenden Bild gezeigt:

Syntax:

ds_grid_get_max(index, x1, y1, x2, y2);

Argument Beschreibung
index Der Index des Gitters.
x1 Die linke Zellenspalte der Region.
y1 Die oberste Zellreihe des Bereichs.
x2 Die rechte Zellenspalte der Region.
y2 Die unterste Zellreihe des Bereichs.

Rückgabe:

Real oder String

Beispiel:

val = ds_grid_get_max(grid, 0, 0, 5, 5)

Der obige Code setzt die Variable "val" auf den Maximalwert, der innerhalb des angegebenen Bereichs des DS-Gitters enthalten ist, das in der Variablen "grid" indiziert ist.
