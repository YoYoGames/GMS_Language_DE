ds_grid_get_disk_max

Diese Funktion kann verwendet werden, um den Maximalwert für alle Zellen zu finden, die sich innerhalb des definierten Scheibenbereichs eines Gitters befinden, wie in der folgenden Abbildung gezeigt:

Syntax:

ds_grid_get_disk_max(index, xm, ym, r);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibenmitte im Raster.
ym Die y-Position der Scheibenmitte im Raster.
r Der Radius der Scheibe im Raster.

Rückgabe:

Real oder String

Beispiel:

val = ds_grid_get_disk_max(grid, 5, 5, 2)

Der obige Code setzt die Variable "val" auf den Maximalwert, der innerhalb der angegebenen Scheibe des in der Variablen "grid" indizierten DS-Gitters enthalten ist.
