ds_grid_multiply_region

Mit dieser Funktion können Sie einen Bereich des Gitters angeben, in dem jeder Zellwert mit einem bestimmten Betrag multipliziert wird.

HINWEIS: Diese Funktion funktioniert nur mit reellen Zahlen, nicht mit Strings.

Syntax:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

Argument Beschreibung
index Der Index des Gitters.
x1 Die x-Position des linken Randes der Region im Raster.
y1 Die y-Position der Oberkante des Bereichs im Raster.
x2 Die x-Position der rechten Seite der Region im Raster.
y2 Die y-Position des Bodens der Region im Raster.
val Der Wert, der mit den Regionszellen multipliziert werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 )

Der obige Code nimmt alle Werte, die innerhalb des definierten rechteckigen Gitterbereichs gefunden werden, und multipliziert jeden mit 2.
