ds_grid_get_disk_mean

Diese Funktion kann verwendet werden, um den Mittelwert für alle Zellen zu finden, die sich innerhalb des definierten Scheibenbereichs eines Gitters befinden (alle Zellenwerte werden addiert und dann durch die Gesamtzahl der Zellen, aus denen die Scheibe besteht, geteilt), wie in der folgenden Abbildung gezeigt:

Syntax:

ds_grid_get_disk_mean(index, xm, ym, r);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.

Rückgabe:

Real oder String

Beispiel:

val = ds_grid_get_disk_mean(grid, 5, 5, 2)

Der obige Code setzt die Variable "val" auf den Mittelwert, der innerhalb der angegebenen Scheibe des in der Variablen "grid" indizierten DS-Gitters enthalten ist.
