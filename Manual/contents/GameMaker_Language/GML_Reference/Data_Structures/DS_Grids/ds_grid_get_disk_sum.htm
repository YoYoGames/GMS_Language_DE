ds_grid_get_disk_sum

Diese Funktion kann verwendet werden, um alle Werte aller Zellen, die sich innerhalb des definierten Scheibenbereichs eines Gitters befinden, zu addieren, wie im folgenden Bild gezeigt:

Syntax:

ds_grid_get_disk_sum(index, xm, ym, r);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.

Rückgabe:

Real oder String

Beispiel:

val = ds_grid_get_disk_sum(grid, 5, 5, 2)

Der obige Code setzt die Variable "val" auf die Summe aller Werte, die innerhalb der angegebenen Scheibe des DS-Gitters enthalten sind, das in der Variablen "grid" indiziert ist.
