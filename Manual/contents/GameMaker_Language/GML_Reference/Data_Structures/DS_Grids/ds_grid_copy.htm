ds_grid_copy

Mit dieser Funktion können Sie den Inhalt eines Gitters in ein anderes kopieren. Beide Grids müssen zuvor mit der ds_grid_create() Funktion.

Syntax:

ds_grid_copy(Ziel, Quelle);

Argument Beschreibung
Ziel Der Index des Rasters, in das kopiert werden soll.
Quelle Dieser Index des Rasters, aus dem kopiert werden soll.

Rückgabe:

N/A

Beispiel:

n_grid = ds_grid_create(ds_grid_width(a_grid), ds_grid_height(a_grid));
ds_grid_copy(n_grid, a_grid);
ds_grid_clear(a_grid, -1)

Der obige Code erstellt ein neues DS-Gitter, basierend auf der Breite und Höhe eines zuvor erstellten Gitters, und kopiert dann die Informationen aus dem vorherigen Gitter in das neue. Schließlich wird das alte Gitter gelöscht, so dass alle Zellen den Wert -1 haben.
