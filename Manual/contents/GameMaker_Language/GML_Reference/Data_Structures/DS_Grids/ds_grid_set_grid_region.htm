ds_grid_set_grid_region

Diese Funktion kann verwendet werden, um den Inhalt eines rechteckigen Bereichs von Gitterzellen von einem (zuvor definierten) DS-Gitter in ein anderes zu kopieren, oder sie kann verwendet werden, um einen Bereich innerhalb desselben Gitters zu kopieren. Die folgenden Bilder veranschaulichen beide Möglichkeiten, wie diese Funktion verwendet werden kann:

Syntax:

ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

Argument Beschreibung
index Der Index des Zielgitters.
Quelle Der Index des Quellgitters.
x1 Die linke Position des zu kopierenden Bereichs von Zellen aus dem Quellgitter.
y1 Die obere Position des Bereichs der Zellen, die aus dem Quellgitter kopiert werden sollen.
x2 Die rechte Position des zu kopierenden Bereichs von Zellen aus dem Quellgitter.
y2 Die untere Position des Bereichs der Zellen, die aus dem Quellgitter kopiert werden sollen.
xpos Die x-Position auf dem Zielraster, in die die Quellregion kopiert werden soll.
ypos Die y-Position auf dem Zielraster, in die die Quellregion kopiert werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10)

Dies würde den Bereich der Zellen von (0,0) bis (5,5) des in der Variablen "t_grid" indizierten Gitters kopieren und an die Position (10,10) in dem in der Variablen "grid" indizierten Gitter kopieren.
