ds_grid_add

Diese Funktion kann verwendet werden, um einen gegebenen Wert (Real oder String) zum Wert der gegebenen Zelle im Raster zu addieren. Der zu addierende Wert muss vom gleichen Typ sein wie der in der Gitterzelle enthaltene Wert, d. h.: Sie können keine Zeichenkette zu einer reellen Zahl addieren oder umgekehrt, und für Strings entspricht dies einer Verkettung.

Syntax:

ds_grid_add(index, x, y, val);

Argument Beschreibung
index Der Index des Gitters.
x Die x-Position der Zelle im Raster.
y Die y-Position der Zelle im Raster.
val Der Wert, der zu der Zelle hinzugefügt werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_add(grid, 5, 5, 6)

Dies würde der angegebenen Zelle innerhalb des DS-Gitters, auf das die Variable "grid" verweist, 6 hinzufügen.
