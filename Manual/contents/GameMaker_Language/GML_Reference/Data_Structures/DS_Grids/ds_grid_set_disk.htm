ds_grid_set_disk

Mit dieser Funktion können Sie einen kreisförmigen Bereich eines Gitters auf einen bestimmten Wert setzen. Sie müssen eine Start-Gitterzelle (als x- und y-Achsenkoordinate) sowie den Radius der zu setzenden Scheibe und den Wert, auf den Sie die Scheibe setzen wollen, angeben, wie wie in der Abbildung unten gezeigt:

Syntax:

ds_grid_set_disk(index, xm, ym, r, val);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.
val Der Wert, der mit den Zellen innerhalb der Platte gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

Der obige Code setzt einen kreisförmigen Bereich mit einem Radius von 5 Zellen im DS-Gitter, das in der Variablen "grid" indiziert ist, auf einen Wert von -4.
