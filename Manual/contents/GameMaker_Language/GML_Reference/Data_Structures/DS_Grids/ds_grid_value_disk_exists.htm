ds_grid_value_disk_exists

Mit dieser Funktion können Sie prüfen, ob ein bestimmter Wert (Real oder String) innerhalb eines kreisförmigen Bereichs eines gegebenen DS-Gitters vorhanden ist. Wenn er vorhanden ist, gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

ds_grid_value_disk_exists(index, xm, ym, r, val);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.
val Der zu findende Wert.

Rückgabe:

Boolesche

Beispiel:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Der obige Code prüft ein DS-Gitter auf einen bestimmten Wert innerhalb einer Plattenregion. Wenn er gefunden wird, speichert er dann die x- und y-Position des Wertes in zwei Variablen zur späteren Verwendung.
