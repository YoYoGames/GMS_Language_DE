ds_grid_multiply_disk

Diese Funktion nimmt alle Werte in einem gegebenen scheibenförmigen Bereich des DS-Gitters und multipliziert jeden einzelnen mit dem angegebenen Betrag.

HINWEIS: Diese Funktion funktioniert nur mit reellen Zahlen, nicht mit Strings.

Syntax:

ds_grid_multiply_disk(index, xm, ym, r, val);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.
val Der Wert, mit dem die Zellen innerhalb der Scheibe multipliziert werden.

Rückgabe:

N/A

Beispiel:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

Der obige Code nimmt alle Werte, die innerhalb des kreisförmigen Gitterbereichs gefunden werden, und multipliziert jeden mit 2.
