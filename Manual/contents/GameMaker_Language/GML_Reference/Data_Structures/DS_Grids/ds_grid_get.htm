ds_grid_get

Mit dieser Funktion können Sie den Wert (entweder eine reelle Zahl oder eine Zeichenkette) von einer beliebigen Zelle innerhalb des angegebenen DS-Gitters abrufen. Wenn Sie der Funktion ungültige Gitterkoordinaten übergeben, ist der zurückgegebene Wert undefiniert und es wird ein Fehler im Ausgabefenster angezeigt.

Syntax:

ds_grid_get(index, x, y);

Argument Beschreibung
index Der Index des Gitters.
x Die x-Position der Zelle, für die Sie den Wert suchen möchten.
y Die y-Position der Zelle, für die Sie den Wert suchen möchten.

Rückgabe:

Real oder String

Beispiel:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Der obige Code wählt eine zufällige Zelle aus dem in der Variablen "grid" indizierten DS-Gitter aus und speichert ihren Wert in der Variablen "val".
