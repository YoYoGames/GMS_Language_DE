ds_grid_value_disk_y

Mit dieser Funktion erhalten Sie die y-Koordinate (innerhalb des angegebenen scheibenförmigen Rasterbereichs) des gesuchten Wertes. Sie geben den DS-Grid-Index (wie er von ds_grid_create()) zusammen mit den x/y-Positionen für die mittlere Zelle der Scheibe. Dann geben Sie den Radius (als Ganzzahlwert) um die zu suchende Mittelzelle an, bevor Sie den zu suchenden Wert angeben. Wenn der gesuchte Wert nicht vorhanden ist, gibt die Funktion -1 zurück.

Syntax:

ds_grid_value_disk_y(index, xm, ym, r, val);

Argument Beschreibung
index Der Index des Gitters.
xm Die x-Position der Scheibe auf dem Raster.
ym Die y-Position der Scheibe auf dem Raster.
r Der Radius der Scheibe auf dem Gitter.
val Der zu findende Wert.

Rückgabe:

Real

Beispiel:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

Der obige Code prüft ein DS-Gitter auf einen bestimmten Wert innerhalb einer Plattenregion. Wenn er gefunden wird, speichert er dann die x- und y-Position des Wertes in zwei Variablen zur späteren Verwendung.
