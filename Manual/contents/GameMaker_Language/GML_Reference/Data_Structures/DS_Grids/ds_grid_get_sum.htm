ds_grid_get_sum

Diese Funktion kann verwendet werden, um alle Werte aller Zellen, die sich innerhalb des definierten Bereichs eines Gitters befinden, zu addieren, wie im folgenden Bild gezeigt:

Syntax:

ds_grid_get_sum(index, x1, y1, x2, y2);

Argument Beschreibung
index Der Index des Gitters.
x1 Die linke Zellenspalte der Region.
y1 Die oberste Zellreihe des Bereichs.
x2 Die rechte Zellenspalte der Region.
y2 Die unterste Zellreihe des Bereichs.

Rückgabe:

Real

Beispiel:

val = ds_grid_get_sum(grid, 0, 0, 5, 5)

Der obige Code setzt die Variable "val" auf die Summe aller Werte, die innerhalb des angegebenen Bereichs des DS-Gitters enthalten sind, das in der Variablen "grid" indiziert ist.
