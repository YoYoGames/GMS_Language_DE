ds_gitter_lesen

Mit dieser Funktion können Sie einen String, der zuvor mit der Funktion ds_grid_write() zurück in ein DS-Gitter. Das DS-Gitter muss zuvor erstellt worden sein (siehe das Beispiel unten). Beachten Sie dass, wenn die Datenstruktur mit früheren Versionen von GameMaker erstellt wurde, sollten Sie das optionale Argument "legacy" hinzufügen und auf true setzen, da sich das String-Format mit dieser Version geändert hat.

Syntax:

ds_grid_read(index, string [, legacy]);

Argument Beschreibung
index Der Index des zu lesenden Gitters.
String Die Zeichenkette, die in das DS-Gitter eingelesen werden soll.
Vermächtnis(optional) Kann entweder wahr oder falsch sein oder komplett weggelassen werden.

Rückgabe:

N/A

Beispiel:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

Der obige Code erstellt ein DS-Gitter basierend auf der Größe des Raums (jedes 32x32-Pixelquadrat stellt eine Gitterzelle dar) und liest dann einen zuvor gespeicherten Satz von Gitterdaten aus einer ini-Datei in das neue DS-Gitter ein.
