ds_grid_height

Diese Funktion gibt die Höhe des angegebenen Gitters zurück. Dieser Wert ist die Anzahl der Zellen, die das Gitter entlang der y-Achse hat, und ist immer eine ganze Zahl, wie im Bild unten gezeigt:

Syntax:

ds_grid_height(index);

Argument Beschreibung
index Dieser Index des Gitters, um die Höhe zu finden.

Rückgabe:

Real

Beispiel:

for (var i = 0; i < ds_grid_width(grid); ++i;)
{
for (var j = 0; j < ds_grid_height(grid); ++j;)
{
if (ds_grid_get(grid, i, j) == 1)
{
instance_create_Layer(i * 32, j * 32, "Walls", obj_Wall);
}
}
}

Der obige Code läuft in einer Schleife durch das DS-Gitter, das in der Variablen "grid" indiziert ist, und wenn der Wert in einer bestimmten Zelle gleich 1 ist, wird eine Instanz von "obj_Wall" an der entsprechenden Position im Raum erstellt.
