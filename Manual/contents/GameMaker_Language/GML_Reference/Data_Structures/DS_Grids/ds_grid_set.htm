ds_grid_set

Diese Funktion kann verwendet werden, um eine bestimmte Zelle innerhalb des gegebenen DS-Gitters auf einen beliebigen Wert zu setzen, der eine reelle Zahl oder eine Zeichenkette sein kann. Das folgende Bild veranschaulicht dies:

Syntax:

ds_grid_set(index, x, y, wert);

Argument Beschreibung
index Dieser Index des Gitters.
x Die x-Position der zu setzenden Zelle.
y Die y-Position der zu setzenden Zelle.
Wert Der Wert, mit dem die Zelle gesetzt werden soll.

Rückgabe:

N/A

Beispiel:

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;
repeat (ds_grid_width(grid))
{
repeat (ds_grid_height(grid))
{
ds_grid_set(grid, i, j, irandom(9));
j += 1;
}
j = 0;
i += 1;
}

Der obige Code erstellt ein Raster und speichert dessen Index in der Variablen "grid". Anschließend wird dieses Raster mit zufälligen Ganzzahlen von 0 bis 9 gefüllt.
