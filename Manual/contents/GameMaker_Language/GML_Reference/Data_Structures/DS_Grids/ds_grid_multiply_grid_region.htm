ds_grid_multiply_grid_region

Mit dieser Funktion können Sie einen Bereich innerhalb eines gegebenen DS-Gitters definieren, dann diese Werte nehmen und sie mit denen multiplizieren, die in einer separaten Region entweder desselben DS-Gitters oder eines anderen (das zuvor erstellt wurde) gefunden wurden. Die ursprüngliche Region wird bleibt unverändert, während der Bereich, mit dem sie multipliziert wurden, nun die neuen Werte für jede Zelle speichert.

Syntax:

ds_grid_multiply_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

Argument Beschreibung
index Der Index des Zielgitters.
Quelle Der Index des Quellgitters.
x1 Die linke Position des zu kopierenden Bereichs von Zellen aus dem Quellgitter.
y1 Die obere Position des Bereichs der Zellen, die aus dem Quellgitter kopiert werden sollen.
x2 Die rechte Position des zu kopierenden Bereichs von Zellen aus dem Quellgitter.
y2 Die untere Position des Bereichs der Zellen, die aus dem Quellgitter kopiert werden sollen.
xpos Die x-Position auf dem Zielraster, mit der die Quellregion multipliziert werden soll.
ypos Die y-Position auf dem Zielraster, mit der die Quellregion multipliziert werden soll.

Rückgabe:

N/A

Beispiel:

ds_grid_multiply_grid_region(mygrid, mygrid, 0, 0, 5, 5, 0, 0)

Dies würde den Bereich der Zellen von (0,0) bis (5,5) des DS-Gitters "mygrid" nehmen und mit den Zellen von der Position (0,0) des gleichen DS-Gitters multiplizieren.
