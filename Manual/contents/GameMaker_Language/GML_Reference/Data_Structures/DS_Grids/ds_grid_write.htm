ds_grid_write

Mit dieser Funktion kann das angegebene ds_grid in einen String umgewandelt werden, der dann z. B. in einer externen Datei gespeichert werden kann. Den von dieser Funktion zurückgegebenen String können Sie wieder in ein ds_grid einlesen, indem Sie die Funktion ds_grid_read().

HINWEIS: Die zurückgegebene Zeichenfolge ist keine von Menschen lesbare Zeichenfolge, sondern ein Dump des Inhalts der Datenstruktur.

Syntax:

ds_grid_write(index);

Argument Beschreibung
index Der Index des zu schreibenden Gitters.

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Der obige Code öffnet eine ini-Datei (und legt sie an, falls sie noch nicht existiert) und schreibt dann das angegebene ds_grid als String in diese Datei.
