ds_stack_size

Diese Funktion gibt die "Größe" des Stapels zurück, d. h. die Anzahl der Elemente, die auf den Stapel geschoben wurden.

Syntax:

ds_stack_size(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Real

Beispiel:

if !ds_stack_empty(control_stack)
{
num = ds_stack_size(control_stack);
}

Der obige Code prüft einen DS-Stack, um zu sehen, ob er leer ist oder nicht, und wenn nicht, wird die Anzahl der enthaltenen Elemente abgerufen und der Wert in einer Variablen gespeichert.
