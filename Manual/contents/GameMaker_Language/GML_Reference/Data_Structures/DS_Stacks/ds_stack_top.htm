ds_stack_top

Diese Funktion liest nur den ersten Wert des Stapels (den, der "oben" liegt). Der Wert wird nicht gepoppt, d. h. er kann auch in Zukunft noch von dieser Funktion oder der Funktion ds_stack_pop(). Wenn der Stack leer ist, gibt die Funktion die Konstante undefiniert zurück, andernfalls gibt sie den im Stack enthaltenen Real- oder String-Wert zurück.

Syntax:

ds_stack_top(id);

Argument Beschreibung
id Die ID der Datenstruktur, aus der gelesen werden soll.

Rückgabe:

Datentypwert im Stack gespeichert oder undefiniert

Beispiel:

num = ds_stack_top(control_stack);

Der obige Code liest den Wert vom Stapel, der in der Variablen "control_stack" indiziert ist, und speichert den Rückgabewert in der Variablen "num".
