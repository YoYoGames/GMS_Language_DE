ds_stack_push

Diese Funktion schiebt (addiert) einen Wert, der von einem beliebigen Datentyp sein kann, an die Spitze des Stacks. Die Funktion kann weitere 14 optionale Argumente aufnehmen (insgesamt 15 mögliche Additionen), so dass Sie mehrere Werte nacheinander in einem einzigen Aufruf auf den Stack legen.

Syntax:

ds_stack_push(id, val [, val2, ... val15]);

Argument Beschreibung
id Die ID der Datenstruktur, auf die geschoben werden soll.
val Der Wert, der auf den Stack geschoben werden soll.
[val2, ... val13] Optionale Werte, die dem Stapel hinzugefügt werden sollen.

Rückgabe:

N/A

Beispiel:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

Der obige Code erzeugt einen neuen DS-Stack und speichert dessen Index in der Variablen "move_stack". Er schiebt dann eine Anzahl von Werten auf den Stack zur zukünftigen Verwendung.
