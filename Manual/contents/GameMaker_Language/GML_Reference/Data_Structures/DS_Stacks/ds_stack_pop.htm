ds_stack_pop

Diese Funktion holt den obersten Wert aus dem DS-Stapel, entfernt ihn vom Stapel und gibt den Wert zurück, der in einer Variablen gespeichert werden soll. Wenn der Stack leer ist, gibt die Funktion die Konstante undefiniert zurück, andernfalls gibt sie den im Stapel enthaltenen Real- oder String-Wert zurück.

Syntax:

ds_stack_pop(id);

Argument Beschreibung
id Die ID der Datenstruktur, aus der gepoppt werden soll.

Rückgabe:

Wert des Datentyps, der auf dem Stapel gespeichert oder undefiniert ist

Beispiel:

if !ds_stack_empty(move_stack)
{
var xx = ds_stack_pop(move_stack);
var yy = ds_stack_pop(move_stack);
move_towards_point(xx, yy, 4);
}

Der obige Code prüft den DS-Stapel, der in der Variablen "move_stack" indiziert ist, um zu sehen, ob er leer ist, und wenn das nicht der Fall ist, dann werden die obersten beiden Werte vom Stapel gepoppt und zum Festlegen einer Bewegungsrichtung verwendet.
