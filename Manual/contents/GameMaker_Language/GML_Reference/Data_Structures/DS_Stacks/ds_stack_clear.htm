ds_stack_clear

Mit dieser Funktion können Sie alle Daten aus der angegebenen Stack-Datenstruktur löschen. Dabei wird die Datenstruktur NICHT zerstört (dafür sollten Sie die Funktion ds_stack_destroy()), sondern löscht nur alle Daten aus ihr und gibt einen leeren Stack zurück.

Syntax:

ds_stack_clear(id);

Argument Beschreibung
id Die ID der zu löschenden Datenstruktur.

Rückgabe:

N/A

Beispiel:

if (ai_count = 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack);
alarm[0] = room_speed;
ai_count = 0;
}

Der obige Code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat, und wenn dies der Fall ist, löscht er den in der Variablen "AI_stack" indizierten DS-Stapel, setzt einen Alarm und setzt die Variable auf 0 zurück.
