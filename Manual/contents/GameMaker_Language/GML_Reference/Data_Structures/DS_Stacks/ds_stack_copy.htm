ds_stack_copy

Mit dieser Funktion kann der Inhalt eines Stacks in einen anderen kopiert werden. Beachten Sie, dass dadurch der Inhalt des ursprünglichen Stapels NICHT entfernt wird und der ursprüngliche Stapel auch nicht zerstört wird. Wenn Sie diese Funktion verwenden, muss der Stapel, in den kopiert wird Wenn der Stapel, in den kopiert werden soll, zuvor erstellt wurde und vor dem Kopieren Elemente enthielt, werden diese zuerst gelöscht (d. h. diese Informationen gehen verloren).

Syntax:

ds_stack_copy(id, source);

Argument Beschreibung
id Die ID des NEUEN Stapels.
Quelle Der ORIGINAL-Stack, der auf id kopiert werden soll.

Rückgabe:

N/A

Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
stack = ds_stack_create();
ds_stack_copy(stack, other.stack);
}

Die obige Funktion erstellt eine neue Instanz und erstellt dann in dieser Instanz einen neuen DS-Stack und kopiert den Inhalt des Stacks in der Instanz, die den Codeblock ausführt, in den neu erstellten Instanz-Stack.
