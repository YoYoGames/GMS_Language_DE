ds_stack_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der ds_stack_read() Funktion.

HINWEIS: Die zurückgegebene Zeichenfolge ist keine von Menschen lesbare Zeichenfolge, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_stack_write(id);

Argument Beschreibung
id Die ID der zu schreibenden Datenstruktur.

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

Der obige Code öffnet eine Ini-Datei und schreibt dann einen String, der die im DS-Stack gespeicherten Informationen enthält, die in der Variablen "stack" indiziert sind. Anschließend wird der Stack geleert und die Ini-Datei geschlossen.
