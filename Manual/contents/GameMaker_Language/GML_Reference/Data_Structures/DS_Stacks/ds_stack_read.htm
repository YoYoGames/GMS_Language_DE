ds_stack_read

Mit dieser Funktion können Sie einen gespeicherten DS-Stapel wiederherstellen (einen, der zuvor als String mit ds_stack_write()). Sie müssen zuerst einen neuen DS-Stack erstellen, um den String einzulesen, und wenn der DS-Stack bereits existiert und Informationen darin gespeichert sind, wird dieser vor dem Lesen gelöscht. Diese Funktion ist von entscheidender Bedeutung, wenn Sie Speicher-/Lademechanismen für Ihr Spiel erstellen. Beachten Sie, dass wenn die Datenstruktur mit früheren Versionen von GameMaker erstellt wurde, sollten Sie das optionale Argument "legacy" hinzufügen, indem Sie es auf true setzen, da sich das String-Format mit dieser Version geändert hat.

Syntax:

ds_stack_read(id, str [, legacy] );

Argument Beschreibung
id Die ID der Datenstruktur, in die eingelesen werden soll.
str Die Zeichenkette, aus der gelesen werden soll.
Vermächtnis(optional) Kann entweder wahr oder falsch sein oder komplett weggelassen werden.

Rückgabe:

N/A

Beispiel:

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
ds_stack_read(stack, str);
}
ini_close();

Der obige Code erzeugt einen Stack und speichert den Index in der Variablen "stack". Er öffnet dann eine ini-Datei und liest eine Zeichenkette aus dieser, wobei er zuerst überprüft, ob die Zeichenkette nicht als leer zurückgegeben wird. Diese Zeichenkette wird dann in den neu angelegten DS-Stack eingelesen.
