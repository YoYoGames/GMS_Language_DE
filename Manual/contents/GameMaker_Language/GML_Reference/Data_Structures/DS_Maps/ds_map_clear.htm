ds_map_clear

Diese Funktion löscht die (zuvor erstellte) DS-Map von allen Schlüssel/Wert-Paaren, aber sie löscht nicht die DS-Map selbst aus dem Speicher. Hierfür sollten Sie die Funktion ds_map_destroy(). Beachten Sie, dass das Löschen einer Map alle in der Map gespeicherten Datenstrukturen de-referenziert, was zu einem Speicherleck führt. Um dies zu verhindern, müssen Sie also alle Datenstrukturelemente manuell löschen, bevor Sie die Map löschen. Dies ist nur dann nicht erforderlich, wenn Wenn Sie Elemente in der Map als DS-Liste oder als eine andere DS-Map gekennzeichnet haben, werden diese Elemente zerstört (nicht gelöscht!) und ihr Speicher wird automatisch bereinigt, wenn die übergeordnete Map gelöscht wird.

Syntax:

ds_map_clear(id);

Argument Beschreibung
id Die ID der zu löschenden Datenstruktur.

Rückgabe:

N/A

Beispiel:

if global.new_game
{
ds_map_clear(inventory);
}

Die obige Funktion prüft eine globale Variable, um zu sehen, ob sie wahr ist, und wenn dies der Fall ist, dann wird die in der Instanzvariablen "inventory" indizierte DS-Map von allen Schlüssel/Wert-Paaren gelöscht (aber nicht zerstört ).
