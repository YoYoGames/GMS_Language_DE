ds_map_write

Diese Funktion wandelt die DS-Map-Daten der angegebenen Map in ein String-Format um, das dann zur einfachen Speicherung in eine *.ini- oder eine *.txt-Datei geschrieben werden kann. Dieser String kann dann später wieder in eine neue ds_map eingelesen werden mit ds_map_read().

HINWEIS: Die zurückgegebene Zeichenfolge ist keine von Menschen lesbare Zeichenfolge, sondern ein Dump des Inhalts der Datenstruktur.

Syntax:

ds_map_write(id);

Argument Beschreibung
id Die ID der zu verwendenden Datenstruktur

Rückgabe:

Zeichenfolge

Beispiel:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Der obige Code öffnet eine Ini-Datei, in die geschrieben werden kann. Er verwendet dann ds_map_write(), um eine Zeichenkette zu erzeugen, die in der temporären Variablen "t_string" gespeichert wird. Schließlich wird diese Zeichenfolge in die Ini-Datei geschrieben, bevor sie geschlossen wird.
