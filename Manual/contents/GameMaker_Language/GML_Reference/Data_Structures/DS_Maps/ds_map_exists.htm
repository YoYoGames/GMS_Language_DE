ds_map_exists

Diese Funktion gibt true zurück, wenn der angegebene Schlüssel in der (zuvor erstellten) DS-Map existiert, und false, wenn er nicht existiert.

Syntax:

ds_map_exists(id, key);

Argument Beschreibung
id die ID der zu prüfenden Datenstruktur
Taste die Taste zur Überprüfung von

Rückgabe:

Boolesche

Beispiel:

if !ds_map_exists(inventar, "tränke")
{
ds_map_add(inventar, "tränke", 1);
}

Der obige Code überprüft die in der Variable "inventory" indizierte DS-Map auf den Schlüssel "trions" und fügt ihn, falls er nicht existiert, der Map hinzu.
