ds_map_empty

Diese Funktion gibt einfach false zurück, wenn die angegebene (zuvor erstellte) DS-Map irgendwelche Schlüssel/Wertpaare enthält, oder true, wenn sie keine enthält.

Syntax:

ds_map_empty(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Boolesche

Beispiel:

if ds_map_empty(inventory)
{
gewicht = 0;
}

Der obige Code prüft, ob die DS-Map, die in der Variablen "inventory" indiziert ist, Schlüssel/Wert-Paare enthält, und wenn nicht, setzt er die Variable "weight" auf 0.
