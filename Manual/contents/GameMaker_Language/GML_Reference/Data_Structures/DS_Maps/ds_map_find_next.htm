ds_map_find_next

Diese Funktion gibt den nächsten Schlüssel zurück, der in der DS-Map nach dem in der Funktion angegebenen Schlüssel gespeichert ist. Dies kann nützlich sein, wenn Sie die DS-Map auf der Suche nach etwas durchlaufen müssen, sollte aber nach Möglichkeit vermieden werden, da es langsam sein kann. Wenn kein solcher Schlüssel existiert, gibt die Funktion einen undefinierten Wert zurück. Sie sollten dies immer mit der is_undefined() Funktion.

Syntax:

ds_map_find_next(id, key);

Argument Beschreibung
id Die ID der zu verwendenden Karte.
Taste Der Schlüssel, um den nächsten zu finden.

Rückgabe:

Real, String oder undefiniert

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if (key != "gold")
{
key = ds_map_find_next(inventory, key);
}
else break;
}

Der obige Code erstellt einige temporäre Variablen und ruft dann die Größe der DS-Map ab und findet den ersten Schlüssel, den der Computer in der Map gespeichert hat. Er verwendet dann eine for-Schleife, um durch die DS-Map zu iterieren und nach dem Schlüsselwert "gold" zu suchen. Wenn er ihn findet, bricht er die Schleife ab.
