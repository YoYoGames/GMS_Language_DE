ds_map_add_list

Mit dieser Funktion können Sie eine (zuvor erstellte) DS-Liste einem Schlüssel innerhalb der angegebenen DS-Map zuordnen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren und sollte nur in Verbindung mit dieser Funktionalität verwendet werden. Wenn einer DS-Map auf diese Weise eine Liste hinzugefügt wurde, werden beim Zerstören der übergeordneten Map auch die enthaltenen Listen zerstört und ihr Speicher freigegeben, und der Aufruf von ds_map_clear() auf der übergeordneten Map zerstört und bereinigt auch alle markierten Listen.

Syntax:

ds_map_add_list(id, schlüssel, wert)

Argument Beschreibung
id Die ID der zu verwendenden Karte.
Taste Die Taste für die hinzugefügte Liste.
Wert Die ID der hinzuzufügenden Liste.

Rückgabe:

N/A

Beispiel:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Der obige Code erstellt eine Liste und füllt sie mit den verschiedenen Werten der globalen Variablen. Diese Liste wird dann in einer DS-Map "verschachtelt", und die Map wird dann in einen JSON-String kodiert, bevor die Map zerstört wird, wodurch sie und alle in ihr enthaltenen Listen aus dem Speicher entfernt werden.
