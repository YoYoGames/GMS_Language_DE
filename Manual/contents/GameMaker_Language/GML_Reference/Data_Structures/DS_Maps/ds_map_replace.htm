ds_map_replace

Mit dieser Funktion können Sie den Wert für den angegebenen Schlüssel innerhalb einer DS-Map ändern. Sie liefern den Index der Map (wie er von der Funktion ds_map_create()) und dann den zu ersetzenden Schlüssel - entweder eine Zeichenkette oder eine Ganzzahl - und den Wert, durch den der Schlüsselwert ersetzt werden soll. Wenn der angegebene Schlüssel nicht existiert, wird er für Sie erstellt, und wenn er existiert, wird der aktuelle Wert durch den neuen Wert ersetzt. Wert ersetzt. Die Funktion gibt true zurück, wenn der Schlüssel existiert und der Wert ersetzt wurde, und false, wenn der Schlüssel nicht existiert und ein neuer Schlüssel mit dem Wert erstellt wurde.

Syntax:

ds_map_replace( id, key, val );

Argument Beschreibung
id Die ID der zu ändernden Karte.
Taste Der Schlüssel mit dem Wert, der durch den neuen Wert ersetzt werden soll
val Der neue Wert, mit dem der angegebene Wert ersetzt wird

Rückgabe:

Boolesche

Beispiel:

ds_map_replace(inventory, "torso", 55);

Der obige Code sucht in der DS-Map nach dem Schlüssel "torso" und wenn er ihn findet (oder er wird erstellt, wenn er nicht existiert), wird der aktuelle Wert durch den angegebenen ersetzt.
