ds_map_secure_save_buffer

Diese Funktion speichert eine zuvor erstellte DS-Map in einem Puffer. Sie geben den DS-Map-ID-Wert (wie von der Funktion ds_map_erstellen()) und die ID des Puffers, in den geschrieben werden soll (zurückgegeben von der Funktion buffer_create()). Beachten Sie, dass, wenn die zu speichernde DS-Map ein Array enthält, dieses beim Speichern stattdessen in eine DS-Liste umgewandelt wird.

WICHTIG! Die sicher gespeicherte DS-Kartendatei kann nur auf dem Gerät geladen werden, auf dem sie erstellt wurde. Wenn Sie versuchen, eine Datei zu laden, die auf einem anderen Gerät gespeichert wurde, dann wird sie nicht funktionieren.

Syntax:

ds_map_secure_save_buffer(Dateiname);

Argument Beschreibung
id Der Wert der DS-Map-ID.
Puffer Der Wert der Puffer-ID.

Rückgabe:

Integer (DS-Kartenindex)

Beispiel:

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map, "bob", "ajob");
ds_map_add(map, "geld",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

Der obige Code erstellt einen Puffer und eine DS-Map, füllt dann die Map mit einigen Werten und schreibt sie in den Puffer, bevor er die Map löscht.
