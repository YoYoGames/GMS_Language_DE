ds_map_replace_map

Mit dieser Funktion können Sie eine DS-Map, die im angegebenen "Schlüssel" gespeichert wurde, durch eine andere, zuvor erstellte Map ersetzen. Diese Funktion ist dafür gedacht, JSON-kompatible Maps zu erstellen, die Sie dann mit json_encode() kodieren und sollte nur in Verbindung mit dieser Funktionalität verwendet werden.

Syntax:

ds_map_replace_map(id, Schlüssel, Wert)

Argument Beschreibung
id Die ID der zu verwendenden ds_map.
Taste Die Taste zum Ersetzen.
Wert Die ID der ds_map, die verwendet werden soll, um die zuvor im angegebenen Schlüssel gespeicherte zu ersetzen.

Rückgabe:

N/A

Beispiel:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Der obige Code erstellt eine DS-Map und füllt sie mit einem Array und einer DS-Liste, bevor er eine zuvor gespeicherte Map in der DS-Map "j_map" ersetzt.
