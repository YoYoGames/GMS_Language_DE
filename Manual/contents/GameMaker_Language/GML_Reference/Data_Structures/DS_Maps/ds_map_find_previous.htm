ds_map_find_previous

Diese Funktion gibt den vorherigen Schlüssel zurück, der in der DS-Map vor dem in der Funktion angegebenen Schlüssel gespeichert ist. Dies kann nützlich sein, wenn Sie durch die DS-Map iterieren müssen, um etwas zu suchen, sollte aber möglichst vermieden werden, da es langsam sein kann. Wenn kein solcher Schlüssel existiert, gibt die Funktion undefiniert zurück. Sie sollten dies immer mit der is_undefined() Funktion.

Syntax:

ds_map_find_previous(id, key);

Argument Beschreibung
id Die ID der zu verwendenden Karte.
Taste Der Schlüssel, um den vorherigen zu finden.

Rückgabe:

Real, String oder undefiniert

Beispiel:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
if (key != "gold")
{
key = ds_map_find_previous(inventar, key);
}
else break;
}

Der obige Code erstellt einige temporäre Variablen und ruft dann die Größe der ds_map ab und findet den letzten Schlüssel, den der Computer in der Map gespeichert hat. Er verwendet dann eine for-Schleife, um die ds_map zu durchlaufen und nach dem Schlüsselwert "gold" zu suchen. Wenn er ihn findet, bricht er die Schleife ab.
