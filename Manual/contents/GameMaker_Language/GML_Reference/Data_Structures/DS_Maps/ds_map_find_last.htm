ds_map_find_last

Diese Funktion gibt den letzten in der DS-Map gespeicherten Schlüssel zurück. Dies ist nicht der letzte Schlüssel in der Reihenfolge, in der Sie ihn hinzugefügt haben! DS-Maps werden nicht in einer linearen Form gespeichert, dafür verwenden Sie DS-Listen, also ist alles, was diese Funktion tut, den letzten Schlüssel zu finden, wie er vom Computer gespeichert wurde. Dies kann nützlich sein, wenn Sie durch die DS-Map iterieren müssen, um etwas zu suchen, sollte aber wenn möglich vermieden werden, da es langsam sein kann.

Beachten Sie, dass diese Funktion undefiniert zurückgibt, wenn die angegebene DS-Map leer ist.

Syntax:

ds_map_find_last(id);

Argument Beschreibung
id Die ID der zu verwendenden Karte.

Rückgabe:

Real, String oder undefiniert

Beispiel:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
if (key != "gold")
{
key = ds_map_find_previous(inventory, key);
}
else break;
}

Der obige Code erstellt einige temporäre Variablen und ruft dann die Größe der DS-Map ab und findet den letzten Schlüssel, der vom Computer in der Map gespeichert wurde. Er verwendet dann eine for-Schleife, um die DS-Map zu durchlaufen und nach dem Schlüsselwert "gold" zu suchen. Wenn er ihn findet, bricht er die Schleife ab.
