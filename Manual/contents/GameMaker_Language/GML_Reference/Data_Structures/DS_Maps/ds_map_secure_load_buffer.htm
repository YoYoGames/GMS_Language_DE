ds_map_secure_load_buffer

Diese Funktion lädt eine sicher gespeicherte DS-Map aus einem Puffer. Sie müssen den Puffer zuvor in den Speicher geladen haben (mit puffer_laden()), und die Übergabe an diese Funktion gibt eine DS Map zurück, die mit dem Inhalt des Puffers gefüllt ist. Beachten Sie, dass der Puffer mit der Funktion ds_map_sicher_speichern_puffer()erstellt worden sein muss, damit dies korrekt funktioniert, und beachten Sie auch, dass, wenn die DS Map ein Array enthält, wird dieses beim Laden stattdessen in eine DS-Liste konvertiert.

WICHTIG! Eine der Eigenschaften einer sicher gespeicherten Datei ist, dass sie an das Gerät gebunden ist, auf dem sie erstellt wurde. Sie können also eine auf einem Gerät gespeicherte Datei nicht in ein Projekt laden, das auf einem anderen Gerät läuft.

Syntax:

ds_map_secure_load_buffer(buffer);

Argument Beschreibung
Dateiname Die Puffer-ID des Puffers, aus dem die Kartendaten geladen werden sollen.

Rückgabe:

Integer (DS-Kartenindex)

Beispiel:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Der obige Code lädt eine sicher gespeicherte DS-Map aus einem Puffer und speichert ihren Indexwert in einer Variablen zur späteren Verwendung.
