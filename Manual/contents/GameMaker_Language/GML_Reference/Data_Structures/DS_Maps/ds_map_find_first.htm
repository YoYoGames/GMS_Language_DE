ds_map_find_first

Diese Funktion gibt den ersten Schlüssel zurück, der in der DS-Map gespeichert ist. Dies ist nicht der erste Schlüssel in der Reihenfolge, in der Sie ihn hinzugefügt haben! DS-Maps werden nicht in einer linearen Form gespeichert, verwenden Sie dafür DS-Liste, so dass alles, was diese Funktion tut, das Finden des ersten Schlüssel, wie er vom Computer gespeichert wurde. Dies kann nützlich sein, wenn Sie durch die DS-Map iterieren müssen, um nach etwas zu suchen, sollte aber nach Möglichkeit vermieden werden, da es langsam sein kann.

Beachten Sie, dass diese Funktion undefiniert zurückgibt, wenn die angegebene DS-Map leer ist.

Syntax:

ds_map_find_first(id);

Argument Beschreibung
id Die ID der zu verwendenden Karte.

Rückgabe:

Real, String oder undefiniert

Beispiel:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if key != "gold"
{
key = ds_map_find_next(inventory, key);
}
else break;
}

Der obige Code erstellt einige temporäre Variablen und ruft dann die Größe der DS-Map ab und findet den ersten Schlüssel, den der Computer in der Map gespeichert hat. Er verwendet dann eine for-Schleife, um durch die DS-Map zu iterieren und nach dem Schlüsselwert "gold" zu suchen. Wenn er ihn findet, bricht er die Schleife ab.
