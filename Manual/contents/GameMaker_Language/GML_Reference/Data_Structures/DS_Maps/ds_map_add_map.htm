ds_map_add_map

Mit dieser Funktion können Sie eine (zuvor erstellte) DS-Map einem Schlüssel innerhalb der angegebenen DS-Map zuordnen. Diese Funktion ist für die Erstellung von JSON-kompatiblen Maps gedacht, die Sie dann mit json_encode() kodieren und sollte nur in Verbindung mit dieser Funktionalität verwendet werden. Wenn einer DS-Map eine andere Map auf diese Weise hinzugefügt wurde, werden beim Zerstören der übergeordneten Map auch die enthaltenen Maps zerstört und ihr Speicher freigegeben, und der Aufruf von ds_map_clear() auf der übergeordneten Map zerstört und bereinigt auch alle markierten Maps.

Syntax:

ds_map_add_map(id, schlüssel, wert)

Argument Beschreibung
id Die ID der zu verwendenden Karte.
Taste Die Taste für das hinzugefügte Kennfeld.
Wert Die ID der hinzuzufügenden Karte.

Rückgabe:

N/A

Beispiel:

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

Der obige Code erstellt zwei DS-Maps und füllt dann eine mit einer Liste und einem Array, bevor er sie in die zweite einfügt, die dann in einen JSON-String kodiert wird. Die Map wird dann zerstört, um sie und alle anderen Maps oder Listen, die sie enthält, zu entfernen, aus dem Speicher zu entfernen.
