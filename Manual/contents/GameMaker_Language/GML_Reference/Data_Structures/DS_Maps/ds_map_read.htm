ds_map_read

Diese Funktion nimmt einen String, der zuvor mit der Funktion ds_map_write() erstellt wurde, und liest sie dann in eine zuvor erstellte DS-Map ein. Wenn die Map, in die die Zeichenkette eingelesen wird, irgendwelche Schlüssel/Wert-Paare enthält, werden diese zuerst gelöscht, bevor die gespeicherte Map neu aufgebaut wird. Beachten Sie, dass Sie, wenn die Datenstruktur mit früheren Versionen von GameMaker Studio 2 erstellt wurde, das optionale Argument "legacy" hinzufügen und es auf true setzen, da sich das String-Format mit dieser Version geändert hat.

Syntax:

ds_map_read(id, str [, legacy]);

Argument Beschreibung
id Die ID der Datenstruktur, in die der String eingelesen werden soll
str Die zu lesende Zeichenkette
Vermächtnis(optional) Kann entweder wahr oder falsch sein oder komplett weggelassen werden.

Rückgabe:

N/A

Beispiel:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
ds_map_read(inventory, t_string);
}
ini_close();

Der obige Code erstellt eine neue DS-Map und speichert ihren ID-Index in der Variablen "inventory". Dann öffnet er eine Ini-Datei und liest eine Zeichenkette aus dieser Datei in die temporäre Variable "t_string". Schließlich wird überprüft, ob die t_string" gültig ist (nicht der Standard-Ini-Wert ""). Ist dies der Fall, wird die Zeichenfolge in die neu erstellte DS-Map eingelesen, bevor die Ini-Datei wieder geschlossen wird.
