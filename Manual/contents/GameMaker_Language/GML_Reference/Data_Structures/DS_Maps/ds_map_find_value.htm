ds_map_find_value

Mit dieser Funktion können Sie den Wert aus einem angegebenen Schlüssel ermitteln. Die Eingabewerte der Funktion sind die zu verwendende (zuvor erstellte) DS-Map und der Schlüssel, auf den geprüft werden soll.

HINWEIS: Wenn kein solcher Schlüssel existiert, gibt die Funktion undefiniert zurück. Sie sollten dies immer mit der is_undefined() Funktion

Syntax:

ds_map_find_value(id, key)

Argument Beschreibung
id Die ID der zu verwendenden Karte.
Taste Der Schlüssel zum Finden.

Rückgabe:

Real, String oder undefiniert

Beispiel:

Menge = ds_map_find_value(inventory, "food");

Oder Sie verwenden den Map-Accessor "?":

Menge = Inventar[? "Essen"];

Der obige Code ermittelt den Wert des Schlüssels "food" und speichert ihn in der Variablen "amount".
