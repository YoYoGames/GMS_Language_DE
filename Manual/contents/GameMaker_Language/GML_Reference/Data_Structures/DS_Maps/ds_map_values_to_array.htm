ds_map_values_to_array

Mit dieser Funktion können Sie alle Werte abrufen, die eine DS-Map für jeden Schlüssel in der Map enthält. Sie geben die ID der DS-Map an, von der Sie die Schlüssel abrufen wollen (wie sie von ds_map_create()) und die Funktion gibt ein Array zurück, bei dem jeder Eintrag im Array ein Wert aus der DS-Map ist (wenn die Map z. B. 10 Schlüssel hat, wird ein Array mit 10 Einträgen zurückgegeben, wobei jeder Eintrag einem Wert für einen Schlüssel einem Wert für einen Schlüssel entspricht). Die Funktion hat ein optionales zweites Argument, in dem Sie ein von Ihnen erstelltes Array angeben können. In diesem Fall werden die Schlüsselwertdaten der Map an die bereits vorhandenen Daten im Array angehängt. Beachten Sie, dass die Funktion das übergebene Array direkt ändert, aber auch einen Verweis darauf zurückgibt (oder einen Verweis auf ein neues Array, wenn keines übergeben wird).

Syntax:

ds_map_values_to_array(id, [array])

Argument Beschreibung
id Die ID der zu verwendenden Karte.
[array] OPTIONAL! Das Array, an das die DS-Map-Werte angehängt werden.

Rückgabe:

Array

Beispiel:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
for (var i = 0; i < array_lengh(values); ++i;)
{
_total += _values[i];
}
draw_text(32, 32, "Gesamtpunktzahl für alle Stufen = " + string(_total"));

Die obige Funktion ruft alle Werte aus einer DS-Map für jeden Schlüssel in der Map ab und speichert sie in einer lokalen Variablen als Array. Dann wird das Array in einer Schleife durchlaufen und jeder der Werte zu einer lokalen Variablen hinzugefügt, die dann verwendet wird, um die Summe aller Werte anzuzeigen. Werte anzeigt.
