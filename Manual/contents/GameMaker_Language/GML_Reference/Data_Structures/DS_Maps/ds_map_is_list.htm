ds_map_ist_liste

Mit dieser Funktion können Sie prüfen, ob eine DS-Liste im angegebenen Map-Schlüssel gespeichert ist. Wenn der angegebene Schlüssel eine DS-Listen-ID enthält, dann gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

ds_map_is_list(id, Schlüssel)

Argument Beschreibung
id Die ID der zu verwendenden ds_map.
Taste Die Taste zum Prüfen.

Rückgabe:

Boolesche

Beispiel:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if ds_map_is_list(inventar, schlüssel)
{
ds_list_destroy(key);
}
key = ds_map_find_next(inventory);
}
ds_map_destroy(inventar);

Der obige Code läuft in einer Schleife durch eine DS-Map und prüft, ob einer der Schlüssel darin für eine DS-Liste ist. Wenn dies der Fall ist, wird die DS-Liste zerstört, und am Ende der Schleife wird auch die DS-Map zerstört.
