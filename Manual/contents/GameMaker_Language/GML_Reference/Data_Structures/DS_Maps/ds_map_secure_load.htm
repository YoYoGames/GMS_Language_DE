ds_map_secure_load

Diese Funktion lädt eine zuvor gespeicherte DS-Map aus der angegebenen Datei. Diese Datei muss zuvor mit der Funktion ds_map_secure_save() Funktion erstellt worden sein, und die Funktion gibt nach dem Laden den Index der DS-Map zurück, die aus den geladenen Daten erstellt wurde. Dieser DS-Map-Index sollte in einer Variablen gespeichert werden und für alle weiteren Funktionsaufrufe zu dieser Map verwendet werden. Beachten Sie, dass, wenn die zu ladende DS-Map mit einem Array als einem der Schlüsselwerten gespeichert wurde, wurde dieses Array beim Laden in eine DS-Liste umgewandelt.

WICHTIG! Eine der Eigenschaften einer sicher gespeicherten Datei ist, dass sie an das Gerät gebunden ist, auf dem sie erstellt wurde. Sie können also eine auf einem Gerät gespeicherte Datei nicht in ein Projekt laden, das auf einem anderen Gerät läuft.

Syntax:

ds_map_secure_load(Dateiname);

Argument Beschreibung
Dateiname Der Name der Datei, aus der die Kartendaten geladen werden sollen.

Rückgabe:

Integer (DS-Kartenindex)

Beispiel:

p_map = ds_map_secure_load("p_data.dat");

Der obige Code lädt eine sicher gespeicherte DS-Map und speichert ihren Indexwert in einer Variablen zur späteren Verwendung.
