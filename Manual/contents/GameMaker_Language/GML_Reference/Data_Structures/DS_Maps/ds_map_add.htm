ds_map_add

Diese Funktion sollte verwendet werden, um Sätze von Schlüssel/Wertpaaren in die angegebene DS-Map hinzuzufügen. Sie können diese Funktion überprüfen, um zu sehen, ob sie erfolgreich war oder nicht (bei Erfolg gibt sie true zurück, ansonsten false ), da sie fehlschlagen kann, wenn derselbe Schlüssel bereits in der DS-Map existiert oder Sie eine nicht existierende DS-Map als ID der Map angeben, zu der hinzugefügt werden soll. Die Schlüssel und Werte, die Sie übergeben, können aus einer beliebigen Kombination von Datentypen bestehen, so dass alle folgenden - und mehr - sind akzeptabel (obwohl Sie in der Praxis am häufigsten eine Zeichenkette für den Schlüssel verwenden werden):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

Sie können auch mit dem Accessor "?" zu einer Karte hinzufügen, wie unten gezeigt:

map[? 5] = undefiniert;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

HINWEIS: Im Gegensatz zu anderen Datenstrukturen in GameMaker Studio 2 geht dieser Schlüssel nicht an den Anfang (und auch nicht an das Ende) der DS-Map, sondern er geht einfach irgendwo in die DS-Map hinein.

Syntax:

ds_map_add(id, key, val);

Argument Beschreibung
id Die ID der Map, zu der hinzugefügt werden soll.
Taste Der Schlüssel des hinzuzufügenden Wertes.
val Der Wert, der zur Karte hinzugefügt werden soll.

Rückgabe:

Boolesche

Beispiel:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Dadurch wird eine neue Karte erstellt, deren ID der Variablen "inventory" zugewiesen wird. Dann fügt es der Karte zwei neue Schlüssel hinzu, "hp potion" und "gold", und setzt ihre Anfangswerte auf 1 und 100.
