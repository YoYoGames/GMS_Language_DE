ds_map_delete

Mit dieser Funktion können Sie einen beliebigen Schlüssel (und den dazugehörigen Wert) aus der gegebenen, zuvor erstellten, DS-Map entfernen.

Syntax:

ds_map_delete(id, key);

Argument Beschreibung
id Die ID der zu ändernden Karte.
Taste Der zu löschende Schlüssel (mit dem zugehörigen Wert).

Rückgabe:

N/A

Beispiel:

ds_map_delete(inventory, "shield");

Der obige Code löscht den Schlüssel "shield" (und den Wert, mit dem er gepaart ist) aus der DS-Map (Inventar).
