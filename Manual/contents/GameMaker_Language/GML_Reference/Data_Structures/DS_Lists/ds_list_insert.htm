ds_list_insert

Diese Funktion fügt den angegebenen Wert an der angegebenen Position in die Liste ein. Wenn die Liste nach der angegebenen Position weitere Werte enthält, wird deren Position um eins nach oben verschoben, um Platz zu schaffen, wodurch die Liste um eins größer wird.

Syntax:

ds_list_insert(id, pos, val);

Argument Beschreibung
id Die ID der Liste, zu der hinzugefügt werden soll.
pos Die Position, an der der Wert hinzugefügt werden soll, wobei 0 dem Anfang der Liste entspricht und die Endposition ds_list_size(id)-1 ist.
val Der Wert, der der Liste hinzugefügt werden soll.

Rückgabe:

N/A

Beispiel:

ds_list_insert(list, 9, score);

Der obige Code fügt den in der Variablen "score" gespeicherten Wert an die 10. Position der durch die Variable "list" indizierten Liste hinzu (Listen werden von 0 an gezählt, eine Liste mit zehn Werten ist also von 0 - 9 nummeriert).
