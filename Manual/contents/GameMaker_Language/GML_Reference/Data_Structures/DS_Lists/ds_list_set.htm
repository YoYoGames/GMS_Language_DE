ds_list_set

Mit dieser Funktion können Sie einen zuvor hinzugefügten Listeneintrag setzen. Sie geben die Listen-ID (wie beim Erstellen der Liste zurückgegeben) und die Position innerhalb der Liste an, die gesetzt werden soll, sowie den Wert, auf den er gesetzt werden soll. Beachten Sie, dass, wenn der zu setzende Eintrag außerhalb der Grenzen der Liste liegt der Liste liegt (d. h., Sie setzen den Listeneintrag 20, aber die aktuelle Liste enthält nur 10 Einträge), dann wird die Liste bis zur angegebenen Position gefüllt und jeder Eintrag auf 0 gesetzt. Diese Funktion entspricht der Verwendung des Listenzugriffs DS.

Syntax:

ds_list_set(id, pos, val);

Argument Beschreibung
id Die ID der Liste, zu der hinzugefügt werden soll.
pos Die zu setzende Position innerhalb der Liste.
[val2, ... max_val] Der Wert, der in der Liste eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

for (var i = 0; i < ds_list_size(list); i++;)
{
ds_list_set(list, i, -1);
}

Der obige Code fügt den in der Variablen "score" gespeicherten Wert in die in der Variablen "sc_list" indizierte Liste ein.
