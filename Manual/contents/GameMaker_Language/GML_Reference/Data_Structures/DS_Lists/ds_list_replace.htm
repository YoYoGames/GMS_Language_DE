ds_list_replace

Mit dieser Funktion wird der Wert an der angegebenen Position durch einen anderen ersetzt.

Syntax:

ds_list_replace(id, pos, val);

Argument Beschreibung
id Die ID der zu ändernden Liste.
pos Die Position, an der der Wert ersetzt werden soll, wobei 0 dem Anfang der Liste entspricht und die Endposition ds_list_size(id)-1 ist.
val Der neue Wert, durch den der angegebene Wert ersetzt werden soll.

Rückgabe:

N/A

Beispiel:

ds_list_replace(n_list, 3, name);

Der obige Code ersetzt den an Position 3 in der Liste gespeicherten Wert durch den in der Variablen "name" gespeicherten Wert.
