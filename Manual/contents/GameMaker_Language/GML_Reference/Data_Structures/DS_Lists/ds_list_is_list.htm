ds_liste_ist_liste

Mit dieser Funktion können Sie prüfen, ob an der angegebenen Position innerhalb einer DS-Liste eine weitere DS-Liste gespeichert ist. Wenn die angegebene Position eine DS-Listen-ID enthält, dann gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

ds_list_is_list(id, pos);

Argument Beschreibung
id Die ID der zu prüfenden Liste.
pos Die zu prüfende Position innerhalb der Liste.

Rückgabe:

Boolesche

Beispiel:

var size = ds_list_size(Schiffe) ;
for (var i = 0; i < size; i++;)
{
if ds_list_is_list(Schiffe, i)
{
ds_list_destroy(i);
}
}
ds_list_destroy(ships);

Der obige Code läuft in einer Schleife durch eine DS-Liste und prüft, ob einer der Einträge andere Listen-IDs enthält. Wenn dies der Fall ist, werden diese Listen zerstört, und die Hauptliste wird nach Beendigung der Schleife zerstört.
