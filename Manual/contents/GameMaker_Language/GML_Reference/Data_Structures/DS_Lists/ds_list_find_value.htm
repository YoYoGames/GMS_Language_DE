ds_list_find_value

Mit dieser Funktion können Sie die angegebene Listenposition überprüfen und der Wert, der innerhalb der Liste für diese Position gehalten wird, wird zurückgegeben. Beachten Sie, dass, wenn Sie eine Position angeben, die außerhalb der angegebenen Listengröße liegt (z. B. Position 11 in einer Liste mit 10 Werten), die Funktion undefiniert oder 0 zurückgeben kann. Das liegt daran, dass beim Erstellen der Liste intern die ersten paar Einträge in der Liste auf 0 gesetzt werden, um Leistungsprobleme beim anfänglichen Hinzufügen von Elementen zur Liste zu minimieren (obwohl die ds_list_size() Funktion auch bei einer neu erstellten Liste 0 zurückgibt). Wenn Sie sicherstellen wollen, dass die Liste beim Erstellen "wirklich" leer ist, sollten Sie ds_list_clear() aufrufen, nachdem Sie die Liste erstellt haben, was dann bedeutet, dass alle Werte, die für nicht ausgefüllte Listenslots zurückgegeben werden, undefiniert sind, was Sie dann konsequent mit der is_undefined() Funktion.

Syntax:

ds_list_find_value(id, pos);

Argument Beschreibung
id Die ID der zu verwendenden Liste.
pos Die zu betrachtende Position, wobei 0 dem Anfang der Liste entspricht und die Endposition ds_list_size(id)-1 ist.

Rückgabe:

Real, String oder Undefiniert

Beispiel:

val = ds_list_find_value(list, ds_list_size(list) - 1);

Der obige Code prüft die in der Variablen "list" indizierte Liste an der letzten Position in der Liste und speichert den zurückgegebenen Wert in der Variablen "val".
