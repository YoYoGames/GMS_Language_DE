ds_list_find_index

Mit dieser Funktion können Sie die angegebene Liste auf einen Wert prüfen und die Position innerhalb der Liste für diesen Wert wird zurückgegeben. Beachten Sie, dass, wenn mehrere Einträge mit demselben Wert in der Liste vorhanden sind, die Position eines beliebigen Eintrags zurückgegeben werden kann, und dass, wenn der Wert nicht existiert, -1 zurückgegeben wird. Beachten Sie, dass der Wert ein Array sein kann, das Sie mit der Funktion is_array().

Syntax:

ds_list_find_index(id, val);

Argument Beschreibung
id Die ID der zu verwendenden Liste.
val Der zu findende Wert.

Rückgabe:

Real

Beispiel:

pos = ds_list_find_index(list, "Player1");

Der obige Code überprüft die in der Variablen "list" indizierte Liste auf den Wert "Player1" und speichert die zurückgegebene Position in der Variablen "pos".
