ds_liste_kopieren

Mit dieser Funktion können Sie den Inhalt einer Liste in eine andere kopieren. Beide Listen müssen zuvor erstellt worden sein, und wenn die Liste, in die kopiert wird, bereits Informationen enthält, wird diese Liste zuerst gelöscht. Das Endergebnis sind zwei unabhängige Listen, die die gleichen Informationen enthalten.

Syntax:

ds_list_copy( id, source );

Argument Beschreibung
id Die ID der Liste, in die kopiert wird.
Quelle Die ID der Liste, aus der kopiert werden soll.

Rückgabe:

N/A

Beispiel:

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list);
}

Der obige Code prüft eine DS-Liste, um zu sehen, ob sie leer ist. Wenn sie nicht leer ist, wird sie in eine andere DS-Liste kopiert (die zuvor erstellt wurde) und dann wird die ursprüngliche Liste gelöscht.
