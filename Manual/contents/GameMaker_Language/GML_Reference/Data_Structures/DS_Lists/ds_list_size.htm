ds_list_size

Diese Funktion gibt die "Größe" der Liste zurück, d. h. die Anzahl der Elemente, die der Liste hinzugefügt wurden.

Syntax:

ds_list_size(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Real

Beispiel:

if !ds_list_empty(control_list)
{
num = ds_list_size(control_list);
}

Der obige Code prüft eine DS-Liste, um zu sehen, ob sie leer ist oder nicht, und wenn nicht, erhält er die Anzahl der darin enthaltenen Elemente und speichert den Wert in einer Variablen.
