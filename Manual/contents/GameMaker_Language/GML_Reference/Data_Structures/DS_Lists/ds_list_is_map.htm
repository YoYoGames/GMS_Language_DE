ds_list_is_map

Mit dieser Funktion können Sie prüfen, ob an der angegebenen Position innerhalb einer DS-Liste ein DS-Map gespeichert ist. Wenn die angegebene Position eine DS-Map-ID enthält, dann gibt die Funktion true zurück, andernfalls gibt sie false zurück.

Syntax:

ds_list_is_map(id, pos);

Argument Beschreibung
id Die ID der zu prüfenden Liste.
pos Die zu prüfende Position innerhalb der Liste.

Rückgabe:

Boolesche

Beispiel:

var size = ds_list_size(ships) ;
for (var i = 0; i < size; i++;)
{
if ds_list_is_map(Schiffe, i)
{
ds_map_destroy(i);
}
}
ds_list_destroy(ships);

Der obige Code läuft in einer Schleife durch eine DS-Liste und prüft, ob einer der Einträge Kennfeld-IDs enthält. Wenn dies der Fall ist, werden diese Kennfelder zerstört, und die Hauptliste wird nach Beendigung der Schleife zerstört.
