ds_list_delete

Mit dieser Funktion können Sie den Wert entfernen, der an einer bestimmten Position innerhalb der Liste gespeichert ist.

Syntax:

ds_list_delete(id, pos);

Argument Beschreibung
id Die ID der zu ändernden Liste.
pos Wo in der Liste soll der Wert gelöscht werden.

Rückgabe:

N/A

Beispiel:

if (ds_list_size(sc_list) > 10)
{
while (ds_list_size(sc_list) > 10)
{
ds_list_delete(sc_list, 0);
}
}

Der obige Code prüft die Größe einer DS-Liste und wenn diese größer als zehn ist, wird die Liste in einer Schleife durchlaufen, wobei der oberste Wert (Position 0) entfernt wird, bis die Liste nur noch 10 Einträge hat.
