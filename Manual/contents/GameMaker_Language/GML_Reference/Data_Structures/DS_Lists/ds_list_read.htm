ds_liste_lesen

Mit dieser Funktion können Sie eine gespeicherte DS-Liste wiederherstellen (eine, die zuvor als String mit ds_list_write()). Sie müssen zunächst eine neue DS-Liste erstellen, in die Sie den String einlesen, und wenn die DS-Liste bereits existiert und Informationen darin gespeichert sind, werden diese vor dem Lesen gelöscht. Diese Funktion ist von entscheidender Bedeutung, wenn Sie Speicher-/Lademechanismen für Ihr Spiel erstellen.Beachten Sie, dass wenn die Datenstruktur mit früheren Versionen von GameMaker erstellt wurde, sollten Sie das optionale Argument "legacy" hinzufügen, indem Sie es auf true setzen, da sich das String-Format mit dieser Version geändert hat.

Syntax:

ds_list_read(id, str [, legacy]);

Argument Beschreibung
id Die ID der Datenstruktur, in die eingelesen werden soll.
str Die Zeichenkette, aus der gelesen werden soll.
Vermächtnis(optional) Kann entweder wahr oder falsch sein oder komplett weggelassen werden.

Rückgabe:

N/A

Beispiel:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Listen", "0", "");
if str != ""
{
ds_list_read(list, str);
}
ini_close();

Der obige Code erstellt eine Liste und speichert den Index in der Variablen "list". Dann wird eine ini-Datei geöffnet und eine Zeichenkette daraus gelesen, wobei zuerst überprüft wird, dass die Zeichenkette nicht als leer zurückgegeben wird. Diese Zeichenkette wird dann in die die neu erstellte ds_list eingelesen.
