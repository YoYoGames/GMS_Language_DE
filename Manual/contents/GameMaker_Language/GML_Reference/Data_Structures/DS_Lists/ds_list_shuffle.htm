ds_list_shuffle

Mit dieser Funktion können Sie eine Liste mischen, wodurch alle Komponentenwerte an zufälligen Positionen neu angeordnet werden, und zwar an denen, an denen sie ursprünglich der Liste hinzugefügt wurden.

HINWEIS: Diese Funktion mischt die Listenelemente bei jedem neuen Durchlauf des Spiels an dieselben Positionen, da GameMaker Studio 2 jedes Mal denselben anfänglichen Zufallswert generiert, um das Debuggen von Code zu erleichtern. Zur Vermeidung von dieses Verhalten zu vermeiden, verwenden Sie randomise() beim Start Ihres Spiels. Dies gilt nur für das Testen und Debuggen des Spiels, da das endgültige ausführbare Paket dieses Verhalten nicht aufweisen wird und bei jedem Spiel zufällig ist.

Syntax:

ds_list_shuffle(id);

Argument Beschreibung
id Die ID der Liste, die gemischt werden soll.

Rückgabe:

N/A

Beispiel:

if restart
{
ds_list_shuffle(card_list);
}

Der obige Code mischt die in der Variablen "card_list" indizierte Liste, wenn die Variable "restart" als true gekennzeichnet ist.
