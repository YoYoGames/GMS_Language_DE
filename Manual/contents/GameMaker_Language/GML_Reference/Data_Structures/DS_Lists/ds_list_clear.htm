ds_list_clear

Mit dieser Funktion können Sie alle Daten aus der angegebenen Listen-Datenstruktur löschen. Dabei wird die Datenstruktur NICHT zerstört (dafür sollten Sie die Funktion ds_list_destroy()), sondern löscht nur alle Daten aus ihr und macht die Liste leer (Größe Null). Beachten Sie, dass das Löschen einer Liste alle in ihr gespeicherten Datenstrukturen de-referenziert, was zu einem Speicherleck führt, so dass Sie die Liste durchgehen und alle Datenstrukturelemente manuell zerstören müssen, bevor Sie sie löschen, um dies zu verhindern dies zu verhindern. Dies ist nur dann nicht erforderlich, wenn Sie Elemente in der Liste als eine andere DS-Liste oder als eine DS-Map gekennzeichnet haben. In diesem Fall werden diese Elemente zerstört (nicht gelöscht!) und ihr Speicher wird automatisch bereinigt, wenn das übergeordnete Element gelöscht wird.

Syntax:

ds_list_clear(id);

Argument Beschreibung
id Die ID der zu löschenden Datenstruktur.

Rückgabe:

N/A

Beispiel:

if (count == 15) && (!ds_list_empty(command_list))
{
ds_list_clear(command_list);
alarm[0] = room_speed;
ai_count = 0;
}

Der obige Code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat, und wenn dies der Fall ist, löscht er die in der Variable "command_list" indizierte DS-Liste, setzt einen Alarm und setzt die Variable auf 0 zurück.
