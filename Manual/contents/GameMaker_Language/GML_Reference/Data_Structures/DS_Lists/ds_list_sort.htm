ds_liste_sortieren

Mit dieser Funktion können Sie alle Werte innerhalb einer Liste entweder aufsteigend oder absteigend sortieren. Wenn die Liste Zeichenketten enthält, werden diese alphabetisch sortiert, basierend auf dem englischen 26-Buchstaben-Alphabet.

Syntax:

ds_list_sort(id, ascend);

Argument Beschreibung
id Die ID der zu sortierenden Liste.
aufsteigen Ob die Werte aufsteigend(true) oder absteigend(false) geordnet sein sollen.

Rückgabe:

N/A

Beispiel:

if newgame
{
ds_list_sort(name_list, true);
}

Der obige Code sortiert die in der Variablen "name_list" indizierte Liste, wenn die Variable "newgame" als wahr gekennzeichnet ist.
