ds_existiert

Mit dieser Funktion können Sie prüfen, ob eine Datenstruktur des angegebenen Typs existiert. Sie übergeben den Wert "index" (wie in einer Variablen gehalten) und den DS "type", der eine der unten aufgeführten Konstanten sein kann, und die Funktion gibt true zurück, wenn die Datenstruktur existiert, und andernfalls false.

Konstant Beschreibung
ds_type_map Eine Map-Datenstruktur
ds_type_list Eine Listendatenstruktur
ds_type_stack Eine Stack-Datenstruktur
ds_type_grid Eine Gitterdatenstruktur
ds_type_queue Eine Warteschlangen-Datenstruktur
ds_type_priorität Eine Prioritätsdatenstruktur

Syntax:

ds_exists(ind, typ);

Argument Beschreibung
ind Der Variablenindex zur Prüfung der Datenstruktur
Typ Der Typ der Datenstruktur, auf den geprüft werden soll (siehe die Liste der Konstanten unten)

Rückgabe:

Boolesche

Beispiel:

if !ds_exists(ai_grid, ds_type_grid)
{
ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

Der obige Code prüft die (zuvor initialisierte) Variable "ai_grid", um zu sehen, ob sie eine Datenstruktur vom Typ DS-Grid indiziert, und wenn nicht, dann erstellt sie eine und speichert ihren Index in der Variablen.
