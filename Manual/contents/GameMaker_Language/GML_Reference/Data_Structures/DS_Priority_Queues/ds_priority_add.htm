ds_prioritaet_hinzufuegen

Mit dieser Funktion können Sie einen Wert (entweder eine reelle Zahl oder eine Zeichenkette) zu einer Prioritätswarteschlange hinzufügen und ihr gleichzeitig einen Prioritätswert zuweisen.

Syntax:

ds_priority_add(id, val, priority);

Argument Beschreibung
id Die ID der Prioritätswarteschlange, zu der hinzugefügt werden soll.
val Der Wert, der der Prioritätswarteschlange hinzugefügt werden soll.
Priorität Die Priorität des hinzuzufügenden Wertes.

Rückgabe:

N/A

Beispiel:

ds_priority_add(ai_priority, AI_Search, 5);

Der obige Code fügt der in der Variablen "ai_priority" indizierten Prioritätswarteschlange eine Skriptfunktion hinzu und weist ihr eine Priorität von 5 zu.
