ds_prioritaet_find_prioritaet

Mit dieser Funktion können Sie die Priorität eines beliebigen Wertes abrufen. Wenn der Wert nicht in der Prioritäts-Warteschlange vorhanden ist, wird undefiniert zurückgegeben.

Syntax:

ds_priority_find_priority(id, val);

Argument Beschreibung
id Die ID der zu verwendenden Prioritätswarteschlange.
val Der Wert, für den die Priorität gefunden werden soll.

Rückgabe:

Real oder undefiniert

Beispiel:

p = ds_priority_find_priority(ai_priority, "intelligence");

Der obige Code speichert die zurückgegebene Priorität für den angegebenen Wert in der Instanzvariablen "p".
