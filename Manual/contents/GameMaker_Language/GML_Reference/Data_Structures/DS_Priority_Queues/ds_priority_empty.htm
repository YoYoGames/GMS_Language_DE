ds_prioritaet_leer

Mit dieser Funktion können Sie die angegebene DS-Prioritätswarteschlange daraufhin überprüfen, ob sie leer ist (gibt true zurück) oder nicht (gibt false zurück).

Syntax:

ds_priority_empty(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Boolesche

Beispiel:

if (count == 15) && (!ds_priority_empty(command_queue))
{
ds_priority_clear(command_queue);
alarm[0] = room_speed;
ai_count = 0;
}

Der obige Code prüft eine Variable, um zu sehen, ob sie einen bestimmten Wert erreicht hat, und wenn dies der Fall ist, löscht er die DS-Prioritätswarteschlange, die in der Variablen "command_queue" indiziert ist, setzt einen Alarm und setzt die Variable auf 0 zurück.
