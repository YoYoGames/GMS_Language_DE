ds_prioritaet_groesse

Diese Funktion gibt die "Größe" der Prioritäts-Warteschlange zurück, d. h. die Anzahl der Elemente, die in der Warteschlange priorisiert wurden.

Syntax:

ds_priority_size(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Real

Beispiel:

if !ds_priority_empty(control_priority)
{
num = ds_priority_size(control_priority);
}

Der obige Code prüft eine DS-Prioritätswarteschlange, um zu sehen, ob sie leer ist oder nicht, und wenn sie es nicht ist, wird die Anzahl der darin enthaltenen Elemente abgerufen und der Wert in einer Variablen gespeichert.
