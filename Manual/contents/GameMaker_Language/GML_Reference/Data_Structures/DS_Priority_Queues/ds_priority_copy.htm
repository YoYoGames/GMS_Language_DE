ds_priorität_kopie

Mit dieser Funktion können Sie den Inhalt einer Prioritäts-Warteschlange in eine andere kopieren. Beachten Sie, dass dadurch der Inhalt der ursprünglichen Prioritäts-Warteschlange NICHT entfernt wird und die ursprüngliche Prioritäts-Warteschlange auch nicht zerstört wird. Wenn Sie diese Funktion verwenden, muss die Prioritäts Wenn diese Funktion verwendet wird, muss die Prioritäts-Warteschlange, in die kopiert wird, zuvor erstellt worden sein, und wenn sie vor dem Kopieren Elemente enthielt, werden diese zuerst gelöscht (d. h. diese Informationen gehen verloren).

Syntax:

ds_priority_copy(id, Quelle);

Argument Beschreibung
id Die ID der Prioritätswarteschlange, in die kopiert werden soll.
Quelle Die ID der Prioritätswarteschlange, aus der kopiert werden soll.

Rückgabe:

N/A

Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create();
ds_priority_copy(p_queue, other.p_queue);
}

Die obige Funktion erstellt eine neue Instanz und erstellt dann in dieser Instanz eine neue DS-Prioritätswarteschlange und kopiert den Inhalt der Prioritätswarteschlange in der Instanz, die den Codeblock ausführt, in die neu erstellte Instanz-Prioritätswarteschlange.
