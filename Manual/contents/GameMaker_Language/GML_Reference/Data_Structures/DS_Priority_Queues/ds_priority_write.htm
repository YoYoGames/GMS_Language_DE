ds_priority_write

Diese Funktion gibt eine Zeichenkette zurück, die dann gespeichert oder mit der ds_prioritaet_lesen() Funktion.

HINWEIS: Die zurückgegebene Zeichenfolge ist keine von Menschen lesbare Zeichenfolge, sondern ein Dump des Inhalts der Datenstruktur

Syntax:

ds_priority_write(id);

Argument Beschreibung
id Die ID der zu prüfenden Datenstruktur.

Rückgabe:

Zeichenfolge

Beispiel:

var str;
ini_open("save.ini");
str =ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

Der obige Code öffnet eine Ini-Datei und schreibt dann einen String, der die in der DS-Prioritätswarteschlange gespeicherten Informationen enthält, die in der Variablen "p_queue" indiziert sind. Anschließend wird die Prioritäts-Warteschlange geleert und die Ini-Datei geschlossen.
