ds_prioritaet_loeschen_wert

Diese Funktion löscht einfach den angegebenen Wert zusammen mit seiner Priorität aus der indizierten Prioritätswarteschlange.

Syntax:

ds_priority_delete_value(id,val);

Argument Beschreibung
id Die ID der zu verwendenden Prioritätswarteschlange.
val Der Wert, der aus der Prioritätswarteschlange gelöscht werden soll.

Rückgabe:

N/A

Beispiel:

if (ai_move == false)
{
ds_priority_delete_value(ai_priority, AI_Move);
}

Der obige Code prüft eine Instanzvariable und wenn er false zurückgibt, wird die indizierte Skriptfunktion aus der Prioritätswarteschlange entfernt.
