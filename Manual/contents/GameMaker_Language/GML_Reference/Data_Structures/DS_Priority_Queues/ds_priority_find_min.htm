ds_prioritaet_find_min

Mit dieser Funktion können Sie den in der Prioritäts-Warteschlange gespeicherten Wert mit der niedrigsten Priorität finden, und wenn mehrere Werte die gleiche Priorität haben, kann jeder von ihnen in beliebiger Reihenfolge zurückgegeben werden. Allerdings wird im Gegensatz zu ds_priority_delete_min(), entfernt diese Funktion jedoch nicht den Wert aus der Warteschlange.

Syntax:

ds_priority_find_min(id);

Argument Beschreibung
id Die ID der zu verwendenden Prioritätswarteschlange.

Rückgabe:

Real oder String

Beispiel:

if ai_move
{
script_execute(ds_priority_find_min(ai_priority));
}

Der obige Code prüft eine Instanzvariable und führt, wenn er true zurückgibt, eine Skriptfunktion aus, die in der Prioritätswarteschlange mit dem niedrigsten Prioritätswert indiziert ist.
