DS-Prioritäts-Warteschlangen

Eine DS-Prioritätswarteschlange ist eine spezielle Datenstruktur ähnlich einer DS-Warteschlange, nur dass hier die Warteschlange der Werte nach den Prioritäten (immer reelle Zahlen) geordnet ist, die den Werten zugewiesen wurden. Diese Struktur kann sehr nützlich sein, um Bestenlisten oder Informationslisten zu erstellen, in denen die Wichtigkeit jedes Eintrags gewichtet werden kann.

HINWEIS: Wie alle dynamischen Ressourcen belegen auch Datenstrukturen Speicherplatz und sollten daher immer zerstört werden, wenn sie nicht mehr benötigt werden, um Speicherlecks zu vermeiden, die Ihr Spiel verlangsamen und schließlich zum Absturz bringen.

HINWEIS: Wenn Sie prüfen müssen, ob eine Datenstruktur existiert, können Sie die ds_exists() Funktion.

Es gibt die folgenden Funktionen, die sich mit DS-Prioritätswarteschlangen befassen:

  • ds_prioritaet_erzeugen
  • ds_prioritaet_zerstoeren
  • ds_prioritaet_klar
  • ds_prioritaet_leer
  • ds_prioritaet_groesse
  • ds_prioritaet_hinzufuegen
  • ds_prioritaet_wechseln_prioritaet
  • ds_prioritaet_loeschen_max
  • ds_prioritaet_loeschen_min
  • ds_prioritaet_loeschen_wert
  • ds_prioritaet_find_max
  • ds_prioritaet_find_min
  • ds_prioritaet_find_prioritaet
  • ds_priorität_kopie
  • ds_prioritaet_lesen
  • ds_priority_write
