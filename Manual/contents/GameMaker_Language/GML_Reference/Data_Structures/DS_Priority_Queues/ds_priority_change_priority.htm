ds_prioritaet_wechseln_prioritaet

Diese Funktion nimmt einen gegebenen Wert und ändert dessen Priorität innerhalb der referenzierten Prioritätswarteschlange.

Syntax:

ds_priority_change_priority(id, val, priority);

Argument Beschreibung
id Die ID der zu ändernden Prioritätswarteschlange.
val Der Wert, für den die Priorität geändert werden soll.
Priorität Die neue Priorität des Wertes.

Rückgabe:

N/A

Beispiel:

if (global.Game_Time < 1000)
{
ds_priority_change(ai_priority, AI_Search, 1);
}

Der obige Code prüft eine globale Variable, und wenn sie unter einem bestimmten Wert liegt, ändert er die Priorität des Skriptfunktionsindex, der in der Prioritätswarteschlange gehalten wird.
