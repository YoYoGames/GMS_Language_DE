ds_prioritaet_loeschen_max

Diese Funktion gibt den Wert zurück, der die höchste Priorität in der Warteschlange hat und entfernt dann den Wert (und die Priorität) aus der Datenstruktur. Wenn mehr als ein Wert die gleiche Priorität hat, kann jeder davon in beliebiger Reihenfolge zurückgegeben werden, aber alle aber alle anderen Werte mit der gleichen Priorität befinden sich noch in der Warteschlange.

Syntax:

ds_priority_delete_max(id);

Argument Beschreibung
id Die ID der zu verwendenden Prioritätswarteschlange.

Rückgabe:

Datentyp, der in der Priorität gespeichert ist

Beispiel:

wenn ai_move
{
script_execute(ds_priority_delete_max(ai_priority));
}

Der obige Code prüft eine Instanzvariable und führt, wenn er "true" zurückgibt, ein in der Prioritätswarteschlange indiziertes Skript mit dem höchsten Prioritätswert aus und entfernt dann dieses Skript aus der Warteschlange.
