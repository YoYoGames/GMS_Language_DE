netzwerk_erzeugen_socket_ext

Diese Funktion wird verwendet, um einen neuen Client-Socket für Ihr Spiel zu erstellen, um über das Netzwerk zu kommunizieren. Sie müssen das Socket-Protokoll definieren (siehe die Liste der Konstanten unten) und einen zu verwendenden Port angeben, und die Funktion gibt eine eindeutige ID zurück, die Die Funktion gibt eine eindeutige ID zurück, die in allen weiteren Funktionsaufrufen für diesen Socket verwendet werden sollte, oder einen Wert kleiner als 0, wenn die Verbindung fehlschlägt.

Konstant Beschreibung
network_socket_tcp Erstellen Sie einen Socket mit TCP.
network_socket_udp Erstellen Sie einen Socket mit UDP.
netzwerk_sockel_ws Erstellen Sie einen Websocket(nur für Projekte, die auf HTML5 laufen), unter Verwendung von TCP.

Syntax:

network_create_socket_ext(protocol, port);

Argument Beschreibung
Protokoll Das zu verwendende Netzwerkprotokoll
Hafen Der zu verwendende Port

Rückgabe:

Real

Beispiel:

client = network_create_socket_ext(network_socket_udp, 6510);

Der obige Code erstellt einen neuen UDP-Socket an Port 6510.
