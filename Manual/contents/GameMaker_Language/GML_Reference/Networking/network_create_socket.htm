netzwerk_erzeugen_socket

Diese Funktion wird verwendet, um einen neuen Client-Socket für Ihr Spiel zu erstellen, um über das Netzwerk zu kommunizieren. Sie müssen den Socket-Typ definieren (siehe die Liste der Konstanten unten) und die Funktion gibt eine eindeutige ID für diesen Socket zurück, die in allen weiteren Funktionsaufrufen für diesen Socket verwendet werden sollte in allen weiteren Funktionsaufrufen für diesen Socket verwendet werden sollte, oder einen Wert kleiner als 0, wenn die Verbindung fehlschlägt.

HINWEIS: Sie können die IP "127.0.0.1" verwenden, um sich wieder mit demselben Gerät zu verbinden, auf dem das Spiel läuft.

Konstant Beschreibung
network_socket_tcp Erstellen Sie einen Socket mit TCP.
network_socket_udp Erstellen Sie einen Socket mit UDP.
netzwerk_sockel_ws Erstellen Sie einen Websocket(nur für Projekte, die auf HTML5 laufen), unter Verwendung von TCP.
netzwerk_steckdose_bluetooth Erstellen Sie eine Bluetooth-Buchse(derzeit nicht verfügbar!).

Syntax:

network_create_socket(type);

Argument Beschreibung
Typ Der Typ der zu erstellenden Socket-Verbindung (siehe die unten aufgeführten Konstanten).

Rückgabe:

Real

Beispiel:

if os_browser == browser_not_a_browser
{
client = network_create_socket(network_socket_tcp);
network_connect( client, "192.134.0.1", 6510 );
}
sonst
{
client = network_create_socket(network_socket_ws);
network_connect( client, "192.134.0.1", 6520 );
}

Der obige Code prüft, ob das Spiel in einem Browser läuft oder nicht, und erstellt einen neuen TCP- oder Web-Socket, bevor er versucht, sich über diesen mit der angegebenen IP-Adresse am angegebenen Port zu verbinden.
