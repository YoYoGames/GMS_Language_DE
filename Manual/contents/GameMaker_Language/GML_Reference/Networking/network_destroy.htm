netzwerk_zerstören

Mit dieser Funktion können Sie eine Netzwerk-Socket-Verbindung aus Ihrem Spiel entfernen.

HINWEIS: Diese Funktion funktioniert nicht, wenn sie in einem Projekt verwendet wird, das auf dem HTML5-Target läuft.

Syntax:

network_destroy(socket);

Argument Beschreibung
Steckdose Die ID des zu entfernenden Sockels.

Rückgabe:

N/A

Beispiel:

if !os_is_network_connected()
{
network_destroy(sock);
}

Der obige Code prüft, ob eine Datenverbindung besteht, und wenn keine gefunden wird, wird der Socket mit der ID "sock" zerstört.
