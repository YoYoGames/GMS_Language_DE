Diese Funktion gibt die IP-Adresse der angegebenen URL zurück.

Syntax:

network_resolve(url);

Argument Beschreibung
url Die URL zum Abrufen der IP von (eine Zeichenkette).

Rückgabe:

Zeichenfolge

Beispiel:

game_ip = network_resolve("www.macsweeneygames.com");

Der obige Code gibt die IP-Adresse der angegebenen URL zurück und speichert sie in der Variablen "game_ip".
