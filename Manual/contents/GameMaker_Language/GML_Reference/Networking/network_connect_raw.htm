netzwerk_verbinden_raw

Mit dieser Funktion können Sie eine Anfrage zur Verbindung mit einem Server senden. Die Funktion nimmt die Socket-ID an, über die die Verbindung hergestellt werden soll (siehe netzwerk_erzeugen_socket()) und verlangt die Angabe der IP-Adresse, zu der die Verbindung aufgebaut werden soll (eine Zeichenkette) sowie den Port, über den die Verbindung hergestellt werden soll. Wenn die Verbindung fehlschlägt, wird ein Wert kleiner 0 zurückgegeben. Der Unterschied zwischen dieser Funktion und netzwerk_verbinden() ist, dass diese Funktion eine Verbindung zu einem beliebigen Server herstellen kann und nichts mit den Rohdaten macht, was bedeutet, dass Sie die Protokolle selbst auf der Serverseite implementieren müssen. Beachten Sie, dass die Funktion standardmäßig synchron ist, was bedeutet, dass Ihr Spiel zu "hängen" scheint, während die Verbindung hergestellt wird. Sie können einen Timeout-Wert für die Verbindung festlegen oder alternativ die Erstellung asynchron machen, indem Sie die Funktion network_set_config()verwenden, oder alternativ die Funktion network_connect_async_raw() stattdessen.

Syntax:

network_connect_raw(socket, url, port);

Argument Beschreibung
Steckdose Die ID des zu verwendenden Sockets.
url Die URL oder IP, zu der eine Verbindung hergestellt werden soll (eine Zeichenfolge).
Hafen Der Anschluss, zu dem eine Verbindung hergestellt werden soll.

Rückgabe:

Real

Beispiel:

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

Der obige Code erstellt einen neuen TCP-Socket und versucht dann, über diesen eine "rohe" Verbindung zu der angegebenen URL an Port 6510 herzustellen.
