netzwerk_zeitlimit_setzen

Mit dieser Funktion können Sie den Timeout für das Lesen und Schreiben von Daten zu/von einem Server über den angegebenen Socket einstellen. Beachten Sie, dass die Zeitüberschreitung keine Art von Ereignis erzeugt, so dass Sie selbst mit Zeitüberschreitungen umgehen müssen, z. B. mit Alarmen. Beachten Sie, dass sich dieser Wert nur auf das Senden und Empfangen von Daten auswirkt. Wenn Sie den Timeout-Wert für die Verbindung ändern möchten, sollten Sie die Funktion network_set_config().

Syntax:

network_set_timeout(socket, read_timeout, write_timeout);

Argument Beschreibung
Steckdose Die ID des zu verwendenden Sockets.
Lesezeitüberschreitung Die Millisekunden, in denen eine Übertragung von einem Server eine Zeitüberschreitung erfährt.
Schreibzeitüberschreitung Die Millisekunden, in denen eine Übertragung zu einem Server eine Zeitüberschreitung erfährt.

Rückgabe:

N/A

Beispiel:

network_set_timeout(sock, 3000, 3000);

Der obige Code setzt den Timeout für das Lesen und Schreiben von Daten über den in der Variablen "sock" indizierten Socket auf 3 Sekunden.
