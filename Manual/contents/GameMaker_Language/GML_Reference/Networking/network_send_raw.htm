netzwerk_senden_raw

Mit dieser Funktion können Sie ein "rohes" Datenpaket über das Netzwerk senden. Die Funktion nimmt die Socket-ID an, über die eine Verbindung hergestellt werden soll, und dann müssen Sie die Puffer-ID angeben, die die zu sendenden Daten enthält (für weitere Informationen zu Puffern siehe Referenz - Puffer) und schließlich die Größe (in Bytes) des Datenpakets. Die gesendeten Daten werden von GameMaker Studio 2 in keiner Weise formatiert und die empfangenden Geräte erhalten die Daten als Stream. was bedeutet, dass Sie es selbst handhaben müssen. Die Funktion gibt die Anzahl der gesendeten Datenbytes zurück, oder eine Zahl kleiner 0, wenn das Senden fehlgeschlagen ist.

Syntax:

network_send_raw(socket, buffer, size);

Argument Beschreibung
Steckdose Die ID des zu verwendenden Sockets.
Puffer Die ID des Puffers, aus dem die Daten geholt werden sollen.
Größe Die Größe (in Bytes) der Daten.

Rückgabe:

Real

Beispiel:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

Die obigen Informationen laden einen zuvor gespeicherten Puffer in den Speicher und geben die Puffer-ID zurück, die in der Variablen "buff" gespeichert wird. Dieser komplette Puffer wird dann als Rohdatenpaket über das Netzwerk unter Verwendung des Sockets, der durch die Variable "sock" identifiziert wird.
