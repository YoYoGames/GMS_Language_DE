netzwerk_verbinden_async

Mit dieser Funktion können Sie eine Anfrage zur Verbindung mit einem Server senden. Die Funktion nimmt die Socket-ID an, über die die Verbindung hergestellt werden soll (siehe network_create_socket()) und verlangt die Angabe der IP-Adresse, zu der die Verbindung aufgebaut werden soll (eine Zeichenkette) sowie den Port, über den die Verbindung hergestellt werden soll. Wenn die Verbindung fehlschlägt, wird ein Wert kleiner 0 zurückgegeben. Die Verbindung verwendet ein spezielles Protokoll, das sicherstellt, dass sich nur GameMaker Studio 2-Spiele miteinander verbinden. mit einem Server verbinden müssen, der kein GameMaker Studio 2-Spiel ist, können Sie network_connect_async_raw(). Beachten Sie, dass diese Funktion asynchron ist und ein Asynchronous Networking-Ereignis vom Typ network_type_non_blocking_connect erzeugt.

Syntax:

network_connect_async(socket, url, port);

Argument Beschreibung
Steckdose Die ID des zu verwendenden Sockets.
url Die URL oder IP, zu der eine Verbindung hergestellt werden soll (eine Zeichenfolge).
Hafen Der Anschluss, zu dem eine Verbindung hergestellt werden soll.

Rückgabe:

Real

Beispiel:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

Der obige Code erstellt einen neuen TCP-Socket und versucht dann, über diesen eine asynchrone Verbindung zu der angegebenen IP-Adresse an Port 6510 herzustellen.
