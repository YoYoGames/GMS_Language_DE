mouse_check_button_released

Diese Funktion gibt true zurück, wenn die überprüfte Maustaste losgelassen wurde, oder false, wenn dies nicht der Fall ist. Diese Funktion wird für jede Maustaste nur einmal ausgelöst, wenn sie losgelassen wird, und um sie erneut auszulösen, muss die Taste muss die Taste gedrückt und wieder losgelassen worden sein. Sie geben die zu prüfende Maustaste aus einer der folgenden Konstanten an:

Konstant Beschreibung
mb_links Die linke Maustaste
mb_middle Die mittlere Maustaste (dies ist möglicherweise nicht für alle Zielplattformen gültig)
mb_rechts Die rechte Maustaste
mb_side1* Seitliche Maustaste 1
mb_side2* Seitliche Maustaste 2
mb_beliebig Eine der Maustasten
mb_keine Keine Maustaste


* HINWEIS: Die Schaltflächen mb_side1 und mb_side2 sind nur unter Windows, macOS, Ubuntu und HTML5 verfügbar.

Syntax:

mouse_check_button_released(numb);

Argument Beschreibung
taub Auf welche Maustastenkonstante geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if mouse_check_button_released(mb_right)
{
speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

Der obige Code prüft, ob die rechte Maustaste losgelassen wurde, und wenn ja, setzt er die Geschwindigkeit der Instanz auf ein Zehntel des Abstands zwischen der aktuellen x/y-Position und der x/y-Position der Maus.
