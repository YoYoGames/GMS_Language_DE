maus_rad_auf

Diese Funktion gibt true zurück, wenn das Mausrad nach oben gedreht wird, sonst false.

Syntax:

mouse_wheel_up();

Rückgabe:

Boolesche

Beispiel:

if mouse_wheel_up()
{
y -= 10;
}

Damit wird die aktuelle Instanz auf dem Bildschirm nach oben verschoben, wenn das Mausrad nach oben gedreht wird.
