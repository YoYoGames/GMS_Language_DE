Maus-Eingabe

Die Mauseingabe wird auf allen Plattformen akzeptiert (auf mobilen Geräten wird sie als Einzelbildschirmberührung akzeptiert - wenn Sie Multitouch verwenden müssen, sollten Sie die gerätespezifischen Funktionen verwenden) und hat einige Konstanten die verwendet werden, um die zu drückenden Tasten zu spezifizieren. Diese Konstanten sind in der folgenden Tabelle aufgeführt:

Konstant Beschreibung
mb_links Die linke Maustaste
mb_middle Die mittlere Maustaste (dies ist möglicherweise nicht für alle Zielplattformen gültig)
mb_rechts Die rechte Maustaste
mb_side1* Seitliche Maustaste 1
mb_side2* Seitliche Maustaste 2
mb_beliebig Eine der Maustasten
mb_keine Keine Maustaste


* HINWEIS: Die Schaltflächen mb_side1 und mb_side2 sind nur unter Windows, macOS, Ubuntu und HTML5 verfügbar.

Für die Standard-Maustastensteuerungen gibt es die folgenden Funktionen:

  • maus_taste
  • maus_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • Maus_klar
  • maus_lastbutton
  • maus_rad_auf
  • Maus_Rad_ab
  • Maus_x
  • Maus_y

Beachten Sie, dass es auch eine Reihe von Fensterfunktionen gibt, die sich auf die Verwendung der Maus auf Desktop-Zielen beziehen:

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
