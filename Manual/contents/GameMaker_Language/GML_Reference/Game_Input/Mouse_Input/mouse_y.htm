Maus_y

Diese schreibgeschützte Variable gibt die aktuelle y-Achsenposition der Maus im Raum zurück.

Syntax:

maus_y;

Rückgabe:

Real

Beispiel:

y = median(64, Maus_y, Raum_Höhe - 64);

Der obige Code hält die Instanz an der y-Position der Maus, solange sie sich innerhalb der Grenzen von 64 Pixeln vom oberen und unteren Rand des Raums befindet.
