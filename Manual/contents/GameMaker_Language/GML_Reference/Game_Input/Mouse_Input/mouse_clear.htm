Maus_klar

Mit dieser Funktion wird der aktuelle Zustand der angegebenen Maustaste gelöscht. Das bedeutet, dass die Prüfung, ob die Taste gedrückt ist, erst dann true zurückgibt, wenn der Spieler die Taste loslässt und erneut drückt (der Loslass-Status wird jedoch weiterhin erkannt wenn der Löschvorgang durchgeführt wird, während die Maustaste gedrückt gehalten wird).

Sie können eine der auf dieser Seite aufgeführten Maustastenkonstanten angeben.

Syntax:

mouse_clear(Taste);

Argument Beschreibung
Taste Auf welche Maustastenkonstante geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

mouse_clear(mb_any);

Der obige Code löscht den Abwärtszustand aller Maustasten.
