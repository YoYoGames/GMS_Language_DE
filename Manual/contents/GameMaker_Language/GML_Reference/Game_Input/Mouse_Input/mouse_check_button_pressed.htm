mouse_check_button_pressed

Diese Funktion gibt true zurück, wenn die zu prüfende Maustaste gedrückt wurde, oder false, wenn sie nicht gedrückt wurde. Diese Funktion wird für jede Maustaste nur einmal ausgelöst, wenn sie zum ersten Mal gedrückt wird, und um sie erneut auszulösen, muss die Taste muss losgelassen und erneut gedrückt werden. Beachten Sie, dass sie für die Dauer des Schritts und für alle Instanzen, die Mausereignisse haben oder die dieselbe Funktion verwenden, als ausgelöst gilt.

Sie liefern die zu prüfende Maustaste aus einer der folgenden Konstanten:

Konstant Beschreibung
mb_links Die linke Maustaste
mb_middle Die mittlere Maustaste (dies ist möglicherweise nicht für alle Zielplattformen gültig)
mb_rechts Die rechte Maustaste
mb_side1* Seitliche Maustaste 1
mb_side2* Seitliche Maustaste 2
mb_beliebig Eine der Maustasten
mb_keine Keine Maustaste


* HINWEIS: Die Schaltflächen mb_side1 und mb_side2 sind nur unter Windows, macOS, Ubuntu und HTML5 verfügbar.

Syntax:

mouse_check_button_pressed(numb);

Argument Beschreibung
taub Auf welche Maustastenkonstante geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if mouse_check_button_pressed(mb_left)
{
score += 50;
}

Der obige Code prüft, ob die linke Maustaste gedrückt wurde, und fügt in diesem Fall 50 zum Punktestand hinzu.
