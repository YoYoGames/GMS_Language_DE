Maus_x

Diese schreibgeschützte Variable gibt die aktuelle x-Achsenposition der Maus im Raum zurück.

Syntax:

Maus_x;

Rückgabe:

Real

Beispiel:

x = median(64, mouse_x, room_width - 64);

Der obige Code hält die Instanz an der Maus-X-Position, solange sie sich innerhalb der Grenzen von 64 Pixeln von jeder Seite des Raums befindet.
