keyboard_virtual_height

Diese Funktion gibt die aktuelle Höhe der virtuellen Tastatur in Pixel zurück, basierend auf der Größe des Displays. Wenn die Tastatur nicht sichtbar ist, wird 0 zurückgegeben.

Syntax

keyboard_virtual_height();

Rückgabe:

Int

Beispiel:

if keyboard_virtual_status() == true
{
key_h = keyboard_virtual_height();
}

Der obige Code prüft den Status der virtuellen Tastatur des Betriebssystems und setzt, wenn sie sichtbar ist, eine Variable auf die Höhe der Tastatur.
