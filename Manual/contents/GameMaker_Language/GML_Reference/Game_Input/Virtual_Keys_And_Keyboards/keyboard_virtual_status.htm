keyboard_virtual_status

Mit dieser Funktion kann der Status der virtuellen Tastatur auf dem Gerät, auf dem das Spiel läuft, abgefragt werden. Die Funktion gibt true zurück, wenn die virtuelle Tastatur des Betriebssystems sichtbar ist/angezeigt wird, oder false, wenn sie verborgen/versteckt ist.

Syntax

keyboard_virtual_status();

Rückgabe:

Boolesche

Beispiel:

var _status = keyboard_virtual_status();
if _status == false
{
keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

Der obige Code zeigt die virtuelle Tastatur des Betriebssystems an, wenn der aktuelle Status false ist.
