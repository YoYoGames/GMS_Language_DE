keyboard_virtual_hide

Mit dieser Funktion können Sie die virtuelle Tastatur auf dem Gerät, auf dem das Spiel läuft, ausblenden. Der Aufruf dieser Funktion erzeugt ein asynchrones Systemereignis, bei dem die async_load DS-Map mit den folgenden Schlüssel/Wertpaaren gefüllt wird:

  • " event_type" - dies wird " virtualkeyboard status" sein, wenn es durch Funktionen der virtuellen Tastatur ausgelöst wird.
  • "screen_height" - die Höhe der virtuellen Tastatur (in Pixel). Dieser Wert ist 0, wenn die Tastatur unsichtbar ist.
  • "keyboard_status" - der aktuelle Status der Tastatur, zurückgegeben als eine der folgenden Zeichenketten:
    • "versteckt"
    • "versteckt"
    • "zeigend"
    • "sichtbar"

Syntax

keyboard_virtual_hide();

Rückgabe:

N/A

Beispiel:

if input == true
{
input = false;
keyboard_virtual_hide();
}

Der obige Code blendet die virtuelle Tastatur des Betriebssystems aus, wenn die angegebene Variable nicht auf false gesetzt ist.
