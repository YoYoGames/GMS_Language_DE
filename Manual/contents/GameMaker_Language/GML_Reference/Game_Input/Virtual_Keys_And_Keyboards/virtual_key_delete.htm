virtual_key_delete

Wenn Ihr Spiel über verschiedene Räume oder Instanzen zur Steuerung von Menüs und Spielabläufen und anderen Dingen verfügt, ist es wahrscheinlich, dass Sie irgendwann die Position und Tastenbelegung Ihrer virtuellen Tasten ändern müssen. Dafür können Sie diese Funktion verwenden, um die alten zu löschen, bevor Sie die neuen erstellen (falls erforderlich). Die Funktion erfordert, dass Sie die ID der zu löschenden virtuellen Taste angeben - wie sie von virtual_key_add().

Beachten Sie, dass alle virtuellen Keys automatisch aus dem Raum entfernt werden, wenn der Raum gewechselt wird, sodass diese Funktion nur notwendig ist, wenn Sie die Keys manuell entfernen möchten, bevor der Raum selbst beendet ist.

Syntax

virtual_key_delete(index);

Argument Beschreibung
index Der Index der zu löschenden virtuellen Taste.

Rückgabe:

N/A

Beispiel:

virtual_key_delete(global.Left);

Der obige Code löscht den virtuellen Schlüssel, der in der globalen Variable "global.Left" indiziert ist.
