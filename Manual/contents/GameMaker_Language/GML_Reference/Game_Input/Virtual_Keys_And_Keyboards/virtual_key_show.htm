virtual_key_show

Nachdem Sie Ihre virtuelle Taste für Ihre Geräte erstellt haben, müssen Sie eventuell debuggen und testen, ob sie korrekt positioniert ist und richtig funktioniert. Dazu rufen Sie diese Funktion auf und verwenden den Index der virtuellen Taste, die Sie sehen möchten (zuvor erstellt und gespeichert mit virtual_key_add()), die ein farbiges Rechteck zeichnet, um die Taste auf dem Bildschirm darzustellen. Wenn Sie damit zufrieden sind, empfiehlt es sich, eine eigene Grafik zu erstellen und in ihrem Bereich zu platzieren.

HINWEIS: Das gezeichnete Rechteck wird von den Blending-, Farb- und Alpha-Optionen beeinflusst, die von den entsprechenden Zeichenfunktionen eingestellt werden.

Syntax

virtual_key_show(index);

Argument Beschreibung
index Der Index der virtuellen Taste, die angezeigt werden soll.

Rückgabe:

N/A

Beispiel:

if global.Debug
{
virtual_key_show(global.Left);
}
sonst
{
virtual_key_hide(global.Left);
}

Der obige Code prüft die globale Variable "global.Debug" und wenn der Test wahr ist, wird die virtuelle Taste, die in der Variable "global.Left" indiziert ist, auf dem Bildschirm gezeichnet, und wenn er falsch ist, wird die Taste ausgeblendet.
