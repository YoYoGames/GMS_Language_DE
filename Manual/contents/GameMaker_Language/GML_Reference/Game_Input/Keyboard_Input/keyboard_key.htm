tastatur_taste

Mit dieser Variablen können Sie den Keycode der Taste abfragen, die gerade gedrückt wird, und es wird 0 zurückgegeben, wenn keine Taste gedrückt wird, wenn die Prüfung abgeschlossen ist.

Syntax:

keyboard_key;

Rückgabe:

Real

Beispiel:

switch (tastatur_taste)
{
case vk_numpad1: gun = weapon[0][0]; break;
case vk_numpad2: gun = weapon[1][0]; break;
case vk_numpad3: gun = weapon[2][0]; break;
}

Der obige Code verwendet den Wert der Variable keyboard_key, um eine Variable auf den gleichen Wert wie ein Array zu setzen.
