keyboard_key_press

Mit dieser Funktion können Sie das Drücken einer beliebigen Taste auf der Tastatur simulieren. Wenn sie verwendet wird, wird die Taste als gedrückt gekennzeichnet, bis die entsprechende Freigabefunktion aufgerufen wird (siehe keyboard_key_release()). Die Funktion nimmt einen Keycode-Wert an, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Beachten Sie, dass nach dem Aufruf dieser Funktion, wenn die Taste physisch auf der Tastatur gedrückt wird, dieses Druckereignis erst wieder registriert wird, wenn die Taste physisch losgelassen wurde (wodurch das Freigabeereignis ausgelöst und diese Funktion gestoppt wird), oder die entsprechende Funktion "Taste loslassen" aufgerufen und die Taste erneut gedrückt wurde.

Syntax:

keyboard_key_press(key);

Argument Beschreibung
Taste Die Taste zum Simulieren eines Drucks von.

Rückgabe:

N/A

Beispiel:

keyboard_key_press(vk_space);

Dadurch wird ein Drücken der Leertaste simuliert.
