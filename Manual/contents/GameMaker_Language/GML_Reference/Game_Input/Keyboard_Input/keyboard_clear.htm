Tastatur_klar

Mit dieser Funktion können Sie den aktuellen Tastaturzustand löschen, was im Wesentlichen bedeutet, dass eine gehaltene Taste nicht mehr erkannt wird, bis sie wieder losgelassen wird (was auch bei dieser Gelegenheit nicht das Ereignis Keyboard Key Released auslösen wird auslöst) und erneut gedrückt wird. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe.

Syntax:

keyboard_clear(key);

Argument Beschreibung
Taste Die Taste zum Löschen.

Rückgabe:

N/A

Beispiel:

keyboard_clear(vk_space);

Der obige Code löscht den Zustand der Leertaste.
