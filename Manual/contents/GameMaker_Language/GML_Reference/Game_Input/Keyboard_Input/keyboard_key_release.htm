tastatur_taste_freigeben

Mit dieser Funktion können Sie das Loslassen einer beliebigen Taste auf der Tastatur simulieren. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_key_release(key);

Argument Beschreibung
Taste Die Taste zum Simulieren einer Freigabe von.

Rückgabe:

N/A

Beispiel:

keyboard_key_release(vk_space);

Dadurch wird das Loslassen der Leertaste simuliert.
