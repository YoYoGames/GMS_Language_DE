keyboard_check_pressed

Mit dieser Funktion können Sie prüfen, ob eine Taste gedrückt wurde oder nicht. Anders als die keyboard_check() Funktion wird diese Funktion nur einmal für jede gedrückte Taste ausgeführt, d. h. damit sie erneut ausgelöst wird muss die Taste erst losgelassen und dann erneut gedrückt werden. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite " Tastatureingabe " aufgeführt sind.

Syntax:

keyboard_check_pressed(key);

Argument Beschreibung
Taste Die Taste, deren gedrückter Zustand geprüft werden soll.

Rückgabe:

Boolesche

Beispiel:

if keyboard_check_pressed(vk_anykey)
{
room_goto_next();
}

Der obige Code führt zum nächsten Raum, wenn der Spieler eine beliebige Taste auf der Tastatur drückt (funktioniert wie eine "Press Any Key to Continue"-Aufforderung).
