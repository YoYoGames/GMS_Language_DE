keyboard_unset_map

Mit dieser Funktion können Sie alle neu zugewiesenen Tasten löschen, so dass sie in ihren Standardzustand zurückkehren, d. h. alle Tasten auf sich selbst zuweisen.

Syntax:

keyboard_get_map(key);

Rückgabe:

N/A

Beispiel:

if keyboard_check_pressed(vk_escape)
{
keyboard_unset_map();
}

Der obige Beispielcode setzt alle zugeordneten Tasten auf ihre Standardeinstellungen zurück, wenn der Benutzer die "Escape"-Taste drückt.
