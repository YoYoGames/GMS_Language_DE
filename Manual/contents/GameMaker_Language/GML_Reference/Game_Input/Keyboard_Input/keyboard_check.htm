keyboard_check

Mit dieser Funktion können Sie prüfen, ob eine Taste gedrückt ist oder nicht. Anders als die keyboard_check_pressed() oder keyboard_check_released() Funktionen, die nur einmal ausgelöst werden, wenn die Taste gedrückt oder losgelassen wird, wird diese Funktion bei jedem Schritt ausgelöst, bei dem die Taste gedrückt gehalten wird. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

Syntax:

keyboard_check(key);

Argument Beschreibung
Taste Die Taste zum Überprüfen des heruntergefahrenen Zustands von.

Rückgabe:

Boolesch

Beispiel:

if keyboard_check(vk_left)
{
x -= 5;
}

Der obige Code prüft, ob die Pfeiltaste gedrückt wird, und verschiebt die Instanz bei jedem Schritt, bei dem er true zurückgibt, um 5 Pixel nach links.
