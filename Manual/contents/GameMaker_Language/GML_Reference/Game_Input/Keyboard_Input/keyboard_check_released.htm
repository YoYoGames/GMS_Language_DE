keyboard_check_released

Mit dieser Funktion können Sie prüfen, ob eine Taste losgelassen wurde oder nicht. Anders als die keyboard_check() Funktion wird diese Funktion nur einmal für jedes Loslassen der Taste ausgeführt, damit sie erneut ausgelöst wird, muss die Taste erst gedrückt und dann wieder losgelassen werden. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite " Tastatureingabe " aufgeführt sind.

Syntax:

keyboard_check_released(key);

Argument Beschreibung
Taste Die Taste zum Prüfen des freigegebenen Zustands von.

Rückgabe:

Boolesche

Beispiel:

if keyboard_check_released(ord("P"))
{
instance_create_layer(0, 0, "Controllers", obj_Pause);
}

Der obige Code prüft, ob die Taste "P" losgelassen wurde, und wenn ja, wird eine Instanz von "obj_Pause" erzeugt.
