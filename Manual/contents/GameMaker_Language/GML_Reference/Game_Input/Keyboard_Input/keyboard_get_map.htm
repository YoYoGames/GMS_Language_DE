keyboard_get_map

Manchmal möchten Sie vielleicht den Ascii-Code für eine zugeordnete Taste erhalten (um z. B. zu sehen, ob sie bereits zugeordnet ist), was diese Funktion zurückgibt. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe aufgelistet sind.

Syntax:

keyboard_get_map(key);

Argument Beschreibung
Taste Dies ist der Schlüssel, von dem Sie den gemappten Wert erhalten möchten

Rückgabe:

Real

Beispiel:

if keyboard_get_map(ord("A")) = ord("A")
{
keyboard_set_map(ord("A"), keyboard_lastkey);
}

Der obige Beispielcode prüft zunächst, ob "A" einer anderen Taste zugewiesen wurde, und wenn nicht, wird es der letzten Taste zugewiesen, die der Benutzer gedrückt hat.
