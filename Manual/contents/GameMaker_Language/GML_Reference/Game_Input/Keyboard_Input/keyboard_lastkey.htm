tastatur_lastkey

Diese Variable bezieht sich auf den Wert, der tastatur_taste im vorherigen Frame war und gibt den Tastencode dieser Taste zurück (alle Standard-Tastencode-Konstanten werden zurückgegeben). Diese Variable ist nicht schreibgeschützt und Sie können ändern, z. B. um sie auf -1 zu setzen, wenn Sie sie bereits behandelt haben.

Syntax:

keyboard_lastkey;

Rückgabe:

Real

Beispiel:

if (keyboard_lastkey != -1)
{
str += keyboard_lastchar;
keyboard_lastkey = -1;
}

Der obige Code prüft, ob die Variable "lastkey" ungleich -1 ist, und wenn dies der Fall ist, fügt er der Variablen "str" den Wert der letzten Taste als Zeichenkette hinzu und setzt dann die Variable " keyboard_lastkey" zurück, um weitere Eingaben zu akzeptieren.
