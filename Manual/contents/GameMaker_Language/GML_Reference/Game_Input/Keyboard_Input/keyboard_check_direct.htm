keyboard_check_direct

Diese Funktion gibt true zurück, wenn die Taste mit dem bestimmten Keycode gedrückt ist, oder false, wenn sie nicht gedrückt ist, indem sie die Hardware direkt überprüft. Sie erlaubt ein paar weitere Überprüfungen, insbesondere können Sie die Tastencodes vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol und vk_ralt prüfen, ob die linke oder rechte Shift-, Control- oder Alt-Taste gedrückt ist. Die Funktion nimmt einen Keycode-Wert, wie er von der Funktion ord() zurückgegeben wird (nur Großbuchstaben von A-Z oder Zahlen von 0-9), oder eine der vk_*-Konstanten, die auf der Hauptseite Tastatureingabe aufgeführt sind.

HINWEIS: Diese Funktion ist nur für das Standard-Windows-Ziel verfügbar und das Ergebnis ist unabhängig davon, welche Anwendung den Fokus hat.

Syntax:

keyboard_check_direct(key);

Argument Beschreibung
Taste Die Taste zum Überprüfen des heruntergefahrenen Zustands von.

Rückgabe:

Boolesche

Beispiel:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
crouch = true;
}

Der obige Code prüft, ob die linke oder rechte Alt-Taste gedrückt wurde, und wenn ja, setzt er die Variable "crouch" auf true.
