keyboard_set_numlock

Mit dieser Funktion können Sie die Zifferntastensperre ein- oder ausschalten ( true für ein und false für aus).

HINWEIS: Diese Funktionalität ist nur in den Windows-Ziel-Builds verfügbar und funktioniert nicht auf anderen Geräten.

Syntax:

keyboard_set_numlock(value);

Argument Beschreibung
Wert Setzen Sie dies auf true für "on" und false für "off"

Rückgabe:

N/A

Beispiel:

if keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
sonst
{
keyboard_set_numlock(true);
}

Der obige Beispielcode ermittelt den Status der Taste "numberlock" und setzt sie, wenn sie eingeschaltet ist (true), auf aus (false) und umgekehrt.
