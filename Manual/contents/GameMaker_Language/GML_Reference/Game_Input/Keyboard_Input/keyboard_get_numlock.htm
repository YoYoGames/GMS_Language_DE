keyboard_get_numlock

Mit dieser Funktion können Sie den Status der Zifferntastensperre ermitteln, wobei true für ein und false für aus zurückgegeben wird.

HINWEIS: Diese Funktionalität ist nur in den Windows-Exe-Builds verfügbar und funktioniert nicht auf anderen Geräten.

Syntax:

keyboard_get_numlock();

Rückgabe:

Boolesche

Beispiel:

if keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
sonst
{
keyboard_set_numlock(true);
}

Der obige Beispielcode ermittelt den Status der Taste "numberlock" und setzt sie, wenn sie eingeschaltet ist (true), auf aus (false) und umgekehrt.
