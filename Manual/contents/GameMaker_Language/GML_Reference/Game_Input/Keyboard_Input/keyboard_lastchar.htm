keyboard_lastchar

Diese Variable speichert eine Zeichenkette der zuletzt gedrückten Taste. Diese Variable ist nicht schreibgeschützt und Sie können sie ändern, z. B. auf "" (eine leere Zeichenfolge) setzen, wenn Sie sie bereits behandelt haben.

Syntax:

keyboard_lastchar;

Rückgabe:

Zeichenfolge

Beispiel:

if keyboard_lastkey != -1
{
str += keyboard_lastchar;
keyboard_lastkey = -1;
}

Der obige Code prüft, ob die Variable "lastkey" größer als -1 ist, und wenn ja, fügt er der Variablen "str" den letzten Schlüssel als String hinzu.
