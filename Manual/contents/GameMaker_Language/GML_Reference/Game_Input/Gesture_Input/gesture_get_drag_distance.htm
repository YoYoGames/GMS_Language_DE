gesture_get_drag_distance

Diese Funktion wird verwendet, um den Abstand zu ermitteln, der benötigt wird, um ein Dragging Gesture-Ereignis durch die Bewegung einer Berührung oder eines Klicks auszulösen. Der Abstand wird in Zoll gemessen gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_get_drag_distance();

Rückgabe:

Real (Zoll)

Beispiel:

if gesture_get_drag_distance() != 0.1
{
gesture_drag_distance(0.1);
}

Der obige Code prüft, ob der Schleppabstand für Gesten auf 0,1 Zoll eingestellt ist, und wenn nicht, setzt er ihn auf diesen Wert.
