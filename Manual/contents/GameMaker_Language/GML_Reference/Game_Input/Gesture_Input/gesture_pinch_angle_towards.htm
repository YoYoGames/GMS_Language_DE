Geste_Zwicken_Winkel_zurück

Mit dieser Funktion wird der Winkel eingestellt, innerhalb dessen sich eine Berührung auf eine andere Berührung zubewegen muss, bevor eine Pinch-In-Geste potenziell gestartet werden kann. Der Winkel wird gemessen in Grad gemessen und hat einen Standardwert von 45°.

Syntax:

gesture_pinch_angle_towards();

Rückgabe:

Real (Zoll)

Beispiel:

if gesture_get_pinch_angle_towards() != 45
{
gesture_pinch_angle_towards(45);
}

Der obige Code prüft, ob der Einklemmwinkel für Gesten auf 45° eingestellt ist, und wenn nicht, setzt er ihn auf diesen Wert.
