Geste_Klemme_Winkel_weg

Mit dieser Funktion wird der Winkel eingestellt, innerhalb dessen sich eine Berührung von einer anderen Berührung entfernen muss, bevor eine Pinch-Out-Geste potenziell gestartet werden kann. Der Winkel wird gemessen in Grad gemessen und hat einen Standardwert von 45°.

Syntax:

gesture_pinch_angle_away();

Rückgabe:

Real (Zoll)

Beispiel:

if gesture_get_pinch_angle_away() != 45
{
gesture_pinch_angle_away(45);
}

Der obige Code prüft, ob der Pinch-Out-Winkel für Gesten auf 45° eingestellt ist, und wenn nicht, setzt er ihn auf diesen Wert
