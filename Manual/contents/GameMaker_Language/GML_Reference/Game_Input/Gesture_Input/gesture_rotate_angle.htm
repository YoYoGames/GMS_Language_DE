gesten_rotieren_Winkel

Mit dieser Funktion wird der Winkel eingestellt, den ein Berührungspaar überschreiten muss, um eine Rotate Start Gesture auszulösen. Der Winkel wird in Grad gemessen und hat einen Standard Wert von 5°.

Syntax:

gesture_rotate_angle();

Rückgabe:

Real (Zoll)

Beispiel:

if gesture_get_rotate_angle() != 5
{
gesture_rotate_angle(5);
}

Der obige Code prüft, ob der Drehwinkel für Gesten auf 5° eingestellt ist, und wenn nicht, setzt er ihn auf diesen Wert.
