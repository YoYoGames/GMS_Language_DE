geste_doppel_tap_zeit

Mit dieser Funktion wird die Zeit eingestellt, die zwischen zwei Berührungen/Klicks vergeht, um ein Double Tap Gesture-Ereignis auszulösen. Die Zeit wird in Sekunden gemessen und hat einen Standardwert von 0,16.

Syntax:

gesture_double_tap_time(time);

Argument Beschreibung
Zeit Die Zeit (in Sekunden), die für die Erkennung von Doppel-Tap-Gestenereignissen eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

if gesture_get_double_tap_time() != 0.16
{
gesture_get_double_tap_time(0.16);
}

Der obige Code prüft, ob die Doppeltippzeit für Gesten auf 0,16 Sekunden eingestellt ist, und wenn nicht, setzt er sie auf diesen Wert.
