geste_get_rotate_time

Mit dieser Funktion wird die Zeit ermittelt, innerhalb der sich ein Berührungspaar in eine konsistente Richtung drehen muss, damit eine Rotate Start Gesture ausgelöst wird. Die Zeit wird wird in Sekunden gemessen und hat einen Standardwert von 0,16s.

WICHTIG! Derzeit können Sie dies für ein Spiel mit 60fps nur auf maximal eine Sekunde einstellen, da sonst keine Rotationen erkannt werden. Bei einer niedrigeren Bildrate erhöht sich dieser Wert, z. B. kann ein Spiel mit 30 Bildern pro Sekunde eine maximale Zeit von 2 Sekunden haben.

Syntax:

gesture_get_rotate_time();

Rückgabe:

Real (Zoll)

Beispiel:

if gesture_get_rotate_time() != 0.1
{
gesture_rotate_time(0.1);
}

Der obige Code prüft, ob die Rotationszeit für Gesten auf 0,1 Sekunden eingestellt ist, und wenn nicht, setzt er sie auf diesen Wert.
