event_data

Diese Variable ist global und wird verwendet, um eine DS-Map zu halten, wenn sie in den Gestenereignissen verwendet wird, und -1 zu allen anderen Zeiten. allen anderen Zeiten. Der tatsächliche Inhalt der DS-Map hängt von der Art des Gestenereignisses ab, das es ausgelöst hat, lesen Sie daher die einzelnen Abschnitte für diese Ereignisse.

Syntax:

event_data;

Rückgabe:

Real (ganzzahliger Wert)

Beispiel:

isFlick = event_data[? "isflick"];
if (isFlick)
{
flickVelX = event_data[? "diffX"];
flickVelY = event_data[? "diffY"];
}
sonst
{
flickVelX = 0;
flickVelY = 0;
}

Der obige Code wird vom Ereignis " Ziehen Ende Geste" übernommen und prüft, ob es sich um ein "Flick"-Ereignis handelt. Wenn ja, extrahiert er die erforderlichen Daten aus der DS-Map "event_data" und verwendet sie, um einige Variablen zu setzen. Wenn ein "Flick"-Ereignis Ereignis nicht erkannt wird, werden die gleichen Variablen auf 0 gesetzt.
