gesture_double_tap_distance

Mit dieser Funktion stellen Sie den Abstand ein, innerhalb dessen Sie den Bildschirm nach einmaligem Antippen erneut berühren/anklicken müssen, um eine Double-Tap-Geste auszulösen. Der Abstand wird in Zoll gemessen und hat einen Standardwert von 0,1.

Syntax:

gesture_double_tap_distance(distance);

Argument Beschreibung
Entfernung Der Abstand (in Zoll), der für die Erkennung von Doppel-Tap-Gestenereignissen eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

if gesture_get_double_tap_distance() != 0.1
{
gesture_get_double_tap_distance(0.1);
}

Der obige Code prüft, ob der Doppel-Tap-Abstand für Gesten auf 0,1 Zoll eingestellt ist, und wenn nicht, setzt er ihn auf diesen Wert.
