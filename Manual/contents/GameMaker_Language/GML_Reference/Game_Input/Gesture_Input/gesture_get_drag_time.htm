gesten_holen_ziehen_zeit

Mit dieser Funktion wird die Zeit ermittelt, die es dauert, bis ein Drag-Start-Gesten-Ereignis durch eine Berührung oder einen Klick ausgelöst wird. Diese Zeit beeinflusst auch, wie das Tap-Ereignis ausgelöst wird ausgelöst wird, da ein Berühren/Klicken und Loslassen vor Ablauf dieser Zeit als Tippen betrachtet wird. Die Zeit wird in Sekunden gemessen und hat einen Standardwert von 0,16.

Syntax:

gesture_get_drag_time();

Rückgabe:

Real (Zeit in Sekunden)

Beispiel:

if gesture_get_drag_time() != 0.16
{
gesture_drag_time(0.16);
}

Der obige Code prüft, ob die Ziehzeit für Gesten auf 0,16 Sekunden eingestellt ist, und wenn nicht, setzt er sie auf diesen Wert.
