Geste_Flick_Geschwindigkeit

Mit dieser Funktion wird die Geschwindigkeit eingestellt, die erforderlich ist, damit beim Loslassen einer Berührung oder eines Klicks ein Flick-Gesture-Ereignis ausgelöst wird. Die Geschwindigkeit wird in Zoll pro Sekunde gemessen und hat einen Standardwert von 2,0.

Syntax:

gesture_flick_speed(speed);

Argument Beschreibung
Geschwindigkeit Die Geschwindigkeit (in Zoll pro Sekunde), die für die Erkennung von Flick-Gesten-Ereignissen eingestellt werden soll.

Rückgabe:

N/A

Beispiel:

if gesture_get_flick_speed() != 2
{
gesture_flick_speed(2);
}

Der obige Code prüft, ob die Bewegungsgeschwindigkeit für Gesten auf 2 Zoll pro Sekunde eingestellt ist, und wenn nicht, setzt er sie auf diesen Wert.
