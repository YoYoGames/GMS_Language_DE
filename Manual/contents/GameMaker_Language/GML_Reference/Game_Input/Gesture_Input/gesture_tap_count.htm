gesten_tap_zähler

Mit dieser Funktion wird eingestellt, ob die Tippzählung aktiviert oder deaktiviert ist. Wenn sie auf "true" gesetzt ist, ist die Tippzählung aktiviert, wenn sie auf "false" gesetzt ist, ist sie deaktiviert, wobei Sie den jeweiligen Zustand mit der Funktion geste_get_tap_count(). Wenn sie aktiviert ist, hat jedes Tap-Ereignis einen zusätzlichen DS-Map-Eintrag "tapcount", der die Anzahl der Taps registriert, und wenn sie aktiviert ist, bedeutet das, dass alle Tap-Ereignisse ausgelöst werden, d. h.: zwei Taps lösen sowohl das Single-Tap-Ereignis als auch das Das heißt, zwei Antippen lösen sowohl das Einzel-Tap-Ereignis als auch das Doppel-Tap-Ereignis aus, wobei die Anzahl der Antippen für das Einzel-Tap-Ereignis 1 und die Anzahl der Antippen für das Doppel-Tap-Ereignis 2 ist. Der Tap-Zählwert wird auf 0 zurückgesetzt, nachdem die für eine Doppel-Tap-Erkennung eingestellte Zeit verstrichen ist (siehe die Funktion gesture_double_tap_time()). Wenn die Zapfwellenzählung deaktiviert ist, wird die erste Zapfung erst registriert, wenn die Zeit für die Doppelzapfung abgelaufen ist und keine zweite Zapfung erkannt wurde. Beachten Sie, dass dies standardmäßig aktiviert ist.

Syntax:

gesture_tap_count(enable);

Argument Beschreibung
aktivieren. Aktivieren (auf true gesetzt) oder deaktivieren (auf false gesetzt) Sie die Tippzählung.

Rückgabe:

N/A

Beispiel:

if !gesture_get_tap_count()
{
getsure_tap_count(true);
}

Der obige Code prüft, ob die Tippzählung aktiviert ist, und wenn nicht, wird sie eingeschaltet.
