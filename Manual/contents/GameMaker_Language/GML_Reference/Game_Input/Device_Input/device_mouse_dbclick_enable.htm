device_mouse_dbclick_enable

Mit dieser Funktion kann das Gerät so eingestellt werden, dass ein Doppeltippen der mb_left (linke Maustaste) als mb_right (rechte Maustaste) Tippen erkannt wird oder nicht. Standardmäßig ist dies auf true gesetzt, was bedeutet, dass jedes Mal, wenn der Benutzer zweimal schnell hintereinander auf den Gerätebildschirm zweimal schnell hintereinander tippt, ist der Rückgabewert derselbe, als ob die rechte Maustaste geklickt worden wäre. Wenn dies eingeschaltet ist, wird das erste Antippen als mb_left und das zweite als mb_right erkannt, also stellen Sie sicher, dass dass jeder Code, den Sie verwenden, dies berücksichtigt.

Syntax:

device_mouse_dbclick_enable(bool);

Argument Beschreibung
bool Stellen Sie die Doppelklick-Erkennung ein(true) oder aus(false).

Rückgabe:

Boolesche

Beispiel:

if os_type == os_windows || os_type == os_mac
{
device_mouse_dbclick_enable(false);
}

Der obige Code prüft, ob es sich bei dem Gerät, auf dem das Spiel läuft, um einen Windows-PC oder einen Mac handelt, und wenn es sich um einen von beiden handelt, wird die Doppeltipp-Funktion deaktiviert.
