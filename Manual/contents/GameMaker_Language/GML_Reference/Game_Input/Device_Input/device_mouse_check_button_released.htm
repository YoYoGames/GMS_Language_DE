device_mouse_check_button_released

Diese Funktion gibt true oder false zurück, je nachdem, ob der von Ihnen angegebene Geräte-Touch ausgelöst wurde oder nicht. Das Argument "Gerät" bezieht sich auf die Berührungsnummer des Geräts, die zwischen 0 und n liegen kann, und beachten Sie, dass die maximale Anzahl von Berührungen, die erkannt werden können, sehr stark von dem verwendeten Gerät und dem darauf laufenden Betriebssystem abhängt (die meisten Geräte unterstützen mindestens 4 gleichzeitige Berührungen). Diese Funktion wird nur einmal durch die eigentliche Freigabeaktion ausgelöst, und die auf dieser Seite aufgeführten Konstanten auf dieser Seite aufgeführten Konstanten können zur Überprüfung der Maustasten verwendet werden. Beachten Sie, dass mb_right nur dann erkannt wurde, wenn ein Doppeltap-Touch erkannt und beim zweiten Tap vor dem Loslassen gehalten wird (dieses Verhalten kann mit der Funktion device_mouse_dbclick_enable() deaktiviert werden).

Syntax:

device_mouse_check_button_released(device, button);

Argument Beschreibung
Gerät Das Gerät (von 0 - n), das geprüft wird.
Taste Die Taste des Geräts, das geprüft wird.

Rückgabe:

Boolesche

Beispiel:

if device_mouse_check_button_released(0, mb_left)
{
press=false;
}

Der obige Code prüft, ob das Gerät gedrückt wurde und setzt eine Variable auf false, wenn dies der Fall ist.
