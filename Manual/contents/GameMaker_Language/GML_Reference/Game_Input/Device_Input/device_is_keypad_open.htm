gerät_ist_tastatur_offen

Diese Funktion prüft das Gerät auf ein Tastenfeld und gibt, wenn eines vorhanden ist, true zurück, andernfalls false. Bitte beachten Sie, dass diese Funktion hauptsächlich für die Verwendung mit Android-Geräten gedacht ist

Für Benutzer mit einem Sony Xperia Play gibt es in GameMaker Studio 2 eine festgelegte Tastenbelegung, so dass Sie die Tastaturkonstanten vk_up, vk_down, vk_left verwenden können, vk_right für die Joypad-Tasten und Triangle ist ord("T"), Square ist ord("S "), Circle ist vk_alt + vk_backspace, während die Back-Taste einfach vk_backspace ist, Cross ist vk_space, Select ist vk_return und Start ist vk_rshift, der L-Trigger ist ord("L") und R-Trigger ist ord("R ").

Syntax:

gerät_ist_tastatur_offen

Rückgabe:

Boolesche

Beispiel:

if device_is_keypad_open()
{
global.Einstellung = 2;
}
sonst
{
global.Einstellung = 1;
}

Der obige Code prüft auf eine Tastatur und ändert dann eine globale Variable in Abhängigkeit vom zurückgegebenen Wert.
