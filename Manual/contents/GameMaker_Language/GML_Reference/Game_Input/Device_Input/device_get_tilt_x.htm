gerät_gekippt_x

Diese Funktion gibt einen Wert zwischen -1 und 1 zurück, abhängig vom "Neigungswinkel" des Geräts. Die tatsächliche Korrelation zwischen Neigungsgrad und dem zurückgegebenen Wert hängt vom Gerät und dem verwendeten Betriebssystem ab, aber im Allgemeinen entspricht ein Wert von 1 oder -1 das gleiche wie +/-90°. Die folgende Abbildung zeigt, wie sich jede der verfügbaren Funktionen auf das Gerät bezieht:

Syntax:

device_get_tilt_x()

Rückgabe:

Real

Beispiel:

if display_get_orientation() = display_landscape
{
x += sign(device_get_tilt_y());
}
sonst
{
x += sign(device_get_tilt_x());
}

Der obige Code prüft die Ausrichtung des Displays und verwendet dann den entsprechenden Neigungswert, um den Player entlang der X-Achse zu bewegen.
