device_mouse_check_button_pressed

Diese Funktion gibt true oder false zurück, je nachdem, ob das von Ihnen angegebene Gerät "berührt" (angeklickt) wurde oder nicht. Das Argument "Gerät" bezieht sich auf die Berührungsnummer, die zwischen 0 und n liegen kann, und die maximale Anzahl von Berührungen, die erkannt werden können, hängt sehr stark von dem verwendeten Gerät und dem darauf laufenden Betriebssystem ab (die meisten Geräte unterstützen mindestens 4 gleichzeitige Berührungen). Diese Funktion wird nur einmal durch den eigentlichen Druckvorgang ausgelöst, und die auf dieser Seite aufgeführten Konstanten die auf dieser Seite aufgeführt sind, können zur Überprüfung der Maustasten verwendet werden. Beachten Sie, dass mb_right nur erkannt wird, wenn eine doppelte Berührung erkannt und beim zweiten Tippen gehalten wird (dieses Verhalten kann deaktiviert werden mit der Funktion device_mouse_dbclick_enable()).

Syntax:

device_mouse_check_button_pressed(device, button);

Argument Beschreibung
Gerät Das Gerät (von 0 - n), das geprüft wird.
Taste Die Taste des Geräts, das geprüft wird.

Rückgabe:

Boolesche

Beispiel:

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

Der obige Code prüft, ob das Gerät gedrückt wurde und setzt eine Variable auf true, wenn dies der Fall ist.
