gamepad_ist_unterstützt

Mit dieser Funktion können Sie herausfinden, ob die Zielplattform Gamepads unterstützt (gibt true zurück) oder nicht (gibt false zurück).

Syntax:

gamepad_is_supported();

Rückgabe:

Boolesche

Beispiel:

global.GP = gamepad_is_supported();

Der obige Code prüft, ob ein Gamepad unterstützt wird, und speichert den Rückgabewert in einer globalen Variablen für zukünftige Prüfungen.
