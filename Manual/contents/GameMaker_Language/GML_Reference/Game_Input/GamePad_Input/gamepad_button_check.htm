gamepad_button_check

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Gamepad-Taste als gedrückt gehalten erkannt wird oder nicht. Wenn Sie eine analoge Taste prüfen, berücksichtigt die Prüfung die Schwellenwerteinstellung und gibt nur true zurück, wenn der rohe Eingangswert über dem angegebenen Schwellenwert liegt (Sie können diesen rohen Wert mit der Funktion gamepad_button_value() erhalten).

Syntax:

gamepad_button_check(device, button);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
Taste Auf welche Gamepad-Tastenkonstante ist zu achten.

Rückgabe:

Boolesche

Beispiel:

if gamepad_button_check(0, gp_face1)
{
if canshoot = true
{
audio_play_sound(snd_Shoot, 0, false);
instance_create_layer(x, y, "Bullets", obj_Bullet)
canshoot = false;
alarm[0] = room_speed / 2;
}
}

Der obige Code erkennt, ob die untere Taste der oberen Fläche (das "X" auf einem ps3-Controller) gedrückt gehalten wird, und wenn ja, schießt er eine "Bullet"-Instanz und setzt einen Alarm.
