gamepad_set_vibration

Mit dieser Funktion können Sie die Vibration der Gamepad-Motoren einstellen, wobei jeder Motor einen Wert von 0 (keine Vibration) bis 1 (volle Vibration) verwendet. Beachten Sie, dass diese Funktion nicht zeitlich begrenzt ist, so dass Sie eine Variable oder einen Alarm verwenden müssen, um die Vibration Sie müssen also eine Variable oder einen Alarm verwenden, um die Vibration abzuschalten (Motoren auf 0 setzen), nachdem eine bestimmte Zeit verstrichen ist, sonst vibriert das Gamepad unbegrenzt weiter.

HINWEIS: Diese Funktion ist derzeit nur für die Zielmodule Standard Windows, PS4 und Xbox One Zielmodule.

Syntax:

gamepad_set_vibration(device, left_motor, right_motor);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
links_motor Die Stärke der Vibration des linken Motors von 0 bis 1.
rechts_motor Die Stärke der Vibration des rechten Motors von 0 bis 1.

Rückgabe:

N/A

Beispiel:

if lives = 0
{
gamepad_set_vibration(0, 1, 1);
alarm[0] = room_speed / 2;
}

Der obige Code würde (z. B.) in einem Kollisionsereignis verwendet werden, um das in "Steckplatz" 0 eingesteckte Gamepad für eine halbe Sekunde vibrieren zu lassen, wobei der eingestellte Alarm verwendet wird, um es nach Ablauf dieser Zeit wieder auszuschalten.
