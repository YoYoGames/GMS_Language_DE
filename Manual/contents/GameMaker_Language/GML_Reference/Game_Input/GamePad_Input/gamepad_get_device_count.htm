gamepad_get_device_count

Diese Funktion teilt Ihnen eines von zwei Dingen mit. Entweder die Anzahl der angeschlossenen Gamepads oder die Anzahl der verfügbaren "Steckplätze" für Gamepads, an die sie angeschlossen werden können. Der tatsächliche Rückgabewert hängt von der Plattform und der internen Konfiguration dieser Plattform ab, und daher sollte diese Funktion in Verbindung mit der Funktion gamepad_ist_verbunden() verwendet werden, um die genaue Anzahl der angeschlossenen Pads jederzeit zu ermitteln.

Syntax:

gamepad_get_device_count();

Rückgabe:

Real

Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i)
{
global.gp[i] = true;
}
else
{
global.gp[i] = false;
}
}

Der obige Code durchläuft die verfügbaren Gamepads (oder Gamepad-Steckplätze) und prüft dann jeden einzelnen auf ein angeschlossenes Gamepad. Der zurückgegebene Wert wird dann verwendet, um ein globales Array auf true oder false für die Verwendung in zukünftigen Prüfungen zu setzen.
