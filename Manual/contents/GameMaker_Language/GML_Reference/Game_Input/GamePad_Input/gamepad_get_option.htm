gamepad_get_option

Diese Funktion kann verwendet werden, um den angegebenen Optionswert abzurufen. Sie geben den zu prüfenden Gamepad-"Steckplatz" sowie die Optionszeichenfolge an und die Funktion gibt den zugehörigen Wert zurück. Die verfügbare Optionszeichenfolge hängt von der Plattform ab, auf der auf der das Projekt ausgeführt wird, wie unten aufgeführt:

Option Plattform Beschreibung Rückgabe
"allow_rotation" tvOS Gibt zurück, ob die Siri Remote drehgesperrt ist (d. h. nur vertikal verwendet werden kann) oder nicht. Wenn die Rotationssperre aufgehoben ist, wird durch Drehen des Steuerelements die Eingabe geändert, so dass bei einer Drehung um 90º nach links die "rechte" Seite des Touchpads nach oben zeigt usw. usw., so dass alle Eingaben relativ zur Ausrichtung des Geräts erfolgen. Boolesche
"dpad_absolute" tvOS Gibt zurück, ob das Touchpad der Siri Remote eine absolute oder eine relative Eingabe akzeptiert. Im absoluten Modus wird die Mitte des Touchpads als (0,0) Position betrachtet - d. h. dort, wo der Analogstick in Ruhe ist oder wo die Mitte des D-Pads sein würde - und die Bewegung um diese Position erzeugt eine Eingabe. Bei der Einstellung "Relativ" wird die Position (0,0) als der Ort betrachtet, an dem die erste Berührung des Pads erfolgt, und alle Eingaben werden von dieser Position aus generiert, nicht von der Mitte des Pads. des Pads. Boolesche

Syntax:

gamepad_get_option(gerät, option_name);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
option_name Der Name der Option, von der der Wert geholt werden soll (eine Zeichenkette, siehe Tabelle oben)

Rückgabe:

Int, Bool, Real, String, N/A (abhängig von der Plattform)

Beispiel:

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

Der obige Code durchläuft alle Gamepad-Slots und sucht nach der "Siri Remote" auf der tvOS-Plattform. Wenn eine erkannt wird, wird die Fernbedienung so eingestellt, dass eine Eingabedrehung möglich ist.
