gamepad_set_option

Mit dieser Funktion können Sie jede der verfügbaren Gamepad-Optionen einstellen. Sie geben den Gamepad-"Steckplatz" an, auf dem die Option eingestellt werden soll, sowie die einzustellende Optionszeichenfolge und den zu verwendenden Wert. Die verfügbare Optionszeichenfolge hängt von der Plattform ab, auf der auf der das Projekt ausgeführt wird, wie unten aufgeführt:

Option Plattform Beschreibung Wert
"allow_rotation" tvOS Legen Sie fest, ob die Siri Remote drehgesperrt sein soll (d. h. nur vertikal verwendet werden kann) oder nicht. Wenn die Rotationssperre aufgehoben ist (Wert true), wird durch Drehen des Steuerelements die Eingabe geändert, d. h. eine Drehung um 90º nach links führt dazu, dass die "rechte" Seite des Touchpads nach oben zeigt usw. Seite des Touchpads "nach oben" zurückkehrt, usw..., so dass alle Eingaben relativ zur Ausrichtung des Geräts erfolgen. Wenn gesperrt (Wert false), wird die Eingabe nicht relativ zur Position des Controllers geändert und wird immer so behandelt, als befände sich der Controller in einer vertikalen Position. Boolesche
"dpad_absolute" tvOS Legen Sie fest, ob das Touchpad der Siri Remote eine absolute oder eine relative Eingabe akzeptiert. Im absoluten Modus (Wert true) wird die Mitte des Touchpads als (0,0) Position betrachtet - d. h. dort, wo sich der Analogstick in Ruhe befindet oder die Mitte des Dpads wäre. des Dpads sein würde - und eine Bewegung um diese Position erzeugt eine Eingabe. Bei der Einstellung "relativ" (Wert " false" ) wird die Position (0,0) als der Ort betrachtet, an dem die erste Berührung des Pads erfolgt, und alle Eingaben werden von dieser Position aus generiert, nicht von der Mitte des Pads. Boolesche

Syntax:

gamepad_set_option(gerät, option_name, wert);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" eingestellt werden soll.
option_name Der Name der zu setzenden Option (eine Zeichenkette, siehe Tabelle oben)
Wert Der Wert, auf den die Option gesetzt werden soll (kann boolesch, real, integer oder string sein - siehe Tabelle oben)

Rückgabe:

N/A

Beispiel:

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

Der obige Code durchläuft alle Gamepad-Slots und sucht nach der "Siri Remote" auf der tvOS-Plattform. Wenn eine erkannt wird, wird die Fernbedienung so eingestellt, dass eine Eingabedrehung möglich ist.
