gamepad_button_check_pressed

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Gamepad-Taste als gedrückt erkannt wurde oder nicht. Beachten Sie, dass diese Funktion nur einmal für die Taste ausgelöst wird, wenn sie zum ersten Mal gedrückt wird. Für Damit sie erneut ausgelöst wird, muss die Taste erst losgelassen und dann erneut gedrückt werden. Wenn Sie eine kontinuierliche Betätigung der Taste prüfen müssen, sollten Sie die Funktion gamepad_button_check().

Wenn Sie eine analoge Taste prüfen, berücksichtigt die Prüfung die Schwellenwerteinstellung und gibt nur dann true zurück, wenn der rohe Eingangswert den angegebenen Schwellenwert überschreitet (Sie können diesen Rohwert mit der Funktion gamepad_button_value() erhalten).

Syntax:

gamepad_button_check_pressed(device, button);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
Taste Auf welche Gamepad-Tastenkonstante ist zu prüfen.

Rückgabe:

Boolesche

Beispiel:

if gamepad_button_check_pressed(0, gp_start)
{
audio_play_sound(snd_Start, 0, false);
room_goto(rm_Level_1);
}

Der obige Code erkennt, ob die "Start"-Taste des an Gerät "Slot" 0 angeschlossenen Gamepads gedrückt wurde oder nicht und wechselt den Raum, wenn dies der Fall ist.
