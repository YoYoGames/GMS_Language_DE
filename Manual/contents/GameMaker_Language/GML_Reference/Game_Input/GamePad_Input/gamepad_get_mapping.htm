gamepad_get_mapping

Mit dieser Funktion können Sie die Zuordnungszeichenfolge für das Gamepad abrufen. Diese Zeichenkette wird entweder sein:

  • Der aktuelle Map-String für das Gamepad, zum Beispiel:

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • Ein String mit "no mapping", wenn kein Mapping gesetzt ist (dies ist nicht dasselbe wie wenn kein Mapping vorhanden ist)
  • Ein String mit "device index out of range", wenn der Slot-Index kein gültiger Gamepad-Index ist
  • Eine leere Zeichenkette "", wenn für das angeschlossene Gamepad kein Mapping vorhanden ist

Weitere Informationen über die Formatierung des zurückgegebenen Map-Strings finden Sie in der Funktion gamepad_test_mapping().

Syntax:

gamepad_get_mapping(index);

Argument Beschreibung
index Aus welchem Gamepad-Index-"Slot" die Zuordnung geholt werden soll.

Rückgabe:

Zeichenfolge

Beispiel:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

Der obige Code ruft den Map-String für den angegebenen Gamepad-Steckplatz ab und gibt ihn dann zur Fehlersuche auf der Konsole aus.
