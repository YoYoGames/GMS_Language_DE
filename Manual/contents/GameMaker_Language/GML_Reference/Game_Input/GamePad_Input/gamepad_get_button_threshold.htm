gamepad_get_button_threshold

Mit dieser Funktion kann die aktuelle Schwellenwerteinstellung der analogen Tasten für ein bestimmtes Gerät ermittelt werden. Der Standardschwellenwert für alle analogen Tasten ist 0,5, wobei der Bereich von 0 bis 1 reicht. Der Schwellenwert definiert, ab wann die Taste als "gedrückt" gilt. als "gedrückt" angesehen wird, wenn sie als digitale Taste fungieren soll.

Syntax:

gamepad_get_button_threshold(device);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.

Rückgabe:

Real

Beispiel:

if gamepad_get_button_threshold(0) != 0.5
{
gamepad_set_button_threshold(0, 0.5);
}

Der obige Code prüft den Schwellenwert der analogen Taste des Gamepads, das an das Gerät "Steckplatz" 0 angeschlossen ist, und wenn er nicht der Standardwert von 0,5 ist, wird er auf diesen Wert gesetzt.
