gamepad_ist_verbunden

Diese Funktion gibt zurück, ob ein Gamepad mit dem angegebenen "Steckplatz" verbunden ist (gibt true zurück) oder nicht (gibt false zurück). Normalerweise würden Sie diese Funktion in Verbindung mit der Funktion gamepad_get_device_count() Funktion verwenden, um die korrekte Anzahl der verfügbaren Gamepads und/oder Gamepad-"Steckplätze" zu ermitteln. Beachten Sie, dass es zu einer leichten Verzögerung kommen kann, wenn der Benutzer das Gamepad anschließt und GameMaker Studio 2 es als angeschlossen erkennt (dies ist insbesondere bei Fall, wenn es sich um über Bluetooth verbundene Controller handelt).

HINWEIS: Auf einigen Plattformen - insbesondere Konsolen - kann diese Funktion unmittelbar nach dem Anschließen/Auswählen eines Gamepads einen falschen Wert zurückgeben und muss möglicherweise einige Frames (Schritte) später in einem Alarm überprüft werden, damit sie das Gamepad korrekt erkennt Gamepad.

Syntax:

gamepad_is_connected(numb);

Argument Beschreibung
Gerät Welcher Gamepad-"Steckplatz" zu prüfen ist.

Rückgabe:

Boolesche

Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
}

Der obige Code durchläuft die verfügbaren Gamepads (oder Gamepad-Steckplätze) und prüft dann jeden einzelnen auf ein angeschlossenes Gamepad. Der zurückgegebene Wert wird dann verwendet, um ein globales Array auf true oder false für die Verwendung in zukünftigen Prüfungen zu setzen.
