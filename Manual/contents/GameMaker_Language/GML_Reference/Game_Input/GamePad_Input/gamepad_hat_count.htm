gamepad_hat_count

Diese Funktion gibt die Gesamtzahl der Hüte zurück, die für das Gamepad verfügbar sind, das an den angegebenen Geräte-"Slot" angeschlossen ist. Hüte beziehen sich im Allgemeinen auf die Auf-/Ab-/Links-/Rechts-Tasten, und beachten Sie, dass auf dem Windows-Zielgerät Hüte nur auf DirectInput Controllern verfügbar sind (also ab Steckplatz 4 aufwärts).

Syntax:

gamepad_hat_count(device);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.

Rückgabe:

Real

Beispiel:

h_num = gamepad_hat_count(4);

Der obige Code gibt die Anzahl der Hüte zurück, die auf dem in Gerät "Steckplatz" 4 eingesteckten Gamepad verfügbar sind, und speichert den Wert in der Variablen "h_num".
