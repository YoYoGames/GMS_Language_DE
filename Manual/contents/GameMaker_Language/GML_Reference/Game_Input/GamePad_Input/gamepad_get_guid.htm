gamepad_get_guid

Mit dieser Funktion können Sie die GUID für das Gamepad abrufen, das mit dem angegebenen Steckplatzindex verbunden ist. Sie geben den zu prüfenden Gamepad-Steckplatz an (von 0 - 11), und die Funktion gibt eine der folgenden Zeichenketten zurück:

  • " keine", wenn keine GUID vorhanden ist oder kein Gamepad angeschlossen ist
  • "Geräteindex außerhalb des Bereichs", wenn der Gamepad-Slot-Index außerhalb des Bereichs liegt
  • Die GUID-Zeichenfolge, etwas wie "050000005e040000fd020000ffff3f00", wenn dem Steckplatz ein Gamepad zugewiesen ist und die GUID dafür abgerufen werden kann

Diese Funktion wird normalerweise in Verbindung mit gamepad_get_description() verwendet, um einen Teil des SDL-Strings zu generieren, der für die Neuzuordnung von Controllern unter Windows, Android und macOS erforderlich ist.

Syntax:

gamepad_get_guid(index);

Argument Beschreibung
index Welcher Gamepad-"Slot"-Index zu prüfen ist

Rückgabe:

Zeichenfolge

Beispiel:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Der obige Code ruft die GUID- und Namensbeschreibungszeichenfolgen ab, verkettet sie und speichert die endgültige Zeichenfolge in einer globalen Variablen für spätere Verwendung.
