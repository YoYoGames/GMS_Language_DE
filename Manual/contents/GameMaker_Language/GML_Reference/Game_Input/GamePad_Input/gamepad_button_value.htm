gamepad_taste_wert

Mit dieser Funktion können Sie den aktuellen Wert einer analogen Taste abfragen, von 0 bis 1, wobei 0 kein Druck und 1 voller Druck ist. Sie geben den zu prüfenden Gamepad-Slot-Index an, zusammen mit entweder einer Tastenkonstante (wie hier aufgeführt), oder einen Integer-Wert zwischen 0 und gamepad_button_count() -1.

Beachten Sie, dass dies den Rohwert für die Schaltfläche zurückgibt und die Einstellung für den Schwellenwert nicht berücksichtigt (weitere Informationen zu Schwellenwerten finden Sie hier ).

Syntax:

gamepad_button_value(Gerät, Taste);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
Taste Auf welche Gamepad-Tastenkonstante ist zu achten.

Rückgabe:

Real

Beispiel:

Geschwindigkeit = gamepad_button_value(0, gp_shoulderrb) * 4;

Der obige Code verwendet den analogen Triggerwert vom Gamepad, das in Gerät "Slot" 0 eingesteckt ist, um die Geschwindigkeit der Instanz einzustellen.
