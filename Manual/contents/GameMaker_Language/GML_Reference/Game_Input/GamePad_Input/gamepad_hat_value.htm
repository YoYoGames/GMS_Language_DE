gamepad_hat_value

Mit dieser Funktion können Sie den aktuellen Wert eines Gamepad-"Hutes" ermitteln. Jeder Hut-Wert ist eine Bit-Maske für die verschiedenen Richtungen, wo:

  • Auf = 1
  • Rechts = 2
  • Abwärts = 4
  • Links = 8

Beachten Sie, dass diese kombiniert werden können (z. B. würde die Angabe eines Hut-Index von 3 eine Prüfung nach oben und nach rechts bedeuten), aber es können nur maximal 2 Bits auf einmal gedrückt werden. Die Funktion gibt einen reellen Wert zwischen 0 und 1 zurück, wobei 0 für nicht gedrückt und 1 für vollständig gedrückt steht (und es kann Werte dazwischen geben, je nachdem, ob das Gamepad die analoge Eingabe für Hüte unterstützt).

Syntax:

gamepad_hat_value(Gerät, hatindex);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
hatindex Welche Gamepad-Hut (oder Hüte) zu prüfen ist.

Rückgabe:

Real

Beispiel:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
case 1: y -= 3; break;
case 2: x += 3; break;
case 3: y -= 3; x += 3; break;
case 4: x -= 3; break;
case 6: y += 3; x += 3; break;
case 8: y += 3; break;
case 9: y -= 3; x -= 3; break;
case 12: y += 3; x -= 3; break;
}

Der obige Code speichert den Zustand des Hutes "0" in einer lokalen Variablen, prüft dann, was der Rückgabewert ist und handelt entsprechend.
