gamepad_get_axis_deadzone

Mit dieser Funktion können Sie den "Totzonen"-Wert der Joystick-Achse ermitteln. Die Funktion gibt einen Wert zwischen 0 und 1 zurück, wobei der Wert den Schwellenwert angibt, unter dem die Joystick-Achse als auf 0 steht.

Syntax:

gamepad_get_axis_deadzone(device);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.

Rückgabe:

Real

Beispiel:

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
gamepad_set_axis_deadzone(global.PadId, 0.5);
}

Der obige Code prüft den Schwellenwert der analogen Achse des an das Gerät "slot" angeschlossenen Gamepads, der in der globalen Variablen gespeichert ist, und wenn er nicht der Standardwert von 0,5 ist, wird er auf diesen Wert gesetzt.
