gamepad_axis_value

Mit dieser Funktion können Sie den Wert der verschiedenen Achsen eines bestimmten Gamepad-Slots ermitteln. Sie geben den zu prüfenden Gamepad-Slot-Index an, zusammen mit einer Achsen-Index-Konstante (wie hier aufgelistet) oder einem Integer-Wert zwischen 0 und gamepad_axis_count()-1. Der zurückgegebene Wert liegt zwischen -1 und 1 für jede der verfügbaren horizontalen und vertikalen Achsen.

Beachten Sie, dass der zurückgegebene Wert normalisiert wird, um den für die Achsen-Totzone eingestellten Wert zu berücksichtigen (mit der Funktion gamepad_set_axis_deadzone()), und daher immer zwischen -1 und 0 bzw. 0 und 1. Wenn Sie z. B. die Totzone auf 0,2 setzen, bedeutet dies, dass das Drücken des Knüppels nach rechts erst dann einen Wert von 0 - 1 zurückgibt, wenn der Rohwert der Achse über 0,2 liegt - wenn also z. B. der Rohwert 0,2 beträgt, ist der Wenn der Rohwert 0,2 ist, ist der Rückgabewert 0, wenn der Rohwert 0,5 ist, ist der Rückgabewert 0,375, oder wenn der Rohwert 0,9 ist, wäre der Rückgabewert 0,875.

Syntax:

gamepad_axis_value(Gerät, axisIndex);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
axisIndex Der zu prüfende Achsenindex (siehe Konstantenliste).

Rückgabe:

real

Beispiel:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Der obige Code setzt die Geschwindigkeit und Richtung der Instanz basierend auf den zurückgegebenen Werten des linken analogen Controllers des Gamepads, das in Gerät "Slot" 0 eingesteckt ist.
