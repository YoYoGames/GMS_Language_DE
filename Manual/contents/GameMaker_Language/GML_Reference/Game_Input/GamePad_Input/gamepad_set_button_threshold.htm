gamepad_set_button_threshold

Mit dieser Funktion können Sie die aktuelle Schwellenwerteinstellung der analogen Tasten für ein bestimmtes Gerät einstellen. Der Standardschwellenwert für alle analogen Tasten ist 0,5, wobei der Bereich von 0 bis 1 reicht. Der Schwellenwert legt fest, ab wann die Taste als "gedrückt" gilt. als "gedrückt" angesehen wird, wenn sie als digitale Taste fungieren soll. Beachten Sie, dass diese Funktion die Zustände "geprüft", "gedrückt" und " losgelassen" für analoge Tasten beeinflusst, aber nicht den Wert, der von der Funktion gamepad_button_check()zurückgegeben wird, die immer den Rohwert für die Taste zurückgibt.

Syntax:

gamepad_set_button_threshold(device, threshold);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
Schwellenwert Der neue Schwellenwert (von 0 - 1, Voreinstellung 0,5).

Rückgabe:

N/A

Beispiel:

if gamepad_get_button_threshold(0) != 0.5
{
gamepad_set_button_threshold(0, 0.5);
}

Der obige Code prüft den Schwellenwert der analogen Taste des Gamepads, das an das Gerät "Steckplatz" 0 angeschlossen ist, und wenn er nicht der Standardwert von 0,5 ist, wird er auf diesen Wert gesetzt.
