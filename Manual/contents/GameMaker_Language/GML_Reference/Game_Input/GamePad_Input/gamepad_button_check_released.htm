gamepad_button_check_released

Diese Funktion gibt true oder false zurück, je nachdem, ob die angegebene Gamepad-Taste als losgelassen erkannt wird oder nicht. Beachten Sie, dass diese Funktion nur einmal für die Taste ausgelöst wird, sobald sie losgelassen wurde. Damit sie erneut ausgelöst wird, muss die Taste erst gedrückt und dann wieder losgelassen werden. Wenn Sie eine analoge Taste prüfen, dann berücksichtigt die Prüfung die Schwellenwerteinstellung und gibt nur true zurück, wenn der rohe Eingangswert unter den angegebenen Schwellenwert fällt (Sie können diesen rohen Wert mit der Funktion gamepad_button_value() ermitteln).

Syntax:

gamepad_button_check_released(device, button);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.
Taste Auf welche Gamepad-Tastenkonstante ist zu achten.

Rückgabe:

Boolesche

Beispiel:

if gamepad_button_check_released(0, gp_select)
{
audio_play_sound(snd_Button, 0, false);
global.Pause = !global.Pause;
}

Der obige Code erkennt, ob die "Select"-Taste des an Gerät "Slot" 0 angeschlossenen Gamepads gedrückt wurde oder nicht und schaltet die globale Variable "Pause" um.
