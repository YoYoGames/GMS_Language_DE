gamepad_set_colour

Mit dieser Funktion können Sie die Farbe der LEDs innerhalb eines PlayStation-Controllers einstellen. Sie geben den einzustellenden Gerätesteckplatz an und geben dann eine Farbe an, die eine der Farbkonstanten oder ein mit den spezifischen Farbfunktionen erstellter Farbwert oder ein HEX-Wert (wie $FFFFFFF) sein kann.

Syntax:

gamepad_set_colour(Gerät, Farbe);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" eingestellt werden soll.
Farbe Die zu verwendende Farbe.

Rückgabe:

N/A

Beispiel:

if health < 10
{
gamepad_set_colour(0, c_red);
}

Der obige Code setzt die LEDs des PlayStation-Controllers auf Rot, wenn die Health-Variable unter 10 fällt.
