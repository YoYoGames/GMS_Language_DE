gamepad_remove_mapping

Diese Funktion kann verwendet werden, um die aktuelle Gerätezuordnung vom angegebenen Gamepad-Slot-Index zu entfernen. Nach dem Aufruf muss der Slot-Index mit der Funktion gamepad_test_mapping() Funktion neu zugeordnet werden, wenn Sie in der Lage sein, die Gamepad-Konstanten zur Erkennung von Eingaben korrekt zu verwenden (direkte Eingaben können immer mit den Funktionen gamepad_axis/button/hat_count() und gamepad_axis/button/hat_value() Funktionen zusammen verwendet werden).

Syntax:

gamepad_remove_mapping(index);

Argument Beschreibung
index Aus welchem Gamepad-Index-"Steckplatz" die Zuordnung entfernt werden soll.

Rückgabe:

N/A

Beispiel:

if remap == true
{
gamepad_remove_mapping(global.PadIndex);
}

Der obige Code entfernt das Mapping aus dem angegebenen Gamepad-Index-Slot basierend auf dem Wert einer Variablen.
