gamepad_axis_count

Diese Funktion gibt die Anzahl der "Achsen"-Bedienelemente auf dem überprüften Gerät zurück. Diese Bedienelemente sind die analogen Richtungs-"Daumensticks" auf den meisten Controllern.

Syntax:

gamepad_axis_count(device);

Argument Beschreibung
Gerät Welcher Gamepad-Geräte-"Steckplatz" zu prüfen ist.

Rückgabe:

Real

Beispiel:

axis = gamepad_axis_count(0))

Der obige Code speichert die Anzahl der verfügbaren Achsen für das an Gerät "Slot" 0 angeschlossene Gamepad in der Variablen "axis".
