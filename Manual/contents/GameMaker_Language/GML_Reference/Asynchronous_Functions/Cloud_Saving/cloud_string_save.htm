cloud_string_save

Diese Funktion überträgt eine Zeichenfolge an den gewählten Cloud-Dienst zur Speicherung. Die Funktion gibt einen eindeutigen ID-Wert zurück, der dann in dem entsprechenden asynchronen Ereignis verwendet werden sollte, um die DS-Map zu identifizieren, die als "Rückruf" zurückgegeben wird vom Cloud-Dienst zurückgegeben wird. Die Zeichenkette sollte alle Informationen enthalten, die Sie für Ihr Spiel speichern müssen, da Sie nur einen einzigen "Daten-Blob" in der Cloud speichern können, und ein erneutes Ausführen dieser Funktion überschreibt alle zuvor gespeicherten Werte (ebenso wie die Verwendung der cloud_file_save() Funktion). Die Beschreibung sollte eine kurze Zeichenkette sein, die das Speichern beschreibt, z. B.: "Level2, Stage2".

Weitere Informationen zu den zurückgegebenen asynchronen Daten finden Sie in der Funktion cloud_synchronise().

Syntax:

cloud_string_save(string, description);

Argument Beschreibung
String Der hochzuladende Datenstring.
Beschreibung Eine kurze Beschreibung der zu speichernden Daten.

Rückgabe:

Real

Beispiel:

var t_str = "";
for (var i = 0; i < 10; i++;)
{
t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Aktuelle Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

Der obige Code erstellt eine Zeichenkette aus den im globalen Array "Highscores" gespeicherten Werten und schreibt diese Zeichenkette dann zur Speicherung in den Cloud-Dienst sowie in eine Datei zur lokalen Speicherung.
