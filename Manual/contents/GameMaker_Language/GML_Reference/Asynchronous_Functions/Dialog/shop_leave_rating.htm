shop_leave_rating

Diese Funktion öffnet einen OS-abhängigen Dialog, in dem Sie den Benutzer auffordern können, eine Bewertung oder einen Kommentar zu einer bestimmten Seite abzugeben. Sie können den Text definieren, der in dem Dialog erscheinen soll, sowie den Text, der auf den beiden Schaltflächen erscheinen soll, und die URL wo der Kommentar gepostet werden soll.

HINWEIS: Diese Funktion ist nur für Android- und iOS-Ziele verfügbar.

WICHTIG! Auf iOSwenn Sie TestFlight verwenden, wird in der Entwicklung die Aufforderung angezeigt, aber Sie können nichts tun. Sobald Ihre App im App Store ist, wird diese Schaltfläche aktiv und der Benutzer hat die Möglichkeit, eine eine Rezension zu schreiben oder eine Bewertung zu hinterlassen. Beachten Sie, dass, wenn Sie das Spiel über TestFlight vertreiben, dieses Fenster möglicherweise gar nicht angezeigt wird.

Syntax:

shop_leave_rating(text, yes_string, no_string, url)

Argument Beschreibung
Text Text, der im Dialogfeld angezeigt wird.
yes_string Text, der auf der "Ja"-Schaltfläche erscheint.
keine_Zeichenfolge Text, der auf der Schaltfläche "Nein" erscheint.
url Die vollständige URL, an die der Kommentar gesendet werden soll.

Rückgabe:

N/A

Beispiel:

if timer <= 0
{
shop_leave_rating("Danke, dass Sie das Spiel gespielt haben! Hinterlassen Sie doch einen Kommentar!", "Okay!", "Heute nicht!", "http://MacSweeney/comments");
timer = 10000000;
}
sonst
{
timer -= 1;
}

Der obige Code fordert den Benutzer auf, einen Kommentar zu hinterlassen, wenn die Variable "timer" auf 0 heruntergezählt hat.
