Leistung_Anzeigen_Leuchttafeln

Mit dieser Funktion wird die Ranglistenseite für die gewählte Plattform geöffnet. Bitte beachten Sie, dass es sich hierbei um eine asynchrone Funktion handelt, d. h. Ihr Spiel läuft im Hintergrund weiter, während die Seite mit den Bestenlisten angezeigt wird. Daher sollten Sie Sie sollten daher vorsichtig sein, wo Sie diese Funktion verwenden, und sicherstellen, dass Sie das Spiel pausieren oder die Funktion nur in Bereichen Ihres Spiels anzeigen lassen, in denen sie den Spielablauf nicht stört.

Syntax:

achievement_show_leaderboards()

Rückgabe:

N/A

Beispiel:

if achievement_available()
{
global.Pause = true;
instance_create_layer(0, 0, "Controllers", obj_Pause);
achievement_show_leaderboards();
}

Der obige Code prüft, ob der gewählte Leaderboard- und Achievement-Dienst verfügbar ist, und wenn ja, setzt er eine globale Variable und erstellt eine Controller-Instanz "pause", bevor er die Funktion zum Anzeigen der Leaderboards auf dem Bildschirm aufruft.
