Errungenschaften und Bestenlisten

Spiele, die für Geräte mit iOS- oder Android-Betriebssystemen entwickelt wurden, können auch mit einer von zwei APIs für Bestenlisten und Erfolge verbunden werden. Für iOS gibt es iTunes Connect und für Android können Sie Google Play verwenden, wobei beide Beide nutzen die Funktionen, die Sie auf dieser Seite finden. Diese Funktionen ermöglichen es Ihrem Spiel, Informationen über Spielstände und Erfolge an die Bestenlisten zu senden, die Sie im Entwicklerkonto für das von Ihnen erstellte Spiel eingerichtet haben, und sind eine hervorragende Möglichkeit, um Dies ist eine hervorragende Möglichkeit, die Spieler in die soziale Seite des Spiels einzubinden, da sie dann sehen können, was andere Personen, die Ihr Spiel gespielt haben, erreicht haben, und versuchen können, sie zu übertreffen.

Wenn Sie diese Funktionen verwenden, sollten Sie beachten, dass sie Achievements und Spielstände nicht automatisch für Sie verfolgen. Sie zeigen nur die Daten an, die Sie ihnen von Ihrem Spiel schicken, also müssen Sie Ihr eigenes Achievement- und Hi-Score-System programmieren, um den Status der Personen im Spiel zu verfolgen, und dann diese Funktionen verwenden, um die Daten zum gewählten Netzwerk hinzuzufügen (Sie können jedoch Fortschrittsinformationen für Erfolge mit der achievement_load_progress() Funktion).

Einige dieser Funktionen lösen einen Callback Social Asynchronous Event aus, der eine DS-Map mit Daten enthält, die sich auf zu der Funktion, die es ausgelöst hat. Die Funktionen, die das Ereignis auslösen, geben auch die Schlüssel/Wertpaare der Map an, die vom Callback zurückgegeben werden, aber ein Callback Social Event kann auch ausgelöst werden, wenn Ihr Spiel externe Benachrichtigungen über Herausforderungen usw. erhält.

HINWEIS: Damit diese Funktionen mit iOS funktionieren, müssen Sie über ein gültiges Game Center-Konto und eine iOS-App-ID für Ihr Spiel verfügen.

HINWEIS: Damit diese Funktionen mit Android funktionieren, müssen Sie Ihr Spiel bei Google Play einreichen und die erforderlichen Abschnitte eingerichtet haben sowie die API in den Android-Spieloptionen für Ihr Spiel aktiviert haben.

Die unten aufgeführten Funktionen sind die grundlegenden, die für die Integration Ihres Spiels mit der gewählten Zielplattform für soziale Spiele verwendet werden:

  • leistung_post_score
  • Leistung_Zunahme
  • leistung_zeigen_erfolge
  • Leistung_Anzeigen_Leuchttafeln
  • achievement_show_challenge_notifications
  • leistung_holen_herausforderungen

  • leistung_laden_freunde
  • leistung_laden_leittafel
  • leistung_laden_fortschritt
