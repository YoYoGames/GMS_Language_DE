Leistung_Zunahme

Spiele können Achievements haben, die keinen Abschlusswert haben, und so können Sie diese Funktion verwenden, um diese Achievements um einen bestimmten Betrag zu erhöhen. Unter Android senden Sie die ID der Errungenschaft als String (dies ist die eindeutige ID der Errungenschaft, die beim Einrichten der Errungenschaft zugewiesen wurde). auf allen anderen Plattformen geben Sie den Namen der definierten Leistung als String an und geben dann den tatsächlichen Wert der Leistung an, um den erhöht werden soll.

Die Funktion löst ein soziales asynchrones Ereignis aus, wobei die zurückgegebene async_load DS-Map die folgenden Schlüssel/Wertpaare enthält:

  • " type"- Dies ist der Typ des ausgelösten Ereignisses, derfür diese Funktion die Zeichenfolge " achievement_increment_result" sein wird.
  • " id" - Dies ist die ID des Ereignisses, und für diese Funktion wird eine GML-Konstante GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT zurückgegeben .
  • "achievement_id" - Dies ist die eindeutige ID der Leistung, die inkrementiert wurde.
  • "Status" - Dies ist 1, wenn die Leistung erfolgreich erhöht wurde, oder 0, wenn die Anfrage fehlgeschlagen ist
  • "complete" - Dies ist 1, wenn die Leistung bis zu diesem Inkrement abgeschlossen wurde, oder 0, wenn die Anfrage fehlgeschlagen ist

Syntax:

leistung_erhöhen(name, wert)

Argument Beschreibung
Name Der Name/die ID der Leistung.
Wert Der Wert, der als Inkrement gesendet werden soll.

Rückgabe:

N/A

Beispiel:

achievement_increment(global.Achievement[0], score);

Der obige Code fügt den aktuellen Punktestand des Spielers zum Achievement mit der im globalen Array gespeicherten ID hinzu.
