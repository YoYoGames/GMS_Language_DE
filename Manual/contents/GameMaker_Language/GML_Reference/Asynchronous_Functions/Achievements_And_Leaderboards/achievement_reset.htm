Leistung_zurücksetzen

Mit dieser Funktion werden alle Achievements auf ihre Anfangswerte für das Spiel zurückgesetzt. Diese Funktion wird als Debug-Funktion bereitgestellt und es wird nicht empfohlen, dass Sie dem Endbenutzer erlauben, dies in Ihren Spielen zu tun.

Syntax:

leistung_zuruecksetzen()

Rückgabe:

N/A

Beispiel:

if debug_mode
{
if keyboard_check_pressed(vk_space)
{
achievement_reset()
}
}

Der obige Code prüft, ob sich das Spiel im Debug-Modus befindet, und wenn dies der Fall ist, werden durch Drücken der Leertaste alle Erfolge zurückgesetzt.
