leistung_anmeldung_status

Diese Funktion ist derzeit nur für Google Play auf der Android-Plattform nützlich. Sie prüft, ob der Benutzer bereits bei Google-Diensten angemeldet ist, in diesem Fall gibt sie true zurück, oder nicht, dann gibt sie false zurück. Wenn sie true zurückgibt, ist es nicht notwendig, die achievement_login() Funktion aufzurufen.

Syntax:

achievement_login_status();

Rückgabe:

Boolesche

Beispiel:

if !achievement_login_status()
{
achievement_login();
}

Der obige Code meldet Sie bei dem gewählten Ranglisten- und Achievement-Dienst an.
