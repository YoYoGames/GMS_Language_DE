leistung_post

Mit dieser Funktion können Sie Ihre Erfolge an den gewählten Leaderboard- und Achievement-Dienst senden. Auf Android benötigen Sie die ID der Errungenschaft als String (dies ist die eindeutige ID der Errungenschaft, die beim Einrichten der Errungenschaft zugewiesen wurde), während Sie auf allen anderen Plattformen den Namen des Achievements als String liefern und dann den Prozentsatz angeben, den Sie auf dem Weg zum Erhalt des Achievements erreicht haben (0 - Keine, 100 - Abgeschlossen). Die Funktion löst ein soziales asynchrones Ereignis aus, bei dem die zurückgegebene async_load DS-Map die folgenden Schlüssel/Wertpaare enthält:

  • " type"- Dies ist der Typ des Ereignisses, das ausgelöst wurde, wasfür diese Funktion die Zeichenfolge " achievement_post_result" sein wird.
  • " id" - Dies ist die ID des Ereignisses, und für diese Funktion wird eine GML-Konstante GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT zurückgegeben .
  • "achievement_id" - Dies ist die eindeutige ID der Leistung, die aktualisiert wurde.
  • "Status" - Dieser Wert ist 1, wenn die Leistung erfolgreich aktualisiert wurde, oder 0, wenn die Anfrage fehlgeschlagen ist

Syntax:

achievement_post(Leistung, Prozent)

Argument Beschreibung
Leistung Der Name/die ID der Leistung.
Prozent Der Prozentsatz der abgeschlossenen Leistung.

Rückgabe:

N/A

Beispiel:

if achievement_available()
{
achievement_post(global.Achievement[0], 100);
}

Der obige Code prüft, ob der gewählte Ranglisten- und Achievement-Dienst verfügbar ist, und wenn ja, dann wird ein Achievement gepostet.
