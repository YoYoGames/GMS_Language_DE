leistung_zeigen_erfolge

Mit dieser Funktion wird die Achievement-Seite für die gewählte Plattform geöffnet. Bitte beachten Sie, dass dies eine asynchrone Funktion ist, d. h. Ihr Spiel läuft im Hintergrund weiter, während die Achievements-Seite angezeigt wird. Daher sollten Sie vorsichtig sein Sie sollten daher vorsichtig sein, wo Sie diese Funktion verwenden, und sicherstellen, dass das Spiel angehalten wird oder dass sie nur in Bereichen Ihres Spiels angezeigt wird, in denen sie den Spielablauf nicht beeinträchtigt.

Syntax:

leistung_zeigen_erfolge()

Rückgabe:

N/A

Beispiel:

if achievement_available()
{
global.Pause = true;
instance_create_layer(0, 0, "Controllers", obj_Pause);
achievement_show_achievements();
}

Der obige Code prüft, ob der gewählte Leaderboard- und Achievement-Dienst verfügbar ist, und wenn ja, setzt er eine globale Variable und erstellt eine Controller-Instanz "pause", bevor er die Funktion zum Anzeigen der Achievements auf dem Bildschirm aufruft.
