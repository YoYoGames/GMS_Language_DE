achievement_show_challenge_notifications

Mit dieser Funktion können Sie die verschiedenen "Toast"-Popup-Benachrichtigungen in Bezug auf Herausforderungen anzeigen oder unterdrücken. Wenn die Argumente auf true gesetzt sind, erscheinen diese Meldungen und informieren den Spieler über alle lokalen oder entfernten Herausforderungen erhaltenen sowie abgeschlossenen Herausforderungen, während die Einstellung false diese Benachrichtigungen unterdrückt.

Syntax:

achievement_show_challenge_notifications(receive, local, remote)

Argument Beschreibung
empfangen. Popup-Fenster für empfangene Herausforderungen anzeigen(true) oder nicht(false).
lokal Popup-Fenster für abgeschlossene lokale Herausforderungen anzeigen(true) oder nicht(false).
remote Popup-Fenster für abgeschlossene Remote-Herausforderungen anzeigen(true) oder nicht(false).

Rückgabe:

N/A

Beispiel:

achievement_show_challenge_notifications(true, false, false);

Der obige Code legt fest, dass Toast-Benachrichtigungen nur für empfangene Herausforderungen angezeigt werden.
