werfen

Mit der throw-Laufzeitanweisung können Sie Ihre eigenen Laufzeitausnahmen erzeugen, die das Spiel beenden und eine Fehlermeldung anzeigen, und zwar mit der folgenden Syntax:

werfen (<Ausdruck>);

Der verwendete Ausdruck kann ein Wert oder eine Zeichenkette oder ein beliebiger anderer Datentyp sein, und dies erzeugt dann einen Ausnahmefehler, der - standardmäßig - auf dem Bildschirm angezeigt wird, und beim Schließen der Fehlermeldung wird das Spiel beendet. Zum Beispiel wird dies aufgerufen:

throw ("Hallo Welt!");

Mit dieser Laufzeitfunktion können Sie eine benutzerdefinierte Methode bereitstellen, die immer dann aufgerufen wird, wenn in Ihrem Spiel unbehandelte Exceptions auftreten.
