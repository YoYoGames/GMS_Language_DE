Pause

Die Anweisung break wird verwendet, um ein vorzeitiges Ende eines für, wiederholen, while oder do / until Schleife vorzeitig zu beenden, oder um eine switch Anweisung zu sagen, dass sie an dieser Stelle enden soll, oder um einen mit Aufruf zu beenden. Bitte lesen Sie die einzelnen Seiten für diese verschiedenen Funktionen, um eine eine genauere Erklärung, wie sie unter den jeweiligen Umständen verwendet werden kann. Beachten Sie, dass die Verwendung von break außerhalb eines dieser Kontexte zu einem Fehler führt.

Nachfolgend sehen Sie einige grundlegende Beispiele, wie break verwendet werden kann, und seine Syntax ist einfach:

Pause;

Pause in einer for-Schleife:

for (var i = 0; i < 10; i += 1)
{
if (array[i] == 234)
{
break;
}
}
num = i;

Pause in einer Wiederholungsschleife:

var i = 0;
var temp = 0;
repeat (10)
{
temp += array[i];
if (temp > max_total)
{
break;
}
sonst
{
i += 1;
}
}

Pause in einer while-Schleife:

var i = 0;
while (!place_free(x, y))
{
x = random(Raum_Breite);
y = random(Raum_Höhe);
if (i > 50)
{
break;
}
sonst
{
i+=1;
}
}

break in einer do / until Schleife:

var _id = noone;
do
{
_id = list[| 0];
if instance_exists(_id)
{
_break;
}
ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

Pause bei Verwendung mit:

var count = 0;
with (obj_Enemy)
{
count++;
if (count > 10)
{
break;
}
hp = 100;
}

break in einer switch:

switch (tastatur_taste)
{
case vk_left:
case ord("A"):
x -= 4;
break;
case vk_right:
case ord("D"):
x += 4;
break;
case vk_up:
case ord("W"):
y -= 4;
break;
case vk_down:
case ord("S"):
y += 4;
break;
}
