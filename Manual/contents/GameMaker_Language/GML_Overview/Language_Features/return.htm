Rückgabe

Die Return-Anweisung hat die folgende Syntax:

return (<Ausdruck>)

Die return-Anweisung wird nur in Skriptfunktionen und -methoden verwendet und dient dazu, einen Wert aus der Funktion zurückzugeben, der in weiterem Code oder Funktionsaufrufen verwendet werden kann Aufrufen verwendet wird. Es ist zu beachten, dass die Ausführung der Funktion an der Anweisung return endet, was bedeutet, dass jeglicher Code, der nach dem Aufruf von return kommt, nicht ausgeführt wird. Hier ist ein kurzes Beispielskript namens sqr_calc, das das Quadrat eines beliebigen Wertes berechnet, der ihm übergeben wird, und es enthält eine Fehlerabfangfunktion, falls der übergebene Parameter keine reelle Zahl ist:

/// @function sqr_calc(val);
/// @param {real} val Der Wert, von dem das Quadrat berechnet werden soll
/// @description Berechne das Quadrat des angegebenen Wertes

function sqr_calc(value)
{
if !is_real(wert)
{
return 0;
}
sonst
{
return (Wert * Wert);
}
}

Um eine Funktion aus einem Stück Code heraus aufzurufen, verwenden Sie sie einfach auf die gleiche Weise wie beim Aufruf von Laufzeitfunktionen - das heißt, Sie schreiben den Funktionsnamen mit den Parameterwerten in Klammern. Die obige Funktion würde also wie folgt aufgerufen werden:

if keyboard_check_pressed(vk_enter)
{
val = scr_sqr(Betrag);
}

Hier wird die Variable val entweder 0 sein (weil die Variable amount keine reelle Zahl ist) oder die Summe für value * value.
