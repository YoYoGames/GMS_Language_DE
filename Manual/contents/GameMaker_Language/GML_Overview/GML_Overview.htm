GameMaker-Sprachübersicht

Dieser Teil des Handbuchs enthält alle Informationen, die zum Verständnis und zur Benutzung von GameMaker Studio 2 Scripting-Sprache GML. Die Laufzeitfunktionen in dieser Sprache können zur Erstellung Ihrer Spiele verwendet werden und werden in Objekte aus dem Objekt-Editorhinzugefügt, normalerweise wenn Sie ein neues GameMaker-Sprachprojekt , obwohl sie auch zusammen mit DnD™.(Drag and Drop). Unten sehen Sie ein typisches Bild eines Objekts mit geöffnetem Code-Editor bei einem Ereignis:

Jedes Ereignis hat seine eigene Registerkarte im Editor, und Sie können jederzeit Code hinzufügen, bearbeiten oder entfernen (weitere Informationen zu Ereignissen finden Sie unter Objekt-Ereignisse). Der Code selbst muss eine Grundstruktur haben und kann Ressourcenindizes, Variablen, Funktionen, Ausdrücke, Schlüsselwörter usw. enthalten, die alle in den folgenden Abschnitten erklärt werden. Wenn Sie ein Anfänger in der Programmierung sind oder von DnD™umsteigen, wird empfohlen, mit der Seite über Grundlegende Code-Struktur zu beginnen und dann alle anderen Seiten in diesem Abschnitt durchzulesen und den Code von jeder Seite in GameMaker Studio 2 selbst zu testen.
