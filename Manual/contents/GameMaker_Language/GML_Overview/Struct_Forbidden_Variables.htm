Struktur Verbotene Variablen

Bei der Erstellung einer struct ist es möglich, bestimmte eingebaute Variablen als Namen von Mitgliedsvariablen zu verwenden, z. B:

mystruct = {
object_index : obj_Player,
speed : 3,
image_blend : c_red
}

Es können jedoch nur lokale und Instanzbereich-Einbauvariablen auf diese Weise verwendet und ihnen Werte zugewiesen werden, als ob sie reguläre Strukturvariablen wären. Die Verwendung von eingebauten Variablen mit globalem Geltungsbereich ist verboten und führt zu Problemen in Ihrem Spiel. Im Folgenden finden Sie eine vollständige Liste dieser Variablen, damit Sie wissen, welche Sie vermeiden sollten.
