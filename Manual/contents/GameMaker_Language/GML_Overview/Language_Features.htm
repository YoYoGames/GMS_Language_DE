Sprachfunktionen

Die GameMaker Language(GML) hat eine Reihe von Funktionen, die zur grundlegenden Struktur aller mit der Sprache geschriebenen Programme gehören. Jedes dieser Features wird in jedem Code, den Sie schreiben, mehrfach verwendet werden, daher sollten Sie wissen, was sie sind und wie sie funktionieren arbeiten, bevor Sie mit der Programmierung eines Projekts beginnen.

  • wenn / sonst und bedingte Operatoren
  • wiederholen.
  • während
  • tun / bis
  • für
  • schalten.
  • Pause
  • weiter
  • Ausgang
  • mit
  • Rückgabe
  • Anfang / Ende
  • try / catch / finally
  • werfen
  • neu
  • löschen.
