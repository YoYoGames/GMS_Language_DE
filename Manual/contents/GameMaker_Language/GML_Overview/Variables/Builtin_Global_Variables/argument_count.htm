argument_count

Diese schreibgeschützte Variable gibt die Anzahl der "Argumente" zurück, die an eine Skriptfunktion oder eine Methode durchgereicht werden. Normalerweise wird sie in Verbindung mit einem Argument-Array (argument[0...15]) verwendet, um unterschiedliche Eingabeargumente für eine bestimmte Funktion zu ermöglichen.

HINWEIS: Dies ist nicht mehr zwingend erforderlich und dient eher der Unterstützung von älteren Funktionen als alles andere. Alle benutzerdefinierten Funktionen akzeptieren variable Argumente, und jedes an eine Funktion übergebene Argument, das nicht Teil der benannten Inputs ist, wird standardmäßig auf undefiniert initialisiert, was mit der Funktion is_undefined() überprüft werden kann.

Syntax:

argument_count;

Rückgabe:

Integer

Beispiel:

function mein_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argument[i]
}
sonst
{
arg[i] = -1;
}
}
}

Der obige Code verwendet die Variable argument_count zusammen mit dem Array argument, um Variablen zu initialisieren, und ist ein Beispiel dafür, wie man einer benutzerdefinierten Funktion erlauben kann, eine variable Anzahl von Argumenten zu akzeptieren.
