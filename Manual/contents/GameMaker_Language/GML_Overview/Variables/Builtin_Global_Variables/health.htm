Gesundheit

Diese Variable hat einen globalen Umfang und wird verwendet, um einen numerischen Wert zu halten, der normalerweise für die Gesundheit des Spielers verwendet wird. Diese Variable ist nur zur Unterstützung von Legacy-Projekten aus früheren Versionen von GameMaker und sollte nicht in neuen Projekten verwendet werden verwendet werden, da sie in Zukunft möglicherweise veraltet sein wird.

Syntax:

Gesundheit;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel

if (health <= 0)
{
global.state = "Game Over";
instance_destroy();
}

Der obige Code prüft die Variable health und wenn sie kleiner oder gleich 0 ist, wird eine globale Variable gesetzt und die Instanz zerstört.
