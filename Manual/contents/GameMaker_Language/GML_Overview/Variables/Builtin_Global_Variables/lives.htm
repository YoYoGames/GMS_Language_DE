lebt

Diese Variable hat einen globalen Gültigkeitsbereich und wird verwendet, um einen numerischen Wert zu halten, der normalerweise für die Spielerleben verwendet wird. Diese Variable ist nur zur Unterstützung von Legacy-Projekten aus früheren Versionen von GameMaker und sollte nicht in neuen Projekten verwendet werden verwendet werden, da sie in Zukunft möglicherweise veraltet sein wird.

Syntax:

lebt;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel

if (Leben < 5)
{
Leben += 1;
}

Der obige Code prüft die Variable lives und wenn sie kleiner als 5 ist, wird 1 zu ihr addiert.
