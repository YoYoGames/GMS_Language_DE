Ergebnis

Diese Variable hat einen globalen Gültigkeitsbereich und wird verwendet, um einen numerischen Wert zu halten, der normalerweise für den Spielerstand verwendet wird. Diese Variable ist nur zur Unterstützung von Legacy-Projekten aus früheren Versionen von GameMaker und sollte darf nicht in neuen Projekten verwendet werden verwendet werden, da sie in Zukunft möglicherweise veraltet sein wird.

Syntax:

Ergebnis;

Rückgabe:

Real (Gleitkommawert mit einfacher Genauigkeit)

Beispiel

switch (object_index)
{
case obj_Enemy_Fighter:
score += 10;
break;
case obj_Enemy_Mage:
score += 25;
break;
case obj_Enemy_Boss:
score += 100;
break;
}
instance_destroy();

Der obige Code prüft den Objektindex der Instanz, die den Code ausführt, mithilfe eines Schalters und fügt dann unterschiedliche Beträge zur Variablen score hinzu, je nachdem, um welches Objekt es sich handelt.
