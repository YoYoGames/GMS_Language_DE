asynchron_laden

Diese Variable hat einen globalen Umfang und wird verwendet, um eine DS-Map zu halten, wenn sie in den Asynchronous Events verwendet wird, und -1 zu allen anderen Zeiten. Der tatsächliche Inhalt der DS-Map hängt vom Typ des asynchronen Ereignis-Callbacks sowie von der Funktion ab, die zum Auslösen des Ereignisses verwendet wurde, daher finden Sie auf den Daher finden Sie auf den einzelnen Seiten für diese Ereignisse ausführliche Informationen zu allen möglichen DS-Map-Inhalten.

Syntax:

async_load;

Rückgabe:

Real (ganzzahliger Wert)

Erweitertes Beispiel:

sprite = sprite_add("site.com/pfad/image.png", 16, true, true, 0, 0);

Der obige Code würde in einem Ereignis aufgerufen werden, um ein Sprite von einer externen URL zu laden. Dies würde dann das asynchrone Ereignis "Image Loaded " auslösen, in dem Sie die Karte async_load parsen würden:

if ds_map_find_value(async_load, "id") == sprite
{
if (ds_map_find_value(async_load, "status") >= 0)
{
sprite_index = sprite;
}
}

Der obige Code prüft zunächst die ID der async_load-Map und dann den Status des Callbacks. Wenn der Wert größer oder gleich 0 ist (was Erfolg signalisiert), wird das Ergebnis des Callbacks verwendet, um den Sprite-Index der Instanz auf das neu geladene Bild.
