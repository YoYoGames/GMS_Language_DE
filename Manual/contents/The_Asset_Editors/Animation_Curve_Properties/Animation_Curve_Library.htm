Die Animationskurven-Bibliothek

Im Fenster Animationskurvenbibliothek können Sie den Kurventyp für die Animationskurve ändern und eine Voreinstellung auf den ausgewählten Kanal anwenden. eine Voreinstellung auf den ausgewählten Kanal anwenden. Es kann geöffnet werden, indem Sie mit der linken Maustaste auf die Schaltfläche im Animationskurven-Editor oder im Sequence Dope Sheet klicken oder indem Sie mit der rechten Maustaste irgendwo im Kanal-Editor klicken.

Kurven-Typ

Der Kurventyp steuert, wie ein Kanal von Punkt zu Punkt fließt, der auf eine der folgenden Optionen eingestellt werden kann:

  • Linear: Dadurch wird ein linearer Verlauf zwischen den Punkten auf dem Kanal erzeugt
  • Glatt: Dies verwendet die Katmull-Rom-Interpolation, um einen glatten Verlauf zwischen den Punkten auf dem Kanal zu erzeugen
  • Bezier: Hier wird die Mathematik der Bezier-Kurve verwendet, um einen sanften Verlauf zwischen den Punkten des Kanals zu erzeugen, wobei Sie das Aussehen der Kurve durch Manipulation der "Griffe" an den Punkten steuern können

Beachten Sie, dass das Ändern des Kurventyps diesen Typ auf alle Kanäle in der Animationskurve anwendet.

Kurvenvoreinstellungen

EineKurvenvoreinstellung ändert die Form Ihrer Kurve. Wenn Sie mit der linken Maustaste auf eine Voreinstellung im Menü "Voreinstellungen" klicken, wird diese Voreinstellung auf den ausgewählten Kanal angewendet. Wenn Sie mit der linken Maustaste auf eine Voreinstellung im Menü Voreinstellungen klicken, wird diese Voreinstellung auf den ausgewählten Kanal angewendet.

HINWEIS: Voreinstellungen können nur angewendet werden, wenn der Kurventyp auf Bezier eingestellt ist.

Standardmäßig wird eine Voreinstellung auf den gesamten Kanal angewendet. Sie können einen Bereich Ihres Kanals auswählen, indem Sie mit der linken Maustaste im Kanal-Editor ziehen und dann das Fenster "Animationskurven-Bibliothek" über die Schaltfläche oder durch einen Rechtsklick auf öffnen, wodurch alle Voreinstellungen nur auf den ausgewählten Bereich angewendet werden.

Beachten Sie, dass Ihr Kurvenkanal (oder eine Auswahl innerhalb dieses Kanals) eine Steigung haben muss, damit eine Voreinstellung eine Wirkung hat, da jede angewandte Interpolation auf der vertikalen Differenz zwischen dem Anfangs- und Endpunkt basiert. Eine flache Kurve hat keinen solchen Unterschied zwischen den beiden Punkten, so dass Sie nach dem Anwenden einer Voreinstellung auf eine solche Kurve keine Ergebnisse sehen werden.

Es gibt zwei Anwendungsmodi, aus denen Sie wählen können, was sich darauf auswirkt, wie eine Voreinstellung auf Ihren Kanal (oder auf Ihre Auswahl innerhalb dieses Kanals) angewendet wird:


  • Überschreiben: Dies ist die Standardeinstellung, wenn das Fenster über die Schaltfläche im Animationskurven-Editor oder im Sequence Dope Sheet geöffnet wird. Es überschreibt alle Punkte zwischen dem Anfangs- und Endpunkt Ihres Kanals oder Ihrer Auswahl.
  • Zwischen: Dies ist die Standardeinstellung, wenn das Fenster durch einen Rechtsklick auf an einer beliebigen Stelle im Kanal-Editor geöffnet wird. Es wendet die Voreinstellung zwischen jedem Paar von ausgewählten Punkten an, und fügt alle neuen Punkte hinzu, die benötigt werden, ohne vorhandene Punkte zu entfernen.

Es gibt auch drei Easing-Modi , aus denen Sie wählen können, die steuern, wie eine Kurve beschleunigt oder verlangsamt wird. Der Standard-Easing-Modus ist " Ease Out".

  • Langsam einfahren: Die Kurve beginnt mit einer schnelleren Beschleunigung und verlangsamt sich dann
  • Ausklingen lassen: Die Kurve beginnt mit einer langsameren Beschleunigung und beschleunigt sich dann mit der Zeit
  • Ease In-Out: Die Kurve beginnt mit langsamerer Beschleunigung, beschleunigt mit der Zeit und verlangsamt sich dann
