Der Kachelsatz-Editor

Ein Kachelsatz ist eine Grafikressource zum Zeichnen von Ebenen und anderen statischen Komponenten Ihres Spiels. Ein Kachelsatz besteht aus einem einzelnen Bild das dann in verschiedene "Zellen" (Kacheln) aufgeteilt wird, und jede Kachel kann dann im Raumeditor platziert werden, um ein vollständiges Bild zu erstellen. Unten sehen Sie zwei Beispiel-Sprites, die als Kachelsätze verwendet werden können:

Bilder von Lost Garden

Dies sind die häufigsten Arten, wie Leute Kachelsätze erstellen - entweder alle zusammengestopft oder mit kleinen Zwischenräumen, die sie voneinander trennen, und GameMaker Studio 2 kann mit beiden Arten umgehen. Im Wesentlichen müssen alle Kachelsätze aus Bildern bestehen, die in ein klar definiertes Raster passen. (Beachten Sie, dass die oben gezeigten Bilder zwar alle aus quadratischen Kacheln bestehen, ein Kachelsatz aber auch aus rechteckigen Kacheln erstellt werden kann).

Denken Sie beim Erstellen Ihrer Kachelsätze daran, dass die obere linke Rasterzelle immer leer sein muss, da dies die Kachel ist, die GameMaker Studio 2 für "leere" Kacheln in einem Raum und zum Löschen vorhandener Kacheln verwendet (und selbst wenn wenn Sie Pixel in diesem Teil des Bildes haben, werden diese ignoriert).

Wenn Sie zum ersten Mal eine Kachelsatz-Ressource erstellen, wird das Fenster "Kachelsatz-Editor" mit den folgenden Optionen geöffnet:

Name

Hier können Sie Ihrem Kachelsatz einen Namen geben. Namen können nur alphanumerisch sein und dürfen nur das Unterstrich-Symbol "_" verwenden, und sie dürfen nicht mit einer Zahl beginnen.


Sprite

Hier können Sie das Sprite-Asset auswählen, aus dem unser Kachelset erstellt werden soll. Wenn Sie auf die Hauptschaltfläche klicken, wird der Asset Explorer geöffnet, in dem Sie das Sprite auswählen können, das Sie für das Kachelset verwenden möchten:

Sie können auch auf die Schaltfläche New Sprite klicken, um ein neues Sprite-Asset zu erstellen (wodurch der Sprite-Editor für das neue Asset innerhalb des Arbeitsbereichs zur Bearbeitung geöffnet wird), oder Sie können das aktuell ausgewählte Sprite mit der Schaltfläche Edit Sprite bearbeiten oder sogar das Sprite-Bild selbst im Bild-Editor bearbeiten, indem Sie auf die Schaltfläche drücken.

Standardmäßig wird beim Öffnen eines Kachelsatzes auch das Fenster " Kachelsatzeigenschaften " angezeigt (siehe unten), aber wenn Sie es schließen, können Sie auf die Schaltfläche hier klicken, um es wieder zu öffnen.


Eigenschaften

Im Fenster " Eigenschaften des Kachelsatzes " können Sie die Basisgröße der Kacheln (die Voreinstellung ist 16 px im Quadrat), den Versatz und den Abstand zwischen den Kacheln festlegen (die letzten beiden Optionen sind für die Verwendung mit Sprites wie dem im Beispielbild weiter unten gezeigten im Beispielbild weiter oben auf dieser Seite, wo es Lücken zwischen den einzelnen Teilen der Kacheln gibt). Sie können auch die Werte für den Ausgaberand des Kachelsatzbildes einstellen, der steuert, wie das Bild auf der Festplatte gespeichert wird und nach Bedarf zusätzliche Pixel um jede Kachel herum hinzufügen.

Aufgrund der Art und Weise, wie Kacheln gerendert werden, müssen sie vorher mit einem Rand um jede Kachel herum vorbereitet werden, um die Kanten zu "verwischen". Als nächstes können Sie die Texturgruppe auswählen, zu der die Tileset-Ressource gehören soll. Grundsätzlich ist eine Texturgruppe (zuvor im Texturgruppen-Manager definiert) etwas, das Sie einrichten können, damit alle Bild-Assets, die Sie für bestimmte Räume oder Ebenen Ihres Spiels benötigen, zusammen gespeichert werden können. So können Sie z. B. alle Bilder der Ebene 1 in einer Texturgruppe haben, alle Bilder der Ebene 2 in einer anderen usw... und GameMaker Studio 2 wird versuchen, alle diese gruppierten Assets auf derselben Texturseite zu platzieren, um das Vertauschen von Texturseiten zu reduzieren, während Ihr Spiel auf der gewählten Zielplattform läuft.

HINWEIS: Dies ist möglicherweise nicht immer erforderlich und die Leistungssteigerung durch diese Methode hängt davon ab, ob das Zielgerät CPU- oder GPU-gebunden ist.

Die letzte Option in den Eigenschaften des Kachelsatzes ist " Disable Source Sprite Export". Diese Option ist standardmäßig aktiviert und bewirkt, dass das Quell-Sprite-Asset bei der Generierung der Texturseite für Ihr Projekt ausgelassen wird. Diese funktioniert unter der Annahme, dass ein Sprite, das als Kachelsatz verwendet wird, nicht für etwas anderes verwendet wird und daher nur einmal exportiert werden muss. Wenn Sie jedoch die Basis-Sprite-Ressource für etwas anderes als Kacheln verwenden, dann sollten Sie deaktivieren, damit sowohl das Basis-Sprite als auch das Kachelset beim Kompilieren zur Textur-Seite hinzugefügt werden.

Herausgeber

Jede der Schaltflächen im Bereich " Editoren" öffnet einen anderen Editor für bestimmte Kachelsatzfunktionen. Sie können Kachelsatzpinsel erstellen, die auf mehreren Kacheln zusammen basieren, animierte Kacheln, die verschiedene Kacheln Sie können auch automatische Kacheln erstellen, die sich auf der Grundlage der sie umgebenden Kacheln "zusammenfügen". Jeder dieser Editoren wird in den Abschnitten, die am Ende dieser Seite verlinkt sind, im Detail erklärt.


Vorschau-Toolbox

Dies sind die Werkzeuge, mit denen Sie eine Vorschau des für den Kachelsatz ausgewählten Sprites anzeigen können. Diese Schaltflächen steuern die aktuelle Zoomstufe der Vorschau und Sie können sie zum Vergrößern oder Verkleinern verwenden. Wenn Sie auf die Schaltfläche klicken, wird die Vorschau auf 1:1 mit dem bearbeiteten Kachelsatz-Sprite zurückgesetzt. Sie können auch auf die Schaltfläche Window Fit ( ) klicken, um das gesamte Kachelset in das aktuelle Vorschaufenster einzupassen (dadurch wird es entsprechend vergrößert/verkleinert, damit es passt). Beachten Sie, dass Sie auch vergrößern und verkleinern können / und das Mausrad verwenden, und durch Drücken von / + wird die Leinwand 1:1 mit dem Kachelsatz-Sprite eingestellt.

Eine weitere Funktion ist, dass Sie die Rasteransicht ein- oder ausschalten können, indem Sie auf die Schaltfläche klicken, ebenso wie Sie die Farbe für das zu verwendende Raster einstellen (durch Klicken auf die Schaltfläche ). Wenn das Raster aktiv ist, wird ein farbiger Umriss für jede der Kacheln im Raster angezeigt, wobei die Einstellungen für die Trennung usw. berücksichtigt werden.


Vorschau

Dieser Bereich ist das Kachelsatz-Vorschaufenster. Um in der Vorschau zu navigieren, können Sie dieselben grundlegenden Steuerelemente wie für allgemeine Arbeitsbereiche verwenden, d. h.: Verwenden Sie die mittlere Maustaste und ziehen Sie, um die Vorschau zu verschieben (oder verwenden Sie alternativ die Kombination + ), und Sie können mit dem Mausrad horizontal scrollen oder mit der Taste / und dem Mausrad vergrößern und verkleinern. Sie können auch die verschiedenen Schaltflächen in der Vorschau-Toolbox verwenden, um den Zoom zu ändern und die Überlagerung der Gitterzellen zu aktivieren/deaktivieren.

Wie im obigen Abschnitt über Editoren erwähnt, stehen weitere Werkzeuge zur Verfügung, um Kachelsätze zu bearbeiten und spezielle Pinsel (Kombinationen von Kacheln, die in einen Raum gemalt werden können), animierte Kacheln und automatische Kacheln (das sind spezielle Kacheln, die sich automatisch verbinden, wenn sie in einem Raum platziert werden). Jeder dieser Editoren wird auf den folgenden Seiten eingehend erläutert:

  • Animierte Kacheln
  • Auto-Kacheln
