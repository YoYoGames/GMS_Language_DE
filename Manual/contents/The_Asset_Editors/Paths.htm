Der Pfad-Editor

Irgendwann beim Erstellen Ihrer Spiele müssen Sie vielleicht eine Ihrer Instanzen einem Pfad durch einen Level folgen lassen. Das kann man im Code machen, indem man einfach ein Array von ein Array von Positionen innerhalb des Raums erstellen und dann eine Instanz zwischen diesen Positionen hin- und herbewegen lassen, aber das kann eine lästige Aufgabe sein und ist schwierig zu ändern, zu testen und für verschiedene Instanzen anzupassen. Aus diesem Grund verfügt GameMaker Studio 2 über Pfadressourcen und einen speziellen Pfad-Editor.

Die Grundidee ist recht einfach - Sie definieren einen Pfad, indem Sie ihn im Pfad-Editor zeichnen, dann können Sie eine Aktion (oder Code) im Erstellungsereignis eines Objekts platzieren, um der Instanz zu sagen, dass sie diesem bestimmten Pfad folgen soll. Sie können auch die Geschwindigkeit festlegen, mit der Sie können auch die Geschwindigkeit festlegen, mit der der Pfad verfolgt wird, sowie eine Reihe anderer Aktionen, die sich auf die Position und Ausrichtung des Pfades im Raum beziehen. Pfad-Eigenschaften

Wie bei allen Ressourcen in GameMaker Studio 2 sollten Sie Ihrem neuen Pfad einen eindeutigen Namen geben, da er im gesamten Spiel, das Sie erstellen, zur Identifizierung verwendet wird. voranzustellen (oder zu ergänzen), damit sie in Code oder Aktionen leicht zu erkennen ist (z. B.: path_Enemy_1 oder PowerUp_path).

Nachdem Sie den Pfad benannt haben, können Sie damit beginnen, ihn zu definieren, indem Sie Punkte im Haupteditor auf der rechten Seite platzieren, wobei jeder Punkt hier zusammen mit seiner Position und Geschwindigkeit in die Liste aufgenommen wird. Der Geschwindigkeitswert für Pfadpunkte wird durch einen Prozentwert definiert der Geschwindigkeit, mit der Sie die Instanz entlang des Pfades bewegen lassen (Standard ist 100 %). Wenn Sie also eine Instanz mit einer Geschwindigkeit von 2 (2 Pixel pro Schritt) entlang eines Pfades starten und dann einer der Pfadpunkte einen Geschwindigkeitswert von 50 % hat, wird die Instanz die Geschwindigkeit wenn sie sich diesem Punkt nähert, auf eine Geschwindigkeit von 1 (50 % der Pfadgeschwindigkeit). Genauso können Sie die Geschwindigkeit auf 150% setzen und die Instanz wird beschleunigen, wenn sie den Punkt erreicht. Auf diese Weise können Sie eine schöne, dynamisch aussehende Bewegung ohne jeglichen Code erstellen.

Es ist erwähnenswert, dass Sie einen schnellen Doppelklick auf einen der definierten Punkte in der Liste machen können, um den Editor auf den ausgewählten Punkt zu schwenken, oder langsam auf einen der Werte doppelklicken, um ihn manuell zu bearbeiten (x, y oder Geschwindigkeit).

Sie können auch die Verbindungsart für den gesamten Pfad angeben und ob der Pfad geschlossen sein soll oder nicht (ein geschlossener Pfad verbindet den letzten Punkt des Pfades mit dem ersten). Die Verbindungsart kann entweder geradlinig oder glatt sein, wobei eine gerade Verbindung einfach die Punkte eines Pfades auf linearer Basis verbindet, mit Winkeländerungen an jedem Punkt, während eine glatte Verbindung einen viel weicheren Pfad erzeugt, der möglicherweise nicht direkt durch einen der Punkte verläuft, sondern eher eine eine ungefähre Linie, die durch weiche Interpolation schöne Kurven erzeugt. Sie können auch die Genauigkeit für die Kurven angeben (Standard ist 4), die zwischen 1 und 8 liegen kann, wobei 8 die am glattesten ist. Pfad-Bearbeitung

Sie können dem Pfad Punkte hinzufügen, indem Sie auf an einer beliebigen Stelle des Bearbeitungsfensters klicken, und alle nachfolgenden Klicks fügen weitere Punkte hinzu. Sobald ein Punkt im Editor platziert ist, kann er verschoben werden, indem Sie Klicken Sie auf und ziehen Sie ihn oder ändern Sie die Werte, die Ihnen links für die X- und Y-Position und die Geschwindigkeit angezeigt werden (siehe Pfadeigenschaften, oben). Sie Sie können / verwenden und mit der linken Maustaste auf mehrere Punkte klicken, um sie gemeinsam auszuwählen, und diese Gruppe von Punkten kann dann beliebig verschoben und platziert werden. Sie können auch drücken, um den ausgewählten Punkt oder die ausgewählten Punkte zu löschen.

Im Editor-Fenster steht Ihnen ein zusätzliches Menü zur Verfügung, wenn Sie mit der rechten Maustaste auf klicken, wie unten gezeigt:

Jeder der Punkte in diesem Menü funktioniert wie folgt:

  • Punkte löschen - Löscht den ausgewählten Punkt oder die ausgewählten Punkte.
  • Auf Pfad schwenken - Schwenkt die Ansicht im Editor auf die Mitte des Pfades.
  • Umkehren - Kehrt die Reihenfolge der Pfadpunkte um. Dadurch werden sie nicht verschoben, sondern die Reihenfolge, in der sie verarbeitet werden, wird so geändert, dass der Anfang zum Ende wird und andersherum.
  • Spiegeln - Spiegelt den Pfad entlang der horizontalen Achse.
  • Spiegeln - Spiegelt den Pfad entlang der vertikalen Achse.

Der Werkzeugkasten

Mit den Schaltflächen des Werkzeugkastens oben können Sie steuern, wie die Dinge im Editorfenster gezeichnet werden. Sie können das Hintergrundraster ein- oder ausschalten und festlegen, ob die hinzugefügten Pfadpunkte an diesem Raster gefangen werden sollen oder nicht:

Die Fangwerte geben die Größe der Gitterzellen an und kann auf jeden gewünschten Wert geändert werden. Der Rest der Werkzeuge dient zum Vergrößern und Verkleinern und zum Schwenken des Editorbereichs. Sie können auf die Schaltfläche klicken, um die Anzeige auf 1:1 zurückzusetzen. Beachten Sie, dass Sie, wenn Sie zu weit von der Stelle schwenken, an der Sie den Pfad zeichnen möchten, das Menü der rechten Maustaste verwenden und dann auf " Pan to Path" (Auf Pfad schwenken) klicken können, um zu der Stelle zurückzuscrollen, an der Sie gearbeitet haben.


Wegpunkte

Pfade sind einfach eine Sammlung von Punkten, die durch eine Linie miteinander verbunden sind. Im Editor platzieren Sie diese Punkte mit der linken Maustaste . Sie können an beliebiger Stelle im Raumeditor Punkte hinzufügen, die dann nacheinander in die Punkteliste aufgenommen werden, um einen Pfad zu bilden. Wenn Sie einen Punkt in der Nähe einer Verbindungslinie oder eines anderen Knotens platzieren, wird der neue Punkt in den Pfad eingefügt und nicht zu ihm hinzugefügt (Sie können den Erkennungsabstand dafür in den Voreinstellungen einstellen). Sie können / drücken und auf auf bereits platzierte Punkte klicken, um sie auszuwählen, und dann werden alle Operationen, die an den Pfadpunkten vorgenommen werden, wie z. B. das Verschieben, auf alle die ausgewählten Punkte als Gruppe angewendet. Um einen Punkt oder eine Auswahl von Punkten zu entfernen, können Sie mit der rechten Maustaste auf klicken und "Delete Punkt" wählen oder die Taste drücken.

Beachten Sie, dass Sie auch die folgenden Tastenkombinationen verwenden können, um einige zusätzliche Operationen durchzuführen, die nicht über das Menü der rechten Maustaste auf dem/den ausgewählten Pfadpunkt(en) verfügbar sind:

  • T + : Versetzen (Verschieben) der Bahnpunkte
  • R + : Drehen der Pfadpunkte
  • S + : Skalieren der Bahnpunkte
