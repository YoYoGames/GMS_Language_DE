Der Font-Editor

Wenn Sie Text in Ihrem Spiel zeichnen möchten, wird dieser Text in einer Standardschriftart Arial 12 Punkt gezeichnet, aber um interessantere oder einzigartig aussehende Texte zu erstellen, werden Sie wahrscheinlich andere Schriftarten verwenden wollen. Um also verschiedene Schriftarten zu verwenden, die Sie auf Ihrem Computer haben, müssen Sie in GameMaker Studio 2 eine Schriftart-Asset erstellen. Dazu klicken Sie mit der rechten Maustaste auf im Asset-Browser und wählen "Erstellen" -> "Schriftart". Für jedes Font-Asset, das Sie erstellen, geben Sie eine bestimmte Schriftart von Ihrem Computer an, die dann in Ihrem Spiel mit Hilfe der dann in Ihrem Spiel mit Hilfe der Aktion oder des Codes zum Setzen einer Schriftart zum Zeichnen verwendet werden kann.

Details

Hier können Sie Ihrer Schrift einen eindeutigen Namen geben (nur alphanumerische Zeichen und das "_"-Unterstrichsymbol) sowie die zu verwendende Schriftart auswählen. Die Schriftart muss auf Ihrem Rechner installiert sein, und aus lizenzrechtlichen Gründen speichert GameMaker Studio 2 nicht Aufgrund von Lizenzproblemen speichert GameMaker Studio 2 die eigentlichen Schriftdateien standardmäßig nicht mit der Projektdatei, sondern die Schrift wird als eine Reihe von Bitmap-Glyphen auf eine Texturseite gerendert. von Bitmap-Glyphen gerendert. Die Texturseite wird einer Texturgruppe zugewiesen, die Sie auch hier über die Option Gruppe auswählen können.

Wenn Sie die eigentliche Schriftdatei ( *.ttf oder nur *.otf ) zusammen mit dem Projekt verteilen möchten, wenn Sie ein Marketplace-Asset erstellen oder aus anderen Gründen, können Sie die Option In Projekt kopieren aktivieren. Diese öffnet den Datei-Explorer, in dem Sie die Schriftdatei auswählen können, die dann zu den mit dem Projekt gebündelten Dateien hinzugefügt wird.

WICHTIG! Sie sollten nur Schriftarten verwenden, deren Weitergabe oder Verwendung in Ihrem Projekt rechtlich zulässig und lizenziert ist.

Voreinstellungen

In diesem Bereich können Sie die Größe (in Punkt - Standard ist 12pt) einstellen und ob die Schrift fett oder kursiv gezeichnet oder mit Anti-Aliasing versehen werden soll. Das Vorschaufenster auf der rechten Seite zeigt Ihnen den Text ungefähr so an, wie er in Ihrem Spiel mit der von Ihnen angegebenen Größe und den Transformationen aussehen wird.


Bereichskontrollen

Wenn Sie eine Font-Ressource erstellen, ist es selten, dass Sie jede einzelne Glyphe des Fonts einbeziehen möchten. Aus diesem Grund verwendet GameMaker Studio 2 standardmäßig nur die Zeichen von 32 bis 127 des gewählten Fonts, da dies die Zahlen und die Klein- und Großbuchstaben sowie bestimmte Satzzeichen. Sie können jedoch den verwendeten Zeichenbereich ändern, um Ihre Spiele zu optimieren, d. h., wenn Sie nur die Zahlen einer bestimmten Schriftart benötigen, wählen Sie nur die Zahlen aus. Um dies Klicken Sie dazu zuerst auf die Schaltfläche " Löschen" , um den aktuellen Bereich zu löschen (möglicherweise müssen Sie den Bereich markieren, damit dies funktioniert, indem Sie im Vorschaufenster auf klicken). Vorschaufenster) und dann auf die Schaltfläche Hinzufügen klicken, um einen neuen Bereich hinzuzufügen. Dadurch wird das folgende verkettete Fenster geöffnet:

Dieses Fenster enthält einige Schaltflächen, mit denen Sie einen Standardbereich für Ihre Schrift festlegen können:

- Der normale Bereich von 32 bis 127

- Der ASCII-Bereich von 0 bis 255

- Der Ziffernbereich , der nur die 10 Ziffern enthält

- Der Buchstabenbereich , der nur alle Groß- und Kleinbuchstaben enthält

Andere Bereiche können verwendet werden, indem Sie den ersten und letzten Zeichenindex des gewünschten Bereichs in die Eingabefelder für den Zeichenbereich eingeben (wenn ein Zeichen nicht im Bereich liegt, wird es durch ein Leerzeichen ersetzt). Abgesehen von diesen Schaltflächen und Eingabefeldern, haben Sie noch zwei weitere Optionen, die besonders nützlich sind:

  • Aus Code: Wenn Sie auf diese Schaltfläche und dann auf OK klicken, analysiert GameMaker Studio 2 automatisch Ihren Spielcode nach Zeichenketten und erstellt dann verschiedene Zeichenbereiche, um den gesamten Text in Ihrem Spiel abzudecken. Beachten Sie, dass es nach alle Zeichenketten (entweder innerhalb von "" oder '') und kann daher auch Dateinamen in die Zeichenbereiche aufnehmen.
  • Aus Datei: Dies funktioniert ähnlich wie die oben erklärte Option "Aus Code". Wenn Sie darauf klicken, werden Sie aufgefordert, eine Datei anzugeben, und wenn das erledigt ist, sollten Sie auf OK klicken. GameMaker Studio 2 parst dann die Datei Datei und erstellt Zeichenbereiche für den darin gefundenen Text.

Beachten Sie, dass Sie den Textbereich manuell bearbeiten können, indem Sie direkt im Vorschaufenster Textglyphen hinzufügen oder löschen.

Textur-Steuerelemente

Wie oben (im Abschnitt über Schriftartdetails) erwähnt, werden standardmäßig alle Schriftglyphen als Teil einer Texturgruppe in eine Texturseite gerendert, um sie in grafische Wenn das Spiel kompiliert wird, wird die Schrift zu einer temporären Textur hinzugefügt, um ein einzelnes Bild mit allen erforderlichen Schriftglyphen zu erstellen, das dann der Haupttexturgruppe zum Exportieren hinzugefügt wird. Die Schaltflächen hier dienen zum Einstellen wie die Schrifttextur generiert wird oder um eine neue Schrifttextur zu generieren.

Mit der Schaltfläche Optionen öffnen Sie die erweiterten Optionen für die Schrifttextur:

Diese Optionen können die Darstellung einer Schriftart im Spiel teilweise radikal verändern, und nicht alle Schriftarten benötigen die gleichen Einstellungen. Die folgenden Optionen sollen Ihnen die größtmögliche Anpassung Ihrer Schriftart ermöglichen:

  • Hinting: Font Hinting (auch als Instruktion bekannt) ist die Verwendung von mathematischen Anweisungen, um die Anzeige einer Umrissschrift so anzupassen, dass sie mit einem gerasterten Raster übereinstimmt. Bei niedrigen Bildschirmauflösungen ist das Hinting entscheidend für die Erzeugung von Hier können Sie zwischen verschiedenen Hinting-Algorithmen wählen, um das bestmögliche Ergebnis zu erzielen. Die Voreinstellung ist "normal".
  • Glyphen-Optionen: Mit diesen Optionen können Sie das Laden der Glyphen durch Aktivieren/Deaktivieren verschiedener Operationen ändern. Diese sind:
    1. Hinting deaktivieren: Damit wird das fonteigene Hinting und der Auto-Hintergrund deaktiviert.
    2. Auto-Hinterlegung deaktivieren: Dies deaktiviert den Auto-Hintergrund und verwendet stattdessen den nativen Hintergrund der Schriftart (wenn diese einen hat). Beachten Sie, dass dies die Option "Auto-Hinterlegung bevorzugen" außer Kraft setzt.
    3. Auto-Hintergrund bevorzugen: Diese Option weist GameMaker Studio 2 an, den Auto-Hintergrund gegenüber dem nativen Hinting der Schriftart zu bevorzugen. Beachten Sie, dass dies durch die Optionen "Auto-Hintergrund deaktivieren" und "Hinting deaktivieren" Optionen überschrieben wird.
    4. Verzerren: Wenn Sie diese Option aktivieren, werden Glyphen leicht verzerrt, um die Rasterausrichtung zu verbessern. Beachten Sie, dass hierfür der Auto-Hintergrund und der Hinting-Modus "Normal" erforderlich sind.
    5. Keine Skalierung: Wenn diese Option aktiviert ist, bleiben die Glyphen in Schrifteinheiten und ignorieren die angeforderte Größe. Dies bedeutet, dass Hinting deaktiviert ist.
    Der Standardzustand für alle oben genannten Optionen ist deaktiviert.
  • Interpreter: Hier können Sie die TrueType-Version auswählen, die GameMaker Studio 2 zum Rendern von Schriften verwenden soll. Standard ist TrueType v40.
  • Rundung: Mit dieser Option wird die Rundungsmethode bei der Konvertierung von einer Festkomma-Dezimalgröße in eine Ganzzahl festgelegt. Standard ist "floor".

Mit der Schaltfläche Neu generieren können Sie die Schrifttextur neu generieren. Dies ist erforderlich, wenn Sie Änderungen, die Sie an einer bestehenden Schrift vorgenommen haben, aktualisieren möchten, oder wenn Sie eine neue Schrift erstellt haben und sehen möchten, wie sie im Vorschaufenster Fenster. Beachten Sie, dass Sie Schriften auch über das Menü der rechten Taste für die Schrift (oder die Schriften) im Asset-Browser regenerieren können.

Es ist wichtig zu beachten, dass beim ersten Erstellen einer neuen Schriftart, wenn Sie die Textur nicht im Editor regenerieren, Wenn Sie die Textur nicht im Editor neu generieren, wird die Textur beim ersten Ausführen oder Testen des Spiels erstellt, aber das ist das einzige Mal, dass sie automatisch generiert wird. Das bedeutet, dass, sobald eine Schriftart eine generierte Textur hat, alle Änderungen an der Schriftart-Ressource erfordern dass die Schriftart neu generiert werden muss, damit die Änderungen im Spiel sichtbar werden, andernfalls wird weiterhin die "alte" Schrifttextur verwendet.

GameMaker Studio 2 rendert die generierte Schrifttextur als PNG Datei zusammen mit den restlichen Projektdateien, und sollten Sie das Projekt laden und die Basisschriftart nicht gefunden werden, dann wird das Spiel den Text trotzdem korrekt anzeigen mit der gespeicherten Textur. Eine fehlende Schriftart bedeutet jedoch, dass Sie außer dem Namen, der Schriftfamilie und der Texturgruppe (über den Texturgruppen-Editor) keine weiteren Eigenschaften mehr ändern können.


Vorschau-Modus

In diesem Bereich können Sie wählen, ob das Vorschaufenster entweder einen Beispieltext der ausgewählten Schriftart oder das gerenderte Texturbild, das für die Schriftart erzeugt wurde, anzeigen soll.

Vorschau

Das Vorschaufenster können Sie eine Vorschau der Schriftart sehen, entweder als Textmuster oder als Textur. Wenn der Vorschaumodus eingestellt ist auf Sample eingestellt ist, können Sie jeden Text, den Sie testen möchten, direkt in das Vorschaufenster eingeben, um zu sehen, wie er aussieht, während die Auswahl von Spielvorschau wird das gerenderte Texturbild angezeigt, das für die Schriftart generiert wurde. Wenn kein Bild vorhanden ist, sollten Sie auf die Schaltfläche Neu generieren klicken, um die Textur zu generieren.

Beachten Sie, dass das Texturbild vom Muster abweichen kann, da die Textur nicht neu generiert wird, es sei denn, dies wird ausdrücklich gewünscht. Das bedeutet, dass nach dem anfänglichen Erzeugen einer Schrifttextur Änderungen, die an der Schriftressource vorgenommen wurden, nicht in das Spiel übernommen werden, es sei denn, die Textur wird neu generiert.

Bereich Vorschau

Hier werden die verschiedenen Zeichenbereiche, die Sie für die Schriftart hinzugefügt haben, aufgelistet. Die Werte der Bereiche werden auf der linken Seite angezeigt und die diesen Werten entsprechenden Bereichsglyphen werden auf der rechten Seite angezeigt.
