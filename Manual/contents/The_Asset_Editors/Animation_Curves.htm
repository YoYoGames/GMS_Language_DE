Der Animationskurven-Editor

Eine Animationskurve ist ein Asset, das eine oder mehrere Kurven enthält, die darstellen, wie sich ein Wert im Laufe der Zeit ändert, und kann lineare Interpolation, glatte Interpolation oder Bezier-Interpolation verwenden, um zwischen verschiedenen Punkten in der Kurve zu wechseln. Die Werte, die Sie auf der vertikalen Achse einstellen, können zwischen zwei beliebigen Werten liegen (standardmäßig zwischen -1 und 1) und die Dauer entlang der horizontalen Achse ist immer von 0 bis 1 normalisiert, was es einfach macht unterschiedliche Zeitbereiche mit Hilfe von Multiplikatoren in Ihrem Spielcode anzusteuern oder sie zu strecken, damit sie in einen Track im Sequenzeditor passen.

Animationskurven bestehen aus "Kanälen" und jeder Kanal kann seine eigenen Kurveneinstellungen haben, was es Ihnen erlaubt, z. B. eine räumliche Position mit zwei Kanälen für die x/y-Lage zu beschreiben oder einen Farbverlauf mit 4 Kanälen, um ein Farbformat ein Farbformat darzustellen. Animationskurven-Assets können bei der Arbeit mit Sequenzen verwendet werden und können auch über Code aufgerufen werden, was sie zu einem leistungsstarken Werkzeug bei der Erstellung Ihrer Spiele macht.

Wenn Sie zum ersten Mal ein Animationskurven-Asset erstellen, öffnet sich das Editorfenster und zeigt Ihnen die folgenden Abschnitte:

Kurve Name

Hier können Sie Ihrer Animationskurve einen eindeutigen Namen geben(nur alphanumerische Zeichen und das Unterstrichsymbol "_"). Dieser Name wird verwendet, um auf die Kurve und alle ihre Kanäle zu verweisen wenn Sie über Code oder in einer Sequenz auf sie zugreifen.

Kurvenbibliothek

Die Schaltfläche öffnet das Animationskurven-Bibliothek Fenster, in dem Sie den Kurventyp der Animationskurve ändern und eine Voreinstellung auf den ausgewählten Kanal anwenden können.

Informationen zu Kurventypen und Voreinstellungen finden Sie im Abschnitt über die Animationskurvenbibliothek. über Kurventypen und Voreinstellungen. Beachten Sie, dass beim Öffnen des Fensters über die Schaltfläche standardmäßig der Anwendungsmodus "Überschreiben" ausgewählt ist.

Kanal-Liste

In der Kanalliste können Sie die verschiedenen Kanäle, aus denen Ihr Animationskurven-Asset bestehen wird, hinzufügen und benennen. Wenn Sie zum ersten Mal eine Animationskurve erstellen, wird standardmäßig ein Kanal für Sie hinzugefügt, und Sie können weitere Kanäle hinzufügen, indem Sie auf die Schaltfläche auf die Schaltfläche links neben dem Kurvennamen klicken. Jeder Kanal muss einen eindeutigen Namen haben ( nur alphanumerische Zeichen und das Unterstrichsymbol "_" verwenden ), und Sie können auch die Farbe für den Kanal festlegen, indem Sie auf auf das Farbfeld links klicken, wodurch sich der Farbwähler öffnet und Sie eine neue Farbe auswählen können. Um den Kanal umzubenennen, können Sie einen langsamen Doppelklick und umbenennen, und Sie können auch die rechte Maustaste verwenden, um ein Menü zu öffnen, in dem Sie wählen können, ob Sie den Kanal umbenennen, die Farbe ändern oder löschen möchten.

Kanäle können durch Klicken auf den Pfeil neben dem Namen erweitert werden, um jeden der Punkte auf der Kanalkurve anzuzeigen, und diese Punkte können manuell bearbeitet werden, indem Sie auf den aktuellen Wert klicken und den neuen Wert eintippen.

Sie können die Sichtbarkeit des Kanals durch Klicken auf das "Auge"-Symbol umschalten oder den Kanal durch Klicken auf das "Mülleimer"-Symbol löschen.

Kanalbereich

Die Schaltfläche "Kanalbereich" dient zum Einstellen des visuellen vertikalen Bereichs für den Kanal-Editor. Standardmäßig ist dieser von -1 bis 1 eingestellt, aber wenn Sie auf diese Schaltfläche klicken, wird das folgende Fenster geöffnet:

Wenn Sie diese Werte ändern, ändert sich die Art der Darstellung der Daten im Kanal-Editor-Bereich, so dass Sie höhere/niedrigere Werte entlang der Kanalkurve als nur -1 bis 1 haben können. Beachten Sie, dass dies rein visuell ist und die Werte für den Kanal nicht festhält.

Kanal-Editor

Der Kanal-Editor zeigt eine visuelle Darstellung der verschiedenen Kanäle, die in der Animationskurve verwendet werden. Standardmäßig werden hier alle Kanäle angezeigt, aber Sie können nur den aktuell in der Kanalliste ausgewählten Kanal bearbeiten, der im Editor hervorgehoben wird im Editor hervorgehoben wird. Sie können die Position jedes der Punkte entlang der Kanalkurve ändern, indem Sie auf klicken und sie im Editor verschieben, aber beachten Sie, dass der Beachten Sie jedoch, dass der erste und der letzte Punkt nur entlang der vertikalen Achse geändert werden können und ihr horizontaler Wert immer 0 bzw. 1 sein wird.

Sie können Punkte direkt zur Kanalkurve hinzufügen, indem Sie die Maus in die Nähe der Hauptkanallinie bewegen und dann auf klicken, wenn der Cursor zum Werkzeug "Punkt hinzufügen" wechselt. Sie können diese Punkte dann bearbeiten, indem Sie auf klicken und ziehen oder indem Sie ihre Werte in den erweiterten Kanaloptionen in der Kanalliste ändern:

Sie können mehrere Punkte auf einmal bearbeiten, indem Sie auf klicken und innerhalb des Editors ziehen, um alle zu bearbeitenden Punkte auszuwählen, oder indem Sie / + verwenden, um Punkte einzeln zur Auswahl hinzuzufügen. Wenn Sie die Maustaste loslassen und dann erneut auf einen der Punkte in der Auswahl klicken und ziehen, werden alle Punkte und ziehen, werden alle Punkte gemeinsam verschoben:

Sie können mit der rechten Maustaste auf eine beliebige Stelle im Kanal-Editor klicken, um das Fenster "Animationskurven-Bibliothek" zu öffnen (nur wenn der ausgewählte Kurventyp Bezier ist). Das Anwenden einer Voreinstellung auf diese Weise wirkt sich auf den gesamten Kanal aus. Um nur einen Teil des Kanals zu ändern, können Sie mit der rechten Maustaste auf eine einzelne Linie klicken, um eine Voreinstellung darauf anzuwenden, oder mehrere Linien auswählen, indem Sie auf klicken und ziehen und dann mit der rechten Maustaste auf die Auswahl klicken, um eine Voreinstellung auf die ausgewählten Linien anzuwenden.

Beachten Sie, dass beim Öffnen des Fensters " Animationskurvenbibliothek" über den Kanal-Editor standardmäßig der Anwendungsmodus "Zwischen" ausgewählt ist.
