Der Bild-Editor

ImBildeditor können Sie eigene Sprites erstellen oder solche bearbeiten, die Sie die Sie zuvor importiert oder erstellt haben. Wenn Sie bereits andere Bildbearbeitungsprogramme verwendet haben, sollte Ihnen dies vertraut sein, aber es gibt ein paar Dinge zu beachten, wenn Sie den GameMaker Studio 2 Bildeditor verwenden, die vielleicht nicht sofort ersichtlich sind:

  • Wenn Sie mehrere Rahmen erstellen, können Sie zu ihnen zeichnen, während sie animiert werden! Wenn Sie also in der Rahmensteuerungsleiste auf die Wiedergabetaste drücken und dann einen Pinsel auswählen und im Editorfenster zu zeichnen beginnen, werden Sie sehen, dass die Rahmen im Editor animiert werden und das, was Sie zeichnen, in diesem Moment dem entsprechenden Rahmen hinzugefügt wird. Sie können mit verschiedenen Bildraten experimentieren Sie können mit verschiedenen Frameraten experimentieren, bis Sie etwas erhalten, das Ihnen gefällt, aber wenn Sie einmal den Dreh raus haben, wie das funktioniert, werden Sie im Handumdrehen dynamische Sprites und Effekte erstellen.
  • Sie können die Registerkarte "Bildeditor" aus dem Hauptfenster des Arbeitsbereichs in ein eigenes Fenster ziehen. Dies kann z. B. bei der Verwendung mehrerer Monitore praktisch sein.
  • Wenn Sie die Kopierfunktionen verwenden, um z. B. eine Auswahl eines Bildes zu kopieren, wird die kopierte Auswahl zu Ihrer Pinselliste hinzugefügt. Das bedeutet, dass Sie einen Teil eines Bildes auswählen und dann mit den übrigen Malwerkzeugen zum Zeichnen von Linien, Polygonen oder einfach zum freien Malen verwenden. Der Pinsel kann auch skaliert und mit einer Farbe überblendet werden. Beachten Sie, dass die Funktion " Einfügen" nicht für die interne Verwendung im Editor gedacht ist, da alles, was Sie kopieren, in die Pinselliste hinzugefügt und kann auf diese Weise zum "Einfügen" ausgewählt werden. Stattdessen ist Einfügen einfach dafür gedacht, dass Sie etwas aus einem externen Programm aufnehmen und in den Bildeditor einfügen.
  • Dank des praktischen Ebenen-Editors können Sie mehrere Ebenen für jedes Sprite haben. Diese Ebenen können geordnet und sogar in Ordnern platziert werden, um das Erstellen komplexerer Bilder zu erleichtern.
  • Sie können im Editor schnell zwischen Teilbildern (Frames) wechseln, indem Sie die Tasten "1" und "2" auf der Tastatur verwenden.

Dies sind nur einige der erweiterten Funktionen, die der Bildeditor bietet. Um den vollen Funktionsumfang kennenzulernen, lesen Sie bitte die unten stehenden Punkte, die die Funktionen jedes Teils des Editors detailliert beschreiben:

Frame-Ansicht

Dieser Teil des Fensters zeigt die Rahmen in der Reihenfolge an, in der sie animiert werden sollen. Sie können auf die Schaltfläche auf der linken Seite klicken, um weitere Bilder hinzuzufügen - oder Sie verwenden das Bildmenü - und Sie können auch auf die Rahmen klicken und sie ziehen, um ihre Position auf der Zeitachse zu ändern. Oben links in der Rahmenansicht können Sie das Onion Skinning einschalten, d. h., Sie können in einem Einzelbild einer Sprite-Animation jedes der vorangehenden oder nachfolgenden Bilder als halbtransparente Ebene unter dem Bildrahmen sehen den Sie gerade bearbeiten. Das Tolle daran ist, dass jedes Bild etwas heller gezeichnet wird als das letzte, so dass Sie sehen können, wie sich die Animation von Bild zu Bild entwickelt hat, während Sie das nächste Bild zeichnen.

Wenn Sie dies aktivieren, werden einige zusätzliche Steuerelemente über den Frames angezeigt, die Sie anklicken und ziehen können, um die Anzahl der anzuzeigenden vorangehenden Frames sowie die die Anzahl der folgenden Frames einzustellen.

Sie können bestimmte Eigenschaften für das Zwiebelhäuten über das Kontextmenü Ansicht einstellen, das weiter unten auf dieser Seite erläutert wird.

Unter der Schaltfläche "Onion Skin" befindet sich die Schaltfläche " Loop Control" , mit der Sie einstellen können, ob das Sprite einen Looping (geht zurück zu Bild 1 zurückgeht, wenn das letzte Bild erreicht ist) oder Ping-Pong (geht rückwärts durch die Bilder, wenn das letzte Bild erreicht ist). Die Play-Taste am am unteren Rand dient zum Starten/Stoppen der Vorschau der Animation.

Sie können auf einen beliebigen Rahmen klicken, um ihn für die Bearbeitung auszuwählen, und ein ausgewählter Rahmen hat in der oberen Ecke eine Schaltfläche "Löschen" , auf die Sie klicken können, um den Rahmen aus dem Sprite zu entfernen. Sie können auch auf / + auf mehrere Frames klicken, um sie gemeinsam auszuwählen; in diesem Fall werden durch Klicken auf die Schaltfläche "Delete" alle ausgewählten Frames gelöscht. Schließlich können Sie Schließlich können Sie die Standard-Tastaturkürzel für Ausschneiden, Kopieren und Einfügen verwenden, um ausgewählte Rahmen zu bearbeiten: / + "X", / + "C", und / + "V".

Ansicht

Das Menü "Ansicht" verfügt über die folgenden Optionen:

  • 1:1 - Skalieren Sie das Bild auf 1:1 mit der Auflösung (d.h.: tatsächliche Größe) und zentrieren Sie es im Bildbearbeitungsfenster.
  • Fit Screen - Skaliert das Bild so, dass es das Bildbearbeitungsfenster ausfüllt.
  • Raster anzeigen - Schaltet das Pixelraster ein. Dies ist dasselbe wie die Schaltfläche "Raster" in den Canvas-Steuerungen (weiter oben auf dieser Seite erklärt).
  • Zwiebelhaut-Einstellungen - Ändern Sie die Einstellungen der Zwiebelhaut. Dadurch wird das folgende Fenster geöffnet, in dem Sie die von der Zwiebelhaut-Schaltfläche angezeigten Rahmen sowie die Deckkraft und Farbe der gehäuteten Rahmen einstellen können.

Effekte

Im Menü "Effekte" können Sie Ihre eigenen benutzerdefinierten Effekt-Plugins hinzufügen, und es verfügt standardmäßig über die folgenden Effekte:

  • Weichzeichnen - Wendet einen Weichzeichnungseffekt auf das Bild an. Sie können die Intensität sowie die Art des Weichzeichners (Rechteck oder Gauß) einstellen und festlegen, ob nur entlang der horizontalen oder vertikalen Achse oder entlang beider Achsen weichgezeichnet werden soll. Sie können dann wählen, ob Sie nur auf die aktuelle Ebene, auf sichtbare Ebenen oder auf alle Ebenen anwenden.
  • Graustufen - Entsättigt das Bild um einen bestimmten Betrag. Sie können wählen, ob dies nur auf die aktuelle Ebene, sichtbare Ebenen oder alle Ebenen angewendet werden soll.
