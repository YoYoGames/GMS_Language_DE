Objekt-Variablen

Wenn Sie auf die Schaltfläche " Variablen" klicken, öffnet sich das Fenster " Objektvariablen ", in dem Sie vor dem Erstellungsereignis des Objekts eine beliebige Anzahl von Variablen erzeugen können:

In diesem Fenster können Sie auf die Schaltfläche Hinzufügen klicken, um eine neue Variable in die Liste aufzunehmen. Sie können die Variable dann benennen Variable benennen sowie ihren Typ festlegen und ihre Eigenschaften anpassen:

Der Variablenname muss mit einem Buchstaben beginnen und darf nur Buchstaben, Zahlen und das Unterstrichsymbol '_' enthalten, mit einer maximalen Länge von 64 Zeichen. Gültige Variablen sind also Dinge wie fish, foo_bar, num1, und ungültige Variablen wären 6fish, foo bar , oder *num. Nachdem Sie den Namen der Variable angegeben haben, müssen Sie ihren Typ festlegen, der einer der folgenden sein kann: Es ist erwähnenswert, dass diese Funktion besonders nützlich ist, wenn Sie mit Parent/Child-Instanzen arbeiten - da Sie ein Parent-Objekt mit einem Satz definierter Variablen haben und diese dann einfach im Child-Objekt ändern können (siehe den Abschnitt Objektvariablen und Child-Objekte weiter unten für weitere Details) - und wenn Sie Instanzen in den Raum-Editor hinzufügen (siehe den Abschnitt über Ebenen > Instanz-Ebene), da Sie diese Variablen auch für einzelne Instanzen ändern können die im Raum platziert wurden.

Die verschiedenen Variablentypen, die Sie erstellen können, sind unten aufgeführt:

  • Real: Eine reelle Zahl ist jede Zahl, die keine ganze Zahl ist und positiv oder negativ sein kann. Also, 124.5, 45639.566546456, 0.9, -45.5 usw... sind alles Beispiele für reelle Zahlen. Alle reellen Zahlen werden als 64-Bit-Fließkommazahlen (Doubles) gespeichert. Die reellen (und ganzzahligen) Datentypen Typen haben eine Option zum Einstellen eines Wertebereichs, die durch Klicken auf die Schaltfläche " Optionen " aufgerufen werden kann:Wenn Sie dies aktivieren, können Sie einen Start- und einen Endwert eingeben, und anstelle eines festen Werts für die Variable im Fenster "Objektvariablen" wird ein Schieberegler angezeigt, wird ein Schieberegler angezeigt, der an diese Werte geklammert ist:
  • Ganzzahl: Eine ganze Zahl ist eine ganze Zahl und kann positiv oder negativ sein, zum Beispiel 30004, 19, 0, -300. Beachten Sie, dass Sie einen Wertebereich für die Ausgangs-Ganzzahl festlegen können, wie oben für reelle Zahlen erklärt.
  • Zeichenkette: Ein String ist alles, was in doppelte Anführungszeichen gesetzt wurde, z. B. "fish", "Hello World" oder "12345".
  • Boolesch: Ein Boolescher Wert ist ein Wert, der entweder wahr oder falsch ist. Im Variablenfenster des Objekteditors wird dies einfach als ein Kästchen angezeigt, das Sie für wahr aktivieren und für falsch deaktivieren.
  • Ausdruck: Ein Ausdruck ist eine mathematische Phrase, die normale Zahlen, Variablen, Strings oder GML-Funktionen sowie einen oder mehrere Operatoren enthalten kann. Zum Beispiel ist sqrt(85 * 6) + 5.5 ein Ausdruck.
  • Ressource: Eine Ressource ist einfach eine der Ressourcen, die Sie im Asset-Browser definiert haben. Wenn Sie diesen Ressourcentyp auswählen, können Sie anschließend auf die Schaltfläche Asset Explorer öffnen klicken, um den Asset Explorer zu öffnen und die gewünschte Ressource auszuwählen.
  • Liste: Die Auswahl eines Listeneingangstyps bedeutet, dass Sie dann eine Auswahl von Werten erstellen können (dies können Strings, Reals, Ausdrücke usw. sein), von denen Sie dann einen oder mehrere für die Variable auswählen können, die sie zurückgeben soll.   • Farbe: Mit dem Farbtyp können Sie einen Farbwert definieren, der in der Variablen gespeichert werden soll. Sie können eine reelle Zahl (von 0 bis 16777216, diese hat einen Alpha-Wert von 255) oder einen Hex-Wert (im Format $RRGGBBAA) eingeben, oder Sie können können auf auf das Farbfeld doppelklicken, um den Farbwähler zu öffnen und die Farbe dort zu definieren.

Objektvariablen und untergeordnete Objekte

Ein wichtiges Merkmal der Objektvariablen ist, dass sie an alle untergeordneten Objekte im Asset-Browser vererbt werden (weitere Informationen finden Sie im Abschnitt über übergeordnete Objekte ), was bedeutet, dass Sie Das bedeutet, dass Sie einzelne oder alle Variablen überschreiben oder ändern können, wenn Sie dies wünschen.

Wenn Sie ein untergeordnetes Objekt einer anderen Objektressource erstellen, für die Variablen definiert sind, werden diese im Fenster "Objektvariablen" wie folgt angezeigt:

Im oberen Bild können Sie sehen, dass das Parent-Objekt vier Objektvariablen hat, und im unteren Bild sind diese werden diese ebenfalls angezeigt, nur "ausgegraut" und mit dem Symbol Vererbt von Parent , um anzuzeigen, dass sie vererbt wurden. Diese geerbten Variablen können dann bearbeitet werden, wenn Sie auf die Schaltfläche Override Variable klicken, so dass Sie dann den Bereichsschieberegler oder Werte ändern, oder verschiedene Elemente aus Listen auswählen usw.. Beachten Sie, dass Sie beim Bearbeiten einer übergeordneten Variable nur den definierten Wert, nicht aber den Namen und auch nicht die Variablenoptionen ändern können. Beachten Sie auch, dass Sie neue Objektvariablen auch zu untergeordneten Objekten hinzufügen können (und im obigen Beispielbild sehen Sie, dass "Zeichentyp" eine neue Objektvariable nur für das untergeordnete Objekt ist).

Wenn Sie eine geerbte Objektvariable bearbeitet haben, können Sie auf die Schaltfläche Löschen klicken, um die Bearbeitung zu entfernen, aber nicht die Variable. Die Variable wird wird weiterhin vom übergeordneten Objekt geerbt, verwendet aber wieder den Standardwert des übergeordneten Objekts. Wenn Sie die Variable vollständig entfernen möchten, müssen Sie dies über das übergeordnete Objekt tun.
