Code-Schnipsel

Ein sehr praktisches Werkzeug, das Ihnen beim Bearbeiten Ihres Codes zur Verfügung steht, ist die Verwendung von Code-Snippets. Wenn Sie den Skript-Editor verwenden, können Sie die Taste F4 drücken, um das Code-Snippet-Popup zu öffnen, mit dem Sie eine der häufig verwendeten Code-Methoden aus einer Liste auswählen können:

In diesem Fenster können Sie mit der Maus das zu verwendende Snippet auswählen oder den zugehörigen Hotkey drücken, der rechts aufgeführt ist. Dies fügt das Snippet zur Bearbeitung in Ihren Code ein.

Sie können auch eigene Codeschnipsel definieren, wenn Sie das möchten. Bevor Sie dies tun, müssen Sie eine Datei mit dem Namen "snippets.txt" in einem der folgenden Verzeichnisse erstellen:

  • Windows: %ProgramData%\GameMakerStudio2\User\
  • macOS: /Users/Shared/GameMakerStudio2/Benutzer/

Dieser Ordner wird bei einem Update auf GameMaker Studio 2 nicht verändert, so dass Ihre bearbeitete Datei intakt bleibt (aber alle Änderungen an der Basis-Snippets-Datei im Installationsverzeichnis werden rückgängig gemacht).

Sobald die Datei im Benutzerverzeichnis erstellt ist, können Sie sie mit einem beliebigen Texteditor bearbeiten, indem Sie diese Regeln befolgen:

  • Jedes Snippet muss in einer eigenen Zeile stehen (es sollten keine Leerzeilen vorhanden sein)
  • Jedes Snippet muss mit dem zu verwendenden Hotkey beginnen, gefolgt von "-" und dann dem Namen des Snippets (das ist das, was im Menü angezeigt wird), gefolgt von einem Doppelpunkt ":", z. B:

I - Instanz erstellen:

Nach dem Doppelpunkt fügen Sie den Codeschnipsel

I - Instanz erzeugen:instance_create_layer(x, y, |layer|, object);

Der Code, den Sie hinzufügen, muss außerdem einem bestimmten Format folgen, wobei:

  • Der gesamte Code steht in einer einzigen Zeile
  • Sie verwenden "$" für einen Zeilenumbruch
  • Sie setzen den Text, der ausgewählt werden soll, zuerst zwischen die Symbole "|" (der Cursor wählt diesen Text zur Bearbeitung aus, wenn das Snippet hinzugefügt wird)

Sehen Sie sich die Beispiele an, die bereits in der Basisdatei "Snippets" enthalten sind, um zu sehen, wie sie nach den oben genannten Regeln aufgebaut ist. Sie finden die Basisdatei im GameMaker Studio 2-Installationsverzeichnis:

  • \GameMaker Studio 2\TextEditor\snippets.txt
