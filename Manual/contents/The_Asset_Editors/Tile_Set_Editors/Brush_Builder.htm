Der Kachelsatz-Bürstengenerator

Standardmäßig wählen Sie beim "Malen" von Kacheln auf eine Kachel-Map-Ebene im Raumeditor eine einzelne Kachel aus und malen mit dieser. Kachelsätze sind jedoch fast immer so konzipiert, dass sie Abschnitte haben die auf unterschiedliche Weise zusammenpassen, um ganze Bereiche oder Objekte zu bilden. Ein RPG-Kachelsatz kann z. B. Landschaftskacheln enthalten, die je nach Anzahl der verwendeten Kacheln zu größeren oder kleineren Objekten - wie Bäumen und Gebäuden - verbunden werden können. verwendet werden. Würden Sie nun mehrere solcher Features auf einer Raumebene platzieren, müssten Sie viele Male hin- und hergehen, um die Kacheln zu wechseln, was nicht gut für Ihren Arbeitsablauf ist. Um dieses Problem zu lösen, ermöglicht GameMaker Studio 2 das Erstellen von Kachelpinseln im dem Kachel-Set-Editor zu erstellen, der verfügbar ist, wenn Sie auf auf die Schaltfläche " Brush Builder " klicken:

Im Brush Builder haben Sie auf der linken Seite das ursprüngliche Kachelset und auf der rechten Seite eine leere "Leinwand". Sie können nun eine Sie können nun eine beliebige Kachel auf der linken Seite auswählen und sie auf der rechten Seite malen, um benutzerdefinierte "Pinsel" zu erstellen, die Sie dann im Raumeditor verwenden können. Beachten Sie, dass Sie die linke Maustaste gedrückt halten und dann auf das Kachelset ziehen können, um mehrere Kacheln auszuwählen, die Sie in die Pinsel-Leinwand malen möchten. Sie können auch / + gedrückt halten, um bestimmte einzelne Kacheln hinzuzufügen oder + gedrückt halten, um bestimmte Kacheln zu entfernen.

Während Sie Ihre Pinsel erstellen, malen Sie mit der linken Maustaste und löschen mit der rechten Maustaste . Oben rechts sehen Sie das aktuell ausgewählte Werkzeug, und Sie können auch die Größe des Pinsels einstellen, mit dem Sie malen möchten. Die Standardgröße ist 1, was einer einzelnen Kachel entspricht, aber wenn Sie sie auf höhere Werte einstellen, können Sie mit einem größeren Pinsel malen (und löschen), der aus der ausgewählten Kachel besteht, die wiederholt wird. Wenn Sie jedoch einen höheren Wert einstellen, können Sie mit einem größeren Pinsel malen (und löschen), der aus der Wiederholung der ausgewählten Kachel besteht, wie in der Abbildung unten gezeigt:

Bleistift Dies ist das Bleistift-Werkzeug. Es verwendet die ausgewählte Kachel, um im Pinsel-Fenster mit der linken Maustaste zu malen, und Sie können mit der rechten Maustaste löschen.
Radiergummi Mit dem Radiergummi-Werkzeug können Sie die linke Maustaste verwenden, um eine bestimmte Kachel im Pinselfenster zu löschen. Im Wesentlichen wird dabei nur der Kachelindex auf 0 gesetzt, was die reservierte "leere" Kachel.
Auswahl Dies ist das Auswahlwerkzeug, mit dem Sie einen Bereich des Pinselfensters für die Bearbeitung festlegen können. Sie können mit der linken Maustaste klicken und dann die Maus ziehen um einen rechteckigen Bereich zu erstellen, oder Sie können / + drücken, um mehrere Auswahlen hinzuzufügen, und + , um einen Teil der Auswahl zu löschen. Um die gesamte Auswahl zu löschen, können Sie drücken. Wenn Sie einen Bereich des Pinselfensters ausgewählt haben, funktionieren die übrigen Werkzeuge (Stiften, Spiegeln, Drehen usw.) nur innerhalb des ausgewählten Bereichs. Beachten Sie, dass Sie auch Kacheln im Fenster auswählen und dann kopieren ( / + "C"), ausschneiden ( / + "X") und einfügen können ( / + " V"), wodurch das Werkzeug auf den Bleistift umgeschaltet wird und Sie damit "malen" können.
AutoTile Wenn Sie auf dieses Werkzeug klicken, wird der Malstil "Autokacheln" aktiviert. Wenn dies aktiv ist, können Sie eine beliebige Kachel aus der Registerkarte "Autokacheln" auswählen und sie dann in das Pinsel-Fenster malen, und GameMaker Studio 2 passt sie automatisch an die umgebenden Kacheln anpassen, sofern Sie die Registerkarte "Autokacheln" korrekt eingerichtet haben.
