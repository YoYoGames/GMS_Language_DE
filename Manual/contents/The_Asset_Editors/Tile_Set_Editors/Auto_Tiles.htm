Auto-Kacheln

Die Funktion " Auto-Kacheln " ist ein unglaublich leistungsfähiges Werkzeug, das die Erstellung von Ebenen erheblich vereinfacht, egal ob es sich um ein Projekt in der Draufsicht oder in der Seitenansicht handelt. Im Wesentlichen erstellen Sie eine automatische Kachelbibliothek, und wenn Sie dann Wenn Sie dann eine Kachel aus dieser Bibliothek verlegen, "verbindet" sie sich mit den umliegenden Kacheln und bildet eine nahtlose Wand oder Plattform.

Bevor Sie jedoch mit der Auto-Kachel-Funktion beginnen, ist es sehr wichtig, dass Sie das Kachelsatz-Sprite korrekt eingerichtet haben und dass Sie einen geeigneten Typ von Auto-Kacheln wählen. Sie können zwischen folgenden Typen wählen: 47-Kachel-Autokacheln oder 16-Kachel-Autokacheln , wobei (im Allgemeinen) 16-Kacheln für Top-Down- oder Landschaftsansichten verwendet werden (da sie schönere Übergänge ergeben) und 47-Kacheln für Platformer/Seitenansichten oder Top-Down-Ansichten, bei denen Wände mehrere Verbindungen erfordern, aber das aber das ist keineswegs eine feste Regel und hängt davon ab, wie Ihr endgültiges Projekt aussehen soll.

Beachten Sie, dass das verwendete Kachelset-Sprite ALLE benötigten Kacheln hat und auch andere Kacheln haben kann, da Sie auswählen können welche Kacheln in der Auto-Kachel sein sollen. Sie können jetzt auf auf die Schaltfläche "47 Kacheln" im Bereich "Vorlagen" klicken, um einen neuen Bibliothekseintrag mit 47 Kacheln zu erstellen, und es wird Sie können nun auf die Schaltfläche "47 Kacheln" im Bereich "Vorlagen" klicken, um einen neuen Eintrag für die Bibliothek mit 47 Kacheln zu erstellen:

In der Vorlage stellt der hellgraue Bereich die Außenkante der verwendeten Kachel dar und das dunklere Grau ist der "Füllbereich". Um eine Kachel in die Schablone einzufügen, klicken Sie einfach auf die erste verfügbare Schablonenzelle und wählen dann die hinzuzufügende Kachel von links aus:

Wenn Sie einen Fehler machen, wählen Sie einfach die Autokachel, die Sie entfernen möchten, und wählen Sie die "leere" Kachel aus dem Kachelsatz. Nachdem Sie alle Kacheln hinzugefügt haben, können Sie die Kacheln mit der Vorlage abgleichen, indem Sie oben auf die Schaltfläche "Vorlage anzeigen" klicken. Dadurch wird das Overlay der Vorlage auf der endgültigen Autokachel eingeblendet, wie in der Abbildung unten gezeigt:

Wenn Sie ein Autotile mit 16 Kacheln erstellen möchten, ist die Vorgehensweise genau wie oben beschrieben, nur dass Sie statt der der 47-Kacheln-Vorlage verwenden Sie die 16-Kacheln-Vorlage:

Wir haben jetzt einige Auto-Kacheln in unserer Auto-Kachelbibliothek, die aus einem Kachelsatz stammen. Diese können im Raumeditor verwendet werden, indem Sie einfach eine Kachel Map-Ebene erstellen und dann eine der Auto-Kacheln aus der Bibliothek auswählen (die automatisch den Auto-Kachel-Pinsel oben im Raumeditor auswählt) und in Ihrem Raum malen. Die Kacheln verbinden sich nun "auto-magisch" und erzeugen richtig aufeinanderfolgende Kachel-Maps.

Bitte beachten Sie, dass das Verhalten einer Autokachel entlang der Raumkanten von der Schaltfläche Offene oder geschlossene Kanten abhängt. Wenn eine Autofliese entlang der Kante eines Raums platziert wird, wählt sie standardmäßig eine Fliese aus, als ob die Außenseite des Raums leer wäre (d. h: Es wird dem Raum eine "Kante" geben). Wenn Sie jedoch auf diese Schaltfläche klicken, wird die Autokachel den Raum so kacheln, als ob der Bereich außerhalb mit Kacheln gefüllt wäre und somit Kacheln wählen, die sich vermischen und keinen "Rand" haben.
