Animierte Kacheln

Kacheln werden im Allgemeinen als statische Zellen innerhalb des Spielraums betrachtet, aber mit GameMaker Studio 2 ist es möglich, sie zu animieren, genau wie Sie es mit einem Sprite tun würden. Warum nicht einfach Sprites verwenden, fragen Sie? Nun, Sprites haben einen gewissen Verarbeitungsaufwand, da sie da sie mit einer Bounding Box verbunden sind und die Art und Weise, wie sie gerendert werden, aber Kacheln haben einen viel geringeren Verarbeitungsaufwand und sind daher schneller zu rendern. Dies hat jedoch den Nachteil, dass Sie mit Kachelsatz-Animationen auf Folgendes beschränkt sind Animationen zu erstellen, die eine Potenz von zwei Frames lang sein müssen - d. h. 2, 4, 8, 16 usw. Frames - und sie müssen natürlich innerhalb des Kachelrasters erstellt werden, und sie müssen alle mit der gleichen Geschwindigkeit animiert werden (auf einer Kachel-Set-Basis). Aber selbst Auch mit diesen Einschränkungen sind animierte Kacheln eine leistungsstarke Funktion, die verwendet werden kann, um ansonsten statischen Umgebungen und Hintergründen Leben und Farbe zu verleihen.

Um eine Kachelanimation zu erstellen, müssen Sie zunächst ein Kachelset haben, das die benötigten Kacheln alle innerhalb eines Sprite-Bildes hat (aber nicht als Sprite-Frames, also ein einziges großes Sprite-Frame mit allen Animationsbildern darin), und dann klicken Sie im Kachelset-Editor auf die Schaltfläche Kachelanimation, um den Animationseditor aufzurufen:

Das obige Beispielbild ist ein einzelnes Bild mit allen darin enthaltenen Animationsframes, aber es ist zu beachten, dass das Bild auch andere Nicht-Animationskacheln enthalten könnte. auch andere Nicht-Animationskacheln enthalten kann, da Sie einen einzigen großen Kachelsatz verwenden und dann die Teile davon "herauspicken" können, aus denen Sie die Animation erstellen möchten. Sie könnten zum Beispiel ein großes Landschaftskachelset mit Bäumen, Erde und Wasser haben, Erde und Wasser, und innerhalb des Kachelsatzes mehrere Kacheln haben, die animiert werden sollen, um das Wasser plätschern zu lassen.

Um eine Kachelanimation zu erstellen, müssen Sie zunächst eine Animation zur Animationsbibliothek hinzufügen, indem Sie auf die Schaltfläche Animation hinzufügen klicken. einen leeren Animationsstreifen zur Bibliothek hinzufügt und Sie auffordert, eine Anzahl von Bildern auszuwählen, die verwendet werden sollen:

Sie erhalten dann eine Ansicht jedes leeren Animationsrahmens, wobei der Anfangsrahmen für Sie hervorgehoben ist:

Wenn Sie dann zum Kachelsatzbild auf der linken Seite des Editors zurückgehen, können Sie auf auf eine beliebige Kachel klicken, um sie in die Animation einzufügen, und das Bild wird zum nächsten weitergeschaltet. Wenn Sie auf auf aufeinanderfolgende Kacheln klicken, wird die Animation ausgefüllt. Animation, die Sie dann durch Klicken auf die "Play"-Schaltfläche neben den Frames in der Vorschau anzeigen können:

Sobald Sie das getan haben, wird Ihre Animation Teil der Animationsbibliothek und wenn Sie den Raumeditor aufrufen, können Sie sie auswählen und im Raum platzieren. Beachten Sie, dass, wenn Sie eine der Kacheln aus einer Animation auswählen, um sie in einem Raum zu platzieren, diese Kachel ab der ausgewählten Kachel animiert wird, so dass Sie eine Animation "desynchronisieren" können, indem Sie separate Rahmen innerhalb des Raums platzieren und sie werden alle animiert. Der einzige Nachteil dabei ist, dass die Animationsgeschwindigkeit immer die gleiche ist, da sie für den gesamten Kachelsatz und nicht für einzelne Animationen innerhalb der Bibliothek definiert ist.

Ein paar Dinge sind bei Kacheln, die in Animationen verwendet werden, zu beachten:

  • Wenn für die Kachel eine eigene Animation festgelegt wurde (es also eine Animation gibt, bei der sie als erste Kachel festgelegt ist), dann wird sie diese Animation verwenden.
  • Wenn die Kachel keine eigene Animation hat, sondern nur in einer anderen Animation verwendet wird, dann wird diese Animation ab der ersten Position verwendet, an der die Kachel in der Animation vorhanden ist.
  • Wenn die Kachel keine eigene Animation hat und in mehreren anderen Animationen verwendet wird, dann wird sie überhaupt nicht animiert. Sie müssen dann explizit eine Animation mit ihr als erste Kachel angeben.
