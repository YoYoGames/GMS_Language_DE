Der Zeitleisten-Editor

In den meisten Spielen möchten Sie, dass bestimmte Dinge zu bestimmten Zeitpunkten passieren. Sie können versuchen, dies mit Hilfe der Alarmereignisse zu erreichen, aber aber wenn die Dinge zu kompliziert werden, wird das nicht mehr funktionieren, zumal Sie auf nur zwölf Alarme beschränkt sind. Deshalb gibt es das Zeitleisten-Asset. In einer Zeitleiste legen Sie fest, welche Aktionen zu bestimmten Zeitpunkten im Spiel passieren müssen. Spielzeit passieren müssen, und Sie können dazu alle Aktionen verwenden, die auch für ein Objekt in seinen verschiedenen Ereignissen oder im Code zur Verfügung stehen. Sobald Sie eine Zeitleiste erstellt haben, können Sie diese einem Objekt zuweisen, und die Instanz dieses Objekts führt dann die Aktionen zu den angegebenen Zeitpunkten aus, wenn sie in einem Raum platziert oder erzeugt wird.

Nehmen Sie z. B. an, dass Sie einen Wächter erstellen möchten und dass sich dieser Wächter 20 Zeitschritte nach links, dann 10 nach oben, dann 20 nach rechts, dann 10 nach unten und schließlich anhalten soll. Um dies zu erreichen, können Sie eine Zeitleiste erstellen, in der Sie mit dem Setzen einer Bewegung nach links. Dann erstellen Sie den Zeitpunkt 20 (ein "Zeitpunkt" ist nur eine andere Bezeichnung für "Spielschritt" oder "Spielframe") und setzen eine Bewegung nach oben, dann bei Zeitpunkt 30 eine Bewegung nach rechts, bei Zeitpunkt 50 eine Bewegung nach unten und schließlich bei Moment 60 würden Sie die Bewegung stoppen. Danach können Sie diese Zeitleiste dem Wächterobjekt zuweisen und die Wächterinstanz wird genau das tun, was Sie geplant haben, wenn sie in einem Raum platziert wird. Sie könnten sogar einen Alarm in der Zeitleiste setzen, so dass dass er nach einer bestimmten Zeitspanne wiederholt wird, und auf diese Weise kann Ihr Wächter einen Bereich patrouillieren. Sie können eine Zeitleiste auch verwenden, um Ihr Spiel globaler zu steuern, indem Sie z. B. ein unsichtbares Controller-Objekt erstellen, dann eine Zeitleiste, die zu bestimmten Zeitpunkten Momenten Feinde erzeugt, und diese dann dem Controller-Objekt zuweisen und eine Instanz in den Raum stellen. Dies ist ein einfacher Weg, um zeitgesteuerte Spawns der Feinde in einem Space-Shooter-Spiel oder ähnlichem zu erhalten.

Um eine Zeitleiste zu erstellen, klicken Sie mit der rechten Maustaste auf im Asset-Browser und wählen "Zeitleiste erstellen". Daraufhin öffnet sich das oben auf dieser Seite abgebildete Fenster oben auf dieser Seite mit den folgenden Abschnitten:

Name

Hier geben Sie Ihrer Timeline-Ressource einen Namen. Dieser darf nur alphanumerische Zeichen oder den Unterstrich "_" enthalten und darf nicht mit einer Zahl beginnen, da er zur Referenzierung Ihrer Zeitleiste durch Code verwendet wird. Dieser Name wird ist auch die eindeutige Zeitleisten-ID, die in DnD™-Aktionenoder Codefunktionen verwendet werden kann, um die Zeitleiste zu identifizieren.


Moment

Hier können Sie den Momentwert einstellen. Der Momentwert wird als die Anzahl der Spielschritte nach dem Start der Zeitleiste berechnet. Wenn Sie also eine Zeitleiste mit Momenten an den Positionen 0 und 60 haben, dann wird der Moment 0 ausgelöst, wenn die Zeitleiste gestartet wird und Moment 60 wird 60 Schritte später ausgelöst. Wenn Ihre Raumgeschwindigkeit ebenfalls auf 60 eingestellt ist, dann wird der Moment 60 eine Sekunde nach dem Start der Zeitleiste ausgelöst. Sie können einen Doppelklick auf den Momentwert machen oder Sie können das Menü der rechten Taste verwenden und wählen " ändern" (siehe den Abschnitt über das Tastenmenü unten), um ihn zu ändern. Um ein neues Moment hinzuzufügen, klicken Sie einfach auf die Schaltfläche " Hinzufügen " am unteren Rand des Editors. Durch das Hinzufügen neuer Momente wird der Code-Editor (oder Aktionseditor) geöffnet, in dem Sie die Funktionen oder DnD™hinzufügen können, die Sie in diesem Moment aufrufen möchten.


Moment Name

Damit Sie besser verstehen können, was Ihre Zeitleiste tut, ist es zulässig, dass Sie Momenten einen Namen geben, um sie zu identifizieren. Dabei handelt es sich lediglich um eine Zeichenkette, die weder im Spielcode noch an anderer Stelle verwendet wird, da sie lediglich dazu dient, zu organisieren und zu verstehen, was was Ihre Zeitleiste tut. Um Ihrem Moment einen Namen/eine Beschreibung hinzuzufügen, müssen Sie das Format /// @description in der ersten Zeile des Codeblocks verwenden, zum Beispiel:

/// @Beschreibung Hallo Welt

Im Zeitleisten-Editor können Sie mit + alle Momente zwischen dem ersten Klick und dem zweiten Klick , oder Sie können mit / + einzelne Momente anklicken, um sie unabhängig voneinander auszuwählen. Timeline-Momente können über das Menü der rechten Maustaste auf einem der ausgewählten Elemente bearbeitet werden:

  • Hinzufügen: Mit dieser Option können Sie einen neuen Moment zur Zeitleiste hinzufügen. Er wird direkt nach dem letzten Moment hinzugefügt. Wenn also der letzte Moment in Ihrer Zeitleiste 120 ist, wird der nächste, den Sie hinzufügen, 121 sein. Sie können die Position des Moments ändern ändern, indem Sie auf auf den Momentwert im Editor doppelklicken oder indem Sie mit der rechten Maustaste auf klicken und "Ändern" wählen.
  • Bearbeiten: Damit wird der aktuelle Timeline-Moment zur Bearbeitung geöffnet.
  • Moment ändern: Ändern Sie den Wert für den ausgewählten Timeline-Moment. Dies ist dasselbe wie ein langsamer Doppelklick auf den Timeline-Wert im Editorfenster.
  • Beschreibung ändern: Ändern Sie den beschreibenden Namen für den ausgewählten Timeline-Moment. Dies ist dasselbe wie ein langsamer Doppelklick auf auf die Timeline-Beschreibung im Editorfenster.
  • Duplizieren: Damit werden alle ausgewählten Timeline-Momente dupliziert und nacheinander an das Ende der aktuellen Timeline angehängt.
  • Zusammenführen: Wenn Sie zwei oder mehr Momente in einer Timeline auswählen, können Sie diese zusammenführen. Dadurch wird ein einziger Timeline-Moment an der Position des zuerst ausgewählten Moments mit allen Codes und Aktionen gleichzeitig zusammengeführt.
  • Entfernen: Damit wird der ausgewählte Moment (oder die ausgewählten Momente) aus der Zeitleiste entfernt.
  • Ausschneiden / Kopieren / Einfügen: Mit diesen Optionen können Sie einen oder mehrere Timeline-Momente ausschneiden, kopieren oder einfügen. Sie können auch hier die Standard-Tastaturkürzel verwenden: / + "X", / + "C" und / + "V".
  • Alles auswählen: Mit dieser Option können Sie alle Momente in einer Timeline auswählen. Ausgewählte Momente können dann entfernt, zusammengeführt, dupliziert, ausgeschnitten oder kopiert werden.
  • In Drag and Drop/ Code konvertieren: Timeline-Momente werden mit der Methode erstellt, die Sie beim Erstellen des Projekts gewählt haben, entweder mit Code oder mit Drag and Drop™-Aktionen. Mit dieser Option können Sie von der einen zur anderen Methode konvertieren, unabhängig des Projekttyps.
