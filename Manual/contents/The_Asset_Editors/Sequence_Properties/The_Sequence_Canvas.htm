Die Sequenz-Leinwand

Beim Erstellen von Sequenzen für Ihre Projekte wird ein großer Teil der Arbeit im Canvas des Sequenzeditors erledigt. Der Canvas ist einfach der Bereich, der verwendet wird, um die Sequenzelemente zu enthalten, und Er definiert auch, was angezeigt wird, wenn die Sequenz zu einem Raum hinzugefügt wird, der angezeigt werden soll. Wenn Sie zum ersten Mal eine Sequenz erstellen, wird die Leinwand auf die Standardgröße von 1366 x 768 Pixel eingestellt, aber dies kann in der Toolbox geändert werden, indem Sie Klicken Sie in der Toolbox auf die Schaltfläche neben der Schaltfläche Canvas Frame Toggle :

Die Leinwandgröße definiert einen visuellen Bereich, in dem die Elemente, aus denen die Sequenz besteht, platziert werden, aber der Rahmen ist Der Rahmen ist jedoch nur ein Richtwert für die Größe und hat keinen Einfluss darauf, wie Dinge gerendert werden, d. h.: Wenn Sie ein Sprite-Asset außerhalb des Rahmens platzieren und die Sequenz zu einem Raum hinzufügen, wird das Asset trotzdem gerendert.

Der Rahmen wird auch zur Aufnahme von Referenzbildern verwendet, die Sie zur Steuerung der Positionierung von Assets innerhalb der Sequenz verwenden möchten. Ein Referenzbild kann über dasselbe Menü " Leinwandoptionen " hinzugefügt werden, in dem Sie auch die Größe des Leinwandrahmens festlegen. Ein Referenzbild kann über dasselbe Menü "Canvas Options" hinzugefügt werden, in dem Sie auch die Größe des Leinwandrahmens einstellen, und wird dann als Hintergrundbild für die Sequenzleinwand verwendet. Beachten Sie, dass Referenzbilder nicht mit dem Projekt gespeichert werden und nicht gerendert werden, wenn die Sequenz zu einem Raum hinzugefügt wird.

Sobald Sie die Leinwand wie gewünscht eingerichtet haben, können Sie damit beginnen, ihr Assets hinzuzufügen. Dies geschieht einfach durch Ziehen aus dem Asset-Browser in die Leinwand, wodurch eine Asset-Spur im Spurbedienfeld erstellt wird:

Die Assets, die einer Sequenz hinzugefügt werden können, sind wie folgt:

  • Sprites
  • Objekte
  • Klingt
  • Sequenzen

  • Indem Sie das Transformieren-Gizmo am Ursprung des Assets verwenden, das Sie in der Canvas Toolbox ändern können:

Ein letzter erwähnenswerter Punkt ist, dass der Canvas die Verwendung und das Hinzufügen von intelligenten Hilfslinien erlaubt, d. h. Hilfslinien, die dem Canvas hinzugefügt werden können, damit Assets an ihnen ausgerichtet werden können. Standardmäßig zeigt der Canvas horizontale und vertikale Hilfslinien um den Mittelpunkt sowie eine rechteckige Hilfslinie um die "Kante" des Sequenz-Canvas, je nach Größe des Canvas. Diese Hilfslinien werden in der endgültigen Sequenz nicht gerendert, wenn sie zu einem Raum hinzugefügt werden. Sie dienen lediglich dazu, Ihnen eine Vorstellung vom Raum zu vermitteln und die Positionierung von Dingen zu erleichtern. RMB-Menü und Navigation

Um in der Canvas-Ansicht zu navigieren, können Sie die gleichen grundlegenden Steuerelemente wie für einen regulären Arbeitsbereich verwenden, d. h.: Verwenden Sie die mittlere Maustaste und ziehen Sie, um die Leinwand zu verschieben (oder alternativ verwenden Sie + ), und Sie können mit dem Mausrad horizontal scrollen Rad horizontal scrollen oder mit der Taste / und dem Mausrad ein- und auszoomen.

Um eine Kühlstelle auszuwählen, klicken Sie einfach mit der linken Maustaste darauf, oder Sie können mehrere Kühlstellen auswählen, indem Sie / und verwenden, um auf jede Kühlstelle zu klicken. Sie können auch mehrere Kühlstellen auswählen, indem Sie verwenden und dann auf klicken und ziehen. Dies wird ein "Kasten" in der Arbeitsfläche erstellt, und jede Kühlstelle, die den Kastenbereich berührt, wird ausgewählt.

Wenn Sie eine oder mehrere Kühlstellen ausgewählt haben, können Sie diese mit den Tastenkombinationen / +"X", / + "C" und / + "V" ausschneiden, kopieren und einfügen.

Beachten Sie, dass, wenn mehrere Kühlstellen im Canvas ausgewählt sind, für jede Kühlstelle ein Begrenzungsrahmen und für alle zusammen ein größerer Begrenzungsrahmen angezeigt wird. Sie können ändern, wie Sie können das Verhalten dieses Begrenzungsrahmens mit den Symbolen in der oberen linken Ecke ändern:

Wenn Sie auf das Kreissymbol klicken, wird das Transformationssteuerungs-Widget auf den Ursprung des gesamten Begrenzungsrahmens für alle ausgewählten Kühlstellen geändert, Klicken Sie auf das Kastensymbol, um ein individuelles Steuerungs-Widget für jede Kühlstelle in der Auswahl anzuzeigen. Sie können gruppierte Kühlstellen verschieben und sie kopieren/einfügen usw., genau wie bei einer einzelnen Kühlstelle.

Wenn Sie mehrere Assets haben, die sich überlappen, und ein bestimmtes auswählen möchten, ohne die anderen zu stören, können Sie mit der Maus darüber fahren, um die Objekt-Toolbox aufzurufen, die eine Liste von Symbolen für jedes Objekt unter der Maus. Sie können dann auf eines dieser Elemente klicken, um es auszuwählen und zu bearbeiten:

Außerdem stehen Ihnen über die rechte Maustaste eine Reihe zusätzlicher Optionen zur Verfügung, wenn Sie mit ihr auf eine beliebige Kühlstelle in der Sequenz klicken:

Die hier verfügbaren Optionen sind:

  • Aufzeichnen: Mit dieser Option können Sie Parameterschlüssel für das/die Asset(s) aufzeichnen, und wenn Sie den Mauszeiger darüber bewegen, sehen Sie Optionen zum Aufzeichnen der Skala, Drehung oder Position.
  • Gruppe: Wenn Sie diese Option wählen, wird ein neuer Gruppenordner im Spurbedienfeld erstellt und das/die ausgewählte(n) Asset(s) in diesen Ordner verschoben.
  • Umbenennen: Wenn Sie darauf klicken, wird die Spur im Spur-Bedienfeld hervorgehoben und der Name wird editierbar gemacht, so dass Sie ihn ändern können.
  • Im Blatt suchen: Wenn Sie hierauf klicken, wird der Schlüssel der ausgewählten Kühlstelle in der Tabelle hervorgehoben.
  • Andere suchen: Wenn Sie hierauf klicken, wird eine Liste aller anderen Kühlstellen desselben Typs angezeigt, aus der Sie eine auswählen können.
  • Auswählen: Wenn Sie hierauf klicken, wird eine Liste aller Kühlstellen angezeigt, die sich an der angeklickten Position überlappen, und Sie können eine beliebige davon zur Bearbeitung auswählen.
  • Schlüssel entfernen: Wenn Sie diese Option wählen, wird der Asset-Spurschlüssel an der aktuellen Abspielkopfposition aus dem Abspielplan entfernt. Die Spur selbst bleibt bestehen, ebenso wie alle Parameterspuren, die sie enthält, und Sie können der Spur ein anderes Asset wieder hinzufügen zu der Spur hinzufügen, indem Sie es aus dem Asset-Browser in das Dope-Sheet ziehen.
  • Spur entfernen: Dies entfernt die Asset-Spur (und ggf. ihr zugewiesene Parameterspuren) aus dem Spurbereich sowie alle mit ihr verbundenen Tasten im Dope Sheet.
  • Editor öffnen: Damit wird das Asset zur Bearbeitung im Standard-Editor für den Asset-Typ geöffnet.
