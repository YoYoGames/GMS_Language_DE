Clipping-Masken

Wie auf der Seite " Spur-Panel" erwähnt, gibt es eine Aktion zum Erstellen einer so genannten Schnittmaskengruppe im Track Editor. Dazu klicken Sie auf die Schaltfläche am unteren Rand des Spurpanels:

Wenn Sie hierauf klicken, wird eine Schnittmaskengruppe erstellt, die aus zwei Untergruppen von Assets besteht, die entweder als das Subjekt oder die Maske der Gruppe. Im Grunde ist das Motiv das Asset, das maskiert werden soll, und die Maske ist das Asset, das die Maskierung vornimmt:

Die Idee hinter der Verwendung einer Clipping-Maske ist, dass Sie ein Motiv-Sprite und dann ein Masken-Sprite hinzufügen, und das Masken-Sprite "versteckt" (maskiert) (Maske) das Motiv. Sie können dann die Animation des Masken-Sprites verwenden oder das Masken-Sprite verschieben oder skalieren, um die Sichtbarkeit des Motivs (oder von Teilen des Motivs) zu ändern. Jede Gruppe (Maske und Motiv) kann mehrere Elemente enthalten, und jedes Element kann mehrere Parameterspuren haben.

Um ein Motiv und eine Maske hinzuzufügen, ziehen Sie einfach das Sprite, Objekt oder die Sequenz aus dem Asset-Browser über den Ordner, dem Sie es hinzufügen möchten:

Das Masken-Sprite Alpha wird verwendet, um das darunter liegende Objekt zu "verstecken", wobei ein Wert von 100 % Alpha (undurchsichtig) es vollständig maskiert und ein Wert von 0 % Alpha (transparent) es sichtbar macht. Zum Beispiel haben wir dieses Sprite für die Maskierung:

Im obigen Beispiel ist "Sprite_2" einfach ein statisches Sprite-Asset, das das Basissymbol darstellt, und darüber haben wir eine Schnittmaskengruppe mit einer Graustufenversion des Sprites sowie einer Maske (unter Verwendung des oben gezeigten Beispiel-Sprites) erstellt. Wenn sie kombiniert und die Sequenz abgespielt wird, enthüllt die Clipping-Masken-Gruppe das darunter liegende Basis-Sprite darunter, wodurch der Abkling-Effekt entsteht.

Beachten Sie, dass Schnittmaskengruppen als ein einzelnes Element in der Spurliste und im Dope Sheet behandelt werden können, und Sie können der Gruppe als Ganzes Rotations- und Positionsparameterspuren hinzufügen. Sie können Beschneidungsmaskengruppen auch auf Gruppenordner von Assets oder sogar auf andere Schnittmaskengruppen anwenden, was sie zu einem sehr leistungsfähigen Werkzeug macht.
