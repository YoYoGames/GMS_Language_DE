Verwenden des Dope Sheets

Die Dope-Blatt werden die Daten der verschiedenen zur Sequenz hinzugefügten Spuren im Zeitverlauf angezeigt, so dass Sie nicht nur einen Überblick über die einzelnen Spuren, sondern auch die Möglichkeit, die Position der Spuren sowie die Zeitpunkte für Parameteränderungen usw. zu bearbeiten.

Wenn Sie ein neues Asset zu einer Sequenz hinzufügen, wird eine Asset-Spur im Spurbedienfeld erstellt und ein Asset-Schlüssel wird dem an der aktuellen Abspielkopfposition hinzugefügt . Dieser Schlüssel gibt die Abspieldauer des Assets in Frames an:

Der Asset-Schlüssel selbst kann dann bearbeitet werden, indem Sie die Maus entweder an den Anfang oder an das Ende des Schlüsselrahmens bewegen und, wenn sich der Cursor ändert, klicken Sie auf und ziehen Sie. Dadurch wird entweder der Anfang des Schlüssels oder das Ende des Schlüssels verschoben, je nachdem, was Sie ausgewählt haben, aber beide beeinflussen die Länge der Taste. Beachten Sie, dass Sie die Position einer Kühlstellentaste auch ändern können, indem Sie einfach auf klicken und nach links oder rechts ziehen:

Neben den Kühlstellenschlüsseln zeigt das Datenblatt auch Parameterschlüsselan. s. Parameterschlüssel sind zunächst "Punkte" in der Zeit, an denen Sie eine Änderung einer Eigenschaft für die Kühlstelle signalisieren, auf die sie sich beziehen, und die Sequenz verwendet den Wert für diesen Punkt, um die Kühlstelle zu ändern. Wie dies geschieht, hängt davon ab, ob Sie die Tasten im Spurbedienfeld auf lineare Interpolation eingestellt haben oder nicht. Parametertasten können wie Spurtasten verschoben werden, indem Sie auf klicken und sie entlang der Timeline an eine neue Position ziehen, und Sie können die Taste auch auf mehr als ein Einzelbild ausdehnen, indem Sie die Taste gleichzeitig mit dem Ziehen verwenden (der Cursor ändert sich, um dies anzuzeigen). Das bedeutet, dass Sie Schlüsselwerte über die Zeit halten können, bevor Sie mit einer Interpolation zum nächsten Schlüssel fortfahren:

Bei der Arbeit mit dem Dope-Sheet können auch die üblichen Tastaturkürzel verwendet werden, d. h.: / und die linke Maustaste , um eine oder mehrere Spuren in beliebiger Reihenfolge auszuwählen, und die linke Maustaste , um aufeinanderfolgende Spuren auszuwählen, und Sie können sie auch mit den Tastenkombinationen / +"X", / + "C" und / + "V" ausschneiden, kopieren und einfügen.

Über das Menü der rechten Maustaste stehen außerdem verschiedene Optionen für Tasten zur Verfügung, die im Folgenden beschrieben werden. Beachten Sie, dass beim Einfügen von Tasten in das Dope Sheet mit diesen Befehle die Tasten an der aktuellen Position des Abspielkopfs eingefügt werden.

Region Start / Region Ende

Diese Markierungen sind die Start- und Endpunkte der Clip-Region. Sie können verwendet werden, um einen bestimmten Abschnitt der Sequenz festzulegen, der im Raum angezeigt werden soll, wenn das Spiel ausgeführt wird, und sie werden auch als Loop-Punkte bei der Vorschau der Sequenz verwendet im Editor verwendet. Diese Punkte können nicht größer oder kleiner sein als die Gesamtlänge der Sequenz, wie sie in den Sequenzsteuerungen des Sequenzeditors eingestellt ist. Um die Position einer Clip-Regionsmarkierung zu ändern, klicken Sie einfach auf und ziehen Sie sie auf den gewünschten Frame.

Zoom-Steuerungen

Diese Leiste dient zur Anzeige des aktuellen Zooms des Dope-Sheets und kann auch zum Ändern dieser Zoomstufe verwendet werden. Sie können auf auf die Endwidgets klicken und dann ziehen, um die Zoomstufe zu ändern:

Beachten Sie auch, dass Sie mit der Taste / und dem Mausrad das Dopeblatt ein- oder auszoomen können.


Asset-Spuren

Die Asset-Spuren werden auf dem Dope-Sheet als "Keys" dargestellt, d. h. als farbige Balken, die die Position des Assets und die Länge der Zeit, die es in der Sequenz aktiv ist, repräsentieren. Wie oben erwähnt, können diese verschoben und nach Bedarf verlängert und verkürzt werden. und verkürzt werden. Beachten Sie, dass ein Asset-Track-Schlüssel, der länger ist als das Asset selbst - zum Beispiel könnte ein Sound 5 Frames dauern und Sie dehnen seinen Schlüssel auf 15 Frames - für die gesamte Dauer seines Schlüssels wiederholt wird. Asset Spuren und Schlüssel können mit den Standard-Tastaturkürzeln kopiert und eingefügt oder mit den Menüoptionen der rechten Maustaste (weiter unten auf dieser Seite erklärt) weiter verändert werden. dieser Seite).

Parameter Tracks

Parameterspuren werden auf dem Dotierungsblatt als farbige Balken unterhalb der Haupt-Kassenspur dargestellt, zu der sie gehören. Die farbigen Balken selbst können nicht bearbeitet werden und sind eine rein visuelle Darstellung der Spur, auf die die Parameterschlüssel angewendet werden. Die Farbbalken selbst können nicht bearbeitet werden und sind eine rein visuelle Darstellung der Spur, auf die die Parametertasten angewendet werden, und es sind die einzelnen Punkte auf der Parameterspur, die bearbeitet werden (siehe Parameter-Tasten, unten).

Der Abspielkopf

Der Abspielkopf ist die Position in der Zeit, an der die Zeitleiste gerade in der Vorschau angezeigt wird. Diese Position wird beim Ablegen von Assets im Sequenzbildschirm verwendet, um die Position festzulegen, an der sie hinzugefügt wurden, sowie sowie eine Reihe anderer Dinge, wie das Hinzufügen von Parametern oder das Aufteilen von Tasten. Wenn Sie im Sequenzeditor-Canvas auf "Abspielen" drücken, bewegt sich der Abspielkopf durch die Sequenz, und Sie können auch auf klicken und ihn ziehen, um zu einem bestimmten Frame innerhalb der Sequenz zu gelangen oder um über mehrere Frames zu "scrubben" und eine schnelle Vorschau zu erhalten, ohne die ganze Sequenz abzuspielen.


Parameter-Tasten

Parameterschlüssel sind die "Punkte", die auf einer Parameterspur überall dort angezeigt werden, wo eine Änderung dieses Parameters stattgefunden hat. Diese Punkte können nach Bedarf verschoben und erweitert werden, so dass sie mehr als ein Bild umfassen (wie weiter oben auf dieser Seite erklärt), und Sie können ihren Wert entweder über das Spurbedienfeld oder durch Ändern des Assets auf der Leinwand (wenn es sich um einen Skalierungs-, Positions- oder Rotationsparameter handelt) bearbeiten. Beachten Sie, dass alle Spurparameter Diese Punkte können jedoch nicht bearbeitet werden und dienen nur als Informationshilfe, wenn Sie die verschiedenen Parameterspuren im Spurbedienfeld zusammengeklappt haben.


Parameter-Kurven

Sie können in Ihren Parameterspuren auchParemeter-Kurven anstelle von Keyframes verwenden, um die Parameterwerte zu ändern. Diese Kurven können entweder als Teil des Sequenz-Assets eingebettet sein, oder können externe Animationskurven-Assets sein, die auf eine Parameterspur angewendet werden. Weitere Informationen finden Sie im Abschnitt über Verwenden von Animationskurven für weitere Informationen.

RMB-Menü

Wenn Sie mit der rechten Maustaste innerhalb des Dope Sheets auf eine Asset-Spur klicken, werden die folgenden Menüoptionen angezeigt (beachten Sie, dass einige Optionen möglicherweise nicht sichtbar sind abhängig davon, ob der angeklickte Teil der Spur eine Taste hat oder nicht):

  • Taste(n)Ausschneiden, Kopieren und Einfügen: Mit diesen Optionen können Sie ein Asset ausschneiden/kopieren/einfügen. Beachten Sie, dass alle Tasten, die mit diesen Optionen in das Abspielblatt eingefügt werden, an der Position des Abspielkopfes eingefügt werden.
  • Schlüssel an Ort und Stelle hinzufügen: Dies fügt einen neuen Asset-Schlüssel zur Asset-Spur an der Frame-Position hinzu, die der Position des Mausklicks am nächsten liegt, und nicht an der Abspielkopfposition.
  • Taste an Ort und Stelle teilen: Wenn diese Option verwendet wird, wird die Asset-Taste unter der Maus in zwei Tasten geteilt, wobei die Teilung am nächstgelegenen Frame zu der Stelle erfolgt, an der die Maus angeklickt wurde, und nicht an der Abspielkopfposition.
  • Taste(n) an Ort und Stelle einfügen: Wenn Sie diese Option verwenden, werden die kopierte(n) oder ausgeschnittene(n) Taste(n) an der Frame-Position eingefügt, die der Position am nächsten liegt, an der die rechte Maustaste angeklickt wurde, und nicht an der Abspielkopfposition.
  • Taste(n) löschen: Wenn Sie diese Option auswählen, wird die aktuell ausgewählte Taste bzw. werden die ausgewählten Tasten gelöscht.
  • Auf Anfang kopieren: Diese Option kopiert den/die ausgewählten Key(s) an den Anfang der Sequenz(nicht an den Anfang der Clip-Region).
  • Bis zum Ende kopieren: Diese Option kopiert den/die ausgewählten Key(s) bis zum Ende der Sequenz(nicht bis zum Ende der Clip-Region).
  • Asset-Key strecken: Wenn Sie diese Option auswählen, wird der ausgewählte Asset-Key bis zum Ende der Sequenz (nicht bis zum Ende der Clip-Region) oder bis zum Beginn eines anderen Asset-Keys auf derselben Spur gestreckt.
  • Asset-Schlüssel löschen: Wenn Sie dies verwenden, wird der ausgewählte Asset-Schlüssel gelöscht und Sie können dann ein neues Asset zur Spur hinzufügen, indem Sie es aus dem Asset-Browser auf die gelöschte Dope-Sheet-Spur ziehen. Dadurch wird ein neuer Asset Taste, die dann bearbeitet werden kann, wobei alle mit der Kühlstelle verbundenen Parameterspuren erhalten bleiben.

Wenn Sie mit der rechten Maustaste innerhalb des Dope-Sheets auf eine Parameterspur klicken, werden die folgenden Menüoptionen angezeigt (beachten Sie, dass einige Optionen möglicherweise nicht sichtbar sind abhängig davon, ob der angeklickte Teil der Spur einen Keyframe hat oder nicht und ob die Spur Keyframes oder Parameterkurven verwendet):

  • Taste(n)Ausschneiden, Kopieren und Einfügen: Mit diesen Optionen können Sie Parameter-Tasten ausschneiden/kopieren/einfügen. Beachten Sie, dass alle Tasten, die mit diesen Optionen in die Abspielmaske eingefügt werden, an der Position des Abspielkopfs eingefügt werden.
  • Taste an Ort und Stelle hinzufügen: Damit wird eine neue Parametertaste zur Parameterspur an der Frame-Position hinzugefügt, die dem Mausklick am nächsten liegt, und nicht an der Abspielkopfposition.
  • Taste an Ort und Stelle aufteilen: Wenn diese Option verwendet wird, wird die Parametertaste unter der Maus in zwei Keyframes aufgeteilt, wobei die Aufteilung am nächstgelegenen Frame zu der Stelle erfolgt, an der die Maus angeklickt wurde, und nicht an der Abspielkopfposition.
  • Key(s) an Ort und Stelle einfügen: Wenn Sie diese Option verwenden, wird/werden der/die kopierte(n) oder ausgeschnittene(n) Keyframe(s) an der Position eingefügt, an der die rechte Maustaste geklickt wurde, und nicht an der Abspielkopfposition.
  • Taste(n) löschen: Wenn Sie dies auswählen, wird der aktuell ausgewählte Keyframe oder die Keyframes gelöscht.
  • In eingebettete Animationskurve konvertieren: Mit dieser Option werden alle Keyframes für die ausgewählte(n) Parameterspur(en) in eine eingebettete Animationskurve umgewandelt (weitere Informationen finden Sie im Abschnitt über die Verwendung von Animationskurven weitere Informationen). Alternativ dazu wird bei Verwendung auf einer Parameterkurve, die ein Animationskurven-Asset verwendet, das Kurven-Asset als eingebettete Parameterkurve in die Spur kopiert.
  • Auf externe Animationskurve setzen: Mit dieser Option können Sie der Parameterspur ein Animationskurven-Asset zuweisen (weitere Informationen finden Sie im Abschnitt " Animationskurven verwenden ").
  • In Keys konvertieren: Mit dieser Option können Sie eine Animationskurve in einzelne Keyframes umwandeln. Beachten Sie, dass, wenn die Kurve eine Katmull-Rom-Interpolation verwendet hat, diese Wenn die Kurve eine Katmull-Rom-Interpolation verwendet hat, geht diese verloren und die Keyframes verwenden stattdessen eine lineare Interpolation zwischen den Punkten auf der Kurve, und beachten Sie auch, dass aus einer Animationskurve erzeugte Keys auf ein Frame fallen müssen. Das bedeutet, dass jede Kurve mit Punkten, die zwischen Frames liegen, bei der Konvertierung in Schlüssel auf den nächstgelegenen ganzzahligen Frame-Wert gesetzt wird, wodurch sich das Verhalten der Kurve möglicherweise ändert.
  • Eingebettete Animationskurve exportieren: Mit dieser Option können Sie eine eingebettete Animationskurve in ein Animationskurven-Asset umwandeln, das im Asset-Browser angezeigt wird und dann an anderer Stelle verwendet werden kann.
  • Editor öffnen: Dadurch wird der Kurveneditor des Dope-Blattes geöffnet, um eingebettete oder externe Animationskurven zu bearbeiten, die auf die Parameterspur angewendet wurden. Dies ist dasselbe wie die Verwendung der Schaltfläche Kurvenmodus umschalten auf dem Dotierungsblatt.
  • Auf Anfang kopieren: Diese Option kopiert den/die ausgewählten Keyframe(s) an den Anfang der Sequenz(nicht an den Anfang der Clip-Region).
  • Bis zum Ende kopieren: Mit dieser Option werden die ausgewählten Keyframes an das Ende der Sequenz kopiert(nicht an das Ende des Clipbereichs).
  • Parameter Key strecken: Wenn Sie diese Option auswählen, wird der ausgewählte Asset-Key bis zum Ende der Sequenz (nicht bis zum Ende der Clip-Region) oder bis zum nächsten Keyframe auf der Spur gestreckt. Ein gestreckter Parameter-Key bedeutet, dass er dass er seinen Wert für so viele Frames beibehält, wie er gestreckt wird. Wenn sie auf einer Parameterkurve verwendet wird, wird die Kurve bis zum Ende der Sequenz oder bis zum Zusammentreffen mit einer anderen Kurve auf der Spur gestreckt.
  • Parametertaste löschen: Wenn Sie dies verwenden, werden der oder die ausgewählten Parameter-Keyframes gelöscht.
