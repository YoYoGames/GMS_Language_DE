Menüs

Der obere Bereich der GameMaker Studio IDE wird von der Menüleiste eingenommen. Hier finden Sie eine Reihe von Einblendmenüs, mit denen Sie in der IDE navigieren der IDE zu navigieren, Aufgaben auszuführen oder zusätzliche Fenster mit Werkzeugen zu öffnen, die Ihnen bei der Entwicklung Ihrer Spiele helfen. Beachten Sie, dass wir hier die standardmäßig verfügbaren Menüs auflisten, aber einige der Asset-Editoren fügen je nach Kontext zusätzliche Menüs hinzu. Diese zusätzlichen Menüs werden auf den Seiten erklärt, die für den Editor relevant sind, der sie erstellt hat.

Die verfügbaren Standardmenüs sind:

  • Das Menü "Datei
  • Das Menü Bearbeiten
  • Das Menü "Erstellen
  • Das Windows-Menü
  • Das Menü "Werkzeuge
  • Das Marktplatz-Menü
  • Das Menü Layouts
  • Das Hilfe-Menü
