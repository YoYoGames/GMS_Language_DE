Aktuelle Fenster

Die Liste der letzten Fenster kann über das Fenster Menü geöffnet werden und zeigt eine kurze Liste (standardmäßig Sie zeigt eine kurze Liste (standardmäßig mit maximal 10 Einträgen) der Fenster, die Sie zuvor geöffnet haben. Zu Beginn ist diese Liste leer, aber wenn Sie verschiedene Fenster in der IDE öffnen (z. B. ein Sprite oder ein Skript oder ein Objekt usw.), wird die Liste aufgefüllt. Sollte Wenn Sie zu einem dieser zuvor geöffneten Fenster zurückkehren möchten, klicken Sie einfach mit der linken Maustaste auf das Element in der Liste und der Arbeitsbereich schwenkt zu dem Fenster. Wenn Sie mit der rechten Maustaste auf einen Eintrag klicken, öffnet sich das folgende Menü: Hier können Sie einen Eintrag (oder mehrere, da Sie mit + oder / + mehrere auswählen können) aus der Liste löschen oder an den Anfang der Liste verschieben lassen. Das RMB-Menü verfügt auch über eine zusätzliche Option, mit der Sie die Liste komplett löschen können. Es ist erwähnenswert, dass die Recent Windows-Liste an die Seiten der IDE angedockt werden kann und Sie können auch bestimmte Einstellungen vornehmen, um ihr Verhalten zu ändern.
