Das Windows-Menü

Dieses Menü dient zum Öffnen der allgemeinen angedockten IDE-Fenster. Da Docks neu angeordnet werden können und der Inhalt als eigenständige Fenster entfernt oder sogar ganz geschlossen werden kann, erlaubt Ihnen dieses Fenster, sie wieder zu öffnen, wenn Sie feststellen, dass Sie sie für etwas brauchen. Die Fenster, die von hier aus geöffnet werden können, sind das Asset-Browser-Fenster und die verschiedenen Ausgabefenster (Ausgabe, Suche, Source Control, Breakpoints und Compile/Syntax-Fehler).

Als nächstes können Sie das Inspektorfenster öffnen. In diesem Fenster können Sie die Eigenschaften des aktuell ausgewählten IDE-Elements untersuchen und diese Eigenschaften dann nach Bedarf bearbeiten.

Nach der Option "Inspektor" gibt es die Liste " Letzte Fenster" und die Möglichkeit, das Fenster " Umgebungsvariablen" zu öffnen, in dem Sie die verschiedenen Pfade sehen können, auf die die verschiedenen in den Voreinstellungen verwendeten Umgebungsvariablen Variablen, die in den Voreinstellungen verwendet werden, angezeigt werden. Diese Variablen können nicht bearbeitet werden (weder ihr Name noch ihr Pfad/ihre Einstellung), aber Sie können jede von ihnen auswählen und zur Verwendung an anderer Stelle kopieren/einfügen an anderer Stelle.

Schließlich haben Sie die Möglichkeit, Tutorial-Fenster zu öffnen. Dadurch werden alle Fenster wieder geöffnet, die mit einem geöffneten Lernprogramm-Projekt zusammenhängen. die Sie geöffnet haben, aber nur, wenn es sich um ein Projekt handelt, das als Teil eines Tutorials erstellt wurde (wenn nicht, ist diese Option nicht verfügbar). Informationen zum Erstellen eigener Tutorial-Projekte finden Sie im Abschnitt über die Marktplatz (da alle Tutorials dort erstellt, gehostet und heruntergeladen werden).
