Das Marktplatz-Menü

Neben der Möglichkeit, eigene Assets und Code für Ihre Projekte zu erstellen, haben Sie auch Zugriff auf den GameMaker Studio 2 Marketplace, wo Sie sowohl kostenlose als auch kostenpflichtige Assets herunterladen können, um sie Ihren Projekten hinzuzufügen. Mehr über den Marketplace erfahren Sie unter dem Abschnitt über den GameMaker Marktplatzund die Optionen in diesem Menü finden Sie unten:

  • Zum Marktplatz gehen - Öffnen Sie das Marktplatz-Fenster, in dem Sie nach Assets suchen können.
  • Meine Bibliothek - Zeigen Sie Ihre aktuelle Marketplace-Inhaltsbibliothek an.
  • Meine Kaufhistorie - Diese Option öffnet einen Browser und bringt Sie zur Kontoseite für den angemeldeten Benutzer, damit Sie die Kaufhistorie für dieses Konto sehen können.
  • Vorhandenes Paket aktualisieren - Diese Option öffnet den Assistenten zum Aktualisieren von Paketen, damit Sie ein bestehendes Marketplace-Paket, das Sie zuvor erstellt haben, aktualisieren können.
  • Paket erstellen - Mit dieser Option können Sie den Assistenten zum Erstellen von Paketen öffnen, um ein neues Asset-Paket oder Tutorial für den Marketplace zu erstellen.
