Das Menü Bearbeiten

Im Bearbeitungsmenü können Sie die allgemeinen Bearbeitungswerkzeuge der GameMaker Studio 2-Oberfläche verwenden. Die verfügbaren Optionen sind:

  • Rückgängig - Macht eine Aktion rückgängig.
  • Wiederholen - Wiederholt eine vorherige Aktion.
  • Suchen und Ersetzen - Damit wird ein Suchfenster geöffnet, mit dem Sie Ihren gesamten Code (einschließlich Raumerstellungscode, Skripte und Ereignisse) nach einer bestimmten Eingabe durchsuchen können - siehe unten für weitere Informationen.

Wenn Sie im Menü "Bearbeiten" die Option "Suchen & Ersetzen" wählen, wird das folgende Fenster geöffnet:

Damit können Sie alle geskripteten Assets in Ihrem Spiel durchsuchen, einschließlich Raumcode, Ereigniscode, Skripts und Shader. Wenn Sie suchen, können Sie können Sie einen neuen Suchbegriff in das Feld oben eingeben, oder Sie können auf das Pfeilsymbol rechts klicken, um eine Liste der zuvor gesuchten Begriffe zu öffnen. Sie haben auch die Möglichkeit, einen Ersatzbegriff anzugeben, der anstelle des ursprünglichen Suchbegriffs verwendet wird. Begriff ersetzt werden soll, und auch hier gibt es die Möglichkeit, über die Pfeilschaltfläche rechts eine zuvor verwendete Ersetzungszeichenfolge zu verwenden.

Die Suche kann allgemein sein oder Sie können sie mit den Optionen unter den Suchbegriffeingaben verfeinern:

  • Sie können die Groß-/Kleinschreibung bei der Suche berücksichtigen. In diesem Fall werden nur Wörter einbezogen, die genau mit der Groß-/Kleinschreibung des Suchstrings übereinstimmen, d. h.: Wenn Sie ohne diese Option nach "My_Var" suchen, erhalten Sie Ergebnisse mit "my_var", "mY_VAR" oder "My_Var", aber wenn diese Option aktiviert ist, wird nur "My_Var" in die Suchergebnisse aufgenommen.
  • In diesem Fall werden alle Ergebnisse, die den Suchbegriff enthalten, aber keine exakte Übereinstimmung sind, ignoriert, d.h.: eine Suche nach "game" ohne diese Option wird Dinge wie die Funktion "game_end()" oder Variablen wie "game_state" anzeigen, während eine Suche nach "game" mit dieser Option nur Ergebnisse zeigt, die "game" selbst sind.
  • Sie können wählen, ob die Suche Kommentare ignorieren soll. Wenn dies aktiviert ist, werden alle Kommentare in Ihrem Code, die den Suchbegriff enthalten, einfach ignoriert und nicht in den Ausgabeergebnissen angezeigt.
  • Sie können einen Filter auf die Suchergebnisse anwenden, der auf den verschiedenen Asset-Typen im Asset-Browser basiert. Standardmäßig ist dies auf "Alles" eingestellt, aber wenn Sie darauf klicken, öffnet sich das folgende Menü, in dem Sie Folgendes an- oder abwählen können Assets, die in die Suche einbezogen werden sollen:

Nachdem Sie einen Suchbegriff (und optional einen Ersetzungsbegriff) eingegeben und die Suchoptionen eingerichtet haben, können Sie mit den verschiedenen Schaltflächen am unteren Rand wählen, wie die Suche durchgeführt werden soll und welche Aktionen durchgeführt werden sollen. Diese Schaltflächen sind:

  • Alle suchen: Findet alle Suchbegriffe und zeigt sie im Ausgabefenster an.
  • Vorherige suchen: Sucht die vorherige Instanz des Suchbegriffs. Dadurch wird das Asset, das den Verweis enthält, im Arbeitsbereich geöffnet (wenn es nicht bereits geöffnet ist) und der Schwenk dorthin ausgeführt.
  • Weiter suchen: Sucht die nächste Instanz des Suchbegriffs. Dadurch wird das Asset, das den Verweis enthält, im Arbeitsbereich geöffnet (wenn es nicht bereits geöffnet ist) und der Schwenk dorthin ausgeführt.
  • Alle ersetzen: Dies führt eine Suche nach dem angegebenen Begriff mit den eingestellten Optionen durch und ersetzt ihn auch in den Assets durch den neuen Ersetzungsbegriff, den Sie angegeben haben.
  • Ersetzen: Damit wird der aktuell ausgewählte Begriff im aktuellen Editor, wo er markiert ist, durch den ausgewählten Ersetzen-Begriff ersetzt.
  • Suche anhalten: Stoppt die aktuelle Suche, wenn eine solche läuft.

Wenn Sie die Escape-Taste drücken, während dieses Fenster geöffnet ist, wird das ausgewählte Feld gelöscht; wenn das Feld jedoch bereits leer ist, wird das Fenster geschlossen.

Alle Ergebnisse der Verwendung des Fensters "Suchen & Ersetzen" werden im Fenster "Suchausgabe" unten in der IDE angezeigt. Jedes Ergebnis kann doppelt angeklickt werden, um das Asset mit dem Begriff im aktuellen Arbeitsbereich zu öffnen und dorthin zu schwenken, um es zu bearbeiten.
