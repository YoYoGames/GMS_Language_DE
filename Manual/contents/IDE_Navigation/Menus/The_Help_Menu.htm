Das Hilfe-Menü

Dieses Menü bezieht sich auf das Abrufen von Hilfe bei der Verwendung von GameMaker Studio 2. Es enthält die folgenden Menüpunkte:

  • Startseite - Damit wird ein neuer Startseiten-Arbeitsbereich geöffnet.
  • Willkommensseite - Damit wird die anfängliche Willkommensseiten-Diashow in einem neuen Arbeitsbereich geöffnet.
  • Handbuch öffnen - Öffnet das Hilfedokument. Beachten Sie, dass auch das Anklicken der Hilfe-Schaltfläche in der Menüleiste das Dokument öffnet, ebenso wie das Drücken von .
  • Knowledge Base - Hier gelangen Sie zum GameMaker Studio 2 Help Center, in dem Sie Artikel über die Einrichtung der Plattform, bekannte Probleme, die Verwendung von GameMaker Studio 2 und eine Reihe anderer Dinge finden können Dinge finden.
  • Report A Bug - Sollten Sie Bugs finden oder Probleme bei der Verwendung von GameMaker Studio 2 haben, können Sie diese hier melden.
  • YoYoGames.com - Öffnet die Hauptseite vonYoYo Games.
  • GameMaker Community - Dies öffnet die GameMaker Community-Foren in Ihrem Browser. Beachten Sie, dass dies ein von der Community betriebenes Forum ist, wenn Sie also einen Fehlerbericht haben oder technische Unterstützung benötigen Wenn Sie also einen Fehlerbericht haben oder technische Unterstützung benötigen, sollten Sie zuerst die offizielle Support-Seite besuchen (siehe oben).
  • Release Notes - Dies öffnet die Release Notes für GameMaker Studio 2 in Ihrem Browser. Sollten Sie ein Update zurücksetzen müssen, können Sie Installationsprogramme für frühere Versionen erhalten, indem Sie den Link zur Versionsnummer auf dieser Seite klicken.
  • Aktuelle Runtime Release Notes - Da die Laufzeit unabhängig von der IDE aktualisieren kann, hat sie ihren eigenen Satz von Runtime Release Notes. Wenn Sie diese Option auswählen, werden sie in Ihrem Browser geöffnet, aber beachten Sie, dass die aktuelle Runtime möglicherweise nicht die ist, die Sie verwenden. Um herauszufinden, welche Runtime Sie verwenden, oder für Informationen zum Hinzufügen, Entfernen oder Aktivieren von Runtimes, lesen Sie bitte den Abschnitt Runtime-Voreinstellungen.
  • Erforderliche SDKs - Aufgrund des plattformübergreifenden Charakters von GameMaker Studio 2 sind eine Reihe von SDKs von Drittanbietern erforderlich. Wenn Sie auf diese Schaltfläche klicken, gelangen Sie zu einem Artikel im Support-Forum, in dem detailliert beschrieben wird, was Sie für die einzelnen Plattformen benötigen.
  • Projekt im Explorer öffnen - Damit wird der Speicherort des aktuell geladenen Projekts im OS-Dateiexplorer geöffnet.
  • Über - Öffnet das Fenster "Über GameMaker Studio 2", in dem Sie Versionsinformationen zum Produkt erhalten (wichtig für das Melden von Fehlern oder Problemen).
