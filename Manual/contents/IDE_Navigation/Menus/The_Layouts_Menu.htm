Das Menü Layouts

Dieses Menü bezieht sich auf das Speichern und Laden verschiedener IDE-Layouts. Sie können die IDE so einrichten, wie Sie es wünschen, wobei bestimmte Elemente (wie der Asset-Browser, oder die Ausgabefenster) anders als in der Voreinstellung angeordnet sind. So können Sie GameMaker Studio 2 an Ihre Arbeitsweise anpassen. Mit den Optionen in diesem Menü können Sie die aktuellen Projektlayouts als *.xml Datei speichern, die dann in jedes beliebige Projekt geladen werden kann, um die gleichen Layouts zu verwenden. Die Optionen in diesem Menü sind wie folgt:

  • Layout laden - Dies öffnet einen Datei-Explorer, in dem Sie eine zuvor gespeicherte Layout *.xml-Datei zum Laden auswählen können.
  • Layout speichern - Dadurch wird ein Datei-Explorer geöffnet, in dem Sie das aktuelle IDE-Layout als *.xml-Datei benennen und speichern können.
  • Layout zurücksetzen - Damit wird das aktuelle IDE-Layout auf die Standardeinstellungen zurückgesetzt.
