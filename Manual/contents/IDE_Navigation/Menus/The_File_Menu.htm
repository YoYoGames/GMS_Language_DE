Das Menü "Datei

Im Dateimenü können Sie Projekte ändern, speichern oder importieren sowie die Einstellungen der GameMaker Studio 2 IDE festlegen. Die verfügbaren Optionen sind:

  • Neues Projekt - Hiermit können Sie ein neues Projekt von Grund auf in GameMaker Studio 2 erstellen. Es öffnet den Dialog "Speichern" und fordert Sie auf, einen Speicherort für das neue Projekt zu wählen. Sobald Sie dem Projekt einen Namen gegeben haben und einen Speicherort angegeben haben, werden Sie zu einem neuen, leeren Arbeitsbereich geführt, um mit der Arbeit daran zu beginnen. Projektdateien werden mit der Dateierweiterung *.yyp gespeichert.
  • Projekt öffnen - Damit wird ein Datei-Explorer geöffnet, in dem Sie eine zuvor erstellte *.yyz Projektdatei zum Laden auswählen können. Bitte beachten Sie, dass beim Import eine Warnmeldung erscheint, wenn das Projekt mit einer niedrigeren Version als 2.3.0 erstellt wurde, Wenn Sie fortfahren, wird ein Datei-Explorer geöffnet, und Sie können dem Projekt einen aktualisierten Namen und Speicherort geben. Sie können dem Projekt einen aktualisierten Namen und einen Speicherort geben, damit das Originalprojekt nicht überschrieben wird. Sobald das Projekt in das neue Format 2.3.0+ konvertiert wurde, wird es in der IDE geöffnet und Sie erhalten eine Meldung, dass die Konvertierung erfolgreich war. Sie sollten den Kompatibilitätsbericht, der durch den Konvertierungsprozess erzeugt wurde, überprüfen und sehen, ob es etwas gibt, das Sie beheben müssen, bevor Sie versuchen, das Projekt auszuführen oder zu kompilieren. Beachten Sie, dass, wenn Sie wiederholt Fehler erhalten beim Versuch, ein Projekt zu öffnen, das mit einer Version vor 2.3.0 erstellt wurde, sollten Sie YoYo Games kontaktieren (über die Option "Fehler melden" im Hilfemenü) und die Originaldatei *.yyp, die Sie zu konvertieren versuchen, angeben.
  • Projekt importieren - Hier können Sie Projekte importieren, die zuvor als *.yyz Archiv exportiert wurden (siehe "Projekt exportieren", unten), oder - wenn Sie ein Projekt haben, das mit dem GameMaker: Studio 1.4 Software von YoYo Games erstellt wurde, können Sie diese Option wählen, um entweder die Datei *.gmx oder die Datei *.gmz aus dieser Produktlinie zu importieren. Wenn die *.yyz mit einer Version von GameMaker Studio 2 vor dem Update 2.3.0 erstellt wurde, muss sie - wie beim Öffnen eines Projekts - konvertiert werden, da sich das Format der Projektdatei mit dem Update geändert hat.

HINWEIS: Um ein früheres GameMaker-Projekt in GameMaker Studio 2 zu importieren, muss es mit der Version 1.4 erstellt worden sein, und selbst dann funktioniert es möglicherweise nicht direkt beim Import. Sehr viele Dinge sind anders zwischen GameMaker Studio 2 und GameMaker: Studio 1.4, und so müssen Ihre älteren Projekte möglicherweise etwas angepasst werden, damit sie funktionieren. Bitte lesen Sie den Artikel Portierung eines GM:S 1.4 Spiels nach GameMaker Studio 2 in der YoYo Games Knowledge Base und den Abschnitt des Handbuchs, der veraltete Funktionen und Kompatibilitätsscripts behandelt.

WICHTIG! Beim Importieren eines *.yyz -Projekts, das Font-Ressourcen enthält, müssen Sie die Fonts möglicherweise "neu generieren", indem Sie sie entfernen/neu hinzufügen oder die Schrift im Font-Editor ändern und dann sie wieder zurücksetzen. Das liegt daran, dass die vorhandenen Schriftinformationen beim Export von *.yyz beibehalten werden und beim späteren Import oder auf einem anderen Rechner das Projekt möglicherweise nicht die benötigten Schriften bereits installiert hat (oder eine andere Schriftart mit demselben Namen haben). Dies sollte nur durchgeführt werden, wenn Sie Probleme mit der Darstellung der Schriftart nach dem Import haben.

  • Letzte Projekte - Mit dieser Option können Sie zuletzt verwendete Projekte anzeigen. Wenn Sie diese Option auswählen, wird die Startseite wieder geöffnet (falls Sie sie geschlossen haben) und Sie können ein kürzlich verwendetes Projekt aus der dortigen Liste auswählen. Projekt aus der dortigen Liste auswählen.
  • Projekt speichern - Damit wird Ihr Spiel an dem Ort gespeichert, den Sie beim Erstellen des Projekts festgelegt haben (oder von dem aus Sie es geöffnet haben).
  • Projekt speichern unter - Dasselbe wie oben, nur dass Sie den Speicherort und den neuen Namen des Projekts angeben.
  • Projekt exportieren - Hiermit wird das Projekt als einzelne *.yyz Datei exportiert. Sehr nützlich zum Versenden von Kopien an andere Personen oder zum Speichern von Sicherungskopien.
  • Voreinstellungen - Wenn Sie dies auswählen, gelangen Sie zum Fenster "Voreinstellungen ", in dem Sie das Aussehen und die Bedienung von GameMaker Studio 2 einstellen können.
  • Neue IDE - Wenn Sie dies auswählen, wird eine neue Instanz der IDE zur Verwendung geöffnet (sie wird auf der Startseite geöffnet).
  • Mein Konto - Diese Option öffnet Ihren Standard-Webbrowser und führt Sie zu Ihrer YoYo-Kontoseite.
  • Abmelden - Wenn Sie dies wählen, werden Sie von GameMaker Studio 2 abgemeldet. Sie müssen sich mit der gleichen Lizenz (oder einer anderen) erneut anmelden, um das Produkt später verwenden zu können.

WICHTIG! Wenn Sie eine Testlizenz für GameMaker Studio 2 verwendet haben und dann eine der anderen Lizenzen (Desktop, Mobile, UWP, Web oder Konsole) kaufen, müssen Sie sich möglicherweise von der IDE abmelden und dann GameMaker Studio 2 neu starten und wieder anmelden, um sicherzustellen, dass die neue Lizenz korrekt angewendet wird (obwohl dies automatisch geschehen sollte).

  • Beenden - Beendet GameMaker Studio 2 auf dem Desktop.
